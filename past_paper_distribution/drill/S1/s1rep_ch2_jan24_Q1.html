Grouped frequency data: histogram, median, quartiles by linear interpolation, and skewness

Question

The table shows the time, $t$ minutes, taken by 120 students to complete an online test.

Time ($t$ minutes)Frequency
$10 \leq t < 20$8
$20 \leq t < 25$17
$25 \leq t < 30$35
$30 \leq t < 40$42
$40 \leq t < 60$18

(a) On graph paper, draw a histogram to represent these data.

(b) Use linear interpolation to estimate the median time.

(c) Use linear interpolation to estimate the interquartile range.

(d) Describe the skewness of the distribution. Give a reason for your answer.

[10 marks]

(a) [3 marks] · 3 steps

STEP 1/3 M1

For a histogram, area ∝ frequency, so we need frequency density:

$$\text{Frequency density} = \dfrac{\text{Frequency}}{\text{Class width}}$$

Calculate class widths and frequency densities:

ClassWidthFreqFreq Density
$10-20$108$8÷10=0.8$
$20-25$517$17÷5=3.4$
$25-30$535$35÷5=7$
$30-40$1042$42÷10=4.2$
$40-60$2018$18÷20=0.9$
Exam pitfall: with unequal class widths, you MUST use frequency density — not raw frequency.
$\boxed{\text{Frequency densities: } 0.8, 3.4, 7, 4.2, 0.9}$
↳ from STEP 1:From STEP 1: Frequency densities: $0.8, 3.4, 7, 4.2, 0.9$
STEP 2/3 A1

Draw histogram with:

  • Horizontal axis: time $t$ (minutes) from 10 to 60
  • Vertical axis: frequency density
  • Bars with NO gaps, heights matching calculated densities
  • Bar widths matching class intervals
Exam pitfall: bars must touch (continuous data) and use class BOUNDARIES on x-axis.
$\boxed{\text{Histogram drawn with correct frequency densities}}$
↳ from STEP 2:From STEP 2: Histogram structure established
STEP 3/3 A1

Check histogram requirements:

  • ✓ Correct scale on both axes with labels
  • ✓ All 5 bars drawn with correct widths
  • ✓ Heights: 0.8, 3.4, 7, 4.2, 0.9
  • ✓ No gaps between bars
Exam pitfall: marks lost for gaps between bars or plotting frequency instead of frequency density.
$\boxed{\text{Fully labelled histogram complete}}$

(b) [2 marks] · 2 steps

STEP 1/2 M1

Linear interpolation formula for median:

$$Q_2 = L + \dfrac{\frac{n}{2} - F}{f} \times w$$

Where: $L$ = lower boundary, $F$ = cumulative frequency before median class, $f$ = frequency of median class, $w$ = class width

Find median position: $\dfrac{n}{2} = \dfrac{120}{2} = 60$

Build cumulative frequencies:

  • $10-20$: CF = 8
  • $20-25$: CF = $8+17=25$
  • $25-30$: CF = $25+35=60$
  • $30-40$: CF = $60+42=102$

Position 60 falls in class $25 \leq t < 30$ (CF reaches exactly 60 at upper boundary)

Exam pitfall: median class contains the 60th item — check where CF first reaches or exceeds n/2.
$\boxed{\text{Median class: } 25 \leq t < 30}$
↳ from STEP 1:From STEP 1: Median class $25-30$, position 60, $F=25$, $f=35$
STEP 2/2 A1

Substitute into formula:

$$Q_2 = 25 + \dfrac{60 - 25}{35} \times 5$$

$$Q_2 = 25 + \dfrac{35}{35} \times 5$$

$$Q_2 = 25 + 1 \times 5$$

$$Q_2 = 25 + 5 = 30$$

Exam pitfall: use class boundaries (25, 30) not class limits; class width is $30-25=5$.
$\boxed{Q_2 = 30 \text{ minutes}}$

(c) [3 marks] · 3 steps

From (b): Cumulative frequencies: 8, 25, 60, 102, 120
STEP 1/3 M1

Find $Q_1$ position: $\dfrac{n}{4} = \dfrac{120}{4} = 30$

From cumulative frequencies: CF = 25 at end of $20-25$, CF = 60 at end of $25-30$

Position 30 falls in class $25 \leq t < 30$

Apply linear interpolation:

$$Q_1 = L + \dfrac{\frac{n}{4} - F}{f} \times w$$

$$Q_1 = 25 + \dfrac{30 - 25}{35} \times 5$$

$$Q_1 = 25 + \dfrac{5}{35} \times 5$$

$$Q_1 = 25 + \dfrac{25}{35} = 25 + 0.714...$$

$$Q_1 = 25.71 \text{ (2 d.p.)}$$

Exam pitfall: $Q_1$ uses position $n/4$, not $n/2$.
$\boxed{Q_1 = 25.71 \text{ minutes}}$
↳ from STEP 1:From STEP 1: $Q_1 = 25.71$
STEP 2/3 M1

Find $Q_3$ position: $\dfrac{3n}{4} = \dfrac{3 \times 120}{4} = 90$

From cumulative frequencies: CF = 60 at end of $25-30$, CF = 102 at end of $30-40$

Position 90 falls in class $30 \leq t < 40$

Apply linear interpolation:

$$Q_3 = L + \dfrac{\frac{3n}{4} - F}{f} \times w$$

$$Q_3 = 30 + \dfrac{90 - 60}{42} \times 10$$

$$Q_3 = 30 + \dfrac{30}{42} \times 10$$

$$Q_3 = 30 + \dfrac{300}{42} = 30 + 7.143...$$

$$Q_3 = 37.14 \text{ (2 d.p.)}$$

Exam pitfall: class $30-40$ has width 10, not 5 — different from previous classes.
$\boxed{Q_3 = 37.14 \text{ minutes}}$
↳ from STEP 2:From STEP 2: $Q_1 = 25.71$, $Q_3 = 37.14$
STEP 3/3 A1

Calculate interquartile range:

$$\text{IQR} = Q_3 - Q_1$$

$$\text{IQR} = 37.14 - 25.71$$

$$\text{IQR} = 11.43 \text{ minutes}$$

Exam pitfall: IQR = $Q_3 - Q_1$ (not $Q_1 - Q_3$); it must be positive.
$\boxed{\text{IQR} = 11.4 \text{ minutes (3 s.f.)}}$

(d) [2 marks] · 2 steps

From (b): $Q_2 = 30$; From (c): $Q_1 = 25.71$, $Q_3 = 37.14$
STEP 1/2 M1

Compare distances from median to quartiles:

$$Q_2 - Q_1 = 30 - 25.71 = 4.29$$

$$Q_3 - Q_2 = 37.14 - 30 = 7.14$$

Since $Q_3 - Q_2 > Q_2 - Q_1$:

The upper tail (from median to $Q_3$) is longer than the lower tail.

Exam pitfall: always compare $Q_2 - Q_1$ with $Q_3 - Q_2$, not the quartile values themselves.
$\boxed{Q_3 - Q_2 = 7.14 > Q_2 - Q_1 = 4.29}$
↳ from STEP 1:From STEP 1: Upper distance $(7.14)$ > Lower distance $(4.29)$
STEP 2/2 A1

Interpretation:

When $Q_3 - Q_2 > Q_2 - Q_1$, the distribution has a longer upper tail.

This indicates positive skewness (skewed to the right).

Alternative reasoning: The histogram shows a longer tail extending towards higher values (right side).

Exam pitfall: must give numerical evidence from quartiles — "the histogram looks skewed" alone scores 0 marks.
$\boxed{\text{Positive skew because } Q_3 - Q_2 > Q_2 - Q_1}$
