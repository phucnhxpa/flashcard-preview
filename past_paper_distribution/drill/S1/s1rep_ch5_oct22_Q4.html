Q4 · CDF and Expected Value

Question

The CDF of discrete random variable $W$ (taking only values 6, 7, 8) is:

$F(w) = \dfrac{(w+3)(w-1)}{77} \quad \text{for } w = 6, 7, 8$

Find $E(W)$. [4]

[4 marks]

[4 marks] · 2 steps

STEP 1/2 M2

Find probabilities from the CDF

Evaluate $F(w)$ at each value:

$F(6) = \dfrac{(6+3)(6-1)}{77} = \dfrac{9 \times 5}{77} = \dfrac{45}{77}$

$F(7) = \dfrac{(7+3)(7-1)}{77} = \dfrac{10 \times 6}{77} = \dfrac{60}{77}$

$F(8) = \dfrac{(8+3)(8-1)}{77} = \dfrac{11 \times 7}{77} = \dfrac{77}{77} = 1$

Find individual probabilities using $P(W=w) = F(w) - F(w-1)$:

$P(W=6) = F(6) = \dfrac{45}{77}$   (smallest value, $F(5)=0$)

$P(W=7) = F(7) - F(6) = \dfrac{60}{77} - \dfrac{45}{77} = \dfrac{15}{77}$

$P(W=8) = 1 - F(7) = 1 - \dfrac{60}{77} = \dfrac{17}{77}$

Check: $\frac{45+15+17}{77} = \frac{77}{77} = 1$ ✓ — probabilities must sum to 1.
$P(W=6)=\frac{45}{77},\quad P(W=7)=\frac{15}{77},\quad P(W=8)=\frac{17}{77}$
Carry-in from previous stepFrom STEP 1: $P(W=6)=\frac{45}{77}$, $P(W=7)=\frac{15}{77}$, $P(W=8)=\frac{17}{77}$
STEP 2/2 A2

Calculate $E(W) = \sum w \cdot P(W=w)$

$E(W) = 6 \times \dfrac{45}{77} + 7 \times \dfrac{15}{77} + 8 \times \dfrac{17}{77}$

$= \dfrac{270}{77} + \dfrac{105}{77} + \dfrac{136}{77}$

$= \dfrac{270 + 105 + 136}{77} = \dfrac{511}{77} = \dfrac{73}{11}$

Use exact fractions until the final step; $511 \div 77 = 6.636...$, which is $\frac{73}{11}$.
$\boxed{E(W) = \dfrac{73}{11} \approx 6.636}$
