Q4 · CDF to Probability Distribution to Expected Value

Question

The CDF of discrete random variable $W$ (taking only values 6, 7, 8) is:

$F(w) = \dfrac{(w+3)(w-1)}{77} \quad \text{for } w = 6, 7, 8$

Find $E(W)$. [4]

[4 marks]

Full Solution [4 marks] · 2 steps

CDF → Probability Distribution → E(W)
The Cumulative Distribution Function (CDF) of a discrete random variable \(W\) is: \[F(w) = P(W \leq w)\] It tells you the probability that \(W\) is at most \(w\).

To recover individual probabilities from the CDF: \[P(W = w) = F(w) - F(w_{\text{prev}})\] where \(w_{\text{prev}}\) is the previous possible value of \(W\). For the smallest value \(w_{\min}\): \[P(W = w_{\min}) = F(w_{\min}) \quad \text{(since nothing comes before it, \(F(w_{\min}-1) = 0\))}\] Then compute the expected value: \[E(W) = \sum_w w \cdot P(W = w)\] This is the weighted average of all possible values, weighted by their probabilities.
STEP 1

Evaluate \(F(w)\) at each value, then find individual probabilities using \(P(W=w) = F(w) - F(w-1)\)

Evaluate the CDF formula \(F(w) = \dfrac{(w+3)(w-1)}{77}\) at each value:

\[F(6) = \frac{(6+3)(6-1)}{77} = \frac{9 \times 5}{77} = \frac{45}{77}\] \[F(7) = \frac{(7+3)(7-1)}{77} = \frac{10 \times 6}{77} = \frac{60}{77}\] \[F(8) = \frac{(8+3)(8-1)}{77} = \frac{11 \times 7}{77} = \frac{77}{77} = 1 \checkmark\]

Now extract individual probabilities:

\[P(W=6) = F(6) - F(5) = \frac{45}{77} - 0 = \frac{45}{77} \quad \text{(6 is the min, so \(F(5)=0\))}\] \[P(W=7) = F(7) - F(6) = \frac{60}{77} - \frac{45}{77} = \frac{15}{77}\] \[P(W=8) = F(8) - F(7) = 1 - \frac{60}{77} = \frac{17}{77}\]
Check: probabilities sum to 1: \(\dfrac{45 + 15 + 17}{77} = \dfrac{77}{77} = 1\) ✓
Why is \(F(5) = 0\)? \(W\) can only take values 6, 7, 8 — it is impossible for \(W \leq 5\), so \(P(W \leq 5) = 0\).
\(P(W=6) = \dfrac{45}{77},\quad P(W=7) = \dfrac{15}{77},\quad P(W=8) = \dfrac{17}{77}\)
From STEP 1:$P(W=6)=\frac{45}{77}$, $P(W=7)=\frac{15}{77}$, $P(W=8)=\frac{17}{77}$
STEP 2

Apply \(E(W) = \sum w \cdot P(W=w)\) with the probability distribution

\[E(W) = 6 \times \frac{45}{77} + 7 \times \frac{15}{77} + 8 \times \frac{17}{77}\] \[= \frac{6 \times 45 + 7 \times 15 + 8 \times 17}{77}\] \[= \frac{270 + 105 + 136}{77} = \frac{511}{77}\]

Simplify: \(\gcd(511, 77) = 7\), so \(\dfrac{511}{77} = \dfrac{73}{11}\).

\[E(W) = \frac{73}{11} \approx 6.636\]
Strategy: Put everything over the common denominator 77 to add the fractions. Then compute \(270 + 105 + 136 = 511\). Finally simplify: \(511 = 7 \times 73\) and \(77 = 7 \times 11\), so \(\frac{511}{77} = \frac{73}{11}\). As a decimal: \(73 \div 11 = 6.636...\)
$\boxed{E(W) = \dfrac{73}{11} \approx 6.64}$
