Discrete random variable $X$ with CDF $F(x)$ — find probabilities, expectation and variance

Question

The discrete random variable $X$ has cumulative distribution function $F(x)$ defined by:

$F(x) = \begin{cases} 0 & x < 1 \\ \dfrac{x^2 + x}{20} & 1 \leq x \leq 4 \\ 1 & x > 4 \end{cases}$

where $x$ takes integer values only.

(a) Find the probability distribution of $X$.

(b) Find $E(X)$.

(c) Find $Var(X)$.

(d) Find $E(3X - 2)$.

[12 marks]

(a) [4 marks] · 2 steps

STEP 1/2 M1

Formula: For a CDF, $P(X = x) = F(x) - F(x-1)$

First evaluate $F(x)$ at each integer value $x = 1, 2, 3, 4$:

$F(1) = \dfrac{1^2 + 1}{20} = \dfrac{2}{20} = \dfrac{1}{10}$

$F(2) = \dfrac{2^2 + 2}{20} = \dfrac{6}{20} = \dfrac{3}{10}$

$F(3) = \dfrac{3^2 + 3}{20} = \dfrac{12}{20} = \dfrac{6}{10}$

$F(4) = \dfrac{4^2 + 4}{20} = \dfrac{20}{20} = 1$

Exam pitfall: $F(0) = 0$ since $x < 1$ gives $F(x) = 0$.
$\boxed{F(1) = \tfrac{1}{10}, F(2) = \tfrac{3}{10}, F(3) = \tfrac{6}{10}, F(4) = 1}$
↳ from STEP 1:From STEP 1: $F(1) = \tfrac{1}{10}, F(2) = \tfrac{3}{10}, F(3) = \tfrac{6}{10}, F(4) = 1$
STEP 2/2 A1

Now find individual probabilities using $P(X = x) = F(x) - F(x-1)$:

$P(X = 1) = F(1) - F(0) = \dfrac{1}{10} - 0 = \dfrac{1}{10}$

$P(X = 2) = F(2) - F(1) = \dfrac{3}{10} - \dfrac{1}{10} = \dfrac{2}{10} = \dfrac{1}{5}$

$P(X = 3) = F(3) - F(2) = \dfrac{6}{10} - \dfrac{3}{10} = \dfrac{3}{10}$

$P(X = 4) = F(4) - F(3) = 1 - \dfrac{6}{10} = \dfrac{4}{10} = \dfrac{2}{5}$

Exam pitfall: Always verify probabilities sum to 1: $\tfrac{1}{10} + \tfrac{2}{10} + \tfrac{3}{10} + \tfrac{4}{10} = 1$ ✓
$\boxed{P(X=1) = \tfrac{1}{10}, P(X=2) = \tfrac{1}{5}, P(X=3) = \tfrac{3}{10}, P(X=4) = \tfrac{2}{5}}$

(b) [3 marks] · 2 steps

From (a): $P(X=1) = \tfrac{1}{10}, P(X=2) = \tfrac{1}{5}, P(X=3) = \tfrac{3}{10}, P(X=4) = \tfrac{2}{5}$
STEP 1/2 M1

Formula: $E(X) = \sum x \cdot P(X = x)$

Substituting all values:

$E(X) = 1 \times \dfrac{1}{10} + 2 \times \dfrac{2}{10} + 3 \times \dfrac{3}{10} + 4 \times \dfrac{4}{10}$

$E(X) = \dfrac{1}{10} + \dfrac{4}{10} + \dfrac{9}{10} + \dfrac{16}{10}$

Exam pitfall: Write each term separately before adding — avoid arithmetic errors.
$\boxed{\text{Sum of terms} = \tfrac{1+4+9+16}{10}}$
↳ from STEP 1:From STEP 1: $E(X) = \dfrac{1+4+9+16}{10}$
STEP 2/2 A1

Completing the arithmetic:

$E(X) = \dfrac{30}{10} = 3$

Exam pitfall: Keep exact fractions throughout — do not round intermediate values.
$\boxed{E(X) = 3}$

(c) [3 marks] · 2 steps

From (a): $P(X=1) = \tfrac{1}{10}, P(X=2) = \tfrac{1}{5}, P(X=3) = \tfrac{3}{10}, P(X=4) = \tfrac{2}{5}$; From (b): $E(X) = 3$
STEP 1/2 M1

Formula: $Var(X) = E(X^2) - [E(X)]^2$

First find $E(X^2) = \sum x^2 \cdot P(X = x)$:

$E(X^2) = 1^2 \times \dfrac{1}{10} + 2^2 \times \dfrac{2}{10} + 3^2 \times \dfrac{3}{10} + 4^2 \times \dfrac{4}{10}$

$E(X^2) = \dfrac{1}{10} + \dfrac{8}{10} + \dfrac{27}{10} + \dfrac{64}{10}$

$E(X^2) = \dfrac{1 + 8 + 27 + 64}{10} = \dfrac{100}{10} = 10$

Exam pitfall: Use $x^2$ values (1, 4, 9, 16) not $x$ values when calculating $E(X^2)$.
$\boxed{E(X^2) = 10}$
↳ from STEP 1:From STEP 1: $E(X^2) = 10$; From (b): $E(X) = 3$
STEP 2/2 A1

Now apply the variance formula:

$Var(X) = E(X^2) - [E(X)]^2$

$Var(X) = 10 - 3^2$

$Var(X) = 10 - 9 = 1$

Exam pitfall: Remember to SQUARE $E(X)$ — a common error is to subtract $E(X)$ instead of $[E(X)]^2$.
$\boxed{Var(X) = 1}$

(d) [2 marks] · 2 steps

From (b): $E(X) = 3$
STEP 1/2 M1

Formula: For linear transformation $E(aX + b) = aE(X) + b$

Here $a = 3$ and $b = -2$

$E(3X - 2) = 3 \times E(X) + (-2)$

Exam pitfall: The constant $b$ is ADDED to the expectation, not multiplied.
$\boxed{E(3X - 2) = 3E(X) - 2}$
↳ from STEP 1:From STEP 1: $E(3X - 2) = 3E(X) - 2$; From (b): $E(X) = 3$
STEP 2/2 A1

Substituting $E(X) = 3$:

$E(3X - 2) = 3 \times 3 - 2$

$E(3X - 2) = 9 - 2 = 7$

Exam pitfall: If asked for $Var(3X - 2)$, remember $Var(aX + b) = a^2 Var(X)$ — the constant $b$ disappears.
$\boxed{E(3X - 2) = 7}$
