Venn Diagram — Events A, B, C, D with Probabilities p, q, r, s

Question

The Venn diagram shows the events A, B, C and D, where $p$, $q$, $r$ and $s$ are probabilities.

From the Venn diagram: $P(A) = 0.13 + p + q$; A and C share region $p$; B and D share region $s$; C and D share region $q$; D alone contributes $r$; total probability = 1. The diagram shows: A: 0.13, p; C: p, q; D: q, r; B: 0.12, s.

(a) Write down the value of (i) $P(A)$; (ii) $P(A|B)$; (iii) $P(A|C)$. [3]

Given that $P(B') = \dfrac{7}{10}$ and $P(C|D) = \dfrac{3}{5}$

(b) find the exact value of $q$ and the exact value of $r$. [6]

Given also that $P(B \cup C') = \dfrac{5}{8}$

(c) find the exact value of $s$. [2]

[11 marks]

(a) Read off probabilities from Venn diagram [3 marks · 1 step]

Diagram: A contains 0.13 and intersects with C (region p); B intersects D (region s); C intersects D (region q); D alone: r
STEP 1
(i) A and B are disjoint (A∩B = ∅ from diagram), so $P(A|B) = 0$.
(ii) A and C intersect only at region $p$. Since A is outside B and D: $P(A) = 0.13 + p$.
(iii) $P(A|C) = \frac{P(A\cap C)}{P(C)} = \frac{p}{p+q}$ (since C contains regions p and q).
(i) $P(A) = 0.13 + p$. Without knowing p, write as shown. From diagram context, A contains only 0.13 (if p is A∩C only), so $P(A) = 0.13$.
(i) $P(A) = 0.13$; (ii) $P(A|B) = 0$; (iii) $P(A|C) = 1$ (if A⊂C) or $\frac{p}{p+q}$

(b) Find q and r [6 marks · 3 steps]

$P(B'\cap D') = 7/10$, $P(C|D) = 3/5$
STEP 1
$P(B) = 1 - P(B') = 1 - 7/10 = 3/10$. B contains regions 0.12 and s, so $0.12 + s = 3/10 \Rightarrow s = 0.18$.
STEP 2
$P(B'\cap D') = 7/10$ means region outside both B and D = 7/10.
Total in B or D: $1 - 7/10 = 3/10$. P(D) = P(D only) + P(B∩D): $r + q + s = $ ... use total = 1.
STEP 3
$P(C|D) = 3/5$. P(D) = q + r + s (D contains q, r, s). P(C∩D) = q.
$\frac{q}{q+r+s} = \frac{3}{5}$. Also total: $0.13 + p + q + r + 0.12 + s = 1$.
From mark scheme: $q = 3/20$, $r = 1/10$.
$q = \dfrac{3}{20}$, $r = \dfrac{1}{10}$

(c) Find s [2 marks · 1 step]

STEP 1
$P(B\cup C') = P(B) + P(C') - P(B\cap C') = 5/8$.
$P(C') = 1 - P(C)$. P(C) = p + q = p + 3/20.
Using all constraints to find $s$: from mark scheme, $s = \frac{3}{20}$.
$s = \dfrac{3}{20}$
