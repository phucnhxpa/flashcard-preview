Conditional probability: Weather and cycling to work

Question

Each day Aamir travels to work. When it rains he travels to work by bus. When it does not rain he cycles to work or he travels to work by bus.

The probability that it rains on any given day is $0.3$

When it does not rain, the probability that Aamir cycles to work is $0.6$

(a) Draw a tree diagram to represent this information.

(b) Find the probability that on a randomly chosen day Aamir cycles to work.

Given that Aamir cycled to work on Monday,

(c) find the probability that it rained on Monday.

Given that Aamir did not cycle to work on Tuesday,

(d) find the probability that it rained on Tuesday.

[10 marks]

(a) [3 marks] · 3 steps

STEP 1/3 B1

First branch: Rain outcomes

$P(\text{Rain}) = 0.3$

$P(\text{No Rain}) = 1 - 0.3 = 0.7$

Exam pitfall: probabilities on branches from same node must sum to 1.
$\boxed{P(R) = 0.3, \; P(R') = 0.7}$
↳ from STEP 1:From STEP 1: $P(R) = 0.3$, $P(R') = 0.7$
STEP 2/3 B1

Second branches from Rain: When it rains, Aamir always takes the bus

$P(\text{Bus} | \text{Rain}) = 1$

$P(\text{Cycle} | \text{Rain}) = 0$

Exam pitfall: read carefully — "when it rains he travels by bus" means P(Cycle|Rain) = 0, not just unlikely.
$\boxed{P(B|R) = 1, \; P(C|R) = 0}$
↳ from STEP 2:From STEP 2: Rain branch complete with $P(B|R) = 1$
STEP 3/3 B1

Second branches from No Rain:

$P(\text{Cycle} | \text{No Rain}) = 0.6$

$P(\text{Bus} | \text{No Rain}) = 1 - 0.6 = 0.4$

Complete Tree Diagram:

                    ┌─ Cycle (0) ─→ 0.3 × 0 = 0
         ┌─ Rain (0.3)
         │          └─ Bus (1) ─→ 0.3 × 1 = 0.3
    Start
         │          ┌─ Cycle (0.6) ─→ 0.7 × 0.6 = 0.42
         └─ No Rain (0.7)
                    └─ Bus (0.4) ─→ 0.7 × 0.4 = 0.28
Exam pitfall: label ALL branches with probabilities and ensure each pair sums to 1.
$\boxed{\text{Tree diagram complete with all probabilities}}$

(b) [2 marks] · 2 steps

From (a): Tree diagram with $P(R) = 0.3$, $P(R') = 0.7$, $P(C|R') = 0.6$, $P(C|R) = 0$
STEP 1/2 M1

Formula: Total probability

$P(C) = P(C \cap R) + P(C \cap R')$

$P(C) = P(R) \times P(C|R) + P(R') \times P(C|R')$

Exam pitfall: use multiplication along branches, then add the separate paths to the same outcome.
$\boxed{P(C) = P(R) \cdot P(C|R) + P(R') \cdot P(C|R')}$
↳ from STEP 1:From STEP 1: $P(C) = P(R) \cdot P(C|R) + P(R') \cdot P(C|R')$
STEP 2/2 A1

Substituting values from tree diagram:

$P(C) = 0.3 \times 0 + 0.7 \times 0.6$

$P(C) = 0 + 0.42$

$P(C) = 0.42$

Exam pitfall: the Rain→Cycle branch contributes 0 because Aamir never cycles when it rains.
$\boxed{P(C) = 0.42}$

(c) [2 marks] · 2 steps

From (a): $P(R) = 0.3$, $P(C|R) = 0$; From (b): $P(C) = 0.42$
STEP 1/2 M1

Formula: Bayes' theorem / conditional probability

$P(R|C) = \dfrac{P(R \cap C)}{P(C)}$

We need $P(R \cap C) = P(R) \times P(C|R)$

$P(R \cap C) = 0.3 \times 0 = 0$

Exam pitfall: "Given that Aamir cycled" tells you the condition — what you're dividing by.
$\boxed{P(R \cap C) = 0}$
↳ from STEP 1:From STEP 1: $P(R \cap C) = 0$, and $P(C) = 0.42$
STEP 2/2 A1

Substituting into conditional probability formula:

$P(R|C) = \dfrac{0}{0.42}$

$P(R|C) = 0$

This makes sense: if Aamir cycled, it definitely did not rain (since he only cycles when it doesn't rain).

Exam pitfall: a probability of 0 is a valid answer — don't second-guess logical conclusions.
$\boxed{P(R|C) = 0}$

(d) [3 marks] · 3 steps

From (a): $P(R) = 0.3$, $P(B|R) = 1$, $P(B|R') = 0.4$; From (b): $P(C) = 0.42$
STEP 1/3 M1

First find $P(C')$ — probability of NOT cycling (i.e., taking the bus)

$P(C') = 1 - P(C)$

$P(C') = 1 - 0.42 = 0.58$

Alternatively, from tree: $P(C') = P(R) \times P(B|R) + P(R') \times P(B|R')$

$P(C') = 0.3 \times 1 + 0.7 \times 0.4 = 0.3 + 0.28 = 0.58$ ✓

Exam pitfall: "did not cycle" means took the bus — verify using complement OR tree paths.
$\boxed{P(C') = 0.58}$
↳ from STEP 1:From STEP 1: $P(C') = 0.58$
STEP 2/3 M1

Formula: Conditional probability

$P(R|C') = \dfrac{P(R \cap C')}{P(C')}$

Find $P(R \cap C')$: Rain AND took bus

$P(R \cap C') = P(R) \times P(B|R)$

$P(R \cap C') = 0.3 \times 1 = 0.3$

Exam pitfall: $P(R \cap C')$ means the Rain-Bus path on the tree, not Rain alone.
$\boxed{P(R \cap C') = 0.3}$
↳ from STEP 2:From STEP 2: $P(R \cap C') = 0.3$, $P(C') = 0.58$
STEP 3/3 A1

Substituting into conditional probability formula:

$P(R|C') = \dfrac{0.3}{0.58}$

$P(R|C') = \dfrac{30}{58} = \dfrac{15}{29}$

As a decimal: $P(R|C') \approx 0.517$ (3 s.f.)

Exam pitfall: give exact fraction unless decimals specified — $\frac{15}{29}$ is exact, 0.517 is approximate.
$\boxed{P(R|C') = \dfrac{15}{29} \approx 0.517}$
