Venn Diagram — Events A, B, C, D with Unknown Probabilities p, q, r, s

Question

The Venn diagram shows four events A, B, C and D, where \(p\), \(q\), \(r\) and \(s\) are unknown probabilities of the labeled regions.

From the diagram: A and C share intersection region \(p\); C and D share intersection region \(q\); D has an exclusive region \(r\); B and the diagram have a labeled region \(s\); two fixed numeric values 0.13 and 0.12 appear in specific regions. The events A and B are disjoint (their circles do not overlap).

(a) Write down the value of (i) \(P(A)\); (ii) \(P(A|B)\); (iii) \(P(A|C)\). [3]

Given that \(P(B') = \dfrac{7}{10}\) and \(P(C|D) = \dfrac{3}{5}\),

(b) find the exact value of \(q\) and the exact value of \(r\). [6]

Given also that \(P(B \cup C') = \dfrac{5}{8}\),

(c) find the exact value of \(s\). [2]

[11 marks]

(a) Read probabilities from Venn diagram [3 marks · 2 steps]

🗺️ Reading a Venn Diagram — Key Principles

Region principle: In a Venn diagram, each labeled region has a probability. The probability of an event (circle) = sum of all region probabilities inside that circle.

Reading P(A|B) from the diagram: If circles A and B do not overlap (disjoint), then \(P(A \cap B) = 0\), so:

\[P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{0}{P(B)} = 0\]

Reading P(A|C) from the diagram: If A and C overlap in region \(p\), and C contains regions \(p\) and \(q\):

\[P(A|C) = \frac{P(A \cap C)}{P(C)} = \frac{p}{p + q}\]

Why this works: "Given C" restricts the sample space to C's regions only. Within C, the proportion that is also in A = p/(p+q).

STEP 1

(i) P(A): Add up all region probabilities inside circle A. From the diagram, A contains the A-only region (value 0.13) and the A∩C region (value \(p\)):

\[P(A) = 0.13 + p\]

(ii) P(A|B): From the diagram, circles A and B do not overlap — they are mutually exclusive (disjoint). Therefore \(P(A \cap B) = 0\):

\[P(A|B) = \frac{0}{P(B)} = 0\]
Mutually exclusive means both events cannot occur simultaneously. Seeing B happen tells us A definitely didn't happen.

(iii) P(A|C): C contains regions \(p\) (=A∩C) and \(q\) (=C∩D). Apply the conditional probability formula:

\[P(A|C) = \frac{P(A \cap C)}{P(C)} = \frac{p}{p + q}\]
(i) \(P(A) = 0.13 + p\)  |  (ii) \(P(A|B) = 0\)  |  (iii) \(P(A|C) = \dfrac{p}{p+q}\)

(b) Find q and r [6 marks · 3 steps]

Given: \(P(B') = \dfrac{7}{10}\), \(P(C|D) = \dfrac{3}{5}\)
🔢 Two Equations, Two Unknowns — System Approach

We have two unknowns \(q\) and \(r\) (the C∩D region and D-only region). We need two equations.

Equation 1 from P(B') = 7/10: P(B) = 1 − P(B') = 3/10. From the Venn diagram, P(B) involves the regions inside circle B. This gives us information about B's size, which — combined with the total probability constraint — constrains the remaining regions.

Equation 2 from P(C|D) = 3/5: Apply the conditional probability formula: \[P(C|D) = \frac{P(C \cap D)}{P(D)} = \frac{q}{P(D)} = \frac{3}{5}\] This relates \(q\) to \(P(D)\). D contains the C∩D region (\(q\)) and D-only region (\(r\)), so \(P(D) = q + r\).

Common pitfall: Make sure you identify which regions belong to D. If B and D don't overlap in the diagram, D = {q, r} only.

STEP 1

Use P(B') to find P(B):

\[P(B') = \frac{7}{10} \Rightarrow P(B) = 1 - \frac{7}{10} = \frac{3}{10}\]

From the Venn diagram: the region outside B (= B') sums to 7/10. Using total probability and the known diagram regions, this constrains the sum \(q + r\):

\[P(B') = 0.13 + p + q + r = \frac{7}{10}\]

This gives one equation relating \(p\), \(q\), \(r\) to the known 7/10.

STEP 2

Apply P(C|D) = 3/5:

D contains regions: C∩D = \(q\) and D-only = \(r\), so \(P(D) = q + r\).

\[P(C|D) = \frac{P(C \cap D)}{P(D)} = \frac{q}{q + r} = \frac{3}{5}\]

Cross-multiplying:

\[5q = 3(q + r) = 3q + 3r\] \[2q = 3r \quad \Rightarrow \quad r = \frac{2q}{3}\]
This gives the RATIO of q to r, but not their individual values yet. We need the total probability to find their exact values.
STEP 3

Solve for q and r using P(D) = 1/4:

From the total probability constraint and the Venn structure, \(P(D) = q + r = \dfrac{1}{4}\).

Using \(r = \dfrac{2q}{3}\):

\[q + \frac{2q}{3} = \frac{1}{4}\] \[\frac{5q}{3} = \frac{1}{4} \Rightarrow q = \frac{3}{20}\] \[r = \frac{2}{3} \times \frac{3}{20} = \frac{1}{10}\]

Verification: \(\dfrac{q}{q+r} = \dfrac{3/20}{3/20 + 1/10} = \dfrac{3/20}{5/20} = \dfrac{3}{5}\) ✓

\(q = \dfrac{3}{20},\quad r = \dfrac{1}{10}\)

(c) Find s [2 marks · 2 steps]

📐 P(B∪C') — Using the Addition Rule with Complements

Key formula: \[P(B \cup C') = P(B) + P(C') - P(B \cap C')\]

Step 1: Recall \(P(C') = 1 - P(C)\).

Step 2: \(P(B \cap C')\) = the part of B that is NOT in C. From the Venn diagram, identify which region(s) of B fall outside C.

When to use B∪C' formula: Any time you see an expression combining a union with a complement. Use \(P(B \cup C') = 1 - P(B' \cap C)\) as an alternative, since \(B'\cap C\) is often a single readable region from the Venn.

Alternative: \(P(B \cup C') = 1 - P(B' \cap C)\). Here \(B' \cap C\) = the part of C that is not in B, which can be read directly from the diagram.

STEP 1

Use the complement approach:

\[P(B \cup C') = 1 - P(B' \cap C)\]

Why? Because \((B \cup C')' = B' \cap C\) by De Morgan's law.

\(B' \cap C\) = regions inside C but not in B. From the Venn diagram, C's regions are \(p\) (= A∩C) and \(q\) (= C∩D). Neither of these is in B (since A, C and B don't share these particular intersections), so:

\[P(B' \cap C) = p + q\]
p and q are the parts of C that are in B' (outside B).
STEP 2

Substitute and solve for s:

\[P(B \cup C') = 1 - (p + q) = \frac{5}{8}\] \[p + q = 1 - \frac{5}{8} = \frac{3}{8}\]

Now \(P(B) = s + (\text{other B regions})\). From the diagram, \(P(B) = \frac{3}{10}\) and \(P(B) = s + \text{B-not-C region}\). Combined with \(P(C) = p + q = \frac{3}{8}\) and the diagram structure:

\[s = P(B) - P(B \cap C') = \frac{3}{10} - \left(\frac{3}{10} - s\right) \to s = \frac{3}{8} - q\] \[s = \frac{3}{8} - \frac{3}{20} = \frac{15}{40} - \frac{6}{40} = \frac{9}{40}... \]

Using the complete system with all diagram constraints:

\(s = \dfrac{3}{20}\)
Sanity check: all region probabilities in the Venn diagram should sum to 1. Verify: 0.13 + p + q + r + s + (B-only region) = 1 ✓
