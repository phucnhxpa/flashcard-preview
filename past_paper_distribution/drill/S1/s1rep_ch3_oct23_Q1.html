Sally's Game — Win/Lose, Tree Diagram & Conditional Probability

Question

Sally plays a game in which she can either win or lose.

A turn consists of up to 3 games. On each turn Sally plays the game up to 3 times. If she wins the first 2 games or loses the first 2 games, then she will not play the 3rd game.

  • The probability that Sally wins the first game in a turn is 0.7
  • If Sally wins a game the probability that she wins the next game is 0.6
  • If Sally loses a game the probability that she wins the next game is 0.2

(a) Use this information to complete the tree diagram. [3]

(b) Find the probability that Sally wins the first 2 games in a turn. [2]

(c) Find the probability that Sally wins exactly 2 games in a turn. [2]

Given that Sally wins 2 games in a turn,

(d) find the probability that she won the first 2 games. [2]

Given that Sally won the first game in a turn,

(e) find the probability that she won 2 games. [2]

[11 marks]

(a) Complete tree diagram [3 marks · 1 step]

P(win 1st)=0.7; P(win next|won)=0.6; P(win next|lost)=0.2
STEP 1
1st game: W(0.7), L(0.3)
2nd game: W→W(0.6), W→L(0.4); L→W(0.2), L→L(0.8)
3rd game (only if 1st≠2nd): WL→W(0.6),WL→L(0.4); LW→W(0.6),LW→L(0.4)
If WW or LL after 2 games, stop — no 3rd game.
Branches: WW(stops), WLW, WLL, LWW, LWL, LL(stops)

(b) P(wins first 2 games) [2 marks · 1 step]

STEP 1
P(WW) = P(W1st)×P(W|won) = 0.7 × 0.6 = 0.42
$P(\text{wins first 2}) = 0.42$

(c) P(wins exactly 2 games) [2 marks · 1 step]

STEP 1
"Exactly 2 wins" can happen as: WW (stop, 2 wins), WLW (2 wins), LWW (stop after game 2? No: LW continues to 3rd). Wait: WW→stop (2 wins ✓); LL→stop (0 wins); WLW (2 wins ✓); LWW→stop (2 wins ✓). Also LWL = 1 win, WLL = 1 win.
P(WW) = 0.42; P(WLW) = 0.7×0.4×0.6 = 0.168; P(LWW) = 0.3×0.2×0.6 = 0.036
P(exactly 2) = 0.42+0.168+0.036 = ... but WW has 2 wins and stops, LWW has 2 wins and stops.
P(exactly 2) = P(WLW)+P(LWW) = 0.168+0.036 = 0.204? Or include WW?
WW = 2 wins. P(exactly 2) = P(WW)+P(WLW)+P(LWW) = 0.42+0.168+0.036 = 0.624
$P(\text{exactly 2 wins}) = 0.624$

(d) P(won first 2 | won exactly 2) [2 marks · 1 step]

STEP 1
$P(\text{first 2 won} | \text{exactly 2}) = \frac{P(WW)}{P(\text{exactly 2})} = \frac{0.42}{0.624} = \frac{420}{624} = \frac{35}{52}$
$P = \dfrac{35}{52} \approx 0.673$

(e) P(won 2 games | won 1st game) [2 marks · 1 step]

STEP 1
Given won 1st: branches WW(0.42) or WLW(0.168) or WLL(0.7×0.4×0.4=0.112). "Won 2 games given won 1st" = WW+WLW.
$P(\text{1st won}) = 0.7$. $P(\text{won 2} \cap \text{1st won}) = P(WW)+P(WLW) = 0.42+0.168 = 0.588$
$P(\text{2 wins}|\text{1st won}) = 0.588/0.7 = 0.84$
$P = 0.84$
