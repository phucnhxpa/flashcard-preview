Sally's Game — Win/Lose, Tree Diagram & Conditional Probability

Question

Sally plays a game in which she can either win (W) or lose (L) each individual game.

A turn consists of up to 3 games. On each turn Sally plays up to 3 times. If she wins the first 2 games OR loses the first 2 games, she does not play the 3rd game.

  • The probability that Sally wins the first game in a turn is 0.7
  • If Sally wins a game, the probability she wins the next game is 0.6
  • If Sally loses a game, the probability she wins the next game is 0.2

(a) Use this information to complete the tree diagram. [3]

(b) Find the probability that Sally wins the first 2 games in a turn. [2]

(c) Find the probability that Sally wins exactly 2 games in a turn. [2]

Given that Sally wins exactly 2 games in a turn,

(d) find the probability that she won the first 2 games. [2]

Given that Sally won the first game in a turn,

(e) find the probability that she won exactly 2 games. [2]

[11 marks]

(a) Complete tree diagram [3 marks · 2 steps]

P(W₁) = 0.7; P(W|prev W) = 0.6; P(W|prev L) = 0.2
🌳 Tree Diagram Structure — Variable-Length Paths

Key rules:

  • Multiply along branches: Each branch shows a conditional probability. The overall path probability = product of branch probabilities.
  • Add across paths: To find P(outcome), sum the probabilities of ALL paths leading to that outcome.

This tree has variable depth — some paths END after 2 games (if WW or LL), others continue to a 3rd game (if WL or LW). This is because winning or losing twice in a row ends the turn early.

Branch probabilities depend on the previous result — this is directly stated in the problem (conditional probabilities). The 3rd game branches use the same conditional probabilities as game 2.

All possible paths: WW (stop), WL→W (WLW), WL→L (WLL), LW→W (LWW), LW→L (LWL), LL (stop). That's 6 terminal outcomes.

STEP 1

Game 1 and Game 2 branches:

Game 1: P(W₁) = 0.7, P(L₁) = 0.3

Game 2 — after W₁: P(W₂|W₁) = 0.6, P(L₂|W₁) = 0.4
→ Path WW: STOP (2 wins → turn ends)
→ Path WL: CONTINUE to game 3

Game 2 — after L₁: P(W₂|L₁) = 0.2, P(L₂|L₁) = 0.8
→ Path LW: CONTINUE to game 3
→ Path LL: STOP (2 losses → turn ends)

WHY stop after WW or LL? The rules say so — these are the early-stopping conditions. The tree must reflect this by having no 3rd-level branches after WW or LL.
STEP 2

Game 3 branches (only for WL and LW paths):

After WL (last result was L): P(W₃|L) = 0.2, P(L₃|L) = 0.8
→ Paths: WLW (prob = 0.7 × 0.4 × 0.2 = 0.056), WLL (prob = 0.7 × 0.4 × 0.8 = 0.224)

After LW (last result was W): P(W₃|W) = 0.6, P(L₃|W) = 0.4
→ Paths: LWW (prob = 0.3 × 0.2 × 0.6 = 0.036), LWL (prob = 0.3 × 0.2 × 0.4 = 0.024)

6 terminal paths: WW (0.42), WLW (0.056), WLL (0.224), LWW (0.036), LWL (0.024), LL (0.24)
Check: 0.42 + 0.056 + 0.224 + 0.036 + 0.024 + 0.24 = 1.000 ✓

(b) P(wins first 2 games) [2 marks · 1 step]

🎯 "Wins First 2 Games" = Path WW Only

"Wins the first 2 games" means: Game 1 = W AND Game 2 = W. This is exactly one path: WW.

Multiply along branches (the events are sequential and each branch is the conditional probability given the previous result):

\[P(WW) = P(W_1) \times P(W_2 \mid W_1)\]

Note: "Wins first 2" does NOT ask about game 3, and since WW stops the turn, there is no game 3 anyway.

STEP 1

Multiply the two branch probabilities along the WW path:

\[P(\text{wins first 2}) = P(WW) = P(W_1) \times P(W_2 \mid W_1)\] \[= 0.7 \times 0.6 = 0.42\]
\(P(\text{wins first 2 games}) = 0.42\)

(c) P(wins exactly 2 games) [2 marks · 2 steps]

🎯 "Exactly 2 Wins" — Which Paths Have Exactly 2 Wins?

"Exactly 2 wins" means the total count of W's in a turn = 2. List all 6 terminal paths and count wins in each:

  • WW: 2 wins ✓ (turn stops here)
  • WLW: 2 wins ✓
  • WLL: 1 win ✗
  • LWW: 2 wins ✓ (turn stops after 3rd game? No — LW means we play game 3, LWW = 2 wins in 3 games)
  • LWL: 1 win ✗
  • LL: 0 wins ✗ (turn stops here)

So "exactly 2 wins" = WW OR WLW OR LWW — these are mutually exclusive paths → ADD their probabilities.

Common pitfall: Don't confuse "wins exactly 2 games" with "wins the first 2 games." WLW and LWW also give 2 total wins but DON'T win the first 2.

STEP 1

Identify and compute all paths with exactly 2 wins:

\[P(WW) = 0.7 \times 0.6 = 0.42\] \[P(WLW) = 0.7 \times 0.4 \times 0.2 = 0.056\] \[P(LWW) = 0.3 \times 0.2 \times 0.6 = 0.036\]
For LWW: L first (prob 0.3), then W (prob 0.2 given prev L), then W (prob 0.6 given prev W). Three sequential multiplications.
STEP 2

Add the three mutually exclusive paths:

\[P(\text{exactly 2 wins}) = P(WW) + P(WLW) + P(LWW)\] \[= 0.42 + 0.056 + 0.036\] \[= 0.512\]
\(P(\text{exactly 2 wins}) = 0.512\)
Sanity check: P(exactly 0 wins) = P(LL) = 0.3×0.8 = 0.24; P(exactly 1 win) = P(WLL)+P(LWL) = 0.224+0.024 = 0.248. Total = 0.24+0.248+0.512 = 1.000 ✓

(d) P(won first 2 | exactly 2 wins) [2 marks · 2 steps]

📐 Conditional Probability — P(A|B) = P(A∩B) / P(B)

Formula: \[P(A \mid B) = \frac{P(A \cap B)}{P(B)}\]

Why it works: "Given B happened" restricts our sample space to only outcomes where B occurs. Within that restricted space, we ask what fraction is also A. Dividing by P(B) re-scales so the probabilities sum to 1.

In this problem:
A = "won the first 2 games" (= path WW)
B = "exactly 2 wins" (= paths WW, WLW, LWW)
A∩B = "won first 2 AND had exactly 2 wins" = just path WW (since WW ⊂ "exactly 2 wins")

When to use: Any time the question says "given that…" before asking for a probability.

STEP 1

Identify A∩B:

"Won first 2" AND "exactly 2 wins" = path WW only. (WLW and LWW don't have the first 2 as wins.)

\[P(A \cap B) = P(WW) = 0.42\] \[P(B) = P(\text{exactly 2 wins}) = 0.512\]
STEP 2

Apply the formula:

\[P(\text{won first 2} \mid \text{exactly 2}) = \frac{P(WW)}{P(\text{exactly 2})} = \frac{0.42}{0.512}\]

Convert to fractions: \(\dfrac{0.42}{0.512} = \dfrac{420}{512} = \dfrac{105}{128}\)

\(P = \dfrac{105}{128} \approx 0.820\)
Makes sense: given Sally won exactly 2 games, the WW path (prob 0.42) is far more likely than WLW (0.056) or LWW (0.036), so P ≈ 0.82 is reasonable.

(e) P(exactly 2 wins | won 1st game) [2 marks · 2 steps]

📐 Conditional Probability — Restricting to "Won 1st Game" Branches

Formula: \[P(\text{exactly 2 wins} \mid W_1) = \frac{P(\text{exactly 2 wins} \cap W_1)}{P(W_1)}\]

Given W₁ occurred: We only consider paths that start with W. The relevant paths from the tree are:

  • WW: 2 wins ✓ (starts with W)
  • WLW: 2 wins ✓ (starts with W)
  • WLL: 1 win ✗ (starts with W, but only 1 win)

"Exactly 2 wins AND won 1st game" = WW or WLW (not LWW, because LWW starts with a loss).

Denominator: P(W₁) = 0.7 (directly from the given probabilities).

STEP 1

Find P(exactly 2 wins ∩ won 1st game):

Paths that start with W and have exactly 2 wins: WW and WLW.

\[P(\text{2 wins} \cap W_1) = P(WW) + P(WLW)\] \[= 0.42 + 0.056 = 0.476\]
LWW is excluded because it starts with a LOSS, not a win — it does not satisfy "won 1st game."
STEP 2

Apply the conditional probability formula:

\[P(\text{exactly 2 wins} \mid W_1) = \frac{0.476}{P(W_1)} = \frac{0.476}{0.7}\] \[= 0.68\]
\(P(\text{exactly 2 wins} \mid \text{won 1st game}) = 0.68\)
Sanity check: given won game 1 (prob 0.7), the paths from W₁ are WW (0.42), WLW (0.056), WLL (0.224). These sum to 0.7 ✓. Of these, WW+WLW = 0.476, giving 0.476/0.7 = 0.68 ✓.
