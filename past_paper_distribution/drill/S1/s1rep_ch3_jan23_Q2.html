Two Bags X & Y — Marbles, Tree Diagram & Conditional Probability

Question

Two bags, X and Y, each contain green marbles (G) and blue marbles (B) only.

  • Bag X contains 5 green marbles and 4 blue marbles
  • Bag Y contains 6 green marbles and 5 blue marbles

A marble is selected at random from bag X and placed in bag Y.
A second marble is selected at random from bag X and placed in bag Y.
A third marble is then selected, this time from bag Y.

(a) Use this information to complete the tree diagram. [3]

(b) Find the probability that the 2 marbles selected from bag X are of different colours. [2]

(c) Find the probability that all 3 marbles selected are the same colour. [2]

Given that all three marbles selected are the same colour,

(d) find the probability that they are all green. [3]

[10 marks]

(a) Tree diagram [3 marks · 2 steps]

🌳 Tree Diagram — Core Rules

Rule 1 — Multiply along branches: The probability of following a specific path (e.g. G then B then G) equals the product of the branch probabilities along that path. This is because each draw is a sequential conditional event.

Rule 2 — Add across branches: To find the probability of an outcome that can happen in multiple ways (e.g. "2 greens"), add up the path probabilities for each distinct path that leads to that outcome.

This tree has 3 levels: Draw 1 from X → Draw 2 from X → Draw 3 from Y.
Key subtlety: each marble moved from X to Y changes the composition of both bags — X shrinks from 9 to 8 balls, Y grows from 11 to 13 balls (receiving 2 balls total).

STEP 1

Draws from Bag X (levels 1 and 2):

Bag X starts with 5G + 4B = 9 total.

Level 1 (1st draw from X): \(P(G_1) = \dfrac{5}{9},\quad P(B_1) = \dfrac{4}{9}\)

After draw 1, one marble is removed from X — now X has 8 balls remaining.

Level 2 (2nd draw from X): probabilities depend on what was drawn first.

  • If 1st was G: X has 4G, 4B left → \(P(G_2|G_1)=\frac{4}{8}=\frac{1}{2},\quad P(B_2|G_1)=\frac{4}{8}=\frac{1}{2}\)
  • If 1st was B: X has 5G, 3B left → \(P(G_2|B_1)=\frac{5}{8},\quad P(B_2|B_1)=\frac{3}{8}\)
WHY different? Without replacement within X — removing a marble changes what's left. The denominator drops from 9 to 8.
STEP 2

Draw from Bag Y (level 3):

Y starts with 6G + 5B = 11 balls. Two marbles from X are transferred to Y, so Y ends up with 13 balls total. The colour composition of Y depends on the 2 balls transferred:

  • GG transferred: Y = 6+2 G, 5 B = 8G 5B (13 total) → \(P(G_3|GG)=\tfrac{8}{13},\ P(B_3|GG)=\tfrac{5}{13}\)
  • GB or BG transferred: Y = 7G, 6B (13 total) → \(P(G_3)=\tfrac{7}{13},\ P(B_3)=\tfrac{6}{13}\)
  • BB transferred: Y = 6G, 7B (13 total) → \(P(G_3|BB)=\tfrac{6}{13},\ P(B_3|BB)=\tfrac{7}{13}\)
Tree complete: 3 levels, 6 terminal branches (GGG, GGB, GBG, GBB, BGG, BGB, BBG, BBB — noting GB and BG share the same Y composition).

(b) P(2 marbles from X are different colours) [2 marks · 2 steps]

🔀 "Different Colours" — Addition of Mutually Exclusive Paths

"Different colours" from X means exactly one of two orderings: Green then Blue (GB) OR Blue then Green (BG). These paths are mutually exclusive (can't both happen at once), so we add their probabilities.

Each path probability = multiply branch probabilities along that path (sequential rule).

Common pitfall: Don't forget there are two orderings — (G,B) and (B,G) — each contributes.

STEP 1

Identify the two different-colour paths and compute each:

\[P(GB) = P(G_1) \times P(B_2|G_1) = \frac{5}{9} \times \frac{4}{8} = \frac{20}{72}\] \[P(BG) = P(B_1) \times P(G_2|B_1) = \frac{4}{9} \times \frac{5}{8} = \frac{20}{72}\]
STEP 2

Add the two mutually exclusive paths:

\[P(\text{different}) = P(GB) + P(BG) = \frac{20}{72} + \frac{20}{72} = \frac{40}{72}\]
\(P(\text{different colours}) = \dfrac{5}{9}\)
Sanity check: \(\frac{40}{72} = \frac{5}{9}\). Also P(same colour from X) = P(GG)+P(BB) = \(\frac{20}{72}+\frac{12}{72}=\frac{32}{72}=\frac{4}{9}\). Total: \(\frac{5}{9}+\frac{4}{9}=1\) ✓

(c) P(all 3 marbles same colour) [2 marks · 2 steps]

🎯 "All Same Colour" — Two Mutually Exclusive 3-Ball Paths

"All same colour" means either all three green (GGG) or all three blue (BBB). These are mutually exclusive events, so:

\[P(\text{all same}) = P(GGG) + P(BBB)\]

Each 3-ball path probability = multiply all three branch probabilities (levels 1 × 2 × 3).

Key: The 3rd draw's branch probability depends on which pair was drawn from X — use the correct Y-composition for each case.

STEP 1

P(GGG): Path G₁→G₂→G₃. After GG transferred, Y has 8G 5B.

\[P(GGG) = \frac{5}{9} \times \frac{4}{8} \times \frac{8}{13} = \frac{160}{936}\]

P(BBB): Path B₁→B₂→B₃. After BB transferred, Y has 6G 7B.

\[P(BBB) = \frac{4}{9} \times \frac{3}{8} \times \frac{7}{13} = \frac{84}{936}\]
STEP 2

Add the two mutually exclusive "all same" paths:

\[P(\text{all same}) = \frac{160}{936} + \frac{84}{936} = \frac{244}{936}\]

Simplify: \(\gcd(244, 936) = 4\), so:

\[\frac{244}{936} = \frac{61}{234}\]
\(P(\text{all same colour}) = \dfrac{61}{234}\)

(d) P(all green | all same colour) [3 marks · 2 steps]

From part (c): \(P(\text{all same}) = \dfrac{61}{234}\); \(P(GGG) = \dfrac{160}{936} = \dfrac{40}{234}\)
📐 Conditional Probability — P(A|B) = P(A∩B) / P(B)

Formula: \[P(A \mid B) = \frac{P(A \cap B)}{P(B)}\]

Why it works: "Given B has occurred" means we restrict our sample space to only the outcomes where B happens. Within that restricted space, we ask: how much of it does A also occupy? Dividing P(A∩B) by P(B) re-scales probabilities so the reduced sample space sums to 1.

In this problem: A = "all green", B = "all same colour". Since GGG ⊂ {all same colour}, P(A∩B) = P(GGG).

When to use: Any time the question says "given that…" — that's a conditional probability.

STEP 1

Identify A∩B and express on a common denominator:

"All green AND all same colour" = GGG (it's already a subset of "all same colour").

\[P(GGG) = \frac{160}{936} = \frac{40}{234}\] \[P(\text{all same}) = \frac{61}{234}\]
Converting to the same denominator (234) makes the division clean.
STEP 2

Apply the conditional probability formula:

\[P(\text{all green} \mid \text{all same}) = \frac{P(GGG)}{P(\text{all same})} = \frac{40/234}{61/234} = \frac{40}{61}\]
\(P(\text{all green} \mid \text{all same colour}) = \dfrac{40}{61} \approx 0.656\)
Sanity check: \(\frac{40}{61} > \frac{1}{2}\) makes sense — given all three are the same colour, green is more likely than blue because X had more green marbles (5G vs 4B).
