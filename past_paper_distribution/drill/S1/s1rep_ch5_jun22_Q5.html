Q5 · Two Spinners — E(X), Var(X), CDF, Probability

Question

Pabel uses a red spinner. The random variable $R$ represents his score:

$r$23456
$P$0.250.300.150.100.20

(a) Show that $E(R^2) = 15.8$  [1]

Given $E(R) = 3.7$, (b) find the standard deviation of $R$ to 2 d.p.  [2]

Jessie uses a yellow spinner. The random variable $Y$ represents her score:

$y$23456
$P(Y=y)$0.100.200.10$a$$b$
$F(y)$0.100.300.40$c$$d$

(c) Write down the value of $d$.  [1]

Given $E(Y) = 4.55$, (d) find the value of $c$.  [5]

Jessie wins if she lands on 2. Pabel wins if he lands on 2. Jessie spins first.

(e) Find the probability that Jessie wins on her second spin.  [2]

(f) Calculate $P(\text{Pabel's first score} = \text{Jessie's first score})$.  [3]

[14 marks]

(a) [1 marks] · 1 step

STEP 1/1 B1

Formula: $E(R^2) = \sum r^2 \cdot P(R=r)$

$E(R^2) = 4(0.25) + 9(0.30) + 16(0.15) + 25(0.10) + 36(0.20)$

$= 1.00 + 2.70 + 2.40 + 2.50 + 7.20$

Square each $r$ value first, then multiply by its probability — don't forget $r=6$ contributes $36 \times 0.20 = 7.2$.
$E(R^2) = \boxed{15.8}$ ✓

(b) [2 marks] · 1 step

$E(R) = 3.7$, $E(R^2) = 15.8$
STEP 1/1 M1 A1

Formula: $\text{Var}(R) = E(R^2) - [E(R)]^2$

$\text{Var}(R) = 15.8 - (3.7)^2 = 15.8 - 13.69 = 2.11$

$\text{SD}(R) = \sqrt{\text{Var}(R)} = \sqrt{2.11}$

Standard deviation = $\sqrt{\text{Variance}}$; don't forget to square-root at the end.
$\boxed{\text{SD}(R) = \sqrt{2.11} \approx 1.45}$

(c) [1 marks] · 1 step

$F(y)$ is the CDF of $Y$; $F(d)$ is the maximum value of the CDF
STEP 1/1 B1

CDF at the maximum value must equal 1

$d = F(6)$ = probability $Y \leq 6$ = probability of any outcome

For any RV, $F(\text{max value}) = 1$ — all outcomes are covered.
$\boxed{d = 1}$

(d) [5 marks] · 2 steps

$E(Y) = 4.55$; $P(Y=5) = a$, $P(Y=6) = b$; $d = 1$
STEP 1/2 M1

Set up two equations using: (1) probabilities sum to 1, (2) $E(Y) = 4.55$

Equation 1 (probs sum to 1): $0.10 + 0.20 + 0.10 + a + b = 1$

$\Rightarrow a + b = 0.60$

Equation 2 (expectation): $E(Y) = 2(0.10) + 3(0.20) + 4(0.10) + 5a + 6b = 4.55$

$\Rightarrow 0.20 + 0.60 + 0.40 + 5a + 6b = 4.55$

$\Rightarrow 5a + 6b = 3.35$

Always write out the full $E(Y)$ sum — easy to misread table entries.
Equations: $a + b = 0.60$ and $5a + 6b = 3.35$
Carry-in from previous stepFrom STEP 1: $a + b = 0.60$ and $5a + 6b = 3.35$
STEP 2/2 M1 A1

Solve simultaneously

From Eq 1: $b = 0.60 - a$. Substitute into Eq 2:

$5a + 6(0.60 - a) = 3.35$

$5a + 3.60 - 6a = 3.35$

$-a = -0.25 \Rightarrow a = 0.25$, $b = 0.35$

$c = F(5) = F(4) + P(Y=5) = 0.40 + 0.25 = 0.65$

$c = F(5)$, not $a$ — remember $c$ is the CDF, not the probability.
$a = 0.25,\; b = 0.35,\; \boxed{c = 0.65}$

(e) [2 marks] · 1 step

$P(Y=2) = 0.10$, $P(R=2) = 0.25$; Jessie wins if $Y=2$, Pabel wins if $R=2$; Jessie spins first
STEP 1/1 M1 A1

Jessie wins on her 2nd spin requires: J loses spin 1, P loses spin 1, J wins spin 2

(Each spin is independent; "wins on 2nd spin" means she gets a 2nd chance)

$P = P(J \neq 2 \text{ on 1st}) \times P(P \neq 2 \text{ on 1st}) \times P(J = 2 \text{ on 2nd})$

$= (1 - 0.10) \times (1 - 0.25) \times 0.10$

$= 0.90 \times 0.75 \times 0.10$

Jessie only spins a 2nd time if neither player won on the 1st round — both must not-win first.
$\boxed{P = 0.9 \times 0.75 \times 0.1 = 0.0675}$

(f) [3 marks] · 1 step

$P(R=2)=0.25, P(R=3)=0.30, P(R=4)=0.15, P(R=5)=0.10, P(R=6)=0.20$; $P(Y=2)=0.10, P(Y=3)=0.20, P(Y=4)=0.10, P(Y=5)=0.25, P(Y=6)=0.35$
STEP 1/1 M1 A2

Formula: $P(R = Y) = \sum_{r} P(R=r) \cdot P(Y=r)$

(scores are equal iff both land on same value; spins are independent)

$r$$P(R=r)$$P(Y=r)$product
20.250.100.025
30.300.200.060
40.150.100.015
50.100.250.025
60.200.350.070

$P = 0.025 + 0.060 + 0.015 + 0.025 + 0.070$

Use updated $P(Y=5)=a=0.25$ and $P(Y=6)=b=0.35$ from part (d) — not any earlier values.
$\boxed{P(\text{same score}) = 0.195}$
