(a) Show that $E(R^2) = 15.8$  [1]

(a) [1 mark] · 1 step

Expected Value: Definition and E(X²)
For a discrete random variable \(X\) with probability distribution \(P(X=x)\): \[E(X) = \sum_x x \cdot P(X = x)\] \[E(X^2) = \sum_x x^2 \cdot P(X = x) \quad \text{(square each } x \text{ value, then weight by its probability)}\] Note: \(E(X^2) \neq [E(X)]^2\) in general. These are two different quantities.
When to use \(E(X^2)\): Computing variance via \(\text{Var}(X) = E(X^2) - [E(X)]^2\).
STEP 1

Apply \(E(R^2) = \sum r^2 \cdot P(R = r)\) to the red spinner's distribution

Square each \(r\) value, then multiply by its probability:

\[E(R^2) = 2^2(0.25) + 3^2(0.30) + 4^2(0.15) + 5^2(0.10) + 6^2(0.20)\] \[= 4(0.25) + 9(0.30) + 16(0.15) + 25(0.10) + 36(0.20)\] \[= 1.00 + 2.70 + 2.40 + 2.50 + 7.20\] \[= 15.80 \checkmark\]
Don't forget \(r=6\): It contributes \(36 \times 0.20 = 7.20\), which is the largest single term. Always work through all rows of the table systematically.
$E(R^2) = \boxed{15.8}$ ✓

(b) [2 marks] · 1 step

Variance and Standard Deviation
\[\text{Var}(X) = E(X^2) - [E(X)]^2\] Derivation idea: \(\text{Var}(X) = E[(X-\mu)^2] = E[X^2 - 2\mu X + \mu^2] = E(X^2) - 2\mu^2 + \mu^2 = E(X^2) - [E(X)]^2\).
The standard deviation is then: \[\text{SD}(X) = \sqrt{\text{Var}(X)}\] Common error: forgetting to take the square root at the end — variance and standard deviation have different units.
$E(R) = 3.7$, $E(R^2) = 15.8$ (from part (a))
STEP 1

Apply \(\text{Var}(R) = E(R^2) - [E(R)]^2\), then take the square root

\[\text{Var}(R) = E(R^2) - [E(R)]^2 = 15.8 - (3.7)^2 = 15.8 - 13.69 = 2.11\] \[\text{SD}(R) = \sqrt{\text{Var}(R)} = \sqrt{2.11} = 1.4526\ldots \approx 1.45\]
Why \([E(R)]^2 = 13.69\)? \((3.7)^2 = 3.7 \times 3.7 = 13.69\). It is important to square first, then subtract. Do not compute \(E(R^2 - [E(R)]^2)\) as if the expectation applies inside.
$\boxed{\text{SD}(R) = \sqrt{2.11} \approx 1.45}$

(c) [1 mark] · 1 step

Cumulative Distribution Function (CDF)
The CDF of a discrete random variable \(Y\) is defined as: \[F(y) = P(Y \leq y) = \sum_{t \leq y} P(Y = t)\] Key properties:
  • \(F\) is non-decreasing: as \(y\) increases, \(F(y)\) can only stay the same or increase
  • \(F(\text{maximum value}) = 1\): the probability of getting at most the largest possible value is 1 (certainty)
  • \(F(\text{minimum value}) = P(Y = \text{min})\)
Recovering probabilities from CDF: \(P(Y = y) = F(y) - F(y_{\text{prev}})\)
$F(y)$ is the CDF of $Y$; maximum value of $Y$ is 6
STEP 1

Apply the CDF property: \(F(\text{max value}) = 1\)

The largest possible value of \(Y\) is 6. Therefore:

\[d = F(6) = P(Y \leq 6) = 1\]

This is because \(Y\) can only take values 2, 3, 4, 5, 6, so the probability of \(Y\) being at most 6 covers all possible outcomes.

Why always 1? \(F(6) = P(Y \leq 6)\) includes all possible outcomes of \(Y\). Since something must happen, this probability must equal 1 (by the axiom of total probability: all probabilities sum to 1).
$\boxed{d = 1}$

(d) [5 marks] · 2 steps

Finding Unknown Probabilities Using Sum = 1 and E(Y)
When a distribution has unknown probabilities \(a\) and \(b\), set up two simultaneous equations:
  1. Equation 1: All probabilities sum to 1: \(\sum P(Y=y) = 1\)
  2. Equation 2: Use the given expected value: \(E(Y) = \sum y \cdot P(Y=y)\)
Then solve the system for \(a\) and \(b\).
Once \(a\) and \(b\) are found, recover the CDF value \(c = F(5) = F(4) + P(Y=5) = F(4) + a\).
$E(Y) = 4.55$; $P(Y=5) = a$, $P(Y=6) = b$; $F(4) = 0.40$
STEP 1

Set up two equations

Equation 1 — probabilities sum to 1:

\[0.10 + 0.20 + 0.10 + a + b = 1 \implies a + b = 0.60\]

Equation 2 — use \(E(Y) = \sum y \cdot P(Y=y)\):

\[E(Y) = 2(0.10) + 3(0.20) + 4(0.10) + 5a + 6b = 4.55\] \[0.20 + 0.60 + 0.40 + 5a + 6b = 4.55\] \[1.20 + 5a + 6b = 4.55 \implies 5a + 6b = 3.35\]
Always write out the full \(E(Y)\) sum — it is easy to misread table entries or miss a term. Work through each row: multiply \(y\) value by probability, then add all contributions.
Equations: $a + b = 0.60$ and $5a + 6b = 3.35$
From STEP 1:$a + b = 0.60$ and $5a + 6b = 3.35$
STEP 2

Solve simultaneously to find \(a\), \(b\), then compute \(c = F(5)\)

From Equation 1: \(b = 0.60 - a\). Substitute into Equation 2:

\[5a + 6(0.60 - a) = 3.35\] \[5a + 3.60 - 6a = 3.35\] \[-a = -0.25 \implies a = 0.25\] \[b = 0.60 - 0.25 = 0.35\]

Now find \(c = F(5)\). The CDF is cumulative, so:

\[c = F(5) = F(4) + P(Y=5) = 0.40 + a = 0.40 + 0.25 = 0.65\]
Careful: \(c = F(5)\), not \(a\)! \(a = P(Y=5) = 0.25\) is the probability at exactly 5. But \(c = F(5) = P(Y \leq 5)\) is the cumulative probability up to and including 5. These are different: \(F(5) = F(4) + P(Y=5) = 0.40 + 0.25 = 0.65\).
$a = 0.25,\; b = 0.35,\; \boxed{c = 0.65}$

(e) [2 marks] · 1 step

Sequential Game: Jessie Wins on Her 2nd Spin
The game proceeds in rounds: each round consists of Jessie spinning (she goes first), then Pabel spinning. The game ends when someone lands on 2.
For Jessie to win on her 2nd spin, the following must happen in order:
  1. Jessie's 1st spin: NOT 2 (game continues)
  2. Pabel's 1st spin: NOT 2 (game continues)
  3. Jessie's 2nd spin: IS 2 (Jessie wins)
Since spins are independent, multiply the probabilities.
$P(Y=2) = 0.10$ (Jessie's probability of 2); $P(R=2) = 0.25$ (Pabel's); Jessie spins first
STEP 1

Identify the sequence of events and multiply probabilities

\[P(\text{Jessie wins on spin 2}) = P(\text{J}\neq 2) \times P(\text{P}\neq 2) \times P(\text{J}= 2)\] \[= (1 - 0.10) \times (1 - 0.25) \times 0.10\] \[= 0.90 \times 0.75 \times 0.10\] \[= 0.0675\]
Why multiply? Each spin is independent of the others. By the multiplication rule for independent events: \(P(A \cap B \cap C) = P(A) \times P(B) \times P(C)\).
Why \(P(\text{J}\neq 2) = 1 - 0.10 = 0.90\)? Jessie fails to win on her first spin with probability \(1 - P(Y=2) = 0.90\). Similarly, \(P(\text{P}\neq 2) = 1 - 0.25 = 0.75\).
$\boxed{P(\text{Jessie wins on 2nd spin}) = 0.0675}$

Probability of Equal Scores (Independent Spinners)
Since \(R\) and \(Y\) are independent, the probability that both land on the same value \(r\) is: \[P(R = r \text{ AND } Y = r) = P(R = r) \times P(Y = r)\] To find the total probability they score the same, sum over all possible common values: \[P(R = Y) = \sum_r P(R = r) \cdot P(Y = r)\] Work through each possible score value from the tables and add up the joint probabilities.
Full distributions of $R$ and $Y$ (with $a=0.25$, $b=0.35$ from part (d))
