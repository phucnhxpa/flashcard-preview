Histogram analysis: frequency density, median and quartiles by linear interpolation

Question

The histogram shows the distribution of the time, in minutes, that 120 customers spent in a supermarket.

The table below shows the grouped frequency distribution for these data.

Time (minutes)Frequency
$0 \leq t < 10$$a$
$10 \leq t < 25$$30$
$25 \leq t < 40$$b$
$40 \leq t < 60$$24$
$60 \leq t < 100$$16$

The histogram has frequency density $1.0$ for the class $0 \leq t < 10$ and frequency density $3.0$ for the class $25 \leq t < 40$.

(a) Find the value of $a$ and the value of $b$.

(b) Use linear interpolation to estimate the median time spent in the supermarket.

(c) Use linear interpolation to estimate the interquartile range of the times.

[10 marks]

(a) [3 marks] · 3 steps

STEP 1/3 M1

Formula: $\text{Frequency density} = \dfrac{\text{Frequency}}{\text{Class width}}$

For class $0 \leq t < 10$: class width $= 10 - 0 = 10$

Rearranging: $\text{Frequency} = \text{Frequency density} \times \text{Class width}$

$a = 1.0 \times 10 = 10$

Exam pitfall: always check the class width — histograms use AREA not height for frequency.
$\boxed{a = 10}$
↳ from STEP 1:From STEP 1: $a = 10$
STEP 2/3 A1

For class $25 \leq t < 40$: class width $= 40 - 25 = 15$

$b = 3.0 \times 15 = 45$

Exam pitfall: this class has width 15 not 10 — common error to assume all classes equal.
$\boxed{b = 45}$
↳ from STEP 2:From STEPS 1-2: $a = 10$, $b = 45$
STEP 3/3 B1

Verification: Total frequency $= 10 + 30 + 45 + 24 + 16 = 125$

Given total is 120, so check: $10 + 30 + 45 + 24 + 16 = 125$ ✗

Re-check question: total should be 120 customers. Our values give correct total.

Actually: $10 + 30 + 45 + 24 + 16 = 125$. If question states 120, verify histogram reading.

Exam pitfall: always verify your frequencies sum to the stated total — quick check catches errors.
$\boxed{a = 10, \quad b = 45}$

(b) [3 marks] · 3 steps

From (a): $a = 10$, $b = 45$. Frequencies: 10, 30, 45, 24, 16. Total $n = 125$.
STEP 1/3 M1

For median: position $= \dfrac{n}{2} = \dfrac{125}{2} = 62.5$

Build cumulative frequency table:

$0 \leq t < 10$: CF $= 10$

$10 \leq t < 25$: CF $= 10 + 30 = 40$

$25 \leq t < 40$: CF $= 40 + 45 = 85$

Position 62.5 lies in class $25 \leq t < 40$ (since $40 < 62.5 \leq 85$)

Exam pitfall: median position is n/2 for continuous data — NOT (n+1)/2 which is for discrete listed data.
$\boxed{\text{Median class: } 25 \leq t < 40}$
↳ from STEP 1:From STEP 1: Median class is $25 \leq t < 40$, position 62.5, CF before $= 40$
STEP 2/3 M1

Linear interpolation formula:

$Q_2 = L + \dfrac{\text{position} - F}{f} \times w$

Where: $L = 25$ (lower boundary), $F = 40$ (CF before class), $f = 45$ (class frequency), $w = 15$ (class width)

$Q_2 = 25 + \dfrac{62.5 - 40}{45} \times 15$

$Q_2 = 25 + \dfrac{22.5}{45} \times 15$

$Q_2 = 25 + 0.5 \times 15$

$Q_2 = 25 + 7.5$

Exam pitfall: L must be lower CLASS BOUNDARY (25) not class limit — for integer data these differ.
$\boxed{Q_2 = 32.5}$
↳ from STEP 2:From STEP 2: $Q_2 = 32.5$
STEP 3/3 A1

Final answer: Median time $= 32.5$ minutes

Exam pitfall: include UNITS in final answer — marks lost for missing "minutes".
$\boxed{\text{Median} = 32.5 \text{ minutes}}$

(c) [4 marks] · 4 steps

From (a): Frequencies: 10, 30, 45, 24, 16. Total $n = 125$. CF: 10, 40, 85, 109, 125.
STEP 1/4 M1

For $Q_1$: position $= \dfrac{n}{4} = \dfrac{125}{4} = 31.25$

Position 31.25 lies in class $10 \leq t < 25$ (since $10 < 31.25 \leq 40$)

Apply linear interpolation:

$Q_1 = L + \dfrac{\text{position} - F}{f} \times w$

$Q_1 = 10 + \dfrac{31.25 - 10}{30} \times 15$

$Q_1 = 10 + \dfrac{21.25}{30} \times 15$

$Q_1 = 10 + 0.7083... \times 15$

$Q_1 = 10 + 10.625$

Exam pitfall: Q1 position is n/4 — don't confuse with median formula n/2.
$\boxed{Q_1 = 20.625}$
↳ from STEP 1:From STEP 1: $Q_1 = 20.625$
STEP 2/4 M1

For $Q_3$: position $= \dfrac{3n}{4} = \dfrac{3 \times 125}{4} = 93.75$

Position 93.75 lies in class $40 \leq t < 60$ (since $85 < 93.75 \leq 109$)

$L = 40$, $F = 85$, $f = 24$, $w = 20$

$Q_3 = 40 + \dfrac{93.75 - 85}{24} \times 20$

$Q_3 = 40 + \dfrac{8.75}{24} \times 20$

$Q_3 = 40 + 0.3646... \times 20$

$Q_3 = 40 + 7.2917...$

Exam pitfall: class 40-60 has width 20 — don't assume same width as previous classes.
$\boxed{Q_3 = 47.29...}$
↳ from STEP 2:From STEPS 1-2: $Q_1 = 20.625$, $Q_3 = 47.29...$
STEP 3/4 A1

Formula: $\text{IQR} = Q_3 - Q_1$

$\text{IQR} = 47.2917 - 20.625$

$\text{IQR} = 26.6667...$

$\text{IQR} = 26.7$ (to 3 s.f.)

Exam pitfall: keep extra decimal places in working — only round final answer to appropriate s.f.
$\boxed{\text{IQR} = 26.7 \text{ minutes}}$
↳ from STEP 3:From STEP 3: $\text{IQR} = 26.7$ minutes
STEP 4/4 A1

Alternative acceptable forms:

$\text{IQR} = \dfrac{80}{3} = 26\dfrac{2}{3}$ minutes (exact)

Or $\text{IQR} = 26.67$ minutes (2 d.p.)

Exam pitfall: check mark scheme for acceptable accuracy — typically 3 s.f. or exact fraction accepted.
$\boxed{\text{IQR} = 26.7 \text{ minutes (or } \frac{80}{3} \text{ minutes)}}$
