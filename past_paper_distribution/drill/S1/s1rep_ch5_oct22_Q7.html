X with P(X=0)=0.1, P(X=5)=0.2, P(X=10)=0.7 — find E(X), Var(X), P(X₁
Question

Adana selects one number at random from the distribution of $X$ which has the following probability distribution.

The discrete random variable $X$ takes values 0, 5, 10 with probabilities:

$x$0510
$P(X=x)$0.10.20.7

(a) Given that the number selected by Adana is not 5, write down the probability it is 0. [1]

$X_1$ and $X_2$ are independent observations of $X$. Bruno and Charlie each independently select one number at random from the distribution of $X$.
(b) Show that $E(X^2) = 75$. [1]   (c) Find $\text{Var}(X)$. [3]   (d) Find $\text{Var}(4-3X)$. [2]
(e) Find the probability that the number Bruno selects is greater than the number Charlie selects. [3]   (f) Determine the probability distribution of $D = \text{Bruno's number} \times \text{Charlie's number}$. [4]

[15 marks]

(a) P(X = 0 | X ≠ 5) [1 mark · 1 step]

STEP 1
Given X ≠ 5, the remaining outcomes are X = 0 (prob 0.1) and X = 10 (prob 0.7), total = 0.8.
$P(X=0 | X\neq 5) = \dfrac{P(X=0)}{P(X\neq5)} = \dfrac{0.1}{0.1+0.7} = \dfrac{0.1}{0.8} = \dfrac{1}{8}$
$P(X=0 \mid X\neq5) = \dfrac{1}{8}$

(b)+(c) [5 marks] · 1 step

STEP 1/1

$E(X) = 0(0.1) + 5(0.2) + 10(0.7) = 0 + 1 + 7 = 8$

$E(X^2) = 0^2(0.1) + 5^2(0.2) + 10^2(0.7) = 0 + 5 + 70 = 75$ ✓

Var formula: $\text{Var}(X) = E(X^2) - [E(X)]^2 = 75 - 8^2 = 75 - 64$

Must SQUARE E(X): $[E(X)]^2 = 64$, not $E(X)=8$
$\boxed{\text{Var}(X) = 11}$

(d) [2 marks] · 1 step

$\text{Var}(X)=11$
Carry-in from previous step$\text{Var}(X)=11$
STEP 1/1

Formula: $\text{Var}(a + bX) = b^2\,\text{Var}(X)$

$\text{Var}(4-3X) = (-3)^2 \times 11 = 9 \times 11$

The constant $+4$ vanishes; coefficient must be squared
$\boxed{\text{Var}(4-3X) = 99}$

(d) [3 marks] · 1 step

$X_1, X_2$ independent copies of $X$
Carry-in from previous step$P(X=0)=0.1,\ P(X=5)=0.2,\ P(X=10)=0.7$
STEP 1/1

Key identity: $P(X_1 = X_2) = \sum [P(X=x)]^2$

$P(X_1=X_2) = 0.1^2 + 0.2^2 + 0.7^2 = 0.01 + 0.04 + 0.49 = 0.54$

By symmetry $P(X_1 < X_2) = P(X_1 > X_2)$, so:

$P(X_1 < X_2) = \dfrac{1 - P(X_1=X_2)}{2} = \dfrac{1 - 0.54}{2} = \dfrac{0.46}{2}$

Symmetry works because $X_1$ and $X_2$ have the same distribution (i.i.d.)
$\boxed{P(X_1 < X_2) = 0.23}$

(e) [4 marks] · 1 step

$X_1, X_2$ independent, $D = X_1 X_2$
Carry-in from previous step$P(X=0)=0.1,\ P(X=5)=0.2,\ P(X=10)=0.7$
STEP 1/1

Products that can arise: $D \in \{0, 25, 50, 100\}$

$P(D=0) = 1 - P(\text{neither is 0}) = 1 - 0.9^2 = 1 - 0.81 = 0.19$

$P(D=25) = P(X_1=5)\cdot P(X_2=5) = 0.2^2 = 0.04$

$P(D=50) = 2 \times P(X=5)\cdot P(X=10) = 2(0.2)(0.7) = 0.28$

$P(D=100) = P(X=10)^2 = 0.7^2 = 0.49$

Check: $0.19+0.04+0.28+0.49 = 1$ ✓
$D$: 0 (0.19), 25 (0.04), 50 (0.28), 100 (0.49)
