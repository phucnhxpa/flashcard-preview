Three Bags A, B, C — Coloured Balls, Tree Diagram & Distribution

Question

Three bags A, B and C each contain coloured balls.

Bag A contains 4 red balls and 2 yellow balls only.
Bag B contains 4 red balls and 1 yellow ball only.
Bag C contains 6 red balls only.

In a game, Mike takes a ball at random from bag A, records the colour and places it in bag C. He then takes a ball at random from bag B, records the colour and places it in bag C. Finally, Mike takes a ball at random from bag C and records the colour.

(a) Complete the tree diagram to illustrate the game by adding the remaining branches and all probabilities. [3]

(b) Show that the probability that Mike records a yellow ball exactly twice is \(\dfrac{1}{10}\). [3]

Given that Mike records exactly 2 yellow balls,

(c) find the probability that the ball drawn from bag A is red. [2]

Mike plays this game a large number of times, each time starting with the bags containing balls as described above. The random variable \(X\) represents the number of yellow balls recorded in a single game.

(d) Find the probability distribution of \(X\). [3]

(e) Find \(E(X)\). [2]

[13 marks]

(a) Tree diagram [3 marks · 2 steps]

A: 4R 2Y (6 total); B: 4R 1Y (5 total); C starts with 6R only
🌳 Tree Structure — 3 Levels, Changing Bag C

The tree has 3 levels:

  1. Draw from A (6 balls: 4R, 2Y) — ball goes into C
  2. Draw from B (5 balls: 4R, 1Y) — ball goes into C
  3. Draw from C — but C now has 8 balls (6 original R + 1 from A + 1 from B)

Critical point: The composition of C depends on what was drawn from A and B. Map out all 4 cases (RR, RY, YR, YY) to get C's contents before the final draw.

Rule: Multiply along branches (sequential probability); add across branches (OR of mutually exclusive paths).

STEP 1

Levels 1 and 2 — draws from A and B:

From A (4R, 2Y out of 6):

\[P(R_A)=\frac{4}{6}=\frac{2}{3},\quad P(Y_A)=\frac{2}{6}=\frac{1}{3}\]

From B (4R, 1Y out of 5) — independent of A's draw:

\[P(R_B)=\frac{4}{5},\quad P(Y_B)=\frac{1}{5}\]
A and B are separate bags drawn independently, so B's probabilities are the same regardless of what was drawn from A.
STEP 2

Level 3 — draw from C (8 balls total, composition varies):

C starts with 6R and receives 1 ball from A and 1 ball from B.

  • RR added: C = 8R, 0Y → \(P(R_C)=1,\ P(Y_C)=0\)
  • RY added: C = 7R, 1Y → \(P(R_C)=\tfrac{7}{8},\ P(Y_C)=\tfrac{1}{8}\)
  • YR added: C = 7R, 1Y → \(P(R_C)=\tfrac{7}{8},\ P(Y_C)=\tfrac{1}{8}\)
  • YY added: C = 6R, 2Y → \(P(R_C)=\tfrac{6}{8}=\tfrac{3}{4},\ P(Y_C)=\tfrac{2}{8}=\tfrac{1}{4}\)
Tree: A branches (2/3 R, 1/3 Y) × B branches (4/5 R, 1/5 Y) × C branches (as above)

(b) Show P(exactly 2 yellow) = 1/10 [3 marks · 3 steps]

🎯 "Exactly 2 Yellows" — Listing All Valid Orderings

Mike records 3 balls (one from A, one from B, one from C). "Exactly 2 yellows" means exactly 2 of those 3 records are Y. The three draws play different roles, so we must check each possible ordering of where the 2 yellows appear:

  • YYR: Yellow from A, Yellow from B, Red from C
  • RYY: Red from A, Yellow from B, Yellow from C
  • YRY: Yellow from A, Red from B, Yellow from C

These 3 outcomes are mutually exclusive → add their probabilities.

STEP 1

P(YYR): Y from A, Y from B → C has 6R+2Y = 8 total; need R from C.

\[P(YYR) = \frac{1}{3} \times \frac{1}{5} \times \frac{6}{8}\] \[= \frac{6}{120} = \frac{1}{20}\]
STEP 2

P(RYY): R from A, Y from B → C has 7R+1Y = 8 total; need Y from C.

\[P(RYY) = \frac{2}{3} \times \frac{1}{5} \times \frac{1}{8}\] \[= \frac{2}{120} = \frac{1}{60}\]

P(YRY): Y from A, R from B → C has 7R+1Y = 8 total; need Y from C.

\[P(YRY) = \frac{1}{3} \times \frac{4}{5} \times \frac{1}{8} = \frac{4}{120} = \frac{1}{30}\]
STEP 3

Add all three mutually exclusive orderings:

\[P(X=2) = \frac{1}{20} + \frac{1}{60} + \frac{1}{30}\]

Common denominator 60:

\[= \frac{3}{60} + \frac{1}{60} + \frac{2}{60} = \frac{6}{60}\]
\(P(X=2) = \dfrac{1}{10}\) ✓

(c) P(A is red | exactly 2 yellow) [2 marks · 2 steps]

📐 Conditional Probability — P(A|B) = P(A∩B) / P(B)

Formula: \[P(A \mid B) = \frac{P(A \cap B)}{P(B)}\]

Why it works: "Given B occurred" restricts our universe to outcomes where B happens. P(A∩B)/P(B) re-scales so the restricted space sums to 1.

Here: A = "ball from bag A is red"; B = "exactly 2 yellows" = {YYR, RYY, YRY}.
"A is red AND exactly 2 yellows" = the only ordering with R from A and 2 yellows total = RYY (R from A, Y from B, Y from C).

STEP 1

Identify the intersection:

"A is red" AND "exactly 2 yellows" = only the ordering RYY (the other orderings with 2 yellows — YYR and YRY — both have Yellow from A, not Red).

\[P(\text{A red} \cap X=2) = P(RYY) = \frac{1}{60}\]
STEP 2

Apply the formula:

\[P(\text{A red} \mid X=2) = \frac{P(RYY)}{P(X=2)} = \frac{1/60}{1/10} = \frac{1}{60} \times \frac{10}{1} = \frac{10}{60}\]
\(P(\text{A red} \mid X=2) = \dfrac{1}{6}\)
Sanity check: P(A yellow | X=2) = 1 − 1/6 = 5/6. YYR gives P=1/20=3/60 and YRY gives P=1/30=2/60, total = 5/60 = 1/12. Then (1/12)/(1/10) = 10/12 = 5/6 ✓

(d) Probability distribution of X [3 marks · 3 steps]

📊 Probability Distribution — Must Sum to 1

X = number of yellow balls recorded (can be 0, 1, 2, or 3).

We need P(X=0), P(X=1), P(X=2), P(X=3). A key fact: all probabilities in a distribution must sum to 1:

\[P(X=0) + P(X=1) + P(X=2) + P(X=3) = 1\]

Strategy: Compute P(X=0) and P(X=3) directly from the tree (each has only 1 path), use P(X=2) = 1/10 from part (b), then find P(X=1) by subtraction.

STEP 1

P(X=0) — all red:

Path: R from A, R from B → C has 8R, 0Y; R from C (certain).

\[P(X=0) = P(RRR) = \frac{2}{3} \times \frac{4}{5} \times 1\] \[= \frac{8}{15}\]

P(X=3) — all yellow:

Path: Y from A, Y from B → C has 6R, 2Y; Y from C.

\[P(X=3) = P(YYY) = \frac{1}{3} \times \frac{1}{5} \times \frac{2}{8} = \frac{2}{120} = \frac{1}{60}\]
STEP 2

P(X=1) by the complement rule:

\[P(X=1) = 1 - P(X=0) - P(X=2) - P(X=3)\]

Convert to a common denominator of 60:

\[P(X=0) = \frac{8}{15} = \frac{32}{60}\] \[P(X=2) = \frac{1}{10} = \frac{6}{60},\quad P(X=3) = \frac{1}{60}\] \[P(X=1) = 1 - \frac{32}{60} - \frac{6}{60} - \frac{1}{60} = \frac{60-39}{60} = \frac{21}{60} = \frac{7}{20}\]
STEP 3

Distribution table:

\(x\) 0 1 2 3
\(P(X=x)\) \(\frac{8}{15}\) \(\frac{7}{20}\) \(\frac{1}{10}\) \(\frac{1}{60}\)
Check: \(\frac{32}{60}+\frac{21}{60}+\frac{6}{60}+\frac{1}{60} = \frac{60}{60} = 1\) ✓

(e) Find E(X) [2 marks · 2 steps]

📈 Expected Value — E(X) = Σ x · P(X = x)

Formula: \[E(X) = \sum_x x \cdot P(X=x) = 0\cdot P(X=0) + 1\cdot P(X=1) + 2\cdot P(X=2) + 3\cdot P(X=3)\]

Interpretation: E(X) is the long-run average number of yellow balls Mike would record per game if he played many times. It does NOT have to be a whole number.

Tip: Multiply each outcome value by its probability, then sum. The \(x=0\) term always contributes 0.

STEP 1

Multiply each value by its probability:

\[E(X) = 0 \cdot \frac{8}{15} + 1 \cdot \frac{7}{20} + 2 \cdot \frac{1}{10} + 3 \cdot \frac{1}{60}\] \[= 0 + \frac{7}{20} + \frac{2}{10} + \frac{3}{60}\]
STEP 2

Convert to common denominator 60 and sum:

\[\frac{7}{20} = \frac{21}{60},\quad \frac{2}{10} = \frac{12}{60},\quad \frac{3}{60} = \frac{3}{60}\] \[E(X) = \frac{21+12+3}{60} = \frac{36}{60}\]
\(E(X) = \dfrac{3}{5} = 0.6\)
Sanity check: X ranges from 0 to 3. Average 0.6 is close to 0, which makes sense because P(X=0) = 8/15 ≈ 0.53 is the dominant term — most games end with 0 or 1 yellow.
