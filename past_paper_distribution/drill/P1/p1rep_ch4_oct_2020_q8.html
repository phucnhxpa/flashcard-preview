Question

The curve C has equation $y = 2x^3 + ax^2 + bx$ where a and b are constants. Given that C passes through the point (-1, 0) and has a stationary point when x = 1, find the values of a and b. Then find the equation of the tangent to C that is parallel to the line $y = 6x - 5$.

[11 marks]

(a) Finding a and b [5 marks] · 3 steps

STEP 1/3Use the condition that C passes through (-1, 0) M1

Strategy: Substitute the point (-1, 0) into the equation

Formula: $y = 2x^3 + ax^2 + bx$

When x = -1, y = 0:

$0 = 2(-1)^3 + a(-1)^2 + b(-1)$

$0 = -2 + a - b$

$a - b = 2$ ... equation (1)
↳ from STEP 1:$a - b = 2$
STEP 2/3Use the stationary point condition at x = 1 M1 A1

Strategy: At a stationary point, $\frac{dy}{dx} = 0$

$\frac{dy}{dx} = 6x^2 + 2ax + b$

When x = 1, $\frac{dy}{dx} = 0$:

$0 = 6(1)^2 + 2a(1) + b$

$0 = 6 + 2a + b$

$2a + b = -6$ ... equation (2)
↳ from STEP 2:$a - b = 2$ and $2a + b = -6$
STEP 3/3Solve the simultaneous equations A1 A1

Adding equations (1) and (2):

$3a = -4$

$a = -\frac{4}{3}$

Substituting back into equation (1):

$-\frac{4}{3} - b = 2$

$b = -\frac{4}{3} - 2 = -\frac{10}{3}$

Watch: Check your signs carefully when solving simultaneous equations
$\boxed{a = -\frac{4}{3}, \quad b = -\frac{10}{3}}$

(b) Finding the tangent parallel to y = 6x - 5 [6 marks] · 3 steps

$y = 2x^3 - \frac{4}{3}x^2 - \frac{10}{3}x$
STEP 1/3Identify the gradient needed for parallel lines M1

Strategy: Parallel lines have equal gradients

The line $y = 6x - 5$ has gradient 6

So we need $\frac{dy}{dx} = 6$

Gradient of tangent = 6
↳ from STEP 1:$\frac{dy}{dx} = 6x^2 - \frac{8}{3}x - \frac{10}{3}$
STEP 2/3Set up and solve the equation for x M1 A1

$6x^2 - \frac{8}{3}x - \frac{10}{3} = 6$

Multiply by 3:

$18x^2 - 8x - 10 = 18$

$18x^2 - 8x - 28 = 0$

$9x^2 - 4x - 14 = 0$

$(9x + 7)(x - 2) = 0$

$x = -\frac{7}{9}$ or $x = 2$
↳ from STEP 2:$x = -\frac{7}{9}$ or $x = 2$
STEP 3/3Find the y-coordinates and equations of tangents M1 A1 A1

When $x = 2$:

$y = 2(8) - \frac{4}{3}(4) - \frac{10}{3}(2) = 16 - \frac{16}{3} - \frac{20}{3} = 16 - 12 = 4$

Tangent: $y - 4 = 6(x - 2)$, giving $y = 6x - 8$

When $x = -\frac{7}{9}$:

$y = 2(-\frac{343}{729}) - \frac{4}{3}(\frac{49}{81}) - \frac{10}{3}(-\frac{7}{9}) = -\frac{686}{729} - \frac{196}{243} + \frac{70}{27} = \frac{700}{729}$

Tangent: $y - \frac{700}{729} = 6(x + \frac{7}{9})$

$\boxed{y = 6x - 8 \text{ and } y = 6x + \frac{5314}{729}}$
