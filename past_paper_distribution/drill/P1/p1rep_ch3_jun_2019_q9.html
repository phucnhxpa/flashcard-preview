Trigonometric Graph & Equations

Question

The curve C has equation $y = 4\sin(2x) - 1$ where $x$ is in degrees.

(a) Sketch C for $0° \leq x \leq 360°$, showing the coordinates of the minimum point.

(b) Solve $4\sin(2x) - 1 = 1$ for $0° \leq x \leq 360°$.

[[7] marks]

Part (a) - Graph sketch and minimum [4 marks] · 3 steps

$y = 4\sin(2x) - 1$
STEP 1/3 B1

Strategy: Identify transformations from standard sine curve

Formula: $y = A\sin(Bx) + C$ has amplitude $|A|$, period $\frac{360°}{B}$, vertical shift $C$

Amplitude = 4, period = $\frac{360°}{2} = 180°$, vertical shift = -1

The coefficient of 2 inside halves the period, not the amplitude
$\boxed{\text{Amplitude } 4, \text{ period } 180°, \text{ shift down } 1}$
↳ from STEP 1:Amplitude 4, vertical shift -1
STEP 2/3 M1 A1

Strategy: Minimum occurs when $\sin(2x) = -1$

Formula: $\sin\theta = -1$ when $\theta = 270°$

$\sin(2x) = -1 \Rightarrow 2x = 270° \Rightarrow x = 135°$

$y_{\min} = 4(-1) - 1 = -5$

Common error: using $x = 270°$ instead of solving $2x = 270°$ — must divide by 2
$\boxed{\text{Minimum at } (135°, -5)}$
↳ from STEP 2:Min at $(135°, -5)$, period $180°$
STEP 3/3 B1

Strategy: Sketch sine curve with correct transformations

Key features:

• Starts at $(0°, -1)$

• Maximum at $(45°, 3)$ and $(225°, 3)$

• Minimum at $(135°, -5)$ and $(315°, -5)$

• Two complete cycles in $[0°, 360°]$

Graph oscillates between $y = 3$ and $y = -5$ (not between 4 and -4)
$\boxed{\text{Sine curve through } (0°,-1), (45°,3), (135°,-5), (225°,3), (315°,-5), (360°,-1)}$

Part (b) - Solve equation [3 marks] · 3 steps

$4\sin(2x) - 1 = 1$
STEP 1/3 M1

Strategy: Isolate the sine term

Working:

$4\sin(2x) - 1 = 1$

$4\sin(2x) = 2$

$\sin(2x) = \frac{1}{2}$

Keep the argument as $2x$ — don't simplify to $\sin x$ yet
$\boxed{\sin(2x) = \frac{1}{2}}$
↳ from STEP 1:$\sin(2x) = \frac{1}{2}$
STEP 2/3 M1

Strategy: Since $0° \leq x \leq 360°$, we need $0° \leq 2x \leq 720°$

Formula: $\sin\theta = \frac{1}{2} \Rightarrow \theta = 30°, 150°$ (+ multiples of $360°$)

$2x = 30°, 150°, 390°, 510°$

Critical step: double the domain for $2x$ before finding solutions, then add $360°$ to get solutions in $[360°, 720°]$
$\boxed{2x = 30°, 150°, 390°, 510°}$
↳ from STEP 2:$2x = 30°, 150°, 390°, 510°$
STEP 3/3 A1

Strategy: Divide each solution by 2

Working:

$x = 15°, 75°, 195°, 255°$

Common error: forgetting to divide by 2, giving $x = 30°, 150°$ which is wrong
$\boxed{x = 15°, 75°, 195°, 255°}$
