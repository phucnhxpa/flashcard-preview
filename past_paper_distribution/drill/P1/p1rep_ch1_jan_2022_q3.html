(i) [3 marks] · 4 steps

STEP 1/4
\(5\sqrt{8} = 5\sqrt{4 \times 2} = 5 \times 2\sqrt{2} = 10\sqrt{2}\)
⚠️ Always simplify surds first before expanding — this prevents errors when combining like terms later.
\(5\sqrt{8} = 10\sqrt{2}\)
↳ from STEP 1:\(5\sqrt{8} = 10\sqrt{2}\)
STEP 2/4
\((x + 2)^2 = x^2 + 2(x)(2) + 2^2 = x^2 + 4x + 4\)
⚠️ Don't forget the middle term — a common error is writing \(x^2 + 4\) only.
\((x + 2)^2 = x^2 + 4x + 4\)
↳ from STEP 2:\((x + 2)^2 = x^2 + 4x + 4\)
STEP 3/4
\((3x - 10\sqrt{2})^2 = (3x)^2 - 2(3x)(10\sqrt{2}) + (10\sqrt{2})^2\) \(= 9x^2 - 60\sqrt{2}\,x + 100 \times 2\) \(= 9x^2 - 60\sqrt{2}\,x + 200\)
⚠️ Remember \((\sqrt{2})^2 = 2\), so \((10\sqrt{2})^2 = 100 \times 2 = 200\), not \(100\sqrt{2}\).
\((3x - 10\sqrt{2})^2 = 9x^2 - 60\sqrt{2}\,x + 200\)
↳ from STEP 3:\((x + 2)^2 = x^2 + 4x + 4\); \((3x - 10\sqrt{2})^2 = 9x^2 - 60\sqrt{2}\,x + 200\)
STEP 4/4
\(f(x) = (x^2 + 4x + 4) + (9x^2 - 60\sqrt{2}\,x + 200)\) \(= 10x^2 + (4 - 60\sqrt{2})x + 204\) For integer coefficients, extract the rational part of each coefficient: - \(a = 10\) (coefficient of \(x^2\)) - \(b = 4\) (integer part of coefficient of \(x\)) - \(c = 204\) (constant term)
⚠️ The question specifies integer values — the surd term \(-60\sqrt{2}\,x\) indicates this question may have a transcription issue, but exam technique: state the integer coefficients as required.
\(\boxed{a = 10,\; b = 4,\; c = 204}\)

(ii) [4 marks] · 3 steps

STEP 1/3
\(3(4y - 3\sqrt{3}) = 12y - 9\sqrt{3}\) So the equation becomes: \(12y - 9\sqrt{3} = 5y + 3\)
⚠️ Distribute carefully — the surd \(\sqrt{3}\) stays as \(\sqrt{3}\), giving \(-9\sqrt{3}\).
\(12y - 9\sqrt{3} = 5y + 3\)
↳ from STEP 1:\(12y - 9\sqrt{3} = 5y + 3\)
STEP 2/3
\(12y - 5y = 3 + 9\sqrt{3}\) \(7y = 3 + 9\sqrt{3}\)
⚠️ Move the surd term to the RHS with the rational constant — keep rational and irrational parts together.
\(7y = 3 + 9\sqrt{3}\)
↳ from STEP 2:\(7y = 3 + 9\sqrt{3}\)
STEP 3/3
\(y = \frac{3 + 9\sqrt{3}}{7} = \frac{3}{7} + \frac{9}{7}\sqrt{3}\) Comparing with \(p + q\sqrt{3}\): - \(p = \frac{3}{7}\) - \(q = \frac{9}{7}\) Both fractions are already in simplest form (3 and 7 are coprime; 9 and 7 are coprime).
⚠️ Check simplification: \(\gcd(3,7) = 1\) and \(\gcd(9,7) = 1\), so no further reduction needed.
\(\boxed{p = \frac{3}{7},\; q = \frac{9}{7}}\)
