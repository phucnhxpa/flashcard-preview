Q9 — Evaluate trig expressions; sketch y=sin(2x); solve sin(2x)=p

Question

Given that $\sin\alpha = p$, where $0 < \alpha < 90°$

(a) State, in terms of $p$, the value of:  (i) $2\sin(180°-\alpha)$  (ii) $\sin(\alpha-180°)$  (iii) $3 + \sin(180°+\alpha)$   [3]

(b) On a copy of the graph of $y = \sin x$, sketch the graph of $y = \sin 2x$   [2]

(c) Hence find, in terms of $\alpha$, the $x$ coordinates of any points in $0 < x < 180°$ where $\sin 2x = p$   [3]

[8 marks]

Solution [8 marks] · 6 steps

STEP 1/6 M1
Use the identity for sine of supplementary angles: $$\sin(180° - \alpha) = \sin\alpha$$ Therefore: $$2\sin(180° - \alpha) = 2\sin\alpha = 2p$$
Pitfall: Forgetting that \(\sin(180° - \theta) = \sin\theta\) — the sine function is positive and symmetric about 90°.
$$2\sin(180° - \alpha) = \boxed{2p}$$
↳ from STEP 1:From (a)(i): \(\sin(180° - \alpha) = \sin\alpha = p\)
STEP 2/6 M1
Use the odd function property and the identity: $$\sin(\alpha - 180°) = -\sin(180° - \alpha)$$ Since \(\sin(180° - \alpha) = \sin\alpha = p\): $$\sin(\alpha - 180°) = -p$$
Pitfall: Sign error — \(\sin(\theta - 180°) = -\sin(180° - \theta)\) due to sine being odd: \(\sin(-x) = -\sin(x)\).
$$\sin(\alpha - 180°) = \boxed{-p}$$
↳ from STEP 2:From (a)(i): \(\sin\alpha = p\)
STEP 3/6 M1
Use the identity for sine in the third quadrant: $$\sin(180° + \alpha) = -\sin\alpha = -p$$ Therefore: $$3 + \sin(180° + \alpha) = 3 + (-p) = 3 - p$$
Pitfall: In the third quadrant (180° to 270°), sine is negative. Don't confuse \(\sin(180° + \alpha)\) with \(\sin(180° - \alpha)\).
$$3 + \sin(180° + \alpha) = \boxed{3 - p}$$
STEP 4/6 M1
For \(y = \sin(2x)\): • **Period**: \(\frac{360°}{2} = 180°\) (completes 2 full cycles in \([0°, 360°]\)) • **Amplitude**: 1 **Zeros** (where curve meets x-axis): $$x = 0°, 90°, 180°, 270°, 360°$$ **Maximum** (\(y = 1\)): $$x = 45°, 225°$$ **Minimum** (\(y = -1\)): $$x = 135°, 315°$$
Pitfall: Common error is using period 360° instead of 180°. The coefficient 2 halves the period.
$$\boxed{\text{Zeros: }(0°,0), (90°,0), (180°,0), (270°,0), (360°,0)}$$ $$\boxed{\text{Max: }(45°,1), (225°,1) \quad \text{Min: }(135°,-1), (315°,-1)}$$
↳ from STEP 4:From (b): \(y = \sin(2x)\) has period 180°, completing 2 cycles in \([0°, 360°]\)
STEP 5/6 M1
The horizontal line \(y = p\) (where \(0 < p < 1\)) intersects \(y = \sin(2x)\): Since \(\sin(2x)\) completes **2 full cycles** in \([0°, 360°]\), and a horizontal line \(0 < p < 1\) crosses each cycle **twice**: $$\text{Number of solutions} = 2 \times 2 = 4$$
Pitfall: Forgetting that \(\sin(2x)\) has double the frequency — count intersections for each complete cycle.
$$\boxed{4 \text{ solutions}}$$
↳ from STEP 5:From (b): maxima of \(y = \sin(2x)\) occur at \(x = 45°\) and \(x = 225°\)
STEP 6/6 M1
Solve \(\sin(2x) = 1\): $$2x = 90° + 360°n \quad \text{(where }\sin\theta = 1\text{)}$$ For \(0 \leq x \leq 360°\), we need \(0 \leq 2x \leq 720°\): $$2x = 90° \Rightarrow x = 45°$$ $$2x = 450° \Rightarrow x = 225°$$ (\(2x = 810°\) gives \(x = 405°\), outside range)
Pitfall: When \(p = 1\), the line touches the maxima only — there are exactly 2 solutions (not 4), at the peaks of the sine curve.
$$\boxed{x = 45°, \quad x = 225°}$$
