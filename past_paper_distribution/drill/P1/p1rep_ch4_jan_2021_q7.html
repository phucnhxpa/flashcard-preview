Question

f(x) = 2x³ - 15x² + 36x - 1. Find f''(x). Solve f''(x) = 0. Find the values of x for which the curve is concave up.

[9 marks]

(a) Find f'(x) [2 marks] · 1 step

↳ from STEP 0:f(x) = 2x³ - 15x² + 36x - 1
STEP 1/1Differentiate f(x) term by term M1

Strategy: Apply the power rule to each term

Formula: $\frac{d}{dx}(x^n) = nx^{n-1}$

$f'(x) = 3(2)x^2 - 2(15)x + 36$

$f'(x) = 6x^2 - 30x + 36$

$\boxed{f'(x) = 6x^2 - 30x + 36}$

(b) Find f''(x) [2 marks] · 1 step

From previous: see above.
↳ from STEP 0:f'(x) = 6x² - 30x + 36
STEP 1/1Differentiate f'(x) to get the second derivative M1 A1

Strategy: Differentiate f'(x) using the power rule

$f''(x) = 2(6)x - 30$

$f''(x) = 12x - 30$

$\boxed{f''(x) = 12x - 30}$

(c) Solve f''(x) = 0 [2 marks] · 1 step

From previous: see above.
↳ from STEP 0:f''(x) = 12x - 30
STEP 1/1Set the second derivative equal to zero and solve for x M1 A1

Strategy: Solve the linear equation 12x - 30 = 0

$12x - 30 = 0$

$12x = 30$

$x = \frac{30}{12} = \frac{5}{2}$

Note: This is the point of inflection where concavity changes
$\boxed{x = \frac{5}{2}}$

(d) Find where curve is concave up [3 marks] · 1 step

From previous: see above.
↳ from STEP 0:f''(x) = 12x - 30
STEP 1/1A curve is concave up where f''(x) > 0 M1 M1 A1

Strategy: Find where the second derivative is positive

Definition: Concave up means f''(x) > 0

$12x - 30 > 0$

$12x > 30$

$x > \frac{30}{12}$

$x > \frac{5}{2}$

Watch: Don't confuse concave up (f'' > 0) with concave down (f'' < 0). Concave up looks like a smile ∪
$\boxed{x > \frac{5}{2}}$
