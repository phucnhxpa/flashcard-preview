Jan 2019 Q7 — Particle A on table, B on rough incline; find $T$, $a$, pulley force [16m]

Question

Particle A (mass $m$) lies on a rough horizontal table. A light inextensible string connects A over a smooth pulley at the table edge to particle B (mass $3m$) on a rough inclined plane. $\tan\alpha = \tfrac{4}{3}$ so $\sin\alpha=\tfrac{4}{5}$, $\cos\alpha=\tfrac{3}{5}$. Coefficient of friction: $\mu_A = \tfrac{1}{5}$ (table), $\mu_B = \tfrac{1}{3}$ (incline). System released from rest; B slides down the slope.

(a) Find the tension $T$ in the string. [11]

(b) State where the light-string assumption is used. [1]

(c) Find the magnitude of the force on the pulley. [4]

[16 marks]

Part (a) — Find $T$ and $a$ [11 marks] · 3 steps

STEP 1/3

Strategy: decode all modelling words before writing equations. Each phrase removes or fixes an unknown.

Assumptions called out:

• Light inextensible string → tension $T$ is uniform throughout; both particles share one acceleration $a$.

• Smooth pulley → no friction at pulley, so tension is same on both sides.

• Particles → model as point masses; no rotational effects.

Geometry (from $\tan\alpha = 4/3$, 3-4-5 triangle):

$$\sin\alpha = \frac{4}{5}, \quad \cos\alpha = \frac{3}{5}$$

Normal reactions and friction (B slides DOWN → friction on B acts UP slope):

On A (horizontal table, no vertical acceleration):

$$N_A = mg, \quad F_A = \mu_A N_A = \frac{1}{5}mg$$

On B (perpendicular to incline, no acceleration perpendicular to slope):

$$N_B = 3mg\cos\alpha = 3mg \cdot \frac{3}{5} = \frac{9mg}{5}$$

$$F_B = \mu_B N_B = \frac{1}{3} \cdot \frac{9mg}{5} = \frac{3mg}{5}$$

Friction direction trap: B slides DOWN the slope, so kinetic friction on B acts UP the slope — it opposes the motion. The tension T also acts up the slope. Both oppose the downward slide of B. The weight component $3mg\sin\alpha$ drives B down.
$N_A = mg$; $F_A = \frac{mg}{5}$; $N_B = \frac{9mg}{5}$; $F_B = \frac{3mg}{5}$
↳ from STEP 1:From STEP 1: $F_A = mg/5$; $F_B = 3mg/5$; shared acceleration $a$.
STEP 2/3

Formula (Newton’s 2nd Law): $\Sigma F = ma$ for each particle separately, taking the direction of motion as positive.

Particle A (moves right toward pulley, positive = right):

$$T - F_A = ma \implies T - \frac{mg}{5} = ma \quad \cdots (1)$$

Particle B (moves down the slope, positive = down slope):

$$3mg\sin\alpha - T - F_B = 3ma$$

$$3mg \cdot \frac{4}{5} - T - \frac{3mg}{5} = 3ma$$

$$\frac{12mg}{5} - \frac{3mg}{5} - T = 3ma$$

$$\frac{9mg}{5} - T = 3ma \quad \cdots (2)$$

Sign logic on B: Weight component $3mg\sin\alpha$ drives B DOWN (+ve), tension $T$ pulls B UP the slope (−ve from B's perspective), friction $F_B$ acts UP the slope (−ve). All three correctly signed for "down-slope = positive".
Eq (1): $T - mg/5 = ma$; Eq (2): $9mg/5 - T = 3ma$
↳ from STEP 2:From STEP 2: Eq(1) $T - mg/5 = ma$; Eq(2) $9mg/5 - T = 3ma$.
STEP 3/3

Strategy: add (1) and (2) to eliminate $T$:

$$(T - \frac{mg}{5}) + (\frac{9mg}{5} - T) = ma + 3ma$$

$$\frac{9mg}{5} - \frac{mg}{5} = 4ma$$

$$\frac{8mg}{5} = 4ma \implies a = \frac{2g}{5}$$

Substitute $a = 2g/5$ back into (1):

$$T = ma + \frac{mg}{5} = m \cdot \frac{2g}{5} + \frac{mg}{5} = \frac{2mg+mg}{5} = \frac{3mg}{5}$$

$$\boxed{\; a = \frac{2g}{5}, \quad T = \frac{3mg}{5} \;}$$

Sense check: $a = 2g/5 < g$ ✓ (system can't accelerate faster than free fall). T is less than B's weight component $3mg\sin\alpha = 12mg/5$ ✓ (B accelerates). T is also less than A's weight $mg$ ✓ (A doesn't accelerate faster than T allows).
$\boxed{\,a = \tfrac{2g}{5}, \quad T = \tfrac{3mg}{5}\,}$

Part (b) — Light-string assumption [1 marks] · 1 step

From (a): $T = 3mg/5$ used as the same tension throughout the string.
STEP 1/1

Answer: The assumption that the string is light (massless) is used when the tension is taken as the same value $T$ on both sides of the pulley.

If the string had mass, different sections would have different tensions (each section must support the weight of string below it plus transmit the load). A light string eliminates this complication so one scalar $T$ applies throughout.

Exam exact phrasing: "The tension is the same throughout the string" — this is the mark-scheme answer. Write it word-for-word in the exam.
The tension $T$ is the same on both sides of the pulley — possible only because the string is light.

Part (c) — Force on pulley [4 marks] · 1 step

From (a): $T = 3mg/5$; string on table side pulls pulley horizontally; string on incline side pulls pulley at angle $\alpha$ below horizontal.
STEP 1/1

Strategy: the pulley is smooth so tension magnitude $T$ is the same on both sides. The string exerts two forces ON the pulley: one from each side. Take rightward = $+x$, upward = $+y$.

Table-side string: pulls pulley horizontally toward A (leftward):

$$\vec{F}_1 = (-T,\; 0) = \left(-\frac{3mg}{5},\; 0\right)$$

Incline-side string: B is down and to the right of the pulley along the slope (the slope descends away from the table). The string pulls the pulley toward B, i.e. at angle $\alpha$ below horizontal, pointing away from the table (rightward + downward):

$$\vec{F}_2 = (+T\cos\alpha,\; -T\sin\alpha) = \left(\frac{3mg}{5} \cdot \frac{3}{5},\; -\frac{3mg}{5} \cdot \frac{4}{5}\right) = \left(\frac{9mg}{25},\; -\frac{12mg}{25}\right)$$

Resultant on pulley:

$$F_x = -\frac{3mg}{5} + \frac{9mg}{25} = -\frac{15mg}{25} + \frac{9mg}{25} = -\frac{6mg}{25}$$

$$F_y = 0 - \frac{12mg}{25} = -\frac{12mg}{25}$$

$$|F| = \frac{mg}{25}\sqrt{6^2 + 12^2} = \frac{mg}{25}\sqrt{36+144} = \frac{mg}{25}\sqrt{180} = \frac{mg}{25} \cdot 6\sqrt{5}$$

$$\boxed{\;|F| = \frac{6\sqrt{5}}{25}\,mg\;}$$

Common error: adding the tensions as scalars gives $2T = 6mg/5$ — wrong! You must resolve into components because the two string segments are NOT parallel. $\sqrt{180} = \sqrt{36\cdot5} = 6\sqrt{5}$ ✓.
$\boxed{\,|F|_{\text{pulley}} = \frac{6\sqrt{5}}{25}mg\,}$
