Logarithms — Equation to Cubic & Solving

Question

(a) Given that

$$2\log_4(x+3) + \log_4 x = \tfrac{1}{2} + \log_4(4x + 2)$$

show that $x^3 + 6x^2 + x - 4 = 0$ [4]

(b) Given also that $-1$ is a root of the equation $x^3 + 6x^2 + x - 4 = 0$,

 (i) use algebra to find the other two roots of the equation. [3]

 (ii) Hence solve $2\log_4(x+3) + \log_4 x = \tfrac{1}{2} + \log_4(4x+2)$ [1]

[8 marks]

Part (a) — Show x³ + 6x² + x − 4 = 0 [4 marks] · 4 steps

$2\log_4(x+3) + \log_4 x = \frac{1}{2} + \log_4(4x+2)$
STEP 1/4 B1

Simplify $\frac{1}{2}$: Note $\frac{1}{2} = \log_4 2$ (since $\log_4 4^{1/2} = \log_4 2$)

Also use power rule: $2\log_4(x+3) = \log_4(x+3)^2$

So equation becomes:

$$\log_4(x+3)^2 + \log_4 x = \log_4 2 + \log_4(4x+2)$$

$\log_4 2 = \frac{1}{2}$ because $4^{1/2} = 2$
Carry-in:$\log_4(x+3)^2 + \log_4 x = \log_4 2 + \log_4(4x+2)$
STEP 2/4 M1

Combine logs (product rule $\log A + \log B = \log AB$):

$$\log_4\!\left[x(x+3)^2\right] = \log_4\!\left[2(4x+2)\right]$$

So: $x(x+3)^2 = 2(4x+2)$

Carry-in:$x(x+3)^2 = 2(4x+2)$
STEP 3/4 A1

Expand LHS: $x(x^2 + 6x + 9) = x^3 + 6x^2 + 9x$

RHS: $8x + 4$

$$x^3 + 6x^2 + 9x = 8x + 4$$

Carry-in:$x^3+6x^2+9x = 8x+4$
STEP 4/4 A1*

$$x^3 + 6x^2 + 9x - 8x - 4 = 0$$

$$\boxed{x^3 + 6x^2 + x - 4 = 0}$$

Part (b)(i) — Other two roots [3 marks] · 2 steps

$x^3 + 6x^2 + x - 4 = 0$; $-1$ is a root
STEP 1/2 M1

Factor out $(x+1)$:

Divide $x^3 + 6x^2 + x - 4$ by $(x+1)$:

$$x^3 + 6x^2 + x - 4 = (x+1)(x^2 + 5x - 4)$$

Use polynomial long division or inspection: $(x+1)(x^2+5x-4) = x^3+5x^2-4x+x^2+5x-4$ ✓
Carry-in:$(x+1)(x^2+5x-4)=0$
STEP 2/2 dM1 A1

Solve $x^2 + 5x - 4 = 0$:

$$x = \frac{-5 \pm \sqrt{25 + 16}}{2} = \frac{-5 \pm \sqrt{41}}{2}$$

Other roots: $x = \dfrac{-5 + \sqrt{41}}{2}$ and $x = \dfrac{-5 - \sqrt{41}}{2}$

Part (b)(ii) — Solve original log equation [1 mark] · 1 step

Roots: $x = -1$, $x = \frac{-5\pm\sqrt{41}}{2}$; log requires $x > 0$, $x+3 > 0$, $4x+2 > 0$
STEP 1/1 B1

Domain restriction: $x > 0$ (for $\log_4 x$ to be defined)

$x = -1 < 0$ ✗; $\frac{-5-\sqrt{41}}{2} \approx -5.7 < 0$ ✗

$\frac{-5+\sqrt{41}}{2} \approx \frac{-5+6.4}{2} \approx 0.70 > 0$ ✓

$x = \dfrac{-5 + \sqrt{41}}{2}$
Only the positive root is valid for the original log equation
