Coordinate Geometry: Tangent Condition

Question

The circle $C$ has centre $X(3, 5)$ and radius $r$.

The line $l$ has equation $y = 2x + k$, where $k$ is a constant.

(a) Show that if $l$ and $C$ intersect, then $5x^2 + (4k - 26)x + k^2 - 10k + 34 - r^2 = 0$ [3]

(b) Hence, given that $l$ is a tangent to $C$, show that $5r^2 = (k + p)^2$, where $p$ is a constant to be found. [3]

(c) Given that $l$ is a tangent to $C$, the line $l$ meets the $y$-axis at the point $A$ and touches $C$ at the point $B$. Given also that $AB = 2r$, find the value of $k$. [6]

[12 marks]

Part (a) — Derive intersection quadratic [3 marks] · 3 steps

STEP 1/3 M1

Circle equation and substitution

Circle $C$: $(x-3)^2 + (y-5)^2 = r^2$

Substitute $y = 2x + k$:

$$(x-3)^2 + (2x + k - 5)^2 = r^2$$

Must substitute the line equation into the circle — not solve separately.
$\boxed{(x-3)^2 + (2x+k-5)^2 = r^2}$
Carry-in from previous step$(x-3)^2 + (2x+k-5)^2 = r^2$
STEP 2/3 A1

Expand each bracket

$$(x-3)^2 = x^2 - 6x + 9$$

$$(2x + k - 5)^2 = 4x^2 + 4(k-5)x + (k-5)^2$$

$$= 4x^2 + (4k - 20)x + k^2 - 10k + 25$$

Common error: writing $4kx$ instead of $4(k-5)x$ — the cross-term must include the $-5$.
$\boxed{4x^2 + (4k-20)x + k^2 - 10k + 25}$
Carry-in from previous stepExpanded brackets
STEP 3/3 A1

Combine and simplify

$$x^2 - 6x + 9 + 4x^2 + (4k-20)x + k^2 - 10k + 25 = r^2$$

$$5x^2 + (4k - 26)x + k^2 - 10k + 34 - r^2 = 0$$

Collect $x^2$ terms: $1 + 4 = 5$. Collect $x$ terms: $-6 + 4k - 20 = 4k - 26$.
$\boxed{5x^2 + (4k-26)x + k^2 - 10k + 34 - r^2 = 0}$

Part (b) — Tangent condition via discriminant [3 marks] · 3 steps

From (a): $5x^2 + (4k-26)x + k^2 - 10k + 34 - r^2 = 0$
Carry-in from previous step$5x^2 + (4k-26)x + k^2 - 10k + 34 - r^2 = 0$
STEP 1/3 M1

Tangent condition: discriminant = 0

For quadratic $ax^2 + bx + c = 0$, tangent means one repeated root:

$$\Delta = b^2 - 4ac = 0$$

Here: $a = 5$, $b = 4k - 26$, $c = k^2 - 10k + 34 - r^2$

Tangent ⟹ line touches circle at exactly one point ⟹ discriminant equals zero.
$\boxed{(4k-26)^2 - 20(k^2 - 10k + 34 - r^2) = 0}$
Carry-in from previous step$(4k-26)^2 - 20(k^2 - 10k + 34 - r^2) = 0$
STEP 2/3 A1

Expand discriminant equation

$$(4k-26)^2 = 16k^2 - 208k + 676$$

$$20(k^2 - 10k + 34 - r^2) = 20k^2 - 200k + 680 - 20r^2$$

$$16k^2 - 208k + 676 - 20k^2 + 200k - 680 + 20r^2 = 0$$

$$-4k^2 - 8k - 4 + 20r^2 = 0$$

Watch signs carefully when distributing the $-20$ through the bracket.
$\boxed{-4k^2 - 8k - 4 + 20r^2 = 0}$
Carry-in from previous step$-4k^2 - 8k - 4 + 20r^2 = 0$
STEP 3/3 A1

Rearrange to find $p$

$$20r^2 = 4k^2 + 8k + 4$$

$$20r^2 = 4(k^2 + 2k + 1) = 4(k+1)^2$$

$$5r^2 = (k+1)^2$$

Recognise perfect square: $k^2 + 2k + 1 = (k+1)^2$.
$\boxed{p = 1}$

Part (c) — Find k using AB = 2r [6 marks] · 5 steps

From (b): $5r^2 = (k+1)^2$
Carry-in from previous step$5r^2 = (k+1)^2$
STEP 1/5 B1

Find coordinates of point A

Line $l$: $y = 2x + k$ meets the $y$-axis where $x = 0$:

$$y = 2(0) + k = k$$

So $A = (0, k)$

The $y$-axis has equation $x = 0$ — substitute this into the line equation.
$\boxed{A = (0, k)}$
Carry-in from previous step$A = (0, k)$, centre $X(3, 5)$
STEP 2/5 M1

Calculate distance AX

Using distance formula:

$$AX^2 = (3-0)^2 + (5-k)^2$$

$$= 9 + 25 - 10k + k^2$$

$$= k^2 - 10k + 34$$

Always write $AX^2$ first to avoid unnecessary square roots.
$\boxed{AX^2 = k^2 - 10k + 34}$
Carry-in from previous step$AX^2 = k^2 - 10k + 34$, $XB = r$
STEP 3/5 M1

Apply Pythagoras in triangle AXB

Since $XB \perp l$ (radius to tangent point), triangle $AXB$ is right-angled at $B$:

$$AX^2 = AB^2 + XB^2$$

$$AB^2 = AX^2 - XB^2 = k^2 - 10k + 34 - r^2$$

Common error: using $AB = AX - XB$ linearly. Must use Pythagoras since angle at $B$ is $90°$.
$\boxed{AB^2 = k^2 - 10k + 34 - r^2}$
Carry-in from previous step$AB^2 = k^2 - 10k + 34 - r^2$, $AB = 2r$
STEP 4/5 M1

Apply condition AB = 2r

Given $AB = 2r$, so $AB^2 = 4r^2$:

$$4r^2 = k^2 - 10k + 34 - r^2$$

$$5r^2 = k^2 - 10k + 34$$

Rearranging gives an expression for $5r^2$ that we can equate with part (b).
$\boxed{5r^2 = k^2 - 10k + 34}$
Carry-in from previous step$5r^2 = k^2 - 10k + 34$ and $5r^2 = (k+1)^2$
STEP 5/5 M1A1

Equate expressions and solve for k

From part (b): $5r^2 = (k+1)^2 = k^2 + 2k + 1$

From geometry: $5r^2 = k^2 - 10k + 34$

Equating:

$$k^2 + 2k + 1 = k^2 - 10k + 34$$

$$12k = 33$$

$$k = \frac{33}{12} = \frac{11}{4}$$

The $k^2$ terms cancel — always check if this simplification occurs.
$\boxed{k = \frac{11}{4}}$
