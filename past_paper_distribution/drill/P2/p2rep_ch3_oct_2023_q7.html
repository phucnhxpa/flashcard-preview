Coordinate Geometry — Circle, Perpendicular & Equation

Question

The circle $C$ has centre $X(4, -3)$. The line $l$ has equation $y = \dfrac{5x}{2} + \dfrac{55}{2}$.

Given that $l$ is the tangent to $C$ at the point $N$,

(a) show that an equation for the straight line passing through $X$ and $N$ is $2x + 5y + 7 = 0$ [3]

(b) Hence find

 (i) the coordinates of $N$,

 (ii) an equation for $C$. [5]

[8 marks]

Part (a) — Show line XN is 2x + 5y + 7 = 0 [3 marks] · 3 steps

Centre $X(4, -3)$, tangent $l: y = \frac{5x}{2} + \frac{55}{2}$
STEP 1/3 M1

Gradient of tangent $l$:

$$m_l = \frac{5}{2}$$

Perpendicular gradient (radius XN ⊥ tangent):

$$m_{XN} = -\frac{1}{m_l} = -\frac{2}{5}$$

Perpendicular gradient = negative reciprocal: flip and negate
Carry-in:$m_{XN} = -\frac{2}{5}$, $X(4,-3)$
STEP 2/3 M1

Line through $X(4,-3)$ with gradient $-\frac{2}{5}$:

$$y - (-3) = -\frac{2}{5}(x - 4)$$

$$y + 3 = -\frac{2}{5}(x - 4)$$

Multiply through by 5:

$$5y + 15 = -2(x - 4) = -2x + 8$$

Carry-in:$5y + 15 = -2x + 8$
STEP 3/3 A1

Rearrange to required form:

$$2x + 5y + 15 - 8 = 0$$

$$\boxed{2x + 5y + 7 = 0}$$

Check: substitute $X(4,-3)$: $8 - 15 + 7 = 0$ ✓

Part (b)(i) — Coordinates of N [3 marks] · 3 steps

Line XN: $2x + 5y + 7 = 0$; tangent $l: y = \frac{5x}{2} + \frac{55}{2}$
STEP 1/3 M1

N is intersection of line XN and tangent $l$.

Substitute $y = \frac{5x}{2} + \frac{55}{2}$ into $2x + 5y + 7 = 0$:

$$2x + 5\!\left(\frac{5x}{2} + \frac{55}{2}\right) + 7 = 0$$

$$2x + \frac{25x}{2} + \frac{275}{2} + 7 = 0$$

Carry-in:$2x + \frac{25x}{2} + \frac{275}{2} + 7 = 0$
STEP 2/3 A1

Multiply by 2: $4x + 25x + 275 + 14 = 0$

$$29x + 289 = 0 \implies x = -\frac{289}{29} = -\frac{289}{29}$$

Wait — simplify: $29x = -289 \implies x = -\frac{289}{29}$

Check: $289 = 29 \times 10 - 1$... Actually $-289/29$ doesn't simplify. Let me re-check:

$4x + 25x = 29x$; $275 + 14 = 289$. So $x = -289/29$.

$\gcd(289, 29)$: $289 = 17^2$, $29$ is prime, so no common factor — keep as fraction
Carry-in:$x = -\frac{289}{29}$
STEP 3/3 A1

$y = \frac{5}{2}\!\left(-\frac{289}{29}\right) + \frac{55}{2} = \frac{-1445}{58} + \frac{1595}{58} = \frac{150}{58} = \frac{75}{29}$

$N = \left(-\dfrac{289}{29},\ \dfrac{75}{29}\right)$
Alternatively, solve by elimination: multiply XN eqn and tangent to cancel $y$, giving same answer

Part (b)(ii) — Equation of C [2 marks] · 2 steps

Centre $X(4,-3)$, $N = \left(-\frac{289}{29}, \frac{75}{29}\right)$
STEP 1/2 M1

Radius $r = |XN|$:

$$r^2 = \left(4 - \left(-\tfrac{289}{29}\right)\right)^2 + \left(-3 - \tfrac{75}{29}\right)^2$$

$$= \left(\tfrac{116 + 289}{29}\right)^2 + \left(\tfrac{-87 - 75}{29}\right)^2$$

$$= \left(\tfrac{405}{29}\right)^2 + \left(\tfrac{-162}{29}\right)^2$$

$$= \frac{164025 + 26244}{841} = \frac{190269}{841} = \frac{190269}{841}$$

$190269 / 841 = 226.24...$; keep exact: $r^2 = \frac{190269}{841}$
Carry-in:$r^2 = \frac{190269}{841}$
STEP 2/2 A1

Equation of circle with centre $(4,-3)$:

$$(x - 4)^2 + (y + 3)^2 = r^2$$

$(x-4)^2 + (y+3)^2 = \dfrac{190269}{841}$
Equivalently $r^2 = \frac{190269}{841}$ — exact form required (not decimal)
