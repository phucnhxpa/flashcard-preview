P2 (WMA12/01) · Q9 · Specimen 2018

Compound Angle Identities — R-formula

Question

(a) Express $12\cos x - 5\sin x$ in the form $R\cos(x + \alpha)$, where $R > 0$ and $0^\circ < \alpha < 90^\circ$. Give the value of $\alpha$ to 2 decimal places. [3]

(b) Hence solve, for $0^\circ \le x < 360^\circ$, the equation $$12\cos x - 5\sin x = 7$$ giving your answers to 1 decimal place. [4]

(c) State the maximum value of $12\cos x - 5\sin x$ and find the smallest positive value of $x$ at which this maximum occurs. [2]

[9 marks]

Part (a) — Express in R-formula form [3 marks] · 3 steps

STEP 1/3 M1

Expand the compound angle formula

$$R\cos(x + \alpha) = R\cos x \cos\alpha - R\sin x \sin\alpha$$

Compare with $12\cos x - 5\sin x$:

$$R\cos\alpha = 12 \quad (1)$$

$$R\sin\alpha = 5 \quad (2)$$

The minus sign in the expansion matches the minus in the original expression, so $R\sin\alpha = +5$, not $-5$.
$\boxed{R\cos\alpha = 12,\; R\sin\alpha = 5}$
Carry-in from previous step$R\cos\alpha = 12,\; R\sin\alpha = 5$
STEP 2/3 A1

Find R by squaring and adding

$$(1)^2 + (2)^2: \quad R^2\cos^2\alpha + R^2\sin^2\alpha = 144 + 25$$

$$R^2(\cos^2\alpha + \sin^2\alpha) = 169$$

$$R^2 = 169 \implies R = 13$$

Since $R > 0$, we take the positive square root only.
$\boxed{R = 13}$
Carry-in from previous step$R = 13$
STEP 3/3 A1

Find α by dividing

$$\frac{(2)}{(1)}: \quad \tan\alpha = \frac{5}{12}$$

$$\alpha = \arctan\left(\frac{5}{12}\right) = 22.62^\circ$$

Therefore: $12\cos x - 5\sin x = 13\cos(x + 22.62^\circ)$

Check $\alpha$ is in the correct quadrant: both $R\cos\alpha$ and $R\sin\alpha$ are positive, so $\alpha$ is in Q1 ✓
$\boxed{R = 13,\; \alpha = 22.62^\circ}$

Part (b) — Solve the equation [4 marks] · 4 steps

$12\cos x - 5\sin x = 13\cos(x + 22.62^\circ)$
Carry-in from previous step$12\cos x - 5\sin x = 13\cos(x + 22.62^\circ)$
STEP 1/4 M1

Substitute and isolate cosine

$$13\cos(x + 22.62^\circ) = 7$$

$$\cos(x + 22.62^\circ) = \frac{7}{13}$$

Always simplify to $\cos(\text{angle}) = \text{value}$ before finding solutions.
$\boxed{\cos(x + 22.62^\circ) = \frac{7}{13}}$
Carry-in from previous step$\cos(x + 22.62^\circ) = \frac{7}{13}$
STEP 2/4 A1

Adjust the range for the transformed variable

If $0^\circ \le x < 360^\circ$, then:

$$22.62^\circ \le (x + 22.62^\circ) < 382.62^\circ$$

Principal value: $\arccos\left(\frac{7}{13}\right) = 57.40^\circ$

Forgetting to adjust the range is a common error — always add $\alpha$ to both bounds.
$\boxed{\text{Principal value} = 57.40^\circ}$
Carry-in from previous stepPrincipal value $= 57.40^\circ$, range $[22.62^\circ, 382.62^\circ)$
STEP 3/4 A1

Find all solutions using CAST

Cosine is positive in Q1 and Q4:

$$x + 22.62^\circ = 57.40^\circ \implies x = 34.78^\circ$$

$$x + 22.62^\circ = 360^\circ - 57.40^\circ = 302.60^\circ \implies x = 280.0^\circ$$

Check: $57.40^\circ + 360^\circ = 417.40^\circ > 382.62^\circ$, so no third solution.
$\boxed{x + 22.62^\circ = 57.40^\circ \text{ or } 302.60^\circ}$
Carry-in from previous step$x + 22.62^\circ = 57.40^\circ$ or $302.60^\circ$
STEP 4/4 A1

State final answers to 1 d.p.

$$x = 57.40^\circ - 22.62^\circ = 34.8^\circ$$

$$x = 302.60^\circ - 22.62^\circ = 280.0^\circ$$

Always verify solutions lie within the original range $0^\circ \le x < 360^\circ$.
$\boxed{x = 34.8^\circ,\; 280.0^\circ}$

Part (c) — Maximum value [2 marks] · 2 steps

$12\cos x - 5\sin x = 13\cos(x + 22.62^\circ)$
Carry-in from previous step$12\cos x - 5\sin x = 13\cos(x + 22.62^\circ)$
STEP 1/2 B1

Find the maximum value

The maximum value of $\cos(\theta)$ is $1$.

$$\text{Maximum of } 13\cos(x + 22.62^\circ) = 13 \times 1 = 13$$

For $R\cos(\theta + \alpha)$, the maximum is always $R$ and minimum is $-R$.
$\boxed{\text{Maximum} = 13}$
Carry-in from previous stepMaximum $= 13$
STEP 2/2 B1

Find smallest positive x for maximum

Maximum occurs when $\cos(x + 22.62^\circ) = 1$:

$$x + 22.62^\circ = 0^\circ \implies x = -22.62^\circ \text{ (negative, reject)}$$

$$x + 22.62^\circ = 360^\circ \implies x = 337.38^\circ$$

Smallest positive value: $x = 337.4^\circ$

$\cos\theta = 1$ when $\theta = 0^\circ, 360^\circ, 720^\circ, \ldots$ — choose the one giving smallest positive $x$.
$\boxed{x = 337.4^\circ}$
