P2 (WMA12/01) · Q9 · Specimen 2018

Trigonometric Equations — Double Angle & Quadratic in cos

Question

(i) Solve, for $0 \leqslant \theta < \pi$, the equation

$\sin 3\theta - \sqrt{3}\cos 3\theta = 0$

giving your answers in terms of $\pi$. [3]

(ii) Given that

$4\sin^2 x + \cos x = 4 - k, \quad 0 \leqslant k \leqslant 3$

(a) find $\cos x$ in terms of $k$. [3]

(b) When $k = 3$, find the values of $x$ in the range $0 \leqslant x < 360°$. [3]

[9 marks]

(i) Solve sin 3θ − √3 cos 3θ = 0 for 0 ≤ θ < π [3 marks] · 3 steps

STEP 1
Divide both sides by cos 3θ
$$\frac{\sin 3\theta}{\cos 3\theta} = \sqrt{3}$$ $$\tan 3\theta = \sqrt{3}$$
Dividing through by $\cos 3\theta$ converts to tan form.
$3\theta = \dfrac{\pi}{3}$ (principal value)
STEP 2
Find range for 3θ and all solutions
Since $0 \leqslant \theta < \pi$, we have $0 \leqslant 3\theta < 3\pi$.
$\tan$ has period $\pi$, so add $\pi$ repeatedly: $$3\theta = \frac{\pi}{3},\quad \frac{\pi}{3}+\pi = \frac{4\pi}{3},\quad \frac{\pi}{3}+2\pi = \frac{7\pi}{3}$$
$3\theta = \dfrac{\pi}{3},\; \dfrac{4\pi}{3},\; \dfrac{7\pi}{3}$
STEP 3
Divide by 3 for final answers
$$\theta = \frac{\pi}{9},\quad \frac{4\pi}{9},\quad \frac{7\pi}{9}$$
All three values lie in $[0,\pi)$ and no extra values in range.
$\theta = \dfrac{\pi}{9},\; \dfrac{4\pi}{9},\; \dfrac{7\pi}{9}$

(ii)(a) Find cos x in terms of k [3 marks] · 2 steps

STEP 1
Use sin²x = 1 − cos²x
$$4(1-\cos^2 x) + \cos x = 4 - k$$ $$4 - 4\cos^2 x + \cos x = 4 - k$$ $$4\cos^2 x - \cos x - k = 0$$
This is a quadratic in $\cos x$.
STEP 2
Apply quadratic formula
$$\cos x = \frac{1 \pm \sqrt{1 + 16k}}{8}$$
Using $a=4, b=-1, c=-k$: discriminant $= 1 + 16k$.
$\cos x = \dfrac{1 \pm \sqrt{1+16k}}{8}$

(ii)(b) When k = 3, find x for 0 ≤ x < 360° [3 marks] · 2 steps

$k = 3$: $\cos x = \dfrac{1 \pm \sqrt{49}}{8} = \dfrac{1 \pm 7}{8}$
STEP 1
Evaluate cos x when k = 3
$$\cos x = \frac{1+7}{8} = 1 \quad \text{or} \quad \cos x = \frac{1-7}{8} = -\frac{3}{4}$$
Discard any values of $\cos x$ outside $[-1, 1]$. Both are valid here.
STEP 2
Find all x in [0°, 360°)
$\cos x = 1 \Rightarrow x = 0°$
$\cos x = -\tfrac{3}{4} \Rightarrow x = \arccos(-\tfrac{3}{4}) \approx 138.6°$ and $360° - 138.6° \approx 221.4°$
$x = 0°,\; 139°,\; 221°$ (allow awrt 139 and 221)
