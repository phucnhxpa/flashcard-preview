Integration — Trapezium Rule

Question

$$y = 1 - \log_{10}(\sin x)$$

The table shows some values of $y$ for $0.5 \leq x \leq 3$

$x$0.511.522.53
$y$1.3191.0011.2231.850

(a) Complete the table, giving the missing values of $y$ to 3 decimal places. [2]

(b) Use the trapezium rule, with all the values of $y$ in the completed table, to obtain an estimate for $$\int_{0.5}^{3} (1 - \log_{10}(\sin x))\,dx$$ giving your answer to 2 decimal places. [4]

(c) Using your answer to part (b), and making your method clear, obtain an estimate for $$\int_{0.5}^{3} (3 + \log_{10}(\sin x))\,dx$$ [2]

[8 marks]

Part (a) — Complete the table [2 marks] · 2 steps

STEP 1/2 M1

Substitute $x = 1$ into $y = 1 - \log_{10}(\sin x)$

Calculator must be in radians:

$$\sin 1 \approx 0.841471$$

$$\log_{10}(0.841471) \approx -0.075001$$

$$y = 1 - (-0.075001) = 1.075$$

Pitfall: Using degrees instead of radians gives completely wrong values — $\sin 1°$ is tiny!
$\boxed{y(1) = 1.075}$
Carry-in from previous step$y(1) = 1.075$
STEP 2/2 A1

Substitute $x = 2$ into $y = 1 - \log_{10}(\sin x)$

$$\sin 2 \approx 0.909297$$

$$\log_{10}(0.909297) \approx -0.041275$$

$$y = 1 - (-0.041275) = 1.041$$

Pitfall: Misreading which $y$ values are missing — check column alignment carefully against $x$ values.
$\boxed{y(2) = 1.041}$

Part (b) — Trapezium rule estimate [4 marks] · 3 steps

$y(1) = 1.075$, $y(2) = 1.041$
STEP 1/3 M1

Identify parameters for trapezium rule

$$h = \frac{b - a}{n} = \frac{3 - 0.5}{5} = 0.5$$

6 ordinates: $y_0 = 1.319,\; y_1 = 1.075,\; y_2 = 1.001,\; y_3 = 1.041,\; y_4 = 1.223,\; y_5 = 1.850$

Pitfall: $n$ = number of strips (5), not number of ordinates (6).
$\boxed{h = 0.5}$
Carry-in from previous step$h = 0.5$, all $y$ values identified
STEP 2/3 M1A1

Apply trapezium rule formula

$$\int_a^b y\,dx \approx \frac{h}{2}\left[y_0 + y_n + 2(y_1 + y_2 + \cdots + y_{n-1})\right]$$

$$= \frac{0.5}{2}\left[1.319 + 1.850 + 2(1.075 + 1.001 + 1.041 + 1.223)\right]$$

$$= 0.25\left[3.169 + 2(4.340)\right]$$

Pitfall: First and last ordinates are NOT doubled — only interior values.
$\boxed{0.25 \times 11.849}$
Carry-in from previous step$0.25 \times 11.849$
STEP 3/3 A1

Calculate final value

$$= 0.25 \times 11.849 = 2.96225$$

$$\approx 2.96 \text{ (2 d.p.)}$$

Pitfall: Keep extra precision in working — only round the final answer to 2 d.p.
$\boxed{2.96}$

Part (c) — Related integral [2 marks] · 2 steps

$\int_{0.5}^{3} (1 - \log_{10}(\sin x))\,dx \approx 2.96$
STEP 1/2 M1

Rewrite integrand using $y = 1 - \log_{10}(\sin x)$

Rearranging: $\log_{10}(\sin x) = 1 - y$

So: $3 + \log_{10}(\sin x) = 3 + (1 - y) = 4 - y$

$$\int_{0.5}^{3} (3 + \log_{10}(\sin x))\,dx = \int_{0.5}^{3} 4\,dx - \int_{0.5}^{3} y\,dx$$

Pitfall: Failing to connect $\log_{10}(\sin x)$ to $y$ from part (a) — this is the key insight!
$\boxed{\int_{0.5}^{3} 4\,dx - 2.96}$
Carry-in from previous step$\int_{0.5}^{3} 4\,dx - 2.96$
STEP 2/2 A1

Evaluate the constant integral

$$\int_{0.5}^{3} 4\,dx = 4(3 - 0.5) = 4 \times 2.5 = 10$$

$$\therefore \int_{0.5}^{3} (3 + \log_{10}(\sin x))\,dx = 10 - 2.96 = 7.04$$

Pitfall: Don't recalculate using trapezium rule — use part (b) result directly as instructed.
$\boxed{7.04}$
