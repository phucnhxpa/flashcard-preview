Q4 — Tree, Total Probability, Bayes

Question

A bag contains a large number of coloured counters. Each counter is labelled A, B or C: 30% A, 45% B, the rest C. Of those labelled A, 2% are red; of those labelled B, 4% are red; of those labelled C, 6% are red. One counter is selected at random.
(a) Complete the tree diagram. (2)
(b) Find $P(\text{labelled A and not red})$. (2)
(c) Find $P(\text{red})$. (2)
(d) Given the counter is red, find $P(\text{labelled C})$. (3)

Needed before revealing this part

$P(A)=0.30$, $P(B)=0.45$, so $P(C)=0.25$. Red conditionals: $0.02,0.04,0.06$; not-red conditionals: $0.98,0.96,0.94$.

STEP 1/1Part (a)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

Setup: $P(A)=0.30$, $P(B)=0.45$, $P(C)=1-0.30-0.45=0.25$. Conditional reds: $0.02, 0.04, 0.06$. Not-reds: $0.98, 0.96, 0.94$.

Mistake guard from the attempt/revision notes.

What went wrong. Bayes-style questions trip students on direction: $P(C \mid R)$ and $P(R \mid C)$ are not interchangeable.

Revision-guide trap (d).. Quoting $P(R \mid C) = 0.06$ as the answer.

Part (b)[2 marks] · 1 step

Needed before revealing this part

For labelled A and not red: $P(A)=0.30$ and $P(\text{not red}\mid A)=0.98$.

STEP 1/1Part (b)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

(b)
$$P(A \cap \text{not red}) = 0.30 \times 0.98 = \boxed{0.294}.$$

Target: $\boxed{0.294}$

Part (c)[2 marks] · 1 step

Needed before revealing this part

Branch products: $A\cap R=0.006$, $B\cap R=0.018$, $C\cap R=0.015$. Total red is their sum.

STEP 1/1Part (c)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

(c) Total probability:
$$P(R) = 0.30(0.02) + 0.45(0.04) + 0.25(0.06) = 0.006 + 0.018 + 0.015 = \boxed{0.039}.$$

Target: $\boxed{0.039}$

Part (d)[3 marks] · 1 step

Needed before revealing this part

From (c): $P(R)=0.039$. Numerator $P(C\cap R)=0.25\times0.06=0.015$. Use $P(C\mid R)=P(C\cap R)/P(R)$.

STEP 1/1Part (d)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

(d) Bayes:
$$P(C \mid R) = \frac{P(C)\,P(R \mid C)}{P(R)} = \frac{0.25 \times 0.06}{0.039} = \frac{0.015}{0.039} = \frac{15}{39} = \frac{5}{13} \approx \boxed{0.385}.$$

Target: $\boxed{0.385}$
