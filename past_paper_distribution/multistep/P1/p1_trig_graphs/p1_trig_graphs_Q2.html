Q2 — Trig transformation: f(x) = cos 2x° and the "exactly two solutions" range

Question

Figure 2 shows a sketch of the curve y = f(x), where f(x) = cos 2x°, 0 ≤ x ≤ k. The point Q and the point R(k, 0) lie on the curve.
(a) State (i) the coordinates of Q, (ii) the value of k. (3)
(b) Given that there are exactly two solutions to the equation cos 2x° = p in the region 0 ≤ x ≤ k, find the range of possible values for p. (2)

[2 marks]

Part (a)2 steps

STEP 1/2

Coordinates of Q and value of k. (3)

Step 1: Understand cos 2x°. It is cos x° with x → 2x: a horizontal squash, scale factor ½, so the period is 360/2 = 180°. Every point maps (x, y) ↦ (x/2, y).

STEP 2/2

Step 2: Find the minimum Q. The minimum has y = −1: cos 2x° = −1 ⟹ 2x = 180 ⟹ x = 90. So Q = (90°, −1).

Step 3: Find k = R. Zeros where 2x = 90, 270, 450 ⟹ x = 45, 135, 225. The curve runs (0,1)→(45,0)→(90,−1)Q→(135,0)→(180,1)→(225,0)R, so R is the third zero: k = 225. (Not 270 — that would be the next minimum, not the x-intercept.)

Result: Q = (90°, −1); k = 225.

Part (b)[2 marks] · 3 steps

STEP 1/3

Range of p for exactly two solutions of cos 2x° = p on 0 ≤ x ≤ 225. (2)

Step 1: Break the curve into monotonic pieces. S1: x∈[0,90], y: 1→−1. S2: x∈[90,180], y: −1→1. S3: x∈[180,225], y: 1→0.

STEP 2/3

Step 2: Count how many pieces the line y = p crosses for each band of p. p = 1 → x = 0 and x = 180 → 2 solutions ✓. 0 < p < 1 → one on each of S1, S2, S3 → 3 solutions ✗. p = 0 → x = 45, 135, 225 → 3 solutions ✗. −1 < p < 0 → one on S1, one on S2, none on S3 → 2 solutions ✓. p = −1 → x = 90 only → 1 solution ✗.

Step 3: Collect the cases that give exactly two. Only −1 < p < 0 and p = 1.

Result: −1 < p < 0, or p = 1.

STEP 3/3

Common trap (this part).

Answered the over-broad −1 < p < 1. That is wrong because the curve rises back to (180, 1) then falls to (225, 0): any positive p < 1 (and p = 0) is met THREE times across the three monotonic pieces.

Exam checklist.

For "exactly n solutions of curve = p", split the curve into monotonic pieces and count how many the line y = p cuts in each band; underline "coordinates" so Q is given as a point.

Common pitfalls.

f(ax) vs f(x/a) confusion — f(ax) is a stretch of scale factor 1/a (a>1 squashes), anchor it with (x, y) ↦ (x/a, y); writing k = 270 instead of 225.
