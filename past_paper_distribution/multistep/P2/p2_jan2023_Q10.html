Strategy (proof by exhaustion): every integer $n$ falls into exactly one of three residue classes modulo 3: $n = 3k$, $n = 3k+1$, or $n = 3k+2$ for some integer $k$. The case $n = 3k$ is done. We must complete the two remaining cases.
STEP 1/3 M1 â€” case n = 3k+1

Recognise the trigger. Proof by exhaustion partitions $\mathbb{Z}$ into the three residue classes mod 3 and checks each separately. The classes $\{3k, 3k+1, 3k+2 \,:\, k \in \mathbb{Z}\}$ cover every integer with no overlap, so verifying the claim on each class proves it for all $n \in \mathbb{Z}$.

Case 2: $n = 3k+1$ for some integer $k$. Substitute into $2n^2 + n + 1$ and expand.

Square first:

$$(3k+1)^2 \;=\; (3k)^2 + 2(3k)(1) + 1^2 \;=\; 9k^2 + 6k + 1.$$

Multiply by 2:

$$2(3k+1)^2 \;=\; 2(9k^2 + 6k + 1) \;=\; 18k^2 + 12k + 2.$$

Add $n + 1 = (3k+1) + 1 = 3k + 2$:

$$2n^2 + n + 1 \;=\; \underbrace{18k^2 + 12k + 2}_{2n^2} \;+\; \underbrace{3k + 2}_{n + 1}$$

$$\;=\; 18k^2 + 15k + 4.$$

Extract a factor of 3. The first two terms are clearly divisible by 3; for the constant 4, split as $4 = 3 + 1$:

$$18k^2 + 15k + 4 \;=\; 3(6k^2 + 5k + 1) + 1.$$

Since $6k^2 + 5k + 1$ is an integer (sum/product of integers), this is in the form $3M + 1$ where $M = 6k^2 + 5k + 1 \in \mathbb{Z}$.

Therefore $2n^2 + n + 1$ leaves remainder $\mathbf{1}$ on division by 3, so is not divisible by 3 in this case. âś“

The structural slip is expanding $(3k+1)^2$ as $9k^2 + 1$ (forgetting the cross term) or $9k^2 + 3k + 1$ (cross term wrong factor). Always write the binomial expansion as three separate terms â€” $(3k)^2 + 2(3k)(1) + 1^2$ â€” to make the arithmetic visible. Splitting the constant $4 = 3 + 1$ at the end is the key algebraic move: you're forcing the expression into the form $3M + r$ with $0 \le r < 3$, which is precisely what "remainder modulo 3" means.

Case 3: $n = 3k+2$ for some integer $k$.

Square first:

$$(3k+2)^2 \;=\; (3k)^2 + 2(3k)(2) + 2^2 \;=\; 9k^2 + 12k + 4.$$

Multiply by 2:

$$2(3k+2)^2 \;=\; 18k^2 + 24k + 8.$$

Add $n + 1 = (3k+2) + 1 = 3k + 3$:

$$2n^2 + n + 1 \;=\; \underbrace{18k^2 + 24k + 8}_{2n^2} \;+\; \underbrace{3k + 3}_{n+1}$$

$$\;=\; 18k^2 + 27k + 11.$$

Extract a factor of 3. The first two terms ($18k^2$ and $27k$) are divisible by 3. For the constant 11, decompose as $11 = 9 + 2$, where $9 = 3 \cdot 3$:

$$18k^2 + 27k + 11 \;=\; (18k^2 + 27k + 9) + 2 \;=\; 3(6k^2 + 9k + 3) + 2.$$

Verify the split: $3 \cdot (6k^2 + 9k + 3) = 18k^2 + 27k + 9$, then $+ 2$ gives $18k^2 + 27k + 11$ âś“.

$6k^2 + 9k + 3$ is an integer (sum/product of integers), so $2n^2 + n + 1$ has the form $3M + 2$ where $M \in \mathbb{Z}$.

Therefore $2n^2 + n + 1$ leaves remainder $\mathbf{2}$ on division by 3, so is not divisible by 3 in this case. âś“

STEP 3/3 A1 â€” conclusion

Recognise the trigger. Every proof by exhaustion ends with a one-line conclusion that ties the cases together and states the conclusion.

State the conclusion explicitly.

Every integer $n$ is of exactly one of the three forms $3k$, $3k+1$, or $3k+2$ for some integer $k$. In each of these three cases, the expression $2n^2 + n + 1$ has been shown to leave a nonzero remainder on division by 3 (remainder 1, 1, and 2 respectively). Hence in every case $2n^2 + n + 1$ is not divisible by 3.

Therefore for all integers $n$, $\;2n^2 + n + 1\;$ is not divisible by 3. $\;\blacksquare$

$$\boxed{\,\text{Proof complete by exhaustion of the three residue classes mod 3.}\,}$$
