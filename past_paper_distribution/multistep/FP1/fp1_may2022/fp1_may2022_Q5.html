Q5 — Roots of a Quadratic, Building a New Equation

Question

The quadratic equation $2x^2 - 3x + 5 = 0$ has roots $\alpha$ and $\beta$.
(a) Write down the value of $(\alpha + \beta)$ and the value of $\alpha\beta$. --- (2)
(b) Determine the value of (i) $\alpha^2 + \beta^2$, (ii) $\alpha^3 + \beta^3$. --- (4)
(c) Find a quadratic equation which has roots $(\alpha^3 - \beta)$ and $(\beta^3 - \alpha)$. Give your answer in the form $px^2 + qx + r = 0$ where $p$, $q$ and $r$ are integers to be determined. --- (5)

[10 marks]

Use the visible question and any carry-in strip first; reveal the grey work boxes only after attempting that part.

Part (a)[2 marks] · 1 step

STEP 1/1Part (a)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

(a) For $2x^2 - 3x + 5 = 0$: $\alpha+\beta = -\dfrac{-3}{2} = \dfrac{3}{2}$, $\alpha\beta = \dfrac{5}{2}$.

Mistake guard from the attempt/revision notes.

What went wrong. From the voice note on (c):

"Because we have two equations and two unknowns, we're free to select the value for one of the variables. After that, because we have two equations and two unknowns, we can just solve for the variable, and that's it."

That description is muddled. The correct framework is not "two equations in two unknowns" --- it is "compute the new sum $S$ and new product $P$, then write $x^2 - Sx + P = 0$". The "free to select one variable" remark only applies at the very end, when clearing fractions: any positive multiplier $p$ works, and you conventionally pick the smallest $p$ that makes $q,r$ integers.

Phuc's incorrect approach (for (c)). He tried to set up "$\alpha + \beta = -q/p$ and $\alpha\beta = r/p$" as two equations in the three unknowns $p,q,r$ and pick a value. That direction is fine in principle but Phuc didn't first compute the new sum and product, so he had no numbers to plug in.

From the Loom. > "Try to see the common factor" --- Mr. Fuat repeatedly emphasises factoring $\tfrac13 n$ or $(n+1)$ out at the right moment so the algebra collapses; same principle applies here when forming the new equation.

Part (b)[4 marks] · 2 steps

Needed before revealing this part

From (a): $\alpha+\beta=\tfrac32$ and $\alpha\beta=\tfrac52$. Use $\alpha^2+\beta^2=(\alpha+\beta)^2-2\alpha\beta$ and $\alpha^3+\beta^3=(\alpha+\beta)^3-3\alpha\beta(\alpha+\beta)$.

STEP 1/2Part (b)(i)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

(b)(i)
$$\alpha^2 + \beta^2 = (\alpha+\beta)^2 - 2\alpha\beta = \left(\tfrac32\right)^2 - 2\cdot\tfrac52 = \tfrac94 - 5 = -\tfrac{11}{4}.$$

STEP 2/2Part (b)(ii)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

(b)(ii)
$$\alpha^3 + \beta^3 = (\alpha+\beta)^3 - 3\alpha\beta(\alpha+\beta) = \left(\tfrac32\right)^3 - 3\cdot\tfrac52\cdot\tfrac32 = \tfrac{27}{8} - \tfrac{45}{4} = \tfrac{27}{8} - \tfrac{90}{8} = -\tfrac{63}{8}.$$

Part (c)[5 marks] · 1 step

Needed before revealing this part

Carry forward: $\alpha+\beta=\tfrac32$, $\alpha\beta=\tfrac52$, $\alpha^2+\beta^2=-\tfrac{11}{4}$, $\alpha^3+\beta^3=-\tfrac{63}{8}$. New roots are $\alpha^3-\beta$ and $\beta^3-\alpha$; build $x^2-Sx+P=0$.

STEP 1/1Part (c)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

(c) Let $S$ = sum of new roots, $P$ = product.

$$S = (\alpha^3 - \beta) + (\beta^3 - \alpha) = (\alpha^3 + \beta^3) - (\alpha + \beta) = -\tfrac{63}{8} - \tfrac{3}{2} = -\tfrac{63}{8} - \tfrac{12}{8} = -\tfrac{75}{8}.$$

For $P$, expand:
$$P = (\alpha^3 - \beta)(\beta^3 - \alpha) = \alpha^3\beta^3 - \alpha^4 - \beta^4 + \alpha\beta = (\alpha\beta)^3 + \alpha\beta - (\alpha^4 + \beta^4).$$

Compute $\alpha^4 + \beta^4$:
$$\alpha^4 + \beta^4 = (\alpha^2+\beta^2)^2 - 2(\alpha\beta)^2 = \left(-\tfrac{11}{4}\right)^2 - 2\cdot\left(\tfrac52\right)^2 = \tfrac{121}{16} - \tfrac{50}{4} = \tfrac{121}{16} - \tfrac{200}{16} = -\tfrac{79}{16}.$$

Therefore
$$P = \left(\tfrac52\right)^3 + \tfrac52 - \left(-\tfrac{79}{16}\right) = \tfrac{125}{8} + \tfrac{5}{2} + \tfrac{79}{16}.$$

Common denominator 16:
$$P = \tfrac{250}{16} + \tfrac{40}{16} + \tfrac{79}{16} = \tfrac{369}{16}.$$

New equation: $x^2 - Sx + P = 0$:
$$x^2 + \tfrac{75}{8}x + \tfrac{369}{16} = 0.$$

Multiply through by 16:
$$\boxed{16x^2 + 150x + 369 = 0}$$

Check $\gcd(16, 150, 369)$: $16 = 2^4$, $150 = 2\cdot 3\cdot 5^2$, $369 = 9\cdot 41$. Gcd $= 1$, so coefficients are in simplest integer form.

Target: $\boxed{16x^2 + 150x + 369 = 0}$
