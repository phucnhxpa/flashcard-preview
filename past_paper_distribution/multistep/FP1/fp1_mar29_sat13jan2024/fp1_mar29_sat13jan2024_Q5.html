Q5 — Quadratic with transformed roots

Question

The quadratic equation
$$2x^2 - 3x + 7 = 0$$
has roots $\alpha$ and $\beta$. Without solving the equation,

(a) write down the value of $(\alpha + \beta)$ and the value of $\alpha\beta$. (1)
(b) determine the value of $\alpha^2 + \beta^2$. (2)
(c) find a quadratic equation which has roots
$$\alpha\left(1 + \frac{1}{\beta^2}\right) \quad \text{and} \quad \beta\left(1 + \frac{1}{\alpha^2}\right)$$
giving your answer in the form $px^2 + qx + r = 0$ where $p$, $q$ and $r$ are integers to be determined. (6)

From the Loom (paraphrased Phuc's own method, Slack)

"1. Use the relationship in the root of the polynomial in terms of the sum of the roots and the product of the roots. 2. We set up a system of equations and try to manipulate that equation into the set of variables that we already know. 3. After that, we can find the sum of the root and the product of the root. 4. Once we have all of that, we can try to find the ABC. 5. Once we have the ABC, we can identify the quadratic equation."

The method is correct --- the failure mode is arithmetic on the symmetric identities.

Correct method

(a) From $2x^2 - 3x + 7 = 0$:
$$\alpha + \beta = -\frac{-3}{2} = \frac{3}{2}, \qquad \alpha\beta = \frac{7}{2}$$

(b) Use the identity $\alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta$:
$$\alpha^2 + \beta^2 = \left(\tfrac{3}{2}\right)^2 - 2 \cdot \tfrac{7}{2} = \tfrac{9}{4} - 7 = \tfrac{9}{4} - \tfrac{28}{4} = -\tfrac{19}{4}$$

(Negative is fine --- the original roots are complex, so $\alpha^2 + \beta^2$ can be any real number.)

(c) Let the new roots be $S_1 = \alpha\left(1 + \tfrac{1}{\beta^2}\right) = \alpha + \tfrac{\alpha}{\beta^2}$ and $S_2 = \beta\left(1 + \tfrac{1}{\alpha^2}\right) = \beta + \tfrac{\beta}{\alpha^2}$.

Sum of new roots:
$$S_1 + S_2 = (\alpha + \beta) + \left(\frac{\alpha}{\beta^2} + \frac{\beta}{\alpha^2}\right)$$

For the second bracket, combine over a common denominator $\alpha^2\beta^2 = (\alpha\beta)^2$:
$$\frac{\alpha}{\beta^2} + \frac{\beta}{\alpha^2} = \frac{\alpha^3 + \beta^3}{(\alpha\beta)^2}$$

Compute $\alpha^3 + \beta^3$ via the identity $\alpha^3 + \beta^3 = (\alpha+\beta)^3 - 3\alpha\beta(\alpha+\beta)$:
$$\alpha^3 + \beta^3 = \left(\tfrac{3}{2}\right)^3 - 3\cdot\tfrac{7}{2}\cdot\tfrac{3}{2} = \tfrac{27}{8} - \tfrac{63}{4} = \tfrac{27}{8} - \tfrac{126}{8} = -\tfrac{99}{8}$$

And $(\alpha\beta)^2 = \left(\tfrac{7}{2}\right)^2 = \tfrac{49}{4}$.

So:
$$\frac{\alpha^3 + \beta^3}{(\alpha\beta)^2} = \frac{-99/8}{49/4} = -\frac{99}{8} \cdot \frac{4}{49} = -\frac{99}{98}$$

Therefore:
$$S_1 + S_2 = \tfrac{3}{2} - \tfrac{99}{98} = \tfrac{147}{98} - \tfrac{99}{98} = \tfrac{48}{98} = \tfrac{24}{49}$$

Product of new roots:
$$S_1 \cdot S_2 = \alpha\beta\left(1 + \frac{1}{\beta^2}\right)\left(1 + \frac{1}{\alpha^2}\right)$$

Expand the bracket:
$$\left(1 + \tfrac{1}{\beta^2}\right)\left(1 + \tfrac{1}{\alpha^2}\right) = 1 + \tfrac{1}{\alpha^2} + \tfrac{1}{\beta^2} + \tfrac{1}{\alpha^2\beta^2}$$

Compute $\tfrac{1}{\alpha^2} + \tfrac{1}{\beta^2} = \dfrac{\alpha^2 + \beta^2}{(\alpha\beta)^2} = \dfrac{-19/4}{49/4} = -\dfrac{19}{49}$

And $\tfrac{1}{\alpha^2\beta^2} = \tfrac{1}{49/4} = \tfrac{4}{49}$.

So the bracket equals:
$$1 - \tfrac{19}{49} + \tfrac{4}{49} = 1 - \tfrac{15}{49} = \tfrac{49 - 15}{49} = \tfrac{34}{49}$$

Therefore:
$$S_1 S_2 = \tfrac{7}{2} \cdot \tfrac{34}{49} = \frac{7 \cdot 34}{2 \cdot 49} = \frac{34}{2 \cdot 7} = \frac{34}{14} = \frac{17}{7}$$

Build the quadratic. Any quadratic with these roots: $x^2 - (S_1 + S_2)x + S_1 S_2 = 0$:
$$x^2 - \tfrac{24}{49}x + \tfrac{17}{7} = 0$$

Multiply through by $49$ to clear denominators:
$$49x^2 - 24x + 49 \cdot \tfrac{17}{7} = 0$$
$$49x^2 - 24x + 7 \cdot 17 = 0$$
$$\boxed{49x^2 - 24x + 119 = 0}$$

So $p = 49$, $q = -24$, $r = 119$. ($\gcd(49, 24, 119) = 1$, so already in lowest terms.)

What went wrong

The procedure is right; the failure mode is fraction arithmetic under exam pressure. Specifically the moment of combining $\tfrac{3}{2} - \tfrac{99}{98}$: students often rewrite $\tfrac{3}{2}$ as $\tfrac{49}{49}\cdot \tfrac{3}{2}$ --- fine --- but then forget to apply the same denominator to both terms before subtracting. Use a single LCD ($98$ here) and convert both numerators in one move.

Key formula / concept

Symmetric function identities (must be memorised --- NOT in the formula booklet):
- $\alpha^2 + \beta^2 = (\alpha+\beta)^2 - 2\alpha\beta$
- $\alpha^3 + \beta^3 = (\alpha+\beta)^3 - 3\alpha\beta(\alpha+\beta)$
- $\dfrac{1}{\alpha} + \dfrac{1}{\beta} = \dfrac{\alpha+\beta}{\alpha\beta}$
- $\dfrac{1}{\alpha^2} + \dfrac{1}{\beta^2} = \dfrac{\alpha^2+\beta^2}{(\alpha\beta)^2}$
- For new quadratic with roots $S_1, S_2$: $x^2 - (S_1+S_2)x + S_1 S_2 = 0$.

Exam checklist

  • [ ] Stated sum and product from (a) with correct signs ($-b/a$, not $+b/a$)
  • [ ] Used the identity for $\alpha^3 + \beta^3$ rather than expanding $(\alpha+\beta)(\alpha^2 - \alpha\beta + \beta^2)$ (both fine, identity is faster)
  • [ ] Combined fractions step-by-step with explicit common denominators shown
  • [ ] Multiplied through to clear denominators at the very end to get integer $p, q, r$
  • [ ] Verified $\gcd(p, q, r) = 1$ --- examiner expects lowest terms

Common pitfalls

  • Computing $\tfrac{1}{\alpha^2} + \tfrac{1}{\beta^2}$ as $\dfrac{\alpha + \beta}{(\alpha\beta)^2}$ (using sum instead of sum-of-squares in the numerator).
  • Forgetting that $(\alpha\beta)^2 \neq \alpha^2 + \beta^2$ --- they're entirely different quantities.
  • Multiplying $\tfrac{7}{2} \cdot \tfrac{34}{49}$ and not simplifying the $7$ with the $49$.
