Worked solution

Thought Process

We have three sequential draws (from A, then B, then C) where the composition of bag C changes depending on earlier draws. This is a classic dependent-events tree diagram problem.

(a) Tree Diagram

Bag A: 4R, 2Y → $P(R_A) = \frac{4}{6} = \frac{2}{3}$, $P(Y_A) = \frac{1}{3}$

Bag B: 4R, 1Y → $P(R_B) = \frac{4}{5}$, $P(Y_B) = \frac{1}{5}$

Bag C starts with 6R. After placing one ball from A and one from B, bag C has 8 balls. The composition depends on what was added:

  • If A=R, B=R: C has 8R, 0Y → $P(R_C)=1$, $P(Y_C)=0$
  • If A=R, B=Y: C has 7R, 1Y → $P(R_C)=\frac{7}{8}$, $P(Y_C)=\frac{1}{8}$
  • If A=Y, B=R: C has 7R, 1Y → $P(R_C)=\frac{7}{8}$, $P(Y_C)=\frac{1}{8}$
  • If A=Y, B=Y: C has 6R, 2Y → $P(R_C)=\frac{6}{8}=\frac{3}{4}$, $P(Y_C)=\frac{2}{8}=\frac{1}{4}$

(b) P(exactly 2 yellow)

The three "recordings" are from A, B, C. Exactly 2 yellow means cases RYY, YRY, YYR:

$$P(RYY) = \frac{2}{3} \times \frac{1}{5} \times \frac{1}{8} = \frac{2}{120}$$

$$P(YRY) = \frac{1}{3} \times \frac{4}{5} \times \frac{1}{8} = \frac{4}{120}$$

$$P(YYR) = \frac{1}{3} \times \frac{1}{5} \times \frac{3}{4} = \frac{6}{120}$$

$$P(\text{exactly 2 yellow}) = \frac{2+4+6}{120} = \frac{12}{120} = \frac{1}{10} \quad \square$$

(c) P(A is red | exactly 2 yellow)

Using conditional probability:

$$P(R_A \mid \text{2Y}) = \frac{P(RYY)}{P(\text{2Y})} = \frac{2/120}{1/10} = \frac{2/120}{12/120} = \frac{2}{12} = \frac{1}{6}$$

(d) Probability distribution of X

$P(X=0)$: all red = $\frac{2}{3}\times\frac{4}{5}\times 1 = \frac{8}{15} = \frac{64}{120}$

$P(X=1)$: exactly one yellow. RRY is impossible (prob 0). So:


  • RYR: $\frac{2}{3}\times\frac{1}{5}\times\frac{7}{8} = \frac{14}{120}$

  • YRR: $\frac{1}{3}\times\frac{4}{5}\times\frac{7}{8} = \frac{28}{120}$

$P(X=1) = \frac{42}{120} = \frac{7}{20}$

$P(X=2) = \frac{12}{120} = \frac{1}{10}$ (from part b)

$P(X=3)$: YYY = $\frac{1}{3}\times\frac{1}{5}\times\frac{1}{4} = \frac{2}{120} = \frac{1}{60}$

Check: $64+42+12+2=120$ ✓

| $x$ | 0 | 1 | 2 | 3 |
|---|---|---|---|---|
| $P(X=x)$ | $\frac{64}{120}$ | $\frac{42}{120}$ | $\frac{12}{120}$ | $\frac{2}{120}$ |

(e) E(X)

$$E(X) = 0\times\frac{64}{120} + 1\times\frac{42}{120} + 2\times\frac{12}{120} + 3\times\frac{2}{120} = \frac{0+42+24+6}{120} = \frac{72}{120} = 0.6$$

Answer: (b) $\frac{1}{10}$, (c) $\frac{1}{6}$, (d) table above, (e) $E(X)=0.6$

Key techniques

  • construct and label a tree diagram with sequential dependent events
  • enumerate cases for exactly k successes across stages
  • apply conditional probability formula P(A|B) = P(A∩B)/P(B)
  • build a probability distribution table from tree diagram paths
  • compute E(X) from a probability distribution

Common pitfalls

  • Forgetting that bag C's composition changes depending on what was added from A and B — each branch of the tree gives different bag C probabilities
  • Including the impossible path RRY (probability 0) as a non-zero case when counting exactly 1 yellow
  • In conditional probability, dividing by the wrong total — must use P(exactly 2 yellow) = 1/10 as denominator, not some other value
