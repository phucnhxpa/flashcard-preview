Worked solution

Thought Process

We have a stem-and-leaf diagram with 24 values, three unknowns $a < b < c$ in the 13-stem row, an outlier definition, coding for mean/SD, and a conditional probability question.

---

Part (a): Show $c = 9$

First, list the data in order. There are 24 values: 2 in the 10-row, 11 in the 11-row, 8 in the 12-row, and 3 in the 13-row.

For $n = 24$, the lower quartile $Q_1$ is the $\frac{24+1}{4} = 6.25$th value, and the upper quartile $Q_3$ is the $18.75$th value.

Counting through: the 6th value is 116 and the 7th is also 116, so $Q_1 = 116$.

The 18th value is 125 and the 19th is also 125, so $Q_3 = 125$.

$$\text{IQR} = 125 - 116 = 9$$

Outlier threshold:
$$Q_3 + 1.5 \times \text{IQR} = 125 + 1.5 \times 9 = 125 + 13.5 = 138.5$$

An outlier is any value greater than 138.5. Since there is only one outlier and $a < b < c$ are digits in the 13-stem:

  • $13c > 138.5$ requires $c = 9$ (giving 139).
  • $13b \leq 138.5$ requires $b \leq 8$.

Since $a < b < c = 9$ and they are positive integers, this is consistent. $c = 9$.

---

Part (b): Find the mean number of deliveries

Coding: $x = d - 125$, so $d = x + 125$.

Given $\sum x = -96$ and $n = 24$:

$$\bar{x} = \frac{-96}{24} = -4$$

$$\bar{d} = \bar{x} + 125 = -4 + 125 = 121$$

Answer: $\bar{d} = 121$

---

Part (c): Find the standard deviation

The standard deviation is unchanged by a linear shift (adding/subtracting a constant), so $\sigma_d = \sigma_x$.

$$\sigma_x = \sqrt{\frac{\sum(x - \bar{x})^2}{n}} = \sqrt{\frac{1306}{24}} = \sqrt{54.4166\ldots} = 7.3767\ldots$$

Answer: $\sigma_d \approx 7.38$

---

Part (d): Find $P(D > 118 \mid X < 0)$

$X = D - 125$, so $X < 0 \Leftrightarrow D < 125$.

We need $P(D > 118 \mid D < 125)$.

Count values with $D < 125$: the first 2 + 11 = 13 values, plus the 14th value is 124. So 14 values satisfy $D < 125$.

Of these 14, count those with $D > 118$: values 119, 119, 119, 119, 124 — that's 5 values. (From the 11-row: 119 appears 4 times; from the 12-row: 124 is the only value below 125.)

Wait — recount: 118 is in the data (two 118s). Values $> 118$ and $< 125$ are: 119, 119, 119, 119, 124 = 5 values.

$$P(D > 118 \mid X < 0) = \frac{5}{14}$$

Answer: $\dfrac{5}{14}$

Key techniques

  • finding quartiles from ordered data
  • using outlier rule Q3 + 1.5 × IQR
  • reversing coding to find original mean
  • computing standard deviation from summary statistics
  • conditional probability from discrete data

Common pitfalls

  • Forgetting that the coding shift does not affect the standard deviation — only scaling would
  • Miscounting the position of quartiles: for n=24, using the wrong interpolation method
  • In the conditional probability, forgetting that X < 0 means D < 125 (not D ≤ 125) and miscounting boundary values
