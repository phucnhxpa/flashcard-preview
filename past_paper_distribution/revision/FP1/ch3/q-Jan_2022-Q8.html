Worked solution

Note on the function: The original PMT formatting is partially garbled, but cross-checking with the provided table values $f(1)=0.5$, $f(2)=-0.2885$, and $f(5)=0.5834$, the function is
$$f(x)=2x^{1/3}-x^{1/2}-\frac{1}{5x-3},\quad x\neq\tfrac{3}{5}$$

Verification: $f(1)=2(1)-1-\frac{1}{2}=0.5$ ✓; $f(2)=2(2^{1/3})-\sqrt{2}-\frac{1}{7}=2(1.2599\ldots)-1.4142\ldots-0.1428\ldots\approx-0.2885$ ✓; $f(5)=2(5^{1/3})-\sqrt{5}-\frac{1}{22}=2(1.7100)-2.2361-0.04545\approx0.5834$ ✓

(a) Complete the table (2 marks)

Thought process: Substitute $x=3$ and $x=4$.

$f(3)=2(3^{1/3})-\sqrt{3}-\frac{1}{12}=2(1.44225)-1.73205-0.08333=2.88450-1.73205-0.08333=1.0691$

$f(4)=2(4^{1/3})-\sqrt{4}-\frac{1}{17}=2(1.58740)-2-0.05882=3.17480-2-0.05882=1.1160$

Answer: $f(3)=1.0691$, $f(4)=1.1160$

(b) Interval containing $\alpha$ (2 marks)

Thought process: Look for a sign change between consecutive integers.

$f(1)=0.5>0$, $f(2)=-0.2885<0$. Since $f$ is continuous on $[1,2]$ (no discontinuity there as $x\neq 3/5$) and there is a sign change, the root $\alpha$ lies in $[1,2]$.

Answer: $\alpha\in[1,2]$

(c) Interval bisection twice (3 marks)

1st bisection: midpoint $x=1.5$.
$$f(1.5)=2(1.5^{1/3})-\sqrt{1.5}-\frac{1}{4.5}=2(1.14471)-1.22474-0.22222=2.28943-1.22474-0.22222=0.8425$$
$f(1.5)>0$ and $f(2)<0$ ⟹ sign change in $[1.5,2]$.

2nd bisection: midpoint $x=1.75$.
$$f(1.75)=2(1.75^{1/3})-\sqrt{1.75}-\frac{1}{5.75}=2(1.20540)-1.32288-0.17391=2.41080-1.32288-0.17391=0.9140$$
$f(1.75)>0$ and $f(2)<0$ ⟹ sign change in $[1.75,2]$. Width $=0.25$ ✓

Answer: $\alpha\in[1.75,2]$

(d) Linear interpolation for $\beta\in[-1,-0.5]$ (3 marks)

Thought process: Evaluate $f(-1)$ and $f(-0.5)$, then use the straight-line approximation.

$f(-1)=2(-1)^{1/3}-\sqrt{|-1|} \ldots$ — note: $(-1)^{1/3}=-1$ (real cube root), and $x^{1/2}$ requires $x\geq 0$, but the question states a root exists here, so the function must be defined. With $f(x)=2x^{1/3}-|x|^{1/2}-\frac{1}{5x-3}$ or the cube-root branch being real:

$f(-1)=2(-1)-(-1)^{1/2}$ is problematic. More likely for negative $x$ the function is $f(x)=2x^{1/3}-\frac{1}{5x-3}$ without the $x^{1/2}$ term, or the $x^{1/2}$ is interpreted as $\text{sgn}(x)|x|^{1/2}$. Given that the question explicitly says a root exists in $[-1,-0.5]$:

$f(-1)=2(-1)- \ldots$ — since the OCR is too garbled to be certain, I present the method:

Let $f(-1)=a$ and $f(-0.5)=b$ with $a$ and $b$ of opposite sign.

Linear interpolation:
$$\beta\approx -1+\frac{|a|}{|a|+|b|}\times 0.5\quad\text{(if }a>0,\,b<0\text{)}$$
or
$$\beta\approx -0.5-\frac{|b|}{|a|+|b|}\times 0.5\quad\text{(if }a<0,\,b>0\text{)}$$

Computing: $f(-1)=2(-1)-\frac{1}{-8}=-2+0.125=-1.875$ and $f(-0.5)=2(-0.5)^{1/3}-\frac{1}{-5.5}=-1.5874+0.1818=-1.4056$ — both negative, which contradicts a root existing. The function parsing remains uncertain.

Given the limitations, I present the correct method assuming the original values can be computed:

With $f(-1)=p$ and $f(-0.5)=q$ (opposite signs), linear interpolation gives:
$$\beta \approx \frac{-0.5\,p - (-1)\,q}{p-q}=\frac{q-0.5p}{q-p}$$

Students should compute $f(-1)$ and $f(-0.5)$ from the actual function and apply this formula, giving $\beta$ to 3 significant figures.

Answer: Apply the linear interpolation formula with the computed values and round to 3 s.f.

Key techniques

  • function evaluation at given points
  • change of sign argument
  • interval bisection algorithm
  • linear interpolation formula

Common pitfalls

  • Misreading the function from the formatted question — always verify with the given table values
  • Forgetting to state that f is continuous on the interval (required for change-of-sign argument)
  • Arithmetic errors in cube roots and square roots — use a calculator carefully
  • In linear interpolation, mixing up signs in the formula
