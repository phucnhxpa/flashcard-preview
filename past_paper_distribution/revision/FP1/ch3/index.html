Assumptions
  • All three methods are applied to the same equation f(x) = 0
  • Results from earlier parts may be needed in later parts (e.g., f(a) and f(b) from part (a) used in part (c))
  • Each method should give an estimate consistent with the others — use this as a sanity check
Approaches
A: Sequential application of all three methods
When: When the question has multiple parts each specifying a different method
  1. Part (a): Evaluate f at endpoints, state continuity, conclude root exists (sign change)
  2. Part (b): Differentiate f(x), apply Newton-Raphson formula from given x₀, compute x₁ (and x₂ if asked)
  3. Part (c): Use the f-values from part (a) (or re-evaluate if needed) in the linear interpolation formula
  4. Compare all estimates as a reasonableness check
+ Pros: Structured question guides you through each technique; marks available for each part independently
– Cons: Long question — errors in early parts can undermine later parts if values are carried forward
Formulas
Sign change rule with continuity statement (MEMORIZE)
$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$ (formula booklet)
$\alpha \approx \frac{a|f(b)| + b|f(a)|}{|f(a)| + |f(b)|}$ (MEMORIZE)
Pitfalls
  • Running out of time — budget time carefully across all parts
  • Forgetting continuity statement in part (a) when rushing to get to Newton-Raphson
  • Using rounded values from Newton-Raphson part in the linear interpolation part when exact endpoint values should be used
  • Not re-reading the question to check which values are to be used in which part

Newton-Raphson method (single or multiple iterations)
#2

Given f(x) and a starting value x₀, apply Newton-Raphson to find x₁ (and sometimes x₂). The function is usually a polynomial or polynomial + trig/exponential. You must differentiate f(x) first.

Assumptions
  • f(x) is differentiable at and near x₀
  • f'(x₀) ≠ 0 (the tangent is not horizontal at the starting point)
  • The starting value x₀ is sufficiently close to the root for convergence
  • Full calculator precision should be used for intermediate steps; only round the final answer
Approaches
A: Standard Newton-Raphson iteration
When: Always — the only approach when Newton-Raphson is specified
  1. Write down the Newton-Raphson formula: $x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$
  2. Differentiate f(x) carefully to obtain f'(x)
  3. Evaluate f(x₀) using full calculator precision
  4. Evaluate f'(x₀) using full calculator precision
  5. Substitute into the formula to find x₁ = x₀ − f(x₀)/f'(x₀)
  6. If a second iteration is required, repeat: evaluate f(x₁) and f'(x₁), then find x₂
  7. Round to the number of decimal places specified in the question only at the very end (or as instructed)
+ Pros: Converges very quickly (quadratic convergence) so usually only 1-2 iterations needed
– Cons: Requires correct differentiation and careful arithmetic; one slip propagates through all subsequent iterations
Formulas
$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$ — Newton-Raphson formula (formula booklet)
Power rule: $\frac{d}{dx}(x^n) = nx^{n-1}$ including negative and fractional powers (MEMORIZE)
$\frac{d}{dx}(\tan x) = \sec^2 x$ (formula booklet)
$\frac{d}{dx}(e^x) = e^x$, $\frac{d}{dx}(\ln x) = \frac{1}{x}$ (formula booklet)
Pitfalls
  • Differentiating f(x) incorrectly — especially with negative powers (e.g., differentiating x⁻² as −2x⁻¹ instead of −2x⁻³)
  • Premature rounding of f(x₀) or f'(x₀) before dividing — use full calculator precision throughout
  • Not quoting the Newton-Raphson formula before substituting — examiners may withhold a method mark
  • Sign errors in f(x₀)/f'(x₀) leading to iteration going the wrong direction
  • Forgetting constant terms when differentiating (derivative of a constant is 0)
  • Using the rounded x₁ value to compute x₂ instead of the unrounded value stored in calculator memory

Assumptions
  • f(x) is continuous on the given interval
  • A root has already been located in [a, b] via sign change
  • Each bisection halves the interval width, so after n bisections the interval has width (b−a)/2ⁿ
Approaches
A: Systematic midpoint evaluation
When: When the question says 'use interval bisection' or 'apply the bisection method'
  1. Start with the interval [a, b] where f(a) and f(b) have opposite signs
  2. Calculate the midpoint: $c = \frac{a+b}{2}$
  3. Evaluate f(c)
  4. If f(c) has the same sign as f(a), the root is in [c, b]; replace a with c
  5. If f(c) has the same sign as f(b), the root is in [a, c]; replace b with c
  6. Repeat for the required number of iterations
  7. State the final interval clearly
+ Pros: Very systematic and reliable — hard to get wrong if arithmetic is correct
– Cons: Convergence is slow (linear); requires multiple iterations to achieve good accuracy
Formulas
Midpoint: $c = \frac{a + b}{2}$ (MEMORIZE)
After $n$ bisections, interval width $= \frac{b - a}{2^n}$ (MEMORIZE)
Pitfalls
  • Selecting the wrong sub-interval by comparing signs incorrectly
  • Arithmetic errors in evaluating f at the midpoint
  • Not clearly stating which sub-interval is selected and why (sign change argument needed each time)
  • Calculator in wrong mode for trig functions

Assumptions
  • f(x) must be continuous on the interval [a, b] — you MUST state this explicitly
  • A sign change means exactly one root only if f is monotonic on the interval; the question assumes a single root unless stated otherwise
  • The function is evaluated exactly or to sufficient decimal places to demonstrate the sign difference
Approaches
A: Direct evaluation at endpoints
When: Always — this is the only standard approach for this sub-question
  1. Substitute x = a into f(x) and evaluate to at least 2-3 significant figures
  2. Substitute x = b into f(x) and evaluate similarly
  3. State that f(a) and f(b) have opposite signs (one positive, one negative)
  4. State that f(x) is continuous on [a, b] (required for full marks)
  5. Conclude: 'Therefore, by the sign change rule (Intermediate Value Theorem), there is a root α in the interval (a, b)'
+ Pros: Quick, straightforward, and low risk of error
– Cons: Students lose marks by forgetting the continuity statement or the conclusion
Formulas
If $f(a) \cdot f(b) < 0$ and $f$ is continuous on $[a,b]$, then $\exists\; \alpha \in (a,b)$ such that $f(\alpha) = 0$ (MEMORIZE)
Pitfalls
  • Forgetting to state that f(x) is continuous on the interval — costs a mark every time
  • Not writing a clear conclusion ('therefore a root exists in (a, b)')
  • Using degrees mode instead of radians mode when f(x) involves trig functions
  • Rounding intermediate values so much that the sign change is obscured

Assumptions
  • f(x) is continuous on [a, b]
  • f(a) and f(b) have already been evaluated and have opposite signs
  • The linear interpolation formula gives an estimate — it is NOT the exact root
  • Values of f(a) and f(b) should be used with full precision in the formula
Approaches
A: Linear interpolation formula
When: When the question says 'use linear interpolation' or 'by interpolation'
  1. Identify the two points: $(a, f(a))$ and $(b, f(b))$ where f(a) and f(b) have opposite signs
  2. Apply the formula: $\alpha \approx a + \frac{-f(a)}{f(b) - f(a)} \times (b - a)$
  3. Alternatively use similar triangles: $\frac{\alpha - a}{b - \alpha} = \frac{|f(a)|}{|f(b)|}$
  4. Calculate the estimate and round as required
+ Pros: Generally gives a better first estimate than bisection
– Cons: Formula is easy to mix up; students must be careful with signs
B: Similar triangles / geometric approach
When: When you prefer a visual/geometric derivation or want to cross-check the formula
  1. Draw (or sketch) the chord from (a, f(a)) to (b, f(b))
  2. The chord crosses the x-axis at the estimated root α
  3. Set up similar triangles: $\frac{\alpha - a}{|f(a)|} = \frac{b - \alpha}{|f(b)|}$
  4. Solve for α: $\alpha = \frac{a|f(b)| + b|f(a)|}{|f(a)| + |f(b)|}$
  5. Calculate and round as required
+ Pros: Conceptually clear and easy to derive from scratch if you forget the formula
– Cons: Slightly longer to set up than direct formula application
Formulas
$\alpha \approx a + \frac{-f(a)}{f(b) - f(a)} \times (b - a)$ — linear interpolation (MEMORIZE — not always in formula booklet)
Equivalent form: $\alpha \approx \frac{a\,|f(b)| + b\,|f(a)|}{|f(a)| + |f(b)|}$ (MEMORIZE)
Similar triangles relation: $\frac{\alpha - a}{|f(a)|} = \frac{b - \alpha}{|f(b)|}$ (MEMORIZE)
Pitfalls
  • Mixing up a and b or f(a) and f(b) in the formula — always label clearly
  • Sign errors: using f(a) and f(b) with wrong signs in the denominator
  • Using the formula with a and b the wrong way round
  • Rounding f(a) or f(b) before substituting into the formula — use full precision

Must memorize

  • $x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$ — Newton-Raphson formula (formula booklet — but know it cold)
  • Linear interpolation: $\alpha \approx \frac{a|f(b)| + b|f(a)|}{|f(a)| + |f(b)|}$ (MEMORIZE)
  • Sign change rule: if $f$ is continuous on $[a,b]$ and $f(a) \cdot f(b) < 0$, then $\exists\; \alpha \in (a,b)$ with $f(\alpha) = 0$ (MEMORIZE)
  • Interval bisection midpoint: $c = \frac{a+b}{2}$ (MEMORIZE)
  • Key derivatives: $\frac{d}{dx}(x^n) = nx^{n-1}$, $\frac{d}{dx}(\tan x) = \sec^2 x$, $\frac{d}{dx}(e^{kx}) = ke^{kx}$, $\frac{d}{dx}(\ln x) = \frac{1}{x}$ (formula booklet but practise rapid recall)

Global strategy

  • Almost every question begins with 'show a root exists in [a, b]' — always evaluate f(a) and f(b), state opposite signs, state continuity, and write a conclusion. This is a 2-3 mark gift.
  • Master all three numerical methods (sign change, Newton-Raphson, linear interpolation) — the exam frequently tests two or three in one question.
  • Newton-Raphson is the most commonly examined method (7/8 sessions). Practise differentiating polynomials, negative powers, and simple trig/exponential functions quickly and accurately.
  • Use full calculator precision throughout; only round at the very end when the question specifies decimal places. Store intermediate results in calculator memory.
  • Always QUOTE the Newton-Raphson or linear interpolation formula BEFORE substituting values — examiners award a method mark for stating the formula.
  • For interval bisection, lay out your working in a clear table: interval, midpoint, f(midpoint), sign, new interval. This avoids confusion.
  • Double-check your calculator is in RADIANS mode whenever the function involves trigonometric terms.
  • Read the question carefully to identify which method is required in each part — do not apply Newton-Raphson when linear interpolation is asked for, and vice versa.

Exam tips

  1. For the sign change argument: always write THREE things — (1) the two function values with opposite signs, (2) 'f is continuous', (3) 'therefore a root exists in (a, b)'. Missing any one can cost a mark.
  2. In Newton-Raphson, write your formula, then show substitution clearly: $x_1 = x_0 - \frac{f(x_0)}{f'(x_0)} = \ldots$. Examiners need to see the method.
  3. Store every intermediate result in your calculator memory (use Ans or M+ keys) to avoid rounding errors propagating through iterations.
  4. If a question asks for a root to, say, 5 decimal places and you get x₁ and x₂ to agree to 5 d.p., you can often stop — but always do as many iterations as the question asks.
  5. For linear interpolation, sketch a quick diagram of the chord between (a, f(a)) and (b, f(b)) — this helps you set up similar triangles correctly and catch sign errors.
  6. Budget about 1 minute per mark. A 10-mark question should take roughly 10 minutes — don't spend 15 minutes perfecting part (a) and rush parts (b) and (c).
  7. If you get a Newton-Raphson answer that diverges wildly from the interval, recheck your differentiation — a sign error in f'(x) is the most common cause.
