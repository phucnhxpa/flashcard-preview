Worked solution

Let $\mathbf{r}$ be the position vector, and $\mathbf{v}$ be the velocity vector.

Part (a): Show that the velocity of the ship is $(10\mathbf{i} – 5\mathbf{j})\text{ km h}^{-1}$

Thought process: We are given two position vectors for the ship at two different times. Since the velocity is constant, we can find the displacement vector and divide it by the time taken to find the velocity. It's crucial to convert the time difference into hours.

Working:
Let the position of the port be $\mathbf{r}_0 = (-2\mathbf{i} + 9\mathbf{j})\text{ km}$ at $t=0$ (7 am).
At 7:36 am, which is $36$ minutes after 7 am, the position is $\mathbf{r}_1 = (4\mathbf{i} + 6\mathbf{j})\text{ km}$.

First, calculate the time elapsed in hours:
$t = 36 \text{ minutes} = \frac{36}{60} \text{ hours} = 0.6 \text{ hours}$.

Next, find the displacement vector, $\Delta\mathbf{r}$, which is the final position minus the initial position:
$$\Delta\mathbf{r} = \mathbf{r}_1 - \mathbf{r}_0$$
$$\Delta\mathbf{r} = (4\mathbf{i} + 6\mathbf{j}) - (-2\mathbf{i} + 9\mathbf{j})$$
$$\Delta\mathbf{r} = (4 - (-2))\mathbf{i} + (6 - 9)\mathbf{j}$$
$$\Delta\mathbf{r} = (6\mathbf{i} - 3\mathbf{j})\text{ km}$$

Now, calculate the constant velocity, $\mathbf{v}$, using the formula $\mathbf{v} = \frac{\Delta\mathbf{r}}{t}$:
$$\mathbf{v} = \frac{6\mathbf{i} - 3\mathbf{j}}{0.6}$$
$$\mathbf{v} = \left(\frac{6}{0.6}\right)\mathbf{i} - \left(\frac{3}{0.6}\right)\mathbf{j}$$
$$\mathbf{v} = (10\mathbf{i} - 5\mathbf{j})\text{ km h}^{-1}$$
This matches the given answer.

Answer: The velocity of the ship is $(10\mathbf{i} - 5\mathbf{j})\text{ km h}^{-1}$.

Part (b): Find the position vector of the ship $t$ hours after leaving port.

Thought process: We have the initial position vector $\mathbf{r}_0$ (at 7 am, when $t=0$) and the constant velocity vector $\mathbf{v}$ from part (a). The general equation for position with constant velocity is $\mathbf{r}(t) = \mathbf{r}_0 + \mathbf{v}t$.

Working:
Using $\mathbf{r}_0 = (-2\mathbf{i} + 9\mathbf{j})$ and $\mathbf{v} = (10\mathbf{i} - 5\mathbf{j})$:
$$\mathbf{r}(t) = (-2\mathbf{i} + 9\mathbf{j}) + t(10\mathbf{i} - 5\mathbf{j})$$
$$\mathbf{r}(t) = (-2 + 10t)\mathbf{i} + (9 - 5t)\mathbf{j}\text{ km}$$

Answer: The position vector of the ship $t$ hours after leaving port is $\mathbf{r}(t) = (-2 + 10t)\mathbf{i} + (9 - 5t)\mathbf{j}\text{ km}$.

Part (c): Find the position vector of the lighthouse.

Thought process: The lighthouse is at a fixed position, say $\mathbf{L} = L_x\mathbf{i} + L_y\mathbf{j}$. We are given two pieces of information relating the ship's position to the lighthouse's position at two different times. We need to calculate the ship's position at these times first, then use the directional information to set up equations for $L_x$ and $L_y$.

Working:
First, calculate the time elapsed from 7 am for each observation:
At 8:48 am: Time $t_1 = 1 \text{ hour } 48 \text{ minutes} = 1 + \frac{48}{60} = 1.8 \text{ hours}$.
At 9:00 am: Time $t_2 = 2 \text{ hours}$.

Calculate the ship's position at $t_1 = 1.8$ hours using the formula from part (b):
$$\mathbf{r}(1.8) = (-2 + 10(1.8))\mathbf{i} + (9 - 5(1.8))\mathbf{j}$$
$$\mathbf{r}(1.8) = (-2 + 18)\mathbf{i} + (9 - 9)\mathbf{j}$$
$$\mathbf{r}(1.8) = 16\mathbf{i} + 0\mathbf{j} = 16\mathbf{i}\text{ km}$$

At 8:48 am, the lighthouse is due east of the ship. This means the ship's $j$-coordinate (north-south position) is the same as the lighthouse's $j$-coordinate. Since the ship is at $16\mathbf{i}$, its $j$-coordinate is $0$. Therefore, the lighthouse's $j$-coordinate, $L_y$, must be $0$.
So, $\mathbf{L} = L_x\mathbf{i} + 0\mathbf{j}$.

Now, calculate the ship's position at $t_2 = 2$ hours:
$$\mathbf{r}(2) = (-2 + 10(2))\mathbf{i} + (9 - 5(2))\mathbf{j}$$
$$\mathbf{r}(2) = (-2 + 20)\mathbf{i} + (9 - 10)\mathbf{j}$$
$$\mathbf{r}(2) = 18\mathbf{i} - 1\mathbf{j}\text{ km}$$

At 9:00 am, the lighthouse is north-east of the ship. This means the vector from the ship to the lighthouse has positive and equal $\mathbf{i}$ and $\mathbf{j}$ components. Let $\mathbf{r}_S$ be the ship's position at this time.

$\mathbf{L} - \mathbf{r}_S = (L_x\mathbf{i} + L_y\mathbf{j}) - (18\mathbf{i} - 1\mathbf{j})$
$\mathbf{L} - \mathbf{r}_S = (L_x - 18)\mathbf{i} + (L_y - (-1))\mathbf{j}$
$\mathbf{L} - \mathbf{r}_S = (L_x - 18)\mathbf{i} + (L_y + 1)\mathbf{j}$

Since $L_y = 0$, substitute this into the equation:
$\mathbf{L} - \mathbf{r}_S = (L_x - 18)\mathbf{i} + (0 + 1)\mathbf{j}$
$\mathbf{L} - \mathbf{r}_S = (L_x - 18)\mathbf{i} + 1\mathbf{j}$

For the lighthouse to be north-east of the ship, the $\mathbf{i}$ and $\mathbf{j}$ components of this relative position vector must be equal and positive:
$L_x - 18 = 1$
$L_x = 19$

So, the position vector of the lighthouse is $\mathbf{L} = 19\mathbf{i} + 0\mathbf{j} = 19\mathbf{i}\text{ km}$.

Answer: The position vector of the lighthouse is $19\mathbf{i}\text{ km}$.

Part (d): Find the position vector of the ship when it is due south of the lighthouse.

Thought process: When the ship is due south of the lighthouse, their $\mathbf{i}$-components (east-west positions) must be the same. We know the fixed $\mathbf{i}$-component of the lighthouse from part (c). We can set the ship's $\mathbf{i}$-component from part (b) equal to this value to find the time $t$. Once $t$ is found, substitute it back into the ship's position vector equation to find its full position.

Working:
Let the position of the ship be $\mathbf{r}(t) = (-2 + 10t)\mathbf{i} + (9 - 5t)\mathbf{j}$.
The position of the lighthouse is $\mathbf{L} = 19\mathbf{i}$.

If the ship is due south of the lighthouse, their $\mathbf{i}$-components are equal:
$(-2 + 10t) = 19$
$10t = 19 + 2$
$10t = 21$
$t = \frac{21}{10} = 2.1 \text{ hours}$

Now, substitute this value of $t$ back into the ship's position vector equation:
$$\mathbf{r}(2.1) = (-2 + 10(2.1))\mathbf{i} + (9 - 5(2.1))\mathbf{j}$$
$$\mathbf{r}(2.1) = (-2 + 21)\mathbf{i} + (9 - 10.5)\mathbf{j}$$
$$\mathbf{r}(2.1) = 19\mathbf{i} - 1.5\mathbf{j}\text{ km}$$

Answer: The position vector of the ship when it is due south of the lighthouse is $(19\mathbf{i} - 1.5\mathbf{j})\text{ km}$.

Key techniques

  • displacement calculation
  • vector equation for position (r = r0 + vt)
  • equating vector components
  • time unit conversion (minutes to hours)

Common pitfalls

  • Incorrectly converting time units (e.g., forgetting to divide minutes by 60 for hours).
  • Mixing up 'due east', 'due north', 'north-east' conditions, especially confusing equality of components with difference.
  • Calculation errors when adding/subtracting vectors or solving linear equations.
