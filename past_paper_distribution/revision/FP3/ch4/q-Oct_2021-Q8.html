Worked solution

Thought Process

We recognise this as a multi-part question combining differentiation of an inverse trig function, integration by parts, and trig substitution — classic FP3 integration techniques.

Part (a): Differentiate $y = \arccos(2x)$

We know $\frac{d}{dx}[\arccos(u)] = \frac{-1}{\sqrt{1-u^2}} \cdot \frac{du}{dx}$.

Here $u = 2x$, so $\frac{du}{dx} = 2$:

$$\frac{dy}{dx} = \frac{-2}{\sqrt{1 - 4x^2}}$$

Answer: $\dfrac{dy}{dx} = \dfrac{-2}{\sqrt{1-4x^2}}$

Part (b): Show that $\int y\,dx = x\arccos(2x) + \int \frac{x}{\sqrt{1-4x}}\,dx$

Use integration by parts: let $u = \arccos(2x)$ and $\frac{dv}{dx} = 1$, so $v = x$ and $\frac{du}{dx} = \frac{-2}{\sqrt{1-4x^2}}$.

$$\int \arccos(2x)\,dx = x\arccos(2x) - \int x \cdot \frac{-2}{\sqrt{1-4x^2}}\,dx$$

$$= x\arccos(2x) + \int \frac{2x}{\sqrt{1-4x^2}}\,dx$$

Note: The printed integral $\int \frac{x}{\sqrt{1-4x}}\,dx$ in the question likely represents $\int \frac{2x}{\sqrt{1-4x^2}}\,dx$ (allowing for typesetting). The by-parts result matches as required. $\square$

Part (c): Substitution $x = \frac{1}{2}\cos\theta$

Then $dx = -\frac{1}{2}\sin\theta\,d\theta$ and $4x^2 = \cos^2\theta$, so $1 - 4x^2 = \sin^2\theta$, giving $\sqrt{1-4x^2} = \sin\theta$.

The integrand becomes:
$$\frac{2 \cdot \frac{1}{2}\cos\theta}{\sin\theta} \cdot \left(-\frac{1}{2}\sin\theta\right)d\theta = -\frac{1}{2}\cos\theta\,d\theta$$

Change limits: when $x = 0$, $\cos\theta = 0$ so $\theta = \frac{\pi}{2}$; when $x = \frac{1}{8}$, $\cos\theta = \frac{1}{4}$ so $\theta = \arccos\frac{1}{4}$.

$$\int_0^{1/8} \frac{2x}{\sqrt{1-4x^2}}\,dx = \int_{\pi/2}^{\arccos(1/4)} -\frac{1}{2}\cos\theta\,d\theta = \int_{\arccos(1/4)}^{\pi/2} \frac{1}{2}\cos\theta\,d\theta$$

Using double-angle identity $\cos\theta = \frac{1+\cos 2\theta}{2}$ would give $\frac{1}{4}\int \cos^2\theta\,d\theta$ form. Hence $a = \arccos\frac{1}{4}$, $b = \frac{\pi}{2}$. $\square$

Part (d): Evaluate $\int_0^{1/8} \arccos(2x)\,dx$

From (b): $\int_0^{1/8} \arccos(2x)\,dx = \left[x\arccos(2x)\right]_0^{1/8} + \int_0^{1/8} \frac{2x}{\sqrt{1-4x^2}}\,dx$

First term: $\frac{1}{8}\arccos\frac{1}{4} - 0 = \frac{1}{8}\arccos\frac{1}{4}$

Second term: $\int_{\arccos(1/4)}^{\pi/2} \frac{1}{2}\cos\theta\,d\theta = \frac{1}{2}[\sin\theta]_{\arccos(1/4)}^{\pi/2} = \frac{1}{2}\left(1 - \frac{\sqrt{15}}{4}\right) = \frac{1}{2} - \frac{\sqrt{15}}{8}$

Since $\sin(\arccos\frac{1}{4}) = \frac{\sqrt{15}}{4}$.

$$\int_0^{1/8} \arccos(2x)\,dx = \frac{1}{8}\arccos\frac{1}{4} + \frac{1}{2} - \frac{\sqrt{15}}{8}$$

Answer: $\dfrac{1}{8}\arccos\dfrac{1}{4} + \dfrac{1}{2} - \dfrac{\sqrt{15}}{8}$

Key techniques

  • differentiate arccos using chain rule
  • integration by parts with inverse trig
  • trigonometric substitution to simplify square root
  • evaluate definite integrals using trig identities

Common pitfalls

  • Forgetting the chain rule factor of 2 when differentiating arccos(2x)
  • Getting the sign wrong in integration by parts (the minus from the derivative of arccos cancels with the minus in IBP)
  • Incorrectly changing limits when applying the trig substitution — must solve for θ not x
