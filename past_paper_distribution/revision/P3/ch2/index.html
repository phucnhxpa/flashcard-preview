R-formula transformation, extrema & solving equations

Express $a\sin\theta + b\cos\theta$ (or similar) in the form $R\sin(\theta \pm \alpha)$ or $R\cos(\theta \pm \alpha)$. Then find max/min of a function, range of a composite function, or solve an equation. May involve a rational function like $\frac{k}{p + q\sin x + r\cos x}$.

Assumptions
  • $R > 0$ and $0 < \alpha < \frac{\pi}{2}$ (or $0° < \alpha < 90°$) unless stated otherwise
  • When finding extrema of $\frac{k}{p + R\sin(\theta - \alpha)}$, the denominator must never be zero within the domain
  • All solutions must lie within the specified domain — adjust the range of the transformed angle $(\theta \pm \alpha)$ accordingly
  • Calculator must be in the correct mode (degrees or radians) as specified by the question
Approaches
A: Expand compound angle and compare coefficients
When: Always — this is the standard and expected method
  1. Choose the target form: e.g. $R\sin(\theta - \alpha) \equiv R\sin\theta\cos\alpha - R\cos\theta\sin\alpha$
  2. Expand the compound angle formula fully
  3. Compare coefficients of $\sin\theta$ and $\cos\theta$ with the original expression: $R\cos\alpha = a$, $R\sin\alpha = b$
  4. Find $R = \sqrt{a^2 + b^2}$
  5. Find $\alpha = \arctan\left(\frac{b}{a}\right)$, using the signs of $R\cos\alpha$ and $R\sin\alpha$ to determine the correct quadrant
  6. For extrema: use $-1 \leq \sin(\theta - \alpha) \leq 1$ (or cos) to find max/min of the entire expression
  7. For solving: set $R\sin(\theta - \alpha) = k$, solve for $(\theta - \alpha)$, then add $\alpha$ back
  8. Check all solutions lie within the original domain
+ Pros: Systematic and always works; directly gives R and α for all subsequent parts
– Cons: Requires careful expansion — sign errors are very common
B: Use $R^2 = a^2 + b^2$ and $\tan\alpha = b/a$ directly (shortcut)
When: When you are confident about which form is required and just need R and α quickly
  1. Identify the target form and which coefficient goes with sin and cos
  2. Compute $R = \sqrt{a^2 + b^2}$
  3. Compute $\alpha = \arctan\left(\frac{b}{a}\right)$ with appropriate quadrant check
  4. Verify by substituting back into the expanded form
+ Pros: Fast — avoids full expansion
– Cons: Easy to get the wrong sign or wrong quadrant for α if not careful; still need to verify
Formulas
$R\sin(\theta + \alpha) \equiv R\sin\theta\cos\alpha + R\cos\theta\sin\alpha$ (formula booklet)
$R\cos(\theta - \alpha) \equiv R\cos\theta\cos\alpha + R\sin\theta\sin\alpha$ (formula booklet)
$R = \sqrt{a^2 + b^2}$ (MEMORIZE — derived from comparison)
$\tan\alpha = \frac{b}{a}$ with quadrant analysis (MEMORIZE)
$-1 \leq \sin\phi \leq 1$ and $-1 \leq \cos\phi \leq 1$ (MEMORIZE)
Pitfalls
  • Incorrect quadrant for α — both $R\cos\alpha$ and $R\sin\alpha$ should be positive when $0 < \alpha < \pi/2$; if not, re-examine your chosen form
  • Forgetting to adjust the domain for the transformed variable: if $0 \leq x \leq 2\pi$ and you substitute $y = x - \alpha$, then $-\alpha \leq y \leq 2\pi - \alpha$
  • For rational functions $\frac{k}{p + R\sin(\theta - \alpha)}$, minimum of denominator gives maximum of f(x) and vice versa — students frequently invert this
  • Forgetting to square R when computing $R^2$ in denominators (e.g. $(\sqrt{13})^2 = 13$ not $\sqrt{13}$)
  • Using degrees when the question specifies radians or vice versa
  • Missing a second solution within the range when solving $\sin(\theta - \alpha) = k$

An equation involving mixed trig functions (e.g. sec and tan, cosec and cot, or sin and cos). Use a Pythagorean identity to reduce to a single trig function, forming a quadratic. Solve the quadratic and find all solutions in the given range.

Assumptions
  • The equation must be reduced to a single trigonometric function before solving
  • Quadratic solutions must be checked: reject values outside the range of the trig function (e.g. $\sin\theta > 1$)
  • Solutions for $\sin\theta = k$ or $\cos\theta = k$ typically give two solutions per period; $\tan\theta = k$ gives one per period
  • The domain is exactly as stated — do not include endpoints if the inequality is strict
Approaches
A: Use Pythagorean identity to form quadratic
When: When the equation mixes sec/tan or cosec/cot — always the primary approach
  1. Identify which pair of reciprocal trig functions appears (sec & tan, or cosec & cot)
  2. Apply the appropriate identity: $\sec^2 x = 1 + \tan^2 x$ or $\csc^2 x = 1 + \cot^2 x$
  3. Substitute to eliminate one function, obtaining a quadratic in the remaining function
  4. Rearrange into standard form $at^2 + bt + c = 0$ and solve by factorisation or quadratic formula
  5. Convert back: e.g. $\tan x = p$ or $\sin x = q$
  6. Find all solutions in the given interval using principal value + periodicity or CAST diagram
  7. Check solutions are valid (not excluded by domain restrictions like denominators being zero)
+ Pros: Reliable and systematic; works for all such problems
– Cons: Must remember correct identity — $\sec^2 = 1 + \tan^2$ not $1 - \tan^2$
B: Convert everything to sin and cos, then simplify
When: When the Pythagorean identity route is not immediately obvious, or when the equation has products of different functions
  1. Replace all trig functions with sin and cos: $\sec x = 1/\cos x$, $\csc x = 1/\sin x$, $\cot x = \cos x/\sin x$
  2. Multiply through by the common denominator to clear fractions
  3. Use $\sin^2 x + \cos^2 x = 1$ to reduce to one function
  4. Solve the resulting equation
  5. Find all solutions in the given range
+ Pros: More intuitive for students who think in sin/cos
– Cons: Can lead to more complex algebra; risk of multiplying by zero and introducing extraneous solutions
Formulas
$\sin^2 x + \cos^2 x = 1$ (MEMORIZE)
$1 + \tan^2 x = \sec^2 x$ (MEMORIZE)
$1 + \cot^2 x = \csc^2 x$ (MEMORIZE)
Quadratic formula: $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ (MEMORIZE)
Pitfalls
  • Using $\cot^2 x = 1 - \csc^2 x$ instead of $\csc^2 x - 1$ — sign error on identity
  • Forgetting $\pm$ when taking square roots: $\tan^2 x = k$ gives $\tan x = \pm\sqrt{k}$
  • Dividing through by $\sin x$ or $\cos x$ and losing solutions where that function is zero
  • Rejecting valid solutions or accepting invalid ones (e.g. $\sin\theta = 2$ is impossible)
  • Missing solutions due to not considering the full range — especially when the range is $0 < x \leq \pi$ (note the strict/non-strict inequality)

Assumptions
  • When the variable is $2\theta$, the effective range doubles: e.g. if $0 < \theta < 2\pi$ then $0 < 2\theta < 4\pi$
  • Choose the correct form of $\cos 2\theta$ based on what the rest of the equation contains
  • Solutions from $\sin 2\theta = k$ give four solutions in $0 < \theta < 2\pi$ (two periods)
Approaches
A: Apply double angle formula, reduce to quadratic
When: Standard approach when double angle terms can be expressed in terms of single-angle functions
  1. Replace $\sin 2\theta$ with $2\sin\theta\cos\theta$ or $\cos 2\theta$ with the appropriate form
  2. Replace any reciprocal functions ($\sec$, $\csc$, $\cot$) with their sin/cos equivalents
  3. Simplify and collect terms
  4. If a quadratic forms, solve using factorisation or the formula
  5. Find all solutions in the specified range
  6. Verify no extraneous solutions were introduced
+ Pros: Direct and follows naturally from the identity substitution
– Cons: Choosing the wrong form of cos 2θ can make algebra much harder
B: Factor and solve directly (when equation factors nicely)
When: When after substitution, terms share a common factor like $\sin\theta$ or $\cos\theta$
  1. After applying double angle formulae, look for common factors
  2. Factor out the common trig function: e.g. $\sin\theta(2\cos\theta - 1) = 0$
  3. Set each factor equal to zero and solve separately
  4. Collect all solutions within the range
+ Pros: Avoids the quadratic formula; often faster
– Cons: Only works when the equation factors — not always the case
Formulas
$\sin 2A = 2\sin A\cos A$ (formula booklet)
$\cos 2A = 1 - 2\sin^2 A$ — use when the equation involves $\sin$ (formula booklet)
$\cos 2A = 2\cos^2 A - 1$ — use when the equation involves $\cos$ (formula booklet)
$\cos 2A = \cos^2 A - \sin^2 A$ — use when both appear (formula booklet)
Pitfalls
  • Using the wrong form of $\cos 2\theta$ — pick the form that matches the other terms in the equation
  • Dividing by $\sin\theta$ or $\cos\theta$ instead of factoring — this loses solutions where $\sin\theta = 0$ or $\cos\theta = 0$
  • Forgetting to extend the range when working with $2\theta$: if $\theta \in [0, 2\pi)$, then $2\theta \in [0, 4\pi)$
  • Writing $\operatorname{cosec}\theta = 1/\cos\theta$ instead of $1/\sin\theta$

Assumptions
  • In a 'show that' proof, you must work from one side to the other (or from LHS to a common form) — do not assume the result
  • All identities in the formula booklet are available for use without proof
  • When an equation reduces to a quadratic, both roots must be checked for validity within the trig function's range
  • The domain is as specified — find ALL solutions in that range
Approaches
A: Start from LHS, manipulate to RHS using identities
When: Standard approach for 'show that' proofs — usually start from the more complex side
  1. Write everything in terms of sin and cos (convert cosec, sec, cot, tan)
  2. Apply double angle formulae where appropriate: $\sin 2A = 2\sin A\cos A$, $\cos 2A = 1 - 2\sin^2 A$ etc.
  3. Combine fractions using common denominators
  4. Simplify using Pythagorean identity $\sin^2 A + \cos^2 A = 1$
  5. Arrive at RHS and write $= \text{RHS}$ QED
+ Pros: Logical and clear; examiners can follow each step for method marks
– Cons: Can get stuck if you choose the wrong starting point or wrong form of double angle identity
B: Work from both sides toward a common expression
When: When neither side simplifies cleanly to the other — meet in the middle
  1. Simplify LHS as far as possible
  2. Separately simplify RHS as far as possible
  3. Show both equal the same expression
  4. Conclude LHS = RHS
+ Pros: Useful when one-directional manipulation is very difficult
– Cons: Must make clear you are not assuming the result — present carefully
Formulas
$\sin 2A = 2\sin A\cos A$ (formula booklet)
$\cos 2A = \cos^2 A - \sin^2 A = 2\cos^2 A - 1 = 1 - 2\sin^2 A$ (formula booklet — know ALL THREE forms)
$\tan 2A = \frac{2\tan A}{1 - \tan^2 A}$ (formula booklet)
$1 + \tan^2 A = \sec^2 A$ (MEMORIZE)
$1 + \cot^2 A = \csc^2 A$ (MEMORIZE)
$\csc A = \frac{1}{\sin A}$, $\sec A = \frac{1}{\cos A}$, $\cot A = \frac{\cos A}{\sin A}$ (MEMORIZE)
Pitfalls
  • Using the wrong form of $\cos 2A$: choose the form that best matches the target (e.g. use $1 - 2\sin^2 A$ if the target involves $\sin$)
  • Misapplying double angle formulae — especially writing $\cos 2A = 1 + \cos^2 A$ instead of $2\cos^2 A - 1$
  • Failing to use the 'hence' result from part (a) in part (b) — if it says 'hence', you MUST use the identity you just proved
  • Forgetting that $\csc 2\theta \neq 2\csc\theta$
  • Algebraic errors when combining fractions over a common denominator

Assumptions
  • Compound angle formulae from the booklet are used directly
  • When the question links parts via 'hence', the substitution $\theta = x + k$ or similar connects them
  • The range must be adjusted for any substitution: if $0 < x < 360°$ and $\theta = x + 20°$ then $20° < \theta < 380°$
Approaches
A: Expand compound angle, isolate tan, then solve
When: When the question asks to show a result involving tan from a compound angle expression
  1. Write out the compound angle formula: e.g. $\sin(A+B) = \sin A\cos B + \cos A\sin B$
  2. Substitute the given values and simplify
  3. To isolate $\tan$, divide numerator and denominator by $\cos A\cos B$ (or appropriate)
  4. For the 'hence' part, identify the substitution that links the angles
  5. Solve $\tan\phi = k$ and find all solutions in the adjusted range
  6. Convert back to the original variable
+ Pros: Clean and elegant; produces exact values
– Cons: The link between parts can be hard to spot
Formulas
$\sin(A \pm B) = \sin A\cos B \pm \cos A\sin B$ (formula booklet)
$\cos(A \pm B) = \cos A\cos B \mp \sin A\sin B$ (formula booklet)
$\tan(A \pm B) = \frac{\tan A \pm \tan B}{1 \mp \tan A\tan B}$ (formula booklet)
Pitfalls
  • Forgetting to adjust the range for the substituted variable
  • Failing to spot the connection between the 'show' and 'hence' parts — look for the angle difference matching
  • Sign errors in compound angle expansion, especially $\cos(A-B)$ vs $\cos(A+B)$

Assumptions
  • Integration of $\sin(kx)$ and $\cos(kx)$ is expected — the chain rule factor $1/k$ must be included
  • The proven identity in part (a) must be used for the integration in part (b)
  • Exact values are expected — leave in terms of surds and fractions
Approaches
A: Prove identity via compound angle expansion, then integrate term-by-term
When: Always — this is the intended approach for these two-part questions
  1. Expand compound angle expressions: e.g. $\sin 3x = \sin(2x + x)$
  2. Apply double angle formulae within the expansion
  3. Simplify and rearrange to match the required identity
  4. For integration: substitute the identity into the integral
  5. Integrate each term: $\int \sin(kx)\,dx = -\frac{1}{k}\cos(kx) + C$, $\int \cos(kx)\,dx = \frac{1}{k}\sin(kx) + C$
  6. Evaluate at the limits and simplify
+ Pros: Turns a difficult integral (power of trig) into straightforward standard integrals
– Cons: The identity proof can be lengthy; sign errors propagate into the integration
Formulas
$\sin 3A = 3\sin A - 4\sin^3 A$ (can be derived from compound angles — MEMORIZE or derive)
$\cos^2 A = \frac{1 + \cos 2A}{2}$ (MEMORIZE — essential for integration)
$\sin^2 A = \frac{1 - \cos 2A}{2}$ (MEMORIZE — essential for integration)
$\int \sin(kx)\,dx = -\frac{1}{k}\cos(kx) + C$ (MEMORIZE)
$\int \cos(kx)\,dx = \frac{1}{k}\sin(kx) + C$ (MEMORIZE)
Pitfalls
  • Sign error when integrating $\sin(3x)$: $\int \sin(3x)\,dx = -\frac{1}{3}\cos(3x)$, NOT $+\frac{1}{3}$
  • Forgetting the $\frac{1}{k}$ factor from the chain rule in integration
  • Not rearranging the proven identity to isolate the expression being integrated (e.g. solving for $\sin^3 x$ in terms of $\sin 3x$ and $\sin x$)

Must memorize

  • $\sin^2 A + \cos^2 A = 1$ — the fundamental Pythagorean identity
  • $1 + \tan^2 A = \sec^2 A$ — derived from dividing Pythagoras by $\cos^2 A$
  • $1 + \cot^2 A = \csc^2 A$ — derived from dividing Pythagoras by $\sin^2 A$
  • $\sin 2A = 2\sin A\cos A$ (in formula booklet but must be instant recall)
  • $\cos 2A = \cos^2 A - \sin^2 A = 2\cos^2 A - 1 = 1 - 2\sin^2 A$ — all three forms (formula booklet)
  • $\tan 2A = \frac{2\tan A}{1 - \tan^2 A}$ (formula booklet)
  • $R = \sqrt{a^2 + b^2}$ and $\tan\alpha = b/a$ for R-formula — with quadrant check
  • Reciprocal definitions: $\sec A = 1/\cos A$, $\csc A = 1/\sin A$, $\cot A = \cos A/\sin A$
  • Compound angle formulae for $\sin(A \pm B)$, $\cos(A \pm B)$, $\tan(A \pm B)$ (formula booklet)
  • $\cos^2 A = \frac{1 + \cos 2A}{2}$ and $\sin^2 A = \frac{1 - \cos 2A}{2}$ — essential for integration

Global strategy

  • Identify the question type immediately: R-formula (look for 'express in the form'), identity proof ('show that'), or equation solving ('solve'). This determines your opening move.
  • For ANY equation with mixed trig functions (sec & tan, cosec & cot), your first instinct should be: apply the Pythagorean identity to reduce to a single trig function.
  • Always choose the correct form of $\cos 2A$: use $1 - 2\sin^2 A$ when the equation involves $\sin$; use $2\cos^2 A - 1$ when it involves $\cos$; use $\cos^2 A - \sin^2 A$ when both appear.
  • In R-formula questions, the value of $\alpha$ is ALWAYS checked against which quadrant makes both $R\cos\alpha$ and $R\sin\alpha$ have the correct signs.
  • When solving trig equations, ALWAYS adjust the working range for any substitution or double angle: if the range is $[0, 2\pi]$ for $\theta$, then for $2\theta$ it becomes $[0, 4\pi]$, and for $(\theta - \alpha)$ it becomes $[-\alpha, 2\pi - \alpha]$.
  • Never divide by a trig function (sin, cos, tan) — instead factor it out, so you don't lose solutions where that function equals zero.
  • For 'hence' questions, you MUST use the result from the previous part. If you can't see the link, look at the angles — often the difference or sum of angles in part (b) matches the compound angle in part (a).
  • Present 'show that' proofs clearly: state which identity you are using at each step. Marks are awarded for method, not just the final line.

Exam tips

  1. Read the question carefully for degree vs radian mode — this affects every calculation. Check BEFORE you start.
  2. In 'show that' questions, if you arrive at the given answer, explicitly state '= RHS as required'. If you don't reach it, check your working — do not fudge the last step.
  3. When finding all solutions of $\sin\theta = k$ in $[0, 2\pi]$: principal value $\theta_1 = \arcsin(k)$, second value $\theta_2 = \pi - \theta_1$. For $\cos\theta = k$: $\theta_1 = \arccos(k)$, $\theta_2 = 2\pi - \theta_1$. For $\tan\theta = k$: $\theta_1 = \arctan(k)$, $\theta_2 = \theta_1 + \pi$.
  4. When a quadratic in $\sin\theta$ gives two values, always check both: reject any value outside $[-1, 1]$ for sin/cos, and note that $\sec\theta$ and $\csc\theta$ can never equal values in $(-1, 1)$.
  5. For R-formula extrema questions on rational functions like $\frac{c}{a + b\sin x}$: the function is maximised when the denominator is minimised (and vice versa). State this reasoning explicitly for full marks.
  6. Leave space for corrections — trig algebra is error-prone. If you spot an error, fix it cleanly rather than starting over.
  7. On 8-9 mark questions, expect 2-3 parts building on each other. Budget roughly 1 minute per mark. If stuck on part (a), move to part (b) using the given result.
  8. Always write down the identity you are using before substituting — this earns method marks even if you make an arithmetic error later.
