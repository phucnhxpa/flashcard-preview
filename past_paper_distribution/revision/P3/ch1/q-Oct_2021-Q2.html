Worked solution

(a) State the coordinates of P.

Thought Process: We recognize that $y = |3x - 13| + 5$ is an absolute value function of the form $y = |Ax+B| + C$. The vertex of such a function occurs when the expression inside the absolute value is zero ($Ax+B=0$). The y-coordinate of the vertex will then be $C$.

Working:
The given function is $f(x) = |3x - 13| + 5$.
The vertex occurs when $3x - 13 = 0$.
$3x = 13 \implies x = \frac{13}{3}$.
Substitute this x-value back into the function to find the y-coordinate:
$f\left(\frac{13}{3}\right) = \left|3\left(\frac{13}{3}\right) - 13\right| + 5 = |13 - 13| + 5 = |0| + 5 = 5$.

Answer: $\left( \frac{13}{3}, 5 \right)$

(b) (i) State the range of f.

Thought Process: The absolute value of any real number is always non-negative, i.e., $|X| \ge 0$. We use this property to determine the minimum value of $f(x)$. Since the absolute value part can be 0, the minimum value of $f(x)$ will occur when $|3x - 13|$ is 0.

Working:
Since $|3x - 13| \ge 0$ for all real $x$.
Adding 5 to both sides, we get $|3x - 13| + 5 \ge 0 + 5$.
So, $f(x) \ge 5$.

Answer: $f(x) \ge 5$ (or $[5, \infty)$)

(b) (ii) Find the value of ff(4).

Thought Process: This is a composite function. We must evaluate the inner function first, $f(4)$, and then use that result as the input for the outer function, $f(\text{result})$.

Working:
First, find $f(4)$:
$f(4) = |3(4) - 13| + 5 = |12 - 13| + 5 = |-1| + 5 = 1 + 5 = 6$.
Now, find $f(f(4))$, which is $f(6)$:
$f(6) = |3(6) - 13| + 5 = |18 - 13| + 5 = |5| + 5 = 5 + 5 = 10$.

Answer: $10$

(c) Solve, using algebra and showing your working, $16 - 2x > |3x - 13| + 5$.

Thought Process: This is an inequality involving an absolute value. The general approach for solving an inequality of the form $B > |A|$ (or $|A| < B$) is to state that $-B < A < B$, provided that $B > 0$. We first need to isolate the absolute value term on one side.

Working:
Given the inequality: $16 - 2x > |3x - 13| + 5$.
Subtract 5 from both sides to isolate the absolute value term:
$11 - 2x > |3x - 13|$.
For this inequality to have solutions, we must have $11 - 2x > 0$, as the absolute value term is always non-negative. So, $11 > 2x \implies x < \frac{11}{2}$. This is a condition for the solution.
Now we can split the inequality into two parts:
Part 1: $3x - 13 < 11 - 2x$
Add $2x$ to both sides: $5x - 13 < 11$.
Add $13$ to both sides: $5x < 24$.
Divide by $5$: $x < \frac{24}{5}$.

Part 2: $3x - 13 > -(11 - 2x)$
$3x - 13 > -11 + 2x$.
Subtract $2x$ from both sides: $x - 13 > -11$.
Add $13$ to both sides: $x > 2$.

Combining the conditions $x < \frac{24}{5}$ and $x > 2$, we get $2 < x < \frac{24}{5}$.
We must also satisfy the initial condition $x < \frac{11}{2}$.
Numerically, $\frac{24}{5} = 4.8$ and $\frac{11}{2} = 5.5$.
Since $4.8 < 5.5$, the condition $x < \frac{11}{2}$ is automatically satisfied if $x < \frac{24}{5}$.

Answer: $2
< x < \frac{24}{5}$

(d) find the value of a and the value of b.

Thought Process: We need to understand how the transformation $y = af(x+b)$ affects the coordinates of the vertex. The original vertex is $(x_0, y_0) = (\frac{13}{3}, 5)$.
A transformation $f(x+b)$ translates the graph horizontally. If $b>0$, it shifts left by $b$ units, so the new x-coordinate becomes $x_0 - b$. (Think: $x+b=0$ shifts the root to $x=-b$).
A transformation $af(x)$ stretches the graph vertically by a factor of $a$. The new y-coordinate becomes $a y_0$.
Combining these, the new vertex will be $(x_0 - b, a y_0)$. We can then equate this to the given new vertex coordinates.

Working:
Original vertex P is $(\frac{13}{3}, 5)$.
The transformed equation is $y = af(x+b)$.
The horizontal translation $f(x+b)$ shifts the x-coordinate of the vertex by $-b$ units.
So, the new x-coordinate of the vertex is $\frac{13}{3} - b$.
The vertical stretch $af(x)$ scales the y-coordinate of the vertex by a factor of $a$.
So, the new y-coordinate of the vertex is $5a$.
The new vertex is $(\frac{13}{3} - b, 5a)$.
We are given that the new vertex is $(4, 20)$.

Equating the x-coordinates:
$\frac{13}{3} - b = 4$.
$b = \frac{13}{3} - 4 = \frac{13}{3} - \frac{12}{3} = \frac{1}{3}$.

Equating the y-coordinates:
$5a = 20$.
$a = \frac{20}{5} = 4$.

Answer: $a = 4, b = \frac{1}{3}$

Key techniques

  • finding vertex of $|Ax+B|+C$
  • determining range of absolute value functions
  • evaluating composite functions
  • solving absolute value inequalities
  • identifying transformation parameters from vertex shift

Common pitfalls

  • Errors in solving absolute value inequalities, particularly forgetting to consider both positive and negative cases or making sign errors with the negative case.
  • Incorrect order of operations for composite functions, e.g., calculating $f(f(4))$ as $(f(4))^2$ or evaluating $f(4)$ then $f(x)$ for the result but using $x=4$ again.
  • Mistakes in identifying transformation parameters, especially confusing horizontal shift directions ($f(x+b)$ means shift left by $b$, not right). Or incorrectly applying scaling to only one coordinate.
