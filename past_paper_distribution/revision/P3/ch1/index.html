Triggersinverse functioncomposite functionfg(x)gf(x)f⁻¹(x)domainrangeone-to-onestate the domain of f⁻¹
Assumptions
  • Domain of fg exists only where the range of g is a subset of (or overlaps with) the domain of f
  • Domain of f⁻¹ = Range of f, and Range of f⁻¹ = Domain of f
  • f⁻¹ exists only when f is one-to-one on the given domain
  • When finding f⁻¹, the ± from a square root must be resolved using the known range
Approaches
A: Direct algebraic rearrangement for inverse
When: When f(x) is a rational function or simple algebraic expression
  1. Write y = f(x)
  2. Swap x and y to get x = f(y)
  3. Rearrange to make y the subject — multiply out denominators, collect y-terms, factorise
  4. Replace y with f⁻¹(x)
  5. State domain of f⁻¹ = range of f (find range by considering asymptotes/turning points of f)
+ Pros: Systematic and always works for rational functions
– Cons: Can produce messy algebra, especially with quadratics under a square root
B: Use completing the square or quadratic formula for inverse
When: When rearranging y = f(x) leads to a quadratic in y (e.g., f involves x² or a rational function that rearranges to quadratic)
  1. Write y = f(x), swap x and y
  2. Rearrange to standard quadratic form ay² + by + c = 0
  3. Apply quadratic formula: y = (-b ± √(b²-4ac)) / 2a
  4. Choose + or − sign by checking which matches the known domain of f (= range of f⁻¹)
  5. State domain of f⁻¹
+ Pros: Handles cases where simple rearrangement fails
– Cons: Must carefully justify the sign choice — marks lost if not explained
Formulas
$fg(x) = f(g(x))$: substitute $g(x)$ into $f$ (MEMORIZE)
Domain of $f^{-1}$ = Range of $f$; Range of $f^{-1}$ = Domain of $f$ (MEMORIZE)
Quadratic formula: $x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$ (formula booklet)
Pitfalls
  • Incorrect order of composite functions: fg(x) means f(g(x)), evaluate g first then f
  • Forgetting to state the domain of f⁻¹ — this is typically a separate mark
  • Choosing the wrong ± sign when inverse involves quadratic formula; must match original domain
  • Sign errors when cross-multiplying to clear denominators in rational functions
  • Not checking discriminant ≥ 0 when solving fg(x) = k leads to quadratic — may need to show no real solutions

Triggersmodulus|absolute valuesketchnumber of solutionsset of valuesinequality
Assumptions
  • |f(x)| ≥ 0 for all x, so the graph never goes below the x-axis for y=|f(x)|
  • Vertex of y=|ax+b| is at x = -b/a with y = 0
  • When solving |A| = B, we need B ≥ 0 for real solutions
  • Each case solution must be checked against its own domain condition
Approaches
A: Case-splitting (algebraic)
When: When solving |f(x)| = g(x) or |f(x)| < g(x) algebraically
  1. Identify the critical point where the expression inside | | equals zero
  2. Case 1: assume expression ≥ 0, remove modulus as-is, solve equation/inequality
  3. Case 2: assume expression < 0, negate the expression inside modulus, solve
  4. For each case, verify the solution satisfies the domain condition of that case
  5. Combine valid solutions; for inequalities, express as intersection of intervals
+ Pros: Rigorous and guaranteed to find all solutions
– Cons: Easy to forget to check case domains, leading to extraneous solutions
B: Graphical/sketch approach
When: When asked for number of solutions, parameter ranges for k, or to justify answers visually
  1. Sketch y = |f(x)| by drawing y = f(x) first, then reflecting negative parts in x-axis
  2. Sketch the other function (e.g., y = k or y = mx + c) on the same axes
  3. Count intersection points for different values of k
  4. Identify critical k values: tangent to vertex, parallel to an arm, passing through intercepts
  5. State ranges of k for required number of intersections using < or ≤ carefully
+ Pros: Excellent for parameter-range questions and gives visual confirmation
– Cons: Not sufficient alone for exact solutions — must be paired with algebra
C: Squaring both sides
When: When both sides can be squared without introducing extraneous roots (e.g., |f(x)| = |g(x)|)
  1. Square both sides: [f(x)]² = [g(x)]²
  2. Rearrange to [f(x)]² - [g(x)]² = 0
  3. Factorise as (f(x)-g(x))(f(x)+g(x)) = 0
  4. Solve each factor = 0
  5. Check solutions in the original equation
+ Pros: Avoids case-splitting entirely
– Cons: Only valid when both sides are non-negative or both have modulus; can introduce extraneous solutions
Formulas
$|x| = \begin{cases} x & x \geq 0 \\ -x & x < 0 \end{cases}$ (MEMORIZE)
Vertex of $y = |ax + b| + c$ is at $\left(-\frac{b}{a},\, c\right)$ (MEMORIZE)
$|f(x)| < k \Leftrightarrow -k < f(x) < k$ for $k > 0$ (MEMORIZE)
$|f(x)| > k \Leftrightarrow f(x) > k \text{ or } f(x) < -k$ for $k > 0$ (MEMORIZE)
Pitfalls
  • Forgetting that a boundary k value (e.g., line tangent to vertex) gives exactly one root, not two
  • Not checking that k > 0 when the question says 'two distinct solutions' — k=0 may be a degenerate case
  • Missing that a line parallel to one arm of the V gives no intersection on that arm
  • For inequalities: forgetting to intersect each case's solution with its domain condition
  • Forgetting to reflect the correct part of the graph below the x-axis when sketching |f(x)|

Assumptions
  • Quotient rule: d/dx [u/v] = (vu' - uv')/v²
  • f'(x) > 0 everywhere implies f is strictly increasing (one-to-one)
  • Integration of 1/(ax+b) gives (1/a)ln|ax+b| + C
  • Area under curve = definite integral; below x-axis contributions are negative
Approaches
A: Quotient rule differentiation then analysis
When: When asked to find f'(x) for a fraction and show monotonicity
  1. Identify u, v, u', v' from f(x) = u/v
  2. Apply quotient rule: f'(x) = (vu' - uv') / v²
  3. Simplify the numerator
  4. Show the sign of f'(x) (e.g., numerator always negative, denominator always positive → f'(x) < 0)
  5. State conclusion: 'f is strictly decreasing' or 'f is one-to-one'
+ Pros: Directly addresses 'show' questions about monotonicity
– Cons: Must simplify fully and explicitly state the sign conclusion for full marks
B: Decompose then integrate
When: When asked to find a definite integral of a rational function
  1. If improper, perform long division first
  2. Decompose proper part into partial fractions
  3. Integrate each term: polynomial terms directly, A/(x-a) → A ln|x-a|
  4. Apply limits and use log laws to simplify
  5. Express answer in form p + ln q if required
+ Pros: Breaks complex integral into manageable standard forms
– Cons: Multiple steps where errors can compound
Formulas
Quotient rule: $\frac{d}{dx}\left(\frac{u}{v}\right) = \frac{vu' - uv'}{v^2}$ (formula booklet)
$\int \frac{1}{ax+b}\,dx = \frac{1}{a}\ln|ax+b| + C$ (MEMORIZE)
$\int \frac{f'(x)}{f(x)}\,dx = \ln|f(x)| + C$ (MEMORIZE)
Chain rule: $\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)$ (formula booklet)
Pitfalls
  • Forgetting the chain rule factor when differentiating (3x-k)⁻¹ — the factor of 3 is essential
  • Only computing f'(x) without explicitly stating and concluding the sign — 'show' means explain
  • Losing the 1/a factor when integrating 1/(ax+b)
  • Sign errors when applying log laws: ln a - ln b = ln(a/b) not ln(b/a)
  • Not using modulus bars in ln|x-a| during indefinite integration

Triggerspartial fractionsexpress in the formA/(x-a) + B/(x-b)repeated factordecompose
Assumptions
  • The fraction must be proper (degree of numerator < degree of denominator) before decomposition — use long division first if improper
  • Distinct linear factors (ax+b)(cx+d) → A/(ax+b) + B/(cx+d)
  • Repeated linear factor (ax+b)² → A/(ax+b) + B/(ax+b)²
  • Irreducible quadratic factor (ax²+bx+c) → (Ax+B)/(ax²+bx+c)
Approaches
A: Substitution method (cover-up)
When: Best for distinct linear factors — quick and efficient
  1. Write the identity: f(x)/[(x-a)(x-b)] ≡ A/(x-a) + B/(x-b)
  2. Multiply both sides by the full denominator
  3. Substitute x = a to eliminate B and find A
  4. Substitute x = b to eliminate A and find B
  5. For repeated factors, two substitutions give two of three constants; use a third convenient x value
+ Pros: Very fast for distinct linear factors
– Cons: For repeated or irreducible quadratic factors, doesn't directly give all constants
B: Equating coefficients
When: When substitution alone doesn't determine all constants (e.g., repeated factors, three or more unknowns)
  1. Write the identity and multiply through by the denominator
  2. Expand the right-hand side fully
  3. Compare coefficients of x², x, and constant terms
  4. Solve the resulting simultaneous equations for A, B, C
+ Pros: Always works regardless of factor type
– Cons: Slower and more prone to algebraic errors
Formulas
Distinct factors: $\frac{f(x)}{(x-a)(x-b)} = \frac{A}{x-a} + \frac{B}{x-b}$ (MEMORIZE)
Repeated factor: $\frac{f(x)}{(x-a)^2(x-b)} = \frac{A}{x-a} + \frac{B}{(x-a)^2} + \frac{C}{x-b}$ (MEMORIZE)
Integration result: $\int \frac{A}{x-a}\,dx = A\ln|x-a| + C$ (formula booklet)
Pitfalls
  • Forgetting to include B/(x-a)² term for repeated factor (x-a)² — only writing A/(x-a) + C/(x-b)
  • Not performing long division first when the fraction is improper (deg numerator ≥ deg denominator)
  • Sign errors when substituting negative values into the identity
  • Arithmetic errors when expanding brackets in the equating-coefficients method

Triggerstransformationsketch the curvenew coordinatesf(2x)f(x+a)af(x)-f(x)describe the transformation
Assumptions
  • Horizontal transformations act on x by the inverse operation: f(x + a) translates left by a
  • Vertical transformations act on y by the direct operation: af(x) stretches y by factor a
  • Order of transformations matters when both horizontal and vertical are combined
  • Asymptotes transform in the same way as points
Approaches
A: Apply transformation rules to coordinates
When: When given specific points and asked for their images
  1. For f(x + a): subtract a from each x-coordinate, y unchanged
  2. For f(ax): divide each x-coordinate by a, y unchanged
  3. For af(x): multiply each y-coordinate by a, x unchanged
  4. For f(x) + b: add b to each y-coordinate, x unchanged
  5. For -f(x): negate y-coordinates (reflection in x-axis)
  6. For f(-x): negate x-coordinates (reflection in y-axis)
  7. Apply transformations in correct order for combined expressions
+ Pros: Quick and reliable for point-by-point transformation
– Cons: Must remember horizontal transformations are inverse — common source of error
B: Describe transformation in words and apply to sketch
When: When asked to sketch the transformed curve or describe the transformation
  1. Identify each transformation component in the expression
  2. Describe each in words (e.g., 'stretch parallel to x-axis, scale factor 1/2')
  3. Apply to key features: intercepts, turning points, asymptotes
  4. Sketch the new curve showing transformed key points
+ Pros: Good for sketch questions and 'describe' questions
– Cons: Less precise for exact coordinate calculations
Formulas
$y = f(x - a)$: translation $\binom{a}{0}$ (MEMORIZE)
$y = f(x) + b$: translation $\binom{0}{b}$ (MEMORIZE)
$y = f(ax)$: stretch parallel to x-axis, scale factor $\frac{1}{a}$ (MEMORIZE)
$y = af(x)$: stretch parallel to y-axis, scale factor $a$ (MEMORIZE)
$y = -f(x)$: reflection in the x-axis (MEMORIZE)
$y = f(-x)$: reflection in the y-axis (MEMORIZE)
Pitfalls
  • For f(ax), multiplying x by a instead of dividing — the horizontal stretch factor is 1/a not a
  • Applying transformations in the wrong order for combined expressions like 2f(3x - 1)
  • Confusing f(x - 2) (translate right 2) with f(x + 2) (translate left 2)
  • Forgetting to transform asymptotes as well as key points

Triggersalgebraic divisionlong divisionquotientremainderoblique asymptoteexpress in the form Q(x) + R(x)/D(x)
Assumptions
  • Include placeholder 0x terms for missing powers when setting up division
  • The oblique asymptote is y = quotient Q(x) as x → ±∞
  • Vertical asymptotes occur where the denominator = 0 (provided no cancellation)
Approaches
A: Polynomial long division
When: Standard method; always applicable
  1. Write dividend and divisor in descending powers, inserting 0 coefficients for missing terms
  2. Divide leading term of dividend by leading term of divisor
  3. Multiply the entire divisor by this result and subtract from the dividend
  4. Bring down the next term and repeat
  5. Continue until the degree of the remainder < degree of divisor
  6. Express result as Quotient + Remainder/Divisor
+ Pros: Mechanical and reliable
– Cons: Sign errors during subtraction are very common
B: Equating coefficients (identity method)
When: When you can guess the form of the quotient and need to find specific coefficients
  1. Write f(x) = (divisor)(ax + b) + (cx + d) where degrees are chosen appropriately
  2. Expand the right-hand side
  3. Compare coefficients of each power of x
  4. Solve for unknowns a, b, c, d
+ Pros: Can be faster when the form is known; avoids long division layout
– Cons: Requires correct initial setup of the form
Formulas
$f(x) = d(x) \cdot q(x) + r(x)$ where $\deg(r) < \deg(d)$ (MEMORIZE)
Oblique asymptote: $y = q(x)$ when $\frac{f(x)}{d(x)} = q(x) + \frac{r(x)}{d(x)}$ (MEMORIZE)
Pitfalls
  • Forgetting to insert 0x placeholder terms (e.g., dividing x³ + 1 by x + 1, must write x³ + 0x² + 0x + 1)
  • Sign errors when subtracting during long division — subtract every term
  • Confusing oblique asymptote with horizontal asymptote when degrees are equal (horizontal) vs. differ by 1 (oblique)
  • Not recognizing that the question is asking for long division when it says 'express in the form'

Triggerssolvefactorisefactorizequadraticroots
Assumptions
  • If asked to factorise, the quadratic has rational roots
  • Null factor law: if AB = 0 then A = 0 or B = 0
Approaches
A: Splitting the middle term / AC method
When: When a ≠ 1 and the quadratic factorises with integer roots
  1. Compute ac (product of leading coefficient and constant)
  2. Find two numbers p, q such that p × q = ac and p + q = b
  3. Rewrite bx as px + qx
  4. Factor by grouping the four terms into two pairs
  5. Extract common binomial factor
+ Pros: Works systematically for any factorisable quadratic
– Cons: Can be slow if ac is large; need correct sign identification
B: Quadratic formula
When: When factorisation is not obvious or as a verification method
  1. Identify a, b, c from ax² + bx + c = 0
  2. Compute discriminant Δ = b² - 4ac
  3. Apply x = (-b ± √Δ) / 2a
  4. Simplify
+ Pros: Always works; also reveals nature of roots via discriminant
– Cons: Question may specifically ask for factorisation, in which case formula alone may not earn method marks
Formulas
$x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$ (formula booklet)
Discriminant: $\Delta = b^2 - 4ac$; $\Delta > 0$ two real roots, $\Delta = 0$ repeated, $\Delta < 0$ no real roots (MEMORIZE)
Pitfalls
  • Sign errors when identifying the pair of numbers for splitting the middle term
  • Forgetting to set each factor equal to zero (null factor law)
  • Arithmetic mistakes in grouping, especially with negative terms
  • Not writing the final factorised form when 'factorise' is explicitly requested

Assumptions
  • Factor theorem: (x - a) is a factor of f(x) if and only if f(a) = 0
  • Remainder theorem: when f(x) is divided by (x - a), the remainder is f(a)
  • A cubic with one known factor reduces to a quadratic that may factorise or require the formula
Approaches
A: Factor theorem then long division
When: Standard approach when one factor is given or hinted
  1. Substitute x = a into f(x) and show f(a) = 0
  2. Perform algebraic long division: f(x) ÷ (x - a) to get quadratic Q(x)
  3. Factorise Q(x) or use quadratic formula
  4. Write f(x) = (x - a)(x - b)(x - c) and state all roots
+ Pros: Straightforward and well-structured for showing working
– Cons: Long division can be error-prone with negative coefficients
B: Factor theorem then inspection/comparison
When: When you can spot the quadratic factor quickly
  1. Write f(x) = (x - a)(ax² + bx + c)
  2. Expand and compare coefficients with original polynomial
  3. Solve for a, b, c
  4. Factorise or solve the quadratic
+ Pros: Faster than formal long division if confident
– Cons: Higher risk of errors if coefficients are large or negative
Formulas
Factor theorem: $(x - a)$ is a factor of $f(x) \Leftrightarrow f(a) = 0$ (MEMORIZE)
Remainder theorem: $f(x) \div (x - a)$ has remainder $f(a)$ (MEMORIZE)
Quadratic formula: $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ (formula booklet)
Pitfalls
  • Substituting negative values incorrectly: e.g., (-2)³ = -8 not 8
  • Errors in signs during polynomial division with negative coefficients
  • Incorrectly factorising the resulting quadratic (wrong sign pair)
  • Forgetting to state all three roots of the cubic

Must memorize

  • $f^{-1}$ exists $\Leftrightarrow$ $f$ is one-to-one; domain of $f^{-1}$ = range of $f$, range of $f^{-1}$ = domain of $f$
  • $fg(x) = f(g(x))$: apply $g$ first, then $f$ — order matters
  • Partial fraction forms: distinct factors $\frac{A}{x-a} + \frac{B}{x-b}$; repeated factor adds $\frac{B}{(x-a)^2}$
  • Graph transformations: $f(x+a)$ translates left $a$; $f(ax)$ horizontal stretch factor $1/a$; vertical transformations act directly
  • $|f(x)| = k \Rightarrow f(x) = k$ or $f(x) = -k$; $|f(x)| < k \Rightarrow -k < f(x) < k$
  • $\int \frac{1}{ax+b}\,dx = \frac{1}{a}\ln|ax+b| + C$ (not in formula booklet — MEMORIZE)
  • Factor theorem: $(x-a)$ is a factor of $f(x) \Leftrightarrow f(a) = 0$
  • Quadratic formula and discriminant conditions (in formula booklet but must be fluent)

Global strategy

  • Master inverse functions: this is the single most frequently examined topic — practice rearranging rational and quadratic functions, always state domain of f⁻¹ = range of f.
  • For modulus questions, always sketch first even if not asked — it prevents missing solutions and helps identify parameter boundaries.
  • Know when to use long division vs. partial fractions: improper fractions always need division first before decomposition.
  • In multi-part questions, earlier parts often feed into later parts — e.g., partial fraction form from (a) is used for integration in (b).
  • For 'show that' questions, every algebraic step must be explicit — jumping to the answer loses method marks.
  • Always check composite function domains: fg(x) only exists where range of g ⊆ domain of f.
  • Practise the quotient rule until it is automatic — it appears frequently combined with inverse function questions to prove one-to-one.
  • Read the question for exact form requirements: 'express in the form a + b ln c' means use log laws to reach that specific structure.

Exam tips

  1. In function questions, circle the domain restriction given — forgetting x > 0 or x ≠ 1 is the #1 mark-losing error.
  2. When finding f⁻¹ leads to ±√, always refer back to the original domain to justify which sign you keep — write one sentence of justification.
  3. For modulus parameter questions (find k for 2 solutions), sketch the graph and identify the critical k values where the line is tangent to the vertex or parallel to an arm.
  4. In partial fractions, after finding constants, substitute a simple value (e.g., x = 0) back into both sides as a quick check.
  5. Show your long division working neatly in columns — examiners need to follow each subtraction step for method marks.
  6. For 'hence integrate' after partial fractions, you must use the partial fraction form — using a different method may score 0 even if correct.
  7. In composite function evaluations, write the intermediate step: fg(2) = f(g(2)) = f(value) = answer. Each step can earn marks.
  8. Time allocation: 1 mark ≈ 1.2 minutes. A 14-mark question deserves ~17 minutes; a 4-mark question only ~5 minutes.
