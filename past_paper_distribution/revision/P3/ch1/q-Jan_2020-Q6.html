Worked solution

Part (a): State the coordinates of P.

Thought Process: The function $f(x) = 2|2x-5|+3$ is a V-shaped graph. The vertex (minimum point) occurs where the expression inside the modulus equals zero, since $|\cdot| \ge 0$ and the minimum value of $|2x-5|$ is $0$.

Working:
Set $2x - 5 = 0 \Rightarrow x = \tfrac{5}{2}$

$f\!\left(\tfrac{5}{2}\right) = 2|0| + 3 = 3$

Answer: $P = \left(\dfrac{5}{2},\, 3\right)$ ✓
(2 marks)

---

Part (b): Solve $f(x) = 3x - 2$

Thought Process: We solve $2|2x-5| + 3 = 3x - 2$, i.e. $2|2x-5| = 3x - 5$. Note that the RHS must be $\ge 0$, so $x \ge \tfrac{5}{3}$. We split into two cases at $x = \tfrac{5}{2}$.

Case 1: $x \ge \tfrac{5}{2}$ so $|2x-5| = 2x-5$:
$$2(2x-5) = 3x-5 \implies 4x-10 = 3x-5 \implies x = 5$$
Check: $x = 5 \ge 2.5$ ✓

Case 2: $\tfrac{5}{3} \le x < \tfrac{5}{2}$ so $|2x-5| = 5-2x$:
$$2(5-2x) = 3x-5 \implies 10-4x = 3x-5 \implies 7x = 15 \implies x = \tfrac{15}{7}$$
Check: $\tfrac{15}{7} \approx 2.14$, and $\tfrac{5}{3} \approx 1.67$, so $\tfrac{5}{3} \le \tfrac{15}{7} < \tfrac{5}{2}$ ✓

Answer: $x = 5$ and $x = \dfrac{15}{7}$
(4 marks)

---

Part (c): Find the range of values of $k$ for which $f(x) = kx + 2$ has exactly two roots.

Thought Process: The line $y = kx + 2$ always passes through $(0, 2)$. We need it to cross the V-shaped graph $y = f(x)$ exactly twice. The two arms of $f(x)$ (for $x \ge 0$) are:


  • Left arm ($0 \le x < 2.5$): $f(x) = -4x + 13$ with gradient $-4$

  • Right arm ($x \ge 2.5$): $f(x) = 4x - 7$ with gradient $4$

Note $f(0) = 13$ and the line passes through $(0,2)$, which is below the graph at $x=0$.

Intersection with left arm: $-4x + 13 = kx + 2 \implies x = \dfrac{11}{k+4}$. Valid when $0 \le x < 2.5$, requiring $k > -4$ and $\dfrac{11}{k+4} < 2.5 \implies k + 4 > 4.4 \implies k > \tfrac{2}{5}$.

Intersection with right arm: $4x - 7 = kx + 2 \implies x = \dfrac{9}{4-k}$. Valid when $x \ge 2.5$, requiring $k < 4$ and $\dfrac{9}{4-k} \ge 2.5 \implies 4-k \le 3.6 \implies k \ge \tfrac{2}{5}$.

Critical value $k = \tfrac{2}{5}$: Both equations give $x = 2.5$, so the line passes through the vertex — only one intersection point.

Critical value $k = 4$: Right arm gives no solution (parallel lines). Left arm: $x = \tfrac{11}{8} = 1.375 < 2.5$ ✓, so only one root.

For exactly two roots, we need both intersections to be valid, which requires $k > \tfrac{2}{5}$ (left arm valid) and $\tfrac{2}{5} \le k < 4$ (right arm valid). Since $k = \tfrac{2}{5}$ gives only one root (at vertex), we need strict inequality on both sides.

Answer: $\dfrac{2}{5}
< k < 4$ (3 marks)

Key techniques

  • identifying vertex of modulus graph
  • splitting modulus equations into cases
  • solving linear equations
  • interpreting number of roots graphically (line-curve intersection)
  • boundary/critical gradient analysis

Common pitfalls

  • Forgetting that k = 2/5 gives a tangent at the vertex (one root, not two)
  • Not checking that k = 4 makes the line parallel to the right arm (no intersection on right arm)
  • Neglecting the domain x ≥ 0 when checking validity of solutions
  • Forgetting to verify solutions satisfy the original modulus equation (not just the case condition)
