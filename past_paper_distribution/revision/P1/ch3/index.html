Area & Perimeter of Composite Shapes (Sectors & Triangles)
15×
7-14m

A diagram with a shape made from sectors, triangles, and sometimes other polygons (kites, trapeziums). The question asks for a perimeter, an area (often shaded), or to 'show that' a value is equal to a given expression.

Core strategy
  1. 1. Identify all the basic shapes that make up the composite figure (e.g., Sector OAB, Triangle OBC).
  2. 2. State your plan. For area, this might be 'Shaded Area = Area of Sector - Area of Triangle'. For perimeter, trace the outer edge and list the segments to sum.
  3. 3. Check the angle units required. For ALL sector/arc length formulas ($A = \frac{1}{2}r^2\theta$, $s = r\theta$), angles MUST be in RADIANS. Set calculator mode accordingly.
  4. 4. Systematically find any missing lengths or angles needed for your plan. This often involves using the Sine Rule, Cosine Rule, or basic SOH CAH TOA in right-angled triangles.
  5. 5. Substitute known values into the relevant formulas for each component shape.
  6. 6. Combine the results as per your plan from step 2 to find the final area or perimeter.
  7. 7. For 'show that' questions, do not round intermediate steps. Keep values exact or stored in your calculator to arrive precisely at the target expression.
Must-know formulas
Area of a sector: $A = \frac{1}{2}r^2\theta$ (for $\theta$ in radians) (formula booklet)
Arc length: $s = r\theta$ (for $\theta$ in radians) (formula booklet)
Area of a triangle: $A = \frac{1}{2}ab\sin C$ (formula booklet)
Cosine Rule: $a^2 = b^2 + c^2 - 2bc \cos A$ (formula booklet)
Sine Rule: $\frac{a}{\sin A} = \frac{b}{\sin B}$ (formula booklet)
Common pitfalls
  • Using degrees in the sector area or arc length formulas which are strictly for radians.
  • Prematurely rounding intermediate values, leading to an inaccurate final answer, especially in 'show that' questions.
  • Forgetting to add the straight-edge radii when calculating the perimeter of a sector, only calculating the arc length.
  • Using the minor sector angle when the major sector is required (or vice versa).

Trigonometric Graphs (Transformations & Solutions)
11×
4-8m

A question involving a function like $y = A \sin(kx) + B$ or $y = A \cos(k(x - c))$. It may ask for a sketch, coordinates of max/min points, intercepts, the period, or the number of solutions to an equation like $f(x) = c$.

Core strategy
  1. 1. Identify the transformations from the parent graph ($y=\sin x$ or $y=\cos x$).
  2. 2. Determine Amplitude ($|A|$): This is the vertical stretch. Max value is $D+|A|$, Min value is $D-|A|$ for $y=A\sin(x)+D$.
  3. 3. Determine Period: This is the horizontal compression. For $y=\sin(kx)$, the period is $360^\circ/k$ or $2\pi/k$. This is one full cycle.
  4. 4. Determine Vertical Shift ($B$): This moves the whole graph up/down. The principal axis is now $y=B$.
  5. 5. To sketch, start with the parent graph's key points (0, 90, 180, 270, 360 degrees) and apply the transformations to find the new key points.
  6. 6. To find the number of solutions to $f(x)=c$, sketch both $y=f(x)$ and the line $y=c$ and count the intersections within the given domain.
  7. 7. For large domains (e.g., $0 < x < 2000^\circ$), calculate the number of solutions in one period and multiply by the number of periods in the domain.
Must-know formulas
For $y = A \cos(kx) + B$: Amplitude is $|A|$, Period is $\frac{360^\circ}{k}$ or $\frac{2\pi}{k}$, Vertical shift is $B$ (MEMORIZE CONCEPTS)
Max value = $B + |A|$
Min value = $B - |A|$
Common pitfalls
  • Incorrectly calculating the period, e.g., using $2\pi \times k$ instead of $2\pi / k$.
  • Confusing amplitude with the maximum value. Amplitude is the distance from the principal axis to the peak.
  • Mixing up degrees and radians when stating coordinates or calculating the period.
  • When counting solutions, only finding them in the first cycle and forgetting to use periodicity to find all solutions in the full domain.

Solving Triangles (Sine & Cosine Rules)
3×
5-7m

A question describing a non-right-angled triangle ABC with some side lengths and angles given. It asks to find a missing side or angle, often with a hint towards an 'obtuse' or 'acute' angle.

Core strategy
  1. 1. Sketch the triangle and label all known sides and angles.
  2. 2. Decide which rule to use. Use Cosine Rule if you have Side-Angle-Side (SAS) to find the opposite side, or Side-Side-Side (SSS) to find an angle.
  3. 3. Use Sine Rule if you have Angle-Side-Angle (ASA) or Angle-Angle-Side (AAS) to find a side, or Side-Side-Angle (SSA) to find an angle.
  4. 4. Be alert for the 'Ambiguous Case' (SSA). When using Sine Rule to find an angle $\theta$, your calculator gives $\theta = \arcsin(k)$. The other possible solution is $180^\circ - \theta$.
  5. 5. Use context from the problem (e.g., 'angle ABC is obtuse', 'side AC is the longest') to decide which of the two possible angles is correct. The largest angle is opposite the longest side.
  6. 6. If finding a side using the Cosine Rule, ensure you use the angle *between* the two known sides.
  7. 7. For area, use the formula $A = \frac{1}{2}ab\sin C$, where C is the angle *included* between sides a and b.
Must-know formulas
Cosine Rule: $c^2 = a^2 + b^2 - 2ab\cos C$ (formula booklet)
Sine Rule (for sides): $\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}$ (formula booklet)
Sine Rule (for angles): $\frac{\sin A}{a} = \frac{\sin B}{b} = \frac{\sin C}{c}$ (formula booklet)
Area of a triangle: $A = \frac{1}{2}ab\sin C$ (formula booklet)
Ambiguous Case solution: $\theta_2 = 180^\circ - \theta_1$ or $\theta_2 = \pi - \theta_1$ (MEMORIZE CONCEPT)
Common pitfalls
  • Forgetting the ambiguous case of the Sine Rule and only finding the acute angle solution.
  • Incorrectly applying the Cosine Rule, usually by matching the wrong angle with the side on the left-hand side of the equation.
  • Calculation errors with squares and square roots in the Cosine Rule.
  • Using the area formula $1/2ab\sin C$ with an angle that is not included between sides a and b.

Solving Trigonometric Equations (Identities & Quadratics)
2×
6-8m

An equation containing trigonometric functions, such as $3\sin^2 x + 7\sin x + 2 = 0$. The question asks for all solutions within a given interval, like $0 \le x < 360^\circ$.

Core strategy
  1. 1. Identify the trigonometric functions involved. If there's a mix (e.g., $\sin$ and $\cos^2$), aim to get the equation in terms of a single function.
  2. 2. Use the identity $\sin^2 x + \cos^2 x = 1$ to substitute one for the other (e.g., replace $\cos^2 x$ with $1 - \sin^2 x$).
  3. 3. Rearrange the equation into a standard quadratic form, like $a y^2 + b y + c = 0$, where $y$ might be $\sin x$ or $\cos x$.
  4. 4. Solve the quadratic for $y$ by factorising or using the quadratic formula. This will give one or two values, e.g., $\sin x = -1/3$ and $\sin x = -2$.
  5. 5. For each valid solution (e.g., $\sin x$ must be between -1 and 1), find the Principal Value (PV) using your calculator (e.g., $x = \arcsin(-1/3)$).
  6. 6. Use a CAST diagram or a sketch of the trigonometric graph to find all other solutions within the required range.
  7. 7. List all solutions clearly, ensuring they are within the specified interval and to the correct degree of accuracy.
Must-know formulas
$\sin^2 x + \cos^2 x = 1$ (formula booklet)
Quadratic Formula: $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ (formula booklet)
Common pitfalls
  • Algebraic errors when rearranging the equation into a quadratic.
  • Forgetting to find all solutions in the given range; often students only state the principal value from the calculator.
  • Incorrectly using the CAST diagram or graph symmetry to find the secondary solutions.
  • Trying to solve an impossible equation like $\sin x = -2$, which has no real solutions.

Must memorize

  • Area of a sector: $A = \frac{1}{2}r^2\theta$ (for $\theta$ in radians) (formula booklet)
  • Arc length: $s = r\theta$ (for $\theta$ in radians) (formula booklet)
  • Area of a triangle: $A = \frac{1}{2}ab\sin C$ (formula booklet)
  • Cosine Rule: $a^2 = b^2 + c^2 - 2bc \cos A$ (formula booklet)
  • Sine Rule: $\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}$ (formula booklet)
  • Pythagorean Identity: $\sin^2 \theta + \cos^2 \theta = 1$ (formula booklet)
  • Period of $f(kx)$ is Period of $f(x)$ divided by $k$. For sin/cos, period is $\frac{360^\circ}{k}$ or $\frac{2\pi}{k}$. (MEMORIZE CONCEPT)
  • Ambiguous Case solution for $\sin x = k$: if $\alpha$ is the principal value, the other solution is $180^\circ - \alpha$ (or $\pi - \alpha$). (MEMORIZE CONCEPT)

Global strategy

  • RADIANS vs DEGREES: This is the single biggest source of error. Always check the question for units (a '°' symbol vs 'radians'). Set your calculator mode BEFORE you start. Sector/arc formulas ONLY work in radians.
  • SKETCH & ANNOTATE: For geometry problems, a large, clear, annotated diagram is your best tool. Mark all given lengths and angles, and add new ones as you find them. For graph problems, a quick sketch helps visualize transformations and solutions.
  • BEWARE THE AMBIGUOUS CASE: When using the Sine Rule to find an angle (SSA), always remember that $\sin(x) = \sin(180^\circ-x)$. Check if the obtuse angle is a geometrically valid solution.
  • 'SHOW THAT' PRECISION: In questions that ask you to show a result, you must provide clear, logical steps. Crucially, do not round any intermediate calculations. Store full values in your calculator memory to avoid rounding errors.
  • BREAK DOWN COMPLEX SHAPES: Don't be intimidated by high-mark composite shape questions. Break the problem into smaller, familiar parts: Area of sector, area of triangle, etc. Write down a clear plan (e.g., 'Shaded Area = Sector Area - Triangle Area') before you start calculating.
  • FIND ALL SOLUTIONS: When solving a trigonometric equation, don't just stop at the principal value from your calculator. Use graph sketches or a CAST diagram to find every single solution within the required domain.

Exam tips

  1. Calculator Mode First: Before reading the question fully, scan for a degree symbol '°' or the word 'radians'. Set your calculator mode immediately. It's the easiest mark-saver.
  2. Don't Round Too Soon: For multi-step calculations, especially in geometry, store intermediate answers in your calculator's memory (e.g., STO A). Only round your final answer to the specified accuracy.
  3. Show Your Method: For 'show that' questions and high-mark geometry problems, clearly state the formulas you are using and show the values you are substituting. This helps secure method marks even if you make a calculation slip.
  4. Check Reasonableness: Does your answer make sense? If you calculate an angle in a triangle to be $200^\circ$, you've made a mistake. If a side length is negative, re-check your working.
  5. Manage Time on Long Questions: The composite shape questions can be long. If you get stuck on one part (e.g., finding an angle), write down a sensible assumed value and use it to attempt the later parts of the question to get 'follow-through' marks.
