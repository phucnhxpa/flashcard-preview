Worked solution
  • Part (a):
  • Thought process: We recognise $y = \cos(2x)$ as a horizontally compressed version of $y = \cos x$. The period is halved to $\pi$. The standard $\cos x$ curve has its first minimum at $x = \pi$ with a value of $-1$. For $y = \cos(2x)$, this minimum occurs when the argument $2x$ equals $\pi$.
  • Working:
The minimum value of $\cos(2x)$ is $-1$. This occurs when $2x = \pi, 3\pi, \dots$. For the point P, which is the first minimum for $x > 0$, we set $2x = \pi$. Solving for $x$, we get $x = \frac{\pi}{2}$. The $y$-coordinate at this minimum is $-1$.
Answer: $(\frac{\pi}{2}, -1)$
  • Part (b):
  • Thought process: We need to sketch the standard $y = \sin x$ curve. Key features are its starting point at the origin $(0,0)$, an amplitude of $1$, and a period of $2\pi$. We'll plot its x-intercepts, maxima, and minima to guide the sketch over the given range.
  • Working:
Sketch $y = \sin x$ with the following key characteristics:
  • Passes through $(0,0)$, $(\pi,0)$, $(2\pi,0)$, $(3\pi,0)$ (x-intercepts).
  • Reaches a maximum value of $1$ at $x = \frac{\pi}{2}$ and $x = \frac{5\pi}{2}$.
  • Reaches a minimum value of $-1$ at $x = \frac{3\pi}{2}$.
Draw a smooth, continuous wave that follows these points and has an amplitude of $1$ relative to the given axes.

  • Part (c)(i):
  • Thought process: To find the number of solutions for $\cos(2x) = \sin(x)$, we should solve the equation algebraically. The key is to use a double angle identity for $\cos(2x)$ to convert the equation into one involving only $\sin x$. This will result in a quadratic equation. After finding solutions for $\sin x$, we determine the corresponding $x$ values in a single $2\pi$ cycle, then extrapolate for the larger $20\pi$ range.
  • Working:
The given equation is $\cos(2x) = \sin(x)$. Using the double angle identity $\cos(2x) = 1 - 2\sin^2(x)$: $$1 - 2\sin^2(x) = \sin(x)$$ Rearrange into a quadratic equation in terms of $\sin(x)$: $$2\sin^2(x) + \sin(x) - 1 = 0$$ Let $u = \sin(x)$. The equation becomes: $$2u^2 + u - 1 = 0$$ Factorise the quadratic: $$(2u - 1)(u + 1) = 0$$ This yields two possible values for $u$: $$u = \frac{1}{2} \quad \text{or} \quad u = -1$$ Substitute back $u = \sin(x)$:
  1. If $\sin(x) = \frac{1}{2}$: In the interval $0 \le x < 2\pi$, the solutions are $x = \frac{\pi}{6}$ and $x = \pi - \frac{\pi}{6} = \frac{5\pi}{6}$.
  2. If $\sin(x) = -1$: In the interval $0 \le x < 2\pi$, the unique solution is $x = \frac{3\pi}{2}$.
Thus, there are $3$ distinct solutions in any interval of $2\pi$ (e.g., $0 \le x < 2\pi$). The region $0 \le x \le 20\pi$ covers exactly $10$ full $2\pi$ cycles ($20\pi / 2\pi = 10$). Total number of solutions = $10 \times 3 = 30$.
Answer: 30
  • Part (c)(ii):
  • Thought process: We know there are $30$ solutions in $0 \le x \le 20\pi$. The new range $0 \le x \le 21\pi$ extends an additional $\pi$ beyond $20\pi$. We need to count how many *new* solutions occur in the interval $20\pi < x \le 21\pi$ by considering the periodicity of the functions.
  • Working:
From part (c)(i), we have $30$ solutions in $0 \le x \le 20\pi$. Now, consider the additional interval $20\pi < x \le 21\pi$. Due to the $2\pi$ periodicity of $\sin(x)$ and $\cos(2x)$ (since $\cos(2(x+2\pi)) = \cos(2x+4\pi) = \cos(2x)$), this interval is equivalent to $0 < x' \le \pi$ for $x' = x - 20\pi$. In the interval $0 < x' \le \pi$:
  • For $\sin(x') = \frac{1}{2}$: The solutions are $x' = \frac{\pi}{6}$ and $x' = \frac{5\pi}{6}$. (2 solutions)
  • For $\sin(x') = -1$: There are no solutions in this interval ($x' = \frac{3\pi}{2}$ is outside $0 < x' \le \pi$).
Therefore, there are $2$ additional solutions in $20\pi < x \le 21\pi$. Total number of solutions = $30 + 2 = 32$.
Answer: 32

Key techniques

  • Identify properties of transformed trig graphs
  • Solve trig equations using identities
  • Apply periodicity to count solutions over large intervals

Common pitfalls

  • Incorrectly determining the period or phase shift for transformed trigonometric graphs (e.g., $y = \cos(2x)$).
  • Algebraic errors when solving the quadratic in $\sin(x)$, or failing to use the double angle identity.
  • Miscounting solutions when extrapolating over large intervals, especially at boundaries or in partial cycles (e.g., $0 \le x \le 21\pi$ vs. $0 \le x \le 20\pi$). Students might mistakenly multiply by $21/2$ or forget to check the partial cycle.
