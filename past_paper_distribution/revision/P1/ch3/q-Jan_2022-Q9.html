Worked solution

Part (a): Write down the value of A.

Thought Process:
We recognize the general form of the curve as $y = A \cos(x - \alpha)$ where $A$ is the amplitude scaling factor and $\alpha$ is the horizontal phase shift. The given equation is $y = A \cos(x - 30)°$, so the phase shift $\alpha = 30°$.
We are given a minimum point $P(30, -3)$. We substitute these coordinates into the equation to find $A$.
When $x = 30$, the argument of the cosine function becomes $(30 - 30)° = 0°$. We know that $\cos(0°) = 1$. So, at this point, $y = A \times 1 = A$.
Since the $y$-coordinate of $P$ is $-3$, we can directly deduce that $A = -3$.
The negative sign for $A$ is crucial; it means the standard cosine graph, which has a maximum at $x=0$, is reflected in the x-axis, turning that maximum into a minimum when $A=-3$.

Working:
Given the equation $y = A \cos(x - 30)°$.
The point $P(30, -3)$ is a minimum point on the curve.
Substitute the coordinates of $P$ into the equation:
$-3 = A \cos(30 - 30)°$
$-3 = A \cos(0)°$
Since $\cos(0)° = 1$:
$-3 = A(1)$
$A = -3$

Answer: $A = -3$

---

Part (b): Find the coordinates of Q.

Thought Process:
Now we know the full equation of the curve is $y = -3 \cos(x - 30)°$.
Point $Q$ is a maximum point. For a maximum point on $y = -3 \cos(x - 30)°$:


  1. Find the y-coordinate: The amplitude of the curve is $|A| = |-3| = 3$. Since the curve is $y = -3 \cos(...)$, a maximum occurs when $\cos(...)$ takes its minimum value, which is $-1$. So, $y_{max} = -3 \times (-1) = 3$.

  2. Find the x-coordinate: We need to find $x$ such that $\cos(x - 30)° = -1$.


The principal value for $\theta$ where $\cos(\theta) = -1$ is $\theta = 180°$.
Due to the periodic nature of the cosine function, the general solution for $\cos(x - 30)° = -1$ is:
$x - 30 = 180 + 360n$, where $n$ is an integer.
So, $x = 210 + 360n$.
We need to select the correct value of $n$ corresponding to point $Q$ as shown in Figure 4. Point $P$ is at $x=30$ (a minimum). Let's list the positions of maxima:

  • For $n=0$: $x = 210 + 360(0) = 210°$. This is the first maximum after $x=0$.

  • For $n=1$: $x = 210 + 360(1) = 570°$. This is the second maximum.

  • For $n=2$: $x = 210 + 360(2) = 210 + 720 = 930°$. This is the third maximum.


Observing Figure 4, the point $Q$ is shown significantly far along the positive x-axis, past several cycles. It appears to be the third maximum shown after $x=0$, which corresponds to $x=930°$. The graph visually supports taking a higher value of $n$.

Working:
The equation of the curve is $y = -3 \cos(x - 30)°$.
For a maximum point $Q$, the value of $\cos(x - 30)°$ must be $-1$.
So, $y_Q = -3 \times (-1) = 3$.

Set $\cos(x - 30)° = -1$:
$x - 30 = 180 + 360n \quad$ (where $n$ is an integer)
$x = 210 + 360n$

We need to find the appropriate $x$-value for $Q$ from the figure.
For $n=0, x = 210°$.
For $n=1, x = 570°$.
For $n=2, x = 930°$.

Looking at Figure 4, point $Q$ is clearly shown to be several cycles after the y-axis, well beyond $x=210$ and $x=570$. The value $x=930°$ is consistent with the visual representation of $Q$ in Figure 4.

Thus, the coordinates of $Q$ are $(930, 3)$.

Answer: $Q = (930, 3)$

Key techniques

  • interpret A in y=Acos(x-alpha)
  • understand phase shift
  • identify min/max points on transformed trig graphs
  • use periodicity to find multiple solutions

Common pitfalls

  • Incorrectly determining the sign of A (e.g., A=3 instead of A=-3) in part (a).
  • Calculating only the first maximum point (e.g., (210, 3)) in part (b) and not using the visual information in Figure 4 to find the correct cycle.
  • Swapping the x and y coordinates of Q (e.g., (3, 930)).
