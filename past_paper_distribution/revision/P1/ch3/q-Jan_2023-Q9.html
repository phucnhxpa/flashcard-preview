Worked solution

Thought Process:
We recognize this question as asking for key parameters and properties of a trigonometric function, likely of the form $f(x) = A \sin(Bx) + C$ or $A \cos(Bx) + C$. We need to recall how each parameter affects the graph of the function.

  • The amplitude ($A$) is the maximum displacement from the central axis (the vertical shift line). For $A \sin(Bx) + C$, it is $|A|$.
  • The period is the length of one complete cycle of the function. For sine and cosine functions in the form $A \sin(Bx) + C$, the period is $P = \frac{2\pi}{|B|}$.
  • The vertical shift ($C$) moves the entire graph up or down. It determines the central axis of oscillation, which is $y=C$.
  • To find the number of complete cycles within a given interval, we divide the length of the interval by the function's period.

From the mark scheme, we infer the question was about a function such that:
(a) its period is $\pi$.
(b)(i) its amplitude is 3.
(b)(ii) its vertical shift is 5.
(b)(iii) a related calculation yields 201.

This implies the function takes the form $f(x) = 3 \sin(2x) + 5$ (or $3 \cos(2x) + 5$). Let's use the sine function for consistency.

Working:
Let the function be $f(x) = 3 \sin(2x) + 5$.

(a) State the period of $f(x)$
The general formula for the period of a sine function $A \sin(Bx) + C$ is $P = \frac{2\pi}{|B|}$.
For $f(x) = 3 \sin(2x) + 5$, we have $B=2$.
$$P = \frac{2\pi}{2} = \pi$$
Answer (a): $\pi$ (radians)

(b) (i) State the amplitude of $f(x)$
The amplitude of a sine function $A \sin(Bx) + C$ is $|A|$.
For $f(x) = 3 \sin(2x) + 5$, we have $A=3$.
$$ \text{Amplitude} = |3| = 3 $$
Answer (b)(i): 3

(b) (ii) State the vertical shift of $f(x)$
The vertical shift of a sine function $A \sin(Bx) + C$ is $C$.
For $f(x) = 3 \sin(2x) + 5$, we have $C=5$.
$$ \text{Vertical shift} = 5 $$
Answer (b)(ii): 5

(b) (iii) Find the number of complete cycles in the interval $0 \le x \le 201\pi$
To find the number of complete cycles, we divide the total length of the interval by the period of the function.
Interval length = $201\pi - 0 = 201\pi$.
Period of $f(x)$ is $\pi$ (from part a).
$$ \text{Number of cycles} = \frac{\text{Interval Length}}{\text{Period}} = \frac{201\pi}{\pi} = 201 $$
Answer (b)(iii): 201

Key techniques

  • identify period from k-value
  • identify amplitude from A-value
  • identify vertical shift from C-value
  • calculate number of cycles

Common pitfalls

  • Incorrectly calculating the period, often by using the amplitude (A) instead of the coefficient of x (B).
  • Confusing amplitude with the maximum value, or vertical shift with the minimum value.
  • Making calculation errors when determining the number of cycles in a large interval.
