Worked solution

Let the curve C have equation $y = f(x) = x^{3/2} - 8x^{-1/2}$.

Part (a): Find the equation of the tangent to the curve at the point where $x=4$.

Thought Process: To find the equation of a tangent line, we need two things: a point $(x_1, y_1)$ on the line and the gradient $m$ of the line. The problem gives us the $x$-coordinate, $x=4$. We'll find the corresponding $y$-coordinate by substituting $x=4$ into the original curve's equation. The gradient of the tangent at any point is given by the derivative $\frac{dy}{dx}$, so we'll differentiate $y$ with respect to $x$ and then substitute $x=4$ into $\frac{dy}{dx}$ to find $m$.

Working:


  1. Find the $y$-coordinate of the point of tangency:


Substitute $x=4$ into the curve's equation $y = x^{3/2} - 8x^{-1/2}$:
$$y = (4)^{3/2} - 8(4)^{-1/2}$$
Recall that $x^{3/2} = (\sqrt{x})^3$ and $x^{-1/2} = \frac{1}{\sqrt{x}}$.
$$y = (\sqrt{4})^3 - 8\left(\frac{1}{\sqrt{4}}\right)$$
$$y = (2)^3 - 8\left(\frac{1}{2}\right)$$
$$y = 8 - 4$$
$$y = 4$$
So, the point of tangency is $(4, 4)$.

  1. Find the derivative $\frac{dy}{dx}$:
Given $y = x^{3/2} - 8x^{-1/2}$. Apply the power rule for differentiation, $\frac{d}{dx}(ax^n) = anx^{n-1}$: $$\frac{dy}{dx} = \frac{3}{2}x^{\frac{3}{2}-1} - 8 \times \left(-\frac{1}{2}\right)x^{-\frac{1}{2}-1}$$ $$\frac{dy}{dx} = \frac{3}{2}x^{1/2} + 4x^{-3/2}$$
  1. Find the gradient $m$ at $x=4$:
Substitute $x=4$ into the derivative $\frac{dy}{dx}$: $$m = \frac{3}{2}(4)^{1/2} + 4(4)^{-3/2}$$ $$m = \frac{3}{2}(\sqrt{4}) + 4\left(\frac{1}{(\sqrt{4})^3}\right)$$ $$m = \frac{3}{2}(2) + 4\left(\frac{1}{2^3}\right)$$ $$m = 3 + 4\left(\frac{1}{8}\right)$$ $$m = 3 + \frac{1}{2}$$ $$m = \frac{7}{2}$$
  1. Find the equation of the tangent line:
Using the point $(x_1, y_1) = (4, 4)$ and the gradient $m = \frac{7}{2}$ in the formula $y - y_1 = m(x - x_1)$: $$y - 4 = \frac{7}{2}(x - 4)$$ Multiply by 2 to clear the fraction: $$2(y - 4) = 7(x - 4)$$ $$2y - 8 = 7x - 28$$ Rearrange into the form $Ax + By + C = 0$: $$7x - 2y - 28 + 8 = 0$$ $$7x - 2y - 20 = 0$$

Part (b): Given that $y = f(x)$, find $\int f(x) dx$, given that the integrated function passes through the point $(4, 12)$.

Thought Process: We need to find the indefinite integral of $f(x)$. This means applying the power rule for integration $\int ax^n dx = \frac{a}{n+1}x^{n+1} + C$. The integration will introduce a constant $C$. We are given a specific point $(4, 12)$ that the integrated function passes through. We will substitute these values into our integrated function to solve for $C$.

Working:


  1. Integrate $f(x) = x^{3/2} - 8x^{-1/2}$:


$$\int (x^{3/2} - 8x^{-1/2}) dx = \frac{x^{3/2+1}}{3/2+1} - 8\frac{x^{-1/2+1}}{-1/2+1} + C$$
$$= \frac{x^{5/2}}{5/2} - 8\frac{x^{1/2}}{1/2} + C$$
$$= \frac{2}{5}x^{5/2} - 16x^{1/2} + C$$

  1. Use the point $(4, 12)$ to find the constant $C$:
Let the integrated function be $F(x) = \frac{2}{5}x^{5/2} - 16x^{1/2} + C$. We are given $F(4) = 12$. $$12 = \frac{2}{5}(4)^{5/2} - 16(4)^{1/2} + C$$ $$12 = \frac{2}{5}(\sqrt{4})^5 - 16(\sqrt{4}) + C$$ $$12 = \frac{2}{5}(2^5) - 16(2) + C$$ $$12 = \frac{2}{5}(32) - 32 + C$$ $$12 = \frac{64}{5} - 32 + C$$ $$12 = 12.8 - 32 + C$$ $$12 = -19.2 + C$$ $$C = 12 + 19.2$$ $$C = 31.2$$ This can also be expressed as a fraction: $C = \frac{312}{10} = \frac{156}{5}$.
  1. Write the full integrated function:
$$F(x) = \frac{2}{5}x^{5/2} - 16x^{1/2} + \frac{156}{5}$$

Answer:
(a) The equation of the tangent is $7x - 2y - 20 = 0$.
(b) The integrated function is $F(x) = \frac{2}{5}x^{5/2} - 16x^{1/2} + \frac{156}{5}$.

Key techniques

  • Apply power rule for differentiation: $d/dx(ax^n) = anx^{n-1}$
  • Calculate gradient of tangent by substituting $x$-value into $dy/dx$
  • Use point-gradient formula $y - y_1 = m(x - x_1)$ to find tangent equation
  • Apply power rule for integration: $\int ax^n dx = \frac{a}{n+1}x^{n+1} + C$
  • Determine constant of integration $C$ using a given point

Common pitfalls

  • Sign errors when differentiating or integrating terms with negative powers (e.g., $-8x^{-1/2}$ becoming $+4x^{-3/2}$).
  • Arithmetic mistakes when evaluating fractional powers or simplifying fractions after substitution (e.g., $(4)^{3/2} \ne 6$).
  • Forgetting to add the constant of integration $+C$ for indefinite integrals, or failing to use the given point to calculate its value.
