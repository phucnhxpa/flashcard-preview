Worked solution

*(a) Thought Process:*
We are given two simultaneous equations and asked to show that they lead to a specific quartic equation in $x$. The target equation, $2x^4 - 15x^2 - 50 = 0$, contains only $x$ terms. This implies that we need to eliminate $y$ from the system. The second equation, $y - x^3 + 6x = 0$, is simpler to rearrange to make $y$ the subject, as $y$ is already a single term. Once we have an expression for $y$ in terms of $x$, we can substitute it into the first equation to obtain an equation solely in $x$.

*(a) Working:*
Given the equations:
1) $2xy - 3x^2 = 50$
2) $y - x^3 + 6x = 0$

From equation (2), we can make $y$ the subject:
$$y = x^3 - 6x$$

Substitute this expression for $y$ into equation (1):
$$2x(x^3 - 6x) - 3x^2 = 50$$

Expand the bracket carefully:
$$2x^4 - 12x^2 - 3x^2 = 50$$

Combine the $x^2$ terms:
$$2x^4 - 15x^2 = 50$$

Rearrange to match the required form by moving the constant term to the left side:
$$2x^4 - 15x^2 - 50 = 0$$
This matches the given equation, as required.

*(b) Thought Process:*
We need to "hence solve" the simultaneous equations, meaning we must use the equation derived in part (a), $2x^4 - 15x^2 - 50 = 0$. This is a quartic equation, but it is in quadratic form with respect to $x^2$. We can treat $x^2$ as a new variable (e.g., $u$) to solve for $x^2$ first. After finding the values for $x^2$, we can find the values for $x$. Remember that $x^2$ must be non-negative for real solutions for $x$. Finally, substitute the found $x$ values back into one of the original equations to find the corresponding $y$ values. The equation $y = x^3 - 6x$ (from part a) is the most straightforward for this. All answers must be in fully simplified surd form, and calculator reliance is not allowed.

*(b) Working:*
We need to solve $2x^4 - 15x^2 - 50 = 0$.
Let $u = x^2$. Substituting this into the equation gives a quadratic in $u$:
$$2u^2 - 15u - 50 = 0$$

We can solve this quadratic by factorisation. We look for two numbers that multiply to $2 \times -50 = -100$ and add to $-15$. These numbers are $-20$ and $5$.
So, rewrite the middle term and factorise by grouping:
$$2u^2 - 20u + 5u - 50 = 0$$
$$2u(u - 10) + 5(u - 10) = 0$$
$$(2u + 5)(u - 10) = 0$$

This gives two possible values for $u$:
$2u + 5 = 0 \Rightarrow 2u = -5 \Rightarrow u = -\frac{5}{2}$
$u - 10 = 0 \Rightarrow u = 10$

Now substitute back $u = x^2$:
Case 1: $x^2 = -\frac{5}{2}$
There are no real solutions for $x$ when $x^2$ is negative. So, we discard this case for real solutions.

Case 2: $x^2 = 10$
Taking the square root of both sides gives:
$$x = \pm\sqrt{10}$$

Now we find the corresponding $y$ values using $y = x^3 - 6x$:

For $x = \sqrt{10}$:
$$y = (\sqrt{10})^3 - 6(\sqrt{10})$$
Recall that $(\sqrt{10})^3 = \sqrt{10} \times \sqrt{10} \times \sqrt{10} = 10\sqrt{10}$.
$$y = 10\sqrt{10} - 6\sqrt{10}$$
$$y = 4\sqrt{10}$$
So, one solution pair is $(x, y) = (\sqrt{10}, 4\sqrt{10})$.

For $x = -\sqrt{10}$:
$$y = (-\sqrt{10})^3 - 6(-\sqrt{10})$$
Recall that $(-\sqrt{10})^3 = (-\sqrt{10}) \times (-\sqrt{10}) \times (-\sqrt{10}) = -10\sqrt{10}$.
$$y = -10\sqrt{10} + 6\sqrt{10}$$
$$y = -4\sqrt{10}$$
So, the other solution pair is $(x, y) = (-\sqrt{10}, -4\sqrt{10})$.

Answer: The solutions are $(\sqrt{10}, 4\sqrt{10})$ and $(-\sqrt{10}, -4\sqrt{10})$.

Key techniques

  • substitution method
  • factorising quadratics
  • simplifying surds
  • handling multiple solutions

Common pitfalls

  • Algebraic errors when expanding brackets after substitution, e.g., $2x(-6x)$ resulting in $-12x$ instead of $-12x^2$.
  • Forgetting the $\pm$ sign when taking the square root of $x^2=k$, thus missing one pair of solutions.
  • Not simplifying surds fully or making errors in surd manipulation, such as $(\sqrt{10})^3 = 10\sqrt{10}$.
