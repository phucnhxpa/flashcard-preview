Worked solution

Thought Process

Part (a): Find the difference at monitoring began.
We are asked to find the difference in share values at the point monitoring began. This corresponds to $t=0$. We need to substitute $t=0$ into both $P_A$ and $P_B$ equations and then find their positive difference.

Part (b): State the maximum share value of company A.
Company A's share value is modelled by $P_A = 53 - 0.4(t-8)^2$. This is a quadratic function in vertex form $y = k - a(x-h)^2$. Since the coefficient of $(t-8)^2$ is negative ($-0.4$), the parabola opens downwards, meaning its vertex represents the maximum point. The maximum value occurs when the squared term $(t-8)^2$ is zero, which happens at $t=8$. The maximum value is $53$. We must confirm that $t=8$ falls within the 15-year monitoring period, which it does ($0 \le 8 \le 15$).

Part (c): Find times when $P_A > P_B$.
We need to set up the inequality $P_A > P_B$ and solve it algebraically for $t$. This will result in a quadratic inequality. We'll expand and simplify, find the roots of the corresponding quadratic equation, and then determine the interval(s) where the inequality holds. Finally, we must restrict the solution to the given domain for the models, which is $0 \le t \le 15$.

Part (d): Explain why the model for company A should not be used for $t=20$.
Mathematical models often have limitations on their applicability. We are told the monitoring was for a '15-year period'. Therefore, predicting beyond $t=15$ is outside the validated domain of the model. Additionally, we can calculate $P_A(20)$ and check if the result is realistic for a share value (which must be non-negative).

Working

Part (a)
At the point monitoring began, $t=0$.
For company A:
$$P_A(0) = 53 - 0.4(0 - 8)^2 = 53 - 0.4(-8)^2 = 53 - 0.4(64) = 53 - 25.6 = 27.4$$
For company B:
$$P_B(0) = -1.6(0) + 44.2 = 44.2$$

Difference in share value:
$$P_B(0) - P_A(0) = 44.2 - 27.4 = 16.8$$

Answer: The difference is $\pounds 16.8$ million.

Part (b)
The share value of company A is $P_A = 53 - 0.4(t-8)^2$.
This is a quadratic function with a negative coefficient for the squared term, so it has a maximum value. The maximum occurs when $(t-8)^2 = 0$, which is when $t=8$.
At $t=8$, $P_A = 53 - 0.4(8-8)^2 = 53 - 0.4(0) = 53$.
Since $t=8$ is within the 15-year monitoring period ($0 \le t \le 15$), this maximum is valid.

Answer: The maximum share value of company A is $\pounds 53$ million.

Part (c)
We need to find when $P_A > P_B$:
$$53 - 0.4(t-8)^2 > -1.6t + 44.2$$
First, expand $(t-8)^2 = t^2 - 16t + 64$:
$$53 - 0.4(t^2 - 16t + 64) > -1.6t + 44.2$$
$$53 - 0.4t^2 + 6.4t - 25.6 > -1.6t + 44.2$$
Rearrange to form a quadratic inequality by moving all terms to one side:
$$-0.4t^2 + 6.4t + 1.6t + 27.4 - 44.2 > 0$$
$$-0.4t^2 + 8t - 16.8 > 0$$
To work with a positive leading coefficient, multiply by $-1$ and reverse the inequality sign:
$$0.4t^2 - 8t + 16.8 < 0$$
To simplify, divide by $0.4$ (which is equivalent to multiplying by $2.5$):
$$t^2 - 20t + 42 < 0$$
Now, find the roots of the quadratic equation $t^2 - 20t + 42 = 0$ using the quadratic formula $t = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$:
$$t = \frac{-(-20) \pm \sqrt{(-20)^2 - 4(1)(42)}}{2(1)}$$
$$t = \frac{20 \pm \sqrt{400 - 168}}{2}$$
$$t = \frac{20 \pm \sqrt{232}}{2}$$
$$t = \frac{20 \pm 2\sqrt{58}}{2}$$
$$t = 10 \pm \sqrt{58}$$
Approximate values for the roots:
$t_1 = 10 - \sqrt{58} \approx 10 - 7.6157 \approx 2.3843$
$t_2 = 10 + \sqrt{58} \approx 10 + 7.6157 \approx 17.6157$

The parabola $y = t^2 - 20t + 42$ opens upwards. Therefore, $t^2 - 20t + 42 < 0$ when $t$ is between its roots:
$$10 - \sqrt{58} < t < 10 + \sqrt{58}$$
$$2.3843 < t < 17.6157$$
The problem states that the monitoring was over a 15-year period, so we must also satisfy $0 \le t \le 15$.
Combining these conditions:
$$(2.3843 < t < 17.6157) \cap (0 \le t \le 15)$$
This gives the interval:
$$2.3843 < t \le 15$$

Answer: The share value of company A was greater than company B when $10 - \sqrt{58}
< t \le 15$ (or $2.38 < t \le 15$, to 3 significant figures).

Part (d)
The model for company A should not be used to predict its share value when $t=20$ for two reasons:


  1. The model was developed and stated to be valid for a 15-year period ($t \le 15$). Predicting for $t=20$ is extrapolating outside this established domain.

  2. Substituting $t=20$ into the model yields:


$P_A(20) = 53 - 0.4(20-8)^2 = 53 - 0.4(12)^2 = 53 - 0.4(144) = 53 - 57.6 = -4.6$.
A share value cannot be negative, indicating that the model is no longer realistic at $t=20$.

Answer: The model is stated to be for a 15-year period, and $t=20$ is beyond this range. Furthermore, $P_A(20) = -4.6$, meaning the predicted share value would be negative, which is not possible.

Key techniques

  • Evaluate functions at a given point
  • Identify maximum/minimum from vertex form
  • Solve algebraic inequalities (quadratic)
  • Interpret practical limitations of mathematical models

Common pitfalls

  • Forgetting to consider the upper bound of the model's domain ($t \le 15$) when providing the final answer for inequalities.
  • Algebraic errors when expanding the quadratic or solving the quadratic inequality, especially when multiplying or dividing by negative numbers.
  • Misinterpreting the maximum/minimum point of the quadratic function (e.g., confusing it with minimum because of the negative leading coefficient without understanding the vertex form).
