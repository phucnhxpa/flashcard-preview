Worked solution

Part (a) State the equation of the asymptote to the curve with equation $y = f(-x)$.

  • Thought Process: We need to identify the effect of the transformation $y = f(-x)$ on the asymptote. The original curve has a horizontal asymptote $y = 4$. The transformation $y = f(-x)$ represents a reflection of the graph in the $y$-axis. A horizontal line, such as $y=4$, is symmetrical with respect to the $y$-axis (it doesn't change its position when reflected). Therefore, the equation of the asymptote will remain the same.
  • Working:
The original asymptote is $y = 4$. A reflection in the $y$-axis maps $(x, y)$ to $(-x, y)$. Since the asymptote is a horizontal line, its $y$-coordinate is constant regardless of the $x$-coordinate, and its equation does not depend on $x$. Hence, reflection in the $y$-axis does not affect horizontal asymptotes.
Answer: $y = 4$

Part (b) State the coordinates of the turning point on the curve with equation $y = f(\frac{1}{4}x)$.

  • Thought Process: We need to find the new coordinates of the turning point after the transformation $y = f(\frac{1}{4}x)$. The original turning point is a maximum at $(4, 9)$. A transformation of the form $y = f(ax)$ is a horizontal stretch or compression by a factor of $\frac{1}{a}$. In this case, $a = \frac{1}{4}$, so the stretch factor is $\frac{1}{1/4} = 4$. This means the $x$-coordinate of any point on the curve will be multiplied by $4$, while the $y$-coordinate remains unchanged.
  • Working:
Original turning point: $(4, 9)$ Transformation: $y = f(\frac{1}{4}x)$ This is a horizontal stretch by a factor of $4$ from the $y$-axis. New $x$-coordinate: $4 \times 4 = 16$ New $y$-coordinate: $9$ (unchanged) The new turning point is $(16, 9)$.
Answer: $(16, 9)$

Part (c) Given that the line with equation $y = k$, where $k$ is a constant, intersects C at exactly one point, state the possible values for $k$.

  • Thought Process: We are looking for values of the constant $k$ such that the horizontal line $y=k$ intersects the curve C at exactly one point. We need to examine the key features of the graph: the maximum turning point and the horizontal asymptote.
From the graph, the maximum point is $(4, 9)$. If $y = 9$, the line is tangent to the curve at the maximum, resulting in exactly one intersection point. The curve also has a horizontal asymptote $y=4$. An asymptote is a line that the curve approaches as $x$ tends to infinity (or negative infinity). For a horizontal line $y=k$ to intersect the curve at exactly one point, $k$ must typically correspond to either a stationary point or an asymptote that the curve only approaches without crossing multiple times. Looking at the graph, as $x \to \infty$, the curve approaches $y=4$ from above, and as $x \to -\infty$, it approaches $y=4$ from below. The specific behaviour for $y=4$ to be a single intersection point is that the curve does not cross $y=4$ more than once, and the approach towards it at infinity counts as a 'single point of contact'.
  • Working:
The maximum turning point of the curve C is $(4, 9)$. A horizontal line $y = k$ passing through this point, i.e., $y = 9$, will intersect the curve at exactly one point (the maximum). The curve C has a horizontal asymptote $y = 4$. A horizontal line $y = k$ that coincides with the asymptote, i.e., $y = 4$, will also intersect the curve at exactly one point (as the curve approaches the asymptote at infinity).
Answer: $k = 4, k = 9$

Part (d) The curve C is transformed to a new curve that passes through the origin.

(i) Given that the new curve has equation $y = f(x) - a$, state the value of the constant $a$.

  • Thought Process: The transformation $y = f(x) - a$ represents a vertical translation. We are given that the new curve passes through the origin $(0, 0)$. The original curve crosses the $y$-axis at $(0, 6)$, meaning $f(0) = 6$. If the new curve passes through $(0,0)$, then when $x=0$, $y=0$. Substituting these into the transformed equation: $0 = f(0) - a$. We can then use $f(0)=6$ to solve for $a$.
  • Working:
The transformation is $y = f(x) - a$. The new curve passes through the origin $(0, 0)$. Substitute $(0, 0)$ into the equation: $0 = f(0) - a$. From the given information and graph, the original curve crosses the $y$-axis at $(0, 6)$, so $f(0) = 6$. Therefore, $0 = 6 - a$. Solving for $a$, we get $a = 6$.
Answer: $a = 6$

(ii) Write down an equation for another single transformation of C that also passes through the origin.

  • Thought Process: We need to find *another* single transformation of C that also results in the curve passing through the origin $(0,0)$. The original curve has an $x$-intercept at $(-3, 0)$. If we translate the curve horizontally so that this $x$-intercept moves to the origin, the new curve will pass through $(0,0)$. To move a point with $x$-coordinate $-3$ to an $x$-coordinate of $0$, we need to shift the graph $3$ units to the right. A translation of $3$ units to the right corresponds to the transformation $y = f(x - 3)$.
  • Working:
The original curve has an $x$-intercept at $(-3, 0)$. To make the new curve pass through the origin $(0, 0)$, we can translate the graph horizontally such that the point $(-3, 0)$ moves to $(0, 0)$. This requires a translation of $3$ units in the positive $x$-direction. The transformation for a translation of $h$ units to the right is $y = f(x - h)$. So, for $h = 3$, the equation is $y = f(x - 3)$. To verify, if $x=0$, $y = f(0-3) = f(-3)$. From the original graph, $f(-3)=0$, so $(0,0)$ is on the new curve.
Answer: $y = f(x - 3)$

Key techniques

  • identify graph transformations
  • apply transformations to coordinates
  • interpret horizontal asymptotes
  • analyse intersections of y=k with a curve

Common pitfalls

  • Confusing the effect of horizontal stretch/compression factors, e.g., applying a factor of 1/4 instead of 4 for $f(x/4)$.
  • Incorrectly identifying all values for $k$ in part (c), often missing the asymptote ($k=4$) or incorrectly including values for multiple intersections.
  • Sign errors in translations, e.g., using $f(x+3)$ for a rightward shift instead of $f(x-3)$.
