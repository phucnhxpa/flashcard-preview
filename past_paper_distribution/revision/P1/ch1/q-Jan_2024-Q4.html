Worked solution

Part (a): Show that the equation $2^{2x+2} - 33(2^x) + 8 = 0$ can be written in the form $4p^2 - 33p + 8 = 0$, where $p = 2^x$.

  • Thought Process: We recognize that the given equation contains terms involving $2^x$ and $2^{2x}$. The goal is to transform this equation into a quadratic form in terms of a new variable $p$, where $p = 2^x$. This requires applying the rules of indices, specifically $a^{m+n} = a^m a^n$ and $a^{mn} = (a^m)^n$, to express all terms as functions of $2^x$ before making the substitution.
  • Working:
The given equation is: $$2^{2x+2} - 33(2^x) + 8 = 0$$ First, we apply the index law $a^{m+n} = a^m a^n$ to the first term, $2^{2x+2}$: $$2^{2x} \cdot 2^2 - 33(2^x) + 8 = 0$$ Next, we simplify $2^2$ to $4$ and rewrite $2^{2x}$ using the index law $a^{mn} = (a^m)^n$, which means $2^{2x} = (2^x)^2$: $$4 \cdot (2^x)^2 - 33(2^x) + 8 = 0$$ Now, we introduce the substitution $p = 2^x$. We replace every instance of $2^x$ with $p$: $$4p^2 - 33p + 8 = 0$$ This is the required form, as stated in the question.
Answer: $4p^2 - 33p + 8 = 0$

Part (b): Hence, solve the equation for $x$, giving your answers to 3 significant figures where appropriate.

  • Thought Process: From part (a), we have a quadratic equation in $p$. Our first step is to solve this quadratic equation for $p$. As this is typically a non-calculator problem, factorization is the most common method. Once we find the values of $p$, we must remember that $p = 2^x$ and substitute back to find the corresponding values of $x$. It's crucial to ensure that any values of $p$ are positive, as $2^x$ is always positive for real $x$.
  • Working:
We start with the quadratic equation derived in part (a): $$4p^2 - 33p + 8 = 0$$ To solve this by factorization, we look for two numbers that multiply to $4 imes 8 = 32$ and add up to $-33$. These numbers are $-1$ and $-32$. We rewrite the middle term, $-33p$, using these numbers: $$4p^2 - p - 32p + 8 = 0$$ Now, we factor by grouping: $$p(4p - 1) - 8(4p - 1) = 0$$ Factor out the common term $(4p - 1)$: $$(4p - 1)(p - 8) = 0$$ This gives us two possible values for $p$: $$4p - 1 = 0 \implies 4p = 1 \implies p = rac{1}{4}$$ $$p - 8 = 0 \implies p = 8$$

Next, we substitute back $p = 2^x$ for each of these values of $p$ to find $x$:

Case 1: $p = rac{1}{4}$
$$2^x = rac{1}{4}$$
Since $ rac{1}{4}$ can be written as $2^{-2}$:
$$2^x = 2^{-2}$$
Equating the exponents, we get:
$$x = -2$$

Case 2: $p = 8$
$$2^x = 8$$
Since $8$ can be written as $2^3$:
$$2^x = 2^3$$
Equating the exponents, we get:
$$x = 3$$
Both values of $p$ are positive, so both values of $x$ are valid solutions.

Answer: $x = -2, 3$

Key techniques

  • apply index laws for powers of same base
  • recognize substitution $p = a^x$ to form a quadratic
  • factorize quadratic equations (non-monic)
  • solve $a^x = k$ using logarithms or by matching bases

Common pitfalls

  • Incorrect application of index laws (e.g., $2^{2x+2} \ne 2^{2x} + 2^2$ or $2^{2x+2} \ne (2^{2x})(2^x)$).
  • Algebraic errors when factorizing the quadratic equation, especially non-monic ones.
  • Forgetting to substitute back $p = 2^x$ and solve for $x$, or making errors in solving $2^x = k$ (e.g. if $k$ is not a simple power of 2, requiring logarithms).
  • Accepting a negative value for $p$ when $p=2^x$, as $2^x$ must always be positive.
