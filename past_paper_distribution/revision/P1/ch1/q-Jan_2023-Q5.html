Worked solution

Thought Process for Part (a):
We are given an exponential equation and asked to show that it can be transformed into a specific quadratic equation in terms of $p$, where $p=3^x$. This is a common strategy for solving exponential equations where multiple terms involve the same base raised to powers of $x$.

  1. Identify the substitution: The instruction to use $p=3^x$ is key. Our first step is to rewrite every term in the given exponential equation so that it can be expressed using $p$.
  2. Apply index laws: We need to recall and apply index laws such as $a^{m+n} = a^m a^n$, $a^{m-n} = a^m/a^n$, and $a^{mn} = (a^m)^n$. For example, $3^{2x}$ will become $(3^x)^2 = p^2$, and $3^{1-x}$ will become $3^1 \cdot 3^{-x} = 3/3^x = 3/p$.
  3. Substitute and simplify: Replace all $3^x$ terms with $p$. This will result in an equation involving $p$ and possibly fractions.
  4. Clear denominators: To obtain a standard quadratic equation (where all terms are integers or simple fractions), we must eliminate any denominators. If there's a $1/p$ term, we multiply the entire equation by $p$.
  5. Rearrange: Finally, move all terms to one side of the equation to match the target quadratic form $9p^2 + 26p - 3 = 0$.

Working for Part (a):
Let's assume the question to be solved is $9 \cdot 3^x + 26 = 3^{1-x}$, which leads to the target quadratic $9p^2 + 26p - 3 = 0$.

The given equation is:
$$9 \cdot 3^x + 26 = 3^{1-x}$$

Step 1: Apply index laws to rewrite each term in terms of $3^x$.
We are given the substitution $p = 3^x$. Let's express each exponential term using $p$:


  • The term $9 \cdot 3^x$ directly becomes $9p$.

  • The term $3^{1-x}$ can be rewritten using the index law $a^{m-n} = a^m / a^n$ (or $a^m a^{-n}$):


$$3^{1-x} = 3^1 imes 3^{-x} = rac{3}{3^x}$$

Step 2: Substitute $p = 3^x$ into the equation.
Substitute the rewritten terms into the original equation:
$$9p + 26 = rac{3}{p}$$

Step 3: Clear the denominator and rearrange into the desired quadratic form.
To eliminate the fraction $ rac{3}{p}$, multiply every term in the entire equation by $p$. This ensures we do not change the equality:
$$p(9p) + p(26) = p\left( rac{3}{p} ight)$$
This simplifies to:
$$9p^2 + 26p = 3$$
Now, move the constant term from the right-hand side to the left-hand side to form a quadratic equation equal to zero. Remember to change its sign when moving:
$$9p^2 + 26p - 3 = 0$$
This matches the target quadratic equation we needed to show.

Answer (a): $9p^2 + 26p - 3 = 0$

---

Thought Process for Part (b):
Part (b) asks us to solve the original exponential equation, which means finding the value(s) of $x$. Since we've already transformed it into a quadratic in $p$, the strategy is:

  1. Solve the quadratic for $p$: Use a reliable method like factorisation, the quadratic formula, or completing the square. Factorisation is often the most efficient if the factors are easy to find.
  2. Convert back to $x$: For each valid value of $p$, substitute it back into $p = 3^x$ and solve for $x$.
  3. Check for extraneous solutions: Remember that $p = 3^x$ implies that $p$ must be a positive value (since a positive base raised to any real power is always positive). Any negative values of $p$ obtained from solving the quadratic must be rejected as they will not yield real solutions for $x$.

Working for Part (b):
We need to solve the quadratic equation derived in part (a):
$$9p^2 + 26p - 3 = 0$$

Step 1: Factorise the quadratic equation.
To factorise $9p^2 + 26p - 3 = 0$, we look for two numbers that multiply to $9 imes (-3) = -27$ and add up to the middle coefficient, $26$. These numbers are $27$ and $-1$.
Rewrite the middle term $26p$ using these two numbers:
$$9p^2 + 27p - p - 3 = 0$$
Now, factor by grouping the terms:
$$9p(p + 3) - 1(p + 3) = 0$$
Notice that $(p+3)$ is a common factor. Factor it out:
$$(9p - 1)(p + 3) = 0$$

Step 2: Solve for the possible values of $p$.
Set each factor equal to zero to find the possible values for $p$:


  • For the first factor:


$$9p - 1 = 0$$
$$9p = 1$$
$$p = rac{1}{9}$$

  • For the second factor:


$$p + 3 = 0$$
$$p = -3$$

Step 3: Convert back to $x$ using $p = 3^x$ and check for valid solutions.
We have two possible values for $p$. Now, we substitute them back into $p = 3^x$:

  • Case 1: $p = rac{1}{9}$
$$3^x = rac{1}{9}$$ We know that $9 = 3^2$, so $ rac{1}{9} = rac{1}{3^2} = 3^{-2}$. $$3^x = 3^{-2}$$ By equating the exponents (since the bases are the same): $$x = -2$$
  • Case 2: $p = -3$
$$3^x = -3$$ An exponential function with a positive base (like $3^x$) always produces a positive result for any real value of $x$. Therefore, $3^x = -3$ has no real solutions for $x$. This value of $p$ is extraneous and must be rejected.

Thus, the only valid real solution for $x$ is $-2$.

Answer (b): $x = -2$

Key techniques

  • apply index laws to simplify terms
  • substitute variable to form quadratic
  • clear denominators in algebraic expressions
  • solve quadratic by factorisation
  • solve $a^x=b$ using logarithms or powers

Common pitfalls

  • Incorrect application of index laws (e.g., $3^{x+2} eq 3^x + 3^2$, or $3^{1-x} eq 3^1 - 3^x$).
  • Algebraic errors when multiplying to clear denominators or rearranging terms.
  • Forgetting that $p=3^x$ must be positive, leading to the acceptance of extraneous solutions for $p$ that do not yield real $x$ values.
