Worked solution

Part (a): Express $y = \frac{1}{2}x^2 - 10x + 22$ in the form $a(x-b)^2+c$.

Thought Process: We need to rewrite the quadratic function into the completed square form. The first step is to factor out the coefficient of $x^2$ from the terms involving $x$, then complete the square for the expression inside the bracket.

Working:
Start with the given equation:
$$y = \frac{1}{2}x^2 - 10x + 22$$
Factor out $\frac{1}{2}$ from the $x^2$ and $x$ terms:
$$y = \frac{1}{2}(x^2 - 20x) + 22$$
Now, complete the square for $x^2 - 20x$. We take half of the coefficient of $x$ (which is $-20/2 = -10$) and square it. So $x^2 - 20x = (x-10)^2 - (-10)^2$:
$$y = \frac{1}{2}((x-10)^2 - 10^2) + 22$$
$$y = \frac{1}{2}((x-10)^2 - 100) + 22$$
Distribute the $\frac{1}{2}$:
$$y = \frac{1}{2}(x-10)^2 - \frac{1}{2}(100) + 22$$
$$y = \frac{1}{2}(x-10)^2 - 50 + 22$$
Combine the constant terms:
$$y = \frac{1}{2}(x-10)^2 - 28$$

Answer: $y = \frac{1}{2}(x-10)^2 - 28$

Part (b): State the coordinates of the minimum point of the curve.

Thought Process: The completed square form $y = a(x-b)^2+c$ directly gives the coordinates of the vertex (minimum or maximum point) as $(b,c)$. Since $a = \frac{1}{2}$ is positive, this parabola opens upwards, and the vertex is a minimum point.

Working:
From part (a), we have $y = \frac{1}{2}(x-10)^2 - 28$.
The minimum value of $(x-10)^2$ is 0, which occurs when $x-10=0$, i.e., $x=10$.
When $x=10$, $y = \frac{1}{2}(10-10)^2 - 28 = \frac{1}{2}(0)^2 - 28 = -28$.

Answer: The minimum point is $(10, -28)$.

Part (c)(i): Find the coordinates of the point on the curve where the gradient of the tangent is 8.

Thought Process: The gradient of the tangent to a curve at any point $(x,y)$ is given by its derivative, $\frac{dy}{dx}$. We need to differentiate the original equation for $y$, set the derivative equal to 8, solve for $x$, and then substitute this $x$-value back into the original equation for $y$.

Working:
Given $y = \frac{1}{2}x^2 - 10x + 22$.
Differentiate $y$ with respect to $x$:
$$\frac{dy}{dx} = \frac{d}{dx}\left(\frac{1}{2}x^2 - 10x + 22\right)$$
$$\frac{dy}{dx} = 2 \cdot \frac{1}{2}x^{2-1} - 10x^{1-1} + 0$$
$$\frac{dy}{dx} = x - 10$$
We are given that the gradient of the tangent is 8. So, set $\frac{dy}{dx} = 8$:
$$x - 10 = 8$$
$$x = 18$$
Now, substitute $x=18$ back into the original equation for $y$ to find the $y$-coordinate:
$$y = \frac{1}{2}(18)^2 - 10(18) + 22$$
$$y = \frac{1}{2}(324) - 180 + 22$$
$$y = 162 - 180 + 22$$
$$y = -18 + 22$$
$$y = 4$$

Answer: The point is $(18, 4)$.

Part (c)(ii): A line $L$ passes through the point found in (c)(i) and is perpendicular to the tangent at that point. Find the value of $k$ if the equation of $L$ is $y = -\frac{1}{8}x + k$.

Thought Process: From (c)(i), the tangent at $(18,4)$ has a gradient of 8. If line $L$ is perpendicular to this tangent, its gradient $m_L$ must satisfy $m_L \cdot m_T = -1$. Once we have $m_L$, we can use the point $(18,4)$ and the equation $y = m_Lx + k$ to solve for $k$.

Working:
The gradient of the tangent at $(18,4)$ is $m_T = 8$.
The line $L$ is perpendicular to this tangent, so its gradient $m_L$ is:
$$m_L = -\frac{1}{m_T} = -\frac{1}{8}$$
The equation of line $L$ is given as $y = -\frac{1}{8}x + k$. We know $L$ passes through $(18,4)$. Substitute these coordinates into the equation:
$$4 = \left(-\frac{1}{8}\right)(18) + k$$
$$4 = -\frac{18}{8} + k$$
Simplify the fraction:
$$4 = -\frac{9}{4} + k$$
Solve for $k$:
$$k = 4 + \frac{9}{4}$$
Convert 4 to a fraction with denominator 4:
$$k = \frac{16}{4} + \frac{9}{4}$$
$$k = \frac{25}{4}$$

Answer: $k = \frac{25}{4}$

Part (d): State the three inequalities that define region $R$, bounded by the curve, the line $L$, and the line $x=10$.

Thought Process: We need to consider the boundaries provided: the curve, the line $L$, and the vertical line $x=10$. We'll determine the direction of the inequality for each boundary to enclose the region $R$.

  1. The curve: The region is *above* the parabola $y = \frac{1}{2}x^2 - 10x + 22$. So, $y \ge \frac{1}{2}x^2 - 10x + 22$.
  2. The line $L$: The region is *below* the line $L$, which is $y = -\frac{1}{8}x + \frac{25}{4}$. So, $y \le -\frac{1}{8}x + \frac{25}{4}$.
  3. The line $x=10$: From part (b), the minimum point of the curve is at $x=10$. The region is bounded by $x=10$ on the left, extending to the right. So, $x \ge 10$.

Working:
The three inequalities defining region $R$ are:


  1. $x \ge 10$

  2. $y \ge \frac{1}{2}x^2 - 10x + 22$

  3. $y \le -\frac{1}{8}x + \frac{25}{4}$


Answer: The region $R$ is defined by $x \ge 10$, $y \ge \frac{1}{2}x^2 - 10x + 22$, and $y \le -\frac{1}{8}x + \frac{25}{4}$.

Key techniques

  • Completing the square $ax^2+bx+c$ to $a(x-p)^2+q$
  • Finding stationary points from completed square form
  • Differentiation to find gradient function $dy/dx$
  • Using $m_1m_2=-1$ for perpendicular lines
  • Formulating inequalities for a bounded region

Common pitfalls

  • Errors in algebraic manipulation, especially when factoring out coefficients or dealing with negative signs during completing the square.
  • Incorrectly calculating the gradient of a perpendicular line (e.g., using $m_1m_2=1$ or just changing the sign).
  • Missing one of the boundary conditions for the region, or using strict inequalities where inclusive ones are required (or vice-versa, without consistency).
