Question 10 continued
Worked solution

Part (a): Sketching C1

Thought Process:
We are given the equation for curve $C_1$ in factorised form: $f(x) = (4x - 3)(x - 5)^2$. To sketch a curve, we need its intercepts with the axes and its general shape.


  1. x-intercepts (roots): These occur when $f(x) = 0$. Since the function is factorised, we set each factor to zero.

  2. y-intercept: This occurs when $x = 0$. We substitute $x=0$ into $f(x)$.

  3. Shape of the curve: The highest power of $x$ when expanded would be $x \times x^2 = x^3$, so it's a cubic function. The leading coefficient is positive (from $4x \times x^2$), so the graph will start from negative infinity on the left and go to positive infinity on the right (general 'N' shape). The term $(x - 5)^2$ indicates that the curve *touches* the x-axis at $x=5$, rather than crossing it. The term $(4x-3)$ indicates it *crosses* at $x=3/4$.

Working:


  • x-intercepts:


Set $f(x) = 0$:
$(4x - 3)(x - 5)^2 = 0$
Either $4x - 3 = 0 \implies 4x = 3 \implies x = \frac{3}{4}$
Or $(x - 5)^2 = 0 \implies x - 5 = 0 \implies x = 5$
So, the curve crosses the x-axis at $(\frac{3}{4}, 0)$ and touches the x-axis at $(5, 0)$.

  • y-intercept:
Set $x = 0$: $f(0) = (4(0) - 3)(0 - 5)^2$ $f(0) = (-3)(-5)^2$ $f(0) = (-3)(25)$ $f(0) = -75$ So, the curve crosses the y-axis at $(0, -75)$.
  • Sketch Description:
A cubic curve starting from bottom-left, crossing the x-axis at $(\frac{3}{4}, 0)$, going down to a local minimum, then turning back up to touch the x-axis at $(5, 0)$, and continuing upwards to the top-right. It crosses the y-axis at $(0, -75)$.

Part (b)(i): Find x for $f(\frac{x}{4}) = 0$

Thought Process:
We know that $f(X) = 0$ when $X = \frac{3}{4}$ or $X = 5$. In this case, the input to the function is $\frac{x}{4}$ instead of $x$. So we set $\frac{x}{4}$ equal to the known roots.

Working:
We have $f(\frac{x}{4}) = 0$.
From part (a), $f(X) = 0$ when $X = \frac{3}{4}$ or $X = 5$.
So, we set $\frac{x}{4}$ equal to these values:


  1. $\frac{x}{4} = \frac{3}{4} \implies x = 3$

  2. $\frac{x}{4} = 5 \implies x = 20$

Answer: $x = 3, 20$

Part (b)(ii): Find $p$ such that $y = f(x) + p$ passes through the origin

Thought Process:
If a curve passes through the origin, it means the point $(0,0)$ lies on the curve. We can substitute $x=0$ and $y=0$ into the equation $y = f(x) + p$ to find the value of $p$. This represents a vertical translation of the original curve.

Working:
The curve is $y = f(x) + p$. It passes through the origin $(0,0)$.
Substitute $x=0, y=0$:
$0 = f(0) + p$
From part (a), we know $f(0) = -75$.
$0 = -75 + p$
$p = 75$

Answer: $p = 75$

Part (c)(i): Find $g(x)$ in simplest form

Thought Process:
We are given $g(x) = f(x + 1)$. This means we need to replace every instance of $x$ in the expression for $f(x)$ with $(x + 1)$. This represents a horizontal translation of the curve $C_1$, 1 unit to the left.

Working:
$f(x) = (4x - 3)(x - 5)^2$
Substitute $(x + 1)$ for $x$:
$g(x) = (4(x + 1) - 3)((x + 1) - 5)^2$
Simplify the terms inside the parentheses:
$g(x) = (4x + 4 - 3)(x - 4)^2$
$g(x) = (4x + 1)(x - 4)^2$

Answer: $g(x) = (4x + 1)(x - 4)^2$

Part (c)(ii): Find the y-intercept of curve C2

Thought Process:
The y-intercept of any curve occurs when $x = 0$. We need to find $g(0)$ using the expression for $g(x)$ we just found.

Working:
Substitute $x = 0$ into $g(x) = (4x + 1)(x - 4)^2$:
$g(0) = (4(0) + 1)(0 - 4)^2$
$g(0) = (1)(-4)^2$
$g(0) = (1)(16)$
$g(0) = 16$

Answer: $16$

Key techniques

  • Identify roots and their multiplicity from factorised form
  • Determine y-intercept by setting x=0
  • Recognize general shape of cubic graphs
  • Apply horizontal stretch/compression transformations
  • Apply vertical and horizontal translation transformations
  • Substitute expressions into function definitions

Common pitfalls

  • Incorrectly identifying whether the curve 'touches' or 'crosses' the x-axis at roots (e.g. treating $(x-5)^2$ as a crossing point).
  • Making algebraic errors when substituting for transformations, especially with signs or distributing terms (e.g. $4(x+1)-3$ vs $4x+1-3$).
  • Confusing horizontal and vertical transformations, or the direction of translation for $f(x+a)$ (left vs right).
