Worked solution

Part (a)(i): Show that $a=3$

Thought Process:
We are given the function $f(x) = ax^3 + (6a + 8)x^2 - a^2x$ and the condition $f(-1) = 32$. The most direct approach is to substitute $x = -1$ into the expression for $f(x)$ and set the result equal to $32$. This will give us an equation in terms of $a$, which we can then solve. We must also remember the condition that $a$ is a positive constant.

Working:
Given $f(x) = ax^3 + (6a + 8)x^2 - a^2x$ and $f(-1) = 32$.

Substitute $x = -1$ into $f(x)$:
$f(-1) = a(-1)^3 + (6a + 8)(-1)^2 - a^2(-1)$
$f(-1) = a(-1) + (6a + 8)(1) + a^2$
$f(-1) = -a + 6a + 8 + a^2$
$f(-1) = a^2 + 5a + 8$

Now, set this equal to $32$:
$a^2 + 5a + 8 = 32$
$a^2 + 5a - 24 = 0$

This is a quadratic equation in $a$. We can solve it by factorization:
$(a + 8)(a - 3) = 0$

This gives two possible values for $a$: $a = -8$ or $a = 3$.

The question states that $a$ is a positive constant. Therefore, we must choose $a=3$.

Answer: $a=3$ (since $a$ is a positive constant)

Part (a)(ii): Solve $f(x) = 0$ using $a=3$

Thought Process:
Now that we know $a=3$, we can substitute this value back into the original function $f(x)$ to get a specific polynomial. Then we need to solve the equation $f(x) = 0$. Since it's a cubic equation, we should look for ways to simplify it. Often, a common factor can be extracted, or the Factor Theorem might be useful. In this case, notice that every term has an $x$.

Working:
Substitute $a=3$ into $f(x)$:
$f(x) = 3x^3 + (6(3) + 8)x^2 - (3^2)x$
$f(x) = 3x^3 + (18 + 8)x^2 - 9x$
$f(x) = 3x^3 + 26x^2 - 9x$

Now, solve $f(x) = 0$:
$3x^3 + 26x^2 - 9x = 0$

Notice that $x$ is a common factor in all terms. Factor out $x$:
$x(3x^2 + 26x - 9) = 0$

This immediately gives one solution: $x=0$. The remaining solutions come from the quadratic factor:
$3x^2 + 26x - 9 = 0$

We can factorize this quadratic. We look for two numbers that multiply to $3 imes -9 = -27$ and add to $26$. These numbers are $27$ and $-1$. So we split the middle term:
$3x^2 + 27x - x - 9 = 0$
$3x(x + 9) - 1(x + 9) = 0$
$(3x - 1)(x + 9) = 0$

This gives two more solutions:
$3x - 1 = 0 \Rightarrow 3x = 1 \Rightarrow x = rac{1}{3}$
$x + 9 = 0 \Rightarrow x = -9$

So, the solutions to $f(x) = 0$ are $x = 0, x = rac{1}{3}, x = -9$.

Answer: $x = 0, x = rac{1}{3}, x = -9$

Part (b)(i): Find all real solutions of $3y + 26y^{ rac{2}{3}} - 9y^{ rac{1}{3}} = 0$

Thought Process:
This equation looks very similar in structure to $f(x) = 0$ where $f(x) = 3x^3 + 26x^2 - 9x$. We can identify a substitution that transforms this equation into the form of $f(x)=0$. The powers are $1, rac{2}{3}, rac{1}{3}$. If we let $x = y^{ rac{1}{3}}$, then $x^2 = (y^{ rac{1}{3}})^2 = y^{ rac{2}{3}}$ and $x^3 = (y^{ rac{1}{3}})^3 = y$. This substitution will convert the given equation into $3x^3 + 26x^2 - 9x = 0$, which we have already solved in part (a)(ii).

Working:
Let $x = y^{ rac{1}{3}}$. Then $x^2 = y^{ rac{2}{3}}$ and $x^3 = y$.

Substitute these into the given equation:
$3(y) + 26(y^{ rac{2}{3}}) - 9(y^{ rac{1}{3}}) = 0$
becomes
$3x^3 + 26x^2 - 9x = 0$

From part (a)(ii), we know the solutions for $x$ are:
$x = 0, x = rac{1}{3}, x = -9$

Now, substitute back $x = y^{ rac{1}{3}}$ to find $y$:

Case 1: $y^{ rac{1}{3}} = 0$
Cubing both sides: $y = 0^3 \Rightarrow y = 0$

Case 2: $y^{ rac{1}{3}} = rac{1}{3}$
Cubing both sides: $y = \left( rac{1}{3} ight)^3 \Rightarrow y = rac{1}{27}$

Case 3: $y^{ rac{1}{3}} = -9$
Cubing both sides: $y = (-9)^3 \Rightarrow y = -729$

All three solutions are real.

Answer: $y = 0, y = rac{1}{27}, y = -729$

Part (b)(ii): Find all real solutions of $3(9^{3z}) + 26(9^{2z}) - 9(9^z) = 0$

Thought Process:
Again, this equation has a similar structure to $f(x) = 0$. The terms involve $9^z$, $9^{2z}$, and $9^{3z}$. We can make a substitution $x = 9^z$. Then $x^2 = (9^z)^2 = 9^{2z}$ and $x^3 = (9^z)^3 = 9^{3z}$. This will transform the equation into $3x^3 + 26x^2 - 9x = 0$, which we already solved. After finding the values of $x$, we need to convert back to $z$. Remember that $9^z$ must always be positive, so any non-positive values of $x$ will not yield real solutions for $z$.

Working:
Let $x = 9^z$. Then $x^2 = 9^{2z}$ and $x^3 = 9^{3z}$.

Substitute these into the given equation:
$3(9^{3z}) + 26(9^{2z}) - 9(9^z) = 0$
becomes
$3x^3 + 26x^2 - 9x = 0$

From part (a)(ii), we know the solutions for $x$ are:
$x = 0, x = rac{1}{3}, x = -9$

Now, substitute back $x = 9^z$ to find $z$:

Case 1: $9^z = 0$
This has no real solution for $z$, as $9^z$ is always positive for any real $z$.

Case 2: $9^z = rac{1}{3}$
We need to express both sides with the same base. Since $9 = 3^2$ and $ rac{1}{3} = 3^{-1}$:
$(3^2)^z = 3^{-1}$
$3^{2z} = 3^{-1}$
Equating the exponents: $2z = -1 \Rightarrow z = - rac{1}{2}$

Case 3: $9^z = -9$
This has no real solution for $z$, as $9^z$ is always positive for any real $z$.

So, there is only one real solution for $z$.

Answer: $z = - rac{1}{2}$

Key techniques

  • substituting values into functions
  • factorizing quadratic expressions
  • factorizing polynomials by common factor
  • identifying and applying appropriate substitutions
  • solving exponential equations

Common pitfalls

  • Forgetting to apply the condition 'a is a positive constant' when solving for 'a'.
  • Algebraic errors when substituting and simplifying, especially with negative numbers or fractions.
  • Not recognizing the common factor 'x' in part (a)(ii), leading to a more complex cubic to solve.
  • Failing to make the correct substitution in parts (b)(i) and (b)(ii), or making errors when converting back to the original variable.
  • Forgetting that $A^z$ must be positive for real $z$ (e.g., $9^z$ cannot be 0 or negative), leading to incorrect solutions in part (b)(ii).
