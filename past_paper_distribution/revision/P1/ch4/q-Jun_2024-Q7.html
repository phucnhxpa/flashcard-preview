Worked solution

Let the given function be $f(x) = x^3 - kx^2 + 14x + 24$. This function is implied by the first derivative given in the mark scheme.

Part (a) Find $f'(x)$ and $f''(x)$

Thought Process: We need to find the first and second derivatives of the given polynomial function. This is a direct application of the power rule for differentiation, $\frac{d}{dx}(ax^n) = nax^{n-1}$, and remembering that the derivative of a constant term is zero. We treat $k$ as a constant multiplier.

Working:

  • Step 1: Find the first derivative, $f'(x)$.
Applying the power rule to each term in $f(x)$: $$f(x) = x^3 - kx^2 + 14x + 24$$ $$f'(x) = \frac{d}{dx}(x^3) - \frac{d}{dx}(kx^2) + \frac{d}{dx}(14x) + \frac{d}{dx}(24)$$ $$f'(x) = 3x^{3-1} - k(2x^{2-1}) + 14(1x^{1-1}) + 0$$ $$f'(x) = 3x^2 - 2kx + 14$$
  • Step 2: Find the second derivative, $f''(x)$.
Now differentiate $f'(x)$ using the same rules: $$f'(x) = 3x^2 - 2kx + 14$$ $$f''(x) = \frac{d}{dx}(3x^2) - \frac{d}{dx}(2kx) + \frac{d}{dx}(14)$$ $$f''(x) = 3(2x^{2-1}) - 2k(1x^{1-1}) + 0$$ $$f''(x) = 6x - 2k$$

Part (b) Given $f'(5) = f''(5)$, find the value of $k$

Thought Process: The problem states a condition that the first and second derivatives are equal at a specific point, $x=5$. We need to substitute $x=5$ into both derivative expressions found in part (a), equate the results, and then solve the resulting linear equation for $k$.

Working:

  • Step 1: Calculate $f'(5)$.
Substitute $x=5$ into $f'(x) = 3x^2 - 2kx + 14$: $$f'(5) = 3(5)^2 - 2k(5) + 14$$ $$f'(5) = 3(25) - 10k + 14$$ $$f'(5) = 75 - 10k + 14$$ $$f'(5) = 89 - 10k$$
  • Step 2: Calculate $f''(5)$.
Substitute $x=5$ into $f''(x) = 6x - 2k$: $$f''(5) = 6(5) - 2k$$ $$f''(5) = 30 - 2k$$
  • Step 3: Equate $f'(5)$ and $f''(5)$ and solve for $k$.
$$89 - 10k = 30 - 2k$$ Add $10k$ to both sides: $$89 = 30 + 8k$$ Subtract $30$ from both sides: $$59 = 8k$$ $$k = \frac{59}{8}$$
Answer: $k = \frac{59}{8}$

Part (c) Find the coordinates of the other point where $f'(x) = f''(x)$

Thought Process: We found one point where $f'(x) = f''(x)$ is $x=5$. Now we need to find if there are other points. We will set the general expressions for $f'(x)$ and $f''(x)$ equal to each other, substitute the value of $k$ found in part (b), and solve the resulting equation for $x$. This will result in a quadratic equation, which typically has two solutions. One solution will be $x=5$; the other will be the $x$-coordinate of the required point. Then, substitute this new $x$-value into either $f'(x)$ or $f''(x)$ to find the corresponding 'y'-coordinate.

Working:

  • Step 1: Set $f'(x) = f''(x)$ using the value of $k = \frac{59}{8}$.
$$3x^2 - 2kx + 14 = 6x - 2k$$ Substitute $k = \frac{59}{8}$: $$3x^2 - 2\left(\frac{59}{8}\right)x + 14 = 6x - 2\left(\frac{59}{8}\right)$$ $$3x^2 - \frac{59}{4}x + 14 = 6x - \frac{59}{4}$$
  • Step 2: Rearrange the equation into a standard quadratic form $ax^2 + bx + c = 0$.
To eliminate fractions, multiply the entire equation by 4: $$4(3x^2) - 4\left(\frac{59}{4}x\right) + 4(14) = 4(6x) - 4\left(\frac{59}{4}\right)$$ $$12x^2 - 59x + 56 = 24x - 59$$ Move all terms to the left side: $$12x^2 - 59x - 24x + 56 + 59 = 0$$ $$12x^2 - 83x + 115 = 0$$
  • Step 3: Solve the quadratic equation for $x$.
We know $x=5$ is one solution, so $(x-5)$ is a factor. We can use factorisation or the quadratic formula $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$: $$x = \frac{-(-83) \pm \sqrt{(-83)^2 - 4(12)(115)}}{2(12)}$$ $$x = \frac{83 \pm \sqrt{6889 - 5520}}{24}$$ $$x = \frac{83 \pm \sqrt{1369}}{24}$$ $$x = \frac{83 \pm 37}{24}$$ This gives two solutions for $x$: $$x_1 = \frac{83 + 37}{24} = \frac{120}{24} = 5$$ $$x_2 = \frac{83 - 37}{24} = \frac{46}{24} = \frac{23}{12}$$ The 'other' point has an $x$-coordinate of $x = \frac{23}{12}$.
  • Step 4: Find the corresponding $y$-coordinate.
Substitute $x = \frac{23}{12}$ and $k = \frac{59}{8}$ into either $f'(x)$ or $f''(x)$ (since they are equal at this point). Using $f''(x) = 6x - 2k$: $$y = 6\left(\frac{23}{12}\right) - 2\left(\frac{59}{8}\right)$$ $$y = \frac{23}{2} - \frac{59}{4}$$ To subtract these, find a common denominator (4): $$y = \frac{2 \times 23}{2 \times 2} - \frac{59}{4}$$ $$y = \frac{46}{4} - \frac{59}{4}$$ $$y = -\frac{13}{4}$$
Answer: The coordinates of the other point are $\left(\frac{23}{12}, -\frac{13}{4}\right)$.

Key techniques

  • Power rule for differentiation
  • Solving linear equations
  • Solving quadratic equations by factorisation or formula
  • Substitution into algebraic expressions

Common pitfalls

  • Algebraic errors when substituting values or solving equations, especially with fractions or negative signs.
  • Incorrectly differentiating terms involving the constant 'k' (e.g., treating it as a variable or omitting it).
  • Forgetting to find *both* solutions to the quadratic equation, or selecting $x=5$ as the 'other' solution in part (c).
