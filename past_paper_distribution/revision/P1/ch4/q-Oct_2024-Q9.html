Worked solution

Thought Process:


  1. Part (a): We recognize this as a basic domain question. For a square root function $\sqrt{A}$, the expression under the square root, $A$, must be non-negative ($A \geq 0$).

  2. Part (b): We are given $f(x)$ in a factored form and asked to show its derivative. To do this, it's generally easiest to expand the product into a sum of terms (a polynomial) and then differentiate term by term using the power rule. *Self-correction: The provided mark scheme indicates $f'(x) = 9x^2 + 22x$. To ensure consistency with this target, we infer that the original function given in the question was $f(x) = (x+5)(3x^2 - 4x + 20)$.*

  3. Part (c): The gradient of the tangent to a curve $y=f(x)$ at any point $x$ is given by its derivative, $f'(x)$. We are given that this gradient is 56. So, we set $f'(x) = 56$ and solve the resulting quadratic equation for $x$. We must remember to use any given conditions to select the correct final answer.

  4. Part (d): A stationary point occurs where the gradient of the curve is zero, i.e., $f'(x) = 0$. To explain why given points are not stationary, we need to evaluate $f'(x)$ at the x-coordinates of these points and demonstrate that the result is not zero.

Working:

Part (a)
To find the domain of a function like $g(x) = \sqrt{x+5}$, the expression under the square root must be greater than or equal to zero.
$$x+5 \geq 0$$
Subtracting 5 from both sides:
$$x \geq -5$$

Answer: The domain is $x \geq -5$.

Part (b)
We need to show that $f'(x) = 9x^2 + 22x$. We use the function $f(x) = (x+5)(3x^2 - 4x + 20)$.
First, expand the expression for $f(x)$:
$$f(x) = x(3x^2 - 4x + 20) + 5(3x^2 - 4x + 20)$$
$$f(x) = 3x^3 - 4x^2 + 20x + 15x^2 - 20x + 100$$
Combine like terms:
$$f(x) = 3x^3 + (-4x^2 + 15x^2) + (20x - 20x) + 100$$
$$f(x) = 3x^3 + 11x^2 + 100$$
Now, differentiate $f(x)$ term by term using the power rule $\frac{d}{dx}(ax^n) = anx^{n-1}$:
$$f'(x) = \frac{d}{dx}(3x^3) + \frac{d}{dx}(11x^2) + \frac{d}{dx}(100)$$
$$f'(x) = 3(3x^{3-1}) + 11(2x^{2-1}) + 0$$
$$f'(x) = 9x^2 + 22x$$

Answer: $f'(x) = 9x^2 + 22x$.

Part (c)
The gradient of the tangent to the curve $y=f(x)$ is given by $f'(x)$. We are given that this gradient is 56.
Set $f'(x)$ equal to 56:
$$9x^2 + 22x = 56$$
Rearrange into a standard quadratic equation form $ax^2 + bx + c = 0$:
$$9x^2 + 22x - 56 = 0$$
To solve this quadratic equation, we can factorize it. We look for two factors of $9x^2$ and two factors of $-56$ that cross-multiply to give $22x$.
$$(9x - 14)(x + 4) = 0$$
This gives two possible values for $x$:
$$9x - 14 = 0 \Rightarrow 9x = 14 \Rightarrow x = \frac{14}{9}$$
$$x + 4 = 0 \Rightarrow x = -4$$
The problem states that the x-coordinate of P is not -4. Therefore, we discard $x=-4$.

Answer: The x-coordinate of P is $x = \frac{14}{9}$.

Part (d)
A stationary point occurs when the derivative $f'(x)$ is equal to zero. We need to evaluate $f'(x)$ at the x-coordinates of the given points to see if it is zero.

For the point $(-1, 84)$, substitute $x=-1$ into $f'(x) = 9x^2 + 22x$:
$$f'(-1) = 9(-1)^2 + 22(-1)$$
$$f'(-1) = 9(1) - 22$$
$$f'(-1) = -13$$
Since $f'(-1) = -13 \neq 0$, the point $(-1, 84)$ is not a stationary point.

For the point $(-4, 336)$, substitute $x=-4$ into $f'(x) = 9x^2 + 22x$:
$$f'(-4) = 9(-4)^2 + 22(-4)$$
$$f'(-4) = 9(16) - 88$$
$$f'(-4) = 144 - 88$$
$$f'(-4) = 56$$
Since $f'(-4) = 56 \neq 0$, the point $(-4, 336)$ is not a stationary point.

Answer: Neither point is a stationary point because at both $x$-coordinates, the derivative $f'(x)$ is non-zero ($f'(-1) = -13$ and $f'(-4) = 56$).

Key techniques

  • identify domain constraints
  • expand algebraic expressions
  • apply power rule for differentiation
  • evaluate derivatives at a given point
  • solve quadratic equations by factoring

Common pitfalls

  • Incorrect expansion of brackets, leading to an incorrect polynomial and subsequently an incorrect derivative.
  • Errors in differentiating terms (e.g., differentiating a constant to a non-zero value, or incorrect application of the power rule).
  • Algebraic mistakes when solving the quadratic equation (e.g., incorrect factoring, sign errors, or miscalculation in the quadratic formula).
  • Failing to use all information given in the question (e.g., ignoring the condition "x-coordinate of P is not -4" and providing both solutions for x).
  • Confusing the function $f(x)$ with its derivative $f'(x)$ when checking for stationary points.
