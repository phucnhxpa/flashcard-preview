Parametric Tangent Condition

1 moves · 4 examples
Convert polar tangent-direction condition into dy/dθ = 0 or dx/dθ = 0 via Cartesian parametric form
$\;r = 2\cos\theta,\quad y = r\sin\theta = 2\cos\theta\sin\theta\;$
$\;\dfrac{dy}{d\theta} = -2\sin^2\theta + 2\cos^2\theta = 0 \Rightarrow \theta = \dfrac{\pi}{4}\;$
4q parametric tangent condition

Polar Area Setup

1 moves · 4 examples
Set up polar area integral with correct limits and symmetry
$\;r = \sin 2\theta,\quad \text{one petal}\;$
$\;A = \tfrac{1}{2}\int_0^{\pi/2} \sin^2 2\theta\, d\theta = \tfrac{1}{4}\int_0^{\pi/2}(1-\cos 4\theta)\, d\theta = \dfrac{\pi}{8}\;$
4q polar area setup

Symmetry Exploitation

1 moves · 3 examples
Exploit reflection symmetry to halve the polar area integration range
$\;A = \tfrac{1}{2}\int_{0}^{2\pi} a^2(1+\cos\theta)^2\,d\theta,\quad r(-\theta)=r(\theta)\;$
$\;A = 2 \cdot \tfrac{1}{2}\int_{0}^{\pi} a^2(1+\cos\theta)^2\,d\theta = a^2\int_{0}^{\pi}(1+\cos\theta)^2\,d\theta = \tfrac{3\pi a^2}{2}\;$
3q symmetry exploitation

$\;r(-\theta)=r(\theta) \Rightarrow A = 2 \cdot \tfrac{1}{2}\int_{0}^{\pi}(1+\cos\theta)^2\,d\theta\;$

Formula Selection

Apply the polar area formula A = ½∫r² dθ with correct limits
$\;r = \cos\theta,\quad 0 \le \theta \le \tfrac{\pi}{2}\;$
$\;A = \tfrac{1}{2}\int_0^{\pi/2} \cos^2\theta\,d\theta = \tfrac{1}{2}\int_0^{\pi/2} \tfrac{1+\cos 2\theta}{2}\,d\theta = \dfrac{\pi}{8}\;$
2q formula selection

Cartesian Polar Bridge

1 moves · 2 examples
Convert horizontal-tangent condition on a polar curve to dy/dθ = 0 via y = r sin θ
$\;r = 1 + \cos\theta\;$
$\;y = (1+\cos\theta)\sin\theta \Rightarrow \dfrac{dy}{d\theta} = \cos\theta + \cos 2\theta = 0\;$
2q cartesian polar bridge

Parametric Bridge

1 moves · 1 examples
Bridge polar to parametric via x=r cosθ, y=r sinθ for dy/dx
$\;r = 2\cos\theta,\quad x = 2\cos^2\theta,\quad y = 2\sin\theta\cos\theta = \sin 2\theta\;$
$\;\dfrac{dy}{dx} = \dfrac{dy/d\theta}{dx/d\theta} = \dfrac{2\cos 2\theta}{-2\sin 2\theta} = -\cot 2\theta\;$
1q parametric bridge

Geometric Reinterpretation

1 moves · 1 examples
Convert 'tangent perpendicular to initial line' into dx/dθ = 0
$\;r = 2\cos\theta,\quad x = 2\cos\theta\cdot\cos\theta = 2\cos^2\theta\;$
$\;\dfrac{dx}{d\theta} = -4\cos\theta\sin\theta = -2\sin 2\theta = 0 \Rightarrow \theta = 0,\;\dfrac{\pi}{2}\;$
1q geometric reinterpretation

Tangent Criterion

1 moves · 1 examples
Use dy/dθ = 0 via product rule for polar horizontal tangents
$\;r = 1 + \cos\theta,\quad y = r\sin\theta\;$
$\;\dfrac{dy}{d\theta} = -\sin\theta\sin\theta + (1+\cos\theta)\cos\theta = (2\cos\theta - 1)(\cos\theta + 1) = 0\;$
1q tangent criterion

$\;A_R = \underbrace{\tfrac{1}{2}\,a\,b}_{\triangle PSQ}\;-\;\Bigl(A_{\text{sector}} - \underbrace{\tfrac{1}{2}|x_P y_Q - x_Q y_P|}_{\triangle OPQ}\Bigr)\;$
1q geometric decomposition
