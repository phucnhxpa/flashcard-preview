Complex Numbers

Find principal argument via reference angle plus quadrant adjustment

Family: Quadrant Adjusted Argument Frequency: 2 questions

Trigger — When to use

Fires whenever you need arg(z) for a complex number z = a + bi whose real or imaginary part is negative, i.e. z does not lie in the first quadrant. Cues include: 'find the argument', 'express in modulus–argument form', or 'write in the form r(cos θ + i sin θ)' with at least one negative component. Equivalent phrasings: 'compute θ in the range −π < θ ≤ π for −4 + 3i' or 'give the argument of −4 − 4√3 i'. Whenever signs of Re(z) and Im(z) are not both positive, do NOT plug them straight into arctan — use this two-step method.

Motivation — Why this works

arctan only returns values in (−π/2, π/2), so feeding signed ratios into it can land you in the wrong quadrant or collapse two distinct points (e.g. z and −z) to the same angle. The safe procedure is to compute the acute reference angle α = arctan(|Im|/|Re|) using positive magnitudes, then adjust α according to the quadrant to land inside the principal range −π < θ ≤ π demanded by Edexcel. This works because every complex number's argument is α measured from the positive or negative real axis, with the sign and offset fixed by quadrant geometry — a fact you can verify by sketching an Argand diagram.

Thought Process — The step-by-step reasoning

  1. I read off Re(z) and Im(z) and check their signs to locate which quadrant z lies in (Q1, Q2, Q3, or Q4).
  2. I rule out plugging arctan(Im/Re) directly: e.g. for −4 + 3i this gives arctan(−3/4) ≈ −0.64, a Q4 angle, which is plainly wrong since z is in Q2.
  3. I commit to the reference-angle method because it cleanly separates 'how far from the real axis' (the magnitude α) from 'which side of the axes' (the quadrant correction).
  4. I compute α = arctan(|Im|/|Re|) using the positive ratio, so α is guaranteed to be in (0, π/2).
  5. I select the quadrant rule: Q1 → θ = α; Q2 → θ = π − α; Q3 → θ = −(π − α) = α − π; Q4 → θ = −α. A quick Argand sketch confirms the sign.
  6. I check the result lies in (−π, π] as required, and round (radians) or convert (exact multiples of π) as the question demands.

Mechanic — Concrete operations

  1. Identify the quadrant from the signs of Re(z) and Im(z).
  2. Compute the reference angle α = arctan(|Im(z)|/|Re(z)|) (always positive, acute).
  3. Apply the quadrant rule: Q1: θ = α; Q2: θ = π − α; Q3: θ = α − π; Q4: θ = −α.
  4. Confirm θ ∈ (−π, π], then state the answer (exact in terms of π if α is a standard angle, otherwise round).

Worked Micro-Example

Find arg(−4 + 3i). Signs: Re < 0, Im > 0 ⇒ Q2. Reference angle α = arctan(3/4) ≈ 0.6435. Quadrant rule for Q2: θ = π − α ≈ 3.1416 − 0.6435 ≈ 2.498, so arg(−4 + 3i) ≈ 2.5 rad (1 d.p.).

Variants — Same move, different surface

Third quadrant with exact-angle ratio, e.g. z = −4 − 4√3 i
α = arctan(√3) = π/3, so θ = α − π = −2π/3 (giving the principal value, not +4π/3).
Fourth quadrant, e.g. z = 5 − 5i
α = arctan(1) = π/4, so θ = −α = −π/4.
Pure imaginary or pure real (on an axis)
Skip arctan: arg(positive real) = 0, arg(negative real) = π, arg(positive imaginary) = π/2, arg(negative imaginary) = −π/2.
Question requires polar/modulus–argument form r(cos θ + i sin θ)
Combine with r = √(Re² + Im²); the argument step is identical.

Common Misreadings

  • Treating arctan(Im/Re) on the calculator as the final answer — for −4 − 4√3 i this gives arctan(√3) = π/3, a Q1 angle, totally wrong.
  • Using the positive-range convention θ ∈ [0, 2π) and writing 4π/3 for a Q3 point instead of the Edexcel principal value −2π/3.
  • Using arctan(Re/Im) (adjacent/opposite) instead of arctan(Im/Re) (opposite/adjacent) for the reference angle.

Pitfalls

  • Mixing up the quadrant formulae — confusing π − α (Q2) with π + α, or writing −(π + α) instead of −(π − α) = α − π for Q3. A 5-second Argand sketch eliminates this.
  • Mode error on the calculator: computing α in degrees when the question expects radians (or vice versa).
  • Forgetting to verify θ ∈ (−π, π]; producing an answer like 5π/4 or −5π/3 that lies outside the principal range.

A-level Scope Notes

Fully within Edexcel IAL P3/FP1 complex-number content; the principal argument convention −π < θ ≤ π is standard for Edexcel. No formula booklet entry is needed — the modulus–argument form r(cos θ + i sin θ) is assumed knowledge. Avoid invoking the atan2 function by name; stick to the reference-angle + quadrant table.
