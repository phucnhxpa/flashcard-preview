Trigonometry

Multiply by x then divide by cos x to expose tan x in target form

Family: Target Driven Rearrangement Frequency: 1 questions

Trigger — When to use

Fires in 'show that' problems where a starting equation mixes cos x, sin x / x, and ln x terms, while the target contains tan x and x ln x. The presence of tan x in the goal signals that a sin x / cos x ratio must be engineered. The 1/x in the source and its absence in the target signals multiplication by x. Any time a target trig identity (tan, cot, sec, csc) does not appear directly in the source, suspect a divide-by-cos or divide-by-sin step.

Motivation — Why this works

Multiplying by x clears the sin x / x fraction without disturbing the other terms (since cos x ln x simply becomes x cos x ln x). Dividing by cos x then converts sin x / cos x into tan x and reduces x cos x ln x / cos x to x ln x, exactly matching the target. The choice of operations is dictated by reading the target backwards: each algebraic move closes a specific gap between source and goal.

Thought Process — The step-by-step reasoning

  1. Notice the target tan x + x ln x = 0 contains tan x = sin x / cos x and x ln x, with no 1/x and no bare cos x factor.
  2. Rule out trig identities like double-angle or Pythagorean substitution — there is no squared term or compound angle to simplify.
  3. Commit to a two-step algebraic rearrangement guided purely by matching source to target: clear the 1/x, then create the sin/cos ratio.
  4. Multiply through by x to eliminate sin x / x, producing x cos x ln x + sin x = 0.
  5. Before dividing by cos x, verify cos x ≠ 0 at any stationary point (else sin x = 0 too, contradicting sin²x + cos²x = 1), then divide and rewrite sin x / cos x as tan x.

Mechanic — Concrete operations

  1. Start from cos x ln x + sin x / x = 0.
  2. Multiply through by x: x cos x ln x + sin x = 0.
  3. Justify cos x ≠ 0: if cos x = 0 then sin x = 0, contradicting sin²x + cos²x = 1.
  4. Divide through by cos x: x ln x + sin x / cos x = 0.
  5. Replace sin x / cos x with tan x: tan x + x ln x = 0.

Worked Micro-Example

Show that $\cos x\,\ln x + \dfrac{\sin x}{x} = 0 \iff \tan x + x\ln x = 0$ for $x>0$.\\ Multiply by $x$: $x\cos x\,\ln x + \sin x = 0$.\\ At a solution, $\cos x = 0$ would force $\sin x = 0$, impossible. So divide by $\cos x$:\\ $x\ln x + \dfrac{\sin x}{\cos x} = 0 \Rightarrow \tan x + x\ln x = 0.$

Variants — Same move, different surface

Source contains cos x and sin x / x²; target contains tan x and a polynomial in x
Multiply by x² first to clear the fraction, then divide by cos x to form tan x.
Target contains cot x instead of tan x
Divide by sin x (after verifying sin x ≠ 0) to convert cos x / sin x into cot x.
Target contains sec x or csc x
Divide by cos x or sin x respectively to produce 1/cos x = sec x or 1/sin x = csc x.
Implicit-differentiation 'show that' where dy/dx must be expressed via tan
After differentiating, divide numerator and denominator by cos x to convert ratios into tan x form.

Common Misreadings

  • Trying to apply a Pythagorean or double-angle identity instead of recognising the route is purely algebraic rearrangement matching the target.
  • Skipping the justification that cos x ≠ 0, treating the division as automatic.
  • Multiplying by cos x (wrong direction) hoping to eliminate it, which only complicates the expression.

Pitfalls

  • Dividing by cos x without checking cos x ≠ 0 — a logical gap penalised in 'show that' questions.
  • Forgetting that the multiplication by x is valid only because x > 0 is given (or x ≠ 0 must be argued).
  • Algebraic slip: failing to multiply every term by x, e.g. leaving cos x ln x unchanged.

A-level Scope Notes

Uses only Edexcel IAL P3/P4 content: the identity tan x = sin x / cos x, the Pythagorean identity sin²x + cos²x = 1, and standard algebraic manipulation within a 'show that' proof.
