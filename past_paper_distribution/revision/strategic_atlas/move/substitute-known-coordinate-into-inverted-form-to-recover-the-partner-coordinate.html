Coordinate Geometry

Substitute known coordinate into inverted form to recover the partner coordinate

Family: Parametric Evaluation Frequency: 1 questions

Trigger — When to use

Fires when a point on a curve is specified by one coordinate (often the $y$-value for an inverse-trig curve, or a parameter $t$ for a parametric curve) and the other coordinate is needed for a tangent, normal, or distance calculation. Common cues: 'the point $P$ where $y = \pi/4$ on $y = \arcsin(x/2)$', or 'the point with $y = \pi/3$ on $y = \arctan(x)$'. Especially natural when the inverted relation $x = f^{-1}(y)$ has already been written down (e.g. from an earlier part computing $\frac{dx}{dy}$).

Motivation — Why this works

If $y = f(x)$ is invertible on the relevant branch, then $x = f^{-1}(y)$ is an equivalent description of the same curve. Substituting the known $y$ into $x = f^{-1}(y)$ is algebraically identical to solving $f(x) = y_0$, but it is presented as a direct evaluation rather than an equation to rearrange. This removes a manipulation step and so reduces the chance of sign or domain errors, and it dovetails with the inverted form already used to differentiate via $\frac{dy}{dx} = 1/\frac{dx}{dy}$.

Thought Process — The step-by-step reasoning

  1. Notice that only one coordinate of $P$ is given, but both are needed (e.g. for the tangent equation).
  2. Rule out solving $\arcsin(x/2) = \pi/4$ as an equation — it is correct but introduces an unnecessary 'apply $\sin$ to both sides' step.
  3. Commit to using the inverted form $x = 2\sin y$ that is already on the page (typically from the differentiation part).
  4. Bridge: simply plug $y = \pi/4$ into $x = 2\sin y$.
  5. Tactically, evaluate the exact trig value carefully ($\sin(\pi/4) = \sqrt{2}/2$) and record $P$ as an ordered pair $(x, y)$ ready for tangent/normal work.

Mechanic — Concrete operations

  1. Identify or write the inverted relation $x = f^{-1}(y)$ (e.g. $x = 2\sin y$ from $y = \arcsin(x/2)$).
  2. Substitute the given $y$-value into this relation.
  3. Evaluate using exact values from the standard trig table.
  4. State the point as an ordered pair $P = (x_0, y_0)$.

Worked Micro-Example

Curve: $y = \arcsin(x/2)$, point $P$ has $y = \pi/4$. Invert: $x = 2\sin y$. Substitute: $x = 2\sin(\pi/4) = 2 \cdot \tfrac{\sqrt{2}}{2} = \sqrt{2}$. Hence $P = (\sqrt{2},\, \pi/4)$.

Variants — Same move, different surface

Curve $y = \arctan x$ with given $y_0$
Use $x = \tan y$ and substitute, e.g. $y_0 = \pi/3 \Rightarrow x = \sqrt{3}$.
Parametric curve $x = f(t),\ y = g(t)$ with known $t_0$
Substitute $t_0$ into both $f$ and $g$ to obtain the point directly; no inversion needed.
Implicit curve where one variable is given
Substitute the known value into the implicit equation and solve for the remaining variable, choosing the branch consistent with the stated point.
Curve $y = \ln x$ with given $y_0$
Invert to $x = e^{y}$ and substitute, e.g. $y_0 = 2 \Rightarrow x = e^2$.

Common Misreadings

  • Treating $y = \pi/4$ as an $x$-value and substituting into $y = \arcsin(x/2)$ directly.
  • Believing the inverted form $x = f^{-1}(y)$ is a different curve rather than the same curve re-expressed.
  • Trying to compute $\frac{dy}{dx}$ at $P$ before locating $P$, then getting stuck without an $x$-coordinate.

Pitfalls

  • Evaluating $\sin(\pi/4)$ as $1/2$ (confusing with $\sin(\pi/6)$) instead of $\sqrt{2}/2$.
  • Forgetting the factor of $2$ in $x = 2\sin y$ and writing $x = \sin(\pi/4)$.
  • Picking the wrong branch when inverting (e.g. ignoring the principal range of $\arcsin$ or $\arctan$).
  • Writing the point as $(\pi/4, \sqrt{2})$ with coordinates swapped.
