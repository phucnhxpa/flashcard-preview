Trigger — When to use

Fires when an area-between-two-polar-curves problem is posed and you must decide whether the region is a single annular sweep or a piecewise patchwork. Typical cues: one curve given as a constant r = k and the other as r = a ± b cos(nθ) or a ± b sin(nθ); the question asks for area enclosed between them or area inside one and outside the other. Before integrating, you need to know if r₁ ≥ r₂ everywhere on the interval, or whether they genuinely cross.

Motivation — Why this works

The polar area-between formula ½∫(r_outer² − r_inner²) dθ is only valid where the same curve plays the outer role throughout. If the curves swap dominance, the integral must be split at the crossing angles. By bounding the variable curve via the range of cos(nθ) or sin(nθ) ∈ [−1, 1], you can prove r_min ≥ r_constant (or vice versa), reducing the problem to a single clean integral over a full period and avoiding the search for crossing angles.

Thought Process — Step-by-step reasoning

  1. Notice that one curve is a constant circle r = k while the other is r = a ± b·trig(nθ), so the second curve oscillates between fixed extremes.
  2. Rule out hunting for intersection angles before checking whether intersections actually occur — they may only touch tangentially.
  3. Commit to bounding the oscillating curve: substitute the extreme values ±1 of the trig term to obtain r_min and r_max.
  4. Bridge: compare r_min with the constant k. If r_min ≥ k, the oscillating curve is always outer; one integral over [0, 2π] (or the relevant period) suffices.
  5. Tactical decision: distinguish 'touches' (r_min = k, isolated tangent points) from 'crosses' (r_min < k, genuine arcs swapped); only the latter forces piecewise integration.

Mechanic — Concrete operations

  1. Identify the trig term and use −1 ≤ cos(nθ) ≤ 1 (or sin) to find r_min and r_max of the variable curve.
  2. Compute r_min by substituting the trig value that minimises r.
  3. Compute r_max similarly for completeness/sanity check.
  4. Compare r_min with the constant curve r = k: if r_min ≥ k, declare r₁ ≥ r₂ for all θ.
  5. Note any θ where equality holds — these are tangency points, not crossings, so no split is needed.
  6. Proceed with a single integral ½∫(r₁² − r₂²) dθ over the full period.

Worked Micro-Example

Given $r_1 = 4 - 3\cos 6\theta$ and $r_2 = 1$. Since $-1 \le \cos 6\theta \le 1$, $r_1 \in [1, 7]$. So $r_1 \ge r_2$ for all $\theta$, with equality only when $\cos 6\theta = 1$ (six tangent points). Hence the enclosed area is $A = \tfrac{1}{2}\int_0^{2\pi}\left[(4-3\cos 6\theta)^2 - 1^2\right]\,d\theta$ as a single integral.

Variants — Same move, different surface

r = a + b sin(nθ) versus r = k constant
Use sin(nθ) ∈ [−1, 1] identically; r_min = a − |b|. Check r_min vs k.
Cardioid r = a(1 + cos θ) versus inner circle r = k
r_min = 0 (at θ = π); if k > 0 the curves genuinely cross, so you must find intersection angles and split.
Limaçon r = a + b cos θ with |b| < a versus the pole
Bounding shows r > 0 always, so the curve never reaches the origin — no inner loop, and no need to restrict θ.
Two oscillating curves r₁ = a₁ + b₁ cos(nθ), r₂ = a₂ + b₂ cos(nθ)
Bound r₁ − r₂ using its trig form; if its minimum is ≥ 0, r₁ dominates throughout.

Common Misreadings

  • Treating tangent contact (r_min = k at isolated θ) as genuine crossings and unnecessarily splitting the integral.
  • Assuming any oscillating polar curve must cross a circle of comparable radius without checking the trig bounds.
  • Confusing the period of cos(nθ) with 2π and so under-counting tangent points.

Pitfalls

  • Substituting cos(nθ) = 1 to get r_max instead of r_min when the coefficient of cosine is negative — sign of b matters.
  • Forgetting that cos(nθ) = 1 occurs at n distinct values of θ in [0, 2π), so 'touches at one point' is wrong.
  • Integrating over [0, 2π/n] (one petal period) when the full enclosed region requires [0, 2π].

A-level Scope Notes
