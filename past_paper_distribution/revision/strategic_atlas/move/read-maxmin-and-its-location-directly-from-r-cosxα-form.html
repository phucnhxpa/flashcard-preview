Trigonometry

Read max/min and its location directly from R cos(x+α) form

Family: Amplitude Phase Extraction Frequency: 1 questions

Trigger — When to use

Fires in part (c)-style questions after a cos x + b sin x has already been rewritten as R cos(x ± α) or R sin(x ± α). Cues include phrases like 'find the maximum/minimum value of f(x) and the value of x at which it occurs', or 'state the greatest value', usually within a stated domain such as 0° ≤ x ≤ 360°. Also fires on shifted versions like k + R cos(x + α). Whenever the expression has been put in single-sinusoid form, calculus is unnecessary.

Motivation — Why this works

Since −1 ≤ cos θ ≤ 1 for all θ, the expression R cos(x + α) is bounded between −R and R, with the bounds attained exactly when cos(x + α) = ±1. The R-formula has already absorbed the amplitude and phase shift, so the extremes and their locations can be read off by solving cos(x + α) = 1 (max) or cos(x + α) = −1 (min). This bypasses differentiation and is the intended payoff of the harmonic-form rewrite.

Thought Process — The step-by-step reasoning

  1. Notice the expression is already in the form R cos(x + α); part (c) asks for an extremum.
  2. Rule out calculus — the bounded range of cosine makes derivatives unnecessary here.
  3. Commit to the bound argument: max of cos is 1, so max of R cos(x + α) is R; min is −R.
  4. Bridge to operations: set x + α equal to the angle within the given domain where cos = 1 (or −1).
  5. Tactically, pick the representative angle (0°, 360°, 180°, etc.) so that the resulting x lies inside the stated interval, then solve for x and round as required.

Mechanic — Concrete operations

  1. Quote the bound: −R ≤ R cos(x + α) ≤ R, so maximum value = R, minimum value = −R.
  2. For the maximum, set cos(x + α) = 1, i.e. x + α = 0° or 360° (choose so x is in the domain).
  3. Solve for x; e.g. x + 22.62° = 360° ⇒ x = 337.4° (1 d.p.).
  4. For the minimum, set cos(x + α) = −1, i.e. x + α = 180°, and solve for x.
  5. If the original expression has a vertical shift k + R cos(x + α), add k to both extremes; the x-locations are unchanged.

Worked Micro-Example

Given $5\cos x + 12\sin x = 13\cos(x - 67.38°)$ on $0° \le x \le 360°$. Maximum value $= 13$, attained when $x - 67.38° = 0°$, i.e. $x = 67.4°$ (1 d.p.). Minimum value $= -13$, attained when $x - 67.38° = 180°$, i.e. $x = 247.4°$.

Variants — Same move, different surface

Expression written as R sin(x + α)
Maximum R occurs when x + α = 90°; minimum −R when x + α = 270° (or −90°).
Vertical shift: f(x) = k + R cos(x + α)
Max = k + R, min = k − R; the x-values where they occur are the same as for R cos(x + α).
Reciprocal form: g(x) = 1/(k + R cos(x + α)) with k > R
Max of g is 1/(k − R) at the x where the denominator is minimised; min of g is 1/(k + R) at the x where the denominator is maximised.
Sign flip: f(x) = −R cos(x + α)
Max R now occurs where cos(x + α) = −1; min −R where cos(x + α) = 1.
Restricted domain that excludes the natural extremum
Check endpoints of the interval; the extremum may be attained at a boundary rather than where cos(x + α) = ±1.

Common Misreadings

  • Treating this as a calculus problem and differentiating, when the harmonic form already exposes the answer.
  • Believing the maximum depends on α (it does not — α only shifts the location, not the amplitude).
  • Thinking the minimum requires the original a cos x + b sin x to be re-derived, rather than just using −R.

Pitfalls

  • Stating max = R but giving the x-value where cos(x + α) = −1 (mixing up max and min locations).
  • Reporting x + α instead of x as the location of the extremum.
  • Choosing x + α = 0° when this gives x = −α outside the stated domain; should use 360° instead.
  • Forgetting to add the vertical shift k when the expression is k + R cos(x + α).

A-level Scope Notes

Fully within Edexcel IAL P3/P4 (Maths) treatment of harmonic form a cos x + b sin x = R cos(x ± α) or R sin(x ± α); the bounded-range argument is the standard non-calculus method expected in part (c) of such questions.
