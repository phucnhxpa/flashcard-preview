Trigonometry

Enumerate all solutions of cos θ = k using Q1/Q4 symmetry

Family: Cast Diagram Analysis Frequency: 1 questions

Trigger — When to use

Fires when an equation has been reduced to the form cos θ = k (with |k| ≤ 1) and a full set of solutions is required over a stated interval such as [0°, 360°) or [−180°, 180°]. Cues include phrases like 'find all values of θ', 'solve for 0° ≤ θ < 360°', or a question worth more than one mark for a single trig equation. Also fires after substitution steps (e.g. cos 2x = k, cos(x − 30°) = k) once the cosine has been isolated.

Motivation — Why this works

The cosine function is even and 360°-periodic, so cos α = cos(−α) = cos(360° − α). On the CAST diagram cosine is positive in quadrants A (Q1) and C (Q4) and negative in S (Q2) and T (Q3); whichever sign k has, exactly two solutions lie in any 360°-wide interval, paired as α and 360° − α. Using arccos alone returns only the principal value in [0°, 180°], so the symmetry rule is what guarantees no marks are dropped for missing solutions.

Thought Process — The step-by-step reasoning

  1. Notice the equation is in the clean form cos θ = k with k a recognisable exact value (e.g. 1/2, √3/2), so a calculator/exact principal value is immediately available.
  2. Rule out sine and tangent symmetry rules: the relevant symmetry for cosine is reflection about the horizontal axis of the unit circle, giving the (360° − α) partner — not (180° − α).
  3. Commit to the two-quadrant picture: sign of k determines the pair (Q1+Q4 if k>0, Q2+Q3 if k<0); no other quadrants can contribute within one 360° period.
  4. Bridge: compute the acute reference angle α = arccos(|k|), then build the two answers from α using the quadrant template.
  5. Tactical check: confirm each candidate lies inside the stated interval, watch endpoint conventions (strict vs inclusive), and if the interval is wider/narrower than 360° add or remove 360° multiples accordingly.

Mechanic — Concrete operations

  1. Identify the sign of k and hence the two active quadrants (k > 0 → Q1, Q4; k < 0 → Q2, Q3).
  2. Compute the reference angle α = arccos(|k|) in [0°, 90°].
  3. Build the two solutions: if k > 0 use θ = α and θ = 360° − α; if k < 0 use θ = 180° − α and θ = 180° + α.
  4. Discard or shift any solution lying outside the required interval; add ±360° to recover solutions if the interval is shifted.
  5. State the final solution set, checking endpoint inequalities (e.g. exclude 360° if the range is < 360°).

Worked Micro-Example

Solve $\cos\theta = \tfrac{1}{2}$ for $0° \le \theta < 360°$. Principal value: $\alpha = \arccos(\tfrac{1}{2}) = 60°$. Since $k>0$, solutions lie in Q1 and Q4: $\theta_1 = 60°$ and $\theta_2 = 360° - 60° = 300°$. Hence $\theta = 60°,\ 300°$.

Variants — Same move, different surface

sin θ = k with k > 0
Use Q1/Q2 symmetry: θ = α and θ = 180° − α, where α = arcsin(k).
tan θ = k
Use 180° periodicity: θ = α and θ = 180° + α, where α = arctan(k); only one pair per 360°.
cos θ = k with k < 0
Active quadrants are Q2 and Q3: θ = 180° − α and θ = 180° + α, with α = arccos(|k|).
cos(nθ) = k over [0°, 360°)
Solve for nθ over [0°, 360°·n), enumerate all (360° − α) + 360°m partners, then divide by n.
cos(θ − φ) = k
Let u = θ − φ, solve cos u = k over the shifted interval, then add φ back to each solution.
Range given in radians, e.g. [0, 2π)
Replace 360° by 2π and 180° by π; partner of α is 2π − α.

Common Misreadings

  • Applying the sine rule (180° − α) instead of the cosine rule (360° − α), placing the second solution in Q2 instead of Q4.
  • Treating cos θ = k as having only one solution because arccos returns a single value.
  • Forgetting to undo a substitution (e.g. solving cos 2θ = k but reporting values of 2θ as θ).
  • Using the reference angle α = arccos(k) directly when k < 0 (which gives an obtuse angle), instead of α = arccos(|k|).

Pitfalls

  • Stopping after the principal value and losing the Q4 (or Q3) solution mark.
  • Including θ = 360° when the range is 0° ≤ θ < 360° (strict upper bound).
  • Rounding the reference angle prematurely, causing the partner 360° − α to be inconsistent.
  • When the equation has been multiplied/divided by a trig factor, missing extra solutions introduced or lost in that algebraic step.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P2/P3) trigonometric equation solving; uses only the unit circle, CAST diagram, and standard symmetries cos(360° − θ) = cos θ and cos(−θ) = cos θ.
