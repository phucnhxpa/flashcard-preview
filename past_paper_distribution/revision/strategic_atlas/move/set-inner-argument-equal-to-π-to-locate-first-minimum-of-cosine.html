Trigonometry

Set inner argument equal to π to locate first minimum of cosine

Family: Argument Analysis Frequency: 1 questions

Trigger — When to use

Fires when a question asks for the coordinates of a minimum (or maximum) point on a transformed cosine curve such as y = cos(kx), y = cos(x − α), or y = a cos(bx) + c. Phrases like 'find the coordinates of the minimum point', 'first minimum for x > 0', or labelling a turning point P on a sketch are typical cues. The function must be a pure cosine (or sine) so that extrema are determined purely by the argument, not by calculus.

Motivation — Why this works

cos(θ) attains its minimum value of −1 exactly when θ = π + 2nπ, regardless of how θ is built from x. Treating the inner expression as a single argument reduces a graph-feature question to a linear equation in x. This bypasses differentiation entirely and is the standard A-level approach for transformed sinusoids.

Thought Process — The step-by-step reasoning

  1. Notices that y = cos(2x) is a pure cosine in a transformed variable, so its extrema are governed by the argument 2x rather than by setting dy/dx = 0.
  2. Rules out using calculus (overkill) and rules out reading the minimum at x = π (that would be the minimum of cos(x), not cos(2x)).
  3. Commits to the principle: cos(θ) = −1 ⇔ θ = π + 2nπ, with the smallest positive x corresponding to n = 0.
  4. Bridges to the first concrete operation by writing 2x = π and isolating x.
  5. Tactically picks the smallest non-negative solution (the 'first' minimum), and pairs it with y = −1 to give coordinates.

Mechanic — Concrete operations

  1. State the principle: min of cos(θ) is −1, occurring at θ = π + 2nπ.
  2. Identify the inner argument (here θ = 2x).
  3. Set the inner argument equal to π for the first minimum with x > 0.
  4. Solve the resulting linear equation for x.
  5. Write coordinates as (x, −1) (adjusting y if there is a vertical shift or scale).

Worked Micro-Example

Find the first minimum of $y = \cos(2x)$ for $x>0$.\\ Set $2x = \pi$, so $x = \dfrac{\pi}{2}$.\\ At this $x$, $y = \cos(\pi) = -1$.\\ Hence $P = \left(\dfrac{\pi}{2},\,-1\right)$.

Variants — Same move, different surface

y = sin(kx), first maximum for x > 0
Set kx = π/2 instead of π; y-value is +1.
y = cos(x − α), phase-shifted cosine
Set x − α = π, giving x = α + π for the first minimum.
y = a cos(bx) + c with a > 0
Argument condition unchanged (bx = π); minimum y-value is c − a.
y = a cos(bx) + c with a < 0
Minimum now occurs where cos(bx) = +1, so set bx = 0 (or 2π for next one).
Finding the n-th minimum or one in a stated interval
Use bx = π + 2nπ and pick the n that places x in the required interval.

Common Misreadings

  • Treating x itself as the argument and writing x = π, forgetting the horizontal scaling factor.
  • Confusing the minimum condition for cos with that for sin (using π/2 or 3π/2 instead of π).
  • Assuming the minimum y-value is still −1 when there is a vertical scale or shift, rather than c − |a|.

Pitfalls

  • Forgetting to divide by the coefficient of x after setting the argument equal to π.
  • Picking 3π or −π instead of π and missing the FIRST minimum for x > 0.
  • Sign error when the cosine is preceded by a negative coefficient — the minimum then aligns with cos = +1, not −1.
  • Writing the y-coordinate as 1 by reflex when the curve has been reflected or shifted.

A-level Scope Notes

Standard Edexcel IAL Pure content (C2/P2 trig graphs and transformations); no calculus required and all manipulations stay within basic equation solving and known exact values of cosine.
