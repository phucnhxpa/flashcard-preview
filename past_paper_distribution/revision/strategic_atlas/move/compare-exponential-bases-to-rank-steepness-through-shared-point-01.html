Algebra & Functions

Compare exponential bases to rank steepness through shared point (0,1)

Family: Graph Transformation Analysis Frequency: 1 questions

Trigger — When to use

Fires when asked to sketch two or more curves of the form y = a^x on the same axes, with no transformations applied (no shifts, scalings, or reflections). Typical cues: 'sketch y = 2^x and y = 4^x on the same diagram', or 'on the same axes draw y = e^x and y = 3^x'. Recognised by all curves being pure exponentials a^x rather than translates like 2^{x-1} or a·b^x.

Motivation — Why this works

Every curve y = a^x with a > 0 satisfies a^0 = 1, so they share the anchor point (0, 1) and the asymptote y = 0. For a > 1, the curve grows; the larger the base, the faster the growth for x > 0 and the faster the decay for x < 0. So the entire relative shape is determined by ordering the bases — no transformation chain is needed.

Thought Process — The step-by-step reasoning

  1. Notice each equation is of the pure form y = a^x with no added constants, scale factors, or shifts.
  2. Rule out transformation reasoning (stretches/translations) since neither curve is written as a modification of the other.
  3. Commit to a base-comparison argument: shared point (0,1), shared asymptote y = 0, ordering by base size.
  4. Bridge to operation: list each base, decide which is larger, and decide growth vs decay regime (a > 1 or 0 < a < 1).
  5. Tactically check a sample x-value (e.g. x = 1) to confirm which curve sits above the other on the right of the y-axis, and mirror that conclusion on the left.

Mechanic — Concrete operations

  1. Confirm each curve has the form y = a^x with a > 0, a ≠ 1.
  2. Mark the common point (0, 1) and the common horizontal asymptote y = 0.
  3. Order the bases; identify each as growth (a > 1) or decay (0 < a < 1).
  4. For x > 0, the curve with the larger base lies above; for x < 0, it lies below (curves cross only at (0,1)).
  5. Sketch both curves through (0,1), approaching y = 0, with the steeper-base curve rising more sharply on the right.

Worked Micro-Example

Sketch $y = 2^x$ and $y = 4^x$ on the same axes. Both pass through $(0,1)$ with asymptote $y = 0$. At $x = 1$: $2^1 = 2$ and $4^1 = 4$, so $4^x$ is above $2^x$ for $x > 0$. At $x = -1$: $2^{-1} = 0.5$ and $4^{-1} = 0.25$, so $4^x$ is below $2^x$ for $x < 0$. Draw $y = 4^x$ steeper, $y = 2^x$ shallower, both crossing at $(0,1)$.

Variants — Same move, different surface

Comparing y = e^x with y = 3^x
Use e ≈ 2.718 < 3, so y = e^x is the shallower curve; same (0,1) anchor and y = 0 asymptote.
Comparing decay curves y = (1/2)^x and y = (1/4)^x
Both decreasing; the smaller base (1/4) decays faster, so y = (1/4)^x lies below y = (1/2)^x for x > 0 and above it for x < 0.
Comparing a growth curve y = 2^x with a decay curve y = (1/2)^x
They are reflections of each other in the y-axis, still meeting at (0,1) with asymptote y = 0.

Common Misreadings

  • Treating y = 2^x as a vertical stretch or translation of y = 4^x instead of a separate exponential with its own base.
  • Assuming the curves are parallel or never cross, missing that they share the point (0,1).

Pitfalls

  • Drawing the curves with different y-intercepts — every y = a^x passes through (0,1).
  • Forgetting the asymptote y = 0 and letting the curve touch or cross the x-axis.
  • Getting the left/right ordering wrong: the steeper-base curve is higher for x > 0 but lower for x < 0.

A-level Scope Notes

Fully within Edexcel IAL Pure scope: sketching y = a^x and y = e^x curves, recognising the shared point (0,1) and asymptote y = 0, and comparing growth rates by base size.
