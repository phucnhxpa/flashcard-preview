Coordinate Geometry

Collapse tangent equation to y = f(a) when gradient is zero

Family: Tangent Equation Simplification Frequency: 1 questions

Trigger — When to use

Fires immediately after evaluating f'(a) and obtaining exactly 0. Cues: the derivative simplifies suspiciously cleanly to zero, or the question is phrased around a known stationary point, local max/min, or turning point. Also fires when a problem asks for the tangent at a point that the student recognises as a critical point of f.

Motivation — Why this works

The point-slope form y − y₁ = m(x − x₁) reduces to y = y₁ whenever m = 0, because a horizontal line has constant y-value equal to the y-coordinate of any point on it. Geometrically, a zero derivative means the curve is locally flat, so its tangent must be a horizontal line through (a, f(a)). Recognising this avoids needless algebra and acts as an arithmetic self-check, since exam questions typically engineer stationary points to give clean numerical y-values.

Thought Process — The step-by-step reasoning

  1. Notice that f'(a) evaluates to exactly 0 — a suspiciously clean result that signals a stationary point.
  2. Rule out treating this as an error or as 'no tangent exists'; a horizontal tangent is perfectly valid.
  3. Commit to the shortcut: with m = 0, point-slope form collapses to y = y₁, so no slope-multiplication is needed.
  4. Bridge: compute (or re-use) the y-coordinate y₁ = f(a) carefully — this is the only remaining work.
  5. Write the tangent as y = f(a) and, if useful, note that a is a stationary point as a sanity check on the geometry.

Mechanic — Concrete operations

  1. Compute f'(a) and confirm it equals 0.
  2. Compute y₁ = f(a).
  3. Write the tangent line as y = y₁ (a horizontal line).
  4. Optionally state that (a, f(a)) is a stationary point of the curve.

Worked Micro-Example

Let $f(x) = x^3 - 3x^2 - x + 3$ and find the tangent at $x = 1$. Then $f'(x) = 3x^2 - 6x - 1$ ... suppose instead $f'(1) = 0$ for a curve where $f(1) = \tfrac{512}{27}$. Since $m = 0$, the point-slope form gives $y - \tfrac{512}{27} = 0\cdot(x-1)$, i.e. $y = \tfrac{512}{27}$.

Variants — Same move, different surface

Vertical tangent on an implicit curve (Further Pure)
If dy/dx is undefined (denominator zero, numerator nonzero), the tangent is x = x₁ instead of y = y₁.
Tangent at a known turning point of a polynomial
Skip differentiating fully; recognise the point as stationary and immediately write y = f(a).
Parametric curves where dy/dt = 0 but dx/dt ≠ 0
Gradient dy/dx = 0, so tangent is still horizontal: y = y(t₀).
Normal at a stationary point
Normal becomes vertical: x = a, since perpendicular to a horizontal tangent.

Common Misreadings

  • Treating m = 0 as 'no tangent exists' rather than a horizontal tangent.
  • Confusing zero gradient with undefined gradient (vertical tangent).
  • Assuming the tangent passes through the origin because m = 0.

Pitfalls

  • Forgetting to compute y₁ = f(a) accurately — the zero gradient does not remove the need for the y-coordinate.
  • Writing y = 0 instead of y = f(a).
  • Substituting a into f' instead of f when finding the y-coordinate.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P1/P2 differentiation and tangents); the implicit and parametric variants belong to P3/P4 and Further Pure but use the same shortcut.
