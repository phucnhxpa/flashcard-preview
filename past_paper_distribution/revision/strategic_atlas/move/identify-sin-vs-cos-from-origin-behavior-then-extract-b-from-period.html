Trigonometry

Identify sin vs cos from origin behavior, then extract B from period

Family: Graph To Equation Identification Frequency: 1 questions

Trigger — When to use

A figure shows a sinusoidal curve and asks for its equation in the form A·sin(Bx) or A·cos(Bx). The curve's value and gradient at x = 0 are visible, and one full cycle can be measured. Phrasings include 'find an equation of the curve', 'state the values of A and B', or 'write g(x) in the form ...'.

Motivation — Why this works

Since sin(0) = 0 and cos(0) = 1, the y-intercept and initial gradient uniquely fix the base function type without needing a phase shift. Once the type is locked, the standard relation T = 2π/B converts the visually-readable period into the frequency multiplier B, while the max/min values give A. This produces the simplest equation matching the graph, which is what mark schemes reward.

Thought Process — The step-by-step reasoning

  1. Notice the y-intercept: does the curve pass through (0, 0) or through (0, ±A)?
  2. Rule out cos if y(0) = 0, since a non-shifted cosine cannot start at zero; rule out sin if y(0) = ±A.
  3. Commit to sin (with sign chosen by the initial gradient: positive slope → +sin, negative slope → −sin).
  4. Read amplitude A from the peak/trough, then measure one full cycle to get the period T.
  5. Apply B = 2π/T and assemble g(x) = A·sin(Bx), checking a second feature (e.g. location of first maximum) for consistency.

Mechanic — Concrete operations

  1. Check y-intercept and initial gradient at x = 0 to choose between ±sin and ±cos.
  2. Read amplitude: A = (max − min)/2 (here max = 1, min = −1 ⇒ A = 1).
  3. Read period T from one full cycle on the graph (here T = π).
  4. Solve T = 2π/B for B (here B = 2π/π = 2).
  5. Write the equation, e.g. g(x) = sin(2x), and verify a checkpoint such as g(π/4) = 1.

Worked Micro-Example

Curve passes through $(0,0)$ with positive gradient, peaks at $y=1$ when $x=\tfrac{\pi}{4}$, returns to $0$ at $x=\tfrac{\pi}{2}$, troughs at $-1$ when $x=\tfrac{3\pi}{4}$, and completes a cycle at $x=\pi$. So $A=1$, $T=\pi$, $B=2\pi/\pi=2$, giving $g(x)=\sin(2x)$. Check: $g(\tfrac{\pi}{4})=\sin(\tfrac{\pi}{2})=1$. ✓

Variants — Same move, different surface

Curve starts at its maximum y = A at x = 0
Choose g(x) = A·cos(Bx); B still from T = 2π/B.
Curve starts at its minimum y = −A at x = 0
Choose g(x) = −A·cos(Bx) (equivalent to A·cos(Bx + π)).
Curve passes through origin but initially decreases
Choose g(x) = −A·sin(Bx).
Zero crossing occurs at x = c ≠ 0
Use a phase shift: g(x) = A·sin(B(x − c)), or rewrite as a cosine if simpler.
Graph oscillates about y = k rather than y = 0
Add a vertical shift: g(x) = A·sin(Bx) + k, with k = (max + min)/2.

Common Misreadings

  • Writing cos(2x − π/2) instead of sin(2x): equivalent but not the simplest form expected.
  • Misreading the period as 2π by default and writing sin(x) with the wrong frequency.
  • Confusing amplitude with peak value when there is a vertical shift, giving the wrong A.

Pitfalls

  • Forgetting the sign: if the curve decreases from the origin, the answer is −sin(2x), not sin(2x).
  • Reading half a cycle (zero to next zero) as a full period, doubling B incorrectly to 4.
  • Ignoring a vertical translation when the midline is not y = 0.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P2/P3 trig graphs and transformations); uses only standard sine/cosine shape, period formula T = 2π/B, and basic transformations y = A f(Bx) + k.
