Differentiation

Take t→∞ limit of decaying exponential to identify horizontal asymptote/model ceiling

Family: Asymptotic Analysis Frequency: 2 questions

Trigger — When to use

Fires when an exponential model of the form A ± Be^{-kt} or L/(1+Ce^{-kt}) (with k>0) is given and the question asks about long-term behaviour. Look for verbal cues such as 'limiting value', 'in the long run', 'maximum possible', 'carrying capacity', or 'show that [quantity] can never reach/exceed [value]'. The structural cue is a decaying exponential term — an exponent that tends to −∞ as t→∞ — embedded in an otherwise constant expression. Equivalent phrasings include 'find the value the model approaches', 'state the horizontal asymptote', or 'explain why the population never reaches…'.

Motivation — Why this works

As t→∞ any term e^{-kt} (k>0) tends to 0, so the exponential contribution vanishes and the model collapses to its constant part — this constant IS the horizontal asymptote and represents the model's intrinsic ceiling (or floor). The technique works because exponential decay is monotone and bounded below by 0 but never equal to 0 for finite t, which simultaneously gives both the limiting value AND a strict inequality bound. This single computation answers 'what value does the model approach?' and, with the strictness argument, 'why can the model never reach value X beyond the ceiling?'.

Thought Process — The step-by-step reasoning

  1. Notice the model has a constant piece plus/minus an exponential piece whose exponent is linear in t with negative leading coefficient — this is the signature of bounded asymptotic behaviour.
  2. Rule out alternatives: setting the model equal to the target value and solving for t (works but messier and weaker for 'never reach' arguments); differentiating to find a maximum (unnecessary since the function is monotone); guessing a numerical large t (not rigorous).
  3. Commit to the limit approach because horizontal asymptotes are extracted in one line by sending the decaying exponential to 0, and this directly produces both the limiting value and (with one extra sentence) a strict bound.
  4. Bridge to operation: isolate the exponential term, confirm its exponent → −∞, replace it with 0 in the formula, and read off the constant that remains.
  5. If the question is a 'show it can never reach' problem, add the strictness step: e^{-kt} > 0 for all finite t, so the model is strictly less than (or greater than) the asymptote, then compare the asymptote to the target value to close the argument.

Mechanic — Concrete operations

  1. Identify the decaying exponential term, e.g. e^{-kt} or e^{a-kt} with k>0.
  2. Confirm the exponent → −∞ as t→∞, so e^{(\cdot)} → 0.
  3. Substitute 0 for the exponential term in the model and simplify to obtain the limiting constant L.
  4. (If needed) State e^{(\cdot)} > 0 for all finite t to deduce a STRICT one-sided inequality between the model and L.
  5. (If 'never reach' is asked) Compare L to the target value to conclude the inequality holds.

Worked Micro-Example

Given $N=\dfrac{900}{2+e^{-0.12t}}$. As $t\to\infty$, $e^{-0.12t}\to 0$, so $N\to \dfrac{900}{2}=450$. Since $e^{-0.12t}>0$ for all finite $t$, the denominator $>2$, hence $N<450<500$ — so $N$ can never reach 500.

Variants — Same move, different surface

Subtractive model G = A − Be^{-kt}
Exponential vanishes, G → A from below; A is the maximum (e.g. total gold extractable).
Logistic model P = L/(1+Ce^{-kt})
Denominator → 1, so P → L; L is the carrying capacity.
Newton's law of cooling T = T_s + (T_0 − T_s)e^{-kt}
Exponential vanishes, T → T_s; the surrounding temperature is the asymptote.
'Show value V is never reached' phrasing
Add the strictness line e^{-kt}>0 to upgrade limit to strict bound, then compare bound to V.

Common Misreadings

  • Treating 'limiting value' as 'maximum rate of change' and differentiating instead of taking a limit.
  • Setting N = target and solving for t to show no solution exists — works but is less elegant and often loses 'explain' marks.
  • Reading the asymptote as an attained value, writing 'N = 450 when t = ∞' rather than 'N approaches but never equals 450'.

Pitfalls

  • Ignoring constant offsets in the exponent: writing e^{1-0.05t} → e^1 by forgetting the −0.05t dominates as t→∞.
  • Forgetting the strictness step (e^{-kt} > 0 for finite t) in 'never reach' problems — without it you've only shown the limit, not a strict bound.
  • Failing to explicitly compare the asymptote to the target value (e.g. stating 450 is the ceiling but not writing 450 < 500), which leaves the conclusion incomplete.
  • Sign confusion: in A + Be^{-kt} the model approaches A from above, in A − Be^{-kt} from below — get the direction right when arguing strict inequality.

A-level Scope Notes

Fully within Edexcel IAL P2/P3 exponential models and IAL Statistics/Mechanics applied contexts. Uses only the standard fact e^{-kt}→0 as t→∞ for k>0; no formal ε-δ or L'Hôpital reasoning is required or expected. Avoid invoking continuity-of-limits theorems or improper integrals — the move is a one-line substitution argument.
