Find x-intercepts by setting y=0 and factoring out the common variable

Family: Root Finding By Factoring Frequency: 1 questions

Trigger — When to use

Fires whenever a problem requires the x-intercept(s) of a curve y = f(x) where f(x) has no constant term, so x is a common factor. Typical cues: 'where C meets the x-axis', 'the positive x-intercept', or a vertical/normal line 'passing through the x-intercept of C'. Also fires when a quadratic or cubic is set equal to zero and inspection shows every term contains x. Recognise it as faster than the quadratic formula whenever a common factor is visible.

Motivation — Why this works

If every term of f(x) contains x, then f(x) = x·g(x), so f(x) = 0 splits cleanly into x = 0 or g(x) = 0 by the zero-product principle. This avoids the overhead of the quadratic formula or completing the square and exposes both roots immediately. It also generalises: any polynomial with no constant term has 0 as a root, and the remaining roots come from the reduced factor. The move is essentially the first step of full polynomial factorisation.

Thought Process — The step-by-step reasoning

  1. I notice the curve's equation has no constant term, so x = 0 is automatically a root and the other root(s) come from a simpler reduced expression.
  2. I rule out the quadratic formula and completing the square as unnecessarily heavy when a common factor is staring at me.
  3. I commit to factoring because the zero-product principle will give all roots in one line, and the question's wording ('positive intercept') tells me to discard x = 0 at the end.
  4. First concrete operation: set y = 0 and pull out the highest common factor of x.
  5. Tactical decision: keep the fraction inside the bracket rather than clearing it prematurely, then solve the linear factor carefully (multiply both sides by 8, not divide).

Mechanic — Concrete operations

  1. Set y = 0 to obtain the root equation f(x) = 0.
  2. Identify x as a common factor of every term and write f(x) = x·g(x).
  3. Apply the zero-product principle: x = 0 or g(x) = 0.
  4. Solve g(x) = 0 for the remaining root(s).
  5. Select the root demanded by the question (e.g. positive, non-zero) and state the result.

Worked Micro-Example

Find the positive x-intercept of $y = 2x - \tfrac{1}{8}x^2$.\\ Set $y=0$: $0 = 2x - \tfrac{1}{8}x^2 = x\left(2 - \tfrac{x}{8}\right)$.\\ So $x = 0$ or $2 - \tfrac{x}{8} = 0 \Rightarrow x = 16$.\\ The positive intercept is $x = 16$, so the vertical line through it is $x = 16$.

Variants — Same move, different surface

Cubic with no constant term, e.g. $y = x^3 - 4x$
Factor out x to get $x(x^2 - 4) = x(x-2)(x+2)$; roots are $0, \pm 2$.
Curve $y = x^2 e^{x}$ or $y = x\sin x$ in Further Pure / IAL contexts
x is still a common factor; the non-polynomial factor is set to zero separately, sometimes giving infinitely many roots (choose the relevant one).
Finding where a tangent or normal meets the x-axis after deriving its equation
Same set y=0 step, but the equation is linear so factoring reduces to one rearrangement.
Intersection of two curves $y = f(x)$ and $y = g(x)$ where $f-g$ has no constant term
Set $f(x) - g(x) = 0$ and factor out x to find x-coordinates of intersection.

Common Misreadings

  • Dividing both sides by x at the outset, silently discarding the root x = 0.
  • Treating 'x-intercept' as a y-value rather than an x-coordinate.
  • Assuming the quadratic formula is required whenever a quadratic appears, missing the common factor.

Pitfalls

  • Arithmetic slip when solving $\tfrac{x}{8} = 2$: writing $x = 4$ instead of $x = 16$.
  • Forgetting that 'positive x-intercept' explicitly excludes x = 0.
  • Sign errors when the leading coefficient is negative, e.g. mishandling $-\tfrac{1}{8}x^2$ inside the bracket.
  • Stating the intercept as a coordinate $(16,0)$ when a line equation $x=16$ is required (or vice versa).

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P1/P2): polynomial root-finding by factorisation and the zero-product principle, applied to curves, tangents, and intersections.
