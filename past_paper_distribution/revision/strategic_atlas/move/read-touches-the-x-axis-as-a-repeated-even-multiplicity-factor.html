Algebra & Functions

Read 'touches the x-axis' as a repeated (even-multiplicity) factor

Family: Root Multiplicity Analysis Frequency: 1 questions

Trigger — When to use

Fires when a question states a curve 'touches' (or 'is tangent to') the x-axis at some point and supplies f(x) in factored form containing a squared (or even-power) factor. Also fires for phrasings like 'meets but does not cross', 'has a stationary point on the x-axis', or when a sketch shows the curve grazing the axis. The cue is the geometric word 'touches' paired with a polynomial whose factorisation reveals multiplicity.

Motivation — Why this works

A factor (x - a)^k contributes a root of multiplicity k at x = a. Even multiplicities (k = 2, 4, ...) make f(x) keep its sign on either side of a, so the curve meets the axis and bounces back — geometrically a 'touch'. Odd multiplicities cause a sign change, hence a 'cross'. So 'touches' is a direct geometric translation of an even-power factor, letting the student read the touching point straight off the factorisation without calculus.

Thought Process — The step-by-step reasoning

  1. Notice the keyword 'touches' and recall it signals tangency to the axis, not crossing.
  2. Rule out the linear (odd-power) factors as candidates for the touch point — these correspond to crossings.
  3. Commit to the squared factor as the location of the touch, since even multiplicity preserves sign locally.
  4. Bridge to operation: read the root from the squared factor, e.g. (x - 1/2)^2 gives x = 1/2.
  5. Since the point lies on the x-axis, set y = 0 and write the coordinates as (root, 0).

Mechanic — Concrete operations

  1. Scan f(x) for any factor raised to an even power.
  2. Set that factor equal to zero to find the x-coordinate of the touch point.
  3. Confirm the remaining (odd-power) factors correspond to crossing points, not the touch.
  4. State the touching point as (x-value, 0).

Worked Micro-Example

Given $f(x) = (x - \tfrac{1}{2})^2(x - 4)$, the curve touches the x-axis where the squared factor vanishes. Setting $(x - \tfrac{1}{2})^2 = 0$ gives $x = \tfrac{1}{2}$. The factor $(x - 4)$ is linear, so the curve crosses at $x = 4$. Hence the touching point is $P = (\tfrac{1}{2}, 0)$.

Variants — Same move, different surface

Quartic with two squared factors, e.g. $(x-a)^2(x-b)^2$
The curve touches at both $x = a$ and $x = b$; identify both as touch points.
Cubic with a cubed factor $(x-a)^3$
Multiplicity 3 is odd, so the curve crosses (with a stationary point of inflection on the axis), not touches.
Factor form hidden, e.g. $f(x) = x^2(x-4)$
Recognise $x^2$ as the squared factor; touch point is the origin $(0,0)$.
Question gives a sketch and asks for the equation
Each touch on the axis contributes a squared factor; each crossing contributes a linear factor.

Common Misreadings

  • Confusing 'touches' with 'crosses' and picking the linear-factor root instead.
  • Forgetting that a point on the x-axis has y-coordinate 0.
  • Treating $(x - 1/2)^2$ as giving two different roots rather than one repeated root.

Pitfalls

  • Sign errors when reading the root from $(x - a)^2$: the root is $x = a$, not $x = -a$.
  • Assuming all even powers behave identically — higher even powers (e.g. fourth) give a flatter touch but still a touch.
  • Using calculus (setting f'(x) = 0) unnecessarily when the factorisation already reveals the touch.
