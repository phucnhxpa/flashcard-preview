Translate gradient/derivative inequality into quadratic inequality and solve

Family: Quadratic Inequality Frequency: 1 questions

Trigger — When to use

Fires when a question asks for the set of x-values where the gradient of a curve, or f'(x), satisfies an inequality such as f'(x) < k, f'(x) > k, or f'(x) ≤ 0, and f'(x) is quadratic. Cue words include 'gradient less than', 'rate of change exceeds', 'find the values of x for which the curve is decreasing/increasing', or a 'hence' linking to a previously found derivative. The presence of a numeric bound k (not just 0) signals that you must rearrange before factorising.

Motivation — Why this works

A quadratic inequality is solved by reducing it to the form (quadratic) <> 0, factorising to find the roots, and then reading off the correct interval based on the sign of the leading coefficient. This works because a quadratic with positive leading coefficient is negative strictly between its real roots and positive outside them (and vice versa for negative leading coefficient). Treating the inequality as 'find where the parabola lies below/above the x-axis' converts an algebraic question into a sketchable one.

Thought Process — The step-by-step reasoning

  1. Notices that 'gradient' means f'(x) and that the bound is a non-zero number, not 0, so the inequality must be rearranged before solving.
  2. Rules out testing individual x-values or using calculus stationary-point logic — the structure is a standard quadratic inequality once rearranged.
  3. Commits to the route: move all terms to one side, simplify, factorise, then use the sign of the leading coefficient to pick the correct interval.
  4. Bridges to the first concrete operation by subtracting the bound from both sides to get (quadratic) < 0 (or > 0).
  5. Inside the move, divides out any common numerical factor to ease factorisation, factorises (or uses the quadratic formula) for critical values, then reads the solution interval from the parabola's orientation.

Mechanic — Concrete operations

  1. Identify f'(x) and write the inequality f'(x) < k (or whatever sense is required).
  2. Rearrange to bring all terms to one side: ax² + bx + c < 0 (or > 0).
  3. Simplify by dividing by any common factor, keeping the inequality direction (since the factor is positive).
  4. Factorise the quadratic, or apply the quadratic formula, to find the critical values x = α, β.
  5. Determine the sign of the leading coefficient: if positive, the expression is negative between the roots; if negative, it is positive between the roots.
  6. Write the solution as an interval (e.g. α < x < β) or as a union of intervals (x < α or x > β), matching the inequality.

Worked Micro-Example

Given $f'(x) = 6x^2 - 10x - 34$, find where the gradient is less than $-18$.\ Solve $6x^2 - 10x - 34 < -18 \Rightarrow 6x^2 - 10x - 16 < 0 \Rightarrow 3x^2 - 5x - 8 < 0$.\ Factorise: $(3x - 8)(x + 1) < 0$, so critical values are $x = -1$ and $x = \tfrac{8}{3}$.\ Leading coefficient is positive, so the expression is negative between the roots: $-1 < x < \tfrac{8}{3}$.

Variants — Same move, different surface

Negative leading coefficient quadratic, e.g. f'(x) = -2x² + 3x + 5 with f'(x) > 0
After factorising, the expression is positive between the roots (since the parabola opens downward), so the solution lies between the critical values rather than outside.
Strict vs non-strict inequality (f'(x) ≤ k)
Include the endpoints in the solution: write α ≤ x ≤ β instead of α < x < β.
'Curve is increasing/decreasing' phrasing
Translate 'increasing' to f'(x) > 0 and 'decreasing' to f'(x) < 0; the bound is 0 so no rearrangement needed before factorising.
Quadratic that does not factorise nicely
Use the quadratic formula to find the critical values, then apply the same sign-of-leading-coefficient reasoning.
Inequality involving a tangent gradient or rate condition (mechanics/kinematics)
Replace f'(x) with v(t) or a(t); same method, but state the answer in terms of t with any domain restrictions (e.g. t ≥ 0).

Common Misreadings

  • Treating 'gradient less than -18' as 'gradient is negative' and solving f'(x) < 0 instead of f'(x) < -18.
  • Reading 'less than' as 'less than or equal to' and writing ≤ in the final answer.
  • Confusing 'gradient' with 'y-value' and solving f(x) < k instead of f'(x) < k.
  • Misinterpreting the 'hence' as a hint to use part (a) or (b) rather than the derivative from part (c).

Pitfalls

  • Forgetting to move the constant on the right to the left side before factorising, leading to a wrong quadratic.
  • Choosing the wrong interval: writing x < α or x > β when the leading coefficient is positive and the expression should be negative between the roots.
  • Arithmetic slip when combining constants (e.g. -34 - (-18) = -16, not -52 or -16's sign mishandled).
  • Reversing the inequality sign when dividing — only an issue if dividing by a negative, but students sometimes flip it incorrectly when dividing by a positive common factor.
  • Stating critical values but failing to write the final answer as an inequality/interval.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P1/P2): differentiation of polynomials, quadratic inequalities, and interval notation. No calculus beyond standard polynomial differentiation is required.
