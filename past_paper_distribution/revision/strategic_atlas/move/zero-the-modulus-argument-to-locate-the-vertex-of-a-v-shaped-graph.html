Algebra & Functions

Zero the modulus argument to locate the vertex of a V-shaped graph

Family: Vertex Identification Frequency: 3 questions

Trigger — When to use

The function has the form f(x) = a|bx + c| + d (or equivalently a|x − h| + k), producing a V-shaped graph, and the question asks for the vertex, turning point, minimum point (when a > 0), or maximum point (when a < 0). Visual cues: a single absolute value bracket around a linear expression, plus optional vertical scale factor and vertical shift. Phrasings include 'find the coordinates of the minimum point', 'state the vertex', 'find the turning point of the graph y = …', or 'where does the graph have its sharp point?'. The move fires whether the modulus argument is presented as bx + c, x − h, or contains an unknown constant k.

Motivation — Why this works

The absolute value satisfies |u| ≥ 0 with equality iff u = 0, so the modulus term contributes its minimum (zero) precisely when its argument vanishes. Adding a constant d shifts the graph vertically without changing the x-coordinate of the vertex, and multiplying by a > 0 scales but does not move it. Therefore solving the linear equation 'argument = 0' immediately yields the x-coordinate of the vertex, and substituting back gives y = d. This bypasses calculus (the function isn't differentiable at the vertex anyway) and avoids any case-splitting of the modulus.

Thought Process — The step-by-step reasoning

  1. Notice the structural form: a single |·| around a linear expression, scaled and shifted — this is a V-shape, not a quadratic or a smooth curve.
  2. Rule out calculus (f is not differentiable at the vertex) and rule out completing the square (no x²); also rule out solving f(x) = 0, which would give x-intercepts, not the vertex.
  3. Commit to the modulus-vertex principle: |g(x)| ≥ 0 with equality iff g(x) = 0, so the V's tip sits exactly where the inner expression is zero.
  4. Translate recognition into action: write down the linear equation 'inner expression = 0' and solve for x.
  5. Decide the sign of the leading coefficient a: if a > 0 the point is a minimum, if a < 0 it is a maximum — this affects how the answer is labelled but not the coordinates.
  6. Substitute the x-value back into f to obtain y = d (since the modulus term is now 0), and state the vertex as an ordered pair.

Mechanic — Concrete operations

  1. Identify the inner linear expression g(x) = bx + c inside the modulus.
  2. Set g(x) = 0 and solve: x = −c/b.
  3. Substitute this x into f(x); the modulus term contributes 0, leaving y = d.
  4. Write the vertex as the ordered pair (−c/b, d), and state whether it is a minimum (a > 0) or maximum (a < 0).

Worked Micro-Example

For $f(x) = 2|2x-5| + 3$: set the inner expression $2x-5 = 0 \Rightarrow x = \tfrac{5}{2}$. Then $f\!\left(\tfrac{5}{2}\right) = 2|0| + 3 = 3$. Since the leading coefficient $2 > 0$, the vertex is a minimum at $P = \left(\tfrac{5}{2},\, 3\right)$.

Variants — Same move, different surface

Function already in vertex form y = a|x − h| + k
Read off the vertex directly as (h, k) — same principle, no equation to solve.
Coefficient of x inside the modulus is an unknown letter, e.g. f(x) = |kx − 9| − 2
Solve kx − 9 = 0 to get x = 9/k; the vertex is (9/k, −2), expressed in terms of k.
Negative leading coefficient, e.g. f(x) = −3|x + 1| + 7
Same x-coordinate (x = −1), but the point is a maximum rather than a minimum, with y = 7.
Modulus around a non-linear expression with no real zero
Different technique needed — minimise the inner expression first, since the modulus may not actually attain 0.

Common Misreadings

  • Setting f(x) = 0 (looking for x-intercepts/roots) instead of setting only the inner expression to zero.
  • Confusing the vertex of |bx + c| with the vertex of a quadratic and trying to complete the square.
  • Assuming the y-coordinate of the vertex is always 0, forgetting the vertical shift d.

Pitfalls

  • Sign error when solving bx + c = 0: e.g. 2x + 7 = 0 gives x = −7/2, not +7/2.
  • Forgetting to substitute x back to compute the y-coordinate, leaving the answer as just an x-value.
  • Mis-labelling a maximum as a minimum (or vice versa) when a < 0.
  • Unnecessarily splitting the modulus into two cases or attempting to differentiate, wasting time.

A-level Scope Notes

This is standard Edexcel IAL Pure (P2/P3) modulus-function content; no formula booklet entry is needed. Stay with linear arguments inside |·| — extending to |quadratic| or |trig| modulus graphs is a different problem (the inner function's minimum may not be 0, so the technique above does not directly apply).
