Differentiation

Build Maclaurin/Taylor series by repeated differentiation at the expansion point

Family: Repeated Differentiation Frequency: 1 questions

Trigger — When to use

The question says 'using Maclaurin's theorem' or 'by repeated differentiation', or the function does not decompose into standard series (e^x, sin x, cos x, ln(1+x), (1+x)^n). Also fires when an implicit relation defines y in terms of x and a series about a known point is required. Cues include being asked for terms 'up to and including x^n' on a non-standard f(x), or a starting condition like y = 0 when x = 0.

Motivation — Why this works

The Maclaurin/Taylor formula f(x) = Σ f^(n)(a)(x−a)^n / n! requires the numerical values of successive derivatives at x = a. When no algebraic shortcut exists, direct differentiation is the only route. Evaluating at x = a after each differentiation collapses constants and keeps subsequent derivatives manageable. A disciplined table-style layout prevents the compounding errors that plague higher-order differentiation.

Thought Process — The step-by-step reasoning

  1. I notice the function is not a clean composition of standard series, or the question explicitly mandates Maclaurin's theorem.
  2. I rule out substitution into known expansions and rule out multiplying/composing standard series, since neither applies cleanly here.
  3. I commit to repeated differentiation because the formula needs f(a), f'(a), f''(a), … as coefficients.
  4. I set up a vertical layout: column for f^(n)(x), column for f^(n)(a), and differentiate the previous row rather than the original each time.
  5. I evaluate at x = a immediately after each differentiation to simplify before the next step, and I stop once I have enough derivatives for the requested order.

Mechanic — Concrete operations

  1. Write f(x) and compute f(a).
  2. Differentiate to get f'(x); evaluate f'(a).
  3. Differentiate f'(x) to get f''(x); evaluate f''(a). Repeat to required order.
  4. If the function is implicit, differentiate the defining equation, substitute the known x = a and previously computed derivative values at each stage.
  5. Assemble the series: f(x) ≈ f(a) + f'(a)(x−a) + f''(a)(x−a)^2/2! + f'''(a)(x−a)^3/3! + …
  6. For Maclaurin (a = 0), this reduces to f(0) + f'(0)x + f''(0)x^2/2! + …

Worked Micro-Example

Find the Maclaurin series of f(x) = sec x up to x^4. f(x) = sec x ⇒ f(0) = 1. f'(x) = sec x tan x ⇒ f'(0) = 0. f''(x) = sec x tan^2 x + sec^3 x ⇒ f''(0) = 1. f'''(x) differentiated and evaluated gives f'''(0) = 0. f^{(4)}(0) = 5. Hence sec x ≈ 1 + x^2/2 + 5x^4/24.

Variants — Same move, different surface

Implicit relation, e.g. y satisfies y = x + sin y with y = 0 at x = 0
Differentiate implicitly; at each stage substitute x = 0 and previously found values of y, dy/dx, … to extract d^n y/dx^n at 0.
Differential equation defining y, e.g. dy/dx = x + y^2 with y(0) given
Differentiate the ODE repeatedly to obtain higher derivatives in terms of lower ones, then evaluate at x = 0 in sequence.
Taylor series about a ≠ 0, e.g. expand ln x about x = 1
Same procedure but evaluate each derivative at x = a and use powers of (x − a).
Composite function where standard-series substitution is messy, e.g. f(x) = e^{sin x}
Differentiate directly using chain/product rule; evaluating at 0 collapses sin 0 = 0 and cos 0 = 1, simplifying each f^{(n)}(0).

Common Misreadings

  • Treating the question as a standard-series substitution problem when it explicitly demands 'by differentiating'.
  • Believing only one or two derivatives are needed when the requested order is higher.
  • Confusing Taylor about x = a with Maclaurin about 0 and substituting at the wrong point.

Pitfalls

  • Algebraic errors in one derivative propagate to all later ones; check each before continuing.
  • Omitting the n! denominator when assembling the final series.
  • Missing a product-rule or chain-rule term in higher derivatives, especially with sec x, tan x, or composite exponentials.
  • For implicit work, forgetting that y, y', y'' depend on x and so require chain/product rule when differentiated again.
  • Substituting x = a too late, leaving cumbersome expressions that invite algebra slips.

A-level Scope Notes

Standard Edexcel IAL Further Pure content: Maclaurin and Taylor series via repeated differentiation, including implicit and ODE-defined functions, to a stated finite order.
