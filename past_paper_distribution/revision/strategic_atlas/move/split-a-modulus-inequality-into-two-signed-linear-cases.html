Split a modulus inequality into two signed linear cases

Family: Case Split Frequency: 1 questions

Trigger — When to use

Fires when an inequality (or equation) contains |ax+b| together with another expression in x that is itself linear or constant, so the inequality cannot be reduced to a single linear inequality. Typical cues: phrases like 'solve |2x+7| < 3x+1' or '½|2x+7| − 10 ≥ ⅓x + 1', or any modulus with x on both sides. Also fires when the RHS depends on x (so the simple |u|<k ⇔ −k<u<k shortcut is unsafe). The critical value where the inside switches sign (here x = −7/2) is usually flagged in an earlier part.

Motivation — Why this works

By definition |u| = u when u ≥ 0 and |u| = −u when u < 0, so any inequality in |u| splits into two linear inequalities on complementary domains. Solving each case and intersecting with its domain gives the full solution set without introducing spurious roots. This is preferred over squaring when both sides are linear, because squaring forces a non-negativity check on the RHS and produces an unnecessary quadratic. It is also more rigorous than reading off a sketch when exact answers are required.

Thought Process — The step-by-step reasoning

  1. Notice that the inequality has |2x+7| on one side and a linear expression in x on the other, so the LHS is piecewise linear with a corner at x = −7/2.
  2. Rule out the shortcut |u| < k ⇔ −k < u < k because the RHS depends on x, and rule out squaring because it would create an unneeded quadratic and require a sign check.
  3. Commit to a case-split on the sign of 2x+7, since this turns the problem into two ordinary linear inequalities each valid on a half-line.
  4. Identify the boundary x = −7/2 and define Case 1: x ≥ −7/2 (drop the bars) and Case 2: x < −7/2 (insert a minus sign).
  5. In each case, solve the linear inequality and then intersect the solution with the case's domain; finally take the union of the two restricted solution sets.

Mechanic — Concrete operations

  1. Find the critical value x = −b/a where the inside of the modulus changes sign.
  2. Case 1: assume ax+b ≥ 0, replace |ax+b| with ax+b, and solve the resulting linear inequality; intersect with x ≥ −b/a.
  3. Case 2: assume ax+b < 0, replace |ax+b| with −(ax+b), and solve the resulting linear inequality; intersect with x < −b/a.
  4. If dividing by a negative number to isolate x, flip the inequality sign.
  5. Take the union of the two restricted solution sets and present in interval or inequality form.

Worked Micro-Example

Solve $|2x+7| < x+5$. Critical value: $x=-7/2$. Case 1 ($x\ge -7/2$): $2x+7 < x+5 \Rightarrow x < -2$, intersect to get $-7/2 \le x < -2$. Case 2 ($x<-7/2$): $-(2x+7) < x+5 \Rightarrow -3x < 12 \Rightarrow x > -4$, intersect to get $-4 < x < -7/2$. Union: $-4 < x < -2$.

Variants — Same move, different surface

Modulus equation $|f(x)| = g(x)$ with $f$ linear
Replace inequalities with equalities: solve $f(x)=g(x)$ and $f(x)=-g(x)$, then check each root lies in the appropriate domain (and that $g(x)\ge 0$).
Two moduli on the same side, e.g. $|2x+7| \ge |x-1|$
Either split into four sign cases on the two interior expressions (using both critical values to partition the line), or square both sides since both are non-negative.
Nested modulus such as $||x-1|-3| = k$
Apply the case split iteratively: outer case split first, then inner, recording the domain at each stage.
Inequality of the form $|ax+b| < k$ or $|ax+b| > k$ with $k$ constant
Use the shortcut $-k<ax+b<k$ (or $ax+b>k$ or $ax+b<-k$) instead of a full case split — equivalent but quicker.
Graphical/transformations question asking for the range where $y=|2x+7|$ lies above a line
Use the case split to find intersection x-coordinates, then read the required region off the V-shape sketch.

Common Misreadings

  • Solving only the positive case and reporting that as the full solution.
  • Treating $|2x+7| \ge \text{(expression in }x)$ as $2x+7 \ge \text{RHS}$ OR $2x+7 \le -\text{RHS}$ without intersecting with the case domains.
  • Forgetting to intersect each case's solution with its defining domain $x \ge -7/2$ or $x < -7/2$.
  • Confusing the critical value: writing $x = 7/2$ instead of $x = -7/2$ for $|2x+7|$.

Pitfalls

  • Sign errors when distributing the minus in Case 2, e.g. $\tfrac12(-(2x+7)) = -x - 7/2$, not $-x+7/2$.
  • Forgetting to flip the inequality when dividing by a negative coefficient.
  • Reporting the union as separate inequalities when they actually merge into a single interval (or vice versa).
  • Including endpoints incorrectly: strict vs non-strict at the critical value $x=-7/2$.

A-level Scope Notes

Standard Edexcel IAL Pure (P3) modulus material: linear-inside-modulus equations and inequalities solved by case split, with exact answers and interval/inequality presentation.
