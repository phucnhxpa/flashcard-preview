Coordinate Geometry

Translate region's position relative to each boundary into directed inequalities

Family: Region Inequality Assignment Frequency: 3 questions

Trigger — When to use

The question asks you to 'define the region R using inequalities' or 'write down the set of inequalities that define R', and a figure shows a shaded region bounded by some combination of curves, lines, and axes. Each boundary needs a direction (≥/≤ or ≥/≤ on x) decided from the diagram. Equivalent phrasings: 'find the inequalities satisfied by points in R', 'state the three inequalities defining the shaded region'. The cue is: boundary equations are already known (often from earlier parts), and only the inequality SIGN is in question.

Motivation — Why this works

Every boundary curve y = f(x) or vertical line x = k splits the plane into two half-regions; the shaded region R is the intersection of one half from each boundary. Algebraically, 'above y = f(x)' is exactly y ≥ f(x) and 'below' is y ≤ f(x), regardless of whether the curve opens up, down, or is straight. By assigning each boundary the correct direction, the geometric description of R is converted into a system of simultaneous inequalities whose solution set is precisely R. This works because of the basic principle that y - f(x) changes sign across the curve y = f(x).

Thought Process — The step-by-step reasoning

  1. Notice the task is purely a translation problem: the boundary equations are given or already derived, so I just need to fix the SIGN of each inequality and decide strict vs non-strict.
  2. Rule out ad-hoc methods (e.g. parameterising or solving simultaneously) — the standard tool here is one inequality per boundary, intersected.
  3. Commit to the rule: for any non-vertical boundary y = f(x), points above satisfy y ≥ f(x) and points below satisfy y ≤ f(x); for vertical x = k, right means x ≥ k and left means x ≤ k. This rule is independent of whether f is linear, a parabola opening up, or a parabola opening down.
  4. For each boundary, locate R relative to it by eye, or pick a clearly-interior test point and substitute to check which side of f the point is on.
  5. Decide strict vs non-strict: solid boundary lines in Edexcel diagrams (the default) include the boundary, so use ≤ or ≥; dashed lines would indicate < or >.
  6. Write all the inequalities together — R is their simultaneous solution, so missing any one loses the region.

Mechanic — Concrete operations

  1. List every boundary equation in the form y = f(x) or x = k (or y = c).
  2. For each non-vertical boundary, decide whether R sits above (y ≥ f(x)) or below (y ≤ f(x)) by inspecting the figure or testing an interior point.
  3. For each vertical boundary, decide whether R lies to the right (x ≥ k) or left (x ≤ k).
  4. Check inclusion: if the boundary is drawn solid (standard), use ≥ or ≤; if dashed, use > or <.
  5. Present the inequalities as a simultaneous system — all must hold at once.

Worked Micro-Example

Suppose R is bounded above by line $y = 5x + 75$, below by parabola $y = x^2 + 2x - 21$, on the left by $x = -6$, and below by the x-axis $y = 0$. Test point $(0,0)$ inside R: $0 \le 5(0)+75$ ✓, $0 \ge 0^2+0-21 = -21$ ✓, $0 \ge -6$ ✓, $0 \ge 0$ ✓. Hence R is defined by $y \le 5x+75$, $y \ge x^2+2x-21$, $x \ge -6$, $y \ge 0$.

Variants — Same move, different surface

Circular boundary $(x-a)^2 + (y-b)^2 = r^2$
Inside the circle gives $(x-a)^2+(y-b)^2 \le r^2$; outside reverses the sign.
Region 'between' two curves with $f(x) \le g(x)$ on R
Use the pair $y \ge f(x)$ and $y \le g(x)$ simultaneously.
Dashed boundary in the figure
Replace ≥/≤ by strict >/< for that boundary only.
Boundary is a horizontal line $y = c$
Above gives $y \ge c$, below gives $y \le c$ — same logic with f(x) = c.

Common Misreadings

  • Reversing the inequality for a downward-opening parabola, thinking 'inside the arch' must be y ≥ f(x); the rule is purely about y vs f(x), not about the curve's shape.
  • Confusing which side of a negatively-sloped line counts as 'above' — always compare the y-coordinate of a test point to f(x) at the same x.
  • Defining the complement of R by mistaking the unshaded region for the shaded one.

Pitfalls

  • Forgetting one of the boundaries (especially the x-axis or a vertical edge) — the system is incomplete and marks are lost.
  • Using strict inequalities (<, >) when boundaries are solid lines, which Edexcel marks as incorrect boundary inclusion.
  • Writing x ≤ f(y) style inequalities when the boundary is given as y = f(x) — keep the inequality in the variable that matches the boundary's natural form.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (linear programming / region-defining questions in P1–P3 and FP). No formula booklet entry required — purely a translation skill. Avoid generalising to multivariable or 3D regions, which are out of scope.
