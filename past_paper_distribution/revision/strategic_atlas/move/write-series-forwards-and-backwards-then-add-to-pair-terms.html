Proof

Write series forwards and backwards then add to pair terms

Family: Reverse And AddFrequency: 1 questions
Use when
A 'prove from first principles' request for the arithmetic series sum formula $S_n = \tfrac{n}{2}[2a + (n-1)d]$ (or the equivalent $S_n = \tfrac{n}{2}(a+L)$).
The move
$\;(1)+(2):\; 2S_n = \underbrace{[2a+(n-1)d] + \cdots + [2a+(n-1)d]}_{n} = n[2a+(n-1)d] \Rightarrow S_n = \tfrac{n}{2}[2a+(n-1)d]\;$
Minimal example
$\;S_n = a + (a+d) + \dots + [a+(n-1)d]\quad(1);\quad S_n = [a+(n-1)d] + \dots + a\quad(2)\;$  →  $\;(1)+(2):\; 2S_n = \underbrace{[2a+(n-1)d] + \cdots + [2a+(n-1)d]}_{n} = n[2a+(n-1)d] \Rightarrow S_n = \tfrac{n}{2}[2a+(n-1)d]\;$

Forgetting to state that the last term is $a+(n-1)d$ via the $n$-th term formula.

Trigger — When to use

A 'prove from first principles' request for the arithmetic series sum formula $S_n = \tfrac{n}{2}[2a + (n-1)d]$ (or the equivalent $S_n = \tfrac{n}{2}(a+L)$). Also fires whenever a finite sum has the symmetry $u_k + u_{n+1-k} = \text{const}$, e.g. summing $1+2+\dots+n$ or any AP-style list. Cue phrases: 'prove that', 'show that', 'derive the formula for $S_n$'.

Motivation — Why this works

Adding the forward sum to its reverse pairs the $k$-th term with the $(n+1-k)$-th term; in an AP these pair sums are all equal to $2a+(n-1)d$. This converts $n$ distinct terms into $n$ identical pair-sums, so the total collapses to a single product $n[2a+(n-1)d]$. The principle exploited is the equidistant-pair symmetry of an arithmetic progression — the structural fact that makes the closed form exist at all.

Thought Process — Step-by-step reasoning

  1. Notice the question says 'prove' or 'derive', so quoting the formula is not allowed — a first-principles argument is required.
  2. Rule out induction: it is valid but longer, less elegant, and on Edexcel IAL the standard expected proof is the reverse-and-add derivation.
  3. Commit to reverse-and-add because the AP's defining property — constant common difference — guarantees that $u_k + u_{n+1-k}$ is constant, the exact symmetry this trick needs.
  4. Bridge to action: write $S_n$ once in increasing order using $a, a+d, \dots, a+(n-1)d$, then write it again in decreasing order, labelling them (1) and (2).
  5. Tactically, when adding term-by-term, make the pairing explicit: state the common pair-sum is $2a+(n-1)d$ and that there are exactly $n$ pairs before dividing by 2.

Mechanic — Concrete operations

  1. Write $S_n = a + (a+d) + (a+2d) + \dots + (a+(n-1)d)$ — label (1).
  2. Reverse the order: $S_n = (a+(n-1)d) + (a+(n-2)d) + \dots + a$ — label (2).
  3. Add (1) and (2) term-by-term; each of the $n$ paired sums equals $2a+(n-1)d$.
  4. Conclude $2S_n = n[2a+(n-1)d]$.
  5. Divide by 2 to obtain $S_n = \tfrac{n}{2}[2a+(n-1)d]$.

Worked Micro-Example

Prove $S_n = \tfrac{n}{2}[2a+(n-1)d]$. (1) $S_n = a + (a+d) + \dots + (a+(n-1)d)$. (2) $S_n = (a+(n-1)d) + (a+(n-2)d) + \dots + a$. Adding: $2S_n = \underbrace{[2a+(n-1)d] + \dots + [2a+(n-1)d]}_{n\text{ terms}} = n[2a+(n-1)d]$. Hence $S_n = \tfrac{n}{2}[2a+(n-1)d]$. $\blacksquare$

Variants — Same move, different surface

Prove $S_n = \tfrac{n}{2}(a+L)$ where $L$ is the last term
Same proof, but keep the last term as $L$ throughout; each pair sums to $a+L$, giving $2S_n = n(a+L)$.
Sum $1+2+3+\dots+n$ (triangular numbers)
Special case with $a=1, d=1$; pair-sum is $n+1$, yielding $\tfrac{n(n+1)}{2}$.
Gauss's $1+2+\dots+100$
Numerical instance: 50 pairs each summing to 101, giving 5050.
Sum of an AP given first term $a$ and last term $L$ but $d$ unknown
Use the $(a+L)$ form directly — $d$ is not needed if both endpoints are stated.

Common Misreadings

  • Treating the question as 'use the formula' rather than 'prove the formula', so quoting $S_n=\tfrac{n}{2}[2a+(n-1)d]$ without derivation.
  • Attempting induction when the mark scheme expects the direct reverse-and-add derivation.
  • Writing only a few terms with '...' on each side without explicitly identifying the constant pair-sum.

Pitfalls

  • Forgetting to state that the last term is $a+(n-1)d$ via the $n$-th term formula.
  • Failing to say there are exactly $n$ pairs — examiners deduct for this omission.
  • Using $L$ for the last term but never linking $L = a+(n-1)d$, leaving the proof incomplete.
  • Adding only some pairs and writing '$\dots$' without justifying that every pair gives the same sum.
  • Dropping the factor of 2 when dividing, or writing $\tfrac{n}{2}[2a+nd]$ instead of $\tfrac{n}{2}[2a+(n-1)d]$.

A-level Scope Notes

This is the standard Edexcel IAL Pure (P2) proof of the arithmetic series sum formula; only AP notation $a, d, n, L$ and basic algebra are used, fully within syllabus.
