Trigger — When to use

Fires whenever a question explicitly says 'use linear interpolation' (or 'method of false position') on a stated interval [a, b] where f(a) and f(b) have opposite signs. Visual cues: a small interval such as [1.4, 1.5], [2.8, 2.9], [4.8, 4.9], often following a part where a sign change has been verified. Alternative phrasings include 'use linear interpolation on the values at the end points', 'find an estimate for the root α correct to 3 d.p.', or 'use linear interpolation once on the interval'. Distinguish carefully from Newton-Raphson (which uses a tangent and the derivative) and from interval bisection (which only uses signs).

Motivation — Why this works

Linear interpolation replaces the curve y = f(x) with the straight chord joining (a, f(a)) and (b, f(b)) and takes the chord's x-intercept as the root estimate. By similar triangles, the horizontal distance from a to the estimate is proportional to |f(a)| relative to |f(a)| + |f(b)|, so the estimate sits closer to whichever endpoint has the smaller |f|. This is more efficient than bisection because it weights by function values rather than just signs, and it is geometrically transparent — no derivative needed. Algebraically it is equivalent to the secant formula β = a − f(a)(b − a)/(f(b) − f(a)).

Thought Process — Step-by-step reasoning

  1. Notice the question explicitly demands linear interpolation on a given interval [a, b], with f(a) and f(b) of opposite signs already established (or to be computed).
  2. Rule out alternatives: bisection would just take the midpoint and ignore the f-values; Newton-Raphson would need f'(x) and a single starting point — neither matches the wording.
  3. Commit to linear interpolation because (i) the question says so, (ii) two bracketing values are provided, and (iii) the mark scheme expects an explicit similar-triangles or weighted-average setup.
  4. Bridge to operations: evaluate f(a) and f(b) to high precision (in radians if trig is involved), record their signs, and take absolute values for the proportion.
  5. Tactical decision: write the proportion as (β − a)/|f(a)| = (b − β)/|f(b)| (similar triangles) — the layout most aligned with mark schemes — or equivalently use the weighted average β = (a|f(b)| + b|f(a)|)/(|f(a)| + |f(b)|).
  6. Sense-check: the estimate must lie strictly between a and b, and closer to whichever endpoint has the smaller |f|.

Mechanic — Concrete operations

  1. Compute f(a) and f(b) accurately; confirm opposite signs.
  2. Let p = |f(a)| and q = |f(b)|.
  3. Set up the similar-triangles proportion: (β − a)/p = (b − β)/q.
  4. Cross-multiply: q(β − a) = p(b − β), giving β(p + q) = a·q + b·p.
  5. Solve: β = (a·q + b·p)/(p + q), equivalently β = a + p(b − a)/(p + q).
  6. Substitute the numerical p, q and simplify to the required number of decimal places (typically 3 d.p.).

Worked Micro-Example

Estimate the root of f(x) = 0 in [2.8, 2.9] given f(2.8) = 0.1420, f(2.9) = −0.8486. Set (α − 2.8)/0.1420 = (2.9 − α)/0.8486, so α = (2.8·0.8486 + 2.9·0.1420)/(0.1420 + 0.8486) = 2.78048/0.9906 ≈ 2.814 (3 d.p.). The estimate lies near 2.8, consistent with |f(2.8)| ≪ |f(2.9)|.

Variants — Same move, different surface

Function involves tan, sin, cos (e.g. g(x) = 4x² + x − tan x on [1.4, 1.5])
Ensure calculator is in radians; otherwise tan(1.5°) ≈ 0.026 instead of tan(1.5 rad) ≈ 14.101, ruining the calculation.
Question asks for two iterations of linear interpolation
After computing β₁, evaluate f(β₁), discard whichever original endpoint has the same sign as f(β₁), and repeat the proportion on the new bracketing pair (regula falsi).
Endpoints are integers like [4, 5] rather than tenths
Same formula β = a + |f(a)|(b − a)/(|f(a)| + |f(b)|); the interval width (b − a) just changes from 0.1 to 1.
Weighted-average phrasing preferred
Use β = (a|f(b)| + b|f(a)|)/(|f(a)| + |f(b)|) directly — avoids cross-multiplication arithmetic.

Common Misreadings

  • Confusing linear interpolation with Newton-Raphson — the former uses a chord between two points, the latter a tangent at one point.
  • Confusing it with interval bisection, which ignores f-values and just takes the midpoint.
  • Reading the proportion as (β − a)/|f(b)| = (b − β)/|f(a)| (swapping numerators), which mismatches the similar triangles.

Pitfalls

  • Calculator in degrees instead of radians when f involves trig functions — catastrophic for tan near π/2.
  • Using signed values instead of absolute values, producing an estimate outside [a, b] or with wrong sign.
  • Getting the proportion upside-down, placing the estimate closer to the endpoint with larger |f| rather than smaller.
  • Premature rounding of f(a), f(b) before the final division — keep at least 4 s.f. through the calculation.
  • Arithmetic slip in cross-multiplication; always sense-check that a < β < b.

A-level Scope Notes

Within Edexcel IAL Further Pure FP1 (numerical methods for locating roots). The formula is not in the formula booklet — students must reproduce the similar-triangles derivation or memorise β = a + |f(a)|(b−a)/(|f(a)|+|f(b)|). Iterated regula falsi and convergence analysis are out of scope; only one or two applications are expected.
