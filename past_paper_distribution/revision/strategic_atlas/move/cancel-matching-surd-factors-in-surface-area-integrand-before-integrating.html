Integration

Cancel matching surd factors in surface-area integrand before integrating

Family: Integrand Collapse Frequency: 1 questions

Trigger — When to use

Fires in surface-area-of-revolution problems where y itself contains a surd that also appears in the arc-length factor √(1 + (dy/dx)²). Concrete cue: after substituting y and simplifying 1 + (dy/dx)², the same surd √(a² − x²) (or similar) appears in both numerator (from y) and denominator (from the arc-length factor). Equivalently, when the curve is a circle/ellipse arc, the product y·√(1+(dy/dx)²) often collapses to a constant or a simple polynomial. Always multiply y into the arc-length factor before reaching for substitution.

Motivation — Why this works

For a circle x² + y² = r², differentiating gives dy/dx = −x/y, so 1 + (dy/dx)² = (x² + y²)/y² = r²/y², hence √(1 + (dy/dx)²) = r/y. The product y · r/y = r is constant — this reflects the fact that a sphere has uniform surface density along its axis (Archimedes' hat-box theorem). Recognising this collapse turns a seemingly hard integral into S = 2π ∫ r dx, a one-line computation. The same mechanism produces clean answers whenever the curve has a quadratic-in-y implicit form.

Thought Process — The step-by-step reasoning

  1. I notice y = √(9 − x²) and the arc-length factor evaluates to 3/√(9 − x²) — the same surd appears top and bottom.
  2. I rule out trig substitution x = 3 sin θ: it would work but is unnecessary once the cancellation is spotted.
  3. I commit to algebraic simplification first because the matching surds guarantee a clean reduction; the geometric sanity check (hemisphere area 2πr²) confirms it.
  4. I write the full integrand y · √(1 + (dy/dx)²) explicitly, substitute both expressions, and cancel.
  5. Tactically I keep the 2π prefactor outside the integral and only insert numerical limits at the very end.

Mechanic — Concrete operations

  1. From y² = 9 − x², differentiate implicitly: 2y(dy/dx) = −2x, so dy/dx = −x/y.
  2. Compute 1 + (dy/dx)² = 1 + x²/y² = (y² + x²)/y² = 9/(9 − x²).
  3. Take the square root: √(1 + (dy/dx)²) = 3/√(9 − x²).
  4. Form the integrand y · 3/√(9 − x²) = √(9 − x²) · 3/√(9 − x²) = 3.
  5. Evaluate S = 2π ∫₀³ 3 dx = 2π · 9 = 18π.

Worked Micro-Example

Rotate y = √(9 − x²), 0 ≤ x ≤ 3, about the x-axis. Then dy/dx = −x/√(9 − x²), so 1 + (dy/dx)² = 9/(9 − x²). Hence S = 2π ∫₀³ √(9 − x²) · 3/√(9 − x²) dx = 2π ∫₀³ 3 dx = 18π, matching the hemisphere formula 2πr².

Variants — Same move, different surface

Full sphere from y = √(r² − x²) on [−r, r]
Same cancellation gives integrand r; S = 2π ∫_{−r}^{r} r dx = 4πr².
Curve given parametrically as x = r cos t, y = r sin t
Use S = 2π ∫ y √((dx/dt)² + (dy/dt)²) dt; the bracket simplifies to r, again giving y · r and a clean integral.
Catenary y = cosh x rotated about x-axis
Here 1 + (dy/dx)² = cosh²x, so √(...) = cosh x and y·cosh x = cosh²x — collapses to ½(1 + cosh 2x), still a clean integrand though not constant.
Ellipse arc y = b√(1 − x²/a²)
The surds no longer match exactly because of the a, b mismatch; expect partial simplification rather than full cancellation.

Common Misreadings

  • Leaving y as a symbol rather than substituting √(9 − x²), so the cancellation is invisible.
  • Treating the problem as a volume of revolution (πy² instead of 2πy√(1+(dy/dx)²)) and missing the surface-area structure entirely.
  • Reaching immediately for x = 3 sin θ before simplifying algebraically.

Pitfalls

  • Cancelling √(9 − x²) when the two occurrences are not at the same algebraic level (e.g. one squared, one not).
  • Forgetting the 2π prefactor in the surface-area formula.
  • Sign error in dy/dx leading to a wrong 1 + (dy/dx)² before squaring saves you.
  • Using limits 0 to 3 but reporting half the value when the question asked for a full sphere (or vice versa).

A-level Scope Notes

Surface area of revolution S = 2π ∫ y √(1 + (dy/dx)²) dx is in Edexcel IAL Further Pure; the algebraic cancellation and verification against 2πr² for a hemisphere are entirely within syllabus.
