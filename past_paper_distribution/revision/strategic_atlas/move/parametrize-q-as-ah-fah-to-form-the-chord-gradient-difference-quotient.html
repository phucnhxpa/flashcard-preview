Differentiation

Parametrize Q as (a+h, f(a+h)) to form the chord-gradient difference quotient

Family: First Principles Setup Frequency: 2 questions

Trigger — When to use

The question gives a fixed point P(a, f(a)) on a curve and defines a second point Q on the same curve with x-coordinate (a + h), then asks for the gradient of the chord PQ in terms of h. Equivalent phrasings include 'Q has x-coordinate a + h' or 'Q is the point on the curve where x = a + δx'. The structure is the classical setup for differentiation from first principles: a known anchor point, a nearby variable point parametrised by a small offset, and a request for the chord gradient.

Motivation — Why this works

Writing Q = (a + h, f(a + h)) turns the chord gradient (y₂ − y₁)/(x₂ − x₁) into the difference quotient [f(a + h) − f(a)] / h. This is the algebraic skeleton of the derivative: by design the constant terms in the numerator cancel, leaving an expression with h as a factor that can be cleanly cancelled with the h in the denominator. The simplification produces a form (typically polynomial in h) whose limit as h → 0 is the tangent gradient, linking secant gradient to derivative.

Thought Process — The step-by-step reasoning

  1. Notice that P and Q both lie on the curve y = f(x), so Q's y-coordinate is forced: y_Q = f(a + h). The only free parameter is the x-offset h.
  2. Rule out alternative tools: the derivative formula dy/dx is not what is being asked for here (that would skip the first-principles structure the examiner is testing); estimating numerically or sketching is also inappropriate because an exact expression in h is required.
  3. Commit to the chord-gradient formula m = (y₂ − y₁)/(x₂ − x₁), recognising it as the difference-quotient template [f(a+h) − f(a)] / h with a explicit.
  4. Bridge to action: substitute x = a + h into every occurrence of x in the curve equation to compute y_Q, expanding (a + h)² (or other h-expressions) carefully with all cross terms.
  5. Tactical step: subtract y_P (a known constant) from y_Q — the constant terms must cancel; if they don't, an algebra error has occurred. Then divide by (a + h) − a = h, which factorises out cleanly.
  6. Leave the answer in simplified form (e.g. h + 9 or 8 + 2h), ready for the limit step in the next part of the question.

Mechanic — Concrete operations

  1. Identify f(a) by substituting x = a into the curve: y_P = f(a).
  2. Compute y_Q = f(a + h) by substituting x = a + h, expanding all powers (e.g. (a + h)² = a² + 2ah + h²) without dropping cross-terms.
  3. Form the chord gradient: m_PQ = [f(a + h) − f(a)] / [(a + h) − a] = [f(a + h) − f(a)] / h.
  4. Cancel the constant terms in the numerator (they will vanish since f(a) is subtracted), leaving an expression with h as a common factor.
  5. Divide numerator and denominator by h to obtain the simplified chord gradient in terms of h.

Worked Micro-Example

Curve y = x² + 3x − 2, P = (3, 16), Q has x-coordinate 3 + h. Then y_Q = (3+h)² + 3(3+h) − 2 = h² + 9h + 16. Chord gradient: m_{PQ} = \frac{(h^2 + 9h + 16) - 16}{(3+h) - 3} = \frac{h^2 + 9h}{h} = h + 9.

Variants — Same move, different surface

Curve given as y = 1/x or other rational form
Substitution still uses x = a + h, but algebra requires a common denominator: f(a+h) − f(a) = [a − (a+h)] / [a(a+h)] = −h / [a(a+h)], so the h cancels rationally rather than polynomially.
Question uses δx instead of h
Identical logic: write Q = (a + δx, f(a + δx)) and form [f(a + δx) − f(a)] / δx. The notation change does not alter the method.
General point x = a left as a variable
Repeat with general a to derive [f(a+h) − f(a)] / h as a function of both a and h, suitable for a full first-principles derivation of f'(x).

Common Misreadings

  • Treating h as the y-increment (the rise) rather than the x-increment (the run).
  • Treating h as a fixed numerical value rather than a variable approaching zero, which undermines the eventual limit step.
  • Forgetting that Q is constrained to lie on the curve and trying to leave y_Q as a free variable.

Pitfalls

  • Expansion error in (a + h)²: writing a² + h² and omitting the cross-term 2ah.
  • Sign error when subtracting y_P from y_Q, especially if y_P is negative.
  • Forgetting to substitute (a + h) into every occurrence of x in the curve equation (e.g. substituting in the x² term but leaving the linear 3x term untouched).
  • Cancelling h prematurely or incorrectly when the numerator has not yet been fully simplified.

A-level Scope Notes

This sits in Edexcel IAL Pure Mathematics P1/P2 (differentiation from first principles). No formula booklet entry is invoked — the difference quotient must be written out by hand. Avoid invoking ε–δ definitions or non-standard analysis; the IAL treatment uses the informal 'as h → 0' limit only.
