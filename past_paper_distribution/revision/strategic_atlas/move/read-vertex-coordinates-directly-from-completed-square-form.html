Trigger — When to use

The quadratic has already been written (or just been written in a previous part) in completed-square form a(x−p)² + q, and the question asks for the minimum/maximum point, vertex, turning point, or the largest/smallest value of y. Cue words include 'hence write down', 'state the coordinates of the minimum/maximum point', or part (b) following a 'complete the square' part (a). Equivalently, the question may ask for the range of f or the value of x at which f is least/greatest. These all signal that no calculus is needed — the structural form is the tool.

Motivation — Why this works

For a real expression, (x − p)² ≥ 0 always, with equality iff x = p. Hence in y = a(x − p)² + q, the term a(x − p)² is bounded below by 0 (if a > 0) or above by 0 (if a < 0), forcing y ≥ q or y ≤ q respectively. The extremum is attained exactly when (x − p)² = 0, giving the vertex (p, q) by inspection. This bypasses differentiation and the formula x = −B/(2A), and is the algebraic identity the examiner has set up.

Thought Process — Step-by-step reasoning

  1. Notice the quadratic is already in the form a(x − p)² + q, either given or produced in a preceding part — and the question now asks for the turning point.
  2. Consider alternatives: differentiating and solving dy/dx = 0, or using x = −B/(2A) on the expanded form. Both work but duplicate effort and ignore the 'hence' structure of the question — examiners expect a direct read-off.
  3. Commit to the structural argument: (x − p)² ≥ 0 with equality at x = p, so the squared term contributes 0 at its extremum and y = q there.
  4. Check the sign of a to classify: a > 0 gives a minimum, a < 0 gives a maximum. State this explicitly because examiners often want to see the parabola's orientation justified.
  5. Read off carefully: from a(x − p)² + q, the x-coordinate is +p (the value that makes the bracket zero), NOT −p. Verify by plugging x = p back in to confirm y = q.
  6. Write the vertex as the ordered pair (p, q), not just the y-value, since the question asks for a 'point'.

Mechanic — Concrete operations

  1. Identify the sign of a: if a > 0, vertex is a minimum; if a < 0, vertex is a maximum.
  2. Use (x − p)² ≥ 0 ⟹ a(x − p)² is bounded by 0 in the appropriate direction ⟹ y is bounded by q.
  3. Set the squared bracket to zero: x − p = 0 ⟹ x = p.
  4. Substitute back: y = a(0) + q = q.
  5. State the turning point as the coordinate pair (p, q).

Worked Micro-Example

Given $y = \tfrac{1}{2}(x-10)^2 - 28$: since $a = \tfrac{1}{2} > 0$ the curve has a minimum. As $(x-10)^2 \geq 0$ with equality at $x=10$, the smallest $y$ is $\tfrac{1}{2}(0) - 28 = -28$. Hence the minimum point is $(10, -28)$.

Variants — Same move, different surface

Coefficient a < 0, e.g. y = 16 − 3(x − 2)²
The squared term now subtracts from the constant, so y ≤ 16 with equality at x = 2 — same read-off, but classify as a maximum point (2, 16).
Form is a(x + p)² + q (plus sign inside bracket)
Set x + p = 0 ⟹ x = −p; vertex is (−p, q). Easy sign trap.
Disguised squared expression, e.g. f(x) = 2sin²(x) + 3 or a square in another variable
Same non-negativity argument: sin²(x) ≥ 0 with equality at x = 0, π, …, giving minimum value 3.
Question asks for the range of f rather than the vertex
Translate the bound directly: a > 0 ⟹ range is f(x) ≥ q; a < 0 ⟹ f(x) ≤ q.

Common Misreadings

  • Reading (x − p)² and writing the vertex x-coordinate as −p instead of +p (or vice versa for (x + p)²).
  • Reporting only the y-value (e.g. '−9') when the question asks for the 'minimum point', losing the coordinate mark.
  • Confusing minimum and maximum because the sign of a was not checked.

Pitfalls

  • Sign error on the x-coordinate: from a(x + p)² + q the vertex x is −p, not +p; from a(x − p)² + q the vertex x is +p.
  • Forgetting to justify with a > 0 (or a < 0) — examiners often want the orientation stated to confirm minimum vs maximum.
  • If the question says 'find' rather than 'write down', a one-line justification using (x − p)² ≥ 0 is expected; pure assertion may lose a method mark.
  • Differentiating unnecessarily after a 'hence' instruction wastes time and ignores the intended structure.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P1/P2) — completing the square and quadratic graphs. No formula booklet entry is required; this is a structural read-off. Do not generalise to cubics or higher: the non-negativity argument only works because squares are involved.
