Algebra & Functions

Rewrite all powers in a common prime base to unlock index laws

Family: Base Unification Frequency: 3 questions

Trigger — When to use

The expression or equation contains two or more numerical bases that are powers of the same prime — typically combinations like {81, 9, 3}, {8, 4, 2}, {25, 5}, or {27, 9, 3}. Index laws (a^m·a^n = a^{m+n}, a^m/a^n = a^{m-n}, (a^m)^n = a^{mn}) only chain when bases are identical, so the visual cue is multiple distinct bases that you can mentally factor into the same prime. Equivalent phrasings: 'I see different numbers but they all sit on the same prime tower', or 'one base is a power of the other'. This move also fires when a substitution like y = 3^x or u = 2^x is anticipated and demands a uniform base.

Motivation — Why this works

The fundamental index laws only operate when bases match, so converting every term to a single prime base p turns a heterogeneous expression into one manipulable exponential. This achieves three things: (i) numerator/denominator can be combined via a^m/a^n = a^{m-n}; (ii) hidden polynomial structure such as (p^x)^2 alongside p^x becomes visible, enabling a quadratic substitution; (iii) arithmetic with awkward large numbers collapses into clean exponent arithmetic. The underlying principle is simply the laws of indices applied with the relation p^{ab} = (p^a)^b to bridge between bases.

Thought Process — The step-by-step reasoning

  1. I notice the expression mixes numerical bases (e.g. 81 and 9, or 4 and 2, or 8 and 2) that I cannot directly combine because index laws need a shared base.
  2. I rule out taking logs immediately — that fails when the equation has sums/differences of exponentials (log doesn't distribute over +) — and I rule out brute numerical evaluation, which produces messy arithmetic.
  3. I commit to base-unification because every base in sight is a power of one prime p; rewriting everything as p^{something} will make all index laws applicable in one stroke.
  4. Concretely, I factor each numerical base: 81 = 3^4, 9 = 3^2, 8 = 2^3, 4 = 2^2, etc., and apply (p^a)^{exp} = p^{a·exp} carefully — distributing the outer exponent across the WHOLE inner exponent.
  5. Tactically, I choose the SMALLEST prime base available (base 3 rather than base 9, base 2 rather than base 4) so a planned substitution like y = p^x lands cleanly without further conversion.
  6. Finally I combine using a^m/a^n = a^{m-n} and, if a quadratic structure is expected, rewrite p^{2x} as (p^x)^2 to expose it.

Mechanic — Concrete operations

  1. Identify the common prime p such that every numerical base equals p^k for some integer k (e.g. 81 = 3^4, 9 = 3^2, or 8 = 2^3, 4 = 2^2).
  2. Substitute each base: replace c by p^k where c = p^k.
  3. Apply power-of-a-power: (p^k)^{f(x)} = p^{k·f(x)}, distributing k across every term inside f(x).
  4. Apply quotient/product laws: p^A · p^B = p^{A+B} and p^A / p^B = p^{A-B}, using brackets carefully when subtracting compound exponents.
  5. If a substitution is intended, rewrite p^{nx} as (p^x)^n so that y = p^x converts the equation to a polynomial in y.

Worked Micro-Example

Simplify $\dfrac{81}{9^{2-3x}}$. Rewrite: $81 = 3^4$ and $9^{2-3x} = (3^2)^{2-3x} = 3^{4-6x}$. Then $\dfrac{3^4}{3^{4-6x}} = 3^{4-(4-6x)} = 3^{6x} = (3^x)^6$. Substituting $y = 3^x$ gives $y^6$.

Variants — Same move, different surface

Equation with 4^{x-1} and 2^x (quadratic-in-disguise)
Write 4 = 2^2 so 4^{x-1} = 2^{2x-2} = (2^x)^2 / 4; substitute y = 2^x to get a quadratic in y.
Binomial expansion term like $2^{10-r}/8^r$
Write 8 = 2^3, so 8^r = 2^{3r}, giving $2^{10-r}/2^{3r} = 2^{10-4r}$ — a single clean power of 2.
Equations involving 9^x and 3^x, or 25^x and 5^x
Use 9 = 3^2 (or 25 = 5^2) to convert to a single base, then substitute y = 3^x (or y = 5^x) to obtain a quadratic.
Mixed bases like 8^x and 4^x
Both reduce to base 2 (8 = 2^3, 4 = 2^2); pick base 2 to unify.

Common Misreadings

  • Choosing a non-minimal common base (e.g. writing 81 = 9^2 to land in base 9) — this works numerically but forces an extra conversion before any base-3 substitution.
  • Trying to take logs of both sides of an equation containing a sum or difference of exponentials — log does not distribute over addition, so this stalls.
  • Failing to recognise that a numerical base is a power of a prime already present (e.g. not seeing 8 as 2^3) and resorting to heavy arithmetic.

Pitfalls

  • Mis-distributing the outer exponent: writing (3^2)^{2-3x} = 3^{4-3x} instead of 3^{4-6x} — every term inside must be multiplied by the outer 2.
  • Sign/bracket error in the quotient law: 4 − (4 − 6x) = 6x, not −6x; always bracket the subtracted exponent.
  • Forgetting that (x/8)^r = x^r/8^r (the exponent applies to BOTH numerator and denominator).
  • Forgetting to rewrite p^{2x} as (p^x)^2 before substituting y = p^x, leaving the quadratic structure hidden.

A-level Scope Notes

Entirely within Edexcel IAL Pure Mathematics (P1/P2 index laws and exponential equations; P3 for log-based follow-ups). Uses only the standard laws of indices — no formula booklet entry is required as these are assumed knowledge. Avoid generalising to non-integer or irrational base relationships requiring logs unless the problem explicitly demands it.
