Coordinate Geometry

Apply tangent-radius perpendicularity via negative-reciprocal gradient

Family: Perpendicular Gradient Frequency: 3 questions

Trigger — When to use

Fires whenever a line is required to be perpendicular to another line or segment whose gradient is computable from two known points. The most common surface form is 'find the equation of the tangent to a circle at the point P', because the tangent at P is perpendicular to the radius CP. Other phrasings include 'the line l is perpendicular to AB and passes through...', 'find the perpendicular bisector of...', or 'the normal to the curve at P meets...'. The visual cue is two known points (so a gradient can be computed) plus an explicit perpendicularity condition.

Motivation — Why this works

Two non-vertical lines are perpendicular iff the product of their gradients is −1, so the gradient of any perpendicular line is the negative reciprocal of the original. For a circle this is reinforced by the circle theorem 'tangent ⟂ radius at the point of contact', which converts an otherwise calculus-style problem (implicit differentiation) into a single algebraic step. The move replaces a hard geometric/analytic question ('what is the tangent direction?') with two trivial ones: compute one gradient from two points, then flip-and-negate.

Thought Process — The step-by-step reasoning

  1. Notice the perpendicularity cue: either the word 'perpendicular'/'normal'/'tangent to a circle', or a configuration where one line must meet another at a right angle.
  2. Identify two known points that define the reference line — for a circle tangent these are the centre C and the point of contact P; for a perpendicular bisector they are the two endpoints of the segment.
  3. Rule out heavier alternatives: implicit differentiation of the circle equation, or substituting y = mx + c and forcing discriminant = 0. These work but are slower and not signalled by the question structure when the contact point is already given.
  4. Commit to the perpendicular-gradient rule m₁·m₂ = −1, i.e. m_perp = −1/m_ref. This is the Edexcel-standard tool for this configuration.
  5. Compute m_ref using the two-point gradient formula (y₂ − y₁)/(x₂ − x₁), simplifying any compound fractions carefully.
  6. Take the negative reciprocal — flip the fraction AND change the sign — and substitute into the point-slope form y − y₀ = m_perp(x − x₀) using the given point on the new line.

Mechanic — Concrete operations

  1. Identify the reference line and two points on it: e.g. centre C(a,b) and contact point P(x₀,y₀).
  2. Compute the reference gradient: m_ref = (y₀ − b)/(x₀ − a).
  3. Apply the perpendicularity rule: m_perp = −1/m_ref (flip the fraction, change the sign).
  4. Write the required line in point-slope form: y − y₀ = m_perp (x − x₀).
  5. If a special case arises (m_ref = 0), declare the perpendicular as the vertical line x = x₀.

Worked Micro-Example

Tangent to the circle with centre $C(2/5, 4/5)$ at $P(2,3)$. Radius gradient: $m_{CP} = \frac{3 - 4/5}{2 - 2/5} = \frac{11/5}{8/5} = \frac{11}{8}$. Tangent gradient: $m_T = -\frac{8}{11}$. Tangent equation: $y - 3 = -\tfrac{8}{11}(x-2)$.

Variants — Same move, different surface

Tangent from an external point (contact point not given)
First locate the contact point — typically by simultaneous equations with discriminant = 0, or by using perpendicular-distance-from-centre = radius. Then apply the same perpendicularity rule.
Reference line given as ax + by + c = 0
Read its gradient directly as −a/b; the perpendicular gradient is then b/a, no two-point computation needed.
Reference segment is horizontal (m_ref = 0)
The perpendicular is vertical: write x = x₀ rather than using the −1/0 formula.
Working with direction vectors instead of gradients
Replace m₁·m₂ = −1 with the dot product condition: direction vectors are perpendicular iff their dot product is 0.

Common Misreadings

  • Treating the gradient of a chord PQ as the radius gradient — only valid if PQ passes through the centre.
  • Assuming 'normal to a curve' uses the same recipe as 'tangent to a circle'; for a general curve the reference gradient comes from dy/dx, not from a two-point formula.
  • Confusing the tangent gradient with the radius gradient and forgetting to negate-and-reciprocate.

Pitfalls

  • Sign error in the negative reciprocal: writing 8/11 instead of −8/11, or 1/3 instead of 3.
  • Fraction arithmetic slips when the centre has fractional coordinates (e.g. subtracting 4/5 from 3 to get 11/5).
  • Double-negative mishandling when m_ref is itself negative: −1 ÷ (−1/3) = +3, not −3.
  • Forgetting the special case m_ref = 0 (or m_ref undefined), where the negative-reciprocal formula breaks.

A-level Scope Notes

Stays within Edexcel IAL Pure 1/Pure 2 coordinate geometry. The perpendicular-gradient rule m₁m₂ = −1 is assumed knowledge (not in the formula booklet). Avoid generalising to 3D, where perpendicularity requires dot products of direction vectors rather than a single gradient relation.
