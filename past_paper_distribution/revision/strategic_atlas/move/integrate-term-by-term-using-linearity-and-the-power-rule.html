Integration

Integrate term-by-term using linearity and the power rule

Family: Term By Term Power Rule Frequency: 1 questions

Trigger — When to use

The integrand is (or has been rewritten as) a finite sum of terms each of the form ax^n with n ≠ −1, with no products, quotients, or compositions remaining. Typical cues: a polynomial, a 'split fraction' like (x^3 + 2)/x^2 expanded into x − 2x^{-2}·... wait, x + 2x^{-2}, or a surd-form expression rewritten using fractional exponents. If every term is a constant times a power of x, the power rule applies independently to each.

Motivation — Why this works

Integration is a linear operator: ∫(f + g) dx = ∫f dx + ∫g dx and ∫af dx = a∫f dx, so a sum of power terms can be handled one term at a time. Each term ax^n integrates to a·x^{n+1}/(n+1) for n ≠ −1, which is the reverse of d/dx(x^{n+1}) = (n+1)x^n. This eliminates any need for substitution or parts when the integrand is already in standard power form.

Thought Process — The step-by-step reasoning

  1. I notice the integrand is a sum of terms, each a constant multiple of a power of x, after any preliminary algebraic tidying.
  2. I rule out substitution, integration by parts, and partial fractions because no inner function, product, or rational structure remains.
  3. I commit to the power rule because every exponent is ≠ −1, so each term has a clean x^{n+1}/(n+1) antiderivative.
  4. I bridge to action by treating each term independently: increment its exponent by 1, then divide by the new exponent, keeping the leading coefficient.
  5. Tactically, I watch sign handling for negative exponents (n+1 negative), keep fractional exponents in exact form, and add a single +C at the end (or apply limits if definite).

Mechanic — Concrete operations

  1. Confirm the integrand is a sum of ax^n terms with all n ≠ −1; rewrite roots/fractions as powers if needed.
  2. For each term ax^n, compute a·x^{n+1}/(n+1).
  3. Simplify any coefficient arithmetic, especially when n+1 is negative or fractional.
  4. Sum the antiderivatives and append +C (indefinite) or evaluate at the limits (definite).
  5. Optionally check by differentiating the result term-by-term to recover the integrand.

Worked Micro-Example

Evaluate $I = \int (3x^2 - 2x^{-3} + 5)\,dx$.\\ Term-by-term: $\int 3x^2\,dx = x^3$; $\int -2x^{-3}\,dx = \dfrac{-2x^{-2}}{-2} = x^{-2}$; $\int 5\,dx = 5x$.\\ Hence $I = x^3 + x^{-2} + 5x + C$.

Variants — Same move, different surface

Polynomial integrand, e.g. ∫(4x^3 − x + 7) dx
Apply the power rule to each monomial; constants integrate to a linear term.
Fractional/negative exponents from surds, e.g. ∫(x^{1/2} + 2x^{-1/2}) dx
Rewrite roots as fractional powers first, then apply the power rule; results are 2x^{3/2}/3 + 4x^{1/2}.
Quotient that splits into powers, e.g. ∫(x^3 + 2)/x^2 dx
Divide first to get x − 2x^{-2}·... i.e. x + 2x^{-2}, then integrate term-by-term.
Definite integral with the same structure
Find the antiderivative term-by-term, then evaluate F(b) − F(a); no +C needed.
Mixed sum where one term has n = −1, e.g. ∫(2x + 3/x) dx
Use linearity to split off the 1/x term, which integrates to 3 ln|x|; the remaining terms still use the power rule.

Common Misreadings

  • Believing a negative or fractional exponent forces substitution, when direct power-rule integration suffices.
  • Treating the entire sum as a single composite expression and trying to find one antiderivative for the whole thing.
  • Confusing the power rule for integration (n+1) with the differentiation rule (multiply by n).

Pitfalls

  • Sign error when dividing by a negative new exponent: −2x^{-3} integrates to +x^{-2}, not −x^{-2}.
  • Forgetting the constant of integration +C in indefinite integrals.
  • Mis-handling the constant term: ∫5 dx = 5x, not 5 or 5x^0.
  • Applying the power rule to x^{-1}; this case gives ln|x|, not x^0/0.
  • Arithmetic slips with fractional exponents, e.g. dividing by 3/2 instead of multiplying by 2/3.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics: linearity of integration and the power rule for x^n with n ≠ −1 are core P2/P3 content, including fractional and negative indices.
