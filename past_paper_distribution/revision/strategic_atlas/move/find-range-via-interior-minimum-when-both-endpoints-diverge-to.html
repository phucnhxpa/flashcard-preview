Differentiation

Find range via interior minimum when both endpoints diverge to +∞

Family: Stationary Point Condition Frequency: 1 questions

Trigger — When to use

Fires when a question asks for the range of a continuous function on an open interval (or equivalently the domain of f⁻¹), and the function tends to +∞ (or both to −∞) at both ends of that interval. Typical surfaces: f(x) = ax + b + c/(x − k) on x > k, or other rational/sum-of-reciprocal forms with a vertical asymptote at one end and unbounded growth at the other. Cue phrases: 'state the domain of f⁻¹', 'find the range of f', 'hence write down the set of values'.

Motivation — Why this works

A continuous function on an open interval whose values tend to +∞ at both ends must attain a global minimum at an interior point, and that point must be stationary (f'(x) = 0). The minimum value is then the infimum of the range, which is attained, so the range is [f(x*), ∞). Because domain of f⁻¹ equals range of f, this same value is the left endpoint of the inverse's domain.

Thought Process — The step-by-step reasoning

  1. Notice that 'domain of f⁻¹' is being asked for, and recall this equals the range of f on its given domain.
  2. Rule out endpoint-only reasoning: check the limits at both ends of the open interval — both go to +∞, so the minimum is interior, not at a boundary.
  3. Commit to the stationary-point route: continuity + both ends → +∞ guarantees an interior global minimum where f'(x) = 0.
  4. Bridge to computation by differentiating f, treating any 1/(x−k) term using the power rule on (x−k)⁻¹.
  5. Inside the move: select the root of f'(x) = 0 lying inside the given domain, evaluate f there carefully, confirm minimum via sign of f'' or sign change of f', and write the range with a closed lower bound (≥).

Mechanic — Concrete operations

  1. Check end-behaviour at both ends of the domain to confirm both limits are +∞ (or both −∞).
  2. Differentiate f, writing reciprocal terms as negative powers: e.g. d/dx[c/(x−k)] = −c(x−k)⁻².
  3. Solve f'(x) = 0 algebraically; select the root that lies in the given domain.
  4. Substitute that x-value back into f(x) to obtain the minimum (or maximum) value.
  5. Verify it is a minimum using f''(x) > 0 or a sign-change argument.
  6. State the range as [f(x*), ∞), and hence the domain of f⁻¹ as x ≥ f(x*).

Worked Micro-Example

Let f(x) = 3x + 11 + 49/(x−4) for x > 4. As x → 4⁺ and as x → ∞, f → ∞. Differentiate: f'(x) = 3 − 49(x−4)⁻². Setting f'(x) = 0 gives (x−4)² = 49/3, so x − 4 = 7/√3 (positive root). Then f = 3(4 + 7/√3) + 11 + 49·√3/7 = 23 + 7√3 + 7√3 = 23 + 14√3. Since f''(x) = 98(x−4)⁻³ > 0, this is a minimum, so the domain of f⁻¹ is x ≥ 23 + 14√3.

Variants — Same move, different surface

Function of form ax + b/x on x > 0
Both ends (x → 0⁺ and x → ∞) tend to +∞ when a, b > 0; same procedure gives minimum at x = √(b/a) with value 2√(ab).
Both endpoints tend to −∞ instead
Interior point is a global maximum; range becomes (−∞, f(x*)] and domain of f⁻¹ is x ≤ f(x*).
f is monotonic on the given interval
No differentiation needed; range is read directly from the two endpoint limits.
Closed-interval domain [a,b] with interior stationary point
Compare f(a), f(b), and f(x*) to determine min and max; range is [min, max].
Completing the square on a quadratic f(x) = ax² + bx + c
Use vertex form instead of calculus; the turning point gives the range endpoint directly.

Common Misreadings

  • Confusing 'domain of f⁻¹' with 'domain of f' and quoting x > 4 instead of computing the range.
  • Treating the open endpoint x = 4 as giving a finite boundary for the range, rather than checking the limit.
  • Trying to invert f algebraically and then solve a discriminant condition; this can work but is usually longer and error-prone.

Pitfalls

  • Taking the wrong sign root, e.g. x − 4 = −7/√3, which lies outside x > 4.
  • Rationalisation slips when simplifying 49/(7/√3) = 7√3.
  • Writing the range with a strict inequality (>) when the minimum is actually attained — should be ≥.
  • Forgetting to verify that the stationary point is a minimum (not a maximum or inflection).

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics: uses standard differentiation of polynomial and reciprocal terms, basic limit reasoning at asymptotes, and the inverse-function fact that domain of f⁻¹ equals range of f.
