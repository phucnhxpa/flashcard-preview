Trigonometry

Scale the interval for the multiple angle and enumerate every branch

Family: Multiple Angle Interval Frequency: 1 questions

Trigger — When to use

The equation has the form sin(nθ) = k, cos(nθ) = k or tan(nθ) = k (with n ≠ 1) on a stated interval for θ such as (0, 2π) or [0°, 360°]. The argument nθ is the actual variable being inverted, so the working interval must be widened by a factor of n. A reliable cue is any equation where the angle inside the trig function is a multiple (or fraction) of θ and the question asks for ALL solutions in a given range.

Motivation — Why this works

Substituting φ = nθ converts sin(nθ) = k into sin φ = k, but the interval for φ becomes n times as wide as that for θ. Each full 2π-cycle of sine or cosine contributes two solutions (and tangent contributes one per π-cycle), so n cycles generally produce 2n solutions. Solving for every φ first and only then dividing by n guarantees no branch is dropped, which is the standard examiner trap on these questions.

Thought Process — The step-by-step reasoning

  1. Notice that the angle inside the trig function is nθ, not θ, so the natural variable to solve for is φ = nθ.
  2. Rule out solving directly in θ — that risks missing branches that only appear in the extended interval.
  3. Commit to listing the φ-interval explicitly (e.g. (0, 4π) when θ ∈ (0, 2π) and n = 2) before doing any arithmetic.
  4. Bridge: compute the reference angle α = arcsin|k| (or arccos|k|, arctan|k|) using the principal value, then place it in the correct quadrants.
  5. Tactically, generate all solutions in the first 2π by quadrant rules, then add 2π, 4π, … to each until they exceed the upper bound, and finally divide every value by n.

Mechanic — Concrete operations

  1. Let φ = nθ and write down the new interval: if θ ∈ (a, b) then φ ∈ (na, nb).
  2. Find the reference angle α from the principal-value inverse trig.
  3. List the base solutions in [0, 2π): for sine these are α and π − α (when k > 0); for cosine they are α and 2π − α; for tangent just α.
  4. Add 2π (or π for tangent) repeatedly to each base solution, keeping every value that lies in the φ-interval.
  5. Divide each surviving φ by n to recover θ, and check each θ lies in the original interval.
  6. Round to the requested accuracy (typically 3 s.f. or to the nearest 0.1°).

Worked Micro-Example

Solve $\sin 2\theta = \tfrac{1}{5}$ for $\theta \in (0, 2\pi)$. Let $\varphi = 2\theta$, so $\varphi \in (0, 4\pi)$ and $\alpha = \arcsin(0.2) \approx 0.2014$. Base solutions in $[0, 2\pi)$: $\varphi = 0.2014,\ \pi - 0.2014 = 2.940$. Add $2\pi$: $\varphi = 6.485,\ 9.224$. Dividing by 2 gives $\theta \approx 0.101,\ 1.47,\ 3.24,\ 4.61$.

Variants — Same move, different surface

$\sin 3\theta = k$ with $\theta \in (0, 2\pi)$
Search interval becomes $(0, 6\pi)$, giving up to six solutions; add $2\pi$ twice to each base value before dividing by 3.
$\cos(\theta/2) = k$ with $\theta \in (0, 2\pi)$
Search interval shrinks to $(0, \pi)$ — only half a cycle, so typically just one solution survives.
$\tan 2\theta = k$ with $\theta \in [0, 2\pi)$
Use period $\pi$: base solution $\alpha$, then add $\pi, 2\pi, 3\pi$ within $[0, 4\pi)$, giving four θ values.
Equation given in degrees, e.g. $\sin 2\theta = k$ for $\theta \in [0°, 360°]$
Widen to $2\theta \in [0°, 720°]$ and use $180° - \alpha$, $360° + \alpha$, $540° - \alpha$.
$\sin(2\theta + c) = k$
Let $\varphi = 2\theta + c$; the φ-interval is shifted as well as scaled, e.g. $\theta \in (0, 2\pi) \Rightarrow \varphi \in (c, 4\pi + c)$.

Common Misreadings

  • Treating the original interval $(0, 2\pi)$ as the interval for $2\theta$ and producing only two of the four solutions.
  • Reading the n in $\sin n\theta$ as merely a coefficient to be divided through at the end, rather than a factor that scales the search interval.
  • Believing 'period $\pi$' for $\sin 2\theta$ means only solutions in $(0, \pi)$ are needed.

Pitfalls

  • Forgetting the $3\pi - \alpha$ branch and replacing it with $\pi + \alpha$, which corresponds to a negative sine value.
  • Failing to divide each φ by n at the end, leaving answers as values of $2\theta$ rather than $\theta$.
  • Including endpoints when the interval is open (or vice versa), producing one too many or too few solutions.
  • Mixing radians and degrees when adding period multiples (e.g. adding $360$ to a radian value).

A-level Scope Notes

This is the standard P3/P4 technique for solving multiple-angle trig equations on a bounded interval and stays entirely within Edexcel IAL Pure Mathematics; no general-solution formulae beyond the syllabus are required.
