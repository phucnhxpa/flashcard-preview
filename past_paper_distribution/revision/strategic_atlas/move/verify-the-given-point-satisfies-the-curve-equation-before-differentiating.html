Coordinate Geometry

Verify the given point satisfies the curve equation before differentiating

Family: Point Verification Frequency: 1 questions

Trigger — When to use

Fires whenever a question states a specific point lies on a curve and asks for a tangent, normal, or gradient there. Typical phrasings: 'show that the point P(a,b) lies on C', 'the tangent to C at (1,1)', or 'verify that (a,b) is on the curve'. Especially common when the curve is given implicitly or involves logs/exponentials where the point is chosen to make terms vanish (e.g. ln 1 = 0, e^0 = 1). Edexcel mark schemes routinely allocate a B1 mark for this verification.

Motivation — Why this works

Substituting the coordinates into the original equation confirms the point genuinely lies on the curve, which is logically required before any tangent/normal calculation is meaningful. It also catches transcription errors early and exposes simplifications (e.g. ln 1 = 0) that will streamline later derivative evaluation. On Edexcel papers this step is often a standalone mark, so omitting it forfeits an easy credit even if the rest of the solution is perfect.

Thought Process — The step-by-step reasoning

  1. Notice the question hands you a specific point and asks for tangent/normal info — a verification mark is almost certainly available.
  2. Rule out jumping straight to implicit differentiation; the point check is faster and protects against later mistakes.
  3. Commit to substituting because it costs one line, banks a mark, and may reveal simplifying values like ln 1 = 0.
  4. Plug x and y into the LHS of the original equation and evaluate term by term.
  5. Compare to RHS, write a clear tick or 'so (a,b) lies on C', then proceed to the gradient calculation.

Mechanic — Concrete operations

  1. Write the original curve equation.
  2. Substitute the given x and y into the LHS.
  3. Evaluate each term, using identities like ln 1 = 0, e^0 = 1 where relevant.
  4. Show the LHS equals the RHS.
  5. State explicitly that the point lies on the curve.

Worked Micro-Example

Curve $C$: $x^2\ln y + \tfrac{1}{x} + 2y = 3$. Check $(1,1)$: LHS $= 1^2\ln 1 + \tfrac{1}{1} + 2(1) = 0 + 1 + 2 = 3 =$ RHS. Hence $(1,1)$ lies on $C$.

Variants — Same move, different surface

Explicit curve y = f(x)
Substitute x into f and confirm the result equals the given y-coordinate.
Parametric curve x = f(t), y = g(t) with point given
Find the t-value from one coordinate and check it reproduces the other.
Implicit curve with logs/exponentials
Exploit ln 1 = 0 or e^0 = 1 to simplify; these are usually why the point was chosen.
'Show that P lies on C' as a standalone part (a)
Same substitution, but write conclusion sentence explicitly — this is the entire mark.
Circle/locus problems with a candidate point
Substitute into (x-a)^2 + (y-b)^2 = r^2 and check equality.

Common Misreadings

  • Treating the verification as optional scratch work and not writing it on the answer paper.
  • Assuming the point must lie on C because the question says so, and skipping the demonstration.
  • Confusing 'show it lies on C' with 'find the tangent' and only doing the latter.

Pitfalls

  • Skipping the substitution entirely and losing an easy B1.
  • Arithmetic slip when evaluating ln 1, 1/x or fractional terms.
  • Forgetting to state the conclusion 'so (a,b) lies on C' after showing LHS = RHS.
  • Substituting into a rearranged form and getting confused signs instead of using the original equation.

A-level Scope Notes

Fully within Edexcel IAL Pure (P2–P4) coordinate geometry and implicit/parametric differentiation; uses only direct substitution and standard log/exponential values.
