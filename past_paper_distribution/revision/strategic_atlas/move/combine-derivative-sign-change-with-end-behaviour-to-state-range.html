Strategic Atlas / Differentiation / Range From Turning Point
Differentiation

Combine derivative sign change with end-behaviour to state range

Family: Range From Turning Point Frequency: 1 questions

Trigger — When to use

Fires when a 'find the range of f' question follows a part that located a single stationary point on a continuous (often open or half-open) domain. Cues include: f' just solved equal to zero giving one critical x, a domain like x > a or x \in (a, \infty), and no obvious horizontal asymptote already named. Alternative phrasings: 'state the set of values f can take', 'write down the range of f', 'find the least value of f and hence its range'.

Motivation — Why this works

A continuous function with exactly one stationary point on a connected domain is monotonic on each side of that point, so the stationary value is either the global min or global max. The sign of f' on either side classifies which it is, and the limits at the domain endpoints decide whether that extremum is the boundary of the range. Together these three checks (sign-change, classification, end-behaviour) give the range without further machinery.

Thought Process — The step-by-step reasoning

  1. Notice that the stationary point is unique on a connected domain, so monotonicity can flip at most once.
  2. Rule out needing the second derivative test — a sign table for f' is faster and avoids messy algebra.
  3. Commit to: classify the stationary point via sign of f', then read off the global extremum.
  4. Bridge to operations: pick one test value on each side of the critical x and evaluate the sign of f'.
  5. Tactically check both endpoints of the domain (including limits to \infty or to a finite endpoint not in the domain) to decide whether the extremum is attained and whether the range is bounded.

Mechanic — Concrete operations

  1. Identify the unique stationary point x = c and the domain D.
  2. Pick a test value on each side of c within D and compute the sign of f'.
  3. Use the sign change (− then +) ⇒ minimum, (+ then −) ⇒ maximum.
  4. Compute f(c) — the candidate extreme value.
  5. Evaluate \lim f(x) at each endpoint of D (one-sided limits at finite endpoints, behaviour as x \to \infty).
  6. Combine: if minimum at c with both endpoints tending to +\infty, range is [f(c), \infty); adjust inequality strictness based on whether endpoints are attained.

Worked Micro-Example

Let f(x) = x + 1/x for x > 0. Then f'(x) = 1 - 1/x^2 = 0 at x = 1. f'(0.5) = 1 - 4 < 0, f'(2) = 1 - 0.25 > 0, so x=1 is a minimum with f(1)=2. As x\to 0^+, f\to +\infty; as x\to\infty, f\to +\infty. Hence range is f(x) \ge 2, i.e. [2, \infty).

Variants — Same move, different surface

Stationary point is a maximum with both ends tending to -\infty
Range becomes (-\infty, f(c)]; sign of f' goes + then −.
Closed bounded domain [a,b]
Compare f(a), f(b), f(c); range is [min, max] of these three values.
Domain (a, b) open, with finite limits at endpoints
Range may be open on one side; e.g. min attained at c gives [f(c), L) where L = \max of endpoint limits.
Function has a horizontal asymptote y = L at one end
End-behaviour gives an open boundary at L; range like [f(c), L) if L > f(c) and not attained.
Exponential / logarithmic f with one stationary point (e.g. xe^{-x})
Same template; check x\to\pm\infty limits, one of which is typically 0.

Common Misreadings

  • Writing strict inequality f(x) > f(c) when the minimum is in fact attained at an interior point.
  • Treating the unique stationary point as automatically a minimum without checking the sign of f'.
  • Confusing the domain endpoint behaviour with the range endpoint — e.g. assuming f \to 0 just because x \to 0.

Pitfalls

  • Forgetting to test both sides of the stationary point when the domain is open.
  • Neglecting the limit at a finite endpoint not in the domain (e.g. x \to (1/4)^+).
  • Using the second derivative test on an awkward expression when a first-derivative sign table is far cleaner.
  • Stating the range without specifying whether the extreme value is included.

A-level Scope Notes

Uses only first-derivative sign analysis and informal limits at endpoints, both standard in Edexcel IAL P3/P4 differentiation; no formal epsilon-delta or L'Hôpital required.
