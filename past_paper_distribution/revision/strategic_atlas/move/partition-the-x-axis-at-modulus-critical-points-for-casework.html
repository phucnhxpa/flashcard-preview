Partition the x-axis at modulus critical points for casework

Family: Modulus Case Analysis Frequency: 1 questions

Trigger — When to use

Fires when an equation or inequality contains two or more modulus expressions that cannot be reduced to a clean |A| = |B| form (so squaring is unsafe). Typical cue phrases: 'solve |...| + ... = |...|', or modulus terms combined with constants/linear terms outside the bars. Each modulus introduces a critical point where its inner expression changes sign. The presence of stray additive constants outside the moduli is the strongest signal that casework, not squaring, is required.

Motivation — Why this works

A modulus function is piecewise-linear, so a sum/equation of moduli is piecewise-linear with breakpoints exactly at the zeros of the inner expressions. Between consecutive breakpoints every |g(x)| has a fixed sign and can be replaced by +g(x) or -g(x), eliminating the modulus signs. Solving the resulting linear equation on each interval, then checking that each candidate lies within its defining interval, gives all solutions. This works precisely because each inner expression has constant sign on each open interval between critical points.

Thought Process — The step-by-step reasoning

  1. Notice two modulus terms with an additive constant outside, so the equation is not in |A| = |B| form.
  2. Rule out squaring both sides — the +a term outside the LHS modulus would create cross terms that re-introduce a modulus.
  3. Commit to casework: locate the critical points where each inner expression vanishes, since these are the only places where the piecewise structure changes.
  4. Order the critical points using the given sign of any parameter (here a > 0) so the real line splits into three intervals.
  5. On each interval, replace each modulus by its signed expression, solve the resulting linear equation, and discard any solution that falls outside its interval.

Mechanic — Concrete operations

  1. Set each inner expression to zero to find critical points x = c_1, c_2, ....
  2. Use any parameter constraints to order the critical points along the real line.
  3. Define the resulting intervals (including the two unbounded ones).
  4. On each interval, determine the sign of each inner expression and replace |g(x)| by +g(x) or -g(x).
  5. Solve the resulting linear equation in that interval.
  6. Check the candidate solution lies within the defining interval; reject if not.
  7. Collect all valid solutions across the cases.

Worked Micro-Example

Solve $|3x + a| + a = |x + 5a|$ with $a > 0$. Critical points: $x = -a/3$ and $x = -5a$, with $-5a < -a/3$. Case $x \ge -a/3$: $(3x+a)+a = x+5a \Rightarrow x = 3a/2$ ✓. Case $-5a \le x < -a/3$: $-(3x+a)+a = x+5a \Rightarrow x = -5a/4$ ✓. Case $x < -5a$: $-(3x+a)+a = -(x+5a) \Rightarrow x = 5a/2$, rejected since $5a/2 \not< -5a$. Solutions: $x = -5a/4$ and $x = 3a/2$.

Variants — Same move, different surface

Single modulus equation $|f(x)| = k$ with $k \ge 0$
Two cases $f(x) = k$ or $f(x) = -k$; no need to partition the x-axis since k is constant.
Equation $|f(x)| = |g(x)|$
Squaring both sides is cleaner; casework still works but is slower.
Modulus inequality such as $|f(x)| < |g(x)| + c$
Same partition into intervals at critical points, but each case yields a linear inequality whose solution must be intersected with its defining interval.
Three modulus terms, e.g. $|x-1| + |x-2| + |x-3| = k$
Three critical points create four intervals; the procedure scales but requires careful sign tables.
Sketching $y = |f(x)| + |g(x)|$
Critical points become the vertices of the piecewise-linear graph; gradients on each piece come from the same sign analysis.

Common Misreadings

  • Treating $|A| + c = |B|$ as $|A| = |B|$ and squaring, dropping the constant $c$.
  • Using only two cases (overall positive vs negative) rather than one case per interval between critical points.
  • Assuming the critical points are ordered without checking the sign of the parameter.

Pitfalls

  • Incorrectly ordering the critical points when parameters are involved (e.g. assuming $-a/3 < -5a$ for $a > 0$).
  • Negating only part of the inner expression, e.g. writing $-3x + a$ instead of $-(3x+a) = -3x - a$.
  • Forgetting to verify each candidate lies in its defining interval, leading to spurious solutions.
  • Mishandling the boundary points (assigning them to the wrong interval) when the inner expression equals zero there.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics, where modulus equations and inequalities of linear expressions are standard; no analytic tools beyond piecewise-linear reasoning are used.
