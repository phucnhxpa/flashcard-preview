Substitute denominator roots to annihilate terms in partial-fraction identities

Family: Strategic Substitution Frequency: 1 questions

Trigger — When to use

Fires when a cleared partial-fractions identity of the form P(x) ≡ A·f₁(x) + B·f₂(x) + C·f₃(x) holds for all x and each constant multiplies a product of denominator factors. The cue is the presence of distinct linear factors (or repeated factors) so that specific x-values zero out multiple terms. Alternative phrasings: 'find A, B, C such that the identity holds', 'express in partial fractions', or 'using the cover-up rule'. Especially efficient when the identity has just been obtained by multiplying through by the common denominator.

Motivation — Why this works

Because the identity is true for ALL x, we may freely choose values of x that simplify it. Choosing x equal to a root of a denominator factor sends every term containing that factor to zero, leaving a single-unknown equation. This avoids assembling and solving a simultaneous system and exploits the fact that polynomial identities permit pointwise evaluation. It is the algebraic skeleton of the Heaviside cover-up rule.

Thought Process — The step-by-step reasoning

  1. I notice the identity has three unknowns A, B, C, each attached to a product of linear factors of the original denominator.
  2. I rule out comparing coefficients first, since that produces a 3×3 system, and I rule out substituting convenient values like x = 0 because they don't annihilate enough terms.
  3. I commit to substituting roots of the denominator factors, since each root will kill every term whose coefficient's product contains that factor.
  4. I scan each constant: which roots zero out the OTHER constants' products but leave this one's surviving? x = 2 leaves only C; x = -1 leaves only A; no single x isolates B because B's product contains both factors.
  5. I plan to harvest A and C by root-substitution, then recover B by an auxiliary method (equating one coefficient, or substituting any third value such as x = 0).

Mechanic — Concrete operations

  1. Identify the roots of each denominator factor in the cleared identity.
  2. For each root r, substitute x = r into both sides; on the RHS, every term containing (x − r) as a factor vanishes.
  3. Solve the resulting one-variable equation for the surviving constant.
  4. Repeat for each root that isolates a single constant.
  5. For any constant not isolable this way (e.g. one whose term contains every factor), substitute a further convenient value (such as x = 0) or compare a single coefficient, using the constants already found.

Worked Micro-Example

Given $x^{2}+10x+1 \equiv A(x-2)^{2}+B(x+1)(x-2)+C(x+1)$. Set $x=2$: LHS $=25$, RHS $=3C$, so $C=\tfrac{25}{3}$. Set $x=-1$: LHS $=-8$, RHS $=9A$, so $A=-\tfrac{8}{9}$. Then set $x=0$: $1 = 4A - 2B + C$, giving $B=\tfrac{1}{2}(4A+C-1)$, which evaluates to $B=\tfrac{17}{9}$.

Variants — Same move, different surface

Three distinct linear factors, e.g. $\tfrac{P(x)}{(x-a)(x-b)(x-c)}$
Substituting x = a, b, c in turn isolates all three constants directly; no auxiliary step needed.
Repeated linear factor $(x-a)^{2}$ in the denominator
Substituting x = a isolates the constant on the $(x-a)^{2}$-free numerator (the highest-power partial fraction); the remaining constant requires a second substitution or coefficient comparison.
Irreducible quadratic factor present
Real root substitution still works for the linear-factor constants; constants over the quadratic must be found by equating coefficients or substituting any further value.
Heaviside cover-up shortcut
Instead of clearing fractions, cover the factor $(x-r)$ in the original fraction and evaluate the rest at $x=r$ to read off the corresponding constant directly.

Common Misreadings

  • Trying x = 0 or another 'easy' value first, which fails to annihilate any factor and yields a messy multi-unknown equation.
  • Believing every constant must be obtainable by root-substitution, then getting stuck on the constant whose term contains all factors.
  • Confusing the cleared identity with the original fractional equation and substituting roots into the latter (where they make denominators zero).

Pitfalls

  • Arithmetic slips when squaring negatives, e.g. writing $(-1-2)^{2}=-9$ instead of $9$.
  • Forgetting that $(x-2)^{2}$ at $x=2$ is $0$, not $2$ or $4$.
  • Sign errors in the LHS evaluation, particularly for negative substitutions.
  • Stopping after two substitutions and leaving B undetermined.

A-level Scope Notes

Standard Edexcel IAL technique for partial fractions in C3/C4 (proper rational functions with distinct or repeated linear factors, plus irreducible quadratics in Further Pure).
