Integration

Recognise f'(x)/f(x) pattern to integrate as ln|f(x)|

Family: Logarithmic Recognition Frequency: 3 questions

Trigger — When to use

Fires whenever the integrand is a fraction whose numerator is (a constant multiple of) the derivative of the denominator. Concrete cues: ∫4/(4x+5) dx (numerator 4 = d/dx(4x+5)); ∫cot y dy = ∫cos y/sin y dy (numerator is derivative of denominator); ∫(2x+5)/(x²+5x+3) dx after a partial-fraction or algebraic split has been engineered. Equivalent phrasings: 'top is the derivative of the bottom', 'reverse chain rule on a logarithm', 'the integrand looks like u'/u'. Also fires after a deliberate setup step (e.g. multiplying/dividing by a constant) that makes the f'/f shape appear.

Motivation — Why this works

The result ∫f'(x)/f(x) dx = ln|f(x)| + C is the reverse chain rule applied to ln|u|: if u = f(x) then d/dx[ln|f(x)|] = f'(x)/f(x). Recognising this pattern lets you write the antiderivative by inspection, avoiding a formal u-substitution and the bookkeeping it brings. It is the standard A-level shortcut for many rational and trigonometric integrals (cot, tan, sec'/sec etc.) and is essential when solving separable ODEs whose separated form contains such fractions.

Thought Process — The step-by-step reasoning

  1. Notice that the integrand is a quotient and ask: 'is the numerator the derivative of the denominator?' — differentiate the denominator mentally and compare.
  2. Rule out alternatives: a u-substitution would work but is overkill; partial fractions are unnecessary if the f'/f form is already visible; integration by parts is wrong-tool here.
  3. Commit when the comparison matches up to a constant factor — i.e. numerator = k · f'(x) for some constant k that can be pulled outside the integral.
  4. Bridge to the first operation: factor out the constant k, then write the integral directly as k · ln|f(x)| + C.
  5. Tactical adjustment: if f(x) is linear (ax+b), the inner derivative a contributes a 1/a factor; if f'(x) carries a sign (e.g. d(1−x)/dx = −1), make sure the resulting log inherits that sign.
  6. Sanity-check by mentally differentiating the answer: d/dx[k ln|f(x)|] should reproduce the integrand exactly.

Mechanic — Concrete operations

  1. Identify f(x) (the denominator) and compute f'(x).
  2. Compare numerator with f'(x); determine the constant multiplier k so that numerator = k · f'(x).
  3. Rewrite the integral as k · ∫f'(x)/f(x) dx.
  4. Apply the standard result: k · ∫f'(x)/f(x) dx = k · ln|f(x)| + C.
  5. If the integrand came from a decomposition (e.g. partial fractions or a (1/2)-factor split), carry the outer multiplier through to the final logarithm.

Worked Micro-Example

Compute $\int \frac{4}{4x+5}\,dx$. Let $f(x)=4x+5$, so $f'(x)=4$. The integrand is exactly $f'(x)/f(x)$, hence $\int \frac{4}{4x+5}\,dx = \ln|4x+5| + C$. Check: $\frac{d}{dx}\ln|4x+5| = \frac{4}{4x+5}.$ ✓

Variants — Same move, different surface

Linear denominator with mismatched constant numerator, e.g. ∫1/(4x+5) dx
Insert the missing factor: write as (1/4)·∫4/(4x+5) dx = (1/4) ln|4x+5| + C.
Trigonometric integrand: ∫cot y dy = ∫cos y/sin y dy
Since d(sin y)/dy = cos y, the integral is ln|sin y| + C. For ∫tan x dx = ∫sin x/cos x dx, note d(cos x)/dx = −sin x, so the integral is −ln|cos x| + C.
Denominator of the form (1−x) or any (a−x) with negative inner derivative
d(1−x)/dx = −1, so ∫1/(1−x) dx = −ln|1−x| + C. Track the sign carefully.
Quadratic denominator, e.g. ∫(2x+5)/(x²+5x+3) dx after splitting
Numerator 2x+5 is exactly d/dx(x²+5x+3), so integral = ln|x²+5x+3| + C; carry any (1/2) or other outer factor from the split through to the log.
Exponential denominator, e.g. ∫eˣ/(eˣ+1) dx
Since d(eˣ+1)/dx = eˣ, the integral is ln|eˣ+1| + C — same f'/f recognition.

Common Misreadings

  • Mistaking the integrand for a partial-fractions problem when it is already in f'/f form, doing extra unnecessary algebra.
  • Trying to use the power rule on 1/(linear) and getting (linear)⁰/0, instead of recognising the log.

Pitfalls

  • Forgetting the chain-rule factor: writing ∫1/(ax+b) dx = ln|ax+b| instead of (1/a) ln|ax+b|.
  • Sign error when the inner derivative is negative: ∫1/(1−x) dx = −ln|1−x| + C, not +ln|1−x|.
  • Dropping a multiplicative factor (e.g. a 1/2 from a previous decomposition step) when writing the final logarithm.
  • Omitting the absolute value signs in |f(x)| when f(x) can change sign on the relevant domain.

A-level Scope Notes

Fully within Edexcel IAL P3/P4 (and FP2) scope. The result ∫f'(x)/f(x) dx = ln|f(x)| + C is a standard listed integral; ∫1/x dx = ln|x| + C appears in the formula booklet, and the f'/f extension is a direct reverse-chain-rule consequence. Avoid generalising to complex-valued logarithms — stay with real f(x) and absolute values.
