Decompose log arguments into atoms log_a(a) and log_a(b)

Family: Logarithmic Decomposition Frequency: 1 questions

Trigger — When to use

Fires when an expression involves logarithms (single base $a$) whose arguments are products, quotients, or powers built from $a$ and $b$, and the only given datum is $\log_a b = k$. The required answer must be expressed purely in terms of $k$. Surface cues: arguments like $\sqrt{a}/b$, $a^2 b^3$, $a/b^n$, or ratios of such logs. Equivalent phrasings: 'express in terms of $k$', 'find the value of $\log_a(\ldots)$ given $\log_a b = k$'.

Motivation — Why this works

Any log with base $a$ whose argument is built multiplicatively from $a$ and $b$ can be reduced via the product, quotient, and power laws to a linear combination of $\log_a a$ and $\log_a b$. Since $\log_a a = 1$ and $\log_a b = k$ are the only evaluable quantities in this setting, these two are the natural 'atoms' at which simplification must terminate. Targeting them up front gives a deterministic route from a compound log expression to an algebraic expression in $k$.

Thought Process — The step-by-step reasoning

  1. Notices that the only known logarithmic value is $\log_a b = k$, and that $\log_a a = 1$ is automatic — so the answer must collapse onto these two quantities.
  2. Rules out trying to evaluate logs numerically or combining a fraction of logs as a single log (a common temptation when the expression is $\log(\cdot)/\log(\cdot)$).
  3. Commits to applying log laws purely structurally: split products into sums, quotients into differences, and pull exponents out front.
  4. Bridges to operation by scanning each argument: identify whether it is a product, quotient, or power of $a$ and $b$, and choose the matching law first.
  5. Tactically applies quotient/product rule before the power rule to expose each $a^p$ and $b^q$ as a separate term, then pulls $p$ and $q$ out as coefficients and substitutes $\log_a a = 1$, $\log_a b = k$.

Mechanic — Concrete operations

  1. Identify each argument's multiplicative structure (product / quotient / power of $a$ and $b$).
  2. Apply the quotient rule: $\log_a(X/Y) = \log_a X - \log_a Y$.
  3. Apply the product rule: $\log_a(XY) = \log_a X + \log_a Y$.
  4. Apply the power rule: $\log_a(X^n) = n\log_a X$, including fractional powers from radicals.
  5. Substitute $\log_a a = 1$ and $\log_a b = k$.
  6. Collect terms into a single linear or rational expression in $k$.
  7. If the original expression is a ratio of logs, simplify numerator and denominator separately — do NOT merge into a single log.

Worked Micro-Example

Given $\log_a b = k$, evaluate $\log_a\!\left(\dfrac{\sqrt{a}}{b}\right)$. \\ $\log_a(\sqrt{a}/b) = \log_a a^{1/2} - \log_a b = \tfrac{1}{2}\log_a a - \log_a b = \tfrac{1}{2} - k$. \\ Similarly $\dfrac{\log_a(a^2 b^3)}{\log_a(b^2)} = \dfrac{2 + 3k}{2k}$.

Variants — Same move, different surface

Numeric base with prime-factored argument, e.g. $\log_2 72$ given $\log_2 3 = c$.
Factorise the argument over primes ($72 = 2^3\cdot 3^2$) and treat $\log_2 2 = 1$ and $\log_2 3 = c$ as the atoms.
Two given data, e.g. $\log_a b = p$ and $\log_a c = q$, asked for $\log_a(b^m c^n / a^r)$.
Same decomposition, now landing on three atoms $\log_a a, \log_a b, \log_a c$ with values $1, p, q$.
Change-of-base required: expression uses $\log_b x$ but the data is in base $a$.
Apply $\log_b x = \log_a x / \log_a b$ first, then decompose the base-$a$ logs as usual.
Argument involves a sum/difference, e.g. $\log_a(a + b)$.
Move does NOT apply — log laws do not split sums; flag and seek a different strategy.

Common Misreadings

  • Treating $\dfrac{\log_a X}{\log_a Y}$ as $\log_a(X/Y)$ — confusing division of logs with the quotient rule.
  • Reading $\log_a(a+b)$ as decomposable, attempting to split the sum inside the log.
  • Mistaking $\sqrt{a}$ for $a$ and dropping the $\tfrac{1}{2}$ exponent.

Pitfalls

  • Forgetting that $\log_a a = 1$ and leaving stray $\log_a a$ terms in the final answer.
  • Misapplying the power rule as $\log(x^n) = (\log x)^n$ instead of $n\log x$.
  • Collapsing $\log_a(X)/\log_a(Y)$ into $\log_a(X/Y)$ — a frequent error in ratio-of-logs questions.
  • Sign errors when applying the quotient rule, especially with nested radicals like $\sqrt{a/b}$.

A-level Scope Notes

Fully within Edexcel IAL P2/P3 logarithm content: laws of logarithms (product, quotient, power), $\log_a a = 1$, and change of base. No calculus or series tools required.
