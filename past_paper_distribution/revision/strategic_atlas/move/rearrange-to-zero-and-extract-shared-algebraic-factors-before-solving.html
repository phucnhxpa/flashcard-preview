Algebra & Functions

Rearrange to zero and extract shared algebraic factors before solving

Family: Factor And Solve Frequency: 1 questions

Trigger — When to use

The equation f(x) = g(x) presents both sides as products that visibly share common algebraic factors (e.g. both contain x and (x−4) as explicit factors). Neither side is fully expanded, so the factored structure is already exposed. This often arises in curve-intersection problems where each curve is given in factored form, or when solving equations like A·P(x) = B·P(x)·Q(x). The cue is: 'I can see the same bracket on both sides.'

Motivation — Why this works

Moving everything to one side and factoring preserves the structural information already encoded in the factored forms, sidestepping the labour of expanding into a cubic (or higher) and then re-factoring via the factor theorem. Crucially, factoring rather than dividing keeps every root: any value that makes the shared factor zero is a legitimate solution. The zero-product property then converts the problem into independent linear (or simpler) equations, each trivially solvable.

Thought Process — The step-by-step reasoning

  1. Notice that both sides of f(x) = g(x) contain the same explicit factors — here x and (x−4) appear on both sides.
  2. Rule out dividing through by the shared factor: this would discard the roots where that factor vanishes (x = 0 and x = 4), which are genuine intersections.
  3. Rule out full expansion into a cubic: it works but throws away the visible structure and forces a factor-theorem hunt.
  4. Commit to: subtract one side from the other to get '... = 0', then pull the common factor out of the resulting expression.
  5. Inside the move, simplify the residual bracket carefully (combining the leftover linear pieces and any rational coefficients) so the final factorisation is fully reduced before applying the zero-product property.

Mechanic — Concrete operations

  1. Rearrange to one side: bring everything to the left so the equation reads (expression) = 0.
  2. Identify the highest common algebraic factor present in every term (here 0.3x(x−4)).
  3. Factor it out, leaving a residual bracket containing the remaining linear/constant pieces.
  4. Simplify the residual bracket by combining like terms and clearing any fractional coefficients.
  5. Apply the zero-product property: set each factor equal to zero and list all roots.

Worked Micro-Example

Solve $1.2x(x-4) = 0.3x(x-4)^2$.\ Rearrange: $0.3x(x-4)^2 - 1.2x(x-4) = 0$.\ Factor: $0.3x(x-4)\big[(x-4) - 4\big] = 0$, i.e. $0.3x(x-4)(x-8) = 0$.\ Hence $x = 0,\ 4,\ 8$.

Variants — Same move, different surface

Trigonometric equation $\sin x \cos x = \sin x \cos^2 x$ on a given interval.
Rearrange to $\sin x \cos x(1 - \cos x) = 0$ and solve each factor separately, rather than dividing by $\sin x \cos x$ (which would lose roots where these vanish).
Exponential equation $e^x f(x) = e^x g(x)$.
Since $e^x \neq 0$, factoring out $e^x$ is safe; the equation reduces immediately to $f(x) - g(x) = 0$.
Curve intersections where both curves are given as $y = k_1 x^a (x-c)^b$ and $y = k_2 x^p (x-c)^q$.
Subtract, factor out $x^{\min(a,p)}(x-c)^{\min(b,q)}$, then solve the residual polynomial factor.
Equation $x^2(x-1) = x(x-1)^2$.
Rearrange and factor out $x(x-1)$ to get $x(x-1)(x - (x-1)) = x(x-1) = 0$, giving $x = 0, 1$ without losing roots.

Common Misreadings

  • Seeing shared factors and immediately dividing both sides by them, silently discarding the roots where those factors are zero.
  • Treating the equation as if it must be expanded into standard polynomial form before any factoring can be attempted.
  • Misidentifying $(x-4)^2$ and $(x-4)$ as different factors with no common part, instead of recognising $(x-4)$ as the shared piece.

Pitfalls

  • Arithmetic slips when combining fractional coefficients inside the residual bracket (e.g. computing $1.2/0.3$ incorrectly).
  • Forgetting a sign when subtracting the two sides, producing the wrong linear residual factor.
  • Stopping after factoring without simplifying the bracket, then mis-reading the remaining roots.
  • Dividing by a variable expression without first checking whether it can be zero, thereby losing valid solutions.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics: uses only basic factorisation, the zero-product property, and arithmetic with rational coefficients. The same move recurs in P2/P3 trigonometric equations and in curve-intersection problems throughout the IAL syllabus.
