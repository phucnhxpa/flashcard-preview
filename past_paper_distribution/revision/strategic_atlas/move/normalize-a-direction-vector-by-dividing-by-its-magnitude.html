Vectors

Normalize a direction vector by dividing by its magnitude

Family: Unit Vector Extraction Frequency: 1 questions

Trigger — When to use

Fires when a question asks for a unit vector parallel (or perpendicular) to a given line, or for a position vector of length 1 in a specified direction. Cues include phrases like 'unit vector in the direction of', 'parallel to l where l: r = a + λd', or 'find OA where |OA| = 1'. The direction is already available (from a line equation, two points, or a cross product) but its magnitude is not 1.

Motivation — Why this works

For any non-zero vector v, the vector v/|v| has magnitude 1 and points in the same direction, since |kv| = |k||v| with k = 1/|v|. This is the canonical way to standardize length while preserving direction. It works in 2D and 3D identically because |v| = √(v·v) is defined the same way.

Thought Process — The step-by-step reasoning

  1. Notice the question demands magnitude exactly 1, not just 'a vector along l'.
  2. Rule out using the position vector a from r = a + λd — that locates a point on the line, not its direction.
  3. Commit to extracting d (the coefficient of λ) as the raw direction, then rescaling.
  4. Bridge: compute |d| = √(d₁² + d₂² + d₃²) so the scalar 1/|d| is ready.
  5. Decide the sign: if a specific endpoint A is named, pick the sign of d/|d| consistent with A's location; otherwise state ±.

Mechanic — Concrete operations

  1. Identify the direction vector d from the given line/segment (coefficient of the parameter, or B − A for a segment).
  2. Compute |d| = √(d₁² + d₂² + d₃²).
  3. Form the unit vector û = d/|d|, simplifying component-wise.
  4. If required, choose sign ±û based on the geometric context (e.g. which endpoint is meant).

Worked Micro-Example

Line $l: \mathbf{r} = (1,0,2) + \lambda(4,4,-2)$. Direction $\mathbf{d} = (4,4,-2)$, so $|\mathbf{d}| = \sqrt{16+16+4} = 6$. Hence the unit vector parallel to $l$ is $\hat{\mathbf{d}} = \tfrac{1}{6}(4,4,-2) = \left(\tfrac{2}{3},\tfrac{2}{3},-\tfrac{1}{3}\right)$.

Variants — Same move, different surface

Unit vector from two points A and B
Use d = B − A, then divide by |AB|; sign is fixed by the order A→B.
Unit normal to a plane in 3D
Take the normal n from the plane equation (or from a cross product) and normalize n/|n|.
Unit normal to a 2D line ax + by = c
Normalize the coefficient vector (a,b) by √(a²+b²).
Unit vector along resultant force/velocity
Compute the resultant vector first (sum of components), then divide by its magnitude.

Common Misreadings

  • Confusing the position vector a in r = a + λd with the direction vector d and normalizing the wrong one.
  • Treating 'parallel to l' as requiring a specific point on l rather than just a direction.
  • Assuming the given d is already a unit vector without checking |d|.

Pitfalls

  • Arithmetic slip in |d|: forgetting that (−2)² = 4, not −4.
  • Leaving the answer as d without dividing, or dividing by |d|² instead of |d|.
  • Ignoring the ± ambiguity when the problem geometry forces one specific direction.
  • Failing to simplify fractions, e.g. leaving 4/6 instead of 2/3.

A-level Scope Notes

Fully within Edexcel IAL Core Pure / FP scope: vector lines r = a + λd, magnitude via Pythagoras, and unit vectors in 2D and 3D.
