Sketch exponential y = a·b^x via intercept and horizontal asymptote

Family: Exponential Curve Analysis Frequency: 1 questions

Trigger — When to use

The question asks to 'sketch' a curve of the form y = a·b^x (or y = a·b^x + c) where b > 0, b ≠ 1, and no calculus or transformation chain is required. Cues include phrases like 'sketch the curve', 'show the graph of y = 3·4^x', or 'draw y = 2^x on the axes'. The move fires whenever the function is purely exponential and the marker expects key features (intercept, asymptote, shape) rather than plotted points.

Motivation — Why this works

For y = a·b^x with a > 0 and b > 1, three features fully determine the sketch: the y-intercept at (0, a) since b^0 = 1, the absence of any x-intercept since b^x > 0 for all real x, and the horizontal asymptote y = 0 arising from b^x → 0 as x → −∞. These features, together with monotonic increase, are exactly what Edexcel mark schemes reward — labelling them earns the available marks without any further computation.

Thought Process — The step-by-step reasoning

  1. Notice the function is of the form y = a·b^x — a pure exponential, so I expect a standard 'J-shape' curve.
  2. Rule out needing a table of values or calculus: 'sketch' means key features only, not an accurate plot.
  3. Commit to the three-feature checklist: y-intercept, x-intercept (or absence thereof), and horizontal asymptote.
  4. Substitute x = 0 to read off the y-intercept as (0, a), and check the sign of b^x to confirm no x-intercept.
  5. Decide curve direction from b: b > 1 gives an increasing curve approaching y = 0 from above on the left; 0 < b < 1 gives a decreasing curve. Label intercept and asymptote on the diagram.

Mechanic — Concrete operations

  1. Identify a and b in y = a·b^x and confirm a > 0, b > 1 (or note 0 < b < 1 for decay).
  2. Compute y-intercept: substitute x = 0 to get y = a, giving point (0, a).
  3. State that b^x > 0 for all x, so y > 0 for all x and there is no x-intercept.
  4. State the horizontal asymptote y = 0 (the x-axis), justified by b^x → 0 as x → −∞.
  5. Draw a smooth monotonic curve through (0, a), approaching the x-axis on the left and rising steeply on the right; label (0, a) and the asymptote y = 0.

Worked Micro-Example

Sketch $y = 3 \cdot 4^x$. At $x = 0$: $y = 3 \cdot 4^0 = 3$, so the curve passes through $(0, 3)$. Since $4^x > 0$ for all $x$, there is no $x$-intercept, and as $x \to -\infty$, $4^x \to 0$, giving horizontal asymptote $y = 0$. The sketch is an increasing curve through $(0, 3)$, approaching the $x$-axis on the left and rising steeply on the right.

Variants — Same move, different surface

y = a·b^x with 0 < b < 1 (exponential decay)
The curve is decreasing: passes through (0, a), rises steeply on the left, approaches y = 0 from above on the right. Asymptote and intercept logic are identical.
y = a·b^x + c (vertical shift)
Y-intercept becomes (0, a + c); horizontal asymptote shifts to y = c. Check whether the curve crosses the x-axis by solving a·b^x + c = 0 (only possible if a and c have opposite signs).
y = a·b^{x − k} (horizontal shift)
Y-intercept becomes (0, a·b^{−k}); asymptote remains y = 0; the 'characteristic point' (k, a) replaces (0, a) as the cleanest reference.
y = e^{kx} or y = a·e^{kx}
Same template with b = e^k; sign of k determines growth (k > 0) or decay (k < 0). Y-intercept is (0, a), asymptote y = 0.
y = a − b·e^{−kx} (bounded growth, e.g. modelling)
Asymptote is y = a as x → ∞; y-intercept is (0, a − b). Curve approaches y = a from below if b > 0.

Common Misreadings

  • Treating the curve as crossing the x-axis far to the left rather than approaching it asymptotically.
  • Confusing y = a·b^x with y = a + b^x and placing the asymptote incorrectly.
  • Reading 'sketch' as 'plot accurately' and wasting time on a table of values rather than labelling features.
  • Mistaking the y-intercept of y = a·b^x as (0, 1) by ignoring the coefficient a.

Pitfalls

  • Forgetting to label the y-intercept (0, a) and the asymptote y = 0 — both are typically required for full marks.
  • Drawing the curve touching or crossing the x-axis instead of approaching it asymptotically.
  • Drawing a straight line or a parabola-like shape rather than the characteristic exponential curvature.
  • Mislabelling the asymptote as x = 0 (the y-axis) instead of y = 0 (the x-axis).

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P2/P3) treatment of exponential functions and curve sketching; no calculus or limits beyond informal end-behaviour reasoning is required.
