Polar Coordinates

Convert 'tangent perpendicular to initial line' into dx/dθ = 0

Family: Geometric Reinterpretation Frequency: 1 questions

Trigger — When to use

Fires whenever a polar question asks for points where the tangent is 'perpendicular to the initial line', is 'vertical', or 'parallel to the y-axis'. Equivalent phrasings include 'tangent at right angles to the polar axis' or 'extreme x-coordinate on the curve'. The cue is a geometric tangent-direction condition stated on a curve given in the form r = f(θ).

Motivation — Why this works

The initial line is the positive x-axis, so a tangent perpendicular to it is vertical. Since x = r cos θ traces the horizontal coordinate as θ varies, a vertical tangent occurs exactly where x is momentarily stationary, i.e. dx/dθ = 0 (with dy/dθ ≠ 0). This converts a geometric condition into a single trigonometric equation in θ, avoiding the indeterminate dy/dx → ∞ route.

Thought Process — The step-by-step reasoning

  1. Notice the condition is about tangent direction, not about distance from the pole, so dr/dθ is the wrong tool.
  2. Rule out dy/dθ = 0 (that gives horizontal tangents, parallel to the initial line) and rule out chasing dy/dx directly (messy infinity).
  3. Commit to the cleanest equivalent: vertical tangent ⇔ x stationary ⇔ dx/dθ = 0.
  4. Bridge: write x = r(θ) cos θ explicitly using the given r.
  5. Tactically use the product rule, then simplify with double-angle or factor identities to solve for θ in the required range.

Mechanic — Concrete operations

  1. Write x = r cos θ, substituting the given r = f(θ).
  2. Differentiate using the product rule: dx/dθ = r'(θ) cos θ − r(θ) sin θ.
  3. Set dx/dθ = 0 and simplify using trig identities (double angle, factorisation).
  4. Solve for θ within the stated range; reject any θ where also dy/dθ = 0 (cusp/pole).
  5. Substitute θ back into r to obtain the polar coordinates of P.

Worked Micro-Example

For $r = 3\sin 2\theta$, set $x = 3\sin 2\theta\cos\theta$. Then $\dfrac{dx}{d\theta} = 6\cos 2\theta\cos\theta - 3\sin 2\theta\sin\theta = 0$. Using $\sin 2\theta = 2\sin\theta\cos\theta$ gives $6\cos\theta(\cos 2\theta - \sin^2\theta) = 0$, so $\cos\theta = 0$ or $\cos 2\theta = \sin^2\theta$. In $0<\theta<\pi/2$ this yields $\tan^2\theta = 1/3$, i.e. $\theta = \pi/6$, giving $r = 3\sin(\pi/3) = 3\sqrt{3}/2$.

Variants — Same move, different surface

Tangent parallel to the initial line (horizontal tangent)
Set dy/dθ = 0 with y = r sin θ instead.
Tangent making a given angle α with the initial line
Set dy/dθ = tan α · dx/dθ (treating gradient = tan α).
Furthest point from the initial line (max |y|)
Same as horizontal-tangent case: dy/dθ = 0, then check it is a maximum.
Furthest point from the line θ = π/2 (max |x|)
Same as vertical-tangent case: dx/dθ = 0.

Common Misreadings

  • Reading 'perpendicular to the initial line' as horizontal and using dy/dθ = 0.
  • Using dr/dθ = 0, which locates extrema of r (distance from pole), not vertical tangents.
  • Trying to set dy/dx = 0 for a vertical tangent (it should be infinite, not zero).

Pitfalls

  • Forgetting the product rule because r depends on θ — writing dx/dθ = r(−sin θ) only.
  • Cancelling a common factor (e.g. cos θ) and losing a valid solution.
  • Failing to check the answer lies in the stated range for θ, or accepting θ at the pole where both dx/dθ and dy/dθ vanish.

A-level Scope Notes

Standard Edexcel IAL Further Pure polar-coordinates technique: tangents parallel/perpendicular to the initial line via dx/dθ and dy/dθ, with x = r cos θ, y = r sin θ.
