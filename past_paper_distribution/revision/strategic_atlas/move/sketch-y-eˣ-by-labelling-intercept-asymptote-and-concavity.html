Integration

Sketch y = eˣ by labelling intercept, asymptote, and concavity

Family: Curve Sketching Checklist Frequency: 1 questions

Trigger — When to use

Fires when a question says 'sketch y = eˣ' (or a simple transformation) and explicitly asks for the exact y-intercept, asymptote, or 'showing key features'. Verbs like 'showing', 'stating', or 'labelling' signal that specific coordinates/lines must appear on the diagram. Also fires when the curve appears as part of a larger sketch (e.g. with a line or another exponential) and reference points are needed for an intersection argument.

Motivation — Why this works

Exponential sketches are graded on a small checklist: correct shape (monotone increasing, concave up), correct intercept (0,1) from e⁰ = 1, and correct asymptotic behaviour (y = 0 as x → −∞, unbounded as x → +∞). Each cue word in the question maps to a specific mark, so identifying which features are mark-bearing prevents wasted effort and missed marks. The underlying principle is that eˣ > 0 everywhere with derivative eˣ > 0, forcing the standard shape.

Thought Process — The step-by-step reasoning

  1. I notice the question pairs 'sketch' with 'showing the exact y-intercept' — that phrasing means a labelled coordinate is mark-bearing.
  2. I rule out plotting tables of values or computing extra points; only the intercept and asymptote earn marks here.
  3. I commit to the standard exponential template because eˣ has fixed shape: positive, increasing, concave up, with y = 0 as the only asymptote.
  4. Bridging to operations: substitute x = 0 to lock in (0,1), then orient the asymptote as x → −∞.
  5. Tactically, I draw the curve hugging the x-axis on the left, passing visibly through (0,1), and steepening to the right — and I label (0,1) and 'y = 0' explicitly.

Mechanic — Concrete operations

  1. Substitute x = 0 into y = eˣ to obtain y = e⁰ = 1, giving the y-intercept (0, 1).
  2. Note that eˣ > 0 for all x, so y = 0 is a horizontal asymptote as x → −∞.
  3. Draw a smooth, increasing, concave-up curve approaching the x-axis on the left and rising steeply on the right.
  4. Label the point (0, 1) and the asymptote y = 0 (if asked) on the diagram.

Worked Micro-Example

Sketch y = eˣ, showing the exact y-intercept. At x = 0: y = e⁰ = 1, so the curve passes through (0, 1). Since eˣ > 0, the x-axis (y = 0) is a horizontal asymptote as x → −∞. Draw an increasing, concave-up curve through (0, 1), labelled.

Variants — Same move, different surface

y = e⁻ˣ
Reflect in the y-axis: still through (0, 1), but decreasing, with y = 0 as asymptote as x → +∞.
y = eˣ + k (vertical shift)
y-intercept becomes (0, 1 + k); horizontal asymptote moves to y = k.
y = e^(x − a) (horizontal shift)
y-intercept becomes (0, e^(−a)); asymptote remains y = 0; shape unchanged.
y = ln x (inverse curve)
Reflect y = eˣ in y = x: passes through (1, 0), vertical asymptote x = 0, increasing and concave down.
y = aˣ with a > 1
Same checklist: through (0, 1), asymptote y = 0, increasing and concave up.

Common Misreadings

  • Treating eˣ as crossing the x-axis and sketching a curve that dips to or below y = 0.
  • Labelling the y-intercept as (0, e) instead of (0, 1), confusing e¹ with e⁰.
  • Reading 'sketch' as needing a precise plot rather than a labelled qualitative diagram.

Pitfalls

  • Drawing the curve as nearly straight, losing the concave-up mark.
  • Omitting the explicit label on (0, 1) so the 'showing' mark is not awarded.
  • Letting the left-hand tail touch or cross the x-axis instead of approaching it asymptotically.

A-level Scope Notes

Within Edexcel IAL Pure scope: standard exponential and logarithm sketches with intercepts, asymptotes, and simple transformations (P2/P3). No formal limit notation required — qualitative asymptotic language suffices.
