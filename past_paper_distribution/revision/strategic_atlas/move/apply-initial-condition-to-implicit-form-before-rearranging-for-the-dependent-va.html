Differential Equations

Apply initial condition to implicit form before rearranging for the dependent variable

Family: Strategic Ic Timing Frequency: 1 questions

Trigger — When to use

After separating variables and integrating, you have an implicit relation like (H−5)/(H+3) = Ae^(−t/5) (or ln|...| = ... + C) containing an unknown constant A (or C), and you are given an initial condition such as H(0) = 13. The implicit form is algebraically tidy, while solving explicitly for the dependent variable would produce a fraction or exponential expression with A entangled in numerator and denominator. The cue is: 'I could rearrange for H now, or I could substitute first — which is cleaner?'

Motivation — Why this works

The constant of integration is just a number; it does not need the explicit form to be pinned down. Substituting (t,H) = (0,13) into the compact implicit equation typically reduces to one arithmetic step because e^0 = 1 and the left side becomes a simple ratio. Rearranging first creates a Möbius-type expression in A whose inversion to find A is algebraically heavier and error-prone. Choosing the order of operations — IC before rearrangement — is a genuine efficiency decision rewarded in IAL mark schemes.

Thought Process — The step-by-step reasoning

  1. Notice that the post-integration form (H−5)/(H+3) = Ae^(−t/5) still contains the unknown A, and the IC H(0)=13 is available.
  2. Rule out the temptation to first 'finish' by solving for H explicitly; that would give H as a Möbius function of Ae^(−t/5), and substituting then solving for A is messier.
  3. Commit to substituting t=0, H=13 directly into the implicit form to extract A in a single line.
  4. Bridge: evaluate (13−5)/(13+3) and set equal to A·e^0 = A.
  5. Tactical: once A is a clean number (here 1/2), then rearrange for H with that numerical A so cross-multiplication produces small integer coefficients.

Mechanic — Concrete operations

  1. Stop after integration at the implicit form, e.g. (H−5)/(H+3) = Ae^(−t/5).
  2. Substitute t = 0 and the given H value: (13−5)/(13+3) = A·e^0.
  3. Simplify arithmetic: 8/16 = A, hence A = 1/2.
  4. Rewrite the particular solution: (H−5)/(H+3) = (1/2)e^(−t/5).
  5. Only now rearrange for H: cross-multiply, collect H terms on one side, factorise, and divide.

Worked Micro-Example

Separating dH/dt = −(1/5)(H−5)(H+3)/8 (say) leads after integration to $\frac{H-5}{H+3} = Ae^{-t/5}$. With $H(0)=13$: $\frac{13-5}{13+3} = A\cdot 1 \Rightarrow A = \tfrac{1}{2}$. Hence $\frac{H-5}{H+3} = \tfrac{1}{2}e^{-t/5}$, then cross-multiply $2(H-5) = (H+3)e^{-t/5}$ and solve for $H$.

Variants — Same move, different surface

Logarithmic implicit form ln|y−a| − ln|y−b| = kt + C
Substitute the IC directly to find C as a single logarithm, then exponentiate; avoid combining into a single fraction with unknown C first.
Newton's law of cooling: ln|θ − θ_room| = −kt + C
Plug t=0, θ=θ₀ to get C = ln|θ₀ − θ_room| immediately, rather than exponentiating with C unknown.
Logistic-type ODE giving (P/(N−P)) = Ae^(kt)
Use IC P(0)=P₀ in the ratio form: A = P₀/(N−P₀), then rearrange for P.
Integrating-factor solution y·μ(x) = ∫... + C
Apply IC before dividing through by μ(x), so C is found from a clean linear equation.

Common Misreadings

  • Thinking the 'general solution' must be presented in explicit form y = f(x) before any constant can be evaluated.
  • Believing the IC must be applied at the very last step of the problem.
  • Confusing this with absorbing C into a new constant A — the timing decision is separate from the relabelling.

Pitfalls

  • Rearranging for H with general A first, producing H = (−5A e^(−t/5) − 3)/(A e^(−t/5) − 1) and then having to solve a fractional equation for A.
  • Forgetting that e^0 = 1 when substituting t = 0, leaving an unsimplified A·e^0.
  • Sign errors when later cross-multiplying with the numerical A if brackets are dropped.

A-level Scope Notes

Fully within Edexcel IAL Pure/Further Pure separable-variables and first-order linear ODE work; uses only standard log/exponential manipulation and rational rearrangement.
