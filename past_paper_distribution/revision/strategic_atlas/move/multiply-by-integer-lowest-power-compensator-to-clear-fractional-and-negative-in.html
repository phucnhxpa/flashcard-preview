Algebra & Functions

Multiply by integer × lowest-power compensator to clear fractional and negative indices

Family: Strategic Multiplier Selection Frequency: 1 questions

Trigger — When to use

Fires when an equation set to zero contains a mix of fractional/negative powers of x (e.g. x^{1/2}, x^{-1/2}, x^{-3/2}) together with fractional coefficients, and the question asks to show it is equivalent to a polynomial in integer powers. Typical phrasings: 'show that f'(x) = 0 reduces to ...', 'hence show that x satisfies 3x² + 10x − 25 = 0'. The cue is a target equation with integer coefficients and integer powers, while the current form has half-integer indices and halved coefficients.

Motivation — Why this works

Multiplying every term by x^k shifts all exponents up by k, so choosing k equal to the magnitude of the most negative exponent converts every power to a non-negative integer. Multiplying additionally by the LCM of the coefficient denominators clears fractions. Combining both into a single multiplier 2x^{3/2} (or analogous) simultaneously kills denominators and lifts indices, producing a clean polynomial in one stroke. The RHS is 0, so the multiplication does not introduce extraneous solutions provided x ≠ 0 (which is automatic here since negative powers already exclude x = 0).

Thought Process — The step-by-step reasoning

  1. Notices the equation has powers x^{1/2}, x^{-1/2}, x^{-3/2} and halved coefficients 3/2, 25/2 — so it is not yet a polynomial.
  2. Rules out factoring out a single power (e.g. x^{-3/2}) because that leaves the fractional coefficients untouched, and rules out squaring because the equation is a sum of three terms, not an isolated surd.
  3. Commits to a single multiplier of the form c·x^k because the target is a polynomial: lifting all powers and clearing all denominators in one move is the cleanest route.
  4. Bridges to operation by setting k = 3/2 (to neutralise the most negative power) and c = 2 (the LCM of the coefficient denominators), giving the multiplier 2x^{3/2}.
  5. Tactically, distributes 2x^{3/2} across each term, using the index law x^{a}·x^{b}=x^{a+b}, and verifies each new exponent lands on a non-negative integer before collecting the result.

Mechanic — Concrete operations

  1. Identify the most negative power of x present (here x^{-3/2}); the compensating power is x^{3/2}.
  2. Identify the LCM of denominators of the coefficients (here LCM(2,1,2) = 2).
  3. Form the multiplier: 2·x^{3/2}.
  4. Multiply each term: 2x^{3/2}·(3/2)x^{1/2} = 3x², 2x^{3/2}·5x^{-1/2} = 10x, 2x^{3/2}·(−25/2)x^{-3/2} = −25.
  5. Write the resulting equation: 3x² + 10x − 25 = 0, confirming it matches the required form.

Worked Micro-Example

Given $\tfrac{3}{2}x^{1/2} + 5x^{-1/2} - \tfrac{25}{2}x^{-3/2} = 0$, multiply through by $2x^{3/2}$: $2x^{3/2}\cdot\tfrac{3}{2}x^{1/2} = 3x^{2}$, $\;2x^{3/2}\cdot 5x^{-1/2} = 10x$, $\;2x^{3/2}\cdot(-\tfrac{25}{2})x^{-3/2} = -25$. Hence $3x^{2} + 10x - 25 = 0$, as required.

Variants — Same move, different surface

Trig equation mixing sec²x and cosx terms set to zero
Multiply by cos²x (or appropriate cos^n x) to clear secants and produce a polynomial in cos x.
Equation in √x and 1/√x, e.g. a√x + b/√x = c
Multiply by √x to obtain ax + b = c√x, then square if needed.
Rational equation with denominators (x−1) and (x+2)
Multiply by the LCM (x−1)(x+2) — same principle, with linear factors instead of fractional powers.
Stationary-point condition from y = ax^{p} + bx^{q} with p,q fractional
After differentiating, multiply by x raised to the magnitude of the most negative resulting index to obtain a polynomial in x.

Common Misreadings

  • Multiplying only by x^{3/2} and forgetting the factor 2, leaving (3/2)x² + 5x − 25/2 = 0 instead of the required integer-coefficient form.
  • Multiplying by x^{1/2}, which only partially lifts the indices and leaves an x^{-1} term.
  • Trying to square the equation to remove surds, which is inappropriate when the surds appear as separate additive terms rather than as an isolated radical.

Pitfalls

  • Index arithmetic slips when adding 3/2 to negative half-integer powers, e.g. writing (−1/2)+(3/2) = 2 instead of 1.
  • Sign errors on the −25/2 term when distributing the multiplier.
  • Forgetting to multiply the RHS; here RHS = 0 so it is harmless, but the habit causes errors elsewhere.
  • Failing to note the implicit domain x > 0 (required for x^{1/2} to be real), which can matter when later checking solutions of the resulting polynomial.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths: combines index laws (P1) with stationary-point work and 'show that' manipulation (P1/P2). No techniques beyond standard A-level algebra are required.
