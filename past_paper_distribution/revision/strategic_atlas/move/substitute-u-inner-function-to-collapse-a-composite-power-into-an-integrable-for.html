Integration

Substitute u = inner function to collapse a composite power into an integrable form

Family: Substitution Bridge Frequency: 2 questions

Trigger — When to use

The integrand contains a composite expression of the form [f(x)]^n in the denominator (or under a root), where f(x) is either linear (like 1−5x) or a low-degree polynomial (like 2x²+3). Either (a) the rest of the integrand is proportional to f'(x), signalling a clean reverse-chain-rule, or (b) f(x) is linear so x itself can be re-expressed in u and the whole integrand rewritten cleanly. Visual cues: a bracketed expression raised to a power ≥ 2, an awkward x dx or polynomial factor multiplying or dividing it, and no obvious antiderivative by inspection.

Motivation — Why this works

Substituting u = f(x) replaces the composite [f(x)]^n with the simple monomial u^n, which is directly integrable by the power rule (or sets up clean partial fractions if other factors remain). When f'(x) appears as a factor, it is absorbed exactly into du, eliminating x from the integrand entirely; when f is linear, x = (1−u)/(coefficient) re-expresses any leftover x in u-space. This is the reverse chain rule in action — the very operation that produced the composite is being undone.

Thought Process — The step-by-step reasoning

  1. Notice the structural cue: a bracketed expression [f(x)]^n raised to a power, sitting in the denominator or under a root.
  2. Check whether the rest of the integrand looks like f'(x) (up to a constant). If yes, this is a textbook reverse-chain-rule. If f(x) is linear, substitution still works because x can be solved in terms of u.
  3. Rule out alternatives: partial fractions directly in x would require expanding [f(x)]^n (laborious and error-prone); integration by parts has no product of unrelated function families to exploit; trig substitution is overkill for polynomial inner functions.
  4. Commit to u = f(x). Compute du = f'(x) dx and rearrange to express dx in terms of du (and x if necessary).
  5. Tactical check: confirm that after substitution every x has either cancelled or been re-expressed in u. If a stray x remains and f is linear, solve x = (constant − u)/coefficient and substitute. If f is nonlinear and a stray x remains, the substitution is not clean — reconsider.
  6. If the integral is definite, transform the limits x = a, x = b into u-values immediately to avoid back-substitution at the end.

Mechanic — Concrete operations

  1. Identify the inner function f(x) (the base of the composite power) and set u = f(x).
  2. Differentiate: du = f'(x) dx, hence dx = du / f'(x).
  3. If a leftover x remains and f is linear (f(x) = ax+b), solve for x = (u−b)/a and substitute.
  4. Rewrite the integrand entirely in u; check that all x's have vanished.
  5. For definite integrals, replace the x-limits with u-limits using u = f(x).
  6. Integrate the resulting u-expression (typically by power rule or partial fractions in u).

Worked Micro-Example

Evaluate $\int \frac{6x}{(2x^2+3)^2}\,dx$. Set $u = 2x^2+3$, so $du = 4x\,dx \Rightarrow x\,dx = \tfrac{1}{4}du$. Then $\int \frac{6x\,dx}{(2x^2+3)^2} = \int \frac{6 \cdot \tfrac{1}{4}\,du}{u^2} = \tfrac{3}{2}\int u^{-2}\,du = -\tfrac{3}{2u} + C = -\tfrac{3}{2(2x^2+3)} + C.$

Variants — Same move, different surface

Reverse-chain-rule type: numerator is proportional to f'(x), e.g. $\int \cos x\, e^{\sin x}\,dx$
Set u = sin x; cos x dx is absorbed into du, leaving $\int e^u\,du$ — pure power/exp integration.
Linear inner function with extra polynomial factor, e.g. $\int \frac{1}{x(1-5x)^3}\,dx$
Set u = 1−5x and also write x = (1−u)/5; integrand becomes a rational function in u, ready for partial fractions in u-space.
Surd composite, e.g. $\int x\sqrt{x^2+1}\,dx$
Set u = x²+1; 2x dx = du absorbs the x, giving $\tfrac{1}{2}\int u^{1/2}\,du$.
Definite integral
Convert the x-limits into u-limits at the substitution step rather than back-substituting at the end.

Common Misreadings

  • Treating the integrand as a candidate for partial fractions in x directly, forcing expansion of [f(x)]^n into a high-degree polynomial.
  • Attempting integration by parts when there is no genuine product of independent function families.
  • Choosing u as a constant multiple of f(x) (e.g. u = 5x instead of u = 1−5x), which fails to simplify the bracketed power.
  • Overlooking the reverse-chain-rule cue because the constant in front of f'(x) is not exactly 1.

Pitfalls

  • Replacing only the bracketed term with u while leaving dx untouched, producing a mixed x-and-u integral.
  • Arithmetic error in du: e.g. writing du = 2x dx when differentiating 2x²+3 (correct: 4x dx).
  • Sign error when f'(x) is negative (e.g. dx = −du/5 for u = 1−5x) propagating through the whole integral.
  • Forgetting to transform the limits in a definite integral, then mixing u-antiderivative with x-limits.
  • Panicking when a residual x appears, instead of re-expressing it via the inverse relation x = f⁻¹(u) for linear f.

A-level Scope Notes

Fully within Edexcel IAL P3/P4 integration scope (substitution method, often listed as 'integration by substitution'). The Edexcel formula booklet does not list specific substitutions — students are expected to choose u themselves. Avoid generalising to trigonometric or hyperbolic substitutions for nonlinear inner functions unless the question explicitly invites them (those belong to Further Pure).
