Trigger — When to use

Fires whenever a question asks for the domain or range of f⁻¹(x), and the corresponding range or domain of f is either given outright in the question stem or has just been computed in an earlier part. Visual cues: a multi-part question where part (a) asks for range/domain of f and a later part asks for domain/range of f⁻¹; an explicitly stated restricted domain like 'x > 0' or 'x ≠ 1' attached to f; the inverse f⁻¹ has been derived as a rational/surd expression whose 'natural' domain may differ from the true one. Alternative phrasings: 'state the domain of f⁻¹', 'write down the range of g⁻¹', 'give the set of values that f⁻¹ can take'.

Motivation — Why this works

By definition of an inverse function, every output of f⁻¹ is an input of f and vice versa, so domain(f⁻¹) = range(f) and range(f⁻¹) = domain(f). This duality lets the student bypass re-analysing the inverse formula (which often has a wider 'natural' domain than the true one — e.g. a horizontal asymptote of f becomes an excluded value for f⁻¹'s domain that is invisible from f⁻¹'s formula alone). It also turns a potentially error-prone asymptote/limit analysis into a one-line lookup, and acts as a consistency check between parts of the question.

Thought Process — Step-by-step reasoning

  1. Notice the question asks for the domain (or range) of f⁻¹, and check whether the dual quantity for f is already available — either stated in the stem (e.g. 'f: x ∈ ℝ, x ≠ 1') or computed in a previous part.
  2. Rule out deriving the answer from the algebraic formula of f⁻¹ directly: this risks missing exclusions caused by horizontal asymptotes of f, or losing endpoint information when f had a restricted domain like x ≥ 2.
  3. Commit to the duality: domain(f⁻¹) = range(f), range(f⁻¹) = domain(f). Decide which direction the question is asking for.
  4. Bridge to the operation: read off the dual set from the question or earlier work, taking care to preserve open vs closed endpoints exactly as they appeared.
  5. Tactical check: compare against the formula of f⁻¹ as a sanity check — vertical asymptotes of f⁻¹ should match excluded values from range(f) (which equal horizontal asymptotes of f).

Mechanic — Concrete operations

  1. Identify what is asked: domain of f⁻¹ or range of f⁻¹.
  2. Locate the dual quantity for f: range(f) if domain of f⁻¹ is asked; domain(f) if range of f⁻¹ is asked.
  3. If range(f) is needed and not yet known, find it via endpoint/limit analysis on f's restricted domain (e.g. as x → a⁺, f → L; as x → ∞, f → M) and via horizontal asymptote y = (leading coeff num)/(leading coeff den) for rationals.
  4. Write the answer in the same interval/set notation, preserving strict vs non-strict inequalities.
  5. Sanity check: vertical asymptote of f⁻¹ should equal the excluded value in range(f).

Worked Micro-Example

Given $f(x) = \frac{x+2}{x-3}$, $x \in \mathbb{R}, x \neq 3$. The horizontal asymptote is $y = 1$, so $\text{range}(f) = \mathbb{R} \setminus \{1\}$. Hence $\text{domain}(f^{-1}) = \mathbb{R} \setminus \{1\}$, and $\text{range}(f^{-1}) = \mathbb{R} \setminus \{3\}$ (the original domain of $f$).

Variants — Same move, different surface

Rational f with horizontal asymptote (e.g. f(x) = (x+5)/(3x−2))
Range of f excludes the horizontal asymptote y = 1/3, so domain of f⁻¹ is ℝ \ {1/3}; this matches the vertical asymptote of f⁻¹.
Restricted-domain f with monotonic behaviour (e.g. f(x) = 12/(x+1) on x > 0)
Compute range via endpoint limits: f(0⁺) = 12, f(∞) = 0⁺, giving range (0, 12); hence domain of f⁻¹ is 0 < x < 12.
Surd-based f with closed lower endpoint (e.g. range(f) = [41/9, 5))
Domain of f⁻¹ inherits both bounds exactly: [41/9, 5). The lower bound would be invisible from the formula of f⁻¹ alone.
Restricted quadratic (e.g. f(x) = x², x ≥ 0)
Range of f⁻¹ = √x's range = domain of f = [0, ∞); same duality, applied to range instead of domain.

Common Misreadings

  • Stating the range of f⁻¹ as the range of f (wrong direction of swap).
  • Reading the domain of f⁻¹ off the formula of f⁻¹ (e.g. setting its denominator ≠ 0) — this gives the natural domain, missing exclusions inherited from range(f).
  • Reusing the domain of f as the domain of f⁻¹ without swapping.

Pitfalls

  • Mixing up vertical and horizontal asymptotes of f: the horizontal asymptote determines exclusions in range(f) (hence domain of f⁻¹), not the vertical one.
  • Losing endpoint information — if domain(f) was x ≥ 2, the closed endpoint must transfer to range(f⁻¹) as ≥ 2.
  • Forgetting to state the domain of f⁻¹ at all — many mark schemes award a separate mark for it.

A-level Scope Notes

Standard Edexcel IAL P3 inverse-function content (Functions chapter). No formula booklet entry; relies only on the definition that f and f⁻¹ swap domain and range. Avoid invoking general bijection theory or topology — stay with the elementary 'inputs become outputs' justification.
