Binomial Expansion

Translate |X|<1 binomial convergence into an interval for x

Family: Convergence Domain Frequency: 1 questions

Trigger — When to use

The question asks for the set (or range) of values of x for which a binomial expansion is valid, or asks to 'state the values of x' after expanding (1+X)^n with n not a non-negative integer. Phrasings include 'for which values of x is the expansion valid?', 'state the range of validity', or 'find the set of values of x'. It also fires whenever a generalised binomial expansion has been written down and a follow-up part references convergence.

Motivation — Why this works

The generalised binomial series (1+X)^n with n negative or fractional is an infinite power series with radius of convergence 1 in X. So the expansion equals the original function only when |X|<1. Substituting the actual expression for X and solving the resulting inequality converts this abstract analytic condition into a concrete interval of x-values that the student can state as the answer.

Thought Process — The step-by-step reasoning

  1. Notices the part asks about validity, signalling the standard |X|<1 convergence condition for the generalised binomial.
  2. Rules out using endpoints or any n-dependent radius — for A-level the radius is always 1 in X regardless of n.
  3. Commits to identifying X exactly as it appeared inside (1+X)^n in part (a), including any sign and coefficient.
  4. Bridges to the inequality |X|<1 with that X substituted, then uses |kx|=|k||x| to isolate |x|.
  5. Tactically rearranges to |x|<1/|k| and writes the answer as a clean interval, optionally as -1/|k|<x<1/|k|.

Mechanic — Concrete operations

  1. Recall the convergence condition: (1+X)^n is valid for |X|<1.
  2. Identify X from the bracket used in the expansion.
  3. Write |X|<1 with that substitution.
  4. Use |kx|=|k||x| to extract the modulus of the coefficient.
  5. Divide both sides by |k| to obtain |x|<1/|k|.
  6. State the answer as |x|<1/|k| or equivalently -1/|k|<x<1/|k|.

Worked Micro-Example

Expand $(1-\tfrac{5x}{2})^{-2}$. Here $X=-\tfrac{5x}{2}$, so validity needs $\left|-\tfrac{5x}{2}\right|<1$, i.e. $\tfrac{5}{2}|x|<1$, giving $|x|<\tfrac{2}{5}$, or $-\tfrac{2}{5}

Variants — Same move, different surface

Expansion of $(1+3x)^{-1}$
Take $X=3x$; $|3x|<1$ yields $|x|<\tfrac{1}{3}$.
Expansion of $(4-x)^{1/2}$ rewritten as $2(1-\tfrac{x}{4})^{1/2}$
Apply $|X|<1$ to the bracketed form, so $\left|-\tfrac{x}{4}\right|<1$ giving $|x|<4$.
Expansion of $(1+x^2)^{-1/2}$
Here $X=x^2$, so $|x^2|<1$ gives $|x|<1$ (the squared variable still resolves to a bound on $|x|$).
Expansion of $\tfrac{1}{(2+3x)^2}$ rewritten as $\tfrac{1}{4}(1+\tfrac{3x}{2})^{-2}$
Use $X=\tfrac{3x}{2}$, giving $|x|<\tfrac{2}{3}$; the external constant factor doesn't affect validity.
Partial fractions producing two separate binomial expansions
Compute the validity interval for each and take the intersection (the smaller interval).

Common Misreadings

  • Stating |x|<1 directly without substituting the actual X, ignoring the coefficient.
  • Treating the minus sign in X as if it changed the inequality, producing a one-sided range.
  • Forgetting to factor out a leading constant when the bracket is not in (1+...) form, so using the wrong X.

Pitfalls

  • Writing |x|<5/2 instead of |x|<2/5 by dividing the wrong way.
  • Applying the condition to x rather than to the whole expression that played the role of X.
  • When two expansions are combined, quoting only one validity interval instead of intersecting them.
  • Including endpoints (writing ≤ instead of <) — the standard A-level statement uses strict inequality.

A-level Scope Notes

Sits squarely in Edexcel IAL C2/C4-style generalised binomial expansion: the radius of convergence in X is always 1, and answers are expressed as |x|<a or an open interval. No proof of convergence or endpoint analysis is required.
