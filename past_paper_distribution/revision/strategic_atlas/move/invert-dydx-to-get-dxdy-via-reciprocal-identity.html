Differentiation

Invert dy/dx to get dx/dy via reciprocal identity

Family: Reciprocal Derivative Frequency: 1 questions

Trigger — When to use

Fires whenever a question asks for dx/dy (or 1/(dy/dx)) after dy/dx has been found, especially with the cue word 'hence'. Also triggers when explicit inversion of y = f(x) is awkward (involves inverse trig, logs of complicated expressions, or implicit forms). Recognisable phrasings: 'hence find dx/dy', 'hence show that dx/dy = ...', or asking for the gradient of the inverse function at a point.

Motivation — Why this works

For a function that is locally one-to-one with dy/dx ≠ 0, the chain rule gives (dx/dy)·(dy/dx) = 1, so dx/dy = 1/(dy/dx). This bypasses the need to solve y = f(x) for x explicitly, which is often impossible or messy. The 'hence' cue is the examiner's signal that the previous derivative is the intended ingredient. The identity is exact, not an approximation, wherever the derivative is nonzero.

Thought Process — The step-by-step reasoning

  1. I notice the question asks for dx/dy and uses 'hence' immediately after a part that produced dy/dx.
  2. I rule out rearranging y = f(x) into x = f⁻¹(y) and differentiating from scratch — slower, error-prone, and ignores the 'hence' instruction.
  3. I commit to the reciprocal route because dx/dy = 1/(dy/dx) is a one-line identity once dy/dx is known.
  4. I write dy/dx in its current (possibly messy) form and prepare to take its reciprocal cleanly, watching for compound fractions.
  5. I simplify by flipping the fraction (a/b → b/a) and, if a target template p/[q y √(...)] is given, match coefficients to identify p and q.

Mechanic — Concrete operations

  1. State the identity dx/dy = 1/(dy/dx), valid where dy/dx ≠ 0.
  2. Substitute the expression for dy/dx obtained in the previous part.
  3. Simplify the compound fraction using 1/(a/b) = b/a.
  4. Tidy the result and, if asked, compare against a target form to read off requested constants.
  5. Note (if relevant) that the formula is valid on the stated domain because dy/dx is nonzero there.

Worked Micro-Example

Given $\dfrac{dy}{dx} = \dfrac{2y\sqrt{y^2-9}}{3}$, then $\dfrac{dx}{dy} = \dfrac{1}{\,2y\sqrt{y^2-9}/3\,} = \dfrac{3}{2y\sqrt{y^2-9}}.$ Matching against $\dfrac{p}{q\,y\sqrt{y^2-9}}$ gives $p=3,\ q=2$.

Variants — Same move, different surface

Parametric curve with x = x(t), y = y(t)
Use dy/dx = (dy/dt)/(dx/dt); to get dx/dy invert: dx/dy = (dx/dt)/(dy/dt).
Implicit relation where dx/dy is simpler
Differentiate implicitly with respect to y to get dx/dy directly, then invert to obtain dy/dx.
Gradient of inverse function at a point
If g = f⁻¹, then g'(b) = 1/f'(a) where b = f(a); evaluate dy/dx at the x-value, then take reciprocal.
Rate problems linking dV/dt and dt/dV
Same reciprocal principle: dt/dV = 1/(dV/dt) when chaining via the chain rule.

Common Misreadings

  • Ignoring the 'hence' cue and differentiating x = f⁻¹(y) from scratch (e.g. via arcsec), which wastes time and invites algebraic errors.
  • Treating dx/dy as a separate derivative needing implicit differentiation rather than a simple reciprocal.
  • Confusing dx/dy with d/dy of x expressed via y in a more complicated chain than necessary.

Pitfalls

  • Flipping a compound fraction incorrectly: 1/(a/b) = b/a, not a/b.
  • Forgetting to check dy/dx ≠ 0 at any boundary point of interest.
  • Leaving the answer as a stacked fraction instead of simplifying to a single fraction the mark scheme expects.
  • Sign errors when the original dy/dx contains a minus sign that must be carried through the reciprocal.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P3/P4) and Further Pure scope: uses only the reciprocal identity dx/dy = 1/(dy/dx) following standard differentiation, with no analysis beyond noting dy/dx ≠ 0 on the stated domain.
