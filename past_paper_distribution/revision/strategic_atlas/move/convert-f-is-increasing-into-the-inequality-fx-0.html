Differentiation

Convert 'f is increasing' into the inequality f'(x) > 0

Family: Sign Analysis Frequency: 1 questions

Trigger — When to use

The question asks for the range/set of values of x for which f(x) is 'increasing' (or 'decreasing', 'strictly increasing'). Any wording linking the behaviour of f to where it goes up or down is a cue. Often appears as a follow-up part after the student has just computed f'(x). The presence of monotonicity language alongside a differentiable function is the signal.

Motivation — Why this works

By definition, a differentiable function is increasing on an interval where its gradient is positive, so the geometric statement 'f is increasing' is equivalent to the algebraic statement f'(x) > 0. This translation reduces a calculus question to solving an inequality, which is a familiar algebraic task. For polynomial f, f'(x) is a polynomial of one degree lower, so the inequality is typically linear or quadratic and tractable by Pure 1 methods.

Thought Process — The step-by-step reasoning

  1. Notices the monotonicity keyword 'increasing' — flags this as a gradient/derivative question rather than a value question.
  2. Rules out solving f(x) > 0 (sign of the function) or f(x) = 0 (roots) — these address different questions.
  3. Commits to the criterion: increasing ⇔ f'(x) > 0 on the relevant interval, with strict inequality.
  4. Bridges by substituting the already-computed (or freshly differentiated) expression for f'(x) into the inequality.
  5. Plans the next move: solve the resulting polynomial inequality (factorise, find critical values, test sign / use a sketch).

Mechanic — Concrete operations

  1. State the criterion: f is increasing where f'(x) > 0.
  2. Differentiate (or quote f'(x) from a previous part).
  3. Write the inequality f'(x) > 0 explicitly.
  4. Solve the inequality using factorisation and a sign diagram or sketch.
  5. Express the answer as a range/set of x-values.

Worked Micro-Example

Given $f(x) = x^3 + 4x^2 - 12x + 5$, find where $f$ is increasing.\\ $f'(x) = 3x^2 + 8x - 12$. Require $f'(x) > 0$: $3x^2 + 8x - 12 > 0$.\\ Roots: $x = \dfrac{-8 \pm \sqrt{64 + 144}}{6} = \dfrac{-8 \pm \sqrt{208}}{6}$. Since the parabola opens upwards, $f'(x) > 0$ outside the roots, giving $x < \dfrac{-4 - \sqrt{52}}{3}$ or $x > \dfrac{-4 + \sqrt{52}}{3}$.

Variants — Same move, different surface

'f is decreasing' question
Set f'(x) < 0 and solve the reversed inequality.
Stationary points requested
Set f'(x) = 0 and solve the equation instead of an inequality.
'Non-decreasing' or 'increasing for all x'
Use f'(x) ≥ 0; for 'for all x' demand the quadratic discriminant ≤ 0 with positive leading coefficient.
Trigonometric f, e.g. f(x) = sin x + 2x
Same translation: solve f'(x) > 0, but the inequality is trigonometric and may need a CAST/sketch approach.
Rational or product function
Apply quotient/product rule first, then analyse the sign of the resulting expression — watch domain exclusions where the denominator vanishes.

Common Misreadings

  • Confusing 'f(x) is increasing' with 'f(x) > 0' and solving the wrong inequality.
  • Solving f'(x) = 0 and reporting only the stationary points instead of the intervals.
  • Reading 'increasing' as 'positive gradient at a single point' rather than over an interval.

Pitfalls

  • Using ≥ instead of > (or vice versa) without thinking about whether stationary points are to be included.
  • Reversing the inequality when dividing by a negative leading coefficient while solving.
  • Forgetting that for an upward parabola f'(x) > 0 holds *outside* the roots, not between them.
  • Quoting f'(x) incorrectly from a previous part — always re-check the differentiation.

A-level Scope Notes

Standard Edexcel IAL Pure (P1/P2) material: monotonicity is defined via the sign of the derivative for differentiable functions, and the resulting inequalities are solved with quadratic-inequality techniques from Pure 1.
