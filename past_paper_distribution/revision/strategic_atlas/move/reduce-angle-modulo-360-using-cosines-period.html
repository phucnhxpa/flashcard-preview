Trigonometry

Reduce angle modulo 360° using cosine's period

Family: Periodicity Reduction Frequency: 1 questions

Trigger — When to use

Fires whenever a trig expression involves an angle outside the principal interval $0° \le \theta \le 360°$ (or $-180° \le \theta \le 180°$), such as $\cos(540°)$, $\sin(810°)$, or $\cos(-450°)$. Also triggers when evaluating $f(x) = a\cos(x)$ or $a\sin(x)$ at a large $x$-value, or when sketching/reading off function values at points beyond one period. The cue is: the angle looks unfamiliar, but subtracting (or adding) a multiple of $360°$ lands it on a known exact value.

Motivation — Why this works

Cosine and sine are periodic with period $360°$, so $\cos(\theta + 360°k) = \cos\theta$ and $\sin(\theta + 360°k) = \sin\theta$ for any integer $k$. This identity lets us replace any angle by its representative in $[0°, 360°)$ without changing the trig value. Doing so converts an opaque input into a standard angle whose exact value (e.g. $0, \pm\tfrac{1}{2}, \pm\tfrac{\sqrt{2}}{2}, \pm\tfrac{\sqrt{3}}{2}, \pm 1$) can be recalled directly. It is the cheapest possible simplification — pure subtraction, no identity expansion.

Thought Process — The step-by-step reasoning

  1. Notice the angle $540°$ lies outside $[0°, 360°)$, so no immediate exact value is on hand.
  2. Rule out compound-angle expansion ($\cos(360° + 180°)$ via addition formula) as overkill — periodicity alone suffices.
  3. Commit to periodicity because cosine repeats every $360°$ and $540° - 360° = 180°$ is a standard angle.
  4. Bridge: rewrite $540° = 180° + 360°$ and drop the $360°$ inside cosine.
  5. Tactical: choose the smallest non-negative representative; if the result is still ugly, apply a further symmetry (supplementary, co-function, or $\cos(-\theta)=\cos\theta$) to land on a quadrant-1 reference angle.

Mechanic — Concrete operations

  1. Identify the angle and check whether it lies in $[0°, 360°)$.
  2. Find integer $k$ with $\theta - 360°k \in [0°, 360°)$.
  3. Replace $\cos\theta$ by $\cos(\theta - 360°k)$ using periodicity.
  4. Read off the exact value from the standard table (or apply a further reduction if needed).
  5. Substitute back into the surrounding expression.

Worked Micro-Example

Evaluate $y = 5\cos(540°)$.\ Write $540° = 180° + 360°$, so by periodicity $\cos(540°) = \cos(180°)$.\ From the standard values, $\cos(180°) = -1$.\ Therefore $y = 5 \times (-1) = -5$.

Variants — Same move, different surface

Sine at a large angle, e.g. $\sin(540°)$
Same subtraction: $\sin(540°) = \sin(180°) = 0$.
Negative angle, e.g. $\cos(-450°)$
Add $360°k$ until inside $[0°,360°)$: $-450° + 720° = 270°$, so $\cos(-450°) = \cos(270°) = 0$.
Angle in radians, e.g. $\cos(7\pi/2)$
Subtract multiples of $2\pi$: $7\pi/2 - 2\pi = 3\pi/2$, so $\cos(7\pi/2) = \cos(3\pi/2) = 0$.
Tangent (period $180°$), e.g. $\tan(390°)$
Reduce mod $180°$ instead of $360°$: $\tan(390°) = \tan(30°) = \tfrac{1}{\sqrt{3}}$.
Reduced angle still > $90°$, e.g. $\cos(210°)$
After modding, apply a supplementary/quadrant identity: $\cos(210°) = -\cos(30°) = -\tfrac{\sqrt{3}}{2}$.

Common Misreadings

  • Treating $\cos(540°)$ as if $540°$ needed the addition formula $\cos(360°+180°) = \cos 360°\cos 180° - \sin 360°\sin 180°$ instead of just periodicity.
  • Confusing periodicity ($\cos(\theta+360°)=\cos\theta$) with the supplementary identity ($\cos(180°-\theta)=-\cos\theta$).
  • Assuming the period of cosine is $180°$ (it isn't — that's tangent).

Pitfalls

  • Subtracting $180°$ rather than $360°$, which is not a period of cosine and changes the sign.
  • Using sine's reduction rules on cosine (or vice versa) when further simplification is needed.
  • Forgetting to reduce all the way: leaving an angle like $420°$ partially reduced to $60°+360°$ but then not dropping the $360°$.
  • Mismanaging negative-angle reductions by subtracting instead of adding $360°$.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics: periodicity of $\sin$, $\cos$, $\tan$ and exact values at standard angles ($0°, 30°, 45°, 60°, 90°, 180°, 270°$) are core P2/P3 content.
