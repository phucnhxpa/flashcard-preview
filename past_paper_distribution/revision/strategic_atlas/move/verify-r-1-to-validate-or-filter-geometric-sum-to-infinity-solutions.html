Sequences & Series

Verify |r| < 1 to validate or filter geometric sum-to-infinity solutions

Family: Validity Check Frequency: 3 questions

Trigger — When to use

Fires whenever the sum to infinity formula S_∞ = a/(1−r) is invoked or implied — either the question states a value of S_∞, asks you to find it, or you have used it to set up an equation. Concrete cues: the symbol S_∞ appears, the phrase 'sum to infinity' or 'the geometric series converges' is used, or you have just solved a quadratic in r and obtained two roots. Also fires when r is expressed in terms of another variable (e.g. r = d/2 or r = x/3) and you need to check the resulting value, or determine the range of the parameter.

Motivation — Why this works

The formula S_∞ = a/(1−r) is only defined when the geometric series converges, which happens if and only if |r| < 1 (so that r^n → 0). Stating and checking this condition is mathematically essential — without it the formula is meaningless and any 'solution' obtained algebraically may be extraneous. In Edexcel IAL mark schemes this is an explicit reasoning/method mark: it justifies rejecting an invalid root from a quadratic, or confirms that the answer just produced is admissible. The principle is the convergence criterion of an infinite geometric progression.

Thought Process — The step-by-step reasoning

  1. I notice S_∞ appears in the problem (or I just used a/(1−r)), so the formula's domain restriction |r| < 1 is in play — this is not optional.
  2. I rule out simply taking the algebraic answer at face value: a quadratic in r might give two roots, and even a single value of r derived via substitution could violate convergence.
  3. I commit to writing the condition |r| < 1 explicitly, because Edexcel reserves a mark for 'states or implies |r| < 1' and because it is the only principled way to discard extraneous roots.
  4. I convert whatever I have into r itself first (e.g. if r = d/2 and d = 25/14, then r = 25/28) — checking on the wrong variable is a classic error.
  5. I evaluate |r| numerically for each candidate, accept those with |r| < 1, and reject the rest, writing the reason: 'S_∞ exists only when |r| < 1'.
  6. If exactly one root survives, that is my answer; if none survive, I state that no sum to infinity exists; if the question asked for a parameter range, I solve |r| < 1 as an inequality.

Mechanic — Concrete operations

  1. Identify r explicitly (perform any conversion such as r = d/2 first).
  2. Write the convergence condition: 'S_∞ exists iff |r| < 1'.
  3. Compute |r| for each candidate value of r.
  4. Compare with 1: accept if |r| < 1, reject if |r| ≥ 1.
  5. State the conclusion in words, e.g. 'r = 6/5 rejected since |6/5| > 1; r = −1/5 accepted since |−1/5| < 1'.
  6. Substitute the surviving r into S_∞ = a/(1−r) (or report the answer / parameter range).

Worked Micro-Example

Suppose solving a problem yields the quadratic $5r^2 - r - 6 = 0$, giving $r = 6/5$ or $r = -1/5$. Since $S_\infty$ is given, require $|r| < 1$. Check: $|6/5| = 1.2 > 1$ — reject; $|-1/5| = 0.2 < 1$ — accept. Hence $r = -1/5$, and $S_\infty = a/(1-(-1/5)) = 5a/6$.

Variants — Same move, different surface

Single value of r derived via substitution (no quadratic)
Still verify |r| < 1 explicitly — it confirms the answer is valid and earns the reasoning mark; if it fails, declare no sum to infinity exists.
Common ratio depends on a parameter, e.g. r = x/3
Solve the inequality |r| < 1 to obtain the range of the parameter (here −3 < x < 3) for which S_∞ exists.
Ratio expressed via another quantity, e.g. r = d/2 in a linked AP/GP problem
Convert to r first, then test |r| < 1 — never test the condition on d.
Negative common ratio (alternating series)
The condition is still |r| < 1, not r < 1; r = −1/3 converges, r = −2 diverges.

Common Misreadings

  • Assuming |r| < 1 is automatic because the question mentions S_∞, and so omitting the explicit check.
  • Rejecting a negative root just because it is negative — negative ratios with |r| < 1 are perfectly valid.
  • Writing only 'r < 1' instead of '|r| < 1', missing roots like r = −2.

Pitfalls

  • Checking the condition on the wrong variable (e.g. on d when r = d/2) — always convert to r first.
  • Computing 1 − r incorrectly when r is negative, e.g. writing 1 − (−1/2) = 1/2 instead of 3/2.
  • Choosing the 'nicer-looking' root without stating the convergence condition — loses the reasoning mark even if the final number is correct.
  • Forgetting to write the conclusion sentence ('reject because |r| ≥ 1') — the mark is for the justification, not just the choice.

A-level Scope Notes

Stays within Edexcel IAL Pure Mathematics (P2) and the geometric series section of the formula booklet, which lists S_∞ = a/(1−r) for |r| < 1. Do not generalise to power series, ratio test, or radius of convergence — those are out of scope.
