Strategic Atlas / Matrices / Singularity Determinant
Matrices

Set determinant to zero to find singular parameter values

Family: Singularity Determinant Frequency: 1 questions

Trigger — When to use

The question asks for value(s) of a parameter (often k) that make a square matrix singular, non-invertible, or have no inverse. Equivalent phrasings include 'matrix has no inverse', 'system has non-unique solutions', or 'rows/columns are linearly dependent'. The matrix is square (typically 2×2 or 3×3) with the parameter appearing in one or more entries. Any wording that flags loss of invertibility should fire this move.

Motivation — Why this works

A square matrix A is singular if and only if det(A) = 0; this is the defining algebraic characterisation of non-invertibility. Computing det(A) as a polynomial in the parameter converts the abstract singularity condition into a concrete equation to solve. For a single-parameter family, this is more systematic than searching for explicit row dependencies. The same equation also characterises when the associated linear system Ax = b fails to have a unique solution.

Thought Process — The step-by-step reasoning

  1. Notice the keyword 'singular' (or 'no inverse', 'non-invertible') and recall the equivalence singular ⟺ det(A) = 0.
  2. Rule out hunting for row/column dependencies by inspection — feasible only for very simple matrices and error-prone with a parameter.
  3. Commit to the determinant route because it is systematic, mechanical, and produces a polynomial equation in the parameter.
  4. Choose the expansion row/column that contains the most zeros or the simplest entries (often the row where k does not appear) to minimise algebra.
  5. After expanding, simplify carefully to a clean polynomial in k, then solve det(A) = 0 and report all roots.

Mechanic — Concrete operations

  1. Identify the matrix as square and locate where the parameter k appears.
  2. Choose an expansion row or column — preferably one with zeros or constants only.
  3. Expand the determinant using the cofactor (Laplace) rule, respecting the +, −, + sign pattern.
  4. Evaluate each 2×2 minor carefully, keeping k symbolic.
  5. Collect like terms to obtain det(A) as a polynomial in k.
  6. Set det(A) = 0 and solve for k; check each root by substitution if time allows.

Worked Micro-Example

For $A = \begin{pmatrix} 2 & 2 & 2 \\ 1 & k & 2 \\ 2 & 2 & k \end{pmatrix}$, expand along row 1: $\det(A) = 2(k^2 - 4) - 2(k - 4) + 2(2 - 2k)$. Simplify: $\det(A) = 2k^2 - 8 - 2k + 8 + 4 - 4k = 2k^2 - 6k + 4$. Set $2k^2 - 6k + 4 = 0 \Rightarrow k^2 - 3k + 2 = 0 \Rightarrow k = 1 \text{ or } k = 2$.

Variants — Same move, different surface

2×2 parametric matrix asked to be singular
Use det = ad − bc directly; usually yields a linear or quadratic in k with no cofactor expansion needed.
System of linear equations with no unique solution
Rephrase as singularity of the coefficient matrix and apply the same det = 0 procedure.
4×4 matrix in Further Pure
Apply row operations to introduce zeros before expanding, or expand along the sparsest row to control complexity.
Find k so that A has a repeated eigenvalue / characteristic equation has a double root
Use det(A − λI) = 0 and impose discriminant = 0, still a determinant-to-zero move at heart.

Common Misreadings

  • Confusing singular (det = 0) with non-singular (det ≠ 0) and solving the wrong condition.
  • Reading 'has an inverse for all k except…' as requiring det ≠ 0 to be solved, instead of finding the k where det = 0.
  • Treating 'no unique solution' as needing a separate technique rather than a singularity condition.

Pitfalls

  • Sign errors in the cofactor checkerboard pattern (+, −, +) when expanding.
  • Arithmetic mistakes in 2×2 minors, especially when k appears in both entries of a column.
  • Failing to fully simplify before solving, leading to messy or incorrect polynomial roots.
  • Reporting only one root of a quadratic in k when two singular values exist.

A-level Scope Notes

Fully within Edexcel IAL Further Pure (FP1/FP2) matrix content: 2×2 and 3×3 determinants, singularity, and inverse existence. No use of rank or higher-dimensional theory beyond syllabus.
