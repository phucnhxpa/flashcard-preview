Strategic Atlas / Differential Equations / Undetermined Coefficients
Differential Equations

Substitute trial PI into ODE and equate coefficients to determine constants

Family: Undetermined Coefficients Frequency: 2 questions

Trigger — When to use

Fires once a trial particular integral (PI) has been written down with one or more undetermined coefficients (e.g. y_PI = ae^{kx}, or λsin ωx + μcos ωx, or a polynomial like ax+b) and you need to pin down the unknown constants. The cue is: the form of the PI is fixed by the RHS forcing function, but the numerical coefficients are still symbolic. Equivalently, you have a trial solution and now must 'plug in and match'. This is the standard second half of the method of undetermined coefficients, after the trial form has been chosen.

Motivation — Why this works

The trial PI is engineered so that y_PI, y'_PI, y''_PI all belong to the same functional family as the RHS (e.g. all multiples of e^{kx}, or all linear combinations of sin ωx and cos ωx). Substituting into the ODE therefore produces an equation between two expressions of the same family. Because functions like e^{kx}, sin ωx, cos ωx, 1, x, x², ... are linearly independent, the equation can hold for all x only if the coefficient of each independent function on the LHS equals that on the RHS. This converts the differential equation into a small algebraic system that is straightforward to solve.

Thought Process — The step-by-step reasoning

  1. Notice that the trial PI has been correctly chosen (no overlap with the complementary function), so substitution should produce a non-degenerate equation in the unknown constants.
  2. Rule out variation of parameters or Laplace methods — these are heavier machinery and not needed when the RHS is a simple exponential, sinusoid, or polynomial.
  3. Commit to substitution: differentiate the trial PI the required number of times, keeping the unknown coefficients symbolic.
  4. Substitute y_PI, y'_PI, y''_PI into the LHS of the ODE and simplify, factoring out the common functional form (e.g. e^{kx}) where possible.
  5. Group LHS terms by independent function type (constants, x, e^{kx}, sin ωx, cos ωx, …) and read off the coefficient of each.
  6. Equate each LHS coefficient to the corresponding RHS coefficient; solve the resulting algebraic system for the unknowns.
  7. Write the complete PI with numerical coefficients substituted in.

Mechanic — Concrete operations

  1. Differentiate the trial y_PI to obtain y'_PI and y''_PI, keeping unknowns (a, λ, μ, …) symbolic.
  2. Substitute into the ODE's LHS: e.g. 2y''_PI − 5y'_PI + 3y_PI.
  3. Simplify and factor out the common functional form (e^{kx}, or collect sin/cos/polynomial terms).
  4. If the RHS is a single exponential Ce^{kx}: divide through by e^{kx} to get an algebraic equation in the unknown.
  5. If the RHS contains multiple independent functions: equate coefficients of each independent function separately to form a simultaneous system.
  6. Solve the algebraic equation(s) for the undetermined coefficient(s).
  7. Substitute the values back into the trial form to write the final PI.

Worked Micro-Example

Solve for PI of $2y'' - 5y' + 3y = 3e^{3x}$. Trial: $y_{PI} = ae^{3x}$, so $y'_{PI} = 3ae^{3x}$, $y''_{PI} = 9ae^{3x}$. Substituting: $2(9a) - 5(3a) + 3a = 18a - 15a + 3a = 6a$, giving $6ae^{3x} = 3e^{3x}$. Hence $6a = 3 \Rightarrow a = \tfrac{1}{2}$, so $y_{PI} = \tfrac{1}{2}e^{3x}$.

Variants — Same move, different surface

RHS is a sinusoid, e.g. $\sin \omega x$ or $\cos \omega x$
Use trial $y_{PI} = \lambda\sin\omega x + \mu\cos\omega x$ (always include BOTH terms). After substitution, equate sin and cos coefficients separately to get a 2×2 linear system in λ, μ.
RHS is a polynomial, e.g. $ax + b$
Use a polynomial trial of the same degree, e.g. $y_{PI} = px + q$. Note y''_{PI} = 0 for a linear trial — track every term carefully and equate coefficients of $x$ and the constant separately.
Trial form duplicates the complementary function (resonance)
Multiply the trial by $x$ (or $x^2$ for double roots) BEFORE substituting; otherwise substitution gives 0 = RHS, which is inconsistent.
RHS is a sum of different families, e.g. $e^{2x} + \sin x$
Form the trial as the sum of the trials for each piece, then substitute and equate coefficients of each independent function separately.

Common Misreadings

  • Mistaking a resonance case (trial duplicates CF) for an ordinary case — leads to 0 = nonzero contradiction.
  • Trying only $\lambda\sin\omega x$ when the RHS is $\sin\omega x$ — derivatives produce cos terms that cannot be matched without $\mu\cos\omega x$.
  • Reading the trial coefficient off by inspection without substitution, missing the contribution of the y' and y'' terms.

Pitfalls

  • Arithmetic slips when combining numerical coefficients (e.g. $18a - 15a + 3a$).
  • Forgetting the leading coefficient on the y'' term (e.g. the 2 in $2y''$) when substituting.
  • Sign errors when collecting sin/cos terms, since differentiating cos gives −sin.
  • For polynomial trials, forgetting that y''_{PI} = 0 for a linear trial and dropping a term unnecessarily.
  • Failing to divide through by the common exponential (or factor it out) before equating, leading to clutter.

A-level Scope Notes

Standard Edexcel IAL Further Pure (FP2) technique for second-order linear ODEs with constant coefficients and standard forcing functions (exponentials, polynomials, sinusoids, and their products/sums). No formula booklet entry — the trial forms must be memorised. Avoid generalising to variable-coefficient ODEs or non-standard forcing (e.g. $\tan x$, $\sec x$), which require variation of parameters and are out of scope.
