Strategic Atlas / Integration / Direct Integration
Integration

Integrate both sides of d/dx[μy]=μQ to obtain the general solution

Family: Direct Integration Frequency: 2 questions

Trigger — When to use

This move fires once a first-order linear ODE has been multiplied by its integrating factor μ(x) and rewritten as d/dx[μ(x)·y] = μ(x)·Q(x). The visual cue is an exact-derivative LHS sitting against a function-of-x-only RHS — at this point no further manipulation of the LHS is needed, only integration. Equivalent phrasings: 'the LHS is a perfect derivative', 'we have collapsed the LHS via the reverse product rule', 'μy is now ready to be recovered by integration'. If you see (μy)' = (something in x), this is the move.

Motivation — Why this works

The integrating-factor construction is engineered so that the LHS is exactly d/dx[μy]; integrating it therefore returns μy directly with no further work. The RHS, ∫μ(x)Q(x) dx, is a single-variable integral that can be tackled with standard techniques (power rule, substitution, or integration by parts). The arbitrary constant +C produced by integration is essential — it parametrises the entire family of solution curves and is what makes this the general solution rather than a particular one. Dividing through by μ(x) finally isolates y, completing the solution to the ODE.

Thought Process — The step-by-step reasoning

  1. Notice the equation is now in the exact form d/dx[μ(x)·y] = μ(x)·Q(x), so the hard structural work is already done.
  2. Rule out further rearrangement, separation of variables, or guessing a particular integral — none are needed because the LHS is already a perfect derivative.
  3. Commit to integrating both sides with respect to x: the LHS antidifferentiates trivially to μ(x)·y.
  4. Inspect the RHS product μ(x)·Q(x) and decide on the integration technique — direct power rule if it is a polynomial, integration by parts if it is x^n·e^{kx} or x^n·sin/cos, or substitution if there is an inner function.
  5. Append +C immediately after integrating — this is the general solution stage and the constant is non-negotiable.
  6. Divide every term on the RHS (including C) by μ(x) to isolate y, giving y = (1/μ(x))[∫μQ dx + C].

Mechanic — Concrete operations

  1. Write μ(x)·y = ∫ μ(x)·Q(x) dx + C.
  2. Evaluate the RHS integral using the appropriate technique (direct integration, integration by parts, or substitution).
  3. Add the arbitrary constant +C on the RHS.
  4. Divide both sides by μ(x): y = (1/μ(x))·[∫ μ(x)·Q(x) dx + C].
  5. Simplify, and if a boundary condition is given, substitute to find C.

Worked Micro-Example

Suppose we have reduced an ODE to $\frac{d}{dx}[x^2 y] = x^2 e^x$. Integrating both sides: $x^2 y = \int x^2 e^x\,dx + C$. Using integration by parts twice, $\int x^2 e^x\,dx = (x^2-2x+2)e^x$, so $x^2 y = (x^2-2x+2)e^x + C$. Dividing by $x^2$: $y = \frac{(x^2-2x+2)e^x + C}{x^2}$.

Variants — Same move, different surface

μ(x)·Q(x) is a polynomial or simple power of x
Apply the power rule term-by-term — no by-parts needed.
μ(x)·Q(x) is x^n·e^{kx}, x^n·sin(kx) or x^n·cos(kx)
Use integration by parts (possibly repeated, or via the tabular method) — a frequent FP2 12-mark setup.
μ(x)·Q(x) contains an obvious inner function f(g(x))·g'(x)
Use substitution u = g(x) — same integration step, different mechanic.
A boundary condition y(x₀) = y₀ is supplied
After isolating y, substitute the condition to determine C and produce the particular solution.

Common Misreadings

  • Treating the LHS as still requiring product-rule expansion instead of recognising it as a single exact derivative.
  • Mistaking the equation for a separable ODE and trying to separate variables again.
  • Thinking the integrating-factor step is the final step and forgetting that integration of both sides still needs to be performed.

Pitfalls

  • Forgetting the constant of integration +C, which collapses the general solution to a particular one and loses marks.
  • Errors in integration by parts — wrong choice of u and dv, or sign mistakes in the tabular method.
  • Failing to divide the constant C by μ(x) when isolating y (writing y = (1/μ)∫μQ dx + C instead of y = (1/μ)[∫μQ dx + C]).
  • Algebraic slips when simplifying (1/μ(x))·[…], particularly when μ(x) is a power of x or an exponential.

A-level Scope Notes

Stays strictly within Edexcel IAL Further Pure 2 (first-order linear ODEs via integrating factor). The integration techniques invoked (by parts, substitution, standard integrals) are all in the IAL formula booklet. Do not generalise to higher-order linear ODEs or non-linear ODEs requiring numerical methods — those are out of scope.
