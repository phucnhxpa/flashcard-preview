Differential Equations

Choose polynomial PI of matching degree when zero is not an auxiliary root

Family: Particular Integral Ansatz Frequency: 1 questions

Trigger — When to use

The RHS of a constant-coefficient linear ODE is a polynomial in the independent variable of degree n (e.g. 15t², 4x − 1, 7). Before choosing a trial, check the auxiliary equation: if m = 0 is not a root, no term of the CF is itself a polynomial, so there is no resonance. In this case the standard trial is a general polynomial of the same degree n with all lower-order terms included.

Motivation — Why this works

For a constant-coefficient operator L, applying L to a degree-n polynomial produces another degree-n polynomial (provided the constant term of L is nonzero, i.e. m = 0 is not a root). So the map from coefficients of the trial to coefficients of the output is invertible, and matching coefficients gives a unique solution. Including all lower-degree terms is essential because differentiation feeds degree-(n−1), (n−2), … terms into the LHS that must be balanced.

Thought Process — The step-by-step reasoning

  1. Notice the RHS is a pure polynomial of degree n with no exponential or trig factor.
  2. Rule out resonance by checking whether m = 0 satisfies the auxiliary equation; if the constant term of the auxiliary is nonzero, m = 0 is not a root.
  3. Commit to the trial y_p = a_n t^n + a_{n−1} t^{n−1} + … + a_0, a full polynomial of the same degree.
  4. Compute y_p' and y_p'' symbolically and substitute into the ODE to obtain an identity in t.
  5. Equate coefficients of like powers of t from highest to lowest, solving the resulting triangular system for a_n, then a_{n−1}, …, a_0.

Mechanic — Concrete operations

  1. Confirm RHS is polynomial of degree n and that m = 0 is not a root of the auxiliary equation.
  2. Write y_p = a t^n + b t^{n−1} + … (all lower terms included down to constant).
  3. Differentiate to get y_p' and y_p''.
  4. Substitute into the LHS and collect like powers of t.
  5. Equate coefficients of t^n, t^{n−1}, …, t^0 with the RHS to form a triangular linear system.
  6. Solve top-down: highest-degree coefficient first, then back-substitute for the rest.

Worked Micro-Example

Solve y'' + 2y' − 15y = 15t² for a PI. Auxiliary m² + 2m − 15 = 0 has roots m = 3, −5; m = 0 is not a root. Try y_p = at² + bt + c, so y_p' = 2at + b, y_p'' = 2a. Substituting: 2a + 2(2at + b) − 15(at² + bt + c) = 15t². Matching: −15a = 15 ⇒ a = −1; 4a − 15b = 0 ⇒ b = −4/15; 2a + 2b − 15c = 0 ⇒ c = −38/225. Hence y_p = −t² − (4/15)t − 38/225.

Variants — Same move, different surface

RHS is a degree-1 polynomial, e.g. 6x + 1
Use trial y_p = ax + b (still include the constant).
RHS is a constant, e.g. 12
Use trial y_p = c, a single constant; equate to the RHS after substitution.
m = 0 is a simple root of the auxiliary (no y term in the ODE)
Multiply the polynomial trial by t: e.g. for RHS = 15t² use y_p = t(at² + bt + c) = at³ + bt² + ct.
m = 0 is a double root (ODE is y'' = polynomial)
Multiply the trial by t² instead.
RHS is polynomial × exponential, e.g. t²e^{kt} with k not an auxiliary root
Use trial y_p = (at² + bt + c)e^{kt}; combine forms.
RHS is polynomial × sine/cosine, e.g. t cos 2t
Use trial y_p = (at + b)cos 2t + (ct + d)sin 2t.

Common Misreadings

  • Taking the trial as y_p = at² only (matching the RHS literally) and omitting bt + c, leaving no way to cancel the lower-degree terms produced by differentiation.
  • Assuming resonance whenever the RHS 'looks like' part of the CF, without actually checking whether m = 0 (or the relevant exponent) satisfies the auxiliary equation.
  • Choosing a trial of degree higher than the RHS, which introduces unnecessary unknowns that all collapse to zero.

Pitfalls

  • Arithmetic slips when solving the triangular system, especially fractional constant terms such as c = −38/225.
  • Forgetting to differentiate fully or dropping the y_p'' = 2a term, which feeds into the constant-coefficient equation.
  • Skipping the resonance check and producing an ansatz that cannot satisfy the ODE.

A-level Scope Notes

This is the standard PI method for second-order linear constant-coefficient ODEs in Edexcel IAL Further Pure (FP2), where polynomial, exponential and trigonometric RHS forms — and their resonance modifications — are all in scope.
