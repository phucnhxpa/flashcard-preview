Trigonometry

Extract period from coefficient B using P = 2π/|B|

Family: Period Extraction Frequency: 3 questions

Trigger — When to use

The function has the form A sin(Bx + φ) + C, A cos(Bx + φ) + C, or A tan(Bx + φ) + C, and you need its period — either stated directly ('find the period'), or implicitly (matching to a graph, finding k from turning points, or selecting the smallest positive solution of a periodic equation). Visual cues: a non-unit coefficient B multiplying x inside a trig function; two turning points whose x-coordinates are given; a general solution u = u₀ + 2kπ that must be unwound for θ. Alternative phrasings: 'state the period', 'find the value of k such that…', 'the curve has a maximum at x = a and the next minimum at x = b'.

Motivation — Why this works

The base period of sin and cos is 2π, and of tan is π. Replacing x by Bx compresses the cycle horizontally by a factor of |B|, so the new period is 2π/|B| (or π/|B| for tan). This is exactly the smallest positive P satisfying sin(B(x+P)) = sin(Bx), i.e. BP = 2π. Phase shifts and vertical translations do not affect the period, only B does — so the formula isolates the one parameter that controls horizontal scaling.

Thought Process — The step-by-step reasoning

  1. Notice that the trig argument is linear in x with a non-unit coefficient B (e.g. 2x, kx, πθ/2), and the question hinges on horizontal periodicity.
  2. Consider alternatives — graphing, solving sin(B(x+P)) = sin(Bx) from scratch, or pattern-matching from a table — and reject them because the formula P = 2π/|B| is faster and exact.
  3. Commit to extracting B cleanly: read off the coefficient of x inside the trig function, ignoring phase shift φ and vertical shift C, since neither affects the period.
  4. Bridge to computation: identify whether the base function is sin/cos (period 2π) or tan (period π), and whether a squared trig term (sin²/cos², period halved) is involved.
  5. Apply P = 2π/|B|, taking absolute value if B is negative, and simplify. If working backwards (period given, find B), invert: |B| = 2π/P.
  6. If the question demands a smallest positive solution from a general form u = u₀ + 2kπ, test k = 0, 1, −1, … and pick the first k that places θ in the required domain.

Mechanic — Concrete operations

  1. Identify B as the coefficient of x in the trig argument; ignore φ and C.
  2. Choose the base period: 2π for sin/cos, π for tan, π for sin²/cos².
  3. Compute P = (base period)/|B|.
  4. If given two adjacent turning points at x = a (max) and x = b (min), set |a − b| = P/2, so P = 2|a − b|, then solve for B via |B| = 2π/P.
  5. If solving sin(Bx + φ) = ±1 for smallest positive x, write the general solution Bx + φ = u₀ + 2kπ, isolate x = (u₀ − φ + 2kπ)/B, and increment k from 0 until x > 0.

Worked Micro-Example

Given $f(x) = 3\sin(2x) + 5$: read off $B = 2$. Apply $P = \frac{2\pi}{|B|} = \frac{2\pi}{2} = \pi$. So the period is $\pi$. (If instead $f(x) = \tan(2x)$, the period would be $\frac{\pi}{2}$.)

Variants — Same move, different surface

Function involves tan(Bx)
Use base period π, so P = π/|B| instead of 2π/|B|.
Function involves sin²(Bx) or cos²(Bx)
Use the identity sin²(Bx) = (1 − cos(2Bx))/2, so the period halves to π/|B|.
Period inferred from graph (max at x = a, adjacent min at x = b)
Set P/2 = |a − b|, then back-solve B = 2π/P (or π/P for tan).
Smallest positive θ from sin(Bθ + φ) = −1
Use general solution Bθ + φ = −π/2 + 2kπ; test k = 0, 1, … and pick the first k giving θ > 0.

Common Misreadings

  • Inverting the formula and writing P = 2π·B instead of P = 2π/|B|.
  • Including the phase shift φ in the period calculation (it has no effect).
  • Treating sin²(Bx) as having period 2π/|B| instead of π/|B|.
  • Using 360°/B in a question that expects radians, or vice versa.

Pitfalls

  • Forgetting the absolute value when B is negative: sin(−2x) still has period π.
  • Mis-identifying turning points as adjacent when a full cycle separates them, causing P to be doubled or halved.
  • When unwinding θ from u = u₀ + 2kπ, forgetting that the factor in front of k scales by 1/B in the θ-domain — so the θ-period is 2π/|B|, not 2π.
  • Reporting a negative θ from k = 0 instead of advancing to k = 1 for the smallest positive solution.

A-level Scope Notes

Fully within Edexcel IAL Pure (P2/P3) trigonometry. The period formulas P = 2π/|B| (sin, cos) and P = π/|B| (tan) are standard and not in the formula booklet — they must be memorised. Avoid generalising to non-linear arguments like sin(x²), which lie outside A-level scope.
