Vectors

Use sine formula directly to obtain the line-plane angle

Family: Complement Angle Selection Frequency: 4 questions

Trigger — When to use

The question asks for the (acute) angle between a line and a plane — not between the line and the normal. The available data are the line's direction vector d and the plane's normal vector n. Equivalent phrasings: 'find the angle the line makes with the plane', 'determine the angle of inclination of the line to the plane', or being given a line equation r = a + λd together with a plane equation r·n = k and asked for the angle between them.

Motivation — Why this works

The dot product formula d · n = |d||n| cos θ measures the angle θ between d and n, i.e. the angle between the line and the normal. The angle α between the line and the plane itself is the complement: α = 90° − θ. Since sin(90° − θ) = cos θ, we can write sin α = |d · n| / (|d||n|) and read α off directly, bypassing a separate '90° − ' step. The absolute value forces the acute angle, removing dependence on the orientation chosen for d or n.

Thought Process — The step-by-step reasoning

  1. Notice the question asks for the angle between a line and a plane, and that the natural vectors available are d (line direction) and n (plane normal).
  2. Recall that d · n = |d||n| cos θ gives the angle to the normal — that is NOT what the question wants, so a naive cos⁻¹ would yield the wrong angle.
  3. Consider the option of computing θ via cos⁻¹ and then doing 90° − θ; reject it as an unnecessary extra step that invites arithmetic error.
  4. Commit to using the identity sin(90° − θ) = cos θ, which lets us write sin α = |d · n| / (|d||n|) — a one-shot formula for the desired angle α.
  5. Decide to take |d · n| (absolute value) so that the result is the acute angle regardless of how d or n was oriented; finish with α = arcsin(...).
  6. Sanity-check: if d is parallel to the plane, d · n = 0 ⇒ α = 0°; if d is perpendicular to the plane (parallel to n), |d · n| = |d||n| ⇒ α = 90°. Both consistent.

Mechanic — Concrete operations

  1. Identify d (direction vector of the line) and n (normal to the plane).
  2. Compute the scalar dot product d · n.
  3. Compute the magnitudes |d| = √(d·d) and |n| = √(n·n).
  4. Apply sin α = |d · n| / (|d| |n|).
  5. Evaluate α = arcsin(|d · n| / (|d| |n|)), converting to degrees (or radians) as required by the question.

Worked Micro-Example

Line direction d = (1, 2, 2), plane normal n = (2, -1, 2). Then d · n = 2 - 2 + 4 = 4, |d| = 3, |n| = 3, so sin α = |4|/(3·3) = 4/9. Hence α = arcsin(4/9) ≈ 26.4°.

Variants — Same move, different surface

Angle between two planes (dihedral angle)
Use cos θ = |n₁ · n₂| / (|n₁||n₂|) directly — no complement is needed because the dihedral angle equals the (acute) angle between the normals.
Angle between two lines
Use cos θ = |d₁ · d₂| / (|d₁||d₂|) on the direction vectors — again no complement, since both vectors lie along the lines themselves.
Plane given in Cartesian form ax + by + cz = d
Read off n = (a, b, c) directly from the coefficients, then proceed with sin α = |d · n| / (|d||n|).

Common Misreadings

  • Using cos α = |d · n|/(|d||n|) — this gives the angle between the line and the normal, not between the line and the plane (the single most common error on this sub-part).
  • Treating the normal n as if it were the line's direction (or vice versa), losing the geometric meaning entirely.
  • Omitting the absolute value, then obtaining an obtuse angle and reporting it without taking 180° − α.

Pitfalls

  • Forgetting the modulus on d · n, leading to a negative sine and a negative/obtuse angle.
  • Mixing up which formula (sin vs cos) belongs to which scenario (line–plane vs line–line vs plane–plane).
  • Reporting the answer in the wrong angular unit (radians when degrees are expected, or vice versa).
  • Arithmetic slips in computing |d| or |n| (forgetting to square-root, or squaring components incorrectly).

A-level Scope Notes

Fully within Edexcel IAL Core Pure / FP3 vectors. The Edexcel formula booklet lists sin θ = |d · n|/(|d||n|) for the line–plane angle and cos θ = |n₁ · n₂|/(|n₁||n₂|) for the plane–plane angle; quote these directly. Do not invoke cross products or projection-matrix arguments — those are out of scope at IAL level.
