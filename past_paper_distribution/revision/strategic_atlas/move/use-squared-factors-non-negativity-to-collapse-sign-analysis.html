Use squared factor's non-negativity to collapse sign analysis

Family: Sign Analysis Frequency: 1 questions

Trigger — When to use

Fires when a polynomial inequality f(x) > 0, ≥ 0, < 0, or ≤ 0 involves a fully factorised expression containing a factor raised to an even power, e.g. (ax−b)²(cx−d) or (x−p)⁴(x−q)(x−r). Cues include phrases like 'find the set of values of x for which f(x) > 0' applied to an already-factorised cubic/quartic. Also fires when a question hints that f(x) touches but does not cross the x-axis at some root.

Motivation — Why this works

Any real expression squared (or raised to any even power) is non-negative, equalling zero only at its own root. Therefore in a product, an even-power factor cannot flip the sign of f(x); it only contributes a possible zero. This reduces the inequality to a check on the remaining odd-power factors, replacing a full sign table or number-line argument with a one-factor analysis, with the even-power root handled as a special boundary case.

Thought Process — The step-by-step reasoning

  1. Notice that f(x) is already factorised and one bracket carries an even exponent (here, the square on (3x−2)).
  2. Rule out a full sign table or sketch: the even-power factor cannot change sign, so most of that machinery is wasted effort.
  3. Commit to the shortcut because the sign of f(x) on x ≠ 2/3 is identical to the sign of the remaining linear factor (x−4).
  4. Translate f(x) > 0 into (3x−2)² > 0 AND (x−4) > 0, which is the bridge to a single linear inequality.
  5. Decide whether the even-power root must be excluded (strict inequality) or included (non-strict), and check whether it already lies inside or outside the solution interval.

Mechanic — Concrete operations

  1. Identify the even-power factor and record the root where it vanishes (here x = 2/3).
  2. State that this factor is ≥ 0 for all real x, with equality only at that root.
  3. Reduce the inequality to a condition on the remaining (odd-power) factors only.
  4. Solve that reduced inequality (here x − 4 > 0 ⇒ x > 4).
  5. Re-examine the even-power root: include it if the inequality is non-strict and the rest of the product is ≥ 0 there; exclude it if strict.
  6. Write the final solution set, checking whether the excluded/included point is already inside the interval.

Worked Micro-Example

Solve $(3x-2)^2(x-4) > 0$. Since $(3x-2)^2 \ge 0$ with equality only at $x = \tfrac{2}{3}$, the sign of $f(x)$ on $x \ne \tfrac{2}{3}$ matches the sign of $(x-4)$. So $f(x) > 0 \iff x - 4 > 0$ and $x \ne \tfrac{2}{3}$, giving $x > 4$ (the excluded point lies outside this interval).

Variants — Same move, different surface

Quartic of the form $(x-a)^2(x-b)(x-c)$ with $b < c$
Sign of f(x) is determined by $(x-b)(x-c)$; standard two-root sign analysis gives $x < b$ or $x > c$, with $x = a$ excluded for strict inequality.
Higher even power, e.g. $(x-a)^4(x-b)$
Same principle — $(x-a)^4 \ge 0$, so reduce to the sign of $(x-b)$; the root $x = a$ is again only a touching point.
Non-strict inequality $f(x) \ge 0$ with $(x-a)^2(x-b)$
Include $x = a$ in the solution set since $f(a) = 0$ satisfies $\ge 0$; final answer is $\{a\} \cup \{x : x \ge b\}$.
Inequality $f(x) < 0$ with $(x-a)^2(x-b)$
Solve $(x-b) < 0$ and exclude $x = a$ since $f(a) = 0$ is not $< 0$; here $a$ does fall inside $x < b$ if $a < b$, so explicit exclusion matters.

Common Misreadings

  • Treating $(3x-2)^2$ as a normal sign-changing factor and inserting a sign change at $x = 2/3$ in the sign table.
  • Forgetting that the even-power root must be excluded for strict inequalities, even when it changes nothing in this particular interval.
  • Confusing 'touches the axis' with 'crosses the axis' when sketching, and reading off the wrong solution set.

Pitfalls

  • Mishandling the strict vs non-strict boundary: $\ge 0$ keeps the even-power root, $> 0$ drops it.
  • Failing to check whether the excluded even-power root lies inside the candidate interval — it must be removed if it does.
  • Cancelling the squared factor from both sides of the inequality, which is invalid because it equals zero at one point and would lose that boundary information.

A-level Scope Notes

Standard for Edexcel IAL P1–P3 polynomial inequality questions where f(x) is given fully factorised with a repeated linear factor; no calculus or graph-sketching beyond touching/crossing behaviour is required.
