Matrices

Set determinant to zero to find singular parameter values

Family: Singularity Condition Frequency: 1 questions

Trigger — When to use

Fires whenever a question presents a square matrix containing a parameter (often p, k, or λ) and asks for the value(s) making it 'singular', 'non-invertible', 'has no inverse', or 'has a non-trivial null space'. Also triggers when asked for which parameter a linear system has 'no unique solution' or 'infinitely many solutions'. The phrase 'find the value of p such that M⁻¹ does not exist' is the canonical cue.

Motivation — Why this works

A square matrix is invertible iff its determinant is non-zero, so 'no inverse' is equivalent to det(M) = 0. This converts an abstract structural property into a polynomial equation in the parameter, which is directly solvable. Row-reduction with a symbolic parameter is messy and case-prone, whereas the determinant route gives a single clean equation. The same det = 0 condition underlies eigenvalue problems and consistency of linear systems.

Thought Process — The step-by-step reasoning

  1. Notice the keywords 'no inverse' / 'singular' attached to a matrix containing a parameter p.
  2. Rule out row-reduction: branching on whether pivots are zero in p is awkward and slow.
  3. Commit to the det(M) = 0 route because it produces one polynomial equation in p directly.
  4. Bridge to action: choose the row or column for cofactor expansion that has the most zeros or simplest entries to minimise algebra.
  5. Inside the move, expand carefully tracking the +/− cofactor signs, collect like powers of p, factorise, and report all roots (a 3×3 typically yields a quadratic or cubic in p).

Mechanic — Concrete operations

  1. Identify M as the matrix whose singularity is required.
  2. Choose a row/column with simple or zero entries for cofactor expansion.
  3. Expand det(M) and simplify into a polynomial in p.
  4. Set the polynomial equal to 0.
  5. Factorise (or use the quadratic formula) to obtain all values of p.
  6. State each value clearly; check by substitution if time permits.

Worked Micro-Example

Let $M=\begin{pmatrix}p & 1 & 2\\ 0 & p-1 & 1\\ 1 & 0 & 1\end{pmatrix}$. Expanding along row 2: $\det M = -(p-1)\,(p\cdot 1 - 2\cdot 1) - (1)(0 - 1) = -(p-1)(p-2) + 1$. Setting $-(p-1)(p-2)+1 = 0$ gives $(p-1)(p-2)=1$, so $p^{2}-3p+1=0$ and $p = \tfrac{3\pm\sqrt5}{2}$.

Variants — Same move, different surface

2×2 parametric matrix
Use ad − bc = 0 directly; usually a linear or quadratic equation in p.
Consistency of a 3×3 linear system Ax = b with parameter
Singular A (det A = 0) signals 'no unique solution'; then check b against column space to distinguish no-solution vs infinite-solution cases.
Eigenvalue problem (Further Pure)
Replace M by M − λI and solve det(M − λI) = 0 for λ; identical mechanic with λ in place of p.
Geometric singularity (transformation collapses area/volume)
det = 0 is reinterpreted as the linear map being non-injective, e.g. mapping a plane onto a line.

Common Misreadings

  • Reading 'no inverse' as 'inverse has irrational/complex entries' instead of det = 0.
  • Reading 'singular' as 'unique' (linguistic confusion) and so requiring det ≠ 0.
  • Treating det(M) = 0 as a condition on x rather than on the parameter p.

Pitfalls

  • Sign errors from the cofactor checkerboard when expanding a 3×3.
  • Algebraic slips when collecting powers of p, especially distributing minus signs.
  • Discarding a valid root (e.g. dropping the negative root of a quadratic in p).
  • Stopping at the polynomial without solving for p, or forgetting to state all solutions.

A-level Scope Notes

Fully within Edexcel IAL Further Pure: 2×2 and 3×3 determinants, inverses, and singularity. Connects naturally to eigenvalue work via det(M − λI) = 0.
