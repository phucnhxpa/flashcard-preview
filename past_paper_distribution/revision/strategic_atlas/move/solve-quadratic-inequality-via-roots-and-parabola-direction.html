Solve quadratic inequality via roots and parabola direction

Family: Sign Analysis Frequency: 4 questions

Trigger — When to use

The question presents a quadratic expression (or a product of two linear factors) compared to zero with <, ≤, >, or ≥. Visual cues: a degree-2 polynomial in one variable set against 0, integer or small-fraction coefficients suggesting factorisation, or a discriminant condition like b² − 4ac > 0 that has been reduced to a quadratic in a parameter. Equivalent phrasings include 'find the set of values of k for which …', 'find the range of values of x satisfying …', or 'for what values of p is the expression positive?'. Once the roots of the quadratic are known (by factorisation or quadratic formula), this move converts the inequality directly into interval notation.

Motivation — Why this works

A quadratic ax² + bx + c with two real roots α < β factors as a(x − α)(x − β), and its graph is a parabola whose concavity is fixed by the sign of a. If a > 0 (opens upward), the expression is negative strictly between the roots and positive outside; if a < 0, the situation reverses. Knowing only the roots and the sign of the leading coefficient therefore determines the solution set without sign tables or test points. This is the standard Edexcel approach: factorise, sketch a quick parabola, read off the interval(s).

Thought Process — The step-by-step reasoning

  1. Notice the inequality is quadratic in one variable compared to 0 — the solution will be either an interval between the roots or the union of two outer rays, never both.
  2. Rule out test-point/sign-table methods as overkill, and rule out solving the equality alone (which would give only the boundary, not the inequality region).
  3. Commit to: (i) find the two roots, (ii) inspect the sign of the leading coefficient, (iii) read off the correct region from a quick U-shape or ∩-shape sketch.
  4. Bridge: try integer factorisation first (factor pair of ac summing to b); if that fails, fall back on the quadratic formula. Either way, get exact roots α < β.
  5. Tactical decision: match inequality direction to parabola shape. Upward parabola (a > 0): '< 0' ⇒ between roots (α, β); '> 0' ⇒ exterior x < α or x > β. Downward parabola (a < 0): reverse.
  6. Boundary decision: if the inequality is strict (<, >), exclude the roots (open interval); if non-strict (≤, ≥), include them (closed interval).

Mechanic — Concrete operations

  1. Rearrange the inequality to the form (quadratic) ⋛ 0 with all terms on one side.
  2. Factorise: e.g. 2x² − 9x + 4 = (2x − 1)(x − 4); if no clean factorisation, apply x = (−b ± √(b² − 4ac))/(2a).
  3. Identify the two roots α < β and the sign of the leading coefficient a.
  4. Sketch a quick parabola crossing the axis at α and β, opening upward if a > 0 or downward if a < 0.
  5. Read the solution region from the sketch: between roots if expression is below axis, exterior if above (or vice versa for downward parabola).
  6. Write the answer in interval/inequality form, using strict or non-strict signs to match the original inequality. Exterior regions must be written as a union (e.g. x < α or x > β), never as a single chained inequality.

Worked Micro-Example

Solve $k^2 - 5k + 4 \le 0$. Factorise: $(k-1)(k-4) \le 0$, so roots are $k=1, k=4$. Leading coefficient $+1 > 0$, so the parabola opens upward and the expression is $\le 0$ between the roots. Therefore $1 \le k \le 4$.

Variants — Same move, different surface

Leading coefficient is negative (downward parabola)
Regions flip: '> 0' gives the interior (α, β); '< 0' gives the exterior x < α or x > β. Alternatively, multiply through by −1 first (and flip the inequality) to restore the standard upward case.
Quadratic does not factorise over the integers
Use the quadratic formula to find α, β exactly (in surd form), then apply identical parabola-direction reasoning.
Non-strict inequality (≤ or ≥)
Include the roots as endpoints (closed interval/closed ray); the underlying region selection is unchanged.
Discriminant condition for distinct real roots, e.g. b² − 4ac > 0 yields a quadratic in a parameter k
Treat the resulting expression in k as the quadratic to be analysed; same move applies to the parameter.
Repeated root (discriminant = 0)
Parabola touches the axis at a single point; '< 0' has no solutions, '≤ 0' gives only that point, '> 0' gives all reals except the root, '≥ 0' gives all reals.

Common Misreadings

  • Treating '< 0' as 'outside the roots' for an upward parabola (this is the '> 0' region) — confusing 'between' with 'outside'.
  • Forgetting to check the sign of the leading coefficient and defaulting to 'always between the roots'.
  • For an exterior solution like x² > 9, writing only x > 3 and missing the x < −3 branch.
  • Confusing strict (<, >) with non-strict (≤, ≥) and handling the endpoints incorrectly.

Pitfalls

  • Writing an exterior region as a single chained inequality such as 'α > x > β' (impossible) instead of the union 'x < α or x > β'.
  • Sign error when splitting the middle term during factorisation, especially when ac > 0 but b < 0 (both factors are negative).
  • Using ≤ or ≥ when the original inequality is strict — Edexcel deducts marks for incorrect boundary inclusion.
  • Dividing through by a negative leading coefficient without flipping the inequality sign.

A-level Scope Notes

Fully within Edexcel IAL P1/P2 (quadratics and inequalities). No formula booklet entry is needed beyond the quadratic formula (booklet, 'Pure Mathematics — Quadratic Equations'). Avoid generalising to higher-degree polynomial inequalities via this exact shortcut — for cubics and beyond, a sign table or factor-by-factor analysis is required.
