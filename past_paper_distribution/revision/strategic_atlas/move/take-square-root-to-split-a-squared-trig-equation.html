Trigonometry

Take ± square root to split a squared-trig equation

Family: Equation Strategy Frequency: 2 questions

Trigger — When to use

The equation has the form [trig(f(x))]² = k with k a positive constant — e.g. cos²(2x+10°) = 1/3, sin²(θ) = 1/4, or tan²(2x+π/4) = 3. The visual cue is a single squared trig function isolated on one side and a positive number on the other, with no other trig terms present. Equivalent phrasings: 'cos² equals a number', 'tan² equals a number', 'a quadratic in trig with no linear term'. This is NOT the trigger when the equation is a genuine quadratic in the trig function (with a linear middle term) — that needs factoring instead.

Motivation — Why this works

If f(x)² = k with k > 0, then f(x) = +√k OR f(x) = −√k; both branches yield valid solutions and dropping either loses half the answers. Square-rooting preserves the original argument (e.g. 2x+10°) so each resulting equation is a standard inverse-trig problem, whereas using a power-reduction identity like cos²A = (1+cos2A)/2 doubles the argument and inflates the working range, multiplying the candidate-solution count. The ± split is the cleanest route into the standard 'principal value + periodicity' machinery.

Thought Process — The step-by-step reasoning

  1. Notice the equation contains a squared trig function equal to a positive constant, with no linear trig term — so it is not a true quadratic to factor.
  2. Consider the alternative of applying a power-reduction / double-angle identity (cos²A = (1+cos2A)/2 or tan²A = sec²A − 1). Reject it: it either doubles the angle (widening the solution range and multiplying branches) or introduces a second trig function, both of which complicate the work.
  3. Commit to taking square roots of both sides, retaining the ± sign because k > 0 means both signed roots are real and admissible.
  4. Bridge to operation: isolate the squared trig (divide by any leading coefficient first), then write trig(argument) = +√k or trig(argument) = −√k as two separate cases.
  5. Tactically, plan for each case to be solved by the standard recipe: principal value from inverse trig, then add the periodic family, then filter by the stated range on x — remembering to translate the range on x into a range on the argument first.

Mechanic — Concrete operations

  1. Isolate the squared trig: e.g. from 3cos²(2x+10°) = 1, divide by 3 to get cos²(2x+10°) = 1/3.
  2. Apply ± square root: cos(2x+10°) = ±1/√3 (or tan(2x+π/4) = ±√3, etc.).
  3. Split into Case 1 (positive root) and Case 2 (negative root); treat each as an independent standard trig equation.
  4. For each case, find the principal value, generate the full family of solutions using the function's period, then filter against the original range on x.

Worked Micro-Example

Solve $\cos^2(2x+10^\circ) = \tfrac{1}{3}$ for $x \in [0^\circ, 90^\circ)$. Take roots: $\cos(2x+10^\circ) = \pm \tfrac{1}{\sqrt{3}}$. Case 1: $2x+10^\circ = \cos^{-1}(1/\sqrt{3}) \approx 54.7^\circ$, giving $x \approx 22.4^\circ$. Case 2: $2x+10^\circ = \cos^{-1}(-1/\sqrt{3}) \approx 125.3^\circ$, giving $x \approx 57.6^\circ$. Both lie in range, so two solutions.

Variants — Same move, different surface

sin²(θ) = k with 0 < k < 1
Same ± logic: sin θ = ±√k; solve each via principal value plus the sine periodicity family.
tan²(θ) = k with k > 0
tan θ = ±√k; note tan θ = −√k often corresponds to π − (positive principal value), but treat as two independent cases for safety.
cos²(3x) = 1 (k = 1, boundary case)
cos(3x) = ±1 gives genuinely different solution families (3x = 2nπ vs 3x = (2n+1)π); both must be listed.
Quadratic in trig WITH a linear term, e.g. 2cos²θ − cos θ − 1 = 0
Do NOT square-root — factor as a quadratic in cos θ instead.

Common Misreadings

  • Treating cos²A = k as if it implies cos A = +√k only — losing the negative-root branch and half the solutions.
  • Mistaking the equation for one needing a power-reduction identity, then doubling the argument unnecessarily.
  • Confusing this trigger with a genuine quadratic in cos/sin/tan that has a linear middle term.

Pitfalls

  • Dropping the ± after square-rooting — the single most common error and a guaranteed loss of marks.
  • Writing trig = ±√k correctly but then only carrying the positive case through the subsequent working.
  • Forgetting to translate the x-range into a range on the argument (e.g. 2x+10°) before listing solutions, leading to missed or extraneous answers.
  • Failing to rationalise or simplify √k cleanly (e.g. leaving 1/√3 instead of √3/3), which can hide that the value is a standard exact-value angle.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P2/P3) trig-equation work. Uses only inverse trig (principal values) and periodicity, both standard. The formula booklet provides exact values for 30°/45°/60° but does not list cos²A identities — students should know cos²A + sin²A = 1 and the double-angle forms but choose the ± route here. Avoid invoking complex roots or general nth-root extensions; this move is strictly real-valued.
