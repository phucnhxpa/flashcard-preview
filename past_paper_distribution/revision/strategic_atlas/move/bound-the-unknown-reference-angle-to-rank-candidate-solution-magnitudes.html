Trigonometry

Bound the unknown reference angle to rank candidate solution magnitudes

Family: Constraint Reasoning Frequency: 1 questions

Trigger — When to use

Fires when a trig equation yields multiple candidate solutions expressed symbolically in terms of an unknown reference angle α (e.g. α − 24π and −12π − α), and the question asks which is closest to zero, largest, or smallest. The value of α is not computed numerically but is constrained — typically as the smallest positive solution. Recognise the cue 'in terms of α' combined with a comparison demand.

Motivation — Why this works

The defining property of α (smallest positive solution) implicitly bounds it: any other positive solution from a sister branch must exceed α, which yields a usable inequality on α. That inequality is exactly what is needed to compare absolute values of the symbolic candidates without numerical evaluation. This converts a seemingly under-determined comparison into a rigorous deduction, which is what examiners reward.

Thought Process — The step-by-step reasoning

  1. I notice I must rank candidates but α has no numerical value.
  2. I rule out plugging in or guessing — the answer must follow from α's defining property.
  3. I commit to extracting an inequality on α from 'α is the smallest positive solution'.
  4. I bridge by writing the next positive sister solution and demanding α be smaller, giving α < 6π.
  5. Inside the move I form |candidate₁| and |candidate₂|, subtract, and check the sign using the bound.

Mechanic — Concrete operations

  1. Identify the defining constraint on α (e.g. α is the smallest positive root).
  2. Write the next positive solution from the other branch (here 12π − α) and impose α < 12π − α to get α < 6π.
  3. Express each candidate's distance from zero: |α − 24π| = 24π − α and |−12π − α| = 12π + α.
  4. Form the comparison (24π − α) − (12π + α) = 12π − 2α and use the bound α < 6π to sign it.
  5. Conclude which candidate has smaller magnitude and state it as the answer.

Worked Micro-Example

Suppose $\cos(\theta/12) = k$ has smallest positive solution $\alpha$, and we need the negative solution closest to $0$ from candidates $\alpha - 24\pi$ and $-12\pi - \alpha$. Since the next positive root is $12\pi - \alpha > \alpha$, we get $\alpha < 6\pi$. Then $|\alpha - 24\pi| - |-12\pi - \alpha| = (24\pi - \alpha) - (12\pi + \alpha) = 12\pi - 2\alpha > 0$. So $-12\pi - \alpha$ is closer to $0$.

Variants — Same move, different surface

Equation $\sin(x/n) = c$ with α the smallest positive root, choosing largest negative solution
Use sister branch $n\pi - \alpha$ instead of $12\pi - \alpha$; bound becomes $\alpha < n\pi/2$.
α defined as smallest root in $(\pi/2, \pi)$ rather than smallest positive
Bound becomes a two-sided interval; use whichever endpoint settles the inequality.
Comparing two positive candidates for 'largest below given bound'
Same idea: subtract candidates symbolically and sign the result using the α-bound.
tan equation where solutions differ by period $\pi$
Sister-branch reasoning is replaced by period reasoning; bound on α typically half a period.

Common Misreadings

  • Treating α as if it had a known numerical value and substituting a guess.
  • Forgetting that 'smallest positive solution' is itself a usable constraint.
  • Confusing 'closest to zero' (smallest |·|) with 'largest' (most positive).

Pitfalls

  • Reversing the inequality direction and selecting the wrong candidate.
  • Dropping absolute value signs and comparing signed quantities directly.
  • Using the wrong sister-branch formula (e.g. $\pi - \alpha$ vs $2\pi - \alpha$) for the trig function involved.

A-level Scope Notes

Uses only the IAL P2/P3 toolkit: general solutions of trig equations, symmetry of sin/cos/tan, and basic inequality manipulation — no calculus or analysis beyond syllabus.
