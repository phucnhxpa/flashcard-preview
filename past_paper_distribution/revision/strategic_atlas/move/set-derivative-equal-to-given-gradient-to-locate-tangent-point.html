Differentiation

Set derivative equal to given gradient to locate tangent point

Family: Gradient Condition Frequency: 1 questions

Trigger — When to use

Fires when a question states a numerical gradient (e.g. 'gradient of the tangent is 8', 'tangent parallel to y = 8x + c', or 'point where the tangent has slope m') and asks for the coordinates of the corresponding point on a known curve. Also triggers on phrasings like 'find the point on the curve at which the gradient is k' or 'where is the tangent parallel to a given line'. The defining cue is a specified slope value attached to a tangent on a differentiable curve.

Motivation — Why this works

The derivative dy/dx evaluated at a point gives the gradient of the tangent there, so equating dy/dx to the target gradient k converts a geometric tangent condition into an algebraic equation in x. Solving yields the x-coordinate(s) of every point on the curve whose tangent has that slope. Substituting back into the original y equation recovers the y-coordinate(s). This is the standard bridge from 'slope information' to 'point information'.

Thought Process — The step-by-step reasoning

  1. Notice the phrase 'gradient of the tangent = k' — this is a slope condition, and dy/dx IS the gradient function.
  2. Rule out tangent-line construction (no need for y − y₁ = m(x − x₁) here) since the question wants the point, not the line equation.
  3. Commit to the equation dy/dx = k as the governing condition; its solutions in x are exactly the x-coordinates required.
  4. Bridge to action: differentiate the curve once to obtain dy/dx as an explicit function of x.
  5. Tactically: solve dy/dx = k (linear if curve is quadratic, quadratic if curve is cubic, etc.), then substitute each x back into the ORIGINAL y, never into dy/dx.

Mechanic — Concrete operations

  1. Differentiate y with respect to x to obtain dy/dx as a function of x.
  2. Set dy/dx equal to the given gradient k, forming an equation in x.
  3. Solve the equation for x (may be linear, quadratic, or higher).
  4. Substitute each x-value into the original equation for y to find y.
  5. State the coordinate(s) (x, y) of the tangent point(s).

Worked Micro-Example

Curve $y = \tfrac{1}{2}x^2 - 10x + 22$; find the point where the tangent has gradient $8$. Then $\frac{dy}{dx} = x - 10$. Set $x - 10 = 8 \Rightarrow x = 18$. Substitute: $y = \tfrac{1}{2}(18)^2 - 10(18) + 22 = 162 - 180 + 22 = 4$. Point: $(18, 4)$.

Variants — Same move, different surface

Cubic curve with specified tangent gradient
dy/dx is quadratic, so dy/dx = k gives a quadratic in x, potentially yielding two tangent points sharing the same gradient.
Tangent parallel to a given line y = mx + c
Read off the gradient m of the line and apply the same move with k = m.
Tangent perpendicular to a given line of gradient m
Use k = -1/m as the target gradient before setting dy/dx = k.
Stationary point (horizontal tangent)
Special case with k = 0; set dy/dx = 0 and proceed identically.
Parametric curve x = f(t), y = g(t)
Use dy/dx = (dy/dt)/(dx/dt), set equal to k, solve for t, then substitute t into x and y.
Implicit curve F(x,y) = 0
Differentiate implicitly to express dy/dx in terms of x and y, set equal to k, then solve simultaneously with the original curve equation.

Common Misreadings

  • Treating the given gradient value as a y-coordinate and substituting it into the curve equation.
  • Confusing 'gradient of the tangent' with 'value of y at the point' and skipping differentiation.
  • Assuming there is always exactly one such point, missing a second solution when dy/dx = k is quadratic.

Pitfalls

  • Substituting x back into dy/dx instead of into y, then reporting the gradient as the y-coordinate (e.g. stating (18, 8) instead of (18, 4)).
  • Algebra slips when differentiating fractional or negative-power terms before forming dy/dx = k.
  • Forgetting to find y for every x-solution when the gradient equation has multiple roots.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P1/P2 differentiation of polynomials; P3/P4 for parametric and implicit extensions). Uses only standard differentiation rules and equation-solving techniques.
