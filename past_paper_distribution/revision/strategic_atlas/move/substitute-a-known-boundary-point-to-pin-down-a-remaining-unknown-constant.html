Algebra & Functions

Substitute a known boundary point to pin down a remaining unknown constant

Family: Boundary Condition Substitution Frequency: 2 questions

Trigger — When to use

Fires whenever a model or curve has been written in a partially-parametrised form (e.g. vertex form y = a(x−h)² + k, exponential model θ = A − Be^{−kt}, factor form, etc.) and exactly one unknown constant remains to be determined. Look for explicit phrasings like 'passes through (...)', 'cuts the x-axis at...', 'initially', 'when t = 0', 'the y-intercept is...', or 'at time t = T the value is V'. Each such phrasing supplies a single (input, output) pair that the model must satisfy. The cue is structural: one parameter free + one data point given = one linear equation to solve.

Motivation — Why this works

A point lying on a curve, or a stated initial/boundary value, simply means the model equation is satisfied at that input. Substituting it converts the geometric/physical condition into an algebraic equation, and because only one parameter is unknown that equation is typically linear and immediately solvable. Choosing the RIGHT point matters: a well-chosen substitution (such as t = 0 in an exponential, where e^{0} = 1) collapses awkward terms and isolates the unknown without creating a nonlinear system. This is the canonical way to fix the remaining degree of freedom in any parametrised expression.

Thought Process — The step-by-step reasoning

  1. Notice the current form of the model has exactly one unknown left (e.g. 'a' in vertex form, or 'A' in θ = A − 180e^{−kt}).
  2. Scan the question for a data point: a coordinate, a root, an intercept, or an 'initial' / 'when t = ...' clause — each is a hidden (x, y) or (t, value) pair.
  3. Rule out heavier alternatives: setting up a simultaneous system using two unknowns at once would produce a nonlinear system (especially with exponentials) or unnecessary algebra.
  4. Commit to substitution because it converts the boundary condition directly into a single linear equation in the lone unknown.
  5. Tactically choose the most arithmetically friendly point: prefer t = 0 in exponentials (since e^0 = 1 kills the rate constant), prefer a root when the form is factored (since the RHS becomes 0), prefer the y-intercept when x = 0 simplifies a polynomial.
  6. Substitute, simplify, solve, and sanity-check the sign/size of the constant against the physical or geometric setup (e.g. a < 0 for a maximum).

Mechanic — Concrete operations

  1. Identify the lone unknown constant C in the current form of the model.
  2. Identify the boundary data point (x₀, y₀) or (t₀, v₀) supplied by the question.
  3. Substitute: replace the variable(s) with the boundary values in the model equation.
  4. Simplify any constants that collapse (e.g. (−5+2)² = 9, or e^{−k·0} = 1).
  5. Solve the resulting linear equation for C.
  6. Substitute C back into the model to obtain the fully determined expression, and verify consistency with any qualitative features (sign, max/min, asymptote).

Worked Micro-Example

Vertex form $f(x) = a(x+2)^2 + 12$ with curve passing through $(-5, 0)$. Substitute: $0 = a(-5+2)^2 + 12 = 9a + 12$, so $a = -\tfrac{4}{3}$. Therefore $f(x) = -\tfrac{4}{3}(x+2)^2 + 12$.

Variants — Same move, different surface

Exponential model θ = A − Be^{−kt} with an 'initial' value given
Substitute t = 0 so e^{−k·0} = 1; the rate constant k disappears and the equation becomes linear in A.
Vertex form with the y-intercept given instead of an x-intercept
Substitute x = 0 into a(x−h)² + k to get a single linear equation for a.
Factored form y = a(x − r₁)(x − r₂) with a third point given
Substitute the third point's coordinates directly to solve for a; no expansion needed.
Newton's law of cooling / population growth with two data points
Use the t = 0 point first to fix the additive/multiplicative constant, then a second point to fix the rate constant — never both simultaneously.

Common Misreadings

  • Treating 'initially' as the first measured time rather than t = 0.
  • Trying to set up a 2×2 system in both unknowns at once when one substitution would already isolate one of them.
  • Confusing 'cuts the x-axis at −5' (meaning the point (−5, 0)) with the value −5 being a y-coordinate.

Pitfalls

  • Sign/squaring errors such as writing (−5+2)² = −9 instead of 9.
  • Failing to simplify the resulting fraction (e.g. leaving −12/9 instead of −4/3).
  • Forgetting that e^{0} = 1 (not 0), which is the whole point of choosing t = 0.
  • Substituting the boundary values into a wrongly-formed model (e.g. forgetting a sign in the vertex form before substitution).

A-level Scope Notes

Fully within Edexcel IAL Pure (P1–P4) scope: applies to quadratic vertex/factor forms and to exponential models of the form A ± Be^{−kt} as seen in P3/P4 modelling questions. No formula booklet entry required; relies only on the definition of a function value and e^0 = 1. Avoid generalising to systems requiring logarithms before the additive constant is fixed.
