Trigonometry

Read off extremum from bounded sine/sine-squared range

Family: Bounded Function Extremum Frequency: 1 questions

Trigger — When to use

The function under study has the shape A + B·sin(·), A + B·cos(·), or A ± [R sin(·)]² (often appearing after an R-form harmonic compression). The question asks for a maximum, minimum, or full range — not a location θ — and there is no domain restriction (or the angle clearly sweeps a full period). Visual cues: a constant added to a single bounded trig term, a squared bracket of the form (a sin θ + b cos θ)², or a follow-on part building on a previous R cos(θ − α) / R sin(θ − α) result. Equivalent phrasings: 'find the greatest/least value of…', 'state the range of h(θ)…', 'write down the minimum value of…'.

Motivation — Why this works

Sine and cosine are exactly bounded: sin(·), cos(·) ∈ [−1, 1], and consequently sin²(·), cos²(·) ∈ [0, 1]. The linear map y = A + B·u sends an interval [m, M] to [A + Bm, A + BM] (reversing if B < 0), so the extrema of A + B·sin(·) and A + B·sin²(·) can be read directly without calculus. This avoids unnecessary differentiation, matches the expected mark-scheme route, and exploits the harmonic R-form (a sin θ + b cos θ = R sin(θ − α), R = √(a² + b²)) so squared composites collapse to R²·sin²(·) cleanly.

Thought Process — The step-by-step reasoning

  1. Notice the function is 'constant + (scaled) trig term' or 'constant + (scaled) trig term squared' — a textbook bounded-range setup.
  2. Rule out differentiation: setting f'(θ) = 0 would work but is heavier and risks algebra slips; the bounded-range read-off is the intended method.
  3. If the trig term is a combination a sin θ + b cos θ (or its square), invoke the R-form result from a prior part: a sin θ + b cos θ = R sin(θ − α), so its square is R² sin²(θ − α).
  4. Check the angle is unrestricted (or covers a full period) so sin(·) really attains ±1 and sin²(·) really attains 0 and 1; if the domain is restricted, plan to test endpoints instead.
  5. Decide which bound gives the extremum sought: for A + B·sin(·) the min is A − |B| and max is A + |B|; for A − R²·sin²(·) the min is A − R² (at sin² = 1) and max is A (at sin² = 0).
  6. Assemble the answer with non-strict inequalities since both extremes are achieved.

Mechanic — Concrete operations

  1. Write the function in the canonical form A + B·sin(φ(θ)) or A ± [R sin(φ(θ))]² = A ± R²·sin²(φ(θ)).
  2. State the bound: −1 ≤ sin(φ(θ)) ≤ 1, hence 0 ≤ sin²(φ(θ)) ≤ 1.
  3. Multiply through by the coefficient (mind the sign — flip the inequality if negative).
  4. Add the constant A to every part to obtain the range of the whole function.
  5. Read off max/min and quote with ≤ (extremes are attained when the angle is unrestricted).

Worked Micro-Example

Given $h(\beta) = 10 - (12\sin\beta - 5\cos\beta)^2$ with $12\sin\beta - 5\cos\beta = 13\sin(\beta-\alpha)$: square to get $h(\beta) = 10 - 169\sin^2(\beta-\alpha)$. Since $0 \le \sin^2(\beta-\alpha) \le 1$, we have $0 \le 169\sin^2(\beta-\alpha) \le 169$, so $10 - 169 \le h(\beta) \le 10$, giving $h(\beta) \in [-159,\,10]$.

Variants — Same move, different surface

Plain A + B sin(·) or A + B cos(·)
Range is [A − |B|, A + |B|]; min/max read off directly.
A + (a sin θ + b cos θ)² (positive square)
Equals A + R² sin²(θ − α) with R² = a² + b²; range [A, A + R²].
A − (a sin θ + b cos θ)² (negative square)
Equals A − R² sin²(θ − α); range [A − R², A].
Restricted domain (e.g. 0 ≤ θ ≤ π/3)
The argument may not reach the values where sin(·) = ±1 or sin²(·) = 0,1; evaluate at endpoints and any interior stationary angle inside the domain.
Reciprocal form k/(A + B sin(·))
Find the range of the denominator first, then invert (taking care over sign and zeros).

Common Misreadings

  • Treating (a sin θ + b cos θ)² as if its inside ranges only over [0, R] and forgetting that the square attains 0 (when the inside is 0) and R² (when the inside is ±R).
  • Confusing the range of (a sin θ + b cos θ) — which is [−R, R] — with the range of its square, which is [0, R²].
  • Reading the question as 'find θ that gives the minimum' (a stationary-point problem) when only the value is needed.

Pitfalls

  • Forgetting to square R when squaring an R-form: writing (R sin(·))² = R sin²(·) instead of R² sin²(·).
  • Failing to flip the inequality when multiplying by a negative coefficient (e.g. for A − B sin(·)).
  • Quoting strict inequalities (−159 < h < 10) when the bounds are actually attained — use ≤.
  • Not verifying that the angle's domain is wide enough for sin(·) to reach ±1; a restricted domain may make the read-off invalid.

A-level Scope Notes

Stays within Edexcel IAL Pure (P2/P3) trigonometry: bounds of sine/cosine and the R-form a sin θ + b cos θ = R sin(θ ± α) (R = √(a² + b²)) are standard. R-form is in the formula booklet under 'Trigonometric identities'. Avoid invoking calculus optimisation or Lagrange multipliers — these are unnecessary and out of scope for this technique.
