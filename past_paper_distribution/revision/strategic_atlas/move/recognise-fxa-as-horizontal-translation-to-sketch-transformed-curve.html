Algebra & Functions

Recognise f(x−a) as horizontal translation to sketch transformed curve

Family: Function Transformation Frequency: 1 questions

Trigger — When to use

The function to sketch has the form y = f(x − a) for some constant a, with the variable x appearing inside a bracket or exponent shifted by a constant (e.g. y = 4^{x−6}, y = ln(x − 2), y = (x − 3)^2). The question typically asks for a sketch relative to a given parent curve y = f(x), or asks for the new intercepts/asymptote. Cue words include 'sketch', 'state the transformation', or 'describe the relationship to y = f(x)'.

Motivation — Why this works

Replacing x with (x − a) in any function shifts its graph exactly a units to the right (a > 0) without changing shape, range, or horizontal asymptote. This is a direct consequence of the input being evaluated at a shifted argument: the value formerly at x is now reached at x + a. Recognising this avoids plotting individual points and immediately gives the geometric relationship to the parent curve, so only key features (intercepts, asymptote, shifted reference point) need to be computed.

Thought Process — The step-by-step reasoning

  1. Notice the variable x appears as (x − a) inside the function rather than as bare x — this matches the f(x − a) template.
  2. Rule out a vertical shift (which would be f(x) − a, i.e. the constant subtracted outside the function) and rule out a stretch (which would require a coefficient on x).
  3. Commit to a horizontal translation by a units to the right, since (x − a) is the standard rightward-shift form.
  4. Bridge to concrete features: the asymptote/range is unchanged, but every key point (x₀, y₀) on y = f(x) moves to (x₀ + a, y₀).
  5. Tactically compute the new y-intercept by substituting x = 0, since sketch questions usually require this labelled coordinate.

Mechanic — Concrete operations

  1. Identify the parent function f(x) and the shift constant a from the form f(x − a).
  2. State the transformation: translation by a units in the positive x-direction (right).
  3. Note unchanged features: shape, horizontal asymptote, range.
  4. Map the parent's key reference point (e.g. (0, 1) for exponentials, (1, 0) for logs) to its translated image (a, 1) or (a + 1, 0).
  5. Compute the new y-intercept by substituting x = 0 into the transformed function and label it on the sketch.
  6. Draw the curve with the same shape as the parent, passing through the translated reference point and the new intercept.

Worked Micro-Example

Sketch $y = 4^{x-6}$ given $y = 4^x$. This is $y = 4^x$ translated 6 right, so the asymptote $y = 0$ is unchanged and $(0,1) \mapsto (6,1)$. New $y$-intercept: $4^{-6} = \tfrac{1}{4096}$, so the curve passes through $\left(0, \tfrac{1}{4096}\right)$ and $(6, 1)$.

Variants — Same move, different surface

y = e^{x+3}
Rewrite as f(x − (−3)); translation 3 units left. New y-intercept e^3, reference point (−3, 1).
y = ln(x − 2)
Logarithmic parent: vertical asymptote shifts from x = 0 to x = 2; reference point (1, 0) moves to (3, 0); no y-intercept (domain x > 2).
y = (x − 4)^2
Parabola translated 4 right; vertex moves from (0,0) to (4,0); y-intercept at (0, 16).
y = sin(x − π/3)
Sine curve translated π/3 right; key zero moves from x = 0 to x = π/3, amplitude and period unchanged.
y = 1/(x − 5)
Reciprocal curve translated 5 right; vertical asymptote moves from x = 0 to x = 5; horizontal asymptote y = 0 unchanged.

Common Misreadings

  • Reading 4^{x−6} as 4^x − 6, treating it as a vertical shift down by 6 instead of a horizontal shift.
  • Confusing f(x − a) with f(x + a) and shifting left instead of right.
  • Mistaking f(x − a) for a horizontal stretch because the argument is altered.
  • For ln(x − a), forgetting that the vertical asymptote also shifts (not just the curve).

Pitfalls

  • Forgetting to compute and label the new y-intercept explicitly — examiners typically require labelled intercept coordinates on sketch questions.
  • Drawing the asymptote in a new location when it should be unchanged (for exponentials) or failing to redraw it when it does shift (for logs and reciprocals).
  • Translating only one point rather than ensuring the whole curve has the correct shape and limiting behaviour.

A-level Scope Notes

Standard content in Edexcel IAL Pure Mathematics (P2/P3) under graph transformations of y = f(x ± a); applies to exponential, logarithmic, polynomial, trigonometric, and reciprocal parents within the syllabus.
