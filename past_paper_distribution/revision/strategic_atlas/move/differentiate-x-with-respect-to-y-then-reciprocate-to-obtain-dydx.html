Differentiation

Differentiate x with respect to y then reciprocate to obtain dy/dx

Family: Reciprocal Derivative Frequency: 4 questions

Trigger — When to use

Fires whenever the curve is presented as x = f(y) (i.e. x is given explicitly as a function of y) but the question asks for dy/dx. Visual cues: the equation has x alone on one side and a clean expression in y on the other (e.g. x = ln(sin y), x = ye^{2y}, x = 6sin²(2y), x = 4sin²y − 1). Equivalent phrasing of the trigger: 'rearranging to y = f⁻¹(x) would force an arcsin/arccos/arctan or other awkward inverse', or 'dx/dy is a one-line chain/product rule but dy/dx via implicit differentiation would be messy'. The 'show that' target is often expressed in x or in mixed form, but the cleanest route is still through dx/dy first.

Motivation — Why this works

The inverse function theorem gives dy/dx = 1/(dx/dy) provided dx/dy ≠ 0. This lets us differentiate in the direction in which the equation is algebraically simplest (here, with respect to y) and then invert at the end. It bypasses two messier alternatives: (i) solving for y explicitly, which typically introduces an inverse trig function and a nested chain rule; (ii) implicit differentiation of x = f(y) with respect to x, which requires multiplying every y-term by dy/dx and gathering. The reciprocal route keeps the algebra linear and produces a clean intermediate that can be re-expressed in x using the original equation.

Thought Process — The step-by-step reasoning

  1. Notice the equation is written as x = f(y), not y = f(x): x is the subject, but dy/dx is required.
  2. Consider rearranging to y = f⁻¹(x) — rule it out because this forces an arcsin/arccos/arctan and a tangled chain rule on the inverse trig composite.
  3. Consider implicit differentiation of x = f(y) with respect to x — workable but bookkeeping-heavy: every y-term picks up a dy/dx factor that must be collected.
  4. Commit to the reciprocal route: differentiate x = f(y) directly with respect to y (a forward chain/product rule), then take 1/(dx/dy).
  5. Tactical decision: after computing dx/dy in terms of y, plan whether to (a) reciprocate immediately and convert to x at the end, or (b) convert dx/dy to x first via the original equation/identities, then reciprocate — choose whichever matches the 'show that' target form.
  6. Final check: verify dx/dy ≠ 0 on the stated domain so the reciprocal is valid.

Mechanic — Concrete operations

  1. Treat y as the independent variable and compute dx/dy by chain/product rule applied to f(y).
  2. Simplify dx/dy using standard identities where helpful (e.g. 2 sin y cos y = sin 2y, factorising common exponentials).
  3. If the target is in x, substitute using the original relation x = f(y) (or a Pythagorean identity) to express dx/dy in terms of x.
  4. Apply the reciprocal rule: dy/dx = 1/(dx/dy).
  5. State (or check) that dx/dy ≠ 0 on the relevant interval so the inversion is justified.

Worked Micro-Example

Given $x = \ln(\sin y)$, find $\dfrac{dy}{dx}$. Differentiate w.r.t. $y$: $\dfrac{dx}{dy} = \dfrac{\cos y}{\sin y} = \cot y$. Reciprocate: $\dfrac{dy}{dx} = \tan y$. If needed in $x$: since $\sin y = e^{x}$, $\cos y = \sqrt{1-e^{2x}}$, giving $\dfrac{dy}{dx} = \dfrac{e^{x}}{\sqrt{1-e^{2x}}}$.

Variants — Same move, different surface

x = f(y) involves a product, e.g. x = y e^{2y}
Use the product rule for dx/dy: dx/dy = e^{2y}(1 + 2y), then dy/dx = 1/[e^{2y}(1+2y)].
x = f(y) is a trig power, e.g. x = 6 sin²(2y) or x = 4 sin²y − 1
Apply chain rule then double-angle: dx/dy collapses to a multiple of sin(4y) or sin(2y); reciprocate and convert to x via the original equation.
Parametric curves x = f(t), y = g(t)
Same reciprocal idea generalised: dy/dx = (dy/dt)/(dx/dt) — differentiate in the natural parameter, then combine.
Implicit curves where neither variable is isolated cleanly
Standard implicit differentiation is preferable; the reciprocal trick only saves work when x = f(y) is genuinely explicit.

Common Misreadings

  • Treating x = f(y) as if it were y = f(x) and differentiating the right-hand side directly with respect to x without the chain rule.
  • Trying to invert to y = f⁻¹(x) first and getting bogged down in arcsin/arccos derivatives.
  • Performing implicit differentiation w.r.t. x correctly but missing the simpler reciprocal shortcut.

Pitfalls

  • Forgetting to take the reciprocal at the end and reporting dx/dy as the answer.
  • Chain-rule slips when differentiating composites (e.g. writing d/dy[e^{2y}] = e^{2y} instead of 2e^{2y}, or missing the inner derivative on sin²(2y)).
  • Failing to check dx/dy ≠ 0 on the given domain, so the inversion is invalid at isolated points.
  • Stopping in y when the question requires the answer in x — forgetting to substitute back using the original equation or a Pythagorean identity.

A-level Scope Notes

Fully within Edexcel IAL P3/P4 differentiation. Uses the reciprocal-derivative result dy/dx = 1/(dx/dy), which is standard and not a separate booklet entry; relies on chain rule, product rule, and basic trig/log derivatives from the formula booklet. Avoid invoking the formal inverse function theorem statement — students need only the reciprocal identity and the non-vanishing condition on dx/dy.
