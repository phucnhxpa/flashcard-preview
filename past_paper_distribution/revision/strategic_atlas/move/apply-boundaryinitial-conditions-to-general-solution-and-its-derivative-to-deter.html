Strategic Atlas / Differential Equations / Constant Determination
Differential Equations

Apply boundary/initial conditions to general solution and its derivative to determine A and B

Family: Constant Determination Frequency: 2 questions

Trigger — When to use

The general solution of a second-order linear ODE has been found in the form y = CF + PI, containing two arbitrary constants A and B. The question supplies two pieces of data — typically y(x₀) and y'(x₀) (an initial value problem), or values of y at two different x-values (a boundary value problem). Look for phrases like 'given that y = ... and dy/dx = ... when x = 0', 'find the particular solution', or two stated function values. This is the final step that converts the general solution into the unique particular solution requested.

Motivation — Why this works

A second-order linear ODE has a two-dimensional solution space, so its general solution carries exactly two free constants. Two independent conditions provide two scalar equations, pinning A and B down uniquely. The mathematical principle is that of imposing constraints on a parameter family: each condition cuts one degree of freedom. The derivative condition requires differentiating the entire general solution (CF + PI together), since both parts depend on x and contribute to y'(x₀).

Thought Process — The step-by-step reasoning

  1. Notice that the general solution y = CF + PI contains two unknown constants A and B, and the question has supplied two conditions — so this is a constant-determination task, not a solving task.
  2. Rule out trying to apply the conditions to the CF alone (a common error): the PI also contributes to y(x₀) and to y'(x₀), so both must be retained.
  3. Commit to the standard procedure: substitute one condition into y and the other into y' to manufacture two simultaneous equations in A and B.
  4. If a derivative condition is given, differentiate the FULL expression y = CF + PI before substituting — keeping product-rule structure intact for terms like (A + Bx)e^{mx} or e^{px}(A cos qx + B sin qx).
  5. Substitute the given x-value carefully, using e^{0}=1, cos 0 = 1, sin 0 = 0 (or the appropriate non-zero values for a BVP) to reduce each equation to a clean linear relation in A and B.
  6. Solve the 2×2 linear system by elimination or substitution, then write the particular solution with the numerical A and B inserted.

Mechanic — Concrete operations

  1. Write the general solution y = y_CF + y_PI with symbolic constants A, B.
  2. Substitute the first condition (e.g. y(x₀) = k) into y to obtain Equation (1) in A and B.
  3. Differentiate the entire general solution: y' = (d/dx)[y_CF] + (d/dx)[y_PI], using product rule where needed.
  4. Substitute the second condition (e.g. y'(x₀) = c) into y' to obtain Equation (2) in A and B.
  5. Solve Equations (1) and (2) simultaneously for A and B.
  6. Replace A and B in the general solution with their numerical values to state the particular solution.

Worked Micro-Example

Suppose $y = (A + Bx)e^{2x} + 3x$ with $y(0)=1$, $y'(0)=0$. Then $y(0)=A=1$. Differentiating: $y' = Be^{2x} + 2(A+Bx)e^{2x} + 3$, so $y'(0) = B + 2A + 3 = 0 \Rightarrow B = -5$. Hence the particular solution is $y = (1 - 5x)e^{2x} + 3x$.

Variants — Same move, different surface

Boundary value problem: conditions on y at two different x-values (e.g. y(0)=p and y(π/2)=q).
Same method, but evaluate exponentials/trig at the non-zero argument; algebra is heavier but structure identical.
Condition involves y'' rather than y'.
Either differentiate twice and substitute, or rearrange the original ODE to express y'' in terms of y, y' and substitute.
CF contains a repeated root term (A + Bx)e^{mx} or complex form e^{px}(A cos qx + B sin qx).
Apply product rule carefully when computing y'; the rest of the procedure is unchanged.

Common Misreadings

  • Applying the conditions to the CF only, ignoring that the PI contributes to y(x₀) and y'(x₀).
  • Treating the two conditions as if they apply to the same function (both to y, or both to y') when one is actually for y'.
  • Mistaking a boundary value problem for an initial value problem and substituting x=0 instead of the stated boundary point.

Pitfalls

  • Forgetting to differentiate the PI when computing y' — the PI is often a polynomial or constant whose derivative is small but non-zero.
  • Product-rule errors on CF terms like (A + Bx)e^{mx} or e^{px}(A cos qx + B sin qx).
  • Arithmetic slips at x = 0: confusing e^{0}=1, cos 0 = 1, sin 0 = 0.
  • Errors solving the 2×2 system, especially when coefficients involve surds, fractions, or trig values from a non-zero boundary point.

A-level Scope Notes

Fully within Edexcel IAL Further Pure scope (FP2/FP3 second-order linear ODEs with constant coefficients). No formula booklet entry is needed — this is a procedural step. Avoid invoking variation of parameters, Laplace transforms, or general existence/uniqueness theorems, which are out of scope.
