Algebra & Functions

Cross-multiply and collect the target variable to invert a Möbius function

Family: Inverse By Rearrangement Frequency: 4 questions

Trigger — When to use

The function is a ratio of two linear expressions f(x) = (ax+b)/(cx+d) — a Möbius transformation — and you are asked to find f⁻¹(x), or asked for the range of f (which is dual to the inverse problem). Visual cue: x appears in BOTH the numerator and the denominator, so you cannot isolate x with a single +/−/× step. Equivalent phrasings: 'find the inverse of f', 'make x the subject of y = (ax+b)/(cx+d)', 'state the range of f' (since the value of y excluded by the inverse's denominator IS the range exclusion). Also fires when the question asks to find the value of x for which f is self-inverse, or to verify f⁻¹ exists on a specified domain.

Motivation — Why this works

Cross-multiplying clears the fraction, turning y = (ax+b)/(cx+d) into y(cx+d) = ax+b — a LINEAR equation in x. Because every term involving x has degree exactly 1, you can collect all x-terms on one side, factor x out, and divide to get x = (dy−b)/(−cy+a). This works precisely because Möbius transformations are degree-1-over-degree-1, so the inverse is again a Möbius transformation (mirroring the inverse of a 2×2 matrix [[a,b],[c,d]]). Swapping x and y at the end converts 'x as a function of y' into the standard f⁻¹(x) notation. As a bonus, the value that makes the new denominator zero is precisely the horizontal asymptote a/c — i.e. the value excluded from the range of f.

Thought Process — The step-by-step reasoning

  1. Notice that x sits in both numerator and denominator, so I cannot isolate x by one inverse operation — a direct 'undo' chain will not work.
  2. Rule out reciprocating (writing 1/f), which is a common confusion of inverse with reciprocal; rule out sketching/asymptote-spotting alone, since the mark scheme expects an algebraic derivation.
  3. Commit to the standard rearrangement recipe: write y = f(x), cross-multiply to clear the denominator, then collect the variable I want to isolate.
  4. First concrete operation: multiply both sides by (cx+d) to get y(cx+d) = ax+b, expanding to cxy + dy = ax + b — now the equation is linear in x.
  5. Tactical decision: gather every term containing x on the LEFT and everything else on the RIGHT, then factor x out as a common factor — this is the only way to isolate x when it appears in two separate terms.
  6. Divide by the bracket (cy−a) or equivalent, then swap x ↔ y to present the answer as f⁻¹(x). For range questions, instead read off the y-value that makes the new denominator vanish — that is the excluded value.

Mechanic — Concrete operations

  1. Write y = (ax+b)/(cx+d).
  2. Multiply both sides by (cx+d): y(cx+d) = ax+b.
  3. Expand: cxy + dy = ax + b.
  4. Move all x-terms to one side and constants to the other: cxy − ax = b − dy.
  5. Factor x out: x(cy − a) = b − dy.
  6. Divide: x = (b − dy)/(cy − a), equivalently x = (dy − b)/(a − cy).
  7. Swap x and y to get f⁻¹(x) = (dx − b)/(a − cx).
  8. (For range:) note that the denominator of f⁻¹ vanishes when x = a/c, so the range of f is ℝ \ {a/c}.

Worked Micro-Example

Find $f^{-1}$ for $f(x) = \dfrac{x+5}{3x-2}$. Set $y = \dfrac{x+5}{3x-2}$, cross-multiply: $y(3x-2) = x+5$, i.e. $3xy - 2y = x + 5$. Collect: $3xy - x = 2y + 5$, factor: $x(3y-1) = 2y+5$, divide: $x = \dfrac{2y+5}{3y-1}$. Swap variables: $f^{-1}(x) = \dfrac{2x+5}{3x-1}$, with $x \neq \tfrac{1}{3}$.

Variants — Same move, different surface

Finding the range of a Möbius f instead of the inverse
Run exactly the same rearrangement, then identify the y-value making the new denominator zero — that single value is excluded from the range (it is the horizontal asymptote a/c).
Function of the form y = (ax² + b)/(cx + d) (numerator quadratic)
Cross-multiplying yields a QUADRATIC in x, so finish with the quadratic formula; the range then comes from the discriminant condition ≥ 0.
Self-inverse check or composition (f∘f = x)
After deriving f⁻¹, compare coefficients with f; for a Möbius f = (ax+b)/(cx+d) it is self-inverse iff a + d = 0.
Further Maths matrix viewpoint
Recognise that f⁻¹ corresponds to the inverse matrix [[a,b],[c,d]]⁻¹ = (1/Δ)[[d,−b],[−c,a]] — same answer, different surface.

Common Misreadings

  • Confusing the inverse f⁻¹ with the reciprocal 1/f, and writing (cx+d)/(ax+b).
  • Confusing range with domain — stating x ≠ d/c (a domain exclusion of f) when asked for the range.
  • Treating the question as if it asks to 'solve for x' for a particular y, rather than to express x as a general function of y.

Pitfalls

  • Sign errors when expanding y(cx+d) or moving terms across the equals sign — particularly with negative constants.
  • Forgetting to collect ALL x-terms on one side before factoring, leaving a stray x.
  • Forgetting the final swap of x and y, so the answer is given in terms of y instead of x.
  • Omitting the domain of f⁻¹ (which equals the range of f), e.g. failing to state x ≠ a/c.
  • Dividing by (cy − a) without checking it is non-zero — the case cy = a corresponds to the excluded range value.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P2/P3 functions chapter). No formula booklet entry is needed — this is a standard rearrangement technique. Avoid invoking advanced ideas like the Möbius group or projective line; the matrix-inverse analogy is only useful for Further Maths students who have met FP1 matrices.
