Trigonometry

Factor trigonometric polynomial then filter roots by sine/cosine range

Family: Algebraic Factorisation With Range Filter Frequency: 2 questions

Trigger — When to use

Fires when an equation, after substitution s = sin x (or c = cos x, or via reciprocal conversion of csc/sec), reduces to a polynomial equation in s. Visual cues: a single trig function appearing to multiple powers (e.g. sin³y, sin²y, sin y), or a reciprocal equation like csc²x + csc x − k = 0 that becomes a quadratic. After factorising, you obtain candidate values s = a, s = b, ... and must check each against |sin x| ≤ 1 (or |cos x| ≤ 1). Equivalent phrasings: 'polynomial in one trig ratio', 'cubic/quadratic in sin x', 'reciprocal trig equation reducible to quadratic'.

Motivation — Why this works

A polynomial in a single trig ratio is solved by ordinary algebraic factorisation, but trig ratios are bounded: sin x, cos x ∈ [−1, 1] and consequently |csc x|, |sec x| ≥ 1. Any algebraic root violating these bounds yields no real angle and is extraneous. Filtering ensures you only spend time inverting genuine cases and prevents losing marks for spurious answers. The underlying principle is range/codomain filtering: an algebraic equivalence is necessary but not sufficient for a trig equation.

Thought Process — The step-by-step reasoning

  1. Notice the equation involves only one trig ratio (e.g. only sin y, or only csc x), to multiple powers — this signals 'treat it as a polynomial in s'.
  2. Rule out direct angle-finding (no addition/subtraction identity is needed) and rule out dividing through by sin y, since that would silently discard sin y = 0 solutions.
  3. Commit to factoring: extract any common trig factor first, then factor the remaining quadratic by inspection (look for factors of ac that sum to b) rather than reaching for the quadratic formula.
  4. List ALL algebraic roots s₁, s₂, ... before solving any of them — this keeps the filtering step visible.
  5. For each root, check the range: if |s| > 1 (for sin/cos) or |s| < 1 (for csc/sec), reject explicitly with a one-line justification ('no solution since |sin x| ≤ 1').
  6. Proceed to invert only the surviving roots, using arcsin/arccos and the symmetry of the trig graph to enumerate solutions in the given interval.

Mechanic — Concrete operations

  1. Substitute s = sin x (or convert csc x = 1/sin x first if needed) to obtain a polynomial P(s) = 0.
  2. Extract any common factor (e.g. s · Q(s) = 0) — never divide it away.
  3. Factor the remaining quadratic: for as² + bs + c, find factors of ac summing to b, giving (αs − p)(βs − q) = 0.
  4. List all roots: s = 0, s = p/α, s = −q/β, ...
  5. Apply the range filter: discard any root with |s| > 1 (sine/cosine) or |s| < 1 (cosecant/secant), stating the reason.
  6. Solve sin x = s (etc.) for each surviving root over the required interval.

Worked Micro-Example

Solve $3\sin^3 y + 10\sin^2 y - 8\sin y = 0$. Factor: $\sin y(3\sin^2 y + 10\sin y - 8) = 0 \Rightarrow \sin y(3\sin y - 2)(\sin y + 4) = 0$. Roots: $\sin y = 0,\ \tfrac{2}{3},\ -4$. Reject $\sin y = -4$ since $|\sin y| \le 1$. Solve only $\sin y = 0$ and $\sin y = \tfrac{2}{3}$.

Variants — Same move, different surface

Reciprocal equation in csc x or sec x
Convert csc x = 1/sin x (or sec x = 1/cos x) and reject any root with |csc x| < 1 or |sec x| < 1, since this forces |sin x| > 1 (impossible).
Polynomial in cos x after using sin²x = 1 − cos²x
Same logic, but the bound is |cos x| ≤ 1; reject roots outside [−1, 1].
Polynomial in tan x
No range filter needed — tan x ∈ ℝ — so all algebraic roots are admissible (still mind the domain x ≠ π/2 + kπ).

Common Misreadings

  • Dividing through by sin y (or cos x) to 'simplify', thereby losing the sin y = 0 solutions.
  • Discarding a large root (e.g. csc x = 5) instead of the out-of-range root (csc x = −1/2), having confused which bound applies to reciprocals.
  • Failing to check validity at all and attempting arcsin(−2) on the calculator.

Pitfalls

  • Using the quadratic formula on a cleanly factorable quadratic — slower and prone to sign errors with a leading coefficient like 3.
  • Forgetting to state the rejection reason explicitly; Edexcel mark schemes often require 'no solution since |sin x| ≤ 1' to be written.
  • Solving sin x = s on the surviving roots but missing the second solution in the period (e.g. π − arcsin s) within the given interval.

A-level Scope Notes

Within Edexcel IAL P2/P3 trigonometry. Uses only the basic identities sin x = 1/csc x, cos x = 1/sec x and the bounds |sin x|, |cos x| ≤ 1 (no formula booklet entry needed). Do not generalise to complex roots — IAL trig solutions are real angles only.
