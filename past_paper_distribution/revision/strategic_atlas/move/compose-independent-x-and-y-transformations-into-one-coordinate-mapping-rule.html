Algebra & Functions

Compose independent x and y transformations into one coordinate mapping rule

Family: Composite Mapping Rule Frequency: 1 questions

Trigger — When to use

A single equation packages two graph transformations, one acting on the input and one on the output, e.g. y = 3f(2x), y = f(-x) - 1, or y = -f(x) + 4. The question typically asks to sketch the new curve from a given graph with labelled key points (intercepts, turning points, asymptotes). Drawing an intermediate graph after each transformation is slow and invites mistakes. Cue phrase: 'sketch the curve y = ...' where the equation contains both an inside change to x and an outside change to y.

Motivation — Why this works

Inside changes (replacing x by bx, -x, x+c) act only on the x-coordinate, and outside changes (multiplying by a, adding d, negating) act only on the y-coordinate, so the two transformations operate on independent slots and commute. This means the whole transformation can be encoded as a single mapping (x, y) → (g(x), h(y)) and applied point-by-point. Working on a small table of key points reduces a multi-step graphical task to arithmetic, and ensures every labelled feature on the original graph is updated consistently. It also sidesteps any debate about 'which transformation comes first' because the two slots are disjoint.

Thought Process — The step-by-step reasoning

  1. I notice the equation has both an inside modification of x and an outside modification of y, so two transformations are at play.
  2. I rule out drawing an intermediate sketch after one transformation — that doubles the work and risks mislabelling key points twice.
  3. I commit to a single mapping rule because the inside and outside changes act on different coordinates, so they are independent and order-free.
  4. I read off the rule directly: inside 'bx' sends x to x/b, '-x' sends x to -x, '+c' inside sends x to x-c; outside 'a·' sends y to ay, '+d' sends y to y+d, '-' sends y to -y. I write (x, y) → (g(x), h(y)).
  5. I tabulate the original key points (intercepts, turning points, endpoints, asymptote anchors) and push each one through the rule, then sketch a curve of the same shape through the new points.

Mechanic — Concrete operations

  1. Identify the inside transformation acting on x and write its effect on the x-coordinate as x → g(x).
  2. Identify the outside transformation acting on y and write its effect on the y-coordinate as y → h(y).
  3. Combine into the single rule (x, y) → (g(x), h(y)).
  4. List every labelled key point of the original graph (intercepts, turning points, endpoints, asymptote markers).
  5. Apply the rule to each key point to get the new coordinates.
  6. Plot the new points and join them with a curve of the same qualitative shape, updating any asymptotes by the same rule.

Worked Micro-Example

Sketch y = 3f(2x) given y = f(x) passes through (0,0), (1,2), (3,0). Mapping rule: (x, y) → (x/2, 3y). Then (0,0) → (0,0), (1,2) → (1/2, 6), (3,0) → (3/2, 0). Plot these three points and join with the same shape as the original.

Variants — Same move, different surface

y = f(-x) - 1 (reflection in y-axis plus vertical translation)
Rule is (x, y) → (-x, y - 1); apply to all labelled points.
y = -f(x) + d (reflection in x-axis plus vertical shift)
Rule is (x, y) → (x, -y + d); only the y-coordinate changes.
y = af(bx) + d with all four ingredients
Rule is (x, y) → (x/b, ay + d); still one pass because horizontal and vertical slots stay independent.
y = f(bx + c) where two transformations act on the same axis
Factorise inside as f(b(x + c/b)); the rule on x is x → (x - c)/b, i.e. translate by -c then divide by b. Order within the same axis matters.
Transforming asymptotes of y = 1/x type curves
Apply the same mapping rule to the asymptote equations, e.g. x = 0 becomes x = 0/b = 0 under x → x/b, and y = 0 becomes y = d under y → ay + d.

Common Misreadings

  • Reading 'inside' transformations the wrong way round, e.g. thinking f(2x) stretches x by factor 2 instead of 1/2.
  • Believing horizontal and vertical transformations must be done in a fixed order even though they act on independent coordinates.
  • Trying to apply the mapping rule to the equation y = f(x) algebraically instead of to the labelled coordinate points.
  • Confusing y = f(x) + d (shift) with y = f(x + d) (different axis).

Pitfalls

  • When both transformations affect the same axis (e.g. y = f(2x - 3)), the correct x-rule is x → (x + 3)/2, not x → x/2 + 3.
  • Forgetting to transform every labelled point — y-intercepts and asymptote anchors are easy to overlook.
  • Not transforming asymptotes alongside points, leading to a curve that crosses where it shouldn't.
  • Connecting transformed points in the wrong order or with reversed concavity, especially after a reflection.
  • Mishandling negative scale factors: y → -3y reflects and stretches; both effects must be applied.

A-level Scope Notes

Fully within Edexcel IAL P2/P3 graph transformations of y = af(bx + c) + d, including reflections and combinations with f(x) given graphically. No matrix or parametric machinery is invoked.
