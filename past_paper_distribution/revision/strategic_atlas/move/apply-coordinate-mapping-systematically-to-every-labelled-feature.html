Algebra & Functions

Apply coordinate mapping systematically to every labelled feature

Family: Transformation Application Frequency: 1 questions

Trigger — When to use

Fires when a question gives the graph of y = f(x) with explicitly labelled features (turning points, intercepts, asymptotes) and asks you to sketch or state the images under a transformation such as y = f(x) + a, y = f(x + a), y = f(-x), y = -f(x), y = af(x) or y = f(ax). Cues include phrases like 'showing the coordinates of the images of P and Q' and 'the equation of the asymptote'. Alternative phrasings: 'state the new coordinates', 'write down the equation of the transformed asymptote'.

Motivation — Why this works

Each transformation y = f(x) + a, y = f(x + a), y = af(x), y = f(ax), y = -f(x), y = f(-x) corresponds to a single coordinate mapping (x, y) → (x', y'). Applying that one rule uniformly to every labelled feature guarantees completeness and consistency, since the mark scheme awards independent marks for each transformed point and for the asymptote. Treating the asymptote as just another y-value (or x-value) under the same rule prevents the most common omission. Systematic mapping also removes the need to re-reason from the graph for each feature.

Thought Process — The step-by-step reasoning

  1. I notice the question lists specific labelled features (P, Q, asymptote) and demands their images, so I must produce one transformed object per labelled object.
  2. I rule out re-sketching from scratch or reasoning feature-by-feature with different methods — that risks inconsistency and missed marks.
  3. I commit to first classifying the transformation into a single mapping rule (x, y) → (x', y'), then applying it uniformly.
  4. I bridge to operation by writing the rule at the top of my working, e.g., y = f(x) - 2 gives (x, y) → (x, y - 2).
  5. Tactically, I treat a horizontal asymptote y = c as a y-value subject to the vertical part of the rule, and a vertical asymptote x = k as an x-value subject to the horizontal part; if the rule only affects the other coordinate, the asymptote is unchanged.

Mechanic — Concrete operations

  1. Identify the transformation type and write its mapping rule (x, y) → (x', y').
  2. List every labelled feature: each turning point, intercept, and asymptote.
  3. Apply the mapping to each point's coordinates, simplifying arithmetic carefully (especially signs).
  4. Apply the mapping to each asymptote: horizontal asymptote y = c → y = c' using the y-rule; vertical asymptote x = k → x = k' using the x-rule.
  5. State all transformed features explicitly in the final answer, matching the labels used in the question.

Worked Micro-Example

Given P(-1, 0), Q(3/2, 2) and asymptote y = 1 on y = f(x). For y = f(x) - 2, mapping (x, y) → (x, y - 2): P → (-1, -2), Q → (3/2, 0), asymptote → y = -1. For y = f(-x), mapping (x, y) → (-x, y): P → (1, 0), Q → (-3/2, 2), asymptote y = 1 unchanged (no y-change).

Variants — Same move, different surface

Combined translation y = f(x - 3) + 1
Use mapping (x, y) → (x + 3, y + 1) on every feature, including shifting both coordinates of the asymptote's defining value if relevant.
Stretches y = 2f(3x)
Use mapping (x, y) → (x/3, 2y); horizontal asymptote y = c becomes y = 2c, vertical asymptote x = k becomes x = k/3.
Reflection y = -f(x)
Use mapping (x, y) → (x, -y); horizontal asymptote y = c becomes y = -c, x-coordinates unchanged.
Modulus graph y = |f(x)|
Not a single linear mapping: reflect any portion below the x-axis; turning points with negative y-values flip sign, asymptote y = c becomes y = |c| (with care if c < 0).

Common Misreadings

  • Treating the asymptote as decorative and not transforming it, instead of as a labelled feature carrying its own mark.
  • Confusing 'inside' vs 'outside' transformations, e.g., applying y = f(x + 2) as a vertical shift rather than a horizontal one.
  • Assuming a horizontal asymptote always stays fixed, missing that vertical transformations move it.

Pitfalls

  • Sign errors when reflecting, e.g., -(-1) written as -1 instead of 1.
  • Forgetting that horizontal transformations invert the sign/factor: y = f(x + 2) shifts left by 2, not right.
  • Applying a stretch factor in the wrong direction, e.g., y = f(3x) compresses horizontally by factor 1/3, not stretches by 3.
  • Omitting to state the equation of the transformed asymptote (writing only points loses the asymptote mark).

A-level Scope Notes

Stays within Edexcel IAL P2/P3 graph transformations: single and combined translations, stretches and reflections of y = f(x), with labelled points and horizontal/vertical asymptotes.
