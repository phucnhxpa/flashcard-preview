Sequences & Series

Link GP first term to final AP term at phase transition

Family: Sequence Transition Frequency: 1 questions

Trigger — When to use

Fires when a problem describes a process in two consecutive phases — an AP for the first n terms and a GP thereafter — and asks you to find the GP's common ratio or later terms. Look for phrases like 'from the kth term onwards the values form a geometric progression' or a stated value at the transition index that matches the AP's last term. The cue is that no independent first term for the GP is given; instead the (k)th term value is supplied and equals the AP's terminal value.

Motivation — Why this works

The two progressions model one continuous quantity, so continuity at the join forces G_1 = A_n where n is the last AP index. Once G_1 is fixed by the AP, any later given term (typically G_2) determines r via r = G_2 / G_1. This avoids inventing a separate first term and keeps the model physically consistent.

Thought Process — The step-by-step reasoning

  1. Notice that the problem describes one sequence split into an AP phase and a GP phase, with the transition index explicitly named.
  2. Rule out treating the GP as a brand-new sequence with an unknown a; the value at the transition is already pinned down by the AP.
  3. Commit to G_1 = A_n (the last AP term) because the physical process is continuous across the join.
  4. Bridge to computation: identify which given numerical value corresponds to G_1 and which to G_2 (off-by-one care).
  5. Use r = G_2 / G_1 directly; if only later terms are given, use r^{m} = G_{1+m}/G_1.

Mechanic — Concrete operations

  1. Compute or read off A_n, the last AP term, where the (n+1)th term is the first GP term.
  2. Set G_1 = A_n.
  3. Read the next given term as G_2 (or G_k for some k > 1).
  4. Solve r = G_2 / G_1, or r^{k-1} = G_k / G_1 if non-adjacent.
  5. Sanity-check 0 < r < 1 if the process is described as decreasing/converging.

Worked Micro-Example

An AP has $A_n = 50 - 2(n-1)$, so $A_{10} = 32$. From the 11th term onwards the sequence is a GP, and the 12th term equals $30$. Then $G_1 = A_{10} = 32$ and $G_2 = 30$, giving $r = \tfrac{30}{32} = \tfrac{15}{16}$.

Variants — Same move, different surface

GP phase first, then AP phase
Set A_1 equal to the last GP term; use given AP term to find common difference d.
Only a later GP term is given (e.g. G_5)
Use r^{4} = G_5 / G_1 with G_1 = A_n, then take the appropriate root.
Sum to infinity of the GP tail is asked
After fixing G_1 = A_n and r, apply S_\infty = G_1/(1-r), provided |r|<1.
Total distance/quantity over both phases
Add AP sum S_n (using A_1, A_n, n) to GP sum from G_1 = A_n with the found r.

Common Misreadings

  • Treating the GP as starting fresh with a separately defined first term, disconnected from the AP.
  • Taking G_1 to be the term given just after the transition (e.g. the 22nd hit) instead of the transition term itself (the 21st hit).
  • Assuming the AP and GP overlap at the transition index, double-counting one term.

Pitfalls

  • Off-by-one error: the (n+1)th term of the whole sequence is G_1, not G_0 or G_2.
  • Computing r as G_1/G_2 instead of G_2/G_1.
  • Forgetting to check |r| < 1 before invoking sum to infinity in a follow-up part.

A-level Scope Notes

Uses only standard AP nth-term and GP common-ratio formulae from Edexcel IAL P1/P2; no material beyond the syllabus is required.
