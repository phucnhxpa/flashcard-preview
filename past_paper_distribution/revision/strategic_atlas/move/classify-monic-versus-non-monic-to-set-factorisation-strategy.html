Algebra & Functions

Classify monic versus non-monic to set factorisation strategy

Family: Problem Classification Frequency: 1 questions

Trigger — When to use

Fires the moment a quadratic $ax^2 + bx + c$ needs to be factorised by inspection. The first thing to check is whether $a = 1$ (monic) or $a \neq 1$ (non-monic), because this single observation dictates the entire search procedure. Also fires when a quadratic appears inside a larger problem (solving an equation, simplifying a rational expression, integrating) and you need to choose between a one-pair search and a two-pair cross-multiplication search.

Motivation — Why this works

For a monic quadratic $x^2 + bx + c = (x + p)(x + q)$, the conditions reduce to $p + q = b$ and $pq = c$ — a single factor-pair search on $c$. For a non-monic quadratic $ax^2 + bx + c = (mx + p)(nx + q)$, we need $mn = a$, $pq = c$, and the cross-term $mq + np = b$ — a simultaneous search over factor pairs of both $a$ and $c$. Recognising the case up front prevents wasted effort in the monic case and structural errors (dropped leading coefficient) in the non-monic case.

Thought Process — The step-by-step reasoning

  1. I notice the leading coefficient $a$ and ask: is it $1$, or something else?
  2. I rule out treating a non-monic as monic — writing $(x+p)(x+q)$ when $a \neq 1$ would lose the leading coefficient entirely.
  3. I commit to a strategy based on $a$: one-pair search if monic, two-pair cross-search if non-monic.
  4. Before searching, I check whether a common factor (including $-1$) can be pulled out to reduce the non-monic case to a simpler one.
  5. Inside the move, I decide whether to use trial cross-multiplication or the AC-method (split the middle term) for the non-monic case.

Mechanic — Concrete operations

  1. Read off the leading coefficient $a$.
  2. If $a < 0$ or all coefficients share a common factor, extract it first.
  3. If $a = 1$: list factor pairs of $c$, find the pair $(p, q)$ with $p + q = b$, write $(x+p)(x+q)$.
  4. If $a \neq 1$: list factor pairs of $a$ and of $c$, systematically test combinations $(mx + p)(nx + q)$ until the cross-sum $mq + np = b$ matches.
  5. Expand to verify the factorisation reproduces all three coefficients.

Worked Micro-Example

Factorise $6x^2 + 11x - 10$. Since $a = 6 \neq 1$, this is non-monic. Factor pairs of $6$: $(1,6), (2,3)$; of $-10$: $(\pm1, \mp10), (\pm2, \mp5)$. Testing $(2x + 5)(3x - 2)$: cross-sum $= 2(-2) + 5(3) = 11$ ✓. So $6x^2 + 11x - 10 = (2x + 5)(3x - 2)$. Contrast with the monic $x^2 + 11x - 10$, which would only need a single pair search on $-10$ (and in fact has no integer factorisation).

Variants — Same move, different surface

Leading coefficient negative, e.g. $-2x^2 + 5x + 3$
Factor out $-1$ first to get $-(2x^2 - 5x - 3)$, then apply the non-monic procedure to the bracket.
All terms share a common factor, e.g. $4x^2 + 10x + 4$
Extract the common factor $2$ to get $2(2x^2 + 5x + 2)$, reducing the search space before classifying the inner quadratic.
Quadratic in disguise, e.g. $6y^4 + 11y^2 - 10$ with $u = y^2$
Substitute, classify the quadratic in $u$ as non-monic, factorise, then back-substitute.
Non-factorisable over integers
If no integer factor pairs satisfy the cross-sum, abandon inspection and use the quadratic formula or completing the square.

Common Misreadings

  • Treating a non-monic quadratic as if it were monic, writing $(x + p)(x + q)$ and silently dropping the leading coefficient.
  • Confusing the leading coefficient $a$ with the constant $c$ when the quadratic is written in a non-standard order.
  • Assuming any quadratic with integer coefficients must factorise over the integers.

Pitfalls

  • For non-monic quadratics, settling on the first factor-pair combination that partially works without verifying the cross-sum.
  • Forgetting to track signs when listing factor pairs of a negative $c$.
  • Failing to extract a common factor first, leading to an unnecessarily large search space.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P1), where factorisation by inspection of quadratics with integer coefficients is standard. Extensions via substitution (quadratic in $x^2$, $e^x$, etc.) appear throughout P2–P4 and Further Pure.
