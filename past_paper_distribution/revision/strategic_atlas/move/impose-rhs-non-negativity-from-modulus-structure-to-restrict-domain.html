Algebra & Functions

Impose RHS non-negativity from modulus structure to restrict domain

Family: Domain Restriction Frequency: 2 questions

Trigger — When to use

Fires whenever an equation has been arranged into the form |f(x)| = g(x) (or k|f(x)| = g(x), or √f(x) = g(x)) with a modulus/square root isolated on one side and a non-modulus expression on the other. Visual cue: one side is intrinsically ≥ 0 (a single modulus, an even power, or a principal square root), while the other side is a linear or polynomial expression whose sign depends on x. Equivalent phrasings: 'the modulus side forces the other side to be non-negative', 'pre-filter the domain before case-splitting'. Does NOT fire for |f(x)| = |g(x)| (both sides automatically ≥ 0).

Motivation — Why this works

A modulus |f(x)| (or √f(x), or [f(x)]²) is always non-negative, so if it equals g(x), then g(x) must itself be ≥ 0. Imposing this necessary condition upfront carves out a restricted domain that any solution must lie in, acting as a free validity filter for candidate roots produced by case-splitting. This prevents accepting algebraically-valid-but-spurious roots that satisfy a single case's equation but violate the original equation. Geometrically it identifies where the V-shape (or inverted V) can possibly meet the other graph — only where the non-modulus side is above the x-axis.

Thought Process — The step-by-step reasoning

  1. Notice the equation has a modulus (or surd) isolated on one side and a non-modulus expression on the other — the structural asymmetry is the cue.
  2. Rule out jumping straight into the two-case split: that would generate candidate solutions which then need to be checked individually, doubling the work.
  3. Rule out squaring both sides: it would clear the modulus but introduce extraneous roots and obscure the sign information already available.
  4. Commit to the domain-restriction move: since LHS ≥ 0 is automatic, RHS ≥ 0 is a necessary condition for any solution to exist.
  5. Bridge to operation: write down 'RHS ≥ 0' as an inequality and solve it for x to obtain an interval.
  6. Decide whether to use this as a pre-filter (intersect with each case's domain) or a post-filter (check candidates at the end). Pre-filtering is cleaner and more reliable.

Mechanic — Concrete operations

  1. Rearrange the equation so a single modulus (or √) sits alone on one side: |f(x)| = g(x).
  2. State the structural fact: |f(x)| ≥ 0 for all x.
  3. Deduce the necessary condition g(x) ≥ 0 and solve this inequality for x to obtain a domain D.
  4. Carry out the usual case-split on the modulus, but intersect each case's working domain with D.
  5. After solving each case, discard any candidate root not lying in D.

Worked Micro-Example

Solve $2|2x-5| + 3 = 3x - 2$. Isolate: $2|2x-5| = 3x - 5$. LHS $\ge 0$, so require $3x - 5 \ge 0 \Rightarrow x \ge \tfrac{5}{3}$. Now case-split: Case 1 ($x \ge \tfrac{5}{2}$): $2(2x-5) = 3x-5 \Rightarrow x = 5$ ✓. Case 2 ($\tfrac{5}{3} \le x < \tfrac{5}{2}$): $-2(2x-5) = 3x-5 \Rightarrow x = \tfrac{15}{7}$ ✓ (lies in restricted interval).

Variants — Same move, different surface

Equation of the form √f(x) = g(x)
Require both f(x) ≥ 0 (radicand) and g(x) ≥ 0 (principal root non-negative) before squaring.
Equation |f(x)| = g(x) where g(x) is a non-linear expression (e.g. quadratic)
Same logic — solve g(x) ≥ 0 to get the admissible domain, which may be a union of intervals.
Equation |f(x)| = |g(x)|
Skip this move — both sides are ≥ 0 automatically; instead square both sides or split into ±.
Modulus equation with parameter, e.g. |2x - k/3| = k - 2|x|
Require k - 2|x| ≥ 0 ⇒ |x| ≤ k/2, restricting solutions to [-k/2, k/2].

Common Misreadings

  • Treating |f(x)| = g(x) as if both sides are symmetric (like |f| = |g|) and skipping the non-negativity check.
  • Believing that case-splitting alone handles all sign issues, so no domain restriction is needed.

Pitfalls

  • Forgetting the constraint and accepting a candidate root x₀ where g(x₀) < 0 — it satisfies one case's linear equation but not the original modulus equation.
  • Reversing the inequality (writing g(x) ≤ 0 or |x| ≥ k/2) and excluding the correct interval.
  • Using strict inequality g(x) > 0 instead of g(x) ≥ 0 and missing boundary solutions where the modulus equals zero.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P2/P3) modulus equation work; no formula booklet entry is needed — this is a structural argument from the definition |x| ≥ 0. Avoid generalizing to complex modulus equations |z| = w, which are out of scope for this technique at IAL level.
