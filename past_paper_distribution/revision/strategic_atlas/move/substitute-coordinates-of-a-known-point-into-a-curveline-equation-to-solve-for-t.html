Coordinate Geometry

Substitute coordinates of a known point into a curve/line equation to solve for the unknown parameter

Family: Point On Line Substitution Frequency: 4 questions

Trigger — When to use

Fires when a specific point is asserted to 'lie on' a curve or line whose equation contains exactly one unknown parameter (a constant c, gradient m, radius/centre parameter k, or an unknown coordinate of the point itself). Visual cues: the question says 'the point P(a,b) lies on …', 'the line passes through …', or 'C is tangent at …', AND the equation of the curve/line has one symbolic constant left undetermined. Equivalent phrasings: 'show that P satisfies …', 'find the value of k such that the circle passes through …', 'the line through (x₀,y₀) with gradient m has equation …'. The move is forced whenever the only remaining link between the named point and the symbolic equation is the membership relation.

Motivation — Why this works

Any point that lies on a locus must satisfy the locus's defining equation — this is the algebraic translation of the geometric statement 'P ∈ C'. Substituting the known coordinates eliminates x and y from the equation, collapsing a two-variable relation into a single-variable equation in the unknown parameter. For linear equations this yields one linear equation (one solution); for quadratic curves (circles, parabolas) it yields a polynomial in the parameter (typically up to degree 2), to be solved by factoring, the quadratic formula, or completing the square. This converts geometry into algebra in one stroke.

Thought Process — The step-by-step reasoning

  1. Notice that the equation of the curve/line still contains an unknown symbolic constant, while a specific point on it has been given or just deduced.
  2. Rule out alternatives: the parameter does not appear in the gradient/derivative or in any other available relation, so neither differentiation nor a distance-from-centre argument introduces new information beyond substitution.
  3. Commit to substitution because 'P lies on the equation' is the only remaining link between P's coordinates and the unknown — this is the forced final equation.
  4. Bridge to action: replace x with the x-coordinate of P and y with the y-coordinate of P verbatim, keeping any squared brackets in completed form when convenient (e.g. (p−3)² rather than expanding immediately).
  5. Tactical decision: if a quadratic in the parameter results, anticipate two roots and reread the question for a sign/range constraint (e.g. p > 0) that selects the correct one; if linear, expect a unique value.
  6. Finish by simplifying carefully — collect like terms, move everything to one side if a 'show that' target is given, and check the answer by re-substituting.

Mechanic — Concrete operations

  1. Write the curve/line equation with the unknown parameter still symbolic.
  2. Replace x by the x-coordinate of the given point and y by its y-coordinate.
  3. Simplify the numeric arithmetic (e.g. (0+2)² = 4, 4(−1) = −4).
  4. Expand any squared brackets only if necessary to combine like terms.
  5. Rearrange into standard form: linear → solve directly; quadratic → factor or use the formula.
  6. Apply any stated constraint (positivity, integer, range) to select the correct root.

Worked Micro-Example

Find c if the line $y = 4x + c$ passes through $(-1,-3)$. Substitute: $-3 = 4(-1) + c \Rightarrow -3 = -4 + c \Rightarrow c = 1$. So the line is $y = 4x + 1$.

Variants — Same move, different surface

Point on a circle with unknown coordinate, e.g. P(p,0) on (x−3)²+(y+2)²=169
Substitute to get (p−3)²+4=169; leave in completed-square form and square-root directly: p = 3 ± √165, then use sign constraint.
Point on a circle whose equation contains an unknown parameter k
Substitution yields a quadratic in k (since k may appear squared); expand fully and rearrange to standard form ak²+bk+c=0.
Gradient m known, one point known, find y-intercept c
Write y = mx + c, substitute the point, solve the resulting linear equation for c.
Tangent point known, line equation y = mx + c has unknown c
Substitute the tangent point into the line — the gradient role is already played by m, so only c remains to solve.
Two points given instead of gradient + point
Use y − y₁ = ((y₂−y₁)/(x₂−x₁))(x − x₁) instead — same membership principle, different surface form.

Common Misreadings

  • Treating y = 0 as if it removes the y-term entirely, e.g. dropping (0+2)² = 4 in a circle equation.
  • Substituting the gradient value for y instead of the y-coordinate of the point.
  • Trying to complete the square on a circle already in standard form before substituting (unnecessary work).

Pitfalls

  • Sign slip when the point has negative coordinates: writing 4(5) instead of 4(−5), or −4+c misread as 4+c.
  • Forgetting the cross-term when expanding (a − bk)² — gives a wrong linear coefficient in k.
  • Failing to bring all terms to one side when the question is 'show that … = 0'.
  • Expanding a clean (p−3)² unnecessarily instead of square-rooting directly.
  • Forgetting to apply a stated sign/range constraint after obtaining two roots from a quadratic.

A-level Scope Notes

Entirely within Edexcel IAL Pure Maths (P1–P2 for lines and circles, P3 for tangents). No formula booklet entry is needed beyond the standard line form y = mx + c and the circle form (x−a)²+(y−b)²=r². Avoid generalising to implicit differentiation or parametric loci, which lie outside the typical context of this move.
