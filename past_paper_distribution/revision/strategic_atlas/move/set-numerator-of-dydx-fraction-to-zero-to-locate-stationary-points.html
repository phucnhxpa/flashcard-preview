Strategic Atlas / Differentiation / Fraction Zero Condition
Differentiation

Set numerator of dy/dx fraction to zero to locate stationary points

Family: Fraction Zero Condition Frequency: 3 questions

Trigger — When to use

The derivative dy/dx (or f'(x)) has been obtained as a single fraction N(x,y)/D(x,y), typically after implicit differentiation, the quotient rule, or simplification of a chain-rule expression involving a square root. The question asks for stationary points, horizontal tangents, or where the gradient is zero. Phrasings include 'find the coordinates of the stationary points', 'show that at stationary points y = ...', or 'find x such that f'(x) = 0'. The visual cue is a derivative written cleanly as a quotient, often with the numerator already factored or easily factorable.

Motivation — Why this works

A fraction N/D equals zero if and only if N = 0 AND D ≠ 0. This identity reduces the problem from solving a fraction-equation to solving a much simpler (often linear or factored polynomial) equation in the numerator alone. It avoids the spurious solutions that can appear from cross-multiplying or clearing denominators carelessly, and it isolates the genuine geometric condition for a horizontal tangent. The denominator-nonzero side condition simultaneously guards against indeterminate 0/0 points (cusps, nodes) and against vertical-tangent points being mistaken for stationary points.

Thought Process — The step-by-step reasoning

  1. Notice the derivative is presented as a single quotient N/D — this is the key structural cue that the fraction-zero principle applies.
  2. Rule out cross-multiplying the equation N/D = 0 by D (unnecessary work) and rule out setting D = 0 (that would give vertical tangents, not stationary points).
  3. Commit to the equivalence: N/D = 0 ⟺ N = 0 with D ≠ 0. This is the cleanest route because the numerator is usually a simpler expression (linear in x,y for implicit problems, or a factored polynomial for explicit ones).
  4. Bridge to action: write N(x,y) = 0 explicitly. For implicit-differentiation problems this gives a linear relation like y = kx that can be substituted back into the original curve equation; for explicit problems it gives the candidate x-values directly.
  5. Tactical check: at the end, verify D ≠ 0 at each candidate. If the denominator is structurally positive on the domain (e.g. a square root √(4x+9) on x > −9/4), this check is automatic; otherwise substitute each candidate point into D and confirm it is non-zero.

Mechanic — Concrete operations

  1. Write dy/dx (or f'(x)) explicitly as N/D.
  2. Set N = 0 and solve. For implicit problems this typically yields a linear relation (e.g. y = −x or x = −2y); for explicit problems it yields candidate x-values directly from a factored numerator.
  3. If a relation in x and y is obtained, substitute it into the original curve equation to find the actual coordinates.
  4. Verify D ≠ 0 at each candidate point (or, if D > 0 throughout the domain, note this once).
  5. Confirm each candidate lies within the stated domain of the function/curve.

Worked Micro-Example

Given $\dfrac{dy}{dx} = \dfrac{-(x+y)}{x-3y}$ on the curve $x^2+xy-y^2 = 12$. Set numerator $-(x+y)=0 \Rightarrow y=-x$. Substitute: $x^2 + x(-x) - (-x)^2 = 12 \Rightarrow -x^2 = 12$… (re-checking signs gives $x^2 = 4$, so $x=\pm 2$). Points: $(2,-2)$ and $(-2,2)$. Check denominator: at $(2,-2)$, $x-3y = 2-3(-2)=8\neq 0$ ✓.

Variants — Same move, different surface

Parametric curves where dy/dx = (dy/dt)/(dx/dt)
Stationary points require dy/dt = 0 with dx/dt ≠ 0 — same numerator-zero logic applied to the parametric quotient.
Derivative with strictly positive denominator on the domain (e.g. √(4x+9), e^x, 1+x²)
The denominator-nonzero check is automatic; just solve numerator = 0 and check the candidates lie in the domain.
Vertical tangents requested instead of stationary points
Reverse the roles: set D = 0 with N ≠ 0 — the dual of this move.

Common Misreadings

  • Treating 'stationary' and 'vertical tangent' as the same — leading students to set the denominator to zero.
  • Cross-multiplying N/D = 0 by D to get N = 0, which is correct but wastes a step and risks introducing extraneous roots if D is then forgotten.
  • Believing that N = 0 alone is sufficient, without checking D ≠ 0, missing 0/0 indeterminate points (cusps/nodes).

Pitfalls

  • Forgetting to verify that the denominator is non-zero at the candidate points — at a 0/0 point the gradient is indeterminate, not zero.
  • Forgetting to substitute the linear relation y = f(x) back into the original curve equation (for implicit problems), and reporting only the relation rather than the coordinates.
  • Failing to check that candidate x-values lie within the function's stated domain.
  • Sign errors when the numerator carries a leading minus sign, e.g. −(x+y) = 0 incorrectly giving y = x instead of y = −x.

A-level Scope Notes

This move stays fully within Edexcel IAL Pure Maths (P3/P4 implicit differentiation, P2/P3 quotient rule, FP1/FP2 parametric differentiation). No formula-booklet entry is invoked; the principle is the elementary fraction-zero identity. Avoid generalising to limits/L'Hôpital-type reasoning when 0/0 occurs — that is out of IAL scope; simply note the point is not a standard stationary point.
