Trigger — When to use

The integrand contains e^{kx} for some non-zero constant k (e.g. e^{0.5x}, e^{-3x}, e^{2x}). The exponent is linear in x with no extra factor of x or other function. This is the cue to apply the standard result ∫e^{kx} dx = (1/k)e^{kx} + C directly, rather than reaching for a u-substitution.

Motivation — Why this works

Differentiating e^{kx} gives k·e^{kx} by the chain rule, so reversing this introduces a factor of 1/k on integration. Recognising the standard form means you read off the antiderivative immediately and only need to handle any external constant multiplier. This is faster, cleaner, and less error-prone than a substitution u = kx, which produces the same answer but with extra bookkeeping.

Thought Process — Step-by-step reasoning

  1. I notice the integrand has the form (constant)·e^{kx} with k a simple number.
  2. I rule out substitution: the exponent is linear in x with no other x-dependence, so a standard result suffices.
  3. I commit to ∫e^{kx} dx = (1/k)e^{kx} + C because it is a memorised result in the formula book / standard list.
  4. I bridge to execution by identifying k explicitly and computing 1/k as a clean number (e.g. 1/0.5 = 2).
  5. I multiply 1/k by any external constant in front of the exponential, keeping signs intact, then add +C at the very end.

Mechanic — Concrete operations

  1. Identify k in the exponent e^{kx}.
  2. Write the antiderivative as (1/k)e^{kx}.
  3. Multiply by any constant coefficient in front of the original exponential term.
  4. Combine with antiderivatives of the other terms in the integrand.
  5. Add a single constant of integration +C at the end (for indefinite integrals).

Worked Micro-Example

Evaluate $\int \left(e^{x} - 4e^{0.5x} + 4\right)\,dx$.\\ $\int e^{x}\,dx = e^{x}$; $\int -4e^{0.5x}\,dx = -4\cdot\tfrac{1}{0.5}e^{0.5x} = -8e^{0.5x}$; $\int 4\,dx = 4x$.\\ Therefore the integral equals $e^{x} - 8e^{0.5x} + 4x + C$.

Variants — Same move, different surface

Negative k, e.g. ∫e^{-3x} dx
Reciprocal is 1/(-3), giving -\tfrac{1}{3}e^{-3x} + C.
Fractional k, e.g. ∫e^{x/2} dx
k = 1/2 so 1/k = 2, giving 2e^{x/2} + C.
External coefficient, e.g. ∫5e^{2x} dx
Combine: 5·(1/2)e^{2x} = \tfrac{5}{2}e^{2x} + C.
Trigonometric analogues ∫cos(kx) dx and ∫sin(kx) dx
Same reciprocal pattern: (1/k)sin(kx) and -(1/k)cos(kx) respectively.
Definite version, e.g. ∫_0^2 e^{0.5x} dx
Apply the same antiderivative 2e^{0.5x} and evaluate at the limits: 2e^{1} - 2e^{0} = 2e - 2.

Common Misreadings

  • Treating e^{kx} as if it integrated to e^{kx} (forgetting the 1/k factor entirely).
  • Reading e^{0.5x} as e^{x}/2 or (e^x)^{0.5} = e^{x/2} but then integrating it as if k were 1.
  • Confusing the integration rule with the differentiation rule and multiplying by k instead of dividing.

Pitfalls

  • Multiplying by k (or by 0.5) instead of dividing by k — i.e. applying the derivative rule by mistake.
  • Dropping the external coefficient (e.g. losing the -4) when bringing 1/k out front.
  • Sign slips when k is negative, especially writing 1/(-3) as -1/3 incorrectly placed.
  • Forgetting the +C in indefinite integrals.

A-level Scope Notes

The result ∫e^{kx} dx = (1/k)e^{kx} + C is a standard formula assumed in Edexcel IAL Pure Maths (P2/P3) and reused throughout C34/Further Pure; no derivation beyond reversing the chain rule is required.
