Substitute data points into parameterised model to form simultaneous equations

Family: Algebraic Modelling Frequency: 1 questions

Trigger — When to use

A model equation contains exactly k unknown constants/parameters and the question supplies k data points (or boundary conditions) that the model must satisfy. Typical phrasings: 'find the values of p and q', 'the model passes through...', 'when t = ..., H = ...'. The model may look nonlinear in the variables, but the unknown parameters appear linearly (e.g. H^3 = pt^2 + q, y = ax^n + b with n given, or N = A + Bk^t with k given).

Motivation — Why this works

Each (input, output) pair is a constraint the model must satisfy, so substituting it produces one equation in the unknown parameters. When the parameters appear linearly, k substitutions yield a k×k linear system with a unique solution (provided the data points are independent). This reduces a modelling problem to standard simultaneous-equations technique, which is core A-level algebra.

Thought Process — The step-by-step reasoning

  1. I notice two unknown constants p and q in the model and exactly two numerical data points supplied.
  2. I rule out fitting from a single point (one equation cannot determine two unknowns) and rule out logarithmic linearisation since p and q already appear linearly.
  3. I commit to substituting each data point into the model to generate one equation per point.
  4. I evaluate the variable powers (e.g. H^3, t^2) numerically first, so each equation reduces to a linear form in p and q.
  5. I label the resulting equations (1) and (2) ready for elimination or substitution in the next move.

Mechanic — Concrete operations

  1. Identify the unknown parameters and confirm they appear linearly in the model.
  2. Count data points and check the number matches the number of unknowns.
  3. Substitute the first data point: evaluate any powers/functions of the variables to numerical values, leaving a linear equation in the parameters. Label as Equation (1).
  4. Substitute the second data point similarly. Label as Equation (2).
  5. Hand off the linear system to the simultaneous-equation solving move (elimination or substitution).

Worked Micro-Example

Model: $H^3 = pt^2 + q$. Given $(t,H) = (3,2)$ and $(5, 2.4)$.\ Sub (3,2): $2^3 = 9p + q \Rightarrow 8 = 9p + q$ … (1)\ Sub (5, 2.4): $2.4^3 = 25p + q \Rightarrow 13.824 = 25p + q$ … (2)\ (2)−(1): $5.824 = 16p \Rightarrow p = 0.364$, then $q = 8 - 9(0.364) = 4.724$.

Variants — Same move, different surface

Linear model $y = ax + b$ through two given points.
Substitute each point to get two linear equations in $a,b$; solve directly.
Power model $y = ax^n + b$ with $n$ given, two data points supplied.
Evaluate $x^n$ numerically at each point first, then equations become linear in $a,b$.
Exponential model $N = A + Bk^t$ with $k$ given, two data points.
Compute $k^t$ at each point; the resulting equations are linear in $A$ and $B$.
Three unknown parameters with three data points.
Same principle but produces a 3×3 linear system; solve by elimination or matrix methods (FP scope).
Boundary/initial conditions in differential-equation modelling.
Substitute the boundary value into the general solution to fix the constant of integration — same logic with one unknown and one condition.

Common Misreadings

  • Believing the model is genuinely nonlinear in p and q and reaching for logarithms or curve-fitting tools when simple substitution suffices.
  • Trying to determine both parameters from a single data point.
  • Confusing which symbol is the variable and which is the parameter, e.g. solving for t instead of p.
  • Reading H^3 = pt^2 + q as (pt)^2 + q or p(t^2 + q) due to bracketing inattention.

Pitfalls

  • Arithmetic slip in evaluating powers (e.g. computing $2.4^3$; correct value is 13.824, not 13.8 or 14.4).
  • Mislabelling equations or losing track of signs when subtracting (1) from (2).
  • Premature rounding of $H^3$ or $t^2$ values, propagating error into p and q.
  • Forgetting to substitute back to find the second parameter once the first is found.

A-level Scope Notes

Sits squarely within Edexcel IAL Pure Maths modelling and simultaneous-equations content (P1–P3); the 3×3 extension belongs to Further Pure matrix work.
