Integration

Divide by inner linear coefficient when integrating 1/(ax+b) forms

Family: Linear Chain Rule Frequency: 1 questions

Trigger — When to use

Fires whenever the integrand contains 1/(ax+b) or 1/(ax+b)^n with a ≠ 1, typically after partial fractions on a rational function with linear factors like (2x−1), (3x+4), or repeated linear factors. Also triggers in reverse-chain integrations of (ax+b)^n. Cue phrases: 'integrate', 'find ∫…dx', or part (ii) following a partial-fraction decomposition. The defining feature is a non-unit leading coefficient inside a linear bracket.

Motivation — Why this works

By the reverse chain rule, ∫1/(ax+b) dx = (1/a) ln|ax+b| + C, because d/dx[ln|ax+b|] = a/(ax+b). The factor 1/a compensates for the inner derivative a that would otherwise appear if we naively wrote ln|ax+b|. The same principle extends to ∫(ax+b)^n dx = (ax+b)^(n+1)/[a(n+1)] + C for n ≠ −1. Skipping the 1/a factor is one of the most penalised slips in IAL integration questions.

Thought Process — The step-by-step reasoning

  1. Notices the integrand has a linear bracket (ax+b) with a ≠ 1, signalling a chain-rule adjustment is needed.
  2. Rules out u-substitution as overkill — the linear case has a fixed shortcut (1/a) factor.
  3. Commits to the template ∫A/(ax+b) dx = (A/a) ln|ax+b| + C after confirming the bracket is linear, not quadratic.
  4. Bridges to computation by reading off A and a directly from the integrand without expanding.
  5. Decides whether each term is a log (n = −1) or a power (n ≠ −1) and applies the matching template.

Mechanic — Concrete operations

  1. Identify the linear bracket (ax+b) and read off the inner derivative a.
  2. If exponent is −1: ∫A/(ax+b) dx = (A/a) ln|ax+b| + C.
  3. If exponent n ≠ −1: ∫A(ax+b)^n dx = A(ax+b)^(n+1)/[a(n+1)] + C.
  4. Simplify the leading coefficient (A/a or A/[a(n+1)]) before combining with other terms.
  5. Add a single constant of integration at the end of the whole expression.

Worked Micro-Example

Compute $\int \dfrac{-52}{25(2x-1)}\,dx$. Here $A=-52/25$ and $a=2$, so the integral is $\dfrac{-52}{25}\cdot\dfrac{1}{2}\ln|2x-1| = -\dfrac{26}{25}\ln|2x-1| + C$. For the partner term $\int \dfrac{2/5}{(x-3)^2}\,dx$, we have $a=1$, $n=-2$, giving $\dfrac{2}{5}\cdot\dfrac{(x-3)^{-1}}{-1} = -\dfrac{2}{5(x-3)} + C$.

Variants — Same move, different surface

Repeated linear factor 1/(ax+b)^2 after partial fractions
Use the power rule: result is −1/[a(ax+b)] · A, i.e. divide by a once for the chain rule.
Integrand (ax+b)^n with n a positive integer or fraction
Result is (ax+b)^(n+1)/[a(n+1)]; the 1/a factor still compensates for the inner derivative.
Definite integral ∫_p^q 1/(ax+b) dx producing ln of a ratio
Apply the (1/a) factor first, then evaluate (1/a)[ln|aq+b| − ln|ap+b|] = (1/a) ln|(aq+b)/(ap+b)|.
Reverse chain on e^(ax+b) or sin(ax+b)
Same 1/a compensation: ∫e^(ax+b) dx = (1/a)e^(ax+b)+C, ∫sin(ax+b) dx = −(1/a)cos(ax+b)+C.

Common Misreadings

  • Treating 1/(2x−1) as if it integrates to ln|2x−1| with no 1/2 factor.
  • Confusing a quadratic denominator like 1/(x^2−1) with a linear case and applying the shortcut directly instead of doing partial fractions first.
  • Mistaking the constant A in front for part of the bracket and dividing it in twice.

Pitfalls

  • Forgetting the 1/a factor entirely when a ≠ 1.
  • Dividing by a twice — once mentally during the antiderivative and again when simplifying.
  • Dropping the modulus signs in ln|ax+b|, which IAL mark schemes generally require.
  • Applying (1/a) to the power-rule term as well when the exponent is positive without also accounting for the (n+1) denominator.

A-level Scope Notes

This is core Edexcel IAL C34/P3 integration material; only linear inner functions need this shortcut, while non-linear inner functions require explicit substitution which is also in scope.
