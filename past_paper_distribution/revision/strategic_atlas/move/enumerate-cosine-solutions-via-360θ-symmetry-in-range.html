Trigonometry

Enumerate cosine solutions via 360°−θ symmetry in range

Family: Cast Enumeration Frequency: 1 questions

Trigger — When to use

Fires when an equation reduces to cosθ = k (with |k| ≤ 1) and the question asks for all θ in a finite range such as [0°, 360°) or [0, 2π). Common cues: 'find all values of θ', 'solve in the interval', or after a rearrangement that isolates cos. Particularly relevant when k is negative, forcing Q2/Q3 solutions, or positive, forcing Q1/Q4 solutions.

Motivation — Why this works

The calculator's cos⁻¹ returns only the principal value in [0°, 180°], giving one solution. Cosine is even and periodic with the symmetry cos(360° − α) = cos(α), so the second solution in [0°, 360°) is obtained by reflecting the principal value about 360°. Using CAST or the cosine graph guarantees the complete solution set in the specified range without missing or duplicating values.

Thought Process — The step-by-step reasoning

  1. Notice the equation has the form cosθ = k with k a fixed numerical value, and a finite range is specified.
  2. Rule out sine-style symmetry (180° − θ): the function is cosine, so the relevant symmetry is about 0°/360°, not 180°.
  3. Commit to the CAST/graph approach because cosine takes each value in (−1, 1) exactly twice per period, so exactly two solutions are expected in [0°, 360°).
  4. Compute the principal value θ₁ = cos⁻¹(k); decide its quadrant from the sign of k (Q1 if k > 0, Q2 if k < 0).
  5. Generate θ₂ = 360° − θ₁, then check both lie within the specified range and round to the accuracy demanded by the mark scheme.

Mechanic — Concrete operations

  1. Identify the form cosθ = k and the required range.
  2. Compute principal value θ₁ = cos⁻¹(k) (in degrees or radians as required).
  3. Apply symmetry: θ₂ = 360° − θ₁ (or 2π − θ₁ in radians).
  4. Check both θ₁ and θ₂ lie inside the stated range; adjust by ±360° if the range is shifted (e.g. [−180°, 180°)).
  5. Present both solutions to the required accuracy (typically awrt 0.1° or 3 s.f.).

Worked Micro-Example

Solve $\cos\theta = -\tfrac{1}{3}$ for $\theta \in [0°, 360°)$. Principal value: $\theta_1 = \cos^{-1}(-\tfrac{1}{3}) \approx 109.47°$ (Q2). Symmetry: $\theta_2 = 360° - 109.47° \approx 250.53°$ (Q3). Both lie in $[0°, 360°)$, so $\theta \approx 109.5°,\ 250.5°$.

Variants — Same move, different surface

cosθ = k with k > 0 in [0°, 360°)
Principal value lies in Q1; second solution is 360° − θ₁ in Q4.
sinθ = k in [0°, 360°)
Use sine symmetry: solutions are θ₁ and 180° − θ₁, not 360° − θ₁.
tanθ = k in [0°, 360°)
Period is 180°, so the second solution is θ₁ + 180° rather than a reflection.
Range [−180°, 180°)
Replace 360° − θ₁ with −θ₁, exploiting cos being even about 0°.
Range in radians, e.g. [0, 2π)
Use θ₂ = 2π − θ₁; principal value from cos⁻¹ is in [0, π].
Composite argument cos(nθ) = k or cos(θ + φ) = k
Solve for the inner argument over an expanded range, then divide/shift back to recover θ.

Common Misreadings

  • Applying the sine rule 180° − θ₁ to a cosine equation, producing a wrong second solution.
  • Believing cos⁻¹ of a negative number returns a negative angle and attempting to 'correct' it.
  • Treating cosθ = k as having only one solution per period.
  • Confusing cosine's symmetry axis (about 0°/180°) with sine's (about 90°/270°).

Pitfalls

  • Omitting the second solution and losing the enumeration mark.
  • Quoting 360° + θ₁, which lies outside [0°, 360°).
  • Premature rounding of θ₁ before computing 360° − θ₁, causing the second answer to fail awrt tolerance.
  • Forgetting to widen the range first when the argument is nθ or θ + φ.

A-level Scope Notes

Standard Edexcel IAL Pure (P2/P3) technique for solving trigonometric equations on a bounded interval; uses only cos⁻¹, basic symmetry, and the unit circle/CAST diagram.
