Algebra & Functions

Complete the square via expanding target form and comparing coefficients

Family: Completing The Square Frequency: 1 questions

Trigger — When to use

The question asks to express a quadratic in the form a(x+b)² + c (or p(x+q)² + r), especially when the leading coefficient is not 1. Phrases like 'express in the form...', 'find the values of a, b, c', or 'hence write down the minimum value' all signal this move. It also fires when the student wants a method that bypasses the 'factor-out-then-complete' route. Any quadratic where vertex form is required is a candidate.

Motivation — Why this works

Expanding a(x+b)² + c yields ax² + 2abx + (ab² + c), so matching against the given quadratic produces three equations — one per power of x — for the three unknowns a, b, c. Because the equations decouple (a from x², then b from x, then c from the constant), the system solves by a clean cascade rather than simultaneous algebra. This works for any leading coefficient and avoids memorising the 'halve the coefficient' shortcut. The identity (x+b)² = x² + 2bx + b² is the only fact needed.

Thought Process — The step-by-step reasoning

  1. Notices the target form a(x+b)² + c has three unknowns and the source quadratic has three coefficients — a determined system.
  2. Rules out the monic shortcut 'halve the x-coefficient' because the leading coefficient is not 1, and rules out factoring-out as merely an alternative route.
  3. Commits to coefficient comparison because expanding the target form and equating powers of x is mechanical and hard to misexecute.
  4. Bridges to action by writing the identity ax² + 2abx + ab² + c ≡ given quadratic and lining up powers of x.
  5. Solves in cascade order: a from x², then b from the x-term using the now-known a, then c from the constant using both.

Mechanic — Concrete operations

  1. Write the identity: given quadratic ≡ a(x+b)² + c, and expand RHS to ax² + 2abx + ab² + c.
  2. Compare x² coefficients to get a directly.
  3. Compare x coefficients: equate to 2ab and solve for b using the known a.
  4. Compare constant terms: equate to ab² + c and solve for c using the known a and b.
  5. State the completed form a(x+b)² + c with the found values, and (if asked) read off the vertex (−b, c) or minimum value c.

Worked Micro-Example

Express $3x^2 + 12x + 13$ as $a(x+b)^2 + c$. Expand: $a(x+b)^2 + c = ax^2 + 2abx + ab^2 + c$. Compare $x^2$: $a = 3$. Compare $x$: $2ab = 12 \Rightarrow 6b = 12 \Rightarrow b = 2$. Compare constants: $ab^2 + c = 13 \Rightarrow 12 + c = 13 \Rightarrow c = 1$. Hence $3x^2 + 12x + 13 = 3(x+2)^2 + 1$.

Variants — Same move, different surface

Monic quadratic x² + px + q
Skip the a-equation (a = 1 automatically); the 'halve p' shortcut gives (x + p/2)² + (q − p²/4) immediately.
Quadratic with negative leading coefficient, e.g. −2x² + 8x − 3
Same procedure; a comes out negative (a = −2), and the parabola opens downward so c is the maximum value, not minimum.
Factor-out-first route
Factor the leading coefficient: 3x² + 12x + 13 = 3[x² + 4x] + 13, complete inside brackets to 3[(x+2)² − 4] + 13 = 3(x+2)² + 1. Same answer, different bookkeeping.
Quadratic in a substituted variable, e.g. 2u² − 12u + 5 where u = e^x or u = sin x
Apply the same coefficient-matching to get 2(u−3)² − 13, then substitute back; useful for finding ranges of trig/exponential expressions.
Discriminant or 'show always positive' problems
After completing the square, argue from a(x+b)² ≥ 0 to deduce minimum value c, hence the sign of the quadratic for all real x.

Common Misreadings

  • Assuming a = 1 and trying to fit the quadratic into (x+b)² + c, ignoring the leading coefficient.
  • Confusing a(x+b)² + c with (ax+b)² + c, which expands to a²x² + 2abx + b² + c and gives wrong values.
  • Treating the requested form as a(x−b)² + c and reading off b with the wrong sign.

Pitfalls

  • Forgetting to distribute a through the b² term: writing the constant as b² + c instead of ab² + c.
  • Sign error in the vertex: reporting it as (b, c) instead of (−b, c).
  • Arithmetic slip when computing ab² (e.g. 3·2² = 12, not 6) before solving for c.
  • Solving for b without first substituting the value of a, leading to a b/a confusion.

A-level Scope Notes

Standard Edexcel IAL Pure content (P1): completing the square for quadratics with any real leading coefficient, used for vertex/min-max, sketching, and feeding into discriminant or quadratic-inequality questions.
