Coordinate Geometry

Find farthest point on circle from a point on it via diameter extension

Family: Extremal Diameter Argument Frequency: 1 questions

Trigger — When to use

Fires when a question asks for the point S on a circle C such that the distance QS is maximised, where Q is a known point lying on C. Watch for phrases like 'greatest distance', 'maximum value of QS', or 'farthest point on the circle from Q'. The centre and radius of C are typically given or easily found by completing the square. No calculus or parametrisation is required once the geometric configuration is recognised.

Motivation — Why this works

The longest chord of a circle is a diameter, of length $2r$. So if Q lies on the circle, the point S on the circle maximising $QS$ must be the other end of the diameter through Q — i.e. the reflection of Q in the centre. This collapses an apparent optimisation problem into a single midpoint-reversal calculation, exploiting the symmetry of the circle about its centre.

Thought Process — The step-by-step reasoning

  1. Notice that both Q and S lie on the same circle, and the question asks to maximise QS — a chord length.
  2. Rule out calculus or parametric optimisation: although $S=(a+r\cos\theta, b+r\sin\theta)$ would work, it's overkill given the symmetry.
  3. Commit to the geometric fact that the longest chord is a diameter, so S must be diametrically opposite Q.
  4. Bridge to computation: the centre $C$ is the midpoint of the diameter $QS$, so $S$ is found by reversing the midpoint formula.
  5. Tactically, just compute $S = 2C - Q$ componentwise; no need to find the radius or write a circle equation.

Mechanic — Concrete operations

  1. Identify the centre $(a,b)$ of circle C (complete the square if needed).
  2. Confirm Q lies on C (substitute and check), so QS is a chord with maximum length when it is a diameter.
  3. Use the fact that the centre is the midpoint of the diameter: $a = \tfrac{x_Q + x_S}{2}$, $b = \tfrac{y_Q + y_S}{2}$.
  4. Solve: $x_S = 2a - x_Q$ and $y_S = 2b - y_Q$.
  5. State $S = (2a - x_Q,\ 2b - y_Q)$ and, if asked, $QS = 2r$.

Worked Micro-Example

Circle C has centre $(8,-6)$ and passes through $Q(15,-30)$. Find the point $S$ on C with $QS$ greatest. Since QS must be a diameter, $C$ is the midpoint of $QS$: $x_S = 2(8) - 15 = 1$, $y_S = 2(-6) - (-30) = 18$. So $S = (1, 18)$, and $QS = 2r = 2\sqrt{7^2 + 24^2} = 50$.

Variants — Same move, different surface

Q is external to the circle, asking for the farthest point on C from Q
The farthest point still lies on the line through Q and the centre, but now it is the far intersection of that line with the circle; distance is $|QC| + r$.
Q external, asking for the nearest point on C from Q
Use the same line through the centre; nearest point is the near intersection, distance $|QC| - r$.
Asked for the maximum length of a chord through a fixed interior point P
Maximum chord through P is the diameter through P; reflect P's role and use centre as midpoint of that diameter's endpoints.
Asked only for the maximum distance $QS$ (not the point)
Skip finding S; answer is simply $2r$ once the radius is known.

Common Misreadings

  • Treating S as the point on C farthest from the centre (which is meaningless — every point on C is at distance r from the centre).
  • Attempting to differentiate $QS^2(\theta)$ from a parametrisation — correct but unnecessarily long.
  • Believing the answer requires the equation of the circle when only centre and Q are needed.

Pitfalls

  • Sign errors when applying $S = 2C - Q$, especially with negative coordinates.
  • Forgetting to verify Q is actually on the circle before invoking the diameter argument.
  • Confusing the midpoint formula direction: writing $S = (Q+C)/2$ instead of $S = 2C - Q$.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P2 coordinate geometry of the circle): uses only the midpoint formula and the fact that the longest chord is a diameter. No calculus or trigonometric parametrisation required.
