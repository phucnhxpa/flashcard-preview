Integration

Drop absolute value in ln when domain forces positivity

Family: Domain Simplification Frequency: 1 questions

Trigger — When to use

Fires immediately after producing a logarithm of the form ln|f(x)| from an integral such as ∫f'(x)/f(x) dx, when the question states a domain restriction (e.g. x > a, x ∈ [p,q]) that makes f(x) strictly positive throughout. Typical cues: 'for x > …', 'on the interval …', or a definite integral whose limits both lie in a region where f(x) > 0. Also fires when integrating reciprocal-linear forms like 1/(ax+b) with a stated domain ensuring ax+b > 0.

Motivation — Why this works

The general antiderivative ∫f'/f dx = ln|f(x)| + C uses absolute values because ln is only defined for positive arguments and f could change sign. If the problem's domain guarantees f(x) > 0, then |f(x)| = f(x) and the bars are redundant. Removing them aligns with mark-scheme presentation and, more importantly, makes subsequent log laws (ln A − ln B = ln(A/B), n ln A = ln A^n) apply cleanly without sign-handling.

Thought Process — The step-by-step reasoning

  1. Notice that the antiderivative has just been written with absolute value bars, e.g. ln|4x+5|.
  2. Rule out leaving the bars in: check whether the question gives a domain restriction; if it does, the bars are likely unnecessary baggage.
  3. Commit once the domain condition is verified to force the argument strictly positive on the entire region of integration.
  4. Bridge to operation: substitute |f(x)| → f(x) in every log term produced by the integration.
  5. Tactically, do this BEFORE combining logs or substituting limits, so that algebraic manipulation (combining, exponentiating to solve for y) is cleanest.

Mechanic — Concrete operations

  1. Identify the stated domain condition (e.g. x > −5/4).
  2. Solve the inequality f(x) > 0 and confirm the stated domain lies inside it.
  3. State briefly: 'since x > −5/4, 4x+5 > 0, so |4x+5| = 4x+5'.
  4. Rewrite ln|f(x)| as ln(f(x)) throughout the working.
  5. Proceed with log laws, limit substitution, or solving for y as required.

Worked Micro-Example

Compute $\int \frac{4}{4x+5}\,dx$ for $x > -\tfrac{5}{4}$.\nGeneral antiderivative: $\ln|4x+5| + C$.\nSince $x > -\tfrac{5}{4} \Rightarrow 4x+5 > 0$, we have $|4x+5| = 4x+5$.\nFinal answer: $\ln(4x+5) + C$.

Variants — Same move, different surface

Integrating $1/x$ on a stated interval $x>0$
Write $\ln x$ directly instead of $\ln|x|$.
Definite integral $\int_a^b f'/f\,dx$ with both limits in a region where $f>0$
Drop bars before substituting limits, giving $\ln f(b) - \ln f(a) = \ln(f(b)/f(a))$.
Separation of variables in a differential equation, e.g. $dy/dx = ky$ with $y>0$ from initial condition
Write $\ln y$ instead of $\ln|y|$ so exponentiating gives $y = Ae^{kx}$ without sign ambiguity.
Partial fractions producing $\ln|x-a| + \ln|x-b|$ on a domain where both factors are positive
Drop both sets of bars, then combine via $\ln((x-a)(x-b))$.

Common Misreadings

  • Believing the absolute value must always be retained because 'that's the formula'.
  • Thinking the domain restriction only affects the limits of integration, not the form of the antiderivative.
  • Confusing 'argument positive' with 'integrand positive' — it is the argument of ln that must be checked.

Pitfalls

  • Dropping the bars without checking — e.g. on $x > -2$ for $\ln|2x+5|$, the argument can still be negative.
  • Forgetting to drop bars before applying log laws, then incorrectly writing $\ln|A|-\ln|B| = \ln|A/B|$ when the signs differ.
  • Stating no justification on the script; examiners expect a brief 'since x > … , …+… > 0' remark.
  • Dropping bars globally when the domain only covers part of the positivity region for a piecewise problem.

A-level Scope Notes

Standard for Edexcel IAL P3/P4 integration of $f'/f$ forms and for first-order separable differential equations where an initial condition or stated domain pins down the sign of the log argument.
