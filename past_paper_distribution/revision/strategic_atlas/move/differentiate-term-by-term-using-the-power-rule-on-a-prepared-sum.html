Differentiation

Differentiate term-by-term using the power rule on a prepared sum

Family: Power Rule Application Frequency: 2 questions

Trigger — When to use

Fires when the function has been written (or rewritten) as a finite sum of terms each in the form ax^n, with no products, quotients, or compositions of x remaining. Visual cues: every term is a constant multiple of a single power of x, possibly with fractional or negative exponents (e.g. 6x^{-1/2} + 2x, or (1/4)x^3 - 3x^{1/2} + 17x^{-1}). Equivalent phrasings: 'the expression is in index form', 'a polynomial-like sum of powers', or 'each term is ax^n'. Whenever you see this layout and the question asks for dy/dx or f'(x), the standalone power rule is the correct and most efficient tool.

Motivation — Why this works

The derivative of a sum equals the sum of the derivatives, so each ax^n term can be differentiated independently with the power rule d/dx(ax^n) = n a x^{n-1}. This avoids the product, quotient, and chain rules entirely, which is why the preparation step (rewriting roots and fractions as powers) was worth doing. The result is a clean general gradient function in one systematic pass, ready for substitution to find gradients, tangents, normals, or stationary points.

Thought Process — The step-by-step reasoning

  1. Notice that every term in the (prepared) function is of the form ax^n — a constant times a single power of x — so the structure matches the power rule template exactly.
  2. Rule out the product rule (no two x-factors multiplied), the quotient rule (any fractions have already been split into separate powers), and the chain rule (no inner function of x). The function is just a sum, so only the sum rule + power rule are needed.
  3. Commit to differentiating term-by-term: this is faster, cleaner, and avoids the most common error of trying to apply quotient rule to something like 17/x when it should just be 17x^{-1}.
  4. For each term, identify the coefficient a and exponent n, then write down n·a·x^{n-1}. Be deliberate: do the coefficient multiplication first, then carefully subtract 1 from the exponent (handling negatives and fractions explicitly).
  5. Recombine the differentiated terms with the same signs they had originally, giving the full derivative dy/dx.

Mechanic — Concrete operations

  1. Identify each term as a_i x^{n_i}.
  2. For each term, compute the new coefficient: n_i · a_i.
  3. For each term, compute the new exponent: n_i - 1 (taking care with negative/fractional values, e.g. -1/2 - 1 = -3/2, and -1 - 1 = -2).
  4. Write the differentiated term as (n_i a_i) x^{n_i - 1}.
  5. Sum all differentiated terms to obtain dy/dx.

Worked Micro-Example

Given $y = 6x^{-1/2} + 2x$: - $\frac{d}{dx}(6x^{-1/2}) = (-\tfrac{1}{2})(6)x^{-1/2 - 1} = -3x^{-3/2}$ - $\frac{d}{dx}(2x) = (1)(2)x^{0} = 2$ - $\therefore \frac{dy}{dx} = -3x^{-3/2} + 2$.

Variants — Same move, different surface

Function originally a single fraction like (x^3 + 2x)/√x
First split the fraction into x^{5/2} + 2x^{1/2}, then apply the power rule term-by-term.
Function contains a constant term c
Treat c as cx^0; its derivative is 0·c·x^{-1} = 0, so constants vanish.
Brackets to expand first, e.g. (x+1)(x-3)/√x
Expand and divide through by x^{1/2} to put each term in ax^n form before differentiating.

Common Misreadings

  • Treating 17/x as a quotient and reaching for the quotient rule instead of rewriting it as 17x^{-1}.
  • Treating √x as needing the chain rule rather than rewriting as x^{1/2}.
  • Believing a constant term needs special treatment instead of simply differentiating to 0.

Pitfalls

  • Subtracting 1 from a fractional exponent incorrectly: -1/2 - 1 = -3/2, not -1/2 or +1/2.
  • Sign errors with negative exponents: -1 - 1 = -2, not 0.
  • Forgetting to multiply the coefficient by the old exponent (e.g. writing -1/2·x^{-3/2} instead of -3x^{-3/2}).
  • Writing x^0 = 0 instead of x^0 = 1, which kills a term that should survive (e.g. d/dx(2x) = 2, not 0).
  • Multiplying fractional coefficients carelessly: (1/2)·(-3) = -3/2, not -3 or -1/2.

A-level Scope Notes

Fully within Edexcel IAL P1/P2 differentiation. The power rule d/dx(x^n) = nx^{n-1} is assumed knowledge (not in the formula booklet). Valid for all real n on the domain where x^n is defined; do not generalize to differentiating a^x or x^x by this rule — those need other techniques.
