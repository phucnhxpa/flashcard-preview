Complex Numbers

Apply zero-product property to a pre-factored polynomial

Family: De Moivre Root Extraction Frequency: 1 questions

Trigger — When to use

The polynomial equation f(x) = 0 is presented already factored as a product of lower-degree polynomials, e.g. f(x) = (x² + 9)(x² − 10x + d²). Cues include explicit brackets multiplied together, or a problem statement that hands you the factorisation. This fires whenever the equation is set to zero and at least one factor is non-trivial. The move is the reflex to split rather than expand.

Motivation — Why this works

Over ℂ (an integral domain), a product equals zero iff at least one factor is zero. This decouples a degree-n problem into independent lower-degree problems, each attackable by standard tools (quadratic formula, completing the square, De Moivre for x^n = c). Expanding the product would destroy the structure and force you back into a harder factorisation problem. Preserving the given factorisation is therefore both faster and safer.

Thought Process — The step-by-step reasoning

  1. Notices that f(x) is already displayed as a product of brackets equal to zero.
  2. Rules out expanding to a single quartic/cubic since that would discard hard-won structure and force a re-factorisation.
  3. Commits to the zero-product principle: solve each factor = 0 independently.
  4. Bridges to first operation by writing each factor on its own line, equated to zero.
  5. Decides which standard technique to use per factor (quadratic formula, x² = −k giving x = ±i√k, or De Moivre for higher-order factors).

Mechanic — Concrete operations

  1. Read off each factor of f(x).
  2. Set each factor equal to zero, producing independent sub-equations.
  3. Solve each sub-equation by the appropriate standard method (quadratic formula, square-root extraction, De Moivre).
  4. Collect all roots; check the count matches the degree of f(x) (counting multiplicity).

Worked Micro-Example

Solve $(x^2+9)(x^2-10x+d^2)=0$ for real $d>5$.\\ Factor 1: $x^2+9=0 \Rightarrow x=\pm 3i$.\\ Factor 2: $x^2-10x+d^2=0 \Rightarrow x=\dfrac{10\pm\sqrt{100-4d^2}}{2}=5\pm i\sqrt{d^2-25}$.\\ Four roots: $\pm 3i,\;5\pm i\sqrt{d^2-25}$.

Variants — Same move, different surface

Cubic given as (linear)(quadratic), e.g. $(x-2)(x^2+x+1)=0$
Read off the linear root directly; apply quadratic formula to the quadratic factor.
Equation of the form $(z^n - a)(z^2+bz+c)=0$
Use De Moivre / nth roots on the first factor and the quadratic formula on the second.
Repeated factor, e.g. $(x^2+1)^2(x-3)=0$
Same split, but record multiplicities so the total root count equals the degree.
Polynomial not pre-factored
First obtain a factor (factor theorem, given root, conjugate-root pairing) before this move can be applied.

Common Misreadings

  • Expanding the product into a single quartic and then attempting to re-factorise from scratch.
  • Treating the equation as if both factors must vanish simultaneously rather than 'either/or'.
  • Discarding the $x^2+9=0$ factor because it has no real roots, forgetting the question is over ℂ.

Pitfalls

  • Sign slip when solving $x^2 = -k$; the roots are $\pm i\sqrt{k}$, not $\pm\sqrt{k}$.
  • Mis-applying the quadratic formula when the discriminant is negative — must write the result as $\alpha \pm i\beta$.
  • Failing to verify that the total number of roots (with multiplicity) equals the degree of f(x).

A-level Scope Notes

Fully within Edexcel IAL FP1/FP2 scope: zero-product reasoning over ℂ, quadratic formula with negative discriminant, and complex roots in conjugate pairs are all standard.
