Trigonometry

Select correct quadrant solutions using sine symmetry and reference angles

Family: Quadrant Selection Frequency: 2 questions

Trigger — When to use

Fires whenever you have solved sin θ = k and need a solution that the calculator's principal value (always in [−90°, 90°]) does NOT directly give. Concrete cues: (i) the question states the angle is obtuse, reflex, or in a specific quadrant; (ii) the question asks for ALL solutions in a range like 0° ≤ x < 360° or 0 ≤ x < 2π; (iii) the geometric context (e.g. an obtuse interior angle of a parallelogram) forces a non-acute answer. Alternative phrasings: 'find the obtuse angle', 'solve for x in the given interval', 'state all values of θ such that...'.

Motivation — Why this works

Sine is positive in quadrants I and II and negative in quadrants III and IV, with the symmetries sin θ = sin(180° − θ) and sin θ = sin(180° + (−θ)) in the negative case. A calculator's arcsin only ever returns the principal value in [−90°, 90°], so any second-quadrant or third-quadrant solution must be recovered by hand. Using a reference angle α = arcsin|k| together with the CAST quadrant rules gives a systematic, error-free way to enumerate every solution in the required range and to pick out the one matching a stated geometric constraint.

Thought Process — The step-by-step reasoning

  1. Notice that the calculator output θ₁ = arcsin(k) is acute (or, for negative k, in Q4 with a negative measure), but the question imposes an extra constraint — either an interval like [0°, 360°) or a descriptor like 'obtuse'.
  2. Rule out simply quoting the principal value: that would silently ignore the second solution that sine's symmetry guarantees exists.
  3. Rule out using the cosine rule rerun (which can return obtuse angles directly) — the problem has already been reduced to sin θ = k, so I must handle the ambiguity at the trig-equation level.
  4. Commit to the symmetry method: identify the reference angle α = arcsin|k|, decide which quadrants are valid from the sign of k and any extra constraint, then write down x = α, 180°−α, 180°+α, or 360°−α as appropriate.
  5. Tactical choice: keep α to full calculator precision until the very last step, only rounding the final answer, to avoid accumulated error in any follow-up part of the question.
  6. Sanity-check each candidate against the stated range and any 'obtuse/acute/reflex' wording, and watch for boundary cases (sin x = ±1 yields a single solution, not two).

Mechanic — Concrete operations

  1. Compute the reference angle α = arcsin|k| from the calculator (keep full precision).
  2. Determine the sign of k and select target quadrants: k > 0 → Q1, Q2; k < 0 → Q3, Q4.
  3. Write candidate solutions using the standard formulas: Q1: α; Q2: 180° − α; Q3: 180° + α; Q4: 360° − α.
  4. Apply any extra constraint from the question (obtuse → take 180° − α; specific interval → discard out-of-range values).
  5. Handle boundary cases separately: sin x = 1 ⇒ x = 90° only; sin x = −1 ⇒ x = 270° only.
  6. Round the surviving solution(s) to the precision required, only at the final step.

Worked Micro-Example

Solve $\sin\theta = \tfrac{2}{3}$ for an obtuse $\theta$. Reference angle $\alpha = \arcsin(2/3) \approx 41.8103°$. Since $\theta$ must be obtuse, take the Q2 value: $\theta = 180° - 41.8103° \approx 138.19°$.

Variants — Same move, different surface

All solutions of sin x = k in 0° ≤ x < 360° with k < 0
Use Q3 and Q4 formulas: x = 180° + α and x = 360° − α, where α = arcsin|k|.
Equation in radians, 0 ≤ x < 2π
Replace 180° with π and 360° with 2π throughout; otherwise the method is identical.
Cosine equation cos x = k
Cosine is symmetric about 0°: solutions are x = α and x = 360° − α (or equivalently ±α). For k < 0, use Q2 (180° − α) and Q3 (180° + α).
Transformed argument, e.g. sin(2x + 30°) = −1/2 in 0° ≤ x < 360°
First find every solution for the inner argument u = 2x + 30° over its EXTENDED range, then back-solve x = (u − 30°)/2 and filter.
Boundary value sin x = 1 or sin x = −1
Only one solution in [0°, 360°): 90° or 270° respectively — do not duplicate via the two-quadrant formula.

Common Misreadings

  • Accepting the calculator's acute principal value when the question explicitly demands an obtuse angle.
  • Believing arcsin returns obtuse values automatically (confusing it with the cosine rule's behaviour).
  • Mis-assigning the sign convention — e.g. placing solutions of sin x = −1/2 into Q1 and Q2 instead of Q3 and Q4.
  • Reporting only the principal value when the question asks for ALL solutions in a stated interval.

Pitfalls

  • Rounding α before subtracting from 180° introduces avoidable error in any subsequent part of the question — keep full precision until the final answer.
  • Writing 180° + α when the constraint is 'obtuse' (this gives a Q3 reflex-like value, not an obtuse one).
  • Mechanically applying a two-quadrant rule to sin x = ±1 and producing a duplicate or out-of-range value.
  • Misreading the interval endpoint: 0° ≤ x < 360° excludes 360° itself, so a candidate like 360° must not be listed.

A-level Scope Notes

Sits squarely in Edexcel IAL P2/P3 trigonometry. Uses the identities sin(180° − θ) = sin θ and sin(180° + θ) = −sin θ, which appear in the formula booklet under standard trig identities. Do not extend to general-solution formulas (x = nπ + (−1)ⁿα) — that compact form is not required at IAL and may confuse interval filtering.
