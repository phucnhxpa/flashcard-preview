Differentiation

Differentiate and evaluate at the given point to obtain the tangent gradient

Family: Tangent Via Derivative Frequency: 2 questions

Trigger — When to use

The question asks for the equation of a tangent (or normal) to a curve at a specified point or x-value. You see phrases like 'find the equation of the tangent to C at the point where x = a' or 'at the point (a, b)'. The curve may be given explicitly y = f(x), implicitly F(x, y) = 0, or parametrically. The presence of a single specified point combined with the word 'tangent' is the unmistakable cue.

Motivation — Why this works

A straight line is fully determined by a point and a gradient, so the task reduces to producing both. The derivative dy/dx is precisely the instantaneous gradient of the curve, so evaluating it at the given point yields the gradient of the tangent there. For explicit y = f(x) this needs only an x-value; for implicit forms dy/dx involves both x and y, so the full point (x₀, y₀) is substituted. Combining the point and gradient with point-slope form y − y₀ = m(x − x₀) assembles the tangent line directly.

Thought Process — The step-by-step reasoning

  1. Notice the request is for a tangent line at a specific location — a line needs a point and a gradient, so identify what is missing.
  2. Rule out alternative techniques: a limit/secant approach is unnecessary when standard differentiation rules apply; solving an implicit equation for y just to differentiate explicitly is wasteful when implicit dy/dx is already available.
  3. Commit to: (i) secure the point on the curve, (ii) obtain dy/dx, (iii) evaluate dy/dx at that point to get m, (iv) substitute into y − y₀ = m(x − x₀).
  4. If only x is given, substitute it into the original curve to recover y₀; if both coordinates are given, use them directly (and optionally verify the point lies on the curve).
  5. Choose the differentiation route that matches the curve's form: power/chain rule for explicit y = f(x), implicit differentiation when y is entangled with x, parametric formula dy/dx = (dy/dt)/(dx/dt) for parametric curves.
  6. After computing m, decide between tangent (slope m) or normal (slope −1/m) based on what the question asked, then assemble the line.

Mechanic — Concrete operations

  1. Identify or compute the point of contact (x₀, y₀): if y₀ is missing, substitute x₀ into the curve equation.
  2. Differentiate to obtain dy/dx using the appropriate technique (power rule, product/quotient/chain rule, implicit differentiation, or parametric formula).
  3. Evaluate dy/dx at (x₀, y₀) — substitute x₀ alone for explicit forms, or both x₀ and y₀ for implicit forms — to obtain the gradient m.
  4. Write the tangent as y − y₀ = m(x − x₀); for a normal, replace m with −1/m.
  5. Simplify into the form requested (e.g. ax + by + c = 0 or y = mx + c).

Worked Micro-Example

Find the tangent to $y = x^{3/2} - 8x^{-1/2}$ at $x = 4$. Point: $y(4) = 8 - 4 = 4$, so $(4,4)$. Derivative: $\frac{dy}{dx} = \tfrac{3}{2}x^{1/2} + 4x^{-3/2}$. At $x=4$: $m = \tfrac{3}{2}(2) + 4 \cdot \tfrac{1}{8} = 3 + \tfrac{1}{2} = \tfrac{7}{2}$. Tangent: $y - 4 = \tfrac{7}{2}(x - 4)$.

Variants — Same move, different surface

Implicit curve, e.g. dy/dx = (4x + 3y)/(2y − 3x) at point (2, 7)
Substitute BOTH x and y into dy/dx: m = (8 + 21)/(14 − 6) = 29/8. No need to solve for y(x).
Normal line instead of tangent
Same procedure for finding m, but use gradient −1/m in the point-slope form.
Parametric curve x = f(t), y = g(t) at parameter t = t₀
Compute dy/dx = (dy/dt)/(dx/dt), evaluate at t₀, and use the point (f(t₀), g(t₀)).

Common Misreadings

  • Confusing 'tangent at x = 4' with 'tangent with gradient 4' — leading to solving dy/dx = 4 rather than evaluating dy/dx at x = 4.
  • Treating implicit dy/dx as a function of x only and forgetting to substitute the y-coordinate.

Pitfalls

  • Forgetting to compute the y-coordinate and accidentally using (x₀, 0) in point-slope form.
  • Sign errors when differentiating negative or fractional powers (e.g. d/dx[−8x^(−1/2)] = +4x^(−3/2), not −4x^(−3/2)).
  • Using the gradient of the normal when the tangent was asked, or vice versa.
  • Not verifying (when prudent) that the given point actually lies on the curve before computing the gradient.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P1–P4): power rule, chain/product/quotient rules, implicit differentiation (P3/P4), and parametric differentiation (P4). No formula booklet entry is required beyond standard derivatives; the point-slope form y − y₁ = m(x − x₁) is assumed knowledge. Avoid invoking higher-level ideas such as tangent planes or differential geometry.
