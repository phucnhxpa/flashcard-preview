Differentiation

Classify a stationary point via first-derivative sign change

Family: Sign Analysis Frequency: 1 questions

Trigger — When to use

Fires when a stationary point has been located (f'(x)=0) and you must classify it as max/min/inflection, AND f'(x) is already in a factored or sign-readable form. Especially attractive when f''(x) would be messy — e.g., when f'(x) is a product of several factors, contains exponentials/logs, or when f''(x_0) = 0 makes the second derivative test inconclusive.

Motivation — Why this works

By the definition of a local extremum, f is increasing on one side and decreasing on the other. The sign of f'(x) directly encodes this: a + → − change is a maximum, − → + is a minimum, no change is a (non-turning) inflection. When f'(x) is factored, each factor's sign on either side of the stationary point can be read off without further differentiation, making this faster and more transparent than the second derivative test.

Thought Process — The step-by-step reasoning

  1. Notice f'(x) is already in fully factored form, so signs of each factor near x_0 are easy to read.
  2. Rule out the second derivative test: differentiating a product of three factors again would be messy and error-prone.
  3. Commit to the first-derivative sign test because each factor's sign just left and just right of x_0 can be determined by inspection.
  4. Pick a convenient test value slightly below x_0 and another slightly above, ensuring no other stationary point lies between.
  5. Multiply the signs (not the values) of the factors at each test point and compare: + → − gives a max, − → + a min, no change an inflection.

Mechanic — Concrete operations

  1. Confirm x_0 is a stationary point by checking f'(x_0) = 0.
  2. Choose a test value x_L slightly less than x_0 (and inside any relevant domain), avoiding crossing another root of f').
  3. Evaluate the sign of each factor of f'(x_L) and combine to get the overall sign.
  4. Choose a test value x_R slightly greater than x_0 and repeat.
  5. Compare: + then − ⇒ local maximum; − then + ⇒ local minimum; same sign ⇒ point of inflection (non-turning).
  6. State the conclusion explicitly with reference to the sign change.

Worked Micro-Example

Given f'(x) = (x-2)e^{3x}(3x-4), so stationary points at x = 2 and x = 4/3. Classify x = 4/3.\\ At x = 1: (1-2)(+)(3-4) = (-)(+)(-) = + so f' > 0.\\ At x = 1.5: (1.5-2)(+)(4.5-4) = (-)(+)(+) = - so f' < 0.\\ Sign change + → −, hence x = 4/3 is a local maximum.

Variants — Same move, different surface

Classifying a minimum
Look for a sign change − → + across x_0.
Stationary point of inflection
f'(x) keeps the same sign on both sides of x_0; no turning occurs.
f''(x_0) = 0 (second derivative test inconclusive)
Fall back on the first-derivative sign test as the definitive classifier.
f'(x) involves a quadratic factor
Factorise (or use the sign of the leading coefficient and roots) to read signs in each region.
f'(x) contains e^{kx} or e^{g(x)}
These factors are always positive, so they don't affect the sign — but still note them so no factor is dropped silently.
Trig derivatives, e.g. f'(x) = cos x near x = π/2
Test values just below and above; cos changes from + to − so x = π/2 is a maximum of sin x.

Common Misreadings

  • Treating the test as evaluating f at the test points rather than f'.
  • Believing the second derivative test is always required for full marks — the first-derivative sign test is equally valid.
  • Reading 'sign change' as a change in the value of f rather than the sign of f'.

Pitfalls

  • Choosing a test value that lies beyond another stationary point, so the sign reflects a different interval.
  • Forgetting to include an always-positive factor (like e^{3x}) in the sign tally — harmless here but a habit-error elsewhere.
  • Picking test values outside the function's domain (e.g., negative x for a ln x problem).
  • Concluding 'maximum' or 'minimum' without explicitly stating the sign change as evidence.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P2/P3/P4) — stationary points and their classification are core content, and the first-derivative sign test is an accepted alternative to the second derivative test.
