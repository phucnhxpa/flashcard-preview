Algebra & Functions

Decompose reciprocal function as translation of y=1/x

Family: Transformation Recognition Frequency: 1 questions

Trigger — When to use

The function has the form y = a/(x−h) + k, with a constant subtracted from x inside the denominator and a constant added outside. Cues include any expression like 1/(x−2)+1, 3/(x+1)−2, or rearrangements such as (x+1)/(x−2) which simplify to 1 + 3/(x−2). Fires whenever a sketch, asymptotes, or transformation description is requested for a single-fraction rational function whose denominator is linear.

Motivation — Why this works

Every function of the form a/(x−h)+k is a horizontal translation by h, vertical translation by k, and vertical stretch by a applied to the parent y=1/x. Recognising this gives both asymptotes (x=h, y=k) and the two-branch hyperbolic shape immediately, without point-plotting. This leverages the syllabus's transformation rules f(x−h)+k rather than treating the curve as unfamiliar.

Thought Process — The step-by-step reasoning

  1. Notices the structure constant/(linear in x) + constant and recalls the parent y=1/x.
  2. Rules out building a table of values or using calculus to find features — transformation reading is faster and exact.
  3. Commits because the form matches f(x−h)+k with f(x)=a/x, so asymptotes and shape are determined.
  4. Bridges to operations: read h and k off directly to write asymptotes x=h and y=k.
  5. Decides to add intercepts (set x=0 and y=0) for precision marks and chooses branch orientation from the sign of a.

Mechanic — Concrete operations

  1. Match the function to the template y = a/(x−h) + k, identifying a, h, k.
  2. Vertical asymptote: set denominator = 0, giving x = h.
  3. Horizontal asymptote: as x → ±∞, the fraction → 0, so y → k.
  4. y-intercept: substitute x = 0 to get y = a/(−h) + k.
  5. x-intercept: set y = 0 and solve a/(x−h) = −k for x.
  6. Sketch two branches separated by the asymptotes, with orientation determined by the sign of a (a>0: branches in upper-right and lower-left of the asymptote cross; a<0: opposite).

Worked Micro-Example

Sketch $y = \dfrac{1}{x-2} + 1$. Match to $a=1, h=2, k=1$. Asymptotes: $x=2$ and $y=1$. y-intercept: $y = \tfrac{1}{-2}+1 = \tfrac{1}{2}$, giving $(0,\tfrac{1}{2})$. x-intercept: $\tfrac{1}{x-2} = -1 \Rightarrow x=1$, giving $(1,0)$. Two branches: one through both intercepts in $x<2, y<1$; one in $x>2, y>1$.

Variants — Same move, different surface

y = a/(x−h) + k with a ≠ 1
Same asymptotes x=h, y=k; the factor a stretches vertically and, if a<0, reflects so branches occupy the opposite pair of regions.
y = 1/(x−h)² + k (parent y=1/x²)
Same shift logic gives asymptotes x=h, y=k, but both branches lie above y=k since 1/(x−h)² > 0.
Improper fraction like y = (x+1)/(x−2)
First perform polynomial division to rewrite as 1 + 3/(x−2), then apply the standard translation reading.
y = 1/(2x−4) + 1
Factor the denominator as 2(x−2) so the function becomes (1/2)/(x−2)+1; asymptotes unchanged at x=2, y=1, with a halved vertical scale.

Common Misreadings

  • Placing the vertical asymptote at x = −h instead of x = h (sign confusion in f(x−h)).
  • Leaving the horizontal asymptote at y = 0 because the +k outside the fraction is overlooked.
  • Treating an improper fraction like (x+1)/(x−2) as already in standard form without first dividing.

Pitfalls

  • Drawing branches in the wrong pair of regions relative to the asymptotes, especially when a is negative.
  • Forgetting to label both intercepts and both asymptotes — each typically carries a mark.
  • Computing the y-intercept with a sign error from 1/(0−h) = −1/h.
