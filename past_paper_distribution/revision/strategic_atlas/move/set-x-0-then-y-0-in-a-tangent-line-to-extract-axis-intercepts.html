Trigger — When to use

Fires when a question asks for the points where a tangent (or normal) meets the coordinate axes, or for any quantity built from those intercepts: area of the triangle cut off by the tangent and the axes, midpoint of the intercept segment, distance between the intercepts, or a locus traced by such a midpoint as the parameter varies. Phrases like 'meets the x-axis at A and the y-axis at B', 'triangle OAB', or 'find the area of the triangle formed' are the standard cues. The move applies whenever you already have a clean linear (or linearised) form of the tangent.

Motivation — Why this works

A line written as (x cos θ)/a + (y sin θ)/b = 1 is linear in x and y, so substituting y=0 or x=0 collapses it to a one-variable equation that solves in a single step — no simultaneous system needed. The resulting intercepts a sec θ and b cosec θ are clean single-parameter trig expressions that drop straight into area, midpoint, distance or locus formulae. This works because an axis intercept is by definition the point on the line with the other coordinate zero; substitution is the algebraic image of that geometric definition.

Thought Process — Step-by-step reasoning

  1. Notices the question is asking for intercepts, or for a quantity (area, midpoint, locus) that secretly needs the intercepts.
  2. Rules out solving a system with the axis equations as a separate pair — that is just substitution dressed up — and rules out parametric/implicit differentiation, which doesn't produce intercepts.
  3. Commits because the tangent is already in a linear form symmetric in x and y, so each intercept costs one substitution.
  4. Bridges to the first operation: write the tangent in (x cos θ)/a + (y sin θ)/b = 1 form (or ax + by = c form for a circle/parabola tangent), then set y=0.
  5. Tactically chooses which intercept to compute first based on what the next part needs (e.g. for midpoint, both; for area, both; for locus of foot of perpendicular, possibly only one).

Mechanic — Concrete operations

  1. Ensure the tangent is in linear form, e.g. (x cos θ)/a + (y sin θ)/b = 1.
  2. Set y = 0: (x cos θ)/a = 1 ⇒ x = a/cos θ = a sec θ. X-intercept A = (a sec θ, 0).
  3. Set x = 0: (y sin θ)/b = 1 ⇒ y = b/sin θ = b cosec θ. Y-intercept B = (0, b cosec θ).
  4. Feed intercepts into the required formula: area of △OAB = ½|a sec θ||b cosec θ| = ab/|sin 2θ|; midpoint = (½ a sec θ, ½ b cosec θ); etc.
  5. State any restrictions on θ where sec θ or cosec θ blow up (θ ≠ 0, π/2, π, …).

Worked Micro-Example

Ellipse x²/a² + y²/b² = 1 has tangent at (a cos θ, b sin θ): (x cos θ)/a + (y sin θ)/b = 1. Setting y=0 gives x = a sec θ, so A = (a sec θ, 0). Setting x=0 gives y = b cosec θ, so B = (0, b cosec θ). Area of △OAB = ½ · a sec θ · b cosec θ = ab/|sin 2θ|, minimised when θ = π/4.

Variants — Same move, different surface

Tangent to a circle x²+y²=r² at (r cos θ, r sin θ): x cos θ + y sin θ = r.
Set y=0 ⇒ x = r sec θ; set x=0 ⇒ y = r cosec θ. Same reciprocal-trig pattern with a=b=r.
Tangent to a parabola y²=4ax at (at², 2at): ty = x + at².
Set y=0 ⇒ x = −at² (X-intercept); set x=0 ⇒ y = at (Y-intercept). Intercepts are now polynomial in t rather than trig.
Tangent to a rectangular hyperbola xy=c² at (ct, c/t): x + t²y = 2ct.
Set y=0 ⇒ x = 2ct; set x=0 ⇒ y = 2c/t. Midpoint of intercepts is (ct, c/t), recovering the point of tangency.
Normal instead of tangent.
Same substitution strategy; algebra is messier because normals lack the symmetric (x/·)+(y/·)=1 form, so simplify after substituting.
Tangent in general form y = mx + c at a non-parametric point.
Set y=0 ⇒ x = −c/m; set x=0 ⇒ y = c. Watch for m=0 (horizontal) or vertical tangents.

Common Misreadings

  • Reading 'meets the axes' as needing simultaneous equations with y=0 and x=0 lines treated as a separate system.
  • Confusing the tangent's intercepts with the curve's own axis intercepts.
  • Thinking parametric differentiation is needed again at this stage — it was already used to get the tangent.

Pitfalls

  • Forgetting sec θ and cosec θ are undefined at θ = 0, π/2, π, 3π/2; must state θ-restrictions.
  • Dropping the ½ or absolute value in the triangle-area formula.
  • Sign errors when the parameter places the intercept on the negative axis — area still uses |·|.
  • Mis-simplifying ab/(sin θ cos θ) instead of using ab/(½ sin 2θ) = 2ab/sin 2θ; double-check the factor of 2.

A-level Scope Notes

Fully within Edexcel IAL scope: tangents to circles (P1/P2), parabolas, ellipses and rectangular hyperbolas (Further Pure parametric coordinate geometry). Uses only basic trig identities (sec, cosec, sin 2θ = 2 sin θ cos θ) and elementary area/midpoint formulae.
