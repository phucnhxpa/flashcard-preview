Integration

Rewrite 1/uⁿ as u⁻ⁿ to apply the reverse power rule

Family: Integrand Rewrite Frequency: 1 questions

Trigger — When to use

The integrand contains a reciprocal power such as 1/u², 1/u³, or 1/√u, where the variable sits in the denominator (or under a root). The expression is not yet in the standard ∫uⁿ du template, but the exponent is not −1, so a logarithmic rule does not apply. This typically arises after a substitution has reduced the integral to a single power of u, or directly when integrating expressions like 1/x² or 1/(2x+1)².

Motivation — Why this works

The reverse power rule ∫uⁿ du = uⁿ⁺¹/(n+1) holds for every real n ≠ −1, including negative and fractional n. Rewriting 1/uⁿ as u⁻ⁿ exposes this structure and lets the rule be applied mechanically. The only exclusion is n = −1, where the antiderivative is ln|u|; for any other negative exponent the power rule remains valid and produces a clean closed form.

Thought Process — The step-by-step reasoning

  1. Notice the integrand is a reciprocal power: the variable lives in the denominator (or under a root), e.g. 1/u².
  2. Rule out the logarithmic case by checking the exponent: ln|u| only applies when the power of u in the denominator is exactly 1.
  3. Commit to the power rule because n = −2 (or whatever the exponent is) satisfies n ≠ −1, so ∫uⁿ du = uⁿ⁺¹/(n+1) is valid.
  4. Bridge to operation: rewrite 1/uⁿ explicitly as u⁻ⁿ so the template ∫uⁿ du is visually present.
  5. Execute carefully: add 1 to the exponent (watching signs), divide by the new exponent, then convert back to reciprocal form if a tidy presentation is wanted.

Mechanic — Concrete operations

  1. Identify the reciprocal power and rewrite with a negative (or fractional) exponent: 1/u² = u⁻².
  2. Apply the reverse power rule: ∫u⁻² du = u⁻²⁺¹/(−2+1) = u⁻¹/(−1) = −u⁻¹ = −1/u.
  3. If definite, substitute the limits: e.g. [−1/u]₃⁴ = −1/4 − (−1/3) = 1/12.
  4. If indefinite, append +C and optionally rewrite back into reciprocal form for clarity.

Worked Micro-Example

Evaluate $\int_{3}^{4} \frac{1}{u^{2}}\,du$. Rewrite: $\frac{1}{u^{2}} = u^{-2}$. Then $\int u^{-2}\,du = \frac{u^{-1}}{-1} = -\frac{1}{u}$. So $\left[-\tfrac{1}{u}\right]_{3}^{4} = -\tfrac{1}{4} + \tfrac{1}{3} = \tfrac{1}{12}$.

Variants — Same move, different surface

Higher reciprocal power, e.g. ∫1/u³ du
Rewrite as u⁻³; result is u⁻²/(−2) = −1/(2u²).
Fractional reciprocal, e.g. ∫1/√u du
Rewrite as u⁻¹ᐟ², giving u¹ᐟ²/(1/2) = 2√u.
Reciprocal of a linear expression squared, e.g. ∫1/(2x+1)² dx
Rewrite as (2x+1)⁻² and apply reverse chain: −(1/2)(2x+1)⁻¹ + C.
Mixed integrand like ∫(u² + 1/u²) du
Split and rewrite the second term as u⁻² before integrating term by term.

Common Misreadings

  • Treating 1/u² as if it were 1/u and writing ln|u| — the logarithm rule is only for the n = −1 case.
  • Believing the power rule fails for negative exponents and reaching unnecessarily for substitution.
  • Reading 1/u² as (1/u)² and trying to integrate by squaring 1/u somehow, instead of using a single exponent −2.

Pitfalls

  • Sign slip when adding 1 to a negative exponent: writing −2+1 = −3 instead of −1.
  • Dropping the negative sign in −1/u, giving +1/u and the wrong definite value.
  • Dividing by the old exponent instead of the new one (n+1).
  • Forgetting +C for indefinite integrals after the rewrite.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths: the reverse power rule for ∫xⁿ dx with n ≠ −1 is core C1/C2 and C4 content, including negative and fractional exponents.
