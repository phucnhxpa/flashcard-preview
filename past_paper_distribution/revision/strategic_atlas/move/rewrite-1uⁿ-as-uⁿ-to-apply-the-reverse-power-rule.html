Trigger вҖ” When to use

The expression is not yet in the standard вҲ«uвҒҝ du template, but the exponent is not вҲ’1, so a logarithmic rule does not apply. Motivation вҖ” Why this works

The reverse power rule вҲ«uвҒҝ du = uвҒҝвҒәВ№/(n+1) holds for every real n вү  вҲ’1, including negative and fractional n. The only exclusion is n = вҲ’1, where the antiderivative is ln|u|; for any other negative exponent the power rule remains valid and produces a clean closed form.

Thought Process вҖ” The step-by-step reasoning

  1. Notice the integrand is a reciprocal power: the variable lives in the denominator (or under a root), e.g. 1/uВІ.
  2. Rule out the logarithmic case by checking the exponent: ln|u| only applies when the power of u in the denominator is exactly 1.
  3. Commit to the power rule because n = вҲ’2 (or whatever the exponent is) satisfies n вү  вҲ’1, so вҲ«uвҒҝ du = uвҒҝвҒәВ№/(n+1) is valid.
  4. Bridge to operation: rewrite 1/uвҒҝ explicitly as uвҒ»вҒҝ so the template вҲ«uвҒҝ du is visually present.
  5. Execute carefully: add 1 to the exponent (watching signs), divide by the new exponent, then convert back to reciprocal form if a tidy presentation is wanted.

Mechanic вҖ” Concrete operations

  1. Identify the reciprocal power and rewrite with a negative (or fractional) exponent: 1/uВІ = uвҒ»ВІ.
  2. Apply the reverse power rule: вҲ«uвҒ»ВІ du = uвҒ»ВІвҒәВ№/(вҲ’2+1) = uвҒ»В№/(вҲ’1) = вҲ’uвҒ»В№ = вҲ’1/u.
  3. If definite, substitute the limits: e.g. [вҲ’1/u]вӮғвҒҙ = вҲ’1/4 вҲ’ (вҲ’1/3) = 1/12.
  4. If indefinite, append +C and optionally rewrite back into reciprocal form for clarity.

Worked Micro-Example

Evaluate $\int_{3}^{4} \frac{1}{u^{2}}\,du$. Rewrite: $\frac{1}{u^{2}} = u^{-2}$. Then $\int u^{-2}\,du = \frac{u^{-1}}{-1} = -\frac{1}{u}$. So $\left[-\tfrac{1}{u}\right]_{3}^{4} = -\tfrac{1}{4} + \tfrac{1}{3} = \tfrac{1}{12}$.

Common Misreadings

  • Believing the power rule fails for negative exponents and reaching unnecessarily for substitution.
  • Reading 1/uВІ as (1/u)ВІ and trying to integrate by squaring 1/u somehow, instead of using a single exponent вҲ’2.

Pitfalls

  • Sign slip when adding 1 to a negative exponent: writing вҲ’2+1 = вҲ’3 instead of вҲ’1.
  • Dropping the negative sign in вҲ’1/u, giving +1/u and the wrong definite value.
  • Dividing by the old exponent instead of the new one (n+1).
  • Forgetting +C for indefinite integrals after the rewrite.
