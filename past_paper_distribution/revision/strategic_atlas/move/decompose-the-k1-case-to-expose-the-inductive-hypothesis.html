Proof

Decompose the (k+1) case to expose the inductive hypothesis

Family: Induction Decomposition Frequency: 2 questions

Trigger — When to use

Fires in the inductive step of any proof by induction where the n=k+1 expression must be related back to the n=k expression. Visual cues: a summation Σ_{r=1}^{n} whose upper limit advances by 1; a product ∏_{r=1}^{n}; a divisibility claim k | f(n) where f(n) contains an exponential term a^n (or a^{n+c}); a matrix power M^n. In all cases you have an assumed form for n=k and need to manufacture it inside the n=k+1 expression. Alternative phrasing: 'split the (k+1) case into [the k case] + [a manageable extra piece]'.

Motivation — Why this works

Induction can only progress if the inductive hypothesis is literally substituted somewhere in the n=k+1 expression. The decomposition manufactures that opportunity: for sums, peeling off the last term leaves Σ_{r=1}^{k} intact; for divisibility with exponential base a, writing f(k+1) = a·f(k) + remainder eliminates the exponential and leaves a checkable polynomial residue. The underlying principle is 'extract the assumed structure, then handle the leftover'. Without this decomposition the (k+1) expression remains an opaque object disconnected from the hypothesis.

Thought Process — The step-by-step reasoning

  1. Notice that the n=k+1 expression contains the n=k expression as a hidden sub-structure — for sums this is obvious (one extra term); for exponentials a^{k+3} = a · a^{k+2}, so a factor of a connects the two cases.
  2. Rule out alternative decompositions: f(k+1) − f(k) often fails for exponential divisibility because it leaves a residual exponential term; splitting a sum at an intermediate index doesn't expose the hypothesis. These dead ends confirm the unique correct multiplier/split.
  3. Commit to the decomposition that makes the hypothesis literally appear: peel off the last term for Σ/∏, or multiply f(k) by the exponential base a so that a·f(k) shares the dominant term of f(k+1).
  4. Bridge to the first concrete operation: write f(k+1) = a·f(k) + R, then compute R explicitly by expansion. For sums: write Σ_{r=1}^{k+1} = Σ_{r=1}^{k} + (k+1)th term, then substitute the assumed closed form.
  5. Tactical choice: the multiplier must match the exponential base exactly (use 5 for 5^n, not 25; use 9 for 3^{2n} since 3^{2(k+1)} = 9·3^{2k}). Any other choice creates avoidable algebra.
  6. Verify the residue R has the required property (divisibility, target closed form), then conclude both pieces have the property so f(k+1) does too.

Mechanic — Concrete operations

  1. Identify the dominant structure: summation upper limit, product range, or exponential base a in f(n).
  2. For sums/products: write Σ_{r=1}^{k+1} g(r) = Σ_{r=1}^{k} g(r) + g(k+1), then substitute the inductive hypothesis for the first part.
  3. For divisibility f(n) with exponential a^n: form f(k+1) − a·f(k) and simplify to get a clean residue R(k).
  4. Rewrite as f(k+1) = a·f(k) + R(k) with R(k) manifestly divisible by the target d.
  5. State explicitly: a·f(k) is divisible by d by the inductive hypothesis; R(k) is divisible by d by direct calculation; therefore f(k+1) is divisible by d.
  6. Close the induction: 'true for n=1, and true for n=k ⟹ true for n=k+1, hence true for all n ∈ ℤ⁺ by induction'.

Worked Micro-Example

Show $16 \mid f(n)$ where $f(n)=5^{n+2}-4n-9$. Assume $16 \mid f(k)$. Then $f(k+1)=5^{k+3}-4(k+1)-9 = 5\cdot 5^{k+2}-4k-13$. Form $f(k+1)-5f(k) = (5\cdot 5^{k+2}-4k-13) - 5(5^{k+2}-4k-9) = 16k+32 = 16(k+2)$. Hence $f(k+1)=5f(k)+16(k+2)$, divisible by 16.

Variants — Same move, different surface

Summation induction (e.g. Σ log₂ r = log₂(n!))
Peel off the (k+1)th addend: Σ_{r=1}^{k+1} = Σ_{r=1}^{k} + (k+1)th term, then collapse using the relevant law (log law, algebraic identity).
Product induction ∏_{r=1}^{n} g(r)
Peel off the (k+1)th factor instead of an addend: ∏_{r=1}^{k+1} = ∏_{r=1}^{k} · g(k+1).
Matrix power induction M^{n+1}
Write M^{k+1} = M^k · M, substitute the assumed closed form for M^k, then multiply out.
Divisibility with base 3^{2n} or 9^n
Multiplier is 9 (since 3^{2(k+1)} = 9·3^{2k}); same decomposition logic, different multiplier matching the effective base.
Divisibility with mixed exponentials e.g. f(n) = 7^n + 2·3^n
Multiple-multiplier trick: consider 7·f(k) or 3·f(k) and add/subtract a correction; sometimes use 7·f(k) − f(k+1) to isolate one base.

Common Misreadings

  • Computing f(k+1) − f(k) for exponential divisibility — leaves a residual exponential term whose divisibility is not obvious.
  • Trying to factor the entire f(n) expression directly (e.g. pulling 5^{k+2} out) — the additive linear terms block clean factorisation.
  • Using a multiplier that doesn't match the exponential base (e.g. 25 for 5^n) — creates unnecessary algebraic complexity.

Pitfalls

  • Sign errors when expanding −a·f(k), especially on negative linear terms (e.g. −5(−4k−9) = +20k+45).
  • Forgetting to multiply ALL terms of f(k) by the multiplier a — students sometimes only multiply the exponential and drop the linear corrections.
  • Failing to state 'using the inductive hypothesis' explicitly when substituting — Edexcel withholds the A-mark without this phrase.
  • Omitting the closing statement 'true for n=1, true for n=k ⟹ true for n=k+1, so true for all n ∈ ℤ⁺' — required for the final mark.

A-level Scope Notes

Within Edexcel IAL Further Pure scope (proof by induction: summation, divisibility, matrix powers, recurrence). No formula booklet entry — induction structure must be memorised. Avoid mentioning strong induction or well-ordering; standard (weak) induction suffices for all IAL FP questions.
