Differential Equations

Find C first, then merge into single log before exponentiating

Family: Log Consolidation And Exponentiation Frequency: 1 questions

Trigger — When to use

Fires when separation of variables has produced an equation of the form $\ln|f(y)| = a\ln|g_1(x)| + a\ln|g_2(x)| + C$ (all logs sharing a common coefficient) and a boundary condition is supplied so that $C$ evaluates to a logarithmic constant. Typical cue phrases: 'find $y$ in terms of $x$', 'express in the form ...', or a mark scheme answer free of $\ln$. Also fires when the target form is a clean algebraic relation like $\sin^4 y = \tfrac{p(x)}{q(x)}$ with no fractional exponent.

Motivation — Why this works

Because $C$ enters additively, substituting the boundary point first turns it into a specific number; if every other term is a log with the same coefficient, $C$ can be rewritten as a log too and absorbed via $a\ln b - a\ln c = a\ln(b/c)$. Consolidating the RHS into one logarithm lets a single exponentiation strip all logs at once, avoiding a stray multiplicative $e^C$. A final integer-power step (e.g. raising to the 4th) clears the fractional exponent and matches the mark-scheme's explicit form.

Thought Process — The step-by-step reasoning

  1. Notice that every term on the RHS is a logarithm with the same coefficient (here $\tfrac14$), and $C$ is the only non-log piece — a strong hint to convert $C$ into a log too.
  2. Rule out exponentiating immediately: that would leave $e^C$ as an unknown multiplicative constant, harder to pin down cleanly with the boundary data.
  3. Commit to: substitute the initial condition first, solve for $C$ as an exact logarithmic value, then merge.
  4. Bridge: plug $x=1/2$, $y=\pi/2$, evaluate $\ln(\sin(\pi/2)) = 0$, and isolate $C$ as a $\tfrac14\ln(\cdot)$ expression.
  5. Inside the move, factor out the shared coefficient before combining logs, then exponentiate, and finally raise to the integer power that clears any fractional exponent.

Mechanic — Concrete operations

  1. Substitute the boundary values into the all-log general solution.
  2. Use $\ln 1 = 0$ and log subtraction to simplify the RHS to a single $a\ln(\cdot)$ term.
  3. Solve for $C$, obtaining $C = a\ln k$ for some constant $k$.
  4. Substitute back and combine all RHS logs: $\ln|f(y)| = a\ln\!\left[\tfrac{g_1(x)}{k\,g_2(x)}\right]$.
  5. Exponentiate to remove the outer logs: $f(y) = \left[\tfrac{g_1(x)}{k\,g_2(x)}\right]^{a}$.
  6. If $a = 1/n$, raise both sides to the $n$th power to clear the fractional exponent.

Worked Micro-Example

From $\ln|\sin y| = \tfrac14\ln|1+3x| - \tfrac14\ln|1-x| + C$ with $y=\pi/2$ at $x=1/2$: $0 = \tfrac14\ln(5/2) - \tfrac14\ln(1/2) + C = \tfrac14\ln 5 + C$, so $C = -\tfrac14\ln 5$. Then $\ln|\sin y| = \tfrac14\ln\!\tfrac{1+3x}{5(1-x)}$, giving $\sin y = \left[\tfrac{1+3x}{5(1-x)}\right]^{1/4}$ and finally $\sin^4 y = \tfrac{1+3x}{5(1-x)}$.

Variants — Same move, different surface

Logistic / population ODE where solution has form $\ln\!\tfrac{P}{N-P} = kt + C$
Write $C = \ln A$ from the start so exponentiation gives $\tfrac{P}{N-P} = A e^{kt}$, then determine $A$ from the initial condition.
Exponential decay $\ln|y| = -kt + C$
Set $A = e^C$; the boundary condition fixes $A$ directly as $y(0)$.
Logs with coefficient 1 (e.g. $\ln|y| = \ln|x| + \ln|x+1| + C$)
Rewrite $C = \ln k$; combine into $\ln|y| = \ln|kx(x+1)|$, then drop logs to get $y = kx(x+1)$.
Mark scheme demands form $y = \dots$ with no fractional powers
After exponentiating, raise both sides to the reciprocal of the fractional exponent to obtain a polynomial/rational form.

Common Misreadings

  • Treating $C$ as a generic constant and exponentiating immediately, producing an unknown $e^C$ that complicates the boundary substitution.
  • Believing $C$ cannot be absorbed into the logs because it 'isn't a log' — missing that any constant can be written as $\ln k$.
  • Reading the fractional-exponent intermediate form as the final answer when the question asks for an explicit algebraic relation.

Pitfalls

  • Subtracting log arguments instead of dividing: writing $\ln(5/2)-\ln(1/2) = \ln 2$ instead of $\ln 5$.
  • Forgetting $\ln(\sin(\pi/2)) = \ln 1 = 0$, not $1$.
  • Failing to factor out the shared coefficient $\tfrac14$ before combining, leading to algebra slips.
  • Stopping at $\sin y = (\cdot)^{1/4}$ when the mark scheme wants $\sin^4 y = \cdot$.
  • Dropping modulus signs without checking that the relevant quantities are positive on the interval of interest.

A-level Scope Notes

Entirely within Edexcel IAL P3/P4 first-order separable ODEs: log laws, exponentiation, and use of a boundary condition to determine the constant of integration. No methods beyond the standard syllabus are required.
