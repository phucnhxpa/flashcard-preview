Algebra & Functions

Refactor x^(kt) as (x^k)^t to expose b^t structure

Family: Index Law Decomposition Frequency: 1 questions

Trigger — When to use

Fires when an exponential expression has the variable multiplied by a constant inside the exponent (e.g. $10^{0.0646t}$, $e^{kt}$, $2^{mt}$) but the target template requires the variable as the sole outer exponent, as in $N = ab^t$. Cue phrases include 'write in the form $ab^t$', 'find the value of $b$', or matching a model $y = AB^x$. The move also fires whenever a regression output gives $\log N = c + dt$ and the student must convert back to multiplicative form.

Motivation — Why this works

The power-of-a-power law $(x^A)^B = x^{AB}$ runs in reverse: $x^{AB} = (x^A)^B$. Reading the exponent $0.0646t$ as the product $0.0646 \times t$ lets us peel $t$ outwards, leaving a fixed numerical base $x^A$ and the variable as the lone outer exponent. This is exactly the structural rearrangement needed to match $b^t$, so $b$ becomes a computable constant rather than a function of $t$.

Thought Process — The step-by-step reasoning

  1. Notice that $t$ appears multiplied by a constant inside the exponent, but the required form needs $t$ alone as the outer exponent.
  2. Rule out distributing or multiplying out the exponent (e.g. treating $10^{0.0646t}$ as $10 \times 0.0646t$) — that would break the index law.
  3. Commit to the reverse power-of-a-power identity $x^{AB} = (x^A)^B$ because it cleanly separates the constant factor from the variable.
  4. Bridge: rewrite $10^{0.0646t} = (10^{0.0646})^t$, so the new base is the numerical value $10^{0.0646}$.
  5. Tactical: evaluate the base to enough precision (usually 3 s.f. or more) and keep the unrounded value in memory if a later part needs it.

Mechanic — Concrete operations

  1. Identify the inner-product exponent: $10^{0.0646t}$ has exponent $0.0646 \times t$.
  2. Apply $x^{AB} = (x^A)^B$ in reverse: $10^{0.0646t} = (10^{0.0646})^t$.
  3. Set $b = 10^{0.0646}$ and evaluate: $b \approx 1.16$ (3 s.f.).
  4. If a constant multiplier is also needed, compute $a$ from the remaining piece (e.g. $a = 10^{1.478} \approx 30$).
  5. Assemble final form: $N = ab^t$, e.g. $N = 30 \times 1.16^t$.

Worked Micro-Example

Given $\log_{10} N = 1.478 + 0.0646t$, exponentiate: $N = 10^{1.478 + 0.0646t} = 10^{1.478} \cdot 10^{0.0646t}$. Apply the reverse power law: $10^{0.0646t} = (10^{0.0646})^t$. Therefore $N = 10^{1.478} \cdot (10^{0.0646})^t \approx 30 \times 1.16^t$.

Variants — Same move, different surface

Natural-base model $e^{kt}$ to be written as $b^t$
Use $e^{kt} = (e^k)^t$ so $b = e^k$.
Base-2 or base-10 growth like $2^{mt}$ in algorithm/half-life contexts
Identical rewrite: $2^{mt} = (2^m)^t$, giving $b = 2^m$.
Reverse direction: model already in $b^t$ form, need to linearise via logs
Apply $\log(b^t) = t \log b$ to convert outer exponent into a coefficient suitable for linear regression.
Negative-rate (decay) form $10^{-kt}$
Rewrite as $(10^{-k})^t$, giving $0 < b < 1$.

Common Misreadings

  • Treating $10^{0.0646t}$ as $10 \cdot 0.0646 \cdot t = 0.646t$ — confusing exponentiation with multiplication.
  • Thinking $b$ must depend on $t$ because the exponent contains $t$.
  • Reading the regression intercept $1.478$ as $a$ directly without exponentiating: writing $a = 1.478$ instead of $a = 10^{1.478}$.

Pitfalls

  • Computing $10^{0.0646}$ incorrectly by entering $10 \times 0.0646$ on the calculator.
  • Rounding $b$ to too few significant figures early, then propagating error when the model is reused in a later part.
  • Forgetting to also exponentiate the constant term when converting from a $\log_{10} N = c + dt$ form, leaving an additive constant instead of a multiplicative $a$.

A-level Scope Notes

Sits inside Edexcel IAL P3/P4 exponentials and logarithms and the S-paper exponential regression model $y = ab^x$; uses only the standard index laws and $\log/\exp$ inverse relationship.
