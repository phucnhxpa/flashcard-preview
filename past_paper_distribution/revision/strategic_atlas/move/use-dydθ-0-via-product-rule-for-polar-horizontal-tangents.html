Polar Coordinates

Use dy/dθ = 0 via product rule for polar horizontal tangents

Family: Tangent Criterion Frequency: 1 questions

Trigger — When to use

Fires when a polar curve r = f(θ) is given and the question asks for tangents parallel to the initial line, perpendicular to the initial line, or for the points where the curve is 'furthest from' / 'highest above' the initial line. Phrasings include 'tangent parallel to the initial line', 'where y is greatest', or 'maximum perpendicular distance from the initial line'. The same machinery is requested when finding tangents at specific θ-values where a Cartesian gradient is needed.

Motivation — Why this works

A tangent parallel to the initial line is a horizontal Cartesian tangent, so dy/dx = 0. Treating θ as a parameter gives dy/dx = (dy/dθ)/(dx/dθ), so dy/dθ = 0 (with dx/dθ ≠ 0) is the working condition. Since y = r sinθ = f(θ) sinθ, the product rule yields dy/dθ = (dr/dθ) sinθ + r cosθ, avoiding any conversion of the whole curve to Cartesian form. The same derivation gives dx/dθ = (dr/dθ) cosθ − r sinθ for vertical tangents.

Thought Process — The step-by-step reasoning

  1. Notices the phrase 'parallel to the initial line' and translates it geometrically to 'horizontal tangent', i.e. dy/dx = 0.
  2. Rules out converting r = f(θ) to a Cartesian equation — usually messy, especially for r involving 1+cosθ, sin2θ, etc.
  3. Commits to the parametric viewpoint with θ as parameter: dy/dx = (dy/dθ)/(dx/dθ), so the condition reduces to dy/dθ = 0.
  4. Bridges to the first operation by writing y = f(θ)sinθ and preparing to product-rule differentiate.
  5. Inside the move, decides whether to factor the resulting trig expression, watches for solutions in the stated θ-range, and earmarks any θ where dx/dθ also vanishes for separate scrutiny.

Mechanic — Concrete operations

  1. Write y = r sinθ with r = f(θ); compute dr/dθ.
  2. Differentiate: dy/dθ = (dr/dθ) sinθ + r cosθ.
  3. Set dy/dθ = 0 and simplify to a single trig equation in θ.
  4. Solve for θ in the given range, discarding any θ where dx/dθ = (dr/dθ) cosθ − r sinθ is also zero.
  5. Substitute each valid θ into r = f(θ) to get r, then compute (x, y) = (r cosθ, r sinθ) if Cartesian coordinates are required.
  6. For perpendicular-to-initial-line tangents, repeat with dx/dθ = 0 instead.

Worked Micro-Example

For r = 1 + cosθ, dr/dθ = −sinθ. Then dy/dθ = −sinθ·sinθ + (1+cosθ)cosθ = cosθ + cos²θ − sin²θ = cosθ + 2cos²θ − 1 = (2cosθ − 1)(cosθ + 1). Setting dy/dθ = 0 gives cosθ = 1/2 (so θ = ±π/3) or cosθ = −1 (θ = π, where r = 0 — pole, excluded). At θ = π/3: r = 3/2, giving the point (3/4, 3√3/4).

Variants — Same move, different surface

Tangent perpendicular to the initial line (vertical tangent)
Use dx/dθ = (dr/dθ)cosθ − r sinθ = 0 instead, with dy/dθ ≠ 0.
Curve given in Cartesian parametric form (x(t), y(t))
Apply dy/dx = (dy/dt)/(dx/dt) directly; no product-rule decomposition is needed because x and y are already separate.
Tangent at the pole r = 0
The formula degenerates; the tangent line at the pole is θ = θ₀ where f(θ₀) = 0, so read the direction off the θ-value rather than from dy/dθ.
Maximum perpendicular distance from the initial line
This is exactly maximising y = r sinθ, so the same equation dy/dθ = 0 applies; verify it is a maximum rather than a minimum.

Common Misreadings

  • Reading 'parallel to the initial line' as 'tangent at θ = 0' — the initial line is a direction, not a point of tangency.
  • Setting dr/dθ = 0 instead of dy/dθ = 0 — that locates extreme values of r, not horizontal tangents.
  • Confusing parallel with perpendicular and using dx/dθ = 0 by mistake.

Pitfalls

  • Forgetting to check dx/dθ ≠ 0 at the solution; simultaneous vanishing typically signals the pole or a cusp.
  • Losing solutions when factoring the trig equation, e.g. dividing by sinθ or cosθ and dropping a root.
  • Ignoring the prescribed θ-range and reporting extra angles, or omitting symmetric solutions like θ = ±π/3.
  • Reporting θ alone when the question asks for the Cartesian coordinates of the point of tangency.

A-level Scope Notes

Fully within Edexcel IAL Further Pure polar coordinates: tangents parallel/perpendicular to the initial line are an explicit syllabus item, using x = r cosθ, y = r sinθ and parametric differentiation.
