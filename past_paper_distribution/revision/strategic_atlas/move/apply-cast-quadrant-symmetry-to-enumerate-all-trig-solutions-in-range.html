Trigger — When to use

Fires whenever a basic trig equation sin θ = k, cos θ = k, or tan θ = k must be solved over a specified interval (e.g. [0°, 360°), [−180°, 180°], or a wider/shifted range), and only the principal value from arcsin/arccos/arctan has been computed. The cue is the phrase 'find all solutions in...' or 'solve for θ in the interval...'. Also fires after a substitution (e.g. X = 3θ − 15°) has produced an extended or shifted range that must be swept for all solutions before back-substitution. Equally applies when an equation reduces to sin²θ = c or |sin θ| = c, producing two sign cases each requiring full enumeration.

Motivation — Why this works

The inverse trig functions return only one principal value (arcsin ∈ [−90°, 90°], arccos ∈ [0°, 180°], arctan ∈ (−90°, 90°)), but each periodic equation has multiple solutions per cycle. The CAST diagram encodes the sign of each trig ratio in the four quadrants, so once the reference (acute) angle α is known, every solution can be written as α, 180°−α, 180°+α, or 360°−α depending on which function and sign apply. This systematic sweep — combined with adding/subtracting 360° (or 180° for tan) for extended ranges — guarantees no solution is missed and none is invented.

Thought Process — Step-by-step reasoning

  1. Notice: the equation has been reduced to sin/cos/tan of a single angle equals a constant, and the question demands ALL solutions in a stated interval — not just one.
  2. Rule out: simply quoting arcsin/arccos/arctan (gives one value); rule out graphical sketching alone (imprecise); rule out general solution formulas with +360°n unless the range is wide.
  3. Commit: use the reference angle α = arcsin/arccos/arctan(|k|) together with the CAST diagram to identify which two quadrants per cycle host solutions, based on the SIGN of k and which trig function is involved.
  4. Bridge: write down the quadrant template for the function — sin: (Q1, Q2) if k>0, (Q3, Q4) if k<0; cos: (Q1, Q4) if k>0, (Q2, Q3) if k<0; tan: (Q1, Q3) if k>0, (Q2, Q4) if k<0 — then plug α into each.
  5. Tactical: if the interval exceeds 360° (or is shifted, e.g. after substitution), generate ±360°n copies of each base solution and keep only those inside the interval. Decide inclusion/exclusion at endpoints carefully.
  6. Verification: count solutions against expectation (linear sin/cos equation: 2 per 360°; tan: 1 per 180°; squared equation: up to 4 per 360°) before reporting.

Mechanic — Concrete operations

  1. Compute the reference angle α = arcsin(|k|), arccos(|k|), or arctan(|k|), keeping full calculator precision.
  2. Identify the two quadrants where the trig ratio has the required sign using CAST.
  3. Write the base solutions: e.g. for sin θ = k>0: θ = α and θ = 180°−α; for sin θ = k<0: θ = 180°+α and θ = 360°−α; analogous templates for cos and tan.
  4. If the interval is wider than one period or shifted, add/subtract 360° (180° for tan) to each base value and keep those lying in the interval.
  5. If the equation produced ± cases (e.g. sin θ = ±√6/4), repeat the enumeration for each sign.
  6. Round only at the final step to the precision required (typically 1 d.p. for degrees).

Worked Micro-Example

Solve $\sin\theta = -\tfrac{\sqrt{6}}{4}$ for $\theta \in [0°, 360°)$.\nReference angle: $\alpha = \arcsin(\sqrt{6}/4) \approx 37.76°$.\nSign is negative ⇒ solutions lie in Q3 and Q4: $\theta = 180°+\alpha \approx 217.8°$ and $\theta = 360°-\alpha \approx 322.2°$.\nBoth lie in $[0°, 360°)$, so the solution set is $\{217.8°,\ 322.2°\}$.

Variants — Same move, different surface

cos θ = k with k > 0
Solutions in Q1 and Q4: θ = α and θ = 360° − α.
cos θ = k with k < 0
Solutions in Q2 and Q3: θ = 180° − α and θ = 180° + α (where α = arccos(|k|)).
tan θ = k (any sign)
Solutions are 180° apart: θ = α + 180°n. Use only one family but two copies per 360°.
sin²θ = c or equation giving sin θ = ±k
Run the full Q1/Q2 enumeration for +k and the full Q3/Q4 enumeration for −k; expect up to 4 solutions per 360°.
Range is wider than 360° (e.g. after substitution X = 3θ − 15°)
Generate base solutions, then add ±360° copies until you exit the interval; back-substitute at the end.
Range is shifted (e.g. −180° < θ ≤ 180°)
Same templates, but a Q3/Q4 solution may be reported as a negative angle (subtract 360°).

Common Misreadings

  • Reporting only the principal value from arcsin/arccos/arctan and stopping there.
  • Using 180° + α instead of 180° − α for the Q2 sine companion (or mixing up cos/sin templates).
  • Only running one sign case after sin²θ = c, missing half the solutions.
  • Forgetting that an extended range after substitution requires adding ±360° copies before back-substituting.

Pitfalls

  • Rounding the reference angle α too early, so derived solutions like 180°−α and 360°−α inherit accumulated error.
  • Misreading the interval endpoints — including 360° when the bracket is half-open [0°, 360°), or excluding a boundary value that is in fact attained.
  • Forgetting to back-substitute (e.g. solving for X = 2θ but reporting X-values as θ).
  • Mixing up which quadrants correspond to which trig sign — always anchor with CAST.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P2/P3) — basic trig equations and the CAST/quadrant rule. No formula booklet entry is needed beyond standard exact values; arcsin/arccos/arctan return principal values only. Avoid invoking complex-valued solutions or general-solution formulas with arbitrary integer n unless the question explicitly asks for them.
