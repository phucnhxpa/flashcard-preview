Algebra & Functions

Introduce proxy variables to reduce log-log plot to linear coordinate geometry

Family: Substitution Bridge Frequency: 1 questions

Trigger — When to use

Fires when a question shows a straight-line graph whose axes are themselves logarithmic expressions (e.g. log₁₀y vs log₁₀x, ln y vs x, log y vs x²) and asks for the equation relating those logarithmic quantities. Cues include explicit axis labels of the form 'log₁₀y' or 'ln y', given coordinates such as intercepts on those axes, and the phrase 'the points (a,b) and (c,d) lie on the line'. It also fires when later parts ask to express y in terms of x via a power law or exponential model.

Motivation — Why this works

Because the plotted quantities themselves are linearly related, the underlying geometry is just y = mx + c in the transformed plane. Substituting Y = log₁₀y and X = log₁₀x converts the problem into a standard gradient-and-intercept exercise, postponing any logarithm manipulation. This separation prevents algebraic errors from conflating the line-finding step with the later step of recovering y in terms of x. It exploits the fact that linearisation by logs is reversible: any line in (X,Y) corresponds to a power law or exponential in (x,y).

Thought Process — The step-by-step reasoning

  1. I notice the axes are not x and y but log₁₀x and log₁₀y, so the linear relationship is between these logged quantities, not between x and y themselves.
  2. I rule out treating the graph as y = mx + c in (x,y); that would misread the axes and give a wrong model.
  3. I commit to a substitution because the line is straight in the logged coordinates, so coordinate geometry applies directly there.
  4. I let Y = log₁₀y and X = log₁₀x, then re-label the given points in (X,Y) form, e.g. (0,4) and (6,0).
  5. I compute gradient and intercept in (X,Y), write Y = mX + c, then substitute back log₁₀y for Y and log₁₀x for X to deliver the requested equation.

Mechanic — Concrete operations

  1. Identify the axis labels and define proxy variables matching them, e.g. Y = log₁₀y, X = log₁₀x.
  2. Rewrite each given point as (X,Y) coordinates.
  3. Compute gradient m = (Y₂ − Y₁)/(X₂ − X₁) and intercept c using one point.
  4. Write the line as Y = mX + c.
  5. Substitute back: replace Y with log₁₀y and X with log₁₀x to obtain the equation in logs.
  6. If a later part asks for y in terms of x, apply 10^(·) to both sides and simplify using log laws.

Worked Micro-Example

A line on axes log₁₀y vs log₁₀x passes through (0,4) and (6,0). Let Y = log₁₀y, X = log₁₀x. Gradient m = (0−4)/(6−0) = −2/3, intercept c = 4, so Y = −\tfrac{2}{3}X + 4. Substituting back: \log_{10}y = -\tfrac{2}{3}\log_{10}x + 4. Then y = 10^{4} x^{-2/3}.

Variants — Same move, different surface

Semi-log plot: log₁₀y vs x is a straight line
Let Y = log₁₀y, X = x; recovering y gives an exponential model y = A·b^x rather than a power law.
ln y vs x² is a straight line
Let Y = ln y, X = x²; back-substitution and exponentiation give y = A·e^{mx²}.
log₁₀y vs log₁₀x with gradient/intercept given instead of two points
Skip the gradient calculation; write Y = mX + c directly and substitute back.
Axes are lg y vs lg x (lg meaning log₁₀)
Same substitution; just match notation used in the question.

Common Misreadings

  • Reading the axes as y and x directly, producing y = mx + c instead of log₁₀y = m log₁₀x + c.
  • Mistaking a semi-log plot (log y vs x) for a log-log plot and assuming a power law instead of an exponential.
  • Swapping which axis is X and which is Y, inverting the gradient.

Pitfalls

  • Leaving the final answer in terms of the proxy variables Y and X rather than log₁₀y and log₁₀x.
  • When recovering y, mishandling 10^{c} as the constant — e.g. writing y = x^{m} + 10^{c} instead of y = 10^{c}·x^{m}.
  • Sign errors in the gradient when one of the points is on an axis.
  • Confusing log₁₀ and ln when the axis label uses ln; the substitution must mirror the exact log used.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P2/P3 logarithms and exponentials), where linearising power-law and exponential models via logs and reading gradient/intercept from a straight-line graph is a standard examined skill.
