Trigonometry

Divide interval length by period to count complete cycles

Family: Cycle Counting Frequency: 1 questions

Trigger — When to use

Fires when a question asks how many complete cycles, oscillations, or full repetitions a periodic function performs over a stated interval [a, b]. Typical phrasings include 'how many complete cycles', 'number of full periods', or 'how many times does the graph repeat'. Usually appears after an earlier part has established (or asks the student to state) the period of the function. The interval is often chosen so that its length is a clean integer multiple of the period.

Motivation — Why this works

A function with period P satisfies f(x+P) = f(x), so its graph on any window of length P is a translated copy of any other. The count of complete cycles in an interval of length L is therefore L/P, which collapses the geometric question into a single division. This is exact whenever L is a multiple of P, and gives ⌊L/P⌋ otherwise. Recognising this avoids any sketching or solving of equations.

Thought Process — The step-by-step reasoning

  1. Notices the phrase 'complete cycles' and reads this as a counting/ratio question, not an equation-solving or calculus question.
  2. Rules out sketching the graph or solving f(x) = 0, since these answer different questions (zeros, turning points) rather than full repetitions.
  3. Commits to the formula (number of cycles) = (interval length)/(period) once the period from an earlier part is available.
  4. Bridges to computation by writing the interval length b − a explicitly and the period P explicitly before dividing.
  5. Tactically checks whether the division is exact; if not, takes the floor for 'complete' cycles and notes any leftover fractional cycle if asked.

Mechanic — Concrete operations

  1. Identify the interval [a, b] and compute its length L = b − a.
  2. State the period P of the function (from an earlier part or by inspection, e.g. P = 2π/k for sin(kx)).
  3. Compute L/P; cancel any common factor of π so the result is a clean number.
  4. If L/P is an integer, that is the number of complete cycles; otherwise take ⌊L/P⌋.

Worked Micro-Example

Let $f(x) = \sin(2x)$, with period $P = \pi$. The number of complete cycles on $[0, 201\pi]$ is $\dfrac{201\pi - 0}{\pi} = 201$. So $f$ completes exactly $201$ full cycles on this interval.

Variants — Same move, different surface

Function of the form $\cos(kx)$ or $\sin(kx)$ where the period must be derived first
Compute $P = 2\pi/k$, then divide the interval length by this $P$.
Function $\tan(kx)$
Use period $P = \pi/k$ instead of $2\pi/k$ before dividing.
Interval not starting at 0, e.g. $[\pi, 201\pi]$
Use length $b - a = 200\pi$, not the right endpoint, then divide by $P$.
Interval length not a multiple of $P$
Number of complete cycles is $\lfloor L/P \rfloor$; the remainder $L - P\lfloor L/P\rfloor$ is a partial cycle.
Modulus-transformed function such as $|\sin x|$
The period halves to $\pi$, so divide by the new period, not the original $2\pi$.

Common Misreadings

  • Dividing $201\pi$ by $2\pi$ by reflex, instead of by the actual period $\pi$ given in the earlier part.
  • Treating 'number of complete cycles' as 'number of zeros' or 'number of maxima', which gives a different count.
  • Using the right endpoint $b$ as the interval length when $a \neq 0$.

Pitfalls

  • Forgetting the period found in part (a) and defaulting to $2\pi$ for any sinusoid.
  • Cancelling $\pi$ incorrectly, e.g. writing $201\pi/\pi = 201\pi$.
  • Counting a partial cycle at the end as a complete cycle when $L/P$ is not an integer.

A-level Scope Notes

Fully within Edexcel IAL Pure scope: relies only on the period of standard trig functions and basic arithmetic; no calculus or advanced analysis required.
