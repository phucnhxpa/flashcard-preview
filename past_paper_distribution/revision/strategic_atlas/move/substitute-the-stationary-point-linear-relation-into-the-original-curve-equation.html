Algebra & Functions

Substitute the stationary-point linear relation into the original curve equation to find exact coordinates

Family: Simultaneous Substitution Frequency: 3 questions

Trigger — When to use

Implicit differentiation has just produced a linear relation between x and y (e.g. x = −2y, y = −(2/3)x, or y = −x) by setting dy/dx = 0 (or numerator of dy/dx = 0). This single relation has two unknowns, so it cannot pin down a point on its own. The original implicit curve equation (typically a quadratic in x and y, plus a constant) is still available as an independent constraint. Whenever a question asks for the actual coordinates of stationary points on an implicit curve — or, more generally, where a line meets a conic — this 'one linear + one quadratic' system is the visual cue.

Motivation — Why this works

A stationary point must (i) lie on the curve and (ii) satisfy dy/dx = 0. Condition (ii) alone gives a line through possible candidates; only condition (i) selects the actual points. Substituting the linear relation into the quadratic curve equation eliminates one variable, collapsing the two-equation system into a single-variable quadratic that is guaranteed solvable by standard methods. This is the same line-meets-conic technique used throughout coordinate geometry.

Thought Process — The step-by-step reasoning

  1. I notice I have two pieces of information about the stationary point: a linear relation between x and y (from dy/dx = 0) and the original curve equation. That's exactly two equations in two unknowns — a solvable system.
  2. I rule out substituting back into the differentiated equation: that's circular, since the linear relation came from it, and would yield 0 = 0.
  3. I rule out solving by elimination via addition/subtraction: with one equation linear and one quadratic, direct substitution is cleaner and standard.
  4. I commit to substituting the linear relation into the original curve equation, because the curve equation is the independent constraint that hasn't been used yet.
  5. Tactically, I substitute whichever variable gives the simplest replacement. If x = −2y, I replace x; if y = −(2/3)x, I replace y. The goal is to leave the equation in one variable only.
  6. I anticipate a pure quadratic in that variable (often with no linear term, by symmetry of the conic), so x² = constant gives ± two solutions — both valid stationary points.
  7. Finally I back-substitute each root into the linear relation to recover the partner coordinate, and present each stationary point as an ordered pair.

Mechanic — Concrete operations

  1. From dy/dx = 0, obtain a linear relation, e.g. y = mx (or x = ky).
  2. In the original curve equation, replace every occurrence of the chosen variable by the expression from the linear relation.
  3. Expand carefully: watch signs, especially (−kx)² = +k²x² (not −k²x²).
  4. Collect like terms to obtain a single-variable equation, typically of the form Ax² + C = 0.
  5. Solve for x² (or y²), then take ± square root; rationalise denominators if a surd appears (e.g. 5/√7 = 5√7/7).
  6. Back-substitute each root into the linear relation to find the partner coordinate.
  7. State both stationary points as coordinate pairs; if relevant, check any denominator from dy/dx is non-zero at each point.

Worked Micro-Example

Curve: $x^2 + 4xy - 3y^2 + 25 = 0$ with stationary condition $x = -2y$. Substitute: $(-2y)^2 + 4(-2y)(y) - 3y^2 + 25 = 4y^2 - 8y^2 - 3y^2 + 25 = -7y^2 + 25 = 0$. So $y^2 = 25/7$, giving $y = \pm \tfrac{5\sqrt{7}}{7}$ and $x = -2y = \mp \tfrac{10\sqrt{7}}{7}$. Stationary points: $\left(-\tfrac{10\sqrt{7}}{7}, \tfrac{5\sqrt{7}}{7}\right)$ and $\left(\tfrac{10\sqrt{7}}{7}, -\tfrac{5\sqrt{7}}{7}\right)$.

Variants — Same move, different surface

Tangent or normal line meets the curve again
Use the tangent/normal equation as the linear constraint and substitute into the curve equation — same line-meets-conic substitution, different source of the linear relation.
Linear relation is given as y = mx + c (non-zero intercept)
Substitution still works; expect a full quadratic Ax² + Bx + C = 0 (linear term no longer vanishes), solve via factorising or quadratic formula.
Parametric curve with stationary points from dy/dt = 0
Solve for t-values directly, then substitute into x(t) and y(t) — no simultaneous substitution needed.
Vertical tangent (denominator of dy/dx = 0)
The constraint may be linear or quadratic; if quadratic, the resulting eliminated equation may need the quadratic formula or factoring of higher degree.

Common Misreadings

  • Substituting the linear relation back into the differentiated equation rather than the original curve — produces a tautology 0 = 0.
  • Treating the linear relation alone as sufficient and reading off a 'point' like (−2, 1) without enforcing that it lies on the curve.
  • Confusing the move with elimination by addition — unnecessary here since one equation is already linear.

Pitfalls

  • Sign error when squaring: (−2y)² = +4y², not −4y²; (−(2/3)x)² = +4x²/9.
  • Dropping the ± when taking the square root, losing one of the two stationary points.
  • Failing to rationalise surd denominators (Edexcel mark schemes typically expect 5√7/7, not 5/√7).
  • Arithmetic slips when combining fractional coefficients, e.g. −x² − 16x²/9 = −25x²/9.
  • Forgetting to back-substitute to recover the partner coordinate, leaving the answer as just x-values.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P3/P4) implicit differentiation and coordinate geometry; no formula booklet entry is required beyond standard quadratic solving. Avoid generalising to Lagrange multipliers or systems of more than two unknowns — those are out of scope.
