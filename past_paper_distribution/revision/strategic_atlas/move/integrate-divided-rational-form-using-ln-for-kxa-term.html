Integration

Integrate divided rational form using ln for k/(x−a) term

Family: Partial Fractions Then Integrate Frequency: 1 questions

Trigger — When to use

Fires when an improper rational function has already been reduced (by long division or partial fractions) to a polynomial plus terms of the form k/(x−a), and a definite or indefinite integral is required. Cues include phrases like 'hence find ∫…' following an earlier 'express in the form…' part, or an answer template 'a + b ln c'. Also fires whenever a rational integrand contains an isolated k/(linear) term whose numerator is a constant.

Motivation — Why this works

Each term k/(x−a) is a constant multiple of f'(x)/f(x) with f(x)=x−a, so it integrates to k ln|x−a|. Once the rational expression is in divided form, every piece is elementary: power rule for the polynomial part and logarithm for the linear-denominator remainder. The earlier algebraic step (division or partial fractions) was engineered precisely to expose this structure, so the integration itself becomes a routine term-by-term application.

Thought Process — The step-by-step reasoning

  1. Notice the integrand is already in 'polynomial + k/(x−a)' form, so no further algebra is needed.
  2. Rule out substitution or by-parts: the k/(x−a) term is the standard log integral, not an unfamiliar shape.
  3. Commit to integrating term-by-term, treating k/(x−a) as k·f'/f with f=x−a.
  4. Check continuity on the interval: the singularity x=a must lie outside the limits, otherwise the integral is improper.
  5. Plug in bounds carefully, keeping absolute values inside ln, and use ln 1 = 0 plus log laws to match the required answer form a + b ln c.

Mechanic — Concrete operations

  1. Write the integral with the divided form, e.g. ∫(px + q + k/(x−a)) dx.
  2. Integrate term-by-term: (p/2)x² + qx + k ln|x−a| (+C if indefinite).
  3. For a definite integral, substitute the upper and lower limits, retaining absolute values inside ln.
  4. Simplify using ln 1 = 0 and ln(A) − ln(B) = ln(A/B) where helpful.
  5. Present the result in the requested form (e.g. a + b ln c with integer a, b, c).

Worked Micro-Example

Evaluate $\int_0^2 \left(4x + 2 + \dfrac{5}{x-3}\right)dx$. Antiderivative: $2x^2 + 2x + 5\ln|x-3|$. At $x=2$: $8+4+5\ln 1 = 12$. At $x=0$: $0+0+5\ln 3$. Result: $12 - 5\ln 3$.

Variants — Same move, different surface

Linear denominator with non-unit coefficient, e.g. k/(2x−3)
Antiderivative is (k/2) ln|2x−3|; divide by the coefficient of x from the chain rule.
Partial fractions giving A/(x−a) + B/(x−b)
Each term integrates to its own logarithm; combine via ln laws if a single ln is required.
Repeated linear factor giving B/(x−a)²
This term integrates to −B/(x−a), not a logarithm; only simple (non-repeated) factors give ln.
Improper rational where division has not yet been done
First perform polynomial long division (or rewrite numerator) to reach the divided form, then apply this move.

Common Misreadings

  • Treating 5/(x−3) as needing a substitution u=x−3 rather than recognising the standard log form directly.
  • Forgetting absolute value bars and writing ln(x−3) when x−3 is negative on the interval.
  • Mistaking the divided form for one still needing partial fractions.

Pitfalls

  • Evaluating ln|2−3| as ln(−1) rather than ln 1 = 0 by dropping the absolute value.
  • Subtracting bounds the wrong way (lower minus upper).
  • Failing to check that the singularity x=a lies outside the interval, leading to an improper integral being treated as proper.
  • Forgetting the chain-rule factor 1/coefficient when the denominator is (bx−a) rather than (x−a).

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P3/P4): integration of 1/(ax+b) as a logarithm and integration following partial fractions or polynomial division are explicit syllabus skills.
