Trigonometry

Map x-bounds to inner-variable bounds before solving compound trig equation

Family: Domain Mapping Frequency: 1 questions

Trigger — When to use

Fires when a trig equation involves an inner linear function such as $\tan(2x)$, $\sin(3x)$, $\cos(x/2)$, or $\sin(x+\pi/6)$ on a restricted $x$-interval. The cue is a substitution $\theta = ax+b$ that simplifies the equation but changes how many periods of the trig function are swept. Whenever the question specifies a domain for $x$ and you intend to solve in terms of $\theta$, you must transform that domain. Equivalent phrasings: 'find all solutions in the given range', 'state all values of $x$ in $[a,b]$'.

Motivation — Why this works

The count and location of solutions to $\tan\theta = k$, $\sin\theta = k$, or $\cos\theta = k$ depend entirely on the $\theta$-interval, because trig functions are periodic. A linear substitution $\theta = ax+b$ stretches/shifts the interval by factor $a$ and offset $b$, so an $x$-window of length $L$ becomes a $\theta$-window of length $|a|L$, which may contain more or fewer period-branches. Solving in $\theta$ over the correctly mapped interval ensures every solution is captured exactly once before back-substituting $x = (\theta-b)/a$.

Thought Process — The step-by-step reasoning

  1. I notice the equation involves $\tan(2x)$, not just $\tan x$, so a substitution $\theta = 2x$ will simplify it.
  2. I rule out solving directly in $x$ via inverse-tan alone, because the doubled argument means the period in $x$ is $\pi/2$, not $\pi$, and I could miss or invent solutions if I forget this.
  3. I commit to the substitution because the $x$-domain $(-\pi/4,\pi/4)$ maps cleanly to $(-\pi/2,\pi/2)$, which is exactly the principal branch of arctan — one solution guaranteed.
  4. First concrete step: rewrite the bounds, then solve $\tan\theta = -3/2$ on the new interval using $\arctan$.
  5. Tactically, I check the candidate $\theta$ lies inside the mapped interval before dividing back to $x$, and I verify the final $x$ lies in the original interval.

Mechanic — Concrete operations

  1. Identify the inner function and set $\theta = 2x$ (or $\theta = ax+b$ generally).
  2. Transform the $x$-bounds: $-\pi/4 < x < \pi/4 \Rightarrow -\pi/2 < \theta < \pi/2$.
  3. Solve $\tan\theta = -3/2$ on the $\theta$-interval: $\theta = \arctan(-3/2) \approx -0.9828$.
  4. Check $\theta$ lies in the mapped interval ($-\pi/2 < -0.9828 < \pi/2$ ✓).
  5. Back-substitute $x = \theta/2 \approx -0.491$ and confirm $x$ lies in the original interval.

Worked Micro-Example

Solve $\tan(2x) = -\tfrac{3}{2}$ for $-\tfrac{\pi}{4} < x < \tfrac{\pi}{4}$.\\ Let $\theta = 2x$, so $-\tfrac{\pi}{2} < \theta < \tfrac{\pi}{2}$.\\ Then $\theta = \arctan(-\tfrac{3}{2}) \approx -0.9828$, which lies in the interval.\\ Hence $x = \theta/2 \approx -0.491$ rad.

Variants — Same move, different surface

$\sin(3x)=k$ on $0 \le x \le 2\pi$
Map to $0 \le \theta \le 6\pi$; expect up to 6 solutions (use $\theta = \arcsin k$ and $\pi - \arcsin k$ plus $2\pi$-shifts).
$\cos(x/2)=k$ on $0 \le x \le 2\pi$
Map to $0 \le \theta \le \pi$; only the principal arccos branch contributes.
$\sin(x+\pi/6)=k$ on $0 \le x \le 2\pi$
Shift bounds to $\pi/6 \le \theta \le 13\pi/6$; solve in $\theta$ then subtract $\pi/6$.
Harmonic form $R\sin(x+\alpha)=c$ on a given $x$-range
After reducing to $\sin\theta = c/R$, transform the $x$-interval by the phase shift $\alpha$ before listing $\theta$-solutions.
Quadratic-in-trig giving $\tan(2x)=k_1, k_2$
Apply the same domain map once; solve each value of $k$ on the mapped $\theta$-interval, collecting all roots before back-substituting.

Common Misreadings

  • Reading the substitution as cosmetic and solving on the original $x$-interval anyway.
  • Treating $\theta = 2x$ as halving (not doubling) the interval.
  • Assuming $\arctan$ alone gives all solutions without checking the mapped interval length.
  • Confusing radian and degree bounds when the calculator is in the wrong mode.

Pitfalls

  • Forgetting to scale bounds by $a$ when $\theta = ax$, leading to an incorrect solution count.
  • Dropping a $\pi$-shifted second solution when the mapped $\theta$-interval spans more than one period of $\tan$ or $\sin$/$\cos$.
  • Reporting $\theta$ as the final answer instead of back-substituting to $x$.
  • Giving answers in degrees when the question is set in radians (or vice versa).

A-level Scope Notes

Standard Edexcel IAL P2/P3 technique for solving $\tan(ax+b)=k$, $\sin(ax+b)=k$, $\cos(ax+b)=k$ on a restricted interval; uses only linear substitution and the standard inverse-trig branches.
