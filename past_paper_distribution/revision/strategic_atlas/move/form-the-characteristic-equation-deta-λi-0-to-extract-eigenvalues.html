Matrices

Form the characteristic equation det(A − λI) = 0 to extract eigenvalues

Family: Eigenvalue Extraction Frequency: 2 questions

Trigger — When to use

Fires whenever the question explicitly asks for eigenvalues of a square matrix A, or asks for any downstream object that requires eigenvalues first — diagonalisation A = PDP⁻¹, computation of A^n, finding invariant lines/planes, or identifying eigenvectors. Visual cues: a 2×2 or 3×3 matrix of numbers given in the stem, together with words like 'eigenvalue', 'characteristic equation', 'diagonalise', or 'A^n'. Equivalent phrasings include 'find λ such that Ax = λx has a non-zero solution' or 'find values of λ for which (A − λI) is singular'.

Motivation — Why this works

Eigenvalues are by definition the scalars λ for which Ax = λx has a non-trivial solution x ≠ 0, equivalently (A − λI)x = 0 has a non-trivial solution. A homogeneous system has non-trivial solutions iff its coefficient matrix is singular, iff its determinant vanishes. Setting det(A − λI) = 0 therefore converts a matrix condition into a single scalar polynomial equation in λ — a quadratic for 2×2, a cubic for 3×3 — solvable by standard algebraic tools (quadratic formula, factor theorem). This is the only systematic route to eigenvalues outside trivial diagonal/triangular cases.

Thought Process — The step-by-step reasoning

  1. Notice the stem contains a numerical 2×2 or 3×3 matrix and either asks directly for eigenvalues, or asks for diagonalisation/matrix powers which require eigenvalues as a prerequisite.
  2. Recall the defining relation Ax = λx, which rearranges to (A − λI)x = 0, and rule out ad-hoc methods (guessing eigenvectors, row-reducing A) — these don't generalise.
  3. Commit to the determinant route: non-trivial solutions ⇔ A − λI singular ⇔ det(A − λI) = 0; this is guaranteed to yield all eigenvalues.
  4. Bridge to operation: subtract λ from every diagonal entry to write down (A − λI), keeping the off-diagonal entries unchanged.
  5. Tactical sub-choice: before expanding, scan rows and columns and expand the determinant along the row/column with the most zeros (or simplest entries) to minimise cofactor arithmetic.
  6. Plan ahead for solving the resulting polynomial: for a cubic, anticipate using the factor theorem with small integers (±1, ±2, ±3) — and check whether the question already supplies one eigenvalue to save work.

Mechanic — Concrete operations

  1. Form (A − λI) by subtracting λ from each diagonal entry of A; leave off-diagonal entries untouched.
  2. Choose a row or column with the most zeros and apply cofactor expansion, respecting the sign pattern (+, −, +; −, +, −; +, −, +) for 3×3.
  3. Collect like terms to obtain the characteristic polynomial: λ² − tr(A)λ + det(A) = 0 for 2×2, or a cubic aλ³ + bλ² + cλ + d = 0 for 3×3.
  4. Solve: for a quadratic, use the quadratic formula; for a cubic, test small integers via the factor theorem to find one root λ = λ₁, divide out (λ − λ₁), then solve the remaining quadratic.
  5. List the eigenvalues λ₁, λ₂, (λ₃) ready for use in subsequent parts (eigenvectors, P, D, A^n).

Worked Micro-Example

Let $A = \begin{pmatrix} 2 & 1 \\ 1 & 2 \end{pmatrix}$. Then $A - \lambda I = \begin{pmatrix} 2-\lambda & 1 \\ 1 & 2-\lambda \end{pmatrix}$, so $\det(A - \lambda I) = (2-\lambda)^2 - 1 = \lambda^2 - 4\lambda + 3 = 0$. Factoring gives $(\lambda - 1)(\lambda - 3) = 0$, hence eigenvalues $\lambda = 1, 3$.

Variants — Same move, different surface

2×2 matrix [[a,b],[c,d]]
Skip cofactor expansion; write the characteristic equation directly as λ² − (a+d)λ + (ad − bc) = 0 using trace and determinant.
Triangular or diagonal matrix
Read eigenvalues straight off the main diagonal — no expansion required, since det(A − λI) = ∏(aᵢᵢ − λ).
Question already states one eigenvalue (e.g. 'show that λ = 2 is an eigenvalue, hence find the others')
Verify by substitution into det(A − λI) = 0, then divide the cubic by (λ − 2) to reduce to a quadratic.

Common Misreadings

  • Forming det(A + λI) = 0 by getting the sign wrong when subtracting λ from the diagonal.
  • Forming det(λI − A) = 0 instead of det(A − λI) = 0; the roots are the same but intermediate signs differ, leading to sign errors in the cubic's coefficients.
  • Confusing eigenvalues with eigenvectors and trying to row-reduce A directly without first finding λ.

Pitfalls

  • Forgetting to subtract λ from every diagonal entry — typically missing the third one in a 3×3.
  • Sign errors in the 3×3 cofactor expansion, especially mishandling the middle (−) sign.
  • Sign errors when a diagonal entry is already negative, e.g. writing (−3 − λ) as (3 − λ).
  • When factoring the cubic, only testing positive integer candidates and missing negative roots.
  • Losing track of the leading sign: for 3×3 the expansion naturally yields −λ³ + … ; multiplying through by −1 is fine but must be done consistently.

A-level Scope Notes

Fully within Edexcel IAL Further Pure scope (FP2/FP3 depending on specification edition) covering eigenvalues of 2×2 and 3×3 matrices and diagonalisation. The formula booklet does not give the characteristic polynomial explicitly — students must derive it. Avoid invoking Cayley–Hamilton as a shortcut for finding eigenvalues, and do not appeal to complex eigenvalues unless the question explicitly permits them.
