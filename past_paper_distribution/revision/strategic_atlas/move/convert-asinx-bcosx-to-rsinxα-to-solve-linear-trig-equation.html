Trigonometry

Convert a·sinx + b·cosx to R·sin(x+α) to solve linear trig equation

Family: R Formula Frequency: 1 questions

Trigger — When to use

The equation contains a linear combination a·sinx + b·cosx set equal to a constant (or another single trig term reducible to a constant). Cues include both sinx and cosx appearing to the first power with no products, and the constraint asking for all solutions in a given interval. Squaring would introduce extraneous roots and the Weierstrass t = tan(x/2) substitution gives a messy quadratic, so the harmonic (R) form is the canonical Edexcel route.

Motivation — Why this works

The identity R·sin(x+α) = R·cosα·sinx + R·sinα·cosx shows any linear combination a·sinx + b·cosx is itself a single sinusoid of amplitude R = √(a²+b²) and phase α with tanα = b/a. Reducing to sin(x+α) = k makes the equation a standard single-trig problem, solvable by arcsin together with the symmetry θ ↦ π−θ within each period. This preserves the solution set exactly (no extraneous roots) and is the form expected by the mark scheme.

Thought Process — The step-by-step reasoning

  1. Notice the equation is linear in sinx and cosx with a constant on the right — the textbook fingerprint of the R-formula.
  2. Rule out squaring (creates extraneous roots) and t = tan(x/2) (messy rational quadratic, awkward for an exam interval).
  3. Commit to R·sin(x+α): the sin form is natural since the leading coefficient is on sinx and a, b are both positive.
  4. Compute R from a²+b² and α from tanα = b/a, then rewrite the equation as a single sine equation.
  5. Solve sin(x+α) = k via principal value plus the π−θ partner, then back-substitute and filter into the required x-interval.

Mechanic — Concrete operations

  1. Expand R·sin(x+α) = R·cosα·sinx + R·sinα·cosx.
  2. Match coefficients: R·cosα = a, R·sinα = b.
  3. Compute R = √(a²+b²).
  4. Compute α = arctan(b/a), choosing α in the first quadrant when a, b > 0.
  5. Rewrite equation as R·sin(x+α) = c, hence sin(x+α) = c/R.
  6. Shift the x-interval by α to get the interval for θ = x+α.
  7. Find θ₁ = arcsin(c/R) and θ₂ = π − θ₁, add 2π multiples as needed to cover the shifted interval.
  8. Subtract α to recover x and discard any values outside the original interval.

Worked Micro-Example

Solve $2\sin x + \cos x = -1$ for $x \in [0, 2\pi)$. Write $2\sin x + \cos x = R\sin(x+\alpha)$ with $R = \sqrt{2^2+1^2} = \sqrt{5}$ and $\alpha = \arctan(1/2) \approx 0.4636$. Then $\sin(x+0.4636) = -1/\sqrt{5} \approx -0.4472$. Principal value $\theta_1 \approx -0.4636$, partner $\theta_2 = \pi - \theta_1 \approx 3.6052$; adding $2\pi$ to $\theta_1$ gives $\theta_1' \approx 5.8195$. Subtracting $\alpha$: $x \approx \pi$ or $x \approx 5.356$.

Variants — Same move, different surface

a·cosx + b·sinx = c with cosx the dominant term
Use R·cos(x−β) form: R·cosβ = a, R·sinβ = b, giving cos(x−β) = c/R.
a·sinx − b·cosx = c (negative cosx coefficient)
Use R·sin(x−α) with R·sinα = b; the sign on α flips but R is unchanged.
Double-angle version e.g. 3sin2x − 4cos2x = 5
Apply R-formula to the 2x argument: R = 5, solve sin(2x − α) = 1, then divide by 2 at the end and adjust the interval to [0, 4π).
Maximum/minimum or range question for f(x) = a·sinx + b·cosx
Same conversion gives extrema ±R directly without solving an equation.

Common Misreadings

  • Treating a·sin x + b·cos x = c as separable, e.g. setting sinx = c/a and cosx = 0 independently.
  • Reading α = arctan(a/b) instead of arctan(b/a) by confusing which coefficient sits with cosα.
  • Believing R·sin(x+α) and R·cos(x−β) give different solution sets; they don't, only the phase label changes.

Pitfalls

  • Choosing the wrong form (e.g. R·sin(x−α) when both a, b are positive) and getting the wrong sign of α.
  • Failing to translate the x-interval into the θ = x+α interval, so missing or gaining solutions.
  • Missing the second solution from the symmetry θ₂ = π − θ₁ within a period.
  • Rounding α too early and propagating error into the final x-values.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P3): the R-formula for a·sinx ± b·cosx with R > 0 and α acute is an explicit specification topic, used both for solving equations and for finding ranges/extrema.
