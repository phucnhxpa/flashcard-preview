Integration

Combine logarithmic terms using log laws for compact exact answer

Family: Log Simplification Frequency: 1 questions

Trigger — When to use

Fires at the end of an integration problem when the antiderivative or evaluated definite integral contains two or more ln terms. Typical cues: partial-fraction integration produces ±ln|...| pieces with matching coefficients, or evaluating at limits leaves expressions like ln a − ln b + ln c − ln d. The question wording 'exact value', 'in the form ln(p/q)', or 'simplest form' is a strong signal. Also fires when a single coefficient (e.g. 1/2, −3) sits in front of multiple ln terms.

Motivation — Why this works

The log laws ln a − ln b = ln(a/b), ln a + ln b = ln(ab), and n ln a = ln(aⁿ) let several logarithmic terms collapse into one ln of a rational expression. This is the canonical 'simplest exact form' expected in Edexcel mark schemes, and dropping it usually costs a final A1 mark. Combining first also reduces arithmetic error when evaluating at limits, since one ratio is computed instead of four separate logs.

Thought Process — The step-by-step reasoning

  1. Notice the answer currently has multiple ln terms — this is rarely the final form on an exam.
  2. Rule out leaving them separate (loses the 'simplify' mark) and rule out converting to decimals (question says exact).
  3. Commit to combining: check whether coefficients match (factor them out) or whether powers need to be absorbed first via n ln a = ln(aⁿ).
  4. Group additions vs subtractions: positive ln's go to the numerator, negative ln's to the denominator of a single ratio.
  5. Cancel common factors inside the resulting fraction (e.g. 135/140 → 27/28) so the final ln has coprime integers or a fully reduced rational expression.

Mechanic — Concrete operations

  1. Identify all ln terms in the current expression and their signs/coefficients.
  2. If coefficients differ, apply n ln a = ln(aⁿ) to absorb each coefficient into its argument.
  3. If a common coefficient is shared, factor it outside a single ln.
  4. Combine using ln a + ln b = ln(ab) for positives and ln a − ln b = ln(a/b) for negatives, forming one ratio.
  5. Reduce the resulting fraction to lowest terms (or simplify the algebraic ratio by cancelling common factors).
  6. Retain absolute value bars for indefinite integrals: |a|/|b| = |a/b|.

Worked Micro-Example

After integrating, $\int_{2}^{3}\!\left(\tfrac{1}{x}-\tfrac{1}{x+1}\right)dx = [\ln|x|-\ln|x+1|]_{2}^{3} = (\ln 3 - \ln 4) - (\ln 2 - \ln 3)$. Combine: $= \ln\tfrac{3}{4} - \ln\tfrac{2}{3} = \ln\!\left(\tfrac{3/4}{2/3}\right) = \ln\tfrac{9}{8}$.

Variants — Same move, different surface

Coefficients differ on each ln term, e.g. $2\ln|x| + 3\ln|x-1|$
Use $n\ln a = \ln(a^n)$ first to get $\ln[x^2(x-1)^3]$ before combining.
Shared rational coefficient, e.g. $\tfrac{26}{25}\ln|x-3| - \tfrac{26}{25}\ln|2x-1|$
Factor the coefficient out: $\tfrac{26}{25}\ln\!\left|\tfrac{x-3}{2x-1}\right|$.
Definite integral with mixed signs at limits
Group positives into numerator, negatives into denominator of a single ratio before reducing.
Answer required in the form $\ln k$ with $k$ rational
Reduce the final fraction to lowest terms; do not leave $\ln(135/140)$.
An additive constant like $\ln 2$ from a substitution
Multiply it into the numerator: $\ln(p/q) + \ln 2 = \ln(2p/q)$.

Common Misreadings

  • Treating $\ln a - \ln b$ as $\ln(a-b)$ instead of $\ln(a/b)$.
  • Believing the answer is already simplified because each ln has a single argument.
  • Thinking the coefficient must be 1 before combining, instead of factoring or absorbing it as a power.
  • Reading 'exact value' as 'decimal to 3sf' and evaluating numerically.

Pitfalls

  • Sign errors when distributing the minus sign across the upper-limit bracket, e.g. $-(\ln 14 - \ln 15)$ becoming $-\ln 14 - \ln 15$.
  • Failing to reduce the final fraction (leaving $\ln(135/140)$ instead of $\ln(27/28)$).
  • Dropping absolute value bars in indefinite integrals when combining.
  • Combining $a\ln x + b\ln y$ with $a\ne b$ as $\ln(xy)^{a}$ or similar invalid manipulations.
  • Cancelling across a sum inside the ln, e.g. wrongly simplifying $\ln\tfrac{x-3}{2x-1}$.

A-level Scope Notes

Uses only the three standard log laws from Edexcel IAL P2/P3 (Pure Mathematics) and applies routinely in C4/P4 partial-fraction integration questions where exact answers are required.
