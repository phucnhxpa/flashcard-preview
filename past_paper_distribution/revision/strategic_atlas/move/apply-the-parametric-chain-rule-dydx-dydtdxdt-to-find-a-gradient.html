Differentiation

Apply the parametric chain rule dy/dx = (dy/dt)/(dx/dt) to find a gradient

Family: Parametric Differentiation Frequency: 3 questions

Trigger — When to use

The curve is defined parametrically — both x and y are given as functions of a parameter t (or θ) — and the question asks for dy/dx, the gradient at a point, the equation of a tangent, or the equation of a normal. Visual cues: a pair of equations 'x = f(t), y = g(t)' often with a stated domain of t, and a target point given either as coordinates (which need to be back-solved for t) or directly as a t-value. Equivalent phrasings include 'find the gradient of C at the point where x = …', 'show that the tangent at P has equation …', or 'find the equation of the normal at the point with parameter t = …'. The move fires before any attempt to eliminate the parameter, especially when (a) and (b) of a question separate the gradient task from the Cartesian task.

Motivation — Why this works

For a curve given parametrically, y is not expressed directly as a function of x, so dy/dx cannot be computed by direct differentiation. The chain rule identity dy/dx = (dy/dt)/(dx/dt) (valid wherever dx/dt ≠ 0) reduces the problem to two independent, usually straightforward, derivatives in t, followed by a division. This avoids the often messy step of eliminating the parameter to obtain a Cartesian equation and then differentiating implicitly, and it typically yields a clean closed form for the gradient as a function of t (e.g. 1/t for the standard parabola x = at², y = 2at).

Thought Process — The step-by-step reasoning

  1. Notice the curve is given as x = f(t) and y = g(t); there is no direct y = h(x), so dy/dx cannot be obtained by differentiating y with respect to x straight away.
  2. Consider the alternatives: (i) eliminate t to get a Cartesian equation, then differentiate (often implicitly); (ii) apply the parametric chain rule. Eliminating t is usually unnecessary work, and when the question separates 'find the gradient' from 'find the Cartesian equation' into different parts, the examiner is signalling that the parametric route is intended.
  3. Commit to dy/dx = (dy/dt)/(dx/dt). This keeps the work in terms of t and exploits the clean derivatives of standard trig/polynomial parameterisations (often factors cancel between numerator and denominator).
  4. Bridge to computation: differentiate x and y separately with respect to t, being careful with chain-rule applications inside (e.g. d/dt[sec²t] = 2sec²t·tan t).
  5. Decide which t-value is needed: either it is given directly, or it must be back-solved from the target point (e.g. from y = 4t = 4 deduce t = 1; check this t lies in any stated domain).
  6. Form the ratio dy/dx, simplify in t, then substitute the parameter value to obtain the numerical gradient. Use this gradient for any subsequent tangent/normal equation.

Mechanic — Concrete operations

  1. Compute dx/dt by differentiating x = f(t) with respect to t.
  2. Compute dy/dt by differentiating y = g(t) with respect to t (apply chain/product rules as needed).
  3. Form dy/dx = (dy/dt) / (dx/dt) and simplify, cancelling common factors in t.
  4. Find the parameter value t₀ corresponding to the given point: solve f(t₀) = x₀ or g(t₀) = y₀, respecting any stated domain of t.
  5. Substitute t = t₀ into the simplified dy/dx to obtain the numerical gradient.
  6. If a tangent or normal is required, use this gradient (or its negative reciprocal for a normal) with the point (x₀, y₀) in y − y₀ = m(x − x₀).

Worked Micro-Example

Curve: $x = 10t^2,\; y = 20t$. Find the gradient at $B(10,20)$. From $y = 20t = 20$, $t = 1$. Then $\frac{dx}{dt} = 20t$ and $\frac{dy}{dt} = 20$, so $\frac{dy}{dx} = \frac{20}{20t} = \frac{1}{t}$. At $t = 1$, $\frac{dy}{dx} = 1$.

Variants — Same move, different surface

Standard parabola x = at², y = 2at (FP1)
dx/dt = 2at, dy/dt = 2a, so dy/dx = 1/t — a memorisable standard result used repeatedly in tangent/normal/chord questions.
Rectangular hyperbola x = ct, y = c/t (FP1)
dx/dt = c, dy/dt = −c/t², giving dy/dx = −1/t² — another standard FP1 gradient.
Trig parameterisation, e.g. x = a + b tan t, y = c sec²t
Differentiate carefully (d/dt[sec²t] = 2sec²t·tan t); common factors of sec²t typically cancel, leaving a clean expression like k tan t.
Implicit Cartesian form available
Implicit differentiation gives the same answer (e.g. y² = 8x ⇒ 2y·dy/dx = 8 ⇒ dy/dx = 4/y) — useful as a check, but the parametric route is usually the intended method.
Polar curve r = r(θ)
Treat θ as the parameter with x = r cos θ, y = r sin θ; then dy/dx = (dr/dθ·sin θ + r cos θ)/(dr/dθ·cos θ − r sin θ) — same chain-rule philosophy.

Common Misreadings

  • Treating the appearance of a Cartesian-form sub-part as a hint to convert first, when the examiner has deliberately separated the gradient and Cartesian tasks.
  • Misreading 'gradient at the point (x₀, y₀)' as needing a Cartesian equation, when the parameter value can be read off directly from x₀ or y₀.

Pitfalls

  • Inverting the ratio: writing dx/dt over dy/dt instead of dy/dt over dx/dt.
  • Differentiating composite trig expressions incorrectly, e.g. writing d/dt[sec²t] = 2sec t·tan t instead of 2sec²t·tan t.
  • Forgetting to substitute the parameter value and leaving the answer as a function of t (e.g. 1/t) rather than a number.
  • Choosing a t-value outside the stated domain when solving for the parameter (e.g. taking t = 3π/4 when the domain forces t = −π/4).
  • Using dy/dx as the normal's gradient instead of taking the negative reciprocal.

A-level Scope Notes

Fully within Edexcel IAL scope: parametric differentiation is in P4 (Pure 4) and underpins standard FP1 results for the parabola and rectangular hyperbola. The identity dy/dx = (dy/dt)/(dx/dt) is in the formula booklet. Avoid out-of-scope generalisations such as parametric surfaces or vector-valued curvature; second derivatives d²y/dx² parametrically are in scope but require d/dt of (dy/dx) divided again by dx/dt — not a simple ratio of second derivatives.
