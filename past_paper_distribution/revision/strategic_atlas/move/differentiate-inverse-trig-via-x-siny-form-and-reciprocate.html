Differentiation

Differentiate inverse trig via x = sin(y) form and reciprocate

Family: Reciprocal Derivative Frequency: 1 questions

Trigger — When to use

Fires when a question presents y = arcsin(x/a) (or another inverse trig function) together with — or invites you to write — the rearranged form x = a sin(y). Cues include the problem explicitly stating 'show that' the derivative equals a particular expression in x via an intermediate step, or providing x = a sin(y) as a given. Also fires whenever you need d/dx of an inverse trig function but prefer not to quote the standard arcsin/arccos/arctan derivative formula. Recognise it any time the inverse function is easier to differentiate after swapping the roles of dependent and independent variables.

Motivation — Why this works

If x = f(y) with f differentiable and f'(y) ≠ 0, then dy/dx = 1/(dx/dy). For inverse trig, the 'forward' trig form (sin, cos, tan of y) differentiates trivially, whereas differentiating arcsin(x/a) directly either requires memorising 1/√(a² − x²) or building it from the chain rule. Reciprocating a clean dx/dy is faster and less error-prone, and the final conversion back to x uses a single Pythagorean identity. This is the standard Edexcel route for deriving inverse trig derivatives.

Thought Process — The step-by-step reasoning

  1. Notice the inverse trig form y = arcsin(x/2) and the partner equation x = 2 sin(y) — the pairing signals the reciprocal route.
  2. Rule out quoting d/dx[arcsin(x/2)] = 1/√(4 − x²) directly, since the question is steering you through a derivation rather than a recall.
  3. Commit to differentiating x with respect to y because sin(y) differentiates instantly to cos(y), giving a clean dx/dy.
  4. Bridge: write dx/dy = 2 cos(y), then invert to dy/dx = 1/(2 cos(y)).
  5. Tactically, convert cos(y) back to x using cos(y) = √(1 − sin²(y)) = √(1 − (x/2)²), choosing the positive root on the principal branch −π/2 ≤ y ≤ π/2.

Mechanic — Concrete operations

  1. Rewrite y = arcsin(x/a) as x = a sin(y).
  2. Differentiate with respect to y: dx/dy = a cos(y).
  3. Reciprocate: dy/dx = 1/(a cos(y)).
  4. Replace cos(y) using cos(y) = √(1 − sin²(y)) = √(1 − (x/a)²).
  5. Simplify to dy/dx = 1/√(a² − x²).

Worked Micro-Example

Given $y = \arcsin(x/2)$, write $x = 2\sin(y)$. Then $\dfrac{dx}{dy} = 2\cos(y)$, so $\dfrac{dy}{dx} = \dfrac{1}{2\cos(y)}$. Since $\cos(y) = \sqrt{1 - \sin^2(y)} = \sqrt{1 - (x/2)^2}$, we get $\dfrac{dy}{dx} = \dfrac{1}{\sqrt{4 - x^2}}$.

Variants — Same move, different surface

y = arccos(x/a)
Write x = a cos(y), differentiate to dx/dy = −a sin(y), reciprocate, then use sin(y) = √(1 − (x/a)²) to obtain dy/dx = −1/√(a² − x²).
y = arctan(x/a)
Write x = a tan(y), differentiate to dx/dy = a sec²(y), reciprocate, then use sec²(y) = 1 + tan²(y) = 1 + (x/a)² to get dy/dx = a/(a² + x²).
y = arcsin(f(x)) with f a function of x
Write sin(y) = f(x), differentiate implicitly: cos(y) dy/dx = f'(x), then dy/dx = f'(x)/cos(y) = f'(x)/√(1 − f(x)²).
Parametric or implicit setups where dx/dy is easier than dy/dx
Same principle: compute dx/dy directly and reciprocate, provided dx/dy ≠ 0.

Common Misreadings

  • Treating sin(y) = x/2 as an implicit equation in x and chain-ruling both sides — valid but longer than the dx/dy reciprocation route.
  • Assuming the question wants the standard formula quoted, when in fact it is asking for the derivation via x = 2 sin(y).
  • Confusing arcsin with 1/sin and trying to differentiate (sin(x/2))⁻¹ instead.

Pitfalls

  • Using dy/dx = 1/(dx/dy) where dx/dy = 0 — at x = ±a, cos(y) = 0 and the derivative is undefined.
  • Choosing the wrong sign when replacing cos(y) by ±√(1 − sin²(y)); on the principal branch −π/2 ≤ y ≤ π/2, cos(y) ≥ 0 so take the positive root.
  • Forgetting the constant a: differentiating x = a sin(y) gives a cos(y), not cos(y).
  • Leaving the answer in terms of y instead of converting back to x via the Pythagorean identity.

A-level Scope Notes

Fully within Edexcel IAL Further Pure: derivation of inverse trig derivatives via x = sin(y), x = cos(y), x = tan(y) and reciprocation is the standard textbook approach, and the Pythagorean identity used for back-substitution is core C2/FP1 content.
