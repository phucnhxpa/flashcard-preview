Trigonometry

Reduce perfect-square trig equation to its base equals zero

Family: Structural Reduction Frequency: 1 questions

Trigger — When to use

The equation has the form (f(x))^2 = 0 (or more generally (f(x))^n = 0 for positive integer n), where f(x) is a trig expression such as 1+2cos2x, sin x − 1, or 2sin x − √3. Common phrasings include 'the curve touches the x-axis', 'find x such that y = 0' for a squared factor, or a stationary value problem reducing to a perfect square equalling zero. The cue is the visible square (or even power) wrapping a single trig expression.

Motivation — Why this works

For real-valued expressions, (f(x))^2 ≥ 0 with equality iff f(x) = 0, so the squared equation is logically equivalent to the linear-in-trig equation f(x) = 0. This collapses what looks like a quadratic-in-trig problem into a single linear case, with no ± branches to track. It also explains the geometric language 'touches': a squared factor produces a repeated root, so the curve is tangent to the axis rather than crossing it.

Thought Process — The step-by-step reasoning

  1. Notice the whole left-hand side is a single trig expression raised to an even power, set equal to 0.
  2. Rule out expanding into a quadratic in cos2x or splitting into ± cases — both are unnecessary work.
  3. Commit to the principle: a square is zero iff its base is zero, so the problem reduces to one linear-in-trig equation.
  4. Drop the square and write f(x) = 0, then rearrange to the standard form cos(kx) = c (or sin/tan analogue).
  5. Solve in the required interval using the usual general-solution method, remembering that each x found is a tangency (double) root of the original.

Mechanic — Concrete operations

  1. Recognise the form (f(x))^2 = 0 with f(x) a trig expression.
  2. Replace by f(x) = 0.
  3. Rearrange to isolate the trig ratio, e.g. cos2x = −1/2.
  4. Solve over the stated interval using standard inverse-trig + symmetry/periodicity.
  5. State solutions; if asked, note these are points where the curve touches (is tangent to) the axis.

Worked Micro-Example

Solve $(1+2\cos 2x)^2 = 0$ for $0 \le x \le \pi$.\\ Since the LHS is a square, it equals 0 iff $1+2\cos 2x = 0$, i.e. $\cos 2x = -\tfrac{1}{2}$.\\ With $2x \in [0,2\pi]$: $2x = \tfrac{2\pi}{3},\ \tfrac{4\pi}{3}$, so $x = \tfrac{\pi}{3},\ \tfrac{2\pi}{3}$.

Variants — Same move, different surface

Curve y = (sin x − 1)^2 touches x-axis
Reduce to sin x = 1, giving x = π/2 + 2πk.
Higher even power, e.g. (2sin x − 1)^4 = 0
Same reduction: base = 0, so sin x = 1/2.
Odd power, e.g. (tan x − 1)^3 = 0
Still reduces to base = 0, but the curve crosses (with inflexion-type contact) rather than 'touches'.
Stationary value of y = (f(x))^2 equals 0
Minimum value 0 is attained exactly when f(x) = 0; same reduction applies.

Common Misreadings

  • Treating (1+2cos2x)^2 = 0 as a quadratic in cos2x and expanding to 4cos^2 2x + 4cos2x + 1 = 0.
  • Introducing ± when 'square-rooting both sides', producing two cases instead of one.
  • Misreading 'touches' as 'crosses' and expecting two distinct roots locally instead of a repeated root.

Pitfalls

  • Forgetting to scale the interval correctly when the argument is 2x (or kx) before solving.
  • Losing solutions by only taking the principal value of the inverse trig.
  • Reporting solutions outside the stated interval, or omitting endpoints when they satisfy the equation.

A-level Scope Notes

Uses only the basic identity that a real square is zero iff its base is zero, plus standard solution of cos(kx)=c / sin(kx)=c on a given interval — fully within Edexcel IAL P2/P3 trigonometry.
