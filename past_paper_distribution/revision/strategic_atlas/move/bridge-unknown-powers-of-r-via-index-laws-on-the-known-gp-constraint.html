Sequences & Series

Bridge unknown powers of r via index laws on the known GP constraint

Family: Index Law Bridging Frequency: 1 questions

Trigger — When to use

Fires in a geometric progression problem where the common ratio r is defined implicitly by an exact relation like r^n = p/q (so r itself is irrational, e.g. r = (37/15)^{1/11}), and you subsequently need a different power r^k for a term formula a r^{k-1} or for r^{n+1} in a sum S_n = a(r^n − 1)/(r − 1). The visual cue is a clean exact value for one power of r alongside a request for another power. Equivalent phrasings: 'I know r^{11} but need r^4', 'I have r^n exactly and need r^{n+1} in the sum'. Don't compute r as a decimal first and then raise it — instead, route through the exact known power.

Motivation — Why this works

Index laws let you keep the exact known quantity r^n = p/q in play for as long as possible, so rounding error in the approximate value of r is not amplified by repeated exponentiation. Two clean tactics fall out: (i) for r^k with k < n, write r^k = (r^n)^{k/n} = (p/q)^{k/n}, a single exponentiation; (ii) for r^{n+j}, factor as r^{n+j} = r^n · r^j = (p/q) · r^j, so the exact fraction multiplies a low-power approximate. Both rest on the basic identity a^{m+k} = a^m · a^k and (a^m)^{k/m} = a^k. The result is fewer rounding stages and cleaner working for the marker.

Thought Process — The step-by-step reasoning

  1. Notice that the problem hands you r in the form r^n = p/q exactly (e.g. r^{11} = 37/15), but the next part asks for r^k or for r^{n+1} inside a sum.
  2. Consider the naive route: compute r ≈ (p/q)^{1/n} as a decimal, then raise that decimal to the required power. Rule this out — rounding in r compounds when raised to a higher power and may shift the final answer.
  3. Commit to staying symbolic with the known exact power. Ask: can I express the required power as (r^n) raised to a rational, or as r^n times a small remaining power?
  4. If k < n, use r^k = (r^n)^{k/n} = (p/q)^{k/n} — one exponentiation, done. If k = n+j with small j, use r^{n+j} = (p/q) · r^j and only ever compute r^j (often r^1) numerically.
  5. Tactical choice: pick whichever decomposition minimises numerical work. For r^{12} with r^{11} known, r^{11}·r is best; for r^4 with r^{11} known, (37/15)^{4/11} is best.
  6. Carry plenty of decimal places (at least 5) on any approximate factor so the final answer is stable.

Mechanic — Concrete operations

  1. Identify the exact relation r^n = p/q from the question setup.
  2. Identify the target power r^k needed by the term formula or sum formula.
  3. Decompose: if k ≤ n, write r^k = (p/q)^{k/n}; if k = n + j, write r^k = (p/q) · r^j with r = (p/q)^{1/n}.
  4. Evaluate the chosen expression in one calculator step (or via ln: (k/n)·ln(p/q) then exp).
  5. Substitute into a r^{k-1} for a specific term, or into S_n = a(r^n − 1)/(r − 1) for a sum, keeping r^n = p/q exact wherever it appears.

Worked Micro-Example

Given $a = 15$, $r^{11} = \tfrac{37}{15}$, find the 5th term $ar^4$. Using index laws, $r^4 = (r^{11})^{4/11} = \left(\tfrac{37}{15}\right)^{4/11} \approx 1.389$. Therefore $ar^4 = 15 \times 1.389 \approx 20.8$.

Variants — Same move, different surface

Need r^{n+1} inside S_n = a(r^n − 1)/(r − 1)
Keep r^n = p/q exact in the numerator; only approximate the single r in the denominator. Do not recompute r^n from a rounded r.
Compound interest / population growth where (1+i)^N = M is given and (1+i)^k is needed
Same identity: (1+i)^k = M^{k/N}. One exponentiation, no need to find i first.
Surd-form ratio, e.g. r^2 = 3 and r^5 needed
Write r^5 = r^4 · r = (r^2)^2 · r = 9r, keeping the exact part exact.

Common Misreadings

  • Treating r as the primary unknown to compute first, then raising it — missing that r^n is the exact object the question handed you.
  • Trying to solve for r algebraically in closed form when the question only ever needs a specific power of r.

Pitfalls

  • Rounding r to 2–3 dp before exponentiating: errors of 0.001 in r become errors of order 0.01 in r^{11}, easily flipping the final integer answer.
  • Forgetting to keep r^n = p/q exact in the sum formula numerator and instead recomputing it from the rounded r, which defeats the whole point of the bridge.

A-level Scope Notes

Within Edexcel IAL P2 geometric sequences (formula booklet: nth term ar^{n-1}, sum S_n = a(r^n − 1)/(r − 1)). Uses only basic index laws a^{m+k} = a^m a^k and (a^m)^{1/m} = a, both assumed known. No logarithmic manipulation beyond optional ln-then-exp evaluation is required.
