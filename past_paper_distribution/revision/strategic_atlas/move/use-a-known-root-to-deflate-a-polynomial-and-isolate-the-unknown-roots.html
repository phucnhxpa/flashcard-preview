Algebra & Functions

Use a known root to deflate a polynomial and isolate the unknown root(s)

Family: Root Selection Frequency: 3 questions

Trigger — When to use

Fires whenever a polynomial equation (typically quadratic or cubic) is set up such that one root is already known or obvious from context: e.g. the question explicitly names an x-intercept, a previous part guarantees a particular x-value satisfies the equation, or the polynomial has no constant term (making x = 0 a visible root). The cue is 'find the OTHER point/root', or 'the curves meet again at...'. Equivalently: you have an equation whose general factorisation looks ugly, but one root is handed to you for free.

Motivation — Why this works

Every root r of a polynomial corresponds to a linear factor (x − r), so knowing one root reduces the problem by one degree. For a quadratic this means the other root is recoverable instantly via Vieta's (sum = −b/a, product = c/a) or by selecting the 'other' branch of the quadratic formula; for a cubic it reduces the problem to solving a residual quadratic. This avoids guesswork, brute factor-pair search, or polynomial division on coefficients that don't factor cleanly. The underlying principle is the Factor Theorem: r is a root ⇔ (x − r) divides the polynomial.

Thought Process — The step-by-step reasoning

  1. Notice that the polynomial has 'awkward' coefficients (e.g. 12x² − 83x + 115) that resist trial-and-error factoring, OR has no constant term so x = 0 is visibly a root.
  2. Recognise from the problem context that one root is already known — either given as an x-intercept, established in a previous part, or visible from the structure (missing constant term).
  3. Rule out: brute-force factor-pair search (slow, error-prone with large products like 12·115); blind quadratic formula without using the known root as a sanity check; expanding from y = ax² + bx + c with three simultaneous equations when factored form is available.
  4. Commit to deflation: write the polynomial as (x − r) × (lower-degree polynomial), where r is the known root. For x = 0 this is just factoring out x; for x = r ≠ 0 use Vieta's or polynomial division.
  5. Tactical choice: if the residual is linear (cubic case with two known roots, or quadratic case with one), Vieta's sum-of-roots is fastest: other root = −b/a − r. If the residual is quadratic (cubic with one known root), apply the quadratic formula to the deflated quadratic.
  6. Cross-check: substitute the known root back, or verify product-of-roots r · (other) = c/a.

Mechanic — Concrete operations

  1. Identify the known root r from the problem's setup (given intercept, previous part, or visible from missing constant term).
  2. Deflate: factor the polynomial as (x − r) · Q(x), either by inspection (e.g. factor out x when r = 0) or by polynomial/synthetic division.
  3. For a quadratic ax² + bx + c with known root r: use Vieta's, other root = (−b/a) − r = c/(ar); or use the quadratic formula and select the branch ≠ r.
  4. For a cubic with known root r: solve the residual quadratic Q(x) = 0 by the quadratic formula or factoring.
  5. Simplify the resulting fraction (e.g. 46/24 → 23/12).
  6. If a y-coordinate is needed, substitute the new x back into the CORRECT function (the curve y = f(x), or y = f'(x), or y = f''(x) — read the question carefully).

Worked Micro-Example

Solve $12x^2 - 83x + 115 = 0$ given $x = 5$ is a root. By Vieta's, sum of roots $= 83/12$, so the other root is $83/12 - 5 = 83/12 - 60/12 = 23/12$. Check: $5 \cdot 23/12 = 115/12 = c/a$. ✓

Variants — Same move, different surface

Cubic with no constant term, e.g. 9x³ − 48x² + 52x = 0
Factor out x to get x(9x² − 48x + 52) = 0; record x = 0 as one root, then apply the quadratic formula to the residual quadratic.
Quadratic with both roots known (e.g. given x-intercepts 0 and 4)
Skip deflation entirely — write directly in factored form f(x) = A·x(x − 4) and use a remaining condition (e.g. vertex value, another point) to fix A.
Known root r ≠ 0 in a higher-degree polynomial
Use polynomial long division or synthetic division by (x − r) instead of simple factoring.
Cubic with three known roots r₁, r₂, r₃
Write f(x) = A(x − r₁)(x − r₂)(x − r₃); only the leading coefficient A remains.

Common Misreadings

  • Forgetting that the known root MUST appear as one of the solutions and panicking when two solutions emerge — discarding the wrong one.
  • Dividing the entire equation by x without first recording x = 0 as a root, thereby losing it.
  • Setting up y = ax² + bx + c with three unknowns and three points instead of using the simpler factored form when roots are given.
  • Missing the leading coefficient A: writing f(x) = x(x − 4) when the parabola is actually scaled.

Pitfalls

  • Arithmetic slips in the discriminant: e.g. 83² = 6889 and 4·12·115 = 5520; 48² = 2304 and 4·9·52 = 1872 — both prone to miscalculation.
  • Failing to simplify the final fraction (e.g. leaving 46/24 instead of 23/12).
  • Substituting the new x into the wrong function — if the 'point' lies on y = f'(x) or y = f''(x), substituting into f(x) gives the wrong y-coordinate.
  • Omitting the scale factor A when writing factored form, forcing the parabola to a shape that doesn't fit the remaining conditions.

A-level Scope Notes

All techniques (Factor Theorem, Vieta's relations for quadratics, quadratic formula, polynomial division) are within Edexcel IAL P1–P4 scope. The quadratic formula is in the formula booklet; Vieta's is not formally listed but the relations sum = −b/a and product = c/a are standard. Avoid invoking general Vieta's for cubics by name — for IAL, derive the residual quadratic explicitly after deflation rather than quoting symmetric-function identities.
