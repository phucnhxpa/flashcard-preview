Algebra & Functions

Decompose fractional powers as root-then-integer-power to evaluate cleanly

Family: Index Law Decomposition Frequency: 1 questions

Trigger — When to use

Fires when an expression like $a^{p/q}$ appears with a numeric base that is a perfect $q$-th power (e.g. $4^{3/2}$, $9^{-1/2}$, $8^{2/3}$, $16^{3/4}$). Typical cues: substitution of an integer into a function involving fractional indices; simplification of surds; evaluating definite integrals or derivatives at perfect-square/cube points. Also triggered when a non-calculator exact answer is required and the fractional exponent must vanish.

Motivation — Why this works

The index law $a^{p/q} = (a^{1/q})^{p} = (\sqrt[q]{a})^{p}$ splits one awkward operation into two familiar ones: extract a root, then raise to an integer power. When the base is a perfect $q$-th power the root is itself an integer, so every term collapses to an integer or simple fraction with no surds remaining. For negative exponents, $a^{-n} = 1/a^{n}$ is applied after the positive power is computed, keeping arithmetic exact.

Thought Process — The step-by-step reasoning

  1. Notice the base $a$ is a perfect square (or cube), so $\sqrt{a}$ (or $\sqrt[3]{a}$) is an integer — the fractional exponent is tractable.
  2. Rule out direct calculator use or naive multiplication of base by exponent; recognise that $a^{p/q}$ is a power, not a product.
  3. Commit to the identity $a^{p/q} = (\sqrt[q]{a})^{p}$ because it converts the problem into integer arithmetic.
  4. Bridge: compute $\sqrt[q]{a}$ first to get a clean integer, then apply the integer exponent $p$.
  5. Tactical: handle the sign of the exponent last — compute $a^{|p|/q}$ first, then reciprocate if the exponent was negative.

Mechanic — Concrete operations

  1. Identify $q$ (denominator of exponent) and verify $a$ is a perfect $q$-th power.
  2. Compute $r = \sqrt[q]{a}$ as an integer.
  3. Raise: compute $r^{|p|}$.
  4. If exponent is negative, take the reciprocal: $1/r^{|p|}$.
  5. Substitute the resulting integer/fraction back into the larger expression.

Worked Micro-Example

Evaluate $4^{3/2} + 4^{-1/2}$. $4^{3/2} = (\sqrt{4})^{3} = 2^{3} = 8$. $4^{-1/2} = 1/\sqrt{4} = 1/2$. Sum $= 8 + 1/2 = 17/2$.

Variants — Same move, different surface

Perfect-cube base with third-power exponent
Use cube root: $8^{2/3} = (\sqrt[3]{8})^{2} = 2^{2} = 4$.
Perfect fourth-power base
$16^{3/4} = (\sqrt[4]{16})^{3} = 2^{3} = 8$.
Negative fractional exponent
Compute the positive power first, then reciprocate: $9^{-3/2} = 1/(\sqrt{9})^{3} = 1/27$.
Algebraic base $x^{2}$ raised to fractional power
$(x^{2})^{3/2} = |x|^{3}$ — careful with sign if $x$ may be negative.
Fractional base
$(1/4)^{3/2} = 1/4^{3/2} = 1/8$ via the same decomposition.

Common Misreadings

  • Reading $a^{p/q}$ as $a \times (p/q)$ — treating the exponent as a multiplier.
  • Reading $a^{-p/q}$ as $-a^{p/q}$ — confusing negative exponent with negation.
  • Assuming $\sqrt{a}$ exists as an integer for any base, when in fact the move requires $a$ to be a perfect $q$-th power.

Pitfalls

  • Computing $4^{-3/2}$ as $-8$ instead of $1/8$.
  • Forgetting to raise after taking the root, e.g. writing $4^{3/2} = \sqrt{4} = 2$.
  • Applying the root to the wrong index, e.g. using $\sqrt{a}$ when the denominator is $3$.
  • Sign-error on $(x^{2})^{1/2} = x$ when $x$ could be negative (it is $|x|$).

A-level Scope Notes

Standard Edexcel IAL Pure (P1) index-law content; uses only $a^{p/q} = (\sqrt[q]{a})^{p}$ and $a^{-n} = 1/a^{n}$, both within the specification.
