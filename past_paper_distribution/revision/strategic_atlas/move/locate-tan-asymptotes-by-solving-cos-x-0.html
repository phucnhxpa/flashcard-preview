Trigonometry

Locate tan asymptotes by solving cos x = 0

Family: Asymptote Identification Frequency: 1 questions

Trigger — When to use

The question asks for the equation of a vertical asymptote of y = tan x (or a transformed version such as y = tan(x − a)) shown on a sketch. A vertical dashed line is drawn at some unmarked x-value, often pinned between two labelled multiples of π. Phrasings include 'find the equation of the asymptote l' or 'write down the value of x for which the curve has an asymptote in the interval ...'. The exact answer is expected in terms of π, not as a decimal.

Motivation — Why this works

Because tan x = sin x / cos x, the function is undefined precisely where cos x = 0, and these are the only vertical asymptotes. So locating asymptotes reduces to solving cos x = 0, whose solutions are x = (2n+1)π/2. This converts a graph-reading task into an exact algebraic one, and the figure simply selects which solution to report.

Thought Process — The step-by-step reasoning

  1. Notice the asymptote is vertical, so its equation must take the form x = c for a constant c.
  2. Rule out reading c off the axis directly: the figure rarely labels the asymptote, and the answer is expected exactly in terms of π.
  3. Commit to the identity tan x = sin x / cos x, so vertical asymptotes occur exactly where cos x = 0.
  4. Bridge: list the cos-zero solutions x = π/2, 3π/2, 5π/2, ... and any negative ones if needed.
  5. Tactical choice: use the interval cues in the figure (e.g. the asymptote lies between π and 2π) to pick the unique correct solution, here x = 3π/2.

Mechanic — Concrete operations

  1. Recognise y = tan x and write tan x = sin x / cos x.
  2. Set the denominator cos x = 0.
  3. Solve to obtain x = π/2 + nπ, equivalently x = (2n+1)π/2.
  4. Use the interval indicated by the figure to select the relevant n.
  5. State the asymptote as x = (chosen value), e.g. x = 3π/2.

Worked Micro-Example

Find the asymptote $l$ of $y = \tan x$ shown between $\pi$ and $2\pi$. Vertical asymptotes satisfy $\cos x = 0$, so $x = \tfrac{(2n+1)\pi}{2}$. The candidates are $\ldots, \tfrac{\pi}{2}, \tfrac{3\pi}{2}, \tfrac{5\pi}{2}, \ldots$ Only $\tfrac{3\pi}{2} \in (\pi, 2\pi)$, so $l: x = \tfrac{3\pi}{2}$.

Variants — Same move, different surface

y = cot x
Asymptotes occur where sin x = 0, giving x = nπ.
y = sec x
Asymptotes occur where cos x = 0, giving x = (2n+1)π/2 (same set as tan).
y = cosec x
Asymptotes occur where sin x = 0, giving x = nπ.
y = tan(x − a) or y = tan(kx)
Solve cos(x − a) = 0 or cos(kx) = 0; e.g. for tan(kx) get x = (2n+1)π/(2k).
Degree-mode question
Solutions of cos x = 0 are x = 90° + 180°n; report 270° instead of 3π/2.

Common Misreadings

  • Treating the dashed vertical line as a y-axis feature rather than an asymptote of the curve.
  • Assuming asymptotes occur where sin x = 0 (confusing tan with cot).
  • Reading the asymptote position from the gridlines and guessing a decimal like x ≈ 4.7.

Pitfalls

  • Writing x = π/2 instead of x = 3π/2 by not checking which interval the figure highlights.
  • Giving a decimal approximation instead of the exact form in terms of π.
  • Forgetting the period: stating only x = π/2 as 'the' asymptote when a general family is requested.
  • For tan(x − a), forgetting to shift and solving cos x = 0 instead of cos(x − a) = 0.

A-level Scope Notes

Fully within Edexcel IAL Pure (P2/P3) trigonometry: graphs of tan, sec, cosec, cot and their asymptotes, with exact answers in terms of π.
