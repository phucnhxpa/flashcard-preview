Complex Numbers

Write every root in explicit a + ib form, including zeros

Family: Canonical Form Compliance Frequency: 1 questions

Trigger — When to use

Fires whenever the question stipulates the answer 'in the form a + ib' (or 'a + bi', 'x + yi') and some of the roots/values turn out to be purely real or purely imaginary. Watch for phrasings like 'give your answers in the form a + ib, where a, b ∈ ℝ' or 'express each root in Cartesian form'. Also triggers when converting from modulus-argument form back to Cartesian and a result lands on the real or imaginary axis.

Motivation — Why this works

Real numbers and purely imaginary numbers are degenerate cases of complex numbers (b = 0 or a = 0 respectively), so the canonical form a + ib still applies. Writing the suppressed zero part explicitly demonstrates awareness that ℝ ⊂ ℂ and matches the exact format the mark scheme requests. Examiners typically reserve a presentation/accuracy mark for compliance with the stated form, so omitting '+ 0i' or '0 +' risks losing it even when the value is correct.

Thought Process — The step-by-step reasoning

  1. Notice the wording 'in the form a + ib' is a formatting instruction, not just a hint about computation.
  2. Rule out the temptation to simplify '0 + 3i' down to '3i' or '8 + 0i' down to '8' — that would strip the requested structure.
  3. Commit to presenting every root with both an explicit real part a and an explicit imaginary coefficient b, even when one is zero.
  4. Bridge: scan the list of roots and tag each as (real-only), (imaginary-only), or (mixed), then rewrite each in the unified template a + ib.
  5. Tactically, keep signs visible: write 0 − 3i rather than 0 + −3i, and write 8 + 0i rather than 8 − 0i, to keep the form readable.

Mechanic — Concrete operations

  1. Identify each root and read off its real part a and imaginary part b.
  2. If b = 0 (real root), write the root as a + 0i (e.g. 8 → 8 + 0i).
  3. If a = 0 (purely imaginary root), write the root as 0 + bi (e.g. 3i → 0 + 3i, −3i → 0 − 3i).
  4. If both are nonzero, leave it as a + bi with signs tidied.
  5. Present all roots in the same uniform template so the format is unambiguous.

Worked Micro-Example

Solve $z^4 - 6z^2 - 72 = 0$ giving roots in the form $a + ib$. Let $w = z^2$: $w^2 - 6w - 72 = 0 \Rightarrow w = 12$ or $w = -6$. Then $z = \pm 2\sqrt{3}$ or $z = \pm i\sqrt{6}$. Final answer: $z = 2\sqrt{3} + 0i,\ -2\sqrt{3} + 0i,\ 0 + \sqrt{6}\,i,\ 0 - \sqrt{6}\,i$.

Variants — Same move, different surface

Converting from modulus-argument or exponential form back to Cartesian
When $r e^{i\theta}$ lands on an axis (e.g. $\theta = 0, \pi/2, \pi$), still write the result as $a + 0i$ or $0 + bi$ rather than collapsing to a bare real or imaginary number.
nth roots of a real or imaginary number
Among the n roots, any that fall on the real or imaginary axis must still be written in $a + ib$ form with the zero component shown.
Roots of a polynomial with one real root and a complex conjugate pair
The real root is written as $a + 0i$ to match the conjugate pair $p \pm qi$ in the same canonical form.

Common Misreadings

  • Treating 'in the form a + ib' as merely descriptive and writing bare reals like '8' or bare imaginaries like '3i'.
  • Assuming that because a real number 'is obviously real', no formatting adjustment is needed.
  • Confusing the instruction with 'in modulus-argument form' and unnecessarily converting.

Pitfalls

  • Writing '0 + −3i' instead of '0 − 3i', which is technically correct but messy and may be penalised for presentation.
  • Forgetting one root in the list and only formatting some of them uniformly.
  • Mislabelling which part is zero (e.g. writing '3 + 0i' for the root 3i).

A-level Scope Notes

Fully within Edexcel IAL Further Pure scope (FP1/FP2 complex numbers): all roots are expressed with $a, b \in \mathbb{R}$ in Cartesian form, no extension beyond standard polynomial root and de Moivre questions.
