Differential Equations

Use target answer form to guide separation exponent

Family: Answer Form Backtracking Frequency: 1 questions

Trigger — When to use

The question states the solution in a specific algebraic shape — e.g. $A=(pt^q+r)^2$, $y=e^{f(x)}$, or $y=(\,\cdot\,)^{1/3}$ — before asking you to solve a separable ODE. This shape tells you in advance what the antiderivative on the dependent-variable side must look like. Fires whenever the prescribed form would clash with a naive separation (e.g. an $\ln$ appearing where a squared polynomial is expected).

Motivation — Why this works

Each separable ODE admits several algebraically-equivalent ways to write $\int g(A)\,dA$, but only one produces an antiderivative compatible with the stated answer shape. If $A=(\,\cdot\,)^2$, then $\sqrt{A}$ is a polynomial in $t$, so the LHS integral must deliver $A^{1/2}$, forcing $\int A^{-1/2}\,dA=2A^{1/2}$. Choosing instead $\int A^{-1}\,dA=\ln A$ would require exponentiating to recover $A$, which cannot yield a perfect square in $t$. The target form thus reverse-engineers the correct power of $A$ to separate.

Thought Process — The step-by-step reasoning

  1. Notice the prescribed answer form, e.g. $A=(pt^q+r)^2$, and read off that $\sqrt{A}=pt^q+r$ is a clean polynomial in $t$.
  2. Rule out separations that would produce $\ln A$ or $A^{-1}$, since exponentiating or reciprocating cannot regenerate a squared polynomial in $t$.
  3. Commit to the separation whose antiderivative on the $A$-side is $A^{1/2}$, i.e. $\int A^{-1/2}\,dA=2A^{1/2}$.
  4. Bridge: rearrange the given ODE so the $A$-side reads $A^{-1/2}\,dA$, which usually means the original RHS had a $\sqrt{A}$ factor.
  5. Tactically, after integrating, square both sides to match the target form and use initial conditions to pin down $p$, $r$ (and $q$ if free).

Mechanic — Concrete operations

  1. Read the target form and identify the power $A^{1/n}$ that must appear after integration.
  2. Choose the separation $\int A^{-(n-1)/n}\,dA = n\,A^{1/n}$ so the antiderivative matches.
  3. Integrate the $t$-side to a single clean power/polynomial in $t$.
  4. Equate, apply the initial condition, then raise to the $n$-th power to recover the prescribed shape.
  5. Read off constants $p,q,r$ by comparing with $(pt^q+r)^n$.

Worked Micro-Example

Solve $\dfrac{dA}{dt}=\dfrac{3\sqrt{A}}{2}\,t^{-2/5}$ given $A=(pt^q+r)^2$. Separate as $A^{-1/2}\,dA=\tfrac{3}{2}t^{-2/5}\,dt$. Integrate: $2A^{1/2}=\tfrac{3}{2}\cdot\tfrac{5}{3}t^{3/5}+C=\tfrac{5}{2}t^{3/5}+C$. So $\sqrt{A}=\tfrac{5}{4}t^{3/5}+\tfrac{C}{2}$, giving $A=\left(\tfrac{5}{4}t^{3/5}+k\right)^2$ with $p=\tfrac{5}{4},\;q=\tfrac{3}{5}$.

Variants — Same move, different surface

Target form $y=e^{f(x)}$
Anticipate $\ln y$ on the LHS, so separate as $\int y^{-1}\,dy$ and exponentiate at the end.
Target form $y=(\,\cdot\,)^3$
Anticipate $y^{1/3}$ from $\int y^{-2/3}\,dy=3y^{1/3}$, then cube.
Target form $y^2=\,\cdot\,$ (implicit square)
Separate as $\int 2y\,dy = y^2$ so the LHS already matches the target without further manipulation.
Target form $\tan y = f(x)$
Anticipate that $\int \sec^2 y\,dy = \tan y$, so rearrange the ODE to expose $\sec^2 y\,dy$.

Common Misreadings

  • Ignoring the given answer form and integrating blindly, then trying to force $\ln A$ into a squared polynomial.
  • Treating $(pt^q+r)^2$ as $p^2t^{2q}+r^2$ and integrating term-by-term instead of preserving the squared structure.
  • Assuming the constant of integration sits outside the square rather than inside as part of $r$.

Pitfalls

  • Dropping the factor of $2$ from $\int A^{-1/2}\,dA=2A^{1/2}$ when matching coefficients.
  • Forgetting to halve the RHS constant when going from $2\sqrt{A}=\dots$ to $\sqrt{A}=\dots$ before squaring.
  • Squaring prematurely before applying the initial condition, which complicates solving for the constant.

A-level Scope Notes

Sits squarely within Edexcel IAL P3/P4 separable first-order ODEs; uses only standard power-rule integration and the algebraic identity $(\sqrt{A})^2=A$.
