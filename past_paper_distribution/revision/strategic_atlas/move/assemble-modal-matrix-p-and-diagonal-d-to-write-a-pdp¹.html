Matrices

Assemble modal matrix P and diagonal D to write A = PDP⁻¹

Family: Eigendecomposition Frequency: 1 questions

Trigger — When to use

The question explicitly asks to diagonalise a 3×3 matrix A, or to express A in the form PDP⁻¹ (or equivalently P⁻¹AP = D). Eigenvalues and three linearly independent eigenvectors have already been computed (often the previous part of the question). Phrasings include 'find a matrix P and a diagonal matrix D such that…', 'hence diagonalise A', or 'show that A = PDP⁻¹'.

Motivation — Why this works

If e₁, e₂, e₃ are eigenvectors of A with eigenvalues λ₁, λ₂, λ₃, then Aeᵢ = λᵢeᵢ. Stacking these as AP = PD with P = (e₁|e₂|e₃) and D = diag(λ₁,λ₂,λ₃) packages all three eigen-relations into one matrix equation. Linear independence of the eigenvectors makes P invertible, so A = PDP⁻¹. This form is the engine for computing Aⁿ, matrix exponentials, and solving coupled recurrences in later parts.

Thought Process — The step-by-step reasoning

  1. I notice three eigenvalues with three eigenvectors are already in hand, so a diagonalisation exists.
  2. I rule out trying to manipulate A directly (e.g. computing Aⁿ entry-wise) — the question is steering me to the PDP⁻¹ structure.
  3. I commit to building P column-by-column from eigenvectors and D from eigenvalues in the same order, because AP = PD then forces A = PDP⁻¹.
  4. First concrete operation: write down P with the eigenvectors as columns and D as the matching diagonal matrix.
  5. Tactical decision: choose a column ordering that keeps eigenvectors with simple entries leftmost to ease the P⁻¹ computation, then invert P by row reduction of [P | I] or by the adjugate formula.

Mechanic — Concrete operations

  1. Write P = (e₁ | e₂ | e₃) with each eigenvector as a column.
  2. Write D = diag(λ₁, λ₂, λ₃) using the SAME ordering as the columns of P.
  3. Compute P⁻¹ via [P | I₃] → [I₃ | P⁻¹] row reduction or via P⁻¹ = (1/det P)·adj(P).
  4. State A = PDP⁻¹ explicitly, presenting P, D and P⁻¹.
  5. If asked, verify by computing PDP⁻¹ and checking it equals A.

Worked Micro-Example

Suppose A has eigenvalues $\lambda_1=1,\ \lambda_2=2,\ \lambda_3=3$ with eigenvectors $e_1=(1,0,0)^T,\ e_2=(1,1,0)^T,\ e_3=(1,1,1)^T$. Then $P=\begin{pmatrix}1&1&1\\0&1&1\\0&0&1\end{pmatrix}$, $D=\begin{pmatrix}1&0&0\\0&2&0\\0&0&3\end{pmatrix}$, and $P^{-1}=\begin{pmatrix}1&-1&0\\0&1&-1\\0&0&1\end{pmatrix}$, giving $A=PDP^{-1}$.

Variants — Same move, different surface

Question asks for P⁻¹AP = D instead of A = PDP⁻¹
Same P and D; just present the rearranged identity — no extra computation.
2×2 matrix diagonalisation
Same recipe with two eigenvectors as columns; P⁻¹ via the simple 2×2 inverse formula (1/det)·swap-and-negate.
Repeated eigenvalue with two independent eigenvectors found
Diagonalisation still works: place both eigenvectors of the repeated λ in adjacent columns of P and repeat λ on the diagonal of D in those positions.
Subsequent part asks for Aⁿ
Use Aⁿ = PDⁿP⁻¹ where Dⁿ = diag(λ₁ⁿ, λ₂ⁿ, λ₃ⁿ); no need to recompute P or P⁻¹.

Common Misreadings

  • Pairing eigenvalues with the wrong eigenvector columns — e.g. putting λ₁ in the (1,1) entry of D when e₁ is in column 2 of P.
  • Treating P as orthogonal and writing P⁻¹ = Pᵀ; this only holds for symmetric A with orthonormal eigenvectors, which is outside required scope.
  • Believing diagonalisation fails just because an eigenvalue is repeated, when in fact two independent eigenvectors for that eigenvalue still suffice.

Pitfalls

  • Arithmetic slips when computing P⁻¹ for a 3×3 matrix — keep determinant and cofactor work tidy.
  • Scaling eigenvectors differently from the mark scheme: P and P⁻¹ will look different, but PDP⁻¹ must still recover A — verify rather than panic.
  • Forgetting that the column order in P must match the diagonal order in D; swapping one without the other breaks the identity.
  • Writing D as a column or row vector of eigenvalues instead of a diagonal matrix.

A-level Scope Notes

Fully within Edexcel IAL Further Pure: diagonalisation of 2×2 and 3×3 matrices with real eigenvalues, modal matrix P, and use of A = PDP⁻¹ to compute powers of A.
