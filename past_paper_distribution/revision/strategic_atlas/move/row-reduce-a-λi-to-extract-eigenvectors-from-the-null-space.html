Matrices

Row-reduce (A − λI) to extract eigenvectors from the null space

Family: Null Space Extraction Frequency: 11 questions

Trigger — When to use

Eigenvalues λ₁, λ₂, … of a square matrix A have already been found (typically from det(A − λI) = 0), and the question now asks for the corresponding eigenvectors, an eigenbasis, or a diagonalising matrix P. Equivalent phrasings: 'find an eigenvector associated with λᵢ', 'find a basis for the eigenspace', 'find the kernel/null space of A − λᵢI', or 'find P such that P⁻¹AP = D'. The structural cue is that for each known λᵢ you face a homogeneous system (A − λᵢI)x = 0 in which the coefficient matrix is guaranteed singular.

Motivation — Why this works

An eigenvector for λᵢ is by definition any non-zero x with Ax = λᵢx, i.e. a non-zero element of ker(A − λᵢI). Because λᵢ satisfies det(A − λᵢI) = 0, the matrix A − λᵢI is singular, so the homogeneous system has at least one free variable and infinitely many solutions. Row reduction (Gaussian elimination) systematically exposes pivot vs free columns, so back-substitution yields the eigenvector parametrically — far more reliable than inspection on a 3×3. The output is a direction vector, unique up to non-zero scalar multiples, which is exactly what is needed.

Thought Process — The step-by-step reasoning

  1. Notice: each eigenvalue λᵢ has been pinned down, so the next task is purely linear-algebraic — solve (A − λᵢI)x = 0 for non-zero x.
  2. Rule out alternatives: I could try to spot a vector by inspection, or solve two scalar equations ad hoc, but for a 3×3 these are error-prone; Cramer's rule is useless here because the system is singular by design.
  3. Commit: row reduction is the systematic, guaranteed method — it must produce at least one zero row because det(A − λᵢI) = 0, confirming a free variable exists.
  4. Bridge to operation: substitute λᵢ numerically into A − λᵢI to obtain a concrete 3×3 matrix, then apply elementary row operations to reach echelon form.
  5. Tactical choice: pick the free variable (the column without a pivot) and assign it a convenient value (usually t = 1, or a small integer that clears fractions) so the resulting eigenvector has tidy integer entries.
  6. Sanity check: verify Aeᵢ = λᵢeᵢ on one component, and confirm eᵢ ≠ 0 — the zero vector is never an eigenvector.

Mechanic — Concrete operations

  1. Substitute λ = λᵢ into A − λI to form the explicit numerical matrix Mᵢ.
  2. Apply elementary row operations (e.g. R₂ ← R₂ − kR₁, R₃ ← R₃ − mR₁, then eliminate in column 2) to reduce Mᵢ to row echelon form.
  3. Identify pivot columns and free column(s); for a simple eigenvalue exactly one column is free.
  4. Set the free variable equal to a parameter t (commonly t = 1) and back-substitute up the echelon form to determine the pivot variables.
  5. Write the eigenvector as a column vector eᵢ, scaling if needed so entries are integers.
  6. Repeat for each distinct eigenvalue; for a repeated eigenvalue with two free variables, extract two linearly independent eigenvectors by setting (t,s) = (1,0) and (0,1).

Worked Micro-Example

For A = [[2,1,0],[0,3,0],[0,1,2]] with eigenvalue λ = 3: A − 3I = [[-1,1,0],[0,0,0],[0,1,-1]]. Row-reduce → pivots in columns 1 and 2; column 3 free. Set z = 1: from row 3, y = 1; from row 1, −x + y = 0 ⇒ x = 1. Eigenvector e = (1,1,1)ᵀ.

Variants — Same move, different surface

Repeated eigenvalue with geometric multiplicity 2 (e.g. λ appears twice and A − λI has rank 1)
Row reduction leaves two free variables; set (s,t) = (1,0) and (0,1) to obtain two linearly independent eigenvectors spanning the 2-D eigenspace.
2×2 matrix
A − λI has rank 1, so a single scalar equation ax + by = 0 is sufficient — read off the eigenvector (b, −a)ᵀ by inspection without full row reduction.
Repeated eigenvalue but only one free variable (defective / non-diagonalisable)
Only one eigenvector is obtainable; the matrix is not diagonalisable. Within IAL scope this signals 'P does not exist'.
Negative or fractional λ
Same procedure, but be especially careful when forming A − λI — subtracting a negative λ adds |λ| on the diagonal.

Common Misreadings

  • Treating (A − λI)x = 0 as a non-homogeneous system and trying to invert (A − λI) — but it is singular by construction.
  • Confusing 'eigenvector' with 'eigenvalue' and trying to re-solve the characteristic equation.
  • Assuming a unique eigenvector exists; in fact any non-zero scalar multiple is also an eigenvector.

Pitfalls

  • Arithmetic slips when forming A − λI, especially with negative or fractional λ (double-negative errors on the diagonal).
  • Writing the zero vector as an eigenvector — invalid by definition.
  • For a repeated eigenvalue, stopping after one eigenvector when the eigenspace is 2-dimensional, leading to an incomplete basis and a wrong P.
  • Row-operation errors propagating through echelon form; always verify by computing Aeᵢ and checking it equals λᵢeᵢ.

A-level Scope Notes

Fully within Edexcel IAL Further Pure (FP2/FP3) treatment of eigenvalues, eigenvectors and diagonalisation of 2×2 and 3×3 matrices. No formula booklet entry is invoked — this is a method. Generalised eigenvectors, Jordan normal form, and complex eigenvectors for non-symmetric real matrices are out of scope and should not be introduced.
