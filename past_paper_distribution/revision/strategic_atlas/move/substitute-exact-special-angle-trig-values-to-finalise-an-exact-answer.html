Trigonometry

Substitute exact special-angle trig values to finalise an exact answer

Family: Exact Value Recall Frequency: 5 questions

Trigger — When to use

This move fires at the final evaluation stage of a problem when arguments inside sin/cos/tan land on standard angles — multiples of π/6, π/4, π/3, π/2, or π — and the question demands an EXACT answer (e.g. 'in the form a + bi', 'give your answer in the form aπ + b√2', 'find the exact value'). Typical surface cues: definite integral bounds like 0, π/12, π/8, π/6, π/4, π/3, π/2, π; de Moivre's theorem producing nθ at a recognisable angle; substitution into an antiderivative producing sin(π) or cos(π/2). Equivalent phrasings: 'special-angle evaluation', 'unit-circle look-up', 'reading off exact surd values'.

Motivation — Why this works

The move replaces symbolic trig expressions with their exact numerical values (0, ±1/2, ±√2/2, ±√3/2, ±1) so the answer can be presented in closed surd/rational form. It works because the unit circle stores these values exactly — derivable from the 30-60-90 and 45-45-90 triangles plus quadrant signs (CAST). Recognising vanishing terms (sin(kπ) = 0, cos(π/2) = 0) also prunes dead algebra. This is the bridge between an analytic result and the marked exact answer the examiner requires; using a calculator decimal here would lose the accuracy/exact-answer marks.

Thought Process — The step-by-step reasoning

  1. I notice the argument(s) of sin/cos after substitution are standard: e.g. 3π/4, π/2, π, 5π/3 — these are unit-circle angles, not arbitrary ones.
  2. I rule out further symbolic manipulation (compound-angle expansion, identities) because the angle is already standard; I also rule out a calculator decimal because the question asks for an exact answer.
  3. I commit to direct look-up: identify the quadrant, find the reference angle, apply the CAST sign, and write the exact surd value.
  4. I scan for vanishing terms first — any sin(kπ) or cos((2k+1)π/2) collapses entire summands and saves work, so I deal with those before evaluating the survivors.
  5. For each remaining term I substitute the exact value, then distribute any prefactor (like r^n in de Moivre, or 1/2 from integration by parts) carefully across both real and imaginary / both numerator and denominator parts.
  6. Finally I combine fractions over a common denominator and check the answer is fully simplified with no trig symbols remaining.

Mechanic — Concrete operations

  1. Identify each trig argument α and locate it on the unit circle: quadrant + reference angle.
  2. Apply CAST to determine the sign; read off the magnitude from {0, 1/2, √2/2, √3/2, 1}.
  3. Substitute the exact value back into the expression in place of sin(α) or cos(α).
  4. Distribute any external scalar (r^n, 1/2, integration constants) across the substituted value.
  5. Evaluate at upper bound minus lower bound (for definite integrals) or sum real/imaginary parts (for de Moivre).
  6. Combine fractions, rationalise if needed, and present in the form requested (a + bi, aπ + b√c, etc.).

Worked Micro-Example

Evaluate $\left[\tfrac{x}{2} - \tfrac{\sin 6x}{18}\right]_{\pi/12}^{\pi/8}$. At $x=\pi/8$: $\sin(6\cdot\pi/8)=\sin(3\pi/4)=\tfrac{\sqrt2}{2}$ (Q2, ref $\pi/4$, sin positive). At $x=\pi/12$: $\sin(6\cdot\pi/12)=\sin(\pi/2)=1$. Result: $\left(\tfrac{\pi}{16}-\tfrac{\sqrt2}{36}\right) - \left(\tfrac{\pi}{24}-\tfrac{1}{18}\right) = \tfrac{\pi}{48} + \tfrac{2-\sqrt2}{36}$.

Variants — Same move, different surface

Argument is not standard but decomposes (e.g. 7π/12 = π/3 + π/4)
Use compound-angle formulae cos(A+B), sin(A+B) on the standard pieces before substituting exact values.
De Moivre roots needing Cartesian form a + bi
Compute $R\cos\alpha_k + iR\sin\alpha_k$ where $R = r^{1/n}$; multiply both components by R after the exact look-up.
Result left in exponential form $re^{i\theta}$
Rewrite as $r\cos\theta + ir\sin\theta$ then substitute exact values exactly as in Cartesian polar form.
Reduction formula or definite integrals with bounds at 0 and π
Spot which sin^n or cos^n terms vanish at the bounds before computing the survivors — saves substantial algebra.

Common Misreadings

  • Reading cos(2·π/2) as cos(π/2) = 0 instead of cos(π) = −1 — failing to evaluate the multiplied argument first.
  • Treating sin(3π/4) as a non-standard angle and reaching for a calculator instead of using the supplementary-angle identity.
  • Mistaking the question's request for 'exact form' as permission to leave trig symbols in the final answer.

Pitfalls

  • Sign errors from misapplying CAST: e.g. sin(3π/4) = −√2/2 (wrong, Q2 sine is positive) or sin(5π/3) = +√3/2 (wrong, Q4 sine is negative).
  • Confusing sin/cos at related angles, e.g. sin(π/3) vs cos(π/3).
  • Evaluating sin(π) as 1 (confusion with sin(π/2)).
  • Forgetting to multiply the exact trig value by the external scalar (r^n, 1/2, etc.).
  • Sign error when subtracting a negative lower-bound contribution: π²/8 − (−1/2) = π²/8 + 1/2, not π²/8 − 1/2.
  • Stopping before combining fractions over a common denominator, leaving an unsimplified answer.

A-level Scope Notes

Within Edexcel IAL P3/P4/FP1/FP2 scope. Exact values for π/6, π/4, π/3 (and their reflections) are NOT in the formula booklet — students must recall them. Compound-angle formulae used for non-standard decompositions ARE in the booklet. Avoid generalising to angles like π/5 or π/8 which require half-angle work beyond routine recall.
