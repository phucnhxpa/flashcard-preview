Differentiation

Substitute stationary x-value into original equation for y-coordinate

Family: Back Substitution Into Original Frequency: 1 questions

Trigger — When to use

The question asks for the coordinates (or y-value) of a stationary/turning point, and solving dy/dx = 0 has yielded only the x-value(s). Phrasings include 'find the coordinates of the stationary point', 'find the exact coordinates of the turning point', or 'show that the minimum point is...'. The cue is that a complete point (x, y) is required, but the derivative alone cannot supply y.

Motivation — Why this works

The derivative dy/dx is the gradient function, not a height function — its value at a stationary point is 0 by construction. The y-coordinate of a point on the curve always comes from the curve's defining equation y = f(x). Returning to the original equation is the only consistent way to recover the height, and it preserves any exact-form structure (logs, surds, e^k) the question demands.

Thought Process — The step-by-step reasoning

  1. Notices that dy/dx = 0 has been solved and an x-value (here x = -1/2) is in hand, but the question asks for coordinates.
  2. Rules out substituting into dy/dx, recognising that this would just verify the gradient is 0, not produce a y-value.
  3. Commits to the original equation y = f(x) as the unique source of the y-coordinate.
  4. Bridges by writing y = f(x-value) cleanly, keeping exact constants (e^{-1}, ln 2, surds) unevaluated.
  5. Tactically simplifies to a clean exact form (e.g. -1/(2e)) and reports the ordered pair, checking against any 'exact form' instruction.

Mechanic — Concrete operations

  1. Take the x-value(s) from dy/dx = 0.
  2. Substitute each x-value into the original y = f(x), not into the derivative.
  3. Simplify using exact-form conventions (keep e^k, ln, surds; do not decimalise).
  4. State the stationary point as the ordered pair (x, y).
  5. If multiple stationary points, repeat for each x-value.

Worked Micro-Example

Given y = xe^{2x}, dy/dx = e^{2x}(2x+1) = 0 gives x = -1/2. Substitute into the original: y = (-\tfrac{1}{2})e^{2(-1/2)} = -\tfrac{1}{2}e^{-1} = -\tfrac{1}{2e}. Stationary point: \left(-\tfrac{1}{2},\ -\tfrac{1}{2e}\right).

Variants — Same move, different surface

Polynomial curve, e.g. y = x^3 - 3x
Solve 3x^2 - 3 = 0 to get x = ±1, then substitute back into y = x^3 - 3x to get y = ∓2.
Logarithmic curve, e.g. y = x - 2\ln x
From dy/dx = 1 - 2/x = 0 get x = 2; substitute into original for y = 2 - 2\ln 2 (keep exact log form).
Trigonometric curve in a specified interval
After solving dy/dx = 0 for x in the given range, substitute each valid x into y = f(x) and keep exact values like \sqrt{3}/2.
Parametric curve with x = f(t), y = g(t)
Solve dy/dx = (dy/dt)/(dx/dt) = 0 for t, then substitute that t into both x = f(t) and y = g(t) — not into dy/dx.
Implicit curve F(x,y) = 0
After finding x at the stationary point, substitute into the original implicit equation and solve for y, rather than using the implicit derivative.

Common Misreadings

  • Substituting the stationary x-value into dy/dx and reporting the resulting 0 as the y-coordinate.
  • Substituting into d^2y/dx^2 (used for the nature test) and treating that value as y.
  • For parametric curves, substituting t into dy/dx instead of into y = g(t).

Pitfalls

  • Giving a decimal approximation (e.g. -0.184) when the question demands exact form -1/(2e).
  • Mishandling e^{2(-1/2)}: writing e^{-1/2} or e^0 instead of e^{-1}.
  • Reporting only the y-value when the question asks for full coordinates, or vice versa.
  • Forgetting to repeat the substitution for each stationary x-value when there is more than one.

A-level Scope Notes

Standard across Edexcel IAL P2–P4 and FP modules: stationary points of explicit, implicit, and parametric curves all require back-substitution into the original equation, with exact-form answers expected when stated.
