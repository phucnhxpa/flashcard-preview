Collapse log difference then apply power rule to expose integer factor

Family: Log Law Simplification Frequency: 1 questions

Trigger — When to use

Fires when an exact answer contains a difference of logarithms with the same base, typically after evaluating a definite integral of the form $\int \frac{f'(x)}{f(x)}\,dx$ between two bounds. Cue phrases: 'give your answer in the form $k\ln 2$', 'exact value', or seeing $\ln A - \ln B$ where $A/B$ is a perfect power of a small integer. Also fires whenever the leading coefficient (e.g. $\tfrac{1}{2}$, $\tfrac{\pi}{2}$) looks like it should cancel with a power that emerges from the log.

Motivation — Why this works

The identities $\ln a - \ln b = \ln(a/b)$ and $\ln(a^n) = n\ln a$ are the only legitimate routes to converting a messy log expression into a single clean term. When the ratio $a/b$ simplifies to $c^n$ for integer $c, n$, the power rule extracts $n$, which usually cancels with the outer fractional coefficient. The chain compresses two arbitrary-looking logs into a single named constant ($\ln 2$, $\ln 3$, etc.), which is what 'exact form' marks demand.

Thought Process — The step-by-step reasoning

  1. I notice two logarithms with the same base separated by a minus sign and a shared coefficient — this screams 'combine first, evaluate later'.
  2. I rule out decimal evaluation; the question asks for exact form, and the bounds were chosen so the ratio is nice.
  3. I commit to $\ln a - \ln b = \ln(a/b)$ because the ratio $80/20 = 4$ is an integer power, signalling further collapse.
  4. I bridge by computing the ratio explicitly and rewriting it as $c^n$ — here $4 = 2^2$ — to set up the power rule.
  5. I check whether the extracted exponent cancels the outer coefficient; if a $2$ comes out and I have a $\tfrac{1}{2}$ outside, the final form is forced and I stop there.

Mechanic — Concrete operations

  1. Factor the shared coefficient: $\tfrac{\pi}{2}(\ln 80 - \ln 20)$.
  2. Apply $\ln a - \ln b = \ln(a/b)$ to get $\tfrac{\pi}{2}\ln 4$.
  3. Rewrite the argument as a power: $4 = 2^2$.
  4. Apply $\ln(a^n) = n\ln a$: $\tfrac{\pi}{2}\cdot 2\ln 2$.
  5. Cancel the coefficient: $\pi \ln 2$.

Worked Micro-Example

Evaluate $\tfrac{1}{2}\ln 80 - \tfrac{1}{2}\ln 20$.\\ $= \tfrac{1}{2}\ln\!\left(\tfrac{80}{20}\right) = \tfrac{1}{2}\ln 4$.\\ $= \tfrac{1}{2}\ln(2^2) = \tfrac{1}{2}\cdot 2\ln 2 = \ln 2$.

Variants — Same move, different surface

Volume of revolution giving $\tfrac{\pi}{2}(\ln 27 - \ln 9)$
Ratio is $3$, not a higher power; result is $\tfrac{\pi}{2}\ln 3$ with no further cancellation.
Definite integral yielding $\ln 32 - \ln 8$
Ratio $= 4 = 2^2$, so result is $2\ln 2$ via the same chain.
Sum form $\ln a + \ln b$ with $ab = c^n$
Use $\ln a + \ln b = \ln(ab)$ first, then power rule.
Coefficient form 'answer is $k\ln 2$, find $k$'
Drive the simplification all the way to a single $\ln 2$ term and read off $k$.
Solving exponential equations e.g. $e^{2x} = 5$
Same identities run in reverse: $2x = \ln 5 \Rightarrow x = \tfrac{1}{2}\ln 5 = \ln\sqrt{5}$.

Common Misreadings

  • Treating $\ln 80 - \ln 20$ as $\ln(80-20) = \ln 60$.
  • Computing $\ln 80$ and $\ln 20$ as decimals and losing the exact form mark.
  • Stopping at $\tfrac{\pi}{2}\ln 4$ without spotting $4 = 2^2$.
  • Reading 'exact answer' as 'numerical answer to 3 s.f.'.

Pitfalls

  • Arithmetic slips evaluating the bounds (e.g. $3(5)^2 + 5 = 80$, $3(\sqrt 5)^2 + 5 = 20$).
  • Misapplying the quotient rule: writing $\ln(80/20) = \ln 80 / \ln 20$.
  • Dropping the outer factor (e.g. $\pi$ or $\tfrac{\pi}{2}$) when manipulating the logs.
  • Cancelling the $2$ from the power rule against the wrong coefficient.

A-level Scope Notes

Uses only the standard log laws $\ln a - \ln b = \ln(a/b)$ and $\ln(a^n) = n\ln a$ from Edexcel IAL P2/P3, applied in a C34-style volume-of-revolution or definite-integral context.
