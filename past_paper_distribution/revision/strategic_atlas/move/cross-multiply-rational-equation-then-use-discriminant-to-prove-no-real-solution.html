Algebra & Functions

Cross-multiply rational equation, then use discriminant to prove no real solutions

Family: Discriminant As Proof Frequency: 1 questions

Trigger — When to use

Fires when a question asks to show that two rational expressions f(x) = p(x)/q(x) and g(x) = r(x)/s(x) never meet, or that an equation like a/x = b/(x−c) has no real solutions. Typical phrasings: 'show that f(x) = g(x) has no real solutions', 'prove the curves do not intersect', 'show there is no value of x for which …'. The presence of two fractions equated, combined with a 'show'/'prove' instruction, is the key cue. If the resulting quadratic factorises easily, this move is unnecessary — discriminant is reserved for the non-factorisable case.

Motivation — Why this works

Cross-multiplication clears both denominators in one step and converts the rational equation into a polynomial equation, usually quadratic. Once in the form ax² + bx + c = 0, the discriminant Δ = b² − 4ac is the canonical Edexcel tool for proving the existence (or non-existence) of real roots without solving. Δ < 0 is a complete, rigorous algebraic proof that no real x satisfies the equation, which is stronger than a graphical or numerical argument and is what the mark scheme rewards.

Thought Process — The step-by-step reasoning

  1. Notice two rational expressions are being equated and the instruction is 'show'/'prove' rather than 'solve' — so the answer is qualitative (no solutions / one solution / two solutions).
  2. Rule out solving directly: there is no obvious factorisation, and graphical/numerical reasoning will not earn proof marks at A-level.
  3. Commit to the discriminant route because it is the standard algebraic certificate for 'no real roots' once a quadratic is in hand.
  4. Bridge: cross-multiply to clear fractions in one stroke, then collect into ax² + bx + c = 0 standard form.
  5. Tactically, identify a, b, c carefully (watch signs), compute Δ = b² − 4ac, and state the implication Δ < 0 ⇒ no real roots ⇒ no intersection. Optionally note that the excluded x-values from the original denominators do not affect the conclusion.

Mechanic — Concrete operations

  1. Set the two expressions equal: p(x)/q(x) = r(x)/s(x).
  2. Cross-multiply: p(x)·s(x) = r(x)·q(x).
  3. Expand and rearrange to standard quadratic form ax² + bx + c = 0.
  4. Identify a, b, c and compute Δ = b² − 4ac.
  5. Conclude from the sign of Δ: Δ < 0 ⇒ no real solutions; Δ = 0 ⇒ exactly one; Δ > 0 ⇒ two distinct.
  6. (If needed) verify excluded denominator values do not coincide with would-be solutions.

Worked Micro-Example

Show that $\dfrac{3x}{x-2} = \dfrac{4}{x}$ has no real solutions. Cross-multiplying: $3x \cdot x = 4(x-2)$, so $3x^2 = 4x - 8$, giving $3x^2 - 4x + 8 = 0$. Then $\Delta = (-4)^2 - 4(3)(8) = 16 - 96 = -80 < 0$. Since $\Delta < 0$, the quadratic has no real roots, so the original equation has no real solutions.

Variants — Same move, different surface

Line and curve non-intersection: show y = mx + c does not meet y = f(x).
Substitute to eliminate y, rearrange to a quadratic in x, then show Δ < 0.
Tangency proof: show a line is tangent to a curve.
Same setup, but conclude Δ = 0 (exactly one repeated root).
Two distinct intersections required.
Show Δ > 0 instead, possibly leading to an inequality in a parameter.
Quadratic in a parameter k (e.g. 'find values of k for which …').
Treat Δ as a function of k and solve the resulting inequality Δ < 0, = 0, or > 0.
Equation reduces to a polynomial of degree > 2 after clearing fractions.
Discriminant is not directly applicable; use factorisation, the factor theorem, or calculus/sketch arguments instead.

Common Misreadings

  • Treating the question as 'solve' and applying the quadratic formula, then being confused by √(negative) instead of stopping at Δ.
  • Reading 'no real solutions' as 'no solutions at all' and worrying about complex roots, which are out of scope here.
  • Confusing 'no intersection' with 'curves are parallel' and trying gradient arguments instead of the discriminant.

Pitfalls

  • Sign error in cross-multiplication, e.g. writing 4(x−2) as 4x + 8, which flips the sign of c and can falsely yield Δ ≥ 0.
  • Misidentifying a, b, c (especially the sign of b) when computing Δ.
  • Forgetting to rearrange to the form ax² + bx + c = 0 before reading off coefficients.
  • Failing to state the conclusion explicitly: a complete answer needs 'Δ < 0, therefore no real roots, therefore no real solutions'.
  • Ignoring domain restrictions x ≠ 0, x ≠ 2 from the original denominators (usually harmless here but should be acknowledged in a rigorous answer).

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P1/P2): manipulation of rational expressions and the discriminant of a quadratic as a test for the nature of roots. No complex numbers or higher-degree theory required.
