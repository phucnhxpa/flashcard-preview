Trigonometry

Use reference angle and CAST quadrants to enumerate all trig solutions

Family: Cast Diagram Analysis Frequency: 3 questions

Trigger — When to use

Fires when you have reduced a trig equation to a basic form sin x = k, cos x = k, or tan x = k (often after factorising a quadratic in sin/cos/tan or after isolating a single ratio) AND are required to give ALL solutions in a specified range such as 0° ≤ x < 360°, 0 ≤ θ < 2π, or [−180°, 180°). Recognise it whenever the calculator's inverse function will give only ONE value but the range spans a full period (so two or more answers are expected). Equivalent phrasings: 'find all values of x in the interval', 'solve for θ in (0, 2π)', 'state all solutions to 1 d.p.'.

Motivation — Why this works

Inverse trig functions on a calculator return only the principal value (arcsin ∈ [−90°, 90°], arccos ∈ [0°, 180°], arctan ∈ (−90°, 90°)), but each of sin, cos, tan takes any non-extreme value at exactly two angles per 360° period. The CAST diagram encodes which quadrants give positive/negative values for each ratio, and the reference (acute) angle α = arc(|k|) measures the angle from the nearest x-axis. Combining the two — pick the quadrants that match the sign of k, then place α inside each — generates every solution in the range systematically without missing or double-counting. This is the standard A-level mechanism for converting a single calculator output into a full solution set.

Thought Process — The step-by-step reasoning

  1. Notice the equation is now in the form (trig ratio) = k with k a specific number, and the question demands ALL solutions in a stated interval that spans (at least) one full period.
  2. Rule out just typing arc-(ratio)(k) and stopping: that gives one value at most, and quadratic-derived problems typically expect 2 or 4 solutions — losing half the marks otherwise.
  3. Commit to the reference-angle + CAST method because it works uniformly for any sign of k and for any of sin/cos/tan, and it scales to non-standard ranges by adding/subtracting full periods.
  4. Compute the acute reference angle α = arc(ratio)(|k|) using |k| (always positive), so α lies in (0°, 90°). Keep α to full calculator precision; do not round yet.
  5. Use the sign of k with CAST to pick the two quadrants: sin>0 → Q1,Q2; sin<0 → Q3,Q4; cos>0 → Q1,Q4; cos<0 → Q2,Q3; tan>0 → Q1,Q3; tan<0 → Q2,Q4.
  6. Place α inside each chosen quadrant via the standard formulas: Q1: α; Q2: 180°−α; Q3: 180°+α; Q4: 360°−α (in radians replace 180° by π and 360° by 2π).
  7. Check every produced value lies in the required interval, then round to the requested precision only at the final step.

Mechanic — Concrete operations

  1. Identify the ratio (sin/cos/tan), the value k, and the required interval.
  2. Compute α = arc(ratio)(|k|) on the calculator in the correct mode (degrees or radians) — keep full precision.
  3. Apply CAST to determine the two quadrants matching the sign of k.
  4. Write the two (or more) solutions using α and the quadrant formulas: Q1 = α, Q2 = 180°−α, Q3 = 180°+α, Q4 = 360°−α (or π/2π in radians).
  5. If the equation arose from a quadratic giving two values of the ratio, repeat the procedure for each value.
  6. Confirm each answer lies within the stated interval; discard or shift by ±360° (±2π) any that don't.
  7. Round to the specified precision (typically 1 d.p. in degrees, 3 s.f. in radians) at the end only.

Worked Micro-Example

Solve $\sin x = -\tfrac{1}{3}$ for $0° \le x < 360°$. Reference angle: $\alpha = \arcsin(\tfrac{1}{3}) \approx 19.471°$. Sine is negative ⇒ Q3 and Q4. Q3: $x = 180° + 19.471° \approx 199.5°$. Q4: $x = 360° - 19.471° \approx 340.5°$. Both lie in $[0°, 360°)$, so $x \approx 199.5°,\ 340.5°$.

Variants — Same move, different surface

cos x = k with k negative (e.g. cos x = −1/4)
α = arccos(|k|); solutions in Q2 and Q3: x = 180°−α and x = 180°+α (equivalently the principal value cos⁻¹(k) and 360° minus it).
tan x = k
Solutions are 180° (or π) apart: x = α and x = 180°+α for k>0; x = 180°−α and x = 360°−α for k<0. Only one quadrant pair, period 180°.
Range is (0, 2π) in radians
Use π and 2π in the quadrant formulas; ensure calculator is in radian mode; round to 3 s.f. typically.
Range is [−180°, 180°)
Same reference angle α, but rewrite Q3/Q4 outputs by subtracting 360°: e.g. Q3 → α − 180°, Q4 → α − 360° (or equivalently −α).
Argument is a transformed angle, e.g. sin(2x) = k on 0° ≤ x < 360°
Solve sin(u) = k on the expanded range 0° ≤ u < 720° first (apply the move and add full periods), then divide each u by 2.

Common Misreadings

  • Reading arcsin(−1/3) directly off the calculator as ≈ −19.47° and stopping there, missing the in-range partners.
  • Using sine symmetry (180° − x) for a cosine equation, or vice versa.
  • Believing the principal value is always in Q1 — for cos x = negative the principal value is already in Q2, so adding 180° to it lands outside the valid quadrant pair.

Pitfalls

  • Computing α with the signed value k instead of |k|, producing a negative reference angle and corrupted quadrant placements.
  • For sin x = 1/2 writing 180° + 30° = 210° (Q3, sine negative) instead of 180° − 30° = 150° (Q2, sine positive).
  • Calculator left in the wrong angle mode (degrees vs radians).
  • Rounding α to 1 d.p. before forming 180°+α and 360°−α, accumulating error in the final answers.
  • Forgetting to repeat the whole procedure for the second root of a quadratic in sin/cos/tan, returning only 2 solutions when 4 are required.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P2/P3) and Further Pure scope; uses only the CAST diagram / unit-circle symmetries and the inverse trig functions on a calculator — no formula-booklet entry is required. Do not introduce general solution formulas like x = nπ + (−1)ⁿα; A-level expects explicit enumeration in a stated finite interval.
