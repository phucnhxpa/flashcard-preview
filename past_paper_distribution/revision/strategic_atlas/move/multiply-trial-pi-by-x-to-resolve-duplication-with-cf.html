Trigger — When to use

Fires immediately after writing down the standard trial particular integral for a non-homogeneous linear ODE, when the trial shares a functional form with any term of the complementary function. Typical cues: RHS is $ke^{mx}$ where $m$ is a root of the auxiliary equation; RHS is $\cos\omega x$ or $\sin\omega x$ where $\pm i\omega$ are roots; RHS is a polynomial when $0$ is a root of the auxiliary equation. Phrased differently in mark schemes as 'since $e^{mx}$ already appears in the CF, try $\lambda x e^{mx}$'.

Motivation — Why this works

Any function in the CF satisfies the homogeneous equation, so substituting it into the LHS produces $0$, which cannot equal a nonzero RHS. Multiplying by $x$ generates a function linearly independent of the CF whose substitution leaves a nonzero residue proportional to the original exponential/trig form, allowing coefficients to be matched. If the auxiliary equation has a repeated root $m$ and the RHS still resonates, both $e^{mx}$ and $xe^{mx}$ lie in the CF, forcing a further $x^2$ multiplication. This is the standard resonance fix for constant-coefficient linear ODEs.

Thought Process — Step-by-step reasoning

  1. Notices that the natural trial PI (e.g. $\lambda e^{2x}$) is already a term of the CF written above.
  2. Rules out simply proceeding with the substitution, because LHS would collapse to $0$ and the coefficient-matching system would be inconsistent.
  3. Commits to the resonance rule: multiply the entire trial by the smallest power of $x$ needed to remove all overlap with the CF.
  4. Bridges to operation by rewriting the trial as $x\lambda e^{2x}$ (or $x^2\lambda e^{2x}$ for repeated roots), then differentiating using the product rule.
  5. Tactically decides whether $x$ suffices or $x^2$ is required by re-checking the new trial against every CF term, including repeated-root partners like $xe^{mx}$.

Mechanic — Concrete operations

  1. Write the CF (from the auxiliary equation) and the standard trial PI side by side.
  2. Compare functional forms, ignoring arbitrary constants and coefficients.
  3. If any overlap exists, replace the trial by $x \times$ (original trial).
  4. Re-check; if overlap persists (repeated-root case), replace by $x^2 \times$ (original trial).
  5. Differentiate the corrected trial using the product rule, substitute into the ODE, and equate coefficients to find the unknown constants.

Worked Micro-Example

Solve $y'' - 3y' + 2y = 4e^{2x}$. Auxiliary: $m^2-3m+2=0 \Rightarrow m=1,2$, so CF $= Ae^{x}+Be^{2x}$. Standard trial $\lambda e^{2x}$ duplicates the CF, so use $y_p = \lambda x e^{2x}$. Then $y_p' = \lambda(1+2x)e^{2x}$, $y_p'' = \lambda(4+4x)e^{2x}$; substituting gives $\lambda e^{2x} = 4e^{2x}$, so $\lambda=4$ and $y_p = 4xe^{2x}$.

Variants — Same move, different surface

Repeated root in CF with matching exponential RHS, e.g. CF $=(A+Bx)e^{mx}$, RHS $=ke^{mx}$
Both $e^{mx}$ and $xe^{mx}$ are in the CF, so use trial $\lambda x^2 e^{mx}$.
Trig RHS resonating with CF, e.g. CF contains $\cos 3x, \sin 3x$ and RHS $=\cos 3x$
Use trial $x(\lambda\cos 3x + \mu\sin 3x)$ — both sine and cosine must be carried through.
Polynomial RHS when $m=0$ is a root of the auxiliary equation (e.g. $y''-y'=x$)
Constants are in the CF, so multiply the polynomial trial by $x$: trial $=x(ax+b)$ instead of $ax+b$.
Mixed RHS like $e^{mx}\cos\omega x$ where $m\pm i\omega$ are CF roots
Multiply the full trial $e^{mx}(\lambda\cos\omega x+\mu\sin\omega x)$ by $x$.

Common Misreadings

  • Thinking the duplication is fine because the CF coefficient is arbitrary and the PI coefficient is to be determined — they live on different sides of the equation but the LHS still annihilates the form.
  • Believing only the matching exponential needs the $x$ factor while leaving sine/cosine partners unchanged in trig cases.
  • Treating $x^2$ as the default fix; $x$ is the minimal power and should be tried first.

Pitfalls

  • Forgetting to check for duplication, leading to an inconsistent linear system when equating coefficients.
  • Multiplying only part of the trial by $x$ (e.g. $\lambda x\cos 3x + \mu\sin 3x$ instead of $x(\lambda\cos 3x+\mu\sin 3x)$).
  • Mis-differentiating $xe^{mx}$ by dropping the product-rule term.
  • Not recognising duplication when the CF term has a different constant multiple, since the form — not the coefficient — is what matters.

A-level Scope Notes

Standard technique for second-order linear constant-coefficient ODEs in Edexcel IAL Further Pure (FP2). Trial forms and the $x$/$x^2$ resonance rule are explicitly examinable; higher-order or variable-coefficient cases are out of scope.
