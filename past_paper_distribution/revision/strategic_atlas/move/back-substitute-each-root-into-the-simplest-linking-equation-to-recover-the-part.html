Algebra & Functions

Back-substitute each root into the simplest linking equation to recover the partner parameter

Family: Parameter Recovery Frequency: 1 questions

Trigger — When to use

Fires after solving for one parameter (e.g. a common ratio r, a gradient m, or an x-coordinate) when the question still asks for a paired parameter (a, c, or y). Cue phrases include 'find the possible values of a and r', 'hence find...', or any system where multiple roots have been retained. Especially relevant when two or more roots survived the discriminant/validity check and each yields a distinct solution pair. The move is signalled whenever an earlier line of working already contains a clean explicit relation linking the two unknowns.

Motivation — Why this works

An earlier step typically produced a one-line relation such as a = 16/r^2 that expresses one parameter directly in terms of the other. Re-using it avoids re-solving the whole system and minimises arithmetic risk. Because each surviving root is a legitimate configuration, every root must be processed to obtain the full solution set the question demands. Presenting all pairs explicitly is what secures the final accuracy marks.

Thought Process — The step-by-step reasoning

  1. Notice that the unknown still required (a) is already expressed in closed form via a previously derived equation, so no fresh system needs solving.
  2. Rule out re-substituting into the original messy simultaneous equations or recomputing d, since the question only asks for a and r.
  3. Commit to the cleanest linking equation (here a = 16/r^2) because it isolates a in one step and reduces error surface.
  4. Bridge: list each surviving root of r as a separate case to keep the bookkeeping transparent.
  5. Tactically, square the fraction carefully and present the two pairs (a, r) explicitly so the marker sees both configurations.

Mechanic — Concrete operations

  1. Identify the simplest equation linking the two parameters from earlier working.
  2. Label each surviving root as a separate case (Case 1, Case 2, ...).
  3. Substitute the root into the linking equation and simplify.
  4. Record the resulting (a, r) pair for that case.
  5. Repeat for every surviving root and present all pairs together as the final answer.

Worked Micro-Example

From earlier work $a = \dfrac{16}{r^2}$ and the quadratic in $r$ gave $r = 4$ or $r = \tfrac{4}{5}$.\\ Case 1: $r = 4 \Rightarrow a = \dfrac{16}{16} = 1$.\\ Case 2: $r = \tfrac{4}{5} \Rightarrow a = \dfrac{16}{16/25} = 25$.\\ Solution pairs: $(a,r) = (1,4)$ or $(25, \tfrac{4}{5})$.

Variants — Same move, different surface

Coordinate geometry: line meets curve at two points
After solving the quadratic for x, substitute each x back into the linear equation (the simpler one) to obtain the matching y.
Trigonometric equation reducible to a quadratic in sin x or cos x
After finding each value of sin x (or cos x), back-substitute into x = arcsin(...) and use the symmetry/periodicity rules to list all angles in range.
Simultaneous equations involving a parameter k with two valid k-values
For each k, substitute into the simpler defining relation (often a linear constraint) to recover the partner unknown.
Mixed AP/GP problems where r and a are linked
Use the GP term equation rather than the AP sum equation when isolating a, since it usually gives a directly.
Logarithmic equations reduced to a quadratic in y = log x
After solving for each y, back-substitute via x = b^y to recover x for every root.

Common Misreadings

  • Assuming only one root is 'the answer' and discarding the other without checking validity.
  • Reading the question as asking for r only, when 'values of a and r' (plural) is required.
  • Confusing this step with a discriminant check — both roots here are valid, no rejection is needed.

Pitfalls

  • Substituting into a more complex equation (e.g. the original sum equation) rather than the clean isolated form.
  • Arithmetic error when squaring a fraction: $(4/5)^2 = 16/25$, not $8/25$ or $4/25$.
  • Forgetting to simplify $16 \div (16/25) = 25$ correctly (dividing by a fraction = multiplying by its reciprocal).
  • Presenting the two a-values without pairing them to the correct r, losing accuracy marks.

A-level Scope Notes

Entirely within Edexcel IAL Pure scope (P1/P2 sequences and series, simultaneous equations); the technique generalises to coordinate geometry and trigonometric equations covered in the same syllabus.
