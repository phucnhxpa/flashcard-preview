Complex Numbers

Apply de Moivre's theorem with +2kπ to enumerate all n distinct nth roots

Family: De Moivre Roots Frequency: 3 questions

Trigger — When to use

An equation of the form z^n = w (with n ≥ 2) where w is a known complex number, and the question asks for ALL roots/solutions — not just one. Visual cues: w is given (or easily put) in polar form r·cis(θ) or exponential form re^{iθ}; the question phrases like 'find the three cube roots', 'solve in the complex plane', 'express each root in the form a+bi'. Equivalent phrasings include 'find all z such that z^n = w' or '(z−c)^n = w'. The presence of an exponent n and the demand for multiple solutions is the core signal.

Motivation — Why this works

A complex number's argument is only defined modulo 2π, so z^n = re^{iθ} is really z^n = re^{i(θ + 2kπ)} for every integer k. Taking the nth root spreads these previously-equivalent arguments into n genuinely distinct values (θ + 2kπ)/n before the cycle repeats at k = n. This delivers exactly the n roots guaranteed by the Fundamental Theorem of Algebra, all on a circle of radius r^{1/n} and equally spaced by 2π/n. The technique exploits trig periodicity to produce every root systematically without factorisation.

Thought Process — The step-by-step reasoning

  1. I notice the equation has the shape z^n = w with w expressible in polar form, and the question demands all n solutions, not just one.
  2. I rule out direct factorisation (z^n − w as a polynomial) — for n ≥ 3 this is messy; I also rule out a single application of z = w^{1/n}, which yields only the principal root and would lose marks.
  3. I commit to de Moivre because it is the standard mechanism: writing arg(w) as θ + 2kπ creates the additive freedom that, once divided by n, produces n distinct arguments.
  4. Bridge: I rewrite w in polar/exponential form r·cis(θ) (or re^{iθ}), then write the general root z_k = r^{1/n}·cis((θ + 2kπ)/n).
  5. Tactical decision on k: I let k run over n consecutive integers. I choose k = 0, 1, …, n−1 by default, but if the answer must lie in the principal range (−π, π], I shift to a symmetric set (e.g. k = −1, 0, 1 for n = 3) so each argument is already principal and no post-adjustment is needed.
  6. Finally, I simplify each argument and, if standard angles appear (multiples of π/6, π/4, π/3), convert to exact Cartesian form a + bi.

Mechanic — Concrete operations

  1. Express w in polar form: w = r·cis(θ) = r e^{iθ}, identifying r = |w| and θ = arg(w).
  2. Write the general nth root: z_k = r^{1/n}\bigl[\cos\tfrac{θ + 2kπ}{n} + i\sin\tfrac{θ + 2kπ}{n}\bigr] = r^{1/n} e^{i(θ + 2kπ)/n}.
  3. Choose n consecutive integer values of k (typically k = 0, 1, …, n−1, or a symmetric set if principal arguments are required).
  4. Substitute each k, simplify the compound fraction (θ + 2kπ)/n, and reduce into the requested argument range (often (−π, π]).
  5. Confirm all roots share modulus r^{1/n}; if asked, convert to a + bi using exact trig values.
  6. Sanity check: the n roots are equally spaced by 2π/n around the circle of radius r^{1/n}.

Worked Micro-Example

Solve $z^3 = 8e^{-i 2\pi/3}$. Write $z_k = 8^{1/3} e^{i(-2\pi/3 + 2k\pi)/3} = 2 e^{i(-2\pi + 6k\pi)/9}$. Taking $k = 0, 1, -1$ gives $z_0 = 2 e^{-i 2\pi/9}$, $z_1 = 2 e^{i 4\pi/9}$, $z_{-1} = 2 e^{-i 8\pi/9}$ — three distinct roots, all with modulus 2, all in $(-\pi, \pi]$.

Variants — Same move, different surface

Roots of unity: z^n = 1
Set r = 1, θ = 0; formula collapses to z_k = cis(2kπ/n), all on the unit circle, equally spaced.
Shifted equation (z − c)^n = w
Apply de Moivre to find z − c, then translate each root by adding c.
Equation given in degrees
Use (θ + 360°k)/n in place of (θ + 2kπ)/n — identical logic in degree measure.
Square roots (n = 2)
Only two roots, diametrically opposite on the circle — k = 0, 1 suffice.

Common Misreadings

  • Treating cube roots like real cube roots and producing only one solution.
  • Confusing '+ 2kπ' with '+ 2kπ/n' — the 2kπ is added to θ BEFORE dividing by n.
  • Using k = 1, 2, …, n instead of k = 0, 1, …, n−1, generating a duplicate (k = n repeats k = 0) and missing a root, or producing arguments outside the principal range.

Pitfalls

  • Forgetting to take the nth root of the modulus (e.g. writing 8 instead of 2 for cube roots of 8e^{iθ}).
  • Arithmetic slips when simplifying compound fractions like (π + 2kπ)/4 across multiple k values.
  • Failing to reduce arguments into the required principal range — e.g. leaving 5π/4 when −3π/4 is expected.
  • Forgetting that all n roots share the same modulus r^{1/n}; varying the modulus by mistake.

A-level Scope Notes

This is core Edexcel IAL Further Pure (FP2) material on de Moivre's theorem and nth roots of a complex number; the formula z_k = r^{1/n}·cis((θ + 2kπ)/n) appears in the formula booklet. Stay within finite n and explicit polar/exponential forms — avoid invoking general field-theoretic statements about roots or complex logarithm branches, which are out of scope.
