Algebra & Functions

Impose discriminant strictly less than zero to enforce no real intersections

Family: Discriminant Condition Frequency: 3 questions

Trigger — When to use

Fires whenever a problem reduces to a quadratic Ax² + Bx + C = 0 (in x) whose coefficients depend on a parameter k (or c, m, etc.) and the question asks for parameter values such that there are NO real solutions. Typical surface phrasings: 'the line does not cross or touch the curve', 'the curve and line do not intersect', 'the equation has no real roots', 'show that … has no real solutions for any value of k'. The key visual cue is a quadratic in one variable with a parameter sitting inside B and/or C, paired with a non-existence demand. The question must EXCLUDE tangency (otherwise a non-strict inequality would be wrong).

Motivation — Why this works

For a genuine quadratic Ax² + Bx + C = 0 with A ≠ 0, the discriminant Δ = B² − 4AC controls root existence: Δ > 0 gives two distinct real roots, Δ = 0 gives a repeated (tangent) root, Δ < 0 gives no real roots. So 'no real intersection / no real root' is exactly equivalent to Δ < 0. This converts a geometric or existence question into a pure algebraic inequality in the parameter, which can then be solved by standard quadratic-inequality techniques. The strict inequality is essential because Δ = 0 still produces a real (repeated) root, i.e. tangency, which counts as 'touching'.

Thought Process — The step-by-step reasoning

  1. Notice the structural cue: after substituting / equating, you get a quadratic in x whose B and/or C depend on the parameter, and the question demands that this equation have NO real solutions (or equivalently, that the line and curve do not meet).
  2. Rule out alternatives: solving for x explicitly is impossible because the parameter is unknown; factorising is unavailable; trying specific values of k only tests, never proves. Hence existence of roots must be controlled by the discriminant.
  3. Commit to Δ < 0 after carefully reading the wording. Reject Δ ≤ 0 (would allow tangency / repeated root, which IS a real root and IS 'touching'); reject Δ > 0 (that gives the OPPOSITE set of parameter values); reject Δ ≥ 0 (would mean 'crosses or touches').
  4. Check the equation is genuinely quadratic: confirm the leading coefficient A ≠ 0 (and ideally is independent of the parameter, so no degenerate linear case). If A could vanish for some parameter value, that case must be handled separately.
  5. Identify A, B, C carefully — beware notational clashes (e.g. when the parameter is itself called c, do NOT confuse it with the constant term in the formula b² − 4ac).
  6. Write down B² − 4AC < 0, then expand and simplify to a quadratic inequality in the parameter, ready for the next step (factorise / complete the square / sketch).

Mechanic — Concrete operations

  1. From the quadratic Ax² + Bx + C = 0, read off A, B, C as expressions in the parameter (e.g. parameter k).
  2. Write the inequality B² − 4AC < 0.
  3. Expand B² fully, taking care with cross terms — e.g. (2 − 2k)² = 4 − 8k + 4k², not 4 − 4k².
  4. Subtract 4AC and collect like terms to get a quadratic inequality in the parameter, e.g. αk² + βk + γ < 0.
  5. Hand off to the next move (factorise / use quadratic formula on the parameter / sketch parabola) to read off the parameter range.

Worked Micro-Example

Find k such that y = kx + 1 does not meet y = x² + 3. Equate: x² − kx + 2 = 0. Here A = 1, B = −k, C = 2. No real intersection ⇔ Δ < 0: $(-k)^2 - 4(1)(2) < 0$, i.e. $k^2 - 8 < 0$, giving $-2\sqrt{2} < k < 2\sqrt{2}$.

Variants — Same move, different surface

Question asks for tangency ('the line is a tangent to the curve')
Replace Δ < 0 with Δ = 0 and solve as an equation, not an inequality.
Question asks for 'two distinct points of intersection'
Use Δ > 0 instead.
Question asks for 'crosses or touches' / 'at least one real root'
Use Δ ≥ 0.
The leading coefficient A itself depends on the parameter
Split into cases: handle A = 0 (linear) separately, then apply Δ < 0 only when A ≠ 0.
Equation arises after clearing a fraction by multiplying by x
Check x = 0 is not introduced as a spurious root (i.e. confirm the constant term is non-zero) before applying the discriminant.

Common Misreadings

  • Reading 'does not cross' as allowing tangency, then using Δ ≤ 0 instead of Δ < 0.
  • Reading 'no real roots' as 'roots are not real numbers we like' and accidentally using Δ > 0 — i.e. solving the complementary problem.
  • When the parameter is named c, treating it as the constant term in 'b² − 4ac' and double-counting it.
  • Forgetting that 'touch' = tangent = repeated root, which is a real root and so must be excluded by a strict inequality.

Pitfalls

  • Algebra slip when expanding (B)² with a binomial parameter — e.g. (2 − 2k)² and (4 − k)² are common error sites.
  • Sign error when subtracting 4AC, especially if C itself is negative or contains the parameter.
  • Dividing the resulting inequality by a negative number without flipping the inequality sign.
  • Failing to verify A ≠ 0, so missing a degenerate linear sub-case where the discriminant test does not apply.
  • Stopping at Δ < 0 without solving the resulting quadratic inequality in the parameter to give the final range.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P1 quadratics, P2/P3 line–curve intersection). The discriminant b² − 4ac is standard bookwork and not given in the formula booklet; students are expected to know it. Avoid invoking complex roots beyond stating 'no real roots' — discussion of complex conjugate pairs belongs to Further Pure, not standard P-unit solutions.
