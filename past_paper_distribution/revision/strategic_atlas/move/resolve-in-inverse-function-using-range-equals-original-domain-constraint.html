Algebra & Functions

Resolve ± in inverse function using range-equals-original-domain constraint

Family: Sign Selection Via Domain Frequency: 1 questions

Trigger — When to use

Fires whenever inverting a function leads to a squared expression and taking a square root introduces ±. Typical surfaces: $f(x) = a \pm b/(x+c)^2$, restricted-domain quadratics like $f(x) = (x-h)^2 + k$ on $x \ge h$, or expressions involving $\sqrt{\,\cdot\,}$ that have been squared during rearrangement. The cue is: 'find $f^{-1}(x)$' on a function with a stated restricted domain, and the algebra produces $(\text{something})^2 = \ldots$.

Motivation — Why this works

An inverse function must be single-valued, so the ± from a square root must be resolved. The defining identity is range$(f^{-1})$ = domain$(f)$: every output of $f^{-1}$ must lie in the original domain of $f$. Substituting the boundary of that domain into the candidate expression for $y$ (or reading off the sign of $y+c$ directly from the domain inequality) forces exactly one branch. Without this step the inverse is ambiguous and the answer loses meaning.

Thought Process — The step-by-step reasoning

  1. Notice that rearranging produced a perfect square, so taking the root introduces a ± that must be pinned down.
  2. Rule out leaving ± in the final answer — an inverse function must be single-valued.
  3. Commit to the principle: range of $f^{-1}$ equals domain of $f$, so any output $y = f^{-1}(x)$ must satisfy the original domain restriction.
  4. Translate the domain restriction into a sign statement about the bracket that was squared (e.g. $x \ge 0 \Rightarrow y+3 \ge 3 > 0$).
  5. Select the matching sign of the square root, then finish the rearrangement to isolate $y$.

Mechanic — Concrete operations

  1. Set $y = f^{-1}(x)$ and rearrange $x = f(y)$ to isolate the squared bracket, e.g. $(y+3)^2 = \tfrac{4}{5-x}$.
  2. Take square roots to get $y + 3 = \pm \tfrac{2}{\sqrt{5-x}}$.
  3. State that range$(f^{-1})$ = domain$(f)$ and translate this into a sign condition on the bracket (here $y \ge 0 \Rightarrow y+3 > 0$).
  4. Discard the branch incompatible with that sign and keep the valid one.
  5. Solve for $y$ to obtain the explicit inverse, e.g. $f^{-1}(x) = \tfrac{2}{\sqrt{5-x}} - 3$.

Worked Micro-Example

Let $f(x) = 5 - \tfrac{4}{(x+3)^2}$, $x \ge 0$. Set $x = 5 - \tfrac{4}{(y+3)^2} \Rightarrow (y+3)^2 = \tfrac{4}{5-x}$, so $y+3 = \pm \tfrac{2}{\sqrt{5-x}}$. Since range$(f^{-1})$ = $[0,\infty)$, we need $y \ge 0$, hence $y+3 \ge 3 > 0$, forcing the $+$ branch. Therefore $f^{-1}(x) = \tfrac{2}{\sqrt{5-x}} - 3$.

Variants — Same move, different surface

Inverting a restricted quadratic, e.g. $f(x) = (x-2)^2 + 1$ on $x \ge 2$.
Rearrange to $(y-2)^2 = x - 1$, so $y - 2 = \pm\sqrt{x-1}$; the domain $y \ge 2$ forces the $+$ root, giving $f^{-1}(x) = 2 + \sqrt{x-1}$.
Same quadratic but on $x \le 2$.
Now $y \le 2 \Rightarrow y - 2 \le 0$, so the $-$ root is selected: $f^{-1}(x) = 2 - \sqrt{x-1}$.
Function of the form $f(x) = a + b\sqrt{x-c}$ inverted by squaring.
Squaring is reversible without ± here, but if rearrangement squares both sides one must check that the candidate $y$ lies in the original domain $x \ge c$ to discard extraneous solutions.
Trig inverse on a restricted interval, e.g. defining $f(x) = \sin x$ on $[-\pi/2, \pi/2]$.
The principal-value range of $\arcsin$ is exactly the original domain, illustrating the same range-equals-domain principle that resolves which branch of the inverse is taken.

Common Misreadings

  • Leaving $\pm$ in the final expression for $f^{-1}(x)$, treating the inverse as multi-valued.
  • Picking the positive root by default without citing the domain of $f$.
  • Confusing domain and range of $f^{-1}$, and using the wrong inequality to choose the sign.

Pitfalls

  • Forgetting to state explicitly 'range of $f^{-1}$ = domain of $f$', which is usually the reasoning mark.
  • Translating the domain inequality incorrectly, e.g. concluding $y+3 \ge 0$ when actually $y+3 \ge 3$, which is harmless here but matters when $0$ is in the bracket's range.
  • Squaring during rearrangement and not checking whether the resulting branch is consistent with the original domain, introducing extraneous solutions.

A-level Scope Notes

Standard Edexcel IAL P2/P3 inverse function technique; uses only the definitions domain$(f^{-1})$ = range$(f)$ and range$(f^{-1})$ = domain$(f)$, with no calculus required.
