Strategic Atlas / Proof / Sign Change Ivt
Proof

Evaluate at endpoints and invoke IVT sign-change for root existence

Family: Sign Change Ivt Frequency: 1 questions

Trigger — When to use

Fires whenever a question says 'show that f(x) = 0 has a root in [a, b]' or 'show that the equation has a solution between x = a and x = b', without asking you to solve it. Also fires when justifying that an iteration converges to a root in a given interval, or when verifying a root to a specified number of decimal places (interval [α − 0.5×10⁻ⁿ, α + 0.5×10⁻ⁿ]). The hallmark is a closed interval with named numerical endpoints and an equation that cannot be solved algebraically.

Motivation — Why this works

If f is continuous on [a, b] and f(a) and f(b) have opposite signs, the Intermediate Value Theorem guarantees at least one c ∈ (a, b) with f(c) = 0. At A-level this is the only rigorous tool for asserting existence of a root in a specified interval without explicit solution. The argument relies on continuity of f on [a, b] and on a strict change of sign — both must be stated explicitly to earn full marks.

Thought Process — The step-by-step reasoning

  1. Notices the phrase 'show there is a root in [a, b]' — instant cue for sign-change/IVT.
  2. Rules out solving f(x) = 0 directly; the equation mixes terms (e.g. negative powers and roots) that make algebraic solution impossible at A-level.
  3. Commits to the IVT route because it is the only rigorous existence argument available.
  4. Bridges by setting up f(a) and f(b) for numerical evaluation, keeping enough decimal places to be sure of the sign.
  5. Tactically packages the conclusion as a four-part statement: numerical values with signs, sign change, continuity on the interval, therefore root exists by IVT.

Mechanic — Concrete operations

  1. Define f(x) by rearranging the equation to f(x) = 0 form if not already given.
  2. Compute f(a) to sufficient accuracy and record its sign.
  3. Compute f(b) to sufficient accuracy and record its sign.
  4. State explicitly that there is a sign change between f(a) and f(b).
  5. State that f is continuous on [a, b] (justify briefly if necessary).
  6. Conclude: by the Intermediate Value Theorem, there exists α ∈ (a, b) with f(α) = 0.

Worked Micro-Example

Let $f(x) = \dfrac{7}{x} - \dfrac{1}{x^{2}} - \dfrac{5}{3}\sqrt[3]{x}$. Then $f(2.8) \approx 0.142 > 0$ and $f(2.9) \approx -0.849 < 0$. Since $f$ is continuous on $[2.8, 2.9]$ and changes sign, by the IVT there exists $\alpha \in (2.8, 2.9)$ with $f(\alpha) = 0$.

Variants — Same move, different surface

'Show the equation g(x) = h(x) has a solution in [a, b]'
Rearrange to f(x) = g(x) − h(x) = 0 first, then apply the same sign-change argument to f.
Verifying a root to n decimal places
Use the narrower interval [α − 0.5×10⁻ⁿ, α + 0.5×10⁻ⁿ] and show sign change there, concluding α is correct to n d.p.
Justifying convergence of an iteration xₙ₊₁ = F(xₙ) to a root
Apply the sign-change argument to f(x) = x − F(x) (or the equivalent rearrangement) on the stated interval.
Trigonometric or exponential equations on a bounded interval
Same template; just check continuity is automatic for sin, cos, eˣ, and polynomial composites on the interval.

Common Misreadings

  • Substituting only one endpoint and arguing from a single value — insufficient for IVT.
  • Omitting the explicit statement of continuity on [a, b], which is a required IVT hypothesis.
  • Treating the conclusion as 'exactly one root' rather than 'at least one root' — IVT alone gives existence, not uniqueness.
  • Forgetting to name the theorem (IVT / sign-change rule) in the conclusion.

Pitfalls

  • Arithmetic slips when evaluating f at endpoints, especially with cube roots, reciprocals, or trig values.
  • Over-rounding intermediate results so that the computed sign becomes ambiguous near zero.
  • Stating values without their signs, or stating signs without the numerical evidence.
  • Applying IVT on an interval where f has a discontinuity (e.g. crossing x = 0 for f involving 1/x).

A-level Scope Notes

Fully within Edexcel IAL scope: standard numerical methods topic in Core Pure / FP1, where IVT (sign-change) is the prescribed justification for root existence. No formal δ-ε continuity is required — a brief appeal to continuity of standard functions suffices.
