Trigger — When to use

The task asks for the maximum or minimum of a rational expression whose numerator and denominator are each linear in sin x and cos x, e.g. y = (p + q sin x)/(r + s cos x). Direct quotient-rule differentiation looks viable but produces a messy trig equation. A cue is when an earlier part of the question established an R sin(x + α) result or an amplitude bound √(a² + b²), suggesting reuse. Also fires whenever the question phrasing 'find the range of values' or 'least value' appears with a single rational trig expression.

Motivation — Why this works

Setting y equal to the expression and cross-multiplying converts the optimisation problem into an existence question: for which y does the equation a sin x + b cos x = c admit a real solution? The known solvability condition c² ≤ a² + b² then yields an inequality in y whose boundary values are the extrema. This sidesteps calculus entirely and leverages the amplitude bound, which is already secured at A-level.

Thought Process — Step-by-step reasoning

  1. Student notices the expression is a ratio where both top and bottom are linear in sin x and cos x, and that an earlier part likely produced an R-form bound.
  2. They rule out quotient-rule differentiation as algebraically heavy and prone to sign errors when setting the derivative to zero.
  3. They commit to the cross-multiply route because it converts the problem into 'when does p sin x + q cos x = c have a solution?', which is a one-line check using √(p² + q²).
  4. Bridge: write y = f(x), clear the denominator, and collect every sin x and cos x term on one side, treating y as a parameter.
  5. Tactical: identify the coefficients of sin x and cos x (one of which now contains y) and the constant (also in y), ready to apply c² ≤ a² + b².

Mechanic — Concrete operations

  1. Let y = (15 + 8 sin x)/(41 − 30 cos x).
  2. Cross-multiply to clear the denominator: y(41 − 30 cos x) = 15 + 8 sin x.
  3. Expand: 41y − 30y cos x = 15 + 8 sin x.
  4. Move all trig terms to one side: 8 sin x + 30y cos x = 41y − 15.
  5. Identify coefficients a = 8, b = 30y, c = 41y − 15 in the form a sin x + b cos x = c, ready for the amplitude bound a² + b² ≥ c².

Worked Micro-Example

Let $y = \dfrac{15 + 8\sin x}{41 - 30\cos x}$. Cross-multiplying: $41y - 30y\cos x = 15 + 8\sin x$, so $8\sin x + 30y\cos x = 41y - 15$. For real $x$ we need $(41y-15)^2 \le 8^2 + (30y)^2 = 64 + 900y^2$. Expanding gives $681y^2 - 1230y + 161 \le 0$, whose roots bound the range of $y$, delivering the minimum directly.

Variants — Same move, different surface

y = (a + b sin x)/(c + d sin x) with sin x only
Cross-multiply and collect to get (b − dy) sin x = cy − a, then use −1 ≤ sin x ≤ 1 instead of the full amplitude bound.
y = (a + b cos x)/(c + d cos x)
Same procedure produces a single cos x equation; bound via |cos x| ≤ 1.
y = (p sin x + q cos x)/(r + s sin x + t cos x) with full mix
Cross-multiply, collect all sin x and cos x terms; the resulting form is still a sin x + b cos x = c with a, b, c linear in y, so the amplitude bound a² + b² ≥ c² still applies.
Range of y rather than just minimum
Solve the quadratic-in-y inequality from a² + b² ≥ c² to get both endpoints simultaneously.

Common Misreadings

  • Attempting quotient-rule differentiation directly — technically valid but algebraically punishing and disconnected from the R-form set up in the earlier part.
  • Trying to substitute the R sin(x + α) result into the numerator or denominator without first cross-multiplying, producing a fraction that still needs optimising.
  • Treating y as a variable to differentiate with respect to rather than a parameter constrained by a solvability condition.

Pitfalls

  • Forgetting to bring the −30y cos x term across to join 8 sin x, leaving the equation in mixed form and unable to apply the amplitude bound.
  • Sign errors when distributing y across (41 − 30 cos x), especially flipping the sign of 30y cos x.
  • Misidentifying the coefficient of cos x as 30 instead of 30y when applying a² + b² ≥ c².
  • Forgetting that the bound is a² + b² ≥ c² (not equality), which would miss the inequality giving the range of y.

A-level Scope Notes

Uses only the harmonic-form amplitude bound √(a² + b²) and basic algebraic manipulation, both standard in Edexcel IAL P3/P4 and Further Pure. No calculus or Weierstrass substitution is required.
