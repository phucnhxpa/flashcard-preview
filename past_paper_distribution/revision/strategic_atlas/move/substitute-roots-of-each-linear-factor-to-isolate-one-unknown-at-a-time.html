Substitute roots of each linear factor to isolate one unknown at a time

Family: Strategic Substitution Frequency: 2 questions

Trigger — When to use

Fires when you have a polynomial identity of the form (numerator) ≡ A·(factor₁) + B·(factor₂) + ... that must hold for ALL x, where each unknown constant multiplies a different linear factor (or product of factors). Typical context: partial-fraction decomposition just after clearing the common denominator, e.g. 11x − 1 ≡ A(x−2) + B(2x+1), or A(x−1)² + B(2x+1)(x−1) + C(2x+1) ≡ 10x²+2x−2. The visual cue is distinct linear factors (ax+b) appearing as multipliers of unknowns. Recognising this lets you avoid expanding everything and solving a simultaneous system.

Motivation — Why this works

Because the identity holds for every x, you may freely pick values that simplify the equation. Choosing x to be the root of one linear factor zeroes that factor, annihilating every term that contains it and leaving a single-variable equation in one unknown. This converts an n-unknown coefficient-comparison system into n quick arithmetic substitutions, which is far faster and less error-prone under exam conditions. It is the algebraic engine behind the 'cover-up rule' for partial fractions.

Thought Process — The step-by-step reasoning

  1. I notice an identity in x with several unknown constants A, B, C, ... where each unknown is attached to a distinct linear factor (or a product containing one).
  2. I consider the alternative: expand all brackets, collect coefficients of x², x, constant, and solve a simultaneous system. This works but is slow and prone to arithmetic slips.
  3. Because the identity is valid for ALL x, I am free to choose convenient values. If I pick x equal to a root of (ax+b), every term carrying that factor vanishes — potentially leaving only one unknown.
  4. I scan the factors and list the roots: e.g. (x−2) → x=2; (2x+1) → x=−1/2; (x−1) → x=1. Each root targets a different unknown.
  5. I commit to substitution. I order the substitutions by arithmetic cleanliness (integer roots before fractional ones) to minimise slips.
  6. I check whether every unknown can be isolated this way. If a constant (e.g. the middle B in a repeated-factor case) sits on a term containing two different factors, no single substitution isolates it — I will need a hybrid step (compare one coefficient, or substitute a third value, after the others are known).

Mechanic — Concrete operations

  1. Clear denominators (if not already) so the identity reads (polynomial) ≡ Σ (unknown)·(factor product).
  2. List the distinct linear factors (ax+b) and compute their roots x = −b/a.
  3. For each root, substitute into both sides. All terms containing that factor vanish; solve the resulting single-variable equation.
  4. If any unknown is not isolated by substitution alone (typical with repeated factors), substitute the already-found values back and either compare one coefficient (e.g. of the highest power of x) or pick a convenient additional x-value to recover the remaining unknown.
  5. State each constant clearly and write the final partial-fraction decomposition.

Worked Micro-Example

Decompose $\dfrac{11x-1}{(2x+1)(x-2)} = \dfrac{A}{2x+1} + \dfrac{B}{x-2}$. Clearing denominators: $11x-1 \equiv A(x-2) + B(2x+1)$. Set $x=2$: $21 = 5B \Rightarrow B = \tfrac{21}{5}$. Set $x=-\tfrac{1}{2}$: $-\tfrac{13}{2} = -\tfrac{5}{2}A \Rightarrow A = \tfrac{13}{5}$. Hence $\dfrac{11x-1}{(2x+1)(x-2)} \equiv \dfrac{13}{5(2x+1)} + \dfrac{21}{5(x-2)}$.

Variants — Same move, different surface

All distinct linear factors in the denominator (no repeats)
Substitution alone finds every constant — no coefficient comparison needed. This is the cleanest case.
Repeated linear factor, e.g. $A(x-1)^2 + B(2x+1)(x-1) + C(2x+1)$
Substitution isolates only the constants attached to a single factor (here A via x=−1/2, and C via x=1). The middle constant B requires a hybrid step: compare the coefficient of x² on both sides, or substitute a third convenient value (e.g. x=0) after A,C are known.
Mental application via the 'cover-up' rule
For distinct linear factors, mentally cover the factor (ax+b) in the original fraction and evaluate the rest at x = −b/a to read off that constant directly.

Common Misreadings

  • Choosing x = 0 or x = 1 'because the arithmetic looks easy' — these usually do not annihilate any factor and yield an equation in all the unknowns, defeating the purpose.
  • Attempting full coefficient comparison when a clean substitution route exists — correct, but strategically slower under timed conditions.
  • Treating the middle term in a repeated-factor decomposition as if it could be isolated by substitution — it cannot, because its term contains both linear factors.

Pitfalls

  • Forgetting that the root of (2x+1) is x = −1/2, not x = −1 — always solve ax+b=0 properly.
  • Arithmetic slips when squaring a fraction, e.g. evaluating $10(-\tfrac{1}{2})^2$ — the square applies before the coefficient.
  • Sign errors when multiplying two negatives, e.g. $A(-\tfrac{5}{2})$ with negative LHS — track signs carefully.
  • When writing the final partial fractions, keeping the leading coefficient inside the denominator: write $\tfrac{A}{2x+1}$ (not $\tfrac{A}{x+1/2}$).

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P3/P4 partial fractions). No formula-booklet entry is needed; this is a standard technique. Avoid generalising to non-linear factors or complex roots — IAL partial fractions are restricted to distinct/repeated linear factors and at most one irreducible quadratic factor (which requires coefficient comparison, not substitution).
