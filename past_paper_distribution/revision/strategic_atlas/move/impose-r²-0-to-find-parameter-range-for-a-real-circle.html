Coordinate Geometry

Impose r² > 0 to find parameter range for a real circle

Family: Validity Condition Frequency: 1 questions

Trigger — When to use

Fires when an equation of the form $(x-a)^2+(y-b)^2 = f(k)$ contains a parameter and the question asks for the values of that parameter making the locus a (genuine) circle. Typical cues: 'find the range/set of values of k for which the equation represents a circle', or follow-up parts after completing the square. Also fires when a circle equation in general form $x^2+y^2+Dx+Ey+F=0$ has a parameter inside $D, E$ or $F$.

Motivation — Why this works

The standard form $(x-a)^2+(y-b)^2=r^2$ represents a real circle only when $r^2>0$: if $r^2=0$ the locus collapses to the single point $(a,b)$, and if $r^2<0$ no real points satisfy the equation. Translating this geometric existence requirement into the algebraic inequality $f(k)>0$ converts a wording condition ('is a circle') into a solvable inequality on the parameter.

Thought Process — The step-by-step reasoning

  1. Notice the completed-square form already gives $r^2$ as an expression in $k$, so the radius is parameter-dependent.
  2. Rule out $r^2 \le 0$: equality gives a point circle and negativity gives an empty locus, neither of which counts as 'a circle'.
  3. Commit to the strict inequality $r^2>0$ because the question specifies a circle, not merely a non-empty locus.
  4. Bridge: substitute the explicit expression for $r^2$ (here $k+61$) into the inequality.
  5. Solve the resulting linear/quadratic inequality in $k$ and present the range, checking the direction of the inequality is preserved.

Mechanic — Concrete operations

  1. Complete the square (if not already done) to obtain $(x-a)^2+(y-b)^2 = f(k)$.
  2. Identify $r^2 = f(k)$.
  3. Write the existence condition: $r^2 > 0$, i.e. $f(k) > 0$.
  4. Solve the inequality for $k$.
  5. State the answer as a range, e.g. $k > -61$.

Worked Micro-Example

Given $x^2+y^2-4x+6y-k=0$, complete the square: $(x-2)^2+(y+3)^2 = k+13$. For a real circle require $r^2>0$, so $k+13>0$, giving $k>-13$.

Variants — Same move, different surface

General form $x^2+y^2+2gx+2fy+c=0$
Real circle requires $g^2+f^2-c > 0$; solve this inequality in the parameter.
Sphere in 3D (Further Maths/coordinate extension)
$(x-a)^2+(y-b)^2+(z-c)^2=r^2$ uses the same condition $r^2>0$.
Question allows a 'point circle'
Use $r^2 \ge 0$ instead of strict inequality.
Parameter appears inside $a$ or $b$ as well as $r^2$
Only the $r^2$ expression governs existence; centre coordinates impose no constraint.

Common Misreadings

  • Using $k>0$ instead of $f(k)>0$ — forgetting that constants absorbed during completing the square contribute to $r^2$.
  • Writing $k \ge -61$ (allowing a point circle) when the question specifies a circle.
  • Treating the original equation's constant term alone as $r^2$ without first completing the square.

Pitfalls

  • Using $\ge$ instead of $>$ when the question requires a genuine circle.
  • Flipping the inequality sign incorrectly when rearranging.
  • Sign errors in completing the square that misstate $r^2$ as $k-61$ instead of $k+61$.

A-level Scope Notes

Fully within Edexcel IAL Pure (Core Maths) circle geometry; the same r²>0 logic extends to spheres in IAL Further Pure coordinate work.
