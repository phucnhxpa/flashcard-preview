Trigger — When to use

Fires when an equation of the form cos(linear in x) = 0 (or similar trig = constant) must be solved over a domain wider than one period — typically anything spanning more than 2π or asking for ALL roots/intercepts. Cue phrases include 'find all x such that...', 'state the x-coordinates of all intercepts', or a domain like −2π ≤ x ≤ 2π. Whenever more than two solutions are expected, prefer general-solution enumeration over a CAST sketch.

Motivation — Why this works

Cosine is zero precisely at odd multiples of π/2, captured by the single family θ = π/2 + nπ. Substituting consecutive integers n generates every zero exactly once, so enumeration is exhaustive and mechanical. This avoids the bookkeeping risk of CAST diagrams when many solutions exist, and matches how Edexcel expects general solutions to be deployed.

Thought Process — Step-by-step reasoning

  1. Notice the equation reduces to cos(argument) = 0 and the domain is wider than one period.
  2. Rule out CAST/quadrant reasoning because four or more solutions would force tedious case work.
  3. Commit to the general solution θ = π/2 + nπ since it parameterises all zeros uniformly.
  4. Bridge: equate the inner argument to π/2 + nπ and solve for x in terms of n.
  5. Tactically pick the integer range of n by bounding x within the stated domain, list every x, and discard any that fall outside.

Mechanic — Concrete operations

  1. Recognise the equation as cos(f(x)) = 0 where f(x) is linear in x.
  2. Write f(x) = π/2 + nπ, n ∈ ℤ.
  3. Solve algebraically for x in terms of n.
  4. Determine the integer range of n by enforcing the domain inequality on x.
  5. Substitute each valid n to list the solutions.
  6. Verify each x lies inside the closed domain; discard endpoints only if strictly excluded.

Worked Micro-Example

Solve $\cos(x+\tfrac{\pi}{4})=0$ for $-2\pi\le x\le 2\pi$.\\ Set $x+\tfrac{\pi}{4}=\tfrac{\pi}{2}+n\pi\Rightarrow x=\tfrac{\pi}{4}+n\pi$.\\ Domain forces $-\tfrac{9}{4}\le n\le \tfrac{7}{4}$, so $n\in\{-2,-1,0,1\}$.\\ Solutions: $x=-\tfrac{7\pi}{4},-\tfrac{3\pi}{4},\tfrac{\pi}{4},\tfrac{5\pi}{4}$.

Variants — Same move, different surface

sin θ = 0 over a wide domain
Use θ = nπ; otherwise the enumerate-and-filter procedure is identical.
cos θ = k with |k| < 1, k ≠ 0
Two families θ = ±arccos(k) + 2nπ; enumerate each separately and merge.
sin θ = k with |k| < 1, k ≠ 0
Use θ = arcsin(k) + 2nπ and θ = π − arcsin(k) + 2nπ.
tan θ = k
Single family θ = arctan(k) + nπ — period π not 2π.
cos(ax+b) = 0 with a ≠ 1
After ax+b = π/2 + nπ, divide by a; the spacing of solutions becomes π/a, so more n values are needed.

Common Misreadings

  • Reporting only the principal value (n = 0) and missing the rest.
  • Using a CAST diagram on the inner variable but forgetting to translate back to x via the substitution.
  • Treating the domain as applying to the inner argument rather than to x itself.

Pitfalls

  • Off-by-one in n: forgetting negative n and losing solutions in the negative-x half of the domain.
  • Arithmetic slips combining π/4 with π/2 + nπ, especially for negative n.
  • Including endpoints when the domain is strict, or excluding them when it is closed.

A-level Scope Notes

General solutions of sin, cos and tan equations and their use over specified domains are standard Edexcel IAL Pure content (P3/P4); this move uses no techniques beyond that syllabus.
