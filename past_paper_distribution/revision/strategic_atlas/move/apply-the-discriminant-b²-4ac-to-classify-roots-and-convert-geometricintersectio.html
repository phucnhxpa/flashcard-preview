Algebra & Functions

Apply the discriminant b² − 4ac to classify roots and convert geometric/intersection conditions into inequalities in a parameter

Family: Discriminant Analysis Frequency: 4 questions

Trigger — When to use

Fires whenever a problem reduces to a quadratic ax² + bx + c = 0 (often after substituting a line into a curve, or eliminating one variable from a system) and the question asks about the NUMBER or NATURE of real solutions/intersections. Cue phrases include: 'does not intersect', 'is tangent to', 'two distinct points of intersection', 'three distinct intersection points', 'show that the equation has no real solutions', 'find the range/values of k (or A) such that...'. Also fires defensively when the quadratic formula produces a negative under the square root, signalling that real roots do not exist. The tell-tale structural cue is a quadratic in one variable whose coefficients depend on a parameter, plus a counting condition on roots.

Motivation — Why this works

For ax² + bx + c = 0 the discriminant Δ = b² − 4ac classifies the roots: Δ > 0 ⇒ two distinct real roots, Δ = 0 ⇒ one repeated real root (tangency), Δ < 0 ⇒ no real roots. This single algebraic quantity converts a geometric or counting statement ('the line doesn't meet the curve', 'three intersections', 'no real solutions') into a clean inequality or equation in the parameter, which can then be solved by standard means. It is also the fastest existence test — no need to invoke the full quadratic formula when only the nature of roots is required.

Thought Process — The step-by-step reasoning

  1. Notice that after substitution/elimination the problem reduces to a single quadratic in one variable, with coefficients possibly involving a parameter (k, A, etc.).
  2. Rule out alternatives: factorising is impractical when coefficients are parametric; completing the square would work but is heavier; the quadratic formula gives more than needed if only a count of roots is asked.
  3. Commit to the discriminant because the question is about EXISTENCE/COUNT of solutions, not their explicit values — Δ = b² − 4ac directly encodes this.
  4. Translate the wording into the correct sign: 'does not intersect' / 'no real roots' ⇒ Δ < 0; 'tangent' / 'repeated root' ⇒ Δ = 0; 'two distinct intersections' ⇒ Δ > 0. Use STRICT inequality when tangency must be excluded.
  5. Check the leading coefficient is genuinely non-zero (else the equation is not quadratic and a separate case is needed); identify a, b, c carefully, especially signs when b is itself a bracketed expression like −(9 + k).
  6. Form Δ, simplify to a quadratic (or linear) inequality in the parameter, and solve. If extra geometric constraints exist (e.g. a known root must be excluded), verify them against the parameter range.

Mechanic — Concrete operations

  1. Write the quadratic in standard form ax² + bx + c = 0 and identify a, b, c (these may depend on the parameter).
  2. Confirm a ≠ 0; if a depends on the parameter, treat a = 0 as a separate case.
  3. Compute Δ = b² − 4ac, expanding carefully — e.g. (−(9+k))² = (9+k)² = k² + 18k + 81.
  4. Set Δ <, =, or > 0 according to the wording: no intersection (<0), tangent (=0), two distinct (>0).
  5. Simplify to a polynomial inequality/equation in the parameter and solve (factorise, quadratic formula, or sketch).
  6. Cross-check any auxiliary conditions (e.g. excluded roots, domain restrictions on the parameter).

Worked Micro-Example

Find k such that y = x + k does not meet y = x² + 9x + 4. Substituting: x² + 9x + 4 = x + k ⇒ x² + 8x + (4 − k) = 0. Then Δ = 8² − 4(1)(4 − k) = 64 − 16 + 4k = 48 + 4k. No intersection ⇒ Δ < 0 ⇒ 48 + 4k < 0 ⇒ k < −12.

Variants — Same move, different surface

Tangency: the line just touches the curve
Use Δ = 0 instead of Δ < 0; same setup, equation rather than inequality.
Two distinct intersection points required
Use Δ > 0; be careful to exclude Δ = 0 (repeated root counts as one point).
Three distinct intersections where one root (e.g. x = 0) is guaranteed
Factor out the known root, then apply Δ > 0 to the remaining quadratic; also verify the known root is not reproduced by the residual quadratic.
Quadratic in y (or another variable) instead of x
Same discriminant logic applies — the variable name is irrelevant; only the structure ay² + by + c = 0 matters.
Negative discriminant inside Further Maths / complex numbers
Instead of stating 'no real solutions', write x = (−b ± i√|Δ|)/(2a) using i = √(−1).
'Show that no real solutions exist' proof question
Compute Δ, demonstrate Δ < 0 explicitly with arithmetic, and state the conclusion — this constitutes a complete proof.

Common Misreadings

  • Using Δ ≤ 0 for 'does not intersect' — this wrongly admits tangency (a tangent line DOES touch the curve).
  • Confusing 'no real roots' with Δ > 0, or 'two distinct roots' with Δ ≥ 0.
  • Forgetting that a repeated root (Δ = 0) gives ONE intersection point, not two — important when counting distinct intersections.
  • Treating an obtained negative discriminant as evidence of an arithmetic mistake and 'fixing' signs to force Δ > 0.

Pitfalls

  • Sign errors when b is itself a negative bracket: (−(9+k))² = (9+k)², not −(9+k)².
  • Arithmetic slips in 4ac when c contains a parameter (e.g. 4(1)(9k+4) = 36k + 16, not 9k + 16).
  • Forgetting to reverse the inequality when dividing by a negative number while solving the resulting inequality (e.g. dividing 16 − 4A > 0 by −4).
  • Failing to check that the leading coefficient a is non-zero when it depends on the parameter — a = 0 collapses the quadratic to linear and must be handled separately.
  • Stopping at the discriminant inequality without checking auxiliary constraints (e.g. that an excluded root such as x = 0 doesn't reappear).

A-level Scope Notes

Fully within Edexcel IAL P1 (quadratics) and recurring across P2–P4 in coordinate geometry and parameter problems. The discriminant b² − 4ac is standard and not in the formula booklet — students are expected to know it. Avoid invoking complex roots unless working in Further Pure (FP1+); in P1–P4 the conclusion for Δ < 0 is simply 'no real solutions'.
