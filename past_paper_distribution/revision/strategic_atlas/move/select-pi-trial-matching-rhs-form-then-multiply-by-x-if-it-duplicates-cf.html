Differential Equations

Select PI trial matching RHS form, then multiply by x if it duplicates CF

Family: Pi Trial Selection Frequency: 2 questions

Trigger — When to use

This move fires whenever you face a non-homogeneous linear ODE ay'' + by' + cy = f(x) and need a particular integral. The cue is a non-zero RHS belonging to a recognisable family: an exponential ke^{αx}, a polynomial, a trig function k cos ωx or k sin ωx, or a product such as xe^{αx}. Equivalently: 'find the general solution' or 'find a particular integral' once the CF has been written down. Before committing to the trial, you must always cross-check whether any term in your proposed trial already appears in the CF.

Motivation — Why this works

The method of undetermined coefficients works because differentiating a function from one of these standard families produces functions of the same family, so a trial with unknown coefficients can be substituted into the LHS and matched term-by-term to the RHS. However, if the trial is itself a CF term, it satisfies the homogeneous equation, so the LHS evaluates to 0 and cannot equal the non-zero RHS — coefficient matching becomes impossible. Multiplying the trial by x (or x² for a repeated root) shifts it out of the homogeneous solution space while keeping it close enough in form that its derivatives still produce matchable terms.

Thought Process — The step-by-step reasoning

  1. Inspect f(x) and identify its family: exponential ke^{αx}, polynomial of degree n, trigonometric k cos ωx + l sin ωx, or a product of these.
  2. Write down the standard trial with undetermined coefficients (e.g. λe^{αx}, or ax^n + bx^{n-1} + … + k, or λ cos ωx + μ sin ωx, or (ax+b)e^{αx}).
  3. Rule out alternatives: variation of parameters or guessing — undetermined coefficients is faster and the RHS lies in a standard family, so commit to it.
  4. Now perform the duplication check: list the CF exponents/frequencies and compare with the trial. For ke^{αx}, ask 'is α a root of the auxiliary equation?'; for trig RHS, ask 'are ±iω the complex CF roots?'.
  5. If there is no duplication, commit to the unmodified trial. If α is a simple root, multiply the entire trial by x. If α is a repeated root, multiply by x².
  6. Even if f(x) only contains cos ωx (or only sin ωx), include BOTH cos and sin in the trial, because differentiation mixes them.
  7. Differentiate the (possibly modified) trial to get y'_PI and y''_PI, then substitute into the full ODE and solve for the coefficients.

Mechanic — Concrete operations

  1. Solve the auxiliary equation am² + bm + c = 0 and write the CF.
  2. Read off the family of f(x) and write the standard trial PI with undetermined coefficients.
  3. Compare each term of the trial with each CF term: check whether the exponent α (or frequency ω) appears in the CF.
  4. If duplication exists, multiply the trial by x; if duplication is doubled (repeated root), multiply by x².
  5. Compute y'_PI and y''_PI by direct differentiation (product rule where needed).
  6. Substitute into ay'' + by' + cy = f(x), collect like terms, and equate coefficients to solve for the unknowns.

Worked Micro-Example

Solve 2y'' - 5y' + 3y = 3e^{3x}. Auxiliary: 2m² - 5m + 3 = 0 → m = 1, 3/2. CF: y_CF = Ae^{x} + Be^{(3/2)x}. RHS exponent is 3; since 3 ≠ 1 and 3 ≠ 3/2, no duplication, so trial is y_PI = ae^{3x}. Then y'_PI = 3ae^{3x}, y''_PI = 9ae^{3x}; substituting: (18a - 15a + 3a)e^{3x} = 6ae^{3x} = 3e^{3x}, giving a = 1/2, so y_PI = (1/2)e^{3x}.

Variants — Same move, different surface

RHS exponent matches a simple CF root (e.g. 3e^{x} with CF containing e^{x})
Multiply the standard trial by x: use y_PI = axe^{x} instead of ae^{x}.
RHS exponent matches a repeated CF root (e.g. 3e^{2x} with CF = (A + Bx)e^{2x})
Multiply by x²: use y_PI = ax²e^{2x}.
RHS is a polynomial of degree n
Trial is a full polynomial of degree n: ax^n + bx^{n-1} + … + k, including all lower-order terms even if RHS has none.
RHS is a product like xe^{2x}
Match the product structure: trial is (ax + b)e^{2x}, then x-multiply if e^{2x} duplicates the CF.
RHS is k cos ωx or k sin ωx and ±iω are roots of the auxiliary equation
Trial becomes x(λ cos ωx + μ sin ωx); always include both cos and sin terms.

Common Misreadings

  • Confusing the RHS exponent with a CF coefficient or root (e.g. seeing exponent 3 and root 3/2 and wrongly declaring duplication).
  • Using only λ cos ωx when f(x) = cos ωx, omitting the sin term — fails because the first derivative produces sin terms.
  • Failing to spot that the exponential in f(x) matches a CF root, then using the unmodified trial and reaching the contradiction 0 = f(x).
  • Misreading a similar-looking mark scheme and forcing the wrong trial form (e.g. forcing axe^{αx} when no duplication exists).

Pitfalls

  • Forgetting the duplication check entirely — the substitution will collapse to 0 = non-zero RHS with no escape.
  • Over-correcting: multiplying by x when no duplication exists, which introduces extra unmatched terms and an inconsistent system.
  • For a polynomial RHS, omitting lower-degree terms in the trial (e.g. trying ax² for RHS = x² + 1 — must use ax² + bx + c).
  • For trig RHS, only x-multiplying one of the two terms instead of the whole bracket x(λ cos ωx + μ sin ωx).

A-level Scope Notes

Stays within Edexcel IAL Further Pure (FP2/FP3) second-order linear ODEs with constant coefficients. The standard families covered are exponentials, polynomials, sin/cos, and simple products thereof; do not invoke variation of parameters or Laplace transforms, which are out of scope. The auxiliary equation method is in the FP formula booklet under second-order ODEs.
