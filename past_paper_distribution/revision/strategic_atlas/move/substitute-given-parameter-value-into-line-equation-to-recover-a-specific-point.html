Vectors

Substitute given parameter value into line equation to recover a specific point

Family: Parametric Evaluation Frequency: 1 questions

Trigger — When to use

Fires whenever a question names a point on a line via a specific parameter value, e.g. 'C lies on l₂ with μ = −1' or 'the point on l where t = 2'. Also triggers when a foot of perpendicular, intersection, or closest point has just been determined as a parameter value and coordinates are now needed. Any subsequent step (distance, displacement vector, angle, dot product) requires Cartesian coordinates, not the parameter.

Motivation — Why this works

A vector line r = p + λd is a bijection between parameter values and points on the line, so substituting the stated λ converts the parametric description directly into explicit coordinates. This is the cheapest possible operation — one scalar multiplication and one vector addition — and it is forced because no later vector calculation (AC, |AC|, AC·AB) can proceed symbolically in λ if a numerical answer is required.

Thought Process — The step-by-step reasoning

  1. I notice the point is specified by a parameter value, not by coordinates.
  2. I rule out solving simultaneous equations or using projection formulas — the parameter is already given.
  3. I commit to direct substitution because the line equation is the definition of the point.
  4. I write r = p + λd with the given λ and prepare to evaluate component-by-component.
  5. I keep signs explicit when λ is negative, and I do not simplify until all three components are written down.

Mechanic — Concrete operations

  1. Identify the line equation l: r = p + λd and the stated parameter value λ₀.
  2. Compute λ₀·d component-by-component, watching the sign of λ₀.
  3. Add to p to get the position vector of the named point.
  4. Use this coordinate triple in the next step (e.g. form AC = c − a).

Worked Micro-Example

Given $l_2: \mathbf{r} = (3,1,4) + \mu(1,5,9)$ and $C$ on $l_2$ with $\mu = -1$:\ $\mathbf{c} = (3,1,4) + (-1)(1,5,9) = (2,-4,-5)$.\ Then with $A = (6,6,2)$, $\overrightarrow{AC} = \mathbf{c} - \mathbf{a} = (-4,-10,-7)$.

Variants — Same move, different surface

Foot of perpendicular from a point to a line, after solving for λ
Substitute the solved λ back into r = p + λd to get the foot's coordinates.
Intersection point of two lines once λ (or μ) is found
Substitute into either line equation to get the intersection point; the other equation provides a check.
Closest point on a line to a fixed point or to another line
After minimising distance to find λ, substitute to recover coordinates before computing the minimum distance.
Line meeting a plane at a parameter value found from the plane equation
Substitute λ into the line to get the point of intersection with the plane.

Common Misreadings

  • Treating the parameter value as a coordinate (e.g. taking μ = −1 to mean C has an x-coordinate of −1).
  • Assuming the named point is the line's reference point p instead of substituting.
  • Confusing which line the parameter belongs to when two lines l₁, l₂ are present.

Pitfalls

  • Sign errors when λ is negative — every component of d must carry the negative.
  • Adding the direction vector instead of (λ × direction vector) when λ ≠ 1.
  • Mixing up the position vector p and the direction vector d in the line equation.
  • Forgetting to subtract a (not c) when forming AC = c − a.

A-level Scope Notes

Fully within Edexcel IAL Core Pure / FP vectors: lines in 3D given in vector form r = p + λd, with downstream use in distances, angles, and dot products.
