Algebra & Functions

Verify continuity on the interval before applying IVT

Family: Domain Validity Check Frequency: 1 questions

Trigger — When to use

Fires when a question asks you to show a root exists in a closed interval [a, b] using a sign change f(a)f(b) < 0, AND the function contains a rational, logarithmic, or trigonometric piece that could be discontinuous. Cues include denominators like 1/(5x − 3), tan x, ln(x − k), or piecewise definitions. Also fires when the mark scheme demands an explicit 'continuous on [a,b]' statement, not just 'sign change'.

Motivation — Why this works

The Intermediate Value Theorem requires f to be continuous on the closed interval [a, b]; without continuity, a sign change can occur across a vertical asymptote with no root in between. By locating any discontinuity and confirming it lies outside [a, b], the IVT hypothesis is fully met. This converts a sign-change observation into a rigorous existence argument that earns full justification marks.

Thought Process — The step-by-step reasoning

  1. Notices the function is not a polynomial — there is a rational term 1/(5x − 3) (or similar) that could be undefined somewhere.
  2. Rules out 'just quote sign change': the IVT premise needs continuity, so a bare sign-change argument is incomplete.
  3. Commits to locating the discontinuity explicitly and testing whether it lies inside [1, 2].
  4. Bridges to the concrete step: solve 5x − 3 = 0 to get x = 3/5, then compare with the interval endpoints.
  5. Decides to write a one-line continuity statement before computing f(1) and f(2), so the IVT citation is watertight.

Mechanic — Concrete operations

  1. Identify all points where f could fail to be continuous (zeros of denominators, arguments of ln, points where tan/sec blow up).
  2. Solve for the discontinuity: e.g. 5x − 3 = 0 ⟹ x = 3/5.
  3. Check whether each such point lies in [a, b]; here 3/5 = 0.6 ∉ [1, 2].
  4. State: 'f is continuous on [1, 2]'.
  5. Compute f(1) and f(2), show opposite signs, then cite IVT to conclude a root α ∈ (1, 2) exists.

Worked Micro-Example

Let $f(x) = x - 2 + \dfrac{1}{5x-3}$. Discontinuity at $x = \tfrac{3}{5} \notin [1,2]$, so $f$ is continuous on $[1,2]$. Then $f(1) = -1 + \tfrac{1}{2} = -\tfrac12 < 0$ and $f(2) = 0 + \tfrac{1}{7} > 0$. By the IVT, there exists $\alpha \in (1,2)$ with $f(\alpha)=0$.

Variants — Same move, different surface

Function involves $\ln(x-k)$
Require $x - k > 0$ throughout [a, b]; state the domain condition before invoking IVT.
Function involves $\tan x$ or $\sec x$
Identify all asymptotes $x = \tfrac{\pi}{2} + n\pi$ in range and confirm none lie in [a, b].
Interval contains a discontinuity
Split into sub-intervals avoiding the discontinuity and apply IVT on a continuous sub-interval where the sign change occurs.
Piecewise-defined function
Check continuity at the join point by matching one-sided limits before quoting IVT across the join.

Common Misreadings

  • Treating 'sign change implies root' as the full IVT statement, omitting continuity.
  • Assuming any 'nice-looking' expression is automatically continuous without checking denominators.
  • Confusing IVT (existence) with a uniqueness claim — IVT alone does not show the root is unique.

Pitfalls

  • Writing 'f is a polynomial so continuous' when f contains a rational or logarithmic term.
  • Forgetting to compare the discontinuity to the interval (e.g., locating x = 3/5 but not stating 3/5 ∉ [1,2]).
  • Quoting IVT on a closed interval but only computing f at one endpoint.
  • Stating the root lies in [1,2] (closed) when IVT yields it in (1,2) (open).

A-level Scope Notes

Standard for Edexcel IAL P3/P4 'show that a root exists in [a, b]' questions; the explicit continuity line is expected when the function is not obviously polynomial.
