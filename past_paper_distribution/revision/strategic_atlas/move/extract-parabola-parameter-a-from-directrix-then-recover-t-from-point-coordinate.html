Coordinate Geometry

Extract parabola parameter a from directrix then recover t from point coordinates

Family: Parameter Identification Frequency: 1 questions

Trigger — When to use

Fires when a parabola problem gives a directrix (e.g. x = −4) and a specific point on the curve, and a general parametric tangent/normal formula from an earlier part must be specialised. Cues: 'directrix x = …', 'point (p, q) lies on the parabola', and a previously derived expression in a and t. Also fires whenever both the scale parameter a and the parameter t at a particular point need to be pinned down before substitution.

Motivation — Why this works

The standard parabola y² = 4ax has directrix x = −a, so comparing the given directrix to this template reads a off directly. Once a is known, the parametric coordinates (at², 2at) give two equations for t from the point's coordinates; t² = (x/a) determines t up to sign, and 2at = y fixes the sign uniquely. Using both coordinates is essential because the x-coordinate alone leaves a ± ambiguity that corresponds to the reflected point (at², −2at).

Thought Process — The step-by-step reasoning

  1. Notices that the general tangent formula ty = x + at² contains two unknowns (a and t) and cannot be used until both are numerical.
  2. Rules out implicit differentiation or ad-hoc tangent finding because part (a) explicitly built a parametric tool meant to be reused.
  3. Commits to a two-stage parameter extraction: first a from the directrix, then t from the point.
  4. Bridges by writing the canonical form y² = 4ax with directrix x = −a and matching x = −4 to get a = 4.
  5. Inside the move, uses x-coordinate for t² then the y-coordinate to disambiguate sign, and only then substitutes into the part (a) formula.

Mechanic — Concrete operations

  1. Match given directrix x = −4 with template x = −a to obtain a = 4.
  2. Substitute a into x = at²: 4t² = 4, so t² = 1 and t = ±1.
  3. Substitute a into y = 2at: 8t = 8, so t = 1 (sign fixed).
  4. Substitute a = 4, t = 1 into the parametric tangent ty = x + at² to get y = x + 4.

Worked Micro-Example

Parabola has directrix $x = -4$ and passes through $(4, 8)$. Matching $x = -a$ gives $a = 4$, so $y^2 = 16x$. Then $at^2 = 4 \Rightarrow t^2 = 1$ and $2at = 8 \Rightarrow 8t = 8 \Rightarrow t = 1$. Substituting into $ty = x + at^2$: $y = x + 4$.

Variants — Same move, different surface

Parabola given by focus (a, 0) instead of directrix
Read a directly from the focus coordinate; rest of the t-extraction is identical.
Rectangular hyperbola xy = c² with a point given
Identify c from the product of coordinates (xy = c²), then recover parameter t via x = ct, y = c/t — sign fixed by matching both coordinates.
Ellipse x²/a² + y²/b² = 1 with point and eccentricity/focus data
Use focus or eccentricity relation to pin a and b, then solve cos θ = x/a, sin θ = y/b to fix θ uniquely (both equations needed for the correct quadrant).
Parabola with latus rectum length given (4a)
Read a from the latus rectum length, then proceed identically for t.

Common Misreadings

  • Verifying that (4, 8) lies on y² = 16x and then doing implicit differentiation, ignoring the parametric tangent built in part (a).
  • Treating a as already known to be 1 because the formula was written as ty = x + at², missing that the coefficient must be determined.

Pitfalls

  • Taking t = −1 from t² = 1 without checking the y-coordinate, producing the tangent at the reflected point (4, −8).
  • Confusing directrix x = −a with focus (a, 0) and setting a = −4, contradicting a > 0.
  • Mis-matching the template: using y² = 4ax with a = 4 but then writing the directrix as x = −4a or x = a.

A-level Scope Notes

Standard Edexcel IAL Further Pure content: parametric form of the parabola y² = 4ax with point (at², 2at), directrix x = −a, and the tangent ty = x + at².
