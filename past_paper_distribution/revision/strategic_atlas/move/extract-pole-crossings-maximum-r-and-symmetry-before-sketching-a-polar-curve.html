Polar Coordinates

Extract pole crossings, maximum r, and symmetry before sketching a polar curve

Family: Feature Extraction Frequency: 1 questions

Trigger — When to use

Fires whenever a question says 'sketch the curve with polar equation r = f(θ)' or asks to identify key features of a polar curve. Also triggers when later parts depend on knowing tangent directions at the pole or the furthest point from O. Phrases like 'show that the curve has a cusp', 'find the tangents at the pole', or 'state the maximum value of r' are explicit cues. Do not just plot a table of points — extract structural features first.

Motivation — Why this works

Setting r = 0 locates the angles where the curve passes through the pole; these are precisely the tangent directions there, and they partition the curve into lobes or petals. Maximising r (by inspection of the trig term or by dr/dθ = 0) gives the curve's furthest reach and the angle at which it occurs. Symmetry tests (θ → −θ, θ → π − θ) halve the work and force a correct global shape. Together these three features determine the sketch up to smooth interpolation, which is the standard expected at A-level.

Thought Process — The step-by-step reasoning

  1. I notice this is a sketch task for r = f(θ), so I need a skeleton of features rather than a dense point plot.
  2. I rule out brute-force tabulation of r at many angles — that risks missing the cusp/petal structure and wastes time.
  3. I commit to the standard triad: pole crossings (r = 0), maximum r, and symmetry, because these pin down shape with minimal calculation.
  4. First concrete operation: solve f(θ) = 0 on [0, 2π) to get tangent directions at O, then find where the trig term hits its max to locate r_max.
  5. Tactically, I check f(−θ) vs f(θ) for initial-line symmetry; if r would go negative on some interval, I note the convention used (typically restrict to r ≥ 0 unless told otherwise) before joining points smoothly.

Mechanic — Concrete operations

  1. Solve f(θ) = 0 for θ ∈ [0, 2π) to find all pole crossings; these give tangent directions at O.
  2. Find max r: either spot where the trig term equals ±1, or solve dr/dθ = 0 and evaluate.
  3. Test symmetry: compute f(−θ) (initial-line symmetry) and/or f(π − θ) (symmetry about θ = π/2).
  4. If r can be negative on the given range, decide whether to restrict to r ≥ 0 or reflect through the pole, per the question's convention.
  5. Plot the pole crossings, the max-r point(s), and a couple of intermediate points; join smoothly using symmetry; label the curve and any axis intercepts.

Worked Micro-Example

Sketch r = a(1 + cosθ), a > 0, for θ ∈ [0, 2π). Pole: 1 + cosθ = 0 ⇒ θ = π, so the curve meets O tangent to the initial line. Max r: cosθ = 1 ⇒ θ = 0, r = 2a. Symmetry: f(−θ) = a(1 + cos(−θ)) = f(θ), so symmetric about the initial line. Sketch: cardioid with cusp at O (along θ = π) and tip at (2a, 0).

Variants — Same move, different surface

Cardioid r = a(1 ± cosθ) or r = a(1 ± sinθ)
Single pole crossing where the bracket vanishes; max r = 2a at the opposite angle; cusp at the pole along that tangent direction.
Rose curve r = a cos(nθ) or a sin(nθ)
Pole crossings at θ = π/(2n) + kπ/n give petal boundaries; n petals if n odd, 2n petals if n even; max r = a at petal centres.
Limaçon r = a + b cosθ with a < b
Inner loop appears; r = 0 gives two angles bounding the inner loop; max r = a + b at θ = 0, min |r| structure must be checked.
Spiral r = aθ on [0, 2π]
r = 0 only at θ = 0; no finite max — endpoints determine extent; no rotational symmetry, so plot more intermediate points.
Curve given parametrically by r and θ in part (a), area in part (b)
Pole crossings give the integration limits for ½∫r² dθ over a single lobe.

Common Misreadings

  • Treating r = 0 as 'the curve is undefined' instead of 'the curve passes through the pole at this θ'.
  • Confusing maximum r with maximum y or maximum x — the question asks for radial distance from O.
  • Reading r = a(1 + cosθ) as a circle because it superficially resembles one near θ = 0.

Pitfalls

  • Plotting only a handful of points without extracting features, then drawing a circle in place of a cardioid or missing petals on a rose curve.
  • Ignoring intervals where f(θ) < 0 and either drawing a phantom branch or omitting a real loop.
  • Forgetting to state the tangent directions at the pole when the question asks for them, even after correctly finding r = 0.
  • Using degrees in dr/dθ when the rest of the working is in radians, leading to scale errors.

A-level Scope Notes

Standard Edexcel IAL Further Pure polar coordinates content: sketches of cardioids, limaçons, roses, and spirals using r = 0, max r, and symmetry. No need for curvature or higher-order tangent analysis beyond identifying tangent lines at the pole.
