Trigonometry

Take arcsin for principal value then halve to find smallest positive θ

Family: General Solution Selection Frequency: 1 questions

Trigger — When to use

The equation has the form sin(kθ) = c with c not a standard exact value (e.g. sin 2θ = 3/4), and the question asks for the smallest positive θ in degrees or radians. The presence of a multiple angle kθ together with a non-exact RHS signals a calculator arcsin followed by branch selection. Phrases like 'smallest positive solution' or 'least positive value' are direct cues. Distinguish from 'find all solutions in [0°, 360°]' which requires enumerating every branch.

Motivation — Why this works

For sin X = c with 0 < c < 1, the two basic-cycle solutions are X = arcsin(c) and X = 180° − arcsin(c), with general form X = arcsin(c) + 360n° or (180° − arcsin(c)) + 360n°. When X = kθ, every solution for X must be divided by k, and the smallest positive θ corresponds to the smallest positive X (because dividing by a positive k preserves order). Thus the principal value arcsin(c) typically yields the smallest positive θ, but this should be confirmed against the second-quadrant branch.

Thought Process — The step-by-step reasoning

  1. Notice the RHS 3/4 is not a standard exact sine value, so a calculator arcsin is unavoidable.
  2. Rule out treating the argument as plain θ — the equation is in 2θ, so any solution for 2θ must later be halved.
  3. Commit to listing the smallest few positive values of 2θ (principal + second-quadrant branch) before halving, because order is preserved under division by a positive constant.
  4. Bridge: compute α = arcsin(3/4) ≈ 48.59° as the principal value, then 180° − α ≈ 131.41° as the partner branch.
  5. Tactical: keep unrounded values until the final step, halve each, then pick the smallest positive result and round to the requested precision.

Mechanic — Concrete operations

  1. Let X = 2θ and compute α = arcsin(3/4) ≈ 48.590°.
  2. List the two basic-cycle solutions for X: X₁ = α ≈ 48.590°, X₂ = 180° − α ≈ 131.410°.
  3. Divide each by 2: θ₁ ≈ 24.295°, θ₂ ≈ 65.705°.
  4. Select the smallest positive value: θ ≈ 24.3° (1 d.p.).
  5. Sanity check: substitute back, sin(2 × 24.3°) = sin(48.6°) ≈ 0.750. ✓

Worked Micro-Example

Solve $\sin 2\theta = \tfrac{3}{4}$ for the smallest positive $\theta$. Principal value: $2\theta = \arcsin(0.75) \approx 48.590°$. Second branch: $2\theta = 180° - 48.590° \approx 131.410°$. Halving: $\theta \approx 24.295°$ or $65.705°$. Smallest positive: $\theta \approx 24.3°$ (1 d.p.).

Variants — Same move, different surface

cos(kθ) = c with 0 < c < 1
Use β = arccos(c) as principal value; partner branch is X = −β (or 360° − β). Smallest positive θ usually still comes from β/k, but check 360° − β branch when k > 1.
tan(kθ) = c
Only one branch per 180° cycle: X = arctan(c) + 180n°. Smallest positive θ = arctan(c)/k provided arctan(c) > 0; otherwise add 180°/k.
sin(kθ + φ) = c (phase shift)
Solve for X = kθ + φ first, then θ = (X − φ)/k. The smallest positive θ may not come from the principal X; check several X-branches.
sin(kθ) = c with c < 0
Principal arcsin(c) is negative; smallest positive θ comes from X = 180° − arcsin(c) (third-quadrant equivalent) or X = arcsin(c) + 360°, whichever is smaller after dividing by k.
Radian version, e.g. sin 2θ = 3/4 for θ ∈ ℝ
Replace 180° with π and 360° with 2π; otherwise identical procedure.

Common Misreadings

  • Solving sin θ = 3/4 directly and reporting θ ≈ 48.6°, ignoring that the argument is 2θ.
  • Treating 'smallest positive' as 'principal value of the calculator' without checking other branches.
  • Confusing the second-quadrant partner (180° − α) with the negative principal (−α).

Pitfalls

  • Rounding 2θ before dividing by 2, which can shift the final 1 d.p. answer.
  • Forgetting the second-quadrant branch when the question later asks for all solutions in a range.
  • Not checking whether a negative-branch solution (e.g. X = α − 360°) wraps around to give a smaller positive θ — irrelevant here but critical for negative c or phase-shifted equations.
  • Using radian mode on the calculator when the question is in degrees (or vice versa).

A-level Scope Notes

Standard Edexcel IAL P2/P3 trigonometry: solving sin/cos/tan of a multiple angle on a restricted interval using arcsin and CAST/quadrant reasoning, with calculator use permitted for non-exact values.
