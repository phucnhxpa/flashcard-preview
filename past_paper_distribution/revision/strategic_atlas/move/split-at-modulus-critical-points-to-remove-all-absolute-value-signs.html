Algebra & Functions

Split at modulus critical points to remove all absolute value signs

Family: Case Splitting Frequency: 2 questions

Trigger — When to use

The equation (or inequality) contains one or more modulus expressions |f(x)| whose interior changes sign at identifiable critical points, e.g. |3 − x|, |2x − k/3|, or a sum like |x − 1| + |x + 2|. The right-hand side is not itself a single modulus (so the symmetric trick |A| = |B| ⇒ A² = B² does not directly apply), and the interiors involve linear or simple expressions where the sign on each subinterval can be determined exactly. Visually: you see vertical bars wrapped around expressions you cannot simplify further, and you need to convert to ordinary algebra. Equivalent phrasings: 'remove the modulus by cases', 'piecewise-linearise the modulus', 'split the real line at the zeros of the interiors'.

Motivation — Why this works

Each modulus |f(x)| is defined piecewise: it equals f(x) when f(x) ≥ 0 and −f(x) when f(x) < 0. By locating the critical points (zeros of each interior) and partitioning the real line into intervals on which every interior has a definite sign, the modulus equation reduces to an ordinary (typically linear) equation on each interval. This is preferred to squaring whenever the RHS is not also a clean modulus, because squaring there would (i) introduce extraneous solutions when the RHS can be negative, and (ii) raise the degree unnecessarily. The underlying principle is the piecewise definition of |·|.

Thought Process — The step-by-step reasoning

  1. Notice every modulus expression in the equation and identify what is inside each one — these interiors are the things whose signs I need to control.
  2. Consider alternatives: could I square both sides? Only safely if both sides are themselves moduli (or the RHS is provably non-negative). Here, the RHS is a plain expression that may go negative, so squaring risks extraneous roots and a higher-degree mess.
  3. Commit to case-splitting: solve each interior = 0 to find the critical points; these subdivide ℝ into intervals on which every modulus has a fixed sign.
  4. On each interval, replace each |f(x)| by either f(x) or −f(x) according to the determined sign, producing a clean (usually linear) equation valid on that interval only.
  5. Solve the equation in each case, then check that each candidate solution actually lies inside the interval that defined the case — if not, discard it. Also handle any case that collapses to a contradiction (no solution there) or an identity (whole interval is solution).
  6. Tactical choice: with n moduli there are up to 2^n sign combinations, but adjacent intervals on ℝ only give n+1 genuine cases — list these in order along the number line to avoid duplication.

Mechanic — Concrete operations

  1. For each modulus |f_i(x)| in the equation, set f_i(x) = 0 and solve to obtain critical points c_1 < c_2 < … < c_m.
  2. Partition ℝ (or the relevant domain) into the intervals (−∞, c_1], [c_1, c_2], …, [c_m, ∞), determining the sign of each f_i on each interval.
  3. On each interval, rewrite the equation by replacing |f_i| with +f_i or −f_i as dictated by the sign, producing a non-modulus equation.
  4. Solve the resulting equation algebraically (usually a single linear step yielding x = …, a contradiction, or an identity).
  5. Validate each candidate against the defining interval; reject any solution outside its case's domain.
  6. Collect all surviving solutions across the cases as the final solution set.

Worked Micro-Example

Solve $|2x - 6| = x + 3$. Critical point: $2x-6=0 \Rightarrow x=3$. Case 1 ($x \geq 3$): $2x-6 = x+3 \Rightarrow x = 9$ ✓ (since $9 \geq 3$). Case 2 ($x < 3$): $-(2x-6) = x+3 \Rightarrow 6-2x = x+3 \Rightarrow x = 1$ ✓ (since $1 < 3$). Solutions: $x = 1, 9$.

Variants — Same move, different surface

Single modulus equation |f(x)| = g(x) with linear f,g
Only one critical point; just two cases. Always check each candidate satisfies its defining inequality.
Two moduli with different interiors, e.g. |2x − k/3| = k − 2|x|
Two critical points (here x = 0 and x = k/6) give three intervals; expect up to three candidate solutions, one of which may be a contradiction.
Both sides are single moduli: |f(x)| = |g(x)|
Squaring is cleaner: f(x)² = g(x)² ⇒ (f−g)(f+g) = 0, giving f = ±g without needing intervals.
Modulus inequality |f(x)| < k or |f(x)| ≥ g(x)
Same piecewise rewrite, but each case yields an inequality whose solution must be intersected with the case's interval.

Common Misreadings

  • Treating only one modulus by cases while leaving another modulus untouched, missing solutions in some sign combinations.
  • Trying to square both sides when the RHS contains a non-modulus term or can be negative, introducing extraneous roots.
  • Writing |2x − k/3| = ±(k − 2|x|) directly without first eliminating |x|, conflating two distinct sign choices into one.

Pitfalls

  • Sign errors when negating the interior — e.g. writing |2x − k/3| = 2x + k/3 instead of −2x + k/3.
  • Forgetting to verify each candidate solution lies in the interval defining its case, retaining solutions that don't actually satisfy the original equation.
  • Mishandling boundary points (e.g. using strict < where ≤ is required) so that an endpoint solution is double-counted or missed.
  • Treating a case that yields a contradiction (like k/3 = k) as a solvable equation instead of declaring 'no solution in this case'.

A-level Scope Notes

This is core Edexcel IAL Pure (P2/P3) modulus material — the piecewise definition |x| = x for x ≥ 0, |x| = −x for x < 0 is assumed, not in the formula booklet. Stay with linear interiors at IAL level; nested moduli or |f(x)| with non-monotonic f are out of scope.
