Back-substitute into the simplest already-isolated equation to recover the remaining unknown

Family: Back Substitution Frequency: 2 questions

Trigger — When to use

After solving a simultaneous or multi-equation system for one variable (e.g. r, x, or a parameter), you still need the partner variable(s) to complete the answer. Several equations are available — the originals and any intermediate rearrangements derived earlier in the question. Recognise this trigger whenever a part (b) or later part says 'hence find...' having just asked you to determine one quantity, or when a coordinate pair / full solution set is required but only one coordinate has been computed. The cue is: 'I have a clean numerical value for one unknown; multiple equations could yield the other; which is cheapest?'

Motivation — Why this works

Back-substitution exploits work already done: an earlier rearrangement (often from part (a)) typically isolates one variable explicitly, so plugging in costs only arithmetic. Choosing the equation that is already solved for the target variable, and that is polynomial rather than rational/surd-laden, minimises algebraic manipulation and error risk. Mathematically this is just the substitution principle for systems — any solution must satisfy every equation, so we pick the one that delivers the answer with fewest operations. The strategic content is the *choice*, not the substitution itself.

Thought Process — The step-by-step reasoning

  1. Notice that one variable is now known numerically (e.g. r = 1/4, or x = ±√10), and the partner variable is still required to finish the question.
  2. List every equation that links the two variables: the two originals, plus any rearranged form produced earlier (e.g. a = 64(1−r), or y = x³ − 6x).
  3. Rule out equations that would force re-isolation of the target variable (dividing through, clearing fractions, taking roots) — these add steps and surface errors, especially with surds or fractions.
  4. Commit to the equation that is already explicit in the target variable AND polynomial / linear in the known variable; this is almost always the rearrangement produced in the earlier sub-part.
  5. Substitute the numerical value(s); if there are multiple cases (e.g. ±√10), process them one at a time and keep signs explicit.
  6. Present the answer in the form the question demands — a single value, a coordinate pair, or a list of pairs.

Mechanic — Concrete operations

  1. Identify the target unknown and the value(s) of the known variable.
  2. Select the equation already isolated for the target unknown (preferring polynomial form over rational/surd form).
  3. Substitute each value of the known variable carefully, retaining signs and surd powers (e.g. (−√10)³ = −10√10).
  4. Simplify to a closed form.
  5. If multiple cases exist, repeat for each and assemble the full solution set (e.g. coordinate pairs).

Worked Micro-Example

Given $a = 64(1-r)$ from part (a) and having found $r = \tfrac{1}{4}$ in part (b): substitute directly, $a = 64\left(1 - \tfrac{1}{4}\right) = 64 \times \tfrac{3}{4} = 48$. No need to revisit $S_3 = a + ar + ar^2 = 63$.

Variants — Same move, different surface

Target variable not previously isolated (no helpful rearrangement exists)
Pick the original equation that is linear in the target, and isolate in one step before substituting (e.g. from $S_\infty = a/(1-r)$ get $a = S_\infty(1-r)$).
Two-variable simultaneous system with surd solutions for x
Substitute into the polynomial relation $y = f(x)$ rather than a rational rearrangement, to avoid surds in denominators.
Trigonometric or parametric simultaneous equations
Same principle: choose whichever equation is already solved for the remaining unknown most cleanly (often the parametric definition itself).

Common Misreadings

  • Substituting back into the equation that was used to eliminate the target variable (e.g. the quartic in x), which yields no information about the partner variable.
  • Treating the question as finished after finding only one variable, when a full solution pair / set is required.

Pitfalls

  • Arithmetic slips with simple fractions, e.g. writing $64 \times \tfrac{3}{4} = 14$ instead of $48$.
  • Sign errors when cubing or odd-powering a negative surd: $(-\sqrt{10})^3 = -10\sqrt{10}$, not $10\sqrt{10}$.
  • Forgetting to present results as coordinate pairs when 'solve the simultaneous equations' is asked.
  • Choosing a more complicated equation out of habit and introducing avoidable algebra.

A-level Scope Notes

Entirely within Edexcel IAL Pure scope (P1–P4). No formula booklet entry is invoked beyond standard geometric series formulae $S_n = a(1-r^n)/(1-r)$ and $S_\infty = a/(1-r)$ where relevant. Avoid generalising to systems requiring matrix inversion or numerical methods — those belong to Further Pure.
