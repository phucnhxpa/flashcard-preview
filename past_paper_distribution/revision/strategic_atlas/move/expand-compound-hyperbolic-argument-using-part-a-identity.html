Expand compound hyperbolic argument using part (a) identity

Family: Use Earlier Result Frequency: 1 questions

Trigger — When to use

An equation or expression contains a hyperbolic function of a sum, e.g. cosh(x + ln k) or sinh(x + a), and an earlier part of the question has just proven (or asked you to prove) the corresponding compound angle identity. The word 'Hence' or 'Use part (a)' explicitly signals the linkage. Numerical values of cosh and sinh of the constant offset are typically clean (e.g. cosh(ln 2) = 5/4).

Motivation — Why this works

The compound argument hides the unknown x inside a single transcendental function, blocking algebraic manipulation. Expanding via cosh(A+B) = cosh A cosh B + sinh A sinh B separates x from the constant, turning the LHS into a linear combination of cosh x and sinh x with known numerical coefficients. This converts the problem into a solvable equation in cosh x and sinh x, after which standard techniques (dividing by cosh x to get tanh x, or converting to exponentials) finish the job.

Thought Process — The step-by-step reasoning

  1. I notice cosh(x + ln 2) — a compound argument that must be broken apart before I can isolate x.
  2. I rule out jumping straight to exponential definitions: it's valid but messier, and the 'Hence' instruction means examiner expects the part (a) identity.
  3. I commit to the compound angle expansion because part (a) hands me the exact tool, and cosh(ln 2), sinh(ln 2) are clean fractions.
  4. First operation: write cosh(x + ln 2) = cosh x cosh(ln 2) + sinh x sinh(ln 2), then substitute the numerical values.
  5. Tactically I keep fractions (5/4, 3/4) rather than decimals, and after substitution I plan to collect cosh x and sinh x terms to reduce to a tanh x equation.

Mechanic — Concrete operations

  1. Identify A = x, B = ln 2 (or the relevant constant offset) in the proven identity.
  2. Write cosh(x + ln 2) = cosh x · cosh(ln 2) + sinh x · sinh(ln 2).
  3. Substitute pre-computed values cosh(ln 2) = 5/4, sinh(ln 2) = 3/4.
  4. Set the expanded LHS equal to the original RHS and collect cosh x, sinh x terms.
  5. Divide through by cosh x (valid since cosh x > 0) to obtain a linear equation in tanh x, then solve.

Worked Micro-Example

Solve $\cosh(x+\ln 2) = 5\sinh x$. Expand: $\cosh x\cosh(\ln 2) + \sinh x\sinh(\ln 2) = 5\sinh x$. Substitute $\cosh(\ln 2)=\tfrac54$, $\sinh(\ln 2)=\tfrac34$: $\tfrac54\cosh x + \tfrac34\sinh x = 5\sinh x$, so $\tfrac54\cosh x = \tfrac{17}{4}\sinh x$, giving $\tanh x = \tfrac{5}{17}$ and $x = \tfrac12\ln\tfrac{22}{12} = \tfrac12\ln\tfrac{11}{6}$.

Variants — Same move, different surface

Equation involving sinh(x + ln k)
Use sinh(A+B) = sinh A cosh B + cosh A sinh B; otherwise identical workflow.
Double-angle form like cosh(2x) embedded in an equation
Use cosh(2x) = 2cosh²x − 1 (or 1 + 2sinh²x) from an earlier part to reduce to a quadratic in cosh x or sinh x.
Trigonometric analogue: cos(x + α) = k after proving the cos compound formula
Apply cos(A+B) = cos A cos B − sin A sin B; mind the minus sign.
R cos(x − α) form for a sinh x + b cosh x
Run the expansion in reverse — match coefficients of cosh x, sinh x to extract R and α.

Common Misreadings

  • Treating 'Hence' as optional and switching straight to exponential definitions, which sacrifices method marks.
  • Confusing the structure with the trig identity for cos(A+B) and inserting a minus sign between the two terms.
  • Forgetting that part (a) has already supplied the numerical values of cosh and sinh of the constant, and recomputing them inefficiently.

Pitfalls

  • Sign error: writing cosh(A+B) = cosh A cosh B − sinh A sinh B (the cos formula).
  • Arithmetic slips when combining fractions like 5/4 cosh x + 3/4 sinh x with the RHS.
  • Dividing by sinh x when sinh x could be zero, instead of dividing by cosh x which is always positive.

A-level Scope Notes

Fully within Edexcel IAL Further Pure (Further Maths) hyperbolic functions: compound angle identities for sinh and cosh, exact values via cosh(ln k) = (k + 1/k)/2, and solving hyperbolic equations using tanh x or exponential form.
