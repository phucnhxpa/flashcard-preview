Algebra & Functions

Rewrite both sides to a common base, then equate exponents

Family: Common Base Reduction Frequency: 1 questions

Trigger — When to use

Fires when an equation equates two exponential expressions whose bases are different but are integer powers of a common base, e.g. $2^x = 4^{x-6}$, $9^{2x} = 27^{x+1}$, or $8^{x} = 32^{x-1}$. The cue is recognising that each base can be written as $b^k$ for some shared $b$. Also applies when one side is a constant that is itself a power of the chosen base. If no common base exists (e.g. $2^x = 5^{x-1}$), this move is ruled out and logarithms are needed instead.

Motivation — Why this works

The injectivity of $a^x$ for $a>0$, $a\neq 1$ guarantees that $a^m = a^n \iff m = n$. Rewriting both sides with the same base therefore collapses an exponential equation into an equation in the exponents, usually linear, which is directly solvable. This keeps answers exact (no logs, no decimals) and avoids the algebraic overhead of $\log$-laws. It is the cleanest A-level technique whenever the structure permits it.

Thought Process — The step-by-step reasoning

  1. Notice that the two bases on either side are not equal, so exponents cannot be compared directly yet.
  2. Rule out logarithms as overkill once it is spotted that both bases are powers of the same integer (here, 2 and 4 are both powers of 2).
  3. Commit to common-base reduction because it yields a one-step rewrite and a linear equation in $x$.
  4. Bridge: choose the smallest natural base (here $2$), and use $(b^k)^m = b^{km}$ to rewrite the side with the larger base.
  5. Tactically, bracket the exponent before multiplying through (write $2^{2(x-6)}$ first, then expand to $2^{2x-12}$) to avoid losing the $-12$.

Mechanic — Concrete operations

  1. Identify a common base $b$ such that each side's base is $b^k$ for some integer $k$.
  2. Rewrite each side using $(b^k)^E = b^{kE}$, keeping the new exponent fully bracketed.
  3. Expand the new exponents to standard linear (or polynomial) form.
  4. Apply $b^M = b^N \implies M = N$ to obtain an equation in the original variable.
  5. Solve the resulting equation and, if required, substitute back into the original expression.

Worked Micro-Example

Solve $2^x = 4^{x-6}$.\ \ Write $4^{x-6} = (2^2)^{x-6} = 2^{2(x-6)} = 2^{2x-12}$.\ \ Then $2^x = 2^{2x-12} \Rightarrow x = 2x - 12 \Rightarrow x = 12$.\ \ Hence $y = 2^{12} = 4096$, giving the point $(12, 4096)$.

Variants — Same move, different surface

$8^{x} = 32^{x-1}$
Use base $2$: $8 = 2^3$, $32 = 2^5$, giving $3x = 5(x-1)$.
$9^{2x} = 27^{x+1}$
Use base $3$: $9 = 3^2$, $27 = 3^3$, giving $4x = 3(x+1)$.
$\left(\tfrac{1}{4}\right)^{x} = 8^{x+2}$
Use base $2$ with negative powers: $\tfrac{1}{4} = 2^{-2}$, $8 = 2^3$, giving $-2x = 3(x+2)$.
$5^{2x-1} = 125$
Rewrite the constant as $5^3$, then equate $2x-1 = 3$.
$2^x = 5^{x-1}$ (no common base)
Common-base reduction fails; switch to taking $\ln$ or $\log$ of both sides.

Common Misreadings

  • Reading $4^{x-6}$ as $4^x - 4^6$, splitting the exponent across subtraction.
  • Thinking the equation must be solved with logarithms because it is 'exponential', missing the common-base shortcut.
  • Believing the bases must already match and giving up when they look different.

Pitfalls

  • Forgetting to bracket the exponent: writing $(2^2)^{x-6} = 2^{2x-6}$ instead of $2^{2x-12}$.
  • Equating bases instead of exponents (e.g. concluding $2 = 4$).
  • Applying $a^m = a^n \Rightarrow m = n$ without checking $a > 0$ and $a \neq 1$.
  • Choosing an awkward base (e.g. base $4$ when base $2$ is cleaner), causing fractional exponents that invite arithmetic slips.

A-level Scope Notes

Standard Edexcel IAL Pure Maths technique (P1/P2 indices and exponentials), used routinely before logarithms are introduced and remaining the preferred method whenever a common base exists.
