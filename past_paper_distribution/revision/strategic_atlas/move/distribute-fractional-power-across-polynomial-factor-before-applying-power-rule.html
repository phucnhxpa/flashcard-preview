Integration

Distribute fractional power across polynomial factor before applying power rule

Family: Algebraic Reduction Frequency: 1 questions

Trigger — When to use

Fires whenever an integrand has the shape (polynomial in u)·u^(p/q), typically appearing right after a substitution like u = ax+b that converted √(ax+b) into a clean u^(1/2). Cue phrases: 'product of a linear (or low-degree) polynomial and a surd/fractional power', 'no standard antiderivative for the product as written'. Also fires when √x or ∛x multiplies a bracket like (x−1), (x+2), or (x²+3x).

Motivation — Why this works

The power rule ∫u^n du = u^(n+1)/(n+1) only handles single monomial powers, not products. Because multiplication distributes over addition, (u−1)·u^(1/2) = u^(3/2) − u^(1/2) is identically equal to a sum of monomials, each integrable in one line. This is strictly cheaper than integration by parts whenever the polynomial factor is low-degree, and avoids the wrong instinct of integrating each factor separately.

Thought Process — The step-by-step reasoning

  1. Notice the integrand is a product: a polynomial bracket times a fractional power, e.g. (u−1)u^(1/2) — no direct antiderivative formula matches.
  2. Rule out integrating the two factors separately and multiplying (that violates ∫fg ≠ (∫f)(∫g)); rule out parts as overkill for a linear bracket.
  3. Commit to expansion because each term of (polynomial)·u^(p/q) becomes a single monomial power, exactly the shape the power rule consumes.
  4. Bridge: rewrite the surd in exponent form u^(p/q), then multiply through the bracket using a^m·a^n = a^(m+n).
  5. Tactically, keep any constant factor (like 1/16 from du-scaling) outside the integral, integrate each monomial, and simplify the reciprocal of the new exponent at the end.

Mechanic — Concrete operations

  1. Rewrite any surd in fractional-exponent form, e.g. √u = u^(1/2).
  2. Distribute the polynomial across the power: (u−1)·u^(1/2) = u^(3/2) − u^(1/2).
  3. Pull out any overall constant from the substitution (e.g. 1/16) so the integral reads k∫(u^(3/2) − u^(1/2)) du.
  4. Apply the power rule term by term: ∫u^n du = u^(n+1)/(n+1).
  5. Simplify each '÷ fraction' as '× reciprocal', e.g. u^(5/2)/(5/2) = (2/5)u^(5/2).
  6. Add +C, then if required substitute back to the original variable.

Worked Micro-Example

Compute ∫ x√(x−1) dx. Let u = x−1, du = dx, so x = u+1 and the integrand becomes (u+1)u^(1/2) = u^(3/2) + u^(1/2). Hence ∫(u^(3/2)+u^(1/2)) du = (2/5)u^(5/2) + (2/3)u^(3/2) + C = (2/5)(x−1)^(5/2) + (2/3)(x−1)^(3/2) + C.

Variants — Same move, different surface

Cubic-root surface, e.g. ∫(u²+3u)u^(1/3) du
Distribute to get u^(7/3) + 3u^(4/3); power rule gives (3/10)u^(10/3) + (9/7)u^(7/3) + C.
Negative fractional power, e.g. ∫(u+2)u^(−1/2) du
Distribute to u^(1/2) + 2u^(−1/2); integrate to (2/3)u^(3/2) + 4u^(1/2) + C.
No substitution needed, e.g. ∫ x(√x − 1) dx directly
Distribute x across the bracket: x^(3/2) − x, then integrate term by term.
Higher-degree polynomial factor, e.g. (u³−2u+1)u^(1/2)
Distribution still works (gives u^(7/2), u^(3/2), u^(1/2)); only switch to parts/another substitution if the bracket itself is non-polynomial.

Common Misreadings

  • Treating (u−1)u^(1/2) as two factors to integrate separately and then multiply, i.e. assuming ∫fg = (∫f)(∫g).
  • Reaching for integration by parts when simple distribution suffices for a linear (or low-degree) bracket.
  • Trying a second substitution like v = √u when straightforward expansion already linearises the problem.

Pitfalls

  • Adding exponents incorrectly: writing u·u^(1/2) = u^(1/2) instead of u^(3/2).
  • Inverting the new exponent wrongly: u^(5/2)/(5/2) becoming (5/2)u^(5/2) rather than (2/5)u^(5/2).
  • Forgetting to carry the overall constant (e.g. the 1/16 from du-scaling) through to the final answer.
  • Forgetting to substitute back to the original variable after integrating in u, when an indefinite answer in x is required.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (C4/P3 integration): power rule with rational exponents and integration by simple substitution u = ax+b, with no techniques beyond the specification.
