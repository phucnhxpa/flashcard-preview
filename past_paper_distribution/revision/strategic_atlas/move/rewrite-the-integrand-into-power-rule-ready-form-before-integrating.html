Integration

Rewrite the integrand into power-rule-ready form before integrating

Family: Algebraic Rewrite Before Integration Frequency: 2 questions

Trigger — When to use

Fires whenever the integrand is written in a form that hides its true power structure: a reciprocal power like 1/u², 1/x³, a surd like 1/√x or ⁴√x, or a product/quotient that can be split into terms of the form k·u^n. Equivalently: you see ∫ dx and the integrand is NOT yet a clean sum of x^n (or u^n after substitution) terms. The cue is 'the power rule almost applies, but the surface notation is in the way.' If the integrand instead has the form 1/(linear) you would use ln, but anything with exponent ≠ −1 on a single power signals this rewrite.

Motivation — Why this works

The power rule ∫ u^n du = u^{n+1}/(n+1) + C (for n ≠ −1) requires the integrand displayed as u^n. Reciprocals and roots are just negative or fractional exponents in disguise; rewriting them exposes n explicitly so the rule becomes mechanical. This avoids guessing antiderivatives and prevents the common slip of treating 1/u² as if it were 1/u (which would call for ln). The underlying principle is simply the identity 1/u^k = u^{−k} and √[m]{u} = u^{1/m}.

Thought Process — The step-by-step reasoning

  1. Notice the integrand contains a reciprocal or radical (e.g. 1/u², 1/√x, x/(x²+1)^3) rather than a clean u^n.
  2. Rule out ln: that only applies when the exponent is exactly −1; here the exponent will be −2, −3, 1/2, etc., so the power rule is the correct tool.
  3. Rule out trig/substitution detours: no nonlinear inner function remains once any obvious u-substitution has been done, so algebraic rewriting is enough.
  4. Commit to the rewrite: convert every reciprocal to a negative exponent and every root to a fractional exponent, so the integrand becomes a sum of k·u^n terms.
  5. Decide the new exponent n+1 and check it is non-zero before dividing — this is the moment to catch sign errors (e.g. n = −2 gives n+1 = −1, so dividing flips the sign).
  6. After integrating, convert back to the original surface form (reciprocal/root) and back-substitute u if a u-substitution was used.

Mechanic — Concrete operations

  1. Identify each problematic factor and rewrite: 1/u^k → u^{−k}; √u → u^{1/2}; ⁿ√(u^m) → u^{m/n}.
  2. Pull constants outside the integral: ∫ (a/u^k) du = a ∫ u^{−k} du.
  3. Apply the power rule: ∫ u^n du = u^{n+1}/(n+1) + C, provided n ≠ −1.
  4. Simplify the resulting exponent and constant; convert back to reciprocal/radical notation if the answer is conventionally written that way.
  5. If a substitution u = g(x) was used, back-substitute to express the answer in x and add + C.

Worked Micro-Example

Compute $\int \frac{3}{2u^2}\,du$ where $u = 2x^2+3$. Rewrite: $\frac{3}{2}\int u^{-2}\,du$. Apply power rule: $\frac{3}{2}\cdot\frac{u^{-1}}{-1} = -\frac{3}{2u}$. Back-substitute: $-\frac{3}{2(2x^2+3)} + C$.

Variants — Same move, different surface

Reciprocal of a higher power, e.g. ∫ 1/x³ dx
Rewrite as ∫ x^{−3} dx = x^{−2}/(−2) = −1/(2x²) + C.
Reciprocal of a square root, e.g. ∫ 1/√x dx
Rewrite as ∫ x^{−1/2} dx = x^{1/2}/(1/2) = 2√x + C.
Root in the numerator, e.g. ∫ ⁴√(x³) dx
Rewrite as ∫ x^{3/4} dx = x^{7/4}/(7/4) = (4/7) x^{7/4} + C.
Quotient that splits, e.g. ∫ (x² + 1)/x² dx
Split first: ∫ (1 + x^{−2}) dx = x − 1/x + C — same rewrite principle, applied termwise.

Common Misreadings

  • Mistaking 1/u² for 1/u and writing ln|u| instead of applying the power rule with n = −2.
  • Treating √u as if it integrates to √u itself, rather than rewriting as u^{1/2} and using the power rule.

Pitfalls

  • Sign error from dividing by (n+1) when n is negative: e.g. for n = −2, n+1 = −1, so the answer carries a minus sign.
  • Forgetting to back-substitute u → g(x) when a substitution was used earlier in the problem.
  • Off-by-one on fractional exponents: for n = 1/2, n+1 = 3/2, and dividing by 3/2 means multiplying by 2/3 — not 1/2.

A-level Scope Notes

This is core Edexcel IAL Pure (P2/P3) integration technique; the power rule ∫ x^n dx = x^{n+1}/(n+1) + C (n ≠ −1) is in the formula booklet. No need to invoke gamma functions or distributional reasoning — stay with real exponents n ≠ −1.
