Strategic Atlas / Differentiation / Tangency Gradient Match
Differentiation

Equate derivative to line's gradient to locate tangent point

Family: Tangency Gradient Match Frequency: 1 questions

Trigger — When to use

Fires when a question states that a line y = mx + c is tangent to a curve y = f(x) and asks for an unknown constant (often c, or sometimes m, or a parameter inside the curve). The gradient of the line is directly readable from the coefficient of x. No specific point of contact is given — it must be located. Phrasings include 'is a tangent to', 'touches the curve', or 'meets the curve at exactly one point' within a calculus chapter.

Motivation — Why this works

At a point of tangency, the curve and line share the same gradient, so f'(x) must equal m at that x-coordinate. This converts the geometric 'touching' condition into an algebraic equation in x alone, pinpointing the tangent point. Once x is known, substituting into the curve gives the y-coordinate, and substituting both into the line equation determines the unknown constant. This is the calculus-route alternative to the discriminant-zero method and is preferred when differentiation is the chapter's focus.

Thought Process — The step-by-step reasoning

  1. Notice the keyword 'tangent' alongside a line in slope-intercept form — the coefficient of x is the gradient m.
  2. Rule out brute substitution (curve = line, discriminant = 0); since the chapter is differentiation, the examiner expects f'(x) = m.
  3. Commit to the gradient-match because it produces a single equation in x, simpler than a quadratic discriminant analysis.
  4. Bridge: differentiate the curve to get f'(x), then set f'(x) = m and solve for x.
  5. Tactically: once x is found, compute y on the curve, then enforce that this point lies on the line to extract the unknown constant.

Mechanic — Concrete operations

  1. Read the line's gradient m as the coefficient of x.
  2. Differentiate the curve to obtain f'(x).
  3. Set f'(x) = m and solve for x — this is the x-coordinate of the tangent point.
  4. Substitute x into the curve equation to find y.
  5. Substitute (x, y) into the line equation and solve for the unknown constant.

Worked Micro-Example

Find c such that y = 4x + c is tangent to y = 2x^2 + 8x + 3.\n dy/dx = 4x + 8. Set 4x + 8 = 4 ⇒ x = -1.\n Then y = 2(-1)^2 + 8(-1) + 3 = -3.\n On the line: -3 = 4(-1) + c ⇒ c = 1.

Variants — Same move, different surface

Tangent to an implicit curve (e.g., x^2 + xy + y^2 = k)
Differentiate implicitly to get dy/dx in terms of x and y, then equate to m; combine with the curve equation to solve simultaneously.
Line gradient unknown but intercept fixed (find m)
Same logic — set f'(x) = m, but now m is the unknown; use the 'point lies on line' equation as the second relation.
Normal rather than tangent
Set f'(x) = -1/m (negative reciprocal of the line's gradient) instead of m.
Parametric curve x = g(t), y = h(t)
Compute dy/dx = h'(t)/g'(t), set equal to m, solve for t, then recover the point.
Unknown parameter inside the curve (e.g., y = x^2 + ax + b)
Use both conditions — gradient match and point on line — to set up two equations in the parameters.

Common Misreadings

  • Confusing 'tangent to the curve' with 'tangent at a stated point' — here the contact point is unknown and must be found via the gradient condition.
  • Treating the problem as a generic intersection and solving curve = line for two roots, missing the tangency (repeated-root) requirement.
  • Misreading the line's gradient when it is given in implicit form like 4x - y + c = 0 instead of slope-intercept form.

Pitfalls

  • Differentiation slips: forgetting to drop the constant term, or mishandling the coefficient of x^2 (e.g., 2x^2 → 2x instead of 4x).
  • Equating f'(x) to c (the intercept) instead of to m (the gradient).
  • Forgetting the second step — finding x is not the answer; the unknown constant requires substituting back.
  • Sign errors when the line is rearranged from a non-standard form before reading off m.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P1/P2 differentiation of polynomials; P3/P4 for implicit and parametric variants). No tools beyond standard differentiation rules are required.
