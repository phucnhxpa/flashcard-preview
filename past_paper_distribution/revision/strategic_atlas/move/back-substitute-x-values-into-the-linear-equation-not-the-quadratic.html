Coordinate Geometry

Back-substitute x-values into the linear equation, not the quadratic

Family: Back Substitution Into Simpler Expression Frequency: 1 questions

Trigger — When to use

Fires after solving a line–curve (or simpler–harder pair of curves) intersection problem where x-coordinates have just been found and y-coordinates are still needed. The cue is that two equivalent expressions for y are available — typically one linear (y = mx + c) and one quadratic (y = ax² + bx + c) — and at intersection points both give the same y. Common phrasings: 'find the points of intersection', 'find the coordinates where the line meets the curve'.

Motivation — Why this works

At an intersection point both equations hold simultaneously, so back-substitution into either yields the same y. Choosing the simpler (linear) expression minimises arithmetic operations: one multiplication and one addition versus a square, two multiplications, and two additions for the quadratic. Fewer operations means fewer opportunities for sign errors, especially when x is negative or non-integer. This is a pure efficiency/accuracy choice with no mathematical cost.

Thought Process — The step-by-step reasoning

  1. Notice that two expressions for y are on the table and either can deliver the y-coordinate.
  2. Rule out the quadratic route because squaring x (particularly negative x) and summing three terms invites arithmetic slips.
  3. Commit to the linear equation: it is the lowest-operation path to the same answer.
  4. Plan to substitute each x-value in turn, keeping the original (un-rearranged) form of the line to avoid propagating earlier algebra.
  5. Decide to verify one point in the quadratic only if a sanity check is desired.

Mechanic — Concrete operations

  1. Identify the original linear equation from the problem (not a rearranged form used during elimination).
  2. Substitute the first x-value into y = mx + c and simplify.
  3. Substitute the second x-value the same way.
  4. Pair each y with its x to write the intersection points (x, y).
  5. Optional: verify by substituting one (x, y) into the quadratic.

Worked Micro-Example

Line $y = 5x + 75$ meets curve $y = x^2 + 3x + 27$ at $x = 8$ and $x = -6$. Using the line: $y = 5(8) + 75 = 115$ and $y = 5(-6) + 75 = 45$. Intersection points: $(8, 115)$ and $(-6, 45)$.

Variants — Same move, different surface

Two quadratics intersecting, one with fewer/simpler terms
Substitute into the quadratic with smaller coefficients or fewer non-zero terms.
Line meets a circle $(x-a)^2 + (y-b)^2 = r^2$
Always back-substitute x-values into the line, never the circle equation, to avoid a second quadratic in y.
Parametric curve intersected with another locus
Substitute the parameter values into the parametric equations directly rather than converting to Cartesian.
Tangent problems where a repeated root x is found
Use the linear (tangent) equation for the single y-value at the point of tangency.

Common Misreadings

  • Believing the two equations might give different y-values and so 'must' use the curve.
  • Thinking the quadratic is 'more accurate' because it defines the curve, when both are equally valid at intersection points.

Pitfalls

  • Substituting into a rearranged equation (e.g. the combined equation set to zero) instead of an original — always return to the given line or curve.
  • Sign errors with negative x-values, particularly if the quadratic route is chosen and $x^2$ is mishandled.
  • Pairing y-values with the wrong x when listing the final coordinates.

A-level Scope Notes

Standard Edexcel IAL Pure technique for line–curve intersection problems in P1/P2 coordinate geometry, including tangents to circles and parabolas.
