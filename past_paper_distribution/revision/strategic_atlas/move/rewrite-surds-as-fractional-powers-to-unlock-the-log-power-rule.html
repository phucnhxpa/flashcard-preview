Rewrite surds as fractional powers to unlock the log power rule

Family: Exponent Rewrite Frequency: 1 questions

Trigger — When to use

A logarithm contains a surd or root in its argument, e.g. $\log_a(\sqrt{a})$, $\log(\sqrt[3]{x^2})$, or $\ln(\sqrt{2x+1})$. The standard log laws (product, quotient, power) provide no direct handle on radical notation, so the argument must first be rewritten as an explicit power. Also fires when a reciprocal $1/x$ inside a log needs to be moved out as a coefficient $-1$.

Motivation — Why this works

The power rule $\log_a(x^n) = n\log_a(x)$ requires an explicit exponent. By the index identity $\sqrt[n]{x^m} = x^{m/n}$, every surd is secretly a power, so converting the notation makes the rule applicable. This single rewrite turns an opaque surd into a coefficient that can be extracted, often collapsing the expression to a numerical value (e.g. $\log_a(a) = 1$).

Thought Process — The step-by-step reasoning

  1. Notice a root symbol sitting inside a logarithm — none of the three log laws act on $\sqrt{\,}$ directly.
  2. Rule out trying to evaluate the surd numerically or applying the product rule prematurely; the surd is the obstruction.
  3. Commit to converting radical notation to index notation because the power rule is the only law that introduces a coefficient out front.
  4. Rewrite $\sqrt[n]{x^m}$ as $x^{m/n}$, taking care with the fraction (denominator = root index, numerator = inner power).
  5. Apply the power rule to bring the fractional exponent in front, then simplify the remaining $\log_a(\text{base})$ — often equal to $1$.

Mechanic — Concrete operations

  1. Identify the surd inside the log and read off its root index $n$ and inner power $m$.
  2. Rewrite $\sqrt[n]{x^m}$ as $x^{m/n}$ (so $\sqrt{a} = a^{1/2}$).
  3. Apply the power rule: $\log_a(x^{m/n}) = \tfrac{m}{n}\log_a(x)$.
  4. Evaluate or simplify the residual logarithm (e.g. $\log_a(a) = 1$).
  5. Combine the coefficient with the simplified log to give the final value.

Worked Micro-Example

Simplify $\log_a(\sqrt{a})$.\\ Rewrite the surd: $\sqrt{a} = a^{1/2}$.\\ Apply the power rule: $\log_a(a^{1/2}) = \tfrac{1}{2}\log_a(a)$.\\ Since $\log_a(a) = 1$, the answer is $\tfrac{1}{2}$.

Variants — Same move, different surface

Cube or higher root, e.g. $\log(\sqrt[3]{x^2})$
Rewrite as $x^{2/3}$, then power rule gives $\tfrac{2}{3}\log(x)$.
Reciprocal inside a log, e.g. $\ln(1/x)$
Rewrite $1/x = x^{-1}$ so the power rule gives $-\ln(x)$, an alternative to the quotient rule.
Surd of a product, e.g. $\log(\sqrt{ab})$
Rewrite as $(ab)^{1/2}$, extract $\tfrac{1}{2}$, then apply the product rule: $\tfrac{1}{2}(\log a + \log b)$.
Surd inside an exponential equation, e.g. $\ln(\sqrt{x+1}) = 2$
Rewrite to $\tfrac{1}{2}\ln(x+1)=2$, then solve $\ln(x+1)=4$.

Common Misreadings

  • Treating $\sqrt{a}$ as if it were a separate factor that the product rule could split.
  • Believing the power rule does not apply because the exponent is fractional rather than an integer.
  • Reading $\log_a(\sqrt{a})$ as $\sqrt{\log_a(a)}$ — the root does not commute with the log.

Pitfalls

  • Writing $\sqrt{a}$ as $a^{2}$ instead of $a^{1/2}$ (inverting the index).
  • Forgetting that $\sqrt[n]{x^m} = x^{m/n}$, not $x^{n/m}$.
  • Dropping the bracket: writing $\log_a a^{1/2}$ as $(\log_a a)^{1/2}$ instead of $\tfrac{1}{2}\log_a a$.
  • After extracting the coefficient, failing to simplify $\log_a(a) = 1$ and leaving the answer un-evaluated.

A-level Scope Notes

Fully within Edexcel IAL P2/P3 logarithm and indices content; uses only the index law $\sqrt[n]{x^m}=x^{m/n}$ and the power rule for logarithms.
