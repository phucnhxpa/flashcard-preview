Algebra & Functions

Compare numerator and denominator degrees before partial fractions

Family: Degree Check Frequency: 1 questions

Trigger — When to use

A rational expression is handed over for partial fraction decomposition, integration of a rational function, or series expansion. Before writing any decomposition template, check whether the fraction is proper (deg numerator < deg denominator) or improper (deg numerator ≥ deg denominator). Cues include phrases like 'express in partial fractions', 'decompose', or seeing a numerator that visibly looks 'large' relative to the denominator. Also fires when the denominator is given in factored form and its total degree is not immediately obvious.

Motivation — Why this works

Partial fraction templates (A/(ax+b), (Ax+B)/(quadratic), etc.) are only valid for proper fractions; they cannot represent the polynomial part of an improper fraction. If deg(num) ≥ deg(denom), polynomial long division must extract a quotient polynomial plus a proper remainder before decomposition. Skipping this check leads to inconsistent simultaneous equations or wrong constants. Confirming the fraction is proper licenses you to write the template directly and saves an unnecessary division step.

Thought Process — The step-by-step reasoning

  1. Notice the task asks for partial fractions, so the very first reflex is: is this fraction proper?
  2. Rule out improperness by reading off the highest power in the numerator and summing the degrees of the factors in the denominator.
  3. Commit to 'proceed directly' once deg(num) < deg(denom) is confirmed; otherwise commit to long division first.
  4. Bridge to action: write the appropriate partial fraction template matching the denominator's factor structure (linear, repeated, irreducible quadratic).
  5. Tactically, when the denominator is factored like (2x+1)(x−1)², add exponents (1+2=3) rather than expanding fully — faster and less error-prone.

Mechanic — Concrete operations

  1. Read the highest power of x in the numerator → degree of numerator.
  2. For the denominator, sum the degrees contributed by each factor (each linear factor contributes 1, repeated linear (x−a)^k contributes k, irreducible quadratic contributes 2).
  3. Compare: if deg(num) < deg(denom), the fraction is proper — proceed to write the partial fraction template.
  4. If deg(num) ≥ deg(denom), perform polynomial long division first; decompose only the proper remainder fraction.

Worked Micro-Example

Decompose $\dfrac{10x^2+2x-2}{(2x+1)(x-1)^2}$. Numerator degree $=2$; denominator degree $=1+2=3$. Since $2<3$, fraction is proper, so write directly $\dfrac{A}{2x+1}+\dfrac{B}{x-1}+\dfrac{C}{(x-1)^2}$ and solve for $A,B,C$ — no long division needed.

Variants — Same move, different surface

Improper fraction such as $\dfrac{x^3+1}{(x-1)(x+2)}$
deg(num)=3 ≥ deg(denom)=2, so perform polynomial long division first, then decompose the proper remainder.
Equal degrees, e.g. $\dfrac{2x^2+3}{x^2-1}$
Still improper; divide to get a constant quotient plus proper remainder, then decompose the remainder.
Denominator with irreducible quadratic, e.g. $(x+1)(x^2+4)$
Total degree is $1+2=3$; remember the quadratic factor contributes 2 to the degree count and demands an $(Ax+B)$ numerator in the template.
Binomial expansion of a rational function for series
Same proper/improper check applies before expanding; if improper, split off the polynomial part first.

Common Misreadings

  • Assuming any rational expression is automatically ready for partial fractions without checking degrees.
  • Treating $(x-1)^2$ as a single degree-1 factor and undercounting denominator degree.
  • Confusing 'proper fraction' (degree-based) with 'fully simplified' (cancellation-based).

Pitfalls

  • Miscounting denominator degree when it is in factored form — forgetting to add exponents on repeated factors.
  • Forgetting that an irreducible quadratic factor contributes degree 2, not 1.
  • Performing long division but then decomposing the original fraction instead of the remainder.

A-level Scope Notes

Fully within Edexcel IAL Core Pure / P3-P4 partial fractions and integration of rational functions; long division of polynomials is also in scope when the fraction is improper.
