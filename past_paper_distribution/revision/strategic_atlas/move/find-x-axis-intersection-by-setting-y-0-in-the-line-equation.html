Coordinate Geometry

Find x-axis intersection by setting y = 0 in the line equation

Family: Axis Intersection Frequency: 1 questions

Trigger — When to use

Fires whenever a point is defined as the intersection of a given line (tangent, normal, chord, or general line) with the x-axis. Cues include phrases like 'meets the x-axis at Q', 'cuts the x-axis', or 'crosses Ox'. The line equation is already available in some explicit or implicit form involving x and y. The y-coordinate of the target point is forced to be 0, collapsing the problem to a single equation in x.

Motivation — Why this works

Any point on the x-axis has y = 0 by definition, so substituting this into the line equation eliminates y and leaves a one-variable linear equation in x. This is the most direct route to the intersection — no simultaneous solving, no projection, no parametrisation. It works because a non-horizontal line meets the x-axis at exactly one point, guaranteeing a unique solution. The resulting x-value is typically a clean function of any parameters present, which then feeds into distance, ratio, or locus calculations.

Thought Process — The step-by-step reasoning

  1. Notice the target point Q is described as lying on the x-axis — its y-coordinate is automatically 0.
  2. Rule out heavier methods: no need to solve a 2×2 system, no need for foot-of-perpendicular formulae or vector projection.
  3. Commit to substitution: replacing y by 0 in the line's equation reduces it instantly to a linear equation in x alone.
  4. Bridge to operation: identify the cleanest form of the line (point-slope, general, or normal form) and substitute y = 0 directly.
  5. Tactically, before dividing through by any factor (e.g. sinθ, cosθ, parameter t), check the domain ensures it is nonzero so the division is valid.

Mechanic — Concrete operations

  1. Take the line's equation in the form already given, e.g. 5x sinθ − 4y cosθ = 9 sinθ cosθ.
  2. Substitute y = 0 to obtain 5x sinθ = 9 sinθ cosθ.
  3. Check that any factor to be cancelled is nonzero (here sinθ ≠ 0 since 0 < θ < π/2).
  4. Divide through to isolate x: x = 9cosθ/5.
  5. Write Q as the ordered pair: Q = (9cosθ/5, 0).

Worked Micro-Example

The normal at P to a curve has equation $5x\sin\theta - 4y\cos\theta = 9\sin\theta\cos\theta$, with $0 < \theta < \pi/2$. To find where it meets the x-axis, set $y = 0$: $5x\sin\theta = 9\sin\theta\cos\theta$. Since $\sin\theta \neq 0$, divide by $5\sin\theta$ to get $x = \tfrac{9\cos\theta}{5}$. Hence $Q = \left(\tfrac{9\cos\theta}{5},\,0\right)$.

Variants — Same move, different surface

Tangent meets the x-axis at T
Same procedure: set y = 0 in the tangent's equation and solve for x.
Line meets the y-axis at R
Set x = 0 instead and solve for y; the roles of the variables swap.
Parametric line (x, y) = (x₀ + at, y₀ + bt) meeting the x-axis
Set y₀ + bt = 0, solve for t, then substitute back to find x.
Chord of an ellipse/parabola in cartesian form crossing Ox
Substitute y = 0 in the chord equation; same one-step elimination.
Normal to a conic at a parametric point, with subsequent distance to focus
After finding Q on the x-axis, distance to a focus also on the x-axis is |x_Q − x_F|, no Pythagoras needed.

Common Misreadings

  • Setting x = 0 instead of y = 0, confusing x-axis with y-axis.
  • Treating the intersection as needing a simultaneous-equation setup with 'y = 0' as a separate line and solving via substitution matrix unnecessarily.
  • Believing one must first rearrange the line into y = mx + c before substituting, when direct substitution into the given form is faster.

Pitfalls

  • Cancelling a trigonometric or parametric factor without checking it is nonzero on the stated domain.
  • Arithmetic slips when dividing both sides by a compound expression — keep the cancellation explicit.
  • Forgetting to write Q as a coordinate pair (x, 0); leaving the answer as just an x-value loses marks.
  • Mis-copying signs from the original line equation when substituting.

A-level Scope Notes

Standard technique within Edexcel IAL Pure Mathematics (P1–P4) and Further Pure for tangents, normals, and chords to lines, circles, parabolas, ellipses, and hyperbolas; uses only substitution and basic algebra.
