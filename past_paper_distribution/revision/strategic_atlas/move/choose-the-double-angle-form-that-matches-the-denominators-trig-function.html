Trigonometry

Choose the double-angle form that matches the denominator's trig function

Family: Identity Selection Frequency: 1 questions

Trigger — When to use

Fires when simplifying or proving an identity where a double-angle term (cos 2x or sin 2x) sits in a numerator over a single trig function like sin x or cos x. The target right-hand side is expressed purely in one function (e.g. cosec x, sec x, cot x), signalling that everything must collapse to that function. Cue phrases include 'show that ... = cosec x + ...' or integrals where a single-function answer is expected.

Motivation — Why this works

cos 2x has three equivalent forms: 1 − 2sin²x, 2cos²x − 1, and cos²x − sin²x. Dividing by sin x cleanly only when the numerator carries a standalone 1 and sin²x terms — i.e. the 1 − 2sin²x form — because 1/sin x = cosec x and sin²x/sin x = sin x. Choosing the form aligned with the denominator avoids mixed trig functions and produces a single-function result by construction.

Thought Process — The step-by-step reasoning

  1. Notice the denominator is a single function (sin x) and the target involves cosec x, so the numerator must yield a constant 1 plus sin-only terms after division.
  2. Rule out cos 2x = 2cos²x − 1 and cos²x − sin²x because they leave cos²x terms that don't cancel against sin x.
  3. Commit to cos 2x = 1 − 2sin²x because it is the unique form producing the required '1' for cosec x.
  4. Bridge: simultaneously rewrite sin 2x = 2 sin x cos x so that, when divided by cos x (if that appears) or sin x, a clean cancellation results.
  5. Tactically, split the fraction term-by-term before simplifying, so each piece reduces to a single trig function independently.

Mechanic — Concrete operations

  1. Identify the denominator's trig function (sin x or cos x) and the target function on the RHS.
  2. Select the cos 2x form whose terms match that function: 1 − 2sin²x for sin x denominators, 2cos²x − 1 for cos x denominators.
  3. If sin 2x is present, expand as 2 sin x cos x to enable cancellation with the denominator.
  4. Split the combined fraction into separate fractions over the single denominator.
  5. Simplify each piece using 1/sin x = cosec x (or 1/cos x = sec x) and cancel matching factors.
  6. Combine like terms to reach the target single-function expression.

Worked Micro-Example

Show that $\dfrac{\cos 2x + \sin 2x \cdot \tan x}{\sin x} = \csc x$ for relevant $x$.\\ Use $\cos 2x = 1 - 2\sin^2 x$ and $\sin 2x = 2\sin x\cos x$.\\ Numerator $= 1 - 2\sin^2 x + 2\sin x\cos x \cdot \tfrac{\sin x}{\cos x} = 1 - 2\sin^2 x + 2\sin^2 x = 1$.\\ Hence the expression $= \dfrac{1}{\sin x} = \csc x$.

Variants — Same move, different surface

Denominator is cos x and target involves sec x
Select cos 2x = 2cos²x − 1 so the constant −1 yields −sec x and the cos²x term reduces to cos x.
Integrating sin²x or cos²x
Use the linearising rearrangements sin²x = (1 − cos 2x)/2 or cos²x = (1 + cos 2x)/2 — the same identity choice driven by which power needs reducing.
Expression involves tan x or cot x denominators
Rewrite tan x = sin x/cos x first, then choose the cos 2x form matching the resulting single-function denominator.
Half-angle problems with cos x in the numerator and sin(x/2) in the denominator
Use cos x = 1 − 2sin²(x/2) to align with the sin(x/2) denominator.

Common Misreadings

  • Defaulting to cos 2x = cos²x − sin²x out of habit, which leaves both functions and blocks cancellation.
  • Treating all three cos 2x forms as interchangeable without inspecting the denominator first.
  • Assuming sin 2x must always be expanded — sometimes leaving it as sin 2x is fine if it appears on the RHS too.

Pitfalls

  • Sign errors when expanding 1 − 2sin²x, particularly inside a larger subtraction.
  • Forgetting the factor 2 in sin 2x = 2 sin x cos x.
  • Splitting the fraction incorrectly when the numerator has multiple terms — ensure every term keeps the original denominator.
  • Cancelling sin x from a sum rather than a factor (a common algebraic slip).

A-level Scope Notes

Entirely within Edexcel IAL Pure Mathematics (P3/P4) double-angle identities and proof-style 'show that' questions; also supports C34 integration of sin²x/cos²x via linearisation.
