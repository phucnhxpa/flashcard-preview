Algebra & Functions

Substitute x = 0 into the simplest available form for the y-intercept

Family: Strategic Factorisation Frequency: 1 questions

Trigger — When to use

A sketch or coordinate question asks explicitly for the y-intercept (or f(0)), and the function is available in more than one algebraic form — typically expanded, completed-square, or factored. Phrasings include 'state the coordinates where the curve crosses the y-axis', 'showing clearly the y-intercept', or simply needing f(0). Fires whenever a constant term is visible in one form but buried inside brackets in another.

Motivation — Why this works

Substituting x = 0 annihilates every term containing x, so the y-intercept is just whatever constant survives. In the expanded form the constant sits in plain view, giving the answer with zero arithmetic. Other forms give the same value but force you to evaluate brackets and combine terms — extra steps mean extra chances for sign or arithmetic error. Choosing the form where the answer is immediate is a deliberate efficiency and accuracy decision.

Thought Process — The step-by-step reasoning

  1. Notices the question asks for the y-intercept, i.e. the value of f(0).
  2. Rules out the completed-square or factored form because evaluating it at 0 still requires squaring/multiplying brackets.
  3. Commits to the expanded (original) form because the constant term is literally the answer.
  4. Bridges to operation: read off the constant term, or substitute x = 0 mentally to confirm.
  5. Records the point as (0, constant) and labels it on the sketch as the question demands.

Mechanic — Concrete operations

  1. Identify the form of f(x) with an exposed constant term (e.g. f(x) = 11 − 4x − 2x²).
  2. Set x = 0; all x and x² terms vanish.
  3. Read f(0) = constant term.
  4. State the y-intercept as the coordinate (0, f(0)).
  5. Label the point clearly on the sketch.

Worked Micro-Example

Given $f(x) = 11 - 4x - 2x^2 = 13 - 2(x+1)^2$, find the y-intercept. Use the expanded form: $f(0) = 11 - 0 - 0 = 11$. Y-intercept: $(0,\,11)$. (Check via completed square: $13 - 2(0+1)^2 = 13 - 2 = 11$ ✓, but this needed an extra step.)

Variants — Same move, different surface

Finding x-intercepts of a quadratic
Switch to the factored form $a(x-p)(x-q)$ or solve via completed-square form; the expanded form is usually the worst choice here.
Evaluating f(a) for a specific non-zero a
Pick whichever form makes (x − a) or the bracketed term collapse — e.g. use $13 - 2(x+1)^2$ at $x = -1$ to get the vertex value 13 instantly.
Cubic or higher polynomial sketch
Same principle: read the constant term of the expanded form for f(0); use factored form for roots.
Rational function $f(x) = \frac{p(x)}{q(x)}$
Substitute x = 0 into numerator and denominator separately, provided $q(0) \neq 0$; otherwise no y-intercept exists.
Transformed function such as $g(x) = f(x-2) + 3$
Compute $g(0) = f(-2) + 3$ using whichever form of f makes f(−2) cleanest.

Common Misreadings

  • Confusing the y-intercept with the maximum/minimum value (the constant outside the bracket in completed-square form).
  • Thinking the question wants the x-intercepts because 'where the curve crosses an axis' is read carelessly.
  • Believing every form must be substituted into to 'check' — wasting time on a single-mark read-off.

Pitfalls

  • Sign slip when the constant is negative, e.g. reading $-11$ as $11$.
  • Forgetting to write the answer as a coordinate pair $(0, c)$ when the question asks for coordinates.
  • Failing to label the point on the sketch — 'showing clearly' marks are lost without annotation.
  • Using the completed-square form and mishandling $-(x+1)^2$ at $x=0$, producing the wrong constant.

A-level Scope Notes

Standard technique throughout Edexcel IAL Pure Mathematics curve-sketching (P1–P4), applicable to polynomials, rational functions, exponentials and trig functions wherever f(0) is defined.
