Polynomial long division to decompose an improper rational fraction

Family: Polynomial Division Frequency: 1 questions

Trigger — When to use

Fires when a rational expression has numerator degree ≥ denominator degree (an improper fraction) and the target form is a polynomial plus a proper fraction, e.g. ax + b + c/(x − d). Typical cues: the question states 'find constants a, b, c, d such that...' or asks to integrate/partial-fraction an expression whose numerator degree is too high. Also fires as a prerequisite step before partial fractions or before integrating an improper rational function.

Motivation — Why this works

Any polynomial N(x) can be written uniquely as N(x) = Q(x)·D(x) + R(x) with deg(R) < deg(D), so N(x)/D(x) = Q(x) + R(x)/D(x). This is the polynomial analogue of the division algorithm for integers and guarantees an exact decomposition into a polynomial part plus a proper fractional remainder. The quotient Q(x) supplies the polynomial coefficients (here a, b) and the remainder R(x) supplies the numerator of the proper fraction (here c).

Thought Process — The step-by-step reasoning

  1. Notice the numerator degree (2) exceeds the denominator degree (1), and the target form is 'linear polynomial + proper fraction' — a textbook signature of long division output.
  2. Rule out direct partial fractions: that technique requires a proper fraction, so it cannot be applied until the improper part is extracted.
  3. Commit to long division (or equivalently, the identity 3x² − x + 5 ≡ (ax + b)(x − 4) + c) because it systematically produces both the quotient and remainder in one pass.
  4. Bridge to the first operation: divide leading term of numerator by leading term of denominator to obtain the first term of the quotient.
  5. Inside the move, track signs carefully on each subtraction, stop when the running remainder has degree < deg(D), and read a, b from Q(x) and c from R.

Mechanic — Concrete operations

  1. Divide leading terms: 3x² ÷ x = 3x. Multiply back: 3x(x − 4) = 3x² − 12x.
  2. Subtract: (3x² − x + 5) − (3x² − 12x) = 11x + 5.
  3. Divide leading terms: 11x ÷ x = 11. Multiply back: 11(x − 4) = 11x − 44.
  4. Subtract: (11x + 5) − (11x − 44) = 49. Degree 0 < degree 1, so stop.
  5. Assemble: (3x² − x + 5)/(x − 4) = 3x + 11 + 49/(x − 4), giving a = 3, b = 11, c = 49, d = 4.

Worked Micro-Example

Divide $\dfrac{3x^2 - x + 5}{x - 4}$. Step 1: $3x^2 \div x = 3x$; subtract $3x(x-4)=3x^2-12x$ to get $11x+5$. Step 2: $11x \div x = 11$; subtract $11(x-4)=11x-44$ to get remainder $49$. Hence $\dfrac{3x^2 - x + 5}{x - 4} = 3x + 11 + \dfrac{49}{x - 4}$.

Variants — Same move, different surface

Cubic numerator over linear denominator, e.g. (x³ + 2x − 1)/(x − 2)
Long division produces a quadratic quotient ax² + bx + c plus a constant remainder over (x − 2); same algorithm, one extra step.
Improper fraction with quadratic denominator before partial fractions, e.g. (2x³ + …)/(x² − 1)
Divide first to get a linear quotient plus a proper fraction (linear)/(x² − 1), then apply partial fractions to the proper part.
Equating-coefficients alternative
Write N(x) ≡ Q(x)·D(x) + R, expand, and match coefficients of each power of x to find the unknowns; algebraically equivalent to long division.
Numerator degree equal to denominator degree
Quotient is a constant; one division step suffices, leaving a proper fraction with the same denominator.

Common Misreadings

  • Trying to apply partial fractions directly to an improper fraction without first dividing.
  • Assuming the remainder must be zero and forcing a polynomial-only answer when a remainder fraction is required.
  • Reading off coefficients from N(x) directly (e.g. claiming a = 3, b = −1) instead of from the quotient produced by division.

Pitfalls

  • Sign errors in subtraction, especially mishandling −(−12x) = +12x.
  • Stopping the division too early, before the running remainder has degree strictly less than the divisor.
  • Forgetting to subtract the full product (e.g. dropping the constant term) at each stage.
  • Misaligning powers of x when the numerator is missing a term (failing to insert a 0 placeholder).

A-level Scope Notes

Standard Edexcel IAL Pure content: polynomial division of a polynomial by a linear or quadratic divisor, used in algebraic fractions, the factor/remainder theorem, partial fractions, and integration of improper rational functions.
