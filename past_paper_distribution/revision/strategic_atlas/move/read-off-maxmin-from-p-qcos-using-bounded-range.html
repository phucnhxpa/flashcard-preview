Trigonometry

Read off max/min from p + q·cos(·) using bounded range

Family: Range Of Trig Function Frequency: 1 questions

Trigger — When to use

Fires when an expression has been (or can be) reduced to the single-trig form $p + q\cos(\theta)$ or $p + q\sin(\theta)$, typically after applying the R-formula to $a\cos x + b\sin x$ or simplifying via a double-angle identity. Cues include phrases like 'find the maximum/minimum value', 'state the greatest value', or 'find the value of $x$ at which the maximum occurs'. Often appears as part (b) of a question whose part (a) established the R-form.

Motivation — Why this works

For all real $\theta$, $-1 \le \cos\theta \le 1$ (and similarly for $\sin$), so $p + q\cos\theta$ is squeezed between $p-|q|$ and $p+|q|$. The R-transformation was performed precisely to collapse a two-term oscillation into one bounded trig function, making the extreme values readable by inspection. This bypasses calculus entirely and gives an exact answer, plus the $x$-value at which it occurs by solving $\cos\theta = \pm 1$.

Thought Process — The step-by-step reasoning

  1. Notice the expression is now a constant plus a single scaled cosine (or sine), e.g. $3 + 35\cos(2x-\alpha)$.
  2. Rule out calculus — differentiating is slower and unnecessary once the form is single-trig.
  3. Commit to the bounded-range argument: $\cos$ ranges over $[-1,1]$, so the whole expression ranges over $[p-|q|,\,p+|q|]$.
  4. Decide which extreme is asked for and check the sign of $q$: if $q>0$, max uses $\cos=1$; if $q<0$, max uses $\cos=-1$.
  5. If asked for the $x$-value, solve $2x-\alpha = 0$ (or $=\pi$ for the other extreme) within the given interval, taking $\bmod\,2\pi$.

Mechanic — Concrete operations

  1. Confirm the expression is in the form $p + q\cos(kx-\alpha)$ (or with $\sin$).
  2. State the bound: $-1 \le \cos(kx-\alpha) \le 1$.
  3. Compute maximum $= p + q$ when $q>0$ (else $p - q$ when $q<0$); minimum is the other endpoint.
  4. If required, solve $kx-\alpha = 0$ (for max with $q>0$) or $kx-\alpha = \pi$ (for min) within the stated domain to find $x$.

Worked Micro-Example

Given $f(x) = 3 + 5\cos 2x + 12\sin 2x$, write the trig part as $13\cos(2x-\alpha)$ where $\tan\alpha = 12/5$. Then $f(x) = 3 + 13\cos(2x-\alpha)$. Since $-1 \le \cos(2x-\alpha) \le 1$, the maximum is $3+13 = 16$, attained when $2x-\alpha = 0$, i.e. $x = \tfrac{1}{2}\alpha$.

Variants — Same move, different surface

Expression in form $p + q\sin(kx+\alpha)$
Same logic: max $= p+|q|$ at $\sin = \mathrm{sgn}(q)$; solve $kx+\alpha = \pi/2$ for the $x$-value.
Reciprocal form $\dfrac{c}{p + q\cos(kx-\alpha)}$ with $p>|q|$
Maximum of the fraction occurs at the minimum of the denominator, i.e. when $\cos = -\mathrm{sgn}(q)$; reverse the extreme.
Negative coefficient, e.g. $p - q\cos(\theta)$ with $q>0$
Maximum occurs when $\cos\theta = -1$, giving $p+q$; do not blindly substitute $\cos = 1$.
Question asks for the value of $x$ giving the extreme
Set the argument equal to $0$ (max) or $\pi$ (min) modulo $2\pi$ and solve within the stated interval.
Range over a restricted domain where the extreme argument is unreachable
The unrestricted bound may not be attained; check that the required argument lies in the domain, otherwise evaluate at the endpoints.

Common Misreadings

  • Reporting the maximum of $\cos$ (i.e. $1$) instead of substituting back to get $p+q$.
  • Confusing the maximum of $f$ with the value of $x$ that produces it.
  • Treating $p + q\cos\theta$ as if its max were $p\cdot q$ or $|p|+|q|$ regardless of the structure.

Pitfalls

  • If $q<0$, the maximum occurs at $\cos = -1$, not $+1$ — track the sign of the amplitude.
  • When solving for $x$, forgetting to add $\alpha$ back or dividing incorrectly by $k$.
  • Ignoring a restricted domain: the theoretical bound may not be attained if the required argument lies outside it.
  • Mixing up degrees and radians when stating $\alpha$ or the $x$-value.

A-level Scope Notes

Standard Edexcel IAL P3/P4 technique following the R-formula for $a\cos x + b\sin x$; uses only the bounded range of sine/cosine and basic equation solving, with no calculus required.
