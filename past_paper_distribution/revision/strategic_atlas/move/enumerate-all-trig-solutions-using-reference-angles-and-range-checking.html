Trigger — When to use

After reducing a trig equation to one or more elementary forms (sin θ = k, cos θ = k, or tan θ = k), you must list every solution in a specified interval. Cues: the question explicitly asks for 'all solutions' or 'all values' in a given range; the variable inside the trig function is a multiple/shift of x (e.g. 2x, x − π/3) so the working range is stretched/translated; k is not always a standard exact value, so a calculator principal value will be needed. Equivalently: 'find x ∈ [a,b] such that …', or 'how many solutions does … have on …'.

Motivation — Why this works

A calculator returns only the principal value of arcsin/arccos/arctan, but sine and cosine each take any value k (with |k|<1) twice per 2π period, while tangent takes it once per π period. To capture every solution without duplication, you exploit the symmetry of the trig function (sine: θ and π−θ; cosine: ±θ; tangent: θ+π) and then translate by the period across the entire required range. Doing this systematically — principal value, companion, then add/subtract 2π (or π for tan) — guarantees a complete, non-redundant list. The CAST diagram is the standard visual mnemonic for which quadrants hold solutions given the sign of k.

Thought Process — Step-by-step reasoning

  1. I notice the equation has been reduced to elementary form sin θ = k (or cos/tan), and the question asks for all solutions in a range that may exceed one period or involve a transformed argument like 2x.
  2. I rule out just taking the calculator's principal value: that gives only one solution and would lose marks. I also rule out general-formula 'θ = nπ + (−1)ⁿ α' style answers since I need explicit numeric values in the stated interval.
  3. I commit to: (i) find the working range for the inner argument (e.g. if x ∈ [−π/2, π/2] and the argument is 2x, the working range is [−π, π]); (ii) compute principal value α = arcsin|k| as a positive reference angle; (iii) use CAST/symmetry to generate companion(s) within one period; (iv) shift by the period to fill the working range.
  4. Bridge to first operation: write down α = arcsin|k| (or arccos|k|) on the calculator, decide which quadrants are active from the sign of k, and list candidate angles symbolically (e.g. α, π−α, α+2π, π−α+2π).
  5. Tactical decisions: handle boundary values sin θ = ±1, cos θ = ±1 specially (one solution per period, not two); respect strict vs non-strict inequalities at endpoints; finally, divide/shift back to recover x and verify each x lies in the original range.

Mechanic — Concrete operations

  1. Identify the working interval for the inner argument θ (e.g. θ = 2x, so multiply the x-range by 2; θ = x − π/3, so shift).
  2. Compute the reference angle α = arcsin|k| (or arccos|k|, arctan|k|) as a positive number.
  3. Apply CAST / symmetry: for sin θ = k with k>0 use θ = α and θ = π−α; with k<0 use θ = −α and θ = −π+α (or equivalently π+α, 2π−α in [0,2π)). For cos θ = k use θ = ±arccos(k). For tan θ = k use θ = arctan(k).
  4. Add integer multiples of the period (2π for sin/cos, π for tan) to each base solution and keep those that lie in the working interval.
  5. Handle boundary cases sin θ = ±1, cos θ = ±1 separately — one solution per 2π period.
  6. Invert any substitution (e.g. divide by 2, add π/3) to recover x, then verify each x is in the original range and respects strict/non-strict endpoints.

Worked Micro-Example

Solve sin 2x = -\tfrac{1}{2} for x \in [-\tfrac{\pi}{2}, \tfrac{\pi}{2}]. Working range: 2x \in [-\pi, \pi]. Reference angle: \alpha = \arcsin(\tfrac{1}{2}) = \tfrac{\pi}{6}. Sine negative ⇒ Q3, Q4 (in [0,2π)) or equivalently 2x = -\tfrac{\pi}{6} and 2x = -\pi + \tfrac{\pi}{6} = -\tfrac{5\pi}{6} in [-\pi,\pi]. Halving: x = -\tfrac{\pi}{12}, -\tfrac{5\pi}{12}.

Variants — Same move, different surface

Equation is cos θ = k in a symmetric range
Use θ = ±arccos(k) + 2nπ — exploits cosine's even symmetry instead of sine's reflection about π/2.
Equation is tan θ = k
Solutions are θ = arctan(k) + nπ; only one base solution per π-period, so enumeration is simpler but range-checking discipline is identical.
Inner argument is a multiple angle (e.g. 2x, 3x)
Stretch the working range by the same factor before enumerating, then divide back at the end. This typically multiplies the number of solutions.
Range given in degrees (e.g. 0° ≤ x < 360°)
Use 180°−α, 180°+α, 360°−α instead of π-based expressions; same CAST logic.
Boundary value sin θ = ±1 or cos θ = ±1
Only one solution per 2π period — do not generate a companion.

Common Misreadings

  • Treating the calculator principal value as the only solution and stopping there.
  • Using x = 180°−α for negative sine, which puts the answer in Q2 where sine is positive — wrong quadrant.
  • Confusing the (signed) calculator output with the (positive) reference angle α.
  • Forgetting that the range for 2x is double the range for x, so only half the solutions are found.

Pitfalls

  • Mixing radians and degrees mid-calculation.
  • Omitting the symmetric companion (π−α for sine, −α for cosine) — the most common single source of lost marks.
  • Forgetting to add/subtract 2π when the working range exceeds one period.
  • Including endpoint values when the inequality is strict (e.g. listing 3π when y < 3π).
  • Forgetting to undo the substitution (divide by 2, add the shift) before reporting x.

A-level Scope Notes

All within Edexcel IAL P2/P3 trigonometry. Uses standard inverse trig (arcsin, arccos, arctan) on the calculator and the symmetry properties listed in the formula booklet. Avoid quoting the general solution θ = nπ + (−1)ⁿ α as a final answer — IAL expects explicit numeric values in the stated interval.
