Sequences & Series

Identify arithmetic model from constant yearly decrease

Family: Sequence Identification Frequency: 1 questions

Trigger — When to use

Fires when a worded context gives successive yearly (or periodic) values whose pairwise differences are equal, e.g. 480, 465, 450, …, especially when the wording explicitly says 'arithmetic sequence' or 'decreases by a fixed amount each year'. Also triggers on phrases like 'reduces by £k each year', 'output falls by a constant amount', or any list where u₂−u₁ = u₃−u₂. The question typically then asks for a specific term (year N), the total over N years, or the year in which the value drops below a threshold.

Motivation — Why this works

A constant first difference is the defining property of an arithmetic sequence, so the closed form u_n = a + (n−1)d applies exactly. Once a (year-1 value) and d (signed common difference) are pinned down, any term is reachable in one step without iterating. This is far more efficient than recursion or tabulation, and it also unlocks S_n = (n/2)(2a+(n−1)d) for cumulative totals.

Thought Process — The step-by-step reasoning

  1. Notice that consecutive given values differ by the same amount: 465−480 = −15 and 450−465 = −15.
  2. Rule out a geometric model (ratios 465/480 ≠ 450/465) and rule out needing recursion since a closed form exists.
  3. Commit to the arithmetic model because the constant difference is confirmed and the wording matches.
  4. Set a = u₁ = 480 (year 1) and d = −15, being careful that d is negative because the quantity decreases.
  5. Map the asked year directly to n (year k ↔ n = k) and substitute into u_n = a + (n−1)d.

Mechanic — Concrete operations

  1. Compute d = u₂ − u₁ = 465 − 480 = −15.
  2. Verify d = u₃ − u₂ = 450 − 465 = −15 to confirm arithmetic.
  3. State a = 480 and d = −15.
  4. Write u_n = a + (n−1)d = 480 + (n−1)(−15).
  5. Substitute the required n (e.g. n = 14): u₁₄ = 480 + 13(−15) = 480 − 195 = 285.

Worked Micro-Example

A well extracts $480$ m³ in year 1, $465$ in year 2, $450$ in year 3, forming an arithmetic sequence. Then $a=480$, $d=-15$, so $u_n = 480 + (n-1)(-15)$. In year $14$: $u_{14} = 480 + 13(-15) = 285$ m³.

Variants — Same move, different surface

Constant ratio instead of constant difference (e.g. 480, 432, 388.8, …)
Switch to geometric model u_n = ar^{n-1} with r = u₂/u₁; verify with u₃/u₂.
Cumulative total over N years requested
Apply S_N = (N/2)(2a + (N−1)d) instead of, or in addition to, the nth-term formula.
Find the year when value first falls below a threshold T
Solve a + (n−1)d < T for n and take the smallest integer n satisfying it.
Increasing sequence (e.g. salary rises by £k per year)
Same formula but with d > 0; check sign carefully from the wording.

Common Misreadings

  • Treating year 1 as n = 0, giving an off-by-one error in u_n.
  • Reading 'decreases by 15' as r = 0.15 (geometric) instead of d = −15 (arithmetic).
  • Using d = 15 (magnitude) and forgetting the negative sign for a declining quantity.

Pitfalls

  • Sign error on d propagates into every term; always re-check with u₂ − u₁.
  • Computing (n−1)d as nd, e.g. using 14·(−15) instead of 13·(−15) for u₁₄.
  • Applying the model beyond its valid range — e.g. a year where u_n becomes negative is physically meaningless even though the formula still returns a value.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P1/P2) sequences and series, using only u_n = a + (n−1)d and, where relevant, S_n = (n/2)(2a + (n−1)d).
