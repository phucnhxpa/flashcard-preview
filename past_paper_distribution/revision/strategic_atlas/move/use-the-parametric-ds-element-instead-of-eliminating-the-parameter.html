Integration

Use the parametric ds element instead of eliminating the parameter

Family: Parametric Formulation Frequency: 1 questions

Trigger — When to use

The curve is presented parametrically as $x = x(t)$, $y = y(t)$ (or via an angle $\theta$), and an arc length $L = \int ds$ or surface area $S = 2\pi\int y\,ds$ is required. Eliminating the parameter would force inverse trig substitutions, fractional powers, or implicit forms. Meanwhile $dx/dt$ and $dy/dt$ are short, clean expressions, and the question already supplies $t$-limits or limits readily converted to $t$.

Motivation — Why this works

The parametric element $ds = \sqrt{(dx/dt)^2 + (dy/dt)^2}\,dt$ is exact and keeps every quantity as a function of $t$, so designed simplifications (often a perfect square under the root) are preserved. Eliminating $t$ typically destroys this structure and introduces square roots that are hard to integrate. Working in $t$ also makes limit conversion automatic. It is the route of least algebraic resistance whenever the curve is parametric.

Thought Process — The step-by-step reasoning

  1. Notice the curve is given as $x(t), y(t)$ rather than $y = f(x)$.
  2. Test mentally whether $t$ can be eliminated cleanly; usually it requires inverse trig or surd manipulation, so reject Cartesian form.
  3. Observe that $dx/dt$ and $dy/dt$ are short and that $(dx/dt)^2 + (dy/dt)^2$ is likely engineered to collapse to a perfect square.
  4. Commit to the parametric arc-length element and translate any $x$-limits into $t$-limits.
  5. Decide whether the integrand needs $|f(t)|$ (e.g. when $dx/dt$ may be negative) and choose the simplification path before integrating.

Mechanic — Concrete operations

  1. Compute $dx/dt$ and $dy/dt$.
  2. Form $(dx/dt)^2 + (dy/dt)^2$ and simplify, aiming for a perfect square so the root disappears.
  3. Write $ds = \sqrt{(dx/dt)^2 + (dy/dt)^2}\,dt$.
  4. Convert the given limits into $t$-values.
  5. Set up $L = \int_{t_1}^{t_2} ds$ (or $S = 2\pi\int y\,ds$, $V$ etc.) and integrate in $t$.

Worked Micro-Example

Find the arc length of $x = t^2$, $y = \tfrac{2}{3}t^3$ for $0 \le t \le 1$.\n$dx/dt = 2t$, $dy/dt = 2t^2$, so $(dx/dt)^2 + (dy/dt)^2 = 4t^2 + 4t^4 = 4t^2(1+t^2)$.\nThus $ds = 2t\sqrt{1+t^2}\,dt$ (using $t \ge 0$).\n$L = \int_0^1 2t\sqrt{1+t^2}\,dt = \left[\tfrac{2}{3}(1+t^2)^{3/2}\right]_0^1 = \tfrac{2}{3}(2\sqrt{2}-1)$.

Variants — Same move, different surface

Polar curve $r = f(\theta)$
Use $\theta$ as the parameter with $x = r\cos\theta$, $y = r\sin\theta$; the element simplifies to $ds = \sqrt{r^2 + (dr/d\theta)^2}\,d\theta$.
Surface of revolution from a parametric curve
Replace $\int ds$ by $2\pi\int y\,ds$ (about $x$-axis) or $2\pi\int x\,ds$ (about $y$-axis), keeping the same parametric $ds$.
Curve given as $y = f(x)$ but with awkward $dy/dx$
Introduce a parameter (e.g. $x = t$ or a trig substitution) so that $(dx/dt)^2 + (dy/dt)^2$ becomes a perfect square.
Trig-parametrised curves such as $x = a\cos^3 t$, $y = a\sin^3 t$
Parametric form is essentially mandatory; identities like $\sin^2 t + \cos^2 t = 1$ produce the perfect square inside the root.

Common Misreadings

  • Treating the curve as Cartesian and trying to find $dy/dx$ explicitly before realising the algebra is intractable.
  • Reading the given $x$-range as the integration range without converting it into $t$-values.
  • Confusing the parametric arc-length formula with the surface-area formula and forgetting the $2\pi y$ factor.

Pitfalls

  • Using $x$-limits instead of $t$-limits after switching to the parametric integrand.
  • Dropping the $dt$ when writing $ds$, leading to a dimensionally wrong integrand.
  • Taking $\sqrt{f(t)^2} = f(t)$ without checking the sign of $f(t)$ on the interval.
  • Mis-simplifying $(dx/dt)^2 + (dy/dt)^2$ and missing the perfect-square factorisation that the question is engineered to produce.

A-level Scope Notes

Edexcel IAL Further Pure includes parametric arc length $\int\sqrt{(dx/dt)^2 + (dy/dt)^2}\,dt$ and surfaces of revolution; polar arc length is also in scope, both as direct applications of this move.
