Algebra & Functions

Compare coefficients of matching linear forms to extract parameters

Family: Coefficient Comparison Frequency: 1 questions

Trigger — When to use

Two expressions for the same quantity are both linear in the same variable, with one side fully numerical (e.g. $0.154t + 3.08$) and the other carrying unknown parameters (e.g. $(\log_{10} b)\,t + \log_{10} a$). This commonly arises after linearising an exponential model $N = ab^t$ via logarithms, or when matching a derived identity to a given form. The cue is 'these must be the same line/polynomial', so corresponding coefficients must agree term-by-term.

Motivation — Why this works

If two polynomials (or linear expressions) are equal for all values of the variable, the identity theorem forces matching coefficients. This converts a single equation in two unknowns into two independent single-variable equations, one per coefficient slot. It avoids substitution, elimination, or solving simultaneous equations, and is exact rather than numerical. The principle generalises to any finite-dimensional basis where representations are unique.

Thought Process — The step-by-step reasoning

  1. I notice both sides are linear in $t$ and equal for all $t$, not just at one value.
  2. I rule out substituting specific $t$-values to build simultaneous equations — that's slower and risks numerical error.
  3. I commit to coefficient comparison because uniqueness of linear representation guarantees gradient matches gradient and intercept matches intercept.
  4. I align the two forms in the same shape: $(\log_{10} b)\,t + \log_{10} a \equiv 0.154t + 3.08$.
  5. I read off $\log_{10} b = 0.154$ and $\log_{10} a = 3.08$, then invert each logarithm independently using $10^{(\cdot)}$.

Mechanic — Concrete operations

  1. Write both sides in the same canonical linear form in $t$.
  2. Equate coefficients of $t$: $\log_{10} b = 0.154$.
  3. Equate constant terms: $\log_{10} a = 3.08$.
  4. Solve $b = 10^{0.154} \approx 1.43$ (3 s.f.).
  5. Solve $a = 10^{3.08} \approx 1200$ (3 s.f.).

Worked Micro-Example

Given $\log_{10} N = 0.154t + 3.08$ and the model $N = ab^t$, take logs: $\log_{10} N = (\log_{10} b)\,t + \log_{10} a$. Matching coefficients of $t$: $\log_{10} b = 0.154 \Rightarrow b = 10^{0.154} \approx 1.43$. Matching constants: $\log_{10} a = 3.08 \Rightarrow a = 10^{3.08} \approx 1200$.

Variants — Same move, different surface

Polynomial identity, e.g. $x^2 + px + q \equiv (x+2)(x+3)$
Expand RHS, then equate coefficients of $x^2$, $x$, and constant separately.
Partial fractions, e.g. $\frac{3x+5}{(x-1)(x+2)} \equiv \frac{A}{x-1} + \frac{B}{x+2}$
Clear denominators to get a polynomial identity, then compare coefficients of each power of $x$ (or use cover-up).
Linearised power model $y = ax^n$ giving $\log y = n \log x + \log a$
Compare against numerical line $\log y = m\log x + c$: $n = m$, $\log a = c$.
Binomial expansion matched to a given series
Equate coefficients of $x^0, x^1, x^2,\dots$ to find unknown constants in the original expression.
Trigonometric identity $R\sin(\theta+\alpha) \equiv a\sin\theta + b\cos\theta$
Expand and compare coefficients of $\sin\theta$ and $\cos\theta$ to get $R\cos\alpha = a$, $R\sin\alpha = b$.

Common Misreadings

  • Treating the equation as holding only at one value of $t$, then trying to solve a single equation in two unknowns.
  • Confusing which side carries the unknowns and which side carries the numerical data.
  • Believing the gradient encodes $a$ and the intercept encodes $b$ (it is the other way round for $N = ab^t$).

Pitfalls

  • Swapping gradient and intercept: assigning $\log_{10} a = 0.154$ and $\log_{10} b = 3.08$.
  • Rounding $10^{3.08}$ prematurely or to the wrong number of significant figures.
  • Forgetting to take antilogs and leaving answers as $\log_{10} a$, $\log_{10} b$.
  • Using natural log antilog ($e^{(\cdot)}$) when the original log was base 10.

A-level Scope Notes

Fully within Edexcel IAL Pure scope: linearisation of exponential/power models with logarithms (P2/P3), polynomial identities, and partial fractions (P4) all rely on this same coefficient-matching principle.
