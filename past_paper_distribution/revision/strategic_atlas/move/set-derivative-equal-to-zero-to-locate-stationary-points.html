Differentiation

Set derivative equal to zero to locate stationary points

Family: Stationary Point Condition Frequency: 3 questions

Trigger — When to use

The question explicitly asks for a 'stationary point', 'turning point', 'maximum/minimum', or 'where the gradient is zero' on a curve y = f(x), x = g(y), or an implicit/parametric curve. It may also fire indirectly: e.g. 'find values of k such that x = k cuts the curve in exactly two places' — where the critical k corresponds to an extremum of x(y). The visual cue is a curve plus a flatness/tangent condition. If the derivative has been computed (or can be computed) as a fraction, expression, or product, the natural next step is to set it to zero and solve.

Motivation — Why this works

A stationary point is defined by gradient = 0, so the geometric condition translates directly into the algebraic equation dy/dx = 0 (or dx/dy = 0, or dy/dt = 0 for parametric forms). When the derivative is a fraction P/Q, the equation reduces to P = 0 with Q ≠ 0 — exploiting the fact that a fraction vanishes iff its numerator does and denominator is finite/nonzero. When the derivative is a product (e.g. e^{ax}·polynomial), one factor (the exponential) is never zero so only the polynomial factor matters. This recasts a geometric question ('where is the curve flat?') into a clean algebraic solve.

Thought Process — The step-by-step reasoning

  1. Notice the keyword 'stationary point' (or an equivalent: turning point, max/min, horizontal tangent). This pins the condition to gradient = 0.
  2. Rule out other tempting interpretations: 'point of inflection' would need d²y/dx² = 0, and 'vertical tangent' would need denominator of dy/dx = 0 — neither is what is asked here.
  3. Commit to dy/dx = 0 (or dx/dy = 0 if the curve is naturally x as a function of y, or dy/dt = 0 with dx/dt ≠ 0 for parametric).
  4. Bridge to operation: differentiate the given function (using product/quotient/chain/implicit rules as appropriate), then write the equation derivative = 0.
  5. Tactical simplification: if the derivative is a fraction, set numerator = 0 only (and remember to verify denominator ≠ 0 at solutions). If it is a product where one factor is strictly positive (like e^{kx}), discard that factor and solve the remaining one.
  6. After solving, substitute back into the original curve to find coordinates, and (if asked) classify using the second derivative or a sign check on dy/dx.

Mechanic — Concrete operations

  1. Compute the relevant derivative: dy/dx, dx/dy, or dy/dt (parametric).
  2. Form the equation derivative = 0.
  3. If derivative = P/Q, reduce to P = 0; if derivative = (positive factor)·R, reduce to R = 0.
  4. Solve the resulting equation for the variable.
  5. Check denominator ≠ 0 (or dx/dt ≠ 0 in parametric case) at each candidate solution.
  6. Substitute back into the original curve to obtain the stationary point coordinates.

Worked Micro-Example

For y = (2x+5)e^{3x}: dy/dx = 2e^{3x} + 3(2x+5)e^{3x} = e^{3x}(6x+17). Set dy/dx = 0. Since e^{3x} > 0, we need 6x+17 = 0, giving x = -17/6.

Variants — Same move, different surface

Derivative is a quotient (x+y)/(2y−x) from implicit differentiation
Set numerator = 0 (e.g. x+y = 0 ⇒ y = −x), then substitute into the original curve; verify denominator ≠ 0 at each candidate.
Curve given as x = f(y), or 'how many y solve f(y) = k'
Use dx/dy = 0 to find extrema of x(y); the critical k-values are the x-coordinates at these extrema.
Parametric curve x = x(t), y = y(t)
Stationary points require dy/dt = 0 with dx/dt ≠ 0; solve for t then find (x, y).
Implicit equation F(x,y) = 0
Differentiate implicitly to obtain dy/dx, then set it to zero (typically reducing to numerator = 0) and solve simultaneously with the original equation.

Common Misreadings

  • Confusing 'stationary point' with 'point of inflection' and setting d²y/dx² = 0 instead.
  • Setting the denominator of dy/dx to zero — this finds vertical tangents, not stationary points.
  • In parametric problems, setting dy/dx = 0 directly without recognising it equals (dy/dt)/(dx/dt).

Pitfalls

  • Forgetting that e^{kx} > 0 always, and so wrongly hunting for an extra stationary point from that factor.
  • Failing to verify the denominator (or dx/dt) is non-zero at candidate points — a zero denominator invalidates the 'stationary' classification.
  • Sign/arithmetic slips when substituting back, e.g. evaluating (−1/2)e^{−1} = −1/(2e).
  • Stopping at the x-value when the question asks for full coordinates or for classification.

A-level Scope Notes

Fully within Edexcel IAL P2/P3/P4 (and FP1) scope. Uses only standard differentiation rules (product, quotient, chain, implicit, parametric) from the formula booklet. Avoid invoking partial derivatives or Lagrange multipliers — these are out of scope.
