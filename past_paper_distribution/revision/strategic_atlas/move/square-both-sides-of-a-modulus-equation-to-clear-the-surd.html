Complex Numbers

Square both sides of a modulus equation to clear the surd

Family: Equation Reduction Frequency: 1 questions

Trigger — When to use

Fires when an equation contains a modulus |·| set equal to a positive constant or surd, so |a+bi| = k expands to √(a²+b²) = k. Typical cues: |w| = 2√5, |z−z₀| = r, or |f(p)| = c with an unknown parameter inside a+bi. Also triggered when both sides are manifestly non-negative and a polynomial equation in the unknown is desired.

Motivation — Why this works

By definition |a+bi| = √(a²+b²), so a modulus equation hides a square root. Squaring sends √(A²+B²) = k to A²+B² = k², which is polynomial and far easier to manipulate. Because |·| ≥ 0 and the RHS is also non-negative, squaring is reversible here and introduces no extraneous roots. This is the standard route to convert geometric/modulus conditions into algebra.

Thought Process — The step-by-step reasoning

  1. Notice the equation has the form √(…) = (positive constant), so direct algebra is awkward.
  2. Rule out expanding the surd or using approximations; also rule out modulus quotient shortcuts if the question structure forces use of the a+bi form already obtained.
  3. Commit to squaring because both sides are non-negative, so the operation is safe and yields a polynomial in the unknown.
  4. Bridge: write |w|² = (RHS)² explicitly, being careful that (k√n)² = k²n.
  5. Inside the move, factor common squared brackets (e.g. (p²+1)²) before expanding, so the polynomial stays compact.

Mechanic — Concrete operations

  1. Express the modulus in surd form: |a+bi| = √(a²+b²).
  2. Square both sides simultaneously: a²+b² = k².
  3. Carefully square any surd on the RHS, e.g. (2√5)² = 4·5 = 20.
  4. Simplify by extracting common squared factors (e.g. (p²+1)²) before expanding.
  5. Solve the resulting polynomial equation in the unknown.

Worked Micro-Example

Given w = \frac{2(p^2+1)}{5} + \frac{p^2+1}{5}i with |w| = 2\sqrt{5}: \;|w|^2 = \frac{4(p^2+1)^2}{25} + \frac{(p^2+1)^2}{25} = \frac{5(p^2+1)^2}{25} = \frac{(p^2+1)^2}{5}. Setting this equal to (2\sqrt{5})^2 = 20 gives (p^2+1)^2 = 100, so p^2+1 = 10 and p = \pm 3.

Variants — Same move, different surface

Locus equation |z − z₀| = r with z = x+iy
Square to obtain the Cartesian circle (x−a)² + (y−b)² = r².
|z₁| = |z₂| with z₁, z₂ in a+bi form
Square both sides to get a₁²+b₁² = a₂²+b₂², avoiding two surds.
|f(z)| = |g(z)| in a perpendicular bisector problem
Squaring removes both moduli simultaneously and yields a linear equation after cancellation.
Modulus equality involving a quotient |z₁/z₂| = k
Either apply |z₁|/|z₂| = k first, then square, or square directly to get |z₁|² = k²|z₂|².

Common Misreadings

  • Reading |w| = 2√5 as |w|² = 2√5 and forgetting to square the RHS.
  • Treating |a+bi| as |a|+|b| rather than √(a²+b²).
  • Believing squaring requires a separate justification for extraneous roots when both sides are already non-negative.

Pitfalls

  • Squaring only the LHS while leaving the RHS unsquared.
  • Computing (2√5)² as 4√5 or 10 instead of 20.
  • Expanding (p²+1)² prematurely and losing the clean factored form.
  • Forgetting the ± when finally taking a square root at the end (e.g. p² = 9 ⇒ p = ±3).

A-level Scope Notes

Fully within Edexcel IAL Further Pure (complex numbers): uses only |a+bi| = √(a²+b²) and basic algebraic manipulation, standard for FP1/FP2 modulus and locus questions.
