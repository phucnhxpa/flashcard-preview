Trigonometry

Divide by cos to convert homogeneous sin–cos equation into tan

Family: Homogeneous To Tan Frequency: 5 questions

Trigger — When to use

The equation has the form a·sin(A) = b·cos(A), or more generally a homogeneous combination of sin and cos with the SAME argument A and no constant term (e.g. a·sin(A) + b·cos(A) = 0, or a·sin²A + b·sinA·cosA + c·cos²A = 0). Visual cues: every term contains a sin or cos of the same compound angle, and the total 'degree' in trig functions is the same in every term. Equivalent phrasings: 'linear (or homogeneous) in sin and cos of one argument', 'no loose constant on either side', 'sin and cos appear only as a ratio in disguise'. If you see this signature, dividing through by the highest power of cos(A) is the canonical first move.

Motivation — Why this works

Dividing both sides by cos(A) (or cos²(A) for degree-2) collapses two unknown trig values into a single tan(A), which has a unique principal value via arctan and a clean 180° period. This works because sin(A)/cos(A) = tan(A) by definition, and homogeneity guarantees every term converts cleanly. The division is safe whenever cos(A) = 0 cannot satisfy the original equation — and a quick check usually confirms this, since cos(A) = 0 would force sin(A) = ±1, contradicting the equation. This route avoids the heavier R·sin(A+α) machinery (which is for non-homogeneous a·sinA + b·cosA = c) and avoids squaring (which introduces spurious roots).

Thought Process — The step-by-step reasoning

  1. Notice that every term involves sin or cos of the SAME argument A, with no standalone constant — this is the homogeneity signature.
  2. Rule out R·sin(A+α) form: that's needed for a·sinA + b·cosA = c with c ≠ 0, but here there's no constant term, so it's overkill.
  3. Rule out squaring both sides: it would let me use sin²+cos²=1, but it introduces extraneous solutions and doesn't simplify the structure.
  4. Commit to dividing by cos(A) (or cos²(A) if degree 2) — this is the lightest tool that exactly matches the structure.
  5. Safety check: could cos(A) = 0 be a solution? Substitute cos(A) = 0 into the original; if it gives a contradiction (e.g. 0 = nonzero), division is safe with no lost solutions. If degree-2 has a sin(A) common factor, factor it out FIRST so those roots aren't lost.
  6. Execute: divide every term by cos(A) [or cos²(A)], replace sin/cos with tan, and solve the resulting linear or quadratic equation in tan(A).
  7. Finish: take arctan to get the principal value, then add 180° (or use CAST) to capture all solutions in the required interval, accounting for the period of tan.

Mechanic — Concrete operations

  1. Confirm homogeneity: every term has the same total degree in sin(A), cos(A); no loose constant.
  2. If degree 2 with a common sin(A) (or cos(A)) factor and RHS = 0, factor it out FIRST: e.g. sin(A)·[a·sinA − b·cosA] = 0; record sin(A) = 0 as one case.
  3. Verify cos(A) = 0 is not a solution by substitution (else division loses roots).
  4. Divide every term by cos(A) for degree-1, or cos²(A) for degree-2.
  5. Replace sin(A)/cos(A) with tan(A); the equation becomes tan(A) = k (linear) or a·tan²(A) + b·tan(A) + c = 0 (quadratic).
  6. Solve for tan(A); take arctan for the principal value.
  7. Generate all solutions in the required interval using period 180°: A = arctan(k) + n·180°, then back-solve for the original variable.

Worked Micro-Example

Solve $3\sin(2x-15°) = \cos(2x-15°)$ for $0° \le x \le 180°$. Both sides have the same argument $A = 2x-15°$ and no constant term, so divide by $\cos A$: $3\tan(2x-15°) = 1 \Rightarrow \tan(2x-15°) = \tfrac{1}{3}$. Then $2x - 15° = \arctan(1/3) + n·180° \approx 18.43° + n·180°$, giving $x \approx 16.7°, 106.7°$.

Variants — Same move, different surface

Degree-2 homogeneous: a·sin²θ + b·sinθcosθ + c·cos²θ = 0
Divide every term by cos²θ to get a·tan²θ + b·tanθ + c = 0, a quadratic in tanθ; solve and take arctan of each root.
Degree-2 with RHS = 0 and a common sinθ (or cosθ) factor, e.g. 3sin²θ − 4sinθcosθ = 0
Factor first: sinθ(3sinθ − 4cosθ) = 0. Record sinθ = 0 separately, then divide the linear factor by cosθ to get tanθ = 4/3.
RHS = 0 form a·sinθ + b·cosθ = 0
Same divide-by-cos trick gives tanθ = −b/a immediately.
Non-homogeneous a·sinθ + b·cosθ = c with c ≠ 0
Divide-by-cos does NOT simplify cleanly — switch to the R·sin(θ+α) (harmonic) form instead.
Compound argument like sin(θ + 30°) and cos(θ + 30°)
Treat the compound argument as a single variable A; divide through, get tan(A) = k, then back-solve A = θ + 30°.

Common Misreadings

  • Treating a·sinθ + b·cosθ = c (with c ≠ 0) as homogeneous and trying to divide by cos — it produces a + b·cotθ = c·secθ, which is worse.
  • Squaring both sides to invoke sin²+cos²=1 — introduces extraneous solutions and a more complex equation.
  • Reaching for R·sin(θ+α) form on a homogeneous equation — correct but unnecessarily heavy.
  • Dividing through by sinθ at the start of a degree-2 equation, losing the sinθ = 0 solutions.

Pitfalls

  • Forgetting to verify cos(A) ≠ 0 before dividing — could lose solutions if the original equation does admit cos(A) = 0.
  • Forgetting the second-period solution: tan has period 180°, so tan(A) = k > 0 has solutions in BOTH Q1 and Q3 within [0°, 360°).
  • Arithmetic slip in the final ratio (e.g. writing tan = 3 instead of tan = 1/3 when 3sinA = cosA).
  • When solving for the original variable from arctan(k) + n·180°, forgetting to scale/shift correctly for compound arguments like 2x − 15°.
  • Rounding arctan to the wrong number of decimal places for the question's stated accuracy.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P2/P3) trigonometry. Uses only the identity tanθ = sinθ/cosθ (not in the formula booklet — must be known) and the period of tan. Avoid extending to complex-argument or hyperbolic analogues; stick to real θ in degrees or radians as the question specifies.
