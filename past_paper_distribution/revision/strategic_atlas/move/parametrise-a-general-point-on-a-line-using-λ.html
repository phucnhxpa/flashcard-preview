Vectors

Parametrise a general point on a line using λ

Family: Parametric Representation Frequency: 1 questions

Trigger — When to use

Fires when a question asks you to find specific point(s) on a line L satisfying some condition — typically a distance from a fixed point, lying on a plane, or making a given angle. The line is given (or convertible) to vector form r = a + λd, and the unknown is a position on L rather than a free point in space. Phrases like 'find the point(s) on L such that...', 'P lies on L and |AP| = k', or 'the point on L closest to...' are standard cues.

Motivation — Why this works

Any point on L is uniquely determined by one scalar λ, so writing p = a + λd collapses three unknown coordinates into one. Every geometric property of P (coordinates, distance, dot product with another vector) then becomes an algebraic expression in λ, and the condition imposed by the question becomes an equation in λ. This is the standard bridge from vector geometry to a solvable 1-variable equation.

Thought Process — The step-by-step reasoning

  1. Notice the unknown point is constrained to lie on L — so it has only one degree of freedom, not three.
  2. Rule out a purely geometric construction (e.g. foot of perpendicular) when the given distance/condition is not the perpendicular one, since that would need an extra offset step.
  3. Commit to the parametric form: introducing λ captures the single degree of freedom directly.
  4. Write p = a + λd componentwise so each coordinate is an explicit linear expression in λ, ready to feed into a distance formula or dot product.
  5. Keep λ as the only unknown; the imposed condition will produce an equation (often quadratic) whose roots give all valid points.

Mechanic — Concrete operations

  1. Identify the position vector a and direction vector d from the line equation r = a + λd.
  2. Write the general point as p = a + λd in component form, e.g. p = (a_1 + λd_1,\ a_2 + λd_2,\ a_3 + λd_3).
  3. Express the required quantity (distance, dot product, plane equation LHS, etc.) as a function of λ.
  4. Set up the equation from the given condition and solve for λ.
  5. Substitute each λ value back into p to obtain the required point(s).

Worked Micro-Example

Line $L: \mathbf{r} = (1,2,3) + \lambda(1,0,-1)$, find points on $L$ at distance $\sqrt{6}$ from $A=(0,2,3)$. General point $P = (1+\lambda,\ 2,\ 3-\lambda)$, so $\vec{AP} = (1+\lambda,\ 0,\ -\lambda)$. Then $|AP|^2 = (1+\lambda)^2 + \lambda^2 = 6 \Rightarrow 2\lambda^2 + 2\lambda - 5 = 0$, giving two values of $\lambda$ and hence two points $P$.

Variants — Same move, different surface

Line in Cartesian symmetric form (x−a)/p = (y−b)/q = (z−c)/r
Set the common ratio equal to λ, then x = a+pλ, y = b+qλ, z = c+rλ — identical parametrisation.
Finding the foot of perpendicular from a point A to L
Parametrise P on L, then impose $\vec{AP}\cdot\mathbf{d}=0$ to solve for λ.
Intersection of line with a plane $\mathbf{r}\cdot\mathbf{n}=k$
Substitute the parametrised P into the plane equation, giving a single linear equation in λ.
Point on L equidistant from two fixed points A and B
Parametrise P, then set $|AP|^2 = |BP|^2$, which reduces to a linear equation in λ.
2D line $\mathbf{r}=\mathbf{a}+\lambda\mathbf{d}$ in the plane
Same parametrisation with two components only; condition becomes an equation in a single λ.

Common Misreadings

  • Treating the required distance as the perpendicular distance from A to L and computing the foot of perpendicular instead of solving for λ via the distance equation.
  • Assuming there is a unique point on L when the distance condition typically yields a quadratic in λ with two solutions.
  • Confusing 'point on L' with 'point on the line segment AB' and restricting λ unnecessarily.

Pitfalls

  • Forgetting to write all three components in terms of λ, especially when a direction component is 0 (the corresponding coordinate is constant, not absent).
  • Algebra slips when squaring: dropping cross terms in $(a_i+\lambda d_i)^2$.
  • Reusing the same λ for two different lines in an intersection problem — use distinct parameters (λ and μ) instead.
  • Stopping after finding λ without substituting back to give the actual point(s) the question asked for.

A-level Scope Notes

Fully within Edexcel IAL Core Pure / FP scope: vector equations of lines in 2D and 3D, distance formula, dot product, and intersection with planes. No matrix or projection machinery needed.
