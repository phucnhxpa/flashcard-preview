Polar Coordinates

Solve r = 0 to find polar area integration limits

Family: Limit Determination Frequency: 1 questions

Trigger — When to use

An area question asks for the region enclosed by a polar curve r = f(θ) but does not state explicit angular limits. Typical phrasings: 'find the area enclosed by the curve', 'the area of one loop', 'the area inside the cardioid'. The curve passes through the pole, so the boundary closes when r = 0. This contrasts with questions that hand you limits like 'between θ = π/6 and θ = π/3', where this step is unnecessary.

Motivation — Why this works

The polar area formula A = ½∫r² dθ needs angular limits α, β at which the swept region begins and ends. A curve enters or leaves the pole exactly when r(θ) = 0, so consecutive zeros of r demarcate one full lobe or loop. Using 0 to 2π by default would either double-count overlapping sweeps (e.g. one petal of a rose) or include empty sweeps. Solving r = 0 therefore plays the same role x-intercepts do for Cartesian area problems.

Thought Process — The step-by-step reasoning

  1. I notice the question asks for an enclosed area but gives no θ-limits — so I must derive them from the curve itself.
  2. I rule out using 0 to 2π by default: for roses, single loops of limaçons, or 'one petal' problems, that range sweeps the curve more than once.
  3. I commit to solving r(θ) = 0 because the curve closes at the pole, making zeros of r the natural boundary angles.
  4. I set f(θ) = 0 and find all solutions in [0, 2π), then pick the consecutive pair that brackets the lobe described.
  5. If the curve is symmetric about the initial line or another axis, I integrate over half the range and double, which simplifies the trig integral.

Mechanic — Concrete operations

  1. Write the equation f(θ) = 0 from r = f(θ).
  2. Solve for all θ ∈ [0, 2π), being careful to list every solution of the trig equation.
  3. Identify the consecutive pair of zeros that bound the required region (one petal, one loop, etc.).
  4. Check for symmetry; if present, halve the interval and multiply the resulting integral by 2.
  5. Substitute the chosen α, β into A = ½∫_α^β [f(θ)]² dθ.

Worked Micro-Example

For the rose $r = \sin(2\theta)$, set $\sin(2\theta) = 0 \Rightarrow 2\theta = 0, \pi, 2\pi, \ldots$, giving $\theta = 0, \tfrac{\pi}{2}, \pi, \ldots$. One petal lies between $\theta = 0$ and $\theta = \tfrac{\pi}{2}$. So its area is $A = \tfrac{1}{2}\int_0^{\pi/2} \sin^2(2\theta)\,d\theta = \tfrac{\pi}{8}$.

Variants — Same move, different surface

Rose curve r = a sin(nθ) or a cos(nθ)
Zeros are equally spaced at θ = kπ/n; consecutive pair bounds one petal.
Inner loop of a limaçon r = a + b cos θ with |a| < |b|
Solve a + b cos θ = 0 to get the two angles where the inner loop pinches at the pole; integrate between them.
Cardioid r = a(1 + cos θ)
r = 0 only at θ = π; use full sweep -π to π (or 0 to 2π) and exploit symmetry by integrating 0 to π and doubling.
Question gives explicit limits
Skip this move entirely and use the stated α, β.

Common Misreadings

  • Assuming the limits are always 0 to 2π regardless of curve shape, leading to multiple-counting of petals.
  • Treating the first zero found as α without checking that the next zero is the correct partner for the lobe asked about.
  • Confusing 'curve passes through pole' with 'curve crosses θ-axis' — the latter is r-axis crossing, not r = 0.

Pitfalls

  • Missing solutions of the trig equation in [0, 2π) (e.g. forgetting the second solution of cos θ = -½).
  • Choosing limits that span a region where r < 0; r² is still positive so the integral evaluates, but the geometric region may not match the one intended.
  • Forgetting the factor ½ or the doubling factor when symmetry is used.

A-level Scope Notes

This is standard Edexcel IAL Further Pure (FP2) polar coordinates material — area enclosed by polar curves using A = ½∫r² dθ with limits found from r = 0.
