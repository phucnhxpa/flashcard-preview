Differential Equations

Apply zero-time condition first to isolate constant of integration before solving for rate

Family: Sequential Boundary Application Frequency: 1 questions

Trigger — When to use

After separating and integrating an ODE, you obtain an implicit solution F(y) = -kt + C (or similar) with two unknowns: the integration constant C and a rate/proportionality constant k. Two data points are supplied, one of which has t = 0 (or another value that annihilates the kt term). Cue phrases: 'initially', 'at the start', 'when t = 0', followed by a later measurement at t = T.

Motivation — Why this works

Substituting t = 0 makes the -kt term vanish, so C is determined by a single evaluation rather than via simultaneous equations. Once C is fixed, the second data point produces one linear equation in k alone. This decouples the two unknowns and replaces a 2x2 system with two one-step calculations, reducing both algebra and the chance of sign/arithmetic error.

Thought Process — The step-by-step reasoning

  1. I notice the integrated form has two unknown constants (C and k) and I am given two boundary conditions.
  2. I rule out substituting both points simultaneously because that produces a coupled pair of equations.
  3. I commit to the t = 0 condition first because it kills the kt term and gives C in one step.
  4. Bridge: evaluate F(y_0) at t = 0 to read off C directly from the left-hand side.
  5. With C locked in, I substitute the second pair (t = T, y = y_1) and solve the resulting linear equation for k.

Mechanic — Concrete operations

  1. Write the integrated implicit solution F(y) = -kt + C.
  2. Substitute the t = 0 condition: F(y_0) = C, giving C immediately.
  3. Substitute the second condition (t = T, y = y_1): F(y_1) = -kT + C.
  4. Rearrange to k = (C - F(y_1))/T and simplify.
  5. Write the particular solution by substituting both constants back into F(y) = -kt + C.

Worked Micro-Example

Given $\tfrac{r^3}{3} = -kt + C$ with $r = 12$ at $t = 0$ and $r = 6$ at $t = 15$. Step 1: $t = 0 \Rightarrow C = \tfrac{12^3}{3} = 576$. Step 2: $t = 15 \Rightarrow \tfrac{6^3}{3} = -15k + 576$, i.e. $72 = -15k + 576$. Step 3: $15k = 504 \Rightarrow k = \tfrac{168}{5}$. Particular solution: $r^3 = 1728 - \tfrac{504}{5}t$.

Variants — Same move, different surface

Newton's law of cooling: $\theta - \theta_a = Ae^{-kt}$ with $\theta(0) = \theta_0$ and $\theta(T) = \theta_1$.
At $t = 0$ the exponential is 1, so $A = \theta_0 - \theta_a$ directly; then use the second point to solve for $k$ via logarithms.
Exponential growth/decay $P = Ae^{kt}$ with initial population and a later measurement.
$t = 0$ gives $A = P_0$ instantly; second condition yields $k = \tfrac{1}{T}\ln(P_1/P_0)$.
Logistic-style separable ODE producing $\ln\left|\tfrac{y}{a-y}\right| = kt + C$.
Apply $t = 0$ to fix $C$ from initial $y$, then the later data point gives a linear equation in $k$ after taking logs.
Condition given at $t = t_0 \neq 0$ but with a clean value (e.g. $y \to 0$ or a known equilibrium).
Substitute the simpler condition first; it may not annihilate kt but still yields a one-variable equation if the other side simplifies.

Common Misreadings

  • Treating both conditions symmetrically and setting up simultaneous equations instead of exploiting t = 0.
  • Reading 'initially' as t = 1 or some non-zero starting time.
  • Forgetting that the constant of integration absorbs any constants from F(y), so C is read directly from F(y_0).

Pitfalls

  • Using the non-zero-time condition first, producing an equation with both k and C — solvable but error-prone.
  • Arithmetic slips evaluating cubes or other powers (e.g. $12^3 = 1728$, $6^3 = 216$).
  • Sign errors when the model is written as $-kt$ versus $+kt$ — losing track of the sign of k.
  • Forgetting to substitute the found constants back to state the particular solution explicitly.

A-level Scope Notes

Standard Edexcel IAL P3/P4 separable ODE technique: integrate, then apply boundary conditions sequentially to determine constants. No methods beyond IAL Maths/Further Maths are required.
