Algebra & Functions

Require radicand non-negative to determine domain

Family: Domain Restriction Frequency: 1 questions

Trigger — When to use

The function involves a real square root (or any even-index radical) such as f(x)=\sqrt{g(x)}, and the question asks for the domain, the largest valid set of inputs, or values of x for which f is defined. Cues include phrases like 'state the domain', 'find the set of values of x for which f(x) is real/defined', or a piecewise definition that includes a surd. It also fires whenever a surd appears as a sub-expression inside a larger function whose domain is being assembled.

Motivation — Why this works

In the reals, \sqrt{A} exists only when A\ge 0, so the domain of f is exactly the solution set of the inequality g(x)\ge 0. This converts a 'domain' question into a standard inequality problem, which A-level students can already solve by linear, quadratic, or sign-table methods. Equality is included because \sqrt{0}=0 is a valid real output.

Thought Process — The step-by-step reasoning

  1. Notice the square root symbol: outputs are only real when the radicand is non-negative.
  2. Rule out other domain restrictions (no denominator, no log, no tan, etc.) so the radicand condition is the only constraint.
  3. Commit to writing the inequality g(x)\ge 0 because every x satisfying it gives a real f(x), and every x violating it does not.
  4. Translate this into the concrete inequality using the actual radicand from the problem.
  5. Choose the right solving tactic: rearrange if linear, factorise/complete the square if quadratic, and decide whether the answer should be in inequality or interval notation.

Mechanic — Concrete operations

  1. Identify the radicand g(x) inside the square root.
  2. Write the condition g(x)\ge 0 (with weak inequality, since \sqrt{0}=0).
  3. Solve the inequality using appropriate algebra (linear rearrangement, quadratic inequality, sign diagram).
  4. State the domain as an inequality in x or in interval/set notation, e.g. x\ge -5 or [-5,\infty).

Worked Micro-Example

Find the domain of g(x)=\sqrt{x+5}.\\ Require x+5\ge 0.\\ Hence x\ge -5, so the domain is [-5,\infty).

Variants — Same move, different surface

Quadratic radicand, e.g. f(x)=\sqrt{x^2-3x-4}
Solve x^2-3x-4\ge 0 by factorising to (x-4)(x+1)\ge 0 and using a sign diagram, giving x\le -1 or x\ge 4.
Surd combined with a reciprocal, e.g. f(x)=\dfrac{1}{\sqrt{x-2}}
Strengthen to strict inequality x-2>0 because the radicand also sits in a denominator and cannot be zero.
Surd inside a composite, e.g. f(x)=\ln(\sqrt{x-1})
Combine constraints: radicand \ge 0 and the log argument >0, which together force x>1.
Sum/intersection of surds, e.g. f(x)=\sqrt{x-1}+\sqrt{5-x}
Impose each radicand non-negative and intersect: x\ge 1 and x\le 5, giving 1\le x\le 5.
Restricted domain stated in the question, e.g. f:x\mapsto\sqrt{x+5},\;x\in\mathbb{R}
Intersect the natural domain x\ge -5 with any stated restriction before quoting the final answer.

Common Misreadings

  • Treating the domain question as a range question and solving for y instead of x.
  • Assuming the domain is all real numbers because no obvious denominator is present, ignoring the surd.
  • Confusing 'domain' with 'where the graph crosses the x-axis' (i.e. setting the radicand equal to zero only).

Pitfalls

  • Using strict inequality x>-5 instead of x\ge -5; \sqrt{0}=0 is defined.
  • Flipping the inequality sign when rearranging (e.g. writing x\le -5).
  • For quadratic radicands, taking the inside-the-roots region instead of the outside region (or vice versa) without checking with a sign diagram.
  • Forgetting to intersect with additional restrictions when the surd is part of a larger expression (denominators, logs, other surds).

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics: domain of real functions, solving linear and quadratic inequalities, and interval notation. No complex-valued square roots are considered.
