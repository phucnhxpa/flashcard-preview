Trigonometry

Apply R-formula to combine a sin x ± b cos x into a single sinusoid

Family: R Formula Frequency: 2 questions

Trigger — When to use

The expression contains a linear combination of sin x and cos x with the SAME argument, e.g. a sin x + b cos x, a cos x − b sin x, or 3 sin x − 4 cos x. Either the question explicitly asks for the form R sin(x ± α) / R cos(x ± α), OR you need to find a maximum/minimum, OR you must solve an equation a sin x + b cos x = c which cannot be inverted directly because two different trig functions are interleaved. Visual cues: two trig terms of the same x with numeric coefficients, often followed by sub-questions about extrema, range, or solving = constant.

Motivation — Why this works

The compound angle formulae give sin(x ± α) = sin x cos α ± cos x sin α and cos(x ∓ α) = cos x cos α ± sin x sin α. So any a sin x + b cos x has exactly the structure of R sin(x ± α) or R cos(x ∓ α) with R cos α and R sin α matching the original coefficients. Collapsing two oscillating terms into one immediately gives the amplitude R = √(a² + b²), the maximum/minimum (±R), and reduces equations to the elementary form sin θ = k or cos θ = k which can be inverted by standard methods.

Thought Process — The step-by-step reasoning

  1. I notice the expression is a linear combination of sin x and cos x with the same argument and constant coefficients — a clear R-formula signature.
  2. I rule out alternatives: squaring both sides would mix sin²x, cos²x and a cross term and risks spurious roots; substituting t = tan(x/2) is heavier and not needed; isolating one term is impossible because both contain x.
  3. I commit to the R-formula because it produces a single sinusoid whose amplitude and phase shift are both directly readable, and which inverts to a standard equation.
  4. I choose the target form to MATCH the sign pattern: a sin x + b cos x → R sin(x + α); a sin x − b cos x → R sin(x − α); a cos x + b sin x → R cos(x − α); a cos x − b sin x → R cos(x + α). If the question dictates a particular form, I use that one and just match coefficients carefully.
  5. I expand the chosen form, equate coefficients of sin x and cos x to give two equations R cos α = … and R sin α = …, both with α acute (first quadrant) so that R > 0.
  6. I solve by (i) square-and-add to get R² = a² + b², so R = √(a² + b²); (ii) divide the two equations to get tan α = (sin-coefficient)/(cos-coefficient), giving α = arctan(…) in the correct units (radians or degrees as the question demands).

Mechanic — Concrete operations

  1. Identify a, b in a sin x + b cos x (or the analogous cos-led form) and pick the matching target: e.g. a sin x − b cos x = R sin(x − α).
  2. Expand the target: R sin(x − α) = (R cos α) sin x − (R sin α) cos x.
  3. Equate coefficients: R cos α = a, R sin α = b (both positive with α acute).
  4. Square and add: R² = a² + b² ⇒ R = √(a² + b²).
  5. Divide: tan α = b/a ⇒ α = arctan(b/a), keeping α in (0, π/2) since both R cos α and R sin α are positive.
  6. Write the final identity, e.g. a sin x − b cos x ≡ √(a² + b²) sin(x − α), and use it to solve the original equation or read off max/min.

Worked Micro-Example

Express $3\sin x - 4\cos x$ as $R\sin(x-\alpha)$. Expand: $R\sin(x-\alpha) = R\cos\alpha\sin x - R\sin\alpha\cos x$. Match: $R\cos\alpha = 3$, $R\sin\alpha = 4$. Then $R^2 = 9+16 = 25 \Rightarrow R=5$, and $\tan\alpha = 4/3 \Rightarrow \alpha = \arctan(4/3) \approx 0.927$ rad. Hence $3\sin x - 4\cos x \equiv 5\sin(x-0.927)$, so $3\sin x - 4\cos x = 2$ becomes $\sin(x-0.927) = 0.4$.

Variants — Same move, different surface

Expression is a cos x + b sin x and form requested is R cos(x − α)
Expand R cos(x − α) = R cos α cos x + R sin α sin x; match R cos α = a, R sin α = b; same square-and-add and divide give R and α.
Expression is a cos x − b sin x
Use R cos(x + α) since cos(x + α) = cos α cos x − sin α sin x preserves the minus sign; match R cos α = a, R sin α = b.
Coefficients are surds or fractions (e.g. √3 sin x + cos x)
Same procedure; R = √(3+1)=2, tan α = 1/√3 ⇒ α = π/6; exact-value answer expected.
Different argument scaling, e.g. a sin(2x) + b cos(2x)
Treat 2x as the single variable; result is R sin(2x + α). When solving, remember to divide the bound on x by 2 only after working in the 2x domain.

Common Misreadings

  • Choosing R sin(x + α) when the cos x term is subtracted (should be R sin(x − α)), producing a sign error in α.
  • Using sine form when the question explicitly demands cosine form (or vice versa) — costs marks even if numerically equivalent.
  • Confusing which coefficient pairs with sin α and which with cos α; always expand the chosen target FIRST and then compare.

Pitfalls

  • Forgetting to give α in the units the question uses — radians vs degrees — especially when later parts use radian bounds.
  • Reporting α outside the acute first-quadrant range; with a, b > 0, α must satisfy 0 < α < π/2.
  • When solving R sin(x − α) = c, forgetting the second solution from the supplementary angle π − (x − α) and missing roots in the given interval.
  • Computing R as a + b or √a + √b instead of √(a² + b²).

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P3) — the compound angle formulae used are listed in the IAL formula booklet, but the R-formula derivation itself is expected to be reproduced. Avoid invoking complex-exponential derivations (Euler's formula) which are out of scope at this stage.
