Integration

Split a trig power as f^(n-1)·f (or f^(n-2)·f²) to set up a clean reduction-formula IBP

Family: Ibp SplittingFrequency: 4 questions
Use when
When a reduction formula is requested for I_n = ∫f^n(x) dx where f is a single trig function (sin, cos, sec, cosec) and the goal is to relate I_n to I_{n-2}.

Trigger — When to use

Fires whenever a reduction formula is requested for I_n = ∫f^n(x) dx where f is a single trig function (sin, cos, sec, cosec) and the goal is to relate I_n to I_{n-2}. Visual cues: a single trig function raised to an integer power n, the phrase 'show that I_n = ... I_{n-2}' or 'derive a reduction formula', and either definite limits like 0 to π/2 (which often kill boundary terms) or an indefinite integral. Equivalently: any 'power-of-one-trig-function' integrand with no obvious u-substitution available.

Motivation — Why this works

The move works because it engineers an IBP pairing where one factor differentiates to lower the power (producing a trig identity opportunity) and the other factor integrates cleanly. For sin^n and cos^n, peeling off a single sin or cos gives dv with a one-line antiderivative (∓cos or sin), and the chain rule on f^(n-1) introduces the partner trig function — the product of the two then becomes a Pythagorean square (cos²x or sin²x) which is rewritten via 1 − sin²x or 1 − cos²x to reproduce I_n itself, allowing algebraic rearrangement. For sec^n, the asymmetric split f^(n-2)·sec²x is forced because sec²x is the one sec-power with an elementary antiderivative (tan x). The underlying principle is 'product rule reversed, chosen so that the leftover integral closes back into the same family'.

Thought Process — Step-by-step reasoning

  1. Notice the structure: integrand is a single trig function raised to power n, and the task wants a recurrence linking I_n to I_{n-2} — this rules out u-substitution and demands IBP.
  2. Rule out alternatives: u = f^n with dv = dx gives v = x (useless); splitting as f^(n-2)·f² leaves dv = f²dx which for sin²/cos² introduces an x term (messy) — except in the sec case, where sec² is precisely the clean dv.
  3. Commit to the (n-1, 1) split for sin^n and cos^n because differentiating f^(n-1) drops the power by 1 and produces the partner trig function, while integrating the lone f gives a one-term antiderivative; for sec^n commit to the (n-2, 2) split because sec²x is the only sec-power with elementary antiderivative tan x.
  4. Bridge to operation: write I_n = ∫f^(n-1)(x)·f(x) dx (or ∫sec^(n-2)x·sec²x dx), set u = the higher power, dv = the lone factor, and apply IBP.
  5. Tactical check on boundary term: for limits 0 to π/2 with sin^n, the term [−sin^(n-1)x cos x] vanishes at both ends, so the recurrence is clean; for indefinite integrals carry the boundary term as sec^(n-2)x tan x or similar.
  6. Anticipate the closure: the leftover integral will contain cos²x = 1 − sin²x (or tan²x = sec²x − 1), which splits into I_{n-2} and I_n, letting you solve algebraically for I_n.

Mechanic — Concrete operations

  1. Rewrite I_n by peeling off one factor: I_n = ∫sin^(n-1)x · sin x dx (or ∫sec^(n-2)x · sec²x dx for sec^n).
  2. Set u = sin^(n-1)x and dv = sin x dx (or u = sec^(n-2)x and dv = sec²x dx).
  3. Differentiate carefully with the chain rule: du = (n−1)sin^(n-2)x cos x dx (or du = (n−2)sec^(n-2)x tan x dx).
  4. Integrate dv: v = −cos x (or v = tan x).
  5. Apply IBP: I_n = [uv]_a^b − ∫v du, evaluate the boundary term (often 0 on [0, π/2] for sin^n).
  6. Convert the remaining integral via Pythagoras: cos²x = 1 − sin²x (or tan²x = sec²x − 1) to express it as a combination of I_n and I_{n-2}.
  7. Solve algebraically for I_n in terms of I_{n-2} to obtain the reduction formula.

Worked Micro-Example

For $I_n = \int_0^{\pi/2} \sin^n x\,dx$: write $I_n = \int_0^{\pi/2} \sin^{n-1}x \cdot \sin x\,dx$. With $u = \sin^{n-1}x$, $dv = \sin x\,dx$, get $du = (n-1)\sin^{n-2}x\cos x\,dx$, $v = -\cos x$. IBP gives $I_n = [-\sin^{n-1}x\cos x]_0^{\pi/2} + (n-1)\int_0^{\pi/2}\sin^{n-2}x\cos^2 x\,dx = 0 + (n-1)(I_{n-2} - I_n)$, so $nI_n = (n-1)I_{n-2}$.

Variants — Same move, different surface

For ∫cos^n x dx
Mirror the logic: u = cos^(n-1)x, dv = cos x dx, v = sin x; the partner identity is sin²x = 1 − cos²x.
For ∫sec^n x dx (or cosec^n x dx)
Use the asymmetric split sec^(n-2)x · sec²x with dv = sec²x dx, v = tan x; close via tan²x = sec²x − 1.
For ∫tan^n x dx (or cot^n x dx)
No IBP needed — write tan^n x = tan^(n-2)x · (sec²x − 1) and split into a direct sub and I_{n-2}.
For ∫x^n e^x dx (analogous power-reduction)
Same philosophy: u = x^n (differentiates to lower the power), dv = e^x dx (integrates cleanly). Each IBP drops the power by 1.

Common Misreadings

  • Trying to halve the power (u = sin^(n/2)x) — only works for even n and produces no clean recurrence.
  • Choosing dv = sec x dx in the sec^n case, which gives v = ln|sec x + tan x| and destroys the sec-power family.
  • Treating the integrand as needing u-substitution because it's 'just one trig function' — a single power has no inner function to substitute.

Pitfalls

  • Forgetting the chain-rule coefficient (n−1) or (n−2) when differentiating sin^(n-1)x or sec^(n-2)x.
  • Sign error in IBP: v = −cos x combined with du containing +cos x gives a +(n−1)∫… term, not minus.
  • Forgetting the chain rule on sec: d/dx[sec^(n-2)x] = (n−2)sec^(n-2)x tan x (not sec^(n-3)x tan x).
  • Failing to evaluate the boundary term explicitly — for indefinite integrals it must be retained, for [0, π/2] with sin^n it vanishes but should be shown.
  • After Pythagorean substitution, forgetting that the resulting I_n term must be moved to the LHS before stating the reduction formula.

A-level Scope Notes

Fully within Edexcel IAL Further Pure (FP2/FP3) reduction-formula material. Uses IBP (in the formula booklet) and standard Pythagorean identities sin²+cos²=1, 1+tan²=sec². Avoid generalizing to ∫sin^m x cos^n x dx with both powers variable — that's a multi-case problem beyond a single reduction formula and not the target of this move.
