Differentiation

Translate tangent gradient conditions into algebraic equations on dy/dx

Family: Geometric To Algebraic Translation Frequency: 2 questions

Trigger — When to use

Fires whenever a question describes a tangent line by a geometric property — 'parallel to', 'perpendicular to', 'horizontal', 'vertical', 'has gradient m', or 'parallel tangents to two curves' — and asks for the x-coordinate, point, or proof of (non-)existence. The cue is a worded geometric constraint about a tangent's direction without an explicit equation. Alternative phrasings: 'find the point where the tangent has slope k', 'show the curve has no vertical tangent', 'find α such that the tangent at α is parallel to l₁'. Whenever you see such phrasing, your first instinct should be to convert the geometric description of the tangent into a numerical/algebraic condition on dy/dx.

Motivation — Why this works

The gradient of a tangent at a point equals dy/dx evaluated there, so any geometric description of the tangent's direction (parallel/perpendicular/horizontal/vertical/slope = k) becomes a direct equation in dy/dx. Parallel ⇒ equal gradients; perpendicular ⇒ product of gradients = −1; horizontal ⇒ dy/dx = 0 (numerator zero for rational forms); vertical ⇒ dy/dx undefined (denominator zero for rational forms). This translation collapses a geometric mystery into a routine algebraic equation that can be solved or shown to have no solution, often combined with the original curve equation.

Thought Process — The step-by-step reasoning

  1. Notice the tangent is described geometrically (parallel, vertical, of given slope) rather than by an equation — so I cannot work with line equations directly yet.
  2. Rule out trying to equate y-values (that would be 'curves meet') or solving simultaneously with l₁ (that finds intersections, not parallel tangents). The geometric cue is about direction, which lives in the gradient.
  3. Commit to the principle: tangent direction at x = α is captured entirely by dy/dx at α. So translate the geometric word into an algebraic statement about dy/dx.
  4. Choose the right translation: 'parallel to a known line' → set dy/dx = (gradient of that line); 'horizontal' → dy/dx = 0; 'vertical' / 'undefined gradient' → if dy/dx = N/D, set D = 0 (and check N ≠ 0); 'perpendicular' → dy/dx = −1/m.
  5. Compute dy/dx (explicit or implicit differentiation as appropriate) and substitute into the chosen condition to get a single equation in x (or in x and y, to be combined with the curve).
  6. Solve the resulting equation. If the original tangent point is already known to satisfy it (e.g. l₁ touches at x = 6), use that root as a known factor to simplify factorisation; reject it if the question demands a distinct second tangent.
  7. Verify the candidate point actually lies on the curve, and check no degenerate cases (e.g. 0/0 in the rational dy/dx) invalidate the geometric interpretation.

Mechanic — Concrete operations

  1. Compute dy/dx for the curve C (directly, or implicitly if F(x,y) = 0).
  2. Translate the geometric condition: parallel/slope = m ⇒ dy/dx = m; horizontal ⇒ numerator of dy/dx = 0; vertical ⇒ denominator of dy/dx = 0; perpendicular ⇒ dy/dx = −1/m.
  3. Form the equation, e.g. 3x² − 20x + 32 = 20, simplifying to 3x² − 20x + 12 = 0; or 2y − 2x = 0 ⇒ y = x.
  4. Solve: factorise (using any known root from earlier parts, e.g. (x − 6) as a factor), or apply the quadratic formula, or substitute y = x into the curve equation for vertical-tangent problems.
  5. Discard extraneous roots (e.g. the root corresponding to the original tangent point if a distinct one is required) and verify the surviving point lies on C.
  6. For 'show no such point exists' tasks, demonstrate the system has no real solution.

Worked Micro-Example

Curve C: y = x³ − 10x² + 32x. l₁ has gradient 20 and is tangent at x = 6. Find α ≠ 6 where the tangent l₂ is parallel to l₁. Set dy/dx = 20: 3x² − 20x + 32 = 20 ⇒ 3x² − 20x + 12 = 0. Since x = 6 is a known root, factor: (x − 6)(3x − 2) = 0. So α = 2/3.

Variants — Same move, different surface

Horizontal tangent (gradient zero) on a curve with rational dy/dx = N(x,y)/D(x,y)
Set N(x,y) = 0 (and check D ≠ 0 there), then combine with the curve equation.
Vertical tangent on an implicit curve
Set the denominator of dy/dx to zero; substitute the resulting relation into the curve equation; if no real solution exists, conclude no vertical tangent.
Parametric curve with dy/dx = (dy/dt)/(dx/dt)
Horizontal tangent ⇒ dy/dt = 0 (with dx/dt ≠ 0); vertical tangent ⇒ dx/dt = 0 (with dy/dt ≠ 0).
Two curves with parallel tangents at a common x
Set dy/dx of curve 1 equal to dy/dx of curve 2 and solve.
Perpendicular tangent to a given line of gradient m
Set dy/dx = −1/m instead of dy/dx = m.

Common Misreadings

  • Treating 'parallel to l₁' as 'meets l₁' and solving y_curve = y_line simultaneously.
  • Setting dy/dx itself to zero when the question says 'vertical' (that produces horizontal tangents — the opposite).
  • Forgetting that a tangent point must lie on the curve, so only setting dy/dx-conditions without checking compatibility with C.

Pitfalls

  • Reporting both roots of the gradient equation when one corresponds to the already-given tangent and must be excluded.
  • Factorisation slips on non-monic quadratics like 3x² − 20x + 12 = 0; using a known root (x = 6) to do polynomial division avoids guesswork.
  • For vertical tangents on rational dy/dx, ignoring the 0/0 case — this can indicate a cusp/node rather than a true vertical tangent.
  • For perpendicular conditions, forgetting the negative sign in −1/m.

A-level Scope Notes

All within Edexcel IAL P2/P3/P4 differentiation: gradient = dy/dx, parallel ⇔ equal gradients, perpendicular ⇔ product of gradients = −1 (formula booklet, coordinate geometry). Implicit differentiation is in P3. Avoid invoking singular-point analysis (cusps, nodes) beyond noting the 0/0 caveat — that is out of scope.
