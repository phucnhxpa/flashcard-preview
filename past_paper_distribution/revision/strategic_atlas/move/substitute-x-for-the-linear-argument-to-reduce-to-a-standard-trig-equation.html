Trigonometry

Substitute X for the linear argument to reduce to a standard trig equation

Family: Argument Substitution Frequency: 1 questions

Trigger — When to use

The equation contains a trig function whose argument is a linear expression in x, e.g. cos(πx/6) = 0, sin(2x + π/3) = 1/2, or tan(3x − π/4) = 1. Solving directly mixes the trig inverse step with algebraic rescaling/shifting, inviting errors. Cue phrases: 'find x such that f(ax+b) = k', 'where does the curve y = A cos(ωx) cross the axis', or any periodic equation with a non-identity inner function.

Motivation — Why this works

Replacing the inner expression with a single letter X separates two independent tasks: (i) solving the standard equation cos X = 0, sin X = k, tan X = k, whose solution set is memorised, and (ii) inverting the linear map X = ax + b to recover x. Because the inner function is linear and bijective, every X-solution corresponds to exactly one x-solution, so no roots are gained or lost. The substitution also makes domain transformation explicit: the x-range maps to a clean X-range in which counting solutions is straightforward.

Thought Process — The step-by-step reasoning

  1. I notice the trig function is applied to a linear expression in x, not to x itself, so the equation isn't yet in a 'lookup' form.
  2. I rule out trying to strip the argument by dividing/multiplying outside the cosine — cos is not linear, so that destroys the equation.
  3. I commit to the substitution X = (inner argument) because the inner map is linear and invertible, guaranteeing a clean two-way correspondence between X- and x-solutions.
  4. First concrete operation: write down X = ax + b, rewrite the equation as a standard trig equation in X, and translate the x-domain into an X-domain.
  5. Tactical decisions: list ALL X-solutions in the transformed range (not just the principal value), then back-substitute x = (X − b)/a, and finally verify each x lies in the original domain.

Mechanic — Concrete operations

  1. Isolate the trig function: divide/transpose so the equation reads cos(ax+b) = k (or sin/tan analogue).
  2. Let X = ax + b; rewrite as cos X = k.
  3. Transform the domain: if α ≤ x ≤ β, then aα + b ≤ X ≤ aβ + b.
  4. List every X in the transformed range satisfying cos X = k, using the general solution X = ±arccos(k) + 2nπ.
  5. Back-substitute x = (X − b)/a for each X-solution.
  6. Check each x lies in the original domain and discard any that do not.

Worked Micro-Example

Solve $\sqrt{3}\cos(\pi x/6) = 0$ for $0 \le x \le 2\pi$. Divide by $\sqrt{3}$: $\cos(\pi x/6)=0$. Let $X=\pi x/6$, so $0 \le X \le \pi^2/3 \approx 3.29$. Then $\cos X = 0 \Rightarrow X = \tfrac{\pi}{2}$ (next root $\tfrac{3\pi}{2}\approx 4.71$ is out of range). Back-substitute: $x = \tfrac{6}{\pi}\cdot\tfrac{\pi}{2}=3$.

Variants — Same move, different surface

sin(2x + π/3) = 1/2 on [0, 2π]
Let U = 2x + π/3; transformed range is [π/3, 4π + π/3]; use sin U = 1/2 ⇒ U = π/6 + 2nπ or 5π/6 + 2nπ; then x = (U − π/3)/2.
tan(3x) = √3 on [0, π]
Let U = 3x; transformed range [0, 3π]; tan U = √3 ⇒ U = π/3 + nπ; then x = U/3.
cos(x − π/4) = −1/2
Let U = x − π/4 (pure shift, a=1); solve cos U = −1/2 then add π/4 back.
Curve y = A sin(ωx + φ) crossing a horizontal line
Same substitution U = ωx + φ; the y-intercept/zero-crossing question becomes a standard sin equation in U.

Common Misreadings

  • Treating cos(πx/6) as (π/6)·cos(x) and dividing through by π/6 before inverting cosine.
  • Reading the substitution as 'solve cos x = 0 then scale x by 6/π', i.e. forgetting that the substitution applies to the argument, not to the eventual x-value naively.
  • Believing the substitution might introduce extraneous roots, prompting unnecessary checking; in fact a linear substitution is bijective.

Pitfalls

  • Failing to transform the domain: solving cos X = 0 over [0, 2π] when the actual X-range is [0, π²/3] gives spurious solutions.
  • Listing only the principal value (e.g. arccos(0) = π/2) and missing the second family 3π/2 + 2nπ, or for sin missing the π − θ partner.
  • Arithmetic slip when inverting the linear map: writing x = X·a/1 instead of x = (X − b)/a.
  • Forgetting the constant factor outside (e.g. dividing by √3 first) before the substitution, leading to cos X = 0 vs cos X = something else.

A-level Scope Notes

Fully within Edexcel IAL P2/P3 trigonometry: solving trig equations with linear arguments and stated domains. The substitution is informal notation, not a change of variable in the calculus sense, and uses only standard exact values and general solutions on the syllabus.
