Trigonometry

Evaluate sin(arccos k) using Pythagorean identity

Family: Inverse Trig Evaluation Frequency: 1 questions

Trigger — When to use

Fires when a composite expression like sin(arccos k), cos(arcsin k), or tan(arccos k) appears — typically after substituting a non-standard angle limit into an evaluated antiderivative. Cues include: the inverse trig argument is a fraction not corresponding to a standard angle (e.g. 1/4, 2/3), and a numeric/exact value is required. Often arises at the bounds of definite integrals after a trig substitution, or when simplifying an expression to surd form.

Motivation — Why this works

Setting θ = arccos k means cosθ = k, and the identity sin²θ + cos²θ = 1 immediately yields sinθ = √(1−k²) (taking the positive root because arccos returns θ ∈ [0,π] where sin is non-negative). This converts a nested inverse-trig expression into a clean surd without ever needing the angle itself. Geometrically it is reading the opposite side of a right triangle with adjacent k and hypotenuse 1.

Thought Process — The step-by-step reasoning

  1. Notices the argument of arccos is not a standard cosine value (½, √2/2, √3/2), so the angle won't simplify.
  2. Rules out evaluating θ numerically or via a calculator approximation — an exact surd is required.
  3. Commits to the identity route: name the angle θ = arccos k, exploit cosθ = k, and recover sinθ from sin²θ + cos²θ = 1.
  4. Bridges to computation: square k, subtract from 1, take the positive square root.
  5. Decides the sign by recalling arccos has range [0,π] where sinθ ≥ 0, so no ± ambiguity remains.

Mechanic — Concrete operations

  1. Let θ = arccos(k), so cosθ = k.
  2. Compute sin²θ = 1 − k².
  3. Take sinθ = √(1 − k²), positive since θ ∈ [0, π].
  4. Substitute back into the original expression.

Worked Micro-Example

Evaluate $\sin(\arccos(1/4))$. Let $\theta=\arccos(1/4)$, so $\cos\theta=1/4$. Then $\sin^2\theta = 1-\tfrac{1}{16}=\tfrac{15}{16}$, and since $\theta\in[0,\pi]$, $\sin\theta=\tfrac{\sqrt{15}}{4}$. Hence e.g. $\tfrac12\bigl[\sin\theta\bigr]_{\arccos(1/4)}^{\pi/2}=\tfrac12\bigl(1-\tfrac{\sqrt{15}}{4}\bigr)=\tfrac12-\tfrac{\sqrt{15}}{8}$.

Variants — Same move, different surface

cos(arcsin k)
Set θ = arcsin k so sinθ = k; then cosθ = √(1−k²), positive since arcsin's range [−π/2,π/2] gives cosθ ≥ 0.
tan(arccos k) with k ≠ 0
After sinθ = √(1−k²), divide by cosθ = k to get tanθ = √(1−k²)/k.
sin(arctan k)
From tanθ = k, build the right triangle with opposite k, adjacent 1, hypotenuse √(1+k²); then sinθ = k/√(1+k²).
Integration bound after x = a sinθ or x = a cosθ substitution
Same identity recovers the missing trig value at a non-standard limit; factor of a from the substitution may need reinstating.

Common Misreadings

  • Reading sin(arccos(1/4)) as 'sin of 1/4' or as 1/4 itself.
  • Treating arccos as the reciprocal 1/cos rather than the inverse function.
  • Assuming the angle must be a standard one and abandoning the problem when it isn't.

Pitfalls

  • Writing sin(arccos(1/4)) = 1/4 by confusing sin/cos roles.
  • Forgetting the square root and reporting 15/16 instead of √15/4.
  • Introducing a ± and not eliminating the negative branch using the range of arccos.
  • Squaring errors, e.g. (1/4)² = 1/4 instead of 1/16.

A-level Scope Notes

Fully within Edexcel IAL Pure scope: uses only sin²θ + cos²θ = 1 and the standard ranges of arcsin/arccos/arctan, both examined in P3/P4 trigonometry and trig-substitution integration.
