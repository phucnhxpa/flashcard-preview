Algebra & Functions

Rewrite rational function by long division to expose asymptote

Family: Algebraic Restructure Frequency: 1 questions

Trigger — When to use

Fires when asked for the range, horizontal/oblique asymptote, or a sketch of a rational function f(x) = (ax+b)/(cx+d) where numerator and denominator have equal (or numerator-greater) degree. Cues include 'find the range', 'state the asymptotes', or 'sketch y = f(x)' for such a quotient. Also fires when integrating an improper rational expression, where the proper-fraction form is required before partial fractions or direct integration.

Motivation — Why this works

Polynomial division decomposes f(x) into a polynomial part plus a proper fraction, separating the asymptotic behaviour from the bounded variation. The constant (or polynomial) part gives the horizontal (or oblique) asymptote directly, and the proper fraction is a translated reciprocal whose range is ℝ \ {0}. Adding the constant shifts the excluded value to the asymptote, so the range follows by inspection. This restructuring turns an opaque quotient into a transparent shifted reciprocal.

Thought Process — The step-by-step reasoning

  1. Notices that f(x) = (2x+3)/(x−1) has equal numerator and denominator degree, so a horizontal asymptote exists but its value isn't visible in this form.
  2. Rules out the algebraic 'set y = f(x), solve for x, find excluded y' approach as slower and more error-prone.
  3. Commits to long division because equal degrees guarantee a constant quotient plus a proper remainder — exactly the shape of a shifted reciprocal.
  4. Bridges to action by writing 2x+3 = 2(x−1) + 5, giving f(x) = 2 + 5/(x−1).
  5. Reads off: horizontal asymptote y = 2, vertical asymptote x = 1, and range ℝ \ {2} since 5/(x−1) is never 0.

Mechanic — Concrete operations

  1. Identify that deg(numerator) ≥ deg(denominator); long division applies.
  2. Divide: write numerator = (quotient)·(denominator) + remainder, e.g. 2x+3 = 2(x−1) + 5.
  3. Rewrite f(x) = quotient + remainder/(denominator), e.g. f(x) = 2 + 5/(x−1).
  4. Read the asymptote from the quotient (y = 2 here; or y = mx+c if quotient is linear).
  5. Determine the range: the proper fraction takes all real values except 0, so f(x) takes all values except the constant in the quotient.

Worked Micro-Example

Find the range of $f(x) = \dfrac{2x+3}{x-1}$, $x \neq 1$.\\ Divide: $2x+3 = 2(x-1) + 5$, so $f(x) = 2 + \dfrac{5}{x-1}$.\\ Since $\dfrac{5}{x-1} \neq 0$ for any real $x$, $f(x) \neq 2$, and it attains every other real value.\\ Hence range $= \mathbb{R} \setminus \{2\}$, with horizontal asymptote $y=2$.

Variants — Same move, different surface

General linear-over-linear f(x) = (ax+b)/(cx+d)
Division gives a/c + (bc−ad)/(c(cx+d)); horizontal asymptote y = a/c and range ℝ \ {a/c}.
Improper quotient with deg(num) = deg(denom) + 1, e.g. (x²+1)/(x−2)
Quotient is linear, giving an oblique asymptote y = mx+c rather than horizontal; range analysis then needs calculus or discriminant, not simple inspection.
Integration of improper rational functions
Long division first, then integrate quotient directly and apply partial fractions to the proper remainder.
Sketching y = (ax+b)/(cx+d)
Divided form exposes both asymptotes (x = −d/c, y = a/c) and shows the curve as a translated/scaled reciprocal 1/x.

Common Misreadings

  • Confusing the excluded domain value (x = 1) with the excluded range value (y = 2).
  • Trying to find the range by plugging in sample x-values, which cannot prove a value is excluded.
  • Reading the horizontal asymptote as the ratio of constants b/d instead of leading coefficients a/c.

Pitfalls

  • Forgetting to justify '5/(x−1) ≠ 0' — the entire range argument depends on this.
  • Algebraic slip in the division step (e.g. writing 2x+3 = 2(x−1) + 1 instead of +5).
  • Stating the range as ℝ without excluding the asymptote value.

A-level Scope Notes

Standard Edexcel IAL Pure (functions, graphs of rational functions) and Further Pure (rational function sketching, asymptotes, integration via long division before partial fractions).
