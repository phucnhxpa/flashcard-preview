Integration

Split a rational fraction term-by-term into standard integrable pieces

Family: Algebraic Decomposition Frequency: 3 questions

Trigger — When to use

The integrand is a rational function whose denominator is a single power of a linear factor (e.g. u², (x+1)) or a monomial, and whose numerator is a low-degree polynomial. Either (i) numerator degree < denominator degree but the fraction is not a single standard form, or (ii) numerator degree = denominator degree (improper) so direct integration fails. Visual cues: 'one fraction bar, monomial or repeated linear denominator, polynomial on top'. Equivalent phrasings: 'divide each term by the denominator', 'write top in terms of bottom', '(x+1)−1 over (x+1)' trick.

Motivation — Why this works

Splitting the fraction converts a non-standard rational expression into a sum of elementary forms — typically 1, 1/u, and u^(−n) — each of which integrates immediately via the power rule or to ln|u|. The underlying principle is just linearity of division: (A+B)/C = A/C + B/C, plus the algebraic identity 'rewrite the numerator as (denominator) ± (remainder)' for improper fractions. This avoids the heavier machinery of formal partial fractions or u-substitution when the denominator is a single repeated factor, and it produces antiderivatives without any further technique.

Thought Process — The step-by-step reasoning

  1. Notice the integrand is a single rational fraction that does not match any standard integral form (not 1/u, not u^n, not 1/(1+u²), etc.).
  2. Check the degrees: if num-degree < denom-degree AND the denominator is a monomial like u² or (x+a)^n, term-by-term division will work directly; if num-degree = denom-degree, rewrite the numerator as (denominator) − (remainder) first.
  3. Rule out partial fractions — they would only re-derive the same split when the denominator is a single repeated factor — and rule out u-substitution as unnecessarily long for a linear denominator.
  4. Commit to the split because every resulting term will be either k/u (→ ln|u|) or k·u^(−n) (→ power rule), both standard.
  5. Execute the algebra carefully, watching the sign on each piece (especially the −u^(−2) → +u^(−1) flip), then integrate term by term and combine.

Mechanic — Concrete operations

  1. Identify the denominator D and the numerator N; compare degrees.
  2. If deg N < deg D and D is a monomial/repeated factor: divide each term of N by D, e.g. (u−1)/u² = u/u² − 1/u² = 1/u − u^(−2).
  3. If deg N = deg D: rewrite N as (D) − r so that N/D = 1 − r/D, e.g. x/(x+1) = [(x+1)−1]/(x+1) = 1 − 1/(x+1).
  4. Integrate each piece using ∫1/u du = ln|u| and ∫u^n du = u^(n+1)/(n+1) (n ≠ −1).
  5. Combine, restore constants/limits, and apply bounds if the integral is definite.

Worked Micro-Example

Evaluate $\int_1^2 \frac{u-1}{u^2}\,du$. Split: $\frac{u-1}{u^2}=\frac{1}{u}-u^{-2}$. Antiderivative: $\ln|u|+\frac{1}{u}$. Evaluate: $\big[\ln 2 + \tfrac12\big] - \big[0+1\big] = \ln 2 - \tfrac12$.

Variants — Same move, different surface

Numerator with several terms over a monomial, e.g. (x²+3x−1)/x³
Divide each numerator term by x³ to get 1/x + 3/x² − 1/x³, then apply power rule / ln to each.
Improper fraction with linear denominator, e.g. x/(x+1)
Write x = (x+1) − 1, giving 1 − 1/(x+1); integrates to x − ln|x+1|.
Improper fraction with quadratic denominator, e.g. (x²+1)/(x²−1)
Write numerator as (denominator) + remainder: 1 + 2/(x²−1); the remainder then needs partial fractions.
Denominator is a product of distinct linear factors, e.g. (u−1)(u+2)
Term-by-term division does NOT work; use formal partial fractions A/(u−1) + B/(u+2).

Common Misreadings

  • Setting up a full partial-fractions decomposition for (au+b)/u² — not wrong but slower than just dividing each term.
  • Trying partial fractions on x/(x+1) — the denominator is a single linear factor, so algebraic division is what is needed.
  • Treating an improper fraction as if num-degree < denom-degree and integrating directly, missing the need to extract the polynomial part first.

Pitfalls

  • Sign error on ∫−u^(−2) du: the answer is +u^(−1) = +1/u, because ∫u^(−2)du = −u^(−1).
  • Misapplying the power rule to u^(−1) (writing u^0/0); this term integrates to ln|u|, not a power.
  • Dropping absolute values: write ln|u|, not ln u, especially in indefinite integrals or when limits could include negatives.
  • Sign error when distributing a leading minus sign through the split, e.g. −∫x/(x+1)dx = −x + ln|x+1|, not −x − ln|x+1|.

A-level Scope Notes

Fully within Edexcel IAL P2/P3/P4 integration. Uses the standard integrals ∫1/x dx = ln|x| and ∫x^n dx = x^(n+1)/(n+1) from the formula booklet. Avoid invoking partial fractions of irreducible quadratics or repeated higher-power factors beyond IAL scope.
