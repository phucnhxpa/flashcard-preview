Trigger — When to use

Fires when dy/dx (or any rational expression) is a fraction with a denominator that is manifestly non-negative — typically a perfect square such as (2+cos x)², (1+cos x)², (e^x+1)², or (1+x²)² — and the task is to solve dy/dx = 0 or to 'show that' the stationary points satisfy a printed numerator equation. Cue phrases include 'show that at stationary points…', 'find the x-coordinates where dy/dx = 0', or any setup where the quotient rule has just been applied. The move also fires when proving monotonicity by sign of the numerator alone.

Motivation — Why this works

A fraction equals zero iff its numerator is zero and its denominator is non-zero. If the denominator can be shown strictly positive on the domain, it cannot contribute roots and may be discarded, collapsing the equation to N(x) = 0. Without this justification a 'show that' answer is logically incomplete — the examiner expects an explicit statement that the denominator is non-zero. The same logic underpins sign-of-derivative arguments: the sign of dy/dx is then determined entirely by the numerator.

Thought Process — Step-by-step reasoning

  1. Notice that after applying the quotient rule, dy/dx = N(x)/[v(x)]² with v(x) a simple expression like 1+cos x, 2+cos x, or e^x+1.
  2. Rule out solving the full equation N(x)/[v(x)]² = 0 directly — multiplying through is wasted work if the denominator is provably non-zero.
  3. Commit to a sign argument on v(x): bound it using a known range (|cos x| ≤ 1, e^x > 0, x² ≥ 0) to show v(x) > 0, hence v(x)² > 0.
  4. Bridge to operation: write 'since [bound], v(x)² > 0, so dy/dx = 0 ⟺ N(x) = 0', then set the numerator to zero.
  5. Tactically check whether v(x) actually vanishes anywhere in the stated domain; if so, flag those points as asymptotes/undefined points rather than stationary points.

Mechanic — Concrete operations

  1. Identify the denominator v(x)² and the elementary expression v(x).
  2. Quote the standard range bound: −1 ≤ cos x ≤ 1, e^x > 0, x² ≥ 0, etc.
  3. Deduce v(x) > 0 (or ≥ 0 with exceptions explicitly noted).
  4. Conclude [v(x)]² > 0 on the domain, so the fraction is zero iff the numerator is zero.
  5. Set N(x) = 0 and rearrange algebraically/trigonometrically to the printed target equation.

Worked Micro-Example

Let $y=\dfrac{\sin x}{2+\cos x}$. Quotient rule gives $\dfrac{dy}{dx}=\dfrac{2\cos x+1}{(2+\cos x)^2}$. Since $-1\le\cos x\le 1$, we have $2+\cos x\ge 1>0$, so $(2+\cos x)^2>0$ for all $x$. Hence $\dfrac{dy}{dx}=0\iff 2\cos x+1=0\iff \cos x=-\tfrac12$.

Variants — Same move, different surface

Denominator (a + cos x)² with a > 1
Strictly positive everywhere; no exceptional points to flag.
Denominator (1 + cos x)² or (1 − cos x)²
Vanishes at x = π or x = 0 respectively; note as vertical asymptote and exclude from stationary-point analysis.
Denominator (e^x + k)² with k ≥ 0
Use e^x > 0 to conclude denominator > 0 everywhere on ℝ.
Denominator (1 + x²)² from differentiating arctan-type or rational functions
Use 1 + x² ≥ 1 > 0; reduce to numerator = 0 immediately.
Sign-of-derivative / monotonicity proofs
Same denominator argument, but conclude that sign(dy/dx) = sign(N(x)) rather than solving = 0.
Implicit differentiation yielding dy/dx = N/D
If D is a sum of squares or otherwise sign-definite, same reduction applies.

Common Misreadings

  • Treating denominator = 0 as a stationary point instead of an asymptote/undefined point.
  • Multiplying both sides by the denominator and solving a polynomial equation, then forgetting to discard spurious roots where the denominator is also zero.
  • Assuming any squared expression is automatically > 0, missing that v(x)² = 0 when v(x) = 0.
  • Omitting the explicit bound statement and jumping straight to 'numerator = 0', losing the justification mark.

Pitfalls

  • Failing to state the range of cos x (or the relevant bound) explicitly — examiners want the inequality written down.
  • Writing v(x)² > 0 without first establishing v(x) ≠ 0 on the domain.
  • Overlooking domain endpoints where v(x) might equal zero (e.g. x = π in [0, 2π]).
  • Confusing 'denominator > 0' with 'function defined' — the function is undefined precisely where v(x) = 0.

A-level Scope Notes

Fully within Edexcel IAL P3/P4 differentiation (quotient rule) and C34 trigonometric/exponential calculus; the bounds used (|cos x|≤1, e^x>0, x²≥0) are standard A-level facts requiring no further justification.
