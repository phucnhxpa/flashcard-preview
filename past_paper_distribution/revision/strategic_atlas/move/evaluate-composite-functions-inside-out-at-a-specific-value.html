Algebra & Functions

Evaluate composite functions inside-out at a specific value

Family: Composite Evaluation Frequency: 4 questions

Trigger — When to use

The question asks for a single numerical value of a composite expression such as fg(a), gf(a), or ff(a) — i.e. f(g(a)), g(f(a)), or f(f(a)) — at one specific input. Visual cues: two function letters written next to each other followed by a numerical or constant argument (e.g. fg(5), ff(4), fg(e²)), with no request for a general formula. Equivalent phrasings include 'find f(g(a))', 'evaluate the composition at x = a', or 'work out fg(a)'. The move fires whenever you only need a number out, not a closed-form expression in x.

Motivation — Why this works

Composite functions are defined right-to-left: fg(x) = f(g(x)) means apply g first, then f. For a single numerical evaluation, computing the inner function at the given input produces a clean number, which is then fed into the outer function — replacing what could be messy algebraic substitution with two simple arithmetic steps. Building the general composite fg(x) and then substituting is unnecessary work when only one value is required, and it introduces extra opportunities for algebraic error. The underlying principle is the definition of function composition: (f∘g)(a) = f(g(a)).

Thought Process — The step-by-step reasoning

  1. Notice the notation fg(a) (or ff(a)): two function symbols juxtaposed with a single numerical argument — this is composition evaluated at a point, not multiplication and not a request for a general formula.
  2. Recognise the order: in fg(a) the right-hand function g acts first on a, then f acts on the result. For gf(a) or ff(a) the same rule applies — work from the innermost outwards.
  3. Rule out the alternative of forming the general composite fg(x) algebraically and then substituting x = a; for a one-off numerical answer this is strictly more work and more error-prone, especially if a is a surd, fraction, or expression like e².
  4. Commit to the two-stage numerical route: compute g(a) to get a single number b, then compute f(b).
  5. Tactical decision: simplify the inner output fully before substituting (e.g. ln(e²) = 2, |−1| = 1) so the outer evaluation starts from the cleanest possible number.
  6. If the function is given in multiple forms (factored vs expanded), pick the form that minimises fraction/surd arithmetic at the substitution stage — typically the factored form for fractional inputs.

Mechanic — Concrete operations

  1. Identify the inner function from the notation: in fg(a), the inner function is g.
  2. Compute the inner value: substitute x = a into g to get g(a) = b, simplifying any logs, exponentials, or absolute values immediately.
  3. Substitute b into the outer function: compute f(b).
  4. Simplify the final expression to a single number or exact form.
  5. State the result as fg(a) = f(b).

Worked Micro-Example

Given $f(x) = \dfrac{12}{x+1}$ and $g(x) = 5 - 2\ln x$, find $fg(e^2)$.\\ Inner: $g(e^2) = 5 - 2\ln(e^2) = 5 - 4 = 1$.\\ Outer: $f(1) = \dfrac{12}{1+1} = 6$.\\ Therefore $fg(e^2) = 6$.

Variants — Same move, different surface

Self-composition ff(a) with an absolute-value or piecewise function
Apply f twice in sequence; resolve the absolute value (or piece selection) at each stage based on the sign of the interior, do not square the output.
Reverse order gf(a)
Compute f(a) first, then substitute into g — same inside-out principle, mirrored.
Input is a fraction or surd and the function is given in both factored and expanded form
Substitute into the factored form so each bracket evaluates to a simple fraction; multiply rather than add fractions with different denominators.
Question asks for the general composite fg(x), not a value
Substitute the expression g(x) symbolically into f wherever x appears, then simplify algebraically.

Common Misreadings

  • Reading fg(a) as g(f(a)) — reversing the order of composition.
  • Reading fg(a) as f(a) × g(a) — treating juxtaposition as multiplication.
  • Reading ff(a) as [f(a)]² — confusing self-composition with squaring.

Pitfalls

  • Forgetting to fully simplify the inner result (e.g. leaving ln(e²) instead of 2) before substituting into the outer function.
  • Sign errors when the inner output is negative, especially inside an absolute value or a squared bracket.
  • Using the expanded polynomial form for a fractional input and miscombining fractions with different denominators, when the factored form would have been cleaner.
  • Substituting into the wrong function due to misreading the order — particularly easy when both letters look similar.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P2/P3) treatment of functions and composite functions; no formula booklet entry is required. Domain/range considerations (ensuring g(a) lies in the domain of f) are part of the syllabus but typically guaranteed in evaluation questions; avoid generalising to infinite iterates or fixed-point analysis, which are out of scope.
