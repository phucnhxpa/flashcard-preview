Algebra & Functions

Substitute given function value to obtain equation in parameter

Family: Parameter Determination Frequency: 1 questions

Trigger — When to use

Fires when a polynomial or rational function f(x) contains one or more unknown constants in its coefficients and the question supplies a specific value f(k) = c. Cue phrases include 'given that f(2) = 7', 'the curve passes through (k, c)', or 'when x = k, y = c'. The numerical input k is chosen so that substitution collapses x out, leaving a single equation (often linear or quadratic) in the unknown parameter. Recognise this whenever you have one unknown plus one numeric condition.

Motivation — Why this works

A function value condition is, by definition, an equation: f(k) − c = 0. Substituting the given x-value evaluates every x-power to a number, so the only symbol surviving is the parameter. The structure of the original polynomial in x typically lowers the degree in the parameter (a cubic in x with a appearing linearly/quadratically in coefficients yields at worst a quadratic in a), making the resulting equation routinely solvable by factorisation or the quadratic formula.

Thought Process — The step-by-step reasoning

  1. I notice f(x) carries an unknown constant in its coefficients and one numerical condition f(k) = c is supplied — exactly the right count to determine the parameter.
  2. I rule out comparing coefficients (no second polynomial to match) and rule out differentiating (no stationary/tangent condition mentioned).
  3. I commit to direct substitution because plugging x = k turns every x-power into a known number, isolating the parameter.
  4. Before substituting I scan how the parameter sits in each coefficient to predict the degree of the resulting equation in the parameter — this tells me whether to expect a linear or quadratic solve.
  5. I substitute carefully, collect like powers of the parameter, move the constant c across, and factorise or apply the quadratic formula.

Mechanic — Concrete operations

  1. Identify the parameter (e.g. a) and the supplied condition f(k) = c.
  2. Substitute x = k into f(x), evaluating each numeric power (k², k³, …) explicitly.
  3. Collect terms by powers of the parameter to obtain a polynomial in a.
  4. Set the expression equal to c and rearrange to standard form = 0.
  5. Factorise (or use the quadratic formula) to obtain all admissible values of the parameter.
  6. If the question constrains a (e.g. a > 0, a ∈ ℤ), discard inadmissible roots.

Worked Micro-Example

Given $f(x) = ax^3 + (6a+8)x^2 - a^2 x$ and $f(-1) = 32$:\ $f(-1) = -a + (6a+8) + a^2 = a^2 + 5a + 8.$\ Set $a^2 + 5a + 8 = 32 \Rightarrow a^2 + 5a - 24 = 0.$\ Factorise: $(a+8)(a-3)=0$, so $a = -8$ or $a = 3$.

Variants — Same move, different surface

Curve passes through a stated point (k, c).
Treat as f(k) = c; substitute x = k and y = c into the equation of the curve.
Two unknown parameters with two function-value conditions.
Substitute each condition to get two equations, then solve the simultaneous system in the parameters.
Remainder Theorem: f(x) divided by (x − k) leaves remainder r.
Use f(k) = r identically — same substitution mechanic.
Factor Theorem: (x − k) is a factor of f(x).
Set f(k) = 0 and solve for the parameter.
Rational function f(x) = p(x)/q(x) with f(k) = c given.
Substitute, then multiply through by q(k) to clear the denominator before solving.

Common Misreadings

  • Reading 'f(−1) = 32' as 'f(x) = 32 has root −1' rather than as a direct evaluation condition.
  • Mistaking the move for coefficient comparison when only one polynomial is in play.
  • Assuming the resulting equation must be linear in the parameter and missing a quadratic term.

Pitfalls

  • Sign errors with odd powers of negatives, e.g. (−1)³ = −1, not +1.
  • Forgetting to apply the power to the entire factor, e.g. writing −a² · (−1) as −a² instead of +a².
  • Subtracting c from the wrong side and producing a sign-flipped quadratic.
  • Failing to discard a root that violates a stated constraint on the parameter.

A-level Scope Notes

Standard P1/P2 technique used throughout Edexcel IAL Pure for determining unknown coefficients via given function values, points on curves, or the Remainder/Factor Theorem.
