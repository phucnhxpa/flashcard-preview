Coordinate Geometry

Apply m₁·m₂ = −1 and cancel shared parametric factors

Family: Perpendicularity Condition Frequency: 1 questions

Trigger — When to use

Fires when a problem on a parametrised curve (typically a parabola y² = 4ax with points P(ap², 2ap), Q(aq², 2aq)) states that two chords/lines are perpendicular, e.g. ON ⊥ PQ. Both gradients can be written as rational expressions in the parameters p and q. Equivalent phrasings include 'OR is at right angles to PQ', 'the chord is perpendicular to OM', or 'angle PNQ = 90°'.

Motivation — Why this works

Perpendicularity translates directly into m₁·m₂ = −1, which is a single scalar equation in p and q. On parametrised conics the chord gradient (e.g. 2/(p+q) for y² = 4ax) and gradients to fixed points share the factor (p+q), so multiplication produces a deliberate cancellation. This cancellation is structural — it is why parabola parameter problems are tractable — and it collapses what looks like a quartic relation into a clean symmetric equation in p and q.

Thought Process — The step-by-step reasoning

  1. Notice the geometric condition is perpendicularity, so the algebraic translation must be m₁·m₂ = −1, not equal gradients or a distance condition.
  2. Rule out vector/dot-product methods: gradients are already in hand from the chord-gradient step, so reusing them is faster.
  3. Commit because both gradients carry a common (p+q) factor, signalling that the product will simplify dramatically.
  4. Bridge: write each gradient explicitly, line them up, and multiply — keeping the (p+q) factors visible so the cancellation is obvious.
  5. Tactically, cancel (p+q) before cross-multiplying, then clear the negative signs together to avoid sign slips.

Mechanic — Concrete operations

  1. Write the gradient of the chord/line through the origin or fixed point in factored parametric form, exposing (p+q).
  2. Write the gradient of PQ as 2/(p+q) (for y² = 4ax).
  3. Form the product and set it equal to −1.
  4. Cancel the common (p+q) factor before any cross-multiplication.
  5. Cross-multiply, cancel matching signs, and rearrange into a symmetric polynomial in p and q.

Worked Micro-Example

On $y^2 = 4x$, $P(p^2,2p)$, $Q(q^2,2q)$, $N$ is the midpoint-style point with $m_{ON} = \dfrac{-2pq(p+q)}{2(p^2+pq+q^2)+4}$ and $m_{PQ} = \dfrac{2}{p+q}$. Setting $m_{ON}\cdot m_{PQ} = -1$ and cancelling $(p+q)$ gives $\dfrac{-4pq}{2(p^2+pq+q^2)+4} = -1$, so $4pq = 2(p^2+pq+q^2)+4$, i.e. $p^2+q^2-pq+2 = 0$.

Variants — Same move, different surface

Tangent ⊥ tangent on y² = 4ax at parameters p, q
Use tangent gradients 1/p and 1/q; condition (1/p)(1/q) = −1 gives pq = −1.
Normal ⊥ normal on the parabola
Use normal gradients −p and −q; perpendicularity gives pq = −1 again.
Chord PQ perpendicular to a fixed direction
Replace m_ON with the fixed slope; same cancellation logic if (p+q) appears.
Parallel chords / parallel to a fixed line
Replace m₁·m₂ = −1 with m₁ = m₂; cancellation pattern often still simplifies the equation.
Rectangular hyperbola xy = c² with parameters t₁, t₂
Chord gradient is −1/(t₁t₂); apply m₁·m₂ = −1 with shared t-factors cancelling analogously.

Common Misreadings

  • Treating ON ⊥ PQ as ON being a normal to the parabola rather than just perpendicular to chord PQ.
  • Assuming O is a generic labelled point rather than the origin, and so using the wrong gradient formula for ON.
  • Reading 'perpendicular' as 'tangent at right angles', invoking tangent formulae unnecessarily.

Pitfalls

  • Cross-multiplying before cancelling (p+q), leading to a degree-3 mess.
  • Sign errors when both sides are negative — losing or doubling a minus sign.
  • Dividing by (p+q) without noting the implicit assumption p + q ≠ 0 (the case p = −q corresponds to a vertical chord).
  • Forgetting to simplify the final equation by dividing through by the common factor 2.

A-level Scope Notes

Fully within Edexcel IAL Further Pure (FP1/FP2) coordinate geometry of the parabola and rectangular hyperbola, using the standard m₁m₂ = −1 perpendicularity criterion.
