Algebra & Functions

Regroup exponent as power-of-a-power to enable substitution

Family: Index Law Decomposition Frequency: 1 questions

Trigger — When to use

Fires when an expression contains a^{kx} (e.g. 3^{3x}, 2^{4x}, 5^{6x}) and the problem invites a substitution y = a^x — typically in 'express in terms of y' tasks, disguised quadratics in a^x, or exponential equations. The cue is that the exponent is an integer (or rational) multiple of x while a single a^x has been named. Also fires when you see a^{kx} sitting next to a^x and want a polynomial in y.

Motivation — Why this works

The index law (a^m)^n = a^{mn} read in reverse gives a^{mn} = (a^m)^n, so a^{kx} = (a^x)^k. This isolates the substitutable unit a^x = y, collapsing the term to y^k in one step. The same identity underlies turning exponential equations into polynomial equations, which is the whole point of the substitution.

Thought Process — The step-by-step reasoning

  1. I notice the exponent kx is a product, and y = a^x has been defined — so a^x must appear explicitly somewhere.
  2. I rule out splitting kx as a sum (a^{kx} \neq k\cdot a^x) and rule out pulling k out as a coefficient of the base.
  3. I commit to the reverse power-of-a-power reading because it is the only manipulation that produces an a^x factor I can name as y.
  4. Concretely I rewrite a^{kx} = a^{x\cdot k} = (a^x)^k, then replace a^x by y to get y^k.
  5. If k is a fraction (e.g. a^{x/2}), the same move gives (a^x)^{1/2} = y^{1/2}; if there is also a coefficient, e.g. 9\cdot 3^{2x}, leave the coefficient outside and only regroup the exponential part.

Mechanic — Concrete operations

  1. Identify the exponent as a product kx with k an integer or rational.
  2. Rewrite a^{kx} = (a^x)^k using the reverse of (a^m)^n = a^{mn}.
  3. Substitute y = a^x to obtain y^k.
  4. Reassemble the full expression (keep any separate coefficients or other terms) as a polynomial / rational expression in y.

Worked Micro-Example

Express $3^{3x} - 4\cdot 3^x$ in terms of $y$ where $y = 3^x$.\\ Regroup: $3^{3x} = 3^{x\cdot 3} = (3^x)^3 = y^3$.\\ Hence $3^{3x} - 4\cdot 3^x = y^3 - 4y$.

Variants — Same move, different surface

Different base, e.g. 2^{4x} with y = 2^x
Rewrite as (2^x)^4 = y^4; identical logic, only the base changes.
Fractional multiple, e.g. a^{x/2} with y = a^x
Use (a^x)^{1/2} = y^{1/2} = \sqrt{y}, valid since a^x > 0.
Reverse direction: y defined as a^{2x}, need a^x
Write a^x = (a^{2x})^{1/2} = y^{1/2} (or y = (a^x)^2 so a^x = \sqrt{y}).
Disguised quadratic, e.g. 3^{2x} - 4\cdot 3^x + 3 = 0
Regroup 3^{2x} = (3^x)^2 = y^2, giving y^2 - 4y + 3 = 0.
Coefficient times exponential, e.g. 9\cdot 3^{2x}
Leave 9 as a multiplicative constant: 9(3^x)^2 = 9y^2; do not absorb 9 into the base.

Common Misreadings

  • Reading 3^{3x} as 3\cdot 3^x = 3y (confusing the exponent product 3x with a coefficient).
  • Reading 3^{3x} as 3^3 \cdot 3^x = 27y (splitting the exponent additively as 3 + x instead of multiplicatively).
  • Treating a^{kx} as (ka)^x, i.e. attaching the multiplier to the base.

Pitfalls

  • Forgetting the reverse reading of (a^m)^n = a^{mn} and so failing to spot that a^{kx} can become (a^x)^k.
  • Mis-handling fractional k, e.g. writing a^{x/2} = (a^x)/2 instead of (a^x)^{1/2}.
  • Losing track of separate coefficients or constant terms when substituting (e.g. dropping the +3 in y^2 - 4y + 3).

A-level Scope Notes

Sits squarely in Edexcel IAL P1/P2 indices and exponentials, including disguised-quadratic equations in a^x; uses only the standard index laws (a^m)^n = a^{mn} and a^{m+n} = a^m a^n.
