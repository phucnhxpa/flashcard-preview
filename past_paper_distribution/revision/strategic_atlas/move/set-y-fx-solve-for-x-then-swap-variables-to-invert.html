Algebra & Functions

Set y = f(x), solve for x, then swap variables to invert

Family: Inverse Function Application Frequency: 1 questions

Trigger — When to use

The question explicitly asks for f⁻¹(x), or asks you to 'find the inverse function' of an algebraically defined f. Typical surfaces are linear, rational (Möbius), or simple composite functions where f is known to be one-to-one on its stated domain. Phrasings include 'find an expression for f⁻¹(x)', 'express the inverse in the form...', or 'state f⁻¹(x) and its domain'. The cue is purely symbolic — you need an exact algebraic formula, not just a graphical description.

Motivation — Why this works

An inverse function reverses the mapping: if f sends x to y, then f⁻¹ sends y back to x. Writing y = f(x) names the output, and solving for x in terms of y produces the rule that undoes f. Renaming y as x at the end converts that rule into the standard f⁻¹(x) notation. The method works because every algebraic step is reversible on the chosen domain, so the rearranged equation defines the genuine inverse mapping.

Thought Process — The step-by-step reasoning

  1. I notice the question asks for f⁻¹(x) of an explicit algebraic f, so I need a formula, not a sketch.
  2. I rule out 1/f(x) (that is the reciprocal, not the inverse) and rule out a graph-reflection argument because it won't yield an exact expression.
  3. I commit to the standard 'set y = f(x), solve for x, swap' procedure since f is one-to-one on its stated domain, guaranteeing an inverse exists.
  4. I prepare to write y = f(x) and treat y as a constant while rearranging to make x the subject.
  5. I plan to finish by replacing y with x and stating the domain of f⁻¹ as the range of f.

Mechanic — Concrete operations

  1. Write y = f(x).
  2. Treat y as a constant and rearrange the equation to isolate x (clear denominators, group x-terms, factor x, divide).
  3. Obtain x = g(y), an explicit formula in y.
  4. Replace every y with x to write f⁻¹(x) = g(x).
  5. State the domain of f⁻¹, which equals the range of f.

Worked Micro-Example

Let $f(x) = \dfrac{3x+2}{x-1}$. Set $y = \dfrac{3x+2}{x-1}$, so $y(x-1) = 3x+2$, giving $yx - 3x = y + 2$, hence $x(y-3) = y+2$ and $x = \dfrac{y+2}{y-3}$. Swap to get $f^{-1}(x) = \dfrac{x+2}{x-3}$, $x \neq 3$.

Variants — Same move, different surface

Linear function f(x) = ax + b
Set y = ax + b, solve x = (y-b)/a, swap to give f⁻¹(x) = (x-b)/a.
Square-root function f(x) = √(x-k) + c on x ≥ k
Set y = √(x-k) + c, square (y-c)² = x-k, so x = (y-c)² + k; swap, and restrict domain to y ≥ c (i.e. x ≥ c) to keep the inverse single-valued.
Exponential/log function f(x) = e^(ax) + b
Set y = e^(ax) + b, isolate e^(ax) = y - b, take ln, divide by a, then swap.
Piecewise function
Apply set-solve-swap on each branch separately and combine, taking care to match each branch's range as the new domain.
Composite inverse (fg)⁻¹
Use (fg)⁻¹ = g⁻¹f⁻¹ rather than inverting fg directly, when each component inverse is easier.

Common Misreadings

  • Confusing the inverse function f⁻¹(x) with the reciprocal 1/f(x).
  • Reading f⁻¹(x) as 'differentiate' or as a power, and trying to manipulate it as f(x)⁻¹.
  • Assuming an inverse exists without checking that f is one-to-one on the stated domain.

Pitfalls

  • Forgetting the final swap of x and y, leaving the answer in terms of y.
  • Failing to state the domain of f⁻¹ (which equals the range of f).
  • Dividing by an expression like (y-3) without noting the corresponding excluded value in the inverse's domain.
  • Squaring both sides without restricting signs, producing a spurious second branch.

A-level Scope Notes

This is the standard Edexcel IAL Pure (P2/P3) procedure for inverse functions; functions are assumed one-to-one on a stated domain, and the inverse's domain is read off as the range of f.
