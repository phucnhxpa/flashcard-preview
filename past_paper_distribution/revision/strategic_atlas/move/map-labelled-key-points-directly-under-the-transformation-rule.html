Algebra & Functions

Map labelled key points directly under the transformation rule

Family: Point Mapping Frequency: 1 questions

Trigger — When to use

The question gives a curve y = f(x) with specific labelled points (intercepts, turning points, endpoints) and asks for sketches of related curves like y = f(2x), y = f(x) + 3, y = -f(x), or y = af(x-b) + c. Marks are explicitly tied to coordinates of the transformed images of those labelled points. The original equation of f may be unknown or messy, but the key points are given. Phrases like 'sketch, showing the coordinates of the images of...' are the strongest cue.

Motivation — Why this works

Every graph transformation y = f(x) → g(x) corresponds to a fixed coordinate map (x, y) → (x', y') that sends each point on the original curve to a point on the new curve. Mapping the labelled points directly applies this rule once per point and is both faster and less error-prone than substituting into a transformed equation. Since the examiner awards marks for the coordinates themselves, anchoring the sketch on these mapped points guarantees the credit. The shape of the curve is preserved (up to the transformation), so connecting the mapped points with the correct shape completes the sketch.

Thought Process — The step-by-step reasoning

  1. Notice that the question labels specific points on y = f(x) and asks for the sketch to show their images — coordinates are the marked items.
  2. Rule out re-deriving f(x) or substituting into the new equation; f may be unknown or unnecessarily complex to manipulate.
  3. Commit to the point-mapping approach because each transformation has a clean coordinate rule that acts on each labelled point independently.
  4. Identify all key points to carry across: origin/intercepts, the named point P, and any turning points or endpoints visible on the original sketch.
  5. Apply the correct coordinate rule (inside-bracket changes affect x in the opposite sense; outside changes affect y directly), then plot and connect with the preserved shape.

Mechanic — Concrete operations

  1. List every key point on the original curve (intercepts, named points, turning points, endpoints).
  2. Identify the coordinate map for the required transformation, e.g. y = f(2x) gives (x, y) → (x/2, y); y = f(x) + 3 gives (x, y) → (x, y + 3).
  3. Apply the map to each key point and write down the new coordinates.
  4. Sketch the new curve through the mapped points, preserving the shape (and reflecting/stretching as the transformation dictates).
  5. Label every transformed key point with its coordinates on the diagram.

Worked Micro-Example

Given y = f(x) passes through O(0,0) and P(9,3). For y = f(2x), use (x, y) → (x/2, y): O → (0,0), P → (4.5, 3). For y = f(x) + 3, use (x, y) → (x, y+3): O → (0, 3), P → (9, 6). Sketch each curve through its mapped points with the same shape as the original.

Variants — Same move, different surface

y = f(-x) (reflection in y-axis)
Apply (x, y) → (-x, y) to each labelled point.
y = -f(x) (reflection in x-axis)
Apply (x, y) → (x, -y).
y = f(x - b) (horizontal translation)
Apply (x, y) → (x + b, y); note the sign flip from inside the bracket.
y = af(x) (vertical stretch)
Apply (x, y) → (x, ay).
y = f(x/k) (horizontal stretch by factor k)
Apply (x, y) → (kx, y).
y = af(x - b) + c (combined)
Apply (x, y) → (x + b, ay + c) to each labelled point in one step.
y = |f(x)| or y = f(|x|) (modulus transformations)
Map points with y < 0 to (x, -y) for |f(x)|; for f(|x|) reflect points with x ≥ 0 to (-x, y) and discard original x < 0 portion.

Common Misreadings

  • Transforming only the named point P but forgetting to also transform the origin, intercepts, or turning points, leaving the sketch poorly anchored.
  • Treating the transformation as acting on the equation only, then trying to recompute coordinates by substitution rather than by the direct map.
  • Confusing y = f(2x) with y = 2f(x) and applying the wrong rule to the wrong coordinate.

Pitfalls

  • Applying the x-coordinate transformation in the wrong direction (multiplying by 2 instead of dividing for f(2x), or shifting +b instead of -b for f(x - b)).
  • Mixing up which coordinate is affected by inside-bracket vs outside-bracket changes.
  • Forgetting to relabel the transformed turning point or asymptote on the new sketch.
  • Losing the shape of the curve (e.g. turning a stretched curve into a different family) when freehand-sketching between mapped points.

A-level Scope Notes

Standard Edexcel IAL Pure (P2/P3) graph-transformation material; covers single and combined translations, stretches, and reflections of y = f(x). Modulus transformations y = |f(x)| and y = f(|x|) appear within IAL scope and are handled by the same point-mapping approach.
