Algebra & Functions

Recognise perfect-square trinomial to extract the repeated root

Family: Completing The Square Frequency: 1 questions

Trigger — When to use

Fires when a quadratic is known (or has just been forced) to have discriminant zero — for example after solving Δ = 0 for a parameter and substituting back. Cues include: 'equal roots', 'repeated root', 'tangent to the x-axis', or a constant term that is exactly (b/2)² of the linear coefficient. Whenever the constant term equals the square of half the (signed) linear coefficient, the trinomial must factor as (x − b/2)².

Motivation — Why this works

If Δ = b² − 4ac = 0 then ax² + bx + c = a(x + b/(2a))², so the quadratic is literally a perfect square and the unique root is x = −b/(2a). Spotting the (x − k)² pattern is faster than the quadratic formula and acts as a self-check: if the pattern fails to appear cleanly, an earlier arithmetic step must be wrong. It also reinforces the link between completing the square and the discriminant.

Thought Process — The step-by-step reasoning

  1. Notice the problem has already enforced Δ = 0, so a repeated root is guaranteed and the quadratic must be a perfect square.
  2. Rule out using the quadratic formula or factor-pair search — both are slower and error-prone with fractional coefficients.
  3. Commit to pattern-matching x² + bx + c against (x + b/2)² by checking that c = (b/2)².
  4. Read off half the linear coefficient as the candidate k, then verify k² equals the constant term.
  5. Set the squared bracket to zero and state the single root x = −b/(2a); if the check fails, return to the discriminant step to hunt the slip.

Mechanic — Concrete operations

  1. Write the quadratic in monic form x² + bx + c (divide through by the leading coefficient if needed).
  2. Compute k = b/2 and check that k² = c.
  3. Rewrite the trinomial as (x + k)².
  4. Set (x + k)² = 0 and conclude x = −k as the repeated root.
  5. Sanity-check by expanding (x + k)² mentally to confirm the original coefficients.

Worked Micro-Example

Given $x^2 - 7x + \tfrac{49}{4} = 0$ with $\Delta = 0$: half the linear coefficient is $-\tfrac{7}{2}$, and $(-\tfrac{7}{2})^2 = \tfrac{49}{4}$ matches the constant term. So $x^2 - 7x + \tfrac{49}{4} = \left(x - \tfrac{7}{2}\right)^2 = 0$, giving the repeated root $x = \tfrac{7}{2}$.

Variants — Same move, different surface

Non-monic quadratic ax² + bx + c with Δ = 0
Either factor out a first to get a(x² + (b/a)x + c/a), or quote the repeated root directly as x = −b/(2a).
Completing the square to find a parabola's vertex (turning point)
Same (x − h)² pattern, but here you keep the constant remainder: y = (x − h)² + k gives vertex (h, k).
Circle equation x² + y² + Dx + Ey + F = 0
Apply the same half-the-coefficient-then-square trick to each variable to read off centre (−D/2, −E/2).
Discriminant condition giving tangency (line tangent to curve)
After imposing Δ = 0, the resulting quadratic in x is a perfect square whose root is the x-coordinate of the point of contact.

Common Misreadings

  • Treating the quadratic as a generic two-root case and applying the formula, missing that Δ = 0 forces a perfect square.
  • Mis-signing k: writing (x + 7/2)² when the linear term is −7x.
  • Forgetting to divide by the leading coefficient before pattern-matching when a ≠ 1.

Pitfalls

  • Arithmetic slips with fractional constants like 49/4 when verifying k² = c.
  • Stating two roots ±k instead of the single repeated root when solving (x − k)² = 0.
  • Using the quadratic formula anyway and risking sign or surd errors despite Δ = 0 making it unnecessary.
