Strategic Atlas / Differentiation / Exponential Never Zero
Differentiation

Discard the exponential factor since e^{f(x)} > 0 to reduce to an algebraic equation

Family: Exponential Never Zero Frequency: 3 questions

Trigger — When to use

Fires whenever you face a product equation of the form e^{f(x)} · g(x) = 0 (typically from setting a derivative equal to zero after factorising). Visual cue: one factor is an exponential e^{kx}, e^{f(x)}, or any always-positive expression (like 1 + x², sec²x), and another factor is algebraic (linear, quadratic, polynomial). Equivalent phrasings: 'solve f'(x) = 0 where f'(x) has been factorised with an exponential outside', or 'find stationary points of a product/exponential function'. The move is also triggered when separating variables in simple ODEs leaves an e^{kx} factor that needs to be eliminated.

Motivation — Why this works

The exponential function e^{f(x)} is strictly positive for every real input — its range is (0, ∞) and it never touches zero. By the zero-product property, only factors that CAN equal zero contribute solutions, so the exponential factor is discarded entirely. This converts a transcendental equation into a purely algebraic one (linear, quadratic, etc.), which is trivially solvable. Examiners explicitly require the justification 'e^{f(x)} > 0 for all real x' to award the reasoning mark — it is not enough to silently drop the factor.

Thought Process — The step-by-step reasoning

  1. Notice the equation has the structure (exponential factor) · (algebraic factor) = 0, e.g. e^{3x}(6x+17) = 0 or (x−2)·e^{3x}·(3x−4) = 0.
  2. Rule out alternative tactics: don't take logs of both sides (you'd get ln(0), undefined); don't divide blindly by e^{3x} without justifying it's nonzero; don't expand the product (that destroys the factorisation).
  3. Commit to the zero-product property: at least one factor must be zero. Examine each factor in turn.
  4. Recall the key identity: e^{anything real} > 0 always. Therefore e^{f(x)} = 0 has NO real solutions and can be eliminated from consideration.
  5. Bridge to concrete work: write the explicit justification 'e^{f(x)} > 0 for all real x, so e^{f(x)} ≠ 0', then set the remaining algebraic factor(s) equal to zero.
  6. Solve the resulting algebraic equation by standard methods (linear rearrangement, factorising a quadratic, etc.).

Mechanic — Concrete operations

  1. Start from the factorised equation, e.g. e^{f(x)} · g(x) = 0.
  2. State explicitly: 'Since e^{f(x)} > 0 for all real x, e^{f(x)} ≠ 0.'
  3. Deduce: g(x) = 0.
  4. Solve g(x) = 0 using algebraic methods (linear: rearrange; quadratic: factorise/formula).
  5. Report all real solutions; no extraneous roots arise from the exponential.

Worked Micro-Example

Solve $e^{3x}(6x+17)=0$. Since $e^{3x}>0$ for all real $x$, we have $e^{3x}\neq 0$, so $6x+17=0$, giving $x=-\tfrac{17}{6}$.

Variants — Same move, different surface

Three-factor product, e.g. (x−2)·e^{3x}·(3x−4) = 0
Discard e^{3x}, then solve the remaining product (x−2)(3x−4) = 0 to get x = 2 or x = 4/3.
Negative or composite exponent, e.g. e^{−x}(3x²−1) = 0 or e^{x²}(2x+1) = 0
Same logic — e^{−x} > 0 and e^{x²} > 0 for all real x, so discard and solve the algebraic factor.
Always-positive non-exponential factor, e.g. (1 + x²)(2x − 5) = 0 or sec²(x)·g(x) = 0
Identical reasoning: 1 + x² > 0 (and sec²x > 0 where defined), so only the other factor contributes roots.
Separable ODE with e^{kx} factor
Since e^{kx} ≠ 0 you may legitimately divide both sides by it without losing solutions.

Common Misreadings

  • Trying to 'solve' e^{f(x)} = 0 by taking logs, producing f(x) = ln(0) which is undefined.
  • Silently dividing both sides by e^{f(x)} without stating it is nonzero — gets the right answer but loses the justification mark.
  • Confusing 'e^{f(x)} > 0' with 'e^{f(x)} ≥ 1' and missing that values can be arbitrarily close to 0 but never equal to 0.

Pitfalls

  • Forgetting to write the explicit justification 'e^{f(x)} > 0 for all real x' — examiners deduct the reasoning mark even when the final answer is correct.
  • Mistakenly assigning a 'solution' to the exponential factor (e.g. claiming e^{3x} = 0 gives x = −∞ or some limiting value).
  • When the algebraic factor is quadratic, forgetting to find BOTH roots after eliminating the exponential.
  • Misapplying the move when the factor is not strictly positive (e.g. (x² − 1) is NOT always positive — it can equal zero, so cannot be discarded).

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P2/P3/P4) — relies on the standard property that the exponential function has range (0, ∞), introduced when e^x is defined. No formula booklet entry is required. Avoid generalising to complex exponentials (e^{iθ} can equal values on the unit circle but never zero either, but this is out of scope).
