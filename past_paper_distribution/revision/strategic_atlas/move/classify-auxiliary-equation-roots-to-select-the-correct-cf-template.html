Differential Equations

Classify auxiliary equation roots to select the correct CF template

Family: Root Classification Frequency: 4 questions

Trigger — When to use

This move fires immediately after solving the auxiliary (characteristic) quadratic am² + bm + c = 0 of a linear second-order homogeneous ODE with constant coefficients. The cue is that you have the roots (or the discriminant) in hand and now need to write down the complementary function. Equivalent phrasings: 'translate roots into CF', 'choose CF form from discriminant sign', 'branch on root type'. The decision is forced because each of the three cases (real distinct, repeated, complex) demands a structurally different CF.

Motivation — Why this works

The three CF templates arise from the requirement of two linearly independent real-valued solutions. Two distinct real roots give two independent exponentials directly. A repeated root yields only one exponential, so reduction of order produces xe^{mx} as the second independent solution — hence the (A + Bx)e^{mx} form. Complex conjugate roots p ± qi would give complex exponentials, but Euler's formula e^{(p+qi)x} = e^{px}(cos qx + i sin qx) lets us recombine them into the real form e^{px}(A cos qx + B sin qx). Choosing the wrong template means the CF fails to span the homogeneous solution space or is not real-valued.

Thought Process — The step-by-step reasoning

  1. Notice that the auxiliary quadratic has been solved; the next task is to write the CF, and the form depends entirely on the nature of the roots.
  2. Rule out trying to write a single universal template — the three cases (Δ > 0, Δ = 0, Δ < 0) are genuinely different and each gives a different family.
  3. Compute or inspect the discriminant Δ = b² − 4ac (or just look at the roots already obtained) to decide which branch applies.
  4. Commit to the matching template: two exponentials for distinct real, polynomial-times-exponential for repeated, exponential-times-trig for complex.
  5. For the complex case, carefully extract p (real part) and q (positive imaginary part) so the trig arguments are qx, not px or (p+q)x.
  6. Write the CF with exactly two arbitrary constants A and B — these will later be fixed by boundary or initial conditions.

Mechanic — Concrete operations

  1. Compute the discriminant Δ = b² − 4ac of am² + bm + c = 0.
  2. Case Δ > 0: roots m₁ ≠ m₂ real and distinct → CF = Ae^{m₁x} + Be^{m₂x}.
  3. Case Δ = 0: repeated root m = −b/(2a) → CF = (A + Bx)e^{mx}.
  4. Case Δ < 0: complex roots p ± qi with p = −b/(2a) and q = √|Δ|/(2a) → CF = e^{px}(A cos qx + B sin qx).
  5. Check the CF contains exactly two arbitrary constants and is real-valued.

Worked Micro-Example

For y'' − 4y' + 13y = 0, the auxiliary equation m² − 4m + 13 = 0 has Δ = 16 − 52 = −36 < 0. Roots: m = (4 ± 6i)/2 = 2 ± 3i, so p = 2, q = 3. Therefore CF: y = e^{2x}(A\cos 3x + B\sin 3x).

Variants — Same move, different surface

One root is m = 0 (e.g. auxiliary m² − 3m = 0)
The corresponding term is e^{0·x} = 1, giving a constant in the CF, e.g. CF = A + Be^{3x}.
No first-derivative term (b = 0) with c < 0, e.g. y'' − k²y = 0
Roots are ±k real → CF = Ae^{kx} + Be^{-kx}, which may equivalently be written as C\cosh kx + D\sinh kx.
No first-derivative term (b = 0) with c > 0, e.g. y'' + q²y = 0
Purely imaginary roots ±qi (so p = 0) → CF = A\cos qx + B\sin qx with no exponential envelope.

Common Misreadings

  • Treating Δ = 0 as if it were Δ > 0 with a 'double-counted' distinct root, leading to Ae^{mx} + Be^{mx} which collapses to one constant.
  • Reading the imaginary part of a complex root as the whole root (e.g. taking q = 3i instead of q = 3).

Pitfalls

  • For the repeated-root case, omitting the factor x in the second term — the CF then has only one effective arbitrary constant.
  • For complex roots, sign errors on p (the real part) or putting px rather than qx inside cos/sin.
  • Arithmetic slips computing Δ — especially mishandling the sign of c or the factor of 4 — leading to the wrong branch entirely.

A-level Scope Notes

Stays within Edexcel IAL Further Pure (FP2/FP3) treatment of second-order linear ODEs with constant coefficients. The three CF templates are standard and appear in the formula booklet section on differential equations. Do not generalise to higher-order ODEs (triple roots requiring x²e^{mx}) or variable-coefficient equations — those are out of scope.
