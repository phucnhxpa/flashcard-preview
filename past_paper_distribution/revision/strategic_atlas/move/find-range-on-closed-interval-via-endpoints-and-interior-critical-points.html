Differentiation

Find range on closed interval via endpoints and interior critical points

Family: Extreme Value Theorem Frequency: 1 questions

Trigger — When to use

A question asks for the range, maximum, or minimum of a continuous function on a closed interval $[a,b]$. Phrases like 'find the range of $f$ on $[a,b]$', 'state the greatest/least value', or 'find the set of values taken by $f$' are typical cues. Often follows a part where stationary points have already been located. May also appear when a transformed function $g = pf + q$ is requested on the same interval.

Motivation — Why this works

The Extreme Value Theorem guarantees that a continuous function on a closed bounded interval attains its absolute maximum and minimum, and these can only occur at endpoints or at interior stationary points. So checking this finite list of candidates is exhaustive. By the Intermediate Value Theorem, every value between the min and max is also attained, so the range is the closed interval $[\min, \max]$. A monotonic linear transformation $g = pf + q$ (with $p>0$) maps this range directly to $[p\min+q, p\max+q]$.

Thought Process — The step-by-step reasoning

  1. Notice the interval is closed and bounded and the function is continuous on it, so EVT applies and the range is a closed interval.
  2. Rule out critical points lying outside the interval — only stationary points strictly inside $[a,b]$ are candidates.
  3. Commit to the finite candidate list: the two endpoints plus the surviving interior stationary points.
  4. Evaluate $f$ at each candidate and read off the smallest and largest values to form $[\min, \max]$.
  5. If a transformed function $g = pf+q$ is asked, check the sign of $p$: positive preserves the order of min/max, negative swaps them.

Mechanic — Concrete operations

  1. Confirm $f$ is continuous on $[a,b]$ (check domains of any surds, logs, fractions).
  2. List interior stationary points from $f'(x)=0$ and discard any outside $[a,b]$.
  3. Evaluate $f$ at $a$, at $b$, and at each surviving interior stationary point.
  4. Identify $\min$ and $\max$ of these values; the range of $f$ is $[\min,\max]$.
  5. If $g = pf + q$, compute $g_{\min}$ and $g_{\max}$; if $p<0$, swap which $f$-value gives min vs max.
  6. State the range using interval or inequality notation.

Worked Micro-Example

Let $f(x)=5(x^2-2)\sqrt{x+9/4}$ on $[-9/4,0]$ with stationary point at $x=-2$ inside. Then $f(-9/4)=0$, $f(-2)=5(2)(1)=10$, $f(0)=5(-2)(3)=-30$. So $f$ has range $[-30,10]$, and $g=2f+4$ has range $[2(-30)+4,\,2(10)+4]=[-56,24]$.

Variants — Same move, different surface

Open or half-open interval $(a,b)$ or $(a,b]$
EVT does not apply at the open end; replace endpoint value with $\lim_{x\to a^+}f(x)$ and the range may be open or unbounded on that side.
Transformation $g = pf+q$ with $p<0$
$g$ is decreasing in $f$, so $g_{\max}=p\cdot f_{\min}+q$ and $g_{\min}=p\cdot f_{\max}+q$.
Non-linear transformation, e.g. $g=f^2$ or $g=\mathrm{e}^{f}$
Monotonicity may fail; apply the move to $f$ first to get $[\min,\max]$, then study $g$ as a function on that image interval.
Endpoint coincides with a stationary point or boundary of the domain
Still evaluate there; just don't double count, and check one-sided behaviour if the derivative is undefined.
Modulus or piecewise function
Add interior points where the derivative is undefined (corners) to the candidate list.

Common Misreadings

  • Only checking the endpoints and forgetting interior stationary points, missing a turning value.
  • Including stationary points that lie outside the given interval.
  • Stating the range as a single inequality like $f(x)\ge -56$ without an upper bound, ignoring that the interval is closed.
  • Confusing range (set of $y$-values) with the domain interval that was given.

Pitfalls

  • Arithmetic slips when an endpoint makes a surd factor vanish — failing to recognise the value is exactly $0$.
  • Forgetting to apply the linear transformation to both ends, or applying it only to the maximum.
  • Writing the range with the wrong inclusion (open vs closed) when EVT actually guarantees both extremes are attained.
  • Sign errors when $p<0$ in $g=pf+q$ leading to swapped or invalid bounds.

A-level Scope Notes

Fully within Edexcel IAL P3/P4 and FP coverage: continuity on closed intervals, stationary points via $f'(x)=0$, and direct evaluation; EVT/IVT are used informally without formal proof.
