Coordinate Geometry

Extract a/e ratio from a given directrix equation

Family: Standard Form Reading Frequency: 1 questions

Trigger — When to use

A conic (hyperbola or ellipse) is centred at the origin with axis along a coordinate axis, and a directrix such as $x = \pm 4/3$ (or $y = \pm k$) is stated. The unknowns typically include $a$ and $e$ (and sometimes the foci or $b$). Phrases like 'directrix is...', 'corresponding directrix', or being given a numerical value of $a/e$ all fire this move.

Motivation — Why this works

For the standard hyperbola $x^2/a^2 - y^2/b^2 = 1$ (and similarly for the ellipse), the directrices are $x = \pm a/e$. Matching the supplied numerical directrix to this template instantly converts a geometric datum into an algebraic equation in $a$ and $e$. This is the only way to use the directrix information, and it is invariably the first equation in a system that will be closed using $b^2 = a^2(e^2 - 1)$ for a hyperbola or $b^2 = a^2(1 - e^2)$ for an ellipse.

Thought Process — The step-by-step reasoning

  1. I notice the problem gives a directrix value, and the conic is in standard position, so the formula $x = \pm a/e$ applies directly.
  2. I rule out trying to use the focus-directrix definition pointwise — that's longer; the standard-form formula is exactly what's needed.
  3. I commit to equating the given directrix value to $a/e$ because this is one of three canonical relations (directrix, focus, $b^2$) and the only one that uses the datum I've been handed.
  4. I bridge to the first operation by writing $a/e = 4/3$ and deciding which variable to make the subject so it slots cleanly into the next relation (usually $b^2 = a^2(e^2-1)$).
  5. Tactically, I keep the equation symbolic ($e = 3a/4$ or $a = 4e/3$) until I see which substitution minimises algebra in the next step.

Mechanic — Concrete operations

  1. Recall the standard directrix formula: $x = \pm a/e$ for a hyperbola/ellipse with transverse (or major) axis along the $x$-axis.
  2. Equate the given numerical directrix to $a/e$: e.g. $a/e = 4/3$.
  3. Rearrange to express one variable in terms of the other, e.g. $e = 3a/4$ or $a = 4e/3$.
  4. Hold this relation aside, ready to substitute into $b^2 = a^2(e^2 - 1)$ or the focus relation $(\pm ae, 0)$.

Worked Micro-Example

Hyperbola: $x^2/a^2 - y^2/b^2 = 1$ with directrix $x = 4/3$ and $b^2 = 9$. Set $a/e = 4/3 \Rightarrow a = 4e/3$. Substitute into $b^2 = a^2(e^2-1)$: $9 = (16e^2/9)(e^2-1)$, giving $e^2 = 9/4$, so $e = 3/2$ and $a = 2$. Foci then at $(\pm ae, 0) = (\pm 3, 0)$.

Variants — Same move, different surface

Ellipse $x^2/a^2 + y^2/b^2 = 1$ with $a > b$
Same formula $x = \pm a/e$ but $e < 1$, and pair with $b^2 = a^2(1 - e^2)$ instead of $a^2(e^2-1)$.
Hyperbola/ellipse with axis along the $y$-axis
Directrices become $y = \pm a/e$ where $a$ is the semi-axis along $y$; foci at $(0, \pm ae)$.
Problem gives the focus instead of the directrix
Use $ae = $ (given focus coordinate) rather than $a/e$; the rest of the workflow is identical.
Problem gives both a directrix and a focus
Two equations $a/e = d$ and $ae = f$ solve directly: $a^2 = df$, $e^2 = f/d$.

Common Misreadings

  • Confusing the directrix formula $x = a/e$ with the focus formula $x = ae$, producing the wrong equation.
  • Forgetting the $\pm$ and assuming only one directrix exists.
  • Applying the hyperbola relation $b^2 = a^2(e^2-1)$ to an ellipse (or vice versa) after extracting the ratio.
  • Misreading the axis orientation and using $x = \pm a/e$ when the conic actually has its transverse axis along $y$.

Pitfalls

  • Inverting the rearrangement: writing $e = 4/(3a)$ instead of $e = 3a/4$ from $a/e = 4/3$.
  • Cancelling $a$ or $e$ prematurely before substitution, losing a needed factor.
  • Plugging $a/e = 4/3$ as $a = 4, e = 3$ — treating a ratio as individual values.

A-level Scope Notes

Fully within Edexcel IAL Further Pure (Further Maths) coordinate geometry of the ellipse and hyperbola; uses only the standard directrix formula $x = \pm a/e$ and the standard $b^2$ relations.
