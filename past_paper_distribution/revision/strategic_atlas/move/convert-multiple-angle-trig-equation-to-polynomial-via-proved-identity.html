Trigger — When to use

Fires when a question says 'hence solve' or 'hence find the roots of' an equation like cos(nθ) = c or sin(nθ) = c, immediately after you have proved an identity expressing cos(nθ) (or sin(nθ)) as a polynomial in cosθ (or sinθ). The keyword 'hence' is the strongest cue — it mandates using the just-proved identity. Also fires when the equation contains a multiple angle whose expansion you know via De Moivre or addition formulae.

Motivation — Why this works

A multiple-angle equation like cos(4θ) = 0 has solutions scattered across [0, 2π] that resist direct enumeration. Substituting the polynomial identity (e.g. cos(4θ) = 8cos⁴θ − 8cos²θ + 1) converts the transcendental equation into an algebraic polynomial in c = cosθ, which factorises or yields to the quadratic formula. The bridge works because the identity is an exact equality, so every root of the polynomial in c corresponds (via cosθ = c) to a family of θ-solutions, and conversely every θ-solution gives a c-root.

Thought Process — Step-by-step reasoning

  1. I notice the equation involves cos(nθ) or sin(nθ) and the question begins with 'hence' — so the previously proved identity is the intended tool.
  2. I rule out direct methods like θ = (1/n)arccos(c), because those typically miss roots or are not what 'hence' asks for.
  3. I commit to the substitution because the identity gives an exact polynomial in cosθ, and polynomial roots are easier to enumerate than scattered angle solutions.
  4. I let c = cosθ (or c = cos²θ if all powers are even) so the equation becomes P(c) = 0, ready to factor.
  5. Once I have the c-values, I invert: discard any with |c| > 1, then for each valid c find every θ in the required range using the symmetry cosθ = cos(2π − θ).

Mechanic — Concrete operations

  1. Replace cos(nθ) (or sin(nθ)) in the equation with the polynomial identity just proved.
  2. Let c = cosθ (or c = cos²θ if the polynomial is even in cosθ) to get a polynomial P(c) = 0.
  3. Solve P(c) = 0 by factoring, quadratic formula, or treating as a quadratic in c².
  4. Reject any root with |c| > 1, since no real θ satisfies cosθ = c there.
  5. For each valid c, list all θ in the specified domain using cosθ = c ⇒ θ = ±arccos(c) + 2kπ.

Worked Micro-Example

Given the proved identity $\cos 4\theta = 8\cos^4\theta - 8\cos^2\theta + 1$, hence solve $\cos 4\theta = 0$ for $\theta \in [0, \pi]$. Let $c = \cos\theta$: $8c^4 - 8c^2 + 1 = 0$. Treating as quadratic in $c^2$: $c^2 = \tfrac{8 \pm \sqrt{32}}{16} = \tfrac{2 \pm \sqrt{2}}{4}$. Both lie in $[0,1]$, so $c = \pm\sqrt{(2\pm\sqrt2)/4}$, giving $\theta = \tfrac{\pi}{8}, \tfrac{3\pi}{8}, \tfrac{5\pi}{8}, \tfrac{7\pi}{8}$.

Variants — Same move, different surface

sin(nθ) = c after proving sin(nθ) as a polynomial in sinθ (odd n)
Let s = sinθ; invert via θ = arcsin(s) and θ = π − arcsin(s) for the second family.
tan(nθ) = c with a tan-multiple-angle identity (rational in tanθ)
Clear denominators first, let t = tanθ, solve the resulting polynomial; θ = arctan(t) + kπ.
Mixed equation like cos(3θ) + cosθ = 0
Substitute cos(3θ) = 4cos³θ − 3cosθ, factor out cosθ, then solve the residual polynomial in cosθ.
De Moivre derivation followed by 'hence solve'
First extract the real or imaginary part of (cosθ + i sinθ)ⁿ to obtain the identity, then apply this move.

Common Misreadings

  • Treating 'hence' as optional and trying to solve cos(nθ) = c directly with θ = (1/n)arccos(c), missing the algebraic route the question demands.
  • Confusing this move with a t = tan(θ/2) Weierstrass substitution, which is a different bridge.
  • Reading the polynomial as defining new variables rather than as an identity in cosθ.

Pitfalls

  • Losing solutions by only taking the principal value of arccos — must include θ and 2π − θ (and add 2kπ as needed).
  • Forgetting the domain restriction stated in the question and listing θ outside it.
  • Failing to discard roots with |c| > 1 when the polynomial admits them.
  • Sign errors when taking square roots of c² to recover c, dropping the negative branch.

A-level Scope Notes

Standard Edexcel IAL Further Pure content: multiple-angle identities derived from De Moivre's theorem (FP2) followed by 'hence solve' tasks producing quartic or cubic equations in cosθ or sinθ.
