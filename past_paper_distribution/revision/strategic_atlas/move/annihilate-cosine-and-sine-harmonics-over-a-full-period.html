Strategic Atlas / Integration / Orthogonality Shortcut
Integration

Annihilate cosine and sine harmonics over a full period

Family: Orthogonality Shortcut Frequency: 1 questions

Trigger — When to use

Fires when integrating over [0, 2π] (or any interval of length 2π) and the integrand has been reduced to a constant plus terms of the form cos(nθ) or sin(nθ) with nonzero integer n. Typical cues: a polar-area integral ½∫r² dθ over a full revolution, or a Fourier-style linearisation of cos^k θ / sin^k θ via double-angle identities. The move says: ignore every oscillatory harmonic and keep only the constant term.

Motivation — Why this works

For any nonzero integer n, ∫₀^{2π} cos(nθ) dθ = [sin(nθ)/n]₀^{2π} = 0 and ∫₀^{2π} sin(nθ) dθ = [−cos(nθ)/n]₀^{2π} = 0, because the trig functions complete exactly n full cycles and return to their starting values. So every harmonic term integrates to zero and only the DC (constant) coefficient contributes. This is the elementary case of Fourier orthogonality and is the entire payoff of choosing a full-period interval.

Thought Process — The step-by-step reasoning

  1. Notice the integration interval is exactly [0, 2π] (a full period) and the integrand is a sum of a constant and cos(nθ)/sin(nθ) terms with integer n.
  2. Rule out evaluating each antiderivative laboriously — the harmonic pieces are guaranteed to vanish.
  3. Commit to the shortcut: only the constant term contributes; write it down as (constant) × 2π.
  4. Bridge to operation: identify the constant (DC) coefficient cleanly from the linearised integrand.
  5. Tactical decision: double-check that every trig argument is an integer multiple of θ, otherwise the orthogonality argument fails.

Mechanic — Concrete operations

  1. Linearise the integrand into the form a₀ + Σ aₙ cos(nθ) + Σ bₙ sin(nθ).
  2. Verify each n is a nonzero integer and the interval is a full period [0, 2π].
  3. Discard all cos(nθ) and sin(nθ) terms — each integrates to 0 over [0, 2π].
  4. Compute ∫₀^{2π} a₀ dθ = 2π a₀.
  5. Apply any outer factor (e.g. ½ for polar area) to obtain the final value.

Worked Micro-Example

Find $A=\tfrac12\int_0^{2\pi}\!\left(\tfrac{39}{2}-24\cos 6\theta+\tfrac{9}{2}\cos 12\theta\right)d\theta$. The cos$6\theta$ and cos$12\theta$ terms each integrate to $0$ over $[0,2\pi]$. So $A=\tfrac12\int_0^{2\pi}\tfrac{39}{2}\,d\theta=\tfrac12\cdot 39\pi=\tfrac{39\pi}{2}.$

Variants — Same move, different surface

Polar area ½∫₀^{2π} r²(θ) dθ where r² linearises to constant + cos(nθ) terms
Read off the constant term and multiply by π (since ½ × 2π = π).
Integrand contains sin(nθ) terms instead of cos
Same conclusion: ∫₀^{2π} sin(nθ) dθ = 0 for nonzero integer n.
Integrand of the form cos^k θ or sin^k θ with k a positive integer
First apply double-angle / power-reduction identities to linearise, then apply the shortcut.
Interval is [−π, π] (still length 2π)
Same shortcut works — any interval of length equal to a full period gives 0 for the harmonics.
Interval shorter than the period, e.g. [0, π]
Shortcut only works if n is even (so nθ still covers an integer number of full cycles); otherwise evaluate explicitly.

Common Misreadings

  • Treating cos(nθ) as having average value ½ (confusing with cos²) instead of 0.
  • Believing the shortcut needs the interval to be exactly [0, 2π]; in fact any full-period interval works.
  • Thinking a constant term inside a cosine, like cos(nθ + φ), breaks the result — it does not, provided n is a nonzero integer.

Pitfalls

  • Forgetting the outer ½ factor in polar-area problems and reporting 39π instead of 39π/2.
  • Applying the shortcut on an interval whose length is not an integer multiple of 2π/n — then the harmonic integral is not zero.
  • Slip in the antiderivative coefficient: ∫cos(nθ) dθ = sin(nθ)/n, not sin(nθ).
  • Misidentifying the constant term after linearising (e.g. dropping a ½ from a cos²θ = ½ + ½cos2θ expansion).

A-level Scope Notes

Fully within Edexcel IAL Further Pure (polar coordinates, areas of polar curves) and Core Pure integration using double-angle identities; no Fourier-series machinery is invoked beyond the elementary fact ∫₀^{2π} cos(nθ) dθ = ∫₀^{2π} sin(nθ) dθ = 0 for nonzero integer n.
