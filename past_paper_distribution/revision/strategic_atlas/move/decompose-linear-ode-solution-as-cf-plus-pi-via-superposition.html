Differential Equations

Decompose linear ODE solution as CF plus PI via superposition

Family: Superposition Principle Frequency: 3 questions

Trigger — When to use

The ODE is linear (no products like y·y', no y² etc.) with a non-zero right-hand side f(x) — i.e. of the form ay'' + by' + cy = f(x). Visual cues: constant (or variable) coefficients on the left, and a forcing term such as cos x, e^{kx}, a polynomial, or sin/cos on the right. Equivalently phrased: 'find the general solution of a non-homogeneous linear ODE'. Whenever you see a second-order linear ODE that is not equal to zero, this decomposition is the standard opening move.

Motivation — Why this works

Because the differential operator L is linear, if L[y_CF] = 0 and L[y_PI] = f(x), then L[y_CF + y_PI] = 0 + f(x) = f(x). The complementary function (CF) supplies the two arbitrary constants required for the general solution of a second-order ODE (one for first-order), and the particular integral (PI) shifts that family to satisfy the forcing term. Together they form a complete general solution: every solution of the non-homogeneous equation can be written this way. This splits one harder problem into two routine subproblems: a homogeneous auxiliary-equation problem and an undetermined-coefficients problem.

Thought Process — The step-by-step reasoning

  1. Notice the ODE is linear (only y, y', y'' appear to the first power, no products) and the RHS is non-zero — this is a non-homogeneous linear ODE.
  2. Rule out direct attacks: separation of variables fails (it's second order with mixed terms), and integrating factors are for first-order linear; tackling the full non-homogeneous equation in one shot is unnecessarily hard.
  3. Commit to the superposition framework: y_general = y_CF + y_PI, justified by the linearity of L.
  4. Bridge to action: set up two subproblems — (i) solve the associated homogeneous equation L[y] = 0 via the auxiliary equation to get y_CF (with arbitrary constants A, B); (ii) guess a trial form for y_PI matching the structure of f(x).
  5. Tactical check before guessing the PI: does any term in f(x) already appear in y_CF? If yes, multiply the trial PI by x (or x²) to avoid resonance/duplication; if no, proceed with the standard trial form.
  6. After finding both, write y = y_CF + y_PI, ensuring exactly two arbitrary constants survive (matching the order of the ODE) and the PI has fully determined numerical coefficients.

Mechanic — Concrete operations

  1. Write the homogeneous equation L[y] = 0 and form the auxiliary equation am² + bm + c = 0; solve for roots and write y_CF (e.g. Ae^{m₁x} + Be^{m₂x}).
  2. Inspect f(x) and choose a trial PI of matching form: f = cos x → try p cos x + q sin x; f = e^{kx} → try pe^{kx}; f = polynomial of degree n → try a general polynomial of degree n.
  3. Check for duplication with y_CF; if a trial term coincides with a CF term, multiply the trial by x.
  4. Substitute the trial into L[y] = f(x), match coefficients, and solve for the unknown constants in the trial — yielding y_PI with definite numerical coefficients.
  5. Write the general solution: y = y_CF + y_PI, retaining only A and B as arbitrary constants.

Worked Micro-Example

For y'' + 5y' + 6y = \cos x: auxiliary m² + 5m + 6 = 0 gives m = -2, -3, so y_{CF} = Ae^{-2x} + Be^{-3x}. Trial y_{PI} = p\cos x + q\sin x; substituting and matching gives p = 1/10, q = 1/10. Hence y = Ae^{-2x} + Be^{-3x} + \tfrac{1}{10}\cos x + \tfrac{1}{10}\sin x.

Variants — Same move, different surface

First-order linear ODE y' + P(x)y = Q(x) solved by integrating factor
Same CF + PI structure emerges: the homogeneous part contributes Ce^{-∫P dx} (one arbitrary constant) and the integrating-factor formula yields the PI.
Forcing term duplicates a CF term (e.g. RHS = e^{-2x} when e^{-2x} is in the CF)
Multiply the trial PI by x (or x² for a repeated root) to break the duplication — same superposition logic, modified trial.
RHS is a sum, e.g. f(x) = cos x + e^{2x}
Find a PI for each piece separately and add them (superposition applies on the right too).

Common Misreadings

  • Treating the equation as separable or attempting an integrating factor on a second-order ODE.
  • Believing the PI alone is the 'solution' because it satisfies the equation, missing that the CF is needed for arbitrary constants.

Pitfalls

  • Omitting the PI and writing only the CF — this solves the homogeneous equation, not the given ODE.
  • Omitting the CF and writing only the PI — this loses the arbitrary constants and cannot satisfy initial/boundary conditions.
  • Failing to check for resonance: if the trial PI duplicates a CF term, the substitution will collapse to 0 = f(x) and produce no solution unless the trial is multiplied by x.
  • Leaving an arbitrary constant inside the PI (e.g. writing Ce^{3x} when the coefficient should be a determined number).

A-level Scope Notes

Fully within Edexcel IAL Further Pure (FP2/FP3) coverage of second-order linear ODEs with constant coefficients. The auxiliary-equation method and standard trial PIs (polynomial, exponential, sin/cos) are in scope; variation of parameters and non-constant-coefficient cases are out of scope and should not be invoked.
