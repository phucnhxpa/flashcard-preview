Read vertex extremum directly from completed-square form

Family: Vertex Form Reading Frequency: 2 questions

Trigger — When to use

The expression is already (or has just been) written in completed-square / vertex form, e.g. f(x) = a(x − h)² + k or k − a(x − h)², and the question asks for a maximum or minimum value (often signalled by 'hence find the minimum/maximum'). Visual cues: a single squared bracket plus or minus a constant, a leading coefficient whose sign is easy to read, and a domain that may or may not be restricted. The word 'hence' in a follow-up part is a strong instruction to reuse the vertex form from the previous part rather than expand or differentiate.

Motivation — Why this works

For any real input, (x − h)² ≥ 0, with equality iff x = h. So a(x − h)² + k achieves its extremum precisely when the squared bracket vanishes: a minimum of k if a > 0, a maximum of k if a < 0 (equivalently k − a(x−h)² with a > 0 has maximum k). This converts an optimisation question into a one-line observation about real squares — no expansion, no calculus, no re-completing — which is exactly the pre-calculus (P1) method examiners reward.

Thought Process — The step-by-step reasoning

  1. Notice the expression is already in the form a(x − h)² + k (or k − a(x − h)²): one squared bracket plus a constant.
  2. Consider alternatives — expand and use −b/(2a), or differentiate and set f′(x) = 0 — but reject them as wasteful, error-prone, and (if 'hence' appears) contrary to the question's instruction.
  3. Commit to the vertex-form reading: read the sign of the leading coefficient to decide whether the vertex is a max (a < 0) or a min (a > 0).
  4. Use (x − h)² ≥ 0 to argue that the squared term is bounded below by 0, so the extremum of the whole expression is k, attained when x = h.
  5. Check that x = h lies inside the stated domain (e.g. 0 ≤ t ≤ 15); if it does, the vertex value is the answer; if not, evaluate at the endpoints instead.
  6. State BOTH the extremum value (the y/P-coordinate, k) and the location (the x/t-coordinate, h), since examiners typically want both.

Mechanic — Concrete operations

  1. Identify h and k from the form a(x − h)² + k (or k − a(x − h)²).
  2. Determine sign of the leading coefficient: a > 0 ⇒ vertex is a minimum; a < 0 (or the '−' in k − a(...)² with a > 0) ⇒ vertex is a maximum.
  3. Assert (x − h)² ≥ 0 with equality at x = h.
  4. Set the squared bracket to 0 to locate the vertex: x = h.
  5. Read off the extremum: f(h) = k.
  6. Verify h lies in the problem's domain; if not, evaluate at the domain endpoints.
  7. Write the final answer as 'minimum/maximum value is k, attained at x = h'.

Worked Micro-Example

Given $f(x) = 2(x-3)^2 - 7$: since $(x-3)^2 \ge 0$, we have $2(x-3)^2 \ge 0$, so $f(x) \ge -7$, with equality when $x = 3$. Hence the minimum value is $-7$, attained at $x = 3$.

Variants — Same move, different surface

Downward parabola written as k − a(t − h)² with a > 0 (e.g. P_A = 53 − 0.4(t − 8)²)
The squared term is subtracted, so the expression is bounded above by k; vertex (h, k) gives the maximum. Same non-negativity argument, flipped direction.
Vertex t-value lies outside the model's domain
The constrained extremum is no longer at the vertex; evaluate the quadratic at both domain endpoints and pick the larger/smaller.
Part (a) asks to complete the square, part (b) says 'hence find the minimum'
Use the part (a) form directly — do not re-derive via calculus or −b/(2a).
Form g(x) = 5 − (x + 2)²
Rewrite mentally as 5 − (x − (−2))²; vertex at x = −2 gives maximum value 5.

Common Misreadings

  • Expanding the bracket and then re-completing the square or differentiating, which wastes time and risks sign errors.
  • Reading the extremum as the x-coordinate (h) instead of the y-coordinate (k).
  • Ignoring the 'hence' instruction and using a method from a later chapter (calculus), losing method marks.

Pitfalls

  • Misidentifying the sign of the leading coefficient and labelling a maximum as a minimum (or vice versa).
  • Forgetting to check that x = h lies within the stated domain — if it does not, the vertex value is unattainable.
  • Quoting only the extremum value without stating where it occurs (or vice versa) when the question asks for both.

A-level Scope Notes

Fully within Edexcel IAL P1 (Quadratics — completing the square). No formula booklet entry is needed; the argument relies only on (x − h)² ≥ 0. Avoid invoking calculus (f′(x) = 0) on 'hence' parts where the completed-square form has just been established.
