Trigonometry

Apply sec²x ≡ 1 + tan²x to reduce equation to a quadratic in tan x

Family: Identity Selection Frequency: 1 questions

Trigger — When to use

The equation contains both sec²x and tan x (or sec x and tan x with sec appearing squared) as the only trig functions of the same angle x. A constant term is typically present, hinting that the '+1' from the Pythagorean identity will combine cleanly. The mixed sec/tan form blocks direct algebraic manipulation, signalling that one function must be eliminated.

Motivation — Why this works

The Pythagorean identity sec²x ≡ 1 + tan²x is the unique bridge between sec²x and tan x, so substituting it collapses the equation onto a single variable u = tan x. This converts a transcendental mixed-function equation into a standard quadratic, which is solvable by factorising or the quadratic formula. The route is preferred over rewriting in sin/cos because it avoids introducing a denominator and keeps the equation polynomial throughout.

Thought Process — The step-by-step reasoning

  1. I notice two trig functions of the same angle: sec²x and tan x — I need them in one function.
  2. I rule out converting to sin/cos: that introduces 1/cos²x and forces multiplication by cos²x, adding steps and a domain caveat.
  3. I commit to sec²x ≡ 1 + tan²x because it directly eliminates sec²x in favour of tan x, the function already present linearly.
  4. Bridge: substitute, expand brackets carefully (preserving any coefficient on sec²x), and collect into the form a tan²x + b tan x + c = 0.
  5. Tactical: let u = tan x mentally so I can factor or apply the quadratic formula cleanly, then solve tan x = each root over the required interval.

Mechanic — Concrete operations

  1. Identify sec²x and tan x as the only trig functions present.
  2. Substitute sec²x = 1 + tan²x, keeping any leading coefficient: e.g. 2 sec²x → 2(1 + tan²x).
  3. Expand and collect like terms; combine constants with the RHS.
  4. Write the result as a quadratic in tan x: a tan²x + b tan x + c = 0.
  5. Factorise (or use the quadratic formula) to find values of tan x.
  6. Solve tan x = k for each root over the specified interval, including all relevant solutions via periodicity.

Worked Micro-Example

Solve $2\sec^2 x - 3\tan x = 2$ for $0 \le x < 2\pi$. Substitute $\sec^2 x = 1 + \tan^2 x$: $2(1+\tan^2 x) - 3\tan x = 2 \Rightarrow 2\tan^2 x - 3\tan x = 0$. Factor: $\tan x(2\tan x - 3) = 0$, so $\tan x = 0$ or $\tan x = \tfrac{3}{2}$. Hence $x = 0, \pi$ or $x = \arctan(3/2),\ \arctan(3/2)+\pi$.

Variants — Same move, different surface

Equation mixes cosec²x and cot x
Use cosec²x ≡ 1 + cot²x to obtain a quadratic in cot x.
Equation mixes tan²x and sec x (sec linear)
Rearrange the identity as tan²x ≡ sec²x − 1 to obtain a quadratic in sec x instead.
Equation mixes sin²x with cos x (or cos²x with sin x)
Use sin²x = 1 − cos²x (or cos²x = 1 − sin²x) to reduce to a quadratic in the linear function.
Double-angle disguise, e.g. sec²(2x) with tan(2x)
Apply the same identity in the argument 2x, solve for tan(2x), then divide the solution set appropriately.

Common Misreadings

  • Trying to convert everything to sin/cos immediately, producing a rational equation that is harder to manage.
  • Confusing sec²x with a double-angle expression and reaching for cos 2x identities.
  • Treating sec²x − tan²x = 1 as if it gives sec²x = tan²x − 1 (sign error in the identity).

Pitfalls

  • Forgetting to distribute a coefficient, e.g. writing sec²x → 1 + tan²x but failing to multiply by the leading 2.
  • Cancelling tan x from tan x(2 tan x − 3) = 0 and losing the root tan x = 0.
  • Stopping after finding tan x values without producing all x in the given interval.
  • Mis-stating the identity as sec²x ≡ tan²x − 1.

A-level Scope Notes

Fully within Edexcel IAL Pure (P2/P3) trigonometric identities and equations; uses only the Pythagorean identity sec²x ≡ 1 + tan²x and standard quadratic techniques.
