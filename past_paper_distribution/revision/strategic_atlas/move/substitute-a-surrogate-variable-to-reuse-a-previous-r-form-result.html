Trigonometry

Substitute a surrogate variable to reuse a previous R-form result

Family: Substitution Bridge Frequency: 1 questions

Trigger — When to use

Fires when a later part of a question contains the same a sin(...) ± b cos(...) structure as an earlier part, but with a composite or transformed argument such as πθ/2 − π/6, 2x, or x + π/4. The coefficients match part (a) exactly; only the argument has changed. Recognise this whenever both trig terms share the identical composite argument and the outer expression is otherwise linear (e.g. constant + trig, or constant × trig).

Motivation — Why this works

The R-form identity a sin X − b cos X = R sin(X − α) is independent of what X actually denotes; it only requires both terms to share one common argument. So once part (a) has established R and α for a sin X − b cos X, any later expression with the same coefficient pattern collapses to R sin(X − α) by relabelling. This saves re-deriving R and α, eliminates expansion errors, and isolates the outer θ-dependence from the inner trig transformation.

Thought Process — The step-by-step reasoning

  1. Notice that both trig terms in the new expression share one identical composite argument, and the coefficients match those in part (a).
  2. Rule out re-expanding via addition formulae or re-deriving R, α — that would duplicate work already done and invite sign or arithmetic slips.
  3. Commit to a substitution because the R-form result is argument-agnostic: whatever X stands for, the identity still holds.
  4. Define X to equal the shared composite argument and rewrite the expression purely in X, so the trig part becomes the exact form from part (a).
  5. After applying part (a), substitute X back in terms of θ and keep the −α phase shift attached to the full composite argument.

Mechanic — Concrete operations

  1. Identify the shared composite argument inside both sin and cos terms.
  2. Let X equal that argument (e.g. X = πθ/2 − π/6).
  3. Rewrite the trig portion of the new function as a sin X − b cos X, matching part (a) exactly.
  4. Quote part (a)'s result to replace it with R sin(X − α).
  5. Substitute X back: write the final form as (outer terms) + R sin(argument − α).
  6. Use this single sinusoidal form for max/min or equation-solving in subsequent parts.

Worked Micro-Example

Given part (a): $12\sin x - 5\cos x = 13\sin(x - \alpha)$ with $\tan\alpha = 5/12$. For $g(\theta) = 10 + 12\sin(\pi\theta/2 - \pi/6) - 5\cos(\pi\theta/2 - \pi/6)$, let $X = \pi\theta/2 - \pi/6$. Then $g(\theta) = 10 + 12\sin X - 5\cos X = 10 + 13\sin(X - \alpha) = 10 + 13\sin(\pi\theta/2 - \pi/6 - \alpha)$. Maximum is $23$, attained when $\pi\theta/2 - \pi/6 - \alpha = \pi/2$.

Variants — Same move, different surface

Later part uses argument 2x while part (a) used x
Let X = 2x; the R-form result applies unchanged, then replace X to get R sin(2x − α).
Later part uses argument x + π/4
Let X = x + π/4; substitute, apply part (a), then resubstitute, keeping −α attached to the full argument.
Outer expression is multiplicative, e.g. k · (a sin X − b cos X)
Bridge gives k · R sin(X − α); the constant simply scales the amplitude to kR.
Cosine R-form was used in part (a): a cos X + b sin X = R cos(X − α)
Same substitution principle; quote the cosine form result instead, preserving the cos(X − α) shape.
Equation-solving part: solve g(θ) = c
After substitution, solve sin(X − α) = (c − constant)/R for X, then back-substitute and unwind to θ within the stated range.

Common Misreadings

  • Treating the new composite argument as requiring a fresh R, α derivation rather than recognising structural reuse.
  • Expanding sin(πθ/2 − π/6) and cos(πθ/2 − π/6) via addition formulae and trying to recombine — correct but wasteful and error-prone.
  • Assuming the substitution changes the value of R or α — only the argument label changes.

Pitfalls

  • Forgetting to carry the −α phase shift through when resubstituting X in terms of θ.
  • Losing the outer constant or coefficient (e.g. the +10) during substitution.
  • When solving equations, failing to adjust the range for X = πθ/2 − π/6 before inverting sin, leading to missed solutions.
  • Mixing degrees and radians once α is reintroduced — keep α in the same units throughout.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P3/P4) Rsin(x ± α) / Rcos(x ± α) harmonic-form questions, where multi-part problems routinely reuse part (a)'s R, α in later parts with transformed arguments.
