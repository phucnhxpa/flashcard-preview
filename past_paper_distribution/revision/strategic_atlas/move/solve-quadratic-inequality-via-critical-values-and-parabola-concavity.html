Solve quadratic inequality via critical values and parabola concavity

Family: Quadratic Inequality Frequency: 5 questions

Trigger — When to use

Fires whenever you have reduced a problem to a quadratic inequality in a single variable, e.g. ax² + bx + c < 0, > 0, ≤ 0 or ≥ 0. Typical surface forms include discriminant conditions like (5 − c)² − 24 < 0 or k² − 18k + 65 < 0 (asking for which parameter values a quadratic has no/two real roots), area/length conditions like 18x² < 72, or any expression that simplifies to a quadratic compared with zero. Recognition cue: the unknown appears squared, you have an inequality (not equation), and you are being asked for a RANGE of values, not a single value.

Motivation — Why this works

A quadratic f(x) = ax² + bx + c crosses zero only at its roots, so its sign can change ONLY at those critical values. Once you know where f = 0 and which way the parabola opens (sign of a), the sign on each region is forced by concavity: an upward parabola (a > 0) is negative BETWEEN the roots and positive OUTSIDE; a downward parabola (a < 0) is the reverse. This avoids ad-hoc test points and converts the inequality directly into an interval. The underlying principle is the Intermediate Value Theorem applied to a continuous quadratic with at most two real roots.

Thought Process — The step-by-step reasoning

  1. Notice the inequality is quadratic in one variable (the variable appears squared and is compared with 0 or a constant). Move everything to one side so it reads f(x) ⋚ 0.
  2. Rule out alternatives: completing the square + square-rooting is risky here because students often forget the ± and get only one bound; testing individual values gives no proof; graphing software is not available — sign analysis is the cleanest A-level tool.
  3. Commit to the standard recipe: (i) find roots of f, (ii) read off concavity from the leading coefficient, (iii) read off the correct region from the parabola sketch.
  4. Bridge to action: solve f(x) = 0 by factoring if obvious (look for integer factor pairs of the constant term that sum to the linear coefficient), or otherwise by quadratic formula / completing the square.
  5. Tactical decision on the region: if a > 0 (upward parabola), then f < 0 between the roots and f > 0 outside; if a < 0, swap. Match this to the original inequality direction.
  6. Tactical decision on strictness: copy < / ≤ from the original inequality — strict inequalities exclude the roots, non-strict include them.

Mechanic — Concrete operations

  1. Rearrange to standard form f(x) = ax² + bx + c ⋚ 0 with everything on one side.
  2. Solve f(x) = 0: try factoring (find p, q with pq = c/a and p + q = b/a after normalising), else use x = (−b ± √(b² − 4ac))/(2a).
  3. Identify the two critical values x = α and x = β with α < β.
  4. Read the sign of the leading coefficient a. If a > 0, parabola opens upward (negative between α and β, positive outside); if a < 0, downward (reverse).
  5. Match to the required inequality direction to select the region: 'between' gives α < x < β (or ≤); 'outside' gives x < α or x > β (or ≤/≥).
  6. Preserve strict vs non-strict from the original inequality in the final answer.

Worked Micro-Example

Solve $k^2 - 18k + 65 < 0$. Factor: $(k-5)(k-13)<0$, so roots are $k=5,\,13$. Leading coefficient $+1>0$ ⇒ upward parabola ⇒ expression is negative BETWEEN the roots. Hence $5 < k < 13$ (strict, since the original inequality was strict).

Variants — Same move, different surface

Discriminant condition like (5 − c)² − 24 < 0 with the parameter inside a square
Either expand to standard form c² − 10c + 1 < 0 and proceed, or take square roots directly: |5 − c| < √24 = 2√6, giving 5 − 2√6 < c < 5 + 2√6.
Downward-opening parabola (a < 0), e.g. −x² + 4 > 0
Either multiply through by −1 (and FLIP the inequality) to get an upward parabola, or apply the rule directly: a < 0 ⇒ expression is positive between the roots.
Quadratic does not factorise over the integers
Use the quadratic formula for critical values, then apply identical concavity logic; surd answers are acceptable.
Pure form like x² < 4
Rewrite as x² − 4 < 0 and factor as (x−2)(x+2) < 0; equivalent to |x| < 2, i.e. −2 < x < 2. Do NOT just write x < 2.
Non-strict inequality (≤ or ≥)
Include the roots as endpoints with ≤ / ≥.

Common Misreadings

  • Choosing the complementary region: writing x < α or x > β when the answer should be α < x < β (or vice versa) — usually from misreading the parabola's concavity.
  • Stopping at the critical values and writing 'k = 5 and k = 13' instead of giving an interval.
  • Square-rooting x² < 4 to give only x < 2, dropping the negative branch.
  • Treating the inequality as an equation and giving only the boundary points.

Pitfalls

  • Using ≤ instead of < (or vice versa) — strictness must match the original inequality.
  • Sign slip when isolating the variable from u = ±√k, particularly the negative branch (e.g. 5 − c = −2√6 ⇒ c = 5 + 2√6, NOT 5 − 2√6).
  • Errors simplifying surds, e.g. writing √24 as 2√3 instead of 2√6.
  • Forgetting to flip the inequality when multiplying or dividing both sides by a negative.
  • Mis-factoring: e.g. (k−10)(k−8) for k² − 18k + 65 — always check the product of constants matches.

A-level Scope Notes

Fully within Edexcel IAL P1/P2 (quadratic inequalities) and reused in discriminant problems for 'two distinct/no real roots' conditions. No formula booklet entry needed beyond the quadratic formula. Avoid generalising to higher-degree polynomial inequalities by the same one-line rule — for cubics and quartics a sign table is required because there can be more than two sign changes.
