Algebra & Functions

Substitute intersection x-values into the simpler of the two functions

Family: Strategic Substitution Frequency: 1 questions

Trigger — When to use

Two expressions f(x) and g(x) have been set equal and solved, yielding intersection x-values, and the y-coordinate is now required. Either function may be evaluated, so a choice exists. Cues include one expression being structurally lighter (fewer terms, no added constant, smaller coefficients, no nested modulus or surd) than the other. Common phrasings: 'find the coordinates of the points of intersection', 'find the y-coordinate', 'state the intersection point'.

Motivation — Why this works

By definition of an intersection, f(x) = g(x) at each solution, so substituting that x into either function returns the same y. The arithmetic cost, however, is not symmetric: an expression with fewer operations, no added constants, or smaller coefficients is faster and less error-prone. For modulus equations especially, evaluating the simpler modulus avoids extra sign decisions inside the bars. The choice is a free optimisation that costs nothing in rigour.

Thought Process — The step-by-step reasoning

  1. Notice that the x-values came from f(x) = g(x), so both functions agree there — I have a free choice.
  2. Rule out the reflex of using the 'original' or first-listed function; that is not a mathematical requirement.
  3. Commit to whichever function has fewer terms, simpler coefficients, or no added constant — here g(x) = |x + 5a| beats f(x) = |3x + a| + a.
  4. Plug each x-value into g(x) and simplify the modulus, checking the sign of the inside to drop the bars correctly.
  5. Optionally cross-check one value in f(x) only if the answer looks suspicious; otherwise trust the equality.

Mechanic — Concrete operations

  1. Confirm that the x-values were obtained from f(x) = g(x), so f and g give the same y there.
  2. Compare the structural complexity of f and g (number of terms, coefficient size, presence of moduli/surds, added constants).
  3. Pick the simpler function — call it h(x).
  4. Substitute each intersection x-value into h(x) and simplify, determining the sign inside any modulus before removing bars.
  5. Pair each y with its x to state the intersection coordinates.

Worked Micro-Example

Let $f(x) = |3x + a| + a$ and $g(x) = |x + 5a|$ meet at $x = \tfrac{3a}{2}$ and $x = -\tfrac{5a}{4}$ (with $a>0$). Using $g$: $g\!\left(\tfrac{3a}{2}\right) = \left|\tfrac{3a}{2} + 5a\right| = \tfrac{13a}{2}$ and $g\!\left(-\tfrac{5a}{4}\right) = \left|-\tfrac{5a}{4} + 5a\right| = \tfrac{15a}{4}$. So the intersections are $\left(\tfrac{3a}{2}, \tfrac{13a}{2}\right)$ and $\left(-\tfrac{5a}{4}, \tfrac{15a}{4}\right)$.

Variants — Same move, different surface

Line meets curve in coordinate geometry (e.g. $y = mx + c$ and a quadratic/cubic)
Substitute the x-roots into the linear equation rather than the curve — one multiplication and one addition versus squaring/cubing.
Two curves where one is $y = k$ (a horizontal line)
The y-coordinate is simply $k$; no substitution is needed at all.
Intersection of $y = f(x)$ and $y = f^{-1}(x)$ on the line $y = x$
Use $y = x$ directly: the y-coordinate equals the x-coordinate, bypassing both $f$ and $f^{-1}$.
Parametric curve meeting a Cartesian line
After finding the parameter value, substitute into whichever of the parametric equations or the line is simpler.
Modulus equation $|p(x)| = |q(x)|$ solved by squaring
Evaluate the y via the modulus expression with the cleaner inside, choosing the sign that matches the x-value.

Common Misreadings

  • Believing the y must be computed from the 'original' function stated first in the question.
  • Thinking both functions must be evaluated and matched as part of the answer — verification is optional, not required.
  • Assuming the simpler-looking function might give a different y and so mistrusting the free choice.

Pitfalls

  • Mishandling the sign inside a modulus when the x-value is negative, e.g. writing $|-5a/4 + 5a|$ as $-15a/4$.
  • Forgetting an added constant outside the modulus (the $+a$ in $|3x+a|+a$) if one does decide to use the more complex function.
  • Pairing y-values with the wrong x-values when there are two intersections.
  • Dropping modulus bars before checking the sign of the interior expression, particularly when $a$ is a parameter whose sign is given.

A-level Scope Notes

Fully within Edexcel IAL Pure scope: appears in modulus equations (P3), coordinate geometry of lines and curves (P1/P2), and intersections involving inverse functions (P3).
