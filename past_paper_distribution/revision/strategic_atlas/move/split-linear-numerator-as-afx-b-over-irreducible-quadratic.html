Integration

Split linear numerator as A·f'(x) + B over irreducible quadratic

Family: Numerator Manipulation Frequency: 1 questions

Trigger — When to use

The integrand is a linear polynomial divided by an irreducible quadratic, e.g. ∫(px + q)/(ax² + bx + c) dx where the discriminant b² − 4ac is negative or non-square so the denominator does not factor over the rationals. Partial fractions into linear factors are unavailable, but a direct u = denominator substitution fails because the numerator is not a scalar multiple of f'(x). This also fires when the denominator is (irreducible quadratic)ⁿ with a linear numerator.

Motivation — Why this works

Any linear polynomial can be written uniquely as A(2ax + b) + B, where 2ax + b = f'(x) for the quadratic denominator f(x). This splits the integral into a 'logarithmic part' ∫f'(x)/f(x) dx = ln|f(x)| and a 'residual part' ∫1/f(x) dx, which is handled by completing the square to give an arctan (or, if the discriminant were positive, an artanh/log). The decomposition is forced and unique because matching two coefficients (x¹ and x⁰) yields a 2×2 linear system in A and B.

Thought Process — The step-by-step reasoning

  1. Notice the denominator is quadratic with negative/non-square discriminant — no rational linear factorisation.
  2. Rule out standard partial fractions and rule out a direct u = denominator substitution because du = f'(x) dx does not match the given numerator.
  3. Commit to engineering f'(x) inside the numerator: write (px + q) = A·f'(x) + B so the integral cleaves into a ln-piece and a constant-over-quadratic piece.
  4. Compute f'(x) = 2ax + b explicitly, then set up the identity px + q ≡ A(2ax + b) + B.
  5. Match coefficients: 2aA = p gives A = p/(2a); then b·A + B = q gives B = q − bA. Use A and B to rewrite the integral as a sum of two standard forms.

Mechanic — Concrete operations

  1. Differentiate the denominator: f'(x) = 2ax + b.
  2. Write the linear numerator as A(2ax + b) + B and expand.
  3. Equate the x-coefficient to find A = p/(2a).
  4. Equate the constant term to find B = q − bA.
  5. Rewrite the integral as A·∫f'(x)/f(x) dx + B·∫1/f(x) dx.
  6. Evaluate the first piece as A·ln|f(x)|; complete the square in the second piece and apply the arctan standard result.

Worked Micro-Example

Evaluate $\int \frac{x-1}{x^2+5x+3}\,dx$. Here $f'(x)=2x+5$, so write $x-1 = \tfrac{1}{2}(2x+5) - \tfrac{7}{2}$. Then $\int \frac{x-1}{x^2+5x+3}\,dx = \tfrac{1}{2}\ln|x^2+5x+3| - \tfrac{7}{2}\int \frac{dx}{(x+5/2)^2 - 13/4}$. Completing the square gives a standard logarithmic form $\tfrac{1}{\sqrt{13}}\ln\left|\tfrac{2x+5-\sqrt{13}}{2x+5+\sqrt{13}}\right|$ (here the discriminant is positive, so it's a log not arctan).

Variants — Same move, different surface

∫(3x+2)/(x²+4x+7) dx with negative discriminant 16−28=−12
Write 3x+2 = A(2x+4)+B with A=3/2, B=−4; the residual integral becomes (1/√3)·arctan((x+2)/√3).
∫(linear)/(ax²+bx+c)ⁿ dx for n ≥ 2
Same A·f'(x)+B split; the f'(x)/f(x)ⁿ piece is handled by power rule, the constant piece by completing the square plus a reduction or trig substitution (Further Maths).
∫(2x+5)/(x²+5x+3) dx where numerator already equals f'(x)
No decomposition needed — A=1, B=0, giving ln|x²+5x+3| immediately.
Improper case ∫(quadratic numerator)/(quadratic denominator) dx
First do polynomial division to reduce to constant + (linear)/(quadratic), then apply this decomposition to the proper part.

Common Misreadings

  • Treating the irreducible quadratic as factorisable using surd roots and pursuing partial fractions over ℝ unnecessarily.
  • Attempting u = denominator directly without checking that du matches the numerator, then getting stuck.
  • Confusing 'irreducible' (no rational factors) with 'no real factors' — the technique works in both subcases; only the residual integral changes between log and arctan.

Pitfalls

  • Mis-differentiating the denominator (e.g. writing f'(x)=x+5 instead of 2x+5).
  • Sign or arithmetic errors when solving B = q − bA, particularly when q is negative.
  • Forgetting to multiply through by the coefficient A when reassembling the integral, e.g. writing ln|f(x)| instead of (1/2)ln|f(x)|.
  • Botching the completing-the-square step on the residual integral and misidentifying whether the answer is arctan or a logarithm.

A-level Scope Notes

Fully within Edexcel IAL scope: linear-over-quadratic integrals using the f'(x)/f(x) recognition appear in Pure 4/Further Pure, with the residual integral evaluated via completing the square and the standard arctan or logarithmic results.
