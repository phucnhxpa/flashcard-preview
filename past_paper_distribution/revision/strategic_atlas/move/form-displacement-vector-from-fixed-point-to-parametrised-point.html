Vectors

Form displacement vector from fixed point to parametrised point

Family: Vector Subtraction For Distance Frequency: 1 questions

Trigger — When to use

Fires when a distance condition is imposed between a fixed point A and a variable point P that lies on a line with parameter λ (e.g. 'find λ such that AP = k' or 'find the point on line l closest to A'). Cue phrases include 'distance from A to the line', 'point on l whose distance from A is...', or any |AP| = constant. The unknown λ must be extracted via a scalar equation, but distance is intrinsically a magnitude of a vector — so a vector must be built first.

Motivation — Why this works

Distance between two points equals the magnitude of the displacement vector joining them: AP = |p − a|. Subtracting position vectors component-wise produces an expression linear in λ in each component, so |AP|² becomes a quadratic in λ — exactly the form solvable by standard algebra. Forming AP is the obligatory bridge from parametric geometry to a scalar equation in λ.

Thought Process — The step-by-step reasoning

  1. Notices that the condition is on a length AP, but A is fixed while P depends on λ — so P has no fixed coordinates yet.
  2. Rules out trying to apply a distance formula directly between coordinates, since one set of coordinates is symbolic in λ; also rules out using PA·d = 0 (that would solve a different problem — foot of perpendicular).
  3. Commits to forming the displacement vector AP = p − a because |AP| is the only object whose square gives a clean quadratic in λ.
  4. Writes p in terms of λ from the line equation, then subtracts a component-wise to get AP(λ).
  5. Chooses AP over PA (either is fine since magnitudes match), and keeps components unsimplified momentarily to check sign handling before squaring.

Mechanic — Concrete operations

  1. Write P's position vector p = a₀ + λd from the line equation.
  2. Compute AP = p − a component-wise, giving each component as a linear expression in λ.
  3. Simplify each component, watching signs carefully, especially constant terms.
  4. Note any component that simplifies to a constant (independent of λ) or to zero — this lightens the magnitude calculation later.

Worked Micro-Example

Line $l: \mathbf{r} = (1,2,3) + \lambda(1,0,-1)$ and fixed point $A=(4,2,5)$. Then $\mathbf{p} = (1+\lambda,\,2,\,3-\lambda)$, so $\overrightarrow{AP} = \mathbf{p}-\mathbf{a} = (1+\lambda-4,\,2-2,\,3-\lambda-5) = (\lambda-3,\,0,\,-\lambda-2)$. The middle zero will simplify $|\overrightarrow{AP}|^2 = (\lambda-3)^2 + (\lambda+2)^2$.

Variants — Same move, different surface

Distance between two parametrised points on different lines (e.g. shortest distance between skew lines via calculus)
Form displacement with two parameters: AP(λ,μ) = p(λ) − q(μ); leads to a system from ∂/∂λ = 0 and ∂/∂μ = 0, or equivalently AP perpendicular to both direction vectors.
Point on a plane satisfying a distance condition from A
Parametrise the plane with two parameters s,t and form AP(s,t); |AP|² is a quadratic surface in (s,t).
Finding foot of perpendicular from A to a line
Same AP(λ) construction, but instead of |AP|=k, impose AP·d = 0 to solve for λ.
2D version with a parametric line in the plane
Identical construction with two components instead of three; quadratic in λ still arises.

Common Misreadings

  • Treating P as if it had fixed coordinates and trying to apply the distance formula directly without first parametrising.
  • Confusing this distance task with the perpendicular-foot task and applying AP·d = 0 when the question actually fixes a length.
  • Believing the choice of AP vs PA changes the answer (it does not, since |AP| = |PA|).

Pitfalls

  • Sign errors in subtraction, especially when the line component itself has a minus sign, e.g. (3−λ) − 5 = −λ−2, not −λ+2.
  • Forgetting to subtract A from every component (often only the λ-bearing components get adjusted).
  • Mixing up which vector is being subtracted from which, producing −AP and then mis-squaring later (harmless for magnitude but harmful if a dot product follows).

A-level Scope Notes

Fully within Edexcel IAL Pure/Further Pure vectors: 3D position vectors, parametric line equations, and magnitudes via $|\mathbf{v}|^2 = v_1^2+v_2^2+v_3^2$. No calculus or matrix machinery needed.
