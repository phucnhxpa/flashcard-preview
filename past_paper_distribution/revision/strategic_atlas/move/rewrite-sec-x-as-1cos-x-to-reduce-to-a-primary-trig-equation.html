Trigonometry

Rewrite sec x as 1/cos x to reduce to a primary trig equation

Family: Reciprocal Rewrite Frequency: 1 questions

Trigger — When to use

The equation contains sec x (or its square, sec² x) appearing in isolation or in a linear combination with constants. There is no direct inverse-secant key on calculators and exact values are tabulated for cos, not sec. Whenever the unknown sits inside a reciprocal trig function with no other trig function present, rewrite to the primary function. Phrasings like 'solve sec x = k', 'find x such that 2 sec x − 3 = 0', or 'sec² x = 4' all signal this move.

Motivation — Why this works

By definition sec x = 1/cos x, so any equation in sec x is equivalent to one in cos x. Cos x has known exact values at standard angles (π/6, π/4, π/3, …) and a standard inverse function, so the rewritten equation is directly solvable. The reciprocal also makes the admissibility check transparent: cos x must lie in [−1, 1], so |sec x| ≥ 1 is enforced automatically once you invert. This converts an unfamiliar surface into a routine 'cos x = c' problem.

Thought Process — The step-by-step reasoning

  1. Notice the equation involves sec x with no other trig function competing for attention.
  2. Rule out trying to invert sec directly — there is no standard arcsec procedure in the IAL toolkit, and exact-value tables are written for cos.
  3. Commit to the reciprocal rewrite because it produces an equation in a primary function with memorised solutions.
  4. Bridge: replace sec x by 1/cos x, then take reciprocals of both sides (or multiply through by cos x) to isolate cos x = 1/k.
  5. Tactically check 1/k ∈ [−1, 1] before solving; if |k| < 1 the equation has no solutions, otherwise read off the reference angle and apply the standard cos-symmetry to get all solutions in the given interval.

Mechanic — Concrete operations

  1. Replace every sec x with 1/cos x in the equation.
  2. Rearrange to the form cos x = c (taking reciprocals of both sides of sec x = k gives cos x = 1/k).
  3. Verify that c ∈ [−1, 1]; if not, state 'no solutions'.
  4. Find the principal value α = arccos(c), using exact values where possible.
  5. Generate all solutions in the required interval using cos's symmetry: x = ±α + 2πn (or in degrees, ±α + 360°n).

Worked Micro-Example

Solve $2\sec x - 4 = 0$ for $x \in [0, 2\pi)$.\n$\sec x = 2 \Rightarrow \cos x = \tfrac{1}{2}$.\nPrincipal value: $x = \arccos(\tfrac{1}{2}) = \tfrac{\pi}{3}$.\nBy cos symmetry on $[0,2\pi)$: $x = \tfrac{\pi}{3}$ or $x = 2\pi - \tfrac{\pi}{3} = \tfrac{5\pi}{3}$.

Variants — Same move, different surface

Equation in cosec x (csc x) alone, e.g. cosec x = k
Rewrite cosec x = 1/sin x, giving sin x = 1/k; solve via arcsin and use sin's symmetry x = α, π − α (+ 2πn).
Equation in cot x alone, e.g. cot x = k
Rewrite cot x = 1/tan x (or cos x/sin x), giving tan x = 1/k; solve via arctan with period π.
Quadratic in sec x, e.g. sec² x = 4
Rewrite as cos² x = 1/4, then cos x = ±1/2, and solve each branch separately.
Mixed identity use, e.g. sec² x = 1 + tan² x given
If both sec and tan appear, prefer the Pythagorean identity over the reciprocal rewrite to keep a single function.

Common Misreadings

  • Treating sec x as sin x's reciprocal (it is cos x's reciprocal).
  • Reading sec x = k as cos x = k instead of cos x = 1/k.
  • Assuming sec x always has a solution; forgetting that |sec x| ≥ 1 forces |k| ≥ 1.

Pitfalls

  • Forgetting to invert: writing cos x = k instead of cos x = 1/k.
  • Sign errors when inverting negative values: sec x = −2/3 has no solution since |−2/3| < 1, not cos x = −3/2.
  • Multiplying through by cos x without noting cos x ≠ 0; this is fine here because sec x is undefined when cos x = 0 anyway, but the exclusion should be stated.
  • Reporting only the principal value and missing the second solution from cos's even symmetry in the given interval.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P3/C3 reciprocal trig functions). Uses only the definition sec x = 1/cos x, exact values at standard angles, and the standard cos solution pattern x = ±α + 2πn.
