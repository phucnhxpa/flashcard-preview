Algebra & Functions

Rewrite into standard transformation form to identify shifts, reflections, and scalings

Family: Transformation Identification Frequency: 2 questions

Trigger — When to use

Fires when a function or graph equation is presented in a form that is *almost* a recognisable parent shape but with extra signs, coefficients, or shifted arguments — e.g. y = 1/(2−x), y = a f(x−b), y = 3/(x+1) + 2. Concrete cues: a linear expression in the denominator with a leading negative or non-unit coefficient, a translated/scaled f(x−b) wrapper around a known graph, or a question asking 'find a and b' / 'sketch and state asymptotes'. Equivalently, any time the student must read off transformations (shift, reflect, scale) or determine parameters by matching transformed features (vertex, asymptote, intercept) to given values.

Motivation — Why this works

Parent graphs (y = 1/x, y = f(x), y = x², trig graphs) have known asymptotes, vertices, and orientations. Rewriting the given expression into the canonical transformation form y = a·f(x−b) + c (or its reciprocal-graph analogue ±1/(x−h)) exposes the sequence of transformations directly, so asymptote positions, branch quadrants, and parameter values can be read off without point-plotting. The underlying principle is that horizontal shifts come from x → x−b inside f, vertical scalings/reflections come from multiplication outside f, and distinguished invariant points (vertices, intersections of asymptotes) map predictably under each operation.

Thought Process — The step-by-step reasoning

  1. Notice the expression is close to a parent form but obstructed: e.g. denominator is 2−x rather than x, or the function is wrapped as a·f(x−b) rather than plain f.
  2. Rule out point-plotting (slow, error-prone) and rule out re-deriving the equation from scratch (unnecessary when transformations are visible).
  3. Commit to the transformation route because the parent's key features (asymptotes for 1/x, vertex for f) are known and transform predictably under shift/scale/reflect.
  4. Bridge: algebraically manipulate to expose the canonical form — factor out −1 from a denominator like 2−x to get −1/(x−2), or read off the (a, b) parameters from the wrapper.
  5. Tactical decision: identify the *distinguished point* to track (asymptote intersection, vertex, max/min) — this single point plus the reflection/scale sign pins everything down.
  6. Verify with a check point (e.g. y-intercept) to confirm branch orientation or sign of a.

Mechanic — Concrete operations

  1. Identify the parent function (y = 1/x, y = f(x), etc.) and its key features (asymptotes at x = 0, y = 0; or vertex at known coordinates).
  2. Rewrite the given expression into canonical form: factor −1 out of denominators (1/(2−x) = −1/(x−2)); or read off a, b, c from the wrapper a·f(x−b) + c.
  3. Apply transformation rules: x → x−b shifts right by b; multiplying by a scales y by factor a (and reflects in x-axis if a<0); adding c shifts up by c.
  4. Map the distinguished point: e.g. vertex (x₀, y₀) → (x₀ + b, a·y₀ + c); asymptote x = 0 → x = b; asymptote y = 0 → y = c.
  5. Solve for unknown parameters by equating mapped coordinates to given new values (e.g. 2 + b = 6 ⟹ b = 4; 21a = 3 ⟹ a = 1/7).
  6. Verify with a second feature: substitute x = 0 to check y-intercept lies on the correct branch / consistent with sign of a.

Worked Micro-Example

Sketch $y = \dfrac{1}{2-x}$. Rewrite: $y = \dfrac{1}{-(x-2)} = -\dfrac{1}{x-2}$. This is $y = 1/x$ shifted right 2 and reflected in the $x$-axis. Asymptotes: $x = 2$, $y = 0$. Branches: upper-left ($x<2, y>0$) and lower-right ($x>2, y<0$). Check: $x=0 \Rightarrow y = 1/2$ ✓.

Variants — Same move, different surface

Reciprocal graph with vertical and horizontal shifts, e.g. y = 3/(x+1) + 2
Read off vertical asymptote x = −1 (from x+1 = 0), horizontal asymptote y = 2 (from the +2), and scale factor 3 (steepness relative to 1/x).
Denominator with non-unit leading coefficient, e.g. y = 1/(a − bx)
Factor out −b: y = −1/(b(x − a/b)). Vertical asymptote at x = a/b; vertical scale 1/b; reflection if b > 0.
Vertex/parameter problem y = a·f(x−b) with given new vertex
Map original vertex (x₀, y₀) to (x₀ + b, a·y₀); equate to given new vertex and solve simultaneously for a and b.
Combined horizontal scale and shift y = f(ax − b)
Rewrite as f(a(x − b/a)): horizontal shift is b/a (not b), scale by 1/a. Read carefully — order matters.
Trigonometric transformations, e.g. y = a sin(x − b) + c
Track a known max/min point: max of sin at x = π/2 maps to x = π/2 + b with new y-value a + c.

Common Misreadings

  • Setting the denominator to zero correctly but failing to determine which side of the asymptote each branch sits on, drawing branches in wrong quadrants.
  • Confusing f(x − b) (shift right by b) with f(x + b) (shift left by b), giving b with the wrong sign.
  • Misreading a·f(x − b) as f(ax − b), conflating vertical scaling with horizontal scaling.
  • Treating 1/(2 − x) as identical to 1/(x − 2), missing the reflection.

Pitfalls

  • Forgetting the reflection when factoring −1 out of a denominator, leading to branches in the wrong quadrants.
  • For y = f(ax − b), incorrectly stating the horizontal shift as b rather than b/a.
  • Applying transformations in the wrong order when both inside and outside operations are present (inside operations affect x in reverse).
  • Forgetting to check sign of a via a y-intercept or second point, leaving ambiguity in the final answer.

A-level Scope Notes

Stays within Edexcel IAL P2/P3 graph transformations and curve sketching. Uses standard transformation rules (no formula booklet entry required). Avoid generalising to non-linear inner transformations like f(x²) — that lies beyond the syllabus's standard transformation set {translate, reflect, stretch}.
