Integration

Peel off one factor of sin x to set up IBP for reduction

Family: Reduction Formula Ibp Frequency: 1 questions

Trigger â€” When to use

The integrand is a single trig function raised to a general integer power n (e.g. $\sin^n x$, $\cos^n x$), and the question asks for a reduction formula linking $I_n$ to $I_{n-2}$. Equivalently, the index in the answer drops by 2, signalling that one factor must be differentiated (lowering the power by 1) and another integrated (which then combines via a Pythagorean identity to drop a further power). Phrasings include 'establish a reduction formula', 'show that $I_n = \ldots I_{n-2}$', or 'hence find $\int_0^{\pi/2} \sin^n x\,dx$'.

Motivation â€” Why this works

Writing $\sin^n x = \sin^{n-1}x \cdot \sin x$ produces a product where $\sin x$ integrates immediately to $-\cos x$, while differentiating $\sin^{n-1}x$ yields $(n-1)\sin^{n-2}x\cos x$. The resulting integrand $\sin^{n-2}x\cos^2 x$ then collapses to $\sin^{n-2}x(1-\sin^2 x)$, which is exactly $I_{n-2} - I_n$ â€” closing the recurrence. No other split achieves this clean two-step drop in index.

Thought Process â€” The step-by-step reasoning

  1. I notice the target relates $I_n$ to $I_{n-2}$, so the index must fall by exactly 2 â€” one drop from differentiation, one from a Pythagorean identity.
  2. I rule out splits like $u = \sin^n x, dv = dx$ (gives $\int \cos x\sin^{n-1}x\,dx$, doesn't loop back) and $u = \sin^{n-2}x, dv = \sin^2 x\,dx$ (the $dv$ doesn't integrate cleanly to a usable $v$).
  3. I commit to $u = \sin^{n-1}x$, $dv = \sin x\,dx$ because $dv$ integrates to $-\cos x$ and $du$ lowers the sine power by one, with a $\cos x$ available to pair via $\cos^2 x = 1 - \sin^2 x$.
  4. First concrete operation: write down $u, du, dv, v$ and apply the IBP formula $\int u\,dv = uv - \int v\,du$.
  5. Tactically, I plan to evaluate the boundary term immediately (it vanishes for the standard limits $0$ to $\pi/2$) and then use $\cos^2 x = 1-\sin^2 x$ to split the residual integral into $I_{n-2} - I_n$, so $I_n$ appears on both sides and I can solve algebraically.

Mechanic â€” Concrete operations

  1. Rewrite the integrand: $\sin^n x = \sin^{n-1}x \cdot \sin x$.
  2. Set $u = \sin^{n-1}x$, $dv = \sin x\,dx$, so $du = (n-1)\sin^{n-2}x\cos x\,dx$ and $v = -\cos x$.
  3. Apply IBP: $I_n = \left[-\sin^{n-1}x\cos x\right]_0^{\pi/2} + (n-1)\int_0^{\pi/2}\sin^{n-2}x\cos^2 x\,dx$.
  4. Evaluate the boundary term â€” it vanishes at both limits.
  5. Replace $\cos^2 x$ with $1-\sin^2 x$ to get $I_n = (n-1)(I_{n-2} - I_n)$.
  6. Solve algebraically: $nI_n = (n-1)I_{n-2}$, hence $I_n = \dfrac{n-1}{n}I_{n-2}$.

Worked Micro-Example

For $I_n = \int_0^{\pi/2}\sin^n x\,dx$, take $u = \sin^{n-1}x$, $dv = \sin x\,dx$. Then $I_n = [-\sin^{n-1}x\cos x]_0^{\pi/2} + (n-1)\int_0^{\pi/2}\sin^{n-2}x(1-\sin^2 x)\,dx = (n-1)I_{n-2} - (n-1)I_n$. Hence $nI_n = (n-1)I_{n-2}$, giving $I_n = \tfrac{n-1}{n}I_{n-2}$.

Variants â€” Same move, different surface

$I_n = \int \cos^n x\,dx$
Split as $\cos^{n-1}x\cdot\cos x$ with $dv = \cos x\,dx$, $v = \sin x$; use $\sin^2 x = 1-\cos^2 x$ to close the recurrence.
$I_n = \int \tan^n x\,dx$
Split as $\tan^{n-2}x\cdot\tan^2 x$ and use $\tan^2 x = \sec^2 x - 1$ directly (no IBP needed) â€” a cousin move where an identity replaces the IBP step.
$I_n = \int x^n e^x\,dx$
Same 'peel one factor that integrates cleanly' principle: $u = x^n$, $dv = e^x\,dx$; recurrence drops index by 1 each time.
$I_n = \int_0^{\pi/2}\sin^n x\cos^m x\,dx$
Peel $\sin x$ from the sine block: $u = \sin^{n-1}x\cos^m x$, $dv = \sin x\,dx$; same logic but with an extra $\cos^m$ carried through.

Common Misreadings

  • Splitting as $u = \sin^{n-2}x$, $dv = \sin^2 x\,dx$ in the hope of using the double-angle formula â€” this produces a $v$ involving $x - \tfrac12\sin 2x$ that does not yield a clean recurrence.
  • Choosing $u = \sin^n x$, $dv = dx$ â€” the resulting integral $\int \cos x \sin^{n-1}x\,dx$ evaluates directly and gives no link to $I_{n-2}$.
  • Treating the question as a definite-integral evaluation rather than first proving the recurrence and then iterating it.

Pitfalls

  • Forgetting the factor $(n-1)$ from differentiating $\sin^{n-1}x$.
  • Sign error: $v = -\cos x$, and the IBP formula already has a minus, so signs must be tracked carefully when combining with the $-\sin^2 x$ from the Pythagorean identity.
  • Failing to algebraically isolate $I_n$ after $I_n$ appears on both sides â€” students sometimes leave the answer as $I_n = (n-1)I_{n-2} - (n-1)I_n$ without solving.
  • Forgetting to verify the boundary term vanishes (it does for $[0,\pi/2]$ but not for arbitrary limits).

A-level Scope Notes

Fully within Edexcel IAL Further Pure scope: reduction formulae for $\int\sin^n x\,dx$, $\int\cos^n x\,dx$, $\int x^n e^x\,dx$ and similar are standard FP examination material, using only IBP and Pythagorean identities.
