Coordinate Geometry

Use focus-directrix property to read off focal distance directly

Family: Parabola Definition Shortcut Frequency: 1 questions

Trigger — When to use

Fires whenever a problem about a point $P$ on the parabola $y^2 = 4ax$ mentions either the distance from $P$ to the focus $S$ or the (perpendicular) distance from $P$ to the directrix $x = -a$. Classic phrasings: 'the shortest distance from $P$ to the directrix is $k$', 'the distance of $P$ from the directrix equals...', or any setup asking for $PS$ where $P$'s coordinates are not yet pinned down. Also fires implicitly whenever a perimeter, focal chord length, or sum-of-focal-distances appears.

Motivation — Why this works

By definition, every point on a parabola is equidistant from focus and directrix, so $PS = x_P + a$ for $y^2 = 4ax$. This converts a two-coordinate Pythagoras computation into a one-line read-off, and often hands you $x_P$ for free as a by-product. It removes any need to solve for $P$ before computing $PS$, which is both faster and immune to sign or algebra slips in $\sqrt{(x_P-a)^2 + y_P^2}$.

Thought Process — The step-by-step reasoning

  1. Notice the problem mentions both the focus $S$ and the directrix (or a distance to one of them) — that pairing is the signature of the focus-directrix definition.
  2. Rule out the brute-force route: solving for $P$'s coordinates from the parabola equation and a side condition, then applying the distance formula, is longer and error-prone.
  3. Commit to the definition $PS = $ (perpendicular distance from $P$ to directrix), because the problem has effectively pre-computed that perpendicular distance for you.
  4. Translate to the algebraic form: for $y^2 = 4ax$, identify $a$, then write $PS = x_P + a$ and equate to the given distance.
  5. If the problem also wants $P$'s coordinates (e.g. for $OP$ in a perimeter), use the same equation to extract $x_P$, then substitute back into $y^2 = 4ax$ for $y_P$.

Mechanic — Concrete operations

  1. Identify $a$ from $y^2 = 4ax$, so the focus is $S=(a,0)$ and directrix is $x = -a$.
  2. Write the focus-directrix relation: $PS = x_P + a$.
  3. Equate $x_P + a$ to the given perpendicular distance from $P$ to the directrix.
  4. Read off $PS$ directly (= that distance), and if needed solve for $x_P$ and then $y_P$ from $y_P^2 = 4ax_P$.

Worked Micro-Example

Let $P$ lie on $y^2 = 18x$, so $4a = 18$ and $a = 9/2$; the directrix is $x = -9/2$. Given the perpendicular distance from $P$ to the directrix is $9$, the focus-directrix property gives $PS = 9$ at once. Furthermore $x_P + 9/2 = 9 \Rightarrow x_P = 9/2$, and $y_P^2 = 18 \cdot 9/2 = 81$ so $y_P = \pm 9$.

Variants — Same move, different surface

Focal chord on $y^2 = 4ax$ with endpoints $P, Q$
Apply the property to each endpoint: $PS + QS = (x_P + a) + (x_Q + a) = x_P + x_Q + 2a$, giving the chord length without coordinate distance work.
Parabola given in shifted form, e.g. $(y-k)^2 = 4a(x-h)$
Use $PS = (x_P - h) + a$ with directrix $x = h - a$; same shortcut after a translation.
Parabola $x^2 = 4ay$ (vertical axis)
Swap roles: directrix is $y = -a$, focus is $(0,a)$, and $PS = y_P + a$.
Ellipse focal-distance question (Further Pure)
Use the ellipse focal property $PS = a - ex_P$ (and $PS' = a + ex_P$) instead — same philosophy of avoiding the distance formula.

Common Misreadings

  • Reading 'shortest distance to the directrix' as a slant or Euclidean distance from $P$ to a specific point on the directrix, rather than the perpendicular distance (which for a vertical directrix is purely horizontal).
  • Failing to recognise that the given directrix distance IS the focal distance $PS$, and instead launching into solving for $P$ first.
  • Confusing the directrix-distance with the distance from the origin or vertex to the directrix.

Pitfalls

  • Sign error: writing $PS = x_P - a$ instead of $x_P + a$ (forgetting the directrix is at $x = -a$, so the perpendicular distance is $x_P - (-a) = x_P + a$).
  • Misidentifying $a$ from $y^2 = 4ax$ — e.g. taking $a = 18$ instead of $a = 9/2$ when $4a = 18$.
  • Applying the formula to a parabola not in standard position without first translating to vertex-at-origin form.

A-level Scope Notes

Fully within Edexcel IAL Further Pure (FP1/FP3) coordinate geometry of the parabola $y^2 = 4ax$, where the focus-directrix definition and standard form are explicitly taught. No conic theory beyond the syllabus is invoked.
