Integration

Split improper rational fraction by numerator surgery before integrating

Family: Algebraic Division Then Integrate Frequency: 1 questions

Trigger — When to use

Fires when the integrand is a rational function with deg(numerator) ≥ deg(denominator), e.g. $\int \frac{x}{3x+3}\,dx$ or $\int \frac{x^2}{x^2+1}\,dx$. The standard log rule $\int \frac{f'(x)}{f(x)}dx = \ln|f(x)|$ does not apply because the numerator is not a constant multiple of the derivative of the denominator. Cue phrases: 'improper fraction', 'top degree equals bottom degree', or noticing that direct $\ln$ or substitution will not close. For linear-over-linear cases this is faster than formal polynomial long division.

Motivation — Why this works

Any improper rational fraction can be written as polynomial + proper fraction, and only the proper-fraction part needs a logarithm or arctan. For linear-over-linear, the trick $x = (x+1) - 1$ rewrites $\frac{x}{x+1} = 1 - \frac{1}{x+1}$, separating the integrand into a constant (integrates to $x$) and a standard reciprocal (integrates to $\ln|x+1|$). This is algebraically identical to polynomial long division but quicker to execute by hand. The same identity-add-and-subtract idea generalises to higher-degree numerators by matching the denominator's structure.

Thought Process — The step-by-step reasoning

  1. I notice the integrand is rational with numerator and denominator of equal degree, so it is improper and no direct standard form applies.
  2. I rule out $\int \frac{1}{f}dx = \ln|f|$ because the numerator is not (a multiple of) $f'$, and I rule out partial fractions because the denominator is a single linear factor.
  3. I commit to numerator surgery: rewrite the numerator to expose a copy of the denominator, leaving a constant remainder.
  4. First concrete operation: factor any constant out of the denominator to clean up, then write $x = (x+1) - 1$ so the fraction splits.
  5. Tactically, I integrate the resulting constant and $\frac{1}{x+1}$ term separately, keep the modulus on the log, carry the $\tfrac{1}{3}$ factor through, and add $+C$.

Mechanic — Concrete operations

  1. Check that deg(num) ≥ deg(denom); confirm the fraction is improper.
  2. Factor any constant from the denominator: $\frac{x}{3x+3} = \frac{1}{3}\cdot\frac{x}{x+1}$.
  3. Add and subtract to match the denominator: $x = (x+1) - 1$.
  4. Split: $\frac{x}{x+1} = 1 - \frac{1}{x+1}$.
  5. Integrate term by term using $\int 1\,dx = x$ and $\int \frac{1}{x+1}dx = \ln|x+1|$.
  6. Reattach the extracted constant and add $+C$.

Worked Micro-Example

Evaluate $\int \frac{x}{3x+3}\,dx$. Factor: $\frac{x}{3x+3} = \tfrac{1}{3}\cdot\frac{x}{x+1}$. Write $x = (x+1)-1$, so $\frac{x}{x+1} = 1 - \frac{1}{x+1}$. Then $\int \frac{x}{3x+3}\,dx = \tfrac{1}{3}\int\!\left(1 - \frac{1}{x+1}\right)dx = \tfrac{1}{3}x - \tfrac{1}{3}\ln|x+1| + C$.

Variants — Same move, different surface

Degree-2 over degree-2, e.g. $\int \frac{x^2}{x^2+1}dx$
Write $x^2 = (x^2+1) - 1$ to get $1 - \frac{1}{x^2+1}$, integrating to $x - \arctan x + C$.
deg(num) exceeds deg(denom) by 2 or more, e.g. $\int \frac{x^2}{x+1}dx$
Use formal polynomial long division to obtain quotient + proper remainder, then integrate; repeated add-and-subtract is awkward here.
Proper fraction with factorable denominator, e.g. $\int \frac{1}{(x-1)(x+2)}dx$
Use partial fractions instead; numerator surgery is unnecessary because the fraction is already proper.
Numerator is nearly a multiple of the derivative of the denominator, e.g. $\int \frac{x+2}{x^2+4x+1}dx$
Split the numerator into $\tfrac{1}{2}(2x+4)$ plus a constant, using the $f'/f$ rule on the first piece.

Common Misreadings

  • Treating $\int \frac{x}{3x+3}dx$ as a direct log: writing $\tfrac{1}{3}\ln|3x+3|$ as if the numerator were the derivative of the denominator.
  • Attempting $u = 3x+3$ substitution without first dividing — this leads to $\int \frac{u-3}{3u}\cdot\tfrac{1}{3}du$ which still demands the same split.
  • Misidentifying the integrand as needing partial fractions, even though the denominator is a single linear factor.

Pitfalls

  • Losing the $\tfrac{1}{3}$ factor when extracting it from the denominator, giving an answer off by a factor of 3.
  • Writing $\ln(x+1)$ without modulus signs; the correct indefinite answer is $\ln|x+1|$.
  • Forgetting $+C$ on the indefinite integral.
  • Algebra slip in the surgery, e.g. writing $x = (x+1)+1$ instead of $(x+1)-1$.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics integration of rational functions (C4/P4 level); uses only $\int 1\,dx$, $\int \frac{1}{x+a}dx = \ln|x+a|$, and basic algebraic manipulation.
