Differentiation

Use exponential-beats-polynomial dominance to extract horizontal asymptote

Family: Dominance Argument Frequency: 1 questions

Trigger — When to use

Fires when a limit as x → ∞ is required and the expression contains a product of a decaying exponential e^(-kx) (k > 0) and a polynomial p(x). Common phrasings: 'find the horizontal asymptote', 'long-term behaviour', 'limiting value of y'. Typical in solutions of linear ODEs (integrating-factor outputs) where the particular solution has form p(x)e^(-kx) plus a constant. Also triggers when reciprocal/quotient forms hide a polynomial × decaying exponential after rearrangement.

Motivation — Why this works

For any polynomial p(x) and constant k > 0, lim_{x→∞} p(x)e^(-kx) = 0 because the exponential decays faster than any polynomial grows. This is a standard A-level limit (justifiable via repeated L'Hôpital but accepted as a known result). Hence such terms can be discarded in the limit, isolating the constant part as the asymptotic value. The principle is essentially that e^x grows faster than every x^n.

Thought Process — The step-by-step reasoning

  1. Notices a product of e^(-kx) (k>0) with a polynomial in the expression for y or y², and the question asks for behaviour as x → ∞.
  2. Rules out keeping the polynomial term — its growth cannot defeat exponential decay, so no algebraic cancellation is needed.
  3. Commits to invoking the standard dominance result: polynomial × decaying exponential → 0.
  4. Bridges by isolating the constant term that survives, since every x-dependent piece is multiplied by the decaying exponential.
  5. Decides whether the surviving statement is for y or y², and selects the correct root using any sign restriction (e.g. y > 0).

Mechanic — Concrete operations

  1. Identify all x-dependent terms and verify each is of the form (polynomial) × e^(-kx) with k > 0.
  2. Quote the standard limit: p(x)e^(-kx) → 0 as x → ∞ for any polynomial p.
  3. Replace each such term by 0 in the expression.
  4. Read off the surviving constant as the limiting value.
  5. If the expression is for y² (or |y|), apply the stated sign condition to select the correct branch.

Worked Micro-Example

Given y^2 = 649 - 36e^{-x/3}(x^2 + 6x + 18) with y > 0, find the horizontal asymptote as x → ∞.\nSince e^{-x/3}(x^2+6x+18) → 0 (exponential decay dominates polynomial growth), y^2 → 649.\nWith y > 0, the asymptote is y = √649.

Variants — Same move, different surface

Solution of dy/dx + ky = polynomial via integrating factor, asked for long-term value
Particular solution form is p(x) + Ce^(-kx); the exponential term vanishes leaving p(x), and if p is itself constant that constant is the asymptote.
Newton's law of cooling / population models: T(t) = T_∞ + (polynomial)e^(-kt)
Same dominance gives T → T_∞ as t → ∞.
Limit as x → -∞ instead of +∞
Decay reverses to growth; the dominance argument fails in that direction, so no horizontal asymptote there (or a different analysis is needed).
Expression involves e^(-kx) × trigonometric × polynomial
Bound the trig factor by 1 in modulus, then apply the same dominance to conclude the product → 0.

Common Misreadings

  • Believing the polynomial 'wins' because it has unbounded degree or large coefficients.
  • Trying to apply L'Hôpital's rule repeatedly when the standard dominance result is sufficient and expected.
  • Confusing decay direction: assuming e^(-kx) → 0 as x → -∞ (it actually blows up there).

Pitfalls

  • Failing to justify the limit — examiners expect a brief statement that exponential decay dominates polynomial growth.
  • Reporting y² = 649 when the question asks for y, or quoting y = ±√649 without using the y > 0 condition.
  • Discarding only some of the x-dependent terms and leaving stray polynomial pieces in the limit.
  • Misreading the sign of k in e^(kx); only k > 0 in e^(-kx) gives decay.

A-level Scope Notes

Uses only the standard A-level fact that e^x grows faster than any polynomial (P3/P4 limits and FP differential equations); no formal ε–N analysis is required.
