Sequences & Series

Set arithmetic sum S_n equal to target to form a quadratic in n

Family: Sum To Target Frequency: 1 questions

Trigger — When to use

The question asks for the number of terms (weeks, days, months, payments) needed for a *cumulative* total to reach or exceed a given value. Cue words include 'total', 'altogether', 'in all', 'enough money', 'accumulated', or 'after how many ... will the sum exceed'. The underlying sequence is arithmetic (constant increment), and the target refers to a running total rather than a single term. Distinguish carefully from 'in which week does the saving itself reach £X', which is a u_n problem.

Motivation — Why this works

For an arithmetic progression, S_n = n/2(2a + (n−1)d) is a quadratic expression in n, so equating it to a numerical target produces a quadratic equation solvable by factorising or the quadratic formula. The alternative form S_n = n/2(a + l) requires the last term l, which is itself unknown when n is the unknown, so the (2a + (n−1)d) form is strategically preferred. Reducing the worded scenario to a single equation in one unknown is the standard A-level move for sum-to-target problems.

Thought Process — The step-by-step reasoning

  1. Notice the word 'total/altogether/enough' — this signals a cumulative sum S_n, not an individual term u_n.
  2. Rule out using S_n = n/2(a + l) because the last term l is not given and would introduce a second unknown.
  3. Commit to S_n = n/2(2a + (n−1)d) because it expresses the sum purely in terms of the known a, d and the unknown n.
  4. Substitute the given a, d and the target value for S_n, then clear the fraction by multiplying through by 2.
  5. Expand and rearrange into standard quadratic form an² + bn + c = 0 ready for factorising or the quadratic formula, discarding any negative or non-integer root at the end.

Mechanic — Concrete operations

  1. Identify a, d and the target sum T from the wording.
  2. Write T = n/2(2a + (n−1)d).
  3. Multiply both sides by 2: 2T = n(2a + (n−1)d).
  4. Expand the right-hand side to obtain a quadratic in n.
  5. Rearrange into the form An² + Bn − 2T = 0 (or = 0 after moving 2T across).
  6. Solve using the quadratic formula or factorisation; reject negative or non-integer solutions.
  7. If the question asks 'enough to reach £T', round up to the next whole n.

Worked Micro-Example

Sara saves £5 in week 1 and £0.25 more each subsequent week. After how many weeks does she have at least £350? Set S_n = 350: 350 = n/2(2(5) + (n−1)(0.25)) ⇒ 700 = n(9.75 + 0.25n) ⇒ 0.25n² + 9.75n − 700 = 0 ⇒ n² + 39n − 2800 = 0. Quadratic formula: n = (−39 + √(1521 + 11200))/2 = (−39 + √12721)/2 ≈ 36.9. So n = 37 weeks (rounded up since 36 weeks gives less than £350).

Variants — Same move, different surface

Loan repayment: fixed first payment, fixed monthly increase, find months to clear a balance.
Set S_n equal to total debt; solve quadratic in n; round up to ensure repayment completes.
Production targets: daily output increases by a constant amount, find days to reach a cumulative quota.
Same equation; interpret n as days and round up to next integer.
'At least' or 'first exceeds' phrasing.
Solve the quadratic equality, then take the smallest integer n strictly greater than the positive root.
Decreasing arithmetic sequence (d < 0) with cumulative target.
Same formula with negative d; check that the target is actually attainable before n becomes ≤ 0.
Last term l is given instead of d.
Use S_n = n/2(a + l) directly, giving a linear equation in n rather than a quadratic.

Common Misreadings

  • Treating 'how many weeks until savings are £350' as u_n = 350 (a single weekly amount) instead of S_n = 350 (cumulative).
  • Reading the first term as the increase d, or vice versa.
  • Assuming the answer must be an integer root of the quadratic; in fact the positive root is usually non-integer and must be rounded up.

Pitfalls

  • Forgetting the factor of n/2 and writing S_n = 2a + (n−1)d.
  • Sign errors when expanding n(n−1)d, especially with decimal d.
  • Rounding down instead of up when the target is 'at least T', giving a total below T.
  • Keeping both roots of the quadratic instead of discarding the negative one.

A-level Scope Notes

This is a core P1/P2 arithmetic series technique in Edexcel IAL Mathematics; only the standard sum formula and quadratic-solving methods (factorising, quadratic formula) are required.
