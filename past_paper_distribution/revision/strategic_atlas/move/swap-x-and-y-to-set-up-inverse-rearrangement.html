Trigger — When to use

The question asks for f⁻¹(x) (or to 'find the inverse function') given an explicit formula y = f(x), most often a rational/Möbius form (ax+b)/(cx+d), a linear-in-radical form, or a simple exponential/logarithmic form. Phrases like 'find the inverse', 'express x in terms of y then …', or 'show that f⁻¹(x) = …' all fire this move. It is the very first algebraic step before any rearrangement begins.

Motivation — Why this works

If y = f(x), then by definition the inverse satisfies x = f(y), so swapping the roles of x and y converts the original relation directly into the equation that defines f⁻¹. Solving the swapped equation for y then gives y = f⁻¹(x) with the correct variable already named. The alternative — solving y = f(x) for x and then relabelling at the end — is algebraically identical but invites notational slips; swapping first keeps 'x is the input of f⁻¹' visible throughout.

Thought Process — Step-by-step reasoning

  1. I notice the question asks for f⁻¹(x), and f is given by an explicit formula in x — so an algebraic inversion is needed, not a graphical or numerical one.
  2. I rule out 'flip the fraction' (that would give 1/f(x), not the inverse) and rule out composition f(g(x)) = x as unnecessarily heavy notation here.
  3. I commit to the swap-and-rearrange template because it produces a single equation in a single unknown y, with x already playing the role of the input to f⁻¹.
  4. I write y = f(x) explicitly, then replace every x by y and every y by x to obtain x = f(y).
  5. I check that I have swapped every occurrence (numerator, denominator, exponent, etc.), so that no 'old' x is left mixed with the 'new' y before rearrangement begins.

Mechanic — Concrete operations

  1. Write the function as y = f(x), e.g. y = (3x+2)/(x−1).
  2. Replace every x with y and every y with x simultaneously: x = (3y+2)/(y−1).
  3. Treat the resulting equation as 'solve for y' — this is now an algebraic rearrangement task.
  4. After isolating y, write the answer as f⁻¹(x) = (expression in x), and state the domain (range of f).

Worked Micro-Example

Given $f(x) = \dfrac{3x+2}{x-1}$, set $y = \dfrac{3x+2}{x-1}$. Swap: $x = \dfrac{3y+2}{y-1}$. Then $x(y-1) = 3y+2 \Rightarrow xy - x = 3y + 2 \Rightarrow y(x-3) = x+2$, so $f^{-1}(x) = \dfrac{x+2}{x-3}$.

Variants — Same move, different surface

Linear-in-radical form, e.g. f(x) = √(x+a) + b
After swapping, isolate the radical first, then square; remember to restrict the domain so the inverse is single-valued.
Exponential form, e.g. f(x) = e^(2x) − 3
After swapping, isolate the exponential and apply ln; the inverse is logarithmic with domain x > −3.
Logarithmic form, e.g. f(x) = ln(x−1) + 2
After swapping, isolate the ln and exponentiate both sides to recover y.
Piecewise-defined function
Apply swap-and-rearrange to each piece separately, then state f⁻¹ piecewise with domains coming from the original ranges.
Self-inverse check (e.g. f(x) = (x+a)/(x−1) type)
After swapping and rearranging, compare with f(x); if identical, conclude f⁻¹ = f without further work.

Common Misreadings

  • Confusing f⁻¹(x) with 1/f(x) and writing the reciprocal of the expression instead of inverting the mapping.
  • Treating the swap as merely cosmetic and forgetting that 'solve for y' is now the genuine task.
  • Believing the inverse exists without first checking that f is one-to-one on its stated domain.

Pitfalls

  • Failing to swap every occurrence of x and y (e.g. swapping in the numerator but not the denominator).
  • Dropping domain/range information — the domain of f⁻¹ is the range of f, and this must be stated for full marks.
  • Multiplying through by (y − 1) without noting y ≠ 1, leading to a missing exclusion in the final domain.

A-level Scope Notes

Standard Edexcel IAL Pure technique for inverse functions (C3/P3 level): applied to one-to-one functions given explicitly, with domain/range stated. No use of implicit differentiation or formal bijection proofs is required.
