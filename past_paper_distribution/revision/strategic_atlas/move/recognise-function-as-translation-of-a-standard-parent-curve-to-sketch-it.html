Algebra & Functions

Recognise function as translation of a standard parent curve to sketch it

Family: Transformation Recognition Frequency: 2 questions

Trigger — When to use

The equation has the structural form y = f(x − a) + b where f is a standard parent curve whose shape, symmetry, and asymptotes are already known (e.g. y = 1/x, y = 1/x², y = √x, y = ln x, y = eˣ). Visual cues include an additive constant outside the function (vertical shift) and/or a constant added to the input variable (horizontal shift). Equivalent phrasings: 'sketch y = 1/(x+1)', 'sketch y = 1/x² − 9', 'sketch y = e^(x−2) + 1' — anything that is a parent curve with linear surface decoration. The move fires whenever the question asks for a SKETCH with asymptotes/intercepts rather than a precise pointwise plot.

Motivation — Why this works

Curve transformations preserve shape, symmetry, and the behaviour near asymptotes; only the LOCATION of these features moves. By identifying the parent function, the student inherits its entire qualitative description for free and only needs to translate the key features (asymptotes, anchor points, intercepts). This avoids tables of values or calculus-based shape analysis, both of which are slower and prone to algebraic slips. The underlying principle is the transformation rule: replacing x by x − a shifts the graph a units in the +x direction, and adding b to f(x) shifts it b units up.

Thought Process — The step-by-step reasoning

  1. Notice the structural form: strip away the additive/inside constants and ask 'what is the bare function underneath?' — e.g. in y = 1/x² − 9 the parent is 1/x²; in y = 1/(x+1) the parent is 1/x.
  2. Rule out alternatives: a table of values gives only isolated points and risks missing the asymptotic behaviour; differentiating to find turning points is overkill for a sketch and may not even produce the asymptotes.
  3. Commit to the transformation reading once you confirm the constants act linearly on input or output (not, say, on x² inside a denominator like 1/(x²−9), which would be an entirely different curve).
  4. Bridge to action: write down the parent's asymptotes and any key anchor points, then apply the translation vector to each — vertical shift moves horizontal asymptotes and y-intercepts; horizontal shift moves vertical asymptotes and x-intercepts.
  5. Tactical check: substitute one or two simple x-values into the actual equation to verify the branch orientations and catch sign errors in the shift direction.
  6. Finally, find any intercepts of the transformed curve algebraically (set y = 0 or x = 0) and label asymptotes explicitly, since mark schemes require them.

Mechanic — Concrete operations

  1. Identify the parent function f(x) and the transformation: y = f(x − a) + b.
  2. List parent's key features: vertical asymptote(s), horizontal asymptote, symmetry, and 1–2 anchor points.
  3. Apply translation: vertical asymptote x = x₀ becomes x = x₀ + a; horizontal asymptote y = y₀ becomes y = y₀ + b; anchor (p, q) becomes (p + a, q + b).
  4. Find intercepts of the transformed curve: set y = 0 to get x-intercepts, set x = 0 (if in domain) to get the y-intercept.
  5. Sketch both branches/regions, drawing asymptotes as dashed lines and labelling them with their equations.
  6. Verify with one substitution: pick a convenient x and check the y-value matches the sketch.

Worked Micro-Example

Sketch $y = \frac{1}{x^2} - 9$. Parent: $y = 1/x^2$ with asymptotes $x=0$, $y=0$, both branches positive. Translate down by 9: asymptotes become $x=0$ and $y=-9$. Intercepts: $1/x^2 - 9 = 0 \Rightarrow x^2 = 1/9 \Rightarrow x = \pm 1/3$. No y-intercept (x=0 excluded). Two branches symmetric in the y-axis, descending to the asymptote $y=-9$.

Variants — Same move, different surface

Horizontal translation only, e.g. y = 1/(x+1)
Vertical asymptote shifts from x = 0 to x = −1; horizontal asymptote y = 0 is unchanged. Anchor point (1, 1) becomes (0, 1).
Combined shift, e.g. y = 1/(x − 2) + 3
Both asymptotes shift: vertical to x = 2, horizontal to y = 3. Anchor and intercepts must be recomputed for the doubly-shifted curve.
Stretch combined with translation, e.g. y = A/x² − 9 or y = 2/(x+1)
Apply the vertical stretch (multiply parent y-values by A) BEFORE the translation; asymptote locations are unchanged by the stretch but anchor points scale.
Reflection involved, e.g. y = −1/x² + 4
Reflect parent in the x-axis first (branches now negative), then translate up by 4; horizontal asymptote becomes y = 4.

Common Misreadings

  • Confusing y = 1/x² − 9 with y = 1/(x² − 9) — the latter has vertical asymptotes at x = ±3 and is not a translation of 1/x².
  • Reading y = 1/(x+1) as a shift right by 1 instead of left by 1 (the sign convention f(x−a) shifts +a, so f(x+1) = f(x−(−1)) shifts left).
  • Treating the additive constant as a stretch — e.g. thinking 'y = 1/x² − 9' scales the curve by 9.

Pitfalls

  • Shifting in the wrong direction: f(x+a) moves LEFT by a, not right.
  • Forgetting that a vertical translation moves BOTH branches — parts of y = 1/x² − 9 now lie below the x-axis.
  • Failing to label asymptotes on the sketch with their equations (e.g. y = −9, x = −1) — Edexcel mark schemes explicitly require this.
  • Forgetting that the y-intercept may not exist if x = 0 is excluded from the domain (as for any 1/x or 1/x² type curve unshifted horizontally).

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P2/P3) curve sketching topics. Uses the standard transformation rules y = f(x) + a (vertical shift), y = f(x + a) (horizontal shift), y = af(x) (vertical stretch), y = f(ax) (horizontal stretch) — these are not in the formula booklet but are core knowledge. Avoid mixing in non-rigid transformations (e.g. y = f(x)² ) which fall outside this move's remit.
