Strategic Atlas / Differentiation / Stationary Point Condition
Differentiation

Set the numerator of dy/dx to zero to locate stationary points

Family: Stationary Point Condition Frequency: 3 questions

Trigger — When to use

The question asks for stationary points (or points of horizontal tangent) of a curve, and the derivative dy/dx has been obtained — typically as a single rational expression of the form N(x,y)/D(x,y), or as a polynomial. Visual cues include phrases like 'find the stationary points', 'find the coordinates where the gradient is zero', or 'find the x-coordinate where dy/dx = 0'. The derivative is already in hand from a previous part (explicit, implicit, or parametric differentiation). When the derivative is a fraction over a domain where the denominator is clearly non-zero (e.g. a positive power, a strictly positive surd), this move fires immediately and bypasses cross-multiplication.

Motivation — Why this works

By definition, a stationary point is where the tangent is horizontal, i.e. dy/dx = 0. A fraction equals zero iff its numerator equals zero AND its denominator is non-zero, so the calculus condition collapses to the single algebraic equation N(x,y) = 0 — usually much simpler than the full fraction. This move converts a geometric condition into a clean algebraic equation (often linear or quadratic), and when paired with the original curve equation it produces a solvable system. The underlying principle is the zero-product/zero-quotient rule: only the numerator can drive a quotient to zero.

Thought Process — The step-by-step reasoning

  1. Notice the keyword 'stationary points' — this is the standard A-level signal that dy/dx must be set to zero (NOT y = 0, NOT d²y/dx² = 0).
  2. Rule out cross-multiplying or clearing the fraction in dy/dx = N/D — that creates unnecessary work and risks extraneous solutions.
  3. Inspect the denominator D: if the domain forces D > 0 (e.g. (4x−1)^{3/2} on x > 1/4) or D is generically non-zero, then the equation reduces cleanly to N = 0; otherwise flag that D ≠ 0 must be verified at the end.
  4. Commit to writing N(x,y) = 0 as the working equation. If the derivative is a polynomial (no fraction), this is just dy/dx = 0 directly.
  5. Decide the next algebraic step: solve a polynomial in x if the derivative is explicit, or treat N(x,y) = 0 as a second equation to combine with the original curve (substitution) for implicit curves.
  6. Anticipate the number of solutions from the degree of N (e.g. a quadratic numerator gives at most two stationary points).

Mechanic — Concrete operations

  1. Write dy/dx in the form N/D (or as a polynomial) from the previous part.
  2. Check/state that D ≠ 0 throughout the relevant domain (or note the condition to verify later).
  3. Set N = 0 and write down the resulting algebraic equation.
  4. Solve the equation: factorise/quadratic formula if a polynomial in x; substitute into the curve equation if N involves both x and y.
  5. For each candidate point, verify it lies in the domain and that D ≠ 0 there.
  6. If coordinates are required, substitute the x-values back into the original curve to get y-values.

Worked Micro-Example

Given $\frac{dy}{dx} = \frac{4x-8}{(4x-1)^{3/2}}$ on $x > 1/4$. Since $(4x-1)^{3/2} > 0$ on the domain, set numerator to zero: $4x - 8 = 0 \Rightarrow x = 2$. Check $2 > 1/4$ ✓, so the unique stationary point is at $x = 2$.

Variants — Same move, different surface

Implicit differentiation giving dy/dx = N(x,y)/D(x,y)
Set N(x,y) = 0 as a linear/algebraic relation between x and y, then substitute into the original curve equation to solve simultaneously. Verify D(x,y) ≠ 0 at each solution.
Parametric curves with dy/dx = (dy/dt)/(dx/dt)
Set dy/dt = 0 (the 'numerator') with dx/dt ≠ 0, then solve for the parameter t.
Vertical tangents instead of stationary points
Set the denominator D = 0 (with N ≠ 0) — this is the dx/dy = 0 condition, not a stationary point.
dy/dx is already a polynomial (no fraction)
Just set dy/dx = 0 directly — there is no denominator to worry about.

Common Misreadings

  • Confusing stationary points with points of inflection — inflection requires d²y/dx² = 0, not dy/dx = 0.
  • Setting y = 0 (finding x-intercepts of the curve) instead of dy/dx = 0.
  • Treating 'stationary point' as 'where the curve is constant', leading to no equation at all.

Pitfalls

  • Cross-multiplying or clearing the fraction unnecessarily, introducing extraneous solutions where the denominator vanishes.
  • Forgetting to check that the denominator is non-zero at the candidate points (a 0/0 form is NOT a stationary point).
  • Forgetting to verify that solutions lie within the stated domain (e.g. x > 1/4).
  • Reporting only x-values when the question asks for coordinates — substitute back to get y.
  • Missing solutions when the numerator factorises into multiple roots (state ALL of them).

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P1–P4) and Further Pure. No formula booklet entry is needed — this is a definitional move. Avoid generalising to critical points of multivariable functions or to complex stationary points; stay with real x in the stated domain.
