Read range of a quadratic from its vertex and opening direction

Family: Range From Vertex Frequency: 1 questions

Trigger — When to use

Fires on a part asking for the range of f(x) when f is quadratic and an earlier part has produced its completed-square form a(x-p)^2 + q (so the vertex (p, q) and sign of a are known). Typical phrasings: 'state the range of f', 'find the set of values f(x) can take', 'write down the range'. Often follows immediately after 'find the minimum/maximum value' or a completed-square question. The move is also cued whenever the domain is the full real line (no restriction stated).

Motivation — Why this works

A quadratic in completed-square form a(x-p)^2 + q has its extreme value at x = p with f(p) = q, and (x-p)^2 takes every non-negative real value. If a > 0 the parabola opens upward so f(x) - q = a(x-p)^2 ≥ 0, giving f(x) ≥ q with equality attained; if a < 0, f(x) ≤ q. As x → ±∞ the magnitude grows without bound, so every value on the correct side of q is attained. This bypasses inverse-function or graphing arguments entirely.

Thought Process — The step-by-step reasoning

  1. Notice f is quadratic and the previous part produced a(x-p)^2 + q, so the vertex (p, q) is already in hand.
  2. Rule out doing calculus, inverse functions, or sketching from scratch — the completed-square form alone determines the range.
  3. Commit because a fixes the opening direction and q is the extreme value, attained at x = p.
  4. Bridge: read off the sign of a, then write y ≥ q (if a > 0) or y ≤ q (if a < 0).
  5. Tactical: use ≥ / ≤ (not strict) since the vertex value is attained, and write the range in terms of f(x) or y, not x.

Mechanic — Concrete operations

  1. From the completed-square form a(x-p)^2 + q, extract the vertex y-coordinate q and the sign of a.
  2. If a > 0, the minimum value of f is q, attained at x = p; if a < 0, the maximum value is q.
  3. Note that (x-p)^2 ranges over [0, ∞), so f(x) takes every value on the correct side of q.
  4. State the range using a non-strict inequality: f(x) ≥ q or f(x) ≤ q (equivalently [q, ∞) or (-∞, q]).
  5. Check the variable: the inequality must be on f(x) or y, not on x.

Worked Micro-Example

Given f(x) = 2x^2 - 12x + 11 = 2(x-3)^2 - 7 with domain \(\mathbb{R}\). Here a = 2 > 0 and the vertex is (3, -7). So f(x) - (-7) = 2(x-3)^2 \ge 0, with equality at x = 3. As x \to \pm\infty, f(x) \to +\infty. Hence the range is \(f(x) \ge -7\).

Variants — Same move, different surface

Downward-opening quadratic, e.g. f(x) = -3(x+1)^2 + 5
Vertex gives the maximum; range is f(x) ≤ 5, i.e. (-∞, 5].
Restricted domain such as x ≥ 0 or a closed interval [a, b]
Check whether the vertex x = p lies in the domain. If yes, the vertex value is still extreme; otherwise the extremes are at the endpoints — evaluate f at the relevant boundary points and use monotonicity on each side of p.
Composite g(x) = f(x) + k or k·f(x) where f is quadratic with known vertex
Shift/scale the vertex value: minimum becomes q + k or k·q (flipping direction if k < 0).
Range of |f(x)| or [f(x)]^2 type questions seen in Further Maths
Use the quadratic range first, then apply the outer operation to the interval; absolute value may fold negative parts.

Common Misreadings

  • Stating the domain (e.g. x ∈ \(\mathbb{R}\)) instead of the range.
  • Giving the vertex coordinates (p, q) as the answer rather than the set of output values.
  • Reporting only the minimum value q without the inequality or interval describing all attained y.

Pitfalls

  • Using strict inequality f(x) > q when the minimum/maximum is actually attained at x = p.
  • Writing the range as q ≤ x instead of q ≤ f(x) or q ≤ y, confusing input and output.
  • Forgetting to flip the inequality direction when a < 0.
  • Ignoring a restricted domain and quoting [q, ∞) when the vertex is outside the allowed x-values.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P1/P2 quadratics and functions): completed-square form, vertex, and range/inequality notation are standard. No calculus or limit machinery is required.
