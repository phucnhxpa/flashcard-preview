Integration

Partition region using geometric boundaries to assign per-piece integration limits

Family: Region Decomposition Frequency: 1 questions

Trigger — When to use

A diagram shows two (or more) shaded regions whose boundaries are described by a mix of curves, lines, and coordinate axes (e.g. R₁ bounded by C, l and the y-axis; R₂ bounded by C and a chord). The limits are not all given by curve–curve intersections — at least one boundary is a coordinate axis or a vertical/horizontal line stated in the question. Wording cues include 'bounded by ... and the y-axis', 'enclosed between C and the line', or labelled points E, F at intersections. Fires before any antiderivative is computed.

Motivation — Why this works

An integral ∫ₐᵇ (top − bottom) dx requires concrete numerical limits, but the problem supplies them implicitly through geometry. The y-axis means x = 0; a labelled intersection point means solving curve = line. By matching each region's verbal/visual boundary description to an x-value (or y-value), the student converts a picture into a definite integral. Treating each region separately avoids cancellation between pieces where the 'top' function changes.

Thought Process — The step-by-step reasoning

  1. Notice that the two shaded regions have different boundary descriptions — one mentions the y-axis, the other does not.
  2. Rule out using a single integral over the whole x-range, since the upper/lower function swaps at the intersection point E.
  3. Commit to splitting at the shared boundary x-value (point E) and treating R₁ and R₂ as independent integrals.
  4. Translate each named boundary into an x-value: y-axis → x = 0; point E → solve C = l; point F → second intersection.
  5. Decide which curve is on top in each subregion (often different) before writing the integrand.

Mechanic — Concrete operations

  1. List every boundary of each region in words: curve C, line l, axis, chord, etc.
  2. Convert each boundary to an equation: y-axis → x = 0, x-axis → y = 0, named points → solve simultaneously.
  3. Find intersection x-coordinates by setting the relevant curves equal.
  4. Assign limits: R₁ from x = 0 to x = 1; R₂ from x = 1 to x = 3 (or analogous).
  5. For each piece, identify top function minus bottom function on that interval.
  6. Write Area = ∫₀¹ (top₁ − bottom₁) dx + ∫₁³ (top₂ − bottom₂) dx.

Worked Micro-Example

Curve C: y = x(3−x) and line l: y = x meet where x(3−x) = x ⇒ x = 0 or x = 2. R₁ is bounded by C, l and the y-axis; R₂ is enclosed by C and l. Then R₁ runs from x = 0 to x = 0 (degenerate here) — instead take y-axis with C: y = x(3−x), line l: y = x+1, meeting at x = 1. R₁ = ∫₀¹ [(x(3−x)) − (x+1)] dx, R₂ = ∫₁ˣ_F [...] dx using the second intersection x_F.

Variants — Same move, different surface

Region bounded by curve and the x-axis up to a vertical line x = k
Lower limit comes from where curve meets x-axis (y = 0); upper limit is the stated k.
Two curves intersecting twice with no axis involved
Both limits are intersection x-values; solve f(x) = g(x) for both.
Integration with respect to y (horizontal strips)
Replace x-axis/y-axis roles; y-axis boundary becomes a y-limit only if region is described in y.
Region containing a point where top/bottom functions swap
Split the integral at the swap point even if it is not a labelled vertex.
Parametric or polar region (Further Maths)
Convert geometric boundaries to parameter or angle limits, e.g. θ = 0 for the initial line.

Common Misreadings

  • Assuming both regions share the same limits (e.g. both from 1 to 3), ignoring that R₁ extends back to the y-axis.
  • Treating a labelled point as an intersection of curve and line when it is actually the foot of a perpendicular or a point on an axis.
  • Including or excluding a triangular sliver (e.g. under chord DF) that the diagram does not actually shade.

Pitfalls

  • Using a single integral from x = 0 to x = 3 then trying to split mid-calculation — error-prone compared with setting up two integrals from the start.
  • Forgetting that on R₁ the line is above the curve while on R₂ the curve is above the line (or vice versa), leading to a sign error.
  • Solving curve = line and using only one root, missing the second intersection that bounds R₂.

A-level Scope Notes

Standard Edexcel IAL P2/P3 area-between-curves work; all boundaries are lines, polynomial/rational/exponential curves, or coordinate axes, and limits are found by elementary simultaneous equations.
