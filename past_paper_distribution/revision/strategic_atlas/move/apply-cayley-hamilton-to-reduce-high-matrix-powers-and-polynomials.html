Trigger — When to use

Fires when the question (i) asks you to verify a matrix satisfies its own characteristic equation, (ii) asks for a high matrix power such as A^4, A^5 or A^{-1}, or (iii) asks to evaluate a matrix polynomial like A^4 - 3A^2 + I. The characteristic polynomial (or the eigenvalues from which it is built) is typically already known from an earlier part of the question. Visual cues: a 2×2 or 3×3 matrix together with a request for a power higher than 2 or 3, or the explicit instruction 'using the Cayley-Hamilton theorem' or 'hence find...'. Equivalent phrasings include 'show that A satisfies p(A) = 0' or 'use the result of part (a) to find A^n'.

Motivation — Why this works

The Cayley-Hamilton theorem says that every square matrix satisfies its own characteristic polynomial: if p(λ) = det(λI − A) is the characteristic polynomial, then p(A) = 0 (the zero matrix). For a 3×3 matrix this gives a cubic relation A^3 + aA^2 + bA + cI = 0, which can be rearranged so that A^3 — and by induction every higher power — is expressible as a linear combination of I, A and A^2. This collapses any matrix polynomial to degree at most n−1, and avoids the heavier machinery of diagonalisation (which needs P^{-1}). Rearranging the same identity also gives a clean route to A^{-1} when c ≠ 0.

Thought Process — Step-by-step reasoning

  1. Notice that the question demands a high power of A, an inverse, or verification of p(A) = 0 — and that the characteristic polynomial is either given or just computed in a previous part.
  2. Rule out brute-force multiplication (computing A^4 directly is three matrix multiplications and error-prone) and rule out diagonalisation if P^{-1} would be awkward to compute.
  3. Commit to Cayley-Hamilton because it converts a degree problem into a single matrix multiplication (A^2) plus scalar arithmetic, which is much cheaper.
  4. Bridge to operation: write down the scalar characteristic equation p(λ) = 0, then carefully replace λ^k by A^k and the constant term by (constant)·I — never by a bare scalar — to obtain the matrix identity p(A) = 0.
  5. Tactical choice: rearrange to isolate the highest power (A^3 = −aA^2 − bA − cI) so that substitution is mechanical, and decide whether the goal is verification (compute A^2, A^3, plug in) or reduction (recursively replace high powers).
  6. For A^n with n large, multiply the isolating relation by A and substitute repeatedly, keeping the running expression in the form αI + βA + γA^2.

Mechanic — Concrete operations

  1. Write the characteristic polynomial p(λ) = λ^3 + aλ^2 + bλ + c (or λ^2 − tr(A)λ + det(A) for 2×2).
  2. State Cayley-Hamilton: p(A) = A^3 + aA^2 + bA + cI = 0, with cI (not c) as the constant term.
  3. Rearrange to A^3 = −aA^2 − bA − cI.
  4. If verifying: compute A^2, then A^3 = A·A^2, substitute into p(A), and confirm it equals the zero matrix.
  5. If finding A^4 (or higher): multiply the rearranged identity by A to get A^4 = −aA^3 − bA^2 − cA, then substitute the expression for A^3 to land in span{I, A, A^2}.
  6. If finding A^{-1}: rearrange p(A) = 0 to I = −(1/c)(A^2 + aA + bI), so A^{-1} = −(1/c)(A^2 + aA + bI), provided c = (−1)^n det(A) ≠ 0.
  7. Compute A^2 once (the only matrix multiplication needed) and assemble the final αI + βA + γA^2.

Worked Micro-Example

Let A be 2×2 with tr(A) = 4 and det(A) = 3, so the characteristic equation is $\lambda^2 - 4\lambda + 3 = 0$. Cayley-Hamilton gives $A^2 - 4A + 3I = 0$, hence $A^2 = 4A - 3I$. Then $A^3 = A\cdot A^2 = 4A^2 - 3A = 4(4A - 3I) - 3A = 13A - 12I$, reducing a cubic in A to a linear combination of A and I.

Variants — Same move, different surface

2×2 matrix
Characteristic equation is quadratic: $A^2 = \operatorname{tr}(A)\,A - \det(A)\,I$. Every higher power reduces to αI + βA.
Finding A^{-1} via Cayley-Hamilton
Isolate I in p(A) = 0 and divide by the constant term: $A^{-1} = -\tfrac{1}{c}(A^2 + aA + bI)$ for 3×3. Avoids row-reduction.
Verifying p(A) = 0 directly
No reduction needed — just compute A^2 and A^3 explicitly and substitute, showing the result is the zero matrix.

Common Misreadings

  • Treating the question as requiring diagonalisation when the characteristic polynomial is already provided — a clear signal that Cayley-Hamilton is the intended route.
  • Reading 'satisfies its characteristic equation' as a scalar identity for the eigenvalues, rather than the matrix identity p(A) = 0.

Pitfalls

  • Forgetting that the constant term c must become cI (the identity matrix scaled), not the scalar c, when passing from p(λ) = 0 to p(A) = 0.
  • Sign errors when rearranging: the characteristic polynomial sign convention matters — write p(λ) = det(λI − A) carefully before substituting.
  • Trying to 'divide' matrices in the Cayley-Hamilton identity; only rearrangement and (for invertibility) multiplication by A^{-1} are legitimate.
  • Arithmetic slips when computing A^2 — every entry feeds the rest of the calculation, so check it before substituting.

A-level Scope Notes

Fully within Edexcel IAL Further Pure (FP1/FP2) for 2×2 and 3×3 matrices; the Cayley-Hamilton theorem and the characteristic equation are explicitly in scope. Do not generalise to minimal polynomials, Jordan forms, or n×n with n ≥ 4 — these are beyond IAL.
