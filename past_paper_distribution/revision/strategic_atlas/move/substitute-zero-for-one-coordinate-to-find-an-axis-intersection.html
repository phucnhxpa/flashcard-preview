Coordinate Geometry

Substitute zero for one coordinate to find an axis intersection

Family: Axis Intersection Frequency: 4 questions

Trigger — When to use

Fires whenever a question asks for the point where a line, tangent, normal, or curve 'meets', 'crosses', or 'cuts' a coordinate axis. Equivalent phrasings include 'the tangent meets the x-axis at R', 'find where the line crosses the y-axis', 'the curve cuts the y-axis at P', or being told a labelled point lies on an axis. The structural cue is that one coordinate of the unknown point is forced to zero by the geometry of the axis itself.

Motivation — Why this works

The y-axis is precisely the locus x = 0 and the x-axis is the locus y = 0, so finding an intersection with an axis is just solving the given equation simultaneously with x = 0 or y = 0. Substituting collapses a two-variable equation to a single-variable one, which is then trivial to solve. This is the standard — and essentially the only — technique for axis intersections in IAL coordinate geometry, and it works identically for lines, tangents, normals, and implicit curves.

Thought Process — The step-by-step reasoning

  1. Notice the question names a point that lies on a coordinate axis (e.g. 'meets the x-axis at R', 'crosses the y-axis at P').
  2. Recall that points on the x-axis have y = 0 and points on the y-axis have x = 0 — so one coordinate of the unknown point is already determined by the geometry.
  3. Rule out alternative methods (rewriting the line as y = mx + c just to read off c, or solving a system by elimination) — direct substitution is faster and works for any form of the equation, including implicit curves.
  4. Commit to substituting the known zero coordinate into the given equation, knowing this reduces it to one variable.
  5. Carry out the substitution and solve; if the resulting equation is a polynomial with multiple roots (as for a curve), discard any root that corresponds to a previously-known intersection (e.g. the origin) to isolate the requested point.
  6. Write the final point as an ordered pair, double-checking which coordinate was set to zero matches the axis named in the question.

Mechanic — Concrete operations

  1. Identify the axis: y-axis ⇒ set x = 0; x-axis ⇒ set y = 0.
  2. Substitute the chosen value into the equation of the line/curve.
  3. Simplify to obtain a single-variable equation in the remaining coordinate.
  4. Solve (linear ⇒ direct; quadratic/polynomial ⇒ factorise and select the relevant root).
  5. Write the intersection as an ordered pair (0, y₀) for the y-axis or (x₀, 0) for the x-axis.
  6. Sanity-check signs and any restrictions (e.g. t ≠ 0 if division by a parameter occurred).

Worked Micro-Example

Find where the tangent $ty = x + at^2$ meets the x-axis. On the x-axis $y=0$, so $t(0) = x + at^2 \Rightarrow x = -at^2$. Hence $R = (-at^2,\,0)$.

Variants — Same move, different surface

Line in general form, e.g. 3x + 4y − 27 = 0, asked for y-axis intercept
Set x = 0 directly: 4y = 27 ⇒ y = 27/4, so intercept is (0, 27/4).
Implicit curve such as y² = ye^{−2x} − 3x crossing the y-axis
Set x = 0 to get a polynomial in y (here y² = y); factorise and discard the already-known intersection (origin) to isolate the new point.
Tangent or normal to a conic (parabola, ellipse, hyperbola) meeting an axis
Identical substitution; the algebra is heavier but the strategic move is unchanged.
Intersection with the directrix x = −a of a parabola
Substitute x = −a (instead of 0) into the line/curve equation — same substitution principle, different constant.

Common Misreadings

  • Confusing 'meets the y-axis' with 'meets the x-axis' and setting the wrong coordinate to zero.
  • Treating an axis intersection as requiring simultaneous solution of two general equations rather than a single substitution.

Pitfalls

  • Dividing through by a parameter (e.g. t) without recording the assumption t ≠ 0; the degenerate case may need separate comment.
  • Dividing both sides by a variable (e.g. y) and silently losing the y = 0 solution when both roots are required.
  • Sign errors when rearranging, e.g. writing x = at² instead of x = −at² after moving a term across the equals sign.
  • Quoting only the non-zero coordinate and forgetting to write the answer as an ordered pair.

A-level Scope Notes

Fully within Edexcel IAL Pure (P1–P4) coordinate geometry of lines, parabolas, ellipses and hyperbolas. No formula-booklet entry is invoked — this is a foundational technique. Avoid generalising to oblique 'axes' or to 3D coordinate intersections, which are out of scope.
