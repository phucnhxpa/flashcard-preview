Algebra & Functions

Factor common x to expose and discard trivial root

Family: Root Selection Frequency: 1 questions

Trigger — When to use

Fires when an equation set to zero has every term sharing a common factor of x (or a simple linear factor), typically mixing a polynomial factor with a transcendental piece such as e^{kx}, sin x, ln x or a power. Cues include forms like ax - x f(x) = 0, x g(x) = h(x) rearranged to x(g(x) - h(x)/x) = 0, or a stationary-point condition where dy/dx factorises. The move is especially indicated when the problem rules out the origin (e.g. 'A is not the origin', 'x > 0') so the trivial root must be visibly identified and rejected.

Motivation — Why this works

Setting a product to zero invokes the null-factor law: x(4 - e^{x/2}) = 0 means x = 0 or 4 - e^{x/2} = 0, with no information lost. Dividing by x instead silently discards the root x = 0 and forces an extra justification clause. Factoring keeps the logical structure transparent for the mark scheme and isolates the genuinely interesting equation 4 - e^{x/2} = 0, which is a standard log-solvable exponential equation.

Thought Process — The step-by-step reasoning

  1. Notice every term contains a factor of x, so the LHS is not yet in product form but easily can be.
  2. Rule out dividing by x: it loses the trivial root and would require an explicit x ≠ 0 justification.
  3. Commit to factoring because the null-factor law gives a clean two-case split that mirrors the mark scheme.
  4. Bridge: write x · (remaining factor) = 0 and immediately read off the two cases.
  5. Tactically, check the problem context (e.g. 'not the origin', x > 0) to justify discarding x = 0, then solve the surviving factor with logs.

Mechanic — Concrete operations

  1. Identify x as a common factor of all terms.
  2. Rewrite the equation as x · (remaining expression) = 0.
  3. Apply the null-factor law to split into x = 0 and remaining = 0.
  4. Discard x = 0 using the stated geometric/contextual restriction.
  5. Solve the surviving equation (typically by taking ln of both sides for an exponential factor).
  6. Simplify the resulting logarithm into its tidiest form, e.g. ln 4 = 2 ln 2.

Worked Micro-Example

Solve $4x - x e^{x/2} = 0$ for $x \neq 0$.\\ Factor: $x(4 - e^{x/2}) = 0$.\\ Reject $x = 0$, so $e^{x/2} = 4$.\\ Take logs: $x/2 = \ln 4$, hence $x = 2\ln 4 = 4\ln 2$.

Variants — Same move, different surface

Stationary points of y = x e^{-x}: dy/dx = e^{-x}(1 - x)
Factor out e^{-x} (always nonzero) instead, leaving 1 - x = 0; the 'discard' step is automatic since the exponential never vanishes.
Equation x sin x = 0 on a bounded interval
Null-factor gives x = 0 or sin x = 0; keep or reject x = 0 according to the stated domain.
x ln x = x in an integration-by-parts boundary
Rearrange to x(ln x - 1) = 0; reject x = 0 since ln x is undefined there, leaving x = e.
Polynomial-only case x^3 - 4x = 0
Factor x(x^2 - 4) = 0 to get x = 0, ±2; same null-factor reasoning, no logs needed.
Parametric dy/dx = 0 giving t(2 - e^t) = 0
Same factoring move on the parameter t; reject t = 0 if it corresponds to an excluded point on the curve.

Common Misreadings

  • Treating 4x - x e^{x/2} = 0 as a transcendental equation requiring numerical methods, missing the algebraic factor.
  • Reading 'A is not the origin' as a hint to ignore x = 0 silently, rather than as justification to formally discard it after factoring.
  • Believing dividing by x and factoring are equivalent moves with no logical difference.

Pitfalls

  • Dividing both sides by x and forgetting to check x = 0 separately, losing a root.
  • Leaving the answer as x = 2 ln 4 instead of simplifying to 4 ln 2.
  • Forgetting the factor of 1/2 when taking logs of e^{x/2}, writing x = ln 4 instead of x = 2 ln 4.
  • Discarding the wrong factor when both roots are admissible (e.g. not checking the contextual restriction).

A-level Scope Notes

Entirely within Edexcel IAL Pure scope: null-factor law, laws of logarithms, and ln/e^x manipulation from C2/C3. The move recurs in stationary-point and intersection problems throughout Pure and Further Pure.
