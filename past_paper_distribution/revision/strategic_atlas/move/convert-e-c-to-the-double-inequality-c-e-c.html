Convert |E| < c to the double inequality −c < E < c

Family: Modulus Inequality Conversion Frequency: 1 questions

Trigger — When to use

Fires whenever a strict or weak modulus inequality of the form |E(x)| < c, |E(x)| ≤ c, or a rearrangement like |E(x)| − c < 0 appears with c a positive constant. Typical surface cues are a single modulus term bounded above by a number, often after a trivial rearrangement. Common phrasings: 'solve |2x − 5| ≤ 3', 'find the set of values of x for which |kx − 9| − 2 < 0', or 'solve |E| < c'. Distinguish from |E| > c, which requires a different (union) treatment.

Motivation — Why this works

For c > 0, |E| < c holds iff −c < E < c, because the modulus measures distance from 0 on the real line and 'distance less than c' is exactly the open interval (−c, c). This identity replaces a case-split with a single compound linear inequality that can be solved by chained operations. It is faster and less error-prone than squaring or splitting into two cases, and it generalises cleanly to ≤ by closing the interval. It also avoids introducing extraneous solutions that squaring can produce when E is not guaranteed non-negative.

Thought Process — The step-by-step reasoning

  1. Notice the inequality reduces, after any obvious rearrangement, to |E(x)| < c with c a positive constant.
  2. Rule out squaring (overkill for linear E) and case-splitting (correct but verbose) since the compound form gives the answer in one chain.
  3. Commit to the equivalence |E| < c ⇔ −c < E < c because c > 0 is clear and E is a single linear expression.
  4. Bridge to the first operation: write the double inequality and prepare to add/subtract constants and divide by the coefficient of x.
  5. Decide the order of operations (constants first, then divide), and check the sign of any divisor before dividing to know whether to flip.

Mechanic — Concrete operations

  1. If needed, rearrange to isolate the modulus on one side: |E(x)| < c.
  2. Confirm c > 0; otherwise the inequality has no solution (for < 0) or trivial behaviour.
  3. Replace with the compound inequality −c < E(x) < c.
  4. Apply the same operation across all three parts to isolate x (add/subtract constants, then divide by the coefficient of x).
  5. If dividing by a negative quantity, reverse both inequality signs; state the final solution as an interval or in set notation.

Worked Micro-Example

Solve $|kx - 9| - 2 < 0$ given $k > 0$.\ Rearrange: $|kx - 9| < 2$.\ Double inequality: $-2 < kx - 9 < 2$.\ Add 9: $7 < kx < 11$. Divide by $k>0$: $\dfrac{7}{k} < x < \dfrac{11}{k}$.

Variants — Same move, different surface

|E(x)| ≤ c with c > 0
Use closed inequalities: −c ≤ E(x) ≤ c, giving a closed interval for x.
|E(x)| > c (or ≥ c)
Split into a union: E(x) > c OR E(x) < −c; the solution is a union of two intervals, not an intersection.
|ax + b| < c with a < 0
Apply the double inequality, then when dividing by a reverse both inequality signs.
|E(x)| < g(x) with g non-constant
Write −g(x) < E(x) < g(x) and additionally require g(x) > 0; often cleaner to square (since both sides non-negative) to get E(x)^2 < g(x)^2.
|f(x)| < |g(x)|
Square both sides to get f(x)^2 < g(x)^2, then factor as a difference of squares (compound form does not directly apply).

Common Misreadings

  • Treating |E| − c < 0 as already in standard form and forgetting to isolate the modulus first.
  • Applying the double-inequality form to |E| > c, producing an incorrect intersection instead of a union.
  • Assuming c > 0 without checking — if c ≤ 0 the inequality |E| < c has no solutions.

Pitfalls

  • Dividing the compound inequality by a negative coefficient without flipping both inequality signs.
  • Performing different operations on different parts of the chain (e.g. adding to only the middle).
  • Mixing strict and non-strict signs when transcribing |E| ≤ c versus |E| < c.
  • Forgetting to state any domain restriction on parameters (e.g. 'k > 0 given') that justifies sign-preserving division.

A-level Scope Notes

Standard Edexcel IAL Pure (modulus function in P3): solving linear modulus inequalities of the form |ax+b| < c is examinable, including parameter-laden cases like |kx − 9| < 2.
