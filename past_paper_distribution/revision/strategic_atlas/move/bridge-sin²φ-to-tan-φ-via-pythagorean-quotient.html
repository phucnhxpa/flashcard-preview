Trigonometry

Bridge sin²φ to tan φ via Pythagorean quotient

Family: Ratio Bridge Frequency: 1 questions

Trigger — When to use

Fires when an equation yields sin²φ = k (or cos²φ = k) but the question asks for tan φ in exact form, often in a 'show that' or 'hence find' part. Cues include exact fractional values like 1/3, 2/5, etc., and a stated domain such as 0 < φ < π/2 that fixes the sign. Also fires when arcsin would produce an inexact angle that must be avoided. The move replaces 'find φ then take tan' with a direct identity-based bridge.

Motivation — Why this works

The Pythagorean identity sin²φ + cos²φ = 1 lets cos²φ be recovered immediately from sin²φ without finding φ. Then tan²φ = sin²φ/cos²φ gives tan φ exactly after a single square root. This preserves surd form, which is essential for show-that marks, and sidesteps calculator approximations. The domain restriction supplies the sign when extracting the root.

Thought Process — The step-by-step reasoning

  1. I notice the result is sin²φ (a squared ratio), and the target is tan φ — both are linked through cos²φ.
  2. I rule out solving sin φ = ±√(1/3) and then computing φ via arcsin — that gives a non-exact angle and loses surd form.
  3. I commit to the identity route because sin²φ → cos²φ → tan²φ is purely algebraic and preserves exactness.
  4. First concrete step: apply cos²φ = 1 − sin²φ to get cos²φ in closed form.
  5. Inside the move: divide to get tan²φ, then use the stated domain (e.g. 0 < φ < π/2) to justify the positive square root.

Mechanic — Concrete operations

  1. From sin²φ = k, write cos²φ = 1 − k.
  2. Form tan²φ = sin²φ / cos²φ = k / (1 − k).
  3. Take the square root: tan φ = ±√(k/(1−k)).
  4. Use the given domain on φ to fix the sign of tan φ.
  5. Simplify the surd (rationalise denominator if required).

Worked Micro-Example

Given $\sin^2\varphi = \tfrac{1}{3}$ with $0 < \varphi < \tfrac{\pi}{2}$. Then $\cos^2\varphi = 1 - \tfrac{1}{3} = \tfrac{2}{3}$, so $\tan^2\varphi = \dfrac{1/3}{2/3} = \tfrac{1}{2}$. Since $\varphi$ is acute, $\tan\varphi > 0$, giving $\tan\varphi = \dfrac{1}{\sqrt{2}} = \dfrac{\sqrt{2}}{2}$.

Variants — Same move, different surface

Given cos²φ = k, find tan φ
Use sin²φ = 1 − k, then tan²φ = (1−k)/k.
Given tan²φ = k, find sin φ or cos φ
Use sec²φ = 1 + tan²φ to get cos²φ = 1/(1+k), then sin²φ = k/(1+k).
Domain extends beyond acute (e.g. π/2 < φ < π)
Sign of tan φ flips; choose the negative root based on quadrant.
Result expressed as sin φ cos φ or sin 2φ
Use sin 2φ = 2 sin φ cos φ with sin²φ and cos²φ already known, taking positive roots per quadrant.

Common Misreadings

  • Treating sin²φ = 1/3 as sin φ = 1/3 and computing tan φ from a 1-3-? right triangle.
  • Believing one must find φ explicitly before computing tan φ.
  • Assuming tan φ is automatically positive without invoking the stated domain.

Pitfalls

  • Forgetting to justify the sign when taking the square root of tan²φ.
  • Arithmetic slip in the quotient (1/3) ÷ (2/3), e.g. writing 2/9 instead of 1/2.
  • Leaving the answer as 1/√2 when the mark scheme expects √2/2 (or vice versa) — match the requested form.
  • Mis-applying the identity as sin²φ − cos²φ = 1.

A-level Scope Notes

Uses only the Pythagorean identity sin²φ + cos²φ = 1 and the definition tan φ = sin φ / cos φ, both core to Edexcel IAL Pure Mathematics (P2/P3). No further-trig results are required.
