Differentiation

Differentiate curve and substitute x-coordinate to obtain tangent gradient

Family: Derivative As Gradient Frequency: 3 questions

Trigger — When to use

This move fires whenever a question asks for the equation of a tangent (or normal) to a curve y = f(x) at a specific point P(x₁, y₁). Typical phrasings include 'Find the equation of the tangent to C at P', 'Show that the tangent at the point where x = a has equation...', or 'Find the gradient of C at the point P'. The structural cue is: a curve equation is given, a specific point on it is named, and a line property (tangent/normal/gradient) at that point is requested. Recognise this also when part (a) of a question asks for dy/dx and part (b) asks for a tangent — the parts are linked deliberately.

Motivation — Why this works

The derivative dy/dx is, by geometric definition, the gradient of the tangent to the curve at any point. Evaluating dy/dx at x = x₁ converts the general gradient function into the single numerical gradient m needed to write the tangent line. Together with the point (x₁, y₁) on the curve, this fully specifies the tangent via y − y₁ = m(x − x₁). The underlying principle is the identification of the analytic derivative with the geometric tangent slope — the very definition of differentiability.

Thought Process — The step-by-step reasoning

  1. Notice that to write any straight line equation I need two ingredients: a point and a gradient. The point is given (or recoverable by substituting x₁ into y); the gradient is what calculus must supply.
  2. Rule out alternatives: a chord/secant gradient is not what's wanted; the limit definition is correct but inefficient for a polynomial; the discriminant-equals-zero method (forcing a repeated root) is valid but far slower. For a polynomial/standard curve, the power rule is the efficient route.
  3. Commit to the strategy: differentiate y with respect to x to get the gradient function dy/dx, then substitute x = x₁ to collapse it to a single number m.
  4. Bridge to operation: apply the power rule term-by-term (or implicit/parametric differentiation if the curve is given in those forms).
  5. Tactical decision: keep the gradient in exact form (e.g. 59/4 rather than 14.75) since downstream the tangent equation is usually expected in exact integer or fractional form.
  6. Distinguish tangent vs normal: if the question asks for a normal, take m_normal = −1/m_tangent AFTER computing m_tangent the same way.

Mechanic — Concrete operations

  1. Identify the curve y = f(x) and the x-coordinate x₁ of the point of contact.
  2. Differentiate term-by-term using the power rule: if y = Σ a_n x^n then dy/dx = Σ n·a_n x^{n−1}.
  3. Substitute x = x₁ into dy/dx to obtain the numerical tangent gradient m = f'(x₁).
  4. Simplify m to exact form (fraction, not decimal).
  5. (Optional) Use m with the point (x₁, y₁) in y − y₁ = m(x − x₁) to write the tangent equation.

Worked Micro-Example

Curve: $y = x^3 - 6x + 5$ at $P(2, 1)$. Differentiate: $\frac{dy}{dx} = 3x^2 - 6$. Substitute $x = 2$: $m = 3(4) - 6 = 6$. Tangent: $y - 1 = 6(x - 2)$, i.e. $y = 6x - 11$.

Variants — Same move, different surface

Curve given implicitly, e.g. x² + y² = 25 at (3, 4)
Use implicit differentiation: 2x + 2y(dy/dx) = 0, then substitute (x, y) = (3, 4) to get dy/dx = −3/4.
Curve given parametrically, x = f(t), y = g(t)
Compute dy/dx = (dy/dt)/(dx/dt), then evaluate at the parameter value t corresponding to the point.
Question asks for the normal instead of the tangent
Compute m_tangent the same way, then take m_normal = −1/m_tangent before writing the line equation.
Negative or fractional powers in y, e.g. y = x² + 16x⁻² − 24x⁻¹
Apply the power rule carefully to negative exponents (n·x^{n−1} with n < 0); double-check arithmetic such as 4⁻³ = 1/64.

Common Misreadings

  • Substituting x = x₁ into the original equation y = f(x) (which gives the y-coordinate, not the gradient) instead of into the derivative.
  • Confusing tangent gradient with normal gradient (using −1/m when m is what's wanted, or vice versa).
  • Treating the gradient of a chord between two nearby points as the tangent gradient.

Pitfalls

  • Forgetting to multiply by the original power when applying the power rule (e.g. writing 2x² instead of 6x² for d/dx(2x³)).
  • Differentiating a constant to a non-zero value.
  • Arithmetic slips with negative exponents: e.g. 4⁻³ = 1/64, not 1/12.
  • Leaving the gradient as a decimal when exact (fractional) form is expected.
  • Forgetting to evaluate at x = x₁ and leaving the answer as a function of x.
  • Sign errors when substituting, especially with subtractions like 2(2) − 6.

A-level Scope Notes

Fully within Edexcel IAL P1/P2/P3 (and P4 for parametric/implicit). Uses the power rule (formula booklet, differentiation section) and, where relevant, implicit/parametric differentiation from P4. Avoid out-of-scope generalisations such as differentiating non-elementary functions or using formal ε-δ limits — the geometric interpretation 'derivative = tangent gradient' is sufficient at this level.
