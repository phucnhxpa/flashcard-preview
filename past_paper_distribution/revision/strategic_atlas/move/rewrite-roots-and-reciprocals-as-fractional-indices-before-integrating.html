Integration

Rewrite roots and reciprocals as fractional indices before integrating

Family: Index Law Preprocessing Frequency: 1 questions

Trigger — When to use

Fires whenever the integrand contains surds (√x, ∛x), reciprocal powers (1/x², 1/x^(3/2)), or roots in denominators (k/√x) that are not yet expressed as ax^n. Cue phrases: 'integrate f'(x)', 'find ∫... dx', or 'find f(x) given f'(x) = ...' where any term is written with a root sign or as a fraction with x in the denominator. Also fires on mixed expressions like (x+1)/√x that must first be split and re-expressed as a sum of powers.

Motivation — Why this works

The power rule ∫ax^n dx = ax^(n+1)/(n+1) + c (for n ≠ −1) only operates on terms in the canonical form ax^n with an explicit numerical exponent. Surds and reciprocals are algebraically equivalent to fractional/negative powers via the index laws 1/x^a = x^(−a) and ⁿ√x = x^(1/n). Rewriting in index form unlocks the rule mechanically, term by term. Without this preprocessing the integral cannot be evaluated by inspection in the IAL toolkit.

Thought Process — The step-by-step reasoning

  1. I notice the integrand contains √x or 1/√x or 1/x² — not yet in the form ax^n.
  2. I rule out trying to integrate it as written: the power rule needs an explicit exponent, and ln|x| only applies if n = −1 exactly.
  3. I commit to a preprocessing pass: convert every term to ax^n using index laws before any integration is attempted.
  4. Concretely, I apply 1/√x = x^(−1/2), √x = x^(1/2), 1/x^k = x^(−k), keeping coefficients attached.
  5. I then check each resulting exponent: if any equals −1, I would switch to ln; otherwise the power rule applies uniformly.

Mechanic — Concrete operations

  1. Identify every non-power-form term (surds, roots, reciprocals of x).
  2. Apply index laws: √x → x^(1/2), ∛x → x^(1/3), 1/√x → x^(−1/2), 1/x^k → x^(−k).
  3. Keep numerical coefficients attached: e.g., 16/√x → 16x^(−1/2).
  4. Rewrite the full integrand as a sum of ax^n terms.
  5. Apply the power rule term by term: ∫ax^n dx = ax^(n+1)/(n+1) + c (checking n ≠ −1).
  6. Optionally convert the answer back to surd form if the question's notation demands it.

Worked Micro-Example

Given f'(x) = \tfrac{1}{4}x^3 - \tfrac{16}{\sqrt{x}}, rewrite as f'(x) = \tfrac{1}{4}x^3 - 16x^{-1/2}. Then f(x) = \tfrac{1}{4} \cdot \tfrac{x^4}{4} - 16 \cdot \tfrac{x^{1/2}}{1/2} + c = \tfrac{x^4}{16} - 32x^{1/2} + c = \tfrac{x^4}{16} - 32\sqrt{x} + c.

Variants — Same move, different surface

Cube roots or higher: integrand contains ∛x or 1/∛x
Use ∛x = x^(1/3) and 1/∛x = x^(−1/3); then power rule gives x^(4/3)/(4/3) or x^(2/3)/(2/3).
Reciprocal powers like 5/x³
Rewrite as 5x^(−3); integral becomes 5x^(−2)/(−2) = −5/(2x²) + c.
Mixed numerator over √x, e.g. (x² + 3)/√x
Split first: x²/√x + 3/√x = x^(3/2) + 3x^(−1/2), then integrate term by term.
Differentiation problems with the same surface
Identical preprocessing: rewrite as x^n then apply d/dx(x^n) = nx^(n−1).
Definite integrals with surds
Same preprocessing, then evaluate [F(x)] between limits; convert back to surd form before substituting if cleaner.

Common Misreadings

  • Treating ∫1/√x dx as a logarithmic integral by analogy with ∫1/x dx — but n = −1/2 ≠ −1 so the power rule applies and gives 2√x + c.
  • Reading 1/√x as having exponent +1/2 instead of −1/2 because the √ is 'positive-looking'.
  • Mistaking √x in a denominator for something requiring substitution, when index conversion alone suffices.

Pitfalls

  • Sign error on the exponent: writing 1/√x as x^(1/2) instead of x^(−1/2).
  • Dropping or mis-multiplying the coefficient when converting (e.g., 16/√x → x^(−1/2) without the 16).
  • After integrating x^(−1/2), forgetting that dividing by 1/2 means multiplying by 2 — common slip giving (1/2)x^(1/2) instead of 2x^(1/2).
  • Failing to split a fraction with multiple terms in the numerator before converting (e.g., trying to integrate (x+1)/√x as a single term).

A-level Scope Notes

Core skill in Edexcel IAL P1/P2 integration; relies only on index laws and the standard power rule, with no substitution or non-elementary techniques required.
