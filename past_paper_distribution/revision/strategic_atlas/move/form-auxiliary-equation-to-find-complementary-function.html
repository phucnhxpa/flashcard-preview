Trigger — When to use

Fires when you face a homogeneous second-order linear ODE with constant coefficients, written as ay'' + by' + cy = 0 with a, b, c numbers (not functions of x). Also fires as the first stage of solving a non-homogeneous equation ay'' + by' + cy = f(x), where you must build the complementary function before adding a particular integral. Look for phrases like 'find the general solution' or 'find the complementary function' attached to a constant-coefficient ODE. The cue is purely structural: every coefficient must be a constant.

Motivation — Why this works

The trial y = e^{mx} satisfies y' = me^{mx} and y'' = m²e^{mx}, so substitution turns each term into a constant multiple of e^{mx}. Since e^{mx} ≠ 0 it cancels, converting the ODE into the algebraic quadratic am² + bm + c = 0. The two roots give two linearly independent solutions, and by linearity their general combination is the full solution space of the homogeneous equation. This reduces a calculus problem to A-level quadratic solving.

Thought Process — Step-by-step reasoning

  1. Notice the ODE is linear, second-order, homogeneous (or homogeneous part of a non-homogeneous one), with constant numerical coefficients in front of y'', y', y.
  2. Rule out integrating factor (that is for first-order linear) and rule out separation of variables (the equation mixes y and its derivatives linearly).
  3. Commit to the auxiliary equation route because constant coefficients guarantee y = e^{mx} is a viable trial — exponentials are eigenfunctions of d/dx.
  4. Bridge to operation: read off a, b, c carefully (watch signs and rearrange so the equation equals zero), then write down am² + bm + c = 0.
  5. Tactically decide how to solve: factorise if obvious, otherwise quadratic formula; then classify the discriminant to pick the correct CF template (distinct real, repeated, or complex conjugate).

Mechanic — Concrete operations

  1. Rearrange the ODE into standard form ay'' + by' + cy = 0 and identify a, b, c.
  2. Replace y'' → m², y' → m, y → 1 to obtain the auxiliary equation am² + bm + c = 0.
  3. Solve the quadratic by factorisation, formula, or completing the square; compute the discriminant if classification is unclear.
  4. Write the complementary function using the matching template: distinct real roots m₁, m₂ → y = Ae^{m₁x} + Be^{m₂x}; repeated root m → y = (A + Bx)e^{mx}; complex roots p ± qi → y = e^{px}(A cos qx + B sin qx).
  5. If the original problem was non-homogeneous, keep this CF aside to add to a particular integral later.

Worked Micro-Example

Solve y'' - 5y' + 6y = 0. Auxiliary equation: m² - 5m + 6 = 0, so (m-2)(m-3) = 0, giving m = 2 or m = 3. Both real and distinct, so the general solution is y = Ae^{2x} + Be^{3x}.

Variants — Same move, different surface

Repeated root, e.g. y'' - 4y' + 4y = 0
Auxiliary (m-2)² = 0 has double root m = 2; CF must include the x factor: y = (A + Bx)e^{2x}.
Complex conjugate roots, e.g. y'' + 4y = 0
m² + 4 = 0 gives m = ±2i, so use the real form y = A cos 2x + B sin 2x (no exponential prefactor since real part is 0).
Non-homogeneous ODE ay'' + by' + cy = f(x)
Use the auxiliary equation only on the homogeneous part to get the CF, then find a particular integral and form y = CF + PI.
Equation given in non-standard form, e.g. 2y'' = 3y' - y
Rearrange to 2y'' - 3y' + y = 0 first, then form 2m² - 3m + 1 = 0.

Common Misreadings

  • Treating a second-order constant-coefficient ODE as first-order and trying an integrating factor.
  • Applying the auxiliary equation to an ODE whose coefficients depend on x (e.g. xy'' + y' + y = 0), where the method does not work directly.
  • Mistaking a non-homogeneous equation's full solution for just the CF, forgetting the particular integral.

Pitfalls

  • Sign errors when copying coefficients into the auxiliary equation, particularly when the ODE was rearranged.
  • Forgetting the Bx factor when the auxiliary equation has a repeated root.
  • Leaving complex-root solutions in the form e^{(p+qi)x}; examiners require the real form e^{px}(A cos qx + B sin qx).
  • Using the same constant for both terms (e.g. writing Ae^{m₁x} + Ae^{m₂x}) instead of two independent constants A and B.

A-level Scope Notes

This is core content in Edexcel IAL Further Pure (FP2/FP3 depending on specification) for second-order linear ODEs with constant coefficients; higher-order or variable-coefficient cases are out of scope.
