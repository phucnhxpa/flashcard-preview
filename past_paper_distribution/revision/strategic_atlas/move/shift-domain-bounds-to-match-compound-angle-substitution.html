Trigonometry

Shift domain bounds to match compound-angle substitution

Family: Range Transformation Frequency: 1 questions

Trigger — When to use

The equation involves a compound angle such as sin(θ+43°), cos(2θ−30°), or tan(θ/2+α), but the stated domain is given on θ. Before solving you intend to substitute X for the compound argument and read off solutions from the standard sine/cosine wave. The cue is any mismatch between the variable inside the trig function and the variable whose range is specified. The move fires the moment you write 'let X = …' and need to know which X-values are admissible.

Motivation — Why this works

Trig equations are easiest to enumerate when the unknown sits alone inside one trig function, e.g. sin X = 1/2. But the original constraint is on θ, so X inherits a shifted (and possibly stretched) interval. Translating the inequality preserves a one-to-one correspondence between admissible X and admissible θ, ensuring every θ-solution is captured and no spurious ones outside the θ-range slip in. The principle is simply that linear substitutions map intervals to intervals by applying the same operations to the endpoints.

Thought Process — The step-by-step reasoning

  1. I notice the trig argument is θ+43° while the domain is on θ — these don't match.
  2. I rule out solving sin X = 1/2 on a default 0°–360° window because that would silently change the problem.
  3. I commit to the substitution X = θ+43° and decide to carry the domain across with it.
  4. Concretely, I add 43° to every part of −180° < θ ≤ 180° to get −137° < X ≤ 223°.
  5. I keep the strict/non-strict inequalities exactly as given, then enumerate X-solutions in this shifted window before subtracting 43° to recover θ.

Mechanic — Concrete operations

  1. Define X equal to the compound argument, e.g. X = θ + 43°.
  2. Apply the same linear operation to each part of the θ-inequality to obtain the X-inequality.
  3. Preserve strict (<) versus non-strict (≤) signs exactly.
  4. Solve the simplified equation sin X = k (or cos/tan) for all X in the new interval using the principal value plus standard symmetry rules.
  5. Reverse the substitution (θ = X − 43°) to list the θ-solutions, checking each lies in the original θ-range.

Variants — Same move, different surface

Multiplier inside the argument, e.g. cos(2θ − 30°) = k on 0° ≤ θ ≤ 360°.
Let X = 2θ − 30°; double each θ-bound then subtract 30° to get −30° ≤ X ≤ 690°, a wider window that will contain more solutions.
Half-angle substitution, e.g. tan(θ/2 + 15°) = k on 0° ≤ θ < 360°.
Let X = θ/2 + 15°; halve each bound then add 15° to obtain 15° ≤ X < 195°, a narrower window.
Radian domain, e.g. sin(x − π/6) = k on 0 ≤ x ≤ 2π.
Let X = x − π/6; subtract π/6 from each bound to get −π/6 ≤ X ≤ 11π/6, working entirely in radians.
After Rsin(θ+α) form from harmonic addition.
Same shift principle: once the equation is reduced to sin(θ+α)=c/R, translate the θ-range by +α to get the X-range.

Common Misreadings

  • Treating the compound argument as if it were θ and solving on the original θ-range, then forgetting to subtract the shift.
  • Assuming the X-range is the 'standard' 0°–360° window regardless of what the question states.
  • Confusing a shift with a stretch — applying the multiplier to only one endpoint when the argument is 2θ+α.

Pitfalls

  • Forgetting to shift the range and solving in the original θ-range, which gives wrong X boundaries.
  • Mishandling strict vs non-strict inequalities at the endpoints (e.g. writing ≤ when the original was <).
  • Dropping solutions near the new boundaries because the shifted interval is non-standard (negative start, or extending past 360°).
  • Failing to subtract the shift at the end, leaving answers as X-values rather than θ-values.

A-level Scope Notes

Standard Edexcel IAL P2/P3 trig-equation technique; only linear substitutions X = aθ+b appear at this level, and answers are listed exhaustively within the given range in degrees or radians.
