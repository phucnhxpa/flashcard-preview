Algebra & Functions

Locate critical values and test sign on each interval

Family: Sign Analysis Frequency: 1 questions

Trigger — When to use

Fires when solving an inequality involving a rational or factorised polynomial expression of the form f(x)/g(x) > 0 (or <, ≥, ≤) where the sign depends on signs of factors. Typical cues include a quotient with linear or quadratic numerator and denominator, or a factored polynomial inequality such as (x−a)(x−b)(x−c) ≥ 0. Also triggered after rearranging an inequality to one side so it reads expression > 0 or expression < 0. Multiplying across by an unknown-sign denominator is unsafe, which forces this method.

Motivation — Why this works

A product or quotient of real factors can only change sign where one of its factors is zero or undefined; between consecutive such points the sign is constant by continuity. So the real line splits into finitely many intervals on which a single test value determines the sign throughout. This converts the inequality into a finite case-check, avoiding the pitfall of multiplying both sides by an expression whose sign is unknown.

Thought Process — The step-by-step reasoning

  1. Notices the inequality is a quotient (or product) compared to zero, so direct algebraic manipulation by cross-multiplying is unsafe.
  2. Rules out multiplying through by x (or any factor) because its sign is not fixed; rules out squaring as it loses sign information.
  3. Commits to sign analysis because the expression is already (or easily made) a product/quotient of factors whose individual signs are easy to track.
  4. Bridges to action by setting each factor of numerator and denominator to zero to list all critical values on the number line.
  5. Tactically picks the simplest test value in each interval (e.g. integers far from boundaries) and decides which boundary points to include based on whether the inequality is strict and whether the point is in the domain.

Mechanic — Concrete operations

  1. Rearrange the inequality so one side is 0, with the other side a single rational/factored expression.
  2. Find all critical values: solve numerator = 0 and denominator = 0.
  3. Mark critical values on a number line; they partition ℝ into open intervals.
  4. Choose a convenient test point in each interval and evaluate the expression's sign.
  5. Select the intervals matching the required sign.
  6. Decide boundary inclusion: include numerator zeros if inequality is non-strict; always exclude denominator zeros.

Worked Micro-Example

Solve $(3-4x)/x > 0$. Critical values: $x = 3/4$ (numerator) and $x = 0$ (denominator). Test $x=-1$: $(7)/(-1) = -7 < 0$. Test $x = 1/2$: $(1)/(1/2) = 2 > 0$. Test $x = 1$: $(-1)/1 = -1 < 0$. Hence the solution is $0 < x < 3/4$.

Variants — Same move, different surface

Polynomial inequality $(x-1)(x-2)(x-5) \ge 0$
Critical values 1, 2, 5 give four intervals; include endpoints since inequality is non-strict, and test signs of the product directly.
Quadratic-over-linear, e.g. $(x^2-1)/(x-3) < 0$
Factor numerator to get critical values $\pm 1$ and $3$, giving four intervals; exclude $x=3$.
Quadratic inequality $x^2 - x - 6 < 0$
Factor to $(x-3)(x+2)<0$; two critical values, three intervals — equivalent to reading off between the roots of an upward parabola.
Sign diagram presentation
Replace test points with a +/− table whose rows are each factor and whose bottom row multiplies the signs; same logic, neater layout.

Common Misreadings

  • Treating the denominator's zero as just another root and including it in the solution.
  • Multiplying both sides by $x$ (or another factor) without case-splitting on its sign, producing a wrong direction of inequality.
  • Reading $(3-4x)/x > 0$ as equivalent to $3 - 4x > 0$, dropping the denominator entirely.

Pitfalls

  • Including $x = 0$ or $x = 3/4$ in the final answer when the inequality is strict.
  • Omitting an interval (often the leftmost or rightmost) and so missing part of the solution set.
  • Testing a value that happens to be a critical value, giving 0 and no sign information.
  • Forgetting to flip the inequality sign in equivalent algebraic manipulations done alongside the sign chart.

A-level Scope Notes

Standard technique for rational and polynomial inequalities in Edexcel IAL Pure Mathematics (P2/P3); all factors encountered are linear or easily factorable quadratics, and sketches/sign tables are accepted methods.
