Audit note

Repeatedly students must set the compound argument Bx+C equal to 0, π/2, π, 3π/2 (or 0°, 90°, 180°, 270°) to find x-coordinates of maxima, minima, and zeros; this is distinct from general solution enumeration and from transformation-based point mapping.

Trigger — When to use

Use this to find the $x$-coordinate of a maximum, minimum or zero of $y = A\,\mathrm{trig}(Bx + C) + D$ (or $\mathrm{trig}(x/12)$, $\cos(2x)$, etc.). Recognition cue: a key point is wanted on a stretched/shifted trig graph and you know what the inner function does at its standard angles.

Motivation — Why this works

$\sin$ peaks at argument $\frac{\pi}{2}$, troughs at $\frac{3\pi}{2}$, and is zero at $0,\pi,2\pi$; $\cos$ peaks at $0,2\pi$, troughs at $\pi$, zero at $\frac{\pi}{2},\frac{3\pi}{2}$. Setting the whole inner argument $Bx+C$ equal to the relevant standard angle and solving for $x$ transfers the known feature location through the transformation.

Thought Process — Step-by-step reasoning

  1. Decide which feature you need (max, min or zero) and recall the standard angle of the inner function that produces it.
  2. Identify the inner argument, e.g. $Bx+C$, $2x$, or $\frac{x}{12}$.
  3. Set the inner argument equal to that standard angle (add $2\pi$ or period multiples if a later cycle is wanted).
  4. Solve the linear equation for $x$, dividing or multiplying by the inner coefficient.
  5. Compute the $y$-value from the outer amplitude/shift: $A\cdot(\pm 1)+D$ for an extremum, $D$ for a zero of the trig part.
  6. Check the resulting $x$ lies in the stated domain and pick the right cycle if several are shown.

Mechanic — Concrete operations

  1. For a $\sin$ maximum set $Bx+C = \frac{\pi}{2}$ (or $90^\circ$); for a min set $= \frac{3\pi}{2}$.
  2. For a $\cos$ minimum set $Bx+C = \pi$ (or $180^\circ$); for a zero set $= \frac{\pi}{2}, \frac{3\pi}{2},\dots$
  3. Solve $x = \frac{(\text{standard angle}) - C}{B}$.
  4. Add multiples of $\frac{2\pi}{B}$ (the period) to reach later maxima/zeros if required.
  5. Substitute to get $y = \pm A + D$ (extremum) and state $(x,y)$.

Worked Micro-Example

For the minimum of $y = \sin\!\left(\frac{x}{12}\right)$: set $\frac{x}{12} = \frac{3\pi}{2}$, so $x = 18\pi$, and $y = \sin\frac{3\pi}{2} = -1$, giving $M = (18\pi, -1)$. For the first minimum of $y=\cos(2x)$ set $2x=\pi$, so $x=\frac{\pi}{2}$, $y=-1$.

Variants — Same move, different surface

Counting to a later zero/turning point, e.g. the third $x$-intercept of $\cos(2x^\circ)$.
List successive standard angles $2x^\circ = 90^\circ,270^\circ,450^\circ,\dots$ and take the required one: $2x^\circ=450^\circ$ gives $x=225$.
Phase-shifted cosine $y = A\cos(x-\alpha)^\circ$ whose maximum lies several cycles out.
Set the argument to the extremum angle plus $360^\circ n$, e.g. $x-30 = 180+360n$, then choose $n$ so $x$ matches the labelled point on the figure.
Stating the period before locating points.
Period $= \frac{2\pi}{|B|}$; e.g. $\sin\frac{x}{12}$ has $B=\frac{1}{12}$, so period $24\pi$, which spaces successive features.

Common Misreadings

  • Setting $x$ (rather than the full inner argument $Bx+C$) equal to the standard angle, ignoring the stretch/shift.
  • Confusing which standard angle gives a max vs a min, especially after a reflection (negative $A$).
  • Using the wrong period when stepping to a later cycle, e.g. taking $2\pi$ instead of $\frac{2\pi}{B}$.

Pitfalls

  • Forgetting to multiply/divide by the inner coefficient when isolating $x$ (e.g. dropping the $\times 12$).
  • Mis-deriving the period (using $2\pi$ instead of $2\pi/B$), which mislocates all subsequent points.
  • Picking the wrong cycle so the point falls outside the given domain, or not finding the smallest positive solution requested.
  • Reporting in degrees when radians are expected (or vice versa), or swapping the coordinate order.
