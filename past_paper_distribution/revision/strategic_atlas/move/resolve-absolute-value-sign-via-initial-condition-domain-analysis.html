Differential Equations

Resolve absolute value sign via initial-condition domain analysis

Family: Sign Analysis Frequency: 1 questions

Trigger — When to use

Fires after integrating a separable ODE of the form ∫ dH/[(H−a)(H−b)] = ∫ k dt, where partial fractions and exponentiation produce |(H−a)/(H−b)| = Ae^(kt). The absolute-value bars must be removed before applying the initial condition and solving for the constant. Cue phrases: 'given H(0) = …', 'find H explicitly', or any time you have |f(H)| = Ce^(λt) and need a clean algebraic solution.

Motivation — Why this works

The absolute value arises because ∫ du/u = ln|u| + C is valid on either sign of u, so exponentiating only recovers |u|. However, solutions of an autonomous ODE cannot cross equilibria (uniqueness), so the sign of each factor (H−a), (H−b) is preserved for all t in the solution interval. Checking the sign at t=0 therefore fixes the sign for all t, allowing the bars to be dropped and A to be assigned a definite sign.

Thought Process — The step-by-step reasoning

  1. Notice the equation has |…| = Ae^(λt) and the next step (apply IC, solve for H) requires a clean rational expression.
  2. Rule out keeping ± as a case-split — wasteful when the IC unambiguously fixes the branch.
  3. Commit to evaluating each factor of the ratio at t=0 to read off signs.
  4. Bridge: substitute H(0) into (H−a) and (H−b) separately, note both signs, and conclude the sign of the ratio.
  5. Tactical: confirm the solution stays on the same side of each equilibrium (no crossing), so the sign persists; then drop bars and write A with the correct sign.

Mechanic — Concrete operations

  1. Identify the factors inside the absolute value, e.g. (H−a) and (H−b).
  2. Substitute the initial value H(0) into each factor and record its sign.
  3. Determine the sign of the ratio (H−a)/(H−b) at t=0.
  4. Argue (via uniqueness / no-crossing of equilibria) that this sign persists on the solution interval.
  5. Drop the absolute value, writing (H−a)/(H−b) = Ae^(λt) with A taking the matching sign.
  6. Apply the IC at t=0 to compute |A| and confirm its sign agrees with step 3.

Worked Micro-Example

From dH/dt = −(H−5)(H+3)/40 with H(0)=13, separation gives ln|(H−5)/(H+3)| = −t/5 + C, so |(H−5)/(H+3)| = Ae^{−t/5}. At t=0: H−5 = 8 > 0 and H+3 = 16 > 0, so the ratio is positive and stays positive (H decreases toward 5 from above). Drop the bars: (H−5)/(H+3) = Ae^{−t/5}, and IC gives A = 8/16 = 1/2.

Variants — Same move, different surface

Logistic ODE dP/dt = kP(1−P/M) with P(0) between 0 and M
After separating one gets |P/(M−P)| = Ae^{kt}; both P>0 and M−P>0, so drop bars with A>0.
Newton's law of cooling: dθ/dt = −k(θ−θ_room) with θ(0) > θ_room
ln|θ−θ_room| = −kt + C; since θ−θ_room > 0 throughout cooling, |θ−θ_room| = (θ−θ_room) and A>0.
Same ODE but initial value below the lower equilibrium (e.g. H(0) = 0 < 5)
Then H−5 < 0 and H+3 > 0, so (H−5)/(H+3) < 0; drop bars but write A as negative (or equivalently flip the sign of the ratio).
Exponential decay dN/dt = −kN with N(0) > 0
|N| = Ae^{−kt} reduces to N = Ae^{−kt} since N stays positive.

Common Misreadings

  • Reading |u| = Ae^{λt} as already meaning u = Ae^{λt} without justification.
  • Thinking A must always be positive because it came from e^{C}; in fact after dropping bars A can absorb a sign.
  • Believing the sign could flip mid-trajectory, forgetting that solutions cannot cross equilibria of an autonomous ODE.

Pitfalls

  • Dropping the absolute value with no written justification — examiners expect a one-line sign argument.
  • Forgetting to recheck signs when the IC places H below an equilibrium, leading to a wrong-sign A.
  • Confusing the constant of integration C with A = e^{C}; A is positive by construction only before the bars are removed.
  • Failing to state the domain of validity (the t-interval on which the sign is preserved).

A-level Scope Notes

Standard for Edexcel IAL P3/FP separable ODE problems involving partial fractions and an initial condition; uses only ln|·|, exponentiation, and elementary sign reasoning.
