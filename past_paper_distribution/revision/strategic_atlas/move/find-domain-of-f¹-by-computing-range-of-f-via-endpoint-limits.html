Find domain of f⁻¹ by computing range of f via endpoint limits

Family: Range Via Limits Frequency: 2 questions

Trigger — When to use

The question asks for the domain (or range) of f⁻¹ where f is a rational/Möbius-type function f(x) = (ax+b)/(cx+d) defined on a one-sided restricted domain such as x > k. The function has been shown (or is easily shown) to be monotonic on that domain, and the range is not visually obvious from the formula. Equivalent phrasings include 'state the range of f', 'find the set of values f can take', or 'write down the domain of f⁻¹'.

Motivation — Why this works

The domain of f⁻¹ is by definition equal to the range of f, so the task reduces to finding the range of f. For a continuous, strictly monotonic function on an interval, the range is itself an interval whose endpoints are the values (or limits) of f at the endpoints of the domain. For Möbius functions, one endpoint typically produces a vertical-asymptote behaviour (±∞) and the other produces the horizontal asymptote y = a/c, which is approached but never attained — giving an open interval bound. This pins down the range using only two limit calculations plus monotonicity.

Thought Process — The step-by-step reasoning

  1. Notice the question is really asking for the range of f, since dom(f⁻¹) = range(f).
  2. Rule out solving y = f(x) for x and reading off restrictions algebraically — for a Möbius function this is possible but slower and obscures which endpoint is open vs closed.
  3. Commit to the limit approach because f has already been shown monotonic on the restricted domain, so the range is fully determined by behaviour at the two endpoints of that domain.
  4. Identify the two endpoints of the domain (one finite boundary k, one at ∞ or −∞) and compute lim f at each: one will give the horizontal asymptote a/c, the other will be a finite value or ±∞ depending on whether x = k makes the denominator vanish.
  5. Decide on open vs closed bounds: a limit value that is approached but never attained gives a strict inequality (open endpoint); a value actually achieved at a closed boundary gives a non-strict inequality.
  6. Combine the two endpoint values using the known monotonicity to write the range as an interval, then state it as the domain of f⁻¹.

Mechanic — Concrete operations

  1. Identify domain of f, e.g. x > k, and confirm f is continuous and strictly monotonic on it.
  2. Compute lim_{x → k⁺} f(x) by direct substitution; if denominator → 0 with non-zero numerator, the limit is ±∞ (check sign).
  3. Compute lim_{x → ∞} f(x) by dividing numerator and denominator by the highest power of x; for (ax+b)/(cx+d) this gives a/c.
  4. Use monotonicity to determine which limit is the lower bound and which is the upper bound of the range.
  5. Write range(f) as an open interval (since asymptotic values are not attained), e.g. (a/c, ∞) or (0, a/c).
  6. State: domain of f⁻¹ = range(f), using strict inequalities.

Worked Micro-Example

Let $f(x) = \dfrac{5x-3}{x-4}$ on $x > 4$, strictly decreasing. As $x \to 4^+$, numerator $\to 17$, denominator $\to 0^+$, so $f(x) \to +\infty$. As $x \to \infty$, $f(x) \to 5/1 = 5$ from above. Hence range$(f) = (5, \infty)$, so domain of $f^{-1}$ is $x > 5$.

Variants — Same move, different surface

Domain is x > 0 and the finite-endpoint limit is finite (no vertical asymptote at x = 0)
Substitute x = 0 directly into f to get the finite endpoint value, e.g. f(x) = 5x/(x+3) gives f → 0, then range is (0, a/c) = (0, 5).
Domain is a closed interval [a, b]
Evaluate f(a) and f(b) directly (not as limits); the range is a closed interval [min, max] using monotonicity.
f is non-monotonic on the domain
In addition to endpoint limits, find turning points via f'(x) = 0 and include those y-values when determining the global max/min.
f is decreasing rather than increasing
Swap which endpoint limit is the upper vs lower bound of the range; the mechanic is otherwise identical.

Common Misreadings

  • Reading 'domain of f⁻¹' as a request to first construct f⁻¹ algebraically and then state its domain — slower and error-prone compared to using range(f).
  • Treating the question as 'state the range of f⁻¹' (which equals the domain of f, a different and usually trivial answer).

Pitfalls

  • Writing x ≥ 5 instead of x > 5 — the horizontal asymptote value is never attained, so the inequality must be strict.
  • Getting the sign of an infinite limit wrong by not checking whether the denominator approaches 0 from above or below.
  • Forgetting to justify (or first establish) monotonicity, without which the range need not be a single interval between the two endpoint limits.
  • Confusing range(f) with range(f⁻¹), or domain(f) with domain(f⁻¹).

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P3 functions chapter). Uses the standard fact dom(f⁻¹) = range(f) and informal limit reasoning at asymptotes — no formal limit definitions or L'Hôpital required. Avoid invoking continuity theorems by name; the argument 'f is continuous and monotonic, so range is an interval between the endpoint values' is the expected level of rigour.
