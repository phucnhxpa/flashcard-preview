Differentiation

Use second derivative test to filter minima from multiple stationary points

Family: Second Derivative Classification Frequency: 1 questions

Trigger — When to use

The question asks specifically for local minima (or maxima) rather than all stationary points, and several candidate x-values have been produced — typically from a trig or polynomial equation dy/dx = 0 with multiple roots in a given interval. Phrasings include 'find the smallest positive x at which y has a minimum', 'the values of x giving local minima', or labelling them α, β, λ. The presence of more candidates than the question asks for is the key cue that classification is required.

Motivation — Why this works

At a stationary point, the sign of d²y/dx² distinguishes minima (positive) from maxima (negative). For trig stationary points, d²y/dx² typically reduces to a constant times cos(kx) or sin(kx), so evaluation is cheap and avoids the bookkeeping of a first-derivative sign-change table. This filters the candidate list down to the requested type so that ordering by x then picks out the named values.

Thought Process — The step-by-step reasoning

  1. I notice the equation dy/dx = 0 has produced more candidate x-values than the question asks for, and the question specifies 'minimum' rather than 'stationary point'.
  2. I rule out a first-derivative sign-change test: with four candidates and a trig derivative, building sign tables is slow and error-prone.
  3. I commit to the second derivative test because d²y/dx² is a single trig expression I can evaluate by inspection at each candidate.
  4. I compute d²y/dx² once, then substitute each candidate x to read off the sign — positive ⇒ min, negative ⇒ max.
  5. I keep only the minima, then sort by x to identify the named values (e.g. β smallest, λ next).

Mechanic — Concrete operations

  1. Differentiate dy/dx to obtain d²y/dx².
  2. Substitute each candidate stationary x-value into d²y/dx², being careful that the argument is kx (not x) for a chain-rule term.
  3. Record the sign: d²y/dx² > 0 ⇒ minimum; d²y/dx² < 0 ⇒ maximum; = 0 ⇒ inconclusive, fall back to sign-change test.
  4. Discard maxima (and inconclusive points unless resolved).
  5. Order the surviving minima by x and assign to the labels requested.

Worked Micro-Example

Given y = 2\sin(3x) with stationary points at x = \tfrac{7\pi}{18}, \tfrac{11\pi}{18}, \tfrac{19\pi}{18}, \tfrac{23\pi}{18}. Then \dfrac{d^2y}{dx^2} = -18\sin(3x). At x = \tfrac{7\pi}{18}: -18\sin(\tfrac{7\pi}{6}) = -18(-\tfrac{1}{2}) = 9 > 0, so minimum. At x = \tfrac{11\pi}{18}: -18\sin(\tfrac{11\pi}{6}) = -18(-\tfrac{1}{2})… (recompute with correct derivative for the actual curve). For y with d²y/dx² = -18\cos(3x): at \tfrac{7\pi}{18}, value is 9\sqrt{3} > 0 (min); at \tfrac{11\pi}{18}, value is -9\sqrt{3} < 0 (max). Hence β = \tfrac{7\pi}{18}, λ = \tfrac{19\pi}{18}.

Variants — Same move, different surface

Polynomial curve with multiple stationary points (e.g. cubic/quartic)
Compute d²y/dx² as a polynomial and substitute numerically; sign of the result classifies each point.
Trig curve where d²y/dx² = 0 at a stationary point (e.g. y = x − sin x at x = 0)
Test inconclusive — revert to a first-derivative sign-change test on either side of the point.
Parametric curve with x = f(t), y = g(t)
Either compute d²y/dx² via the parametric formula \dfrac{d}{dt}(dy/dx) ÷ (dx/dt), or classify using sign changes of dy/dx as t passes through the stationary parameter.
Implicit curve from F(x,y) = 0
Differentiate dy/dx implicitly a second time, substituting dy/dx = 0 at the stationary point to simplify, then test the sign.
Optimisation in context (max area/volume)
Same test on the objective function in one variable; sign of d²A/dx² or d²V/dx² confirms a maximum was found.

Common Misreadings

  • Treating every solution of dy/dx = 0 as a minimum and reporting the smallest x-values without classification.
  • Confusing 'smallest positive x with a stationary point' with 'smallest positive x with a minimum'.
  • Reading d²y/dx² > 0 as 'maximum' by analogy with 'biggest', reversing the rule.

Pitfalls

  • Sign error in d²y/dx²: forgetting the leading minus in expressions like −18cos(3x) and inverting the classification.
  • Evaluating the trig function at x rather than at kx — e.g. cos(7π/18) instead of cos(3·7π/18) = cos(7π/6).
  • Failing to reduce 3x mod 2π before reading off cos/sin from the unit circle, leading to arithmetic slips.
  • Stopping at the first minimum found and missing that the question asks for two (β and λ).

A-level Scope Notes

Fully within Edexcel IAL P2/P3/P4 differentiation and trigonometry: chain rule, exact values of cos and sin at standard angles, and the standard second-derivative test for classifying stationary points.
