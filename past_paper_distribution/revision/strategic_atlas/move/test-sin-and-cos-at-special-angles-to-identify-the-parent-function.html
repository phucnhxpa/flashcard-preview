Trigonometry

Test sin and cos at special angles to identify the parent function

Family: Function Identification Frequency: 1 questions

Trigger — When to use

A point $(x_0, y_0)$ is given on an unidentified trig graph (often labelled $y = A\sin x$ or $y = A\cos x$), and $x_0$ is a multiple of $90°$ or $180°$ (e.g. $0°, 90°, 180°, 270°, 360°, 540°$, or $\tfrac{\pi}{2}, \pi, \tfrac{3\pi}{2}$). Equivalently, the question asks 'which of $y = A\sin x$ or $y = A\cos x$ passes through this point?'. The cue is that $|y_0|$ equals the stated amplitude or $y_0 = 0$, signalling an extremum or root.

Motivation — Why this works

At integer multiples of $90°$, sine and cosine only take values in $\{-1, 0, 1\}$, and crucially $\sin$ and $\cos$ never take a non-zero value at the same standard angle — wherever one is $\pm 1$, the other is $0$. So substituting $x_0$ into each candidate produces exactly one match with $y_0$, uniquely fixing the parent function. This avoids any need for graph-shape reasoning or calculus.

Thought Process — The step-by-step reasoning

  1. Notice $|y_0|$ equals the amplitude (or is $0$), so $x_0$ must be an extremum (or a root) of the trig function.
  2. Rule out one candidate by evaluating $\sin x_0$ — if it gives the wrong value (typically $0$ when $y_0 \neq 0$), discard $y = A\sin x$.
  3. Commit to the surviving candidate after checking that $\cos x_0$ produces the correct sign and magnitude.
  4. Bridge: write $y_0 = A \cdot (\text{value of trig at } x_0)$ and verify equality including sign.
  5. Tactical: reduce $x_0$ modulo $360°$ first (e.g. $540° \equiv 180°$) to land on a memorised special-angle value.

Mechanic — Concrete operations

  1. Reduce $x_0$ modulo $360°$ to a standard reference angle.
  2. Compute $\sin x_0$ using the unit circle / standard values.
  3. Test: does $A \sin x_0 = y_0$? If not, reject $y = A\sin x$.
  4. Compute $\cos x_0$ similarly and test $A \cos x_0 = y_0$.
  5. Conclude with the candidate that matches both sign and magnitude.

Worked Micro-Example

Point $(540°, -5)$ lies on $y = A\sin x$ or $y = A\cos x$ with $A = 5$. Reduce: $540° \equiv 180°$. Then $\sin 180° = 0$, so $5\sin 540° = 0 \neq -5$ — reject sine. And $\cos 180° = -1$, so $5\cos 540° = -5$ ✓. Hence the point lies on $y = 5\cos x$.

Variants — Same move, different surface

$y_0 = 0$ at a standard angle
Use roots instead of extrema: $\sin x = 0$ at multiples of $180°$, $\cos x = 0$ at odd multiples of $90°$ — match accordingly.
Transformed graph $y = A\sin(Bx + C) + D$
Subtract $D$ first, then test the inner argument $Bx_0 + C$ at standard angles rather than $x_0$ itself.
Radian-valued $x_0$ (e.g. $x_0 = \tfrac{3\pi}{2}$)
Use the radian-form special values; the elimination logic is unchanged.
Choosing between $\tan x$ and $\sin/\cos$
Check whether $x_0$ is an asymptote of $\tan$ (odd multiples of $90°$); otherwise evaluate $\tan x_0 \in \{-1, 0, 1\}$ at standard angles.

Common Misreadings

  • Assuming the graph is sine by default because it is the more familiar function, without testing cosine.
  • Reading $|y_0| = A$ as 'any trig function with amplitude $A$ works' rather than as a constraint forcing an extremum.
  • Treating the amplitude as unknown when it is already given, and trying to solve for $A$ instead of identifying the function.

Pitfalls

  • Forgetting $\sin 180° = 0$ (confusing with $\sin 90° = 1$).
  • Sign error: $\cos 180° = -1$, not $+1$; $\cos 270° = 0$, not $-1$.
  • Failing to reduce $x_0$ modulo $360°$ before evaluating, leading to arithmetic slips.
  • Mixing degree and radian modes when the question uses one consistently.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics P2 (trig graphs and standard values). Uses only memorised values of $\sin$ and $\cos$ at multiples of $90°$ and basic amplitude scaling.
