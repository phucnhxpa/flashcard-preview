Differentiation

Differentiate the inverse relation and reciprocate to obtain dy/dx

Family: Inverse Function Differentiation Frequency: 2 questions

Trigger — When to use

Fires when y is given as an awkward function of x (e.g. log₁₀(2x+1), arccos(2x), arcsin(kx)) AND a previous part has produced — or it is easy to produce — an inverse expression x = g(y) that is much simpler to differentiate. Typical surface cues: the word 'Hence' linking to a prior 'show that x = …' part, a logarithm with a non-natural base, or an inverse trig function whose direct derivative the student is unsure of. Equivalently: any time dx/dy is cleaner than dy/dx, this move is the right tool.

Motivation — Why this works

The reciprocal rule dy/dx = 1/(dx/dy) (valid when dx/dy ≠ 0) lets us swap a hard differentiation for an easy one by exploiting the inverse relationship. It works because if y = f(x) and x = g(y) are inverses, the local slopes are reciprocals — geometrically the graphs are reflections in y = x. Mathematically this converts log_a or inverse-trig derivatives (which students often misremember) into clean exponential or basic trig derivatives from the formula booklet, and it honours any 'Hence' instruction by directly using the previous part's result.

Thought Process — The step-by-step reasoning

  1. Notice: y is expressed in a form (log base 10, arccos, arcsin, …) whose direct derivative is either awkward or not obviously memorised, while x is — or can easily be — written as a clean function of y.
  2. Rule out direct differentiation: it would either ignore the 'Hence' instruction (losing method marks) or force a change-of-base / inverse-trig identity that is more error-prone.
  3. Recall the inverse-function rule dy/dx = 1/(dx/dy), valid where dx/dy ≠ 0, and confirm dx/dy will be non-zero on the relevant domain.
  4. Commit: differentiate x with respect to y first, using a standard result (d/dy(a^y) = a^y ln a, d/dy(sin y) = cos y, etc.).
  5. Reciprocate, then — if the answer is required in terms of x — substitute back using the original relation (e.g. replace 10^y by 2x+1, or sin y by √(1−x²)) so the final dy/dx is expressed in the variable the question asks for.

Mechanic — Concrete operations

  1. Write x = g(y) (taken from the previous part or by rearranging y = f(x)).
  2. Differentiate with respect to y to obtain dx/dy = g'(y), using standard derivatives such as d/dy(a^y) = a^y ln a or d/dy(sin y) = cos y.
  3. Apply the reciprocal rule: dy/dx = 1/g'(y).
  4. If required, re-express dy/dx in terms of x by substituting g(y) ↔ x or using a Pythagorean identity (e.g. cos y = √(1 − sin²y) = √(1 − x²)).
  5. State the domain restriction where dx/dy ≠ 0 if the question asks for it.

Worked Micro-Example

Given $y = \log_{10}(2x+1)$, part (a) gives $x = \tfrac{1}{2}(10^{y}-1)$. Then $\dfrac{dx}{dy} = \tfrac{1}{2}\,10^{y}\ln 10$, so $\dfrac{dy}{dx} = \dfrac{2}{10^{y}\ln 10} = \dfrac{2}{(2x+1)\ln 10}$.

Variants — Same move, different surface

Inverse trig: y = arcsin(x) or y = arccos(x)
Write x = sin y (or cos y), differentiate dx/dy = cos y (or −sin y), reciprocate, then use cos y = √(1−x²) to return to x.
Inverse trig with a linear inner function, e.g. y = arccos(2x)
Either set u = 2x and use the formula-book derivative −1/√(1−u²) with chain rule (giving −2/√(1−4x²)), or write 2x = cos y, differentiate 2 = −sin y · dy/dx, and solve — both routes agree.
Logarithm to a non-natural base, y = log_a(h(x))
Invert to x = h⁻¹(a^y), use d/dy(a^y) = a^y ln a, then reciprocate; the ln a appears naturally in the denominator.
Parametric-style 'Hence' questions where x = g(t), y = h(t) is replaced by an explicit x = g(y)
Same idea — differentiate the explicit relation, reciprocate; no parametric chain needed.

Common Misreadings

  • Treating 'Hence find dy/dx' as a free choice and differentiating y = f(x) directly — may yield a correct answer but loses the method marks tied to using the previous part.
  • Confusing dx/dy with dy/dx and quoting the un-reciprocated expression as the final answer.
  • Mistaking arccos for arcsin (or vice versa) and dropping the negative sign in the inverse-trig case.

Pitfalls

  • Forgetting the ln a factor when differentiating a^y — writing d/dy(10^y) = 10^y instead of 10^y ln 10.
  • Forgetting the chain-rule factor for a linear inner function (e.g. giving −1/√(1−4x²) instead of −2/√(1−4x²) for arccos(2x)).
  • Leaving the answer in terms of y when the question asks for it in terms of x (or vice versa).
  • Ignoring where dx/dy = 0, which is precisely where dy/dx fails to exist.

A-level Scope Notes

All steps stay within Edexcel IAL P3/P4 and Further Pure: the derivatives d/dy(a^y) = a^y ln a, d/dy(sin y) = cos y, and the standard inverse-trig derivatives are all in the IAL formula booklet. Avoid invoking implicit differentiation theorems or general inverse-function theorems beyond the simple reciprocal rule dy/dx = 1/(dx/dy).
