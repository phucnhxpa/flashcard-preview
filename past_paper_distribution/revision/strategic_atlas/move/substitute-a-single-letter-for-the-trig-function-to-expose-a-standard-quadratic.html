Trigonometry

Substitute a single letter for the trig function to expose a standard quadratic

Family: Substitution Bridge Frequency: 5 questions

Trigger — When to use

Fires when, after any identity work, the equation contains only one trig function appearing as a squared term, a linear term, and a constant — i.e. the structure a·f(x)² + b·f(x) + c = 0 where f is sin, cos, tan, sec, cosec, cot, etc. Visual cues: the presence of sin²x alongside sin x (or cos²x alongside cos x, sec²θ alongside sec θ, …) with no other distinct trig function present. Equivalent phrasings of the trigger: 'the equation is quadratic in sin x', 'a quadratic in disguise', 'only one trig ratio appears, but to two different powers'. If two different trig functions remain (e.g. sin x AND cos x), the trigger has NOT fired — an identity must be applied first.

Motivation — Why this works

The substitution u = f(x) strips away the trigonometric wrapping and exposes a purely algebraic quadratic au² + bu + c = 0, which can be handled by routine factorisation or the quadratic formula from the formula booklet. This cleanly decouples the problem into two independent phases: (1) an algebraic phase to find the roots u₁, u₂, and (2) a trigonometric phase to solve f(x) = u₁ and f(x) = u₂ for x in the required interval. The principle being invoked is simply that polynomial structure is preserved under any one-to-one symbolic relabelling, so the standard quadratic toolkit applies verbatim.

Thought Process — The step-by-step reasoning

  1. Notice the equation contains only ONE trig function, but raised to two different powers (squared and linear) plus a constant — this is the visual signature of a quadratic in disguise.
  2. Rule out direct factorisation in sin x / cos x / etc. — bracket-matching with compound symbols like sin²x is error-prone, and rule out trying to 'square root both sides' which only works for pure square equations.
  3. Commit to the substitution because it converts the problem into the most familiar shape in A-level algebra (au² + bu + c = 0) and makes the factor pairs or discriminant calculation immediately visible.
  4. Bridge to action: introduce a single letter (u, y, s, c, …) and rewrite the equation, optionally multiplying through by −1 first to make the leading coefficient positive.
  5. Tactical choice: try factorisation first (look for factor pairs of a·c summing to b); if no neat pair exists, switch to the quadratic formula.
  6. After solving for u, immediately check the range constraint: for sin/cos roots must lie in [−1, 1]; for sec/cosec roots must satisfy |u| ≥ 1; for tan/cot any real root is admissible. Discard inadmissible roots before converting back to angles.

Mechanic — Concrete operations

  1. Identify the single trig function f(x) appearing in the equation and let u = f(x).
  2. Rewrite the equation as au² + bu + c = 0 (multiply through by −1 if it tidies the leading coefficient).
  3. Solve the quadratic: try factorisation (au + p)(u + q) = 0 using factor pairs of ac summing to b; otherwise use u = (−b ± √(b² − 4ac))/(2a).
  4. List the roots u₁, u₂ and reject any that violate the range of f (e.g. |u| > 1 for sin/cos; |u| < 1 for sec/cosec).
  5. Substitute back: solve f(x) = u₁ and f(x) = u₂ separately for x in the required interval, using the standard angle techniques (principal value + symmetry/period).

Worked Micro-Example

Solve $2\sin^2 x + \sin x - 1 = 0$ for $x \in [0, 2\pi)$. Let $u = \sin x$, giving $2u^2 + u - 1 = 0 \Rightarrow (2u-1)(u+1) = 0$, so $u = \tfrac{1}{2}$ or $u = -1$. Both lie in $[-1, 1]$. Back-substitute: $\sin x = \tfrac{1}{2} \Rightarrow x = \tfrac{\pi}{6}, \tfrac{5\pi}{6}$; $\sin x = -1 \Rightarrow x = \tfrac{3\pi}{2}$.

Variants — Same move, different surface

Quadratic in cos x or tan x
Identical procedure with c = cos x or t = tan x. For tan there is no [−1, 1] restriction, so no roots are rejected on range grounds.
Quadratic in sec θ or cosec θ
Let s = sec θ (or cosec θ). Reject any root with |s| < 1, since sec and cosec take values only outside (−1, 1). Convert back via cos θ = 1/s.
Coefficients do not factorise nicely
Use the quadratic formula u = (−b ± √(b² − 4ac))/(2a) from the formula booklet; keep surd form unless a decimal is required for inverse-trig evaluation.
Negative leading coefficient, e.g. −2sin²x + sin x + 1 = 0
Multiply through by −1 first to get 2y² − y − 1 = 0 before factorising, to avoid sign errors.
Exponential analogue, e.g. e^{2x} + 3e^x − 4 = 0
Same idea with u = e^x; reject any non-positive root since e^x > 0.

Common Misreadings

  • Treating sin²x as if it were sin(x²) and trying to apply inverse-sin too early.
  • Trying to 'square root both sides' instead of recognising the full quadratic structure.
  • Skipping the substitution and attempting to factorise directly with the compound trig symbol, leading to bracket errors.
  • Mistaking an equation containing two different trig functions (e.g. both sin x and cos x) for this pattern — an identity must be applied first to reduce to a single function.

Pitfalls

  • Forgetting the range check: sin and cos roots outside [−1, 1] must be rejected; sec and cosec roots inside (−1, 1) must be rejected.
  • Sign errors when splitting the middle term, especially when the leading coefficient is not 1 or is negative.
  • Stopping at the u-values and forgetting to convert back to x — the question asks for angles, not for u.
  • Losing the second family of solutions: each valid root of the quadratic typically yields multiple angles in the given interval (principal value plus its symmetric partner, then add periods).
  • Arithmetic slips in the discriminant or in dividing the whole numerator by 2a when using the quadratic formula.

A-level Scope Notes

Fully within Edexcel IAL P2/P3 (trigonometric equations) and Further Pure (extending to sec, cosec, cot). The quadratic formula is in the formula booklet; the trig identities used to reach the single-function form (e.g. cos 2x = 1 − 2sin²x, sec²θ = 1 + tan²θ) are standard. Avoid generalising to higher-degree trig polynomials or to systems mixing two trig functions without an identity bridge — those require different techniques.
