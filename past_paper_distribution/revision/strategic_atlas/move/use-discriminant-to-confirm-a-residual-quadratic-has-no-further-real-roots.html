Algebra & Functions

Use discriminant to confirm a residual quadratic has no further real roots

Family: Discriminant Completeness Check Frequency: 1 questions

Trigger — When to use

Fires after a polynomial (typically cubic or quartic) has been factorised into a known linear factor times a residual quadratic, and the question demands 'all values', 'the possible values', or a 'show that' completeness argument. The residual quadratic does not factor by inspection, so its real-root status is uncertain. Also fires whenever you must justify that a candidate solution is unique without solving the quadratic in full.

Motivation — Why this works

The discriminant Δ = b² − 4ac of a real quadratic determines real-root existence: Δ < 0 means no real roots, so the quadratic factor contributes nothing further to the real solution set. This is cheaper than the full quadratic formula and is exactly what examiners expect for a completeness step. It converts an open-ended 'are there more solutions?' question into a one-line numerical check.

Thought Process — The step-by-step reasoning

  1. Notices that after dividing out the known linear factor, a quadratic factor remains and the question asks for all real solutions.
  2. Rules out solving the quadratic fully — only its real-root existence matters here, not the actual roots.
  3. Commits to the discriminant test as the minimal-cost justification of completeness.
  4. Identifies a, b, c from the residual quadratic and prepares to compute b² − 4ac carefully.
  5. Plans to state the sign of Δ explicitly and to write a concluding sentence tying it back to the original variable's possible values.

Mechanic — Concrete operations

  1. Write the residual quadratic in the form ar² + br + c = 0 and identify a, b, c.
  2. Compute Δ = b² − 4ac, taking care with signs and with 4ac when a ≠ 1.
  3. Compare Δ to 0: if Δ < 0, no real roots; if Δ = 0, a repeated real root; if Δ > 0, two distinct real roots.
  4. State the conclusion in words, e.g. 'no further real solutions, so r = … is the only real value'.
  5. Feed this back into the original problem to deliver a single, complete answer (or full case list).

Worked Micro-Example

After factoring, $(r+2)(3r^2 - 10r + 16) = 0$. For the quadratic factor, $\Delta = (-10)^2 - 4(3)(16) = 100 - 192 = -92 < 0$, so it has no real roots. Hence $r = -2$ is the unique real solution, giving the single answer pair $(a, r) = (13, -2)$.

Variants — Same move, different surface

Quartic factorised as (quadratic)(quadratic), one factor already solved
Apply the discriminant to the remaining quadratic factor to decide whether it contributes additional real roots.
'Show that the equation has exactly one real root' style question
Extract the obvious root, then use Δ < 0 on the residual quadratic to prove uniqueness.
Geometric progression / sequence problem with ratio r constrained to be real
After the discriminant rules out further real r, also recheck context constraints (r ≠ 0, |r| < 1 for sum to infinity, etc.).
Parameter problem: find values of k for which a cubic has only one real root
Set up Δ(k) < 0 on the residual quadratic and solve the inequality in k.

Common Misreadings

  • Treating the discriminant check as optional decoration rather than a required completeness step in 'find all values' questions.
  • Believing that finding one root of a cubic automatically means it is the only real root.
  • Confusing Δ < 0 (no real roots) with 'no solutions at all' — complex roots still exist but are out of A-level real scope here.

Pitfalls

  • Sign errors in b², especially when b is negative; (−10)² = 100, not −100.
  • Arithmetic slips in 4ac when a ≠ 1, e.g. forgetting the factor of a.
  • Failing to write a concluding sentence that names the unique real solution, losing the communication mark.
  • Applying the discriminant to a non-quadratic expression (e.g. a cubic factor) — it is only valid for ar² + br + c.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics: discriminant of a real quadratic and factorisation of cubics via the factor theorem. No use of complex roots or higher-degree discriminants is required.
