Differentiation

Bridge between variables using the defining equation or a shared parameter

Family: Substitution Bridge Frequency: 2 questions

Trigger — When to use

Fires when a derivative has been (or can be) computed in one variable, but the question demands the answer in a different variable. Two classic surface forms: (a) 'find dy/dx in terms of y' after differentiating an equation y = f(x) — the result is naturally in x; (b) 'find dV/dA' (or similar) when V and A are both given as functions of a common parameter x or t, with no direct V(A) relation supplied. Recognition cues: the variable demanded in the answer does not match the variable produced by direct differentiation, AND there exists either an original defining equation or a shared parameter that links them.

Motivation — Why this works

The move works because a 'bridge' equation already in hand — either the original curve equation y = f(x) or a parametric link through x/t — lets you swap variables without solving messy inverse functions. For trig/hyperbolic cases the Pythagorean-type identity (sec²θ − 1 = tan²θ, cosh²θ − sinh²θ = 1) converts a second trig factor into the target variable. For parametric cases the chain rule dV/dA = (dV/dx)/(dA/dx) bypasses the need to eliminate x algebraically. In both surfaces, the principle is the same: use an existing relation to translate the derivative into the requested variable cleanly.

Thought Process — The step-by-step reasoning

  1. Notice the mismatch: my differentiation produced an expression in variable A, but the question asks for the answer in variable B.
  2. Consider and rule out the brute-force route: solving the defining equation for one variable in terms of the other and re-substituting often introduces surds, fractions, or inverse trig — error-prone and sometimes not even algebraically clean.
  3. Commit to the bridge: the defining equation (e.g. y = 3sec(2x), so sec(2x) = y/3) or the shared parameter (V(x), A(x)) is already a ready-made translator — use it.
  4. For trig/hyperbolic: substitute the direct piece first (sec(2x) → y/3), then identify which Pythagorean identity converts the remaining factor (tan² = sec² − 1) into the target variable.
  5. For parametric: write dV/dA = (dV/dx)/(dA/dx); both numerator and denominator are simple power-rule derivatives.
  6. Tactical sign choice: when taking a square root from the identity, check the given domain (e.g. 0 < 2x < π/2 ⇒ first quadrant ⇒ positive root).
  7. Simplify carefully — cancel common factors of x, combine fractions, and verify the final form genuinely contains only the requested variable.

Mechanic — Concrete operations

  1. Differentiate the given equation(s) using chain/power rule, retaining all chain factors (e.g. d/dx[sec(2x)] = 2sec(2x)tan(2x)).
  2. Read off the direct bridge from the original equation (e.g. sec(2x) = y/3, or note both V and A share parameter x).
  3. If a second factor remains in the wrong variable, apply the relevant identity: sec²θ − 1 = tan²θ, 1 − sin²θ = cos²θ, cosh²θ − sinh²θ = 1.
  4. Solve for the second factor and take the appropriate signed root using the stated domain.
  5. For parametric form: compute dV/dA = (dV/dx) ÷ (dA/dx) and simplify (cancel powers of x, reduce numeric fractions).
  6. Substitute all bridge expressions into the derivative; verify the result contains only the requested variable (or is in the form the question expects).

Worked Micro-Example

Given $y = 3\sec(2x)$ on $0 < 2x < \pi/2$, find $\frac{dy}{dx}$ in terms of $y$. Differentiate: $\frac{dy}{dx} = 6\sec(2x)\tan(2x)$. Bridge: $\sec(2x) = y/3$, and $\tan^2(2x) = \sec^2(2x) - 1 = \frac{y^2-9}{9}$, so $\tan(2x) = \frac{\sqrt{y^2-9}}{3}$. Substitute: $\frac{dy}{dx} = 6 \cdot \frac{y}{3} \cdot \frac{\sqrt{y^2-9}}{3} = \frac{2y\sqrt{y^2-9}}{3}$.

Variants — Same move, different surface

Hyperbolic curve y = a·cosh(bx) with answer required in y
Differentiate to get sinh(bx), then use cosh²θ − sinh²θ = 1 with cosh(bx) = y/a to express sinh(bx) in terms of y.
Parametric curve x(t), y(t) with dy/dx required
Use dy/dx = (dy/dt)/(dx/dt); the parameter t plays the bridging role that x did in the V/A example.
Related rates with V and r, A and r (sphere/cone/cylinder)
Apply dV/dA = (dV/dr)/(dA/dr); same chain-rule bridge, geometric context.
Inverse trig curves where answer needed in y
Use sin²+cos²=1 (or 1+tan²=sec²) to convert the residual trig factor — same principle.

Common Misreadings

  • Treating 'in terms of y' as a hint to use implicit differentiation, when in fact direct differentiation followed by substitution is cleaner.
  • Trying to eliminate x algebraically by inverting one of the equations (introducing unnecessary surds) instead of using the chain rule via the parameter.
  • Computing dA/dV when dV/dA was asked, inverting the answer.

Pitfalls

  • Forgetting the chain rule factor from inner functions (e.g. the factor 2 in d/dx[sec(2x)]).
  • Taking the wrong sign when square-rooting from a Pythagorean identity — always check the stated domain/quadrant.
  • Algebra slips when combining fractions like (y/3)·(√(y²−9)/3) or simplifying 15/12 to 5/4.
  • Cancelling x² with x incorrectly, leaving the wrong power in the simplified form.

A-level Scope Notes

Fully within Edexcel IAL P3/P4 (chain rule, parametric differentiation) and FP3 (hyperbolic identities). The identities sec²θ = 1 + tan²θ and cosh²θ − sinh²θ = 1 are in the formula booklet. Avoid generalising to implicit-function-theorem language or multivariable Jacobians — those are out of scope.
