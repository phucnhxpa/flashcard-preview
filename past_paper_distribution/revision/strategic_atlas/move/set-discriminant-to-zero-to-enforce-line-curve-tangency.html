Coordinate Geometry

Set discriminant to zero to enforce line-curve tangency

Family: Discriminant Tangency Frequency: 2 questions

Trigger — When to use

Fires whenever a line is required to be tangent to a curve (circle, parabola, ellipse, or any conic) — equivalently, the line meets the curve at exactly one point. Cue phrases include 'l is a tangent to C', 'find the values of k for which y = mx + c touches the curve', 'horizontal tangent y = k', or 'find the radius for which the line just touches'. Structurally: substituting the line into the curve produces a quadratic in one variable. Also fires for horizontal tangents y = k or vertical tangents x = c on implicit conics, where fixing one coordinate yields a quadratic in the other.

Motivation — Why this works

A quadratic ax² + bx + c = 0 has exactly one (repeated) root precisely when Δ = b² − 4ac = 0. Geometric tangency — a single contact point with multiplicity two — is therefore algebraically equivalent to the discriminant of the substitution quadratic vanishing. Imposing Δ = 0 converts a geometric constraint into a clean algebraic relation between the parameters of the line and the curve (e.g. between k and r, or m and c). This route avoids implicit differentiation and the perpendicular-distance formula, both of which are valid alternatives but typically involve more steps.

Thought Process — The step-by-step reasoning

  1. Notice the keyword 'tangent' (or equivalent: 'touches at exactly one point', 'horizontal/vertical tangent y = k'): tangency ↔ one repeated intersection.
  2. Check that substituting the line into the curve produces a genuine quadratic in one variable — confirm the leading coefficient is non-zero.
  3. Rule out alternatives: implicit differentiation (dy/dx = 0) is heavier and brings a simultaneous system; the perpendicular-distance d = r formula works for circles only. The discriminant route handles all conics uniformly.
  4. Commit to Δ = 0 because the question often already supplies, or strongly hints at, the substitution quadratic (e.g. part (a) sets it up).
  5. Identify a, b, c carefully from the quadratic — paying attention to signs and parameter dependence — then expand b² − 4ac.
  6. Anticipate the target form: if the answer is to look like 5r² = (k + p)², expect the parameter terms to collapse into a perfect square; this guides your simplification.

Mechanic — Concrete operations

  1. Substitute the line equation into the curve equation to obtain a quadratic in one variable, ax² + bx + c = 0.
  2. Read off a, b, c (often functions of the line/curve parameters such as k, m, r).
  3. Form the discriminant Δ = b² − 4ac and set Δ = 0.
  4. Expand and simplify, watching cross-term signs (e.g. middle term of (4k − 26)² is −208k).
  5. Recognise and factor any perfect square in the parameters to reach the required form.
  6. If needed, verify the leading coefficient is non-zero for the parameter values found, and check the contact point lies on the curve.

Worked Micro-Example

Line y = 2x + c is tangent to circle x² + y² = 5. Substitute: x² + (2x + c)² = 5 ⇒ 5x² + 4cx + (c² − 5) = 0. Set Δ = 0: (4c)² − 4·5·(c² − 5) = 0 ⇒ 16c² − 20c² + 100 = 0 ⇒ −4c² + 100 = 0 ⇒ c² = 25 ⇒ c = ±5.

Variants — Same move, different surface

Line tangent to a parabola y² = 4ax
Substitute y = mx + c into y² = 4ax to get a quadratic in x; set Δ = 0 to obtain the tangency condition c = a/m.
Horizontal tangent y = k to an implicit conic
Substitute y = k to get a quadratic in x; setting Δ(k) = 0 yields the y-coordinates of horizontal tangent points without implicit differentiation.
Vertical tangent x = c
Substitute x = c to obtain a quadratic in y; set its discriminant to zero.
Tangent to a circle, alternative method
Use the perpendicular-distance formula d = |ax₀ + by₀ + c|/√(a² + b²) = r; equivalent to Δ = 0 but avoids forming the quadratic.

Common Misreadings

  • Setting Δ > 0 (two intersections) instead of Δ = 0, confusing tangency with the secant case.
  • Setting Δ < 0 (no intersections), confusing tangency with non-intersection.
  • Confusing 'horizontal tangent' with 'tangent at the origin' or 'tangent at a given point', leading to a point-based rather than parameter-based approach.
  • Reaching for implicit differentiation when the discriminant route is shorter.

Pitfalls

  • Arithmetic errors in expanding b², especially the middle term in (4k − 26)² = 16k² − 208k + 676.
  • Dropping the −4ac term or mishandling its sign — e.g. forgetting that −4·5·(−r²) = +20r².
  • Failing to recognise a perfect square such as k² + 2k + 1 = (k + 1)², leaving the answer in unsimplified form.
  • Assuming the substitution is quadratic when the leading coefficient depends on a parameter and may vanish — check it is non-zero.
  • Forgetting to verify that the resulting contact point actually lies on the curve, particularly after squaring steps.

A-level Scope Notes

Fully within Edexcel IAL Pure (P1–P4) and Further Pure scope. Uses the discriminant b² − 4ac (no formula booklet entry needed) and standard conic equations from FP1/FP3. Avoid generalising to higher-degree curves where 'tangency = repeated root' still holds algebraically but the discriminant of a quadratic no longer captures it.
