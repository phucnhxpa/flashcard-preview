Strategic Atlas / Integration / Initial Condition Substitution
Integration

Substitute known point into antiderivative to determine constant of integration

Family: Initial Condition Substitution Frequency: 3 questions

Trigger — When to use

After performing an indefinite integration, the result contains an unknown constant of integration $+c$, and the question separately states that the curve passes through a specific point $(x_0, y_0)$ or gives a function value such as $f(x_0) = y_0$. Common phrasings include 'the curve passes through $(4, 12)$', 'given that $f(3) = \tfrac{67}{24}$', 'with initial condition $y(0) = 1$', or 'the gradient function is ... and the curve passes through ...'. The structural cue is: one unknown ($c$) plus one supplied numerical condition.

Motivation — Why this works

Indefinite integration recovers an antiderivative only up to an additive constant, producing a whole family of vertically-shifted curves $F(x) + c$. A single known point on the curve pins down which member of that family is intended, because substituting $x_0$ and the known value into $F(x_0) + c = y_0$ yields a single linear equation in $c$. Solving it converts the general antiderivative into the particular solution required by the question.

Thought Process — The step-by-step reasoning

  1. I notice that after integrating I have an expression of the form $F(x) + c$ with exactly one unknown, and the question separately supplies a numerical condition like $f(x_0) = y_0$ — that's a one-equation-one-unknown setup.
  2. I rule out treating this as a definite integral (no bounds were given) and rule out substituting the point into $f'(x)$ — the condition is on the integrated function $f(x)$, not its derivative.
  3. I commit to the initial-condition substitution because the structure 'integrated expression + $c$ + one point' is the canonical signature of this move; any other approach (e.g. trying to match coefficients) ignores the data given.
  4. Bridge to operation: I write $F(x_0) + c = y_0$, plug in $x = x_0$ on the left, evaluate each term carefully (often the trickiest part — fractional powers, surds, or large powers like $4^4 = 256$), then isolate $c$.
  5. Tactical decision: if the supplied value is a mixed number or sum like $3 - \tfrac{5}{24}$, I convert it to a single fraction first (here $\tfrac{67}{24}$) and pick a common denominator early to keep the arithmetic clean.
  6. Finally, I substitute the determined $c$ back into $F(x)$ and write the complete particular solution — not just the value of $c$.

Mechanic — Concrete operations

  1. Write the integrated expression: $f(x) = F(x) + c$.
  2. Form the equation $F(x_0) + c = y_0$ by substituting the given $x$-coordinate.
  3. Evaluate each term of $F(x_0)$ carefully (powers, roots, fractional coefficients), simplifying to a single number or fraction.
  4. Solve the resulting linear equation $c = y_0 - F(x_0)$, using a common denominator if fractions are present.
  5. Substitute the value of $c$ back into $f(x)$ and state the full particular solution.

Worked Micro-Example

Given $f(x) = \tfrac{1}{16}x^4 - 32\sqrt{x} + c$ and $f(4) = 12$: substitute $x=4$ to get $12 = \tfrac{1}{16}(256) - 32(2) + c = 16 - 64 + c = -48 + c$, so $c = 60$ and $f(x) = \tfrac{1}{16}x^4 - 32\sqrt{x} + 60$.

Variants — Same move, different surface

Curve passes through the origin, $f(0) = 0$
Substitution kills all $x$-dependent terms, so $c$ is read off directly with no fraction arithmetic.
Differential equation (ODE) context with initial condition $y(x_0) = y_0$
Same move — integrate the ODE to get the general solution with $+c$, then substitute the initial condition.
Second-order problem starting from $f''(x)$
Two integrations produce two constants $c_1, c_2$; apply two boundary conditions sequentially (typically $f'(x_0)$ first, then $f(x_0)$).
Definite integral with explicit bounds
Move is unnecessary — the bounds eliminate $c$ during evaluation.

Common Misreadings

  • Substituting the point into $f'(x)$ (the integrand) instead of into $f(x)$ (the antiderivative).
  • Misreading a value like $f(3) = 3 - \tfrac{5}{24}$ as $f(3) = -\tfrac{5}{24}$ or treating it as two separate conditions.
  • Confusing 'the curve passes through $(a,b)$' with a point on $y = f'(x)$ rather than $y = f(x)$.

Pitfalls

  • Arithmetic slips evaluating fractional or surd terms — e.g. forgetting $\tfrac{4(27)}{3} = 36$, miscomputing $4^4 = 256$, or $\sqrt{4} = 2$.
  • Sign errors when combining terms (e.g. $16 - 64 = -48$, not $-80$ or $+48$), especially when $c$ comes out negative.
  • Failing to use a common denominator when fractions appear, leading to subtraction errors like $\tfrac{67}{24} - \tfrac{45}{2}$.
  • Leaving $c$ as a decimal (e.g. $31.2$) when an exact fraction (e.g. $\tfrac{156}{5}$) is required.
  • Stating only the value of $c$ and forgetting to write out the final $f(x)$ with $c$ substituted back in.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (P2/P3 integration) and the FP differential equations chapters. No formula-booklet entry is needed beyond standard integration results. Avoid invoking the language of 'general vs particular solution' from higher-level ODE theory — at IAL it is sufficient to say 'use the given point to find $c$'.
