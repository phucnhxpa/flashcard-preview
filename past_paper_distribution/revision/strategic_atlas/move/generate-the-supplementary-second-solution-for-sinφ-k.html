Trigonometry

Generate the supplementary second solution for sin(φ) = k

Family: Cast Symmetry Frequency: 1 questions

Trigger — When to use

Fires whenever an equation reduces to sin(φ) = k with k ∈ (0,1) (or more generally |k| < 1) and the solution domain spans more than 90° of the sine curve. Calculator gives only the principal value φ₁ = arcsin(k); a second solution exists because sine is non-injective. Cue phrases: 'find all values of θ in the interval', 'solve for 0° ≤ θ < 360°', or any domain wider than a single quadrant. Also fires after a substitution like φ = θ − α where the shifted domain still contains both Q1 and Q2.

Motivation — Why this works

Sine has the symmetry sin(180° − x) = sin(x), so for any k with 0 < k < 1 there are exactly two solutions per 360° period: the Q1 principal value and its Q2 reflection across φ = 90°. The CAST diagram shows sine is positive in Q1 and Q2, confirming both must be reported. Missing the second solution is the single most penalised slip in IAL trig-equation questions, and the supplementary identity is the cleanest way to recover it without sketching.

Thought Process — The step-by-step reasoning

  1. Notice the equation has been reduced to sin(φ) = k with k positive, and the working domain covers more than 90°.
  2. Rule out Q3/Q4 contributions because sine is negative there and k > 0, so only Q1 and Q2 can supply solutions.
  3. Commit to the supplementary identity sin(180° − φ) = sin(φ) rather than the cosine rule 360° − φ — these are easily confused.
  4. Compute the principal value φ₁ = arcsin(k) to enough decimal places to survive later rounding, then form φ₂ = 180° − φ₁.
  5. Check each candidate against the (possibly shifted) domain, then undo any substitution to return θ-values, and finally consider whether ±360° copies are needed if the domain exceeds one full cycle.

Mechanic — Concrete operations

  1. Compute principal value: φ₁ = arcsin(k), keeping ≥ 4 significant figures.
  2. Apply supplementary symmetry: φ₂ = 180° − φ₁.
  3. Add/subtract 360° to either value as needed to land inside the working domain.
  4. Discard any candidate outside the domain.
  5. Reverse any substitution (e.g. θ = φ + α) to express answers in the original variable.
  6. Round only at the final step, typically to 1 d.p. for degrees.

Worked Micro-Example

Solve $\sin\theta = 0.4$ for $0° \le \theta < 360°$. Principal: $\theta_1 = \arcsin(0.4) = 23.578°$. Supplementary: $\theta_2 = 180° - 23.578° = 156.422°$. Both lie in $[0°,360°)$, so $\theta = 23.6°,\ 156.4°$ (1 d.p.).

Variants — Same move, different surface

cos(φ) = k with k > 0
Use the reflection symmetry φ₂ = 360° − φ₁ instead of 180° − φ₁ (cosine is even about 0°/360°, not about 90°).
sin(φ) = −k (k > 0)
Solutions live in Q3 and Q4: φ₁ = 180° + arcsin(k) and φ₂ = 360° − arcsin(k); equivalently take arcsin(−k) and add 360° to get into range.
tan(φ) = k
Symmetry is φ₂ = 180° + φ₁ (period 180°), not supplementary; only one extra solution per 360°.
sin(nθ) = k with multiplier n
Solve in the expanded domain [0°, n·360°) for nθ, generate all supplementary + period copies there, then divide by n — typically yields 2n solutions.
sin(θ − α) = k with phase shift
Work in the shifted domain [−α, 360° − α) for φ = θ − α, apply supplementary rule, then add α at the end.
Domain given in radians, e.g. 0 ≤ θ < 2π
Replace 180° by π: supplementary value is φ₂ = π − φ₁.

Common Misreadings

  • Treating the equation as if sine behaved like cosine and writing 360° − φ₁ for the second solution.
  • Believing arcsin already returns all solutions in the domain.
  • Reading sin(2θ) = k as needing only two solutions instead of four.
  • Mistaking a negative k for requiring the supplementary rule rather than the Q3/Q4 construction.

Pitfalls

  • Reporting only the principal value and losing the 'all solutions' mark.
  • Writing 180° + φ₁ (the tangent rule) instead of 180° − φ₁.
  • Rounding arcsin(k) to 1 d.p. before subtracting from 180°, pushing the final answer outside the ±0.1° tolerance.
  • Forgetting to undo a substitution (e.g. leaving answers as φ rather than θ).
  • Failing to add or remove 360° when the working domain is shifted and a valid solution lies just outside [0°, 360°).

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics (P2/P3) trig-equation solving; uses only the standard identity sin(180° − x) = sin(x) and the CAST/quadrant framework expected at A level.
