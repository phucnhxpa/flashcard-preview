Proof

Disprove a universal claim by exhibiting one counterexample

Family: Proof By Counterexample Frequency: 1 questions

Trigger — When to use

Fires whenever a statement is universally quantified — phrased as 'for all n ∈ ℕ', 'every integer', 'any real x', or 'whenever ...' — and the question asks you to disprove, refute, or 'show by counterexample that the statement is false'. The presence of the word 'disprove' or 'show that ... is not true' is the strongest cue. Also fires when intuition or small-case checking suggests a pattern breaks somewhere.

Motivation — Why this works

The negation of '∀n, P(n)' is '∃n, ¬P(n)', so a single witness where the property fails logically destroys the universal claim. No algebraic generality is required — one explicit failure is a complete proof. This is far cheaper than attempting a general argument and is the standard A-level technique for refuting conjectures involving primes, divisibility, or number-theoretic properties.

Thought Process — The step-by-step reasoning

  1. I notice the claim is universal ('for all n ∈ ℕ'), and the question asks me to show it is false, so logically I need only one failing case.
  2. I rule out attempting a general algebraic disproof or expanding (n+1)³ − n³ = 3n² + 3n + 1 to argue compositeness in general — this is harder than just hunting.
  3. I commit to a counterexample search because the negation of ∀ is ∃, and exhibiting one n suffices.
  4. I bridge to action by substituting small natural numbers n = 1, 2, 3, ... and computing the expression, checking each output against the asserted property (primality).
  5. Tactically, I keep going past the first few primes; if early values all satisfy the property I do not give up — composite outputs often appear once n is large enough that 3n² + 3n + 1 has small prime factors.

Mechanic — Concrete operations

  1. Identify the universal quantifier and the property P(n) being claimed.
  2. Substitute small values n = 1, 2, 3, ... into the expression and evaluate.
  3. For each output, test the claimed property explicitly (e.g. test primality by trial division by 2, 3, 5, 7, ...).
  4. On finding a failure, state the value of n, the value of the expression, and the explicit factorisation or reason the property fails.
  5. Conclude: 'Therefore n = ... is a counterexample, so the statement is false.'

Worked Micro-Example

Disprove: 'For all $n \in \mathbb{N}$, $(n+1)^3 - n^3$ is prime.' Try $n = 5$: $(5+1)^3 - 5^3 = 216 - 125 = 91$. But $91 = 7 \times 13$, so $91$ is not prime. Hence $n = 5$ is a counterexample, and the statement is false.

Variants — Same move, different surface

Disprove 'for all n ∈ ℕ, n² + n + 41 is prime'
Search until n = 40 or use the structural observation that n = 41 forces 41 | n² + n + 41.
Disprove 'for all real x, f(x) ≥ g(x)'
Substitute one specific x-value where g(x) > f(x); evaluate both sides numerically and state the inequality reverses there.
Disprove 'every quadrilateral with equal diagonals is a rectangle'
Construct one explicit non-rectangle counterexample (e.g. an isosceles trapezium) and verify both that diagonals are equal and that it is not a rectangle.
Disprove 'for all n, 2ⁿ − 1 is prime' (Mersenne)
Try n = 4: 2⁴ − 1 = 15 = 3 × 5, composite.
Disprove a divisibility claim 'for all n, k | f(n)'
Compute f(n) for small n and exhibit one where the remainder mod k is nonzero.

Common Misreadings

  • Treating the question as 'verify' and showing the claim holds for several n, then concluding it is true — this is the wrong logical direction.
  • Providing a counterexample value but not showing why the property fails (e.g. quoting 91 without exhibiting 91 = 7 × 13).
  • Believing a counterexample must be 'small' and giving up when n = 1, 2, 3, 4 all satisfy the property.

Pitfalls

  • Stopping the search too early when initial cases happen to satisfy the property and incorrectly concluding the statement is true.
  • Arithmetic slips when computing the expression — double-check (n+1)³ − n³ at the chosen n.
  • Failing to verify compositeness properly: 91 can look prime if divisibility by 7 is not tested.
  • Omitting the explicit closing sentence identifying n as a counterexample; examiners require a clear conclusion.

A-level Scope Notes

Fully within Edexcel IAL Pure Mathematics proof content (P1/P2 proof section). Counterexample disproof is an explicitly named technique alongside direct proof, proof by exhaustion, and proof by contradiction.
