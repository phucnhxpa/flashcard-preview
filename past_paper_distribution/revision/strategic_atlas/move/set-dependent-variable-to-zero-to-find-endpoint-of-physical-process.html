Differential Equations

Set dependent variable to zero to find endpoint of physical process

Family: Boundary Interpretation Frequency: 1 questions

Trigger — When to use

Fires when an ODE-modelling question asks for the time at which a quantity 'completely' disappears, 'runs out', 'empties', 'melts', 'evaporates', or 'reaches the ground'. The cue is a physical event corresponding to the dependent variable hitting a specific boundary value (typically zero). After obtaining the particular solution $y=f(t)$ (or implicit form), substitute the boundary value to solve for $t$. Distinguish carefully from 'maximum/minimum' phrasing, which would instead set $\frac{dy}{dt}=0$.

Motivation — Why this works

In modelling, mathematical conditions encode physical events: a sphere is 'completely melted' iff its radius $r=0$, a tank is 'empty' iff volume $V=0$, a particle 'hits the ground' iff height $h=0$. Substituting the boundary value into the particular solution converts the problem into a single-variable equation for $t$. This works because the particular solution already encodes initial conditions, so the only unknown left is the time at which the terminal state is reached.

Thought Process — The step-by-step reasoning

  1. Notice the phrase 'completely melted' / 'runs out' — this is a physical terminal event, not a rate condition.
  2. Rule out setting $\frac{dr}{dt}=0$ (that would locate a stationary radius, irrelevant here since $r$ is monotonically decreasing).
  3. Commit to translating the event into the algebraic condition $r=0$ on the dependent variable.
  4. Bridge: take the particular solution $r^3 = 1728 - \tfrac{504}{5}t$ and substitute $r=0$.
  5. Solve the resulting linear equation in $t$, then sanity-check that $t$ exceeds any earlier time-point given in the problem (e.g. $t=15$ when $r=6$).

Mechanic — Concrete operations

  1. Identify the physical event and translate it to a value of the dependent variable (usually $0$).
  2. Substitute that value into the particular solution obtained earlier.
  3. Rearrange to isolate $t$.
  4. Simplify the fraction and, if needed, convert to decimal or minutes/seconds.
  5. Sense-check: confirm $t$ is positive and consistent with the direction of change.

Worked Micro-Example

Given $r^3 = 1728 - \tfrac{504}{5}t$, find when the ball has completely melted. Set $r=0$: $0 = 1728 - \tfrac{504}{5}t$, so $\tfrac{504}{5}t = 1728$, giving $t = \tfrac{1728 \times 5}{504} = \tfrac{8640}{504} = \tfrac{120}{7} \approx 17.1$ minutes.

Variants — Same move, different surface

Tank emptying: volume $V(t)$ given, find when tank is empty.
Set $V=0$ in the particular solution and solve for $t$.
Newton's law of cooling reaching ambient temperature.
Strictly $\theta \to \theta_{\text{ambient}}$ asymptotically; instead solve $\theta = \theta_{\text{ambient}} + \varepsilon$ or a stated target temperature.
Population/decay reaching a target value $N_0$.
Set $N = N_0$ (not necessarily $0$) and solve for $t$.
Projectile/height model: 'hits the ground'.
Set $h=0$ in the solution $h(t)$ and take the positive root.
Implicit solution $f(y)=g(t)+C$.
Substitute the boundary value of $y$ on the LHS, then solve the resulting equation for $t$.

Common Misreadings

  • Reading 'completely melted' as a stationary-point condition and setting $\frac{dr}{dt}=0$.
  • Confusing 'completely melted' with 'half melted' (would require $r$ at half its initial value or volume halved — check carefully).
  • Thinking the question wants the radius at a given time rather than the time at a given radius.

Pitfalls

  • Substituting into the differential equation instead of the integrated particular solution.
  • Forgetting to apply the initial condition first, leaving an unknown constant $C$ in the equation.
  • Arithmetic slip when simplifying fractions like $\tfrac{8640}{504}$ — divide by common factor $72$ to get $\tfrac{120}{7}$.
  • Taking the negative root in models like $h(t)=0$ where a quadratic gives two solutions.

A-level Scope Notes

Standard Edexcel IAL modelling step in first-order separable ODE questions (P3/P4 and FP topics); requires only substitution and linear/quadratic solving, no techniques beyond syllabus.
