Sequences & Series

Validate candidate r against |r|<1 before accepting solutions

Family: Domain Restriction Check Frequency: 1 questions

Trigger — When to use

Fires whenever a problem uses the sum-to-infinity formula $S_\infty = \frac{a}{1-r}$ and yields algebraic solutions for $r$ (often via a quadratic). Any time you've solved an equation derived from $S_\infty$, $S_n \to S_\infty$, or relationships between $S_\infty$ and partial sums, the convergence condition $|r|<1$ must be re-imposed. Cue phrases include 'sum to infinity exists', 'series converges', or being given $S_\infty$ as a finite value. The check is mandatory even when all roots happen to satisfy it.

Motivation — Why this works

The formula $S_\infty = \frac{a}{1-r}$ is only valid when $|r|<1$; outside this range the geometric series diverges and the equation you solved was built on an assumption that no longer holds. Algebraic manipulation (especially squaring or forming quadratics) can introduce extraneous roots that satisfy the derived equation but violate the original convergence premise. Examiners award an explicit mark for stating the validity check, regardless of whether any root is rejected. This is a special case of the general principle: re-impose every domain restriction that was assumed when invoking a formula.

Thought Process — The step-by-step reasoning

  1. I notice the problem invoked $S_\infty$, so my working tacitly assumed $|r|<1$.
  2. I rule out treating algebraic roots as automatically valid — the quadratic doesn't 'know' about the convergence requirement.
  3. I commit to checking $|r|<1$ for each candidate root before reporting an answer.
  4. I compute $|r|$ for each solution and compare to 1, writing the comparison explicitly.
  5. If a root fails, I reject it with a one-line reason; if all pass, I still state the check to secure the verification mark.

Mechanic — Concrete operations

  1. List all candidate values of $r$ obtained from the algebra.
  2. For each candidate, compute $|r|$ and compare to 1.
  3. Reject any $r$ with $|r|\geq 1$, citing 'series would not converge'.
  4. State the surviving value(s) explicitly as the valid common ratio(s).
  5. If the question demands a unique answer, use any extra context (e.g. positive $r$, decreasing series) to select between survivors.

Worked Micro-Example

Suppose solving for $r$ gives $5r^2 - 7r + 2 = 0$, so $r = \tfrac{2}{5}$ or $r = 1$. Check: $\left|\tfrac{2}{5}\right| = 0.4 < 1$ ✓, but $|1| = 1 \not< 1$, so $r=1$ is rejected (the series would not converge). Therefore $r = \tfrac{2}{5}$.

Variants — Same move, different surface

Quadratic in $r$ from $S_\infty$ equations yielding two roots both satisfying $|r|<1$
Both are mathematically valid; use additional problem context (sign of $r$, monotonicity, given inequality) to pick the intended one, but still write the $|r|<1$ check.
Binomial expansion $(1+x)^n$ for non-integer/negative $n$
Replace $|r|<1$ with the validity range $|x|<1$ (or the rescaled equivalent like $|2x|<1 \Rightarrow |x|<\tfrac{1}{2}$).
Problem gives $S_\infty$ in terms of an unknown (e.g. find values of $x$ for which $S_\infty$ exists)
Solve $|r(x)|<1$ as an inequality in $x$ rather than checking discrete roots.
Recurrence-defined sequence claimed to have a limit
Convergence to a fixed point typically requires $|r|<1$ in the linear part; check this before equating $L = f(L)$.

Common Misreadings

  • Checking $0 < r < 1$ instead of $|r|<1$, thereby wrongly rejecting valid negative common ratios.
  • Treating $|r|\leq 1$ as the condition and accepting $r = \pm 1$ (the series diverges or oscillates at the boundary).
  • Assuming the quadratic's solutions are automatically valid because the problem mentioned $S_\infty$.

Pitfalls

  • Omitting the explicit check — losing the verification mark even when both roots are valid.
  • Writing $|r|<1$ but failing to actually compute $|r|$ for each candidate.
  • Forgetting to reject $r=1$ (which makes the formula undefined via division by zero) when it appears as a quadratic root.

A-level Scope Notes

Standard Edexcel IAL P2/C2 geometric series content; the $|r|<1$ condition for $S_\infty$ is examined directly and the analogous validity range appears in P4 binomial expansions.
