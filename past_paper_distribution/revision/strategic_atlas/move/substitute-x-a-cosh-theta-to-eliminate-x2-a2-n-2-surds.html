Trigger — When to use

The integrand contains a factor of the form (x² − a²)^{n/2} (typically n = 1, 3) with |x| ≥ a, e.g. √(x² − 16) or (x² − 16)^{3/2} in numerator or denominator. Often the question explicitly directs the substitution x = a cosh θ, but it should also be the student's reflex when they see x² − a² under a fractional power. Pure rational integrands of x² − a² (no surd) usually want partial fractions instead, so the surd is the key cue.

Motivation — Why this works

The identity cosh²θ − 1 = sinh²θ converts x² − a² into a²sinh²θ, a perfect square, so any half-integer power becomes a clean polynomial in sinh θ. Combined with dx = a sinh θ dθ, one factor of sinh θ from dx typically cancels into the surd, leaving an integrand expressible purely in cosh θ and sinh θ that can be tackled by hyperbolic identities or double-angle formulae. This is preferred over x = a sec θ because it avoids absolute values and sec/tan acrobatics.

Thought Process — Step-by-step reasoning

  1. Notices a surd of the form (x² − a²)^{n/2} with |x| > a — a classic hyperbolic-sub signal.
  2. Rules out x = a sinh θ (gives x² + a²) and partial fractions (no surd would be needed); also notes x = a sec θ is messier.
  3. Commits to x = a cosh θ because cosh²θ − 1 = sinh²θ kills the surd cleanly and the domain |x| ≥ a matches cosh θ ≥ 1.
  4. Computes the three ingredients: x² in cosh², (x² − a²)^{n/2} as a^n sinh^n θ, and dx = a sinh θ dθ.
  5. Assembles and simplifies, expecting cancellation of one sinh factor from dx against sinh's in the surd, leaving a hyperbolic integrand to be reduced by identities like cosh²θ = (1 + cosh 2θ)/2 or coth²θ = 1 + cosech²θ.

Mechanic — Concrete operations

  1. Set x = a cosh θ, so dx = a sinh θ dθ.
  2. Compute x² − a² = a²(cosh²θ − 1) = a² sinh²θ.
  3. Raise to the required power: (a² sinh²θ)^{n/2} = a^n sinh^n θ (valid since sinh θ ≥ 0 on the chosen branch).
  4. Rewrite any x² in numerator as a² cosh²θ.
  5. Substitute everything, cancel one sinh θ from dx with the surd, and simplify to a pure hyperbolic integral.
  6. Convert limits if definite: x = a ↔ θ = 0, and x = X ↔ θ = arcosh(X/a).

Worked Micro-Example

Evaluate $\int \dfrac{x^2}{(x^2-16)^{3/2}}\,dx$. Let $x = 4\cosh\theta$, $dx = 4\sinh\theta\,d\theta$. Then $(x^2-16)^{3/2} = (16\sinh^2\theta)^{3/2} = 64\sinh^3\theta$ and $x^2 = 16\cosh^2\theta$. The integral becomes $\int \dfrac{16\cosh^2\theta}{64\sinh^3\theta}\cdot 4\sinh\theta\,d\theta = \int \coth^2\theta\,d\theta = \int(1+\operatorname{cosech}^2\theta)\,d\theta = \theta - \coth\theta + C$.

Variants — Same move, different surface

Integrand contains (x² + a²)^{n/2}
Use x = a sinh θ; then x² + a² = a² cosh²θ and dx = a cosh θ dθ.
Integrand contains (a² − x²)^{n/2} with |x| < a
Use x = a sin θ (standard trig sub); hyperbolic does not apply cleanly here.
Surd of form √(x² − a²) alone (n = 1)
Same x = a cosh θ; the integral typically reduces to ∫ sinh²θ dθ via the double-angle identity.
Shifted form like ((x − b)² − a²)^{n/2}
First let u = x − b, then apply u = a cosh θ.
Trig alternative x = a sec θ
Also valid but produces ∫ involving sec, tan and absolute values; hyperbolic is usually cleaner in IAL FP.

Common Misreadings

  • Choosing x = 4 sinh θ for x² − 16, which actually generates x² + 16.
  • Treating (x² − 16)^{3/2} as if it were x² − 16 and trying partial fractions.
  • Forgetting the dx = 4 sinh θ dθ factor when substituting.
  • Reading the surd as needing x = 4 sin θ (only valid for a² − x²).

Pitfalls

  • Mis-evaluating 16^{3/2}: the correct value is 64, not 48.
  • Dropping the absolute value: (sinh²θ)^{3/2} = sinh³θ only when sinh θ ≥ 0, which is fine for θ ≥ 0 / x ≥ a but needs care if x ≤ −a.
  • Forgetting to convert limits to θ in definite integrals, or converting back to x sloppily — recall cosh θ = x/a, sinh θ = √(x² − a²)/a.
  • Failing to apply the right hyperbolic identity at the simplification stage (e.g., cosh²θ = (1 + cosh 2θ)/2).

A-level Scope Notes

Fully within Edexcel IAL Further Pure: hyperbolic substitutions, the identity cosh²θ − sinh²θ = 1, and inverse hyperbolic functions (arcosh) are all syllabus content in FP2/FP3.
