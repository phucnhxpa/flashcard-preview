Trigonometry

Read period of sin/cos from coefficient of x

Family: Graph Transformation Frequency: 1 questions

Trigger — When to use

Fires when a question asks for the period (or equivalently the smallest positive value of x for which the curve repeats) of a function in the form y = sin(Bx), y = cos(Bx), or y = tan(Bx), possibly with phase shift or vertical transformations. Typical cues: 'state the period', 'find the smallest positive x such that f(x+a) = f(x)', or sketching tasks where the horizontal scale must be marked. The coefficient B may be a fraction (e.g. x/12), a decimal, or include a phase term inside the bracket.

Motivation — Why this works

Sine and cosine complete one full cycle whenever their argument increases by 2π, so for y = sin(Bx) the input x must increase by 2π/|B| to advance the argument by 2π. This gives the period formula directly: 2π/|B| for sin and cos, π/|B| for tan (which repeats every π). Vertical scalings, vertical shifts, and horizontal phase shifts do not affect the period — only the coefficient of x inside the trig bracket matters.

Thought Process — The step-by-step reasoning

  1. Notice the function has the form sin/cos/tan of a linear expression in x, and the question targets period.
  2. Rule out plotting or solving f(x) = f(x+T) algebraically — these are slower and error-prone.
  3. Commit to the standard period formula because the coefficient of x is clearly visible.
  4. Rewrite the argument so B is explicit; e.g. sin(x/12) = sin((1/12)x), so B = 1/12.
  5. Apply period = 2π/|B| (or π/|B| for tan), being careful with reciprocals when B is a fraction.

Mechanic — Concrete operations

  1. Write the function in the form sin(Bx + φ), cos(Bx + φ), or tan(Bx + φ).
  2. Identify B as the coefficient of x (not the constant term).
  3. Choose the correct base period: 2π for sin/cos, π for tan.
  4. Compute period = (base period)/|B|, simplifying any compound fraction.

Worked Micro-Example

Find the period of $y = \sin(x/12)$.\ Write as $y = \sin\!\left(\tfrac{1}{12}x\right)$, so $B = \tfrac{1}{12}$.\ Period $= \dfrac{2\pi}{|B|} = \dfrac{2\pi}{1/12} = 2\pi \times 12 = 24\pi.$

Variants — Same move, different surface

y = cos(3x + π)
B = 3, period = 2π/3; the phase shift π does not affect the period.
y = tan(Bx)
Use base period π, so period = π/|B|.
y = 5sin(2x) − 1
Vertical stretch and shift do not change the period; period = 2π/2 = π.
y = sin(−2x)
Use |B|: period = 2π/|−2| = π. Negative B reflects the graph but preserves the period.
Argument given in degrees, e.g. y = sin(Bx°)
Use base period 360° instead of 2π, so period = 360°/|B|.

Common Misreadings

  • Confusing sin(x/12) with sin(12x), giving B = 12 and period π/6 rather than 24π.
  • Treating a phase shift inside the bracket (e.g. the π in cos(3x+π)) as if it changed the period.
  • Using the 2π formula for tan, or the π formula for sin/cos.

Pitfalls

  • Dividing incorrectly by a fractional B: writing 2π × (1/12) instead of 2π ÷ (1/12).
  • Dropping the absolute value when B is negative.
  • Forgetting to convert when the argument is in degrees, leaving 2π where 360° was needed.

A-level Scope Notes

Fully within Edexcel IAL Pure Maths (trig graphs and transformations in P2/P3); applies to standard sin, cos, tan transformations without exceeding syllabus.
