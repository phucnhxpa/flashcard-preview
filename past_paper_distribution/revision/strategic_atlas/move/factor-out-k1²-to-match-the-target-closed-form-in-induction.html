Proof

Factor out (k+1)² to match the target closed form in induction

Family: Induction Target Factoring Frequency: 1 questions

Trigger — When to use

Fires in the inductive step of a sum-of-cubes (or analogous Σr^p) proof when you have assumed Σ_{r=1}^{k} r³ = ¼k²(k+1)² and added (k+1)³, giving ¼k²(k+1)² + (k+1)³. The target ¼(k+1)²(k+2)² already displays (k+1)² as a factor, signalling that (k+1)² should be extracted from both terms. Cue phrases: 'show the result holds for n=k+1', 'assume true for n=k', and a target whose factorisation visibly shares (k+1)² with the hypothesis term.

Motivation — Why this works

The closed form ¼n²(n+1)² evaluated at n=k+1 is ¼(k+1)²(k+2)², so the (k+1)² factor must appear in any successful manipulation. Pulling (k+1)² out of ¼k²(k+1)² + (k+1)³ leaves a single quadratic bracket k²/4 + (k+1) = (k+2)²/4, which is the missing piece of the target. This works because (k+1)³ = (k+1)²·(k+1), so the shared factor is structurally guaranteed — not coincidence — for any sum-of-powers identity.

Thought Process — The step-by-step reasoning

  1. Notices that the target ¼(k+1)²(k+2)² contains (k+1)², and the hypothesis term ¼k²(k+1)² already contains (k+1)².
  2. Rules out brute-force expansion of both sides to a quartic in k — algebraically valid but slow and error-prone, and harder to present as a clean factorisation.
  3. Commits to factoring (k+1)² from ¼k²(k+1)² + (k+1)³ because (k+1)³ = (k+1)²(k+1), so the common factor is immediate.
  4. Bridges to the first operation: write the sum as (k+1)²[k²/4 + (k+1)] and aim to simplify the bracket to (k+2)²/4.
  5. Inside the move, decides to put the bracket over a common denominator 4 to expose k² + 4k + 4, then recognises the perfect square (k+2)².

Mechanic — Concrete operations

  1. State sum at n=k+1: Σ_{r=1}^{k+1} r³ = ¼k²(k+1)² + (k+1)³ using the inductive hypothesis.
  2. Factor (k+1)²: = (k+1)²[¼k² + (k+1)].
  3. Combine over denominator 4: ¼k² + k + 1 = (k² + 4k + 4)/4.
  4. Recognise perfect square: k² + 4k + 4 = (k+2)², so bracket = (k+2)²/4.
  5. Conclude: Σ_{r=1}^{k+1} r³ = ¼(k+1)²(k+2)², matching the target form.
  6. Write the formal closing sentence: 'true for n=1, and if true for n=k then true for n=k+1, hence true for all n∈ℤ⁺ by induction.'

Worked Micro-Example

Assume Σ_{r=1}^{k} r³ = ¼k²(k+1)². Then Σ_{r=1}^{k+1} r³ = ¼k²(k+1)² + (k+1)³ = (k+1)²[¼k² + (k+1)] = (k+1)²·(k² + 4k + 4)/4 = ¼(k+1)²(k+2)², as required.

Variants — Same move, different surface

Sum of squares: Σ r² = ⅙n(n+1)(2k+1)
Factor (k+1) (not (k+1)²) from ⅙k(k+1)(2k+1) + (k+1)², then simplify the bracket to ⅙(k+2)(2k+3).
Sum Σ r(r+1) or Σ r(r+1)(r+2) telescoping-style identities
Factor the highest common polynomial factor visible in the target — typically (k+1) or (k+1)(k+2) — and reduce the residual bracket.
Σ ar^{r-1} or geometric-style sums in Further Pure induction
Common factor is a power of r or (r-1); extract it and simplify the residual to match the closed form at n=k+1.
Matrix induction Aⁿ with target containing (n+1) in entries
After computing A^k · A using the hypothesis, factor scalar coefficients (often involving 2^{k+1} or similar) so each entry matches the target with k replaced by k+1.

Common Misreadings

  • Treating (k+1)³ as not sharing a factor with ¼k²(k+1)² and expanding everything into a quartic.
  • Factoring only (k+1) instead of (k+1)², leaving an awkward cubic bracket that hides the perfect square.
  • Believing the move is a clever trick rather than a structural consequence of (k+1)^{p+1} = (k+1)^p·(k+1).

Pitfalls

  • Forgetting to put ¼k² + (k+1) over a common denominator of 4 before trying to factor the bracket.
  • Mis-expanding (k+2)² as k² + 2k + 4 or k² + 4k + 2 instead of k² + 4k + 4.
  • Stopping at ¼(k+1)²(k+2)² without writing the formal induction conclusion (base case verified, inductive step shown, hence true for all n∈ℤ⁺).
  • Dropping the ¼ when factoring, producing (k+1)²[k² + 4(k+1)] without tracking the denominator.

A-level Scope Notes

Standard Edexcel IAL Further Pure 1 induction technique for Σr³ = ¼n²(n+1)²; the same factor-matching strategy underlies all sum-of-powers and matrix-power induction proofs in FP1/FP2.
