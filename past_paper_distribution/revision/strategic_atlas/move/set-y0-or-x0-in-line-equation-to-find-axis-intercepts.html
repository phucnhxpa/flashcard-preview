Coordinate Geometry

Set y=0 or x=0 in line equation to find axis intercepts

Family: Intercept Finding Frequency: 1 questions

Trigger — When to use

The question names a point as where a line 'crosses', 'meets', or 'cuts' the x-axis or y-axis, or labels A, B, C as specific axis intercepts. Phrasings include 'the line meets the x-axis at', 'the y-intercept is', or diagrams showing a line hitting an axis. It also fires implicitly when a later subpart needs the base/height of a triangle whose vertices lie on the axes.

Motivation — Why this works

Every point on the x-axis has y-coordinate 0, and every point on the y-axis has x-coordinate 0. Imposing one of these constraints on the line equation collapses it from a relation in two variables to a one-variable linear equation, which solves directly for the remaining coordinate. This is the defining algebraic characterisation of an axis intercept.

Thought Process — The step-by-step reasoning

  1. I notice the point is described as lying on an axis, so one of its coordinates is forced to be 0.
  2. I rule out solving simultaneous equations or using a gradient formula — intersecting with an axis is a one-line substitution, not a two-line intersection.
  3. I commit because the constraint y=0 (or x=0) is exact and converts the line equation into a one-step solve.
  4. I read the equation, decide which variable to zero out (axis name tells me the OTHER variable is the one I keep), and substitute.
  5. I keep the coordinates in (x,y) form for use downstream, and note any shared zero-coordinates that may simplify a later area or distance step.

Mechanic — Concrete operations

  1. Identify which axis the point lies on: x-axis ⇒ set y=0; y-axis ⇒ set x=0.
  2. Substitute the zero into the line equation.
  3. Solve the resulting one-variable linear equation for the remaining coordinate.
  4. Write the intercept as an ordered pair (x,0) or (0,y).
  5. Repeat for each named intercept; record any shared coordinate (e.g. two points on the x-axis) for later use.

Worked Micro-Example

Find where $3x-2y+12=0$ meets the axes. x-intercept: set $y=0$, giving $3x+12=0$, so $x=-4$, point $(-4,0)$. y-intercept: set $x=0$, giving $-2y+12=0$, so $y=6$, point $(0,6)$.

Variants — Same move, different surface

Line given in $y=mx+c$ form
Read the y-intercept directly as $(0,c)$; for the x-intercept set $y=0$ to get $x=-c/m$.
Line given in two-point or point–gradient form
First rearrange into general or slope-intercept form, then substitute $y=0$ or $x=0$.
Curve (e.g. quadratic $y=ax^2+bx+c$) meeting an axis
Setting $y=0$ now gives a quadratic to solve (factorise/formula); setting $x=0$ still gives the y-intercept directly as $(0,c)$.
Circle $(x-a)^2+(y-b)^2=r^2$ meeting an axis
Substituting $y=0$ or $x=0$ yields a quadratic in the remaining variable; expect 0, 1, or 2 intercepts depending on the discriminant.
Tangent or normal line derived in calculus questions
Once the tangent/normal equation is formed, apply the same $y=0$ / $x=0$ substitution to find where it cuts the axes.

Common Misreadings

  • Treating 'crosses the x-axis' as meaning x=0 instead of y=0 (confusing the axis name with the coordinate to zero out).
  • Assuming an axis intercept must be found by solving simultaneous equations with another line.
  • Reading the constant term of a general-form equation $ax+by+c=0$ directly as an intercept without dividing by $a$ or $b$.

Pitfalls

  • Setting the wrong variable to zero (x=0 when seeking the x-intercept).
  • Sign errors when isolating the variable, especially with negative coefficients.
  • Forgetting to write the answer as a coordinate pair, leaving it as a bare number.
  • For curves, missing a second intercept by stopping after one root of the quadratic.

A-level Scope Notes

Fully within Edexcel IAL P1 coordinate geometry of straight lines, and extends naturally to quadratics (P1) and circles (P2) where the same zero-substitution applies.
