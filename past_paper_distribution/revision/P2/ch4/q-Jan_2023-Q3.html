Worked solution

To solve this question, we'll break it down into its three parts, addressing each one systematically.

Part (a): Find the value of $u_3$ and $u_4$.

Thought Process: We are given the first term, $u_1=3$, and a recurrence relation, $u_{n+1} = 2u_n - 5$. This means each subsequent term is defined by the previous term. To find $u_3$ and $u_4$, we need to calculate the terms in order: first $u_2$ using $u_1$, then $u_3$ using $u_2$, and finally $u_4$ using $u_3$.

Working:
Given $u_1 = 3$ and $u_{n+1} = 2u_n - 5$.

To find $u_2$, substitute $n=1$ into the recurrence relation:
$$u_2 = 2u_1 - 5 = 2(3) - 5 = 6 - 5 = 1$$

To find $u_3$, substitute $n=2$ into the recurrence relation (using $u_2$):
$$u_3 = 2u_2 - 5 = 2(1) - 5 = 2 - 5 = -3$$

To find $u_4$, substitute $n=3$ into the recurrence relation (using $u_3$):
$$u_4 = 2u_3 - 5 = 2(-3) - 5 = -6 - 5 = -11$$

Part (b): Evaluate $\sum_{r=1}^4 (2u_r + 1)$.

Thought Process: The summation notation $\sum_{r=1}^4 (2u_r + 1)$ means we need to calculate the expression $(2u_r + 1)$ for each value of $r$ from 1 to 4, and then add all these results together. We have already found $u_1, u_2, u_3,$ and $u_4$ in part (a).

Working:
We need to calculate $(2u_r + 1)$ for $r=1, 2, 3, 4$:
For $r=1$: $2u_1 + 1 = 2(3) + 1 = 6 + 1 = 7$
For $r=2$: $2u_2 + 1 = 2(1) + 1 = 2 + 1 = 3$
For $r=3$: $2u_3 + 1 = 2(-3) + 1 = -6 + 1 = -5$
For $r=4$: $2u_4 + 1 = 2(-11) + 1 = -22 + 1 = -21$

Now, sum these values:
$$\sum_{r=1}^4 (2u_r + 1) = (2u_1+1) + (2u_2+1) + (2u_3+1) + (2u_4+1)$$
$$= 7 + 3 + (-5) + (-21)$$
$$= 10 - 5 - 21$$
$$= 5 - 21 = -16$$

Part (c): Find the limit of this sequence as $n \to \infty$.

Thought Process: For a sequence defined by a recurrence relation $u_{n+1} = f(u_n)$ to converge to a limit $L$, it must be that as $n$ approaches infinity, both $u_n$ and $u_{n+1}$ approach the same limit $L$. Therefore, we can replace $u_n$ with $L$ and $u_{n+1}$ with $L$ in the recurrence relation and solve for $L$.

Working:
Given the recurrence relation $u_{n+1} = 2u_n - 5$.
If the sequence converges to a limit $L$ as $n \to \infty$, then we can set $u_{n+1} = L$ and $u_n = L$.
$$L = 2L - 5$$
Now, we solve this algebraic equation for $L$:
Subtract $2L$ from both sides:
$$L - 2L = -5$$
$$-L = -5$$
Multiply by $-1$:
$$L = 5$$

Answer:
(a) $u_3 = -3$, $u_4 = -11$
(b) $-16$
(c) $L = 5$

Key techniques

  • Iterate sequence terms
  • Apply summation notation
  • Solve for fixed point of recurrence

Common pitfalls

  • Calculation errors when substituting negative terms into the recurrence relation.
  • Errors in sign or order of operations when evaluating the summation.
  • Incorrectly setting up the equation for the limit, e.g., forgetting to replace both $u_n$ and $u_{n+1}$ with $L$.
  • Algebraic errors when solving for the limit $L$.
