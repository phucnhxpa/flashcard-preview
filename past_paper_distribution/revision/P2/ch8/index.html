Trapezium Rule numerical integration
#1 most important
8×
3-8m

A table of x and y values (some may be missing and need calculating), or a function to evaluate at given x-values. Often followed by a part (b) asking you to use the trapezium rule result combined with integral properties (linearity, logarithm laws) to find a related integral.

Triggerstrapezium ruleestimateapproximate5 ordinates4 stripstableh =numerical integration
Assumptions
  • Strips are of equal width h = (b - a) / n where n is the number of strips
  • y-values must be evaluated in radians if trigonometric functions are involved
  • The trapezium rule gives an approximation — it overestimates for concave-up curves and underestimates for concave-down curves
  • More strips (smaller h) generally improves accuracy
Approaches
A: Direct application of trapezium rule formula
When: Always — this is the standard and only approach for trapezium rule questions
  1. Identify the number of strips n and calculate h = (b - a) / n
  2. List all y-values (ordinates) y₀, y₁, y₂, ..., yₙ, calculating any missing values to required decimal places
  3. Apply the formula: ∫ ≈ (h/2)[y₀ + yₙ + 2(y₁ + y₂ + ... + yₙ₋₁)]
  4. Compute the sum carefully, showing working
  5. Round only the final answer to the required number of decimal places or significant figures
+ Pros: Straightforward mechanical process; easy marks if careful
– Cons: Arithmetic-heavy; one slip ruins the answer
B: Using the trapezium result with integral properties (follow-up parts)
When: When part (b) asks for a related integral such as ∫(af(x) + b) dx or uses logarithm laws
  1. Recognise linearity: ∫[af(x) + g(x)] dx = a∫f(x) dx + ∫g(x) dx
  2. Integrate any simple terms exactly (e.g., ∫k dx = k(b-a), ∫1/x dx = ln|x|)
  3. Substitute the trapezium rule estimate from part (a) into the expression
  4. Apply logarithm laws if needed (e.g., ln A + ln B = ln(AB)) to simplify the final answer
+ Pros: Builds on part (a) result so partial marks are available even if (a) is slightly off
– Cons: Must correctly identify which part of the integral is estimated and which is exact
Formulas
$\int_a^b y\,dx \approx \frac{h}{2}\left[y_0 + y_n + 2(y_1 + y_2 + \cdots + y_{n-1})\right]$ (formula booklet)
$h = \frac{b-a}{n}$ where $n$ = number of strips (MEMORIZE)
Pitfalls
  • Using h instead of h/2 in the formula
  • Doubling the first and last ordinates instead of the intermediate ones
  • Using h = total interval width instead of dividing by number of strips
  • Confusing '5 ordinates' (= 4 strips) with '5 strips'
  • Using degrees instead of radians for trigonometric functions
  • Rounding intermediate y-values too early, causing accumulated error

Integration by substitution (indefinite and definite)

A substitution is usually given (e.g., u = 2x + 1, u² = 3x - 1). You must express the entire integrand in terms of u and du, integrate, then either convert back to x (indefinite) or change limits (definite). May involve partial fractions after substitution.

Triggersusing the substitutionu =show thathence findby substitution
Assumptions
  • The given substitution transforms the integral into a simpler standard form
  • For definite integrals, limits must be converted from x-values to u-values using the substitution
  • For indefinite integrals, the final answer must be expressed back in terms of x
  • dx must be fully replaced using du/dx or dx = f(u) du
Approaches
A: Standard u-substitution with given substitution
When: When the substitution is provided in the question (most common scenario in P2)
  1. Write down the substitution u = g(x) and find du/dx
  2. Rearrange to get dx in terms of du: dx = du / g'(x)
  3. Express x in terms of u if needed (rearrange the substitution)
  4. Replace ALL x's and dx in the integrand with expressions in u and du
  5. Simplify the resulting integral in u
  6. If partial fractions are needed, decompose and integrate each term
  7. Integrate with respect to u using standard results
  8. For indefinite integrals: substitute back u = g(x) to give answer in x, add + C
  9. For definite integrals: change limits using u = g(x) at each x-limit, then evaluate
+ Pros: Systematic process; substitution is given so no guesswork
– Cons: Algebraic manipulation can be complex; easy to leave an x in the integrand by mistake
B: Reverse chain rule (special case)
When: When the integrand is of the form f'(g(x)) · g'(x) — i.e., the derivative of the 'inner function' appears as a factor
  1. Recognise the integrand as f'(g(x)) · g'(x)
  2. Write the integral as f(g(x)) + C directly
  3. Verify by differentiating your answer using the chain rule
+ Pros: Very fast when the pattern is spotted
– Cons: Only works for specific forms; not always obvious
Formulas
$\int f(g(x))\,g'(x)\,dx = F(g(x)) + C$ where $F' = f$ (MEMORIZE — reverse chain rule)
$\int \frac{1}{u}\,du = \ln|u| + C$ (MEMORIZE)
$\int u^n\,du = \frac{u^{n+1}}{n+1} + C$ for $n \neq -1$ (formula booklet)
Partial fractions: $\frac{A}{u} + \frac{B}{u+k}$ for linear factors (MEMORIZE)
Pitfalls
  • Forgetting to change limits from x to u for definite integrals
  • Leaving x terms in the integrand after substitution — everything must be in u and du
  • Errors when expressing x in terms of u (e.g., if u² = 3x - 1 then x = (u² + 1)/3)
  • Forgetting dx = (du/dx)⁻¹ du — not just replacing dx with du
  • Not converting back to x for indefinite integrals
  • Algebraic errors with fractional or negative powers after substitution

Assumptions
  • Curves are continuous in the region of interest
  • Intersection points define the limits of integration
  • If a cubic must be solved, one root is often given or obvious (e.g., x = 0 or x = 1) — use this to factorise
  • The shaded region is typically between two consecutive intersection points
Approaches
A: Single integral of (upper − lower) between intersection points
When: When one curve is consistently above the other in the region
  1. Equate the two curve equations to find intersection points
  2. If cubic: use a known root to factor, then solve the quadratic factor
  3. Determine which curve is on top in the region (substitute a test x-value)
  4. Set up ∫ₐᵇ [upper(x) - lower(x)] dx
  5. Expand and simplify the integrand
  6. Integrate term by term
  7. Evaluate: F(b) - F(a)
  8. Ensure the final answer is positive
+ Pros: Single integral; most efficient
– Cons: Must correctly identify upper vs lower curve
B: Split region into sub-areas
When: When the curves cross within the region, or when the region is bounded by different curves on different sides
  1. Find all intersection points including any crossing within the region
  2. Split the area into sub-regions where one curve is consistently above the other
  3. Integrate each sub-region separately
  4. Add the absolute values of all sub-areas
+ Pros: Handles crossing curves correctly
– Cons: More integrals to compute; more chances for error
Formulas
$\text{Area} = \int_a^b |f(x) - g(x)|\,dx$ (MEMORIZE)
$\int (ax^3 + bx^2 + cx + d)\,dx = \frac{a}{4}x^4 + \frac{b}{3}x^3 + \frac{c}{2}x^2 + dx + C$ (MEMORIZE)
Pitfalls
  • Subtracting the curves in the wrong order, yielding a negative 'area'
  • Algebraic errors when solving the cubic or factorising
  • Errors when evaluating polynomial expressions at fractional limits (e.g., (1-m)^3)
  • Forgetting to check the answer is positive and makes sense geometrically
  • Not simplifying the integrand before integrating, leading to unnecessary complexity

#4
3×
3-7m

A function given as a fraction or with roots/negative powers. You must rewrite it in the form ax^n before integrating term by term. May ask you to find f(x) given f'(x) and a point, or compare your integral with a given form to find constants.

Triggersfind ∫integratefind f(x)constant of integrationgiven that f'(x)
Assumptions
  • Always add + C for indefinite integrals
  • The power rule applies for all n ≠ -1
  • For n = -1, ∫x⁻¹ dx = ln|x| + C
  • Expressions must be written as sums of terms ax^n before integrating — you cannot integrate a product or quotient directly using the power rule
Approaches
A: Rewrite then apply power rule term by term
When: When the integrand is a sum of terms that can be written as ax^n after algebraic manipulation
  1. Rewrite roots as fractional powers: √x = x^(1/2), ∛x = x^(1/3), etc.
  2. Rewrite fractions: split numerator term by term, e.g., (8 - 5x²)/(2x²) = 4x⁻² - 5/2
  3. Apply ∫ax^n dx = (a·x^(n+1))/(n+1) to each term
  4. For ∫x⁻¹ dx, write ln|x|
  5. Add + C
  6. If given a point on the curve, substitute to find C
+ Pros: Works for any rational power; systematic and reliable
– Cons: Requires careful algebraic manipulation before integrating
B: Compare coefficients with given form
When: When the question gives the integral in a specific form and asks you to find unknown constants (e.g., 'show that ∫f(x)dx = Ax^p + Bx^q + C')
  1. Integrate normally using approach A
  2. Write your result in the same form as the given expression
  3. Compare coefficients or powers to identify the unknown constants
  4. State each constant clearly
+ Pros: Built-in check — your integral must match the given form
– Cons: If your integration is wrong, you cannot recover the constants
Formulas
$\int x^n\,dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$ (formula booklet)
$\int x^{-1}\,dx = \ln|x| + C$ (MEMORIZE)
$\int e^{ax}\,dx = \frac{1}{a}e^{ax} + C$ (formula booklet)
Pitfalls
  • Forgetting to rewrite 1/x^n as x^(-n) before integrating
  • Not splitting a fraction into separate terms before integrating
  • Forgetting the + C for indefinite integrals (costs a mark every time)
  • Confusing the power rule with differentiation (adding 1 vs subtracting 1)
  • Errors with negative or fractional indices when adding 1 to the power

Triggersturning pointstationary pointdy/dx = 0exact areahence findproduct rule
Assumptions
  • Turning points occur where dy/dx = 0
  • The x-coordinate of the turning point is often used as a limit of integration
  • The area is exact — do not use decimal approximations
Approaches
A: Differentiate, find turning points, then integrate for area
When: Standard approach for these combined questions
  1. Differentiate using the appropriate rule (product rule, chain rule, quotient rule)
  2. Set dy/dx = 0 and solve — factorise if possible
  3. Find the y-coordinate(s) of turning point(s) by substituting x back into the original
  4. Set up the definite integral with correct limits (often 0 to the turning point x-value)
  5. Integrate using the appropriate technique (substitution, by parts, or standard forms)
  6. Evaluate the definite integral for the exact area
+ Pros: Logical flow from differentiation to integration; tests both skills
– Cons: Long question; errors in differentiation propagate to wrong limits for integration
Formulas
Product rule: $\frac{d}{dx}[uv] = u\frac{dv}{dx} + v\frac{du}{dx}$ (formula booklet)
Chain rule: $\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$ (formula booklet)
Pitfalls
  • Incorrectly applying the product rule for differentiation (mixing up terms)
  • Failing to factorise dy/dx properly to find all turning points
  • Using approximate x-values as limits instead of exact values
  • Choosing the wrong integration technique for the second part

Triggersintegration by parts∫ x² e^x dx∫ x sin x dx∫ x ln x dxhence find the exact areashow that
Assumptions
  • The formula for integration by parts is given in the formula booklet
  • Choose u as the function that simplifies when differentiated (usually the polynomial or ln x)
  • dv/dx should be the function that is easily integrated (usually e^x, sin x, cos x)
  • For ln x integrands, let u = ln x since ln x cannot be easily integrated directly
Approaches
A: Standard integration by parts (LIATE rule for choosing u)
When: For products of polynomials with exponentials, trig, or ln x
  1. Identify u and dv/dx using LIATE priority: Logarithmic > Inverse trig > Algebraic > Trigonometric > Exponential (u is chosen from earliest in list)
  2. Differentiate u to get du/dx
  3. Integrate dv/dx to get v
  4. Apply: ∫u(dv/dx)dx = uv - ∫v(du/dx)dx
  5. If the resulting integral is still a product, apply integration by parts again
  6. For definite integrals, apply limits [a,b] to the uv term AND to the remaining integral
  7. Simplify and evaluate
+ Pros: Systematic; works for all standard product integrals in P2
– Cons: Repeated application can be lengthy; sign errors accumulate
B: Tabular method (for repeated integration by parts with polynomials × e^x or trig)
When: When integrating x^n · e^(ax) or x^n · sin(ax)/cos(ax) where n ≥ 2, requiring multiple applications
  1. Create a table with columns: sign (+/-), derivatives of u (polynomial), integrals of dv/dx
  2. Alternate signs starting with +
  3. Differentiate the polynomial column until you reach 0
  4. Integrate the other function column the same number of times
  5. Multiply diagonally and sum all terms
  6. Add + C (or apply limits for definite integrals)
+ Pros: Very fast for repeated by-parts; reduces sign errors
– Cons: Must be familiar with the method; examiners expect standard working so show the formula
Formulas
$\int u\,\frac{dv}{dx}\,dx = uv - \int v\,\frac{du}{dx}\,dx$ (formula booklet)
$\int e^{ax}\,dx = \frac{1}{a}e^{ax} + C$ (formula booklet)
$\int \sin(ax)\,dx = -\frac{1}{a}\cos(ax) + C$ (formula booklet)
$\int \cos(ax)\,dx = \frac{1}{a}\sin(ax) + C$ (formula booklet)
$\int \ln x\,dx = x\ln x - x + C$ (MEMORIZE — derived by parts with u = ln x, dv = dx)
Pitfalls
  • Choosing u and dv/dx the wrong way round (e.g., trying to integrate ln x as dv/dx)
  • Sign errors when applying the formula, especially the minus sign before the second integral
  • Forgetting to apply limits to BOTH the uv term and the remaining integral in definite integration
  • Algebraic errors when simplifying e^0 = 1 or evaluating trig at 0 and π
  • Not recognising that a second (or third) application of by-parts is needed
  • Dropping the negative sign when integrating -∫v(du/dx)dx

Triggersareaenclosedboundedregionshadedbetween the curve and the linex-axis
Assumptions
  • Area is always positive — take the absolute value or ensure you subtract the lower curve from the upper
  • Intersection points must be found exactly (not estimated) for definite integration limits
  • If a region crosses the x-axis, split into separate integrals for portions above and below
Approaches
A: Integrate (upper curve − lower curve) between intersection points
When: When finding area between two curves or between a curve and a line
  1. Find intersection points by setting the two functions equal and solving
  2. Determine which function is upper (greater y-value) in the region of interest — sketch if unsure
  3. Set up ∫ₐᵇ [f(x) - g(x)] dx where f(x) ≥ g(x) on [a,b]
  4. Expand and simplify the integrand
  5. Integrate term by term
  6. Evaluate at the limits: F(b) - F(a)
  7. Check the answer is positive
+ Pros: Directly gives the area as a positive value when done correctly
– Cons: Must correctly identify which curve is on top; sign errors when evaluating at limits
B: Compute separate areas and subtract
When: When one region is a simple geometric shape (e.g., triangle, trapezium) that can be calculated without integration
  1. Find intersection points
  2. Calculate the area under the curve using ∫ₐᵇ f(x) dx
  3. Calculate the area under the line using integration or the geometric formula for a triangle/trapezium
  4. Subtract the smaller area from the larger to get the enclosed area
+ Pros: Useful when one area is a simple shape; reduces integration work
– Cons: More steps; must be careful about which area to subtract from which
Formulas
$\text{Area} = \int_a^b [f(x) - g(x)]\,dx$ where $f(x) \geq g(x)$ (MEMORIZE)
$\int_a^b x^n\,dx = \left[\frac{x^{n+1}}{n+1}\right]_a^b$ (formula booklet)
Pitfalls
  • Subtracting the wrong way round (lower minus upper) giving a negative area
  • Forgetting to find intersection points and using wrong limits
  • Algebraic errors when expanding (f(x) - g(x)) especially with cubic or quartic terms
  • Not splitting the integral when the curves cross within the region
  • Evaluating F(a) - F(b) instead of F(b) - F(a)

Must memorize

  • $\int x^n\,dx = \frac{x^{n+1}}{n+1} + C,\quad n \neq -1$ (formula booklet)
  • $\int \frac{1}{x}\,dx = \ln|x| + C$ (MEMORIZE)
  • $\int e^{ax}\,dx = \frac{1}{a}e^{ax} + C$ (formula booklet)
  • $\int \sin(ax)\,dx = -\frac{1}{a}\cos(ax) + C$ (formula booklet)
  • $\int \cos(ax)\,dx = \frac{1}{a}\sin(ax) + C$ (formula booklet)
  • Integration by parts: $\int u\frac{dv}{dx}\,dx = uv - \int v\frac{du}{dx}\,dx$ (formula booklet)
  • Trapezium rule: $\int_a^b y\,dx \approx \frac{h}{2}[y_0 + y_n + 2(y_1 + y_2 + \cdots + y_{n-1})]$ (formula booklet)
  • $\int \ln x\,dx = x\ln x - x + C$ (MEMORIZE — derived via by parts)
  • Linearity: $\int [af(x) + bg(x)]\,dx = a\int f(x)\,dx + b\int g(x)\,dx$ (MEMORIZE)
  • Partial fractions: $\frac{P}{(ax+b)(cx+d)} = \frac{A}{ax+b} + \frac{B}{cx+d}$ then integrate to get logs (MEMORIZE)

Global strategy

  • Classify the question immediately: Is it numerical (trapezium rule), algebraic (power rule), by substitution, by parts, or an area question? This determines your entire approach.
  • For trapezium rule: write out ALL y-values in a clear table before substituting into the formula — arithmetic accuracy is everything.
  • For integration by substitution: your #1 priority is to eliminate ALL x's from the integrand. If any x remains, you've made an error.
  • For integration by parts: use LIATE to choose u (Logs > Inverse trig > Algebraic > Trig > Exponentials). u = the one you differentiate.
  • For area questions: ALWAYS sketch the curves to identify which is upper and which is lower. A wrong subtraction order loses all method marks.
  • Show every line of working for definite integrals — write the antiderivative in square brackets with limits, THEN substitute. Examiners award marks for structure.
  • Check dimensions: area must be positive, indefinite integrals must have + C, and substitution integrals must have all variables consistent.
  • In multi-part questions, later parts often use earlier results — if you're stuck, use your answer from part (a) even if you think it might be slightly wrong.

Exam tips

  1. For trapezium rule: double-check your h value FIRST. The most common error is using the whole interval instead of dividing by the number of strips.
  2. When the question says 'show that', you must arrive at the given answer — work towards it and if your answer differs, recheck your algebra rather than fudging.
  3. Always write the integrated expression in brackets with limits BEFORE substituting values — this earns method marks even if your evaluation has an arithmetic slip.
  4. In substitution questions, systematically write: u = ..., du/dx = ..., dx = ..., and show the limits changing. Examiners specifically look for these steps.
  5. For integration by parts, if you need to apply it twice, keep your working organised — clearly label 'first application' and 'second application'.
  6. If an area comes out negative, take the absolute value but also recheck which curve is on top — a negative often signals you subtracted the wrong way.
  7. Don't forget + C for indefinite integrals — it's typically worth 1 mark and is the easiest mark to lose.
  8. Use exact values (fractions, surds, e, ln) not decimals unless the question specifically asks for a decimal approximation.
  9. In multi-step area questions, a quick sketch (even a rough one) prevents you from making the fatal error of integrating with wrong limits or wrong curve order.
