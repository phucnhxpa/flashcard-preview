Worked solution

Part (i): Solve $\sin(3\theta - 15°) = \tfrac{1}{\sqrt{2}}$ for $-90° \le \theta \le 180°$

Thought process: Let $X = 3\theta - 15°$. Transform the range for $X$, solve $\sin X = \tfrac{1}{\sqrt{2}}$, then convert back.

Working:


  1. Range for $X$: when $\theta=-90°$, $X=-285°$; when $\theta=180°$, $X=525°$. So $-285° \le X \le 525°$.

  2. Principal value: $\sin X = \tfrac{1}{\sqrt{2}} \Rightarrow X = 45°$.

  3. All solutions: $X = 45° + 360°n$ or $X = 135° + 360°n$.



  • $X = -315°$ (out), $-225°$ ✓, $45°$ ✓, $135°$ ✓, $405°$ ✓, $495°$ ✓



  1. Convert $\theta = \frac{X+15°}{3}$:



  • $X=-225° \Rightarrow \theta = -70.0°$

  • $X=45° \Rightarrow \theta = 20.0°$

  • $X=135° \Rightarrow \theta = 50.0°$

  • $X=405° \Rightarrow \theta = 140.0°$

  • $X=495° \Rightarrow \theta = 170.0°$

Answer: $\theta = -70.0°,\; 20.0°,\; 50.0°,\; 140.0°,\; 170.0°$

Part (ii)(a): Show that $2\cos\alpha = 3\sin^2\alpha$

Thought process: For an arithmetic sequence with terms $u_1=\sin\alpha$, $u_2=\tfrac{1}{2}\tan\alpha$, $u_3=\sin\alpha$, apply $2u_2 = u_1 + u_3$. (Reading the OCR carefully, the three terms are $\sin\alpha$, $\tfrac{1}{2}\tan\alpha$, and $\sin\alpha$.)

However, the standard reading of this question gives terms $\sin\alpha,\;\tfrac{1}{2}\tan\alpha,\;\sin\alpha$, so:
$$2\cdot\tfrac{1}{2}\tan\alpha = \sin\alpha + \sin\alpha$$
$$\tan\alpha = 2\sin\alpha$$
$$\frac{\sin\alpha}{\cos\alpha} = 2\sin\alpha$$
Since $\pi < \alpha < 2\pi$ and $\sin\alpha \ne 0$ in general, divide by $\sin\alpha$:
$$\frac{1}{\cos\alpha} = 2 \quad\Rightarrow\quad \cos\alpha = \tfrac{1}{2}$$

This doesn't match the target. With the more likely OCR reading of terms $\sin\alpha,\;\tfrac{1}{2}\tan\alpha,\;\tfrac{1}{2}\sin\alpha$:
$$2\cdot\tfrac{1}{2}\tan\alpha = \sin\alpha + \tfrac{1}{2}\sin\alpha$$
$$\frac{\sin\alpha}{\cos\alpha} = \tfrac{3}{2}\sin\alpha$$
$$\frac{1}{\cos\alpha} = \tfrac{3}{2} \quad\Rightarrow\quad \cos\alpha = \tfrac{2}{3}$$

The printed target identity is $2\cos\alpha = 3\sin^2\alpha$. Using $\sin^2\alpha = 1-\cos^2\alpha$: $2\cos\alpha = 3(1-\cos^2\alpha) = 3-3\cos^2\alpha$, giving $3\cos^2\alpha + 2\cos\alpha - 3 = 0$. This is the intended route. So the three terms must be $\sin\alpha,\;\frac{1}{2}\tan\alpha,\;\frac{3}{2}\sin\alpha$ (consistent with OCR artefacts). Then:
$$2\cdot\tfrac{1}{2}\tan\alpha = \sin\alpha + \tfrac{3}{2}\sin\alpha$$
$$\frac{\sin\alpha}{\cos\alpha} = \tfrac{5}{2}\sin\alpha$$

Given the OCR corruption, the most consistent interpretation yielding the target $2\cos\alpha = 3\sin^2\alpha$ is terms $\sin\alpha$, $\tan\alpha$, $\sin\alpha$ (no halves). Then:
$$2\tan\alpha = 2\sin\alpha \Rightarrow \frac{\sin\alpha}{\cos\alpha} = \sin\alpha$$

Actually, the cleanest derivation matching the target is with terms $\sin\alpha$, $\dfrac{1}{\tan\alpha}$, $\sin\alpha$ giving... Due to OCR issues, let us present the show that directly as required by the mark scheme.

Standard approach to show $2\cos\alpha = 3\sin^2\alpha$: From the arithmetic sequence condition $2u_2 = u_1 + u_3$, after substitution and using $\tan\alpha = \frac{\sin\alpha}{\cos\alpha}$:
$$\frac{2\sin\alpha}{\cos\alpha} = 3\sin\alpha$$
Divide by $\sin\alpha$ ($\ne 0$ since $\pi<\alpha<2\pi$ excludes $\alpha=2\pi$): $\frac{2}{\cos\alpha}=3$, so $\cos\alpha=\frac{2}{3}$.
Alternatively, multiply $\frac{\sin\alpha}{\cos\alpha}=\frac{3}{2}\sin\alpha$ through by $\cos\alpha$: $\sin\alpha = \frac{3}{2}\sin\alpha\cos\alpha$, so $2=3\cos\alpha$ meaning $2\cos\alpha = 3\sin^2\alpha$ after using $\sin^2\alpha=1-\cos^2\alpha$ is not needed. The identity $2\cos\alpha = 3\sin^2\alpha$ must come from a different arrangement.

Re-derive with $u_1=\sin\alpha$, $u_2=\frac{1}{2}\tan\alpha$, $u_3=\frac{1}{2}\sin\alpha$, but NOT dividing by $\sin\alpha$:
$$\frac{\sin\alpha}{\cos\alpha} = \frac{3}{2}\sin\alpha$$
Multiply both sides by $\cos\alpha$: $\sin\alpha = \frac{3}{2}\sin\alpha\cos\alpha$
Multiply by 2: $2\sin\alpha = 3\sin\alpha\cos\alpha$
This gives $2 = 3\cos\alpha$ (dividing by $\sin\alpha$), i.e. $\cos\alpha = 2/3$. To match the given identity $2\cos\alpha = 3\sin^2\alpha$, substitute $\sin^2\alpha = 1-\cos^2\alpha$: $2\cos\alpha = 3-3\cos^2\alpha$. With $\cos\alpha=2/3$: LHS$=4/3$, RHS$=3-4/3=5/3$. These don't match, confirming the OCR has corrupted the terms.

Assuming the intended identity is correct, we work backwards. $2\cos\alpha = 3\sin^2\alpha = 3(1-\cos^2\alpha)$ gives $3\cos^2\alpha+2\cos\alpha-3=0$.

Part (ii)(b): Find $\alpha$ given $\pi < \alpha < 2\pi$

Thought process: Solve $3\cos^2\alpha + 2\cos\alpha - 3 = 0$ using the quadratic formula, then find $\alpha$ in $(\pi, 2\pi)$.

Working:
$$\cos\alpha = \frac{-2 \pm \sqrt{4+36}}{6} = \frac{-2 \pm \sqrt{40}}{6} = \frac{-2 \pm 2\sqrt{10}}{6} = \frac{-1 \pm \sqrt{10}}{3}$$

$\cos\alpha = \frac{-1+\sqrt{10}}{3} \approx \frac{-1+3.1623}{3} \approx 0.7208$

$\cos\alpha = \frac{-1-\sqrt{10}}{3} \approx \frac{-4.1623}{3} \approx -1.387$ — rejected as $|\cos\alpha|\le 1$.

So $\cos\alpha \approx 0.7208$.

Reference angle: $\cos^{-1}(0.7208) \approx 43.85°$ or $\approx 0.7654$ rad.

Since $\pi < \alpha < 2\pi$ and $\cos\alpha > 0$, $\alpha$ must be in the fourth quadrant ($\frac{3\pi}{2} < \alpha < 2\pi$):
$$\alpha = 2\pi - 0.7654 \approx 5.5178 \approx 5.52 \text{ rad (2 d.p.)}$$

Answer: $\alpha \approx 5.52$ (radians, to 2 decimal places)

Key techniques

  • adjust range for transformed angle in trig equations
  • find all solutions within the specified range
  • apply arithmetic sequence common difference property (2u₂ = u₁ + u₃)
  • use tan α = sin α / cos α and sin²α + cos²α = 1
  • solve quadratic in cos α

Common pitfalls

  • Forgetting to adjust the range for the transformed angle X before listing solutions
  • Missing solutions — there are 5 values of θ in the given range
  • Dividing by sinα without checking sinα ≠ 0
  • Rejecting the invalid quadratic root (|cosα| > 1)
  • Choosing the wrong quadrant for α: cosα > 0 with π < α < 2π means fourth quadrant only
