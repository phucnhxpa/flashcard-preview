TriggersSolve$\cos 2x$$\sin 2\theta$Give your answers in terms of $\pi$
Assumptions
  • The goal is to eliminate the double angle to get an equation purely in terms of a single angle (e.g., $x$).
  • The correct choice of the $\cos 2x$ identity is crucial for simplification.
Approaches
Double Angle Substitution & Solve
When: When you see a mix of angles like $2x$ and $x$ in the same equation.
  1. Identify the double angle term (e.g., $\cos 2x$).
  2. Choose the most suitable version of the double angle identity. If the rest of the equation involves $\sin x$, use $\cos 2x = 1 - 2\sin^2 x$. If it involves $\cos x$, use $\cos 2x = 2\cos^2 x - 1$.
  3. Substitute the identity into the equation.
  4. Simplify and rearrange the equation, which will typically result in a quadratic or factorisable expression.
  5. Solve for the trigonometric function (e.g., find values for $\sin x$).
  6. Find the principal value and all subsequent solutions within the given range using CAST or graphs.
+ Pros: Directly simplifies the problem into a more familiar (often quadratic) form.
– Cons: Choosing the wrong version of the $\cos 2x$ identity can lead to a dead end or a more complicated equation.
Formulas
$\sin 2A \equiv 2\sin A \cos A$ (formula booklet)
$\cos 2A \equiv \cos^2 A - \sin^2 A$ (formula booklet)
$\cos 2A \equiv 2\cos^2 A - 1$ (formula booklet)
$\cos 2A \equiv 1 - 2\sin^2 A$ (formula booklet)
Pitfalls
  • Using an incorrect version of the $\cos 2x$ formula, making the problem harder.
  • Algebraic errors during substitution and rearrangement.
  • Dividing by a term like $\sin x$ or $\cos x$ and losing solutions, instead of factoring.
  • Forgetting to find all solutions within the full range.

TriggersSolvefor $\theta$ in the interval $0 \le \theta < 180°$$\cos(2x - \frac{\pi}{6})$
Assumptions
  • The core of the problem is managing the transformation of the angle and, critically, the corresponding transformation of the solution range.
  • The periodicity of the function is used to find all possible solutions before solving for the final variable.
Approaches
Range Transformation Method
When: Any time the angle inside the trig function is not just a single variable (e.g., not just 'x').
  1. Let a new variable, say $Y$, equal the transformed angle. E.g., $Y = 2x - 15°$.
  2. Transform the original range for $x$ into the new range for $Y$. If $0° \le x < 360°$, then $2(0)-15 \le Y < 2(360)-15$, so $-15° \le Y < 705°$.
  3. Solve the simplified equation for $Y$. E.g., solve $\sin(Y) = 0.5$.
  4. Find the principal value for $Y$ from your calculator: $Y_1 = \arcsin(0.5) = 30°$.
  5. Find all other solutions for $Y$ within its transformed range ($-15°$ to $705°$). Using symmetry: $Y_2 = 180° - 30° = 150°$. Using periodicity: $Y_3 = 30°+360° = 390°$, $Y_4 = 150°+360° = 510°$.
  6. For each valid value of $Y$, solve for $x$. E.g., $2x - 15° = 30° \implies x=22.5°$.
  7. List all final values for $x$, ensuring they fall within the original range.
+ Pros: A robust method that ensures no solutions are missed.
– Cons: The most common source of error in all trigonometry questions; requires careful bookkeeping of the range.
Formulas
Periodicity of $\sin x$ and $\cos x$ is $360°$ or $2\pi$ radians.
Periodicity of $\tan x$ is $180°$ or $\pi$ radians.
Pitfalls
  • Failing to adjust the range for the transformed angle, leading to missed solutions.
  • Adjusting the range incorrectly (e.g., forgetting to multiply/divide *before* adding/subtracting).
  • Finding all solutions for $x$ and then trying to filter them, instead of finding all solutions for the transformed angle $Y$ first.
  • Forgetting the $\pm$ sign when taking a square root, e.g., if solving $\sin^2(Y) = 0.25$.

TriggersExpress in the form $R\sin(x+\alpha)$where $R>0$ and $0 < \alpha < 90°$Hence solve...
Assumptions
  • The expression `a sin x + b cos x` can be written as a single sine or cosine wave with a phase shift.
  • The values of $R$ and $\alpha$ are constants that depend on $a$ and $b$.
Approaches
Convert to R-Form and Solve
When: Whenever you see a sum of sine and cosine terms with the same angle.
  1. Identify coefficients $a$ and $b$ from the expression $a \sin x + b \cos x$.
  2. Calculate $R = \sqrt{a^2 + b^2}$.
  3. Calculate $\alpha$ by setting up the correct identity. For $R\sin(x+\alpha) = R(\sin x \cos \alpha + \cos x \sin \alpha)$, compare coefficients: $a = R\cos\alpha$ and $b = R\sin\alpha$. Then find $\alpha = \arctan(\frac{b}{a})$.
  4. Rewrite the original equation as $R\sin(x+\alpha) = c$.
  5. Isolate the trig function: $\sin(x+\alpha) = \frac{c}{R}$.
  6. Let $Y = x+\alpha$. Adjust the original range for $x$ to find the new range for $Y$.
  7. Find all solutions for $Y$ in its new range.
  8. Subtract $\alpha$ from each solution for $Y$ to find the corresponding values for $x$.
  9. Check that your final answers for $x$ are in the original range.
+ Pros: Reduces a complex expression into a very simple trigonometric equation.
– Cons: Prone to sign errors when finding $\alpha$ and calculation errors with $R$.
Formulas
$a\sin\theta + b\cos\theta = R\sin(\theta + \alpha)$ where $R=\sqrt{a^2+b^2}$ and $\tan\alpha=\frac{b}{a}$ (formula booklet)
$a\sin\theta + b\cos\theta = R\cos(\theta - \alpha)$ where $R=\sqrt{a^2+b^2}$ and $\tan\alpha=\frac{a}{b}$ (formula booklet)
Pitfalls
  • Mixing up the formulae for $\alpha$, e.g., using $\tan\alpha = a/b$ when converting to $R\sin(x+\alpha)$.
  • Forgetting to adjust the angular range for the new variable $(x+\alpha)$, leading to missed or extra solutions.
  • Calculator in the wrong mode (degrees/radians) when finding $\alpha$ or the final angles.

Assumptions
  • The primary identity needed is either $\sin^2\theta + \cos^2\theta = 1$ or $\tan\theta = \frac{\sin\theta}{\cos\theta}$.
  • The equation will simplify to a standard quadratic form $ay^2 + by + c = 0$, where $y$ is a trigonometric function.
  • All solutions within the specified range must be found.
Approaches
Identity Substitution & Quadratic Solution
When: This is the standard method for this question type.
  1. Identify the trigonometric functions present (e.g., $\sin^2 x$ and $\cos x$).
  2. Use the identity $\sin^2 x + \cos^2 x = 1$ to replace the squared term, creating an equation with only one type of trig function (e.g., replace $\sin^2 x$ with $1 - \cos^2 x$).
  3. Rearrange the equation into the standard quadratic form $a\cos^2 x + b\cos x + c = 0$.
  4. Let $y = \cos x$ (or whatever the function is) and solve the quadratic $ay^2 + by + c = 0$ by factorising or using the quadratic formula.
  5. For each valid solution for $y$ (where $|y| \le 1$), solve the equation (e.g., $\cos x = y$).
  6. Find the principal value for $x$ using $x = \arccos(y)$.
  7. Use a CAST diagram or graph sketch to find all other solutions within the required range (e.g., for $\cos x = y$, the other solution is $360° - x$).
  8. List all solutions to the required degree of accuracy.
+ Pros: A systematic, reliable process that works for all such problems.
– Cons: Can involve messy algebra or quadratic formula calculations.
Formulas
$\sin^2\theta + \cos^2\theta \equiv 1$ (formula booklet)
$\tan\theta \equiv \frac{\sin\theta}{\cos\theta}$ (formula booklet)
Pitfalls
  • Using the wrong identity, e.g., replacing $\cos x$ with $1-\sin x$ instead of $\cos^2 x$ with $1-\sin^2 x$.
  • Making sign errors when rearranging the equation into a quadratic.
  • Forgetting to find all solutions in the given range, only giving the principal value from the calculator.
  • Incorrectly solving the quadratic equation.

Must memorize

  • The CAST diagram: Which quadrants All, Sin, Tan, Cos are positive in. (MEMORIZE)
  • The three forms of $\cos 2A$: $\cos^2 A - \sin^2 A$, $2\cos^2 A - 1$, and $1 - 2\sin^2 A$. Knowing which to use to simplify an equation is a key skill. (formula booklet, but knowing applications is key)
  • The graphs of $y=\sin x$, $y=\cos x$, and $y=\tan x$, including their periods and key values. (MEMORIZE)
  • $\sin^2\theta + \cos^2\theta \equiv 1$ (formula booklet)
  • $\tan\theta \equiv \frac{\sin\theta}{\cos\theta}$ (formula booklet)

Global strategy

  • **1. Identify the Goal:** First, determine the type of equation. Is it a quadratic in disguise? Does it have a double angle? Is it an `a sin x + b cos x` form? This dictates your first move.
  • **2. Unify the Angle:** Use double angle or compound angle identities to make sure every trig function has the same angle (e.g., all `x`, not a mix of `x` and `2x`).
  • **3. Unify the Function:** Use identities like `sin²x + cos²x = 1` to convert the equation into terms of a single trig function (e.g., all `sin x`).
  • **4. Transform the Range:** If the angle is transformed (e.g., `2x - 30°`), you MUST calculate the new range for this transformed angle BEFORE you start finding solutions. Write it down explicitly.
  • **5. Solve for the Function:** Solve the simplified equation (often a quadratic) to find values for the trig function (e.g., `sin x = 0.5`).
  • **6. Find ALL Solutions:** Use the principal value from your calculator and the CAST diagram (or graph) to find ALL solutions for the angle within your (possibly transformed) range.
  • **7. NEVER Divide by a Trig Function:** To solve `sin x cos x = sin x`, rearrange to `sin x (cos x - 1) = 0`. This gives solutions from `sin x = 0` AND `cos x = 1`. Dividing by `sin x` loses the `sin x = 0` solutions.
  • **8. Check Calculator Mode:** Always check if your calculator is in DEGREES or RADIANS, as required by the question.

Exam tips

  1. For 'Show that...' questions, show every step of your algebraic manipulation clearly. Don't jump from the start to the end expression.
  2. When solving a quadratic like $3\sin^2 x - 5\sin x - 2 = 0$, you can make a temporary substitution $y=\sin x$ to get $3y^2 - 5y - 2 = 0$. This can help avoid algebraic slips.
  3. Always check that your final answers are within the range specified in the question and are given to the required accuracy.
  4. If you solve for a trig function and get a value outside the range [-1, 1] (e.g., $\sin x = 1.5$), write down 'no solutions' for that branch of your working.
  5. When using the R-formula, draw a small right-angled triangle to verify your value for $\alpha$. If $R\sin(x+\alpha)$ is needed for $3\sin x + 4\cos x$, then $R=5$ and $\tan\alpha = 4/3$. The triangle helps confirm this.
  6. If you have to take a square root (e.g., from $\cos^2 x = 0.25$), remember both the positive and negative roots: $\cos x = +0.5$ and $\cos x = -0.5$. This is a very common way to lose half the solutions.
