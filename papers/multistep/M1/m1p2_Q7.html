Q7 — Ship $S$ vs lighthouse: closest approach and safety check

Question

At midnight, ship $S$ has position vector $(19\mathbf{i} + 22\mathbf{j})$ km, and travels with constant velocity $(12\mathbf{i} - 16\mathbf{j})$ km/h. A lighthouse $L$ is at $(26\mathbf{i} + 15\mathbf{j})$ km. If $S$ is within $1.3$ km of $L$, it is in danger.

(a) Find the speed of $S$. [2]

(b) Position vector of $S$ at time $t$ hours after midnight. [2]

(c)(i) Find the value of $t$ at which $S$ is closest to $L$. (ii) Hence determine if it is safe. [7]

[11 marks]

Part (a) — Speed of $S$ [2 marks] · 1 step

STEP 1/1 M1 A1

Strategy: speed is the magnitude of the velocity vector. Apply Pythagoras to the components of $\mathbf{v}$.

Formula (speed = magnitude of velocity):

$$|\mathbf{v}| = \sqrt{v_i^2 + v_j^2}$$

Substitute $\mathbf{v} = 12\mathbf{i} - 16\mathbf{j}$:

$$|\mathbf{v}| = \sqrt{12^2 + (-16)^2} = \sqrt{144 + 256} = \sqrt{400}$$

$$|\mathbf{v}| = 20\ \text{km/h}$$

$$\boxed{\;|\mathbf{v}| = 20\ \text{km/h}\;}$$

Sign convention: the negative $\mathbf{j}$-component doesn’t affect speed (squared away). It only affects direction (which is south-east here).
$\boxed{\,|\mathbf{v}| = 20\ \text{km/h}\,}$

Part (b) — Position $\mathbf{s}(t)$ [2 marks] · 1 step

Setup (from question): $\mathbf{r}_S(0) = 19\mathbf{i} + 22\mathbf{j}$, $\mathbf{v}_S = 12\mathbf{i} - 16\mathbf{j}$ km/h.
STEP 1/1 M1 A1

Strategy: constant velocity → position is linear in time. Plug into the standard kinematic formula and collect components.

Formula (constant velocity):

$$\mathbf{r}(t) = \mathbf{r}_0 + \mathbf{v}\, t$$

Substitute $\mathbf{r}_0 = 19\mathbf{i} + 22\mathbf{j}$, $\mathbf{v} = 12\mathbf{i} - 16\mathbf{j}$:

$$\mathbf{s}(t) = (19\mathbf{i} + 22\mathbf{j}) + (12\mathbf{i} - 16\mathbf{j})\, t$$

Collect $\mathbf{i}$ and $\mathbf{j}$ separately:

$\;\;\mathbf{i}$-component: $19 + 12t$

$\;\;\mathbf{j}$-component: $22 - 16t$

$$\boxed{\;\mathbf{s}(t) = (19 + 12t)\mathbf{i} + (22 - 16t)\mathbf{j}\;}$$

Sense check: at $t = 0$, $\mathbf{s}(0) = 19\mathbf{i} + 22\mathbf{j}$ ✓ (matches $\mathbf{r}_0$). The $\mathbf{j}$-component decreases with time (negative coefficient) — consistent with $\mathbf{v}$ having $-16\mathbf{j}$.
$\mathbf{s}(t) = (19+12t)\mathbf{i} + (22-16t)\mathbf{j}$

Part (c)(i) — $t$ at closest approach [4 marks] · 2 steps

$\overrightarrow{LS}(t) = \mathbf{s}(t) - \mathbf{r}_L = (19+12t-26)\mathbf{i} + (22-16t-15)\mathbf{j} = (-7+12t)\mathbf{i} + (7-16t)\mathbf{j}$
STEP 1/2 M1

Formula (closest-approach trick): minimise the squared distance:

$$f(t) = |\overrightarrow{LS}|^2 = x(t)^2 + y(t)^2$$

Substitute:

$$f(t) = (-7+12t)^2 + (7-16t)^2$$

Phuc nailed this trick already. Minimising $|d|^2$ has the same minimiser as $|d|$ but avoids the awkward square-root in the derivative.
$f(t) = (-7+12t)^2 + (7-16t)^2$
↳ from STEP 1:From STEP 1: $f(t) = (-7+12t)^2 + (7-16t)^2$.
STEP 2/2 M1 A1 A1

Strategy: minimum of $f(t)$ occurs where $f'(t) = 0$. Differentiate using the chain rule on each squared bracket, then solve for $t$. Confirm it’s a min via $f''(t) > 0$.

Chain rule: $\tfrac{d}{dt}(at+b)^2 = 2a(at+b)$.

For the first bracket $(-7 + 12t)^2$: $a = 12 \Rightarrow$ derivative $= 2(12)(-7+12t) = 24(-7+12t)$.

For the second bracket $(7 - 16t)^2$: $a = -16 \Rightarrow$ derivative $= 2(-16)(7-16t) = -32(7-16t)$.

$$f'(t) = 24(-7+12t) - 32(7-16t)$$

Expand:

$$f'(t) = -168 + 288t - 224 + 512t$$

$$f'(t) = 800t - 392$$

Set $f'(t) = 0$:

$$800t = 392$$

$$t = \tfrac{392}{800} = 0.49\ \text{h}$$

Confirm minimum: $f''(t) = 800 > 0$ ✓ (concave up).

$$\boxed{\;t = 0.49\ \text{h}\;}$$

Sense check: $0.49$ h $\approx$ 29 minutes after midnight. $t > 0$ → the closest approach happens in the future (after midnight), not in the past — physically meaningful.
$\boxed{\,t = 0.49\ \text{h}\,}$ (minimum)

Part (c)(ii) — Safety conclusion [3 marks] · 2 steps

From (i): $t = 0.49$. Plug back: $-7 + 12(0.49) = -1.12$; $7 - 16(0.49) = -0.84$.
STEP 1/2 M1 A1

$$|\overrightarrow{LS}|_{\min} = \sqrt{1.12^2 + 0.84^2} = \sqrt{1.2544 + 0.7056} = \sqrt{1.96}$$

$$|\overrightarrow{LS}|_{\min} = 1.4\ \text{km}$$

$|\overrightarrow{LS}|_{\min} = 1.4$ km
↳ from STEP 1:From STEP 1: $|\overrightarrow{LS}|_{\min} = 1.4$ km.
STEP 2/2 A1

Compare to safety threshold $1.3$ km:

$$1.4 > 1.3 \;\therefore\; \text{ship stays \emph{outside} the } 1.3\ \text{km zone}$$

$\Rightarrow$ safe to continue. ✓
