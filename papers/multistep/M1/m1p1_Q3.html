Q3 — Find tension $T_1$ in the upper string

Question

Particles $P$ (mass $4m$) and $Q$ (mass $2m$) are connected by a light inextensible string. A second light inextensible string is attached above $Q$, hanging from a fixed support. Both strings are taut and vertical; the system accelerates vertically downwards.

Given that the tension in the string between $P$ and $Q$ is $3mg$, find the tension $T_1$ in the upper string in terms of $m$ and $g$.

[6 marks]

Solution [6 marks] · 3 steps

STEP 1/3key facts

Strategy: before writing any equation, decode the modelling phrases. Each adjective on a string adds (or removes) an unknown.

Definition (light string): mass = 0.

Consequence: applying $\Sigma F = ma$ to a small element of the string gives $\Delta T = (\Delta m) a = 0 \cdot a = 0$, so tension is the same throughout one continuous string. (One string → one tension.)

Definition (inextensible string): length is fixed; doesn’t stretch.

Consequence: if particle $P$ moves down by $\delta$, the connected particle $Q$ must move with the same speed and acceleration along the string — they share one common $a$.

Apply to this problem:

• Two strings here (upper $T_1$ and lower $T_{PQ} = 3mg$). Each is internally uniform.

• Both strings inextensible → $P$, $Q$, and the support point all move together → same $a$ for the whole system.

$$\boxed{\;a_P = a_Q = a_\text{system} = a\;}$$

Phuc's gap (verbatim): “For this question, I need to be able to identify the list of forces, and I need to know what an inextensible string means. In the case of an inextensible string, that means that the acceleration throughout the string is the same.” That insight is the key — it lets us treat $P$ and $Q$ as one 6$m$ block in the system equation later.
Same $a$ for $P$, $Q$, and the whole system.
↳ from STEP 1:From STEP 1: Same $a$ for $P$, $Q$, and the whole system.
STEP 2/3 M1 A1

Strategy: isolate $P$. The upper string ($T_1$, our unknown) doesn’t touch $P$ — only the lower string does. So $P$’s equation has only one unknown: $a$. Solve that first, then come back for $T_1$.

Free-body diagram for $P$ (mass $4m$): taking downward as positive (system accelerates down).

• Weight $4mg$ acts down $\Rightarrow$ contributes $+4mg$.

• Tension $T_{PQ} = 3mg$ in the lower string pulls $P$ up (toward $Q$ above) $\Rightarrow$ contributes $-3mg$.

• No other forces (string is light; no friction; no air resistance).

Formula (Newton’s 2nd law):

$$\Sigma F = m_P\, a$$

Substitute:

$$4mg - 3mg = (4m)\, a$$

$$mg = 4ma$$

Divide both sides by $4m$ (note $m \ne 0$):

$$\boxed{\;a = \tfrac{g}{4}\;}$$

Why isolate P first: the upper string’s tension doesn’t act on $P$, so $P$’s equation has only one unknown ($a$). Tackling the whole system first would give one equation in two unknowns ($a$ and $T_1$). Sense check: the system is heavier on the bottom and net pulled down, so $a < g$ makes sense ($g/4 < g$ ✓). If $T_{PQ} > 4mg$, $P$ would accelerate up; here $T_{PQ} = 3mg < 4mg$, so $P$ accelerates down — consistent.
$a = g/4$
↳ from STEP 2:From STEP 2: $a = g/4$.
STEP 3/3 M1 A1 A1

Strategy: now treat the whole system ($P$ + $Q$ together, mass $6m$) as one combined body. The internal tension $T_{PQ}$ between them cancels (Newton’s 3rd law: internal forces come in pairs that sum to zero). Only external forces matter.

External forces on the combined system (down positive, since system accelerates down):

• Total weight: $(4m + 2m)g = 6mg$ down $\Rightarrow +6mg$.

• Upper-string tension $T_1$ pulls the system up $\Rightarrow -T_1$.

(Note: $T_{PQ}$ is internal — it pulls $Q$ down and $P$ up by the same amount, so they cancel in the combined equation.)

Formula (Newton’s 2nd law for system):

$$\Sigma F_\text{ext} = M_\text{total}\, a$$

Using $a = g/4$ from STEP 2:

$$6mg - T_1 = 6m \cdot \tfrac{g}{4} = \tfrac{6mg}{4} = \tfrac{3mg}{2}$$

Rearrange for $T_1$:

$$T_1 = 6mg - \tfrac{3mg}{2} = \tfrac{12mg}{2} - \tfrac{3mg}{2} = \tfrac{9mg}{2}$$

$$\boxed{\;T_1 = \dfrac{9mg}{2}\;}$$

Trick (internal forces cancel): internal tension $T_{PQ}=3mg$ doesn’t appear in the system equation — you don’t need to write it. Sense check: if the system were in equilibrium (no acceleration), $T_1$ would equal $6mg$. Since the system accelerates down, the supporting tension must be less than the total weight — and $\tfrac{9mg}{2} = 4.5mg < 6mg$ ✓.
$\boxed{\,T_1 = \dfrac{9mg}{2}\,}$
