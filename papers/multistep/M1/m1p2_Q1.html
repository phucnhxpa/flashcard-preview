Q1 — Ring on string AB; horizontal 2 N at $C$

Question

A small smooth ring of mass $M$ kg is threaded on a light inextensible string. The ends of the string are attached to fixed points $A$ and $B$ on a horizontal ceiling. A horizontal force of $2$ N is applied to the ring at $C$ (which is vertically below $B$). The portion $AC$ makes angle $\theta$ with the ceiling, where $\tan\theta = 3/4$ (so $\sin\theta = 3/5$, $\cos\theta = 4/5$). The tension in the string is $T$ N.

(a) Find $T$. [3]

(b) Find $M$. [3]

[6 marks]

Part (a) — Find $T$ [3 marks] · 2 steps

STEP 1/2key fact

Smooth ring on light string: the string slides freely → tension is the same on both portions ($CA$ and $CB$).

Trap: treating the two halves as different tensions would need extra unknowns. The smooth-ring assumption equates them.
$T_{CA} = T_{CB} = T$
↳ from STEP 1:From STEP 1: $T_{CA} = T_{CB} = T$ (smooth ring, equal tension).
STEP 2/2 M1 A1 A1

Strategy: at point $C$ (the smooth ring), all forces meet. To find $T$, resolve horizontally — in this direction $CB$ contributes nothing (vertical), so the equation isolates $T$ cleanly.

Force breakdown at $C$:

• String $CA$: tension $T$ at angle $\theta$ to ceiling → horizontal component $T\cos\theta$ towards $A$.

• String $CB$: vertical only → NO horizontal component.

• Applied force: $2$ N horizontal, away from $A$.

Formula (horizontal equilibrium):

$$\Sigma F_x = 0$$

Substitute (towards $A$ as +):

$$T\cos\theta - 2 = 0 \;\Rightarrow\; T\cos\theta = 2$$

With $\cos\theta = 4/5$:

$$\tfrac{4T}{5} = 2$$

$$T = \tfrac{5 \cdot 2}{4} = 2.5\ \text{N}$$

$$\boxed{\;T = 2.5\ \text{N}\;}$$

Why horizontal first? Choosing the direction in which the unknown vertical-string CB contributes ZERO eliminates one variable. Same trick as choosing a clever pivot for moments.
$\boxed{\,T = 2.5\ \text{N}\,}$

Part (b) — Find $M$ [3 marks] · 1 step

From (a): $T = 2.5$ N (same in both portions — smooth ring). $CB$ is vertical (pulls up with full $T$); $CA$ at angle $\theta$ to ceiling, $\sin\theta = 3/5$.
STEP 1/1 M1 A1 A1

Strategy: resolve vertically to find $M$ — weight is the only vertical force tied to $M$, and both string segments contribute upward components.

Vertical components at $C$:

• $CA$: vertical pull = $T\sin\theta$ up.

• $CB$: pulls straight up = $T$ (the WHOLE tension, since $CB$ is vertical).

• Weight: $Mg$ down.

Formula (vertical equilibrium):

$$\Sigma F_y = 0$$

Substitute:

$$T\sin\theta + T - Mg = 0$$

$$T(\sin\theta + 1) = Mg$$

Using $T = 2.5$ N from (a) and $\sin\theta = 3/5$:

$$2.5 \cdot \tfrac{3}{5} + 2.5 = Mg$$

$$1.5 + 2.5 = Mg$$

$$Mg = 4 \;\Rightarrow\; M = \tfrac{4}{9.8} \approx 0.41\ \text{kg}$$

$$\boxed{\;M \approx 0.41\ \text{kg}\;}$$

Easy miss: forgetting that string $CB$ is vertical, so its FULL $T$ contributes to the upward force — not $T\cos\theta$. Sense check: $0.41$ kg is roughly an apple — reasonable for a small object held by light strings under tension a few newtons.
$\boxed{\,M = 4/9.8 \approx 0.41\ \text{kg}\,}$
