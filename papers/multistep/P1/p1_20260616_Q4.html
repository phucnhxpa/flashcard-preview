P1 Q4 · Integration — find $f(x)$ with $f(12)=8\sqrt{3}$

Question

Given $f'(x) = 2x^{-1/2} - 4x^{-2} + 3$ (with $A = -4$ from part (a)) and $f(12) = 8\sqrt{3}$,

find $f(x)$, giving each term in its simplest form. [5 marks]

[5 marks]

Part (b) — Integrate and find c from f(12) = 8√3 [5 marks] · 6 steps

With $A = -4$ from part (a): $f'(x) = 2x^{-1/2} - 4x^{-2} + 3$. Given $f(12) = 8\sqrt{3}$, find $f(x)$.
STEP 1/5integrate f'(x) term by term M1

Strategy. With $A = -4$: $f'(x) = 2x^{-1/2} - 4x^{-2} + 3$. Integrate term by term using the power rule: add 1 to the power, divide by the new power. Add $+c$.

Integrate each term

$$\int 2x^{-1/2}\,dx = 2\cdot\frac{x^{1/2}}{1/2} = 4x^{1/2} = 4\sqrt{x}.$$$$\int -4x^{-2}\,dx = -4\cdot\frac{x^{-1}}{-1} = +4x^{-1} = \frac{4}{x}.$$$$\int 3\,dx = 3x.$$
Sign flip on the second term. Dividing $-4$ by $-1$ gives $+4$. The sign flip comes from dividing by the new (negative) power $-1$, NOT from any external rule. Phuc's mistake: dropped the power on this term (wrote $+4$ instead of $+4/x$), losing the accuracy mark.
↳ from STEP 1: $\int 2x^{-1/2}=4\sqrt{x},\; \int -4x^{-2}=\tfrac{4}{x},\; \int 3=3x$
STEP 2/5write f(x) with +c A1

Strategy. Combine the integrated terms and add the constant of integration $+c$.

General antiderivative

$$f(x) = 4\sqrt{x} + \frac{4}{x} + 3x + c.$$
Why $+c$ is mandatory. Differentiation destroys constants ($\tfrac{d}{dx}[c] = 0$), so infinitely many functions share the same derivative. Integration cannot recover which constant was lost — we represent the lost family with $+c$. Forgetting $+c$ on an indefinite integral loses a mark even if the rest is perfect.
↳ from STEP 2: $f(x) = 4\sqrt{x} + \tfrac{4}{x} + 3x + c$
STEP 3/5substitute x = 12 M1

Strategy. Use the boundary condition $f(12) = 8\sqrt{3}$ to find $c$. Substitute $x = 12$ into $f(x)$. Simplify $\sqrt{12}$ first: $\sqrt{12} = \sqrt{4\cdot 3} = 2\sqrt{3}$.

Simplify $4\sqrt{12}$

$$4\sqrt{12} = 4\cdot 2\sqrt{3} = 8\sqrt{3}.$$

Evaluate each term at $x=12$

$$f(12) = 4\sqrt{12} + \frac{4}{12} + 3(12) + c = 8\sqrt{3} + \frac{1}{3} + 36 + c.$$
Phuc's algebraic error here. Mishandled the constant term after $4/12 = 1/3$. The surd terms are designed to cancel: $4\sqrt{12} = 8\sqrt{3}$ equals the RHS $8\sqrt{3}$, leaving only rational numbers to solve for $c$. Spotted by simplifying $\sqrt{12}$ before substituting.
↳ from STEP 3: $f(12) = 8\sqrt{3} + \tfrac13 + 36 + c$
STEP 4/5equate and cancel surds M1

Strategy. Set $f(12) = 8\sqrt{3}$ and watch the surd terms cancel by design.

Set up

$$8\sqrt{3} + \frac{1}{3} + 36 + c = 8\sqrt{3}.$$

Cancel $8\sqrt{3}$

$$\frac{1}{3} + 36 + c = 0 \quad\Longrightarrow\quad \frac{109}{3} + c = 0.$$
By design, not coincidence. The question setter chose $f(12) = 8\sqrt{3}$ precisely because $4\sqrt{12} = 8\sqrt{3}$, making the surd terms cancel and leaving a clean rational equation for $c$. Without this, $c$ would contain a surd. Diagnostic: if your $c$ comes out irrational on a “show that” question, re-check your surd simplification.
↳ from STEP 4: $\tfrac{109}{3} + c = 0$
STEP 5/5solve for c and write f(x) A1

Strategy. Solve the linear equation for $c$ and write the final $f(x)$.

Solve for $c$

$$c = -\frac{109}{3}.$$

Final $f(x)$

$$\boxed{f(x) = 4\sqrt{x} + \frac{4}{x} + 3x - \frac{109}{3}.}$$

Sense check. Verify: $f(12) = 8\sqrt{3} + \tfrac13 + 36 - \tfrac{109}{3} = 8\sqrt{3} + \tfrac{1+108-109}{3} = 8\sqrt{3}$ — matches the boundary condition.
