Q3 — Solve $21x^2 + 10x + 1 = 0$

Question

After substituting $y = -4x - 1$ into a quadratic curve, the equation $21x^2 + 10x + 1 = 0$ was reached. Solve for $x$.

[2 marks]

Solution [2 marks] · 2 steps

STEP 1/2 M1

Strategy: a quadratic with integer coefficients usually factors over the integers. Try factorisation first; only fall back on the quadratic formula if it doesn't split.

Step 1 — Read coefficients by matching $ax^2 + bx + c = 0$:

$\;\;a = 21,\quad b = 10,\quad c = 1$.

Step 2 — Split-the-middle method: find two integers with

$\;\;\text{product} = ac = 21 \cdot 1 = 21$

$\;\;\text{sum} = b = 10$

Factor pairs of $21$: $(1, 21)$ sum $22$; $(3, 7)$ sum $10$ ✓. Pick $(3, 7)$.

Step 3 — Rewrite the middle term $10x = 7x + 3x$:

$$21x^2 + 10x + 1 = 21x^2 + 7x + 3x + 1$$

Step 4 — Factor by grouping (pair the first two and last two):

$\;\;= 7x(3x + 1) + 1(3x + 1)$

$\;\;= (7x + 1)(3x + 1)$  (common bracket $3x+1$)

$$\boxed{\;(7x + 1)(3x + 1) = 0\;}$$

Why read $a,b,c$ first: both factorisation and the quadratic formula need these three. Note $c = 1$ (not $0$) — easy to misread when the constant is small. Quick check: expand $(7x+1)(3x+1) = 21x^2 + 7x + 3x + 1 = 21x^2 + 10x + 1$ ✓.
$(7x + 1)(3x + 1) = 0$

STEP 2/2 A1

Zero-product property: if $A \cdot B = 0$, then $A = 0$ or $B = 0$ (or both). So if a product of brackets equals zero, set each bracket to zero separately.

Bracket 1:

$\;\;7x + 1 = 0$

$\;\;7x = -1$

$\;\;x = -\tfrac{1}{7}$

Bracket 2:

$\;\;3x + 1 = 0$

$\;\;3x = -1$

$\;\;x = -\tfrac{1}{3}$

$$\boxed{\;x = -\tfrac{1}{7}\quad\text{or}\quad x = -\tfrac{1}{3}\;}$$

Fallback (memorise — NOT in booklet): $x = \dfrac{-b \pm \sqrt{b^2-4ac}}{2a}$. With $a=21, b=10, c=1$: $b^2 - 4ac = 100 - 84 = 16$, $\sqrt{16} = 4$, so $x = \dfrac{-10 \pm 4}{42}$ $= -\tfrac{6}{42} = -\tfrac{1}{7}$ or $-\tfrac{14}{42} = -\tfrac{1}{3}$ — same answers ✓.
$x = -\tfrac{1}{7}$ or $x = -\tfrac{1}{3}$
