$f(x) = x^2 - 6x + 3$ has roots $\alpha$ and $\beta$. Without solving the equation:

(a) Determine $(\alpha^2 + 1)(\beta^2 + 1)$. [4]

(b) Find a quadratic equation with roots $\dfrac{\alpha}{\alpha^2+1}$ and $\dfrac{\beta}{\beta^2+1}$, in form $px^2 + qx + r = 0$ with integer $p,q,r$. [6]

STEP 1/3 B1

Strategy: the question explicitly says without solving, so we work with the symmetric functions $\alpha+\beta$ and $\alpha\beta$ directly. Vieta's formulas extract these from the coefficients in one line — the foundation for every "new quadratic from $\alpha,\beta$" question.

Formula (Vieta for $ax^2 + bx + c = 0$):

$$\alpha + \beta \;=\; -\frac{b}{a}, \qquad \alpha\beta \;=\; \frac{c}{a}$$

Read off coefficients from $f(x) = x^2 - 6x + 3$:

$$a = 1,\quad b = -6,\quad c = 3$$

Substitute term-by-term:

$$\alpha + \beta \;=\; -\frac{-6}{1} \;=\; 6$$

$$\alpha\beta \;=\; \frac{3}{1} \;=\; 3$$

$$\boxed{\;\alpha + \beta = 6,\quad \alpha\beta = 3\;}$$

Sign-of-$b$ trap: Vieta gives $-b/a$, NOT $+b/a$. Here $b = -6$, so $\alpha+\beta = -(-6)/1 = +6$. Students who forget the leading minus get $\alpha+\beta = -6$, breaking every downstream calculation. Always copy $b$ with its sign first, then negate.
$\alpha+\beta=6,\;\alpha\beta=3$
