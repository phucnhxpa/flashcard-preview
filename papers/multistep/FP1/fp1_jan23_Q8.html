Q8 — Parabola: Focus, Directrix, Line Intersection, Area

Question

Parabola $y^2 = 4ax$. Line $y = k$ meets $C$ at $P$. (a) Show $PS$ formula. (b) Show $y$-coordinate of $A$. (c) Given area ratio, find $a$.

[14 marks]

Solution 4 steps

STEP 1/4

Part (a) — Find $PS$ (distance from $P$ to focus):

Parabola $y^2 = 4ax$, focus $S = (a, 0)$, directrix $x = -a$.

$y = k$ meets $y^2 = 4ax$: $k^2 = 4ax \Rightarrow x = \frac{k^2}{4a}$

So $P = (\frac{k^2}{4a}, k)$.

$PS = \sqrt{(\frac{k^2}{4a} - a)^2 + k^2}$. Simplifies to $\frac{k^2 + 4a^2}{4a}$.

Key property: Distance from any point on parabola to focus = distance to directrix.
$PS = \frac{k^2 + 4a^2}{4a}$
↳ from STEP 1:$P = (\frac{k^2}{4a}, k)$
STEP 2/4

Part (b) — Line $L_2$ and point $A$:

$L_2$ passes through $P(\frac{k^2}{4a}, k)$ and $(-a, 0)$ (directrix meets $x$-axis).

Gradient: $m = \frac{k}{\frac{k^2}{4a} + a} = \frac{4ak}{k^2 + 4a^2}$

Line through $(-a, 0)$: $y = m(x + a)$

At $x = 0$: $y_A = ma = \frac{4a^2k}{k^2 + 4a^2}$ ✓

$y_A = \frac{4a^2k}{k^2 + 4a^2}$
↳ from STEP 2:$y_A$ from part (b), $B = (-a, k)$
STEP 3/4

Part (c) — Triangle areas:

$B = (-a, k)$ (intersection of $y = k$ with directrix $x = -a$)

$\triangle OSP$: $O(0,0)$, $S(a,0)$, $P(\frac{k^2}{4a}, k)$

$$\text{Area}(\triangle OSP) = \frac{1}{2}|ak| = \frac{ak}{2}$$

Tutor: "This has nothing to do with the Parabola. This is finding the area of a triangle using the given information."
Areas computed using base × height or determinant formula
↳ from STEP 3:Triangle areas and ratio
STEP 4/4

Part (c) — Solve for $a$:

Set up the given area ratio: $\frac{\text{Area}(\triangle BPA)}{\text{Area}(\triangle OSP)} = \text{given ratio}$

Solve for $a$.

Parabola toolkit:
• Parametric: $(at^2, 2at)$
• Focus: $(a, 0)$, Directrix: $x = -a$
• Tangent: $ty = x + at^2$
• Normal: $y + tx = 2at + at^3$
Exact value of $a$ determined ✓
