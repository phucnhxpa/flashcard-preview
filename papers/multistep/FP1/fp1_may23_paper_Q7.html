Q7 — $f(x) = x^{3/2} + x - 3$

Question

$f(x) = x^{3/2} + x - 3$.

(a) Show $f(x)=0$ has a root $\alpha \in [1,\,2]$. [2]

(b) Use interval bisection twice on $[1,2]$ to show $\alpha\in[1.25,\,1.5]$. [3]

(c)(i) Find $f'(x)$. (ii) Use Newton–Raphson once with $x_0=1.375$ to give a 2nd approx to 3 d.p. [3]

(d) Use linear interpolation once on $[1.25,\,1.5]$ for a different approximation to 3 d.p. [3]

[11 marks]

Part (a) — Sign change on $[1,2]$ [2 marks] · 1 step

STEP 1/1 M1 A1

Method (sign-change theorem, IVT): if $f$ continuous on $[a,b]$ and $f(a),f(b)$ have opposite signs, then $f$ has a root in $(a,b)$.

Compute at the endpoints:

$$f(1) = 1^{3/2} + 1 - 3 = -1 < 0$$

$$f(2) = 2^{3/2} + 2 - 3 = 2\sqrt 2 - 1 \approx 1.828 > 0$$

$f$ continuous (sum of power and linear); sign change ⇒ root $\alpha\in(1,2)$. ■

Always state continuity + the sign of each endpoint value explicitly. Examiners reward verbatim phrasing.
$f(1)<0,\,f(2)>0$ ⇒ $\alpha\in(1,2)$

Part (b) — Two bisection iterations [3 marks] · 2 steps

From (a): $f(x) = x^{3/2} + x - 3$ with $f(1) = -1 < 0$, $f(2) = 2\sqrt 2 - 1 \approx 1.828 > 0$, so root $\alpha \in (1,2)$. Bisect twice.
STEP 1/2 M1

Iteration 1: midpoint of $[1,2]$ is $m_1=1.5$.

$$f(1.5) = 1.5^{1.5} + 1.5 - 3 \approx 1.837 + 1.5 - 3 = 0.337 > 0$$

$f(1)<0,\,f(1.5)>0$ ⇒ root in $[1,\,1.5]$.

Carry-in from previous stepFrom STEP 1: $f(1.5) = 1.5^{1.5} + 1.5 - 3 \approx 1.837 + 1.5 - 3 = 0.337 > 0$
STEP 2/2 M1 A1

Iteration 2: midpoint of $[1,1.5]$ is $m_2=1.25$.

$$f(1.25) = 1.25^{1.5} + 1.25 - 3 \approx 1.398 + 1.25 - 3 = -0.352 < 0$$

$f(1.25)<0,\,f(1.5)>0$ ⇒ root in $\boxed{[1.25,\,1.5]}$. ■

Bookkeeping: always state which side's sign rules out which half. Phuc lost marks on earlier papers for skipping the explicit "sign change → keep this half" sentence.
$\alpha \in [1.25,\,1.5]$

Part (c) — Newton-Raphson [3 marks] · 2 steps

From the question: $f(x) = x^{3/2} + x - 3$. (i) asks for $f'(x)$; (ii) applies Newton-Raphson once with $x_0 = 1.375$ (midpoint of $[1.25, 1.5]$ from (b)).
STEP 1/2 B1

Strategy: Newton-Raphson needs $f'(x_n)$ in the denominator, so we differentiate first. The fractional power $x^{3/2}$ is just like any other power for the power rule.

Formula (power rule):

$$\frac{d}{dx}\bigl(x^n\bigr) \;=\; n\,x^{n-1}$$

Apply term-by-term to $f(x) = x^{3/2} + x - 3$:

$$\frac{d}{dx}\bigl(x^{3/2}\bigr) \;=\; \tfrac{3}{2}\,x^{(3/2) - 1} \;=\; \tfrac{3}{2}\,x^{1/2}$$

$$\frac{d}{dx}(x) \;=\; 1$$

$$\frac{d}{dx}(-3) \;=\; 0 \quad(\text{constant has zero slope})$$

Sum the results:

$$\boxed{\;f'(x) \;=\; \tfrac{3}{2}\,x^{1/2} + 1 \;=\; \tfrac{3}{2}\sqrt x + 1\;}$$

Fractional-exponent slip: $\tfrac{3}{2} - 1 = \tfrac{1}{2}$, NOT $\tfrac{3}{2}$. Common error is to leave the exponent unchanged: $\tfrac{3}{2}x^{3/2}$. Always subtract $1$ from the exponent, even when it's a fraction. Equivalent forms: $x^{1/2} = \sqrt{x}$ — either is acceptable.
$f'(x) = \tfrac{3}{2}\sqrt x + 1$
Carry-in from previous stepFrom STEP 1: $f'(x) = \tfrac{3}{2}\sqrt x + 1$
STEP 2/2 M1 A1

Formula (Newton-Raphson — IN booklet):

$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

Substitute $x_0 = 1.375$:

$f(1.375) = 1.375^{1.5} + 1.375 - 3 \approx 1.6125 + 1.375 - 3 = -0.01246$

$f'(1.375) = \tfrac{3}{2}\sqrt{1.375} + 1 \approx 1.5 \cdot 1.1726 + 1 = 2.7588$

$$x_1 = 1.375 - \frac{-0.01246}{2.7588} = 1.375 + 0.004516 \approx \boxed{1.380}$$ (3 d.p.)

Calculator hygiene: store $x_0$ in memory, use ANS for both $f$ and $f'$ to avoid rounding propagation. Round only at the end.
$x_1 \approx 1.380$

Part (d) — Linear interpolation on $[1.25,\,1.5]$ [3 marks] · 2 steps

From (b): $f(1.25)\approx -0.352$, $f(1.5)\approx 0.337$.
STEP 1/2 M1

Formula (linear interpolation — NOT in booklet, MEMORIZE):

$$x_1 = a - \frac{f(a)\,(b-a)}{f(b) - f(a)}$$

Phuc's slip: forgot the closed form on first pass; had to derive via similar triangles. This is now the THIRD FP1 paper with linear interp — commit to memory once and for all.
Carry-in from previous stepFrom STEP 1: $x_1 = a - \frac{f(a)\,(b-a)}{f(b) - f(a)}$
STEP 2/2 M1 A1

Substitute $a=1.25$, $b=1.5$, $f(a)=-0.352$, $f(b)=0.337$:

$$x_1 = 1.25 - \frac{(-0.352)(1.5-1.25)}{0.337 - (-0.352)} = 1.25 - \frac{(-0.352)(0.25)}{0.689}$$

$$= 1.25 - \frac{-0.088}{0.689} = 1.25 + 0.1278 \approx \boxed{1.378}$$ (3 d.p.)

Compare methods: Newton–Raphson → 1.380; linear interp → 1.378. Both close to the true root $\approx 1.378$, but NR starts from a single seed inside the bracket so converges quadratically; LI uses both endpoints linearly.
$x_1 \approx 1.378$
