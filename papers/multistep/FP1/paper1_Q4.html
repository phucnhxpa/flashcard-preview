Q4 — $A=\!\begin{pmatrix}1&0\\0&3\end{pmatrix}$; describe $A$; rotation $210^\circ$; composition; area

Question

$A = \begin{pmatrix} 1 & 0 \\ 0 & 3 \end{pmatrix}$. $B$ represents a rotation of $210^\circ$ anticlockwise about the origin.

(a) Describe the single transformation represented by $A$. [2]

(b) Write $B$ in exact form. [1]

(c) The matrix $C$ represents the transformation "$A$ followed by $B$". Find $C$. [2]

(d) Hexagon $H$ has area $5$. $H'$ is the image of $H$ under $C$. Find area($H'$). [2]

[7 marks]

Part (a) — Describe $A$ [2 marks] · 1 step

STEP 1/1 M1 A1

$$A\!\begin{pmatrix}1\\0\end{pmatrix} = \begin{pmatrix}1\\0\end{pmatrix} \quad (\text{$x$-axis fixed})$$

$$A\!\begin{pmatrix}0\\1\end{pmatrix} = \begin{pmatrix}0\\3\end{pmatrix} \quad (\text{$y$-coord scaled by 3})$$

The image of the basis vectors gives the columns of the matrix — read the geometry directly off them.

Stretch, scale factor $3$, parallel to the $y$-axis (equivalently, $x$-axis is the invariant line).

Exam-standard wording: always include all three pieces: type + scale factor + invariant line/axis. Mark scheme penalises "stretch by 3" without the direction.
Stretch, SF $3$, parallel to $y$-axis

Part (b) — Rotation 210° in exact form [1 marks] · 1 step

From (a): $A = \begin{pmatrix} 1 & 0 \\ 0 & 3 \end{pmatrix}$ (stretch SF $3$ parallel to $y$-axis). $B$ is the $210^\circ$ anticlockwise rotation matrix to find now.
STEP 1/1 B1

Formula (booklet) — rotation by $\theta$ anticlockwise:

$$R_\theta = \begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix}$$

$210^\circ$ is in Q3. Reference angle $30^\circ$:

$$\cos 210^\circ = -\cos 30^\circ = -\tfrac{\sqrt{3}}{2}, \quad \sin 210^\circ = -\sin 30^\circ = -\tfrac{1}{2}$$

Substitute:

$$B = \begin{pmatrix} -\tfrac{\sqrt{3}}{2} & \tfrac{1}{2} \\ -\tfrac{1}{2} & -\tfrac{\sqrt{3}}{2} \end{pmatrix}$$

Calculator trap: calculator gives decimals; exam wants surds. Memorise unit-circle exact values for multiples of $30^\circ$ and $45^\circ$.
$B = \begin{pmatrix} -\tfrac{\sqrt{3}}{2} & \tfrac{1}{2} \\ -\tfrac{1}{2} & -\tfrac{\sqrt{3}}{2} \end{pmatrix}$

Part (c) — Composition $C$ [2 marks] · 2 steps

Order rule: "$A$ followed by $B$" $\Rightarrow C = BA$ (B on the LEFT).
STEP 1/2 M1

Strategy: when transformations are applied in sequence, the matrices multiply in REVERSE order — the matrix nearest the vector acts first. This is the single biggest source of marks lost on composition questions.

Order rule: if $T_1$ is applied first and $T_2$ second, the combined transformation has matrix $T_2 T_1$ (NOT $T_1 T_2$).

For a column vector $\mathbf{x}$, "$A$ then $B$" means:

$$\mathbf{x} \;\xrightarrow{\;A\;}\; A\mathbf{x} \;\xrightarrow{\;B\;}\; B(A\mathbf{x}) = (BA)\,\mathbf{x}$$

So the composition matrix is:

$$\boxed{\;C = BA\;}$$

Why reverse? Function composition reads right-to-left: $f\circ g$ means "do $g$ first". Same convention here: in $BA\mathbf{x}$, the matrix nearest $\mathbf{x}$ acts first. Writing $C = AB$ produces a different matrix entirely — matrix multiplication is non-commutative, so $AB \ne BA$ in general.
Carry-in from previous stepFrom STEP 1: $\boxed{\;C = BA\;}$
STEP 2/2 A1

$$C = BA = \begin{pmatrix} -\tfrac{\sqrt{3}}{2} & \tfrac{1}{2} \\ -\tfrac{1}{2} & -\tfrac{\sqrt{3}}{2} \end{pmatrix}\!\begin{pmatrix} 1 & 0 \\ 0 & 3 \end{pmatrix}$$

Row × column:

$$C = \begin{pmatrix} -\tfrac{\sqrt{3}}{2}\cdot 1 + \tfrac{1}{2}\cdot 0 & -\tfrac{\sqrt{3}}{2}\cdot 0 + \tfrac{1}{2}\cdot 3 \\ -\tfrac{1}{2}\cdot 1 + (-\tfrac{\sqrt{3}}{2})\cdot 0 & -\tfrac{1}{2}\cdot 0 + (-\tfrac{\sqrt{3}}{2})\cdot 3 \end{pmatrix}$$

$$C = \begin{pmatrix} -\tfrac{\sqrt{3}}{2} & \tfrac{3}{2} \\ -\tfrac{1}{2} & -\tfrac{3\sqrt{3}}{2} \end{pmatrix}$$

$C = \begin{pmatrix} -\tfrac{\sqrt{3}}{2} & \tfrac{3}{2} \\ -\tfrac{1}{2} & -\tfrac{3\sqrt{3}}{2} \end{pmatrix}$

Part (d) — Area of $H'$ [2 marks] · 1 step

From (a)/(c): $\det A = 3$, $\det B = 1$ (rotation), so $\det C = \det(BA) = 3$. Hexagon $H$ has area $5$.
STEP 1/1 M1 A1

Key fact: for any 2x2 matrix $C$,

$$\text{Area}(H') = |\det C|\cdot \text{Area}(H)$$

Shortcut: $\det(BA) = \det(B)\det(A)$.

$\det(B) = 1$ (any rotation has $\det = 1$); $\det(A) = 1\cdot 3 - 0 = 3$.

$$\det(C) = 1 \cdot 3 = 3$$

$$\text{Area}(H') = |\det C|\cdot \text{Area}(H) = 3 \times 5 = \boxed{15}\ \text{square units}$$

Sanity check via direct computation: $\det(C) = (-\tfrac{\sqrt{3}}{2})(-\tfrac{3\sqrt{3}}{2}) - (\tfrac{3}{2})(-\tfrac{1}{2}) = \tfrac{9}{4} + \tfrac{3}{4} = 3$ ✓
Area$(H') = 15$
