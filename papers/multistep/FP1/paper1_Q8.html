Q8 — Show $\sum_{r=1}^n r(2r^2-3r-1) = \tfrac{1}{2}n(n+1)^2(n-2)$; shifted limits

Question

(a) Use standard results to show $\displaystyle\sum_{r=1}^{n} r(2r^2 - 3r - 1) = \tfrac{1}{2}n(n+1)^2(n-2)$. [4]

(b) Hence show $\displaystyle\sum_{r=n}^{2n} r(2r^2 - 3r - 1) = \tfrac{1}{2}n(n-1)(an+b)(cn+d)$ where $a,b,c,d$ are integers. [4]

[8 marks]

Part (a) — Closed form [4 marks] · 2 steps

STEP 1/2 M1

Standard sums:

$$\sum_{r=1}^{n} r = \tfrac{1}{2}n(n+1) \quad (\text{NOT in booklet})$$

$$\sum_{r=1}^{n} r^2 = \tfrac{1}{6}n(n+1)(2n+1), \quad \sum_{r=1}^{n} r^3 = \tfrac{1}{4}n^2(n+1)^2$$

Tutor warning: "They only have $r^2$ and $r^3$. We need to know the $r$ one then — always consult the formula booklet first."

$$r(2r^2 - 3r - 1) = 2r^3 - 3r^2 - r$$

$$\sum_{r=1}^{n} \bigl(2r^3 - 3r^2 - r\bigr) = 2\sum r^3 - 3\sum r^2 - \sum r$$

Substitute the closed forms:

$$= \tfrac{1}{2}n^2(n+1)^2 - \tfrac{1}{2}n(n+1)(2n+1) - \tfrac{1}{2}n(n+1)$$

Carry-in from previous stepFrom STEP 1: $= \tfrac{1}{2}n^2(n+1)^2 - \tfrac{1}{2}n(n+1)(2n+1) - \tfrac{1}{2}n(n+1)$
STEP 2/2 M1 A1

$$= \tfrac{1}{2}n(n+1)\Bigl[\,n(n+1) - (2n+1) - 1\,\Bigr]$$

Simplify inside:

$$n(n+1) - (2n+1) - 1 = n^2 + n - 2n - 2 = n^2 - n - 2 = (n-2)(n+1)$$

So:

$$\sum_{r=1}^{n} r(2r^2 - 3r - 1) = \tfrac{1}{2}n(n+1)(n-2)(n+1) = \boxed{\tfrac{1}{2}n(n+1)^2(n-2)}\ \blacksquare$$

Tutor key technique: "Before any multiplication, take this common factor. If you multiply out as it is, it will be very, very complicated. Always look at what they want, then factorise early."
$\tfrac{1}{2}n(n+1)^2(n-2)$

Part (b) — Shifted limits $r=n$ to $2n$ [4 marks] · 3 steps

Closed form: $S(N) := \sum_{r=1}^{N} r(2r^2-3r-1) = \tfrac{1}{2}N(N+1)^2(N-2)$.
STEP 1/3 M1 M1

Identity:

$$\sum_{r=n}^{2n} = \sum_{r=1}^{2n} - \sum_{r=1}^{n-1} = S(2n) - S(n-1)$$

Plug into $S(N) = \tfrac{1}{2}N(N+1)^2(N-2)$:

$$S(2n) = \tfrac{1}{2}(2n)(2n+1)^2(2n-2) = \tfrac{1}{2}(2n)(2n+1)^2 \cdot 2(n-1) = 2n(n-1)(2n+1)^2$$

$$S(n-1) = \tfrac{1}{2}(n-1)(n-1+1)^2((n-1)-2) = \tfrac{1}{2}(n-1)\,n^2\,(n-3)$$

Carry-in from previous stepFrom STEP 1: $S(n-1) = \tfrac{1}{2}(n-1)(n-1+1)^2((n-1)-2) = \tfrac{1}{2}(n-1)\,n^2\,(n-3)$
STEP 2/3 A1

$$\sum_{r=n}^{2n} = 2n(n-1)(2n+1)^2 - \tfrac{1}{2}n^2(n-1)(n-3)$$

$$= \tfrac{1}{2}n(n-1)\Bigl[\,4(2n+1)^2 - n(n-3)\,\Bigr]$$

Expand bracket:

$$4(4n^2+4n+1) - (n^2-3n) = 16n^2 + 16n + 4 - n^2 + 3n = 15n^2 + 19n + 4$$

Carry-in from previous stepFrom STEP 2: $4(4n^2+4n+1) - (n^2-3n) = 16n^2 + 16n + 4 - n^2 + 3n = 15n^2 + 19n + 4$
STEP 3/3 A1

Factorise $15n^2 + 19n + 4$. Need product $15\cdot 4 = 60$, sum $19$: $15$ and $4$.

$$15n^2 + 15n + 4n + 4 = 15n(n+1) + 4(n+1) = (15n+4)(n+1)$$

$$\sum_{r=n}^{2n} r(2r^2 - 3r - 1) = \tfrac{1}{2}n(n-1)(n+1)(15n+4)$$

Match $\tfrac{1}{2}n(n-1)(an+b)(cn+d)$:

$$\boxed{\;a=1,\ b=1,\ c=15,\ d=4\;}$$

$a=1,\ b=1,\ c=15,\ d=4$
