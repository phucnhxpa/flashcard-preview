Q9 — $\dfrac{3z-1}{2} = \dfrac{\lambda+5i}{\lambda - 4i}$; find $z$, then $\lambda$ when $\arg z = \pi/4$

Question

$\dfrac{3z-1}{2} = \dfrac{\lambda + 5i}{\lambda - 4i}$, where $\lambda \in \mathbb{R}$.

(a) Determine $z$ in the form $x + yi$, with $x,y$ real and in terms of $\lambda$. [4]

(b) Given that $\arg z = \dfrac{\pi}{4}$, find the possible values of $\lambda$. [2]

[6 marks]

Part (a) — Solve for $z$, then rationalise [4 marks] · 2 steps

STEP 1/2 M1

Multiply by $2$:

$$3z - 1 = \frac{2\lambda + 10i}{\lambda - 4i}$$

Add $1 = \dfrac{\lambda - 4i}{\lambda - 4i}$:

$$3z = \frac{(\lambda - 4i) + (2\lambda + 10i)}{\lambda - 4i} = \frac{3\lambda + 6i}{\lambda - 4i}$$

Divide by $3$:

$$z = \frac{\lambda + 2i}{\lambda - 4i}$$

$z = \dfrac{\lambda + 2i}{\lambda - 4i}$
Carry-in from previous stepFrom STEP 1: $z = \dfrac{\lambda + 2i}{\lambda - 4i}$
STEP 2/2 M1 A1 A1

Formula (sum-of-squares): $(a-bi)(a+bi) = a^2 + b^2$.

Denominator: $(\lambda - 4i)(\lambda + 4i) = \lambda^2 + 16$.

Numerator:

$$(\lambda + 2i)(\lambda + 4i) = \lambda^2 + 4\lambda i + 2\lambda i + 8i^2 = (\lambda^2 - 8) + 6\lambda i$$

$$z = \frac{\lambda^2 - 8}{\lambda^2 + 16} + \frac{6\lambda}{\lambda^2 + 16}\,i$$

$$\boxed{\;x = \frac{\lambda^2 - 8}{\lambda^2 + 16},\quad y = \frac{6\lambda}{\lambda^2 + 16}\;}$$

$x = \tfrac{\lambda^2-8}{\lambda^2+16},\ y = \tfrac{6\lambda}{\lambda^2+16}$

Part (b) — $\arg z = \pi/4$ [2 marks] · 2 steps

From (a): $x = \tfrac{\lambda^2-8}{\lambda^2+16},\ y = \tfrac{6\lambda}{\lambda^2+16}$.
STEP 1/2 M1

Formula: $\arg z = \theta \Rightarrow \tan\theta = y/x$ (in correct quadrant).

$\tan(\pi/4) = 1$ and $\pi/4$ is in Q1 (so need $x>0$ AND $y>0$).

Both $x$ and $y$ have denominator $\lambda^2 + 16 > 0$, so set numerators equal:

$$6\lambda = \lambda^2 - 8 \;\Rightarrow\; \lambda^2 - 6\lambda - 8 = 0$$

$$\lambda = \frac{6 \pm \sqrt{36 + 32}}{2} = \frac{6 \pm \sqrt{68}}{2} = 3 \pm \sqrt{17}$$

$\lambda = 3 \pm \sqrt{17}$
Carry-in from previous stepFrom STEP 1: $\lambda = 3 \pm \sqrt{17}$
STEP 2/2 A1

$\lambda = 3 + \sqrt{17} \approx 7.12$: $\lambda^2 \approx 50.7 > 8$ so $x > 0$; $y = 6\lambda > 0$. ✓ In Q1, so $\arg z = \pi/4$. Valid.

$\lambda = 3 - \sqrt{17} \approx -1.12$: $\lambda^2 \approx 1.25 < 8$ so $x<0$; $y = 6\lambda < 0$. In Q3, so $\arg z = -3\pi/4 \ne \pi/4$. Reject.

$$\boxed{\;\lambda = 3 + \sqrt{17}\;}$$
