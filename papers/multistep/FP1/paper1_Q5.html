Q5 — $2x^2 - 3x + 7 = 0$ has roots $\alpha,\beta$; build new quadratic

Question

The equation $2x^2 - 3x + 7 = 0$ has roots $\alpha$ and $\beta$.

(a) State $\alpha+\beta$ and $\alpha\beta$. [1]

(b) Find $\alpha^2 + \beta^2$. [2]

(c) Find a quadratic equation with integer coefficients having roots $\alpha - \dfrac{1}{\beta^2}$ and $\beta - \dfrac{1}{\alpha^2}$, in the form $px^2 + qx + r = 0$ with integer $p,q,r$. [6]

[9 marks]

Part (a) — Vieta [1 marks] · 1 step

STEP 1/1 B1

Formula (Vieta for $ax^2+bx+c=0$):

$$\alpha+\beta = -\frac{b}{a}, \qquad \alpha\beta = \frac{c}{a}$$

Substitute $a=2,\ b=-3,\ c=7$:

$$\alpha+\beta = -\frac{-3}{2} = \frac{3}{2}, \quad \alpha\beta = \frac{7}{2}$$

$\alpha+\beta = \tfrac{3}{2},\ \alpha\beta = \tfrac{7}{2}$

Part (b) — $\alpha^2 + \beta^2$ [2 marks] · 1 step

From (a): $\alpha+\beta = \tfrac{3}{2}$, $\alpha\beta = \tfrac{7}{2}$.
STEP 1/1 M1 A1

Identity (MEMORIZE):

$$\alpha^2 + \beta^2 = (\alpha+\beta)^2 - 2\alpha\beta$$

Substitute:

$$= \left(\tfrac{3}{2}\right)^2 - 2\cdot\tfrac{7}{2} = \tfrac{9}{4} - 7 = -\tfrac{19}{4}$$

Negative is fine — this is just a useful intermediate quantity, not a length. Real roots would have $\alpha^2+\beta^2 > 0$, but here the discriminant $9-56<0$ tells us the roots are complex.
$\alpha^2 + \beta^2 = -\tfrac{19}{4}$

Part (c) — Build the new quadratic [6 marks] · 3 steps

From (a)/(b): $\alpha+\beta=\tfrac{3}{2}$, $\alpha\beta=\tfrac{7}{2}$, $\alpha^2+\beta^2=-\tfrac{19}{4}$.
STEP 1/3 M1 A1

Let $A = \alpha - \tfrac{1}{\beta^2}$, $B = \beta - \tfrac{1}{\alpha^2}$.

Identity: $\dfrac{1}{\alpha^2} + \dfrac{1}{\beta^2} = \dfrac{\alpha^2+\beta^2}{(\alpha\beta)^2}$.

$$A + B = (\alpha+\beta) - \frac{\alpha^2+\beta^2}{(\alpha\beta)^2}$$

Substitute $\alpha+\beta=\tfrac{3}{2}$, $\alpha^2+\beta^2=-\tfrac{19}{4}$, $\alpha\beta=\tfrac{7}{2}$:

$$A + B = \frac{3}{2} - \frac{-19/4}{(7/2)^2} = \frac{3}{2} - \frac{-19/4}{49/4} = \frac{3}{2} + \frac{19}{49}$$

Common denominator $98$:

$$A + B = \frac{147}{98} + \frac{38}{98} = \frac{185}{98}$$

$A + B = \dfrac{185}{98}$
Carry-in from previous stepFrom STEP 1: $A + B = \dfrac{185}{98}$
STEP 2/3 M1 A1

$$AB = \!\left(\alpha - \tfrac{1}{\beta^2}\right)\!\!\left(\beta - \tfrac{1}{\alpha^2}\right) = \alpha\beta - \tfrac{\alpha}{\alpha^2} - \tfrac{\beta}{\beta^2} + \tfrac{1}{(\alpha\beta)^2}$$

$$= \alpha\beta - \tfrac{1}{\alpha} - \tfrac{1}{\beta} + \tfrac{1}{(\alpha\beta)^2} = \alpha\beta - \frac{\alpha+\beta}{\alpha\beta} + \frac{1}{(\alpha\beta)^2}$$

Substitute $\alpha+\beta=\tfrac{3}{2}$, $\alpha\beta=\tfrac{7}{2}$:

$$AB = \tfrac{7}{2} - \frac{3/2}{7/2} + \frac{1}{(7/2)^2} = \tfrac{7}{2} - \tfrac{3}{7} + \tfrac{4}{49}$$

Common denominator $98$:

$$AB = \tfrac{343}{98} - \tfrac{42}{98} + \tfrac{8}{98} = \tfrac{309}{98}$$

$AB = \dfrac{309}{98}$
Carry-in from previous stepFrom STEP 2: $AB = \dfrac{309}{98}$
STEP 3/3 M1 A1

Formula: if roots are $A,B$ then $x^2 - (A+B)x + AB = 0$.

$$x^2 - \tfrac{185}{98}x + \tfrac{309}{98} = 0$$

Multiply by $98$ to clear fractions:

$$\boxed{\;98x^2 - 185x + 309 = 0\;}$$

Tutor: 3-unknowns/1-equation trick: "We are free to select one variable freely — generally we say $p=1$. Scaling preserves the roots." Here we scaled by $98$ to make $p,q,r$ integers.
$98x^2 - 185x + 309 = 0$
