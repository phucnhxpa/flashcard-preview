Q9 — Prove by induction: $18 \mid (4^n + 6n - 10)$ for $n\in\mathbb{Z}^+,\,n\ge 2$

Question

Prove, by induction, that for $n \in \mathbb{Z}^+,\;n \ge 2$:

$$4^n + 6n - 10\;\;\text{is divisible by 18.}$$

[5 marks]

[5 marks]

Solution — 4-step induction template [5 marks] · 5 steps

4-step template (memorize): BASE → ASSUME → INDUCT → CONCLUDE. Define $f(n) = 4^n + 6n - 10$.
STEP 1/5 B1

STEP 1 — Base case ($n=2$, the paper’s start):

$$f(2) = 4^2 + 6(2) - 10 = 16 + 12 - 10 = 18 = 18 \times 1$$

So $18 \mid f(2)$. ✓

Note: $f(1) = 4+6-10 = 0 = 18\times 0$ also works (0 is divisible by every integer), so the result actually holds for $n\ge 1$. The paper picks $n\ge 2$ to avoid edge-case discussion.
$f(2) = 18$ ✓
Carry-in from previous stepFrom STEP 1: $f(2) = 18$ ✓
STEP 2/5 M1

STEP 2 — Inductive hypothesis: assume true for some $n=k\ge 2$:

$$f(k) = 4^k + 6k - 10 = 18m,\;\;\text{some } m\in\mathbb{Z}$$

Carry-in from previous stepFrom STEP 2: $f(k) = 4^k + 6k - 10 = 18m,\;\;\text{some } m\in\mathbb{Z}$
STEP 3/5 M1

STEP 3 — Inductive step: compute $f(k+1)$ and express it in terms of $f(k)$ + a clean multiple of 18.

$$f(k+1) = 4^{k+1} + 6(k+1) - 10 = 4\cdot 4^k + 6k - 4$$

Engineer $f(k)$ on the RHS:

$$f(k+1) = \underbrace{(4^k + 6k - 10)}_{f(k)} + (3\cdot 4^k + 6)$$

The trick that tripped Fuat in Loom: the residual $3\cdot 4^k + 6$ alone isn’t obviously $18\times(\text{integer})$. We need to combine it with $f(k)$ to get an $18\cdot(\ldots)$ form. Re-express it via $3 f(k)$ to absorb the awkward $3\cdot 4^k$.
Carry-in from previous stepFrom STEP 3: $f(k+1) = \underbrace{(4^k + 6k - 10)}_{f(k)} + (3\cdot 4^k + 6)$
STEP 4/5 A1

Move (express residual via $3 f(k)$):

$$3\,f(k) = 3\cdot 4^k + 18k - 30$$

$$\Rightarrow\; 3\cdot 4^k + 6 = 3 f(k) - 18k + 30 + 6 = 3 f(k) - 18(k-2)$$

So:

$$f(k+1) = f(k) + 3 f(k) - 18(k-2) = 4\,f(k) - 18(k-2)$$

By IH: $f(k) = 18m$. Then:

$$f(k+1) = 4(18m) - 18(k-2) = 18\,(4m - k + 2)$$

$4m - k + 2 \in \mathbb{Z}$, so $18 \mid f(k+1)$. ✓

Fuat (Loom) verbatim: "$f(k+1)$ is going to be $3 f(k)$, and $f(k)$ is going to be $4 f(k)$ minus $18(k-2)$. So we know that $f(k)$ is divisible by 18, no matter which number we multiply out. Also, 18 is divisible by 18, so all terms together are divisible by 18."
$f(k+1) = 18(4m - k + 2)$
Carry-in from previous stepFrom STEP 4: $f(k+1) = 18(4m - k + 2)$
STEP 5/5 A1

STEP 4 — Conclusion:

The statement is true for $n=2$. We have shown that if it is true for $n=k$, then it is true for $n=k+1$. Therefore, by the principle of mathematical induction, $4^n + 6n - 10$ is divisible by 18 for all $n\in\mathbb{Z}^+,\,n\ge 2$. $\blacksquare$

Examiners reward verbatim phrasing — always include "true for $n=k$ ⇒ true for $n=k+1$" and "by the principle of mathematical induction". Even if the algebra wobbles, the structure marks (B1/A1 final) are bankable.
True $\forall\,n\in\mathbb{Z}^+,\,n\ge 2$. $\blacksquare$
