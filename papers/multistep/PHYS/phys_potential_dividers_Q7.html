Self-Test Q5 — loaded vs unloaded output, and the loading rule

Question

A divider has $R_1 = 5.0\ \text{k}\Omega$ (top) and $R_2 = 10.0\ \text{k}\Omega$ (bottom) across a $9.0\ \text{V}$ supply. A load of $R_L = 30.0\ \text{k}\Omega$ is connected across $R_2$.

(a) Find the parallel combination of $R_2$ and the load. [2 marks]
(b) Find the loaded $V_{\text{out}}$. [2 marks]
(c) Find $V_{\text{out}}$ with no load and comment on the difference. [2 marks]

[6 marks]

Part (a) — parallel combination [2 marks] · 1 step

STEP 1/1 M1 A1

Formula (load in parallel with $R_2$):

$$R_{BC} = \dfrac{R_2 R_L}{R_2 + R_L}$$

Substitute $R_2 = 10.0\ \text{k}\Omega$, $R_L = 30.0\ \text{k}\Omega$:

$$R_{BC} = \dfrac{10.0 \times 30.0}{10.0 + 30.0} = \dfrac{300}{40} = 7.5\ \text{k}\Omega$$

Trap: the parallel value is always smaller than the smaller resistor, so $R_{BC} = 7.5\ \text{k}\Omega < R_2 = 10.0\ \text{k}\Omega$. The product-over-sum form is quickest for two resistors.
$R_{BC} = 7.5\ \text{k}\Omega$

Part (b) — loaded output [2 marks] · 1 step

Carry-in from (a): effective bottom resistance $R_{BC} = 7.5\ \text{k}\Omega$; $R_1 = 5.0\ \text{k}\Omega$, $V_{\text{in}} = 9.0\ \text{V}$.
↳ from STEP 1: $R_{BC} = 7.5\ \text{k}\Omega$, $R_1 = 5.0\ \text{k}\Omega$, $V_{\text{in}} = 9.0\ \text{V}$
STEP 1/1 M1 A1

Formula (divider with the loaded bottom resistance):

$$V_{\text{out}} = V_{\text{in}} \cdot \dfrac{R_{BC}}{R_1 + R_{BC}}$$

Substitute $V_{\text{in}} = 9.0$, $R_1 = 5.0$, $R_{BC} = 7.5$:

$$V_{\text{out}} = 9.0 \times \dfrac{7.5}{5.0 + 7.5} = 9.0 \times \dfrac{7.5}{12.5} = 5.4\ \text{V}$$

Use $R_{BC}$, not $R_2$: the bottom of the divider is now the parallel combination $7.5\ \text{k}\Omega$. The k$\Omega$ cancel, so the result is in volts.
$V_{\text{out}} = 5.4\ \text{V}$ (loaded)

Part (c) — unloaded output and comment [2 marks] · 1 step

Carry-in from (a) & (b): loaded $V_{\text{out}} = 5.4\ \text{V}$; unloaded uses $R_2 = 10.0\ \text{k}\Omega$, $R_1 = 5.0\ \text{k}\Omega$, $V_{\text{in}} = 9.0\ \text{V}$. Loading rule of thumb: $R_L \ge 10 R_2$.
↳ from STEP 1: unloaded uses $R_2 = 10.0\ \text{k}\Omega$; loaded answer was $5.4\ \text{V}$; rule $R_L \ge 10 R_2$
STEP 1/1 A1 B1

Unloaded output (bottom is $R_2$):

$$V_{\text{out}} = 9.0 \times \dfrac{10.0}{5.0 + 10.0} = 9.0 \times \dfrac{10}{15} = 6.0\ \text{V}$$

Comment: the load reduced $V_{\text{out}}$ from $6.0\ \text{V}$ to $5.4\ \text{V}$, a drop of $0.6\ \text{V}$.

Why the drop is noticeable: the load $R_L = 30\ \text{k}\Omega$ is only $3\times R_2$, well below the $R_L \ge 10 R_2$ rule of thumb. A load of $\ge 100\ \text{k}\Omega$ would have kept $V_{\text{out}}$ much closer to $6.0\ \text{V}$.
Unloaded $V_{\text{out}} = 6.0\ \text{V}$; load drops it by $0.6\ \text{V}$ ($R_L = 3R_2 < 10R_2$)
