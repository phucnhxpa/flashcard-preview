Tutor Q — Find $V_K$, $I_L$ and each resistance in a series–parallel network

Question

A $9.0\ \text{V}$ source drives resistor $K$ in series with a parallel group of two resistors $L$ and $M$. The measured quantities are: current through $K$ is $I_K = 3.0\ \text{A}$, current through $M$ is $I_M = 2.0\ \text{A}$, and the p.d. across the parallel section is $V_{LM} = 2.0\ \text{V}$.

(a) Find the p.d. across $K$, $V_K$. [2 marks]
(b) Find the current through $L$, $I_L$. [2 marks]
(c) Hence find the three resistances $R_K$, $R_L$, $R_M$. [3 marks]

[7 marks]

Part (a) — p.d. across K [2 marks] · 1 step

STEP 1/1 M1 A1

Formula (Kirchhoff's second law around one loop):

$$\sum \varepsilon = \sum IR \quad\Rightarrow\quad \varepsilon = V_K + V_{LM}$$

A complete path is: source $\to K \to$ (either $L$ or $M$) $\to$ back. Rearrange for $V_K$ and substitute $\varepsilon = 9.0\ \text{V}$, $V_{LM} = 2.0\ \text{V}$:

$$V_K = \varepsilon - V_{LM} = 9.0 - 2.0 = 7.0\ \text{V}$$

Phuc's slip (lesson 35:25): first assumed $V_K = 2.0\ \text{V}$ (the parallel-section voltage). He self-corrected with KVL: $V_K = 9.0 - 2.0 = 7.0\ \text{V}$. The $2.0\ \text{V}$ is across $L\!\parallel\!M$, not across $K$.
Phuc's double-count trap (lesson 36:04): do NOT add $V_K + V_L + V_M = 7 + 2 + 2 = 11\ \text{V}$. A charge passes through $K$ then through one branch only. Any single loop: $7 + 2 = 9.0\ \text{V} = \varepsilon$ ✓.
$V_K = 7.0\ \text{V}$

Part (b) — current through L [2 marks] · 1 step

Carry-in from (a): $V_K = 7.0\ \text{V}$ and $V_{LM} = 2.0\ \text{V}$. Given currents: $I_K = 3.0\ \text{A}$ (into the parallel section), $I_M = 2.0\ \text{A}$ (through branch $M$).
↳ from STEP 1: $I_K = 3.0\ \text{A}$ enters the $L\parallel M$ node; $I_M = 2.0\ \text{A}$
STEP 1/1 M1 A1

Formula (Kirchhoff's first law at the split node):

$$\sum I_{\text{in}} = \sum I_{\text{out}} \quad\Rightarrow\quad I_K = I_L + I_M$$

The current $I_K$ entering the node splits between the two branches. Rearrange for $I_L$ and substitute $I_K = 3.0\ \text{A}$, $I_M = 2.0\ \text{A}$:

$$I_L = I_K - I_M = 3.0 - 2.0 = 1.0\ \text{A}$$

Why KCL here: the same total current $I_K$ that flows through the series resistor $K$ arrives at the parallel node and divides. Charge is conserved, so the two branch currents must sum to $I_K$.
$I_L = 1.0\ \text{A}$

Part (c) — the three resistances [3 marks] · 1 step

Carry-in from (a) & (b): $V_K = 7.0\ \text{V}$ with $I_K = 3.0\ \text{A}$; parallel-section voltage $V_L = V_M = V_{LM} = 2.0\ \text{V}$; branch currents $I_L = 1.0\ \text{A}$, $I_M = 2.0\ \text{A}$.
↳ from STEP 1: $V_K = 7.0\ \text{V}$, $I_K = 3.0\ \text{A}$; $V_L = V_M = 2.0\ \text{V}$; $I_L = 1.0$, $I_M = 2.0\ \text{A}$
STEP 1/1 A1 A1 A1

Formula (definition of resistance — never $R = IV$):

$$R = \dfrac{V}{I}$$

Resistor $K$: $\;R_K = \dfrac{V_K}{I_K} = \dfrac{7.0}{3.0} = 2.33\ \Omega$

Resistor $L$: $\;R_L = \dfrac{V_L}{I_L} = \dfrac{2.0}{1.0} = 2.0\ \Omega$

Resistor $M$: $\;R_M = \dfrac{V_M}{I_M} = \dfrac{2.0}{2.0} = 1.0\ \Omega$

Common trap (R = IV): resistance is $R = V/I$, never $R = IV$ (that gives units $\text{V·A}$, not $\Omega$). Use the parallel rule $V_L = V_M = 2.0\ \text{V}$ — both branches share the node voltage.
Sense-check (KVL & KCL): $V_K + V_L = 7.0 + 2.0 = 9.0\ \text{V} = \varepsilon$ ✓ (one loop); $I_L + I_M = 1.0 + 2.0 = 3.0\ \text{A} = I_K$ ✓.
$R_K = 2.33\ \Omega,\ R_L = 2.0\ \Omega,\ R_M = 1.0\ \Omega$
