Q — Drift-velocity ratio for two series wires

Question

Two copper wires of the same material are connected in series. Wire X has diameter $d$ and wire Y has diameter $2d$. The same current flows through both.

(a) State which wire has the larger drift velocity, with reasoning. [1 mark]
(b) Calculate the ratio $v_X : v_Y$. [3 marks]

[4 marks]

Part (a) — which is larger [1 marks] · 1 step

STEP 1/1 B1

Reasoning: in series the current $I$ is the same in both wires, and $n$, $q$ are the same (same material). From $v = \dfrac{I}{nqA}$, $v \propto \dfrac{1}{A}$.

Wire X is thinner (smaller $A$), so wire X has the larger drift velocity.

Trap: thinner does NOT mean less current — the current is identical; the thinner wire compensates with faster drift.
Wire X (smaller area) has the larger drift velocity

Part (b) — the ratio [3 marks] · 1 step

Carry-in from (a): same $I$, $n$, $q$, so $v \propto 1/A$. Diameters: X has $d$, Y has $2d$; areas use $A = \pi(d/2)^2$.
↳ from STEP 1: $v \propto 1/A$; $A_X = \pi(d/2)^2$, $A_Y = \pi(2d/2)^2 = \pi d^2$
STEP 1/1 M1 M1 A1

Areas: $A_X = \pi\left(\dfrac{d}{2}\right)^2 = \dfrac{\pi d^2}{4}$, $A_Y = \pi\left(\dfrac{2d}{2}\right)^2 = \pi d^2 = 4A_X$.

Apply $v \propto 1/A$:

$$\dfrac{v_X}{v_Y} = \dfrac{A_Y}{A_X} = \dfrac{4A_X}{A_X} = 4$$

$$\therefore\; v_X : v_Y = 4 : 1$$

Sense-check: doubling the diameter quadruples the area, so the drift velocity drops to a quarter — the $4:1$ ratio is correct.
$v_X : v_Y = 4 : 1$
