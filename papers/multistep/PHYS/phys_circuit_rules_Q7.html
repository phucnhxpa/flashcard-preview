Q — Mixed circuit-rules self-test (series, parallel, KVL, unknown branch)

Question

(a) Resistors $4.0\ \Omega$ and $12\ \Omega$ are in parallel. Find the combined resistance. [2 marks]
(b) The same $4.0\ \Omega$ and $12\ \Omega$ are in series across a $6.0\ \text{V}$ supply. Find the current and the p.d. across each resistor. [2 marks]
(c) In a combination circuit the p.d. across the parallel section is $5.0\ \text{V}$ and the total e.m.f. is $9.0\ \text{V}$. Find the p.d. across the series element. [1 mark]
(d) A $3.0\ \Omega$ branch and an unknown branch are in parallel with $4.8\ \text{V}$ across them; the total supply current is $2.4\ \text{A}$. Find the unknown branch's resistance. [2 marks]

[7 marks]

Part (a) — parallel combination [2 marks] · 1 step

STEP 1/1 M1 A1

Formula (resistors in parallel):

$$\dfrac{1}{R_T} = \dfrac{1}{R_1} + \dfrac{1}{R_2}$$

Substitute $R_1 = 4.0\ \Omega$, $R_2 = 12\ \Omega$:

$$\dfrac{1}{R_T} = \dfrac{1}{4.0} + \dfrac{1}{12} = \dfrac{3}{12} + \dfrac{1}{12} = \dfrac{4}{12} = \dfrac{1}{3.0}$$

$$R_T = 3.0\ \Omega$$

Check: $3.0\ \Omega < 4.0\ \Omega$ (smallest branch) ✓. Parallel resistance is always less than the smallest branch.
$R_T = 3.0\ \Omega$

Part (b) — series current and shared p.d. [2 marks] · 1 step

Carry-in: now the same two resistors are in series ($R_T = 4.0 + 12 = 16\ \Omega$) across $6.0\ \text{V}$. Series → same current.
↳ from STEP 1: series: $R_T = 4.0 + 12 = 16\ \Omega$ across $6.0\ \text{V}$
STEP 1/1 M1 A1

Formula (Ohm's law, then series p.d. share):

$$I = \dfrac{\varepsilon}{R_T}, \qquad V = IR\ \text{per resistor}$$

Substitute $\varepsilon = 6.0\ \text{V}$, $R_T = 16\ \Omega$:

$$I = \dfrac{6.0}{16} = 0.375\ \text{A}$$

$$V_1 = 0.375 \times 4.0 = 1.5\ \text{V}, \qquad V_2 = 0.375 \times 12 = 4.5\ \text{V}$$

Common trap (V same in series): the p.d. is not the same across each series resistor — $V = IR$, same $I$ but different $R$ gives different $V$. (Phuc reasoned 'I is same so V is same' at 13:49 — wrong.) Check: $1.5 + 4.5 = 6.0\ \text{V} = \varepsilon$ ✓.
$I = 0.375\ \text{A};\ V_1 = 1.5\ \text{V},\ V_2 = 4.5\ \text{V}$

Part (c) — p.d. across the series element (KVL) [1 marks] · 1 step

Carry-in: total e.m.f. $\varepsilon = 9.0\ \text{V}$; p.d. across the parallel section $V_{\text{parallel}} = 5.0\ \text{V}$.
↳ from STEP 1: $\varepsilon = 9.0\ \text{V}$, $V_{\text{parallel}} = 5.0\ \text{V}$
STEP 1/1 A1

Formula (Kirchhoff's second law / KVL on one loop):

$$\varepsilon = V_{\text{series}} + V_{\text{parallel}} \quad\Rightarrow\quad V_{\text{series}} = \varepsilon - V_{\text{parallel}}$$

Substitute $\varepsilon = 9.0\ \text{V}$, $V_{\text{parallel}} = 5.0\ \text{V}$:

$$V_{\text{series}} = 9.0 - 5.0 = 4.0\ \text{V}$$

Common trap (double-count): one loop is series element + parallel section, not both parallel branches added. The series drop plus the single parallel-section drop equals the e.m.f.
$V_{\text{series}} = 4.0\ \text{V}$

Part (d) — unknown parallel branch [2 marks] · 1 step

Carry-in: the two parallel branches share $V = 4.8\ \text{V}$; known branch $R_1 = 3.0\ \Omega$; total supply current $I_T = 2.4\ \text{A}$.
↳ from STEP 1: $V = 4.8\ \text{V}$ across both branches; $R_1 = 3.0\ \Omega$; $I_T = 2.4\ \text{A}$
STEP 1/1 M1 A1

Step 1 — known branch current ($I = V/R$):

$$I_1 = \dfrac{V}{R_1} = \dfrac{4.8}{3.0} = 1.6\ \text{A}$$

Step 2 — KCL for the unknown branch:

$$I_2 = I_T - I_1 = 2.4 - 1.6 = 0.8\ \text{A}$$

Step 3 — resistance of the unknown branch ($R = V/I$):

$$R_2 = \dfrac{V}{I_2} = \dfrac{4.8}{0.8} = 6.0\ \Omega$$

Method: both branches share $V = 4.8\ \text{V}$ (parallel rule); KCL splits the total current; then $R = V/I$ (never $R = IV$).
$R_2 = 6.0\ \Omega$
