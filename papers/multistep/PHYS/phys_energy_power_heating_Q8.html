Self-Test Q7 — Input power and efficiency of an appliance

Question

An appliance converts $500\ \text{J}$ of electrical energy into $350\ \text{J}$ of useful work in $10\ \text{s}$.

(a) Calculate the input power. [2 marks]
(b) Calculate the efficiency. [2 marks]

[4 marks]

Part (a) — input power [2 marks] · 1 step

STEP 1/1 M1 A1

Formula (power = energy per unit time):

$$P = \dfrac{W}{t}$$

Substitute the input energy $W = 500\ \text{J}$, $t = 10\ \text{s}$:

$$P_{in} = \dfrac{500}{10} = 50\ \text{W}$$

Note (algebra-only; no diagram needed): the input power uses the total electrical energy supplied ($500\ \text{J}$), not the useful output. Always give the unit (W).
$P_{in} = 50\ \text{W}$

Part (b) — efficiency [2 marks] · 1 step

Carry-in from (a): input power $P_{in} = 50\ \text{W}$ (from $500\ \text{J}$ in $10\ \text{s}$); useful output energy $= 350\ \text{J}$ in the same $10\ \text{s}$.
↳ from STEP 1: $P_{in} = 50\ \text{W}$; useful output $= 350\ \text{J}$ in $10\ \text{s}$
STEP 1/1 M1 A1

Useful output power:

$$P_{useful} = \dfrac{350}{10} = 35\ \text{W}$$

Formula (efficiency):

$$\eta = \dfrac{P_{useful}}{P_{total}} \times 100\%$$

Substitute $P_{useful} = 35\ \text{W}$, $P_{total} = 50\ \text{W}$:

$$\eta = \dfrac{35}{50} \times 100\% = 70\%$$

Shortcut: efficiency can also use energies directly: $\eta = 350/500 \times 100\% = 70\%$ — the $\div 10\ \text{s}$ cancels. Either route is valid; always express as a percentage and label which quantity is useful and which is total.
$\eta = 70\%$
