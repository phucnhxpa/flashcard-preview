Length of each spring in a series and a parallel arrangement

Question

Two identical springs each have spring constant $240\,\mathrm{N\,m^{-1}}$ and natural length 210 mm. Calculate the length of each spring when (a) the two are joined vertically in series and stretched by an 8.0 N load, and (b) the two are joined in parallel and stretched by an 8.0 N load.

[6 marks]

Part (a) — each spring in series [3 marks] · 3 steps

STEP 1/3force per spring

Key idea. In a vertical series chain the same force passes through every spring. To find the length of each spring, use each spring's own $k = 240$, not the effective $k$ of the chain.

Force in each series spring $= 8.0\,\mathrm{N}$.
↳ from STEP 2: $F = 8.0\,\mathrm{N},\; k = 240\,\mathrm{N\,m^{-1}}$
STEP 2/3extension

Extension of each spring.

$$x = \frac{F}{k} = \frac{8.0}{240} = 0.0333\,\mathrm{m}$$
$x = 0.0333\,\mathrm{m}$
↳ from STEP 3: $x = 0.0333\,\mathrm{m},\; L_0 = 0.210\,\mathrm{m}$
STEP 3/3length

Length of each spring.

$$L = L_0 + x = 0.210 + 0.0333$$
$L = 0.243\,\mathrm{m}$ (each spring)

Part (b) — each spring in parallel [3 marks] · 3 steps

Carry-in: $k = 240\,\mathrm{N\,m^{-1}}$, $L_0 = 210\,\mathrm{mm} = 0.210\,\mathrm{m}$, total load $8.0\,\mathrm{N}$.
STEP 1/3force per spring

Key idea. In parallel the load is shared equally between the two springs, so each carries half.

$$F_{\text{each}} = \frac{8.0}{2} = 4.0\,\mathrm{N}$$
Force in each parallel spring $= 4.0\,\mathrm{N}$.
↳ from STEP 2: $F_{\text{each}} = 4.0\,\mathrm{N},\; k = 240$
STEP 2/3extension

Extension of each spring.

$$x = \frac{4.0}{240} = 0.0167\,\mathrm{m}$$
$x = 0.0167\,\mathrm{m}$
↳ from STEP 3: $x = 0.0167\,\mathrm{m},\; L_0 = 0.210\,\mathrm{m}$
STEP 3/3length

Length of each spring.

$$L = 0.210 + 0.0167$$
Revision-guide trap: do not use the effective stiffness and add the whole extension to one spring — that gives the chain length, not the length of each spring.
$L = 0.227\,\mathrm{m}$ (each spring)
