Derive — $R_T = R_1 + R_2 + \cdots$ for resistors in series

Question

Derive the formula for the total resistance of resistors in series, $R_T = R_1 + R_2 + \cdots$, starting from a conservation principle. (Edexcel IAL spec point 68 — 'derive and use'.)

(a) State the correct starting principle and write the first equation. [2 marks]
(b) Complete the derivation to reach $R_T = R_1 + R_2$. [2 marks]

[4 marks]

Part (a) — starting principle and first equation [2 marks] · 1 step

STEP 1/1 B1 M1

Starting principle: in a series circuit, conservation of energy means the voltages add. Always start a derivation from the quantity that adds up in that circuit type.

Formula (Kirchhoff's second law / KVL):

$$V_{\text{total}} = V_1 + V_2$$

Phuc's mistake (lesson 43:40): for the parallel proof he started by writing $R_1 + R_2$ (the series move) and got stuck in a complex fraction. Rule (Alex, 52:53): start from the quantity that adds — series → voltage adds ($V_T = V_1 + V_2$); parallel → current adds.
Start: conservation of energy → $V_T = V_1 + V_2$

Part (b) — complete the derivation [2 marks] · 1 step

Carry-in from (a): $V_T = V_1 + V_2$ (KVL), and in series the current is the same everywhere: $I_T = I_1 = I_2 = I$.
↳ from STEP 1: $V_T = V_1 + V_2$; series current rule $I_1 = I_2 = I$
STEP 1/1 M1 A1

Substitute Ohm's law $V = IR$ for each component into $V_T = V_1 + V_2$:

$$I_T R_T = I_1 R_1 + I_2 R_2$$

Apply the series current rule $I_T = I_1 = I_2 = I$:

$$I R_T = I R_1 + I R_2$$

Cancel $I$:

$$R_T = R_1 + R_2$$

Mark-scheme route: must show $V_T = V_1 + V_2$, then $IR_T = IR_1 + IR_2$, then cancel $I$. Total resistance always increases when a resistor is added in series.
$R_T = R_1 + R_2 + \cdots$
