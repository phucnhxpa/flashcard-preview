Tutor Q — Resistance of a non-ohmic component at a point

Question

A non-ohmic component has the curved I–V characteristic shown. Using the graph, find the resistance of the component when the current through it is $I = 40\ \text{mA}$.

(a) Read the corresponding p.d. from the graph and state the correct defining relation for resistance. [2 marks]
(b) Hence calculate the resistance at this point. [2 marks]

[4 marks]

Part (a) — read the point & state R = V/I [2 marks] · 1 step

STEP 1/1 B1 B1

Read the operating point from the graph at $I = 40\ \text{mA}$:

$$V = 0.70\ \text{V}, \qquad I = 40\ \text{mA}$$

Defining relation (true for ANY component, ohmic or not):

$$R = \dfrac{V}{I}$$

Common trap (gradient): the graph is curved (non-ohmic), so you must NOT use the gradient of the curve to get $R$. Read $V$ and $I$ at the actual point and divide.
$V = 0.70\ \text{V}$ at $I = 40\ \text{mA}$; use $R = V/I$

Part (b) — compute R [2 marks] · 1 step

Carry-in from (a): at the operating point $V = 0.70\ \text{V}$ and $I = 40\ \text{mA} = 40\times10^{-3}\ \text{A}$; use $R = V/I$.
↳ from STEP 1: $V = 0.70\ \text{V}$, $I = 40\ \text{mA} = 0.040\ \text{A}$
STEP 1/1 M1 A1

Convert the current to base SI units first:

$$I = 40\ \text{mA} = 40\times10^{-3}\ \text{A} = 0.040\ \text{A}$$

Substitute into $R = V/I$:

$$R = \dfrac{0.70}{0.040} = 17.5\ \Omega$$

Phuc's slip (lesson video 1, 44:01): he wrote $R = I\times V = 0.040\times0.70 = 0.028\ \Omega$ — the wrong formula. Marta corrected it: resistance is the quotient $V\div I$, never the product $IV$.
Unit check: $\text{V}/\text{A} = \Omega$ ✓
$R = 17.5\ \Omega$
