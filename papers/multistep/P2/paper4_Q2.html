Q2 — $f(x) = 4x^3 - 8x^2 + 5x + a$, $(2x-3)$ factor: show $a=-3$; only one real root

Question

$f(x) = 4x^3 - 8x^2 + 5x + a$, where $a$ is a constant. Given that $(2x - 3)$ is a factor of $f(x)$:

(a) Show that $a = -3$ using the factor theorem. [2]

(b) Hence show that $f(x) = 0$ has only one real root. [4]

[6 marks]

Part (a) — show $a = -3$ [2 marks] · 1 step

STEP 1/1 M1 A1

Strategy: the Factor Theorem says: linear factor ⇔ corresponding root is a zero. Here the factor is $(2x - 3)$, which equals zero at $x = \tfrac{3}{2}$. So $f(\tfrac{3}{2}) = 0$ — substitute, simplify, and read off $a$.

Formula (Factor Theorem):

$$(bx - c) \text{ is a factor of } f(x) \;\Longleftrightarrow\; f\!\left(\tfrac{c}{b}\right) = 0$$

Here $b = 2$, $c = 3$, so the test point is $x = \tfrac{3}{2}$.

Substitute $x = \tfrac{3}{2}$ into $f(x) = 4x^3 - 8x^2 + 5x + a$, term by term:

$\;\;4x^3 = 4 \cdot \left(\tfrac{3}{2}\right)^3 = 4 \cdot \tfrac{27}{8} = \tfrac{27}{2}$

$\;\;-8x^2 = -8 \cdot \left(\tfrac{3}{2}\right)^2 = -8 \cdot \tfrac{9}{4} = -18$

$\;\;5x = 5 \cdot \tfrac{3}{2} = \tfrac{15}{2}$

$\;\;+a$ stays as $a$.

Sum: $f(\tfrac{3}{2}) = \tfrac{27}{2} - 18 + \tfrac{15}{2} + a = \tfrac{27 + 15}{2} - 18 + a = 21 - 18 + a = 3 + a$.

Set equal to zero (Factor Theorem):

$$3 + a = 0 \;\Longrightarrow\; \boxed{\;a = -3\;}\quad \blacksquare$$

Trap: using $x = \tfrac{2}{3}$ (swapping) instead of $\tfrac{3}{2}$. The root comes from $bx - c = 0 \Rightarrow x = c/b$, so for $(2x - 3)$ it's $x = 3/2$. Sense check: $\tfrac{27}{2} + \tfrac{15}{2} = \tfrac{42}{2} = 21$ — clean integer arithmetic, a good sign no slip occurred.
$\boxed{a = -3}$

Part (b) — only one real root [4 marks] · 2 steps

From (a): $a = -3$, so $f(x) = 4x^3 - 8x^2 + 5x - 3$ has factor $(2x-3)$. Polynomial division gives $f(x) = (2x-3)(2x^2 - x + 1)$. To prove only ONE real root, show the quadratic factor has no real roots (discriminant $< 0$).
Carry-in from previous stepFrom STEP 1: $a = -3$
STEP 1/2 M1 A1

Strategy: $f(x)$ is a cubic with $(2x-3)$ as a known factor (from part (a)). Dividing the cubic by the linear factor gives a quadratic quotient. The roots of $f$ are then the $x = \tfrac{3}{2}$ from the linear factor PLUS any real roots of the quadratic. We can decide "only one real root" by checking the quadratic's discriminant in the next step.

Degree check: dividend (cubic, deg $3$) ÷ divisor (linear, deg $1$) ⇒ quotient is deg $2$ (quadratic). So we expect $f(x) = (2x - 3)(Ax^2 + Bx + C)$.

Long division of $4x^3 - 8x^2 + 5x - 3$ by $(2x - 3)$:

$\;\;$ Leading term: $\dfrac{4x^3}{2x} = 2x^2$ — multiply back: $2x^2(2x - 3) = 4x^3 - 6x^2$. Subtract: leaves $-2x^2 + 5x - 3$.

$\;\;$ Next: $\dfrac{-2x^2}{2x} = -x$ — multiply back: $-x(2x - 3) = -2x^2 + 3x$. Subtract: leaves $2x - 3$.

$\;\;$ Last: $\dfrac{2x}{2x} = +1$ — multiply back: $1 \cdot (2x - 3) = 2x - 3$. Subtract: leaves $0$ ✓.

$$\boxed{\;f(x) = (2x - 3)(2x^2 - x + 1)\;}$$

Verify by re-expansion: $(2x-3)(2x^2-x+1) = 4x^3 - 2x^2 + 2x - 6x^2 + 3x - 3 = 4x^3 - 8x^2 + 5x - 3$ ✓.

Phuc's instinct: "We factor it out?" Mr Fuat: "No, you need polynomial division." For "only one real root" questions, divide out the known factor, then test the discriminant of the quadratic factor. Polynomial division check: degree of dividend = degree of divisor + degree of quotient ($3 = 1 + 2$ ✓). Remainder must be zero — if not, $(2x-3)$ wasn't a factor.
STEP 2/2 M1 A1

Strategy: for any quadratic $ax^2 + bx + c$, the discriminant $\Delta$ tells us how many real roots it has: $\Delta > 0$ ⇒ two distinct real roots, $\Delta = 0$ ⇒ one repeated root, $\Delta < 0$ ⇒ no real roots (complex conjugate pair). To prove $f(x) = 0$ has "only one real root", show the quadratic factor has $\Delta < 0$.

Formula (discriminant — IN booklet):

$$\Delta = b^2 - 4ac$$

Identify coefficients in $2x^2 - x + 1$:

$\;\;a = 2,\quad b = -1,\quad c = 1$

Substitute:

$$\Delta = (-1)^2 - 4(2)(1)$$

$$= 1 - 8 = -7$$

$\Delta = -7 < 0$ ⇒ $2x^2 - x + 1 = 0$ has no real roots.

Conclusion: the only real solution of $f(x) = 0$ comes from the linear factor:

$$2x - 3 = 0 \;\Longrightarrow\; x = \tfrac{3}{2}$$

$$\boxed{\;f(x) = 0 \text{ has exactly one real root, } x = \tfrac{3}{2}\;}\quad \blacksquare$$

Sign of $b$ trap: $b = -1$ in $2x^2 - x + 1$ (NOT $b = 1$). Then $b^2 = (-1)^2 = +1$ — squaring kills the sign anyway, but get the substitution right. Why $\Delta$ rules: $x = \tfrac{-b \pm \sqrt{\Delta}}{2a}$ — if $\Delta < 0$, $\sqrt{\Delta}$ is imaginary, so no real roots.
$f(x) = 0$ has exactly one real root, $x = \tfrac{3}{2}$ ∎
