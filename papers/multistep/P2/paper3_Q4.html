Q4 — $f(x) = 4x^3 + ax^2 - 29x + b$: $(2x+1)$ factor; $f(2) = -25$

Question

$f(x) = 4x^3 + ax^2 - 29x + b$, where $a$ and $b$ are constants.

(a) Given $(2x + 1)$ is a factor of $f(x)$, show that $a + 4b = -56$. [2]

(b) Given also that the remainder when $f(x) \div (x - 2)$ is $-25$, find a second equation linking $a$ and $b$. [2]

(c)(i) Hence find the values of $a$ and $b$. (ii) Factorise $f(x)$ completely. [5]

[9 marks]

Part (a) — show $a + 4b = -56$ [2 marks] · 1 step

STEP 1/1 M1 A1

Strategy: the Factor Theorem says if $(2x + 1)$ is a factor of $f(x)$, then $f$ vanishes at the root $x = -\tfrac{1}{2}$. Substitute, simplify, and clear fractions to get a linear equation in $a$ and $b$.

Formula (Factor Theorem): $(ax + b) \text{ is a factor of } f(x) \Leftrightarrow f\!\left(-\dfrac{b}{a}\right) = 0$.

Set $f\!\left(-\tfrac{1}{2}\right) = 0$:

$\;\;4\!\left(-\tfrac{1}{2}\right)^3 + a\!\left(-\tfrac{1}{2}\right)^2 - 29\!\left(-\tfrac{1}{2}\right) + b = 0$

Compute powers:

$\;\;4\!\left(-\tfrac{1}{8}\right) + a\!\left(\tfrac{1}{4}\right) - 29\!\left(-\tfrac{1}{2}\right) + b = 0$

Simplify each term:

$\;\;-\tfrac{1}{2} + \tfrac{a}{4} + \tfrac{29}{2} + b = 0$

Combine the constants $-\tfrac{1}{2} + \tfrac{29}{2} = \tfrac{28}{2} = 14$:

$\;\;\tfrac{a}{4} + b + 14 = 0$

Multiply through by $4$ to clear the fraction:

$\;\;a + 4b + 56 = 0$

$\;\;a + 4b = -56$

$$\boxed{\;a + 4b = -56\;}\quad\blacksquare$$

Sign trap on $-29x$ term: at $x = -\tfrac{1}{2}$, the $-29x$ becomes $-29 \cdot (-\tfrac{1}{2}) = +\tfrac{29}{2}$. Forgetting the double negative gives the wrong final equation.

Part (b) — second equation [2 marks] · 1 step

From (a): first equation is $a + 4b = -56$. Now use the Remainder Theorem: remainder of $f(x) \div (x - 2)$ is $f(2)$. Set $f(2) = -25$ and simplify to get a second linear equation in $a$ and $b$.
STEP 1/1 M1 A1

Strategy: the Remainder Theorem states that $f(x) \div (x - c)$ leaves remainder $f(c)$. Here $c = 2$ and the remainder is given as $-25$, so set $f(2) = -25$ to get a second linear equation.

Formula (Remainder Theorem): remainder of $f(x) \div (x - c)$ is $f(c)$.

Compute $f(2)$:

$\;\;f(2) = 4(2)^3 + a(2)^2 - 29(2) + b$

$\;\;= 4(8) + a(4) - 58 + b$

$\;\;= 32 + 4a - 58 + b$

$\;\;= 4a + b - 26$

Set equal to $-25$:

$\;\;4a + b - 26 = -25$

Add $26$ to both sides:

$\;\;4a + b = 1$

$$\boxed{\;4a + b = 1\;}$$

Factor vs Remainder: the Factor Theorem is just the Remainder Theorem with the remainder set to $0$. Both work the same way \u2014 substitute the root and read off the value.
$4a + b = 1$

Part (c)(i) — solve simultaneously [2 marks] · 1 step

System: $a + 4b = -56$  (1),   $4a + b = 1$  (2).
STEP 1/1 M1

Strategy: two equations, two unknowns. Solve eq (2) for $b$ in terms of $a$, substitute into eq (1), and solve the resulting linear equation.

System:

$\;\;(1):\ a + 4b = -56$

$\;\;(2):\ 4a + b = 1$

From (2): $b = 1 - 4a$.

Substitute into (1):

$\;\;a + 4(1 - 4a) = -56$

Expand:

$\;\;a + 4 - 16a = -56$

$\;\;-15a + 4 = -56$

Subtract $4$:

$\;\;-15a = -60$

Divide by $-15$:

$\;\;a = 4$

Back-substitute into $b = 1 - 4a$:

$\;\;b = 1 - 16 = -15$

$$\boxed{\;a = 4,\ b = -15\;}$$

Verify: check eq (1): $4 + 4(-15) = 4 - 60 = -56$ ✓. Check eq (2): $4(4) + (-15) = 16 - 15 = 1$ ✓. Both equations satisfied.
$\boxed{a = 4,\ b = -15}$

Part (c)(ii) — full factorisation [3 marks] · 2 steps

From (c)(i): $a = 4$, $b = -15$, so $f(x) = 4x^3 + 4x^2 - 29x - 15$. From (a) we know $(2x + 1)$ is a factor — divide it out to get a quadratic, then factor that quadratic to obtain three linear factors.
STEP 1/2 M1

Strategy: we know $(2x + 1)$ is one factor. Divide $f(x)$ by it to get the remaining quadratic factor. The cubic becomes a product of $(2x + 1)$ and a quadratic.

Formula (factor extraction): if $(2x + 1)$ is a factor, then $f(x) = (2x + 1) \cdot Q(x)$ for some quadratic $Q(x)$.

With $a = 4$ and $b = -15$, we have $f(x) = 4x^3 + 4x^2 - 29x - 15$.

Divide by $(2x + 1)$ (long division or compare coefficients):

$\;\;4x^3 + 4x^2 - 29x - 15 = (2x + 1)(2x^2 + x - 15)$

Verify by expansion:

$\;\;(2x + 1)(2x^2 + x - 15) = 4x^3 + 2x^2 - 30x + 2x^2 + x - 15$

$\;\;= 4x^3 + 4x^2 - 29x - 15$ ✓

$$\boxed{\;f(x) = (2x + 1)(2x^2 + x - 15)\;}$$

Verification habit: always expand the product after polynomial division \u2014 a single sign slip in long division can ripple through and ruin the next factorisation. The expansion check is cheap insurance.
Carry-in from previous stepFrom STEP 1: $\boxed{\;f(x) = (2x + 1)(2x^2 + x - 15)\;}$
STEP 2/2 A1 A1

Strategy: factorise $2x^2 + x - 15$ using the AC method: find two numbers that multiply to $a \cdot c = -30$ and add to $b = 1$. Split the middle term, then factor by grouping.

Formula (AC method): for $ax^2 + bx + c$, find $p, q$ with $pq = ac$ and $p + q = b$.

Here $a = 2$, $b = 1$, $c = -15$, so $ac = -30$.

Find two numbers with product $-30$ and sum $1$: $+6$ and $-5$ (since $6 \cdot (-5) = -30$ and $6 + (-5) = 1$).

Split the middle term:

$\;\;2x^2 + x - 15 = 2x^2 + 6x - 5x - 15$

Group and factor:

$\;\;= 2x(x + 3) - 5(x + 3)$

$\;\;= (2x - 5)(x + 3)$

Combine with $(2x + 1)$ from the previous step:

$$\boxed{\;f(x) = (2x + 1)(2x - 5)(x + 3)\;}$$

AC-method check: the two numbers must multiply to $ac$ (NOT just $c$) when $a \ne 1$. Forgetting this gives wrong splits. Always double-check by expanding the final factorisation back to the original quadratic.
