Q6(a) — Show $2\log_4(x+3) + \log_4 x = \log_4(4x+2) + \tfrac{1}{2}$ ⇒ $x^3 + 6x^2 + x - 4 = 0$

Question

Solve, for $x$, the equation

$$2\log_4(x + 3) + \log_4 x = \log_4(4x + 2) + \tfrac{1}{2}$$

(a) Show that this leads to $x^3 + 6x^2 + x - 4 = 0$. [4]

[4 marks]

Solution [4 marks] · 2 steps

STEP 1/2 M1

Strategy: the equation has a coefficient log ($2\log_4(\cdot)$), a sum of logs (LHS), and a bare constant $\tfrac{1}{2}$ (RHS). Convert ALL pieces into a single $\log_4$ on each side, then drop the logs.

Formulas (log laws): $n\log_a x = \log_a(x^n)$;   $\log_a x + \log_a y = \log_a(xy)$;   $k = \log_a(a^k)$.

LHS: bring coefficient $2$ into the power, then combine via product law:

$$2\log_4(x+3) + \log_4 x = \log_4(x+3)^2 + \log_4 x = \log_4\!\big[x(x+3)^2\big]$$

RHS: first convert $\tfrac{1}{2}$ to a $\log_4$:

$$\tfrac{1}{2} = \log_4\!\big(4^{1/2}\big) = \log_4 2$$

Then combine via product law:

$$\log_4(4x+2) + \log_4 2 = \log_4\!\big[2(4x+2)\big] = \log_4(8x+4)$$

$$\boxed{\;\log_4\!\big[x(x+3)^2\big] = \log_4(8x+4)\;}$$

Tutor (loom): "The number in front of logarithm must be one, otherwise we cannot use it." Always bring the coefficient into the power BEFORE combining or dropping logs. Trap: $\log(a + b) \ne \log a + \log b$ — only $\log(ab) = \log a + \log b$.
Carry-in from previous stepFrom STEP 1: $\boxed{\;\log_4\!\big[x(x+3)^2\big] = \log_4(8x+4)\;}$
STEP 2/2 M1 M1 A1

Strategy: $\log_4$ is one-to-one. Equal logs ⇒ equal arguments. Drop the logs, expand, collect into the required form.

Drop logs (both sides have $\log_4$):

$$x(x+3)^2 = 8x + 4$$

Expand $(x+3)^2$ using $(a+b)^2 = a^2 + 2ab + b^2$:

$$(x+3)^2 = x^2 + 6x + 9$$

Multiply by $x$:

$$x(x^2 + 6x + 9) = x^3 + 6x^2 + 9x$$

Set up the equation:

$$x^3 + 6x^2 + 9x = 8x + 4$$

Move everything to LHS (subtract $8x$, subtract $4$):

$$x^3 + 6x^2 + (9x - 8x) - 4 = 0$$

$$\boxed{\;x^3 + 6x^2 + x - 4 = 0\;} \quad \blacksquare$$

"Show that" tactic: arrange your final line to MATCH the printed form exactly — same coefficients, same order. Examiners look for the literal target equation.
$x^3 + 6x^2 + x - 4 = 0$ ∎
