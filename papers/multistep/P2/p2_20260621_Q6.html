Q6 · Algebraic manipulation — compare coefficients, verify by substitution

Question

Find constants $b$ and $c$ such that $$\frac{x^{3}+2x^{2}-5x-6}{x+1} = x + b + \frac{c}{x+1}.$$

Harry's tip: if the algebra looks messy, verify rather than abandon.

[4 marks]

Solution [4 marks] · 3 steps

Find constants $b, c$ such that $\dfrac{x^{3}+2x^{2}-5x-6}{x+1} = x + b + \dfrac{c}{x+1}$.
STEP 1/3expand the RHS fully M1

Strategy. To compare coefficients, expand $(x+1)(x^{2}+bx+c)$ so both sides are polynomials in $x$.

Expand

$$\frac{x^{3}+2x^{2}-5x-6}{x+1} = x + b + \frac{c}{x+1} \;\Rightarrow\; x^{3}+2x^{2}-5x-6 = (x+1)(x+bx+c) + c\cdot\frac{x+1}{1}\;\text{(mult. both sides by } x+1).$$
Phuc's habit (Harry #6). Stopped at the first sign of messiness. Keep going: multiply both sides by $(x+1)$ to clear the denominator, then compare coefficients.
↳ from STEP 1: $x^{3}+2x^{2}-5x-6 = (x+1)(x+bx+c)+c$
STEP 2/3equate coefficients of x^k M1

Strategy. Expand the RHS and match coefficients of $x^{3}, x^{2}, x^{1}, x^{0}$.

Expand RHS

$(x+1)(x+bx+c) = x^{2} + bx^{2} + cx + x + bx + c = (1+b)x^{2} + (b+c+1)x + c$.

Equate

$x^{3}$: $1 = 1$ ✓
$x^{2}$: $2 = 1+b \Rightarrow b=1$
$x^{1}$: $-5 = b+c+1 = 1+c+1 \Rightarrow c = -7$
$x^{0}$: $-6 = c = -7$ — contradiction!

Phuc's mistake. Abandoned the working at the contradiction. The contradiction is REAL: the equation is inconsistent — verify by substituting $x=0$ into both sides.
↳ from STEP 2: $b=1,\ c=-7$ but $x^{0}$ mismatch
STEP 3/3verify by substitution A1

Strategy. Sub $x=0$ into the original equation: LHS $= -6$, RHS $= 0 + b + c = 1 - 7 = -6$ — but only if $c=-7$, contradicting the $x^0$ match. So the proposed identity is not valid.

Substitution check

$x=0$: LHS $=-6$, RHS $= b + c = 1 + (-7) = -6$ ✓ (constant only)
$x=1$: LHS $= 1+2-5-6 = -8$, RHS $= 1 + b + c/2 = 1 + 1 - 3.5 = -1.5 \ne -8$.

The identity fails: there are no constants $b, c$ satisfying the equation. Verify-by-substitution caught it — don't abandon at complexity.
