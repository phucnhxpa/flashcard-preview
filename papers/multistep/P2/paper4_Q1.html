Q1 — Trapezium rule for $\int_4^5 f(x)\,dx$ from 6 ordinates (3 d.p.)

Question

The table below shows values of $y = f(x)$:

$x$:     4.0   4.2   4.4   4.6   4.8   5.0
$y$:     9.2   8.4556   3.8512   5.0342   7.8297   8.6

Use the trapezium rule with all six values to estimate $\int_4^5 f(x)\,dx$, giving the answer to 3 d.p. [3]

[3 marks]

Solution [3 marks] · 1 step

STEP 1/1 B1 M1 A1

Strategy: the trapezium rule estimates a definite integral by replacing the curve with straight-line segments between consecutive ordinates. With $n$ strips you have $n+1$ ordinates — here $6$ ordinates ⇒ $5$ strips. Steps: find the strip width $h$, identify endpoints vs interiors, plug into the booklet formula.

Step A — strip width $h$. Six equally-spaced ordinates from $4.0$ to $5.0$ ⇒ $5$ strips:

$$h = \dfrac{b - a}{n} = \dfrac{5 - 4}{5} = 0.2$$

Step B — endpoints. $y_0 = f(4.0) = 9.2$, $y_5 = f(5.0) = 8.6$.

Step C — interior sum. $y_1, y_2, y_3, y_4$ contribute (each weighted $\times 2$):

$$\Sigma_{\text{int}} = 8.4556 + 3.8512 + 5.0342 + 7.8297 = 25.1707$$

Formula (trapezium rule — IN booklet):

$$\int_a^b y\,dx \approx \tfrac{h}{2}\big[y_0 + y_n + 2(y_1 + \ldots + y_{n-1})\big]$$

Substitute $h = 0.2$, endpoints $9.2 + 8.6 = 17.8$, interior sum $25.1707$:

$$\approx \tfrac{0.2}{2}\big[17.8 + 2(25.1707)\big] = 0.1 \times \big[17.8 + 50.3414\big]$$

$$= 0.1 \times 68.1414 = 6.81414$$

$$\boxed{\;\int_4^5 f(x)\,dx \approx 6.814\text{ (3 d.p.)}\;}$$
