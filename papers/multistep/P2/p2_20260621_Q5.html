Question

Solve $\cos 2\theta = 0.5$ for $0 \le \theta < 360^{\circ}$. List ALL solutions.

[4 marks]

Solution [4 marks] · 4 steps

Solve $\cos 2\theta = 0.5$ for $0 \le \theta < 360^{\circ}$. List ALL solutions.
STEP 1/4halve the angle first M1

Strategy. $\cos 2\theta = 0.5$. Set $\alpha = 2\theta$ so $\cos\alpha = 0.5$; solve for $\alpha \in [0, 720)$, then halve to recover $\theta$.

Substitute

$$\text{Let } \alpha = 2\theta. \quad \cos\alpha = 0.5, \quad \alpha \in [0, 720^{\circ}).$$
Phuc's mistake. Solved $\cos\theta = 0.5$ directly and forgot that $2\theta$ spans $[0, 720)$, not $[0, 360)$. The doubled range gives an extra pair of solutions.
↳ from STEP 1: $\cos\alpha=0.5$, $\alpha\in[0,720)$
STEP 2/4principal + second solution M1

Strategy. For $\cos\alpha = c$, the two solutions in $[0, 360)$ are $\alpha_1 = \arccos c$ and $\alpha_2 = 360^{\circ} - \alpha_1$ (cosine symmetry: $\cos\theta = \cos(-\theta) = \cos(360-\theta)$).

First-period solutions

$$\alpha_1 = \arccos(0.5) = 60^{\circ}, \qquad \alpha_2 = 360^{\circ} - 60^{\circ} = 300^{\circ}.$$
↳ from STEP 2: $\alpha_1=60^{\circ},\ \alpha_2=300^{\circ}$
STEP 3/4second period (add 360) A1

Strategy. Since $\alpha$ runs up to $720^{\circ}$, add $360^{\circ}$ to both first-period solutions to get the second-period pair.

All $\alpha$ solutions

$$\alpha \in \{60^{\circ},\ 300^{\circ},\ 420^{\circ},\ 660^{\circ}\}.$$
↳ from STEP 3: $\alpha\in\{60,300,420,660\}^{\circ}$
STEP 4/4halve and check range A1

Strategy. Halve each $\alpha$ to recover $\theta = \alpha/2$. Check every value lies in $[0, 360)$.

Halve

$$\theta \in \{30^{\circ},\ 150^{\circ},\ 210^{\circ},\ 330^{\circ}\}.$$
Phuc's mistake. Stopped after $\{30^{\circ}, 150^{\circ}\}$, missing $210^{\circ}$ and $330^{\circ}$. The second period of $\alpha$ produces the second period of $\theta$.

Sense check. All four values lie in $[0, 360)$ ✓ Back-substitute: $\cos(2\cdot 210^{\circ})=\cos 420^{\circ}=\cos 60^{\circ}=0.5$ ✓

Worked diagram

$y=\cos 2\theta$ meets $y=0.5$ four times on $[0^\circ,360^\circ)$: $\theta=30^\circ,150^\circ,210^\circ,330^\circ$. Period is $180^\circ$, so don't stop at the first two.
