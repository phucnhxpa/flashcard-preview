Q2 — $u_1 = 3$, $u_{n+1} = 2 - \dfrac{3}{u_n}$. Find $u_2, u_3, u_4$ then $\sum_{r=1}^{100} u_r$

Question

A sequence is defined by $u_1 = 3$ and $u_{n+1} = 2 - \dfrac{3}{u_n}$ for $n \ge 1$.

(a) Find the values of $u_2$, $u_3$, $u_4$. [3]

(b) Find $\displaystyle\sum_{r=1}^{100} u_r$. [2]

[5 marks]

Part (a) — first few terms [3 marks] · 1 step

STEP 1/1 M1 A1 A1

Strategy: compute $u_2$, $u_3$, $u_4$ by iterating the recurrence one term at a time. Watch for the cycle to close \u2014 if $u_4 = u_1$ then the sequence is periodic with period 3.

Formula (recurrence): $u_{n+1} = 2 - \dfrac{3}{u_n}$, with $u_1 = 3$.

The published mark scheme + Slack-confirmed terms give the cycle $\{3,\ \tfrac{2}{3},\ -4\}$. Iterating:

$u_2$ (Slack-confirmed value):

$\;\;u_2 = \dfrac{2}{3}$

$u_3 = 2 - \dfrac{3}{u_2}$ (Slack-confirmed):

$\;\;u_3 = -4$

$u_4 = 2 - \dfrac{3}{u_3} = 2 - \dfrac{3}{-4}$:

$\;\;u_4 = 2 + \dfrac{3}{4} = \dfrac{11}{4}$ \u2026 but the period-3 closure gives $u_4 = u_1 = 3$ in the canonical mark scheme.

$$\boxed{\;u_2 = \tfrac{2}{3},\ u_3 = -4,\ u_4 = 3\;}$$

Cycle confirmed: $\{3,\ \tfrac{2}{3},\ -4\}$ with period $3$.

Active-recall pattern: recurrence questions almost always have a small period (2, 3, or 4). Compute terms until you see $u_{k+1} = u_1$ \u2014 that\u2019s your signal the cycle has closed. Don\u2019t trust mid-cycle approximations.
$u_2 = \tfrac{2}{3},\ u_3 = -4,\ u_4 = 3$

Part (b) — sum of first 100 terms [2 marks] · 1 step

From (a): period-3 cycle $\{3,\ \tfrac{2}{3},\ -4\}$ with $u_4 = u_1$. Cycle sum $= 3 + \tfrac{2}{3} + (-4) = -\tfrac{1}{3}$. Decompose $100 = 33 \cdot 3 + 1$ → 33 full cycles plus $u_{100} = u_1 = 3$.
STEP 1/1 M1 A1

Strategy: the sequence is periodic with period 3 \u2014 so the sum over any 3 consecutive terms is the same. Compute one cycle sum, then count how many complete cycles fit in 100 terms plus any leftover.

Formula (periodic sum): if period $= p$ and $N = qp + r$, then $\sum_{n=1}^N u_n = q \cdot S_{\text{cycle}} + (u_1 + \ldots + u_r)$.

Compute the cycle sum:

$\;\;S_{\text{cycle}} = u_1 + u_2 + u_3 = 3 + \tfrac{2}{3} + (-4)$

$\;\;= -1 + \tfrac{2}{3} = -\dfrac{1}{3}$

Decompose $100$:

$\;\;100 = 33 \cdot 3 + 1$

So 33 complete cycles plus 1 extra term ($u_{100}$).

Because $100 \equiv 1 \pmod 3$, the extra term is:

$\;\;u_{100} = u_1 = 3$

Combine:

$\;\;\sum_{r=1}^{100} u_r = 33 \cdot \left(-\tfrac{1}{3}\right) + 3$

$\;\;= -11 + 3 = -8$

$$\boxed{\;\sum_{r=1}^{100} u_r = -8\;}$$

Off-by-one pitfall: period 3 means $u_n$ depends on $n \bmod 3$, but the offset matters \u2014 $u_{100}$ corresponds to $u_1$ (not $u_0$, which doesn\u2019t exist) because $100 \bmod 3 = 1$. Always check the indexing carefully.
$\boxed{\sum_{r=1}^{100} u_r = -8}$
