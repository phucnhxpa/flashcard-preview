Q11 — $u_{n+1} = b - au_n$, $u_1=3$. With $\Sigma_{n=1}^3 u_n = 153$ and $b = a+9$, show $a^2-5a-66=0$; find larger $u_2$

Question

$u_{n+1} = b - au_n$, $u_1 = 3$.

(a) Find $u_2$ and $u_3$ in terms of $a$ and $b$. [2]

Given that $\displaystyle\sum_{n=1}^{3} u_n = 153$ and $b = a + 9$:

(b) Show $a^2 - 5a - 66 = 0$. [3]

(c) Find the larger possible value of $u_2$. [3]

[8 marks]

Part (a) — $u_2$, $u_3$ [2 marks] · 1 step

STEP 1/1 M1 A1

Strategy: a recurrence defines each term using the previous one. Apply $u_{n+1} = b - au_n$ once with $n = 1$ to get $u_2$, then again with $n = 2$ to get $u_3$. Watch the negative sign on $a u_n$ — it's the most common slip.

Compute $u_2$ ($n = 1$): $u_2 = b - a\,u_1 = b - a(3) = b - 3a$.

$$\boxed{\;u_2 = b - 3a\;}$$

Compute $u_3$ ($n = 2$): $u_3 = b - a\,u_2 = b - a(b - 3a)$.

Distribute $-a$ over $(b - 3a)$:

$$u_3 = b - ab - a(-3a) = b - ab + 3a^2$$

$$\boxed{\;u_3 = 3a^2 - ab + b\;}$$

Recurrence pitfall: $u_{n+1} = b - au_n$ — the negative sign on $au_n$ matters; $u_3$ depends on $u_2$ which already has $-3a$. Double negative gives $+3a^2$. Sign trap: $-a(b - 3a) = -ab + 3a^2$ (NOT $-ab - 3a^2$). Distribute carefully.
$u_2 = b - 3a,\ \ u_3 = 3a^2 - ab + b$

Part (b) — show $a^2 - 5a - 66 = 0$ [3 marks] · 2 steps

Sum: $u_1 + u_2 + u_3 = 3 + (b - 3a) + (3a^2 - ab + b) = 153$.
STEP 1/2 M1

Strategy: form the sum $u_1 + u_2 + u_3$, set it equal to $153$, then collect like terms (constants together, $a$ together, $b$ together, etc.) before substituting $b = a + 9$ in the next step.

From part (a): $u_1 = 3$, $u_2 = b - 3a$, $u_3 = 3a^2 - ab + b$.

Sum and set $\sum_{n=1}^3 u_n = 153$:

$$3 + (b - 3a) + (3a^2 - ab + b) = 153$$

Collect like terms by type:

$\;\;$ $a^2$ terms: $3a^2$.

$\;\;$ $ab$ terms: $-ab$.

$\;\;$ $b$ terms: $b + b = 2b$.

$\;\;$ $a$ terms: $-3a$.

$\;\;$ constants: $3$.

$$\boxed{\;3a^2 - ab + 2b - 3a + 3 = 153\;}$$

Trap: $\sum_{n=1}^N u_n$ uses $S_N$ formula, not $u_N$. They differ by accumulation. Here we're computing $S_3 = u_1 + u_2 + u_3$ directly because the recurrence isn't a GP/AP.
Carry-in from previous stepFrom STEP 1: $\boxed{\;3a^2 - ab + 2b - 3a + 3 = 153\;}$
STEP 2/2 M1 A1

Strategy: we have $b$ in terms of $a$ ($b = a + 9$). Substitute to get a single equation in $a$. Then expand, collect, and divide by the GCD to match the target $a^2 - 5a - 66 = 0$.

Substitute $b = a + 9$ into $3a^2 - ab + 2b - 3a + 3 = 153$:

$$3a^2 - a(a + 9) + 2(a + 9) - 3a + 3 = 153$$

Expand each bracket carefully:

$\;\;-a(a + 9) = -a^2 - 9a$

$\;\;+2(a + 9) = 2a + 18$

$$3a^2 - a^2 - 9a + 2a + 18 - 3a + 3 = 153$$

Combine like terms:

$\;\;$ $a^2$: $3a^2 - a^2 = 2a^2$.

$\;\;$ $a$: $-9a + 2a - 3a = -10a$.

$\;\;$ constants: $18 + 3 = 21$.

$$2a^2 - 10a + 21 = 153$$

Move $153$ to LHS:

$$2a^2 - 10a + 21 - 153 = 0$$

$$2a^2 - 10a - 132 = 0$$

Divide through by $2$:

$$\boxed{\;a^2 - 5a - 66 = 0\;}\quad \blacksquare$$

Sign-tracking: three separate $-9a$, $+2a$, $-3a$ terms — combine slowly: $-9a + 2a = -7a$, then $-7a - 3a = -10a$. Sense check: $-132 / 2 = -66$, $-10 / 2 = -5$, $2 / 2 = 1$ ✓ matches target.
$a^2 - 5a - 66 = 0$ ∎

Part (c) — larger value of $u_2$ [3 marks] · 3 steps

From (a)/(b): $u_2 = b - 3a$, $u_3 = 3a^2 - ab + b$. Combined with $\sum_{n=1}^3 u_n = 153$ and $b = a+9$, we showed $a^2 - 5a - 66 = 0$. Solve for $a$, take the larger root, then compute $u_2 = b - 3a = (a+9) - 3a = 9 - 2a$.
STEP 1/3 M1

Strategy: $a^2 - 5a - 66 = 0$ is monic so factoring beats the formula. Find two integers multiplying to $-66$ and summing to $-5$. List $-66$ pairs: $(\pm 1, \mp 66)$, $(\pm 2, \mp 33)$, $(\pm 3, \mp 22)$, $(\pm 6, \mp 11)$. The pair $(-11, +6)$: $(-11)(6) = -66$ ✓ and $-11 + 6 = -5$ ✓.

$$(a - 11)(a + 6) = 0$$

$$\boxed{\;a = 11\quad\text{or}\quad a = -6\;}$$

Verify by expansion: $(a - 11)(a + 6) = a^2 + 6a - 11a - 66 = a^2 - 5a - 66$ ✓.
Carry-in from previous stepFrom STEP 1: $\boxed{\;a = 11\quad\text{or}\quad a = -6\;}$
STEP 2/3 M1

Strategy: use $u_2 = b - 3a$ from part (a). Substitute $b = a + 9$ to write $u_2$ as a function of $a$ alone, then plug in each candidate.

Express $u_2$ in terms of $a$:

$$u_2 = b - 3a = (a + 9) - 3a = 9 - 2a$$

Evaluate at each root:

$\;\;a = 11$: $u_2 = 9 - 2(11) = 9 - 22 = -13$.

$\;\;a = -6$: $u_2 = 9 - 2(-6) = 9 + 12 = 21$.

Note: the LARGER value of $a$ ($a = 11$) gives the SMALLER value of $u_2$ ($-13$), because $u_2 = 9 - 2a$ is decreasing in $a$. Don't conflate "larger $a$" with "larger $u_2$".
Carry-in from previous stepFrom STEP 2: $u_2 = b - 3a = (a + 9) - 3a = 9 - 2a$
STEP 3/3 A1

Strategy: the question asks for the "larger value of $u_2$" — directly compare $-13$ and $21$. Larger means greater, so $21 > -13$ wins.

$$\boxed{\;u_2 = 21\;}$$

Phuc's question: "Why do they say 'larger'?" Because the quadratic gives two roots — both valid for the recurrence — yielding different $u_2$. "Larger" disambiguates which to report. Sense check: the corresponding $a$ value is $-6$ (negative); $b = a + 9 = 3$; sequence is $u_1 = 3$, $u_2 = 21$, $u_3 = 3(36) - (-6)(3) + 3 = 108 + 18 + 3 = 129$; sum $= 3 + 21 + 129 = 153$ ✓.
$\boxed{u_2 = 21}$
