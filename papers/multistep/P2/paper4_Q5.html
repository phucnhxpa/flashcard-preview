Q5 — Solve $\log_2(16x) + \log_2(x+1) = 3 + \log_2(x+6)$

Question

Solve $\log_2(16x) + \log_2(x + 1) = 3 + \log_2(x + 6)$. [5]

[5 marks]

Solution [5 marks] · 3 steps

STEP 1/3 M1 M1 A1

Strategy: the equation has two log terms on the LHS and a bare constant $3$ plus one log on the RHS. To equate arguments we need each side as a single $\log_2$. Use the product law on the LHS, and convert $3 = \log_2 8$ to merge the constant into a log on the RHS. Once both sides are $\log_2(\cdot)$, we can drop the logs (since $\log_2$ is one-to-one).

Formulas (log laws):

$$\log_a x + \log_a y = \log_a(xy), \qquad k = \log_a(a^k)$$

LHS: apply product law to $\log_2(16x) + \log_2(x+1)$:

$$\log_2(16x) + \log_2(x+1) = \log_2\!\big[16x(x+1)\big]$$

RHS: convert the bare $3$ to $\log_2 8$ (since $2^3 = 8$):

$$3 + \log_2(x+6) = \log_2 8 + \log_2(x+6) = \log_2\!\big[8(x+6)\big]$$

Drop the logs (both sides are $\log_2$ of something positive):

$$16x(x+1) = 8(x+6)$$

Expand both sides:

$$16x^2 + 16x = 8x + 48$$

Collect into standard quadratic form (everything to LHS):

$$16x^2 + 16x - 8x - 48 = 0$$

$$16x^2 + 8x - 48 = 0$$

Divide through by $8$:

$$\boxed{\;2x^2 + x - 6 = 0\;}$$

Phuc's slip (verbatim): "I missed that it should be the product rule, instead of addition." When you remove the log: $\log A + \log B = \log(AB)$ — multiply, don't add. Trap: $3$ is NOT $\log_2 3$ — it's $\log_2(2^3) = \log_2 8$. The base matters: $3 = \log_{10} 1000$ in base 10.
Carry-in from previous stepFrom STEP 1: $\boxed{\;2x^2 + x - 6 = 0\;}$
STEP 2/3 M1

Strategy: for $ax^2 + bx + c = 0$ with $a \ne 1$, use the "split the middle term" / AC method: find two integers that multiply to $a \cdot c$ and sum to $b$, then split the middle term and factor by grouping.

Here $a = 2$, $b = 1$, $c = -6$, so we need $a \cdot c = 2 \cdot (-6) = -12$, summing to $b = 1$.

Test pairs: $(+4, -3)$: $4 \cdot (-3) = -12$ ✓ and $4 + (-3) = 1$ ✓.

Split the middle term $x = 4x - 3x$:

$$2x^2 + 4x - 3x - 6 = 0$$

Factor by grouping (first pair $2x^2 + 4x$, second pair $-3x - 6$):

$$2x(x + 2) - 3(x + 2) = 0$$

$$\boxed{\;(2x - 3)(x + 2) = 0\;}$$

So $2x - 3 = 0$ or $x + 2 = 0$:

$$x = \tfrac{3}{2} \quad\text{or}\quad x = -2$$

Sense check via formula: $x = \dfrac{-1 \pm \sqrt{1 + 48}}{4} = \dfrac{-1 \pm 7}{4} = \tfrac{6}{4}$ or $\tfrac{-8}{4}$ = $\tfrac{3}{2}$ or $-2$ ✓.
Carry-in from previous stepFrom STEP 2: $x = \tfrac{3}{2} \quad\text{or}\quad x = -2$
STEP 3/3 A1

Strategy: $\log_a y$ is only defined for $y > 0$. The original equation has three log arguments: $16x$, $x+1$, $x+6$ — all three must be positive at any valid solution. Combining logs CAN introduce spurious roots, so we must check each candidate against the ORIGINAL log arguments.

Domain conditions:

$\;\;16x > 0 \;\Rightarrow\; x > 0$

$\;\;x + 1 > 0 \;\Rightarrow\; x > -1$

$\;\;x + 6 > 0 \;\Rightarrow\; x > -6$

Combined: $x > 0$ (the strictest).

Test $x = \tfrac{3}{2}$ (need $x > 0$): $\tfrac{3}{2} > 0$ ✓.

$\;\;16x = 24 > 0$ ✓, $\;\;x+1 = \tfrac{5}{2} > 0$ ✓, $\;\;x+6 = \tfrac{15}{2} > 0$ ✓ → accept.

Test $x = -2$ (need $x > 0$): $-2 < 0$ ✗ — fails immediately.

$\;\;16x = -32 < 0$ → $\log_2(-32)$ is undefined → reject.

$$\boxed{\;x = \tfrac{3}{2}\;}$$

Mark-loss point: writing both roots without domain check loses the final A1. Why does the spurious root appear? When we wrote $\log_2[16x(x+1)] = \log_2[8(x+6)]$, the product $16x(x+1)$ is positive even when both $16x$ and $x+1$ are negative (or both positive) — the equation has "forgotten" the original sign restrictions. Domain check restores them.
$\boxed{x = \tfrac{3}{2}}$
