Q3 · Trapezium rule — 8 strips, $n$ vs $n+1$ trap

Question

Estimate $\displaystyle\int_0^4 \sqrt{x}\,dx$ using 8 strips.

How many y-values do you need? Verify your count before computing.

[4 marks]

Solution [4 marks] · 4 steps

$\displaystyle\int_0^4 \sqrt{x}\,dx$ using 8 strips. State the number of y-values needed before computing.
STEP 1/4count strips vs y-values B1

Strategy. 8 strips means 8+1 = 9 y-values: $y_0, y_1, \ldots, y_8$. Never confuse the strip count with the y-value count.

Strip / y-value count

$$n = 8 \text{ strips} \;\Rightarrow\; 9 \text{ y-values } (y_0 \text{ to } y_8), \quad h = \frac{4-0}{8} = 0.5.$$
Phuc's mistake (Harry's red flag). Counted $n=8$ and wrote $y_0,\ldots,y_7$ (only 8 values). $n$ strips need $n+1$ y-values — the last value $y_n$ is easy to forget.

Sense check. Count the fence-posts: $0, 0.5, 1.0, \ldots, 4.0$ gives $9$ $x$-values and $9$ $y$-values ✓

↳ from STEP 1: $h=0.5$, 9 y-values $y_0\ldots y_8$
STEP 2/4tabulate y_i M1

Strategy. Compute $y_i = \sqrt{x_i}$ at each $x_i = 0, 0.5, 1, \ldots, 4$. Keep exact surd forms where possible.

Table of values

$y_0=\sqrt0=0$, $y_1=\sqrt{0.5}=\tfrac{\sqrt2}{2}\approx0.7071$, $y_2=\sqrt1=1$, $y_3=\sqrt{1.5}\approx1.2247$, $y_4=\sqrt2\approx1.4142$, $y_5=\sqrt{2.5}\approx1.5811$, $y_6=\sqrt3\approx1.7321$, $y_7=\sqrt{3.5}\approx1.8708$, $y_8=\sqrt4=2$.

↳ from STEP 2: table of $y_i$
STEP 3/4apply trapezium rule M1

Strategy. $\displaystyle\int_0^4 \sqrt{x}\,dx \approx \frac{h}{2}\bigl[y_0 + y_8 + 2(y_1+\cdots+y_7)\bigr]$. Sum the interior $y$-values twice.

Trapezium rule

$$\int_0^4 \sqrt{x}\,dx \approx \frac{0.5}{2}\bigl[0 + 2 + 2(0.7071+1+1.2247+1.4142+1.5811+1.7321+1.8708)\bigr].$$

Sum interior

$$\sum_{i=1}^{7} y_i \approx 9.5300, \quad \text{so } 2\sum \approx 19.0600.$$
↳ from STEP 3: $\tfrac{0.5}{2}[y_0+y_8+2\sum y_i]$
STEP 4/4evaluate and compare A1

Strategy. Combine: $\tfrac{0.5}{2}[0+2+19.0600] = 0.25 \times 21.0600 \approx 5.265$. Compare with the exact value $\tfrac23 x^{3/2}\big|_0^4 = \tfrac23\cdot 8 = \tfrac{16}{3}\approx 5.333$.

Final estimate

$$\int_0^4 \sqrt{x}\,dx \approx 5.265 \quad (\text{exact} = \tfrac{16}{3} \approx 5.333).$$
Phuc's mistake. Wrote $n=8$ trapeziums but used only $y_0,\ldots,y_7$, dropping $y_8=2$. That under-counts the top strip and shrinks the answer by $\approx 0.25$.

Sense check. The trapezium rule under-estimates $\sqrt{x}$ because the curve is concave-down; our $5.265 < 5.333$ matches the expected sign ✓
