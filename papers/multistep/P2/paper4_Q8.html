Q8 — Disprove '$a^2+b^2$ never multiple of 10 for consecutive primes'; prove $1 < x^2 - xy/4 < 15$

Question

(i) Use a counter example to disprove: 'When $a$ and $b$ are consecutive prime numbers, $a^2 + b^2$ is never a multiple of $10$.' [2]

(ii) Given $x$ and $y$ are even integers with $0 < x < 6$ and $0 < y < 6$, prove by exhaustion that $1 < x^2 - \tfrac{xy}{4} < 15$. [3]

[5 marks]

Part (i) — counter example [2 marks] · 1 step

STEP 1/1 M1 A1

Strategy: a counter example is a single instance where the claim FAILS. The claim says $a^2 + b^2$ is never a multiple of $10$ for consecutive primes — find ONE pair $(a, b)$ for which it IS a multiple of $10$.

Plan: systematically work through consecutive prime pairs, computing $a^2 + b^2$ until we hit a multiple of $10$. A number is a multiple of $10$ iff its last digit is $0$.

List consecutive prime pairs: $(2,3), (3,5), (5,7), (7,11), (11,13), (13,17), \ldots$

Test each:

$\;\;(2, 3)$: $4 + 9 = 13$ — last digit $3$, not mult of $10$.

$\;\;(3, 5)$: $9 + 25 = 34$ — last digit $4$, not mult of $10$.

$\;\;(5, 7)$: $25 + 49 = 74$ — last digit $4$, not mult of $10$.

$\;\;(7, 11)$: $49 + 121 = 170$ — last digit $\mathbf{0}$ ✓ — $170 = 17 \times 10$.

Counter example found: $a = 7$, $b = 11$ are consecutive primes, and $a^2 + b^2 = 170$ IS a multiple of $10$. The statement is disproved.

$$\boxed{\;a = 7,\ b = 11 \;\Rightarrow\; a^2 + b^2 = 170 = 17 \times 10\;}\quad \blacksquare$$

Phuc's slip: tried only (3,5) and (5,7) and stopped. Tutor: "We can try the five and seven... Forty-nine plus hundred and twenty-one." Keep going systematically. Why this pair works: $7^2 = 49$ ends in $9$ and $11^2 = 121$ ends in $1$; $9 + 1 = 10$, so the last digit is $0$. That's the structural reason.
$\boxed{a = 7,\ b = 11 \Rightarrow a^2+b^2 = 170 = 17 \times 10}$ ∎

Part (ii) — proof by exhaustion [3 marks] · 2 steps

Even integers $\in (0, 6)$ ⇒ $x, y \in \{2, 4\}$. Four pairs total.
STEP 1/2 M1 M1

Strategy: "proof by exhaustion" means listing ALL cases and verifying the claim for each. The constraint "even integers strictly between $0$ and $6$" gives $x \in \{2, 4\}$ and $y \in \{2, 4\}$ — exactly $2 \times 2 = 4$ pairs to check.

Define $E(x, y) = x^2 - \dfrac{xy}{4}$ and compute for all four pairs:

$\;\;(x, y) = (2, 2)$: $E = 2^2 - \dfrac{2 \cdot 2}{4} = 4 - 1 = 3$.

$\;\;(x, y) = (2, 4)$: $E = 2^2 - \dfrac{2 \cdot 4}{4} = 4 - 2 = 2$.

$\;\;(x, y) = (4, 2)$: $E = 4^2 - \dfrac{4 \cdot 2}{4} = 16 - 2 = 14$.

$\;\;(x, y) = (4, 4)$: $E = 4^2 - \dfrac{4 \cdot 4}{4} = 16 - 4 = 12$.

Set of values: $\{3, 2, 14, 12\}$.

Phuc's slip: initially started from $x = 1$. The constraint is even integers strictly between 0 and 6, so $x, y \in \{2, 4\}$ — only 4 pairs. Why exhaustion works here: the domain is FINITE (only $4$ pairs). For infinite domains, exhaustion is not possible and you'd need a general argument.
Carry-in from previous stepFrom STEP 1: $4$
STEP 2/2 A1

Strategy: compare each computed $E$ value against the target bounds $1 < E < 15$. If ALL four values satisfy the bounds, the proof is complete by exhaustion.

Check each value against $1 < E < 15$:

$\;\;E = 3$: $1 < 3 < 15$ ✓

$\;\;E = 2$: $1 < 2 < 15$ ✓

$\;\;E = 14$: $1 < 14 < 15$ ✓

$\;\;E = 12$: $1 < 12 < 15$ ✓

All four pairs satisfy $1 < E < 15$. Since these are the ONLY possible $(x, y)$ pairs under the given constraints, the claim holds for all such pairs.

$$\boxed{\;\text{For all even } x, y \text{ with } 0 < x, y < 6: \;\; 1 < x^2 - \tfrac{xy}{4} < 15\;}\quad \blacksquare$$

Tightness check: the minimum value is $E = 2$ (just above the lower bound $1$) and the maximum is $E = 14$ (just below the upper bound $15$). The bounds are tight — they're the smallest and largest enclosing integers.
$1 < x^2 - \tfrac{xy}{4} < 15$ ∎
