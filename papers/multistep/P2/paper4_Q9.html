Q9 — Show $3\cos\theta(\tan\theta\sin\theta+3) = 11-5\cos\theta \Rightarrow 3\cos^2\theta - 14\cos\theta + 8 = 0$; solve

Question

(a) Show that $3\cos\theta(\tan\theta\sin\theta + 3) = 11 - 5\cos\theta$ leads to $3\cos^2\theta - 14\cos\theta + 8 = 0$. [3]

(b) Hence solve $3\cos 2x(\tan 2x \sin 2x + 3) = 11 - 5\cos 2x$ for $0 < x < 360°$, to 1 d.p. [4]

[7 marks]

Part (a) — derive the quadratic [3 marks] · 2 steps

STEP 1/2 M1

Strategy: the equation has $\tan$, $\sin$, and $\cos$ all mixed. To collapse to a single trig function, replace $\tan\theta = \sin\theta / \cos\theta$. Then look for cancellations and apply the Pythagorean identity in the next step.

Formula (tan as sin/cos):

$$\tan\theta = \dfrac{\sin\theta}{\cos\theta}$$

Substitute into $3\cos\theta(\tan\theta\sin\theta + 3) = 11 - 5\cos\theta$:

$$3\cos\theta\!\left(\dfrac{\sin\theta}{\cos\theta}\cdot\sin\theta + 3\right) = 11 - 5\cos\theta$$

Distribute the $3\cos\theta$:

$$3\cos\theta \cdot \dfrac{\sin^2\theta}{\cos\theta} + 3\cos\theta \cdot 3 = 11 - 5\cos\theta$$

$\cos\theta$ cancels in the first term:

$$\boxed{\;3\sin^2\theta + 9\cos\theta = 11 - 5\cos\theta\;}$$

Domain note: the cancellation requires $\cos\theta \ne 0$. We'll need to remember any $\cos\theta = 0$ cases when we drop back to specific solutions, but the resulting quadratic in $\cos\theta$ won't hit $\cos\theta = 0$ anyway (verified by the discriminant later).
Carry-in from previous stepFrom STEP 1: $\boxed{\;3\sin^2\theta + 9\cos\theta = 11 - 5\cos\theta\;}$
STEP 2/2 M1 A1

Strategy: the equation now has $\sin^2\theta$ and $\cos\theta$ — to collapse to ONE variable, use the Pythagorean identity to write $\sin^2$ in terms of $\cos$. The result is a quadratic in $\cos\theta$ matching the target form.

Formula (Pythagorean identity):

$$\sin^2\theta + \cos^2\theta = 1 \;\Longrightarrow\; \sin^2\theta = 1 - \cos^2\theta$$

Substitute:

$$3(1 - \cos^2\theta) + 9\cos\theta = 11 - 5\cos\theta$$

Expand:

$$3 - 3\cos^2\theta + 9\cos\theta = 11 - 5\cos\theta$$

Collect everything to the LHS (move RHS over with sign flips):

$$3 - 3\cos^2\theta + 9\cos\theta + 5\cos\theta - 11 = 0$$

Combine like terms ($9\cos + 5\cos = 14\cos$, $3 - 11 = -8$):

$$-3\cos^2\theta + 14\cos\theta - 8 = 0$$

Multiply by $-1$ (flip every sign) to match target form:

$$\boxed{\;3\cos^2\theta - 14\cos\theta + 8 = 0\;}\quad \blacksquare$$

Sign-flip discipline: when multiplying by $-1$, EVERY term changes sign — $-3\cos^2 \to +3\cos^2$, $+14\cos \to -14\cos$, $-8 \to +8$. Why convert $\sin^2$ to $\cos$: the rest of the equation only has $\cos\theta$ — going to $\sin^2$ in terms of $\cos$ keeps everything in one variable.

Part (b) — solve for $0 < x < 360°$ [4 marks] · 3 steps

Replace $\theta$ by $2x$. Domain: $0 < x < 360°$ ⇒ $0 < 2x < 720°$.
STEP 1/3 M1 A1

Strategy: the result of part (a) was for general $\theta$. Replacing $\theta = 2x$ gives the same quadratic in $\cos 2x$. Substitute $u = \cos 2x$ to make this a standard quadratic, solve for $u$, then reject any $u$ outside $[-1, 1]$ since $|\cos| \le 1$.

Let $u = \cos 2x$: $3u^2 - 14u + 8 = 0$.

Apply quadratic formula with $a = 3$, $b = -14$, $c = 8$:

$$u = \dfrac{-(-14) \pm \sqrt{(-14)^2 - 4(3)(8)}}{2(3)} = \dfrac{14 \pm \sqrt{196 - 96}}{6} = \dfrac{14 \pm \sqrt{100}}{6} = \dfrac{14 \pm 10}{6}$$

Two roots:

$\;\;u_1 = \dfrac{14 + 10}{6} = \dfrac{24}{6} = 4$ ✗ reject (since $|\cos 2x| \le 1 < 4$).

$\;\;u_2 = \dfrac{14 - 10}{6} = \dfrac{4}{6} = \dfrac{2}{3}$ ✓.

$$\boxed{\;\cos 2x = \tfrac{2}{3}\;}$$

Domain on cosine: $\cos$ takes values in $[-1, 1]$ for any real angle. So any candidate $u$ outside this range CANNOT come from $\cos 2x$ — reject immediately.
Carry-in from previous stepFrom STEP 1: $\boxed{\;\cos 2x = \tfrac{2}{3}\;}$
STEP 2/3 M1

Strategy: the original domain is $0 < x < 360°$. With the substitution $\theta = 2x$, the working domain becomes $0 < 2x < 720°$ — TWICE the size. So we expect about twice as many solutions for $2x$ (4 instead of 2). Use cosine's symmetry: $\cos(\phi) = \cos(-\phi) = \cos(360° - \phi)$, plus $360°$ periodicity.

Formula (cosine general solution):

$$\cos\phi = k \;\Longrightarrow\; \phi = \pm\cos^{-1}(k) + 360°n, \;\; n \in \mathbb{Z}$$

Principal value: $\cos^{-1}(2/3) \approx 48.19°$.

Generate solutions for $2x \in (0°, 720°)$ from $\pm 48.19° + 360°n$:

$\;\;n = 0$: $2x = +48.19°$ ✓; $2x = -48.19°$ ✗ (outside).

$\;\;n = 1$: $2x = 360° - 48.19° = 311.81°$ ✓; $2x = 360° + 48.19° = 408.19°$ ✓.

$\;\;n = 2$: $2x = 720° - 48.19° = 671.81°$ ✓; $2x = 720° + 48.19° = 768.19°$ ✗ (outside).

$$2x \in \{48.19°,\;\; 311.81°,\;\; 408.19°,\;\; 671.81°\}$$

STEP 3/3 A1 A1

Strategy: we have all four values of $2x$ in $(0°, 720°)$. Divide each by $2$ to recover $x$ in $(0°, 360°)$, rounded to $1$ d.p.

$\;\;2x = 48.19° \;\Rightarrow\; x = 24.095° \approx 24.1°$.

$\;\;2x = 311.81° \;\Rightarrow\; x = 155.905° \approx 155.9°$.

$\;\;2x = 408.19° \;\Rightarrow\; x = 204.095° \approx 204.1°$.

$\;\;2x = 671.81° \;\Rightarrow\; x = 335.905° \approx 335.9°$.

$$\boxed{\;x \in \{24.1°,\;155.9°,\;204.1°,\;335.9°\}\text{ (1 d.p.)}\;}$$

Sense check: 4 solutions in $(0°, 360°)$, consistent with $\cos 2x = $ const for $2x \in (0°, 720°)$ (two periods). Symmetry check: $24.1° + 335.9° = 360°$ and $155.9° + 204.1° = 360°$ — solutions pair up nicely.
$\boxed{x = 24.1°,\ 155.9°,\ 204.1°,\ 335.9°}$
