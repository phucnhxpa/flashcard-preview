Q7 — Normal Distribution, Inverse Normal, Two-Equation System

Question

$J \sim N(\mu, \sigma^2)$ models the volume of juice extracted from 1 kg of apples. With $\mu = 500$, $\sigma = 25$, use standardisation to:
(a)(i) Show that $P(J > 510) = 0.3446$. (2)
(a)(ii) Find $d$ such that $P(J > d) = 0.9192$. (3)
(b) For 5 independent bags, find the probability that each yields less than 510 ml. (2)
After adjustment, $R \sim N(520, k^2)$ with $P(R < r) = 0.15$ and $P(R > 3r - 800) = 0.005$.
(c) Find the values of $r$ and $k$. (7)

Part (a)(i)[2 marks] · 1 step

Needed before revealing this part

$J\sim N(500,25^2)$, so standardise with $Z=(J-500)/25$. For 510, $z=0.4$.

STEP 1/1Part (a)(i)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

(a)(i)
$$z = \frac{510-500}{25} = 0.4,\qquad P(Z > 0.4) = 1 - \Phi(0.4) = 1 - 0.6554 = \boxed{0.3446}.\ ✓$$

Mistake guard from the attempt/revision notes.

What went wrong. Three predictable errors:
1. In (a)(ii), $P(J > d) = 0.9192$ is more than 0.5, so $d$ lies below the mean $\Rightarrow$ $z$ is negative. Students forgetting the sketch make $z = +1.40$ and end up with $d = 535$.
2. In (b), multiplying $P(J<510)$ by 5 instead of raising to the 5th power.
3. In (c), wrong $z$-value because of confusion between body table $\Phi(z)$ and the percentage-point table.

Revision-guide trap (a)(ii).. Looking up $\Phi(z) = 0.9192$ $\Rightarrow$ $z = 1.40$, then writing $d = 500 + 1.40 \times 25 = 535$ --- wrong sign of $z$.

Target: $\boxed{0.3446}$

Part (a)(ii)[3 marks] · 1 step

Needed before revealing this part

$P(J>d)=0.9192$ means $P(J\le d)=0.0808$, so the z-value is negative: $z=-1.40$.

STEP 1/1Part (a)(ii)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

(a)(ii) $P(J > d) = 0.9192 \Rightarrow P(J \le d) = 0.0808$. This is a lower-tail probability < 0.5, so $z < 0$. Solve $\Phi(-z) = 0.9192 \Rightarrow -z = 1.40 \Rightarrow z = -1.40$.
$$d = \mu + z\sigma = 500 + (-1.40)(25) = 500 - 35 = \boxed{465}.$$

Target: $\boxed{465}$

Part (b)[2 marks] · 1 step

Needed before revealing this part

From (a)(i): $P(J>510)=0.3446$, so $P(J<510)=0.6554$. Five bags are independent, so raise to power 5.

STEP 1/1Part (b)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

(b)
$$P(J < 510) = 1 - 0.3446 = 0.6554.$$
For 5 independent bags:
$$P(\text{all 5} < 510) = (0.6554)^5 = 0.12095 \approx \boxed{0.121}.$$

Target: $\boxed{0.121}$

Part (c)[7 marks] · 1 step

Needed before revealing this part

$R\sim N(520,k^2)$. Use $P(R3r-800)=0.005\Rightarrow z=+2.5758$.

STEP 1/1Part (c)

Formula first. State the rule symbolically, then substitute values and box the exam answer.

(c) Translate the two probabilities into $z$-equations.

For $P(R < r) = 0.15$ (lower tail < 0.5 $\Rightarrow$ $z<0$): $\Phi(-z) = 0.85$, percentage-point/body lookup gives $-z \approx 1.0364$, so $z_1 = -1.0364$:
$$\frac{r - 520}{k} = -1.0364 \implies r = 520 - 1.0364k. \tag{$\star$}$$

For $P(R > 3r - 800) = 0.005$ (upper tail < 0.5 $\Rightarrow$ $z>0$): from the percentage-point table, $z_{0.005} = 2.5758$:
$$\frac{(3r - 800) - 520}{k} = 2.5758 \implies \frac{3r - 1320}{k} = 2.5758 \implies 3r = 1320 + 2.5758k$$
$$\implies r = 440 + 0.8586k. \tag{$\star\star$}$$

Equate $(\star)$ and $(\star\star)$:
$$520 - 1.0364k = 440 + 0.8586k \implies 80 = 1.8950k$$
$$\implies k = \frac{80}{1.8950} = 42.2163 \approx \boxed{42.2}.$$

Substitute back:
$$r = 520 - 1.0364 \times 42.2163 = 520 - 43.7531 = 476.2469 \approx \boxed{476}.$$

Check: $3r - 800 = 1428.74 - 800 = 628.74$, $z = (628.74 - 520)/42.22 = 2.5758$ ✓; $z = (476.25 - 520)/42.22 = -1.0364$ ✓.

Target: $\boxed{476}$

Key formula / concept.
