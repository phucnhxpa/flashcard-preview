Question

Part (D) — interpret the gradient in context [2 marks] · 2 steps

The gradient of $w = a + 3.27\,h$ is $b = 3.27$. The word "interpret" means write a sentence a non-mathematician would understand, naming the actual variables (height, weight, kg, cm) and the actual number (3.27).
STEP 1/2identify the four ingredients

Strategy. The interpretation sentence needs four ingredients: (1) the numerical value 3.27, (2) the rate "for each 1 cm increase in height", (3) the real-world variable names "height" and "weight", (4) the units "kg" and "cm". Also include "on average" because the line is a best fit, not an exact law.

Template (memorise this)

"On average, for each 1 [unit of x] increase in [name of x], [name of y] increases by [gradient] [unit of y]."

Phuc's mistake. Phuc wrote: "Positive correlation; as height increases, weight increases." This has no rate, no "on average", no number 3.27, no units. Harry: "it's not quite specific enough… they actually want you to say what the gradient is."
↳ from STEP 1: template = "On average, for each 1 [unit x] increase in [name x], [name y] increases by [b] [unit y]."
STEP 2/2write the sentence

Strategy. Fill in the template with: x = height (cm), y = weight (kg), b = 3.27.

Mark-scheme answer

"On average, for each 1 cm increase in height, the weight increases by 3.27 kg."

Why "on average" matters. The regression line is a best fit — the data scatter around it. "On average" signals this; omitting it treats the line as an exact physical law, which it is not.

Sense check. A horse 1 cm taller is ~3.27 kg heavier on average — a small but sensible rate for a large animal.

Target: "On average, for each 1 cm increase in height, the weight increases by 3.27 kg."

Part (F) — interpolation vs extrapolation [2 marks] · 2 steps

Part (E) produced two estimates of $w$: one for $h$ inside the data range and one for $h$ outside the data range. Part (F) asks which is more reliable, giving a reason.
STEP 1/2identify the data range

Strategy. Let $[h_{\min}, h_{\max}]$ be the range of heights in the original data set. Compare each estimate's $h$-value against this range.

  • $h^*$ inside $[h_{\min}, h_{\max}]$ ⇒ interpolation ⇒ generally reliable.
  • $h^*$ outside $[h_{\min}, h_{\max}]$ ⇒ extrapolation ⇒ unreliable.
Phuc's mistake. Phuc did not use the keywords "interpolation" and "extrapolation" by name. Harry: "the key words they're looking for you to use… are interpolation versus extrapolation." The words are the marking points.
↳ from STEP 1: inside range = interpolation (reliable); outside = extrapolation (unreliable).
STEP 2/2write the answer using keywords

Strategy. State which estimate is which, using the exact words "interpolation" and "extrapolation", then say which is more reliable.

Mark-scheme answer

"The first estimate is an interpolation (within the data range), so it is more reliable. The second is an extrapolation (outside the data range), so it is less reliable."

Why extrapolation is unreliable. The regression model was fitted to data within a specific range of $h$-values. Outside that range there is no evidence the linear relationship holds — the true relationship could curve, flatten, or reverse.

Sense check. The sentence uses both keywords by name and states a clear reason — earns both marks.

Target: "The first estimate is interpolation (reliable); the second is extrapolation (unreliable)."
