Q6 — Venn Diagram & Independence

Question

The Venn diagram shows the probabilities related to teenagers playing 3 particular board games. $C$ is the event that a teenager plays Chess, $S$ that a teenager plays Scrabble, $G$ that a teenager plays Go.

Regions (from the diagram): $C$ only $=0.04$; $G$ only $=0$; $C\cap G\cap S = 0.15$; $C\cap G$ only $=p$; $C\cap S$ only $=0.12$; $S\cap G$ only $=0.10$; $S$ only $=0.23$; outside all three $=q$.

(a) Find the probability that a randomly selected teenager plays Chess but does not play Go. (1)
Given that the events $C$ and $S$ are independent,
(b) find the value of $p$. (4)
(c) Hence find the value of $q$. (2)
(d) Find (i) $P((C \cup S) \cap G')$; (ii) $P(C \mid S \cap G)$. (1, 2)
A youth club consists of a large number of teenagers. In this youth club 76 teenagers play Chess and Go.
(e) Use the information in the Venn diagram to estimate how many of the teenagers in the youth club do not play Scrabble. (3)

[10 marks]

Part (a)[1 mark] · 3 steps

STEP 1/3

Worked solution

STEP 2/3

$P(C \cap G') = P(C\text{ only}) + P(C \cap S\text{ only, not }G) = 0.04 + 0.12 = \boxed{0.16}$

Target: $\boxed{0.16}$
STEP 3/3

Exam checklist.

  • [ ] Identify the regions of $C$ that lie outside $G$.
  • [ ] Add only these two regions — exclude both $G$-containing regions ($p$ and 0.15).

Common pitfalls.

Including $C \cap G$ regions; using $P(C) \cdot P(G')$ (only valid if independent — not assumed in (a)).

Part (b)[4 marks] · 3 steps

Carry-in: Part (a) result: $\boxed{0.16}$
STEP 1/3

Worked solution — Find $p$

$P(C) = 0.04 + p + 0.15 + 0.12 = 0.31 + p$
$P(S) = 0.12 + 0.15 + 0.10 + 0.23 = 0.60$
$P(C \cap S) = 0.12 + 0.15 = 0.27$ (both regions inside the $C\cap S$ overlap)

STEP 2/3

Independence: $P(C \cap S) = P(C) \cdot P(S)$
$$0.27 = (0.31 + p)(0.60)$$
$$0.31 + p = \frac{0.27}{0.60} = 0.45 \;\Longrightarrow\; \boxed{p = 0.14}$$

Key formula: $A, B$ independent ⇔ $P(A\cap B) = P(A)P(B)$.

Target: $\boxed{p = 0.14}$
STEP 3/3

Exam checklist.

  • [ ] List $P(C), P(S), P(C\cap S)$ separately.
  • [ ] $P(C \cap S)$ includes the whole overlap of $C$ and $S$, i.e. both the triple-overlap (0.15) and the C∩S-only region (0.12).
  • [ ] Solve linear equation for $p$.

Common pitfalls.

Using only 0.12 (or only 0.15) for $P(C\cap S)$ — must include both subregions of the $C$-$S$ intersection.

Part (c)[2 marks] · 3 steps

Carry-in: Part (a) result: $\boxed{0.16}$
Part (b) result: $\boxed{p = 0.14}$
STEP 1/3

Worked solution — Find $q$

STEP 2/3

All probabilities sum to 1:
$$0.04 + 0 + 0.15 + p + 0.12 + 0.10 + 0.23 + q = 1$$
$$0.64 + p + q = 1 \;\Longrightarrow\; p + q = 0.36$$
$$q = 0.36 - 0.14 = \boxed{0.22}$$

Target: $\boxed{0.22}$
STEP 3/3

Exam checklist.

  • [ ] Sum all 8 regions (including $q$ outside).
  • [ ] Set equal to 1.
  • [ ] Use $p$ from (b).

Common pitfalls.

Forgetting the outside region; forgetting the $G$-only region even when its value is 0.

Key Formulas

NOT in the formula booklet (memorise)

  1. Variance from raw sums: $\sigma^2 = \dfrac{\sum x^2}{n} - \bar{x}^2$ (Phuc saved this Apr 24 — not in IAL booklet).
  2. Median by linear interpolation: $\text{Median} = L + \dfrac{N/2 - F_b}{f_m}\times c$.
  3. Outlier definition: $x < Q_1 - 1.5\,\text{IQR}$ or $x > Q_3 + 1.5\,\text{IQR}$.
  4. Edexcel quartile rule: $n/4$ integer → average of $(n/4)$th and $(n/4+1)$th; not integer → round up.
  5. Frequency density: FD = frequency ÷ class width (histogram area ∝ frequency).

IN the formula booklet (still know how to use)

  1. $S_{xy} = \sum xy - \dfrac{\sum x \sum y}{n}$; analogous for $S_{xx}, S_{yy}$.
  2. $r = \dfrac{S_{xy}}{\sqrt{S_{xx} S_{yy}}}$.
  3. Regression $y$ on $x$: gradient $b = S_{xy}/S_{xx}$, intercept $a = \bar{y} - b\bar{x}$.
  4. $E(aX+b) = aE(X)+b$; $\operatorname{Var}(aX+b) = a^2\operatorname{Var}(X)$.
  5. $\operatorname{Var}(X) = E(X^2) - [E(X)]^2$.
  6. $P(A|B) = \dfrac{P(A\cap B)}{P(B)}$.
  7. Standard normal $\Phi$ and $\Phi^{-1}$ from tables.

Exam Strategy (tactical)

  1. Read the key first on any stem-and-leaf diagram. Back-to-back diagrams need the left-hand leaves read right-to-left — confirm this before you transcribe values. A single mis-read costs every downstream mark.
  2. For "fastest / slowest" normal questions, sketch and shade the curve before you do any algebra. Fastest = shortest time = lower tail (left). This is the single mistake that wrecked Q5(b).
  3. For $\sigma$ from a symmetric interval, always sanity-check: $\sigma$ should be of the same order of magnitude as the interval half-width. $\sigma = 10/1.96$ is ~5, not 0.5 (the decimal slip in Q5c).
  4. For "no change to box plot" problems (Q1f), list all five statistics (min, $Q_1$, median, $Q_3$, max) as constraints and check each one — don't stop after the ordering constraint.
  5. For Venn diagram "rest of universe" questions, always use the shortcut $P(A') = 1 - P(A)$ when "not $A$" includes the outside-the-circles region. Phuc's 47 vs 105 mistake was forgetting $q$.
  6. For E(Y), Var(Y) with $Y=aX-b$, do variance first: gives $a^2$, eliminates $b$. Then plug $a$ into $E(Y)$ for $b$.
  7. Quartile positions: Memorise the two rules. For $n=35$ (Q1b), Q1 is position 9, not 8.75 (round up). For $n=32$ (Maple), Q1 is the average of positions 8 and 9 (integer rule).
  8. Histogram dimensions: Always derive two scale constants (cm per mm of class width, and cm per unit of frequency density) from the reference bar.
  9. Show working in "show that" parts — even when the answer is given, marks are for the derivation. Write each line of the variance computation.
  10. Round only at the final answer. Use unrounded values when computing regression intercepts and conditional probabilities (carrying $b=0.72184$ vs 0.722 affects $a$ at the 3rd s.f.).

Pattern Summary

The May 2024 WST01/01 paper rewards careful reading of definitions and clean separation of constraints. The "expensive" errors all cluster around translation steps: translating "fastest 60%" into the correct tail (Q5b), translating "no change to box plot" into five separate statistical constraints (Q1f), translating "not Scrabble" into the whole complement including the outside region (Q6e), and translating decimal arithmetic without losing factor-of-10 (Q5c). The technical bookwork (PMCC, regression, variance from a discrete table, linear interpolation, coding) is straightforward once formulas are deployed correctly. Phuc's pattern across this paper is strong on mechanical execution but loses marks at the interpretation boundary between English and probability/statistics — every redo flag in this paper traces back to a translation slip, not a computation error. The drill: slow down on words like "fastest", "no change", and "does not play"; sketch a diagram; list the constraints before computing.
