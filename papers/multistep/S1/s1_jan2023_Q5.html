Q5 — Normal distribution — wing lengths, quartiles, outlier conditional

Question

The lengths, $L$ mm, of housefly wings are normally distributed with $L \sim N(4.5, 0.4^2)$.

(a) Find the probability that a randomly selected housefly has a wing length of less than 3.86 mm. (3)

(b) Find
(i) the upper quartile ($Q_3$) of $L$
(ii) the lower quartile ($Q_1$) of $L$ (4)

A value that is greater than $Q_3 + 1.5\times(Q_3 - Q_1)$ or smaller than $Q_1 - 1.5\times(Q_3 - Q_1)$ is defined as an outlier.

(c) Find these two outlier limits. (3)

A housefly is selected at random.

(d) Using standardisation, show that the probability that this housefly is not an outlier is 0.993 to 3 decimal places. (3)

Given that this housefly is not an outlier,

(e) showing your working, find the probability that the wing length of this housefly is greater than 5 mm. (4)

(e) Conditional $P(L > 5\mid \text{not outlier})$ — Phuc's redo

Let $N$ = "not an outlier" = ${3.4208 < L < 5.5792}$.

$$P(L > 5 \mid N) = \frac{P(L > 5 \cap N)}{P(N)}$$

Crucial step: $L > 5 \cap N$ is not just $L > 5$ — it is the intersection with the not-outlier band, so $5 < L < 5.5792$.

$$Z_{\text{lower}} = \frac{5 - 4.5}{0.4} = 1.25, \qquad Z_{\text{upper}} = 2.70$$

$$P(5 < L < 5.5792) = P(1.25 < Z < 2.70) = 0.9965 - 0.8944 = 0.1021$$

$$P(L > 5 \mid N) = \frac{0.1021}{0.993} = 0.10282\ldots = \boxed{0.103}$$

What went wrong

Phuc's note describes treating this as an inverse normal ("we can only find less than or equal … 1 minus the smaller equals the larger") — that's the wrong framework entirely. This is not inverse normal; it's a direct conditional probability where you must (i) recognise the conditioning event caps $L$ at $5.5792$ on the top end, (ii) compute the intersection $5 < L < 5.5792$, not $L > 5$. Forgetting the cap and using $P(L > 5) = P(Z > 1.25) = 0.1056$ in the numerator would give $0.1056/0.993 \approx 0.106$ — wrong, and the single most-common conditional-normal mistake.

Key concept (IN booklet). Standardisation: $Z = \dfrac{X - \mu}{\sigma} \sim N(0,1)$. Use the table $\Phi(z) = P(Z < z)$.
Key concept (NOT in booklet). Quartile $z$-value: $z = 0.6745$ for $Q_3$, $-0.6745$ for $Q_1$. Memorise this.
