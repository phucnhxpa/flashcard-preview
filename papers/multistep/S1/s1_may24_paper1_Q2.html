Q2 — Spinner $X \in \{2,4,5,7,8\}$: $\Pr(2X-3>5)$, Var, $Y=aX-b$, $P(\text{Sam wins})$

Question

The discrete random variable $X$ is the score on a spinner with probability table:

(a) Find $\Pr(2X - 3 > 5)$. [1]

(b) Given $E(X) = 4.6$, show that $\text{Var}(X) = 4.14$. [3]

(c) $Y = aX - b$ with $a, b > 0$, $E(Y) = 13.4$, $\text{Var}(Y) = 66.24$. Find $a$ and $b$. [4]

(d) Sam: $S = X_1$; Roger: $R = 2X_2 - 3$, with $X_1, X_2$ independent copies of $X$. Higher score wins. Find $\Pr(\text{Sam wins})$. [4]

  x        2     4     5     7     8
P(X=x)   0.25  0.30  0.20  0.10  0.15

[12 marks]

Part (a) — $\Pr(2X - 3 > 5)$ [1 marks] · 1 step

STEP 1/1translate to $X$ B1

$2X - 3 > 5 \Longleftrightarrow X > 4$. So sum probabilities for $X \in \{5, 7, 8\}$:

$$\Pr(X > 4) = 0.20 + 0.10 + 0.15 = 0.45$$

$\Pr(2X - 3 > 5) = 0.45$

Part (b) — Show $\text{Var}(X) = 4.14$ [3 marks] · 2 steps

$E(X) = 4.6$ given. Use $\text{Var}(X) = E(X^2) - [E(X)]^2$.
STEP 1/2$E(X^2)$ M1 M1

Formula: $E(X^2) = \sum x^2 \Pr(X=x)$ — square first, then weight.

$$E(X^2) = 4(0.25) + 16(0.30) + 25(0.20) + 49(0.10) + 64(0.15)$$

$$= 1 + 4.8 + 5 + 4.9 + 9.6 = 25.3$$

$E(X^2) = 25.3$
↳ from STEP 1:From STEP 1: $E(X^2) = 25.3$.
STEP 2/2apply formula A1

$$\text{Var}(X) = E(X^2) - [E(X)]^2 = 25.3 - (4.6)^2 = 25.3 - 21.16 = 4.14\ \checkmark$$

Show-that discipline: finish with the printed answer 4.14. Don't round intermediate values — keep the full 21.16.
$\boxed{\,\text{Var}(X) = 4.14\,}$

Part (c) — Find $a$ and $b$ in $Y = aX - b$ [4 marks] · 2 steps

$E(X) = 4.6,\ \text{Var}(X) = 4.14,\ E(Y) = 13.4,\ \text{Var}(Y) = 66.24,\ a,b > 0$.
STEP 1/2variance ⇒ $a$ M1 A1

Linear coding rule: $\text{Var}(aX-b) = a^2\,\text{Var}(X)$ — the constant $-b$ vanishes.

$$a^2 \cdot 4.14 = 66.24 \;\Longrightarrow\; a^2 = \frac{66.24}{4.14} = 16$$

Since $a > 0$:

$\boxed{\,a = 4\,}$
↳ from STEP 1:From STEP 1: $a = 4$.
STEP 2/2mean ⇒ $b$ M1 A1

$E(aX - b) = aE(X) - b$:

$$4(4.6) - b = 13.4 \;\Longrightarrow\; 18.4 - b = 13.4 \;\Longrightarrow\; b = 5$$

Trap: some students use $a$ instead of $a^2$ in the variance equation (gives $a = 16$ → $b = 60$ — wrong). The $+b$ shift never affects spread; only the multiplicative $a$ does, and it appears squared.
$\boxed{\,a = 4,\ b = 5\,}$

Part (d) — $\Pr(\text{Sam wins})$ with $S = X_1$, $R = 2X_2 - 3$ [4 marks] · 3 steps

Sam wins iff $S > R$ (strict, ties don't count). Condition on $X_2$.
STEP 1/3condition on $X_2$ M1

Strategy: for each value of $X_2 = x_2$, compute $R = 2x_2 - 3$, then $\Pr(X_1 > R)$, then weight by $\Pr(X_2 = x_2)$.

$$\Pr(\text{Sam wins}) = \sum_{x_2} \Pr(X_1 > 2x_2 - 3)\cdot \Pr(X_2=x_2)$$
