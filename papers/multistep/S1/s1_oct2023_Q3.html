Q3 — Venn Diagrams, Mutually Exclusive, Independence

Question

(i) Bob shops at a market each week.
- $C$ = event "Bob buys carrots", $P(C) = 0.65$.
- $O$ = event "Bob buys onions", $P(O') = 0.3$.
- $P(O \cap C') = 0.15$ (Bob buys onions but not carrots).
The Venn diagram has regions $w$ (in $O$ only), $x$ (in $O \cap C$), $y$ (in $C$ only), $z$ (outside both).

(a) Find the value of $w$, $x$, $y$ and $z$. (4)
(b) Find the probability that Bob buys carrots or onions but not both. (1)
(c) Show that the events $C$ and $O$ are not statistically independent. (2)

(ii) $F$, $G$, $H$ are three events. $F$ and $H$ are mutually exclusive. $F$ and $G$ are independent. $P(F) = \tfrac{2}{7}$, $P(H) = \tfrac{1}{4}$, $P(F \cup G) = \tfrac{5}{8}$.

(a) Find $P(F \cup H)$. (1)
(b) Find $P(G)$. (3)
(c) Find $P(F \cap G')$. (1)

[12 marks]

Solution4 steps

STEP 1/4

(i)(a) Region definitions:
- $w = P(O \cap C') = 0.15$ (given directly).
- $P(O) = 1 - P(O') = 1 - 0.3 = 0.7$, and $P(O) = w + x$, so $x = 0.7 - 0.15 = 0.55$.
- $P(C) = x + y = 0.65$, so $y = 0.65 - 0.55 = 0.10$.
- Total $= w + x + y + z = 1$, so $z = 1 - (0.15 + 0.55 + 0.10) = 0.20$.

$$\boxed{w = 0.15,\quad x = 0.55,\quad y = 0.10,\quad z = 0.20.}$$

(i)(b) "Carrots or onions but not both" = the symmetric difference = $w + y = 0.15 + 0.10 = \boxed{0.25}$.

What went wrong. Phuc swapped the labels $w$ and $x$ in part (i)(a) --- he assigned $w = 0.55$ (the intersection value) when the diagram labelling makes $w$ the "$O$ only" region with value $0.15$. In (ii)(b) he wrote the union formula correctly but stumbled in factoring $P(G)$ out cleanly.

Phuc's incorrect approach (annotated). (i)(a) --- written: $w = 0.55, \ x = 0.15$. The diagram reads left-to-right: $w$ = $O$-only, $x$ = intersection, $y$ = $C$-only. So $w$ must be $P(O \cap C') = 0.15$, not the intersection $0.55$.

(ii)(b) --- wrote $\tfrac{5}{8} = \tfrac{2}{7} + P(G) - P(F\cap G)$ without using independence to substitute $P(F\cap G) = \tfrac{2}{7}P(G)$.

Target: $\boxed{0.25}$
STEP 2/4

(i)(c) Test independence:
- $P(C \cap O) = x = 0.55$.
- $P(C) \cdot P(O) = 0.65 \times 0.7 = 0.455$.

Since $0.55 \neq 0.455$, $C$ and $O$ are not independent. ∎

(ii)(a) Mutually exclusive means $P(F \cap H) = 0$:
$$P(F \cup H) = P(F) + P(H) - 0 = \tfrac{2}{7} + \tfrac{1}{4} = \tfrac{8}{28} + \tfrac{7}{28} = \boxed{\tfrac{15}{28}}.$$

Target: $\boxed{\tfrac{15}{28}}$
STEP 3/4

(ii)(b) Independence of $F, G$ gives $P(F \cap G) = P(F)\,P(G) = \tfrac{2}{7} P(G)$.
$$P(F \cup G) = P(F) + P(G) - P(F\cap G)$$
$$\tfrac{5}{8} = \tfrac{2}{7} + P(G) - \tfrac{2}{7} P(G) = \tfrac{2}{7} + \tfrac{5}{7} P(G).$$
$$\tfrac{5}{7} P(G) = \tfrac{5}{8} - \tfrac{2}{7} = \tfrac{35 - 16}{56} = \tfrac{19}{56}.$$
$$P(G) = \tfrac{19}{56} \times \tfrac{7}{5} = \tfrac{19}{40} = \boxed{0.475}.$$

Check: $P(F\cap G) = \tfrac{2}{7} \cdot \tfrac{19}{40} = \tfrac{38}{280} = \tfrac{19}{140}$. Union: $\tfrac{2}{7} + \tfrac{19}{40} - \tfrac{19}{140} = \tfrac{80 + 133 - 38}{280} = \tfrac{175}{280} = \tfrac{5}{8}.$ ✓

(ii)(c) $P(F \cap G') = P(F) - P(F \cap G) = \tfrac{2}{7} - \tfrac{19}{140} = \tfrac{40 - 19}{140} = \tfrac{21}{140} = \boxed{\tfrac{3}{20}}.$

Target: $\boxed{\tfrac{3}{20}}$
STEP 4/4

(Alternative: $P(F\cap G') = P(F) P(G') = \tfrac{2}{7} \cdot \tfrac{21}{40} = \tfrac{42}{280} = \tfrac{3}{20}.$)

Audit checklist1 steps

STEP 1/1

Key formula / concept.

Key formula / concept

  • Independence: $P(A \cap B) = P(A) P(B)$.
  • Mutually exclusive: $P(A \cap B) = 0$.
  • $P(A \cup B) = P(A) + P(B) - P(A\cap B)$ always.
  • $P(A \cap B') = P(A) - P(A \cap B)$.

Exam checklist before moving on.

Exam checklist

  • [ ] Define each Venn region explicitly before substituting.
  • [ ] Use $P(O) = 1 - P(O')$ when complement probability is given.
  • [ ] All four regions sum to $1$.
  • [ ] Independence test: compare $P(A\cap B)$ to $P(A)P(B)$ --- state both numerical values.
  • [ ] For "not independent" --- show numerical inequality, then conclude.

Common pitfalls.

Common pitfalls

  • Labelling $w$ and $x$ in the wrong order --- always re-read the printed diagram orientation.
  • Confusing mutually exclusive with independent. Two non-trivial events cannot be both: if $P(A\cap B) = 0$ but $P(A), P(B) > 0$, then $P(A)P(B) > 0 \neq 0$, so not independent.
  • Forgetting to convert $P(F\cap G)$ via independence in the union equation.
