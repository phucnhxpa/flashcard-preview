Q7 · Long division & inverse function with restricted domain

Question

The function $g$ is defined by $g(x) = \dfrac{5x + 3}{x + 2}$, with domain $x > -2$.

(a) Find the range of $g$.

(b) Find $g^{-1}(x)$ and state its domain.

[7 marks total: (a) 3 marks, (b) 4 marks]

(a) Find the range of $g$ [3 marks] · 4 steps

$g(x) = \frac{5x+3}{x+2}$, domain $x > -2$. The degree of numerator equals the degree of denominator, so do long division first. The quotient + remainder/divisor form reveals the graph as a transformation of $h(x) = -\frac{1}{x}$.
STEP 1/4long division

Strategy. Divide $5x + 3$ by $x + 2$: how many times does $x$ go into $5x$? Answer: $5$. Then multiply $5(x + 2) = 5x + 10$ and subtract to find the remainder.

Divide

$$5x + 3 = 5(x + 2) - 7.$$

Quotient + remainder/divisor form

$$\frac{5x + 3}{x + 2} = 5 + \frac{-7}{x + 2} = 5 - \frac{7}{x + 2}.$$
Trap 1 (watch-list). The output of long division must be written as quotient + remainder/divisor, i.e. $5 + \frac{-7}{x+2}$, NOT as the multiplied form $5(x+2) - 7$. The multiplied form is just an identity check; only the quotient+remainder form lets you read off the graph transformations (asymptotes, stretches, translations).

Sense check. Verify: $5 + \frac{-7}{x+2} = \frac{5(x+2) - 7}{x+2} = \frac{5x + 10 - 7}{x+2} = \frac{5x + 3}{x+2}$. ✓

$g(x) = 5 - \dfrac{7}{x + 2}$.
↳ from STEP 1: $g(x) = 5 - \dfrac{7}{x + 2}$, domain $x > -2$.
STEP 2/4identify asymptotes

Strategy. Read the asymptotes from the quotient+remainder form. $g(x) = 5 - \frac{7}{x+2}$ is the parent $h(x) = -\frac{1}{x}$ stretched vertically by $7$, shifted left by $2$, and shifted up by $5$.

Vertical asymptote

$$x + 2 = 0 \;\Rightarrow\; x = -2 \quad\text{(denominator zero)}.$$

Horizontal asymptote

$$g(x) \to 5 \quad\text{as } x \to \pm\infty \;\Rightarrow\; y = 5.$$
Graph transformations from $h(x) = -1/x$. (1) Vertical stretch by $7$: $-7/x$. (2) Replace $x$ with $x + 2$ (horizontal shift left by $2$): $-7/(x+2)$. (3) Add $5$ (vertical shift up by $5$): $5 - 7/(x+2)$. The vertical asymptote $x = 0$ shifts to $x = -2$; the horizontal asymptote $y = 0$ shifts to $y = 5$.

Sense check. The domain restriction $x > -2$ means we are on the RIGHT branch only (to the right of the vertical asymptote $x = -2$). On this branch, $x + 2 > 0$, so $\frac{7}{x+2} > 0$, hence $g(x) = 5 - \frac{7}{x+2} < 5$.

Asymptotes: $x = -2$ (vertical), $y = 5$ (horizontal).
↳ from STEP 2: asymptotes $x = -2$, $y = 5$; domain $x > -2$ (right branch only).
STEP 3/4determine range on x > -2

Strategy. On the right branch ($x > -2$): as $x \to -2^+$, $x + 2 \to 0^+$, so $\frac{7}{x+2} \to +\infty$ and $g(x) = 5 - \frac{7}{x+2} \to -\infty$. As $x \to +\infty$, $\frac{7}{x+2} \to 0^+$, so $g(x) \to 5^-$ (approaches $5$ from below). So $g$ takes all values below $5$.

Behaviour at the boundaries of the domain

$$x \to -2^+:\quad g(x) \to -\infty.$$ $$x \to +\infty:\quad g(x) \to 5^-.$$

Range

$$g(x) < 5.$$
Trap 4 (watch-list) — x vs y in range. State the range as "$y < 5$" or "$g(x) < 5$" — NEVER "$x < 5$". Edexcel marks "range: $x < 5$" as wrong even though the inequality value is correct. Ranges use $y$ (or $f(x)$, $g(x)$); domains use $x$.

Sense check. On $x > -2$, $g$ is continuous and strictly increasing (the $-\frac{7}{x+2}$ term increases toward $0$ as $x$ grows). So $g$ sweeps from $-\infty$ up to (but not reaching) $5$. Every value $< 5$ is attained exactly once — confirming $g$ is one-to-one on this domain.

Range: $g(x) < 5$, i.e. $\boxed{y < 5}$.
↳ from STEP 3: range $y < 5$.
STEP 4/4state the range

Strategy. Box the range in the correct wording. Since $g$ is one-to-one on $x > -2$ (strictly increasing), the inverse exists and its domain will be this range.

Range

$$\boxed{\text{Range of } g: \quad y < 5}.$$
Why the range matters. The domain of $g^{-1}$ equals the range of $g$ (the inverse undoes $g$, so its inputs must be exactly the outputs $g$ can produce). We will use $y < 5$ as the domain of $g^{-1}$ in part (b).

Sense check. $g$ never reaches $5$ (horizontal asymptote), so $y = 5$ is excluded. $g$ goes to $-\infty$ near $x = -2$, so all values below $5$ are covered.

$\boxed{\text{Range: } y < 5}$ — this becomes the domain of $g^{-1}$.

(b) Find $g^{-1}(x)$ and state its domain [4 marks] · 5 steps

To find $g^{-1}$: set $y = \frac{5x+3}{x+2}$ and solve for $x$ in terms of $y$. Then swap $x \leftrightarrow y$ to get $g^{-1}(x)$. The domain of $g^{-1}$ = range of $g$ = $y < 5$ (from part (a)).
STEP 1/5set y = g(x) and cross-multiply

Strategy. Write $y = \frac{5x+3}{x+2}$ and multiply both sides by $(x + 2)$ to clear the denominator.

Cross-multiply

$$y(x + 2) = 5x + 3.$$ $$xy + 2y = 5x + 3.$$

Sense check. The denominator $x + 2 \neq 0$ on the domain $x > -2$, so multiplying is safe (no division by zero).

$xy + 2y = 5x + 3$.
↳ from STEP 1: $xy + 2y = 5x + 3$.
STEP 2/5collect x-terms

Strategy. Move all $x$-terms to one side, all non-$x$ terms to the other. $xy - 5x = 3 - 2y$.

Rearrange

$$xy - 5x = 3 - 2y.$$

Factor out $x$

$$x(y - 5) = 3 - 2y.$$

Sense check. $xy - 5x = x(y - 5)$. The factor $(y - 5)$ is the key — it is non-zero because $y < 5$ (the range), so we can divide.

$x(y - 5) = 3 - 2y$.
↳ from STEP 2: $x(y - 5) = 3 - 2y$.
STEP 3/5solve for x

Strategy. Divide both sides by $(y - 5)$ to isolate $x$. The condition $y \neq 5$ is guaranteed by the range $y < 5$.

Divide

$$x = \frac{3 - 2y}{y - 5}.$$

Sense check. Numerator $3 - 2y$; denominator $y - 5$. At $y = 0$: $x = \frac{3}{-5} = -\frac{3}{5}$. Check: $g(-\frac{3}{5}) = \frac{5(-\frac{3}{5}) + 3}{-\frac{3}{5} + 2} = \frac{-3 + 3}{\frac{7}{5}} = 0$. ✓

$x = \dfrac{3 - 2y}{y - 5}$.
↳ from STEP 3: $x = \dfrac{3 - 2y}{y - 5}$.
STEP 4/5swap to get g⁻¹(x)

Strategy. Swap $x$ and $y$ to write the inverse function in standard notation $g^{-1}(x)$. This gives $g^{-1}(x) = \frac{3 - 2x}{x - 5}$.

Swap $x \leftrightarrow y$

$$g^{-1}(x) = \frac{3 - 2x}{x - 5}.$$
Trap 2 (watch-list) — composite $fg$ vs $gf$. Last week on Q3d Phuc ended up with $\ln$ on both sides because $f$ and $g$ were swapped. Read $fg(x)$ as "$f$ of $g$ of $x$": the rightmost function acts first. When finding an inverse, there is no composition — just solve and swap — but the same "which function acts first" discipline applies.

Sense check. Verify $g(g^{-1}(x)) = x$: $g\!\left(\frac{3-2x}{x-5}\right) = \frac{5\cdot\frac{3-2x}{x-5} + 3}{\frac{3-2x}{x-5} + 2}$. Multiply top and bottom by $(x - 5)$: $\frac{5(3-2x) + 3(x-5)}{(3-2x) + 2(x-5)} = \frac{15 - 10x + 3x - 15}{3 - 2x + 2x - 10} = \frac{-7x}{-7} = x$. ✓

$g^{-1}(x) = \dfrac{3 - 2x}{x - 5}$.
↳ from STEP 4: $g^{-1}(x) = \dfrac{3 - 2x}{x - 5}$.
STEP 5/5state the domain of g⁻¹

Strategy. The domain of $g^{-1}$ = the range of $g$ = $y < 5$ (from part a). Restate using $x$: domain of $g^{-1}$ is $x < 5$. This is worth one mark — do not forget to state it.

Domain of $g^{-1}$

$$\text{Domain of } g^{-1}: \quad x < 5.$$
Trap 3 (watch-list) — forgetting the domain of $g^{-1}$. Question phrasing is sneaky; the domain is marked anyway. Whenever you write an inverse function, the very next line is its domain (= range of the original). Phuc lost this mark before — always write it.
Trap 4 (watch-list) — x vs y wording. Domains use $x$: "domain: $x < 5$". Ranges use $y$: "range: $y < 5$". Mixing them loses the mark even if the number $5$ is correct.

Sense check. The denominator of $g^{-1}(x) = \frac{3-2x}{x-5}$ is zero at $x = 5$, which is excluded by $x < 5$. ✓ The range of $g$ ($y < 5$) becomes the domain of $g^{-1}$ ($x < 5$) — the swap is consistent.

$\boxed{g^{-1}(x) = \dfrac{3 - 2x}{x - 5},\;\;\text{domain } x < 5}$.
