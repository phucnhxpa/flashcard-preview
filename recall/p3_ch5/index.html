P3 Ch 5 — Exponentials and Logarithms

The whole chapter in one mental model. Exponential functions $a^x$ are the unique family where the rate of change is proportional to the value ($\mathrm{f}'(x)\propto\mathrm{f}(x)$). The constant $\mathrm{e}\approx 2.718$ is special: $\dfrac{\mathrm d}{\mathrm dx}\mathrm{e}^{x}=\mathrm{e}^x$ exactly, and more generally $\dfrac{\mathrm d}{\mathrm dx}\mathrm{e}^{kx}=k\mathrm{e}^{kx}$. The natural logarithm $\ln x=\log_\mathrm{e} x$ is the inverse of $\mathrm{e}^x$. Logarithm linearisation turns power laws ($y=ax^n$) and exponential laws ($y=ab^x$) into straight-line graphs ($\log y$ vs $\log x$ or $x$). This unlocks exponential modelling: anything obeying "rate ∝ amount" — radioactive decay, population growth, cooling, drug clearance — is governed by $y=A\mathrm{e}^{kt}$.

1. Jargon glossary

Term Meaning Where
Exponential function $\mathrm{f}(x)=a^x$ where the base $a$ is constant and the exponent $x$ is variable. Contrast with power function $x^n$ (variable base, constant exponent). All exponentials pass through $(0,1)$ since $a^0=1$. 5.1
Base $a$ The constant being raised to a power. If $a>1$: increasing function (grows without limit). If $0 5.1
Index / power / exponent All synonyms for the variable $x$ in $a^x$. The same number; different name in different contexts (combinatorics often "index", physics often "exponent"). 5.1
The constant $\mathrm{e}$ $\mathrm{e}\approx 2.71828\dots$ — irrational, transcendental. Defined by the property that $\dfrac{\mathrm d}{\mathrm dx}\mathrm{e}^x=\mathrm{e}^x$, i.e. the gradient of $y=\mathrm{e}^x$ equals $y$ itself. Sits between $a=2$ ($\mathrm f'(x)\approx 0.693\cdot 2^x$) and $a=3$ ($\mathrm f'(x)\approx 1.099\cdot 3^x$). 5.2
$\mathrm{e}^x$ The exponential function with base $\mathrm{e}$ — the unique exponential function whose derivative equals itself. Sometimes called "the" exponential function. Always positive, asymptote $y=0$ as $x\to-\infty$. 5.2
Natural logarithm $\ln x$ $\ln x=\log_\mathrm{e}x$ — the inverse function of $\mathrm{e}^x$. Defined only for $x>0$. Passes through $(1,0)$. Asymptote: $y$-axis. Grows slowly without bound as $x\to\infty$. 5.3
Logarithm laws $\log(MN)=\log M+\log N$, $\log(M/N)=\log M-\log N$, $\log(M^k)=k\log M$, $\log_a a=1$, $\log_a 1=0$. Same for $\ln$. The inverse pair: $\mathrm{e}^{\ln x}=x$ ($x>0$) and $\ln(\mathrm{e}^x)=x$. 5.3
Asymptote A line the graph approaches but never reaches. For $y=\mathrm{e}^{ax+b}+c$: horizontal asymptote $y=c$. For $y=\ln(x-d)$: vertical asymptote $x=d$. 5.2
Differentiating $\mathrm{e}^{kx}$ $\dfrac{\mathrm d}{\mathrm dx}\mathrm{e}^{kx}=k\mathrm{e}^{kx}$. Generalises to any constant multiplier: $\dfrac{\mathrm d}{\mathrm dx}A\mathrm{e}^{kx}=Ak\mathrm{e}^{kx}$. 5.2
Quadratic-in-$\mathrm{e}^x$ Equations like $\mathrm{e}^{2x}+5\mathrm{e}^x-14=0$ — substitute $u=\mathrm{e}^x$ to get $u^2+5u-14=0$. Solve, then convert each $u$ back via $x=\ln u$. Discard $u\le 0$ (since $\mathrm{e}^x>0$). 5.3 Ex 6
Quadratic-in-$\ln x$ Equations like $(\ln x)^2+2\ln x-15=0$ — substitute $u=\ln x$, solve quadratic, convert back via $x=\mathrm{e}^u$. 5.3 Ex 3c
Linearising data If $y=ax^n$ then $\log y=\log a+n\log x$ — plot $\log y$ vs $\log x$, get a straight line; gradient $n$, $y$-intercept $\log a$. If $y=ab^x$ then $\log y=\log a+x\log b$ — plot $\log y$ vs $x$ (NOT $\log x$). 5.4
Power law A relation $y=ax^n$ between variables. On a log-log plot it's a straight line. Examples: Kepler's 3rd law, Kleiber's law for metabolic rate. 5.4
Exponential law A relation $y=ab^x$ between variables. On a semi-log plot ($\log y$ vs $x$) it's a straight line. Examples: bacterial growth, radioactive decay, compound interest. 5.4
Initial value The value of a model when $t=0$. For $y=A\mathrm{e}^{kt}$: $y(0)=A$. So the constant $A$ is always the starting amount. 5.5
Growth / decay rate $k$ In $y=A\mathrm{e}^{kt}$: if $k>0$ exponential growth; if $k<0$ exponential decay. The instantaneous rate is $\dfrac{\mathrm dy}{\mathrm dt}=ky$ — the defining property of exponential models. Magnitude $\lvert k\rvert$ controls speed. 5.5
Half-life / doubling time For decay: time $t_{1/2}$ for value to halve, $\mathrm{e}^{kt_{1/2}}=\tfrac12\Rightarrow t_{1/2}=\dfrac{-\ln 2}{k}$ (k<0). For growth: doubling time $t_{2}=\dfrac{\ln 2}{k}$ (k>0). 5.5
Reflection in $y=x$ Geometric statement of inverse-function relationship: graphs of $y=\mathrm{e}^x$ and $y=\ln x$ are reflections of each other across the line $y=x$. 5.3

2. Graph atlas — what each shape looks like

Function Shape Key features
$y=a^x$, $a>1$ Increasing through $(0,1)$ Asymptote $y=0$ as $x\to-\infty$; grows fast for large $x$
$y=a^x$, $0 Decreasing through $(0,1)$ Asymptote $y=0$ as $x\to+\infty$; same as $y=(1/a)^{-x}$
$y=\mathrm{e}^x$ Through $(0,1)$, gradient $=1$ at $x=0$ Self-derivative property
$y=\mathrm{e}^{-x}$ Reflection of $y=\mathrm{e}^x$ in $y$-axis Decay shape
$y=A\mathrm{e}^{kx}+c$ Same shape as $\mathrm{e}^{kx}$, scaled by $A$, shifted up by $c$ Asymptote $y=c$; $y$-intercept $A+c$
$y=\ln x$ Through $(1,0)$, slow growth Asymptote $x=0$ ($y$-axis); only defined for $x>0$
$y=\ln(x-d)+c$ Translation of $\ln x$ Asymptote $x=d$; passes through $(d+1, c)$

Sketching $y=A\mathrm{e}^{kx}+c$:

  1. Sketch asymptote $y=c$ first (dashed).
  2. Find $y$-intercept: $y(0)=A+c$.
  3. Determine direction: $k>0$ → grows away from asymptote; $k<0$ → decays toward asymptote.
  4. Sign of $A$ flips above/below asymptote.

3. Differentiating exponentials — rules

Function Derivative
$\mathrm{e}^x$ $\mathrm{e}^x$
$\mathrm{e}^{kx}$ $k\mathrm{e}^{kx}$
$A\mathrm{e}^{kx}$ $kA\mathrm{e}^{kx}$
$A\mathrm{e}^{kx}+c$ $kA\mathrm{e}^{kx}$ (constant disappears)
$\mathrm{e}^{kx+b}$ $k\mathrm{e}^{kx+b}$
$\mathrm{e}^{f(x)}$ (chain rule) $\mathrm f'(x)\mathrm{e}^{f(x)}$ — needed for P3 Ch6

Why $\mathrm{e}^x$ is special: the exponential is the unique function (up to scalar) satisfying $\mathrm{f}'=\mathrm{f}$. This is the defining property of $\mathrm{e}$.

4. Solving equations: exponentials and logarithms

Type Method Example
$\mathrm{e}^x=k$ ($k>0$) Take $\ln$: $x=\ln k$ $\mathrm{e}^x=5\Rightarrow x=\ln 5$
$\mathrm{e}^{ax+b}=k$ $ax+b=\ln k$, then solve linearly $\mathrm{e}^{2x+3}=7\Rightarrow x=\tfrac{\ln 7-3}{2}$
$\ln x=k$ Exponentiate: $x=\mathrm{e}^k$ $\ln x=3\Rightarrow x=\mathrm{e}^3$
$\ln(f(x))=k$ $f(x)=\mathrm{e}^k$, then solve $\ln(2x+3)=4\Rightarrow x=\tfrac{\mathrm{e}^4-3}{2}$
$a\,\mathrm{e}^{2x}+b\,\mathrm{e}^x+c=0$ Substitute $u=\mathrm{e}^x$ → solve quadratic in $u$ → take $\ln$. Reject $u\le 0$ $\mathrm{e}^{2x}+5\mathrm{e}^x-14=0$: $u=2$ or $u=-7$ (reject). $x=\ln 2$
$a(\ln x)^2+b\ln x+c=0$ Substitute $u=\ln x$ → solve quadratic → $x=\mathrm{e}^u$. (No rejection needed: any $u$ valid) $(\ln x)^2+2\ln x-15=0$: $u=3$ or $u=-5$. $x=\mathrm{e}^3$ or $\mathrm{e}^{-5}$
$\mathrm{e}^x+k\mathrm{e}^{-x}=c$ Multiply through by $\mathrm{e}^x$: $\mathrm{e}^{2x}-c\mathrm{e}^x+k=0$, quadratic in $\mathrm{e}^x$ as above
$A^x B^x = C$ Take $\ln$ of both sides: $x\ln A+x\ln B=\ln C$, so $x=\ln C/(\ln A+\ln B)$ $3^x\mathrm{e}^{4x-1}=5$: $\ln 5=x\ln 3+4x-1$, solve

5. Linearising non-linear data

If you suspect a power law or exponential law, take logs to get a straight line and read off constants from gradient/intercept:

Suspected model Take logs to get Plot Gradient Vertical intercept
$y=ax^n$ (power law) $\log y=\log a+n\log x$ $\log y$ vs $\log x$ (log-log) $n$ $\log a$ → $a=10^{\text{intercept}}$
$y=ab^x$ (exponential law) $\log y=\log a+x\log b$ $\log y$ vs $x$ (semi-log) $\log b$ → $b=10^{\text{gradient}}$ $\log a$ → $a=10^{\text{intercept}}$
$y=a\mathrm{e}^{kx}$ $\ln y=\ln a+kx$ $\ln y$ vs $x$ $k$ $\ln a$ → $a=\mathrm{e}^{\text{intercept}}$

Workflow:

  1. Suspect a model form from theory or data shape.
  2. Take logs (base 10 or $\mathrm{e}$ — whichever the question dictates).
  3. Tabulate $(\log x,\log y)$ or $(x,\log y)$.
  4. Plot, draw line of best fit.
  5. Read gradient → exponent $n$ or $\log b$/$k$.
  6. Read $y$-intercept → $\log a$ → $a=10^{(\text{intercept})}$ or $\mathrm{e}^{(\text{intercept})}$.

WARNING: for $y=ab^x$, plot $\log y$ vs $x$ (NOT $\log x$). For $y=ax^n$, plot $\log y$ vs $\log x$.

6. Exponential modelling — anatomy of $y=A\mathrm{e}^{kt}+c$

Parameter Meaning How to read it
$A$ Initial offset above asymptote $y(0)=A+c$, so $A=y(0)-c$
$c$ Long-term limit (asymptote) $\lim_{t\to\infty}y=c$ if $k<0$; $-\infty$ if $k>0,A<0$, etc.
$k$ Growth/decay rate $\dfrac{\mathrm dy}{\mathrm dt}=k(y-c)$ — rate proportional to deviation from limit
$\dfrac{\mathrm dy}{\mathrm dt}=k(y-c)$ Rate equation Source of the exponential form
$A>0,k>0$ Growth (e.g. population, interest) $y$ grows without bound
$A>0,k<0$ Decay (e.g. radioactive, cooling) $y\to c$ from above
Half-life $t_{1/2}=\dfrac{-\ln 2}{k}$ ($k<0$) Solve $\mathrm{e}^{kt}=\tfrac12$
Doubling time $t_2=\dfrac{\ln 2}{k}$ ($k>0$) Solve $\mathrm{e}^{kt}=2$

Examples in textbook:

  • Pesticide density: $P=160\mathrm{e}^{-0.006t}$ — $A=160$ initial, $k=-0.006$ slow decay.
  • Radioactive half-life, drug clearance, bacteria growth.
  • $\dfrac{\mathrm dP}{\mathrm dt}=kP$ shown by computing derivative.

8. Common exam pitfalls

Pitfall Fix
Treating $\log(a+b)=\log a+\log b$ WRONG — only $\log(ab)=\log a+\log b$. Sums become products under log only when via the multiplication law of exponents
Forgetting that $\mathrm{e}^x>0$ always When solving quadratic in $\mathrm{e}^x$, discard any non-positive roots
Forgetting $\ln x$ defined only for $x>0$ If algebra gives $\ln(\text{negative})$, that branch is rejected
For $y=ab^x$ data, plotting $\log y$ vs $\log x$ instead of $x$ Power law uses $\log y$ vs $\log x$; exponential law uses $\log y$ vs $x$
Assuming $a$ is gradient when actually it's the intercept Re-derive each time: $\log y=\log a+\dots\Rightarrow$ intercept is $\log a$
For $\log a=2.6$, writing $a=2.6$ Need $a=10^{2.6}$ (or $\mathrm{e}^{2.6}$ if natural log)
Differentiating $\mathrm{e}^{kx}$ as $\mathrm{e}^{kx}$ (forgetting the chain factor $k$) $\dfrac{\mathrm d}{\mathrm dx}\mathrm{e}^{kx}=k\mathrm{e}^{kx}$, always
In modelling problems, swapping $A$ and $A+c$ — confusing initial value with asymptote $y(0)$ = $A+c$; long-term limit = $c$; "deviation from asymptote" = $A$
$\dfrac{\mathrm dy}{\mathrm dt}=ky\Rightarrow y=A\mathrm{e}^{kt}$ — losing the $A$ Constant of integration always present — sets the initial condition
Solving $\mathrm{e}^{2x}=k$ as $x=\tfrac12\ln k$ but writing $\ln(\tfrac12 k)$ $2x=\ln k$, so $x=\dfrac{\ln k}{2}$ — NOT $\ln(k/2)$
Confusing $\ln(x^k)=k\ln x$ with $(\ln x)^k$ Very different. $(\ln 4)^2\ne\ln(16)$
Forgetting that $\log_a 1=0$ and $\log_a a=1$ for any base Useful for spot-checking
Treating $y=\mathrm{e}^{-x}$ as decreasing because of negative sign on $x$, but mistakenly assuming asymptote on left Asymptote is $y=0$; as $x\to+\infty$, $y\to 0$ from above

9. Big picture (30-second mental model)

This chapter has three pillars:

  1. The function $\mathrm{e}^x$ and its inverse $\ln x$. $\mathrm{e}\approx 2.718$ is the unique base where $(\mathrm{e}^x)'=\mathrm{e}^x$. More generally $\dfrac{\mathrm d}{\mathrm dx}\mathrm{e}^{kx}=k\mathrm{e}^{kx}$. This makes $\mathrm{e}^x$ the natural language for any process where rate is proportional to amount.

  2. Solving exponential & logarithmic equations. Take $\ln$ to undo $\mathrm{e}$; exponentiate to undo $\ln$. Quadratic-in-$\mathrm{e}^x$ via $u=\mathrm{e}^x$. Quadratic-in-$\ln x$ via $u=\ln x$. Discard $\mathrm{e}^x\le 0$ branches.

  3. Linearising data. Power law $y=ax^n$ → straight line on log-log. Exponential law $y=ab^x$ → straight line on semi-log. Read parameters off gradient and intercept.

  4. Modelling. Anything obeying $\dfrac{\mathrm dy}{\mathrm dt}=ky$ has solution $y=A\mathrm{e}^{kt}$ (initial value $A$, rate $k$). Add a long-term constant $c$ if asymptote ≠ 0. Half-life and doubling-time formulae from $\ln 2 / |k|$.

The chapter is the bridge to calculus of $\mathrm{e}^x$ and $\ln x$ (P3 Ch6) and exponential differential equations (FP). Master the five rules — sketch, differentiate, solve, linearise, model — and the rest is bookkeeping.
