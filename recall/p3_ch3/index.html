P3 Ch 3 — Exponentials and Logarithms | Recall Sheet

P3 · Ch 3 · Exam: 2026-06-02 · Recall sheet for spaced repetition

The whole chapter in one mental model. Exponentials and logarithms are inverses of each other: $y = a^x \Leftrightarrow x = \log_a y$. The chapter has three modes: (1) solving equations — using log laws and substitution to reduce to linear or quadratic form; (2) modelling — converting $y = Ab^x$ into $\ln y = \ln A + x\ln b$ to get a straight line on a log-linear graph; (3) differentiating/integrating — the special properties of $e^x$ and $\ln x$. The unifying thread: every exponential problem either becomes linear via logs, or quadratic via substitution.

1. Jargon glossary

Term Definition Exam note
Exponential function $f(x) = a^x$, $a > 0$, $a \neq 1$ Always positive: $a^x > 0$ for all $x$. Never touches x-axis.
Natural exponential $e^x$ where $e \approx 2.718$ Special: derivative of $e^x$ is $e^x$ (unchanged). Appears constantly in P3 calculus.
Logarithm $\log_a x$ = power to which $a$ must be raised to get $x$ $\log_a(a^x) = x$ and $a^{\log_a x} = x$ — these are the core inverse relationships.
Natural log $\ln x = \log_e x$ Only log you need for calculus in P3. $\dfrac{d}{dx}\ln x = \dfrac{1}{x}$.
Log laws Rules for manipulating logarithms Product: $\log(ab) = \log a + \log b$. Quotient: $\log(a/b) = \log a - \log b$. Power: $\log(a^n) = n\log a$.
Change of base $\log_a x = \dfrac{\ln x}{\ln a}$ Used when bases differ. In IAL: convert everything to $\ln$.
Exponential model $y = Ab^x$ or $y = Ae^{kx}$ Linearise: $\ln y = \ln A + kx$ gives straight line with gradient $k$, intercept $\ln A$.
Hidden quadratic An equation like $e^{2x} - 5e^x + 6 = 0$ that becomes quadratic when you let $u = e^x$ Trigger: see $a^{2x}$ and $a^x$ in same equation.

2. Log laws — complete reference table

Law Formula Example Pitfall
Product rule $\log_a(mn) = \log_a m + \log_a n$ $\ln(3x) = \ln 3 + \ln x$ $\ln(m + n) \neq \ln m + \ln n$ — no rule for sums
Quotient rule $\log_a(m/n) = \log_a m - \log_a n$ $\ln(x/5) = \ln x - \ln 5$ $\ln(m - n) \neq \ln m - \ln n$
Power rule $\log_a(m^n) = n\log_a m$ $\ln(x^3) = 3\ln x$ $\ln(3x^2) \neq 2\ln(3x)$ — apply to $x^2$ only, not $3x^2$ as a whole
Inverse $a^{\log_a x} = x$ and $\log_a(a^x) = x$ $e^{\ln 7} = 7$, $\ln(e^{5x}) = 5x$ These cancel each other — most useful simplification in the chapter
Change of base $\log_a b = \dfrac{\ln b}{\ln a}$ $\log_3 8 = \dfrac{\ln 8}{\ln 3}$ Always convert to $\ln$ for IAL questions
Log of 1 $\log_a 1 = 0$ $\ln 1 = 0$ Common substitution: when $x=0$, $\ln(\text{something}) = 0$ means "something" $= 1$
Log of base $\log_a a = 1$ $\ln e = 1$ Used constantly in exponential model linearisation

3. The 6 question types

Type How to spot it Approach Key formula Worked example Pitfall
1. Hidden quadratic substitution $a^{2x}$ and $a^x$ in same equation, or $e^{2x} - ke^x + c = 0$ Let $u = a^x$ (or $u = e^x$). Write equation in $u$. Factorise or use quadratic formula. Solve for $u$. Back-substitute: $a^x = u \Rightarrow x = \dfrac{\ln u}{\ln a}$. Reject negative $u$ (since $a^x > 0$). $a^{2x} = (a^x)^2$ $e^{2x} - 5e^x + 6 = 0$. Let $u=e^x$: $u^2-5u+6=0 \Rightarrow (u-2)(u-3)=0 \Rightarrow u=2$ or $u=3 \Rightarrow x=\ln 2$ or $x=\ln 3$ Don't forget to back-substitute. Can't take $\ln$ of a negative — always check $u > 0$.
2. Solve log equation (single base) Equation with $\ln$ or $\log$ terms; combine to single log then exponentiate Use log laws to combine all log terms into ONE log expression. Then exponentiate both sides: $\ln f(x) = c \Rightarrow f(x) = e^c$. Solve and check domain ($\ln$ undefined for $x \leq 0$). $\ln A + \ln B = \ln(AB)$, then $e^{\ln X} = X$ $\ln(x+1) + \ln(x-2) = \ln(2x+4)$. LHS $= \ln((x+1)(x-2))$. So $(x+1)(x-2) = 2x+4 \Rightarrow x^2-x-2=2x+4 \Rightarrow x^2-3x-6=0$. Check both roots are in domain. Must check domain — reject any solution where an argument of $\ln$ is $\leq 0$.
3. Exponential equation, different bases $a^{f(x)} = b^{g(x)}$ where $a \neq b$ (e.g. $3^{2x+1} = 5^{x-1}$) Take $\ln$ of both sides. Apply power rule: $(2x+1)\ln 3 = (x-1)\ln 5$. Expand, collect $x$ terms, solve linearly. $\ln(a^n) = n\ln a$ $3^{2x+1} = 5^{x-1}$. $(2x+1)\ln 3 = (x-1)\ln 5$. $2x\ln 3 + \ln 3 = x\ln 5 - \ln 5$. $x(2\ln 3 - \ln 5) = -\ln 5 - \ln 3$. $x = \dfrac{-\ln 15}{2\ln 3 - \ln 5}$. Bracket the whole exponent before applying power rule: $(2x+1)\ln 3$, NOT $2x\ln 3 + 1$.
4. Logarithmic model — linearisation Table of $(x, y)$ values from exponential model; asked to plot $\ln y$ against $x$, find $A$ and $b$ in $y = Ab^x$ Take $\ln$ of both sides of $y = Ab^x$: $\ln y = \ln A + x\ln b$. This is $Y = mX + c$ where $Y=\ln y$, $m=\ln b$, $c=\ln A$. Read gradient $= \ln b$, y-intercept $= \ln A$. Back-calculate: $b = e^m$, $A = e^c$. $y = Ab^x \Rightarrow \ln y = \ln A + (\ln b)x$ Graph of $\ln y$ vs $x$ has gradient $0.3$ and $y$-intercept $1.6$. So $\ln b = 0.3 \Rightarrow b = e^{0.3} \approx 1.35$. $\ln A = 1.6 \Rightarrow A = e^{1.6} \approx 4.95$. Gradient is $\ln b$ not $b$. Intercept is $\ln A$ not $A$. Always decode via $e^{\text{value}}$.
5. Convert log model to exponential form Given $\ln y = ax + b$ (or from graph), find $y$ in terms of $x$ explicitly Exponentiate both sides: $y = e^{ax+b} = e^b \cdot e^{ax}$. Write as $y = Ae^{ax}$ where $A = e^b$. $e^{ax+b} = e^b \cdot e^{ax}$ $\ln y = 2x + 3 \Rightarrow y = e^{2x+3} = e^3 \cdot e^{2x} \approx 20.1 \cdot e^{2x}$ $e^{ax+b} \neq e^{ax} + e^b$ — exponent rules, not addition.
6. Differentiate/integrate $e^x$ and $\ln x$ Asked to differentiate/integrate expressions involving $e^{f(x)}$ or $\ln(f(x))$ Differentiation: $\dfrac{d}{dx}e^{f(x)} = f'(x)e^{f(x)}$ (chain rule). $\dfrac{d}{dx}\ln(f(x)) = \dfrac{f'(x)}{f(x)}$. Integration: $\int e^{ax+b}\,dx = \dfrac{1}{a}e^{ax+b}+c$. $\int \dfrac{1}{x}\,dx = \ln|x|+c$. $\dfrac{d}{dx}e^x = e^x$. $\dfrac{d}{dx}\ln x = \dfrac{1}{x}$. $\displaystyle\int\dfrac{f'(x)}{f(x)}\,dx = \ln|f(x)|+c$. $\dfrac{d}{dx}(e^{3x^2}) = 6x \cdot e^{3x^2}$. $\displaystyle\int\dfrac{2x}{x^2+1}\,dx = \ln|x^2+1|+c$ (numerator is derivative of denominator). $\int e^{x^2}\,dx$ cannot be done by standard methods — only $e^{ax+b}$ integrates cleanly. For $\int\frac{f'}{f}$: numerator must be EXACTLY the derivative of denominator.

4. Key calculus formulas

Expression Derivative Integral Note
$e^x$ $e^x$ $e^x + c$ Unchanged by differentiation — unique property
$e^{ax+b}$ $a\,e^{ax+b}$ $\dfrac{1}{a}e^{ax+b}+c$ Chain rule: multiply by inner derivative $a$
$e^{f(x)}$ $f'(x)\,e^{f(x)}$ No simple form unless $f(x)=ax+b$ Always use chain rule
$\ln x$ $\dfrac{1}{x}$ $x\ln x - x + c$ Integral by parts; in practice just differentiate
$\ln(ax+b)$ $\dfrac{a}{ax+b}$ $\dfrac{1}{a}\ln|ax+b|+c$ Chain rule on differentiation; reverse chain rule on integration
$\ln(f(x))$ $\dfrac{f'(x)}{f(x)}$ $\ln|f(x)|+c$ only if numerator $= f'(x)$ Spot: if integrand looks like $\dfrac{\text{derivative}}{\text{function}}$
$a^x$ $a^x \ln a$ $\dfrac{a^x}{\ln a}+c$ Rarely tested but formula booklet has it

5. Common pitfalls

Mistake Correct approach
$\ln(m+n) = \ln m + \ln n$ FALSE. No log rule for sums. Use $\ln(mn) = \ln m + \ln n$ only for PRODUCTS.
$\ln(x^2) = 2\ln x$ for all $x$ Only if $x > 0$. For any $x$: $\ln(x^2) = 2\ln|x|$.
Accepting $e^x = -3$ as a solution IMPOSSIBLE — $e^x > 0$ always. Reject negative values of $e^x$.
$(2x+1)\ln 3$ becomes $2x\ln 3 + 1$ Bracket error: it's $2x\ln 3 + \ln 3$, not $+1$.
Forgetting to check domain after solving log equation $\ln(f(x))$ requires $f(x) > 0$. Always substitute back and reject invalid roots.
$\int e^{x^2}\,dx = \dfrac{1}{2x}e^{x^2}$ Wrong. $e^{x^2}$ has no elementary integral. Only $e^{\text{linear in }x}$ integrates cleanly.
Gradient of $\ln y$ vs $x$ graph $= b$ (in $y=Ab^x$) Gradient $= \ln b$, not $b$. Must exponentiate: $b = e^{\text{gradient}}$.

6. Big picture — 30-second mental model

  1. Every exponential equation becomes linear or quadratic via logs. If you see $e^{2x}$ and $e^x$, substitute $u = e^x$. If you see $a^{f(x)} = b^{g(x)}$, take $\ln$ of both sides.
  2. Log laws are algebra tools. Use them to combine multiple log terms into one, then exponentiate once to solve.
  3. Linearisation is the modelling recipe. $y = Ab^x \Rightarrow \ln y = (\ln b)x + \ln A$. Always decode back: gradient is $\ln b$ (not $b$), intercept is $\ln A$ (not $A$).
  4. $e^x$ is the only function equal to its own derivative. This makes calculus with $e^x$ clean — the chain rule just multiplies by the inner derivative.
  5. $\ln|f(x)| + c$ is the integral of $\dfrac{f'(x)}{f(x)}$. Recognise when the numerator is (a multiple of) the derivative of the denominator.
  6. Domain check is mandatory whenever you solve a log equation. Reject any solution making an argument of $\ln$ non-positive.
