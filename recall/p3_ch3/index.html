P3 Ch 3 — Trigonometric Functions | Recall Sheet

P3 · Ch 3 · Exam: 2026-06-02 · Recall sheet for spaced repetition

The whole chapter in one mental model. Three new trig functions: $\sec x = 1/\cos x$, $\cosec x = 1/\sin x$, $\cot x = \cos x/\sin x$. Three new Pythagorean identities: start from $\sin^2x+\cos^2x\equiv 1$ and divide by $\cos^2x$ or $\sin^2x$ to get the sec/tan and cosec/cot forms. Three inverse functions: $\arcsin$, $\arccos$, $\arctan$ — each defined on a restricted domain to produce a unique output. The unifying skill: replace awkward sec/cosec/cot expressions using identities, then solve standard trig equations.

1. Jargon glossary

Term Definition Exam note
sec x $\sec x = \dfrac{1}{\cos x}$ (reciprocal of cosine) Undefined when $\cos x = 0$. Range: $|\sec x| \ge 1$.
cosec x $\cosec x = \dfrac{1}{\sin x}$ (reciprocal of sine) Undefined when $\sin x = 0$. Range: $|\cosec x| \ge 1$.
cot x $\cot x = \dfrac{\cos x}{\sin x} = \dfrac{1}{\tan x}$ Undefined when $\sin x = 0$. All real numbers in range.
arcsin x Inverse of sin (restricted to $[-\pi/2,\, \pi/2]$). Output always in $[-\pi/2,\, \pi/2]$. Domain: $[-1, 1]$. $\sin^{-1}$ notation means INVERSE, not reciprocal.
arccos x Inverse of cos (restricted to $[0,\, \pi]$). Output always in $[0,\, \pi]$. Domain: $[-1, 1]$. Output never negative.
arctan x Inverse of tan (restricted to $(-\pi/2,\, \pi/2)$). Output always in $(-\pi/2,\, \pi/2)$. Domain: all reals. Output never reaches $\pm\pi/2$.
Pythagorean identity An identity derived from $\sin^2x+\cos^2x\equiv 1$ by dividing by $\sin^2x$ or $\cos^2x$. $\sec^2x\equiv 1+\tan^2x$ and $\cosec^2x\equiv 1+\cot^2x$ are both NOT in the formula booklet — must memorise.

2. Pythagorean identities (⚠️ NOT in formula booklet — memorise all three)

Base: $\sin^2x + \cos^2x \equiv 1$.
Divide both sides by $\cos^2x$ → $\tan^2x + 1 \equiv \sec^2x$.
Divide both sides by $\sin^2x$ → $1 + \cot^2x \equiv \cosec^2x$.
These three identities and their rearrangements cover every identity question in this chapter.
Identity Derived by Rearrangements Use when
$\sin^2x + \cos^2x \equiv 1$ Fundamental $\sin^2x \equiv 1-\cos^2x$; $\cos^2x \equiv 1-\sin^2x$ Both sin and cos present
$\sec^2x \equiv 1 + \tan^2x$ Divide by $\cos^2x$ $\tan^2x \equiv \sec^2x - 1$ sec and tan present together
$\cosec^2x \equiv 1 + \cot^2x$ Divide by $\sin^2x$ $\cot^2x \equiv \cosec^2x - 1$ cosec and cot present together

3. Reciprocal functions — graphs and properties

Function Definition Undefined at Range Period Asymptotes
sec x $1/\cos x$ $x = \pm\pi/2,\, \pm3\pi/2,\, \ldots$ (where $\cos x = 0$) $(-\infty,-1] \cup [1,\infty)$ $2\pi$ Vertical lines at $x = (2k+1)\pi/2$
cosec x $1/\sin x$ $x = 0,\, \pm\pi,\, \pm2\pi,\, \ldots$ (where $\sin x = 0$) $(-\infty,-1] \cup [1,\infty)$ $2\pi$ Vertical lines at $x = k\pi$
cot x $\cos x / \sin x$ $x = 0,\, \pm\pi,\, \pm2\pi,\, \ldots$ (where $\sin x = 0$) All real numbers $\pi$ Vertical lines at $x = k\pi$
  • sec x graph: U-shapes above x-axis (at cos x peaks) and ∩-shapes below (at cos x troughs). Passes through $(0,1)$.
  • cosec x graph: similar shape but shifted — U-shapes above and ∩-shapes below, passing through $(\pi/2, 1)$.
  • cot x graph: decreasing curves between each pair of asymptotes (opposite gradient direction to tan x).
  • All three are undefined at points where the denominator equals zero — NEVER evaluate at those points.

4. Inverse trig functions

An inverse function only exists if the original is one-to-one. We RESTRICT the domain of sin/cos/tan to make them one-to-one, then define the inverse on that restricted domain. The restricted domain becomes the RANGE of the inverse.
Function Restricted domain of original Output range Key exact values Graph feature
arcsin x sin restricted to $[-\pi/2,\, \pi/2]$ $[-\pi/2,\, \pi/2]$ $\arcsin(0)=0$, $\arcsin(\tfrac{1}{2})=\tfrac{\pi}{6}$, $\arcsin(1)=\tfrac{\pi}{2}$, $\arcsin(-\tfrac{1}{2})=-\tfrac{\pi}{6}$ Increasing curve through origin; reflection of sin in $y=x$
arccos x cos restricted to $[0,\, \pi]$ $[0,\, \pi]$ $\arccos(1)=0$, $\arccos(\tfrac{1}{2})=\tfrac{\pi}{3}$, $\arccos(0)=\tfrac{\pi}{2}$, $\arccos(-\tfrac{1}{2})=\tfrac{2\pi}{3}$ Decreasing curve; reflection of cos (restricted) in $y=x$
arctan x tan restricted to $(-\pi/2,\, \pi/2)$ $(-\pi/2,\, \pi/2)$ $\arctan(0)=0$, $\arctan(1)=\tfrac{\pi}{4}$, $\arctan(\sqrt{3})=\tfrac{\pi}{3}$, $\arctan(-1)=-\tfrac{\pi}{4}$ Increasing S-curve with horizontal asymptotes at $\pm\pi/2$

5. The 5 question types — approaches table

Type How to spot Step-by-step approach Example Pitfall
1. Simplify using identity $\sec^2$, $\cosec^2$, $\cot^2$, $\tan^2$, $\sin^2$, $\cos^2$ mixed in one expression 1. Identify which Pythagorean identity applies. 2. Substitute to replace the awkward term. 3. Simplify. Simplify $\sec^2x - \tan^2x$: $= (1+\tan^2x) - \tan^2x = 1$ Memorise which pairs go together: $\sec^2/\tan^2$, $\cosec^2/\cot^2$. Don't mix them.
2. Prove a trig identity Told to show LHS $\equiv$ RHS involving sec/cosec/cot 1. Work on ONE side only (usually the more complex). 2. Convert sec/cosec/cot to sin/cos. 3. Combine fractions, factorise, simplify until it matches the other side. Prove $\cosec x - \sin x \equiv \cos x \cot x$: LHS $= \tfrac{1}{\sin x} - \sin x = \tfrac{1-\sin^2x}{\sin x} = \tfrac{\cos^2x}{\sin x} = \cos x \cdot \tfrac{\cos x}{\sin x} \equiv \cos x \cot x$ ✓ Never move terms to the other side or cross-multiply. One side only.
3. Solve equation with sec/cosec/cot Equation contains $\sec x = k$, $\cosec x = k$, or $\cot x = k$ 1. Rewrite: $\cos x = 1/k$ (for sec), $\sin x = 1/k$ (for cosec), $\tan x = 1/k$ (for cot). 2. Solve the standard trig equation. 3. Check no solutions are at undefined points. Solve $\sec x = 2$ for $0 \le x \le 2\pi$: $\cos x = \tfrac{1}{2}$ → $x = \tfrac{\pi}{3}$ or $x = \tfrac{5\pi}{3}$ $\sec x = k$ means $\cos x = 1/k$, NOT $\cos x = k$. Flip the value.
4. Quadratic in sec²/tan² or cosec²/cot² Equation mixes $\sec^2x$ and $\tan x$ (or $\cosec^2x$ and $\cot x$) 1. Use $\sec^2x = 1+\tan^2x$ to express all in terms of $\tan x$. 2. Expand and collect. 3. Solve quadratic in $\tan x$. $2\sec^2x + \tan x - 3 = 0$: $2(1+\tan^2x)+\tan x-3=0$ → $2\tan^2x+\tan x-1=0$ → $(2\tan x-1)(\tan x+1)=0$ → $\tan x=\tfrac{1}{2}$ or $\tan x=-1$ Always substitute the identity FIRST before expanding. Easier algebra.
5. Evaluate arcsin/arccos/arctan exactly Asked for exact value of $\arcsin(k)$, $\arccos(k)$, $\arctan(k)$ 1. Identify which exact angle gives $\sin/\cos/\tan = k$. 2. Check the output is in the correct range for that function. $\arcsin(-\tfrac{1}{2}) = -\tfrac{\pi}{6}$ (in $[-\pi/2,\pi/2]$ ✓). $\arccos(-\tfrac{1}{2}) = \tfrac{2\pi}{3}$ (in $[0,\pi]$ ✓, not $-\pi/3$). arcsin and arccos of negative values give opposite signs. arccos output is never negative.

7. Common exam pitfalls

Mistake Correct
$\sin^{-1}x = \dfrac{1}{\sin x}$ $\sin^{-1}x = \arcsin x$ (inverse). $\dfrac{1}{\sin x} = \cosec x$. Completely different.
$\arcsin(\sin(5\pi/6)) = 5\pi/6$ $= \arcsin(\tfrac{1}{2}) = \tfrac{\pi}{6}$. arcsin output always in $[-\pi/2, \pi/2]$.
$\sec^2x = 1 + \sin^2x$ $\sec^2x = 1 + \tan^2x$. Mixed up the identity — memorise the pairs.
Solving $\sec x = 2$ as $\cos x = 2$ $\cos x = \dfrac{1}{\sec x} = \dfrac{1}{2}$. Reciprocal the VALUE, not the function name.
$\arccos(-\tfrac{1}{2}) = -\tfrac{\pi}{3}$ arccos output is in $[0, \pi]$ — never negative. Correct: $\arccos(-\tfrac{1}{2}) = \dfrac{2\pi}{3}$.
Proving identities: working on both sides Work ONE side only. Examiners need to see a chain of equivalent forms from one side to reach the other.

8. Big picture (30-second mental model)

  1. Three new functions = three reciprocals. $\sec=1/\cos$, $\cosec=1/\sin$, $\cot=\cos/\sin$. Undefined where denominator $=0$.
  2. Three identities from $\sin^2+\cos^2=1$. Divide by $\cos^2$ → $\sec^2=1+\tan^2$. Divide by $\sin^2$ → $\cosec^2=1+\cot^2$. BOTH must be memorised.
  3. Solving sec/cosec/cot equations: rewrite as $1/\cos$, $1/\sin$, $\cos/\sin$ → solve the underlying sin/cos/tan equation.
  4. Quadratic equations: substitute the Pythagorean identity to get everything in one trig function → factorise.
  5. Inverse trig outputs are restricted. arcsin: $[-\pi/2, \pi/2]$. arccos: $[0, \pi]$. arctan: $(-\pi/2, \pi/2)$. Each gives exactly one value.
  6. Identity proofs: convert to sin and cos, then simplify one side to match the other. Never cross-multiply.

9. Worked archetypes

# Exam setup Type Full solution
1 Simplify $\dfrac{1-\sin^2x}{1-\cos^2x}$ Identity simplification $= \dfrac{\cos^2x}{\sin^2x} = \cot^2x$
2 Prove $\cot x + \tan x \equiv \cosec x \sec x$ Identity proof LHS $= \dfrac{\cos x}{\sin x} + \dfrac{\sin x}{\cos x} = \dfrac{\cos^2x+\sin^2x}{\sin x \cos x} = \dfrac{1}{\sin x \cos x} = \cosec x \cdot \sec x$ ✓
3 Solve $3\cosec^2x + \cot x = 5$, $0 \le x \le 2\pi$ Quadratic $3(1+\cot^2x)+\cot x=5$ → $3\cot^2x+\cot x-2=0$ → $(3\cot x-2)(\cot x+1)=0$ → $\cot x=\tfrac{2}{3}$ or $\cot x=-1$ → $\tan x=\tfrac{3}{2}$ or $\tan x=-1$ → find $x$ values in $[0,2\pi]$
4 Evaluate $\arctan(-1) + \arcsin\!\left(\dfrac{\sqrt{3}}{2}\right)$ Inverse trig exact $\arctan(-1)=-\dfrac{\pi}{4}$ (in $(-\pi/2,\pi/2)$ ✓). $\arcsin\!\left(\dfrac{\sqrt{3}}{2}\right)=\dfrac{\pi}{3}$. Sum $= -\dfrac{\pi}{4}+\dfrac{\pi}{3} = \dfrac{-3\pi+4\pi}{12} = \dfrac{\pi}{12}$
