P3 Ch 8 — Numerical Methods

The whole chapter in two ideas. P3 Numerical Methods = sign-change method (locate/verify roots by checking $f(a) \cdot f(b) < 0$ on a continuous function) + fixed-point iteration (rearrange $f(x)=0$ to $x = g(x)$, iterate $x_{n+1} = g(x_n)$). Master the three pitfalls (asymptotes, even roots, diverging rearrangements) and the verification interval formula $[\alpha \pm 0.5 \times 10^{-n}]$, and every exam question becomes a checklist.

1. Jargon Glossary

Term Definition
Root A value $x = \alpha$ where $f(\alpha) = 0$; graphically, where $y = f(x)$ crosses the $x$-axis
Continuous A function with no jumps or breaks on an interval; its graph can be drawn without lifting the pen
Change of sign $f(a)$ and $f(b)$ have opposite signs (one positive, one negative), indicating the curve crossed the $x$-axis
Interval A set of real numbers between two bounds; written $[a,b]$ (closed) or $(a,b)$ (open)
Iterative formula A recurrence relation $x_{n+1} = g(x_n)$ used to generate successive approximations to a root
Staircase diagram Graphical iteration where successive values approach the root from the same side — steps form a "staircase"
Cobweb diagram Graphical iteration where successive values alternate either side of the root — steps form a "cobweb"
Convergence Iterates $x_0, x_1, x_2, \ldots$ get progressively closer to the root $\alpha$
Divergence Iterates move progressively further away from the root
Fixed point A value $\alpha$ where $g(\alpha) = \alpha$; every root of $f(x) = 0$ is a fixed point of the rearrangement $x = g(x)$

2. Method Table — Change-of-Sign Method

Step Action Key rule
1 Define $f(x)$ so that the root satisfies $f(x) = 0$ May need to rearrange
2 Choose an interval $[a, b]$ that you suspect contains a root Pick integers or obvious candidates first
3 Evaluate $f(a)$ and $f(b)$ Keep enough decimal places
4 Check $f(a) \cdot f(b) < 0$ (opposite signs) If same sign, method gives no conclusion
5 State: "There is a change of sign in $[a,b]$ and $f$ is continuous, so there is at least one root in $(a,b)$" Must mention continuity and change of sign

When change of sign fails

Failure mode Why it fails What actually happens
Multiple roots, same sign Odd number of roots in $[a,b]$ can still give a sign change Sign change proves ≥1 root, not exactly 1
Even number of roots Curve dips below and returns — same sign at both ends No sign change even though roots exist
Vertical asymptote Discontinuity causes sign flip with no root Sign change is misleading — function not continuous

3. Fixed-Point Iteration — Setup Recipes

$f(x) = 0$ form Rearrangement strategy Example iterative formula
$x^2 - ax + b = 0$ Divide all by $x$: $x = a - b/x$ $x_{n+1} = a - b/x_n$
$x^2 - ax + b = 0$ Isolate $x^2$: $x = \sqrt{ax - b}$ $x_{n+1} = \sqrt{ax_n - b}$
$x^3 = p x^2 + q x - r$ Divide by $x$: $x^2 = px + q - r/x$, then $\sqrt{}$ $x_{n+1} = \sqrt{px_n + q - r/x_n}$
$x^3 + ax^2 = b$ Rearrange cubic: $x = \sqrt[3]{b - ax^2}$ $x_{n+1} = \sqrt[3]{b - ax_n^2}$
Trigonometric $f(x) = 0$ Isolate trig term, apply inverse $x_{n+1} = \arccos(\cdots)^{1/2}$ etc.
Exponential/log $f(x) = 0$ Isolate $e^x$ or $\ln x$, apply inverse $x_{n+1} = \ln(\text{expression in } x_n) + c$

Note: The same $f(x) = 0$ can yield multiple valid rearrangements; different rearrangements may converge to different roots or diverge entirely.

4. Edge-Case Table — 3 Sign-Change Pitfalls

# Pitfall Diagram clue Conclusion to draw
1 Odd multiple roots in $[a,b]$ Curve crosses axis multiple times, same sign at endpoints? No — sign does change Sign change ⇒ at least one root (could be 3, 5, …)
2 Even number of roots in $[a,b]$ Curve dips below/above and returns; $f(a)$ and $f(b)$ same sign No sign change ⇒ cannot conclude no root — roots may still exist
3 Asymptote in $(a,b)$ Vertical asymptote (e.g. $\tan x$ near $\pi/2$) causes $f \to \pm\infty$ Sign change ≠ root; function not continuous on $[a,b]$, so theorem does not apply

5. Algorithm Table — Verify Root to $n$ Decimal Places

To show that root $\alpha = k$ is correct to $n$ d.p.:

Step Action Example ($\alpha = 1.763$ to 3 d.p.)
1 Identify the "rounding interval": $\left[\alpha - 0.5 \times 10^{-n},\; \alpha + 0.5 \times 10^{-n}\right]$ $[1.7625,\; 1.7635]$
2 Evaluate $f$ at both endpoints $f(1.7625) = -0.00064\ldots$, $f(1.7635) = +0.00024\ldots$
3 Confirm opposite signs and state continuity Change of sign, $f$ continuous
4 Conclude: root lies in $[1.7625, 1.7635]$, so $\alpha = 1.763$ to 3 d.p. ✓

Memory hook: half-unit in the last decimal place — go one digit further than the quoted accuracy.

7. Decision Flow — Spot This → Do This

GIVEN: f(x) = 0,  find / verify a root
         │
         ├─ "Show a root exists in [a,b]"
         │       → Evaluate f(a) and f(b)
         │       → Check opposite signs
         │       → State continuity + change of sign
         │
         ├─ "Verify root α = k to n d.p."
         │       → Interval = [k − 0.5×10⁻ⁿ,  k + 0.5×10⁻ⁿ]
         │       → Evaluate f at both ends, check opposite signs
         │       → State conclusion with change of sign
         │
         ├─ "Find root by iteration"
         │       → Rearrange f(x)=0 to x = g(x)
         │       → Write x_{n+1} = g(x_n)
         │       → Choose x₀ near the root
         │       → Iterate until convergence
         │
         ├─ "Sign change but is there really a root?"
         │       → Check for asymptote in (a,b)
         │       → If yes: f is NOT continuous → theorem fails
         │
         └─ "No sign change — does a root exist?"
                 → Check for even number of roots (graph sketch)
                 → Sign change is sufficient but not necessary

8. Five Archetype Examples

Archetype 1 — Basic sign-change location

Problem: Show $f(x) = x^3 - 4x^2 + 3x + 1$ has a root in $(2, 3)$.
Solution: $f(2) = 8 - 16 + 6 + 1 = -1 < 0$; $f(3) = 27 - 36 + 9 + 1 = 1 > 0$.
Change of sign + continuous ⇒ root in $(2, 3)$. ✓

Archetype 2 — Verify root to 3 d.p.

Problem: Verify $\alpha = 1.864$ correct to 3 d.p. for $f(x) = 3 + x^2 - x^3$.
Interval: $[1.8635, 1.8645]$
$f(1.8635) > 0$, $f(1.8645) < 0$ → change of sign → $\alpha = 1.864$ to 3 d.p. ✓

Archetype 3 — Build and run an iterative formula

Problem: $f(x) = x^2 - 4x + 1 = 0$, root in $(3,4)$.
Rearrange: $x^2 = 4x - 1 \Rightarrow x = 4 - 1/x$.
Formula: $x_{n+1} = 4 - 1/x_n$, $x_0 = 3$.
$x_1 = 3.6\overline{6}$, $x_2 = 3.727\ldots$, $x_3 = 3.7317\ldots$ → converging to $\alpha = 3.732\ldots$

Archetype 4 — Cobweb vs staircase (same function, two rearrangements)

$f(x) = x^2 - x - 1$:

Rearrangement Formula Behaviour
$x = \sqrt{x+1}$ $x_{n+1} = \sqrt{x_n + 1}$, $x_0 = 0.5$ Staircase — monotone approach from below
$x = \dfrac{1}{x-1}$ $x_{n+1} = \dfrac{1}{x_n-1}$, $x_0 = -2$ Cobweb — alternating either side
$x = x^2 - 1$ $x_{n+1} = x_n^2 - 1$, $x_0 = 2$ Diverges — moves away from root

Archetype 5 — Asymptote pitfall

Problem: $f(x) = 2 + \tan x$, interval $[1.5, 1.6]$.
$f(1.5) = 2 + \tan(1.5) \approx 16.1 > 0$; $f(1.6) = 2 + \tan(1.6) \approx -32.5 < 0$.
Sign change exists, but $\tan x$ has a vertical asymptote at $x = \pi/2 \approx 1.5708 \in (1.5, 1.6)$.
$f$ is not continuous on $[1.5, 1.6]$ → no root in this interval.

9. Common Pitfalls (Exam Mark-Losers)

# Pitfall Fix
1 Forgetting to state continuity when concluding a root exists Always write: "… and $f$ is continuous on $[a,b]$, so …"
2 Forgetting to state change of sign in the conclusion Explicitly say "there is a change of sign"
3 Wrong verification interval: using $[\alpha - 0.5, \alpha + 0.5]$ instead of $[\alpha \pm 0.5 \times 10^{-n}]$ Half a unit in the last quoted decimal place
4 Assuming sign change ⇒ exactly one root Sign change ⇒ at least one root
5 Assuming no sign change ⇒ no roots Even-multiplicity roots don't give sign changes
6 Using a rearrangement that diverges for the given starting value Try a different rearrangement or different $x_0$
7 Not checking whether the iterative formula is valid (e.g. $x \neq 0$ after dividing) State domain restriction
8 Rounding intermediate iterations — accumulates error Keep full calculator precision during iteration
9 Describing the diagram type wrong: cobweb vs staircase Staircase = same-side approach; cobweb = alternating sides
10 Missing sign-change when $f$ has an asymptote in the interval Always sketch or check for discontinuities

10. Big Picture — 30-Second Mental Model

NUMERICAL METHODS: two big ideas
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① SIGN CHANGE  ──────────────────────────────
   Continuous f: opposite signs at endpoints
       ⟹  at least one root between them
   Use to: LOCATE a root, VERIFY to n d.p.
   Pitfalls: asymptotes, even roots

② FIXED-POINT ITERATION  ────────────────────
   Rearrange f(x) = 0  →  x = g(x)
   Iterate: x_{n+1} = g(x_n)
   Converges ─┬─ staircase (same-side)
              └─ cobweb   (alternating)
   Diverges: iterations run away from root
   Different rearrangements → may find
   different roots or diverge!

KEY FORMULA PAIR:
   Root in [a,b] : f(a)·f(b) < 0  AND  f continuous
   Root = α to n d.p. : sign change in
        [α − 0.5×10⁻ⁿ,  α + 0.5×10⁻ⁿ]
