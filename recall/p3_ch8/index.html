P3 Chapter 8 — Numerical Methods · Recall Sheet

1. Jargon Glossary

TermDefinition
RootA value $x = \alpha$ where $f(\alpha) = 0$; graphically, where $y=f(x)$ crosses the $x$-axis
ContinuousA function with no jumps or breaks on an interval — graph drawn without lifting the pen
Change of sign$f(a)$ and $f(b)$ have opposite signs (one +, one −), indicating the curve crossed the $x$-axis
IntervalA set of real numbers between two bounds; $[a,b]$ (closed) or $(a,b)$ (open)
Iterative formulaA recurrence relation $x_{n+1} = g(x_n)$ generating successive approximations to a root
Staircase diagramGraphical iteration where iterates approach the root from the same side — steps look like stairs
Cobweb diagramGraphical iteration where iterates alternate either side of the root — pattern looks like a cobweb
ConvergenceIterates $x_0, x_1, x_2, \ldots$ get progressively closer to the root $\alpha$
DivergenceIterates move progressively further away from the root
Fixed pointA value $\alpha$ where $g(\alpha) = \alpha$; every root of $f(x)=0$ is a fixed point of $x = g(x)$

2. Method Table — Change-of-Sign Method

StepActionKey rule
1Define $f(x)$ so the root satisfies $f(x)=0$Rearrange if needed
2Choose interval $[a,b]$ suspected to contain a rootTry integers first
3Evaluate $f(a)$ and $f(b)$Keep enough decimal places
4Check $f(a) \cdot f(b) < 0$Same sign ⇒ no conclusion
5State: change of sign + $f$ continuous ⇒ at least one root in $(a,b)$Must mention both continuity and sign change

When Change of Sign Fails

#Failure modeWhy it failsReality
1Multiple roots, same signOdd number of roots in $[a,b]$ still gives a sign changeSign change ⇒ ≥1 root, not exactly 1
2Even number of rootsCurve dips and returns — same sign at both endsNo sign change, yet roots do exist
3Asymptote in $(a,b)$Vertical asymptote causes sign flip with no crossingSign change ≠ root; $f$ not continuous ⇒ theorem fails

3. Fixed-Point Iteration — Setup Recipes

$f(x)=0$ formRearrangement strategyExample iterative formula
$x^2 - ax + b = 0$Divide by $x$: $x = a - b/x$$x_{n+1} = a - b/x_n$
$x^2 - ax + b = 0$Isolate $x^2$: $x = \sqrt{ax-b}$$x_{n+1} = \sqrt{ax_n - b}$
$x^3 = px^2 + qx - r$Divide by $x$, take square root$x_{n+1} = \sqrt{px_n + q - r/x_n}$
$x^3 + ax^2 = b$Cube root: $x = \sqrt[3]{b - ax^2}$$x_{n+1} = \sqrt[3]{b - ax_n^2}$
Trig $f(x)=0$Isolate trig term, apply inverse$x_{n+1} = (\arccos(\cdots))^{1/2}$
Exp/log $f(x)=0$Isolate $e^x$ or $\ln x$, apply inverse$x_{n+1} = \ln(\text{expr})+ c$
⚠️ Same $f(x)=0$ can yield multiple rearrangements — different ones may find different roots or diverge.

4. Edge-Case Table — 3 Sign-Change Pitfalls

#PitfallDiagram clueCorrect conclusion
1Odd multiple roots in $[a,b]$Curve crosses axis >1 time; sign does changeSign change ⇒ at least one root (could be 3, 5, …)
2Even number of roots in $[a,b]$Curve dips below/above and returns; $f(a)$, $f(b)$ same signNo sign change ⇒ cannot rule out roots — they exist!
3Asymptote in $(a,b)$Vertical asymptote (e.g. $\tan x$ near $\pi/2$) → $f\to\pm\infty$Sign change ≠ root; $f$ not continuous → theorem doesn't apply

5. Algorithm — Verify Root to $n$ Decimal Places

To show root $\alpha = k$ is correct to $n$ d.p.:

StepActionExample ($\alpha=1.763$ to 3 d.p.)
1Rounding interval: $\left[\alpha - \tfrac{1}{2}\times10^{-n},\; \alpha + \tfrac{1}{2}\times10^{-n}\right]$$[1.7625,\; 1.7635]$
2Evaluate $f$ at both endpoints$f(1.7625)=-0.00064\ldots$, $f(1.7635)=+0.00024\ldots$
3Confirm opposite signs + state continuityChange of sign, $f$ continuous on interval
4Conclude: root lies in $[1.7625, 1.7635]$ so $\alpha=1.763$ to 3 d.p.✓
💡 Memory hook: half a unit in the last quoted decimal place — always go one digit beyond the stated accuracy.

7. Decision Flow — Spot This → Do This

GIVEN: f(x) = 0,  find / verify a root
         │
         ├─ "Show a root exists in [a,b]"
         │       → Evaluate f(a) and f(b)
         │       → Check opposite signs
         │       → State continuity + change of sign
         │
         ├─ "Verify root α = k to n d.p."
         │       → Interval = [k − 0.5×10⁻ⁿ,  k + 0.5×10⁻ⁿ]
         │       → Evaluate f at both ends, check opposite signs
         │       → State conclusion with change of sign
         │
         ├─ "Find root by iteration"
         │       → Rearrange f(x)=0 to x = g(x)
         │       → Write x_{n+1} = g(x_n)
         │       → Choose x₀ near the root
         │       → Iterate until convergence
         │
         ├─ "Sign change but is there really a root?"
         │       → Check for asymptote in (a,b)
         │       → If yes: f is NOT continuous → theorem fails
         │
         └─ "No sign change — does a root exist?"
                 → Check for even number of roots (sketch)
                 → Sign change sufficient but NOT necessary

8. Five Archetype Examples

① Basic Sign-Change Location

Problem: Show $f(x)=x^3-4x^2+3x+1$ has a root in $(2,3)$.

$f(2)=8-16+6+1=-1<0$;  $f(3)=27-36+9+1=1>0$

Change of sign + $f$ continuous ⇒ root in $(2,3)$. ✓

② Verify Root to 3 d.p.

Problem: Verify $\alpha=1.864$ to 3 d.p. for $f(x)=3+x^2-x^3$.

Interval: $[1.8635,\; 1.8645]$

$f(1.8635)>0$, $f(1.8645)<0$ → change of sign → $\alpha=1.864$ to 3 d.p. ✓

③ Build and Run an Iterative Formula

Problem: $f(x)=x^2-4x+1=0$, root in $(3,4)$.

Rearrange: $x^2=4x-1 \Rightarrow x = 4 - 1/x$

Formula: $x_{n+1}=4-1/x_n$, $x_0=3$

$x_1=3.6\overline{6}$,  $x_2=3.727\ldots$,  $x_3=3.7317\ldots$  → converging to $\alpha=3.732\ldots$

④ Cobweb vs Staircase — Same Function, Two Rearrangements

$f(x)=x^2-x-1$:

RearrangementFormulaBehaviour
$x=\sqrt{x+1}$$x_{n+1}=\sqrt{x_n+1}$, $x_0=0.5$Staircase — monotone approach from below
$x=\dfrac{1}{x-1}$$x_{n+1}=\dfrac{1}{x_n-1}$, $x_0=-2$Cobweb — alternating either side of root
$x=x^2-1$$x_{n+1}=x_n^2-1$, $x_0=2$Diverges — moves away from root

⑤ Asymptote Pitfall

Problem: $f(x)=2+\tan x$, interval $[1.5, 1.6]$.

$f(1.5)=2+\tan(1.5)\approx16.1>0$;  $f(1.6)=2+\tan(1.6)\approx-32.5<0$

Sign change exists, but $\tan x$ has a vertical asymptote at $x=\pi/2\approx1.5708\in(1.5,1.6)$.

$f$ is not continuous on $[1.5,1.6]$ → no root in this interval. The theorem does not apply.

9. Common Pitfalls — Exam Mark-Losers

#PitfallFix
1Forgetting to state continuityAlways write: "… and $f$ is continuous on $[a,b]$, so …"
2Forgetting to state change of signExplicitly say "there is a change of sign"
3Wrong verification interval (using $\pm0.5$ instead of $\pm0.5\times10^{-n}$)Half a unit in the last quoted decimal place
4Assuming sign change ⇒ exactly one rootSign change ⇒ at least one root
5Assuming no sign change ⇒ no rootsEven-multiplicity roots don't give sign changes
6Using a rearrangement that divergesTry a different rearrangement or different $x_0$
7Not stating domain restrictions (e.g. $x\neq0$)State any restriction introduced during rearrangement
8Rounding intermediate iterationsKeep full calculator precision; round only the final answer
9Cobweb vs staircase confusionStaircase = same-side approach; cobweb = alternating sides
10Missing asymptote inside intervalSketch first or check for discontinuities before applying theorem

① SIGN CHANGE  ──────────────────────────────────
   Continuous f: opposite signs at endpoints
       ⟹  at least one root between them
   Use to: LOCATE a root, VERIFY to n d.p.
   Pitfalls: asymptotes, even roots

② FIXED-POINT ITERATION  ────────────────────────
   Rearrange f(x) = 0  →  x = g(x)
   Iterate: x_{n+1} = g(x_n)
   Converges ─┬─ staircase (same-side approach)
              └─ cobweb   (alternating sides)
   Diverges: iterations run away from root
   Different rearrangements → may find
   different roots or diverge!

KEY FORMULA PAIR:
   Root in [a,b]:  f(a)·f(b) < 0  AND  f continuous
   Root = α to n d.p.:  sign change in
        [α − 0.5×10⁻ⁿ,  α + 0.5×10⁻ⁿ]
