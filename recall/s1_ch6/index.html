S1 Ch 6 — Discrete Random Variables

The whole chapter in one mental model. A discrete random variable $X$ has a list of possible values $x$ and a probability distribution $P(X=x)$ summing to 1. From this you compute three signature statistics: the expected value $E(X)=\sum xP(X=x)$ (the long-run mean), the variance $\text{Var}(X)=E(X^2)-(E(X))^2$ (spread squared), and the cumulative distribution function $F(x)=P(X\le x)$. Linear transformations $Y=aX+b$ rescale predictably: $E(aX+b)=aE(X)+b$ but $\text{Var}(aX+b)=a^2\text{Var}(X)$ — the constant shift drops from variance.

1. Jargon glossary

Term Meaning Where
Random variable $X$ A variable whose value is determined by a random experiment. Capital letter for the variable, lower case for a particular value. Discrete if it takes a countable set of values (typically integers); continuous otherwise. 6.1
Sample space The set of all possible values $X$ can take, e.g. $\{0,1,2,3\}$ for "number of heads in 3 coin flips". Probabilities over this space must sum to 1. 6.1
Probability distribution A complete specification of $P(X=x)$ for every $x$ in the sample space. Can be given as a table, a probability function $P(X=x)=f(x)$, or a diagram (bar chart). The defining identity: $\sum P(X=x)=1$. 6.1
Probability function A formula for $P(X=x)$, e.g. $P(X=x)=k/x$ for $x=1,2,3,4$. Find $k$ by enforcing $\sum P=1$. 6.1
Discrete uniform distribution Every value in the sample space is equally likely: $P(X=x)=1/n$. Examples: fair dice, picking a card. Has tidy formulae for $E$ and $\text{Var}$ when $X\in\{1,2,\dots,n\}$. 6.7
Cumulative distribution function (CDF), $F(x)$ $F(x)=P(X\le x)$ — running total of probabilities up to and including $x$. Built by cumulating the probability table. Always non-decreasing, ends at 1. For non-integer arguments, $F(2.6)=F(2)$ for integer-valued $X$ — don't interpolate. 6.2
Expected value $E(X)$ (mean $\mu$) $E(X)=\sum x\,P(X=x)$. The "centre of mass" of the distribution; the long-run average if the experiment is repeated many times. Doesn't have to be a value $X$ actually takes (e.g. fair dice gives $E(X)=3.5$). 6.3
Variance $\text{Var}(X)=\sigma^2$ $\text{Var}(X)=E((X-E(X))^2) = E(X^2)-(E(X))^2$. Always $\ge 0$. Larger ⇒ wider spread of $X$ around its mean. Has units of $X^2$. 6.4
Standard deviation $\sigma$ $\sigma=\sqrt{\text{Var}(X)}$. Same units as $X$ — the natural "average distance from the mean" measure. 6.4
$E(X^2)$ The expected value of $X^2$: $E(X^2)=\sum x^2 P(X=x)$. Not the same as $(E(X))^2$ — their difference is exactly $\text{Var}(X)$. 6.4
Function of a random variable $g(X)$ If $X$ is a random variable and $g$ a function, $g(X)$ is also a random variable. Its expected value is $E(g(X))=\sum g(x)P(X=x)$ — use the same probabilities, just apply $g$ to each $x$. 6.5
Linear transformation $Y=aX+b$ Rescaling and shifting. $E(aX+b)=aE(X)+b$. $\text{Var}(aX+b)=a^2\text{Var}(X)$ — the constant $b$ vanishes from variance because shifting doesn't change spread. 6.5
Mean of a sum $E(X+Y)=E(X)+E(Y)$ — always, even if $X,Y$ are dependent. (Variance does NOT add this simply unless $X,Y$ are independent — beyond this chapter.) 6.5
Mode (of a discrete RV) The most-likely value, i.e. the $x$ that maximises $P(X=x)$. Read off the distribution table directly. 6.1

2. Probability distribution — must-know identities

Identity Statement Use
Total probability $\displaystyle\sum_{x} P(X=x)=1$ Solve for unknown $k$ in a probability function
Non-negative $0\le P(X=x)\le 1$ for all $x$ Sanity check
Inequality probabilities $P(X<3)=P(X=1)+P(X=2)+\dots$ Add the matching cells of the table
CDF $F(x)=P(X\le x)=\sum_{t\le x}P(X=t)$ Running total
Recover PMF from CDF $P(X=x)=F(x)-F(x-1)$ Subtract consecutive CDF values
Sample-space rule $F(\max(X))=1$, $F(\text{below min})=0$ Use to find unknowns in a CDF formula

3. Expected value $E(X)$ — formulas and rules

Setting Formula
Definition $E(X)=\sum x\,P(X=x)$
Function $g$ $E(g(X))=\sum g(x)\,P(X=x)$ — same probabilities, transformed values
Square $E(X^2)=\sum x^2 P(X=x)$
Linearity (constant) $E(aX+b)=aE(X)+b$
Linearity (sum) $E(X+Y)=E(X)+E(Y)$ (always)
Constant $E(c)=c$ for any constant $c$
Discrete uniform on $\{1,\dots,n\}$ $E(X)=\dfrac{n+1}{2}$

4. Variance $\text{Var}(X)$ — formulas and rules

Setting Formula
Definition $\text{Var}(X)=E((X-E(X))^2)$
Computational shortcut $\text{Var}(X)=E(X^2)-(E(X))^2$
Linear transformation $\text{Var}(aX+b)=a^2\text{Var}(X)$ — shift drops
Constant $\text{Var}(c)=0$
Standard deviation $\sigma=\sqrt{\text{Var}(X)}$
Discrete uniform on $\{1,\dots,n\}$ $\text{Var}(X)=\dfrac{(n+1)(n-1)}{12}=\dfrac{n^2-1}{12}$

Recover $E(X^2)$ from $E(X)$ and $\text{Var}(X)$: $$E(X^2)=\text{Var}(X)+(E(X))^2.$$

5. CDF — building and using

Action Method
Build CDF from probability table Cumulative sum: $F(x_1)=p_1,\ F(x_2)=p_1+p_2,\ \dots$
Recover PMF from CDF $P(X=x_i)=F(x_i)-F(x_{i-1})$
Find $P(a $F(b)-F(a)$
Find $P(a\le X\le b)$ $F(b)-F(a-1)$ if $X$ integer-valued
Find unknown in $F(x)$ formula Use $F(\max)=1$
$F(\text{non-integer})$ for integer $X$ $F(2.6)=F(2)$ — never interpolate for discrete variables

6. Functions of $X$ — workflow

For $Y=g(X)$:

Step Action
1 Apply $g$ to each value of $x$ → get values of $y$
2 The probabilities don't change: $P(Y=g(x))=P(X=x)$
3 $E(Y)=\sum g(x)\,P(X=x)$
4 $\text{Var}(Y)$: build $E(Y),E(Y^2)$, use $\text{Var}(Y)=E(Y^2)-(E(Y))^2$
Linear shortcut If $g(x)=ax+b$: $E(Y)=aE(X)+b$, $\text{Var}(Y)=a^2\text{Var}(X)$

Examples of $g$: $X^2,\ \sqrt X,\ \sin X,\ 2^X,\ \log(X+1),\ \dfrac{4-3X}{2}$.

7. Discrete uniform distribution — special-case formulas

For $X$ uniform on $\{1,2,\dots,n\}$:

Quantity Formula
$P(X=x)$ $1/n$
$E(X)$ $\dfrac{n+1}{2}$
$\text{Var}(X)$ $\dfrac{n^2-1}{12}=\dfrac{(n+1)(n-1)}{12}$
$\sigma$ $\sqrt{(n^2-1)/12}$

For uniform on $\{0,1,\dots,n-1\}$ (e.g. random digit 0–9): translate to $X-1$ where $X\in\{1,\dots,n\}$, use $E(X-1)=E(X)-1=\dfrac{n-1}{2}$ and $\text{Var}(X-1)=\text{Var}(X)$.

8. Decision flow — what's being asked?

Spot this Do this
Probability function with unknown $k$ Set $\sum P=1$, solve for $k$
"Find $E(X)$" Compute $\sum x\,P(X=x)$ from the table
"Find $\text{Var}(X)$" Compute $E(X^2)$, then $\text{Var}=E(X^2)-(E(X))^2$
"Find $E(g(X))$" for some $g$ Apply $g$ to each $x$, multiply by original probabilities, sum
Linear transform $Y=aX+b$ $E(Y)=aE(X)+b$, $\text{Var}(Y)=a^2\text{Var}(X)$
Two equations needed (e.g. unknowns $p,q$) One from $\sum P=1$, one from given $E(X)$
CDF given, find PMF $P(X=x)=F(x)-F(x-1)$
Discrete uniform on $\{1,\dots,n\}$ Use formulas $\dfrac{n+1}{2},\dfrac{n^2-1}{12}$
Recover $E(X^2)$ $E(X^2)=\text{Var}(X)+(E(X))^2$

9. Worked archetypes

# Setup Method
1 $P(X=x)=k/x$, $x=1,2,3,4$. Find $k$. $k(1+\tfrac12+\tfrac13+\tfrac14)=1\to k=12/25$
2 $X$ uniform on $\{1,\dots,6\}$. $E(X),\text{Var}(X)$. $E(X)=3.5,\ \text{Var}(X)=35/12$
3 Given $E(X)=3$ and $P$ table with unknowns $p,q$. Eq1: $\sum P=1$; Eq2: $\sum xP=3$. Solve simultaneously
4 $X$ has $E(X)=4,\text{Var}(X)=3$. Find $E(3X),\text{Var}(3X),E(X^2)$. $E(3X)=12$; $\text{Var}(3X)=9\cdot 3=27$; $E(X^2)=3+16=19$
5 $Y=2X+1$. PMF of $Y$ from PMF of $X$. $y$ values shift; probabilities unchanged
6 $E(\sin X^\circ)$ for $X\in\{0,30,60,90\}$ with given probabilities. Apply $\sin$ to each $x$, multiply by $P$, sum
7 $Y=(X-150)/50,\ E(Y)=5.1,\text{Var}(Y)=2.5$. Find $E(X),\text{Var}(X)$. $X=50Y+150$; $E(X)=50(5.1)+150=405$; $\text{Var}(X)=2500\cdot 2.5=6250$
8 CDF $F(x)=(x+k)/8$ for $x=1,2,3$. Find $k$. $F(3)=1\to k=5$

10. Common exam pitfalls

Pitfall Fix
Including $b$ in $\text{Var}(aX+b)$ $\text{Var}(aX+b)=a^2\text{Var}(X)$ — the $b$ drops!
Forgetting to square $a$ $\text{Var}(3X)=9\,\text{Var}(X)$, not $3\,\text{Var}(X)$
$E(X^2)=(E(X))^2$ — wrong These differ by $\text{Var}(X)$
Interpolating CDF for discrete $X$ $F(2.6)=F(2)$ for integer $X$. Discrete = no interpolation
Calling a value of $X$ unattainable when $E(X)$ ≠ a possible value $E(X)=3.5$ for fair dice is fine; the expected value is theoretical
Using the wrong formula for non-uniform-on-$\{1..n\}$ $E=\tfrac{n+1}{2}$ formula only works for uniform on $\{1,\dots,n\}$. Otherwise compute from the table
Probabilities not summing to 1 Always check $\sum P=1$ — common reason a problem fails
Forgetting that $E(X+Y)=E(X)+E(Y)$ even when not independent Mean is always linear
Using $E(g(X)) = g(E(X))$ — wrong (Jensen) Apply $g$ inside the sum: $E(g(X))=\sum g(x)P(X=x)$. Only true for linear $g$
Forgetting to update probabilities for $g(X)$ Probabilities stay tied to $X$; only the $x$-values transform under $g$

11. Big picture (30-second mental model)

A discrete random variable is fully described by its probability table $\{(x,P(X=x))\}$:

  1. Sum-to-1 is the master constraint: solve any unknown $k$ from $\sum P=1$.
  2. Three statistics capture the table: mean $E(X)$, variance $\text{Var}(X)$, and the running total CDF $F(x)$.
  3. Computational variance: $\text{Var}(X)=E(X^2)-(E(X))^2$ — easier than the deviation form.
  4. Linear transforms $Y=aX+b$ rescale the centre and spread predictably: $E$ adds $b$, $\text{Var}$ scales by $a^2$ (the shift drops).
  5. Discrete uniform: the cleanest case — closed-form $E,\text{Var}$ when $X\in\{1,\dots,n\}$.
  6. CDF tricks: $P(X=x)=F(x)-F(x-1)$, and don't interpolate for discrete arguments.

Three statistics, two transformation rules, one master identity — that's the chapter.
