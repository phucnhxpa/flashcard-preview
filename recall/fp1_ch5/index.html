FP1 Ch 5 — Matrices

The whole chapter in one mental model. A matrix is a rectangular grid of numbers with its own arithmetic: add/subtract elementwise (same shape required), multiply row × column (inner sizes must match, order matters), and for $2\times 2$ matrices compute a single number called the determinant $ad-bc$ which decides whether the matrix has an inverse. The inverse formula $\mathbf{M}^{-1}=\frac{1}{\det\mathbf{M}}\begin{pmatrix}d & -b \\ -c & a\end{pmatrix}$ is the entire chapter compressed into one line.

1. Jargon glossary

Term Meaning Where
Matrix A rectangular array of numbers (called elements or entries) inside brackets. Described by its size $n\times m$: $n$ rows × $m$ columns. Bold capital letters denote matrices ($\mathbf{A},\mathbf{M}$). A column vector is a $n\times 1$ matrix; a row vector is a $1\times m$ matrix. 5.1
Square matrix A matrix where the number of rows equals the number of columns ($n=m$). Only square matrices have determinants and inverses. The chapter focuses on $2\times 2$ squares. 5.1
Zero matrix $\mathbf{0}$ A matrix in which every element is 0. Acts like 0 under addition: $\mathbf{A}+\mathbf{0}=\mathbf{A}$. Note: $\mathbf{A}\mathbf{B}=\mathbf{0}$ does NOT imply $\mathbf{A}=\mathbf{0}$ or $\mathbf{B}=\mathbf{0}$ — matrix arithmetic differs from real-number arithmetic. 5.1
Identity matrix $\mathbf{I}$ The square matrix with 1s on the main diagonal and 0s elsewhere. Acts like 1 under multiplication: $\mathbf{A}\mathbf{I}=\mathbf{I}\mathbf{A}=\mathbf{A}$. The $2\times 2$ identity is $\mathbf{I}_2=\begin{pmatrix}1&0\\0&1\end{pmatrix}$. 5.1
Scalar A plain number (not a matrix). To multiply a matrix by a scalar $k$, multiply every element by $k$. Distinguished from matrices by being non-bold ($k$ vs $\mathbf{A}$). 5.1
Additively conformable Two matrices the same size — they can be added or subtracted. $2\times 3$ + $2\times 3$ ✓; $2\times 3$ + $3\times 2$ ✗. 5.1
Multiplicatively conformable $\mathbf{AB}$ exists iff the number of columns of $\mathbf{A}$ equals the number of rows of $\mathbf{B}$. $(n\times m)\times(m\times k)=(n\times k)$. The "inner" dimensions must match; the "outer" dimensions become the product's size. 5.2
Matrix product (row × column rule) The $(i,j)$ entry of $\mathbf{AB}$ is the dot product of $\mathbf{A}$'s row $i$ with $\mathbf{B}$'s column $j$: sum the pairwise products. E.g. $\begin{pmatrix}a&b\\c&d\end{pmatrix}\begin{pmatrix}e&f\\g&h\end{pmatrix}=\begin{pmatrix}ae+bg & af+bh\\ce+dg & cf+dh\end{pmatrix}$. 5.2
Non-commutativity In general $\mathbf{AB}\neq\mathbf{BA}$. Even if both products exist they can be different sizes (e.g. $\mathbf{A}$ is $2\times 3$, $\mathbf{B}$ is $3\times 2$ → $\mathbf{AB}$ is $2\times 2$, $\mathbf{BA}$ is $3\times 3$). Order matters everywhere in matrix algebra. 5.2
Associativity Bracketing doesn't matter as long as the order is preserved: $(\mathbf{AB})\mathbf{C}=\mathbf{A}(\mathbf{BC})$. So $\mathbf{ABC}$ is unambiguous. Distributivity also holds: $\mathbf{A}(\mathbf{B}+\mathbf{C})=\mathbf{AB}+\mathbf{AC}$. 5.2
Determinant $\det\mathbf{M}$ or $\lvert\mathbf{M}\rvert$ A scalar associated with a square matrix. For $\mathbf{M}=\begin{pmatrix}a&b\\c&d\end{pmatrix}$: $\det\mathbf{M}=ad-bc$. Geometrically (FP3) it's the signed area scale factor of the linear map. Algebraically it tells you if $\mathbf{M}$ is invertible. 5.3
Singular matrix A square matrix with $\det\mathbf{M}=0$. Has no inverse. Equivalently, the rows (or columns) are linearly dependent — one is a scalar multiple of the other for $2\times 2$. 5.3
Non-singular matrix A square matrix with $\det\mathbf{M}\neq 0$. Invertible — $\mathbf{M}^{-1}$ exists. 5.3
Inverse matrix $\mathbf{M}^{-1}$ The unique matrix satisfying $\mathbf{MM}^{-1}=\mathbf{M}^{-1}\mathbf{M}=\mathbf{I}$. Acts as the "matrix reciprocal". For $2\times 2$: $\mathbf{M}^{-1}=\dfrac{1}{\det\mathbf{M}}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}$ — swap the diagonal, negate the off-diagonal, divide by determinant. 5.4
Inverse-of-product rule $(\mathbf{AB})^{-1}=\mathbf{B}^{-1}\mathbf{A}^{-1}$. Order reverses under inversion (think: socks-and-shoes — last on, first off). The most-tested algebraic identity in this chapter. 5.4
Solving a matrix equation Use inverses to "divide": from $\mathbf{AB}=\mathbf{C}$ get $\mathbf{B}=\mathbf{A}^{-1}\mathbf{C}$ (multiply on the LEFT by $\mathbf{A}^{-1}$). From $\mathbf{BA}=\mathbf{C}$ get $\mathbf{B}=\mathbf{CA}^{-1}$ (multiply on the RIGHT). Side matters because of non-commutativity. 5.4 Ex 12–13

2. The four operations — head-to-head

Operation Rule Size requirement Result size
Addition $\mathbf{A}+\mathbf{B}$ Add elementwise Same size $n\times m$ $n\times m$
Subtraction $\mathbf{A}-\mathbf{B}$ Subtract elementwise Same size $n\times m$ $n\times m$
Scalar mult. $k\mathbf{A}$ Multiply every entry by $k$ Any Same as $\mathbf{A}$
Matrix mult. $\mathbf{AB}$ Row $i$ of $\mathbf{A}$ · column $j$ of $\mathbf{B}$ $\mathbf{A}$ has $n\times m$, $\mathbf{B}$ has $m\times k$ $n\times k$

3. The six properties — what does and doesn't hold

Property Real numbers Matrices
Addition commutative $\mathbf{A}+\mathbf{B}=\mathbf{B}+\mathbf{A}$ ✓ ✓
Multiplication commutative $\mathbf{AB}=\mathbf{BA}$ ✓ ✗ (in general)
Associative $(\mathbf{AB})\mathbf{C}=\mathbf{A}(\mathbf{BC})$ ✓ ✓
Distributive $\mathbf{A}(\mathbf{B}+\mathbf{C})=\mathbf{AB}+\mathbf{AC}$ ✓ ✓
Cancellation $\mathbf{AB}=\mathbf{0}\Rightarrow\mathbf{A}=\mathbf{0}$ or $\mathbf{B}=\mathbf{0}$ ✓ ✗
Cancellation $\mathbf{AB}=\mathbf{AC}\Rightarrow\mathbf{B}=\mathbf{C}$ ✓ (if $\mathbf{A}\neq 0$) ✗ in general; ✓ if $\mathbf{A}$ non-singular (then mult. by $\mathbf{A}^{-1}$)

4. Determinant & inverse formulas — the entire chapter on one line

For $\mathbf{M}=\begin{pmatrix}a&b\\c&d\end{pmatrix}$:

Quantity Formula Notes
Determinant $\det\mathbf{M}=ad-bc$ Memorise: leading diagonal product − anti-diagonal product
Inverse $\mathbf{M}^{-1}=\dfrac{1}{ad-bc}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}$ Swap $a\leftrightarrow d$, negate $b,c$, divide by $\det$
Singular condition $ad-bc=0$ No inverse; $\mathbf{M}^{-1}$ undefined
Inverse of product $(\mathbf{AB})^{-1}=\mathbf{B}^{-1}\mathbf{A}^{-1}$ Order reverses
Determinant of product $\det(\mathbf{AB})=\det\mathbf{A}\cdot\det\mathbf{B}$ Useful identity
$\mathbf{I}$ self-inverse $\mathbf{I}\mathbf{I}=\mathbf{I}$ $\det\mathbf{I}=1$

5. Worked archetypes (memorise the shape)

# Setup Key step Result
1 $\mathbf{A}=\begin{pmatrix}1&-2\\3&4\end{pmatrix}$, $\mathbf{B}=\begin{pmatrix}-3\\2\end{pmatrix}$, find $\mathbf{AB}$ Sizes $(2\times2)(2\times1)\to 2\times 1$; row · col $\begin{pmatrix}-7\\-1\end{pmatrix}$
2 Same matrices, find $\mathbf{BA}$ Sizes $(2\times1)(2\times2)$ — inner dimensions don't match Doesn't exist
3 $\mathbf{A}=\begin{pmatrix}6&5\\1&2\end{pmatrix}$, find $\det\mathbf{A}$ $6\cdot 2-5\cdot 1=12-5$ $\det=7$
4 $\mathbf{A}=\begin{pmatrix}3&2\\-1&1\end{pmatrix}$, find $\mathbf{A}^{-1}$ $\det=3\cdot1-2\cdot(-1)=5$; swap diag, negate off-diag $\frac{1}{5}\begin{pmatrix}1&-2\\1&3\end{pmatrix}$
5 $\mathbf{A}=\begin{pmatrix}2&5\\1&3\end{pmatrix}$, $\det=1$ $\mathbf{A}^{-1}=\begin{pmatrix}3&-5\\-1&2\end{pmatrix}$ (no $1/\det$ scaling — already 1)
6 $\mathbf{A}=\begin{pmatrix}4&p+2\\-1&3-p\end{pmatrix}$ singular $\det=4(3-p)+(p+2)=14-3p=0$ $p=14/3$
7 $\mathbf{A}=\begin{pmatrix}1&2\\0&1\end{pmatrix}$, find $\mathbf{A}^k$ Compute $\mathbf{A}^2,\mathbf{A}^3$, spot pattern $\mathbf{A}^k=\begin{pmatrix}1&2k\\0&1\end{pmatrix}$
8 $\mathbf{BAB}=\mathbf{I}$, find $\mathbf{A}$ Multiply by $\mathbf{B}^{-1}$ on both sides: $\mathbf{A}=\mathbf{B}^{-1}\mathbf{B}^{-1}$ Plug in $\mathbf{B}$ values

6. Solving matrix equations — the side-of-multiplication rule

Given equation Solve by multiplying by $\mathbf{A}^{-1}$ on the… Result
$\mathbf{AX}=\mathbf{B}$ LEFT (both sides): $\mathbf{A}^{-1}\mathbf{AX}=\mathbf{A}^{-1}\mathbf{B}$ $\mathbf{X}=\mathbf{A}^{-1}\mathbf{B}$
$\mathbf{XA}=\mathbf{B}$ RIGHT (both sides): $\mathbf{XAA}^{-1}=\mathbf{BA}^{-1}$ $\mathbf{X}=\mathbf{BA}^{-1}$
$\mathbf{AXB}=\mathbf{C}$ $\mathbf{A}^{-1}$ on LEFT, $\mathbf{B}^{-1}$ on RIGHT $\mathbf{X}=\mathbf{A}^{-1}\mathbf{CB}^{-1}$
$\mathbf{ABC}=\mathbf{I}$ Pre-mult by $\mathbf{A}^{-1}$, post-mult by $\mathbf{C}^{-1}$ $\mathbf{B}=\mathbf{A}^{-1}\mathbf{C}^{-1}$ — or use $(\mathbf{ABC})^{-1}=\mathbf{C}^{-1}\mathbf{B}^{-1}\mathbf{A}^{-1}$
$\mathbf{BAB}=\mathbf{I}$ Pre-mult by $\mathbf{B}^{-1}$, post-mult by $\mathbf{B}^{-1}$ $\mathbf{A}=\mathbf{B}^{-1}\mathbf{B}^{-1}=(\mathbf{B}^{-1})^2$

Mnemonic: to undo something on the left, multiply on the left; to undo on the right, multiply on the right. Same side as where the matrix sits.

7. Decision flow — common questions

Spot this Do this
"Find $\det$ of a matrix with unknown $k$" Apply $ad-bc$, simplify; solve $=0$ for singular condition
"$\mathbf{M}$ is singular, find $k$" Set $\det\mathbf{M}=0$, solve for $k$
"Find inverse of $\mathbf{M}$ in terms of $a,b$" Compute $\det$ first; use the swap-and-negate formula
"$\mathbf{AB}=\mathbf{C}$, find $\mathbf{B}$" $\mathbf{B}=\mathbf{A}^{-1}\mathbf{C}$ — pre-multiply (if $\mathbf{A}$ non-singular)
"$\mathbf{BA}=\mathbf{C}$, find $\mathbf{B}$" $\mathbf{B}=\mathbf{CA}^{-1}$ — post-multiply
"Show $\mathbf{A}^2=\mathbf{I}$" Compute $\mathbf{A}^2$ directly, check it equals identity
"$\mathbf{A}^2=2\mathbf{A}+5\mathbf{I}$" type Compute $\mathbf{A}^2$, RHS=$2\mathbf{A}+5\mathbf{I}$, equate elements
"$(\mathbf{AB})^{-1}=$ ?" $\mathbf{B}^{-1}\mathbf{A}^{-1}$ — order reverses
"Find $\mathbf{A}^k$ for general $k$" Compute $\mathbf{A}^2,\mathbf{A}^3$; conjecture pattern; (sometimes prove by induction)
Mixed product $\mathbf{ABC}$ where some don't fit Group via associativity: try $\mathbf{(BC)A}$ or $\mathbf{B(CA)}$ — sometimes one ordering is conformable, others not

8. Common exam pitfalls

Pitfall Fix
Multiplying matrices in the wrong order $\mathbf{AB}\neq\mathbf{BA}$ in general — read the question carefully
Adding matrices of different sizes They must be identical $n\times m$; otherwise undefined
Trying to invert a singular matrix $\det=0$ → no inverse exists; check first
Forgetting the $1/\det$ in the inverse formula $\mathbf{M}^{-1}=\frac{1}{ad-bc}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}$ — the scale factor is essential
Sign error in the inverse formula Swap $a\leftrightarrow d$ AND negate $b,c$ (not all four)
Reversing $(\mathbf{AB})^{-1}$ as $\mathbf{A}^{-1}\mathbf{B}^{-1}$ Wrong — it's $\mathbf{B}^{-1}\mathbf{A}^{-1}$ (order reverses)
Cancelling $\mathbf{AB}=\mathbf{AC}\Rightarrow\mathbf{B}=\mathbf{C}$ blindly Only valid if $\mathbf{A}$ is non-singular (so $\mathbf{A}^{-1}$ exists)
Treating "size $n\times m$" as $m\times n$ Convention is rows × columns. Check with calculator notation.
Multiplying $\binom{a}{b}\binom{c}{d}$ as if both were column vectors Two column vectors cannot multiply directly; you may need a transpose
Forgetting to pre/post-multiply on the SAME side both ways If you multiply by $\mathbf{A}^{-1}$ on left of LHS, must also do so on RHS

9. Big picture (the 30-second mental model)

  1. Matrices = grids of numbers, named by size $n\times m$. Special cases: square, zero, identity.
  2. Addition/scalar-mult: elementwise, easy. Matrix multiplication: row-by-column dot products, non-commutative, size-restrictive.
  3. The $2\times 2$ determinant $ad-bc$ tells you everything: zero → singular (no inverse), non-zero → non-singular (has inverse).
  4. The inverse is given by the swap-and-negate formula $\dfrac{1}{ad-bc}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}$, and satisfies $(\mathbf{AB})^{-1}=\mathbf{B}^{-1}\mathbf{A}^{-1}$ (order reverses).
  5. Solving matrix equations means multiplying by inverses on the correct side to isolate the unknown — left for left, right for right.

Everything else (powers, identities like $\mathbf{A}^2=2\mathbf{A}+5\mathbf{I}$, products $\mathbf{ABC}$) is just careful application of these basics. (FP3 will give matrices a geometric interpretation as transformations of the plane and 3D space.)
