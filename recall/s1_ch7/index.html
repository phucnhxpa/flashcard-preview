S1 Ch 7 — The Normal Distribution

The whole chapter in one mental model. Many continuous variables (heights, errors, weights) follow a bell curve $X\sim N(\mu,\sigma^2)$. To find $P(\text{event})$ we standardise via $Z=(X-\mu)/\sigma$ to land on the standard normal $Z\sim N(0,1)$, then look up $\Phi(z)=P(Z\le z)$ in the tables (or use percentage points). Symmetry $\Phi(-z)=1-\Phi(z)$ saves table lookups for negative $z$. To find unknowns $\mu,\sigma$, set up equations of the form $z_i=(x_i-\mu)/\sigma$ and solve simultaneously. The 68–95–99.7 rule sketches probabilities at $\mu\pm 1\sigma,2\sigma,3\sigma$.

6. Solving for $\mu$ and $\sigma$ — simultaneous equations

Given two probability statements about $X\sim N(\mu,\sigma^2)$:

Step Action
1 Translate each to "$P(X\lessgtr x_i)=p_i$"
2 Find $z_i$ from $p_i$ (inverse normal); pay attention to sign
3 Form linear equations: $z_i\sigma + \mu = x_i$ for each $i$
4 Solve simultaneously: subtract to eliminate $\mu$, find $\sigma$, back-substitute for $\mu$
5 Quote to 3 s.f.

Example: $P(X>35)=0.025$ and $P(X<15)=0.1469$.

  • $z_1=1.96$ (from $p=0.025$): $1.96\sigma+\mu=35$.
  • $z_2=-1.05$ (from $\Phi(z_2)=0.1469$): $-1.05\sigma+\mu=15$.
  • Subtract: $3.01\sigma=20\Rightarrow\sigma=6.64$. Back-sub: $\mu=22.0$.

9. Big picture (30-second mental model)

  1. One distribution, two parameters: $X\sim N(\mu,\sigma^2)$ has a bell shape centred at $\mu$ with width controlled by $\sigma$.
  2. Standardise to a single reference: $Z=(X-\mu)/\sigma\sim N(0,1)$. All probabilities reduce to lookups of $\Phi(z)$.
  3. Two tables: the main table lists $\Phi(z)$ for $z\ge 0$ in steps of 0.01; the percentage points table lists $z$ for round right-tail probabilities like 0.05, 0.025, 0.01.
  4. Symmetry: $\Phi(-z)=1-\Phi(z)$ — handles negative arguments without a second table.
  5. Inverse problems: Read off $z$, then de-standardise $x=\mu+z\sigma$.
  6. Two unknowns ($\mu,\sigma$): Set up two equations $z_i\sigma+\mu=x_i$, solve simultaneously.
  7. Empirical 68/95/99.7 rule: quick mental check for "within 1, 2, or 3 SDs of mean".

Standardise → look up → de-standardise — that's the chapter.
