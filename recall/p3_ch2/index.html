P3 Ch 2 — Functions and Graphs

The whole chapter in one mental model. A function is a rule that maps each input to a single output (one-to-one or many-to-one mapping). The chapter formalises domain (allowed inputs) and range (resulting outputs), then introduces three big operations: composition $\mathrm{fg}(x)=\mathrm{f}(\mathrm{g}(x))$, inversion $\mathrm{f}^{-1}$ (which only exists for one-to-one functions and is the reflection of $\mathrm{f}$ in $y=x$), and modulus transformations $|\mathrm{f}(x)|$ and $\mathrm{f}(|x|)$. Combining these with the standard set of graph transformations (translate, stretch, reflect) lets you sketch and solve problems involving piecewise/modulus equations and inequalities.

1. Jargon glossary

Term Meaning Where
Modulus / absolute value $\mid a\mid$ = the non-negative numerical value of $a$. So $\mid 5\mid=5$, $\mid -5\mid=5$. Defined piecewise: if $\mathrm{f}(x)\ge 0$ then $\mid\mathrm{f}(x)\mid=\mathrm{f}(x)$; if $\mathrm{f}(x)<0$ then $\mid\mathrm{f}(x)\mid=-\mathrm{f}(x)$. Geometrically: distance from 0 on the number line. 2.1
Mapping A rule pairing each element of one set (the domain) with one or more elements of another set (the range). Types: one-to-one, many-to-one, one-to-many, many-to-many. 2.2
Function A special mapping where every input has exactly one output. So one-to-one and many-to-one mappings are functions; one-to-many and many-to-many are NOT. 2.2
Domain The set of allowed inputs. Often written $x\in\mathbb{R}$ (any real) or restricted, e.g. $x\ge 2$. A mapping isn't a function until its domain is specified. 2.2
Range The set of resulting outputs (also called "image"). Read off a sketch as the set of $y$-values attained. Range depends on both the rule and the domain. 2.2
One-to-one Each input maps to a unique output AND each output comes from a unique input. The graph passes the horizontal line test. Such functions have inverses. 2.2
Many-to-one Multiple inputs can give the same output. E.g. $\mathrm{f}(x)=x^2$ on $\mathbb{R}$: $2$ and $-2$ both map to $4$. Doesn't have a (full) inverse — needs domain restriction first. 2.2
Piecewise function A function defined by different formulas on different parts of the domain, e.g. $\mathrm{f}(x)=5-2x$ for $x<1$, $\mathrm{f}(x)=x^2+3$ for $x\ge 1$. May be discontinuous at the join. 2.2
Composite function $\mathrm{fg}(x)$ $\mathrm{fg}(x)=\mathrm{f}(\mathrm{g}(x))$ — apply $\mathrm{g}$ first, then $\mathrm{f}$. Order matters: $\mathrm{fg}\neq\mathrm{gf}$ in general. The output of $\mathrm{g}$ must lie in the domain of $\mathrm{f}$. 2.3
Inverse function $\mathrm{f}^{-1}$ The function that undoes $\mathrm{f}$: $\mathrm{ff}^{-1}(x)=\mathrm{f}^{-1}\mathrm{f}(x)=x$. Exists only if $\mathrm{f}$ is one-to-one. Domain of $\mathrm{f}^{-1}$ = range of $\mathrm{f}$; range of $\mathrm{f}^{-1}$ = domain of $\mathrm{f}$. 2.4
Self-inverse A function $\mathrm{f}$ with $\mathrm{f}^{-1}=\mathrm{f}$, i.e. $\mathrm{ff}(x)=x$. Examples: $\mathrm{f}(x)=10-x$, $\mathrm{f}(x)=\frac{3}{x}$. Graph is symmetric in $y=x$. 2.4
Reflection in $y=x$ The geometric meaning of inversion: the graphs of $\mathrm{f}$ and $\mathrm{f}^{-1}$ are mirror images across the diagonal line $y=x$. So an intersection $\mathrm{f}(x)=\mathrm{f}^{-1}(x)$ lies on $y=x$ — solve $\mathrm{f}(x)=x$ instead. 2.4
$y=\mid\mathrm{f}(x)\mid$ transformation Reflect any part of $y=\mathrm{f}(x)$ that lies below the $x$-axis up into the $x$-axis. Parts above stay put. Output is always non-negative. 2.5
$y=\mathrm{f}(\mid x\mid)$ transformation Take the part of $y=\mathrm{f}(x)$ for $x\ge 0$ (right half) and reflect it in the $y$-axis to fill in the left half. Always produces an even (symmetric in $y$-axis) graph. 2.5
Argument of modulus The expression inside the modulus bars. To solve modulus equations, consider both signs of the argument: "argument $\ge 0$" and "argument $<0$" cases. 2.1
Translation $\mathrm{f}(x+a)$ Horizontal translation by vector $\binom{-a}{0}$ — graph shifts $a$ units to the left if $a>0$. (Sign flip caught most students once.) 2.6 (P1 recap)
Translation $\mathrm{f}(x)+a$ Vertical translation by vector $\binom{0}{a}$ — graph shifts $a$ units up if $a>0$. 2.6
Stretch $\mathrm{f}(ax)$ Horizontal stretch by scale factor $\dfrac{1}{a}$ (squashes by factor $a$ if $a>1$). 2.6
Stretch $a\mathrm{f}(x)$ Vertical stretch by scale factor $a$. 2.6
Reflection $\mathrm{f}(-x)$ Reflect graph in the $y$-axis. 2.6
Reflection $-\mathrm{f}(x)$ Reflect graph in the $x$-axis. 2.6
Order of transformations When applying $y=k\mathrm{f}(ax+b)+c$: do horizontal stuff first inside the bracket (translation before stretch/reflection inside); then vertical outside (stretch/reflection before translation outside). For modulus: transformations inside $\mid\dots\mid$ → modulus → transformations outside. 2.6

2. The modulus function — solving equations and inequalities

Setup Method Reason
$\mid\mathrm{f}(x)\mid=k$ for constant $k$ Solve two linear equations: $\mathrm{f}(x)=k$ AND $\mathrm{f}(x)=-k$. Reject $k<0$ (no solutions) The argument can be either sign
$\mid\mathrm{f}(x)\mid=\mathrm{g}(x)$ Sketch both, find intersections; algebraically solve $\mathrm{f}(x)=\mathrm{g}(x)$ AND $-\mathrm{f}(x)=\mathrm{g}(x)$, check each against sketch (some may be extraneous) Original-curve and reflected-curve intersections
$\mid\mathrm{f}(x)\mid<\mathrm{g}(x)$ Solve the boundary $\mid\mathrm{f}(x)\mid=\mathrm{g}(x)$ first, then read inequality off the sketch (where is $\mid\mathrm{f}\mid$ below $\mathrm{g}$?) Inequality direction depends on shape
$\mid\mathrm{f}(x)\mid>\mathrm{g}(x)$ Boundaries first, then read off where $\mid\mathrm{f}\mid$ is above $\mathrm{g}$. Often two regions: $x<\alpha$ or $x>\beta$ Same idea
Algebraic shortcut for $\mid X\mid=k$ ($k\ge 0$) $X=\pm k$ Standard

Pitfall (Example 10 of textbook): for $\mid 3x+4\mid=x$ a student wrote $-(3x+4)=x\Rightarrow x=-1$, but $\mid 3(-1)+4\mid=1\neq -1$. Always check algebraic solutions back in the original equation — modulus algebra can produce extraneous roots.

3. Composite functions

Notation Meaning Domain rule
$\mathrm{fg}(x)$ $\mathrm{f}(\mathrm{g}(x))$ — apply $\mathrm{g}$ first Output of $\mathrm{g}$ must lie in domain of $\mathrm{f}$
$\mathrm{f}^2(x)$ $\mathrm{ff}(x)=\mathrm{f}(\mathrm{f}(x))$ — apply $\mathrm{f}$ twice. (NOT $(\mathrm{f}(x))^2$.) Same as $\mathrm{ff}$
$\mathrm{fg}=\mathrm{gf}$? Almost never — composition is not commutative Test: try numbers
$(\mathrm{f}\circ\mathrm{g})^{-1}$ $\mathrm{g}^{-1}\circ\mathrm{f}^{-1}$ — "socks before shoes" inverted Standard property

Worked Example 12 (textbook): $\mathrm{f}(x)=\mid 2x-8\mid$, $\mathrm{g}(x)=\dfrac{x+1}{2}$. Then $\mathrm{fg}(x)=\mid 2\cdot\tfrac{x+1}{2}-8\mid=\mid x-7\mid$. Solving $\mathrm{fg}(x)=x$: from sketch, only one intersection on reflected part: $-(x-7)=x\Rightarrow x=3.5$.

4. Inverse functions — recipe

Step Action Detail
0 Confirm one-to-one (else inverse doesn't exist over the full domain) If many-to-one, restrict the domain (e.g., $x\ge 0$)
1 Write $y=\mathrm{f}(x)$ Just relabel
2 Make $x$ the subject — solve algebraically for $x$ in terms of $y$ Use rearrangement; for quadratics complete the square first
3 Swap the letters: write $x$ where $y$ was, etc. $\mathrm{f}^{-1}(x)=\dots$
4 State the domain of $\mathrm{f}^{-1}$ = range of $\mathrm{f}$ Read off original sketch
5 (Optional) State the range of $\mathrm{f}^{-1}$ = domain of $\mathrm{f}$ Symmetric

Key facts:

  • Domain $\mathrm{f}$ ↔ Range $\mathrm{f}^{-1}$
  • Range $\mathrm{f}$ ↔ Domain $\mathrm{f}^{-1}$
  • Graphs are reflections in $y=x$
  • Solutions to $\mathrm{f}(x)=\mathrm{f}^{-1}(x)$ lie on $y=x$, so equivalent to solve $\mathrm{f}(x)=x$

Worked Example 14: $\mathrm{f}(x)=\dfrac{3}{x-1}$. $y(x-1)=3\Rightarrow x=\dfrac{3+y}{y}$. Hence $\mathrm{f}^{-1}(x)=\dfrac{3+x}{x}$, $x\neq 0$.

5. Two modulus graphical transformations

Target graph Recipe (from $y=\mathrm{f}(x)$) What it does to features
$y=\mid\mathrm{f}(x)\mid$ 1. Sketch $y=\mathrm{f}(x)$. 2. Reflect any part below the $x$-axis into the $x$-axis. 3. Delete the original-below-axis part Always non-negative output. $x$-intercepts of $\mathrm{f}$ become corners of $\mid\mathrm{f}\mid$ (cusps)
$y=\mathrm{f}(\mid x\mid)$ 1. Sketch $y=\mathrm{f}(x)$ for $x\ge 0$ only. 2. Reflect this right-half in the $y$-axis to get the left-half Output is even function. The behaviour for $x<0$ in original $\mathrm{f}$ is irrelevant

Worked Example 17: $\mathrm{f}(x)=x^2-3x-10=(x-5)(x+2)$.

  • $y=\mathrm{f}(x)$: parabola, $x$-intercepts $-2,5$; $y$-intercept $-10$.
  • $y=\mid\mathrm{f}(x)\mid$: same parabola but the dip below axis (from $-2$ to $5$) is flipped up; new $y$-intercept $\mid -10\mid=10$.
  • $y=\mathrm{f}(\mid x\mid)$: take right half ($x\ge 0$), reflect to fill left half. Symmetric in $y$-axis.

6. Combining transformations — order matters

For $y=k\mathrm{f}(ax+b)+c$ (single inside, single outside):

Position Operation Apply order (within group)
Inside brackets $\mathrm{f}(ax+b)$ Translation (b) before stretch/reflection (a)
Outside brackets $k\mathrm{f}(\dots)+c$ Stretch/reflection (k) before translation (c)

Composite procedure: apply inside transformations first (then evaluate $\mathrm{f}$), then outside. Modulus inside: apply transformations inside $\mid\dots\mid$, then modulus, then transformations outside.

Worked Example 20a: $y=2\mathrm{f}(x)-1$ with key points $(0,0),(2,-1),(6,4)$. Apply vertical stretch ×2 first → $(0,0),(2,-2),(6,8)$. Then translate down 1 → $(0,-1),(2,-3),(6,7)$.

Why order matters: $2\mathrm{f}(x)-1\ne 2(\mathrm{f}(x)-1)=2\mathrm{f}(x)-2$.

7. Decision flow — what method to use

Question phrasing / spotted pattern Method
"Sketch $y=\mid ax+b\mid$" Sketch line, reflect below-axis piece in $x$-axis
"Solve $\mid f(x)\mid=k$" Two cases: $f(x)=k$ and $f(x)=-k$, check
"Solve $\mid f(x)\mid=g(x)$" Sketch both; or solve $f=g$ and $-f=g$; check answers
"Find domain / range of $\mathrm{f}$" Domain = inputs allowed; range = output set — read from sketch
"Find $\mathrm{fg}(x)$" $\mathrm{f}(\mathrm{g}(x))$ — substitute $\mathrm{g}(x)$ for $x$ in $\mathrm{f}$, simplify
"Find $\mathrm{f}^{-1}(x)$" Set $y=\mathrm{f}(x)$; solve for $x$ in terms of $y$; swap letters; state domain (= range of $\mathrm{f}$)
"Solve $\mathrm{f}(x)=\mathrm{f}^{-1}(x)$" Solve $\mathrm{f}(x)=x$ (intersection on $y=x$)
"Sketch $y=\mid\mathrm{f}(x)\mid$" Sketch $\mathrm{f}$; flip below-axis part up
"Sketch $y=\mathrm{f}(\mid x\mid)$" Sketch $\mathrm{f}$ for $x\ge 0$; reflect in $y$-axis
"Sketch $y=k\mathrm{f}(ax+b)+c$" Apply transformations: inside (translation b, then stretch a), then $\mathrm{f}$, then outside (stretch k, then translation c)

10 $\mathrm{t}(x)=3\mid x-1\mid -2$. Solve $\mathrm{t}(x)=\tfrac12 x+3$ Two cases: $3(x-1)-2=\tfrac12 x+3\Rightarrow x=\tfrac{16}{5}$; $-3(x-1)-2=\tfrac12 x+3\Rightarrow x=-\tfrac47$ $x=\tfrac{16}{5},-\tfrac47$

9. Common exam pitfalls

Pitfall Fix
Treating $\mid X\mid=k$ as just $X=k$ — missing the $X=-k$ branch Both signs of the argument always
Treating $\mid f\mid=g$ as $f=g$ only Solve $f=g$ AND $-f=g$, then check both back in original
Mixing $\mathrm{fg}$ and $\mathrm{gf}$: composition is not commutative $\mathrm{fg}\neq\mathrm{gf}$ in general — read carefully
$\mathrm{f}^2(x)$ misread as $(\mathrm{f}(x))^2$ $\mathrm{f}^2$ means $\mathrm{f}\mathrm{f}$ — apply $\mathrm{f}$ twice, not square the output
Trying to invert a many-to-one function without restricting domain Restrict to a one-to-one piece first (e.g. $x\ge 0$ for $x^2$)
Forgetting to state domain of $\mathrm{f}^{-1}$ Always: domain $\mathrm{f}^{-1}$ = range $\mathrm{f}$
Translation $\mathrm{f}(x+a)$ thought to shift right by $a$ Actually shifts left by $a$ if $a>0$
Order of transformations swapped: doing translation before stretch outside Outside the bracket: stretch/reflection FIRST, then translation
For $y=\mid\mathrm{f}(x)\mid$ vs $y=\mathrm{f}(\mid x\mid)$: confusing the two $\mid\mathrm{f}(x)\mid$ flips negative-$y$ regions up; $\mathrm{f}(\mid x\mid)$ replaces left half with reflection of right half
Solving $\mathrm{f}=\mathrm{f}^{-1}$ by computing $\mathrm{f}^{-1}$ first (slow) Easier: $\mathrm{f}=\mathrm{f}^{-1}$ on $y=x$, so just solve $\mathrm{f}(x)=x$
Forgetting graph corners (cusps) at modulus reflection points Mark these clearly on sketches; they're not smooth
Ignoring asymptotes when transforming Asymptotes also translate/stretch with the rest of the graph
Trying to solve $\mid f\mid<\mid g\mid$ algebraically without sketch Sketch first, find boundary intersections, read off intervals

10. Big picture (30-second mental model)

A function is a labelled rule with a stated domain that produces a unique output for each input.

Three operations combine functions:

  1. Composition $\mathrm{fg}(x)=\mathrm{f}(\mathrm{g}(x))$ — order matters; output of inner must lie in domain of outer.
  2. Inversion $\mathrm{f}^{-1}$ — only for one-to-one functions; reflect graph in $y=x$; swap domain↔range.
  3. Modulus transformations $\mid\mathrm{f}(x)\mid$ (flip below-axis pieces up) and $\mathrm{f}(\mid x\mid)$ (mirror right-half in $y$-axis).

Combining transformations ($y=k\mathrm{f}(ax+b)+c$): apply inside (translate, then stretch) → evaluate $\mathrm{f}$ → apply outside (stretch, then translate).

Solving modulus equations/inequalities: split into the two argument-sign cases, OR sketch both sides and read intersections off; always verify algebraic solutions.

The chapter is the algebra-graphical bridge: every problem reduces to (i) identifying the right kind of transformation/operation, (ii) applying it carefully, and (iii) reading the answer off a sketch.
