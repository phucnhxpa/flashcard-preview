FP2 Ch 1 — Inequalities

The whole chapter in one mental model. Solving inequalities boils down to finding critical values (where equality holds) and using a sketch (or sign analysis) to read off which intervals satisfy the inequality. The trap with algebraic fractions is that multiplying by a variable expression can flip the inequality sign — sidestep it by multiplying by the square of the denominator (always non-negative). Modulus inequalities use the same recipe but with the modulus equation $|f(x)|=g(x)$ split into two cases ($f=g$ and $-f=g$).

2. The four-step recipe — algebraic fractions

For inequalities like $\dfrac{f(x)}{g(x)}\lessgtr h(x)$:

Step Action
1 Multiply both sides by $(g(x))^2$ (always $\geq 0$, doesn't flip sign) — sometimes by the product of all denominator squares
2 Cancel & rearrange to put 0 on one side: $F(x)\lessgtr 0$
3 Factor $F(x)$ as a product of linear factors
4 Find critical values (where each factor = 0) and use a sketch (or sign table) to identify which intervals satisfy the inequality

Key non-strict subtlety: if the inequality is $\leq$ or $\geq$, include critical values where the LHS is defined and equality holds. Exclude any critical value that makes a denominator zero (undefined).

3. The two-case method for modulus inequalities

For $|f(x)|\lessgtr g(x)$:

Step Action
1 Solve modulus equation $|f(x)|=g(x)$ by considering $f(x)=g(x)$ AND $-f(x)=g(x)$
2 Each case yields some candidate $x$-values
3 Sketch $y=|f(x)|$ and $y=g(x)$ together to identify genuine intersections (some candidates may be extraneous)
4 Read off the inequality from the sketch — where one graph is above/below the other
5 Note: critical values include $x$ where $f$ changes sign (so the modulus structure changes)

Alternative (simple cases): $|f(x)|0$ converts directly to $-kk$ converts to $f(x)<-k$ or $f(x)>k$.

7. Modulus graphs cheatsheet

Curve Key shape
$y=\lvert x-a \rvert$ V-shape with vertex at $(a,0)$, slope $\pm 1$
$y=\lvert mx+c \rvert$ V-shape with vertex at $(-c/m,0)$, slope $\pm m$
$y=\lvert x^2-a \rvert$ Parabola $y=x^2-a$ with bits below axis flipped above (W-like for $a>0$)
$y=\lvert (x-p)(x-q) \rvert$ Reflect the dip between roots up to make a "double bump"
$y=\lvert f(x) \rvert$ general Reflect every part of $y=f(x)$ that's below the $x$-axis up above

Sketching trick: first sketch $y=f(x)$ normally (with axes intercepts, asymptotes, turning points), then fold up the negative pieces.

8. Common pitfalls (and the marks they cost)

Pitfall Fix
Multiplying by $(x-a)$ rather than $(x-a)^2$ $(x-a)$ may be negative; multiplying by it flips the sign uncontrolledly. Always use the squared form
Missing the asymptote as a critical value Vertical asymptotes split the number line into separate solution regions. Add them to your critical-value list
Including the asymptote in the solution set Function undefined → asymptote can't satisfy the inequality
For $\leq$, including a critical value that's at an asymptote Same fix — asymptote excluded
Modulus equation: keeping all "case" solutions blindly Sketch to identify which are real intersections; some are extraneous
Sketching too quickly when "use algebra" specified "Use algebra" = show factored form & critical values explicitly. Sketch is allowed only to identify intervals
Forgetting that $\lvert x\rvert\geq 0$ always $\lvert f(x)\rvertk$ for $k\leq 0$ holds for all $x$ in domain
Sign error when multiplying out factors after squaring Slow down; double check by plugging in a sample value at the end
Writing answer in interval form when set notation requested "Express in set notation" → use $\{x:\dots\}\cup\{x:\dots\}$
Off-by-one with strict/non-strict at endpoints Strict $<$: open intervals (no endpoint). Non-strict $\leq$: closed (include endpoint if defined)
Treating $\lvert f(x)\rvert If $g(x)<0$, no solutions; this isn't a non-issue, must check sign of $g$
Multiplying by $\lvert x\rvert+c$ (positive) OK because $\lvert x\rvert+c>0$ always (for $c>0$) — doesn't flip sign

9. Big picture (the 30-second mental model)

  1. Inequalities reduce to finding critical values (zeros of $f(x)-g(x)$ and asymptotes), then deciding sign of an expression on each interval.
  2. Algebraic fractions: multiply by squared denominators (preserves sign), rearrange, factor, find critical values, use sketch/sign-table to read off intervals.
  3. Graphical method: sketch both sides on same axes, intersection points = critical values, "above/below" comparison gives the solution.
  4. Modulus: split equation $\lvert f\rvert=g$ into two cases ($f=g$, $-f=g$), find candidates, sketch both sides to keep only genuine intersections, read off inequality.
  5. Endpoint care: strict $<$ → exclude. Non-strict $\leq$ → include UNLESS function is undefined (e.g. denominator zero) at that endpoint.
  6. Set notation $\{x:\dots\}\cup\{x:\dots\}$ joins disjoint intervals.

Three methods, one philosophy: every inequality becomes a sign analysis on intervals between critical values.
