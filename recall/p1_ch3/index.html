P1 Ch 3 — Equations and Inequalities

The whole chapter in one mental model. Combine equations (linear+linear → 1 point; linear+quadratic → 0/1/2 points) by elimination or substitution, with the discriminant of the resulting quadratic deciding "how many intersections". For inequalities, rearrange to RHS = 0, find critical values from the equation, then sketch the graph and read off the regions where the curve sits on the correct side of the $x$-axis. Multiple inequalities → use a number line (AND = overlap, OR = union); 2-D inequalities → shade regions on a coordinate grid.

1. Jargon glossary

Term Meaning Where
Simultaneous equations A system of equations that must all be true at the same time, in the same unknowns. The "solution" is the set of values that satisfies every equation in the system, not just one. Geometrically: the intersection point(s) of all the curves. 3.1
Elimination method Solve a linear system by scaling one (or both) equations so a chosen variable has the same coefficient, then add or subtract to make it disappear. Add when signs differ, subtract when signs match. Leaves a one-variable equation. 3.1 Ex 1
Substitution method Rearrange one equation to $x=\dots$ or $y=\dots$, then plug that expression into the other equation. Mandatory when one equation is non-linear (quadratic): you must substitute from the linear into the quadratic, never the other way. 3.1 Ex 2
Quadratic simultaneous equations A linear-and-quadratic system $\{L=0,\ Q=0\}$ where $Q$ may contain $x^2,y^2,xy$. Solved by substituting from the linear into the quadratic, getting a single-variable quadratic, then pairing $(x,y)$ values correctly. Up to 2 solution pairs. 3.2
Pairing solutions After getting two values of $y$ (say), each one must be put back into the linear equation to find its matching $x$. Don't shuffle — $(x_1,y_1)$ and $(x_2,y_2)$ stay together. 3.2 Ex 3
Discriminant test for intersections After substituting a line into a curve you get $ax^2+bx+c=0$. The sign of $b^2-4ac$ tells you the number of real intersection points: $>0$ two, $=0$ one (tangency), $<0$ none. The same algebra that classifies roots classifies intersections. 3.3
Tangent condition A line meets a curve "at exactly one point" $\iff$ the substitution-quadratic has $b^2-4ac=0$. This is the standard recipe for problems that say "find $k$ such that the line is tangent". 3.3 Ex 6
Linear inequality An inequality of the form $ax+b\,(\square)\,c$ with $\square\in\{<,\le,>,\ge\}$, solved like an equation except that multiplying or dividing by a negative number flips the inequality sign. The solution is a half-line, not a single number. 3.4
Set notation The compact way to write solution sets: $\{x:x\ge 2.75\}$, $\{x:-23\}$. $\cap$ = intersection (AND), $\cup$ = union (OR). Round-bracket end means strict ($<$ or $>$); square-bracket end means inclusive ($\le$ or $\ge$). 3.4
Open vs closed end On a number line, an open circle ◦ marks a strict inequality ($<$ or $>$) — the endpoint is not included. A closed (filled) circle ● marks $\le$ or $\ge$ — endpoint is included. Same convention as round vs square brackets in interval notation. 3.4
AND vs OR (combined inequalities) "AND" = both must hold = take the intersection = overlap on the number line. "OR" = at least one holds = take the union = combined shaded region. Result might be a single interval, two disjoint intervals, the empty set, or all reals. 3.4 Ex 8
Quadratic inequality $ax^2+bx+c\,(\square)\,0$. Solved by: (1) rearrange to RHS $=0$; (2) factorise/solve to find critical values; (3) sketch the parabola; (4) read off where the curve is above/below the $x$-axis as required. 3.5
Critical values The roots of the corresponding equation $ax^2+bx+c=0$. They split the real line into intervals, on each of which the sign of $ax^2+bx+c$ is constant. Mark them on the sketch as $x$-intercepts. 3.5
"Above" vs "below" the $x$-axis After sketching $y=ax^2+bx+c$, "$y>0$" means the curve is above the $x$-axis (use the outer parts if the parabola opens up, the middle part if it opens down). "$y<0$" means below — opposite regions. Visual check beats sign tables. 3.5
Multiplying inequality by $x$ Forbidden if you don't know the sign of $x$ — multiplying by a negative number flips the inequality. Trick: multiply by $x^2$ (always non-negative, never zero if $x\ne 0$), so the sign is preserved. Then proceed as a polynomial inequality. 3.5 Ex 11
No-real-roots condition "Equation $ax^2+bx+c=0$ has no real roots" $\iff b^2-4ac<0$. This is itself an inequality in the parameters $a,b,c$ — solve it (often a quadratic in $k$) to find the parameter range. 3.5 Q5, Q9
Inequality on graphs (curve-vs-curve) $\mathrm{f}(x)<\mathrm{g}(x)$ means: find all $x$ where the graph of $f$ sits below the graph of $g$. Solve $f=g$ for the $x$-coordinates of intersections, then read off which intervals satisfy the strict inequality. 3.6
Region (2-D inequality) A shaded subset of the plane satisfying conditions like $y<\mathrm{f}(x)$. $y<\mathrm{f}(x)$ = below the curve; $y>\mathrm{f}(x)$ = above. Boundary curve = dotted if strict ($<,>$), solid if inclusive ($\le,\ge$). Test a point inside to confirm. 3.7
Vertex of a region A corner of a polygonal shaded region — intersection of two boundary lines. A vertex belongs to the region only if both of its bounding lines are themselves included (solid). If either is dotted, the vertex is excluded. 3.7 Q8

2. Solving simultaneous equations — choose the method

System type Method Steps Number of solutions Trigger example
Both linear (easy elimination) Elimination Scale to match a coefficient, add/subtract to kill it, back-substitute Exactly 1 (unless parallel) $2x+3y=8,\ 3x-y=23$
Both linear (one already isolated) Substitution Use $y=\dots$, plug in, solve, back-sub Exactly 1 $2x-y=1,\ 4x+2y=-30$
One linear + one quadratic Substitution only From linear get $x=\dots$ or $y=\dots$; sub into quadratic; solve resulting 1-variable quadratic; pair $(x,y)$ 0, 1, or 2 pairs $x+2y=3,\ x^2+3xy=10$
Linear + quadratic with parameter $k$ Sub $\to$ quadratic in $x$; apply discriminant condition Same as above + use $b^2-4ac=0$ for tangency, $>0$ for 2 intersections Use discriminant to count $y=2x+1,\ kx^2+2y+(k-2)=0$ "exactly one point"

Pitfall: when using brackets for substitution, e.g. $(3-2y)^2$, expand carefully — common arithmetic loss-of-marks. Always check by substituting answer into both original equations.

3. Discriminant decision table (line vs curve)

After substituting the line into the curve, you have $ax^2+bx+c=0$. Then:

Discriminant Geometric meaning Algebraic meaning Typical exam phrasing
$b^2-4ac>0$ Line cuts curve at two distinct points Two real distinct roots "the line intersects the curve at two points"
$b^2-4ac=0$ Line is tangent to curve (touches once) One repeated real root "exactly one point of intersection", "the line is a tangent"
$b^2-4ac<0$ Line misses the curve entirely No real roots "the line does not meet the curve"

For "find $k$ such that …" problems: write $b^2-4ac\ \square\ 0$ as an inequality/equation in $k$ and solve it (typically itself a quadratic in $k$).

4. Linear inequality manipulation rules

Operation Effect on inequality sign Example
Add/subtract any quantity from both sides Preserved $5x+9\ge x+20 \to 4x+9\ge 20$
Multiply/divide by a positive constant Preserved $4x\ge 11 \to x\ge 2.75$
Multiply/divide by a negative constant Flip $-3x<15 \to x>-5$
Multiply both sides by $x$ when sign of $x$ unknown Forbidden — case-split or use $x^2$ $\dfrac{6}{x}>2$: multiply by $x^2$, not $x$
Multiply both sides by $x^2$ ($x\ne 0$) Preserved (since $x^2>0$) $\dfrac{6}{x}>2 \to 6x>2x^2$
Multiply by $(x-3)^2$ ($x\ne 3$) Preserved $\dfrac{5}{x-3}<2 \to 5(x-3)<2(x-3)^2$

5. Quadratic inequality recipe (Section 3.5)

Step Action Output
1 Rearrange so RHS $=0$. If leading coefficient is negative, optionally multiply by $-1$ and flip sign $ax^2+bx+c\ \square\ 0$ with $a>0$ ideal
2 Solve the equation $ax^2+bx+c=0$ (factorise / formula) Two critical values $\alpha<\beta$
3 Sketch the parabola: $a>0$ opens up, $a<0$ opens down; mark roots on $x$-axis Picture
4 Read regions where $y$ has the required sign — between the roots = "middle bit", outside the roots = "outer bits" Solution set
5 Translate to set notation if asked $\{x:\alpha\beta\}$

Quick lookup (for $a>0$, distinct roots $\alpha<\beta$):

Inequality Solution set
$ax^2+bx+c<0$ $\alpha
$ax^2+bx+c\le 0$ $\alpha\le x\le\beta$ (middle, inclusive)
$ax^2+bx+c>0$ $x<\alpha$ or $x>\beta$
$ax^2+bx+c\ge 0$ $x\le\alpha$ or $x\ge\beta$

For $a<0$, swap the answers (or pre-multiply by $-1$ and flip sign first).

6. Combining inequalities (Section 3.4)

Logical connective Meaning Number-line picture Set notation
AND ($\cap$) Both inequalities hold Overlap of the two shaded segments $\{x:a
OR ($\cup$) At least one holds Combined shading (any covered point is in) $\{x:xb\}$
AND with no overlap Empty intersection No point shaded twice $\varnothing$
OR covers everything Union = $\mathbb{R}$ All of the line shaded $\mathbb{R}$

Inequality-with-quadratic-and-linear problems (e.g. $x^2-7x+10<0$ AND $3x+5<17$): solve each inequality separately, draw both ranges on the same number line, take overlap (AND) or union (OR).

7. Rational/reciprocal inequalities

Form Rewrite Method
$\dfrac{p}{x}\,\square\,q$ ($x\ne 0$) Multiply by $x^2$ to clear the fraction safely $px\,\square\,qx^2$ → polynomial inequality → critical values include $x=0$ as a divider
$\dfrac{p}{x^2}+\dfrac{q}{x}\,\square\,r$ Multiply by $x^2$ $p+qx\,\square\,rx^2$ → quadratic inequality
$\dfrac{p}{x-a}\,\square\,q$ ($x\ne a$) Multiply by $(x-a)^2$ Polynomial inequality with critical values around $x=a$

Watch out: $x=0$ (or $x=a$) is always excluded from the solution even if the algebra suggests it's a critical value, because the original expression is undefined there.

8. Curve-vs-curve / line-vs-curve inequalities (Section 3.6)

For $\mathrm{f}(x)\,\square\,\mathrm{g}(x)$:

Step Action
1 Solve $\mathrm{f}(x)=\mathrm{g}(x)$ for $x$-coordinates of intersection points
2 Sketch both graphs on the same axes (or rearrange to $\mathrm{f}(x)-\mathrm{g}(x)\,\square\,0$ and treat as a single curve vs $x$-axis)
3 Read off the $x$-intervals where $f$ is below $g$ (for $fg$)

Equivalent: rearrange $\mathrm{f}(x)<\mathrm{g}(x)$ to $\mathrm{f}(x)-\mathrm{g}(x)<0$ and apply the quadratic inequality recipe (§5) to the difference function.

9. 2-D regions (Section 3.7)

Inequality Region Boundary line/curve
$y>\mathrm{f}(x)$ Above the curve Dotted (strict)
$y\ge\mathrm{f}(x)$ Above + on the curve Solid (inclusive)
$y<\mathrm{f}(x)$ Below the curve Dotted
$y\le\mathrm{f}(x)$ Below + on the curve Solid
$x>k$ Right of vertical line $x=k$ Dotted vertical
$x\ge k$ Right + on line Solid vertical

Procedure for "shade the region satisfying these inequalities":

  1. Draw each boundary (dotted/solid as per strictness).
  2. For each inequality, decide which side it labels.
  3. The required region is the intersection of all sides.
  4. Sanity check: pick one point clearly inside the candidate region; substitute into every inequality. All should hold.

For "write down the inequalities" reverse problem: identify each boundary curve, work out which side is shaded (test a point), pick $<$/$\le$/$>$/$\ge$ from line style.

10. Common exam pitfalls

Pitfall Fix
Mis-pairing $(x,y)$ values when one linear + one quadratic gives 2 pairs Sub each $y$-value into the linear equation (simpler) to recover its matching $x$
Forgetting to flip inequality when dividing by a negative Underline "$-$" sign in the divisor as a flag before dividing
Multiplying inequality by $x$ assuming $x>0$ Use $x^2$ instead, OR case-split $x>0$ / $x<0$
Quadratic inequality: confusing "above" and "below" axis Always sketch first; label the parabola direction (cup/cap); shade where $y$ has the sign you need
Reading the wrong region of the parabola sketch Mark on the sketch which part is $y>0$ and $y<0$ before reading off
Discriminant test: forgetting to fully simplify the substituted equation before reading $a,b,c$ Always rearrange to standard form $ax^2+bx+c=0$ first; identify $a,b,c$ once
Set notation: writing $\{x<-1\cup x>5\}$ Correct form: $\{x:x<-1\}\cup\{x:x>5\}$ — each set needs its own builder
Strict vs inclusive endpoint in 2-D region Match boundary line style to inequality strictness: $<,>$ → dotted; $\le,\ge$ → solid
"No real roots" requires $b^2-4ac<0$ — students write $\le$ Strict: $b^2-4ac<0$ ($=0$ would be one repeated root, which is a real root)

12. Big picture (30-second mental model)

Two giant moves underlie the whole chapter:

  1. Substitution + discriminant is the master technique for "how many intersections" — line and curve become a quadratic in one variable, and $b^2-4ac$ counts the meeting points.
  2. Rearrange-to-zero + critical-values + sketch is the master technique for inequalities — turn the inequality into "is this expression $>0$ or $<0$ between which roots", then read regions off a sketch.

For combinations (multi-step / multi-inequality problems): use a number line for 1-D (overlap = AND, union = OR) or a 2-D grid for $(x,y)$ regions (intersection of half-planes/curve sides; dotted vs solid boundaries match the inequality strictness). Every problem in this chapter is one of these patterns.
