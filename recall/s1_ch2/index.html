S1 Ch 2 — Measures of Location and Spread

The whole chapter in one mental model. Take any data set and squeeze out two numbers: a location (centre — mean / median / mode / quartile / percentile) and a spread (range / IQR / interpercentile / variance / SD). For grouped data you estimate centre & spread using class midpoints and interpolation. Coding $y=\tfrac{x-a}{b}$ shifts the mean by $-a/b$ but only scales the SD by $1/b$ — additive shifts vanish from spread.

1. Jargon glossary

Term Meaning Where
Quantitative vs qualitative Quantitative variables produce numbers (heights, times, scores); qualitative ones produce categories (colours, names, brands). Only quantitative data has a meaningful mean/median/SD; qualitative data is restricted to mode and proportions. 2.1
Discrete vs continuous Discrete variables take isolated values from a list (number of coins, shoe size — gaps allowed in between). Continuous variables can take any value in a range, limited only by measuring precision (mass, time, length). The distinction drives whether you draw bar charts or histograms. 2.1
Frequency table / grouped frequency table A compact way of writing many observations: each row holds a value (or class) $x$ alongside how often it occurred, $f$. Grouped tables bin continuous data into classes; you lose the exact data but gain readability. 2.1
Class boundaries The actual numerical limits of a class. If the table reads "55–65" with no gaps, boundaries are exactly 55 and 65; if rounded data has gaps like "55–65, 66–70", boundaries shift to 65.5 / 65.5 (halfway). Used for class width and interpolation. 2.1 Ex 2
Midpoint Centre of a class: $\tfrac{1}{2}(L+U)$ where $L,U$ are class boundaries. Used as the value of $x$ in any formula on grouped data — assumes data are evenly spread within each class. 2.1 Ex 2
Mean $\bar x$ The arithmetic average $\bar x = \dfrac{\sum x}{n}$ (raw) or $\bar x = \dfrac{\sum fx}{\sum f}$ (frequency table). Uses every value, so distorted by outliers. The "balance point" of the distribution — but not necessarily a value that exists in the data. 2.2
Median $Q_2$ The middle value once the data are ordered: the $\tfrac{n+1}{2}$-th value for discrete data, the $\tfrac{n}{2}$-th value for grouped/cumulative data. Robust to outliers — splits the dataset 50/50. 2.2/2.3
Mode / modal class The most-frequent value (or class with the highest frequency). The only "average" usable for qualitative data. A dataset can be unimodal, bimodal, or have no mode. 2.2
Combined mean When two sub-samples are merged: $\bar x = \dfrac{n_1\bar x_1+n_2\bar x_2}{n_1+n_2}$. Reweights each group's mean by its size — not the simple average of the means unless $n_1=n_2$. 2.2
Quartiles $Q_1,Q_2,Q_3$ The values that split ordered data into four equal blocks. Discrete rule: compute $\tfrac{n}{4}$ or $\tfrac{3n}{4}$; if integer → midway between that value and the next, if not → round up. Grouped: use $\tfrac{n}{4}$-th value with interpolation. 2.3
Percentile $P_k$ Value $k\%$ of the way through the ordered data; $P_{10}$ is the 10th percentile, $Q_2=P_{50}$. Found by interpolation in grouped tables: $\frac{P_k - L}{U-L} = \frac{kn/100 - F_<}{f}$. 2.3 Ex 7
Linear interpolation Estimating a value inside a class by assuming uniform spread. If the value you want sits a fraction $p$ of the way through a class, place it $p$ of the way between its boundaries: $Q = L + p(U-L)$ where $p=\frac{\text{rank}-F_<}{f}$. 2.3
Range $\text{max}-\text{min}$. Crude — uses only two values, distorted by outliers. Ideal for "what's the worst-case spread?" but not robust. 2.4
Interquartile range (IQR) $Q_3-Q_1$ — spread of the middle 50%. Robust to outliers because it ignores both tails. The default "honest spread" in box-plot summaries. 2.4
Interpercentile range $P_b-P_a$ for any chosen $a 2.4
Variance $\sigma^2$ Mean squared deviation from the mean: $\sigma^2 = \dfrac{\sum(x-\bar x)^2}{n} = \dfrac{\sum x^2}{n}-\left(\dfrac{\sum x}{n}\right)^2 = \dfrac{S_{xx}}{n}$. "Mean of the squares minus the square of the mean." Units are squared, which is why we usually report $\sigma$. 2.5
Standard deviation $\sigma$ $\sigma=\sqrt{\sigma^2}$ — same units as the data, so directly comparable to $\bar x$. About 2/3 of a normal-shaped dataset lies within $\bar x\pm\sigma$. The single most-used measure of spread. 2.5
$S_{xx}$ Compact notation: $S_{xx}=\sum(x-\bar x)^2 = \sum x^2 - \dfrac{(\sum x)^2}{n}$. Lets you write variance as $\sigma^2=S_{xx}/n$. Often given pre-computed in exam questions. 2.5
Coding A linear transformation $y=\dfrac{x-a}{b}$ applied before computing statistics, to keep the numbers tidy. After computing $\bar y,\sigma_y$, decode: $\bar x = b\bar y+a$, $\sigma_x=b\sigma_y$. The shift $a$ vanishes from $\sigma$ — only the scale $b$ matters for spread. 2.6

2. Measures of central tendency — formula table

The "best" average depends on the data type and presence of outliers.

Measure Raw data Ungrouped frequency table Grouped frequency table When to use Outlier-robust?
Mode Most-frequent value Value with highest $f$ Modal class (highest $f$) Qualitative data, "most popular" questions Yes
Median $Q_2$ $\tfrac{n+1}{2}$-th ordered value $\tfrac{n+1}{2}$-th value (cumulative $f$) $\tfrac{n}{2}$-th value via interpolation Skewed / outlier-prone data Yes
Mean $\bar x$ $\dfrac{\sum x}{n}$ $\dfrac{\sum fx}{\sum f}$ $\dfrac{\sum fx}{\sum f}$ with $x=$midpoint (estimate) Clean, symmetric data — uses every value No
Combined mean — — — Merging two groups $\bar x=\dfrac{n_1\bar x_1+n_2\bar x_2}{n_1+n_2}$

3. Other measures of location — quartile & percentile rules

Different rules for discrete vs grouped data — don't mix them up.

Quantity Discrete data rule Grouped/cumulative data rule Interpolation formula
$Q_1$ (lower quartile) $\tfrac{n}{4}$. Integer → midway with next value; non-integer → round up $\tfrac{n}{4}$-th value $\dfrac{Q_1-L}{U-L}=\dfrac{n/4-F_<}{f}$
$Q_2$ (median) $\tfrac{n+1}{2}$-th value $\tfrac{n}{2}$-th value $\dfrac{Q_2-L}{U-L}=\dfrac{n/2-F_<}{f}$
$Q_3$ (upper quartile) $\tfrac{3n}{4}$. Integer → midway; non-integer → round up $\tfrac{3n}{4}$-th value $\dfrac{Q_3-L}{U-L}=\dfrac{3n/4-F_<}{f}$
$P_k$ (kth percentile) $\tfrac{kn}{100}$, same rounding rule $\tfrac{kn}{100}$-th value $\dfrac{P_k-L}{U-L}=\dfrac{kn/100-F_<}{f}$

Notation: $L,U$ = lower/upper class boundaries containing the rank; $f$ = frequency of that class; $F_<$ = cumulative frequency before the class.

4. Measures of spread — formulas

Measure Formula Robustness Uses
Range $\text{max}-\text{min}$ Worst — single outlier ruins it Quick spread, error bounds
IQR $Q_3-Q_1$ Excellent — ignores both tails Box plots, comparison
10–90 IPR $P_{90}-P_{10}$ Strong — ignores extreme 10% each side Robust spread using 80% of data
Variance $\sigma^2$ $\dfrac{\sum(x-\bar x)^2}{n}=\dfrac{\sum x^2}{n}-\bar x^2 = \dfrac{S_{xx}}{n}$ Sensitive to outliers (squares amplify) Building block for SD, normal model
Standard deviation $\sigma$ $\sqrt{\sigma^2}$ — same units as data Sensitive "Average distance from $\bar x$"
Frequency-table SD $\sigma=\sqrt{\dfrac{\sum fx^2}{\sum f}-\left(\dfrac{\sum fx}{\sum f}\right)^2}$ — Apply to midpoints for grouped data (gives estimate)

Mnemonic for variance: "Mean of the squares minus the square of the mean." $$\sigma^2=\overline{x^2}-(\bar x)^2.$$

5. Variance — three equivalent forms (when to use which)

Form Best when
$\dfrac{\sum(x-\bar x)^2}{n}$ You only have a few values and want to show working with deviations
$\dfrac{\sum x^2}{n}-\left(\dfrac{\sum x}{n}\right)^2$ Question gives you $\sum x$ and $\sum x^2$ — plug-and-chug
$\dfrac{S_{xx}}{n}$ Question gives $S_{xx}$ pre-computed (or you ran calculator stats)

6. Coding — how each statistic transforms

Transform: $y=\dfrac{x-a}{b}$ (or rearranged $x=by+a$).

Original statistic Coded statistic Decode (recover $x$-stat) Why
Mean $\bar x$ $\bar y = \dfrac{\bar x - a}{b}$ $\bar x = b\bar y + a$ Mean is linear in $x$
Variance $\sigma_x^2$ $\sigma_y^2 = \dfrac{\sigma_x^2}{b^2}$ $\sigma_x^2 = b^2\sigma_y^2$ Squaring the scaling factor
Standard deviation $\sigma_x$ $\sigma_y = \dfrac{\sigma_x}{b}$ $\sigma_x = b\sigma_y$ The shift $a$ doesn't affect spread
$S_{xx}$ $S_{yy} = \dfrac{S_{xx}}{b^2}$ $S_{xx} = b^2 S_{yy}$ —
Median, quartiles $Q_y = \dfrac{Q_x-a}{b}$ $Q_x = b Q_y + a$ Order-statistics transform like the mean

Key insight: adding a constant changes the centre but not the spread. Multiplying by a constant scales both centre and spread.

7. Decision flow — choosing centre & spread

Situation Centre to use Spread to use Reason
Qualitative data Mode — No arithmetic possible
Symmetric quantitative, no outliers Mean SD Mean uses every datum
Skewed / has outliers Median IQR Both robust
Grouped frequency table Estimated mean (midpoints) or median class Estimated SD or IQR (interpolate) Exact values lost — must estimate
Two combined groups Combined mean formula Re-pool $\sum x,\sum x^2$ then SD Don't average SDs!
Coded data given Decode: $\bar x=b\bar y+a$ Decode: $\sigma_x=b\sigma_y$ Constant shift drops

8. Worked archetypes (memorise the shape)

# Setup Method
1 Raw list of 7 marks: $3,4,6,2,8,8,5$. Find $\sigma$. $\sum x=36,\sum x^2=218 \to \sigma^2=\tfrac{218}{7}-\left(\tfrac{36}{7}\right)^2=4.69 \to \sigma=2.17$
2 Frequency table of shirt sizes; find mean. $\bar x = \dfrac{\sum fx}{\sum f}$ — multiply each $x$ by $f$, sum, divide
3 Combined groups: $n_1=8,\bar x_1=65;\ n_2=12,\bar x_2=72$. $\bar x=\dfrac{8\cdot 65+12\cdot 72}{20}=69.2$
4 Grouped table; find $Q_3$ via interpolation. Locate class containing $\tfrac{3n}{4}$th value; $Q_3=L+\dfrac{3n/4 - F_<}{f}(U-L)$
5 Coded $y=(x-300)/10$, $\bar y=3,\sigma_y=1.72$. Find $\bar x,\sigma_x$. $\bar x = 10(3)+300=330$; $\sigma_x=10(1.72)=17.2$
6 $\bar x,\sigma$ given for $x$. New variable $z=1.04 x$. Find new $\bar z,\sigma_z$. $\bar z=1.04\bar x$; $\sigma_z=1.04\sigma$ (purely scaling)

9. Common exam pitfalls

Pitfall Fix
Using $\tfrac{n+1}{2}$ for grouped median instead of $\tfrac{n}{2}$ Discrete: $\tfrac{n+1}{2}$. Grouped: $\tfrac{n}{2}$.
Forgetting to round up non-integer $\tfrac{n}{4}$ for discrete quartile Integer → midway. Non-integer → round up.
Including the shift $a$ when finding $\sigma_x$ from $\sigma_y$ $\sigma$ ignores $a$: $\sigma_x=b\sigma_y$ only.
Adding the SDs of two groups NEVER. Re-compute $\sum x,\sum x^2$ for the pooled set.
Using class boundaries instead of midpoints for mean Mean uses midpoints. Boundaries are for interpolation only.
Forgetting class boundaries shift by $\pm 0.5$ when data has gaps "5.0–5.9, 6.0–6.9" → boundaries 4.95/5.95/6.95 (rounded data has gaps)
Variance has wrong units in answer Variance $=$ data units$^2$. Answer should report $\sigma$ (same units as data) when comparing to $\bar x$.
Square-rooting before subtracting in $\sigma^2 = \overline{x^2}-\bar x^2$ Order: square the values first, then subtract, then square-root for $\sigma$.

10. Big picture (30-second mental model)

  1. Two numbers per dataset: a centre and a spread. Pick the pair (mean+SD or median+IQR) based on outliers.
  2. Grouped data → estimate. Use midpoints for $\bar x$ and $\sigma$; use linear interpolation for $Q_1,Q_2,Q_3,P_k$ — assume uniform spread within each class.
  3. Variance shortcut: $\sigma^2=\overline{x^2}-\bar x^2 = \dfrac{S_{xx}}{n}$. For frequency data, every $\sum$ becomes $\sum f\cdot$.
  4. Coding rescues messy numbers: subtract a base $a$, divide by a step $b$. Mean shifts and scales; SD only scales (constants don't change spread).
  5. Combining groups: weight means by size; pool $\sum x$ and $\sum x^2$ before re-computing $\sigma$.

Two formulae and three rounding rules — that's the chapter.
