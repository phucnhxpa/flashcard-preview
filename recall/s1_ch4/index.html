S1 Ch 4 — Probability

The whole chapter in one mental model. Three rules drive every problem: (1) Addition $P(A\cup B)=P(A)+P(B)-P(A\cap B)$ — overlap is double-counted; (2) Multiplication $P(A\cap B)=P(A)\cdot P(B|A)$ — chain conditional probabilities; (3) Complement $P(A')=1-P(A)$. Independent ⇒ multiply directly. Mutually exclusive ⇒ add directly. Use Venn diagrams for $\cap/\cup/'$, two-way tables for conditional, tree diagrams for sequential events.

1. Jargon glossary

Term Meaning Where
Experiment A repeatable process whose result can be observed/recorded — flipping a coin, rolling dice, drawing a card. The "physical" thing happening before any maths. 4.1
Outcome A single possible result of an experiment (e.g. "rolled a 3"). Every outcome is atomic — it cannot be split further. 4.1
Event A collection of one or more outcomes — the thing we're interested in. "Rolled an even number" is an event with three outcomes $\{2,4,6\}$. 4.1
Sample space $\mathscr{E}$ The set of all possible outcomes. Often the rectangle in a Venn diagram. Usually labelled $\mathscr{E}$ (or $S,\xi$) and has total probability 1. 4.1
Probability $P(A)$ A number in $[0,1]$ measuring how likely event $A$ is. For equally-likely outcomes: $P(A)=\dfrac{n(A)}{n(\mathscr{E})}$. 0 = impossible, 1 = certain. 4.1
Intersection $A\cap B$ "$A$ AND $B$" — the event that both $A$ and $B$ occur. Shown as the overlap of two circles in a Venn diagram. 4.4
Union $A\cup B$ "$A$ OR $B$" — at least one of $A,B$ occurs (inclusive or — both is allowed). The combined area of the two circles. 4.4
Complement $A'$ "NOT $A$" — the event that $A$ does not occur. Always $P(A)+P(A')=1$. Used constantly: "find probability of at least one ⇒ $1-P(\text{none})$". 4.4
Empty set $\varnothing$ The set with no outcomes. If $A\cap B=\varnothing$, the events are mutually exclusive. 4.4
Mutually exclusive Events that cannot occur together: $A\cap B=\varnothing$, so $P(A\cap B)=0$. Their addition rule simplifies: $P(A\cup B)=P(A)+P(B)$. Disjoint circles in a Venn diagram. 4.3
Independent Events where one's occurrence does not affect the other: $P(A\cap B)=P(A)P(B)$. Equivalently, $P(A\mid B)=P(A)$. Test for independence by checking if this product equation holds. 4.3
Conditional probability $P(B\mid A)$ The probability of $B$ given that $A$ has occurred — restricts the sample space to $A$: $P(B\mid A)=\dfrac{P(A\cap B)}{P(A)}$. The defining engine of "tree diagram" branches. 4.5
Restricted sample space The sub-region of $\mathscr{E}$ corresponding to a given event. To find $P(B\mid A)$, "zoom in" on $A$ and ask what fraction of that zoomed view is in $B$. 4.6
Two-way table A grid showing frequencies (or probabilities) cross-tabulated by two variables. Row totals + column totals + grand total. Reading conditional probabilities = restricting to one row or column. 4.5
Tree diagram Branching diagram for sequential events. Multiply along branches for $P(\text{path})$; add across paths for compound events. Probabilities on the second-tier branches are conditional. 4.7
Set notation $\cap$ = intersection (and), $\cup$ = union (or), $'$ = complement (not), $\varnothing$ = empty, $n(A)$ = number of outcomes in $A$. Compact substitute for English. 4.4
Addition formula $P(A\cup B)=P(A)+P(B)-P(A\cap B)$ — universal (no independence/exclusion needed). The $-P(A\cap B)$ removes double-counting. 4.7
Multiplication formula $P(A\cap B)=P(A)\cdot P(B\mid A)=P(B)\cdot P(A\mid B)$. For independent: $=P(A)P(B)$. 4.7

2. Core probability rules — comparison table

Rule Always true If mutually exclusive If independent
Union $P(A\cup B)=P(A)+P(B)-P(A\cap B)$ $P(A\cup B)=P(A)+P(B)$ $P(A\cup B)=P(A)+P(B)-P(A)P(B)$
Intersection $P(A\cap B)=P(A)P(B\mid A)$ $P(A\cap B)=0$ $P(A\cap B)=P(A)P(B)$
Complement $P(A')=1-P(A)$ — —
Conditional $P(B\mid A)=\dfrac{P(A\cap B)}{P(A)}$ $P(B\mid A)=0$ $P(B\mid A)=P(B)$
Independence test Check $P(A\cap B)\stackrel{?}{=}P(A)P(B)$ always 0 — independence usually fails passes by definition

3. Venn-diagram regions — set algebra cheat sheet

Region (English) Set notation Formula
Both $A$ and $B$ $A\cap B$ given or product/multiplication formula
Either $A$ or $B$ (or both) $A\cup B$ $P(A)+P(B)-P(A\cap B)$
$A$ only (not $B$) $A\cap B'$ $P(A)-P(A\cap B)$
Neither $A$ nor $B$ $A'\cap B' = (A\cup B)'$ $1-P(A\cup B)$
Exactly one of $A,B$ $(A\cap B')\cup(A'\cap B)$ $P(A)+P(B)-2P(A\cap B)$
At least one of $A,B$ $A\cup B$ $1-P(A'\cap B')$
At most one of $A,B$ $(A\cap B)'$ $1-P(A\cap B)$
Complement of $A$ $A'$ $1-P(A)$

Filling Venn diagrams: start from the innermost intersection ($A\cap B\cap C$) and work outward, subtracting overlapping pieces.

4. Conditional probability — three views

View Formula Setup
Definition $P(B\mid A)=\dfrac{P(A\cap B)}{P(A)}$ Two events $A,B$
From two-way table $P(B\mid A)=\dfrac{n(A\cap B)}{n(A)}$ Restrict to row/column $A$
From Venn diagram $P(B\mid A)=\dfrac{\text{prob in }A\cap B}{\text{prob in }A}$ Zoom into circle $A$
From tree diagram $P(B\mid A) = $ probability on the second-tier branch from $A$ to $B$ Branches encode conditionals
Independence shortcut $P(B\mid A)=P(B)$ Iff independent

Multiplication law (rearranged): $P(A\cap B)=P(A)\cdot P(B\mid A)$ — this is the rule on tree diagrams.

5. Tree diagram — workflow

Step Action
1 Identify the two (or more) experiments and their outcomes
2 First-tier branches: write $P(A)$ and $P(A')$ on the two branches; they sum to 1
3 Second-tier: write the conditional probabilities $P(B\mid A),P(B'\mid A),P(B\mid A'),P(B'\mid A')$
4 Each path's probability = product of probabilities on that path (multiplication law)
5 For a compound event covering multiple paths: add path probabilities (mutually exclusive)
6 Cross-check: all path probabilities sum to 1

Common compound events on a tree:

Compound event How to compute
Exactly one occurs $P(A)P(B'\mid A)+P(A')P(B\mid A')$
At least one $1-P(A')P(B'\mid A')$
Both $P(A)P(B\mid A)$
None $P(A')P(B'\mid A')$

6. Two-way tables — reading conditional probabilities

A row/column total restricts the sample space.

Quantity wanted Restrict to Numerator Denominator
$P(A)$ All $n(A)$ grand total
$P(A\cap B)$ All $n(A\cap B)$ grand total
$P(B\mid A)$ Row/col $A$ $n(A\cap B)$ $n(A)$ (the row/col total)
$P(A\mid B)$ Row/col $B$ $n(A\cap B)$ $n(B)$
$P(A'\cap B')$ All $n(A'\cap B')$ (corner cell) grand total

Filling a two-way table: rows and columns must sum correctly to their totals; use this to back out missing entries.

7. Independence — three equivalent tests

A pair of events $A,B$ is independent iff any one of the following holds (all three are equivalent):

Test Condition When to apply
Multiplication $P(A\cap B)=P(A)\cdot P(B)$ When you have $P(A),P(B),P(A\cap B)$
Conditional unchanged $P(A\mid B)=P(A)$ When conditional is given
Symmetric conditional $P(B\mid A)=P(B)$ When the other conditional is given

Don't confuse with mutually exclusive! Mutually exclusive events with positive probability cannot be independent (one occurring kills the other entirely).

8. Worked archetypes

# Setup Method
1 Two fair 4-spinners; sum = 5. Sample-space grid: 4 of 16 outcomes ⇒ $P=1/4$
2 Mutually exclusive: $P(A)=0.2,P(B)=0.4$. Find $P(A\cup B)$. $0.2+0.4=0.6$
3 Independent: $P(A)=1/3,P(B)=1/5$. Find $P(A\cap B)$. $1/3 \cdot 1/5 = 1/15$
4 Addition formula given $P(A)=0.6,P(B)=0.7,P(A\cup B)=0.9$. Find $P(A\cap B)$. $0.6+0.7-0.9 = 0.4$
5 Conditional: $P(C)=0.2,P(D)=0.6,P(C\mid D)=0.3$. Find $P(C\cap D),P(D\mid C)$. $0.3\cdot 0.6=0.18$; then $0.18/0.2=0.9$
6 Test independence given $P(A)=7/30,P(B)=19/30,P(A\cap B)=4/30$. $P(A)P(B)=133/900$; $P(A\cap B)=4/30=120/900$ ⇒ NOT independent
7 Tree: probability of "wins speedboat" with 3 attempts having decreasing miss probabilities. $1-P(\text{all miss})$
8 Two-way table: 75 students, 25 humanities-only, 37 science-only, 11 both. Find $P(S\mid H)$. Restrict to $H$ column (36 students): $11/36$

9. Common exam pitfalls

Pitfall Fix
Using $P(A\cap B)=P(A)P(B)$ when independence isn't established Default to addition formula; only use product when independence is stated or proved
Confusing mutually exclusive with independent Mutually exclusive ⇒ $P(A\cap B)=0$; Independent ⇒ $P(A\cap B)=P(A)P(B)$. Different concepts
Not subtracting the intersection in $P(A\cup B)$ $P(A\cup B)=P(A)+P(B)-P(A\cap B)$, NOT just $P(A)+P(B)$ unless ME
Reversing conditional: $P(A\mid B)$ vs $P(B\mid A)$ $P(A\mid B)=\dfrac{P(A\cap B)}{P(B)}$; check the denominator matches the given event
Filling Venn diagram from outside in Always start from innermost intersection $A\cap B\cap C$ and subtract outward
For tree diagrams, putting unconditional probabilities on the second tier Second-tier branches must be conditional on the first-tier outcome
$P(A')$ = $-P(A)$ $P(A')=1-P(A)$. Probability is never negative
Saying "$A$ and $B$ are independent" without testing Show $P(A\cap B)=P(A)P(B)$ explicitly
Mixing $n(A)$ (count) with $P(A)$ (probability) $P(A)=n(A)/n(\mathscr{E})$. Both notations valid in Venn diagrams; pick one

10. Big picture (30-second mental model)

Probability is set arithmetic with a normalisation ($P(\mathscr{E})=1$):

  1. Set algebra ($\cap,\cup,'$) translates English ("and / or / not") into formulas.
  2. Three workhorse rules: addition (with $-P(A\cap B)$ to fix double-counting), multiplication ($P(A\cap B)=P(A)P(B\mid A)$), and complement ($P(A')=1-P(A)$).
  3. Special cases simplify: mutually exclusive ⇒ $P(A\cap B)=0$ (drop the subtraction); independent ⇒ $P(A\cap B)=P(A)P(B)$ (no conditioning).
  4. Conditional = restricted sample space: $P(B\mid A)$ asks "what fraction of $A$'s outcomes are also in $B$?"
  5. Three diagrams, one job each: Venn (for $\cap,\cup$), two-way table (for conditional), tree (for sequential).

Three rules, three diagrams — that's the chapter.
