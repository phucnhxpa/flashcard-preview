FP1 Ch 6 — Transformations Using Matrices

The whole chapter in one mental model. Every linear transformation of the plane (one that fixes the origin and acts linearly on $x,y$) corresponds to a $2\times 2$ matrix $\mathbf{M}$ whose columns are the images of $\binom{1}{0}$ and $\binom{0}{1}$. Reflections, rotations, stretches, enlargements all sit in standard matrices — combine them by multiplication ($\mathbf{PQ}$ = "do $\mathbf{Q}$ first, then $\mathbf{P}$"), undo them with $\mathbf{M}^{-1}$, and read off the area scale factor from $\det\mathbf{M}$ (negative det → orientation flipped, i.e. a reflection is involved).

1. Jargon glossary

Term Meaning Where
Transformation A rule mapping each point $\binom{x}{y}$ in the plane to a new point $\binom{x'}{y'}$ called its image. The rule can be written $\mathbf{T}:\binom{x}{y}\mapsto\binom{f(x,y)}{g(x,y)}$. Examples include translations, reflections, rotations, stretches. 6.1
Linear transformation A transformation in which both new coordinates are linear in $x,y$: i.e. of the form $x'=ax+by,\ y'=cx+dy$ — no constants, no $x^2,y^2,xy,\sin x$ etc. Two equivalent properties: (i) origin maps to origin; (ii) $\mathbf{T}(k\mathbf{u}+\mathbf{v})=k\mathbf{T}\mathbf{u}+\mathbf{T}\mathbf{v}$ (preserves linear combinations). 6.1
Translation (NOT linear) Mapping $\binom{x}{y}\mapsto\binom{x+a}{y+b}$ with $a,b\neq 0$. Origin moves to $(a,b)$, so it's not linear and can't be represented by a $2\times 2$ matrix on column vectors. 6.1
Matrix of a linear transformation The unique $2\times 2$ matrix $\mathbf{M}=\begin{pmatrix}a&b\\c&d\end{pmatrix}$ such that $\mathbf{T}\binom{x}{y}=\mathbf{M}\binom{x}{y}$. The first column is $\mathbf{T}\binom{1}{0}$, second column is $\mathbf{T}\binom{0}{1}$. So you build the matrix by tracking where the unit vectors go. 6.1
Image of a shape After applying the transformation to every vertex (and thus every point) of a shape, the new shape is the image, often denoted $S'$ or $T'$. To find image vertices: multiply the matrix by each vertex written as a column vector. 6.1
Invariant point A point that maps to itself: $\mathbf{T}P=P$. The origin is always invariant under any linear transformation. The full set of invariant points is found by solving $(\mathbf{M}-\mathbf{I})\binom{x}{y}=\binom{0}{0}$. 6.2
Invariant line A line that maps to itself as a set (each point on the line goes to some point still on the same line — not necessarily the same point). E.g. for reflection in the $y$-axis, every horizontal line $y=k$ is invariant: it maps onto itself, but individual points slide along it. 6.2
Reflection matrices (the four standard) $y$-axis: $\begin{pmatrix}-1&0\\0&1\end{pmatrix}$. $x$-axis: $\begin{pmatrix}1&0\\0&-1\end{pmatrix}$. Line $y=x$: $\begin{pmatrix}0&1\\1&0\end{pmatrix}$. Line $y=-x$: $\begin{pmatrix}0&-1\\-1&0\end{pmatrix}$. Each has $\det=-1$. Each is self-inverse: $\mathbf{R}^2=\mathbf{I}$. 6.2
Rotation matrix $R_\theta$ $R_\theta=\begin{pmatrix}\cos\theta&-\sin\theta\\\sin\theta&\cos\theta\end{pmatrix}$ for an anticlockwise rotation about the origin by angle $\theta$. $\det=1$ (preserves area & orientation). For clockwise rotations, replace $\theta$ by $-\theta$ (or equivalently use $360°-\theta$ anticlockwise). Composition: $R_\alpha R_\beta=R_{\alpha+\beta}$. 6.2
Stretch matrix $\begin{pmatrix}a&0\\0&b\end{pmatrix}$ — stretch by factor $a$ parallel to the $x$-axis and factor $b$ parallel to the $y$-axis. If $a=b$ this is an enlargement with scale factor $a$. Negative entries flip the corresponding axis (reflection combined with stretch). 6.3
Enlargement Special case of stretch with $a=b$. Matrix $\begin{pmatrix}k&0\\0&k\end{pmatrix}=k\mathbf{I}$. Linear scale factor $k$, area scale factor $k^2$. Negative $k$ also rotates by $180°$. 6.3
Area scale factor $\lvert\det\mathbf{M}\rvert$. Every region's area is multiplied by this factor under the transformation. If $\det\mathbf{M}<0$, orientation is reversed (a reflection-type transformation has happened); if $\det=0$, every point collapses onto a single line through the origin (singular transformation). 6.3
Successive transformations Apply $\mathbf{Q}$ first, then $\mathbf{P}$: the combined matrix is $\mathbf{PQ}$. Note the right-to-left reading order — the matrix nearer the column vector acts first. 6.4
Inverse transformation The transformation that reverses $\mathbf{T}$. Its matrix is $\mathbf{M}^{-1}$. To find the pre-image $P$ of a known image $P'$: solve $\mathbf{M}\binom{x}{y}=\binom{x'}{y'}$ by left-multiplying by $\mathbf{M}^{-1}$. Reflections and $180°$ rotations are self-inverse ($\mathbf{M}^{-1}=\mathbf{M}$). 6.5
Self-inverse matrix $\mathbf{M}\mathbf{M}=\mathbf{I}$, i.e. $\mathbf{M}^{-1}=\mathbf{M}$. Includes all reflections, $180°$ rotation, and identity. Equivalently, applying the transformation twice returns the original point. 6.5

3. The two-step recipe — building the matrix from a description

Step Action
1 Identify where $\binom{1}{0}$ goes — sketch a unit square, watch the bottom-right corner
2 Identify where $\binom{0}{1}$ goes — watch the top-left corner
3 First column = image of $\binom{1}{0}$; second column = image of $\binom{0}{1}$
4 Verify by checking a third point (e.g. $\binom{1}{1}$ should land on the sum of the columns)

Worked check: rotation by $90°$ anticlockwise: $\binom{1}{0}\to\binom{0}{1}$, $\binom{0}{1}\to\binom{-1}{0}$, so $\mathbf{M}=\begin{pmatrix}0&-1\\1&0\end{pmatrix}$. ✓

4. Finding invariant points and invariant lines

Goal Equation Solution method
Invariant points (mapped to themselves) $\mathbf{M}\binom{x}{y}=\binom{x}{y}$, i.e. $(\mathbf{M}-\mathbf{I})\binom{x}{y}=\binom{0}{0}$ If $\mathbf{M}-\mathbf{I}$ non-singular, only origin; if singular, a whole line of fixed points
Invariant lines through origin $y=mx$ Substitute $\binom{x}{mx}$, multiply by $\mathbf{M}$, set image $=\binom{x'}{mx'}$, solve for $m$ Get a quadratic-in-$m$ equation; up to two values
Invariant lines NOT through origin $y=mx+c$ Same idea but with constant — usually appears for reflections (parallels to mirror line)
Special case: stretches $x$ and $y$ axes are always invariant lines
Special case: rotations $\theta\neq 0,180°$ Only origin is invariant; no invariant lines
Special case: $180°$ rotation Every line through origin is invariant

5. Composition rules — read right to left

Construction Single matrix
Apply $\mathbf{Q}$, then $\mathbf{P}$ $\mathbf{PQ}$ (P on the left because it acts last)
Apply $\mathbf{R}$, then $\mathbf{Q}$, then $\mathbf{P}$ $\mathbf{PQR}$
Reverse a transformation $\mathbf{M}$ $\mathbf{M}^{-1}$
Apply $\mathbf{M}$ then reverse it $\mathbf{M}^{-1}\mathbf{M}=\mathbf{I}$ — back to start
Apply $\mathbf{M}$ twice (squared) $\mathbf{M}^2$
Apply $\mathbf{M}$ $n$ times $\mathbf{M}^n$ — for rotation $R_\theta$: $R_\theta^n=R_{n\theta}$

Two reflections compose to a rotation: reflecting in line $\ell_1$ then in line $\ell_2$ where the angle between them is $\alpha$ gives a rotation by $2\alpha$ about their intersection.

6. Decoding a "rotation × enlargement" matrix (Section 6.4 trick)

Given $\mathbf{M}$ that's a rotation by $\theta$ then enlargement by $k$ (or vice versa — they commute when both are about origin):

Step Action Output
1 Compute $\det\mathbf{M}$ This equals $k^2$ if pure rotation+enlargement
2 $k=\sqrt{\det\mathbf{M}}$ (positive scale factor) Extracts the enlargement
3 Divide each entry of $\mathbf{M}$ by $k$ → pure rotation matrix $\frac{1}{k}\mathbf{M}=\begin{pmatrix}\cos\theta&-\sin\theta\\\sin\theta&\cos\theta\end{pmatrix}$
4 Read $\theta$ off using both $\cos$ and $\sin$ entries Use both to fix the quadrant

# Setup Method
1 Find image of $(1,1),(3,1),(3,3),(1,3)$ under $\begin{pmatrix}-1&2\\2&1\end{pmatrix}$ Multiply each vertex; image vertices $(1,3),(-1,7),(3,9),(5,5)$
2 Identify the transformation $\mathbf{M}=\begin{pmatrix}0&-1\\-1&0\end{pmatrix}$ $\binom{1}{0}\to\binom{0}{-1}$, $\binom{0}{1}\to\binom{-1}{0}$ — reflection in $y=-x$

5 Triangle area $A$, transformed by $\mathbf{M}$ with $\det=8$ New area $=8A$
6 $\mathbf{M}=\begin{pmatrix}-2\sqrt2 & -2\sqrt2\\2\sqrt2 & -2\sqrt2\end{pmatrix}$ — rotation+enlargement $\det=16$, $k=4$, $\frac{1}{4}\mathbf{M}=\begin{pmatrix}-\sqrt2/2 & -\sqrt2/2\\\sqrt2/2 & -\sqrt2/2\end{pmatrix}$ → $\theta=135°$

8. Decision flow — pick the right approach

Spot this Do this
Linear transformation given as $\binom{x}{y}\mapsto\binom{ax+by}{cx+dy}$ Matrix is $\begin{pmatrix}a&b\\c&d\end{pmatrix}$ — read off coefficients
Description of a transformation (rotation, reflection, stretch) Use standard matrix from §2 table
Need matrix for rotation by specific $\theta$ $\begin{pmatrix}\cos\theta&-\sin\theta\\\sin\theta&\cos\theta\end{pmatrix}$
Asked for area scale factor $\lvert\det\mathbf{M}\rvert$
Asked whether orientation flipped $\det\mathbf{M}<0$ → yes, includes a reflection
Asked for invariant lines Substitute $\binom{x}{mx+c}$, equate, solve for $m,c$
Asked for the pre-image (where did the point come from?) Multiply image by $\mathbf{M}^{-1}$
Composition "do A then B" Matrix is $\mathbf{BA}$ — right-to-left
"$\mathbf{M}=\mathbf{rotation}\times\mathbf{enlargement}$, find $\theta,k$" $k=\sqrt{\det\mathbf{M}}$, then $(1/k)\mathbf{M}$ is pure rotation; read $\theta$
Squared/repeated transformation Use $\mathbf{M}^n$; for rotations $R_\theta^n=R_{n\theta}$

9. Common exam pitfalls

Pitfall Fix
Multiplying matrices in wrong order for composition "Do $\mathbf{Q}$ then $\mathbf{P}$" = $\mathbf{PQ}$ — the LAST transformation is on the LEFT
Confusing rotation direction (clockwise vs anticlockwise) Standard formula is anticlockwise; for clockwise $\theta$, use $-\theta$ or $360°-\theta$
Treating translation as a $2\times 2$ matrix transformation It's not linear — origin doesn't fix; you'd need $3\times 3$ homogeneous coordinates
Reporting $\det\mathbf{M}$ as area scale factor when it's negative Area scale factor is $\lvert\det\mathbf{M}\rvert$; sign tells you about orientation flip
Writing first column as image of $(0,1)$ First column is image of $\binom{1}{0}$, second column is image of $\binom{0}{1}$
Forgetting that reflection matrices have $\det=-1$ All four standard reflections do — useful sanity check
Forgetting reflections are self-inverse $\mathbf{R}^{-1}=\mathbf{R}$ for any reflection, simplifies inverse problems
Using rotation matrix for rotation about a point other than origin Standard formula only works for rotation about origin; for other centres, translate first
Swapping signs on reflection in $y=-x$ Correct: $\begin{pmatrix}0&-1\\-1&0\end{pmatrix}$ — both off-diagonal entries negative
Reporting answer as position vector instead of coordinates Write $(x,y)$ for coordinates; $\binom{x}{y}$ is the column vector

10. Big picture (the 30-second mental model)

  1. A linear transformation = a $2\times 2$ matrix $\mathbf{M}$, where $\mathbf{M}\binom{x}{y}$ gives the image. Built from the unit-vector images: column 1 = $\mathbf{M}\binom{1}{0}$, column 2 = $\mathbf{M}\binom{0}{1}$.
  2. Reflections, rotations, stretches each have standard matrices (Section §2). Reflections have $\det=-1$ (orientation flip), rotations have $\det=1$ (rigid), stretches have $\det=ab$ (area $\times ab$).
  3. Composition = matrix multiplication. "Do $\mathbf{Q}$ then $\mathbf{P}$" composes to $\mathbf{PQ}$ — read right to left.
  4. Inverse transformation = inverse matrix. Reflections and $180°$ rotations are self-inverse.
  5. Determinant is the universal area scale factor (sign = orientation). $\det\mathbf{M}=0$ means the transformation collapses everything onto a line.
  6. Invariant points/lines come from solving $\mathbf{M}\mathbf{v}=\mathbf{v}$ (point) or $\mathbf{M}\mathbf{v}=\lambda\mathbf{v}$ for some scalar (line direction), foreshadowing eigenvectors in FP3.

Two tables (standard matrices, area effects), one rule (right-to-left composition), one inverse (matrix inverse). That's the chapter.
