Q001: When is distance equal to the magnitude of displacement?

Answer

Only when the object travels directly along a straight line from start to finish without reversing direction. If there is any change in direction, distance > |displacement|.

Q002: An object moves away from its start, then returns to exactly the start point. Describe its displacement and distance.

Answer

Displacement = 0 (final position equals initial position). Distance > 0 (sum of outward and return path lengths). Displacement depends only on initial and final position, not the path.

Record 03

Q003: An a-t graph: acceleration starts positive, decreases through zero and becomes negative. Initial velocity u = −5 m s⁻¹. What is required for the object to reverse direction?

Answer

Reversal occurs when the cumulative signed area under a-t (Δv) brings velocity to zero. The required positive area from a-t must equal +5 m s⁻¹. Positive acceleration alone is not enough; if u is negative, the object first slows, then reverses only after the total Δv exceeds 5 m s⁻¹.

Record 04

Q004: v-t graph: velocity rises linearly from 0 to 20 m s⁻¹ in 2 s, then stays at 20 m s⁻¹ for the next 2 s. Find total distance in 4 s.

Answer

Distance = area under v-t = triangle (½×2×20) + rectangle (2×20) = 20 + 40 = 60 m. (Signed area equals distance because v never goes negative).

Record 05

Q005: Bouncing golf ball v-t graph (falling velocity positive). Which point marks the first maximum height after the bounce? Options: A (zero during contact), B, C, D.

Answer

C. A is a zero-velocity instant during floor contact, not the post-bounce apex. The first maximum height after the bounce occurs where the ball momentarily stops rising (v = 0) in free flight, which is C. Never pick v = 0 blindly; identify the physical phase before and after the point.

Record 06

Q006: Displacement-time graph with several points: which point represents the maximum (instantaneous) velocity?

Answer

The point with the steepest positive tangent (in the question, point B). Using s/t from origin gives average velocity, not instantaneous. A steep negative tangent is a large negative velocity, not the maximum positive. Always use tangent slope for instantaneous velocity.

Record 07

Q007: An object dropped from rest. Its v-t curve rises and flattens. Which statement is correct?

Answer

C: Acceleration is 9.81 m s⁻² at t = 0. At release v = 0 → drag = 0 → resultant force = weight → initial acceleration = g. At terminal velocity: drag = weight, resultant force = 0, a = 0, v constant >0. (Acceleration does not stay constant; it decreases.)
