Recall promptAn object travels away from its start and returns. What are its displacement and distance?
Concept sketch: derived from the graph method on this card.

Reveal the method

STEP 1/3Anchor positions
Set the initial position as the reference. The final position is the same reference point.
STEP 2/3Calculate displacement
Final minus initial position is zero, so the displacement vector is zero.
STEP 3/3Keep distance separate
Distance counts every piece of path travelled, so the outward and return lengths add.
Recall answer: Displacement = 0; distance is non-zero (twice the outward length if it retraces the route).
Diagnostic / redo: If you cancelled distance, redo with absolute path lengths before adding.
