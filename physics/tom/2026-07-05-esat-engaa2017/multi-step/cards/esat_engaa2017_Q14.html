Q14 · Nuclear Decay Chain — Impossible Nuclide

Question

The nuclide $^{P}_{Q}$X decays to the stable nuclide $Y$. During this process four particles are emitted: one $\alpha$-particle and three $\beta^{-}$ particles.

Which of the following is NOT a nuclide that could be formed at any stage during this process?

[1 mark]

Solution 4 steps

STEP 1/4

Strategy. Recall the two decay rules: alpha changes both $A$ and $Z$; beta changes only $Z$.

Alpha decay

$$\Delta A = -4, \qquad \Delta Z = -2$$

Beta-minus decay

$$\Delta A = 0, \qquad \Delta Z = +1$$
The one fact that cracks decay-chain questions. Only alpha decay changes the mass number. Beta leaves $A$ untouched. So in any decay chain, the mass number can only ever sit at its starting value or drop in steps of 4 — the number of alphas fixes the mass number completely. Read the mass number first, then worry about the atomic number.
Alpha: $\Delta A = -4, \Delta Z = -2$. Beta: $\Delta A = 0, \Delta Z = +1$.
STEP 2/4

Strategy. If the mass number is still $P$, the alpha has not happened yet — only betas have occurred. Betas raise $Z$, so the atomic number can only be $Q, Q+1, Q+2, Q+3$ (0 to 3 betas).

Mass number = $P$ (no alpha yet):

0 betas: $(P, \; Q)$
1 beta: $(P, \; Q+1)$ — option B
2 betas: $(P, \; Q+2)$ — option C
3 betas: $(P, \; Q+3)$ — option D

It can never be below $Q$. Betas only increase $Z$. So with mass $= P$, the atomic number must be $\ge Q$. Any option showing $(P, Z < Q)$ is impossible.
With $A = P$: reachable $Z \in \{Q, Q+1, Q+2, Q+3\}$ — options B, C, D.
STEP 3/4

Strategy. If the mass number is $P-4$, the alpha has happened ($Z$ dropped by 2), plus 0–3 betas ($Z$ up by 0–3). So $Z$ ranges from $Q-2$ to $Q+1$.

Mass number = $P-4$ (alpha done):

alpha only: $(P-4, \; Q-2)$ — option E
alpha + 1 beta: $(P-4, \; Q-1)$ — option F
alpha + 2 betas: $(P-4, \; Q)$ — option G
alpha + 3 betas: $(P-4, \; Q+1)$ — option H

After the alpha, betas push $Z$ back up. The alpha drops $Z$ by 2 (to $Q-2$), and then 0–3 betas raise it back up by 0, 1, 2, or 3. So the reachable atomic numbers after the alpha are $Q-2, Q-1, Q, Q+1$.
With $A = P-4$: reachable $Z \in \{Q-2, Q-1, Q, Q+1\}$ — options E, F, G, H.
STEP 4/4

Strategy. Scan the options. Options B, C, D ($P$ with $Q+1, Q+2, Q+3$) are all reachable before the alpha. Options E, F, G, H ($P-4$ with $Q-2, Q-1, Q, Q+1$) are all reachable after it. Option A is $(P, Q-1)$: mass still $P$ but atomic number below $Q$.

Option A is $(P, \; Q-1)$: mass still $P$ but atomic number below $Q$.

To drop the atomic number you need an alpha — but an alpha would also drop the mass to $P-4$. You cannot have a reduced atomic number and the original mass number.

The two-line reasoning Tom wrote on the board. "Atomic number is down, so you need an alpha — but you can't have an alpha and still have the same mass." That single sentence is the whole solution. Whenever a decay-chain option shows the original mass number but a lowered atomic number, it is impossible, because the only thing that lowers $Z$ (the alpha) also lowers $A$.

Sense check. Option A $(P, Q-1)$ falls outside both reachable sets ($\{Q, Q+1, Q+2, Q+3\}$ for mass $P$, and $\{Q-2, Q-1, Q, Q+1\}$ for mass $P-4$). Note: $Q-1$ is reachable but only with $A = P-4$, not with $A = P$.

$\boxed{(P, \; Q-1) \text{ is impossible}}$ — Answer A
