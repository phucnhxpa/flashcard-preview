Standalone question

For the October 2025 WMA13/01 question, f(x) = ln(x^2 + 3), x in R. State the range of f.

Concept visual - supports the stated method; it does not add a different question or solution.

Multi-step working

Step 1 - Domain is given

\(x\in\mathbb{R},\ \text{so }x^2\ge 0.\)

Step 2 - Smallest input

\(x^2+3\ge 3.\)

Step 3 - Take logs (ln increasing)

\(f(x)=\ln(x^2+3)\ge\ln 3.\)

Recall answer: Range f(x) >= ln 3, i.e. [ln 3, infinity).
