Q4 — Find $c$ so that $y = 4x + c$ is tangent to $y = 2x^2 + 8x + 3$

Question

Find the value of $c$ such that the line $y = 4x + c$ is a tangent to the curve $y = 2x^2 + 8x + 3$.

[5 marks]

Solution [5 marks] · 5 steps

STEP 1/5 M1

Strategy: a tangent has the same slope as the curve at the touch point. So I need the curve's slope function $\dfrac{dy}{dx}$.

Formula (power rule): $\dfrac{d}{dx}\!\left(x^n\right) = nx^{n-1}$.

Apply term-by-term to $y = 2x^2 + 8x + 3$:

$\;\;\dfrac{d}{dx}(2x^2) = 2 \cdot 2x^{2-1} = 4x$

$\;\;\dfrac{d}{dx}(8x) = 8 \cdot 1 \cdot x^0 = 8$

$\;\;\dfrac{d}{dx}(3) = 0$  (constant has zero slope)

$$\boxed{\;\dfrac{dy}{dx} = 4x + 8\;}$$

Why constants vanish: $3 = 3x^0$ → derivative is $0 \cdot 3 \cdot x^{-1} = 0$. The +3 shifts the curve up but doesn't change its slope at any point.
$\dfrac{dy}{dx} = 4x + 8$
From STEP 1: $\dfrac{dy}{dx} = 4x + 8$
STEP 2/5 B1

Recall: any line in the form $y = mx + k$ has gradient $m$ — literally the number multiplying $x$.

Compare $y = 4x + c$ to $y = mx + k$:

$\;\;m = 4$ (coefficient of $x$)

$\;\;k = c$ (the unknown constant we're solving for)

$$\boxed{\;\text{gradient of line} = 4\;}$$

Trap: writing $\dfrac{dy}{dx} = 4x + c$ is the classic mistake — students confuse the line's equation with its derivative. But $c$ is just a vertical shift; it cannot change the slope. Differentiating $4x+c$ would give $4 + 0 = 4$ anyway. The gradient is always just $m$.
Gradient of line $= 4$
From STEP 1–2: $\dfrac{dy}{dx} = 4x + 8$; gradient of line $= 4$
STEP 3/5 M1

Tangent condition: at the point of tangency, the curve's slope must equal the line's slope. Set $\dfrac{dy}{dx} = m$:

$$4x + 8 = 4$$

Subtract $8$ from both sides:

$$4x = -4$$

Divide by $4$:

$$\boxed{\;x = -1\;}$$

Sense check (vertex location): the parabola $2x^2+8x+3$ has vertex at $x = -\tfrac{b}{2a} = -\tfrac{8}{4} = -2$. A mildly positive tangent slope ($+4$) should touch slightly to the right of the vertex (where slopes start positive) — $x = -1$ is just one unit right of $-2$ ✓.
$x = -1$
From STEP 3: $x = -1$
STEP 4/5 A1

Why curve, not line? The tangent point lies on BOTH, but we only know the line up to the unknown $c$. The curve is fully known.

Substitute $x = -1$ into $y = 2x^2 + 8x + 3$:

$\;\;y = 2(-1)^2 + 8(-1) + 3$

$\;\;\phantom{y} = 2(1) - 8 + 3$

$\;\;\phantom{y} = 2 - 8 + 3 = -3$

$$\boxed{\;\text{Tangent point} = (-1,\, -3)\;}$$

Watch the signs: $(-1)^2 = +1$ (square is positive); $8 \times (-1) = -8$. Easy slip is to write $-2 - 8 + 3 = -7$.
Tangent point $= (-1,\, -3)$
From STEP 4: tangent point $= (-1,\, -3)$
STEP 5/5 A1

The tangent point $(-1, -3)$ also satisfies $y = 4x + c$. Plug in $x = -1$, $y = -3$:

$$-3 = 4(-1) + c$$

$$-3 = -4 + c$$

Add $4$ to both sides:

$$\boxed{\;c = 1\;}$$

Final check: the tangent line is $y = 4x + 1$. At $x = -1$: $y = -4 + 1 = -3$ ✓. Matches the tangent point we found in STEP 4.
$\boxed{\,c = 1\,}$
