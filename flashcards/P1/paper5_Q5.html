Q5 — Definite Integration with Horizontal Translation

Question

Given that $\displaystyle\int_{-1}^{1} f(x)\,dx = 6$,

(a) find $\displaystyle\int_{-1}^{1} \bigl[f(x) - 2x\bigr]\,dx$. (3 marks)

(b) Hence, or otherwise, find $\displaystyle\int_{2}^{4} f(x-3)\,dx$. (2 marks)

[5 marks total]

Part (a) — Linearity of integration [3 marks] · 2 steps

Given: $\displaystyle\int_{-1}^{1} f(x)\,dx = 6$. We need $\displaystyle\int_{-1}^{1} [f(x) - 2x]\,dx$.
STEP 1/2Split using linearity M1

Strategy: The integral of a sum (or difference) of functions can always be split into individual integrals. This is the linearity property of integration, and it is the single most important tool for handling composite integrands where one part is a known quantity and the other is directly computable. In this problem, the integrand is $f(x) - 2x$. We do not know the explicit formula for $f(x)$, but we are told the value of its definite integral over the interval $[-1, 1]$. Meanwhile, $2x$ is a simple polynomial that we can integrate directly. So the strategy is to separate the two pieces and handle each independently.

Working — apply linearity:

$$\int_{-1}^{1} \bigl[f(x) - 2x\bigr]\,dx = \int_{-1}^{1} f(x)\,dx \;-\; \int_{-1}^{1} 2x\,dx$$

The first integral on the right-hand side is given directly by the question: $\displaystyle\int_{-1}^{1} f(x)\,dx = 6$. We do not need to know anything else about $f(x)$ to use this value — the question has effectively handed it to us. The second integral is a standard polynomial integral which we will evaluate in Step 2.

Sense check: We have not yet done any calculation — all we have done is recognise that the integral of a difference is the difference of the integrals. This is always valid for definite integrals and does not require any special conditions on $f$. The linearity property works for any finite sum of integrable functions over the same limits.

So we split into integration between negative 1 and 1 for f(x), minus the integration between negative 1 and 1 for 2x separately. And for the first part, we can use the estimation from the question." This shows correct recognition of the technique — split first, then substitute the given value.
$\displaystyle\int_{-1}^{1}[f(x)-2x]\,dx = 6 - \int_{-1}^{1}2x\,dx$
↳ from STEP 1: Split integral = $6 - \int_{-1}^{1} 2x\,dx$.
STEP 2/2Evaluate $\int_{-1}^{1}2x\,dx$ (odd function shortcut) A1 A1

Strategy: We need to compute $\int_{-1}^{1} 2x\,dx$. There are two ways to see this: (1) direct computation via the power rule, or (2) recognise that $2x$ is an odd function being integrated over a symmetric interval $[-1, 1]$, which always gives zero. Both approaches confirm the same answer, and it is valuable to know both — the direct computation provides practice, while the odd-function shortcut is a time-saver on exam day. An odd function is one where $g(-x) = -g(x)$ for all $x$; examples include $x$, $x^3$, $\sin x$. When integrated over $[-a, a]$, the positive area on one side of the origin exactly cancels the negative area on the other.

Direct computation:

$\displaystyle\int_{-1}^{1} 2x\,dx = \Bigl[x^2\Bigr]_{-1}^{1}$

Evaluate at the upper limit: $(1)^2 = 1$.

Evaluate at the lower limit: $(-1)^2 = 1$.

Subtract: $1 - 1 = 0$.

Odd function confirmation: $g(x) = 2x$ satisfies $g(-x) = 2(-x) = -2x = -g(x)$. The interval $[-1, 1]$ is symmetric about $0$, so $\int_{-1}^{1} g(x)\,dx = 0$ automatically. This saves time in the exam — you can write "odd function on symmetric interval $\Rightarrow$ integral $= 0$" and move on.

Combine from Step 1:

$$\int_{-1}^{1}[f(x) - 2x]\,dx = 6 - 0 = \boxed{6}$$

Sense check: Since $2x$ contributes zero, the answer is the same as the original integral of $f(x)$. This makes geometric sense: subtracting a function whose net signed area is zero does not change the total. The answer $6$ is clean and matches the given value, which is reassuring.

Key exam insight: Whenever you see an odd function ($x$, $x^3$, $\sin x$, etc.) integrated over a symmetric interval $[-a, a]$, write $= 0$ immediately. Examiners love testing whether students spot this shortcut — it appears frequently in P1 integration questions.
Answer (a): $\boxed{\displaystyle\int_{-1}^{1}[f(x)-2x]\,dx = 6}$

Part (b) — Horizontal translation preserves area [2 marks] · 2 steps

Given: $\displaystyle\int_{-1}^{1} f(x)\,dx = 6$. Find $\displaystyle\int_{2}^{4} f(x-3)\,dx$.
STEP 1/2Recognise the horizontal translation M1

Strategy: The integrand is $f(x - 3)$, and the limits are $[2, 4]$ instead of $[-1, 1]$. We need to connect these two facts. The function $f(x - 3)$ is obtained from $f(x)$ by replacing $x$ with $x - 3$, which is a horizontal translation of the graph 3 units to the right. This is a standard transformation rule: inside the function, "$- 3$" means shift right by 3. When a graph shifts horizontally, every point on the curve moves left or right, but the shape and height of the curve are completely unchanged. The area under the curve is simply relocated, not altered in magnitude.

Checking the limits: The original integral runs from $-1$ to $1$. If we shift every $x$-value right by $3$:

$\;\;$Lower limit: $-1 + 3 = 2$ ✓

$\;\;$Upper limit: $1 + 3 = 4$ ✓

The new limits $[2, 4]$ are exactly the old limits $[-1, 1]$ shifted right by $3$. This confirms that the entire "window" of integration has moved with the curve. Since the shape is identical and the window has moved alongside it, the definite integral value must remain the same.

Sense check: If you imagine the graph of $f(x)$ drawn on a piece of tracing paper, then slide that paper $3$ units to the right, the area under the curve between the marks at $2$ and $4$ is exactly the same as the original area between $-1$ and $1$. Nothing has been stretched, compressed, or reflected — just slid sideways.

Mr Fuat (from loom, verbatim): "Because we are moving the graph horizontally, the area shouldn't change. We are translating the function in the x-direction with three units to the right. So overall, area is not going to change." This is the key geometric insight that Phuc initially missed.
$f(x-3)$ is $f(x)$ shifted right by 3; limits $[2,4]$ match $[-1,1]+3$.
↳ from STEP 1: Horizontal shift by 3; limits match perfectly.
STEP 2/2Formal substitution proof A1

Strategy: While the geometric argument from Step 1 is convincing, the exam expects a formal algebraic justification. We use a $u$-substitution to rigorously transform $\int_{2}^{4} f(x-3)\,dx$ back into $\int_{-1}^{1} f(u)\,du$. This is the standard technique for proving that horizontal translations preserve the definite integral: let $u$ equal the expression inside $f$, compute $du$, and transform the limits accordingly. The substitution is clean because the derivative of a linear expression $x - 3$ is simply $1$, so $du = dx$ with no extra factors.

Working — $u$-substitution:

Let $u = x - 3$, so $du = dx$.

When $x = 2$: $\;\;u = 2 - 3 = -1$.

When $x = 4$: $\;\;u = 4 - 3 = 1$.

$$\int_{2}^{4} f(x-3)\,dx = \int_{-1}^{1} f(u)\,du = 6$$

The dummy variable $u$ can be renamed back to $x$ — it makes no difference. The integral $\int_{-1}^{1} f(u)\,du$ is identical to $\int_{-1}^{1} f(x)\,dx$, which was given as $6$. Therefore the answer is $6$.

General rule to remember: For any horizontal translation $f(x - a)$ with appropriately shifted limits:

$$\int_{p+a}^{q+a} f(x-a)\,dx = \int_{p}^{q} f(x)\,dx$$

The area is preserved because horizontal shifting does not change the shape or height of the curve — it only relocates it along the $x$-axis.

Sense check: The answer $6$ is the same as part (a) and the original given integral. This is expected: the question is testing whether students recognise that translation preserves area. A different answer would indicate a mistake in the substitution or a misidentification of the transformation.

Answer (b): $\boxed{\displaystyle\int_{2}^{4} f(x-3)\,dx = 6}$
