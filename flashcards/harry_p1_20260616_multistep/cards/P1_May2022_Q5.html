Question

(c) Define the shaded region \(R\) bounded by \(C,\ell_1,\ell_2\), using inequalities. (9)

Final reference answer

Official answer: (a) \(12-\tfrac43(x+2)^2=-\tfrac43(x-1)(x+5)\)
\(=-\tfrac43x^2-\tfrac{16}{3}x+\tfrac{20}{3}\).
(b) \(\ell_2: y=-\tfrac54x-\tfrac{25}{4}\).
(c) \(y\ge-\tfrac54x-\tfrac{25}{4}\), \(y\ge\tfrac45x\),
\(y\le-\tfrac43x^2-\tfrac{16}{3}x+\tfrac{20}{3}\).
