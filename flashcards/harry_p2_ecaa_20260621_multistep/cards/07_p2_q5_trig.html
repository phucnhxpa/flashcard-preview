Standalone question

(i) Solve 4 tan(theta) + 5 sin(theta) = 0 for 0 < theta <= 360 degrees (5). (ii) Solve sin(x)/cos(x) + cos(x)/sin(x) = 5/cos(x) for 0 < x < pi, to 3 s.f. (4).

Concept visual - supports the stated method; it does not add a different question or solution.

Multi-step working

Step 1 - (i) Respect the domain: cos(theta) not equal to 0

\(\text{Requires }\cos\theta\ne 0.\quad 4\tan\theta+5\sin\theta=\frac{\sin\theta(4+5\cos\theta)}{\cos\theta}=0\Rightarrow\sin\theta(4+5\cos\theta)=0.\)

Step 2 - (i) Both families of roots

\(\sin\theta=0\Rightarrow\theta=180^\circ,360^\circ;\quad \cos\theta=-\tfrac45\Rightarrow\theta=143.1^\circ,\ 360^\circ-143.1^\circ=216.9^\circ.\)

Step 3 - (ii) Respect the domain: sin(x) not 0 and cos(x) not 0

\(\text{Requires }\sin x\ne 0,\ \cos x\ne 0;\ \text{multiply by the non-zero product }\sin x\cos x.\)

Step 4 - (ii) Use the Pythagorean identity

\(\sin^2 x+\cos^2 x=5\sin x\Rightarrow 1=5\sin x\Rightarrow \sin x=0.2;\quad x=0.201,\ \pi-0.201=2.94\ (\text{3 s.f., radians}).\)

Recall answer: (i) theta = 143.1, 180, 216.9, 360 degrees (requires cos theta not 0); (ii) x = 0.201, 2.94 radians (requires sin x not 0 and cos x not 0). The missing second cosine root comes from 360 - alpha, not the sine rule; keep the unrounded stored value for pi - x.
