Packet Q4 / Official WME01 Q5 — vehicle SUVAT (P–Q–R)

Question

From the official paper. Three points P, Q and R are on a horizontal road where PQR is a straight line. Q is between P and R, with \(PQ=6x\) metres and \(QR=5x\) metres. A vehicle, modelled as a particle, moves from P to Q with constant acceleration. At \(t=0\) it passes P with speed \(u\ \mathrm{m\,s^{-1}}\); at \(t=12\ \mathrm s\) it passes Q with speed \(2u\ \mathrm{m\,s^{-1}}\). Using the model, (a) show that \(x=3u\). [2 marks] As the vehicle passes Q, its acceleration changes instantaneously to \(1.5\ \mathrm{m\,s^{-2}}\). It continues with this constant acceleration and passes R with speed \(3u\ \mathrm{m\,s^{-1}}\). Using the model, (b) find the value of \(u\). [3 marks] (c) Find the distance travelled by the vehicle during the first \(14\) seconds after passing P. [4 marks]

[Total: 9 marks]

Part (a) · show x = 3u [MS reference]2 marks · official reference only

Carry-in / given: P–Q only: \(s=6x\), speeds \(u\) and \(2u\), time \(12\ \mathrm s\), constant acceleration.
STEP 1/1

Exam trigger and stopping point: when you see a multi-stage SUVAT journey, choose one positive direction and tabulate the data for each stage before selecting an equation. Status and stage data. This item was reviewed and deferred; the Tutor’s “okay too” comment is not a live derivation. The complete calculation here is OFFICIAL MS REFERENCE ONLY. On P–Q the vehicle has local data \(s=6x\), initial speed \(u\), final speed \(2u\), and time \(12\ \mathrm s\). Acceleration is constant only on this stage; no Q–R acceleration is used in part (a).

OFFICIAL MS REFERENCE ONLY — full derivation. For constant acceleration, displacement equals average speed multiplied by time: \[s=\frac{u+v}{2}t.\] Therefore \[6x=\frac{u+2u}{2}(12)=\frac{3u}{2}(12)=18u.\] Divide both sides by \(6\): \[x=3u,\] exactly the required result. This starts from the independent P–Q data, so it is not circular. An equivalent SUVAT route first obtains \(a=(2u-u)/12=u/12\) and then uses \(6x=12u+\tfrac12(u/12)12^2=18u\).

Units and interpretation. Although \(x\) is a numerical parameter in the distance \(6x\) metres and \(u\) is a numerical parameter in the speed \(u\ \mathrm{m\,s^{-1}}\), the printed units complete the dimensional statements; the relation is between their numerical values. The average speed \(3u/2\) lies between \(u\) and \(2u\), which checks the increasing-speed stage.

OFFICIAL MS REFERENCE ONLY: \(\boxed{x=3u}\). Lesson status remains REVIEWED ONLY / DEFERRED. [2 marks]
Exam trigger. For a “show that” constant-acceleration stage with both endpoint speeds and time, use average speed first. Keep P–Q separate from Q–R: the acceleration changes at Q, so there is no single-acceleration model over P–R.

Part (b) · find u [MS reference]3 marks · official reference only

Carry-in / given: \(x=3u\), \(QR=5x\), Q-speed \(2u\), R-speed \(3u\), and Q–R acceleration \(1.5\ \mathrm{m\,s^{-2}}\).
STEP 1/1

OFFICIAL MS REFERENCE ONLY — choose the Q–R model. At Q the acceleration changes instantaneously to \(1.5\ \mathrm{m\,s^{-2}}\), then stays constant on Q–R. This is a different acceleration regime from P–Q. Since Q–R gives endpoint speeds and displacement but no time, use \(v^2=u_0^2+2as\), with local initial speed \(u_0=2u\), final speed \(v=3u\), acceleration \(a=1.5\), and displacement \(s=5x\).

Substitute the official data and the part (a) result \(x=3u\): \[(3u)^2=(2u)^2+2(1.5)(5x)\] \[9u^2=4u^2+15x=4u^2+15(3u)=4u^2+45u.\] Hence \[5u^2=45u,\qquad 5u(u-9)=0.\] The vehicle passes P with speed \(u\) and later has speeds \(2u\) and \(3u\), so the intended moving model requires \(u>0\); reject \(u=0\). Therefore \[u=9\ \mathrm{m\,s^{-1}},\qquad x=3u=27.\] Check. Then \(QR=5x=135\ \mathrm m\), Q-speed is \(18\ \mathrm{m\,s^{-1}}\), and R-speed is \(27\ \mathrm{m\,s^{-1}}\). Substitution gives \(27^2-18^2=729-324=405\), while \(2as=2(1.5)(135)=405\). Both sides agree exactly.

OFFICIAL MS REFERENCE ONLY: \(\boxed{u=9\ \mathrm{m\,s^{-1}}}\). REVIEWED ONLY / DEFERRED in the lesson. [3 marks]
Trap. Do not carry a P–Q acceleration through Q. The explicit instantaneous change creates a new constant-acceleration stage. Label local initial speed as \(2u\) before selecting SUVAT, and apply the positive-speed condition when an algebraic zero root appears.

Part (c) · distance in first 14 s [MS reference]4 marks · official reference only

Carry-in / given: \(u=9\), \(x=27\), P–Q takes \(12\ \mathrm s\), and the next \(2\ \mathrm s\) use acceleration \(1.5\ \mathrm{m\,s^{-2}}\).
STEP 1/1

OFFICIAL MS REFERENCE ONLY — split at Q. The first \(14\) seconds consist of all \(12\) seconds from P to Q and then \(2\) seconds after Q. A single SUVAT equation cannot span this interval because acceleration changes at Q. From part (a), \[PQ=6x=18u.\] With \(u=9\), \[PQ=18(9)=162\ \mathrm m.\]

Let S be the position reached at absolute time \(t=14\ \mathrm s\). The local Q–S time is \(14-12=2\ \mathrm s\). On this second stage, initial speed is the Q-speed \(2u=18\ \mathrm{m\,s^{-1}}\), acceleration is \(1.5\ \mathrm{m\,s^{-2}}\), and local time is \(2\ \mathrm s\). Thus \[QS=(2u)(2)+\frac12(1.5)(2^2)=4u+3=36+3=39\ \mathrm m.\]

Add the non-overlapping stage distances: \[PS=PQ+QS=162+39=201\ \mathrm m.\] The two time widths are \(12+2=14\), so the requested endpoint is correct. As a second check, speed after two seconds beyond Q is \(18+1.5(2)=21\ \mathrm{m\,s^{-1}}\); average speed on Q–S is \((18+21)/2=19.5\), and \(19.5(2)=39\ \mathrm m\).

OFFICIAL MS REFERENCE ONLY: \(\boxed{201\ \mathrm m}\). Lesson status remains REVIEWED ONLY / DEFERRED. [4 marks]
Transfer trigger. Mark the absolute target time, convert it to local time after each transition, calculate each stage with its own acceleration, and sum each displacement once. Here the decisive conversion is \(14-12=2\) seconds after Q.
