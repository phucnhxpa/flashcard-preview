Q6 — Rectangular Hyperbola: $xy = 20$

Question

Rectangular hyperbola $xy = 20$ with general point $P(2t\sqrt{a}, \frac{2\sqrt{a}}{t})$. (a) Find $a$. (b) Show normal equation. (c) Find possible values of $C$ for intersection.

[12 marks]

Solution 4 steps

STEP 1/4

Part (a) — Find $a$ from parametric form:

Rectangular hyperbola $xy = 20$, so $c^2 = 20$.

Parametric form: $(ct, c/t)$. Given $P = (2t\sqrt{a}, \frac{2\sqrt{a}}{t})$.

Compare: $c = 2\sqrt{a}$, so $c^2 = 4a = 20$.

$\boxed{a = 5}$, so $c = 2\sqrt{5}$
↳ from STEP 1:$c = 2\sqrt{5}$
STEP 2/4

Part (b) — Derive the normal equation:

1. Differentiate $xy = 20$ implicitly: $\frac{dy}{dx} = -\frac{y}{x}$

2. At $P(2t\sqrt{5}, \frac{2\sqrt{5}}{t})$: tangent gradient $= -\frac{1}{t^2}$

3. Normal gradient $= t^2$ (negative reciprocal)

4. Normal: $y - \frac{2\sqrt{5}}{t} = t^2(x - 2t\sqrt{5})$

Rect. hyperbola formulae: Tangent at $P$: $x + t^2y = 2ct$. Normal: $t^3x - ty = c(t^4 - 1)$.
Normal equation at $P$ derived ✓

Point $B$: parameter $T = -\frac{1}{2C}$, coords $(-\frac{\sqrt{5}}{C}, -4C\sqrt{5})$

Substitute $B$'s coordinates into the normal at $A$.
